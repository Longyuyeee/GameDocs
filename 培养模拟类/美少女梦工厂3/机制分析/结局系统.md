## 4. 设计特点

1. **60结局>PM2的79但更有结构**: PM3按11类别组织——比PM2的7步判定更清晰
2. **特殊结局=事件触发而非纯属性**: 10种特殊结局依赖事件链——不只是"属性到了就触发"
3. **妖精结局=8年全休息**: 最极端的"不作为"路线——192回合全部休息——对养成游戏的哲学反驳
4. **信任(Trust)=新的关系维度**: PM2用"亲密度"→PM3用"信任"——妖精与女儿的关系是"信任"而非"父爱"
