# 日程与养成系统 - 美少女梦工厂3(Princess Maker 3: Fairy Tales Come True)

> 游戏版本: Princess Maker 3: Fairy Tales Come True (PC/Steam, 1997年原版/2019年Refine版)
> 数据来源: princessmaker.fandom.com, SteamSolo Guide, Steam社区指南

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 培养周期 | **8年**(10岁→18岁) |
| 每年 | 12个月 |
| 每月 | **2阶段**(上半月/下半月 = 15天×2) |
| 总计行动回合 | 8年×12月×2 = **192回合** |
| 假期 | 2次/年(春季+夏季) |

## 2. 每月流程

```
上半月 → 安排活动(课程/打工/休息/度假)
下半月 → 安排活动
月末 → 属性结算
假期 → 春季/夏季度假(特殊结局触发)
```

## 3. 课程系统(9门 + 教会 + 减肥中心)

| 课程 | 学费 | 主要属性↑ | 副作用 | 特殊条件 |
|------|:--:|------|------|------|
| **学校** | 10G | 智力/自尊/道德/魅力 | 态度↓(傲慢时) | — |
| **武术** | 13G | 体力/自尊/勇气 | 态度↓ | — |
| **舞蹈** | 13G | 体力/自尊/气质/魅力 | 道德↓/态度↓ | — |
| **料理** | 10G | 活力/态度/感受 | 自尊↓ | 心情差时活力↓ |
| **音乐** | 15G | 自尊/气质/感受 | 态度↓ | — |
| **绘画** | 15G | 智力/感受/自尊 | 活力↓(心情差) | — |
| **礼仪** | 15G | 道德/气质/魅力 | 自尊↓/态度↓ | — |
| **教会** | 3G | 智力/道德/气质 | 魅力↓/态度↓ | 最便宜的课程 |
| **减肥中心** | 5G | 魅力↑ | 体力↓/体重↓ | 减重专用 |

> 任一课程学习≥20次 → 解锁对应结局类别。192回合÷9门≈每门21次 → 高度专业化。

## 5. 心情/心态系统(Mood)

| 心态 | 触发条件 | 影响 |
|------|------|------|
| **普通** | 默认 | 全部打工可用 |
| **傲慢** | (自尊+气质) > (态度+道德)×2 | 课程降低态度; 拒绝商人服装(除黑暗) |
| **叛逆** | 道德<300 + 公式 | 不能做农场/家教/育幼院/市场 |
| **漂亮** | 魅力 > 道德×2 | 不能做矿工 |
| **贫穷** | 接近负债线 | 负面效果 |
| **生病** | 疲劳>体力或活力 | **无法行动, 全属性下降** |

## 6. 设计特点

1. **192回合=PM2的2/3密度**: 每月2次替代3次 → 更精炼
2. **20次课程=结局方向**: 比PM2的隐藏评价更透明
3. **心情系统=PM2不良化的精细化**: 6种心态替代二元状态
4. **度假=特殊结局钥匙**: 春季/夏季/黑暗三种度假各自解锁不同结局
