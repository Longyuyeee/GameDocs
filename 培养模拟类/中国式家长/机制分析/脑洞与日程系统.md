# 脑洞与日程系统 - 中国式家长(Chinese Parents)

> 游戏版本: Chinese Parents (PC/Steam, 2018年)
> 数据来源: 萌娘百科, 3loumao全成就指南, 游民星空攻略, 18183悟性公式, Bilibili 2.0抉择攻略

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 总回合 | **48回合**(出生→高考) |
| 每回合 | 脑洞挖掘→学习技能→安排日程→社交→商店→随机事件 |
| 关键节点 | 小学→初中→高中→**高考**(最终检验) |

## 2. 脑洞挖掘系统(核心机制)

### 2.1 基础规则
- 消耗**行动力**→点击消除格子→获得属性点+悟性
- 格子类型: 属性格(彩色)/悟性格/天赋格/随机格/爆炸(清周围)/洞察(揭示隐藏)/闪电(回行动力)/下一层

### 2.2 层系统
- 每4层一个"死胡同"→最优: 1→4层或5→8层一次性完成
- 层数上限受**体魄**影响(体魄低→挖不到深层)

### 2.3 悟性消耗公式
```
悟性消耗 ≈ 大系数 × (属性总和)^(-2.199)
```
- 不是线性的! 属性越高→消耗越低→鼓励集中培养→"专业化的数学奖励"

## 3. 六维属性

| 属性 | 影响 | 获取方式 |
|------|------|------|
| **智商(IQ)** | 理科/信息技术/成绩 | 脑洞+学习 |
| **情商(EQ)** | 语文/文学/社交 | 脑洞+学习 |
| **体魄** | 体育/脑洞深度上限 | 脑洞+体育 |
| **记忆力** | 英语/文科/学习效率 | 脑洞+学习 |
| **想象力** | 艺术/创造力 | 脑洞+艺术 |
| **魅力** | 特殊技能/恋爱结局 | **无法通过脑洞自然增长!** 需特定活动 |

## 4. 压力与心理创伤系统

| 阈值 | 效果 |
|:--:|------|
| 压力>100 | 心理创伤+10, 压力重置到50 |
| **心理创伤>100** | **Game Over**(失败结局) |
| 满意度>100 | 心理创伤-5 |
| 满意度>80% | 获得1次索要机会 |

## 5. 日程安排(每回合6个选择)

```
学习类(提升属性) + 娱乐类(降压力) = 6个
关键道具: 黄冈密卷(6倍刷属性)
```

## 6. 技能树(20+类别)

| 类别 | 代表技能 | 稀有度 |
|------|------|:--:|
| 普通 | 爬行/走路/单车 | 普通 |
| 学科 | 语文/数学/英语/物理 | 普通→稀有 |
| 编程 | PHP/数据库/游戏开发/算法 | 稀有→传说 |
| 经济 | 理财/价值投资 | 稀有→史诗 |

### 面子对决(Face Battle)
- 伤害=7的倍数; 稀有度越高→伤害越高
- 才艺表演: 史诗可能赢; **传说必赢**

## 9. 设计特点

1. **脑洞=养成游戏中最独特的属性获取机制**: 消除类小游戏→养成属性→打破"选项→属性"的单调
2. **心理创伤=中国的"压力锅"隐喻**: 与游戏的主题高度一致
3. **面子对决=中国式攀比的游戏化**: 家长拿孩子的成就互相比较→精准的文化观察
4. **代际传承=无限周目的驱动力**: 每周目不是"结束了"→是"下一代更好"
