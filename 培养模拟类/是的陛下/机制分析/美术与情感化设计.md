# 美术与情感化设计 - 是的陛下(Yes, Your Grace)

> 游戏版本: Yes, Your Grace v1.0 (PC/Steam, 2020年3月6日)
> 数据来源: Steam商店页, 腾讯游戏评测, XBLAFans评测, Noisy Pixel评测, Godisageek评测

## 1. 美术风格

### 1.1 整体风格
| 要素 | 特点 |
|------|------|
| 绘画风格 | **像素艺术(Pixel Art)**——复古但精致 |
| 色调 | 中世纪暗色调+暖色点缀 |
| 画面比例 | 16:9宽屏 |
| 视觉类比 | 复古PC策略游戏 + 现代像素艺术 |

### 1.2 场景设计
| 场景 | 视觉特征 |
|------|------|
| **王座厅** | 庄严的中央大厅，国王坐在王座上听取请愿 |
| **城堡内部** | 不同房间(女儿们的房间/王后的房间/地牢) |
| **探索地图** | 达文王国周边的像素地图 |
| **战斗场景** | 围城战的动态像素画面 |

### 1.3 角色设计
- 主要角色(国王/王后/三位女儿)有独特的像素立绘
- 请愿者有各具特色的像素形象
- NPC设计受**斯拉夫民俗文化**影响
- 像素风格降低了美术成本但保留了表现力

### 1.4 界面设计
- 打字机效果的文字呈现
- 叙事主要通过文字+像素画面组合
- "国王的桌面"式UI——像是在看国王的政务记录

## 2. UI/UX设计

### 2.1 界面特征
| 要素 | 说明 |
|------|------|
| 风格 | **简洁+功能性** |
| 主画面 | 王座厅(请愿阶段)/城堡(周末) |

## 4. 情感化设计

### 4.1 "国王的无力感"设计

| 设计手法 | 效果 |
|------|------|
| **资源永远不够** | 不可能帮助所有人→持续的选择焦虑 |
| **女儿必死** | 即使你是国王→你也无法保护所有人→情感冲击 |
| **每周自动存档** | 你可以反悔→但有些事情反悔也没用(女儿的死亡) |
| **请愿者的故事** | 每个请愿者带着一个故事来→你只能选择和谁的故事继续 |

### 4.2 家庭的情感锚点

| 家庭成员 | 情感角色 | 设计意图 |
|------|------|------|
| 王后 | 成年人的支柱 | "你并不孤单" |
| 洛苏莉亚 | 无法挽回的失去 | "即使国王也有限" |
| 阿萨莉亚 | 你可以做对的选择 | "有些事你还能掌控" |
| 塞达妮 | 纯真的温暖 | "值得为之战斗的存在" |

### 4.3 像素艺术的独特情感价值
- 像素艺术有一种**怀旧的、手工的温暖**
- 不是追求写实→而是追求**情感本质的传达**
- 像素的角色更"符号化"→玩家更容易投射情感

## 7. 设计特点分析

1. **像素艺术=预算约束下的品味选择**: 3人核心团队→像素比3D或高清2D更可控→同时具有艺术性和怀旧感
2. **斯拉夫民俗美学=差异化**: 不是标准的中世纪西欧→而是东欧/斯拉夫风格→视觉上的独特性
3. **打字机音效+逐字显示=叙事的仪式感**: 文字不是瞬间出现→而是像有人在打字→增加了"这是国王的记录/日志"的感觉
4. **家庭=像素角色也有温度**: 尽管是像素艺术→女儿们的对话和互动仍然动人→证明了美术风格不是情感传达的障碍
5. **底部资源四柱=始终可见的焦虑**: 四个资源条永远在屏幕上→持续的视觉提醒→"你的人民需要什么"
