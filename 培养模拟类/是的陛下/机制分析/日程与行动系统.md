# 日程与行动系统 - 是的陛下(Yes, Your Grace)

> 游戏版本: Yes, Your Grace v1.0 (PC/Steam, 2020年3月6日)
> 数据来源: Steam社区, TalkAndroid攻略, namu.wiki, entertainment14攻略, PTT, 各种评测

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 游戏周期 | **52周**(约1年) |
| 每周阶段 | **4阶段**: 请愿(Petition)→分配(Allocation)→战斗/事件→周末(Family) |
| 每周可处理请愿数 | 视资源而定，队列中有多个请愿者 |
| 游戏总时长 | 约**5小时/周目** |

## 2. 每周流程

```
Week N 流程:
  1. 请愿阶段(Throne Room) — 听取臣民请愿，决定帮助/拒绝
  2. 资源分配阶段 — 分配金币/补给用于建设、备战、家庭需求
  3. 事件/战斗阶段 — 执行将军/女巫/猎人的任务派遣
  4. 周末阶段 — 与家人互动(王后+三位女儿)
```

### 2.1 请愿阶段详解

| 参数 | 说明 |
|------|------|
| 每周请愿队列 | 多个NPC排队等待 |
| 处理方式 | 金币援助/补给援助/派遣代理人(将军/女巫/猎人)/拒绝 |
| 拒绝后果 | 民心(Contentment)下降 |
| 可跳过 | 某些请愿可暂时不处理(但有限制) |

### 2.2 资源分配阶段
- 分配金币用于: 城堡升级/女儿婚礼准备/投资(旅馆/酒馆)
- 分配补给用于: 维持军队/应对突发事件
- 选择本周雇佣的代理人(将军/女巫/猎人)

### 2.3 事件/战斗阶段
- 派遣代理人执行任务(侦查/营救/调查)
- 处理来自其他领主的消息/联盟提议
- 战斗事件(围城战/遭遇战)

### 2.4 周末阶段(家庭)
- 与王后对话(了解她的担忧/需求)
- 与三个女儿互动(了解她们的想法/愿望)
- 关键家庭剧情推进

## 3. 核心资源(四大"属性")

| 资源 | 获取方式 | 消耗方式 | 归零后果 |
|------|------|------|------|
| **金币(Gold)** | 税收/投资回报/银行贷款/联盟援助 | 帮助请愿者/城堡升级/婚礼/雇佣代理人 | — |
| **补给(Supplies)** | 将军任务/联盟援助/税收 | 维持军队/帮助请愿者 | — |
| **兵力(Army)** | 联盟联姻/帮助特定请愿者 | 战斗/将军任务 | — |
| **民心(Contentment)** | 帮助请愿者/成功的事件解决 | 拒绝请愿者/失败的事件 | **归零=游戏失败** |

> 四大资源任一归零→Game Over。这是核心的约束机制。

## 4. 代理人系统

### 4.1 三种代理人

| 代理人 | 每周雇佣费 | 解锁周 | 功能 |
|------|:--:|:--:|------|
| **将军(General)** | 金币 | 初始 | 军队调度/侦查地图/营救/军事任务 |
| **女巫(Witch)** | 金币 | 前期解锁 | 神秘调查/巫术仪式/超自然威胁 |
| **猎人(Hunter)** | 金币 | 中期解锁 | 追踪怪兽/野生动物/战斗支援 |

### 4.2 代理人行动
- 地图上的探索点(第5周解锁探索地图)
- 每个探索点消耗1周时间派遣代理人
- 代理人执行任务期间不可用于其他请愿
- 最终围城战中两个代理人会牺牲(非必然)

## 5. 联盟与外交系统(第3周解锁)

| 外交行动 | 消耗 | 可能的回报 |
|------|:--:|------|
| 发送信件 | 免费 | 建立联系 |
| 联姻(嫁女儿) | 女儿的未来 | 兵力+金币+补给 |
| 接受/拒绝同盟 | — | 获得/失去兵力支持 |

## 6. 投资系统

| 投资项目 | 金币成本 | 每周回报 |
|------|:--:|:--:|
| **旅馆(Inn)** | 早期投资 | 每周固定金币 |
| **酒馆(Tavern)** | 早期投资 | 每周固定金币(可谈判) |
| **训练场** | 中期投资 | 每周+10兵力 |

## 7. 设计特点分析

1. **52周=紧迫的时间压力**: 不是无限时间→每周的选择有紧迫感→"我现在必须做决定"
2. **请愿队列=资源分配的道德困境**: 不可能帮助所有人→被迫做出"谁值得帮"的艰难选择
3. **四资源归零=多维度失败条件**: 不是单一的血条→四个维度都需要维持→防止单一策略
4. **家庭成员=情感锚点**: 王后和女儿们不是属性数字→她们是游戏中最重要的情感驱动力
5. **代理人=有限的问题解决能力**: 每周只能雇佣有限的代理人→必须优先处理最关键的威胁
6. **与纯养成游戏的本质区别**: 没有"培养Chilia的属性"→你是在有限资源下**做决策**→国王的困境而非父亲的规划
