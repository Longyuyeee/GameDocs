# 事件与叙事系统 - 是的陛下(Yes, Your Grace)

> 游戏版本: Yes, Your Grace v1.0 (PC/Steam, 2020年3月6日)
> 数据来源: entertainment14图文攻略, Steam社区, 腾讯游戏评测, PTT, 多篇评测

## 1. 叙事结构

| 参数 | 说明 |
|------|------|
| 叙事类型 | **线性主线+分支请愿事件** |
| 章节 | 2大阶段(对抗蛮族→对抗伊沃) |
| 叙事驱动 | 时间线性推进(52周=必然到达关键节点) |
| 分支性 | 请愿选择影响部分剧情线，但**主线核心事件不可改变** |

## 2. 主线事件时间线

### 阶段一: 对抗拉多维亚蛮族(前26周)

| 大致周次 | 关键事件 |
|:--:|------|
| 第1周 | 新手引导: 第一次请愿 |
| 第3周 | 解锁联盟/外交系统 |
| 第5周 | 解锁探索地图 |
| 前10周 | 建立基础经济(投资旅馆/酒馆) |
| 第15-20周 | 蛮族威胁加剧 |
| 第20-26周 | 集结盟友→对抗蛮族决战 |

### 阶段二: 对抗伊沃国王(后26周)

| 大致周次 | 关键事件 |
|:--:|------|
| 第27-30周 | 伊沃背叛→洛苏莉亚**剧情杀**(不可避免) |
| 第31-40周 | 集结新盟友→应对伊沃的5000大军 |
| 第41-46周 | 紧张备战(资源/兵力/盟友) |
| 第47-51周 | 关键决策(审判/继承人/女儿命运) |
| 第52周 | **最终围城战**→结局判定 |

## 7. 评测指出的叙事问题

| 问题 | 表现 |
|------|------|
| **线性化** | 核心剧情不可改变(洛苏莉亚必死) |
| **固定最优解** | 有明确的"最佳选择"(不嫁二女儿/留熊/做巫术) |
| **伪选择** | 某些看似有分支的决定实际走向相同结果 |
| **难度陡增** | 后期资源压力突然加大→部分玩家挫败 |

## 8. 设计特点分析

1. **线性主线+分支请愿=电影式叙事**: 核心故事像一部电影→玩家的选择在边缘(请愿/联盟)而不在核心剧情
2. **女儿的剧情杀=情感上的必然牺牲**: 洛苏莉亚无论如何都会死→制造"即使我是国王也有无法保护的人"的无力感→这是设计意图
3. **家庭互动=国王人性的锚点**: 在冰冷的外交/战争决策之间→与女儿的互动提醒你"你是谁"→情感平衡
4. **可S/L的决策=玩家自由度**: 每周自动存档→允许玩家反悔→降低了永久决策的焦虑
5. **5小时时长=精炼的叙事体验**: 不是无限时长→52周后故事结束→紧凑的节奏
