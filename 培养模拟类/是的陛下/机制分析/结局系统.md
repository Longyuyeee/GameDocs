# 结局系统 - 是的陛下(Yes, Your Grace)

> 游戏版本: Yes, Your Grace v1.0 (PC/Steam, 2020年3月6日)
> 数据来源: entertainment14图文攻略, Steam社区, PTT

## 1. 结局总览

| 参数 | 说明 |
|------|------|
| 结局数量 | 多个(具体数字未公开)，基于关键决策组合 |
| 结局决定因素 | 家庭决策 + 联盟选择 + 最终战胜负 |
| 单周目时长 | 约5小时 |
| 失败结局 | 四资源任一归零→Game Over |

## 2. 好结局达成条件

### 2.1 家庭维度

| 条件 | 具体行动 |
|------|------|
| 二女儿自由 | **不嫁**给老国王瓦里德→让她去环游世界 |
| 小女儿安全 | **必须留宠物熊**→最终战熊杀死猎犬保护她 |
| 王后存活 | 进行巫术仪式→王后顺利生下儿子 |
| 有继承人 | 成功完成巫术仪式→获得儿子 |
| 大女儿 | 无论如何都会死(不可避免) |

### 2.2 政治/审判维度

| 条件 | 具体行动 |
|------|------|
| 审判嫁祸事件 | 选**女领主替罪**(她试图嫁祸给二女儿) |
| 管家 | 公正审判即可 |
| 贝兰(拉多维亚残兵) | **放过他**→两个民族达成和解 |

### 2.3 军事维度

| 条件 | 具体行动 |
|------|------|
| 最终围城战胜 | 兵力300+ |
| 补给充足 | 攒10箱补给 |
| 关键战术决策 | 派猎人应对地道/派女巫应对烟雾 |
| 最后的选择 | "和士兵在一起"→援军到来夹击敌军 |

### 2.4 外交维度

| 条件 | 具体行动 |
|------|------|
| 伊沃 | **杀死他**(不死则手下会煽动暴乱) |
| 拉多维亚残兵 | 放过→和解 |
| 各领主同盟 | 根据需要选择(不影响好结局核心) |

## 3. 坏结局触发条件

| 坏结局 | 触发 |
|------|------|
| 民心归零 | 持续拒绝请愿者→人民起义 |
| 金币归零 | 财政崩溃 |
| 兵力归零 | 无法应对外敌 |
| 补给归零 | 军队崩溃+人民饥饿 |
| 二女儿不幸福 | 嫁给老国王瓦里德 |
| 小女儿遇害 | 不留宠物熊→猎犬事件无人保护 |
| 王后死亡 | 不做巫术+怀孕并发症 |

## 4. 结局=决策组合(推断模型)

```
最终结局 = f(
  家庭决策(二女儿婚姻, 小女儿宠物, 巫术仪式, 审判),
  军事决策(最终战胜负, 兵力, 补给),
  外交决策(伊沃生死, 贝兰生死, 领主同盟),
  资源状态(四维资源是否归零)
)
```

## 5. 与养成类游戏的结局设计对比

| 维度 | 传统养成游戏 | 是的陛下 |
|------|------|------|
| 结局决定因素 | 属性数值+事件完成 | **关键决策的集合** |
| 多结局性质 | 不同职业/路线 | 好结局的条件组合 |
| 最优结局 | 通常存在"真结局" | **明确的好结局条件** |
| 失败状态 | 通常无Game Over | **四维资源归零=Game Over** |
| 玩家的情感目标 | "让女儿成为最好的" | **"保护我的家人和王国"** |

## 6. 结局评价

评测反映:
- 游戏的结局虽然有多分支，但核心主线固定
- 有明确的"最优解"→削弱了重玩意愿
- 洛苏莉亚无论怎样都会死→降低了"选择改变命运"的满足感
- 但结局的情感冲击力很强→即使知道"怎么做最好"，经历过程仍然感人

## 7. 设计特点分析

1. **Game Over作为失败条件**: 不同于养成游戏的"总有结局"(即使是家里蹲)→这里你真的会失败→增加了紧张感
2. **明确的好结局=攻略透明**: 社区已经总结出精确的好结局条件→没有隐藏的秘密结局→线性但清晰
3. **女儿的必死=反"养成"设计**: 养成游戏的核心承诺是"培养=改变命运"→但洛苏莉亚无论如何都会死→打破了养成游戏的隐含契约
4. **组合式结局减少了内容量感**: 不是51个不同的结局→而是好/坏的组合→5小时的游戏→符合小团队的范围控制
5. **情感结局>机制结局**: 不是"属性组合的排列组合"→而是你与家人之间关系的自然结果→情感驱动的结局设计
