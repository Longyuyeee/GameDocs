# 资源与决策系统 - 是的陛下(Yes, Your Grace)

> 游戏版本: Yes, Your Grace v1.0 (PC/Steam, 2020年3月6日)
> 数据来源: TalkAndroid攻略, namu.wiki, entertainment14图文攻略, Steam社区

注: 本游戏使用资源管理替代传统养成游戏的属性培养，用"决策"替代"日程安排"。此为适应游戏独特机制的维度调整。

## 1. 四维资源体系

### 1.1 金币(Gold)

| 参数 | 说明 |
|------|------|
| **主要获取** | 征税(受民心影响)、投资回报(旅馆/酒馆)、银行贷款、联盟援助 |
| **主要消耗** | 帮助请愿者、雇佣代理人、城堡升级、女儿婚礼(25金币底线) |
| **管理策略** | 早期投资旅馆+酒馆→每周被动收入→减轻经济压力 |
| **特殊操作** | "粉尘交易": 7金币买粉尘袋→11金币卖给艾顿领主(利润4金币/袋) |

### 1.2 补给(Supplies)

| 参数 | 说明 |
|------|------|
| **主要获取** | 将军任务收获、联盟援助、税收 |
| **主要消耗** | 维持军队、帮助请愿者(饥荒/灾难) |
| **管理策略** | 军队不吃饭衰减极慢→没钱时优先断军队的补给 |
| **关键用途** | 最终围城: 攒10箱补给=充足 |

### 1.3 兵力(Army)

| 参数 | 说明 |
|------|------|
| **主要获取** | 联盟联姻(嫁女儿)、帮助特定请愿者(难民参军) |
| **主要消耗** | 战斗损失、将军任务 |
| **新兵源** | 训练场每10周+10兵力(积少成多) |
| **最终需求** | 围城战300+兵力足够 |

### 1.4 民心(Contentment)

| 参数 | 说明 |
|------|------|
| **显示方式** | 屏幕底部两张脸(开心/不开心) |
| **提升** | 成功帮助请愿者、解决问题 |
| **下降** | 拒绝请愿者、放任问题不解决 |
| **归零后果** | **Game Over**(人民起义/革命) |
| **与税收关联** | 民心越高→税收收入越高 |

## 2. 请愿决策系统

### 2.1 决策类型

| 决策类型 | 消耗 | 回报 | 风险 |
|------|:--:|------|------|
| **金币援助** | 金币 | 民心↑(+后续回报) | 可能被骗 |
| **补给援助** | 补给 | 民心↑(+兵力/资源回报) | 可能被骗 |
| **派遣代理人** | 代理人本周不可用 | 问题解决+民心 | 代理人可能有损失 |
| **拒绝** | — | — | 民心↓+可能连锁后果 |

### 2.2 请愿者可信度判断

| 信号 | 可靠性 |
|------|:--:|
| 有明确回报承诺 | 中(部分兑现) |
| 单纯求助 | 高(纯消耗但民心回报确定) |
| 反复出现 | 高(长期投资) |
| 与主线关联 | 高(影响结局) |
| 不合理的需求 | 低(可能是骗局) |

### 2.3 请愿优先度策略

| 优先级 | 请愿者类型 | 理由 |
|:--:|------|------|
| ★★★★★ | 投资类(旅馆老板/酒馆) | 每周被动收入→长期收益 |
| ★★★★ | 兵力贡献类 | 直接影响最终战斗 |
| ★★★★ | 主线关联类 | 影响剧情走向和结局 |
| ★★★ | 民心急需类 | 防止民心过低 |
| ★★ | 纯消耗类 | 看资源情况决定 |
| ★ | 可疑/骗局类 | 审问地牢先行调查 |

## 6. 设计特点分析

1. **四维资源=多维失败条件**: 不是"HP归零→死亡"这个单一失败→4个维度任一归零都失败→强迫全面管理
2. **请愿队列=含道德权重的资源分配**: 不是"哪个收益高选哪个"→还有"谁的求助更值得同情"→机制与道德交织
3. **女儿=最强的情感杠杆**: 女儿的命运不是属性数字→她们会死/会被嫁掉/会幸福→你做出的选择有情感代价
4. **粉尘交易=系统性的"作弊"机制**: 7买11卖→玩家社区的"财富密码"→降低了经济难度但弱化了资源紧张的体验
5. **巫术仪式=超自然元素的玩法化**: 生继承人是王国延续的核心→但需要通过巫术→增加了奇幻色彩和玩法多样性
