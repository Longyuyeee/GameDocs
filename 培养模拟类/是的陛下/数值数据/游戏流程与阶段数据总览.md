# 游戏流程与阶段数据总览 - 是的陛下(Yes, Your Grace)

> 游戏版本: Yes, Your Grace v1.0 (PC/Steam, 2020年3月6日)
> 数据来源: TalkAndroid攻略, entertainment14攻略

## 一、完整流程

| 参数 | 数值 |
|------|:--:|
| 总周数 | **52周** |
| 每周阶段 | 请愿→分配→行动→家庭 |
| 主线大战 | 2场(蛮族+伊沃) |
| 单周目时长 | ~5小时 |

## 二、阶段划分

| 阶段 | 周次 | 核心目标 | 关键解锁 |
|:--|:--:|------|:--:|
| **开端** | 1-5 | 新手引导+建立经济基础 | 外交(3周)/探索地图(5周)/第一笔贷款 |
| **早期** | 6-15 | 投资+初步联盟+积蓄 | 旅馆+酒馆投资、领主态度探测 |
| **中期一** | 16-26 | 蛮族之战准备+外交深化 | 联盟选择、第一次大战 |
| **中期二** | 27-35 | 应对伊沃背叛+女儿命运 | 洛苏莉亚之死、新联盟 |
| **后期** | 36-46 | 备战+关键决策 | 女儿联姻/旅行、贝兰事件、审判 |
| **终局** | 47-52 | 最终围城+结局 | 巫术仪式、围城战 |

## 三、每周行动清单

### 每周流程

```
1. 王座厅(Throne Room)
   → 听取请愿者(队列)
   → 决定帮助/拒绝每个请愿者
   → 消耗: 金币/补给/派遣代理人
   → 获得: 民心↑/资源↑/情报

2. 资源分配(Weekly Allocation)
   → 选择本周雇佣的代理人
   → 分配金币用于投资/建设/婚礼
   → 分配补给用于军队/请愿者

3. 行动/探索(Action Phase)
   → 派遣闲置代理人去地图探索
   → 处理外交信件
   → 应对突发事件

4. 周末(Family Time)
   → 访问家庭成员
   → 推进家庭剧情
   → 关键家庭决策
```
