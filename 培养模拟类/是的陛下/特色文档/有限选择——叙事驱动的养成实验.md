# 有限选择——叙事驱动的养成实验 - 是的陛下(Yes, Your Grace)

> 游戏版本: Yes, Your Grace v1.0 (PC/Steam, 2020年3月6日)
> 数据来源: entertainment14攻略, Steam社区, 多篇评测

## 一、核心矛盾: 这是养成游戏吗?

从严格意义上说，是的陛下不是一款养成游戏——它没有属性培养、没有日程安排、没有通过重复活动提升数值。但它提出了一个值得关注的问题: **如果一个游戏以"家人的成长与命运"为情感核心，但使用完全不同的机制来表达，它属于养成品类吗?**

## 二、"有限选择"的设计

### 2.1 与传统养成的对比

| 维度 | 养成游戏 | 是的陛下 |
|------|------|------|
| 选择数量 | 极高(数百次课程/活动安排) | **有限(~20个关键决策)** |
| 选择粒度 | 微(每节课/每次活动) | 宏(嫁/不嫁、杀/放过) |
| 选择可见性 | 看得到进度(属性数字变化) | 看不到"点数"→只有后果 |
| 不可改变之事 | 几乎没有 | **洛苏莉亚必死** |
| 失败状态 | 通常无 | **四资源归零=Game Over** |

### 2.2 有限选择带来的不同体验

**养成游戏**: "我通过每一次微小的选择，逐步塑造她的未来。"
**是的陛下**: "我在关键的时刻做关键的决定，而有些事情我无法改变。"

前者是"耕耘"，后者是"抉择"——两种完全不同的情感体验。

## 三、叙事驱动的"养成"

### 3.1 没有属性，但有"成长"

传统养成: 属性从100→500→2000→结局
是的陛下: 通过一系列叙事事件→女儿们的命运被逐渐决定

在这个意义上，女儿的"成长"不是属性的累积，而是**叙事弧线的推进**。

### 3.2 洛苏莉亚"没养成"=设计上的意图

一个养成游戏的玩家预期是: "我好好培养她→她会有一个好的未来。"
洛苏莉亚的必然死亡打破了这一点: 你无法通过任何"培养"来改变她的命运。

这不是设计的失误——这是**有意的叙事设计**。它传达的信息是: "有些事情，即使你是国王，也改变不了。"

## 四、游戏的关键选择数据分析

### 4.1 真正影响结局的选择

在约150-200个请愿决策中，真正决定结局好坏的选择只有约**8-10个**:

| 选择 | 影响权重 |
|------|:--:|
| 二女儿婚姻 | ★★★★★ |
| 小女儿宠物 | ★★★★★ |
| 巫术仪式 | ★★★★★ |
| 贝兰生死 | ★★★★ |
| 伊沃生死 | ★★★★ |
| 审判选择 | ★★★★ |
| 最终战战术 | ★★★★ |
| 联盟选择 | ★★★ |
| 投资决策(旅馆/酒馆) | ★★★ |

### 4.2 大量的"伪选择"

评测指出: 许多请愿的后果不改变最终结局。它们影响的是:
- 短期资源状况
- 民心(Contentment)
- 玩家的道德感受

这些不是"无意义"的——它们提供了**沉浸感**和**道德重量**——但它们不是改变结局的变量。

## 五、5小时的设计意图

| 时长 | 体验 |
|------|------|
| 5小时 | "一口气讲完一个完整的故事" |
| 20小时(如重玩4次) | 尝试不同的关键选择 |
| 400小时(如莱尔兰德物语) | 全结局收集+全属性路线 |

5小时的简短时间意味着:
- 不需要"投入100小时"的承诺
- 适合叙事驱动的体验(像看一部电影)
- 重玩成本低→鼓励尝试不同选择
- 但重玩的新内容有限→降低了"全收集"的驱动力

## 六、设计启示

1. **"有限选择×叙事驱动"=养成品类的另一种可能**: 不是所有含"女儿"的游戏都需要属性培养→有时候"在关键时刻做出选择"本身就是有意义的养成体验
2. **必然的失去=品类的情感创新**: 养成游戏很少让玩家"必然失去"→是的陛下的洛苏莉亚是一个勇敢的设计→它打破了养成游戏的隐含承诺
3. **5小时=精炼vs单薄**: 叙事驱动的养成体验在5小时内讲完一个完整故事→比100小时的重复循环更适合故事导向的养成
4. **对采集策略的反思**: 这款游戏确实不是"标准养成游戏"→但在P2(边缘参考/跨界创新)中收录它是正确的→它展示了养成元素的另一种可能
