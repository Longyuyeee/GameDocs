# 战斗系统 - 魔女之泉R(Witch Spring R)

> 游戏版本: Witch Spring R (PC/Switch/PS5, 2023年)
> 数据来源: witch-spring.fandom.com, GameFAQs, Steam社区

## 1. 战斗类型

| 参数 | 说明 |
|------|------|
| 类型 | 回合制RPG(经典日式) |
| 视角 | 第一人称(看到敌人) |
| 触发 | 探索中遭遇→进入战斗 |
| 核心 | 训练获得的属性决定战斗力 |

## 2. 战斗系统

| 要素 | 说明 |
|------|------|
| 普通攻击 | 物理伤害(STR决定) |
| 魔法攻击 | 魔法伤害(MAG决定) |
| 技能(Skills) | 特殊技能(学习/道具获得) |
| 道具使用 | 战斗中可以使用制作的道具 |
| 宠物协助 | 召唤宠物→辅助战斗 |

## 3. 物理vs魔法Build

| Build | 核心属性 | 优势 | 劣势 |
|:--:|:--:|------|------|
| 物理型 | STR优先 | 稳定伤害/高HP | 对高防敌人弱 |
| 魔法型 | MAG优先 | 高爆发/AOE | MP消耗大 |
| 均衡型 | STR+MAG | 灵活 | 都不极致 |
| 敏捷型 | AGI优先 | 高速/暴击 | 伤害偏低 |

## 4. 宠物系统

| 宠物 | 获取方式 | 战斗功能 |
|:--:|------|:--:|
| 各类宠物 | 探索中招募/事件 | 辅助攻击/防御/治疗 |

## 5. 设计特点

1. **训练→战斗力=直接映射**: 不是"训练属性→升级"——而是"训练的每1点STR决定每次普攻伤害"
2. **回合制=策略型**: 不是动作游戏——战斗结果由属性+技能选择决定——属性准备>操作
3. **宠物=养成×RPG的交叉点**: 宠物既是养成对象也是战斗伙伴——类似月石岛的精灵
4. **道具战斗使用=制作系统的战斗价值**: 制作的高级道具可在战斗中使用——制作不只是"卖钱"
