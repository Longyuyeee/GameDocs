|------|------|:--:|
| HP | 生命值 | — |

## 7. 结局系统(9种)

| 结局 | 条件 |
|------|------|
| 连续远征 | 100天内完成, Fame≤-6 |
| 臭名昭著黑魔女 | 100天内, Fame<-6 |
| 无处可去的希望 | 最终战失败 |
| **爱给予希望** | 主线+后日谈「正义的邂逅」完成 |
| 派莓的统治 | 主线完成, Fame≥13, 跳过后日谈 |
| 和平的一天 | 主线完成, Fame<13, 跳过后日谈 |
| 超级派莓 | 击败梦魇后回家 |

> 仅"爱给予希望"和"和平的一天"解锁后日谈内容。

## 8. 设计特点

1. **训练=种田→炼金=合成→战斗=验证**: 三个系统的完美循环
2. **宠物=探索×战斗的双重资源**: 每个宠物有野外能力和战斗支援
3. **9结局=声望(Fame)的分类器**: 与人类的关系决定最终结局类型
4. **法杖进化=养成路线的可视化**: 物理/魔法/生存→三选一→你的"培养"可见
