# 团队与开发历程 — 魔女之泉R(WitchSpring R)

> 开发商: Kiwi Walks | 韩国
> 发行年份: 2023年(PC)/2024年(主机) | PC/Switch/PS5/Xbox | 团队规模: **3人**

## 1. 开发团队

| 成员 | 角色 |
|------|------|
| **Suyoung Jang (张水英)** | 导演/设计/编程/剧本 |
| 1人 | 3D建模 |
| 1人 | 市场营销/对外关系 |

Kiwi Walks成立于2015年。工作室名来自新西兰的Kiwi鸟——一种不会飞、几近灭绝的鸟类。Suyoung Jang以此比喻"付费买断制手游"，他觉得在免费游戏泛滥的市场中，买断制也"几近灭绝"。

## 2. 系列历程

| 年份 | 事件 |
|:--:|------|
| 2014年 | Suyoung Jang的第一款免费手游收入仅**2万韩元(~$12)** |
| 2015年 | 反思后决定做付费买断手游，成立Kiwi Walks |
| 2015年7月 | 魔女之泉1——**单人6个月**开发——爆红，多国付费榜第一 |
| 2016年 | 魔女之泉2 |
| 2017年 | 魔女之泉3——期间每天只睡2-3小时。系列累计**100万** |
| 2019年 | 魔女之泉4——首次雇佣3D建模师 |
| **2023年9月** | 魔女之泉R(PC)——**3人团队**3年开发——首款主机级重制 |
| 2024年8月 | PS5/Switch/Xbox版发售 |

魔女之泉1最初的原型几乎是一个**塔防游戏**——Suyoung Jang最后决定改成RPG。原计划20个结局，因单人开发限制缩减为9个。

## 5. 设计启示

1. **从$12到500万美元 = 9年的坚持** — 2014年第一款游戏收入$12 → 2024年系列累计200万份。不是一夜暴富的故事，是9年持续迭代
2. **手游→PC/主机 = 独立开发者的升级路径** — 先在移动端积累IP和粉丝基础 → 再升级到PC/主机做高品质重制
3. **Kiwi鸟的名字 = 独立开发者的自我定位** — "不会飞、几近灭绝" = 在免费游戏时代坚持做付费买断的独立开发者
4. **每天2-3小时睡眠 = 不可持续的开发模式** — WS3期间的极端工作强度不健康，WS4开始扩招是正确方向
