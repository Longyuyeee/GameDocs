# 美术、音频与情感设计 - 大正对称爱丽丝(Taisho x Alice)

> 游戏版本: Taisho x Alice Ep 1-3 + Epilogue (PC/Steam, 2019-2021)
> 数据来源: Noisy Pixel评测, VNDB, Steam社区, 知乎

## 1. 美术风格

### 1.1 整体风格
| 要素 | 特点 |
|------|------|
| 绘画风格 | **日系唯美2D** |
| 时代背景 | **大正浪漫**(1912-1926) + 西洋童话元素 |
| 视觉概念 | "镜之国"——大正年代的日本×西方童话的奇幻融合 |
| CG质量 | **精美**(评测一致赞美CG) |
| 分辨率 | 原始分辨率较低(2015年原版的限制) |

### 1.2 角色设计
| 设计要点 | 说明 |
|------|------|
| 性转童话角色 | 灰姑娘/小红帽/辉夜姬等→转化为**男性**角色 |
| 大正服饰 | 和服×西装的混搭→大正浪漫的视觉特征 |
| 立绘 | 每个角色有多套表情/服装 |
| 女主角 | 金发大小姐→视觉上突出(在日系游戏中较少见) |

### 1.3 场景设计
- 镜之国的奇幻场景
- 大正时代的街景/建筑
- 童话元素的视觉融入(城堡/森林/月宫)
- 高质量的场景CG

## 3. 音频设计

### 3.1 配乐
| 要素 | 评价 |
|------|------|
| 风格 | 古典/童话/大正浪漫 |
| 质量 | 评测赞美("beautiful, memorable") |
| 氛围 | 配合章节色调(轻松→紧张→黑暗→治愈) |

### 3.2 语音
| 角色 | 语音 |
|------|:--:|
| 所有可攻略男性角色 | ✅ 全语音 |
| **女主角** | ❌ **无语音** |
| 配角 | 部分有 |

> 女主角无语音是一个常见的批评点——在男主全语音的对比下，"她"的沉默更加明显。

### 3.3 声优表演

顶级声优阵容的表演是游戏的核心卖点之一:
- 松冈祯丞(爱丽丝)——核心角色的多层次表演
- 平川大辅/前野智昭/增田俊树/江口拓也/苍井翔太/羽多野涉

## 4. 情感化设计

### 4.1 章节间的情感递进

| 章节 | 情感色调 | 玩家情感状态 |
|:--:|:--:|------|
| Ep 1 | 轻松/幽默/甜蜜 | "这个游戏好有趣" |
| Ep 2 | 开始出现裂痕 | "等等，好像有点不对劲" |
| Ep 3 | 黑暗/心理创伤 | "这比我想象的深得多" |
| Epilogue | 真相/治愈/整合 | "原来一切都是..." |

### 4.2 认知失调驱动的叙事

评测中有一个重要的观察: "Something feels not quite right. Don't discount your instincts."

从一开始→游戏就在制造**微妙的违和感**:
- 这些男性角色的"问题"有些奇怪
- 镜之国本身不太对劲
- 魔法师的出场方式很奇怪(他总是观察)

玩家的"有哪里不对"的感觉→被Epilogue验证→这是精心设计的情感弧线。

## 5. 内容警告

后期的重度主题:
- 精神疾病(DID/解离性身份障碍)
- 自我伤害/自杀意念
- 身体畸形恐惧症
- 饮食障碍
- 性侵害(非攻略角色施加)
- 暴力/血腥

> Episode 1相对轻松→但从Episode 2开始逐渐进入黑暗的心理领域。

## 6. 设计特点分析

1. **大正浪漫×童话=独特的视觉身份**: 和服与洋装的混搭→不是标准的中世纪也不是现代→视觉上的记忆点
2. **男主视角SIDE=对称性叙事**: 女主的视角+男主的视角→"对称爱丽丝"的标题在UI层面也体现
3. **女主角无语音=争议性设计**: 评测有分歧→有人认为这加强了"代入感"，有人认为破坏了"她是一个完整角色"的感觉
4. **章节色调的递进=情感的可控释放**: 不是"从头到尾都黑暗"→而是从轻松开始→逐步深入→最后的治愈才有力量
