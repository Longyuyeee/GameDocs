# 角色与关系系统 - 大正对称爱丽丝(Taisho x Alice)

> 游戏版本: Taisho x Alice Ep 1-3 + Epilogue (PC/Steam, 2019-2021)
> 数据来源: VNDB, 知乎, PTT RomanceGame, rekowiki, Noisy Pixel, Bilibili

## 1. 女主角: 有栖百合花(Yurika)

### 1.1 角色特征

| 维度 | 特征 |
|------|------|
| 性格 | **积极主动/毒舌/机智/肉食系** |
| 与常规乙女女主的区别 | 不是"被动等攻略"→是**她先去追他** |
| 能力 | 智商情商双高、料理家务万能 |
| 背景 | 金发大小姐 |
| 语音 | **无**(与全语音的男主们形成对比) |
| 可改名 | 可→但她更像"定义好的角色"而非自插入 |

### 1.2 "拯救者"角色
与传统童话的"王子拯救公主"相反:
- **她**是拯救者
- **男性角色**是需要被拯救的对象
- "公主拯救王子"=标题"对称爱丽丝"的含义之一

## 2. 可攻略角色

### 2.1 角色一览

| 章节 | 角色 | 原型童话 | 核心创伤/问题 | 攻略难度 |
|:--:|------|------|------|:--:|
| Ep1 | **仙杜瑞拉(Cinderella)** | 灰姑娘 | 自我价值/他人期待 | ★★ |
| Ep1 | **小红帽(Red Riding Hood)** | 小红帽 | 信任/被背叛 | ★★ |
| Ep2 | **辉夜姬(Kaguya)** | 辉夜姬/竹取物语 | 自我伤害/孤独 | ★★★★★(11个BE) |
| Ep2 | **格雷特(Gretel)** | 糖果屋(汉塞尔与格蕾特) | 姐弟羁绊/被遗弃 | ★★★ |
| Ep3 | **白雪(Snow White)** | 白雪公主 | 梦境/真实/自我认同 | ★★★★ |
| Ep3 | **魔法师(Wizard)** | —(原创?) | 观察者/幕后的痛苦 | ★★★★ |
| Epilogue | **爱丽丝(Alice)** | 爱丽丝梦游仙境 | **所有创伤的源头(DID)** | —(真相线) |

### 2.2 角色=人格碎片

所有可攻略角色实际上是**爱丽丝泰尔(アリステア)**的DID分裂人格。
- 仙杜瑞拉=他想要被爱的部分
- 小红帽=他无法信任的部分
- 辉夜姬=他自我伤害的部分
- 格雷特=他被遗弃恐惧的部分
- 白雪=他无法面对现实的部分
- 魔法师=他观察自己的部分
- 爱丽丝=他的核心(需要被唤醒)

## 3. 关系发展机制

### 3.1 无"好感度数值"

| 传统养成/乙女 | 大正爱丽丝 |
|------|------|
| 好感度点数(可见或隐藏) | **无** |
| 送礼/约会增加好感 | **无**(纯对话选择) |
| 好感度达到阈值→事件触发 | **选项直接导致HE/BE** |

### 3.2 关系=叙事弧线

| 阶段 | 叙事内容 |
|:--:|------|
| 相遇 | 女主角发现"这个人有问题" |
| 深入 | 通过对话和事件→发现创伤/问题的根源 |
| 危机 | 关键时刻→你的选择决定他的命运 |
| 解决/失败 | HE=治愈/BE=伤害加深 |

## 4. 各角色攻略策略

### 4.1 仙杜瑞拉
| 策略 | 怎么做 |
|------|------|
| 态度 | **跟他作对抬杠**→不要顺从 |
| 陷阱选项 | "他是否应该结婚?"→**不能选"让他自己决定"** |
| 核心 | 帮助他找到自己的价值(而非活在他人期待中) |

### 4.2 小红帽
| 策略 | 怎么做 |
|------|------|
| 关键选项 | **对他说"原谅他"** |
| 核心 | 帮助他克服被背叛后的信任问题 |

### 4.3 辉夜姬(最难)
| 策略 | 怎么做 |
|------|------|
| BE数量 | **11个**(需要反复SL) |
| 最关键选择 | 在月都场景: **用菜刀刺向自己**(不是日记本!)→HE |
| 核心 | 极端的自我伤害倾向→需要极端的"我愿意为你"来化解 |

### 4.4 格雷特
| 策略 | 怎么做 |
|------|------|
| 关键 | **不要舍弃"宝物"**(姐弟之间的羁绊) |
| 陷阱 | 选"让他吃掉自己"→BE |
| 核心 | 被遗弃的恐惧→证明你不会放弃他 |

### 4.5 白雪
| 策略 | 怎么做 |
|------|------|
| 特点 | 整条线实际上是梦境 |
| 核心 | 自我认同(我是谁?我是真实的吗?) |

### 4.6 魔法师
| 策略 | 怎么做 |
|------|------|
| 前提 | 需要**回收前5个男主的BE**后才能完整体验 |
| 定位 | 最接近真相的角色(观察者) |

## 6. 设计特点分析

1. **女主角拯救王子=性转的养成**: 不是"她被培养"，而是**她在拯救和治愈**→养成的主动性从"被动被养成"转向了"主动去治愈"
2. **DID=养成最极端的隐喻**: 一个人的心灵破碎成7个人格→通过你的"培养"(爱/选择/理解)→整合为完整的人→这就是养成
3. **无好感度数值=纯叙事的关系**: 不需要"你提升了5点好感度"→关系发展完全通过故事和对话→最纯粹的叙事化养成
4. **辉夜姬11个BE=最难治愈的创伤**: 自我伤害是最难修复的→游戏用最多的BE反映这一点→机制本身就是叙事
