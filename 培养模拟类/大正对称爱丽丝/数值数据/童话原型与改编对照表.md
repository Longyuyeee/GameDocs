# 童话原型与改编对照表 - 大正对称爱丽丝(Taisho x Alice)

> 游戏版本: Taisho x Alice Ep 1-3 + Epilogue (PC/Steam, 2019-2021)
> 数据来源: 知乎, rekowiki, VNDB

## 一、改编总览

| 童话原型 | 原版特征 | 游戏中的改编 |
|------|------|------|
| 灰姑娘 | 女性/被动/等待拯救 | **男性/自卑/需要被看见** |
| 小红帽 | 女性/天真/被狼欺骗 | **男性/被信任的人背叛** |
| 辉夜姬 | 女性/来自月球/最终离开 | **男性/自我伤害/不相信被爱** |
| 糖果屋(格雷特) | 女性(妹妹)/被遗弃/聪明 | **男性/对姐姐的执念/被抛弃恐惧** |
| 白雪公主 | 女性/纯真/被嫉妒/沉睡 | **男性/活在梦境中/身份困惑** |

## 二、改编维度分析

### 2.1 性别翻转

| 原版 | 游戏改编 |
|------|------|
| 公主等待王子 | **王子等待公主**(来拯救他) |
| 女性是被动的 | **女性是主动的**(她去救他) |
| "从此幸福地生活在一起" | **"我治愈了你→我们共同面对真相"** |

### 2.2 "童话问题"→"心理创伤"

| 童话中的"魔法问题" | 游戏中的"心理创伤" |
|------|------|
| 灰姑娘=魔法消失 | **=自我价值感缺失**(不被看见) |
| 小红帽=被狼吃掉 | **=被信任的人背叛**(信任破碎) |
| 辉夜姬=回到月亮 | **=孤独+自我伤害**(不相信有人会留住她) |
| 糖果屋=被巫婆抓住 | **=被遗弃恐惧**(怕被放弃) |
| 白雪公主=被毒苹果毒死 | **=无法面对真实**(活在梦中) |

### 2.3 "魔法解药"→"爱的回应"

| 童话解药 | 游戏的"解药" |
|------|------|
| 灰姑娘=王子找到她 | **=你看见了他真实的自己**(不抬杠就是看不见他) |
| 小红帽=猎人杀死狼 | **=你说"我原谅你"**(打破背叛的循环) |
| 辉夜姬=她回到月亮(无解) | **=你愿意为她付出一切**(刺向自己) |

## 三、性别翻转的深层含义

### 3.1 乙女游戏的性别政治
传统乙女游戏: 女性玩家=被攻略的对象(被动)
大正爱丽丝: 女性玩家=攻略者(主动)

女主角百合花是**肉食系**——她主动出击，主动追求，主动拯救。这是乙女游戏品类中较少见的设计。

### 3.2 "公主拯救王子"的叙事颠覆
- 不是在颠覆"公主"——而是在颠覆"被动"
- 不是在颠覆"王子"——而是在赋予男性角色"脆弱"的权利
- 男性角色可以哭、可以破碎、可以需要被拯救

### 3.3 最终真相=骗过了所有人

> 所有可攻略角色都是**一个人的分裂人格**。

这个真相的意义: 你不是在拯救7个不同的男人——你是在拯救**一个破碎的人**的7个碎片。童话的"魔法"=心理学的"解离"。

## 四、改编中的不变核心

| 童话元素 | 在游戏中的保留 |
|------|------|
| 灰姑娘=被低估的人 | 仙杜瑞拉=不被看见的人 |
| 小红帽=纯真遇到危险 | 小红帽=纯真的信任被打破 |
| 辉夜姬=不属于这个世界 | 辉夜姬=不属于这个世界→自我伤害 |
| 糖果屋=兄妹/姐弟的羁绊 | 格雷特=姐弟羁绊是唯一的光 |
| 白雪=纯粹的美 | 白雪="我是真实的吗?" |
| 爱丽丝=穿越镜子 | 爱丽丝=穿过镜子进入另一个世界→进入自己的心灵 |

## 五、设计启示

1. **童话=心理剧本**: 每个经典童话都可以被解读为一个心理剧本→游戏做了这个转化
2. **性别翻转=权力翻转**: 让女主角主动→打破了乙女游戏"女性=被动"的隐含假设
3. **"魔法"→"心理学"=成年人视角的童话**: 童话是儿童的心理剧本→成年人看到的是心理创伤→游戏用成年人的视角重新讲述童话
