## 三、BE触发特征

| 特征 | 说明 |
|------|------|
| 触发时机 | **即时**(选项后立即进入) |
| 提示 | 魔法师出场举手→"这是坏结局" |
| 重复进入 | 可通过选项追踪UI避免 |
| 收集要求 | 需要全部BE+HE→全成就 |

## 四、推荐游玩顺序

```
Episode 1:
  先: 仙杜瑞拉(较轻松)
  后: 小红帽(略带黑)

Episode 2:
  先: 格雷特(中等难度)
  后: 辉夜姬(最难→11个BE)

Episode 3:
  先: 白雪
  后: 魔法师(前提: 前5人BE全回收)

Epilogue:
  爱丽丝(真相线——最后!)
```

> ⚠️ Steam上Epilogue在字母排序中可能出现在Episode之前→不要误先玩!
