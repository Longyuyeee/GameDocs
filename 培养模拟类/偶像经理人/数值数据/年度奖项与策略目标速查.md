# 年度奖项与策略目标速查 - 偶像经理人(Idol Manager)

> 游戏版本: Idol Manager v1.0 (PC/Steam, 2021年7月27日)
> 数据来源: GamePretty, NGA, Steam社区

## 一、年度奖项一览

| 奖项 | 提名条件 | 获奖奖励 | 难度 |
|------|------|------|:--:|
| **最多公演** | 年公演场次≥160 | 公演收入×2 | ★★ |
| **最佳演唱会** | 热度≥120%(200%=奖项条件) | C位体力消耗-20 | ★★★★ |
| **最佳新人** | 新人知名度达3星 | 获粉速度×2 | ★★★ |
| **最活跃团体** | 年活动≥12次 | 所有活动费用-50% | ★★ |
| **最佳雇主** | 偶像高体力+高满意度 | 试镜稀有率×2 | ★★★★ |

## 二、奖项→策略路线

| 目标奖项 | 需要做什么 | 推荐的偶像/设施配置 |
|------|------|------|
| 最多公演 | **大量接写真/媒体合同** | 高体力偶像+休息室充足 |
| 最佳演唱会 | 10首歌+演唱会编排+控制事故率 | 核心团+C位体力管理+减事故卡 |
| 最佳新人 | 招募新人→快速推至3星 | 给新人大量资源+单曲C位 |
| 最活跃团体 | 全年保持高频活动 | 足够的偶像体力储备 |
| 最佳雇主 | 控制工作时长+高薪+设施完善 | SPA+医疗室+合理薪水 |

## 三、知名度升级参考

| 知名度(星) | 大致所需条件 | 解锁内容 |
|:--:|------|------|
| 0星 | 初始 | 写真合同 |
| 1星 | 一定粉丝量+单曲 | 更高级合同 |
| 2星 | 多首单曲+粉丝增长 | 高级媒体合同 |
| 3星 | 演唱会+奖项 | 顶级合同+最佳新人奖提名 |
| 4星+ | 持续成功 | 行业顶端 |

## 四、研究点数优先度

| 研究项目 | 效果 | 优先度 |
|------|------|:--:|
| 数字发售 | 无印刷成本→单曲必赚 | ★★★★★ |
| 病毒营销 | 低成本的粉丝获取 | ★★★★ |
| 新设施 | 扩展事务所功能 | ★★★ |
| 员工效率 | 各方面提升 | ★★★ |
| 其他 | — | ★★ |

## 五、偶像代际规划

| 偶像代 | 年龄范围 | 主力定位 | 策略 |
|:--:|:--:|------|------|
| 第1代 | 12-16岁入场 | 可爱路线→快速吸粉 | 3-5年后开始衰退→准备替换 |
| 第2代 | 与第1代错开2-3年 | 补充+转型 | 接手第1代衰退后的粉丝群 |
| 第3代+ | 持续招募 | 维持新鲜度 | 大器晚成型可跨代保留 |

## 六、长期目标路线图

```
初期(生存)
  → 不破产 → 攒粉丝 → 出第一张单曲
    ↓
中期(成长)
  → 稳定盈利 → 10首歌 → 第一次演唱会 → 首次年度奖项
    ↓
后期(帝国)
  → 多代偶像 → 多奖项 → 行业顶端
```
