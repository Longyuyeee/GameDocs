# 服装、外观与MOD系统 - 偶像经理人(Idol Manager)

> 游戏版本: Idol Manager v1.0 (PC/Steam, 2021年7月27日)
> 数据来源: Steam商店页, Steam社区, 多篇评测

## 1. 服装系统

### 1.1 服装类型

| 服装类别 | 获取 | 功能 |
|------|------|------|
| 日常服 | 默认/商店 | 外观变化 |
| 演出服 | 默认/商店/解锁 | 演出视觉效果 |
| 写真服装 | 商店 | 写真合同效果可能受影响 |
| 泳装/特殊 | 商店/MOD | 特定活动/季节 |

### 1.2 服装选择的影响
- 主要影响**外观**
- 特定服装搭配可能影响某些**粉丝群的吸引力**(如短裙无打底裤→核心男性粉↑)
- 服装=粉丝策略的一部分

## 2. 发型与化妆

### 2.1 化妆室功能
- 训练外表属性(可爱/帅气/性感/美丽)
- 变更偶像发型和妆容
- 外观变化=偶像品牌形象的调整

### 2.2 形象策略
| 形象定位 | 推荐属性搭配 | 目标粉丝群 |
|------|------|------|
| 可爱系 | 可爱+舞蹈 | 核心男性(青少年) |
| 性感系 | 性感+演唱 | 核心男性(成人) |
| 帅气系 | 帅气+舞蹈 | 核心女性 |
| 清纯系 | 美丽+聪明 | 核心女性+普通女性 |

## 3. Steam创意工坊(MOD系统)

### 3.1 可MOD内容

| MOD类型 | 说明 |
|------|------|
| 偶像角色模型 | 替换为其他角色(二次元/现实偶像/虚拟主播) |
| 服装 | 新服装设计 |
| 发型 | 新发型选项 |
| 音乐 | 替换游戏内音乐 |
| UI | 界面改进 |

### 3.2 MOD的生态价值
- 创意工坊极大扩展了游戏的内容量
- 玩家社区制作了大量现实偶像团体的角色包
- MOD让"养成自己喜欢的偶像"成为可能

## 4. 单曲的艺术设计

### 4.1 音乐风格选择

| 风格 | 吸引力修正 |
|------|------|
| 可爱/流行 | 核心男性(青少年)↑ |
| 前卫/实验 | 核心男性(成人)↑(?) |
| 帅气/摇滚 | 核心女性↑ |
| 抒情/治愈 | 普通女性↑ |

### 4.2 舞蹈风格
- 与音乐风格配合
- 可爱舞蹈+可爱歌曲→协同效应

## 5. 咖啡厅菜单设计
- 偶像可以设计咖啡厅菜单
- 需要**定期更新**避免审美疲劳
- 咖啡厅=被动收入+粉丝维护

## 6. 设计特点分析

1. **外观=商业策略而非自我表达**: 不同于养成游戏中"打扮女儿"的个人化→这里的服装选择是商业决策
2. **MOD=社区驱动的无限内容**: 创意工坊是这款游戏保持生命力的核心→玩家不断添加新的角色和内容
3. **音乐风格→粉丝获取的协同**: 可爱歌曲+可爱服装+可爱舞蹈→三个层面的协同→叠加粉丝获取效率
4. **俄罗斯团队做日本偶像=外观的文化翻译**: 非日本开发者对偶像"外观"的呈现→有一定程度的"文化凝视"
