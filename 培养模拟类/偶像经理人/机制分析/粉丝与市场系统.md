# 粉丝与市场系统 - 偶像经理人(Idol Manager)

> 游戏版本: Idol Manager v1.0 (PC/Steam, 2021年7月27日)
> 数据来源: GamePretty攻略, NGA攻略, PTT, Steam社区

## 1. 粉丝群体分类

### 1.1 三维度×二元=12种粉丝类型

| 维度 | 二元分类 |
|------|------|
| **忠诚度** | 核心粉(Hardcore) / 普通粉(Casual) |
| **性别** | 男性 / 女性 |
| **年龄** | 青少年(Teen) / 青年人(Young Adult) / 成人(Adult) |

### 1.2 核心粉 vs 普通粉

| 特征 | 核心粉(Hardcore) | 普通粉(Casual) |
|------|------|------|
| **获取难度** | 高 | 低 |
| **流失速度** | 慢(忠诚) | 快(易流失) |
| **购买力** | **极强**(可买超过自身人数的CD) | 一般(1人=1张) |
| **商业价值** | ★★★★★ | ★★ |
| **策略** | 集中培养→核心粉是利润引擎 | 维持基本盘 |

### 1.3 性别×年龄的消费特征

| 粉丝群 | 吸引属性 | 消费特点 |
|------|------|------|
| 核心男性×青少年 | 可爱 | 有热情但消费力有限 |
| 核心男性×青年 | 可爱+性感 | 消费力强 |
| 核心男性×成人 | 性感 | **消费力最强** |
| 核心女性×青少年 | 帅气 | 有热情 |
| 核心女性×青年 | 帅气+美丽 | 消费力强 |
| 核心女性×成人 | 美丽 | 消费力强 |
| 普通男性×各年龄 | 舞蹈+幽默 | 大规模但低消费 |
| 普通女性×各年龄 | 演唱+聪明 | 大规模但低消费 |

## 2. 属性→粉丝吸引力映射

### 2.1 直接映射

| 偶像属性 | 吸引的粉丝群 |
|------|------|
| 可爱(Cute) | 核心男性 |
| 性感(Sexy) | 核心男性 |
| 帅气(Cool) | 核心女性 |
| 美丽(Pretty) | 核心女性 |
| 舞蹈(Dance) | 普通男性 |
| 幽默(Funny) | 普通男性 |
| 演唱(Vocal) | 普通女性 |

### 2.2 叠加效应
```
偶像A: 可爱100 + 舞蹈50
  → 核心男性粉: ★★★★★ (可爱高)
  → 普通男性粉: ★★ (舞蹈中等)
  → 总粉丝=核心男性主导+普通男性辅助

偶像B: 演唱100 + 美丽80
  → 普通女性粉: ★★★★★
  → 核心女性粉: ★★★★
  → 总粉丝=普通女性主导+核心女性辅助
```

## 3. 聚焦 vs 泛化的数学

| 粉丝策略 | 优点 | 缺点 |
|------|------|------|
| **聚焦**(单一粉丝群) | 该群满意度极高→购买力集中→收入最大化 | 粉丝群单一→抗风险能力低(如丑闻打击该群) |
| **泛化**(多种粉丝群) | 风险分散→稳定 | 每种都不够满意→整体收入低 |
| **推荐策略** | **聚焦1-2个最有价值的粉丝群**(如核心男性成人) | — |

## 4. 粉丝获取与维持

### 4.1 获取方式

| 方式 | 效率 | 成本 |
|------|:--:|:--:|
| 单曲发行 | ★★★★★ | 高(制作+营销) |
| 写真合同 | ★★★ | 低 |
| 宣传(Daily) | ★★★ | 低(每日1次) |
| 剧场演出 | ★★ | 无额外成本 |
| 咖啡厅 | ★★ | 需定期更新菜单 |
| 演唱会 | ★★★★★ | 高(但收入也高) |
| 病毒营销 | ★★★★(随机) | 低-中 |

### 4.2 粉丝流失因素
- 长期无新作品→粉丝流失
- 丑闻事件→粉丝大量流失
- 偶像负面行为(如厌男特性触发)→特定粉丝群流失
- 年龄增长→可爱属性自然下降→相关粉丝群缩小

## 5. 演唱会经济学

| 参数 | 要求 |
|------|:--:|
| 最低曲目 | **10首单曲** |
| 上座率底线 | **≥60%**(低于则亏本) |
| 票价策略 | 聚焦核心粉→票价可拉满(¥100,000/张) |

### 演唱会小游戏
- 最多10个节目(单曲+休息交替)
- 事故率>30%→用减事故率卡
- **最好的节目排最后**(最后失败=热度-10%)
- 热度200%→最佳演唱会奖项条件

## 6. 设计特点分析

1. **粉丝系统的数学化**: 不是模糊的"知名度"→而是二元的性别×年龄×忠诚度的精确组合→属性直接映射到具体粉丝群
2. **核心粉=利润引擎**: 1个核心粉的购买力>10个普通粉→战略必须聚焦→分散=穷人
3. **年龄=粉丝获取的自然约束**: 可爱属性随年龄衰减→你需要不断为年龄增长的偶像寻找新定位→或招募新偶像替代
4. **粉丝满意度的隐藏层**: 粉丝不只是"有"或"没有"→他们有不满意度→不满意会流失→属性下降→满意度再下降→恶性循环
5. **俄罗斯视角下的日本偶像产业**: 开发者是俄罗斯团队→他们对日本偶像产业的观察有一定的外部视角→游戏批判了产业的黑暗面(剥削/年龄歧视/性化)
