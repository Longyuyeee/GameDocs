# 经营与经济系统 - 偶像经理人(Idol Manager)

> 游戏版本: Idol Manager v1.0 (PC/Steam, 2021年7月27日)
> 数据来源: GamePretty攻略, NGA攻略, Bilibili攻略, PTT, Steam社区

## 1. 经济体系总览

| 收入来源 | 利润率 | 稳定性 | 阶段 |
|------|:--:|:--:|:--:|
| **写真合同** | ★★★★ | ★★★★★ | 前-中期主力 |
| **单曲销售** | ★★★★★ | ★★★★ | 中期起主力 |
| **演唱会** | ★★★★★ | ★★★ | 中后期(需10首歌+≥60%上座率) |
| **剧场/咖啡厅** | ★★★ | ★★★★★ | 全期(稳定被动收入) |
| **媒体合同** | ★★★ | ★★★ | 全期(提升知名度) |
| **巡演** | ★★ | ★★ | 后期(基本亏钱赚名声) |

## 2. 成本结构

### 2.1 固定成本

| 成本项 | 大致/周 | 节约策略 |
|------|:--:|------|
| **楼层租金** | 中-高(除经理楼层) | 免费层(经理层)尽量多放设施 |
| **偶像薪水** | 浮动 | 0星偶像压到1元→满意度仍200% |
| **员工工资** | 低-高(按等级) | 初期全用新手级(周薪最低) |
| **设施维护** | 低 | — |

### 2.2 可变成本

| 成本项 | 说明 |
|------|------|
| 单曲制作(作词/作曲/编曲/编舞/营销) | 前期出单曲=必然亏本(无粉丝基础) |
| 试镜费用 | 全国选拔最贵但SSR概率最高 |
| 营销推广 | 与单曲绑定 |
| 公关危机处理 | 不可预期 |

## 3. 设施系统

### 3.1 设施一览

| 设施 | 占用格子 | 功能 | 优先级 |
|------|:--:|------|:--:|
| **休息室** | 1 | 偶像每日+5体力回复 | ★★★★★(开局必建×2) |
| **化妆室** | 1 | 训练外表属性(可爱/帅气/性感/美丽) | ★★★★ |
| **舞蹈室** | 2 | 训练Dance | ★★★ |
| **录音室** | 2 | 训练Vocal | ★★★ |
| **剧场** | — | 稳定收入+训练Funny/Smart | ★★★★ |
| **咖啡厅** | — | 偶像设计菜单→稳定收入 | ★★★ |
| **医疗室** | 1 | 回复体力+精神力 | ★★(后期体力短缺时重要) |
| **SPA** | — | 最多5名偶像3天恢复体力 | ★★★ |

### 3.2 布局策略
- **经理层(免租金)**: 尽量塞满→舞蹈室+录音室(最大格但最贵)
- **付费层**: 化妆室+休息室(1格→灵活)
- 合理规划减少租金→前期生存关键

## 4. 单曲经济学

### 4.1 制作流程
```
作词 → 作曲 → 编曲 → 编舞 → 市场营销(选择策略)
```

### 4.2 营销策略

| 策略 | 成本 | 效果 | 风险 |
|------|:--:|------|:--:|
| **数字发售** | 低(无印刷成本) | 几乎不会亏本 | 无 |
| 个人握手会 | 中 | 硬核粉丝销量↑(附赠握手券) | 低 |
| 色情PV | 中 | 男性粉丝+销量↑ | 成功与否有随机性 |
| 病毒营销 | 低-中 | 网络传播↑ | 受Meme Queen特性影响 |

### 4.3 出单曲时机
- ❌ 0星知名度时: 无粉丝基础→必然亏本
- ✅ 有一定粉丝基础+知名度升星后: 开始盈利
- ✅ 优先数字发售(零印刷成本)→试水→再加大投入

## 5. 融资系统

### 5.1 投资人

| 投资人 | 条件 | 特点 |
|------|------|------|
| **不二(Fujimoto)** | 随时可借 | **无利息!** 前期救命稻草 |
| 其他 | — | 可能有利息 |

### 5.2 借款策略
- 借不二的→最长还款期(6个月)→投资设施
- 第一笔贷款=免息=白送的启动资金
- 适度负债→快速扩张→收入增长→还款

## 6. 员工系统

| 等级 | 周薪 | 能力 | 使用阶段 |
|:--:|:--:|:--:|:--:|
| 新手(Novice) | 低(~2万) | 基础 | 初期首选 |
| 职业(Pro) | 中 | 中等 | 中期(有稳定收入后) |
| 专家(Expert) | 高 | 顶级 | 后期(自动化管理) |

### 员工类型
- **商务部长**: 接写真/媒体合同(前期必雇)
- **研究部长**: 攒研究点数→解锁新功能
- **其他**: 根据需求

## 7. 各阶段经营策略

### 初期(破局期)
```
1. 建休息室×2(经理免费层)
2. 偶像薪水压到1元
3. 不出单曲→只接写真合同攒钱
4. 雇1-2名新手级商务部长
5. 向不二借款→最长还款期
6. 控制偶像数≤5人
目标: 不破产+攒初始资金
```

### 中期(增长期)
```
1. 开始出数字单曲(有粉丝基础后)
2. 逐步提升偶像知名度→升星
3. 升星后涨薪水(维持满意度)
4. 攒10首歌→准备演唱会
目标: 建立稳定收入流
```

### 后期(扩张期)
```
1. 演唱会=核心盈利
2. 全国选拔招募SSR偶像
3. 专家级员工→自动化运营
4. 冲击年度奖项
目标: 偶像帝国
```
