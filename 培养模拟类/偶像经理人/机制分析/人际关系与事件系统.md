# 人际关系与事件系统 - 偶像经理人(Idol Manager)

> 游戏版本: Idol Manager v1.0 (PC/Steam, 2021年7月27日)
> 数据来源: GamePretty攻略, Steam社区, 豆瓣评测, 贴吧, NGA

## 1. 偶像间关系

### 1.1 关系类型

| 关系 | 效果 | 管理方式 |
|------|------|------|
| **挚友** | 正面(互相支持) | 维持 |
| **朋友** | 中性偏正面 | — |
| **霸凌者/被霸凌者** | **严重负面** | 需要经理人介入→消耗影响力解决 |
| **竞争者** | 负面(可能影响团队协作) | 调解 |

### 1.2 关系侦查
- 需要**常与偶像聊天**获取内部情报
- 关系不是公开可见的→必须主动探索
- 忽视内部关系→霸凌蔓延→团内瓦解

## 2. 经理人与偶像的关系

### 2.1 好感度系统

| 好感度等级 | 解锁内容 |
|:--:|------|
| 低 | 基础互动 |
| 中 | 个人故事 |
| 高 | **恋爱可能**(成年偶像) |
| 表白 | 好感度不足→关系倒退 |

### 2.2 恋爱规则
- **不能与未成年偶像恋爱**
- 表白前需确保好感度够高→否则关系倒退
- 结婚后→不再有调情选项
- 恋爱关系可能影响事务所运营(利益冲突)

### 2.3 影响力系统
- 通过满足偶像愿望获得
- 消耗影响力解决团内问题(霸凌/争端)
- 影响力=经理人的"政治资本"

## 3. 随机事件/公关危机

### 3.1 危机类型

| 危机 | 严重性 | 处理难度 |
|------|:--:|:--:|
| 偶像与议员出轨 | ★★★★★ | 极高 |
| 与男团成员约会被拍 | ★★★★ | 高 |
| 小号被扒(不当言论) | ★★★ | 中 |
| 粉丝握手会事故 | ★★ | 低 |
| 单曲抄袭指控 | ★★★ | 中 |

### 3.2 处理方式
- 道歉/否认/沉默→不同选择影响舆论走向
- 消耗资源(金币/时间/影响力)
- 影响与当事人的关系
- 处理得当→危机变机遇(话题度↑) / 处理失当→粉丝流失+偶像离队

### 3.3 丑闻点(Scandal Points)机制
- 丑闻累积→团体有"丑闻点"
- 特性「忧虑症(Anxiety)」: 有丑闻点时→全属性下降→连锁反应
- 需要主动管理(公关/道歉/冷却)

## 4. 偶像愿望系统

| 愿望类型 | 满足方式 | 奖励 |
|------|------|------|
| 个人目标 | 达成特定条件 | 影响力+好感度 |
| 团体目标 | 团队成就 | 全员满意度↑ |
| 职业发展 | C位/个人单曲 | 忠诚度↑ |

## 5. 总选举系统

- 类似AKB48的年度人气投票
- 经理人可**"幕后操作"排名**(分配资源影响结果)
- 排名影响偶像士气+粉丝活跃度
- 是重要的年度事件

## 6. 设计特点分析

1. **霸凌机制=对偶像产业黑暗面的模拟**: 不是所有关系都是正面的→团内霸凌是真实存在的→游戏不做美化
2. **经理人恋爱=权力不对等关系的呈现**: 经理人(雇主)与偶像(雇员)的恋爱→内在的权力不对等→游戏没有回避这个问题
3. **公关危机=偶像产业的"潜规则"暴露**: 出轨/小号/抄袭→这些是真实偶像产业中会发生的→游戏让玩家体验"如何处理"
4. **愿望系统=偶像不是工具**: 偶像有个人愿望→满足她们的愿望→她们不只是"赚钱机器"→增加了人性化
5. **俄罗斯开发者的"外人视角"**: 对日本偶像产业的观察没有本地化的"美化"→更直接地展现了产业的剥削和不公
