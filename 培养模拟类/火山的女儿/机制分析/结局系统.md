# 结局系统 - 火山的女儿(Volcano Princess)

> 游戏版本: Volcano Princess v1.0+ (PC, 2023年)
> 数据来源: Steam社区指南(58全结局), cozygamingpanda, LastWordOnGaming

## 1. 结局结构

| 参数 | 数值 |
|------|:--:|
| **总结局数** | **58个** |
| 职业结局 | 50+个(7大分支) |
| 情感结局 | ~18个(独立于职业) |
| 职业+情感组合 | **独立选择**(先选职业→再触发情感) |
| 互斥机制 | 部分职业结局屏蔽婚姻(如修女/农夫/妓女) |

## 4. 结局判定优先级

```
1. 特殊结局(火山女神/魔族女王——最高优先级)
2. 黑暗值判定(黑暗值条件→黑暗分支)
3. 属性阈值判定(高星→低星排列)
4. 默认结局(无业游民——最低1★)
```

## 5. 情感结局

| 机制 | 说明 |
|------|------|
| 触发条件 | 该NPC好感度**140+(10♥)** + 完成**3个角色事件** |
| 选择规则 | 好感度最高的NPC(除Claire)决定情感结局 |
| 友情锁 | 部分NPC在较低好感度时触发"挚友"锁(如Hackett~90/Kenneth~110) |
| 屏蔽 | 部分职业结局屏蔽婚姻(修女/农夫/妓女/名媛/王妃等) |

## 6. 全结局收集策略

```
一周目: 熟悉流程→积累继承点→铁匠结婚
二周目: 刷大部分结局(最后一月存档→回档换结局)
三周目+: 全结局收集(包括高星结局)
全成就: 最少3-5周目
```

## 7. 设计特点

1. **58结局=超大分支树**: 7大分支×8学科×5星级×18情感≈极限组合数
2. **职业+情感独立**: 不是"结婚=唯一结局"——职业和恋爱是两条独立的结局线
3. **黑暗值=隐藏分支**: 光明路线(黑暗<30)vs黑暗路线(黑暗>150)——道德系统的双线结局
4. **432回合的有限资源→必然不是全结局**: 一周目只能达成组合中的一个——驱动多周目
