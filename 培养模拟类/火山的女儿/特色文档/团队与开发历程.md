## 1. 开发团队 — 养蛋人工作室 (Egg Hatcher)

### 1.1 核心成员

| 成员 | 角色 | 教育背景 | 说明 |
|:----:|:----:|:--------:|------|
| **Fry (邓鹤)** | 美术 / 动画 / 设计 | 中国传媒大学动画专业 (2019 届) | 负责全部角色的绘画、动画、UI |
| **Zoi (吴雨桐)** | 游戏设计 / 编程 | 中国传媒大学游戏设计专业 (2019 届) | 自学 Unity C#，负责全部程序实现 |

两人于 2013 年在北京的艺考培训班相识，因共同喜爱动漫（尤其是《命运/冠位指定》和《崩坏3》）结缘。工作室名称"养蛋人"来自她们最喜爱的动漫中两个角色一起养蛋的场景——这个意象暗示了她们创造与培育的理念。

### 1.2 相遇与创业: "毕业即创业"

| 时间 | 事件 |
|:----:|------|
| 2013年 | 北京艺考培训班相识（Fry 学动画、Zoi 学游戏） |
| 2015年 | 一同考入中国传媒大学 |
| 2019年 (毕业) | 48 小时 Game Jam 做出首款合作作品《大撕裂》 |
| 2019年夏 | **决定创业** — 没有去大厂找"稳定工作"，直接开始全职开发火山的女儿 |
| 2023年4月 | 游戏正式发售 |

### 1.3 Fry 与 Zoi 的分工模式

"养蛋人"是一个极简的二人团队，但分工极为明确:

| 方面 | Fry | Zoi |
|:----:|:---:|:---:|
| **美术** | 全部角色设计、CG、UI、动画 | — |
| **程序** | — | 全部 Unity C# 代码 |
| **策划** | 部分（角色/剧情/事件） | 部分（数值/系统/玩法） |
| **工具** | Photoshop、Clip Studio Paint | Unity、VS Code、Git |
| **社区维护** | 部分 | 部分 |

两人共同参与策划决策和剧情讨论。所有重要的设计决策都需要"双人共识"——这是极小型团队避免方向性错误的关键机制。

### 2.2 "4-6 次玩法大改"的放弃过程

火山的女儿在开发过程中经历了多次几乎完全推倒重来的玩法方向变更:

**版本 1 — 卡牌战斗 (2019-2020.08)**
- 核心玩法借鉴《杀戮尖塔》的爬塔式卡牌战斗
- 女儿的战斗 = 卡牌对战，养成 = 卡组构筑
- **放弃原因**: 卡牌战斗的开发量太大（需要 100+ 张卡牌 × 平衡性测试），2 人团队无法完成

**版本 2 — 回合制战斗 (2020.08-2021.01)**
- 简化为传统的回合制 RPG 战斗
- 加入小镇模拟经营作为副系统
- **放弃原因**: 小镇模拟的"经营感"与养成的"情感叙事"不匹配，两个系统的目标冲突

**版本 3 — 最终框架 (2021.07 定稿)**
- 养成主系统 + 轻量战斗辅助系统
- 战斗简化为数值对比（而非操作），重心放在"养成选择→影响数值→战斗结果"的单向因果链上
- 去除了小镇模拟等副系统，集中资源做情感叙事

Zoi 在开发者访谈中的原话是: **"放弃得越快，找到对的路越快。每放弃一次，我们就离正确的玩法近了一步。"**

### 2.3 摩点众筹 (2022)

| 维度 | 数据 |
|:----:|:----:|
| 众筹平台 | 摩点网 (中国) |
| 目标金额 | ￥100,000 |
| 实际金额 | **￥320,000** (320% 达成) |
| 支持人数 | 2,000+ 人 |
| 众筹档位 | ￥36(游戏本体) ~ ￥499(全部 DLC + 实体周边) |

众筹金额（32 万人民币）对于一款 2 人开发的游戏来说并不算大，但它的价值在于: 验证了游戏概念、建立了初始玩家群体、提供了早期的测试反馈。

### 4.3 市场成功的关键因素

1. **定价策略** — ￥35 / $5 的定价是整个市场营销中最关键的决策之一。这个价格让"尝试一下"几乎没有心理门槛，大大加速了口碑传播

2. **题材红利** — "养女儿"在中国玩家群体中是一个极其强势的题材。美少女梦工厂系列在 1990s-2000s 积累的粉丝基础 + 中国独特的"女儿情结"文化构成了天然的需求

3. **社交媒体效应** — Bilibili 上的"女儿结局"视频大量传播，形成社交货币效应:"你养出了什么结局？"

## 5. 设计启示

1. **2 个人 × 4 年 = 国产养成爆款** — 没有大厂背景、没有技术积累（Zoi 自学 Unity）、没有预算（众筹仅 32 万）——但做出了 Steam 好评率国产前三的作品。这证明了在当代游戏市场，**设计能力和情感表达比技术和资源更重要**

2. **4-6 次玩法大改 = 不怕推倒重来** — 早期版本的卡牌战斗和回合制战斗都被果断放弃。对于小团队来说，最大的风险不是"改太多"，而是"将一个不合适的方案坚持太久"

3. **￥35 定价 = 降低门槛的最大化传播** — 对于一款情感体验为核心的游戏，"让人愿意尝试"是比"最大化每份收入"更重要的事情

4. **毕业即创业 = 新世代的勇气** — Fry 和 Zoi 选择在 2019 年毕业后直接创业，没有先去大厂积累经验。这种选择的风险极高，但回报也可能是巨大的

5. **对培养模拟品类的意义** — 火山的女儿证明了美少女梦工厂的核心模式在 2020 年代仍然有巨大的市场。300 万 + 销量意味着"养成女儿"不是一个小众需求——这个品类等待的只是合适的作品
