# 日程时间数据总览 - 梦幻魔法公主(Magical Princess)

> 游戏版本: Magical Princess v1.0 (PC/Steam, 2026年4月27日)
> 数据来源: Steam社区指南, 18183

## 一、全年日程结构

| 时间维度 | 结构 | 说明 |
|------|------|------|
| 培养周期 | **3年** | 魔法学院入学到毕业 |
| 每年 | **12个月** | 无季节概念 |
| 每月 | **3阶段** | 上课→白天自由行动→夜晚自由行动 |
| 总计行动回合 | 3年×12月×3阶段 = **108个阶段** | 含36个上课阶段+36个白天+36个夜晚 |

## 二、行动力(AP)分配

### 2.1 每日AP构成

| 来源 | AP量 | 时机 |
|------|:--:|------|
| 早晨恢复(满) | **10** | 每天早上 |
| 夜晚恢复 | **+5** | 夜晚自由行动时 |
| 每日总AP | **15** | — |
| 天赋「活力充沛」后 | **16** | — |

### 2.2 月度AP估算

| 情景 | AP总量 | 说明 |
|------|:--:|------|
| 标准月 | ~15/天(上课+白天+晚上) | 上课阶段AP自动分配 |
| 赤月之夜月(2月/8月) | ~10(白天部分) | 夜晚禁用所有行动指令 |
| 校庆月(5月) | 白天部分用于比赛 | 需分配AP给比赛vs其他行动 |
| 面包节月(11月) | 白天部分用于比赛 | 同上 |

### 2.3 全程AP总量估算

```
36个月 × ~15 AP/月(含上课自动消耗) ≈ 540 AP(总行动预算)

实际上课阶段AP由系统自动分配，玩家实际控制:
36白天 × 10AP + 36夜晚 × 5AP = 540 AP
减去6次赤月之夜夜晚(2月/8月各3年×2=6次)
= 540 - 30 = 510 AP(实际可用)
```

## 四、压力-行动力关联

| 压力值 | 行动力上限 | 影响 |

| 0-49% | 正常(15) | 无影响 |
| **50%+** | **减半(~7-8)** | **当天几乎所有行动受限** |

## 六、最优日程排布参考

### 智力/魔法优先路线(玛奇亚学级)
```
第1年: 魔术学/文学/算术课程 + 聊天交朋友(修可拉优先) + 面包房打工
第2年: 继续智力课程 + 讨伐魔物(古树林) + 父女出游(压力管理)
第3年: 冲刺属性阈值 + 血月商人购物 + 节日比赛
```

### 体力/战斗优先路线(格拉迪乌学级)
```
第1年: 剑术/体术课程 + 骑士团训练 + 聊天(夏尔优先) + 农场打工
第2年: 继续体力课程 + 讨伐魔物高频 + 街头打架(东亚币)
第3年: 冲刺勇者/将军阈值 + 赤月全参战 + 幻彩龙讨伐
```

### 魅力/社交优先路线(诺比利斯学级)
```
第1年: 礼仪/社交术课程 + 广泛交友 + 餐馆打工
第2年: 美容术课程 + 约会 + 父女出游 + 面包节准备
第3年: 冲刺魅力5000 + 舞蹈冠军 + 面包节
```
