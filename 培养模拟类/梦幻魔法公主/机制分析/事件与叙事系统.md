# 事件与叙事系统 - 梦幻魔法公主(Magical Princess)

> 游戏版本: Magical Princess v1.0 (PC/Steam, 2026年4月27日)
> 数据来源: Steam社区指南, pinogamer, 找攻略, PTT C_Chat, diablofans

## 1. 事件体系总览

| 参数 | 数值 |
|------|:--:|
| 总事件量 | **170+约会场景** + 大量友情事件 + 主线事件 |
| 事件类型 | 主线/友情/约会/季节/随机/善恶事件 |
| 多周目 | 同一角色一周目无法看全所有对话和事件 |

## 2. 事件触发机制

### 2.1 属性阈值触发
- 属性达到特定值后自动触发新事件
- 例如: 体力3000+触发勇者路线事件，魅力5000触发公主路线事件

### 2.2 日期/月份触发
- **校庆**(5月): 学级对抗模拟战事件
- **面包节**(11月): 舞蹈/绘画比赛事件
- **赤月之夜**(2月/8月): 保卫王都/侦察事件
- **生日**: 善恶≤30有额外生日CG

### 2.3 好感度阈值触发
- 每个NPC有多级好感事件
- 好感度100触发恋爱CG
- 友谊事件有群体触发条件

### 2.4 前置事件触发
- 隐藏结局需要严格的周目间事件链:
  - 一周目: 正常通关
  - 二周目: 触发女王相认剧情→好感100→「与女王同行」→获得「继承碎片」
  - 三周目: 禁书研究Lv5→大迷宫→「同意继承」

### 2.5 善恶路线事件
- 善路线(善恶≥50): 常规事件+Cosmos
- 恶路线(善恶≤30): 解锁邪恶专属事件(违法面包、黑市等)
- 善恶≤30过生日有额外CG

## 3. 事件选择→后果分支

### 3.1 选项后果类型
| 后果类型 | 典型影响范围 |
|------|------|
| 属性变化 | ±10-100某子属性 |
| 好感度变化 | ±5-20某NPC好感 |
| 善恶偏移 | ±5-15善恶值 |
| 金币变化 | ±100-1000G |
| 路线锁定 | 开启/关闭特定结局路线 |
| 解锁后续事件 | 前置条件满足后解锁 |
| 道具获得 | 稀有/唯一道具 |

### 3.2 女儿聊天的对话选择
- 每月3个聊天窗口(早/白天/晚)
- 对话选项影响**父女感情**
- 错误选择降感情而非升感情
- 是压力管理的主要手段(0 AP)

## 4. 事件类型详解

### 4.1 主线事件
- **入学选择**: 选择学级(格拉迪乌/玛奇亚/诺比利斯)
- **赤月之谜**: 围绕绯红之月现象的主线剧情
- **毕业结局**: 第3年末根据所有条件触发

### 4.2 友情事件(群体)
- 多个NPC同时参与的场景
- 有特定触发条件(时间+好感度组合)
- 全部友情事件可查攻略触发

### 4.3 约会/恋爱事件(170+)
- 好感度阶段性解锁
- 每个角色有一周目看不全的丰富内容
- 好感度100触发恋爱CG
- 最终只能选择1个伴侣结局

### 4.4 季节/节日事件
- 校庆(5月): 比赛/观战
- 面包节(11月): 舞蹈/绘画/散步
- 赤月之夜(2月+8月): 战斗/侦察
- 生日事件

### 4.5 随机事件
- 翻垃圾: 随机获得物品/被发现
- 偷窃: 随机成功/被发现(影响善恶)
- 街头事件: 随机触发的遭遇

### 4.6 善恶专属事件
- 恶路线: 违法面包店、黑市交易、地下赌博
- 善路线: 教堂事件、骑士团训练

## 5. 多周目事件差异

| 周目 | 新增事件 |
|:--:|------|
| 一周目 | 全部基础事件，女王不可攻略 |
| 二周目 | 解锁女王相认剧情线，女王好感可提升至100，「幼年的约定」事件线可用 |
| 三周目+ | 禁书研究→大迷宫→继承者路线 |

## 6. 设计特点分析

1. **170+约会场景=关系深度重于广度**: 不是简单的"送礼物→涨好感→告白"，而是大量场景化互动建立情感联系
2. **善恶=内容解锁而非道德评判**: 恶路线有专属CG和事件，鼓励玩家两条路线都体验
3. **周目锁=叙事层的渐进揭示**: 女王线和继承者路线被周目锁限制→一周目的"困惑"在二三周目逐步被解答→叙事驱动多周目
4. **群体友谊事件=角色网络而非孤岛**: 不只有1对1的关系，也有多角色互动的场景→增强世界真实感
5. **女儿聊天的选项后果=微妙的情感设计**: 不是每句话都讨好女儿→正确的父爱表达才是关键
