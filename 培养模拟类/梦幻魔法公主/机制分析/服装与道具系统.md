# 服装与道具系统 - 梦幻魔法公主(Magical Princess)

> 游戏版本: Magical Princess v1.0 (PC/Steam, 2026年4月27日)
> 数据来源: Steam社区指南, 18183, diablofans, pinogamer

## 1. 服装系统

### 1.1 服装功能
| 功能 | 说明 |
|------|------|
| 外观变化 | 女儿立绘/模型更换服装外观 |
| 属性加成 | 部分服装附带属性修正 |
| 结局条件 | 购买5件以上解锁「时尚设计师」结局 |
| 特殊服装 | 女仆装(「女仆」结局必需) |

### 1.2 服装获取
| 方式 | 说明 |
|------|------|
| 商店购买 | 主要获取方式，金币购买 |
| 事件奖励 | 特定事件完成后赠送 |
| 节日奖励 | 面包节/校庆奖励 |
| 成就解锁 | — |

### 1.3 服装种类
| 类别 | 代表 | 用途 |
|------|------|------|
| 日常服 | 常服/学园制服 | 基础外观 |
| 职业服装 | 女仆装/骑士装 | 特定结局必需 |
| 节日服装 | 舞蹈服/礼服 | 面包节加成 |
| 特殊服装 | 伪造学生证变装 | 间谍路线 |

### 1.4 服装与活动效率
- 特定服装提升特定活动效率(类似装备加成)
- 服装属性加成影响打工和课程效率
- 没有详细的服装→属性映射表公开

## 4. 料理/烹饪系统

### 4.1 料理作用
| 效果 | 说明 |
|------|------|
| 恢复行动力 | 增加额外可用AP |
| 减轻压力 | 降低压力值 |
| 永久提升属性 | 特定料理提升魅力/感性等 |

### 4.2 烹饪等级
- 等级通过重复制作料理提升
- Lv3: 可制作提升属性的料理
- Lv7: 「企业家」结局需要

### 4.3 关键食材与来源

| 食材 | 获取 | 用途 |
|------|------|------|
| **小麦粉** | 面包房打工(100%)/农场打工 | 全期核心食材，做面包刷烹饪等级 |
| 蔬菜 | 农场打工 | 各类料理 |
| 黄油 | 面包房打工 | 高级料理 |
| 鸡蛋 | 餐馆购买 | 高级料理 |
| 牛奶 | 餐馆购买 | 高级料理 |
| 魔兽肉 | 讨伐魔物(古树林) | 战斗料理 |

### 4.4 料理策略
- 初期: 小麦粉→纺锤面包(刷等级+降压力)
- 中期: Lv3后可制作属性提升料理
- 芝士汉堡: 高效控制压力值

## 6. 设计特点分析

1. **服装=结局条件+视觉奖励**: 不仅仅是属性加成的装备→解锁特定结局和视觉变化
2. **双货币→双商店体系**: 金币商店+东亚硬币商店→正常消费vs灰色消费的区隔
3. **伪造学生证=身份转换道具**: 改变学级→解锁间谍路线→功能性道具改变游戏结构
4. **料理=自给自足的经济/成长循环**: 打工→食材→料理→属性/金币→更好的打工→正向循环
5. **核心食材小麦粉的贯穿设计**: 从初期到后期都需要→简单的经济锚点维持稳定性
