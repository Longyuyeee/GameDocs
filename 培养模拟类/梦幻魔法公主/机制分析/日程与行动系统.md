# 日程与行动系统 - 梦幻魔法公主(Magical Princess)

> 游戏版本: Magical Princess v1.0 (PC/Steam, 2026年4月27日)
> 数据来源: Steam社区指南(SteamCommunity), 找攻略, PTT C_Chat, Stage1st, 18183, pinogamer

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 培养周期 | **3年**(魔法学院入学→毕业) |
| 每年 | 12个月 |
| 每月 | 3个阶段: **上课** → **白天自由行动** → **夜晚自由行动** |
| 无周/日细分 | 月度是最小时间单位 |

## 2. 月度流程

```
每月固定流程:
  上课(Class) → 白天自由行动(Daytime Free) → 夜晚自由行动(Night Free)
```

- **上课阶段**: 推进已选课程的进度条
- **白天自由行动**: 打工/社交/探索/讨伐魔物等主动行动
- **夜晚自由行动**: 夜间专属打工/购物/赌博/街头打架等

## 3. 行动力(AP)系统

| 要素 | 数值 |
|------|:--:|
| 基础AP上限(白天) | **10点** |
| 夜间恢复 | **+5点** |
| 每日总计 | **15点** |
| 天赋「活力充沛」提升后 | **16点** |
| 一次标准行动消耗 | 2-4点 |
| 与朋友聊天消耗 | 1-2点/次 |

- AP每天早上回满
- 夜晚额外+5点
- 推荐用光所有行动力，不保留到次日

## 4. 行动类型总览

| 类型 | 时段 | 代表活动 | AP消耗 | 主要收益 |
|:--:|:--:|------|:--:|------|
| **课程(Class)** | 上课阶段 | 各学科课程 | 固定 | 主属性+子属性经验 |
| **打工(Job)** | 白天/夜晚 | 农场/面包房/餐馆/夜间打工 | 2-3 | 金币+食材+部分属性 |
| **社交(Social)** | 白天/夜晚 | 对话/约会/出游 | 1-2 | 好感度+降压力 |
| **讨伐魔物(Hunt)** | 白天 | 古树林5关 | 高(4-5) | 战斗经验+金币+食材 |
| **购物(Shop)** | 夜晚 | 商店/苏菲摊贩 | 0-1 | 道具/装备/食材 |
| **赌博(Gamble)** | 夜晚 | 掷骰游戏 | 1-2 | 东亚硬币 |
| **街头打架** | 夜晚(中期) | 街头乱打 | 2-3 | 东亚硬币+战斗经验 |
| **特殊活动** | 白天/夜晚 | 教堂祷告/翻垃圾 | 1-2 | 善恶/随机物品 |
| **休息/聊天** | 任意 | 与女儿聊天/父女出游 | 0-1 | 降压力+父女感情 |

## 5. 压力系统

| 参数 | 数值 |
|------|------|
| 压力阈值 | **50%** |
| 超过阈值惩罚 | **行动力上限减半**(15→7-8) |
| 聊天降压力 | **免费**(0 AP)，但对话选项可能影响父女感情 |
| 父女出游降压力 | 花费金币，0 AP，大幅降压力+特殊CG |
| 与朋友聊天降压力 | 少量AP消耗 |
| 道具降压力 | 消耗道具 |

### 聊天时机
- 早课前(早晨)
- 白天自由时间
- 夜晚自由时间
- 每月可与女儿免费聊天

## 6. 课程系统

### 课程进度机制
- 每门课程有**进度条**(0→100%)
- 进度达到100%时**属性才提升**
- 单次上课进度取决于相关属性等级
- 效率标准: **≥50%/次**(2次完成)或**≥34%/次**(3次完成)
- 如进度卡在49%附近→先提升相关属性

### 学级(学院)选择

| 学级 | 侧重主属性 | 课程效率加成 |
|------|:--:|------|
| **格拉迪乌(Gladius)** | 体力 | 本学级课程耗时减少 |
| **玛奇亚(Machia)** | 智力 | 本学级课程耗时减少 |
| **诺比利斯(Nobilis)** | 魅力 | 本学级课程耗时减少 |

- 感性(Sensibility)课程不属于任何学级，所有人都一样
- 可选修其他学级的课程，但进度较慢

### 课程类型

| 主属性 | 代表课程 |
|:--:|------|
| 体力 | 剑术、体术、战斗训练等 |
| 智力 | 魔术学、信仰学、算术、文学等 |
| 魅力 | 礼仪、社交术、美容术等 |
| 感性 | 音乐、美术、创意写作等 |

## 7. 特殊月度事件

### 5月 - 校庆(School Festival)
- 模拟战(学级对抗)
- 选项: 参战 / 观战+赌博
- 第一年极难获胜
- 冠军获得强力道具

### 11月 - 面包节(Bread Festival)
- 舞蹈比赛(魅力属性)
- 绘画比赛(感性属性)
- 散步选项: 获得稀有属性道具(初期推荐)
- 第一年极难夺冠

### 2月/8月 - 赤月之夜(Crimson Moon Night)
- 夜晚所有行动指令**禁用**
- 两个选择:
  - **保卫王都**: 战斗事件，可选难度(Easy/Normal/Hard)，奖励名望+道具+金币+战斗经验
  - **侦察中央广场**: 不战斗，可访问特殊商人苏菲(稀有道具)，但设施可能受损、朋友可能受伤

## 8. 设计特点分析

1. **月度=稀缺资源**: 仅36个月→每个月的课程和行动选择都有长期后果
2. **学级决定路线但不锁定**: 选择学级提供效率加成但不禁止跨学级学习→保留了培养自由度
3. **压力/行动力双约束**: 压力超50%→行动力减半→正向激励玩家管理压力而非惩罚
4. **赤月之夜=月度特殊选择**: 每半年一次的战略决策点→战斗拿名望 vs 购物拿道具
5. **课程进度条机制=策略深度**: 卡在49%时需思考"先提升属性还是先继续上课"
