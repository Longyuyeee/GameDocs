## 3. 3人如何做出50+结局×170+约会事件×全语音?

| 效率策略 | 详情 |
|------|------|
| 世界观复用 | 基于MAGI已有桌游IP——角色/剧情框架已有基础 |
| 漫画分镜替代3D | 大宫选择2D立绘+漫画分镜式演出，大幅降低动画成本 |
| 声优全部外包 | 16位知名声优为签约制，非内部员工 |
| 养成框架模版化 | 类PM的养成游戏=有限美术资产产生大量分支——系统化生成结局 |
| 堆料策略 | 40万+字剧本 + 700+漫画画面 + 100张CG + Live2D全演出 |

## 5. 设计启示

1. **3人=养成游戏的最小可行团队** — 1人编程+1人制作+1人美术——但前提是有现存IP世界观可以复用
2. **¥48定价=口碑裂变引擎** — 极低价格降低了尝试门槛→直播传播→病毒式增长
3. **"养女儿"天然适合直播** — 主播的情感投入=最好的广告
4. **桌游公司跨界做电子游戏=品类的意外创新** — 桌游的"叙事+系统"DNA恰好适合养成品类
