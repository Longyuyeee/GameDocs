# 死亡方式与条件数据 - 女王万岁(Long Live the Queen)

> 游戏版本: Long Live the Queen (PC, 2012年)
> 数据来源: longlivethequeen.fandom.com, GameFAQs

## 二、死亡概率按周(估算)

| 阶段 | 死亡风险 | 说明 |
|:--:|:--:|------|
| 周1-10 | ★★★ | 蛇咬+早期毒杀+水晶获取失败 |
| 周11-20 | ★★★★ | 游行+舞会+阴谋→检定密集 |
| 周21-30 | ★★★ | 精神试炼(周28)为主要威胁 |
| 周31-40 | **★★★★★** | **决斗(周35!)+终局事件** |

## 三、最常见的死亡原因(社区统计推论)

| 排名 | 死亡方式 | 原因 |
|:--:|------|------|
| 1 | **Togami决斗(周35)** | Wield Magic≥100未达到→无法应对 |
| 2 | 蛇咬(周2-3) | 新手没训练Reflexes |
| 3 | Lucille毒(周17) | Sense Magic不够高 |
| 4 | 暗杀 | 多种方式/难以全部预防 |
| 5 | 精神崩溃(周28) | Composure/Meditation忽略 |

## 四、生存率提升策略

| 策略 | 效果 |
|------|------|
| Lumen路线 | 提供最强防御(Wield Magic应对决斗+Sense Magic察觉毒) |
| 平衡训练 | 不要专精→多种威胁需要多种技能 |
| 服装切换 | 事件前换对应服装→临时+10-30检定 |
| 存档策略 | 每周存档→失败回档→调整训练 |
