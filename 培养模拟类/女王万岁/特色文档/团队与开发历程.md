# 团队与开发历程 — 女王万岁(Long Live the Queen)

> 开发商: Hanako Games (Georgina Bensley) | 英国
> 发行年份: 2012年 | PC/主机 | 团队规模: **1人**(核心)

## 1. 开发团队

| 成员 | 角色 |
|------|------|
| **Georgina Bensley ("Hanako")** | 设计/编程/剧本——全部一人完成 |
| DeviantArt自由画师 | 美术外包 |
| Spiky Caterpillar | 额外程序支持 |

Bensley从2005年开始做游戏。她的第一款游戏《Cute Knight》开发预算仅**$50**。PM2是她的启蒙游戏:
> "任何看过我游戏开发历程的人都能看出来，我是PM2的超级粉丝……我沉迷于玩那个被废弃的PM2英文翻译beta版。"

## 5. 设计启示

1. **1人×Ren'Py×DeviantArt = 最纯粹的独立游戏开发** — 没有团队、没有预算、没有技术门槛——只有"我想做一个PM2的精神续作"
2. **PM2的20年级联影响** — 一个1993年的日本游戏→启发了一个英国独立开发者在2012年做出爆款→这是品类的文化传承
3. **Ren'Py不是游戏引擎=但足够用了** — 用视觉小说引擎做养成游戏——工具不是限制，创意才是
4. **高死亡率=品类的独特定位** — 11种死亡方式让LLTQ在所有养成游戏中独树一帜——"你的每个选择都可能致命"
