## 2. 技能检定机制

### 2.1 学习公式

```
每天技能点 = (2 + 心情加成) × 5天/周
  + 技能集点数 × 0.01
  + 类别点数 × 0.001

最高: (2+1)×5 = 15点/周
最低: (2-2)×5 = 0点/周
```

### 2.2 技能上限: 每项100点

### 2.3 服装系统

掌握任一小类的全部3项技能→解锁对应服装→提供额外加成。

## 3. 心情系统(9可见+1隐藏)

### 3.1 四轴八极

| 轴 | 左极 | 右极 |
|------|------|------|
| 1 | Afraid(恐惧) | Angry(愤怒) |
| 2 | Depressed(抑郁) | Cheerful(开朗) |
| 3 | Yielding(顺从) | Willful(任性) |
| 4 | Lonely(孤独) | Pressured(压力) |

### 3.2 心情→技能加成/惩罚矩阵(关键!)

| 心情 | 最佳技能(+1) | 最差技能(-2~-3) |
|------|------|------|
| **Angry** | 武器+1, 军事+1 | — |
| **Willful** | 谋略+1, 军事+1, 光能+1 | **皇家风范-2, 历史-2, 经济-2** |
| **Yielding** | 皇家风范+1, 历史+1, 信仰+1 | **武器-3, 光能-3**(最严重惩罚!) |
| **Cheerful** | 对话+1, 运动+1 | 军事-2 |
| **Depressed** | 表达+1, 动物+1 | — |
| **Pressured** | 运动+1, 信仰+1 | — |
| **Lonely** | 对话+1, 医药+1 | — |
| **Afraid** | 敏捷+1, 信仰+1 | — |
| **Injured**(隐藏) | — | 全部体能技能-3(敏捷/武器/运动/动物) |

> 心情只取"最高峰"(最大值对应的极)，不关心具体数值。

## 5. 设计特点

1. **42技能×100点=4200点的有限资源**: 40周内不可能全技能100→专业化是必然的
2. **心情=技能的乘数**: 不是独立系统→心情直接决定你今天学什么最有效
3. **Yielding=最危险的陷阱**: 武器-3意味着你在需要战斗的事件中几乎必然失败
4. **服装=技能集成的奖励**: 3项技能全掌握→解锁服装→视觉+数值的正反馈
5. **每周固定事件=知识的价值**: 知道第X周有什么检定→提前培养对应技能→"元游戏"的核心
