## 1. 开发团队

### Gainax（盖纳克斯）

**Gainax** 是日本最具传奇色彩的动画与游戏公司之一，由冈田斗司夫、赤井孝美、贞本义行等人于1984年创立。在2007年开发《美少女梦工厂5》时，Gainax正处于历史上最动荡的时期——表面上看，《天元突破》正在电视上热播，公司声誉如日中天，但内部矛盾已经暗流涌动。

### 赤井孝美——PM系列的生父

**赤井孝美 (Takami Akai)** 是Gainax的联合创始人、《美少女梦工厂》系列的创造者。他在PM5中担任角色设计和原画。

然而，PM5的发售恰好与Gainax历史上最大的内部危机之一重合。

| 维度 | 说明 |
|------|------|
| **加入Gainax** | 1984年，大学时期即参与创立 |
| **代表作品** | 美少女梦工厂系列、王立宇宙军、天元突破（前期） |
| **独立工作室** | 1996年创立 NineLives（以个人名义运营） |
| **离开Gainax** | 2007年4月27日（天元突破争议辞职） |

### 开发团队

PM5由Gainax内部团队开发，与系列之前作品不同，这是 **首个赤井孝美未担任导演和制作人的PM正作**。游戏的实际开发由Gainax的游戏部门负责，赤井的角色被限定在角色设计和原画领域。

## 2. 天元突破争议——赤井孝美辞职始末

这是日本游戏/动画史上一个影响深远的事件，直接决定了PM系列的未来走向。

### 事件时间线

| 时间 | 事件 |
|------|------|
| 2007年4月1日 | 《天元突破》第1话播出 |
| 2007年4月初 | 2channel 用户批评第1话的作画质量 |
| 2007年4月初 | Gainax员工 三森圭子（Keiko Mimori）在Mixi上发帖，称2channel用户是"愚蠢恶心的宅男、应该去死" |
| 2007年4月初 | 赤井孝美（网名"Magi no Suke"）跟帖回应，称看2channel评论"就像把脸凑到屁眼上深呼吸" |
| 2007年4月中旬 | 第4话播出，作画风格明显不同，引发更大规模批评 |
| 2007年4月27日 | Gainax发布赤井孝美的公开辞职信，他辞去董事职务和《天元突破》制片人职位，从第5话起从演职员表中消失 |

### 对PM5的影响

- PM5于2007年3月3日发售，仅比天元突破争议爆发早1个月
- 赤井辞职后，PM系列失去了灵魂人物
- Gainax在游戏部门的投入迅速减少
- PM5成为 **Gainax制作的最后一款PM正作**

## 3. 与系列前作的关键区别

### 从幻想走向现代

PM5是系列中唯一将背景设定在 **现代日本** 的作品。这是一个颠覆性的选择：

| 维度 | PM1-PM4 | PM5 |
|------|---------|-----|
| 世界观 | 奇幻中世纪 | **现代日本** |
| 学校 | 无 | 有（小学→中学→高中） |
| 日程 | 月/半月制 | **周制（可细化至每日）** |
| 家长 | 父亲（勇者/王/战士等） | 可选父亲或**母亲** |
| 属性 | 约15种基础属性 | 11显+6隐属性，13种技能 |

### 系统深度——系列最复杂

PM5被公认为拥有系列最复杂的系统：

**日程系统**
- 引入周制日程，可细分到每天放学后的活动安排
- 学校的上课/社团活动/考试构成固定日程
- 周末和假期的自由活动更加丰富

**属性体系**
- 11种显性属性：体力、智力、精神力、魅力、道德、社交、家务、烹饪、感性、运动、艺术
- 6种隐性属性：疲劳、压力、好感度、知名度、运气、罪恶感
- 13种技能：涵盖学习、运动、艺术、家务等多个领域

**结局系统**
- **真公主结局**：需要特定条件达成，是历代最难解锁的结局之一
- **回归社会结局**：对应现代背景的角色代入
- 总体结局数量达到系列最高水准，涵盖职业、婚姻、个人成长等多维度

## 4. PM系列的终结与传承

### 销售与评价

PM5在商业上表现中规中矩，未达到PM2的巅峰水平。Famitsu评分大约在7/10区间，玩家评价两极分化：
- 支持者认为系统深度和现代设定是系列进化
- 反对者认为失去了奇幻氛围和系列特色

### 系列的沉寂

赤井孝美离开Gainax后：
- Gainax失去了继续开发PM的动力和能力
- 系列进入长达10年以上的休眠期
- 赤井孝美通过自己的NineLives工作室保持独立创作

### Steam重生

2018年，韩国发行商 **CFK** 将PM5带到了Steam平台，让新一代玩家能够体验这部"最复杂PM"的魅力。同时CFK还陆续发布了PM1-PM4的Steam版本，实际上在数字发行时代接过了PM系列的发行权。

### 系列遗产

虽然Gainax作为游戏开发商的生涯以黯然收场告终（公司本身也于2024年宣布破产），但《美少女梦工厂》系列留下的遗产不可磨灭——它开创了"女儿养成模拟"这一子类型，深刻影响了中国和韩国的养成游戏产业，包括《明星志愿》《兰岛物语》等中文游戏都直接受益于PM系列的开创性贡献。

PM5作为Gainax时代的终章，以其极致的系统复杂性和充满野心的现代日本设定，成为了系列中最独特、也最富争议的一部作品。
