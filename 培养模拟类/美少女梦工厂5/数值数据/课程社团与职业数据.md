# 课程社团与职业数据 - 美少女梦工厂5(Princess Maker 5)

> 游戏版本: Princess Maker 5 (PC, 2007年) | 开发商: Gainax | Steam版: 2018年
> 数据来源: princessmaker.fandom.com, GameFAQs Complete Guide, 萌娘百科, 百度贴吧

## 一、父母职业(开局选择)

| 职业 | 月收入 | 隐藏属性加成 | 特点 |
|------|:--:|:--:|------|
| 高学历专业 | ¥50,000固定 | 自尊+20 | 最稳定 |
| 创业者 | ¥40,000(Y1-4)→¥60,000(Y5+) | 独立心+20 | 后期收入最高 |
| 公务员 | ¥40,000+年奖金¥60,000×2 | 细心-20 | 有额外奖金 |
| 上班族 | ¥30,000→¥40,000+奖金 | 独立心-20 | 收入最低但稳定 |
| 自由职业 | ¥20,000~80,000(不定) | 细心+20 | 高风险高回报 |

### 隐藏属性(6项)
| 属性 | 影响 |
|------|------|
| 独立心 | 过高→女儿反抗/离家出走 |
| 细心 | 自发行动/降低压力 |
| 社交性 | 交友容易度 |
| 逻辑性 | 高→减少发展兴趣 |
| 自尊心 | 自信/自我形象 |
| 创造性 | 画家结局必需 |
