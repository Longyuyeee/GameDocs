# 属性与结局数据总览 - 美少女梦工厂5(Princess Maker 5)

> 游戏版本: Princess Maker 5 (PC, 2007年)
> 数据来源: princessmaker.fandom.com, GameFAQs Guide, 萌娘百科

## 一、属性体系

### 1.1 11项可见属性

| # | 属性 | 说明 | 关键提升方式 |
|:--:|------|------|------|
| 1 | 体力 | 健康/抗疲劳 | 体育/运动社团 |
| 2 | 精神力 | 心理抗压 | 自由时间/成功体验 |
| 3 | 运动能力 | 体能/运动表现 | 体育/运动社团 |
| 4 | 智力 | 学业/考试成绩 | 学校课程/补习 |
| 5 | 领导力 | 社交影响力 | 学生会/演剧部 |
| 6 | 气质 | 优雅/教养 | 礼仪课程/芭蕾 |
| 7 | 魅力 | 外表吸引力 | 服装/化妆 |
| 8 | 爱情 | 同理心/善良 | 志愿者/与朋友相处 |
| 9 | 道德 | 是非判断 | 学校/志愿者 |
| 10 | 感受 | 艺术敏感/(高时)脆弱 | 音乐/绘画 |
| 11 | 压力 | 累积疲劳 | —（需管理） |

### 1.2 6项隐藏属性

| 属性 | 影响 |
|------|------|
| **独立心** | 过高→不听从安排/离家出走 |
| **细心** | 自发行动/降低压力 |
| **社交性** | 交朋友容易度/情人节送巧克力 |
| **逻辑性** | 高→减少发展兴趣的可能 |
| **自尊心** | 自信/自我形象 |
| **创造性** | 独立心+细心双高时上升→画家结局必需 |

## 五、设计特点

1. **隐藏属性=玩家无法完全控制**: 独立心过高→女儿会反抗→你不再是"全能的父亲"
2. **～53结局=系列最复杂的判定**: 结合了课程次数+隐藏属性+关系+技能
3. **青春期=养成中最真实的成长模拟**: 身体+心理双重变化→女儿不总是"乖"的
4. **周制=极致的微观管理**: 416周的细腻程度在养成品类中无出其右
