# 日程与学校系统 - 美少女梦工厂5(Princess Maker 5)

> 游戏版本: Princess Maker 5 (PC, 2007年)
> 数据来源: princessmaker.fandom.com, GameFAQs Complete Guide, 萌娘百科, 百度贴吧

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 培养周期 | **8年**(小学2年→初中3年→高中3年) |
| 时间单位 | **周**(非PM2/PM3的旬/半月) |
| 每周 | 平日固定上课 + 课后自由安排 + 周末全自由 |
| 月末 | 设定教育方针(影响属性成长方向) |
| 假期 | 寒暑假(完全自由安排) |

> PM5是PM系列中**唯一采用周制**的作品——8年≈416周，比PM2的288回合更多。

## 2. 学校阶段

| 阶段 | 年数 | 年龄 | 特点 |
|:--:|:--:|:--:|------|
| 小学 | 2年 | 10-11 | 基础课程/社团入门 |
| 初中 | 3年 | 12-14 | 社团活动解锁更多/青春期开始 |
| 高中 | 3年 | 15-17 | 冒险解锁(初三起)/结局冲刺 |

## 3. 每月方针系统

月末可设定**教育政策**，影响本月属性成长模式:

| 方针 | 效果 |
|------|------|
| 严格教育 | 属性成长↑ 但压力↑ |
| 自由放任 | 压力↓ 但属性成长↓ |
| 均衡 | 中庸 |

## 7. 青春期系统

PM5独有的**青春期(Puberty)**系统:
- 身体变化: 身高/体重/胸围/腰围/臀围跟踪
- 心理变化: 独立心上升→可能不听从安排
- 反抗期: 独立心过高→女儿跳过日程/离家出走/未经允许花钱
- 与父亲/母亲关系: 青春期可能暂时冷淡→需要耐心维护

## 8. 冒险系统(初三解锁)

- 5个世界地图: 人间界/妖精界/灵界/魔界/天界
- 4个Boss: Hydra(Y4), Gelato(Y5), Kenpi(Y6), Waffel(Y7)
- Boss属性公式: Charm=周数×2, Elegance=周数×1.8, Charisma=周数×1.6, Willpower=周数×1.4
- 最终天界Boss需极高战力

## 9. 父母职业选择(开局)

| 职业 | 月收入 | 隐藏属性加成 |
|------|:--:|------|
| 高学历专业 | ¥50,000固定 | 自尊+20 |
| 创业者 | ¥40,000(Y1-4)→¥60,000(Y5+) | 独立心+20 |
| 公务员 | ¥40,000+年奖金¥60,000×2 | 细心-20 |
| 上班族 | ¥30,000→¥40,000+奖金 | 独立心-20 |
| 自由职业 | ¥20,000~80,000(不定) | 细心+20 |

## 10. 设计特点

1. **周制=最细腻的时间粒度**: 8年416周→每次选择的影响更小但累积效应更强
2. **青春期=最复杂的情感模拟**: 不是简单的"不良化"→是真实的青春期心理变化
3. **父母职业=社会经济背景**: 初始选择影响8年的经济状况→增加了社会阶层模拟
4. **冒险=系列最丰富的战斗系统**: 5个世界+4个Boss+最终Boss
