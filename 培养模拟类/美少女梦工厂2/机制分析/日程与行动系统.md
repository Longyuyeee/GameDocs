# 日程与行动系统 - 美少女梦工厂2(Princess Maker 2 Refine)

> 游戏版本: Princess Maker 2 Refine (PC/Steam, 1993年原版/2016年Refine版)
> 数据来源: princessmaker.fandom.com, SteamSolo Guide, arrpeegeez.com, PTT PM版, 3DM

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 培养周期 | **8年**(10岁→18岁) |
| 每年 | 12个月 |
| 每月 | **3旬**(上旬/中旬/下旬) |
| 每旬 | 安排1项活动(课程/打工/休息/武者修行) |
| 总计行动回合 | 8年×12月×3旬 = **288回合** |
| 节日 | 每年1月(新年)、10月(收获祭) |

## 2. 每月行动流程

```
月初 → 安排本月的饮食方针(正餐/便当/增强体力/减肥)
     → 安排本旬活动(课程/打工/休息/武者修行/自由活动)
     → 中旬安排
     → 下旬安排
月末 → 结算属性变化/收入/疲劳
```

### 3.3 武者修行(Adventure)
- 四大地图: 东部森林/西部沙漠/南部水乡/北部冰川
- 每旬可选择出发→全旬用于探索
- 战斗→提升战技/攻击/防御/魔法技能
- 特殊事件→属性大幅波动
- 收入: 战斗掉落+宝箱(西部沙漠最赚钱)

### 3.4 休息(Free Time/Rest)
| 方式 | 效果 |
|------|------|
| 自由活动 | 疲劳↓/压力↓ |
| 度假(山脉/海边) | 疲劳↓↓/感受↑(山脉)/特定事件触发 |
| 给零用钱 | 压力↓↓(每旬花费金币) |

## 4. 压力/疲劳/健康系统

| 状态 | 积累方式 | 危险阈值 | 后果 |
|------|------|:--:|------|
| **疲劳(Stress)** | 课程/打工/武者修行 | — | — |
| **压力(累积疲劳)** | 持续工作/课程 | 疲劳>体力 | **生病**(可导致死亡) |
| **不良化(Delinquency)** | — | 压力>道德或信仰(取高者) | 女儿变成不良少女 |
| **体重** | 饮食方针 | — | 魅力↓(超重时) |

### 压力管理策略
- 自由活动: 每次降大量压力
- 度假: 降压力+特殊事件
- 零用钱: 最快速降压力(消耗金币)
- 饮食方针「便当」: 每月压力小幅降低

## 5. 血型系统

| 血型 | 每月效果 |
|:--:|------|
| **A型** | 道德+2/月, 压力+2/月 |
| **B型** | 压力-2/月, 魅力-2/月 |
| **O型** | 压力>0时压力+1/月 |
| **AB型** | 压力>0时压力+1/月 |

## 6. 设计特点分析

1. **288回合=8年的密度**: 每年36次选择(12月×3旬)→每个选择都重要
2. **压力>体力=死亡**: 不是"降低效率"→是**真的会死**→硬核的压力管理
3. **打工=属性双刃剑**: 几乎所有打工同时提升和降低属性→没有完美的活动→永远在做权衡
4. **武者修行=高风险高回报**: 可能获得海量财富(西部沙漠)或遭遇灭团→冒险是可变因素
5. **血型=先天不可改变的差异**: 初始选择影响整个8年→A型更好(道德+2/月)但压力管理更难
