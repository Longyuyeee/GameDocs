# 社交与关系系统 - 美少女梦工厂2(Princess Maker 2 Refine)

> 游戏版本: Princess Maker 2 Refine (PC/Steam, 1993/2016)
> 数据来源: princessmaker.fandom.com, SteamSolo Guide, PTT PM, 3DM攻略

## 1. 关系系统总览

PM2没有"好感度数值表"——关系是通过**事件选择**和**隐藏数值**来推进的。这种设计与现代养成游戏的"送礼物→涨好感度"模式完全不同。

## 2. 核心关系对象

### 2.1 王子(青年武官)

| 参数 | 说明 |
|------|------|
| 初次见面 | 每年1月31日→去王宫 |
| 好感累积方式 | 每年1次对话→选"愿意"陪他 |
| 需要次数 | 共8次(10岁→17岁，每年1次) |
| 隐藏好感度上限 | 100(显示为"王子关系") |
| 求婚条件 | 好感≥90 + 魅力≥200 + 气质≥250 + 道德≥30 + 罪孽<100 |

> 如果漏掉一年，王嫁路线直接失败——这是PM2最严苛的设计之一

### 2.2 父亲(玩家)

| 参数 | 说明 |
|------|------|
| 关系值名称 | 父感(Paternal Bond) |
| 提升方式 | 家庭管理打工(+父感)/与女儿对话/度假 |
| 上限 | 100 |
| 父嫁条件 | 父感=100 + 道德≥100 + 信仰≥100 + 罪孽<100 + (智力+气质+魅力)≥道德 + (智力+气质+魅力)×2>信仰+道德 |

### 2.3 管家吉普(Cube)

| 参数 | 说明 |
|------|------|
| 定位 | 管家/女儿的管理者 |
| 关系推进 | 长期互动→特殊事件 |
| 结局 | 管家嫁(特定条件) |

### 2.4 魔王

| 参数 | 说明 |
|------|------|
| 定位 | 西部沙漠的BOSS→可能成为丈夫 |
| 条件 | 道德<30或罪孽高/魅力高/气质高 |
| 结局 | 魔嫁(魔王来抢婚) |

### 2.5 小龙

| 参数 | 说明 |
|------|------|
| 定位 | 东部森林的龙→可能成为丈夫 |
| 结局 | 龙嫁(答应小龙的求婚) |

## 3. 求婚系统

当气质≥200且魅力≥200时，随机出现求婚者。女儿可以**接受**或**拒绝**。

| 求婚者类型 | 出现条件 | 结局类型 |
|------|------|:--:|
| 随机求婚者 | 气质≥200/魅力≥200 | 社交结局(伯爵夫人/富豪夫人等) |
| 王子 | 王子关系≥90 | **王嫁**(特殊婚姻) |
| 父亲 | 父感=100 | **父嫁**(特殊婚姻) |
| 管家 | 吉普关系高 | **管家嫁**(特殊婚姻) |
| 魔王 | 道德<30或高罪孽 | **魔嫁**(特殊婚姻) |
| 小龙 | 龙的好感 | **龙嫁**(特殊婚姻) |

### 求婚/特殊婚姻优先级
```
王子 > 父亲 > 管家 > 魔王 > 小龙
```
即如果同时满足王嫁和父嫁的条件→王嫁优先触发。

## 4. NPC生态

| NPC类型 | 出现场景 | 功能 |
|------|------|------|
| 神秘商人 | 随机访问 | 出售稀有道具 |
| 青年武官(王子) | 每年1月王宫 | 王嫁路线 |
| 各店老板 | 打工时 | 打工成功/失败判定 |
| 武者修行NPC | 各地图 | 战斗/事件/交易 |
| 求婚者 | 随机触发 | 社交结局 |

## 5. 关系设计的独特性

### 5.1 无"送礼"系统
与现代养成游戏不同，PM2**没有礼物系统**——你不能通过送礼来提升关系。关系的推进完全依赖:
- 特定日期的对话选择(王子)
- 打工类型(父感通过家庭管理提升)
- 属性门槛(求婚者出现)

### 5.2 关系=隐藏数值
- 王子关系、父感、吉普关系都是**隐藏的**(玩家看不到精确值)
- 玩家只能通过事件反馈猜测当前关系状态
- 这增加了神秘感和"不知能否成功"的紧张

## 6. 设计特点分析

1. **每年1次=稀缺性带来的珍贵感**: 王子每年只见1次→玩家必须记住"1月31日去王宫"→8年=8次→错过=永远失去
2. **父嫁=品类的禁忌与创新**: 允许女儿嫁给养育她的父亲→这是PM2最具争议性的设计→也是品类的独特实验
3. **求婚=被动触发而非主动攻略**: 不是"我要追谁"→而是"条件满足了自然有人来求婚"→更接近中世纪背景的真实
4. **无礼物系统=关系不是交易**: 不能用钱买好感度→关系必须通过时间和选择来建立→更纯粹但更不透明
