## 2. 结局判定机制

### 2.1 四大评价公式

| 评价 | 计算公式 |
|------|------|
| **战士评价** | 战技 + 攻击 + 防御 |
| **魔法评价** | 魔法技能 + 魔法攻击 + 抗魔 |
| **社交评价** | 礼仪表现 + 谈话技巧 + 艺术修养 |
| **家事评价** | 烹饪技巧 + 扫地洗衣 + 待人接物 |

> 最高评价 = max(战士, 魔法, 社交, 家事)

### 2.2 判定优先级

```
罪孽结局 (最高优先 —— 罪孽>100 且 道德≤0)
  ↓ 不触发
艺术家结局 (艺术修养≥80 且 为最高技能 —— 极容易意外触发!)
  ↓ 不触发
职业结局 (根据最高评价决定职业方向)
  ↓
战士/魔法师/社交/家事 各系结局 (根据具体属性阈值细分)
  ↓
特殊婚姻结局 (叠加在职业结局之上，按王子>父亲>管家>魔王>小龙优先级)
```

### 2.3 高低差规则

| 高低差 | 含义 |
|:--:|------|
| ≤50 | **均衡发展** → 可以达成该系最高等级结局 |
| >50 | **偏科发展** → 降级，只能达成该系中低等级结局 |

## 6. 设计特点分析

1. **74结局=系统化生成的先驱**: 不是74个手工结局→而是属性+评价+技能+关系组合生成的→1993年的程序化叙事
2. **高低差=对"均衡"的奖励**: 偏科无法达到最高结局→这是游戏在说"全面发展最重要"
3. **艺术家优先=反直觉的陷阱**: 艺术修养最容易无意中成为最高技能→许多玩家意外触发艺术家结局
4. **五层特殊婚姻=关系的终极验证**: 王子/父亲/管家/魔王/小龙→不同的"爱"的形式→最丰富的关系结局系统
5. **1999分=社区的终极挑战**: 理论最高分→成为PM2玩家社区30年来的攻略目标
