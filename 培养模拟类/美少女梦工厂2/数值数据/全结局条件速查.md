# 全结局条件速查 - 美少女梦工厂2(Princess Maker 2 Refine)

> 游戏版本: Princess Maker 2 Refine (PC/Steam, 1993/2016)
> 数据来源: princessmaker.fandom.com, arrpeegeez.com, SteamSolo Guide, 3DM攻略, PTT PM

## 一、结局判定流程

```
1. 计算四大评价 → 确定最高评价
2. 计算高低差 → 确定等级上限
3. 检查罪孽 → 触发罪孽结局?
4. 检查艺术家条件 → 触发艺术家结局?
5. 根据最高评价 → 职业结局
6. 检查特殊婚姻条件 → 叠加婚姻结局
```
