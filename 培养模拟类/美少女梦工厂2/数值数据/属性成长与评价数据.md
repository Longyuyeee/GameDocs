## 五、1999分路径参考

达成理论最高分1999需要的条件:
```
1. 8属性全999 + 罪孽0 = 999分
   → 需要极高效率的培养计划+大量道具+合理的时间分配
2. 女王结局 + 道德>500 = 800分
   → 需要均衡发展的四大评价(高低差≤50) + 最高评价>420
3. 王嫁 = 200分
   → 需要8年不间断每年1月见王子 + 魅力≥200 + 气质≥250
```

这个目标被社区认为是PM2的终极挑战，极少有人能达成。
