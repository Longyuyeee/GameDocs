# 团队与开发历程 — 美少女梦工厂2(Princess Maker 2)

> 开发商: Gainax | 日本
> 发行年份: 1993年(原版)/2016年(Refine) | PC-98/多平台 | 团队规模: **约10-15人**

## 1. 开发团队

| 成员 | 角色 |
|------|------|
| **赤井孝美 (Takami Akai)** | 设计/原画/角色设计(Gainax联合创始人) |
| **加藤正人 (Masato Kato)** | 设计/剧本/图形/副导演 |
| 梶原正裕 (Masahiro Kajihara) | 作曲 |
| 橋本律郎 (Ritsuro Hashimoto) | 编程 |

团队极小——成员身兼多职。赤井孝美和加藤正人各担任了2-3个核心角色。这是90年代初日本PC游戏开发的典型模式。

## 2. 开发历程

| 阶段 | 时间 | 关键事件 |
|------|:--:|------|
| Gainax时期 | 1980s-90s | 以动画起家(《王立宇宙军》《飞跃巅峰》) |
| PM2开发 | 1992-1993 | 原定名"My Fair Child"——因觉得"在电脑里养真人女孩太危险"，改为奇幻设定 |
| 原版发售 | 1993年6月15日 | NEC PC-9801——定价**14,800日元**(对标光荣定价) |
| 英文版夭折 | 1996年 | SoftEgg完成翻译→发行商Intracorp**破产**→翻译版泄露为abandonware |
| Refine版 | **2016年** | 30年来**首次官方英文版**Steam发售 |
| Gainax破产 | 2024年 | Gainax负债3.8亿日元→2024年5月29日申请破产→12月解散 |

## 3. 市场表现

| 指标 | 数据 |
|------|:--:|
| PM2前3月 | **3万份**销量 + **1万封**粉丝信(PC-98时代 = 天文数字) |
| 系列累计(1996) | 20万份 |
| 系列累计(2005) | **100万+份** |
| 品类影响 | 定义了"养成模拟"(Raising Sim)品类——后来的心跳回忆、卒业等均受其影响 |

## 4. Gainax的崩坏与IP的漂流

赤井孝美2007年因《天元突破》争议辞去Gainax董事→回老家鸟取县创立**米子GAINAX**。PM的IP版权因此分裂:
- 米子GAINAX持有: PM1/2/3/5/Q
- PM4是与GeneX合作→版权混乱
- 2025年D-ZARD获授权开发新PM→但**不能使用PM4角色**——因版权碎片化

## 5. 设计启示

1. **"在电脑里养真人太危险"→奇幻设定 = 品类的起点** — Gainax的"危险感"让他们选择了奇幻背景，这反而成为PM系列的视觉DNA
2. **14,800日元 = 对标光荣的自信心** — 一个动画公司做游戏，敢定光荣级价格→说明Gainax对PM2极度自信
3. **版权碎片化 = IP的慢性死亡** — PM系列被Gainax的崩坏拖累，IP分裂导致30年只出了5部正作
4. **3万份×1万封信 = 核心粉丝的力量** — 在那个没有Steam/社交媒体/主播的时代，1万封手写信意味着什么——真正的"铁粉"
