## 1. 开发团队 — Gainax

### 1.1 公司背景

| 信息 | 内容 |
|:----:|------|
| **公司名称** | Gainax Co., Ltd. (株式会社ガイナックス) |
| **成立时间** | 1984年12月24日 |
| **总部** | 日本东京都 |
| **创始人** | 冈田斗司夫、武田康广、赤井孝美、贞本义行、樋口真嗣、庵野秀明等 |
| **核心业务** | 动画制作 (以《王立宇宙军》《飞跃巅峰》《新世纪福音战士》闻名) |
| **游戏业务** | 1990年代作为副业——PC-98 平台的成人/养成游戏 |
| **破产日期** | 2024年5月29日 (申请破产) / 2024年12月 (解散) |

Gainax 以动画制作闻名于世，Game 部门是公司在 1990 年代初期的一个"副业"。然而，正是这个副业创造了一个定义了整个"养成模拟"品类的游戏。

### 1.2 PM2 核心团队

| 成员 | 角色 | 背景 |
|:----:|:----:|------|
| **赤井孝美 (Takami Akai)** | 制作人 / 原画 / 角色设计 | Gainax 联合创始人，GAINAX 四天王之一，负责全部角色立绘和 CG |
| **加藤正人 (Masato Kato)** | 剧本 / 图形 / 副导演 | 后加入 Square，参与《时空之轮》《异度装甲》等神作剧本 |
| **梶原正裕 (Masahiro Kajihara)** | 作曲 | 配乐风格被誉为"PM2 灵魂" |
| **橋本律郎 (Ritsuro Hashimoto)** | 编程 | PC-98 平台的系统实现 |

赤井孝美是 PM2 的绝对灵魂人物。他不仅是《美少女梦工厂》系列的创造者，也是 Gainax 动画部门的奠基人之一。他定义了"女儿培养模拟"的视觉风格——柔和的水彩风、精致的哥特/奇幻混搭服装、标志性的"女儿"形象——这些元素在后来的 30 多年中成为了品类的视觉基准。

### 1.3 赤井孝美: 从 Gainax 到鸟取县

| 时间 | 事件 |
|:----:|------|
| 1960s | 出生于日本鸟取县 |
| 1984 | 共同创立 Gainax，参与《王立宇宙军》制作 |
| 1991 | 制作初代《美少女梦工厂》 (PC-98) |
| **1993** | **《美少女梦工厂 2》发布** — 获得巨大成功 |
| 1990s-2000s | 持续参与 Gainax 动画（《天元突破》等） |
| 2007 | 因《天元突破》制作方针争议辞去 Gainax 董事 |
| 2008 | 回归故乡鸟取县，创立**米子 GAINAX (Yonago Gainax)** |
| 2014 | 参与 PM2 Refine 版监修 |
| 2020s | 持续从事绘画和角色设计工作 |

## 2. 开发历程

### 2.1 初代《美少女梦工厂》 (1991)

1991 年，赤井孝美在 Gainax 内部主导开发了初代《美少女梦工厂》。这款游戏在当时极为独特:
- 是首批将"女儿模拟养成"作为核心玩法的游戏之一
- 战斗系统使用类《勇者斗恶龙》式的第一人称回合制
- 约 40 种不同结局
- 定价 9,800 日元

初代获得了不错的市场反馈（PC-98 平台销量约 2-3 万份），促使 Gainax 批准了续作的开发。

### 2.2 PM2 的开发过程

| 阶段 | 时间 | 关键事件 |
|:----:|:----:|---------|
| 立项 | 1992年 | 初代成功后立即启动 |
| 原定名 | 1992年 | **"My Fair Child"** — 被否决，原因是"在电脑里养一个真人女孩太危险" |
| 最终设定 | 1992年 | 改为奇幻背景——女儿是"战败魔王的后裔" |
| 核心开发 | 1992-1993年 | 赤井孝美主导全部美术、加藤正人主导剧本和图形 |
| 原版发售 | **1993年6月15日** | NEC PC-9801 平台，定价 **14,800 日元** |
| DOS/V 版 | 1995年 | 日本 PC/AT 兼容机版 |
| 英文版夭折 | 1996年 | SoftEgg 完成翻译 → 发行商 Intracorp 破产 → 翻译版泄露 |
| Refine 版 | **2016年1月28日** | Steam 发售 — 首次官方英文版 + 高清美术 + 新 CG |
| 手游版 | 2018年 | 基于 Refine 版的移动端移植 |

### 2.3 "My Fair Child" — 被放弃的原案

PM2 最初计划使用"My Fair Child"（窈窕淑女→我的好孩子）作为标题，故事设定是玩家作为一个管家/监护人，将一个小女孩培养成"淑女"。赤井孝美后来在采访中回忆:

> "当时我们觉得在电脑里养一个真人女孩太危险了——这不是绅士游戏，但那个设定会让很多人误解。所以我们把整个故事搬到了奇幻世界，女儿是魔王的孩子，你作为勇者收养了她。"

这个决策不仅改变了游戏的故事基调，也成就了 PM 系列最核心的视觉符号——奇幻风格的城堡、铠甲、魔法——这些元素成为了系列延续 30 年的设计 DNA。

## 3. 市场表现

### 3.1 销售数据

| 时间节点 | 销量/数据 | 备注 |
|:--------:|:---------:|------|
| 发售前 3 个月 (1993) | **3 万份** + **1 万封粉丝信** | PC-98 平台 — 万封来信是天文数字 |
| 系列累计 (1996) | 20 万份 | 包含 1 代和 3 代 |
| 系列累计 (2005) | **100 万+ 份** | 全系列、全平台 |
| Steam Refine 版 | 约 10 万+ 份 (估算) | 2016 年后 |

### 3.2 1 万封粉丝信的意义

对于一个在 PC-98 平台发售、定价 14,800 日元的游戏来说，1 万封手写信件是极其罕见的现象。这说明:
- PM2 的核心用户群具有极高的忠诚度
- 游戏触动了玩家的情感——人们不是因为"好玩"而写信，而是因为"感动"
- 这在社交媒体出现之前是衡量"情感影响力"的最直接指标

## 4. Gainax 的崩坏与 IP 的漂流

### 4.1 赤井孝美的出走与 IP 分割

2007 年，赤井孝美因《天元突破》的制作纠纷辞去 Gainax 董事职务，回到故乡鸟取县创立**米子 GAINAX (Yonago Gainax)**。他带走了美少女梦工厂系列的版权，但版权分割并不完整:

| 子系列 | 版权持有方 |
|:------:|:----------:|
| PM1 / PM2 / PM3 / PM5 / PM Q | **米子 GAINAX** (赤井孝美) |
| PM4 | **GeneX / CyberFront** (合作开发，版权复杂) |
| PM4 角色 | 不可在新作中使用 |

### 4.2 Gainax 破产与 PM 系列的未来

2024 年 5 月 29 日，Gainax 在负债 3.8 亿日元后申请破产，同年 12 月解散。这一事件对 PM 系列的直接影响有限（因为版权不在 Gainax 手中），但标志着 Gainax 时代的彻底终结。

2025 年，D-ZARD 公司获得授权开发全新的美少女梦工厂系列作品，但版权碎片化问题导致其**不能使用 PM4 的角色**。

### 4.3 版权碎片化的教训

PM 系列的版权故事是游戏行业中关于"IP 管理"的最佳反面教材：
- 一个成功的 IP 因创始人内斗而分裂
- 版权分散导致 30 年仅出 5 部正作
- 新作因版权限制不能使用系列最具人气的部分角色

## 5. 设计启示

1. **"在电脑里养真人太危险"→ 奇幻设定 = 品类的起点** — Gainax 的"危险感"让他们选择了奇幻背景，这反而成为 PM 系列最独特的视觉 DNA

2. **￥14,800 = 对标光荣的自信心** — 一个动画公司做游戏，敢定光荣级（行业最高）的价格——这说明了 Gainax 对 PM2 的极度自信

3. **70+ 结局 = 内容深度的复利** — PM2 之所以能在 PC-98 时代引发"粉丝信海啸"，不是因为画面（不如 SFC）、不是因为音乐（好但有限），而是玩家每次通关都想"再试一次，看看另一个结局"

4. **版权碎片化 = IP 的慢性死亡** — 美少女梦工厂系列的衰落不是因为市场需求消失，而是版权持有者的内斗让产品供给断裂

5. **"女儿养成"的普世吸引力** — PM2 的成功证明"养女儿"不是一个日本本土化的文化现象，它在韩国、中国、欧美都有巨大的市场——这为后来者（火山的女儿、中国式家长等）提供了信心
