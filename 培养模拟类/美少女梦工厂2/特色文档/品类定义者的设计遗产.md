# 品类定义者的设计遗产 - 美少女梦工厂2(Princess Maker 2 Refine)

> 游戏版本: Princess Maker 2 Refine (PC/Steam, 1993/2016)
> 数据来源: 跨游戏对比分析, 基于本采集系列全部12款培养模拟游戏

## 一、PM2定义了品类的"语法"

几乎后续所有养成游戏的核心系统，都能在PM2中找到原型:

| PM2系统 | 后续继承者 | 现在的名称 |
|------|------|------|
| 每旬安排1项活动 | 全部养成游戏 | 日程安排系统 |
| 10大属性+6战斗+6技能 | 全部养成游戏 | 多维属性体系 |
| 打工/课程/休息/冒险 四选一 | 火山的女儿/梦幻魔法公主 | 活动分类系统 |
| 属性→评价→结局的映射 | 全部养成游戏 | 结局判定系统 |
| 疲劳>体力=死亡 | 女王万岁(生存压力)/是的陛下(资源归零) | 失败条件 |
| 王子(每年1月见1次) | 梦幻魔法公主(女王周目锁) | 长期攻略路线 |
| 父嫁(可嫁给养育者) | —(PM2独有/极少数模仿) | 品类的禁果 |
| 74结局的属性组合判定 | 火山的女儿(58结局)/梦幻魔法公主(51结局) | 多结局系统 |
| 血型+守护神=先天差异 | 梦幻魔法公主(学级选择)/偶像经理人(随机特性) | 初始差异化 |

## 二、PM2的设计贡献

### 2.1 活动系统 = 属性交易
PM2首创了"每项活动同时提升和降低属性"的设计——这是资源管理思维在养成游戏中的应用。

### 2.2 隐藏评价系统 = 你不知道你在培养她成为什么
四大评价(战士/魔法/社交/家事)是隐藏的——玩家只能通过结局来"看到"自己8年选择的结果。这种不透明性是PM2最独特的设计之一。

### 2.3 罪孽系统 = 道德约束的量化
罪孽>100 = 永远无法达成善结局。这是游戏设计中最早的"道德不可逆"机制。

### 2.4 不良化 = 属性下降的情感化
当压力超过道德/信仰→女儿真的会"变坏"。这不是数字变化——她会在对话中表达不满、拒绝你的安排、甚至离家出走。

### 2.5 武者修行 = 开放世界的雏形
四大地图各自有独特的生态和事件。这在1993年是极为超前的设计。

## 三、PM2的局限(被后续游戏改进的)

| PM2的局限 | 后续游戏的改进 |
|------|------|
| 无多周目继承 | 火山的女儿(NG+继承点)/梦幻魔法公主(继承碎片) |
| 无"日程模板" | 偶像经理人(可批量安排)/中国式家长(一键重复) |
| 无路线提示 | 梦幻魔法公主(画廊→结局条件可见) |
| 评价完全隐藏 | 火山的女儿(属性面板可见所有数值) |
| 288回合=无弹性 | 梦幻魔法公主(36月=更精炼) |
| 课程学费无明细 | 后续游戏(透明化所有成本) |

## 四、PM2与当代养成游戏的对比

| 维度 | PM2 (1993) | 梦幻魔法公主 (2026) |
|------|------|------|
| 属性数 | 10+6+6=22 | 4×4=16 |
| 培养周期 | 8年×36旬=288回合 | 3年×36月=36月(上课+白天+晚上) |
| 结局数 | 74 | 51+8+1 |
| 战斗 | 武者修行(可选) | 回合制RPG(深度整合) |
| 多周目 | 无 | 三周目揭示/继承碎片 |
| 美术 | 16色/全彩重绘 | Live2D/手绘/全语音 |
| 信息透明 | 评价隐藏/无攻略辅助 | 画廊内置结局条件/选项追踪 |

33年的时间跨度 → PM2的设计DNA依然清晰可辨 → 这就是"品类定义者"的意义。

## 五、设计启示

1. **PM2是养成游戏设计的"第一性原理"**: 当你在设计一款新的养成游戏时 → 你需要解释的是"为什么和PM2不同" → 而不是"为什么像PM2"
2. **隐藏评价 = 有争议但有力的设计**: 后续游戏普遍选择了更透明的属性→结局映射 → 但PM2的不透明性创造了独特的紧张感
3. **288回合 = 时间密度太高**: 后续游戏普遍选择了更少的回合数 → 每个选择更重 → PM2的微观管理在当代看来过于细致
4. **罪孽系统 = 不可逆设计的力量**: 没有任何当代养成游戏敢像PM2一样 → 让你"永远锁死"善结局 → 这是设计勇气的体现
