# 日程与行动系统 - 莱尔兰德物语(Lair Land Story)

> 游戏版本: Lair Land Story v1.0 (PC/Steam, 2020年4月; Nintendo Switch, 2022年8月)
> 原始版本: 2007年 (中文名: 兰岛物语)
> 数据来源: Steam商店页, Steam社区, LadiesGamers, Rice Digital, DigitallyDownloaded, Siliconera

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 培养周期 | **4年** |
| 时间单位 | **季节**(Season) |
| 每季动作 | **每周**安排活动 |
| 每周安排 | 最多**3项活动** |
| 每季结束 | **奖励目标(Bonus Goal)**结算 |

## 2. 每周行动分配

每周可为Chilia安排最多**3项活动**，从4大类中选择:

### 2.1 四大活动类型

| 类型 | 效果 | 附加收益 | 经验成长 |
|------|------|------|:--:|
| **学习(Study)** | 主要属性提升 | 知识积累 | ✅ 等级提升→更大属性增益 |
| **工作(Work)** | 较小属性提升 | **金币收入** | ✅ 等级提升→更多收入+属性 |
| **城市重建(Rebuild)** | 城市经济↑ | 解锁新事件+场景变化 | — |
| **休息(Rest/Other)** | 降低Exhaustion+Tension | 恢复状态 | — |

### 2.3 活动→经验等级机制
- 学习和工作各自有**经验等级**
- 重复同一类活动→等级提升→每次属性增益和金币收入提高
- **专业化**(Specialization)比通才更高效
- 一周目时间不足以培养全能型Chilia→必须每轮专注特定方向

## 3. 状态管理系统

### 3.1 双重疲劳机制

| 状态 | 积累方式 | 惩罚 |
|------|------|------|
| **Exhaustion(疲劳)** | 学习/工作持续累积 | **性能下降**(属性增益减少) |
| **Enthusiasm(热情/Tension)** | 持续单调活动导致下降 | **性能下降**(属性增益减少) |

### 3.2 休息的价值
- 缓解Exhaustion(降低疲劳)
- 恢复Enthusiasm(提高热情)
- 休息不止是"不做事的空隙"——是维持效率的核心策略
- 平衡学/工/休三者是关键

## 4. 城镇访问系统

每周除安排活动外，可选择**3个地点**访问:

| 地点 | 功能 |
|------|------|
| **广场(Plaza)** | 触发公共事件/偶遇NPC |
| **宫殿(Palace Gates)** | 政治/贵族线剧情 |
| **酒馆(Bar)** | 社交/信息收集 |
| **美容院(Salon)** | 发型/外观变更 |
| **裁缝店(Sewing Shop)** | 服装/泳装购买 |
| **公共浴场(Bath)** | 特殊事件(战后改为混浴) |

- 不同**时段**访问同一地点可能触发不同事件
- 地点访问是触发**故事事件**和**角色关系**的主要方式
- 某些路线进展需要访问特定地点，但游戏没有明确指引→存在试错成本

## 5. 奖励目标系统(Bonus Goal)

| 要素 | 说明 |
|------|------|
| 设定时机 | 每季开始时 |
| 目标类型 | 属性达到特定值/城市经济水平/特定事件完成等 |
| 达成奖励 | **Bonus Points**(跨周目继承) |
| 失败 | 无惩罚，但失去Bonus Points机会 |

## 6. 设计特点分析

1. **经验等级→专业化驱动**: 不像固定效率的课程→重复同一领域会成长→自然的专业化路径而非强迫
2. **双重疲劳系统(Tension+Exhaustion)**: 传统养成只有"疲劳"一个指标→Tension的设计意味着"无聊"也是问题→不只是身体的疲惫也要关注心理状态
3. **3次城镇访问=叙事发现层**: 不是菜单式选地点→需要自己探索→但缺乏指引带来挫折
4. **Bonus Goal=短期目标锚点**: 4年太长→每季1个短期目标→给玩家持续的进度感
5. **战后重建=养成之外的宏观叙事**: 不只是养女儿→也是在重建一个战后城市→双线叙事
