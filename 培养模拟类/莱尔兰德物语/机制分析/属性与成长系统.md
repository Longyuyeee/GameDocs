# 属性与成长系统 - 莱尔兰德物语(Lair Land Story)

> 游戏版本: Lair Land Story v1.0 (PC/Steam, 2020; Switch, 2022)
> 原始版本: 2007年
> 数据来源: Steam商店页, LadiesGamers, Rice Digital, DigitallyDownloaded, Siliconera

## 1. 属性体系架构

### 1.1 属性数量
- 约**12项**("a dozen or so categories")
- 每个属性影响Chilia的个性、能力和结局方向

### 1.2 已知属性类型(基于多个评测推断)

| 属性领域 | 关联活动 | 典型影响 |
|------|------|------|
| **智力/学识(Wisdom/Intelligence)** | 学习课程 | 学术类结局/魔法能力 |
| **体力/力量(Strength)** | 体力工作 | 体力类工作收入 |
| **魅力/气质(Charisma/Charm)** | 社交活动 | 社交事件/好感度 |
| **信仰/虔诚(Faith/Piety)** | 宗教活动 | 神殿相关事件/结局 |
| **艺术/创作(Arts/Creativity)** | 艺术课程/研究 | 艺术类结局 |
| **哲学/思想(Philosophy)** | 哲学课程 | 思想类结局 |
| **社交/交际(Social)** | 城镇访问/社交事件 | 人际关系 |
| **家政/生活(Housekeeping)** | 家务/打工 | 生活类结局 |
| **魔法/神力(Divine Affinity)** | 特殊研究 | 神力相关事件 |
| **道德/善良(Morality/Kindness)** | 城市重建/慈善 | 善路线结局 |

注: 具体属性名称在公开资料中未完整列出。以上基于评测中提到的"arts/philosophy/piety"等培养方向推断。

## 2. 属性提升机制

### 2.1 主要途径

| 方式 | 效率 | 副作用 | 经验成长 |
|------|:--:|------|:--:|
| **学习** | ★★★★★ | Exhaustion↑/Enthusiasm↓ | ✅ |
| **工作** | ★★★ | Exhaustion↑/Enthusiasm↓ | ✅ |
| **城市重建** | ★★ | Exhaustion↑ | — |
| **特殊事件** | ★★★★ | 少量/无 | — |
| **道具使用** | ★★ | 无 | — |

### 2.2 属性提升的代价机制
- 大多数活动**提升1-2项属性**，但同时**微量降低其他1-2项属性**
- 这意味着不可能培养"全能型"Chilia——必须在属性之间做取舍
- 专业化(specialization)是设计的核心意图

### 2.3 经验等级→效率成长
```
学习Lv1: +10智力/次(假设)
学习Lv2: +13智力/次
学习Lv3: +16智力/次
...
```
具体数值未公开，但趋势是专业领域效率递增。

## 3. 双重状态系统

### 3.1 Exhaustion(疲劳)

| 参数 | 说明 |
|------|------|
| 积累方式 | 每项活动(学习/工作/重建)增加 |
| 过高后果 | 属性增益大幅减少 |
| 降低方式 | 休息/放松活动 |

### 3.2 Enthusiasm/Tension(热情/紧张)

| 参数 | 说明 |
|------|------|
| 变化方式 | 重复单调活动→Tension**下降**(热情降低) |
| 特殊设计 | Tension是**下降**值(down=bad)——这在养成游戏中较少见 |
| 过低后果 | 性能下降，Chilia"无聊"会影响效率 |
| 恢复方式 | 休息/多样化活动安排 |

### 3.3 双状态管理策略
```
不是: 学习→学习→学习→休息
而是: 学习→工作→重建→休息 (多样化维持Enthusiasm)
```

## 4. 神力亲和系统(Divine Affinity)

| 要素 | 说明 |
|------|------|
| 定位 | 特殊属性/解锁神力相关祝福 |
| 获取 | 特定宗教/神力活动 |
| 效果 | 解锁特殊祝福/增益/事件 |

## 5. 属性→Chilia的行为反馈

通过对Chilia的提问回答，影响她的**特质(traits)**和**观点(opinions)**:
- 属性数值影响可选的回答
- 回答影响Chilia的世界观和个性
- 个性+属性组合→结局判定

## 6. 设计特点分析

1. **~12属性×专业化的零和博弈**: 不像梦幻魔法公主的"4×16都能培养"，莱尔兰德的属性间是零和的——提升某些属性会微降其他——打破了养成游戏中"全都要"的幻想
2. **双重疲劳=身体+心理的完整模拟**: Exhaustion(身体)+Enthusiasm(心理)→比单一疲劳更立体的状态模拟
3. **回答问题→培养性格而非数值**: Chilia会提问，回答塑造她的性格而不仅仅是数值→这是经典Princess Maker式的互动
4. **经验等级=隐性引导专业化**: 不需要提示"请选择专业方向"→自然的重复杂活动获得更高回报→行为经济学式设计
5. **神力亲和=养成+信仰的融合**: 将"信仰"系统化而非仅仅是一个属性→有独立的祝福/奖励体系
