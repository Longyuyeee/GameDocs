> 原始版本: 2007年
> 数据来源: Rice Digital, Steam商店页, LadiesGamers, DigitallyDownloaded

| 货币 | 说明 |

## 4. 城市经济系统

### 4.1 战后经济衰退
- 游戏开始时城市处于战后状态
- 物价可能较高/资源匮乏

### 4.2 重建→经济恢复
```
重建投入 → 城市经济↑ → 物价合理化 → 更多消费选择 → 解锁新事件
```

### 4.3 经济与事件的关联
- 城市经济水平影响可触发的事件类型
- 经济繁荣后新商店/新角色出现

## 5. 金钱管理策略

### 5.1 平衡挑战
- 评测反映: "starting with a comfortable sum that can quickly deplete"
- 初期看似有足够金币→但如果忽略工作→很快陷入财务困境
- 必须在**学习/工作/重建/休息**之间分配有限的每周3个活动槽

### 5.2 工作vs学习的权衡
```
学习 = 高属性增益 + 0金币
工作 = 低属性增益 + 金币收入
→ 纯学习路线=高属性但可能财务困难
→ 纯工作路线=有钱但可能触发不了高贵结局
```

### 5.3 推荐策略
- 初期: 适当工作积累金币基础
- 中期: 交替学习+工作，保持财务健康
- 后期: 根据目标结局调整(某些结局重属性轻财富，某些相反)

## 7. 设计特点分析

1. **金币→学习材料的隐性成本**: 不是所有活动都直接消耗金币→但纯学习路线资源不足→自然的平衡约束
2. **城市重建=公共物品投资**: 花钱重建城市→降低所有人的物价→利他的行为最终利己
3. **房间装饰=经济富余的展示**: 初始房间"空荡荡的只有箱子"→装饰后温馨→金币不仅是数字也是情感空间
4. **战后经济=叙事中的经济模拟**: 不是"正常城市"→战后通货膨胀/资源匮乏→重建的满足感
5. **短期充裕→长期紧张**: 舒适的起始资金让人放松→快速消耗→促使面对现实→设计的经济张力
