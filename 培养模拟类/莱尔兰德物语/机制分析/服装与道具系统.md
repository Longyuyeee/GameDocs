# 服装与道具系统 - 莱尔兰德物语(Lair Land Story)

> 游戏版本: Lair Land Story v1.0 (PC/Steam, 2020; Switch, 2022)
> 原始版本: 2007年
> 数据来源: Rice Digital, LadiesGamers, Steam商店页, DigitallyDownloaded

## 1. 服装系统

### 1.1 获取方式

| 方式 | 地点 | 消耗 |
|------|------|:--:|
| 裁缝店购买 | Sewing Shop | 金币 |
| 事件奖励 | 特定事件完成后 | — |
| 泳装 | 裁缝店 | 金币 |

### 1.2 外观变化
- Chilia的立绘/外观会反映当前服装
- "costumes and hairstyles"是可收集的视觉元素
- 服装不仅影响外观，可能也影响社交活动

### 1.3 服装类型(推断)

| 类型 | 用途 |
|------|------|
| 日常服 | 默认外观 |
| 正式礼服 | 重要场合/城镇活动 |
| 泳装 | 浴场/夏季事件 |
| 职业装 | 对应不同职业路线 |

## 3. 房间装饰系统(Home Decoration)

### 3.1 初始状态
- "quite depressingly empty and filled with boxes"
- 战后空旷的房间

### 3.2 装饰意义
- 购买的装饰品**实际展示在Chilia的房间中**
- 这是她的**私人空间的改善**
- 从"空荡荡的箱子堆"→温馨的房间
- 视觉上反映Chilia成长和Herol关怀的情感空间

### 3.3 装饰品来源
- 商店购买
- 事件奖励
- 研究制作

- 战后物价高→重建后物价更合理

## 7. 设计特点分析

1. **房间装饰=情感空间的设计**: 初始的"空箱子堆"→装饰温馨→这是养成游戏中最直接的空间情感表达
2. **研发系统=Chilia的自主性**: 不是所有道具都靠买→Chilia自己可以研发→她不只是"被养的人"
3. **发型+服装=视觉养成的乐趣**: 看到女儿穿着你买的衣服、梳着你付款的发型→视觉上的养成成就感
4. **泳装=品类的趣味性**: 评测专门提到"cute swimsuits"→轻松的养成乐趣
5. **战后经济→消费体验的阶段性**: 初期物价高/选择少→重建后商店更多/价格更合理→消费体验的渐进解锁
