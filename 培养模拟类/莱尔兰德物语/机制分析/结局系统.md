# 结局系统 - 莱尔兰德物语(Lair Land Story)

> 游戏版本: Lair Land Story v1.0 (PC/Steam, 2020; Switch, 2022)
> 原始版本: 2007年
> 数据来源: Rice Digital, Siliconera, LadiesGamers, DigitallyDownloaded, Steam商店页

## 1. 结局总览

| 参数 | 说明 |
|------|------|
| 结局数量 | **大量**("a large number of endings available") |
| 结局类型 | 职业结局 + 感情结局 + 特殊结局 |
| 判定方式 | 属性组合+事件完成+关系选择 |
| 有无"最好"结局 | **没有**——结局只是"不同"，没有最优 |

> 注: 公开资料中未列出具体结局数量。Steam商店页标注"Multiple Endings"标签，评测一致确认有大量结局。

## 2. 结局判定机制

### 2.1 多因素判定
```
结局 = f(属性组合, 完成的事件线, Chilia的感情选择, Herol的感情选择, 城市重建程度, 神力亲和)
```

### 2.2 结局方向推断(基于评测)

| 结局方向 | 核心决定因素 | 说明 |
|------|------|------|
| **职业结局** | 最高属性领域 | 学者/艺术家/神职者/战士/商人等 |
| **Chilia感情结局** | Chilia与追求者的好感度 | 与哪位追求者在一起 |
| **Herol感情结局** | Herol与对象的进展 | 与谁结婚/独身 |
| **城市结局** | 城市重建程度 | 城市繁荣度/政治稳定 |
| **特殊结局** | 隐藏条件 | 真结局/隐藏路线 |

## 4. 多周目与结局收集

### 4.1 单周目限制
- **不可能一周目全收集**
- 属性零和→每次培养只能专注少数方向
- 多条叙事线竞争→一次只能深入少数路线
- "simply not possible to do everything in a single playthrough"

### 4.2 全收集时长
- 评测估计: 全结局/全收集需**400+小时**

### 4.3 为什么要多周目
1. 不同属性培养方向→不同结局群
2. 不同关系选择
3. Bonus Points购买新特权→尝试新策略
4. 城市重建的不同侧重

## 5. 结局设计哲学

来自评测: "the game is about the journey, not the ending"

| 设计思想 | 含义 |
|------|------|
| 无最好结局 | 不评判玩家的选择 |
| 结局=旅程的自然终点 | 不是"目标"，而是你选择的累积 |
| 全部值得体验 | 每种路线都有其独特之处 |

## 6. 设计特点分析

1. **400+小时全收集=极端的重玩驱动**: 不是轻度多周目→是"你需要投入400小时"的硬核承诺
2. **没有最优结局=去中心化设计**: 不像"真结局是最好结局"→每个结局平等→尊重玩家选择
3. **双主角感情=结局的组合爆炸**: Herol选择×Chilia选择→大量组合→但可能稀释每个结局的力度
4. **城市重建=第四维度结局**: 不只是养成对象的结局→城市本身也有结局状态
5. **单周目限制=诚实的边界**: 游戏明确告诉玩家"做不完所有事"→减少完美主义焦虑
