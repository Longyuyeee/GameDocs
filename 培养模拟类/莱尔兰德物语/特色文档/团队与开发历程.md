# 团队与开发历程 — 莱尔兰德物语(Lair Land Story)

> 开发商: CIRCLE Entertainment (香港) | Initory Studio
> 发行年份: 2007年(原版)/2022年(重制) | PC/Switch | 团队规模: **原版2人**

## 1. 开发团队(原版2007年)

| 成员 | 角色 |
|------|------|
| **Chris** | 制作人/编剧/编程——21岁辞去工作后创立CIRCLE |
| **Manny** | 原画师——公司唯一的另一名员工 |

仅2人。音乐和上色外包。开发时间仅**10个月**。

## 2. 开发历程

| 阶段 | 关键事件 |
|------|------|
| 2007年 | PC原版发布——繁体中文(台湾EA)/简体中文(北京娱乐通) |
| 2009年 | **PSP移植**——22岁的Chris花掉最后几千港币**飞日本向Arc System Works总裁推销**→成交! 加入日本声优(含早见沙织) |
| 2013年 | 续作《奇沧的幻海》宣布→推进至50%(50万+字剧本)→停滞 |
| 2020年 | **PC重制版**Steam发布 |
| 2022年 | **Switch版**全球发布(PQube发行) |

## 3. 商业表现

| 指标 | 数据 |
|------|:--:|
| PC原版 | ~5万份 |
| PSP版 | ~1万份 |
| 是否回本 | **从未完全收回成本** |

Chris后来转型发行——在中国大陆创立**薪火游戏**(CIRCLE的子公司)，将日本游戏引入中国市场。

## 4. 设计启示

1. **2人×10个月 = 一个品类的跨界实验** — 两人做出了城市重建+养成的双重系统——野心远大于团队规模
2. **22岁飞日本推销 = 独立开发者的销售勇气** — Chris花最后几千港币飞日本，说服Arc System Works做PSP版——这不是"商业计划"，这是拼尽全力
3. **从未回本 ≠ 失败** — 续作虽然停滞，但Chris通过发行找到了生存之道。游戏开发者的职业路径不是只有"做出爆款"
4. **续作停滞=完美主义的陷阱** — "觉得续作无法超越原版"而放弃——这可能是正确的，但也可能是在50%处放弃了一个好作品
