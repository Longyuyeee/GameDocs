# 结局数据总览 - 莱尔兰德物语(Lair Land Story)

> 游戏版本: Lair Land Story v1.0 (PC/Steam, 2020; Switch, 2022)
> 原始版本: 2007年
> 数据来源: Rice Digital, Siliconera, LadiesGamers, DigitallyDownloaded, Steam商店页

> 注: 本游戏未公开具体结局数量和触发条件。以下基于评测描述和游戏机制推断。

## 一、结局总览

| 参数 | 说明 |
|------|------|
| 结局总数 | **大量**(具体数字未公开) |
| 结局分类 | 职业结局 + Chilia感情结局 + Herol感情结局 + 城市结局 |

## 三、感情结局组合矩阵(推断)

|  | Herol单身 | Herol×青梅竹马 | Herol×卫队长 | Herol×其他 |
|------|:--:|:--:|:--:|:--:|
| Chilia单身 | 双独立结局 | 只有Herol结婚 | 只有Herol结婚 | 只有Herol结婚 |
| Chilia×追求者A | 只有Chilia恋爱 | 双重恋爱 | 双重恋爱 | 双重恋爱 |

注: 双重恋爱结局(Herol和Chilia各自有伴侣)是此游戏的独特设计。

## 四、城市结局维度(推断)

| 城市状态 | 判定因素 |
|------|------|
| 繁荣 | 重建投入高 + 经济等级高 |
| 恢复中 | 中等重建投入 |
| 萧条 | 重建被忽视 |
| 政治稳定 | 政治事件线完成 |
| 政治动荡 | 潜伏的邪恶未被平息 |

## 五、结局判定优先级(推断)

```
当多个结局条件同时满足时:
1. 隐藏/真结局 > 普通结局
2. 职业结局 + 感情结局 可以并存(组合式结局)
3. 属性最高→决定职业方向
4. 未完成的关系/事件线→对应方向结局不触发
```

## 六、结局收集路线建议

| 周目 | 推荐方向 | 关键行动 |
|:--:|------|------|
| 一周目 | 学术/智力路线 | 专注学习→学者/魔法师结局 |
