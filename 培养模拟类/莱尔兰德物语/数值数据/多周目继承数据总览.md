# 多周目继承数据总览 - 莱尔兰德物语(Lair Land Story)

> 游戏版本: Lair Land Story v1.0 (PC/Steam, 2020; Switch, 2022)
> 原始版本: 2007年
> 数据来源: Rice Digital, Siliconera, LadiesGamers, DigitallyDownloaded

## 一、多周目继承概述

莱尔兰德物语使用**Bonus Points系统**作为跨周目的继承机制。玩家在多周目中购买特权来改变游戏体验，而非直接继承属性/金币。

## 二、Bonus Points系统

### 2.1 获取方式

| 获取方式 | 数量 | 频率 |
|------|:--:|:--:|
| **达成Bonus Goal** | 固定 | 每季1次 |
| 完成特定结局 | 额外? | 每次结局 |

### 2.2 累计与消耗

| 操作 | 说明 |
|------|------|
| 存入 | 每季达标后自动获得 |
| 消耗 | 新周目开始时购买特权 |
| 跨周目 | 完全继承(不会消失) |

### 2.3 可购买特权(推断)

| 特权类别 | 效果 |
|------|------|
| 初始金币加成 | 新周目起始金币↑ |
| 属性加成 | 初始属性值↑ 或 成长倍率↑ |
| 活动效率↑ | 学习/工作效果提升 |
| 疲劳抗性 | Exhaustion累积速度↓ |
| 热情保持 | Enthusiasm下降速度↓ |
| 新选项解锁 | 新的活动/对话选项 |
| 隐藏路线 | 可能解锁特殊结局路线 |
| 好感度加成 | 初始NPC好感度↑ |

## 三、继承对比

| 游戏 | 继承类型 | 继承内容 |
|------|:--:|------|
| 火山的女儿 | NG+点数+服装保留 | 服装+继承点换加成 |
| 梦幻魔法公主 | 关键道具传递 | 继承碎片(唯一跨周目道具) |
| **莱尔兰德物语** | **Bonus Points购买制** | **玩家自己选择继承什么** |

## 四、Bonus Points的经济设计

### 4.1 每季可获得上限估算
```
每年4季 × 4年 = 16个Bonus Goal
假设每达成1个Goal获得10-50 Bonus Points:
一周目: 160-800 Bonus Points
```

### 4.2 特权消费(推断)

| 特权 | 大致成本(BP) | 优先级 |
|------|:--:|:--:|
| 小初始金币 | 50-100 | ★★ |
| 属性加成(小) | 100-200 | ★★★ |
| 活动效率↑ | 200-400 | ★★★★★ |
| 新选项解锁 | 300-500 | ★★★★ |
| 隐藏路线 | 500+ | ★★★★★ |

## 五、多周目策略

### 5.1 一周目建议
- 专注**达成尽可能多的Bonus Goals**(最大化BP积累)
- 不追求特定结局→探索系统
- 积累BP为后续周目打基础

### 5.2 二周目建议
- 使用BP购买**活动效率提升**特权
- 尝试与一周目不同的培养方向
- 开始探索不同的关系线

### 5.3 三周目+建议
- BP积累充足→购买高级特权
- 冲刺隐藏结局/真结局
- 全收集路线(400+小时)

## 六、设计特点分析

1. **购买制而非自动继承**: 玩家主动选择继承什么→继承了PM/Princess Maker系列的Bonus Point设计
2. **Bonus Goal=短期目标锚点**: 每季1个可达成的小目标→长期的4年变得可管理
3. **BP=每周目的效率增量**: 一周目最难→后续越来越轻松→但内容驱动玩家继续
4. **特权而非数值=保留新鲜感**: 不直接继承属性→每周目仍需从头培养→避免"NG+碾压"
5. **400+小时全收集=极端硬核**: 不是轻度多周目→是为最硬核的养成游戏玩家设计的终极挑战
