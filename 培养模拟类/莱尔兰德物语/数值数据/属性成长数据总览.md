# 属性成长数据总览 - 莱尔兰德物语(Lair Land Story)

> 游戏版本: Lair Land Story v1.0 (PC/Steam, 2020; Switch, 2022)
> 原始版本: 2007年
> 数据来源: Rice Digital, LadiesGamers, DigitallyDownloaded, Siliconera

> 注: 本游戏未公开具体的属性数值表和成长曲线。以下基于机制描述推断。

## 一、属性体系

### 1.1 属性架构
约**12项属性**("a dozen or so categories")，大致分类:

| 属性 | 类型 | 主要提升方式 | 关联结局领域 |
|------|:--:|------|------|
| 智力/学识 | 核心 | 学术课程 | 学术/魔法 |
| 体力/力量 | 核心 | 体力工作/训练 | 战士/体能职业 |
| 魅力/气质 | 核心 | 社交活动/礼仪课程 | 社交/名媛 |
| 信仰/虔诚 | 核心 | 宗教活动 | 神职/圣女 |
| 艺术/创作 | 核心 | 艺术课程 | 艺术/创作职业 |
| 哲学/思想 | 核心 | 哲学课程 | 哲学家/导师 |
| 社交/交际 | 核心 | 城镇访问/社交活动 | 外交/社交圈 |
| 家政/生活 | 生活 | 家务/打工 | 生活/家庭结局 |
| 神力亲和 | 特殊 | 宗教/特殊研究 | 神力祝福/特殊结局 |
| 道德/善良 | 品行 | 城市重建/善行 | 善/恶路线 |
| (其他~2项) | — | — | — |

## 二、属性提升的计算(推断)

### 2.1 单次活动属性增益
```
属性增益 = 活动基础值 × 经验等级系数 × (1 - 疲劳惩罚) × (1 + 热情加成) × (1 + BP特权加成)
```

### 2.2 经验等级系数(推断)
| 活动经验等级 | 效率系数 |
|:--:|:--:|
| Lv1 | 1.0× |
| Lv2 | 1.2× |
| Lv3 | 1.4× |
| Lv4 | 1.6× |
| Lv5+(Max) | 2.0× |

### 2.3 疲劳惩罚(推断)
| 疲劳程度 | 增益惩罚 |
|:--:|:--:|
| 低(0-30%) | 0% |
| 中(30-60%) | -20% |
| 高(60-90%) | -50% |
| 极高(90%+) | -80% |

### 2.4 热情加成(推断)
| 热情程度 | 增益加成 |
|:--:|:--:|
| 高 | +20% |
| 中 | +0% |
| 低 | -30% |

## 四、属性互斥设计

评测明确指出: 大多数活动提升1-2项属性**同时微量降低其他1-2项属性**。

| 活动 | 提升属性 | 降低属性 |
|------|------|------|
| 学术课程 | 智力↑↑ | 体力↓(微) |
| 体力训练 | 体力↑↑ | 智力↓(微) |
| 宗教活动 | 信仰↑↑ | 哲学↓(微) |
| 艺术课程 | 艺术↑↑ | 智力↓(微) |

这种零和设计意味着:
- **不可能培养全能型Chilia**
- 每次选择都有代价
- 专业化是必然的(而非可选的)

## 五、隐藏属性(推断)

| 隐藏属性 | 说明 | 影响 |
|------|------|------|
| Chilia的"性格" | 通过回答提问塑造 | 对话选项+事件触发 |
| 父女亲密度 | Herol-Chilia关系 | 特殊事件+结局 |
| 城市繁荣度 | 重建投入的结果 | 物价+解锁 |
| 政治稳定度 | 阴谋线处理结果 | 结局 |

## 六、4年的属性成长预期(推断)

### 学术路线
```
第1年: 智力 200→400
第2年: 智力 400→800
第3年: 智力 800→1400
第4年: 智力 1400→2000(顶级学者结局阈值)
```

### 均衡路线(不推荐)
```
第1年: 各项 100→200
第2年: 各项 200→350
第3年: 各项 350→500
第4年: 各项 500→700(只能达到中级结局)
```

注: 均衡路线由于属性互斥机制，各项属性都达不到顶级阈值。

## 七、属性培养效率最大化策略

1. **尽早确定专业方向**(第1年上半)
2. **同一类活动重复**(获取经验等级加成)
3. **管理疲劳和热情**(保持在高效区间)
4. **利用Bonus Points特权**(加速成长)
5. **不要贪多**(专注于少数核心属性)
