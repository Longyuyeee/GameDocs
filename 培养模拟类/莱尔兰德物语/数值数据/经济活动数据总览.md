## 四、初期经济策略(推断)

### 阶段一: 资金积累(第1年)
```
重点: 适当工作→金币储备(评测反映初期钱"舒适但快速耗尽")
建议: 2学习+1工作 或 1学习+1工作+1休息
目标: 积累基础金币储备(~?)
```

### 阶段二: 均衡发展(第2-3年)
```
重点: 工作+学习均衡→保持财务健康
建议: 根据目标结局调整比例
目标: 开始城市重建投入
```

### 阶段三: 冲刺(第3-4年)
```
重点: 根据目标结局方向→最大化关键属性
低财富需求结局: 增加学习比例
高财富需求结局: 保持工作比例
```

## 五、跨周目经济(Bonus Points)

| 获取方式 | 每季达成Bonus Goal |
|:--|:--|
| 用途 | 购买下周目特权 |
| 可能特权 | 初始金币↑/属性加成/解锁新选项/活动效率↑ |

## 六、经济系统评价

基于评测:
- **优势**: 双重劳动回报(金币+属性)+城市重建投资→有层次的经济系统
- **劣势**: 如果忽略工作→财务将快速恶化→自由活动受限
- **最佳实践**: 永远不要让金币归零，保持一定工作频率
