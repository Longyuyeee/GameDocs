# 团队与开发历程 — 温泉物语

> 开发商: 开罗社(Kairosoft Co., Ltd.)
> 发行年份: 2011(iOS/Android) → 2022(Steam) | 平台: PC/Steam/Switch/iOS/Android | 团队规模: ~9-15人

---

## 1. 开发团队

温泉物语由**开罗社(Kairosoft)**同一团队开发。详见[游戏开发物语-团队与开发历程](../游戏开发物语/特色文档/团队与开发历程.md)中的开罗社背景。

温泉物语是在Game Dev Story全球爆红后的**第二款现象级作品**——它证明了开罗不是"一招鲜"，而是一套可以迁移到不同主题的设计公式。

## 2. 开发历程

### 时间线

| 年份 | 事件 |
|:--:|------|
| 2010 | Game Dev Story iOS版全球爆红，Metacritic 86分，GDC提名 |
| 2011 | **温泉物语 iOS/Android发布**——选择了与"游戏开发公司"完全不同的主题(温泉旅馆) |
| 2018 | Nintendo Switch版发布 |
| 2022.3.27 | **Steam版发布**(与Game Dev Story Steam版同日) |
| 2022.12 | 续作《温泉物语2》Steam发布 |

### 从Game Dev Story到温泉物语的设计进化

| 设计维度 | Game Dev Story | 温泉物语 | 进化方向 |
|------|:--:|:--:|------|
| 空间系统 | 无(纯工位制) | **完整的景观辐射+契合度系统** | 加入了"空间"维度 |
| 员工系统 | 复杂(6职业线转职树) | 简单(分配制) | 弱化以平衡新增的空间复杂度 |
| 客人系统 | 统计数据(无个体) | **49种独立偏好客人** | 从"统计"到"个体" |
| 竞争系统 | GGA年度大奖 | 8大杂志排名 | 从"单一年奖"到"多维度评比" |
| 终局 | 21年固定结算 | 15年结算+可自由继续 | 更灵活 |

> 开罗在温泉物语中证明了一件事：**可以抛弃前作最成功的核心系统(转职树)，用全新的核心系统(空间布局)替代，同时保持"开罗公式"的本质**——组合触发、半隐藏规则、阶段门设计。这是开罗跨主题设计能力的有力证明。

## 4. 设计启示

### 对商业经营品类的贡献

1. **"空间即产品"** — 温泉物语是开罗系列中第一个将空间布局提升为核心系统的游戏。后续的百货商场物语2、都市大亨物语都继承了这种"空间=品质=收入"的设计哲学。

2. **客层分化设计** — 49种不是数字堆砌，每种客人有独立偏好、消费力和解锁条件。这种"个体化顾客"模式在双点医院(不同病症的病人)中得到验证。

3. **杂志排名=多维度评分** — 不是单一评分，而是8个评分维度(美食/风景/女性/综合)，每个维度要求不同的经营策略。这让"通关"从单一目标变成了8个并行挑战。

4. **浴池功效跨周目叠加** — 在轻量经营游戏中塞入"多周目产生新玩法"的后门，这是一个被低估的设计创意。
