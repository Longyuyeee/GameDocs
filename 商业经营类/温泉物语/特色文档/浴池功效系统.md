## 二、大温泉：功效累积的"容器"

### 2.1 大温泉的特殊规则

- **解锁**: 全8杂志排名第一 + 满足Chimpan Z(猩猩客人)
- **数量**: 全游戏仅1间
- **景观**: 仅**最左侧2×2区域**享受景观加成——这是极重要的布局约束
- **功效**: 继承所有已解锁的浴池功效

### 2.2 跨周目功效叠加

大温泉的**核心终局玩法**：已解锁的浴池功效可以通过大温泉**跨周目继承**。

```
一周目: 解锁了 按摩浴 + 丝滑 + Springerade (3个)
    ↓ 继承
二周目: 大温泉自带3个功效 + 新解锁 不老 + 潮流 + 联谊 (6个)
    ↓ 再继承
三周目: 大温泉自带全部6个功效
```

> **设计分析**: 这是开罗系列中极少数"多周目产生质变"的机制。大多数开罗游戏的多周目只是"带一些钱和道具重新开始"，但温泉物语的浴池功效叠加让**二周目的大温泉先天强于一周目**。这给了完美玩家一个清晰的多周目目标——"让大温泉拥有全部6种功效"。

## 三、功效的布局策略

### 3.1 浴池周围的"稀有土地"

每个浴池周围2格范围内的土地是**功效触发的唯一载体**——所有配方成分必须放在这个范围内。

| 功效 | 占用周围格子数(约) | 布局策略 |
|------|:--:|------|
| 按摩浴 | 2格 | 容易——可嵌入任何布局 |
| 丝滑 | 4格 | 中等——美容院占地较大 |
| **不老** | 3格(含池塘) | **极难**——池塘仅1个，必须精确计算池塘的位置 |
| 潮流 | 2格 | 容易 |
| Springerade | 3格 | 容易 |
| 联谊 | 2格 | 容易 |

> "不老"是六种功效中最难触发的——因为池塘全游戏仅1个，而它需要在浴池2格范围内。这意味着你必须在**规划阶段就决定池塘的位置**——放在地图中心可以辐射最多区域，但会占用浴池周围最珍贵的格子。放在浴池旁边可以触发不老，但池塘的大范围辐射会浪费。

### 3.2 单浴池多功效的物理约束

一个浴池周围同时容纳多种功效的配方成分是可能的，但受限于：

1. 浴池周围2格范围的总格子数(约20-24格)
2. 每种功效配方成分的占地
3. 配方成分之间的功能冲突(美容院不能挨着卡拉OK太近)

> 理论上一个浴池最多可同时触发全部6种功效——但需要的空间规划精度极高，通常需要专门的重构式"推倒重来"才能实现。

## 五、与其他开罗游戏的"多周目"对比

| 游戏 | 多周目继承 |
|------|------|
| **温泉物语** | 浴池功效叠加(跨周目质变)+ 满意度部分继承 + 特殊建筑继承 |
| 游戏开发物语 | 已解锁的类型/题材(NG+中必须重新解锁) + 部分道具 |
| 冒险村物语2 | — |

> 温泉物语的跨周目功效叠加是整个开罗系列中**最多周目深度的终局设计**——它给了玩家一个在物理上可见的多周目目标("大温泉越来越强")，而不只是"重新打一遍，这次更熟练了"。
