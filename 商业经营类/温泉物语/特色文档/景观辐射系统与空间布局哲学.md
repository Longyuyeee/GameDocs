## 一、景观辐射系统：一个极简但极深的空间模型

### 1.1 为什么2格+衰减是"恰好"的设计

```
景观以2格半径向周围建筑辐射风景值
——不多不少的设计：
  如果3格: 辐射面太大→景观的"位置感"消失→布局不重要
  如果1格: 辐射面太小→只能贴脸放→无策略空间
  2格: 刚好让一个建筑最多被8个景观环绕(上下左右+对角+距离2)
       → 策略深度 = 如何在2格半径内挤进最多"有效景观"
```

### 1.2 百分比衰减创造"近多远少"的空间经济学

```
100%(相邻) → 50%(对角) → 25%(距离2) → 0%(>2格)
```

这种衰减不是线性的——对角格只拿一半，距离2格只拿四分之一。它创造了一个"空间竞价"机制：

| 景观位置 | 风景效率 | 每格成本 |
|------|:--:|:--:|
| 紧贴建筑(4方向) | 100%×4=400% | ★★★★★ |
| 对角(4角) | 50%×4=200% | ★★★ |
| 距离2(最多8格) | 25%×8=200% | ★★ |

> **策略含义**: 最稀缺的资源是建筑**紧邻的4个格子**——它们提供的风景效率是对角的2倍，是距离2格的4倍。完美布局的核心就是最大化每个建筑的"紧邻优质景观数"。

### 1.3 季节性叠加创造"轮作"策略空间

景观不是全年一致的——梅树在春季+24但在冬季几乎无效。这创造了"季节轮作"策略：

```
一个建筑在冬季的风景值可能骤降50%
→ 如果该建筑依赖景观加成来支撑VIP客层的满意度
→ 冬季VIP客层满意度可能因景观衰减而下降
→ 策略: 冬季改用石灯笼(全年夜间+16)替代落叶植物
```

这种"季节轮作"在模拟经营游戏中极为罕见——大多数游戏要么无季节机制，要么季节只影响作物(如星露谷)，而温泉物语让景观**品质随季节动态变化**。

## 二、空间布局的三大哲学

### 2.1 辐射密度最大化

> **目标**: 让每个建筑被尽可能多的高效景观环绕

**最优单元**: "建筑居中+四面环绕景观+四角补充"的3×3布局

```
  L  L  L        L=景观
  L  建  L        建=建筑
  L  L  L
```
景观效率: 4×100% + 4×50% = 600%风景加成

**扩展**: 将多个3×3单元拼接在一起，共享景观。相邻建筑之间共享同一个景观格——这是"共用景观"策略。

### 2.2 动线与消费漏斗

> **目标**: 让客人在去浴池的路上经过尽可能多的消费设施

```
前台 → 商业街(小卖部/咖啡厅/土特产店) → 浴池 → 休闲区(KTV/游戏室) → 客房
       ↑──────────消费漏斗──────────────↑
```

客人在入住后第一件事通常是去浴池——这条"必经之路"上放置商店可最大化消费转化。

### 2.3 分区-客层匹配

> **目标**: 不同客层有独立的"专区"，每区内的设施/景观/契合度针对该客层优化

| 专区 | 客层 | 核心景观 | 核心设施 | 核心浴池功效 |
|------|------|------|------|------|
| 女性专区 | OL/贵妇/女演员 | 绣球花/樱花/杜鹃花 | 美容院/花店/咖啡厅 | 丝滑 |
| 男性专区 | 上班族/社长/政治家 | 竹/松/自然石 | 酒吧/游戏室 | Springerade |
| 老年专区 | 老夫妇/茶道家/理事长 | 梅树/枫树/石灯笼 | 和式客房/按摩椅/读书室/茶室 | 按摩浴 |
| 年轻专区 | 学生/程序员/年轻恋人 | 樱花/现代装饰 | 自动贩卖机/卡拉OK/乒乓球 | 潮流+联谊 |
| VIP专区 | 财阀/第一夫人/会长 | **池塘**+梅树+枫树环绕 | VIP室+高级餐厅+酒吧 | 不老+丝滑+MORE |

## 三、池塘——全游戏只有1个的"空间奇点"

池塘(需购买20块自然石+20盏石灯笼的前置)全游戏仅1个。它的特殊性在于：

1. **辐射范围大于标准2格**(有特殊大范围模式)——放在地图中央可同时惠及最多8+个建筑
2. **全年无季节衰减**——不像梅树/枫树有季节波动
3. **不可替代**——不能建第二个池塘，必须精确计算摆放位置
4. **浴池功效"不老"的前置**——池+梅树+占卜馆触发"不老"

> 池塘相当于一个"空间奇点"——地图上有一个位置的价值远超其他任何格子。把它放在哪决定了整个温泉旅馆的格局。这种"空间上的稀缺性"在整个开罗系列中独一无二。

## 四、从温泉物语到过山车之星——空间设计的两极

| 维度 | 温泉物语 | 过山车之星 |
|------|:--:|:--:|
| 空间模型 | **离散网格**(2D像素) | **连续空间**(3D地形) |
| 景观机制 | 精确衰减(100/50/25%) | 视觉评判(视线+主题) |
| 策略工具 | 数学计算 | 视觉审美 |
| 设计哲学 | "效率最优排列" | "最美的乐园" |
| 复杂度 | 低(规则清晰) | 极高(完全自由) |

温泉物语代表了一种"用简单规则造深度策略"的设计范式——2格+衰减的规则在几分钟内就能学会，但要找到最优布局可能需要数十小时的实验。这与过山车之星的"用复杂规则造自由表达"是两种截然不同的设计哲学。
