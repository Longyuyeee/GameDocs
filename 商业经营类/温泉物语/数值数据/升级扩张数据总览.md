## 四、二周目(New Game+)继承完整表

| 继承项 | 继承比例 | 可跨多周目累积 | 对二周目策略的影响 |
|------|:--:|:--:|------|
| **大温泉** | 100%(如已解锁) | 是 | 二周目起始即拥有——提前10年享受终极设施 |
| **贵宾室** | 100%(如已解锁) | 是 | 配合大温泉=二周目起始即有双终极建筑 |
| **浴池功效(已解锁)** | 100% | **是(关键)** | 二周目大温泉自带全部上周目功效 |
| **客人满意度** | 100→20(折减) | 否 | 满意度需重新培养，但有二周目设施加持更快 |
| **景观植物(各1个)** | 1个/种 | 否 | 二周目初期即可使用枫树/梅树/池塘 |
| **土地扩展** | 不继承 | — | 需重新购买 |
| **大部分设施解锁** | 需重新通过投资解锁 | — | 特殊建筑除外 |
| **道具库存** | 不继承(有限) | — | 需重新购买 |

### 二周目速通策略

```
二周目起始状态:
  ✓ 大温泉(全功效继承)
  ✓ 贵宾室
  ✓ 基础景观(枫树/梅树/池塘各1)
  ✓ 客人满意度从20起步(非0)

速通路线:
  1. 立即布置大温泉+贵宾室+池塘核心三角
  2. 利用景观存量快速提升设施人气
  3. 满意度从20起步→更快触发30投资解锁
  4. 第一年即可开始攻优泉杂志
  5. 预计全杂志制霸缩短至10-12年(vs一周目15年)
```
