### 功效触发的技术约束
- 所有配方成分必须在浴池**2格范围内**(上下左右/对角线)
- 一个浴池理论上可同时触发全部6种功效(需要极度精确的空间规划)
- 大温泉继承所有已解锁功效——二周目叠加策略的基础

### 功效的跨周目叠加

```
一周目: 解锁按摩浴 + 丝滑 + Springerade = 大温泉自带3功效
二周目: 继承3功效 + 新解锁不老 + 潮流 + 联谊 = 大温泉自带6功效
三周目+: 大温泉起始即有全部6功效，且可继续在新浴池中叠加新解锁功效
```

## 三、设施人气提升效率曲线

| 设施人气 | 满意度输出效率 | 对客人满意度的相对速度 | 推荐行动 |
|:--:|:--:|:--:|------|
| 0-30 | 极低(<20%) | 几乎没有满意度增长 | 紧急——道具冲到50+ |
| 30-50 | 低(20-30%) | 缓慢增长 | 基础道具(绿茶/蘑菇) |
| 50-100 | 中(30-60%) | 正常增长 | 中价道具(蜂蜜/罐装咖啡/牛奶) |
| 100-150 | 较高(60-80%) | 较快增长 | 高价道具(花圃海报/经典漫画) |
| 150-200 | 高(80-100%) | **峰值效率** | 顶级道具+人气种子 |
| 200-250 | 递减(100→70%) | 边际效益开始下降 | 停止普通道具→改用种子 |
| 250+ | 严重递减(50%→) | 追加人气几乎无效 | 停止一切人气投入 |

> **关键转折点**: 人气200是"硬件上限"——之后普通道具无效，仅种子可突破。大多数设施推荐停在150-200(性价比峰值)，仅最关键的设施(VIP室/大温泉)值得用种子突破200。
