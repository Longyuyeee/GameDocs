## 一、三大建筑类别

温泉物语的建筑分为三类，各自在空间布局中扮演不同角色：

### 1.1 设施(Facilities)

| 子类 | 功能 | 典型建筑 | 客容量 |
|------|------|------|:--:|
| **客房** | 客人住宿，核心收入来源之一 | 和式客房、西式客房、VIP室 | 3-5人 |
| **浴池** | 温泉核心体验，客人必用 | 男浴池、女浴池、大温泉 | 4人 |
| **休闲设施** | 娱乐消费，满足不同类型客人偏好 | 读书室、游戏室、按摩椅、卡拉OK、乒乓球 | 2-4人 |
| **服务设施** | 前台、自动贩卖机等辅助功能 | 前台(Reception)、自动贩卖机 | — |

### 1.2 商店(Stores)

| 子类 | 功能 | 典型建筑 |
|------|------|------|
| **餐饮** | 核心消费场所 | 食堂、咖啡厅、面馆、寿司店 |
| **零售** | 额外消费 | 小卖部、土特产店、花店 |
| **服务** | 美容/娱乐服务 | 美容院、占卜馆、酒吧、卡拉OK |

### 1.3 景观(Environmental)

| 子类 | 效果 | 典型 |
|------|------|------|
| **植物** | 季节性风景加成 | 梅树(春季+24)、枫树(秋季+22)、绣球花(夏季+8)、樱花(春季) |
| **装饰** | 非季节性风景加成 | 石灯笼(夜晚+16)、自然石、竹篱笆 |
| **特殊景观** | 大面积/高加成 | 池塘(特殊大范围辐射，全游戏仅1个) |

## 二、景观辐射系统（核心空间机制）

这是温泉物语**最重要的系统设计**——景观向周围建筑辐射风景值。

### 2.1 辐射范围与衰减

```
景观的风景值以2格为半径向周围建筑辐射
衰减规则:
  相邻格(上下左右):   100% 风景值加成
  对角格:             50% 风景值加成
  距离2格:            25% 风景值加成
  距离>2格:            0% (无影响)
```

### 2.2 季节性风景加成

| 植物 | 春季 | 夏季 | 秋季 | 冬季 |
|------|:--:|:--:|:--:|:--:|
| **梅树** | +24 | +20 | +10 | — |
| **樱花** | +20 | — | — | — |
| **枫树** | — | +10 | +22 | — |
| **绣球花** | — | +8 | — | — |
| **石灯笼** | — | — | — | +(夜晚+16) |

> **设计分析**: 季节性风景加成是温泉物语的隐形深度——不是所有植物在所有季节都有效。高级玩家会在不同季节重新布置景观(拆除季末的、种上当季的)，这相当于在一年周期内进行"季节轮作"。这与星露谷物语的"不同季节种不同作物"有相同的设计基因。

### 2.3 景观叠加效应

多个景观的风景值可以**叠加**到同一个建筑上。一个被4棵枫树+2盏石灯笼环绕的西式客房，其风景值远超被1棵枫树环绕的同类型客房。

## 三、建筑契合度系统(Combo/相性)

### 3.1 触发条件

当两个特定建筑位于**2格范围内**(上下左右/对角线均可)，触发契合度加成。Combo效果**影响2格范围内的所有建筑**。

### 3.2 核心契合度组合

#### 客房类组合

| 组合 | 效果 | 关联客层 |
|------|------|------|
| 西式客房 + 自动贩卖机 + 按摩椅 + 餐厅 | 综合提升 | 年轻人/上班族 |
| 和式客房 + 按摩椅 | 人气显著提升 | 年长者/老人 |
| 西式客房 + 咖啡厅 + 自动贩卖机 | 年轻人/男性 | 学生/程序员 |
| VIP室 + 高级餐厅 + 酒吧 | 高消费客层 | 社长/财阀/理事长 |

#### 浴池类组合

| 组合 | 效果 | 触发浴池功效 |
|------|------|------|
| 男浴池 + 咖啡厅 + 自动贩卖机 + 竹 | 男性浴客人气提升 | **Springerade** |
| 女浴池 + 美容院 + 杜鹃花 + 自然石 + 牛奶 | 女性浴客人气提升 | **丝滑(Silky Smooth)** |
| 浴池 + 自然石 + 自动贩卖机 | 年长浴客人气提升 | **按摩浴(Jacuzzi)** |
| 浴池 + 占卜馆 + 梅树 + 池塘 | 年长/成人客 | **不老(Forever Young)** |
| 浴池 + 卡拉OK + 乒乓球 | 团体客 | **联谊(Fast Friends)** |
| 浴池 + 土特产店 + 杜鹃花 | 年轻人 | **潮流(In-Crowd)** |

#### 需避免的负面组合

| 组合 | 效果 |
|------|------|
| 卡拉OK 靠近 和式客房 | 人气**下降** |
| 游戏室 紧邻 读书室 | 相互干扰 |

> **设计分析**: 契合度系统是"半隐藏规则"的又一次经典应用——哪些建筑组合会触发Combo不完全显示，驱动玩家自行实验和社区分享。与游戏开发物语的"类型×题材"公式相同，这种半隐藏信息是玩家社区UGC的完美触发器。

## 四、空间布局的设计维度

### 4.1 前台与动线

| 设计维度 | 最优策略 | 原因 |
|------|------|------|
| **前台位置** | 紧邻入口 | 减少客人入住排队时间 |
| **前台数量** | 2-3个(中后期) | 黄金周(每年5月)客人暴增，单一前台形成瓶颈 |
| **客房到浴池距离** | ≤5-8格 | 客人入住后第一件事是去浴池 |
| **商业区集中** | 浴池-客房动线中段 | 客人去浴池的路上经过商店=更多消费 |

### 4.2 分区策略

| 分区 | 目标客层 | 核心设施 | 景观偏好 |
|------|------|------|------|
| **女性区** | OL/主妇/贵妇/女演员 | 女浴池+美容院+花店+咖啡厅 | 绣球花/樱花/杜鹃花 |
| **男性区** | 上班族/社长/程序员 | 男浴池+酒吧+卡拉OK+游戏室 | 竹/自然石/松 |
| **老年区** | 老夫妇/茶道家/地主/理事长 | 和式客房+按摩椅+读书室+茶室 | 梅树/枫树/石灯笼 |
| **青年区** | 学生/年轻恋人/程序员 | 西式客房+自动贩卖机+乒乓球+卡拉OK | 樱花/现代装饰 |
| **VIP区** | 财阀/社长/第一夫人/贵妇 | VIP室+高级餐厅+酒吧+高级景观花园 | 池塘(唯一)+枫树+梅树环绕 |

### 4.3 3×7网格布局模板

```
X O X        X = 设施/商店(产生消费)
O T O        O = 客房(产生住宿收入)
O T O        T = 景观(辐射风景值)
X T X
O T O
O T O
X T X
```

这种布局的核心理念：**每一间客房都被至少3-4个景观和2-3个设施环绕**，且客房到任何设施的距离不超过3格。适合中前期，后期逐渐打破网格进行分区化重构。

## 五、空间系统对比：温泉物语在整个品类中的独特位置

| 游戏 | 空间系统 | 布局深度 | 景观因素 |
|------|:--:|:--:|:--:|
| **温泉物语** | 辐射范围+契合度连携+季节性 | ★★★★★ | **核心机制**——景观直接决定经营效率 |
| 游戏开发物语 | 纯工位制(无布局) | ★ | 无 |
| 百货商场物语2 | 多楼层+商铺搭配 | ★★★★ | 中 |
| 过山车之星 | 完全自由3D地形+建筑 | ★★★★★ | 高(美化评分) |
| 双点医院 | 房间内部布局+房间位置 | ★★★ | 低(仅房间声望范围) |

> **核心洞察**: 温泉物语的景观辐射系统(2格范围×百分比衰减)是整个开罗系列中**最精确的空间机制**。它比百货商场物语的"组合触发"更复杂(需要计算衰减)，又比过山车之星的"自由3D"更简洁(2格半径的离散空间)。这个系统的精妙之处在于：**它让玩家在一个简单的2D网格上做出有深度的空间决策**。
