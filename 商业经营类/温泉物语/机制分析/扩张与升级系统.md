# 扩张与升级系统 — 温泉物语

> 游戏版本: Hot Springs Story Steam版 v2022.3.27 (ver.2.79)
> 数据来源: [Kairosoft Fandom Wiki](https://kairosoft.fandom.com/wiki/Hot_Springs_Story), [GameFAQs攻略](https://gamefaqs.gamespot.com/pc/360006-hot-springs-story/faqs/61941), [B站专栏攻略](https://www.bilibili.com/read/cv41919728)

## 一、扩张的三个维度

温泉物语的扩张是三维的——不像游戏开发物语的纯"工位数量增长"：

| 维度 | 载体 | 机制 | 频率 |
|------|------|------|:--:|
| **物理扩张** | 土地扩展 | 购买周边土地扩大旅馆用地 | 阶段性(价格递增) |
| **产品扩张** | 新设施/商店解锁 | 通过投资(Issue Investment)解锁 | 持续(与客层满意度关联) |
| **客层扩张** | 新客人解锁 | 现有客层满意度≥30→解锁投资→解锁新客层 | 持续连锁 |

---

## 二、土地扩展系统

温泉物语的地图是可扩展的——初始用地较小，需要逐步购买周边土地。

| 扩展次数 | 费用(G) | 解锁内容 |
|:--:|:--:|------|
| 初始地块 | — | 约6×8格(可建基础配置) |
| 第1次扩展 | 递增 | 更多空间布置景观和设施 |
| 第2次扩展 | 更高 | 开始分区布局 |
| 第3次+ | 继续递增 | 后期完美布局必须 |

> **设计分析**: 土地扩展的定价递增不是线性的——它制造了一个"何时该扩地"的策略时刻。扩地太早(设施还填不满)=浪费钱买了空地；扩地太晚=空间挤不下=不得不拆旧建新。这与真实旅馆经营的"扩建时机"决策类似。

---

## 三、投资系统——温泉物语最核心的解锁链

投资是温泉物语**核心的解锁机制**。与游戏开发物语的"职业等级解锁"不同，温泉物语的一切解锁都通过**投资**串成一条链：

```
经营旅馆(获得收入)
    ↓
使用宣传/设施吸引新的基础客层
    ↓
该客层满意度提升至30
    ↓
**触发投资机会** → 花费金币执行投资
    ↓
投资完成 → 解锁新设施/新道具/新客层/新景观
    ↓
新客层到访 → 提升至30 → 触发下一轮投资
    ↓
循环往复，直到所有49客层全部解锁
```

### 投资分类

| 投资类型 | 数量(约) | 典型效果 |
|------|:--:|------|
| **设施解锁** | 15+ | 解锁新建筑(读书室/游戏室/VIP室等) |
| **客层解锁** | 30+ | 解锁新来访者(从散步大师到财阀) |
| **道具/服务** | 5+ | 解锁新道具/新宣传手段 |
| **基础设施** | 3+ | 花圃开发/徒步旅行路线/图书馆引入 |
| **最终解锁** | 2 | VIP室(全49客层90+)、大温泉(全8杂志第一) |

### 投资优先级建议

| 优先级 | 投资名称 | 原因 |
|:--:|------|------|
| 1 | **花圃开发** | 解锁多种景观植物(梅树/枫树/绣球花)——前期最稀缺的资源 |
| 2 | **徒步旅行路线** | 吸引年轻客层(扩大顾客基础) |
| 3 | **图书馆引入** | 解锁读书室——基础人气32(前期人气最高的设施) |
| 4 | **旅馆宣传** | 中期再做(前期客房不够，广告吸引来的客人住不下) |

## 四、设施升级路径

| 设施层级 | 代表 | 解锁阶段 | 每间上限 |
|------|------|:--:|:--:|
| **基础客房** | 和式客房、西式客房 | 初始 | 不限 |
| **高级客房** | VIP室 | 全49客层90+ → 解锁投资 | **仅1间** |
| **基础浴池** | 男浴池、女浴池 | 初始 | 不限 |
| **终极浴池** | 大温泉 | 全8杂志第一+Chimpan Z满足 | **仅1间** |
| **基础设施** | 食堂、小卖部、自动贩卖机 | 初始 | 不限 |
| **主题设施** | 卡拉OK、酒吧、美容院、占卜馆 | 投资解锁 | 不限 |

## 五、规模效应

| 规模效应 | 描述 |
|------|------|
| **正面: 景观乘数** | 地越大→能种的景观越多→风景叠加越强→所有设施品质越高 |
| **正面: 分区可行性** | 大地图可以划分客层专区(女性区/老年区/VIP区)——提升各客层的满意度提升效率 |
| **正面: 瓶颈消除** | 更多前台→消除排队；更多浴池→消除拥挤 |
| **负面: 维护费** | 地大→设施多→每日维护费飙升→需要足够高消费客层支撑 |
| **负面: 动线变长** | 地太大→客人走太远→设施利用率下降→ROI下降 |

## 六、二周目(New Game+)继承

| 继承内容 | 继承方式 |
|------|------|
| **浴池功效** | 大温泉继承+已解锁功效继承，**二周目可叠加新功效** |
| **满意度** | 100满意度客人→20满意度；其他→部分继承 |
| **特殊景观** | 各种景观各继承1个(重要！二周目初期即可使用高级景观) |
| **奖励建筑** | 继承猩猩温泉、贵宾室(Kairo君解锁) |
| **设施解锁** | 大部分需重新解锁，但核心特殊建筑可继承 |

> **设计分析**: 二周目继承的核心是让完美玩家能做"跨周目浴池功效叠加"(温泉物语的最硬核玩法)。由于大温泉功效可在不同周目叠加，理论上可以通过多周目将一家旅馆打造成全部6种功效覆盖的终极温泉。这给了完美玩家"无限继续"的动力。
