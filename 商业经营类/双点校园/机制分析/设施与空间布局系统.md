## 二、房间声望(Prestige)系统

| 声望等级 | 效果 | 提升方式 |
|:--|------|------|
| Lv1-3 | 基准 | 基础家具+少量装饰 |
| Lv4-6 | 学生/教师幸福度+ | 更多装饰+多种物品类型 |
| Lv7-8 | 成绩间接提升 | 大量装饰+高级设备 |
| **Lv9-10** | **最强加成** | **满装饰+全高级设备+多种物品类型混合** |

### 声望提升的关键原则

| 原则 | 说明 |
|------|------|
| **物品多样化(非单一堆叠)** | 8种不同的窗户>8个相同的窗户(边际递减) |
| 海报+画像=高性价比声望 | 便宜且提供大量声望值 |
| 高级学习设备=高声望 | VR投影仪/动能弹珠等高级设备自带高声望 |
| 房间越大=可放越多物品=更高声望潜力 | — |

## 三、校园布局原则

| 原则 | 说明 |
|------|------|
| **学术区** | 阶梯教室+专业实验室+图书馆集中(减少学生步行时间) |
| **生活区** | 宿舍+淋浴+洗手间集中(日常生活需求) |
| **社交区居中** | 学生休息室+学生会+自助餐厅在交通枢纽(中心位置) |
| **每区域配洗手间** | 洗手间+洗手液+垃圾桶放每个门附近 |
| **户外装饰** | 植物+长椅+海报提升校园美观度(鼓励户外活动=健康度↑) |

## 四、设计特点分析

**物品多样化>堆叠——对抗"最优物品堆满"的meta:** 放置8种不同的窗户获得的声望>放置8个相同的窗户。这种"边际递减"设计强制玩家配置**多样化的物品组合**——你需要在有限的房间空间中平衡"功能家具"(教学设备)和"装饰品"(美观物品)。

**声望=从"有房间就行"到"把房间做漂亮":** 双点校园的声望系统将"美观设计"从"可有可无的审美"变为"直接影响成绩和收入的游戏机制"。一个高声望的图书馆不仅看起来更好——学生的自习效率实际上更高。这种"美观=效率"的因果链是双点系列设计DNA的一部分。

---

> 跨品类对比: 与双点医院的"房间=功能单元"(诊断室/治疗室)相同——但双点校园多了"宿舍"(生活空间)和"社团房间"(组织空间)。与过山车之星的"景观评分提升Prestige"类似——都是"美观=收入"的设计逻辑。
