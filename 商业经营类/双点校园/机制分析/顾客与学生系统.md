# 顾客/学生系统 — 双点校园

> 游戏版本: Two Point Campus v1.0+ (Steam/PC, 2022)
> 数据来源: [Two Point Campus Fandom Wiki](https://two-point-campus.fandom.com/), [IGN Beginner's Guide](https://www.ign.com/wikis/two-point-campus/), [TheGamer Guide](https://www.thegamer.com/two-point-campus-complete-guide/)

---

## 一、学生——不是"病人"而是"长期培养对象"

与双点医院的"病人=一次性消费"不同，双点校园的学生在校园中停留**多年**(每年约20分钟)——有完整的学习轨迹。

| 学生属性 | 说明 | 影响因素 |
|------|------|------|
| **成绩(Grades)** | 学业表现——决定毕业等级 | 教师技能+设施品质+幸福度+图书馆使用 |
| **幸福度(Happiness)** | 综合满意度——影响成绩和缴费 | 宿舍/餐饮/娱乐/社交/卫生 |
| **精力(Energy)** | 体能——影响学习效率 | 宿舍/睡眠/饮料 |
| **卫生(Hygiene)** | 清洁度——低=医疗问题 | 淋浴/洗手间/清洁工 |

## 二、学生生命周期

| 阶段 | 时长 | 核心活动 |
|:--|:--|------|
| 入学 | 每学年开始 | 分配宿舍/注册课程 |
| 学习期(秋-春) | 约9个月 | 上课/自习/社团/社交 |
| 考试期 | 学年末 | 成绩结算/升级或留级 |
| **暑期(Summer)** | **7月后** | **停课——学生回家/校园纯建设期** |
| 毕业 | 课程年限制(1-4年) | 毕业等级(A+/A/B/C/F)→声望贡献+学费返还 |

## 三、学生幸福度——核心管理指标

| 幸福度来源 | 影响权重 | 提升方式 |
|------|:--:|------|
| 宿舍品质 | 高 | 高声望宿舍(声望Lv9-10)+充足床位 |
| 餐饮/饮料 | 中 | 自助餐厅+自动售货机+咖啡摊 |
| 娱乐 | 中高 | 学生休息室+游戏机+学生社团活动 |
| 卫生 | 中 | 充足淋浴+洗手间+清洁工 |
| 社交 | 中 | 社团+学生会+活动 |
| 学费合理度 | 低-中 | 降低学费=幸福度+10%(代价=收入-20%) |

### 关键幸福度事件

| 事件 | 效果 | 推荐 |
|------|:--:|:--:|
| **Jasmine Odyssey**(茉莉花奥德赛) | **全体+50%幸福度** | **尽可能频繁举办** |
| 电影之夜 | +20%幸福度 | 定期举办 |
| 社团活动 | 持续小幅提升 | 保持所有社团活跃 |

## 四、个人目标(Personal Goals)

学生会在信箱中提交个人目标——如在校园中放置特定物品、建造特定房间。完成目标=该学生幸福度大幅提升。未完成=幸福度下降。个人目标系统是"学生个性化需求"的游戏化——每个学生有自己独特的需求。

---

## 五、社团系统——5社团×全校加成

| 社团 | 解锁校园 | 全校加成 |
|------|:--|------|
| **强力小睡(Power-Nap)** | Piazza Lanatra | 成员可在任何地方恢复精力 |
| **竞走(Speed Walking)** | 同上 | 成员走路速度加快 |
| **读书会(Book Club)** | 同上 | **成员学习速度加快**(最直接影响成绩) |
| **自然社(Nature Club)** | 同上 | 成员维护校园美观度 |
| **球社(Orb Club)** | 同上 | **成员升级=校园获得现金**(但降低幸福度) |

> 球社是一把双刃剑——现金收入vs幸福度惩罚。达到Lv5后考虑删除(持续的幸福度惩罚>现金收入)。

## 六、设计特点分析

**学生="长期资产"而非"一次性顾客":** 学生在校园中停留多年——这意味着你的"服务质量"投资会在学生的整个在校期间产生持续回报(而非一次性)。一个学生在第一年因为糟糕的宿舍而低幸福度→成绩下降→毕业等级低→声望受损→影响接下来数年的招生质量。这种"长期关系"与双点医院的"一次性治疗"形成鲜明对比。

**暑期=经营的"呼吸间隙":** 每年7月后学生停课回家——这是双点校园独有的设计节奏。暑期的收入停止=你不能靠收学费来"续命"。但暑期也是最好的"重建设窗口"——你可以趁学生不在时大规模改建校园而不会影响任何人的幸福度或成绩。这种"经营期→建设期"的交替是双点校园vs双点医院的最大结构差异。

**社团=从"房间设施"到"组织文化"的升级:** 双点医院的"科室"是房间——双点校园多了"社团"这一组织层面的系统。社团提供的加成是全校性的(不是房间性的)——这意味着你在"组织"维度上有了新的优化空间。

---

> 跨品类对比: 与双点医院的"病人=一次性治疗"不同——双点校园的学生是"多年度培养对象"。与口袋学院的"3年制学生"不同——双点校园的学生周期更灵活(1-4年不等/取决于课程)。
