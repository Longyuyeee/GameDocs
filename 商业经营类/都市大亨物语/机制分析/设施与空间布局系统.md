# 设施与空间布局系统 — 都市大亨物语

> 游戏版本: Dream Town Island (都市大亨物语) Steam版 v2022 (Steam/PC, 2022)
> 数据来源: [9k9k攻略站](https://m.9k9k.com/czdsdwy/), [ZOL新手布局](https://m.zol.com.cn/article/10283014.html), [B站通关布局](https://www.bilibili.com/opus/793179154433441842)

---

## 一、地图空间结构

都市大亨物语的地图是**开放网格**——没有预设的分区和道路，所有建筑由玩家手动放置。

| 参数 | 数值 | 说明 |
|------|:--:|------|
| 地图大小 | 约20×20格 | 可逐步扩展(购买相邻土地) |
| 放置方式 | 自由放置 | 手动选择位置(不像SimCity的"区划自动生成") |
| 专业街判定 | **九宫格(3×3)** | 建筑相性在九宫格范围内生效 |

---

## 二、分区布局策略

| 区域 | 建筑类型 | 布局原则 |
|------|------|------|
| 商业核心区 | 商店+产业+办公楼 | 九宫格密集叠加专业街→最高地价 |
| 住宅区 | 住宅(穿插在商业区中) | **不要集中**——分散穿插以享受各地段的地价加成 |
| 绿化区 | 公园+树木+种植园 | 提升周边地价+提供产业复兴点 |
| 市政区 | 市政厅+特殊建筑 | 市政厅放角落(初期无相性) |

### 核心布局原则

| 原则 | 说明 |
|------|------|
| **住宅穿插在商业街中** | 专业街九宫格的空位放住宅——享受叠加地价 |
| **不集中住宅区** | 集中住宅=只有基础地价/穿插住宅=享受多条专业街叠加 |
| **办公楼居中** | 办公楼覆盖范围越大→可发现的职业越多 |
| **市政厅放角落** | 初期市政厅无相性——让出核心地段 |
| **公园/树木穿插** | 提升周边地价+居民能力增长 |

## 三、建筑相性(Combo)系统

当两个有相性的建筑放置在九宫格范围内→触发"相性加成"→地价上涨+解锁职业:

| 建筑A | 建筑B | 效果 |
|------|------|------|
| 公园 | 足球场 | 运动相性 |
| 美容院 | 美术沙龙 | 美容相性 |
| 咖啡厅 | 花店 | 餐饮相性 |
| 便利商店 | 邮局 | 便民相性 |
| 超市 | 鱼店/蔬果店 | 生鲜相性 |
| 游戏公司 | 音乐店/开罗大楼 | 科技相性 |

> 总计约**170+种建筑相性组合**——放置时看到"气泡"图标=这两个建筑有相性，应放在一起。

---

## 四、设计特点分析

**"网格自由放置"vs"区划自动生成"**：都市大亨物语的建筑是手动逐格放置的——与SimCity的"区划自动生成建筑"形成鲜明对比。这种设计将"城市建造"从"宏观规划"（区划=指挥）变为"微观设计"（每个建筑的位置=手动决策）。尺度感的缩小带来了控制感的提升。

**住宅穿插=对抗"功能分区"的直觉**：真实城市规划强调"功能分区"(工业区/商业区/住宅区分离)。但都市大亨物语的唯一正确策略是**住宅穿插在商业区中**——因为地价是共享的。这种设计打破了"分区=效率"的传统规划直觉——最优布局不是"有序的"，而是"混合的"。

---

> 跨品类对比: 与SimCity的"区划=自动生成建筑"不同——都市大亨物语是手动逐格放置。与百货商场物语2的"垂直楼层"不同——都市大亨物语是水平展开的二维网格布局。
