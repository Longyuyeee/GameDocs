## 二、社团系统——10社团×7对手

### 社团竞赛机制

| 阶段 | 时间 | 行动 |
|------|:--|------|
| 4-7月 | 外校对战 | 赢7局→解锁隐藏职业→星探关注 |
| 8月 | **全国大赛** | 社团成员参加→冠军=特殊职业+声望 |

### 社团策略

| 策略 | 说明 |
|------|------|
| **同时保持3个社团** | 打完一个换下一个——不需要10个同时 |
| **不良学生在社团有加成** | 练习比赛和全国大赛表现更好 |
| **全国冠军最少需要5人** | 等级高可减为4人 |
| **一年专注一个社团** | 挑人多的+有2年级学生的社团 |

### 社团相关设施与教师解锁

| 设施(升至LV3) | 解锁教师 |
|------|------|
| 棒球场 | 体育社团教师 |
| 天文台 | **开罗姬**(社团教师) |
| 桌球台 | 桌球社团教师 |
| 家庭科室 | 家政社团教师 |

## 四、设计特点分析

**社团竞赛=年度的"目标冲刺"**: 4-7月的对外对战→8月全国大赛形成了自然的年度节奏——每年上学期为比赛做准备，暑假进行最终对决。这种"学习期→竞赛期"的循环是真实学校生活的缩影。

**不良学生=隐藏的社团王牌**: 在大多数学校游戏中"不良学生"是负面标签——但在口袋学院物语3中，不良学生在社团竞赛中有重大加成。这是设计中的"反标签化"——标签不定义价值，场景决定价值。

**66景点=设施+植物的空间化学**: 景点的形成需要设施与植物的空间配合——这比百货商场物语2的"专柜组合"多了一个"植物"维度(植物提供相性加成但不占商铺位)。

---

> 跨品类对比: 与游戏开发物语的"类型×题材组合"（知识组合）不同——口袋学院的社团是"学生×设施×竞赛"的行动组合。与都市大亨物语的"专业街"（静态的建筑组合）不同——社团是**动态的**(学生在社团中的活动和竞赛表现随时间变化)。
