## 一、66个景点(相性组合)系统

### 什么是景点

将特定设施与植物在空间上配合→形成"景点"(Popular Spot)——景点提升周边区域的人气和学生属性成长速度。共有**66种景点组合**需要发现。

### 景点相性类型

| 相性类型 | 典型组合 | 效果 |
|------|------|------|
| 运动类 | 操场+樱花树+铜像 | 运动属性成长↑ |
| 艺术类 | 美术室+向日葵+金像 | 艺术属性成长↑ |
| 科技类 | 电脑室+IT设施+开罗像 | IT属性成长↑ |
| 社交类 | 食堂+喷泉+花坛 | 社交属性成长↑ |
| 精神类 | 神社+传说的树+希望之钟 | 精神属性成长↑ |

## 二、校园布局原则

### 核心布局原则

| 原则 | 说明 |
|------|------|
| **设施围绕教室** | 最小化学生从教室到设施的移动时间 |
| 景点叠加 | 多个景点覆盖同一区域=属性成长加速叠加 |
| 雕像慎用 | 雕像维护费极高(不要过量建造) |
| 临时刷级区 | 在空地建门+教室+教师+6设施→48学生一个月三级 |
| 空白处栽花种草 | 简单便宜但提供相性加成 |

### 一周目特别警告

| 设施 | 数量限制 | 说明 |
|------|:--:|------|
| 金银铜雕像 | **各1个**(不可额外获取) | 放置位置需要精心规划 |
| 迷你美术馆 | **1个** | 不可重复建造 |
| 传说的树 | **1个** | 最强精神/自然相性设施 |
| 希望之钟 | **第11年赠送** | 时间限定 |
| 开罗像 | **第16年** | 时间限定 |
| 金开罗像 | **第20年** | 时间限定 |

## 四、设计特点分析

**66景点=设施+植物的"空间化学":** 口袋学院的景点形成需要设施与植物的空间配合——这比百货商场物语2的"专柜组合"多了一个"植物"维度。植物不占用核心建筑位(可以在空地/边缘放置)，但它们提供的相性加成是必不可少的。这种"小投入大回报"的设计让植物的配置成为"花最少的代价获取最大的布局优化"。

**一周目雕像限制=不可逆的空间决策:** 金银铜雕像在一周目只能各获得1个——它们的放置位置是**不可逆的**(不能移动后重新放置)。这种稀缺性+不可逆性让布局决策有了真正的重量——一旦决策错误就没有修正机会。二周目继承多一套雕像=对第一周目"布局经验"的奖励。

---

> 跨品类对比: 与百货商场物语2的"专柜组合"(同楼层商铺组合)不同——口袋学院的景点组合是**设施+植物的空间配合**(非商铺组合)。与温泉物语的"设施相性加成"(相邻=加成)最相似——但口袋学院多了一个"植物≠设施"的类型区分。
