## 六、设计视角：资金流时序分析

```
月初: 政府拨款+赞助费入账 → [现金流峰值]
月中: 门票+捐赠+礼品店日常收入 → [现金流平稳]
月末: 员工薪资统一扣除 → [现金流低谷]
探险期: 一次性大额支出 → [注意储备金充足]
```

**策略启示**：月初是大额投资（新建筑/高费用探险）的最佳时机；月末前确保留有足够的薪资储备；任何时候保持$5,000+的探险备用金。
