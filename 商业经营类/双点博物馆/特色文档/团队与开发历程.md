## 二、开发历程

### 关键时间线

| 时间 | 事件 |
|------|------|
| 2016年7月 | 三位Lionhead元老创立Two Point Studios |
| 2017年1月 | 与SEGA欧洲签订Searchlight出版合作协议 |
| 2018年8月 | **《双点医院》**发售——首作即爆款，Steam同时在线峰值超1.5万，好评率90% |
| 2019年5月 | SEGA完成对Two Point Studios的全资收购 |
| 2022年8月 | **《双点校园》**发售——疫情远程开发，吸引了比医院多4倍的玩家 |
| 2025年3月 | **《双点博物馆》**发售——回归办公室办公后开发，被系列粉丝和评论家评价为最佳一作 |

### 疫情与办公模式的影响

- 《双点校园》几乎全程远程开发（COVID-19封锁期间）
- 《双点博物馆》**回归办公室**(每周4天)，团队表示开发的顺畅度"远优于远程期"
- 创始人Mark Webley在GamesIndustry.biz采访中提到：
  > "在办公室里的即兴交流、随时拉人过来看屏幕上的bug、走廊里的设计讨论——这些东西在Zoom上根本没法重现。"

### 开发哲学

- **"用一个日常概念 + 英式幽默 + 经营模拟公式"**
- 三部作品验证了同一套核心DNA可以在不同产业中成功移植
- 每次迭代保持核心循环（房间化布局+员工技能树+需求驱动），但机械创新足够让每作感觉不同
- 博物馆的"探险获取展品"是三代中最大的核心循环创新

## 五、设计启示

### 对独立游戏开发者的启示

1. **30-40人团队的"甜蜜点"**：Two Point Studios证明了一个40人左右的团队可以在3-4年的开发周期内做出评分85+、销量26万+的优质经营游戏——这个规模对于有野心的独立工作室是可达的。
2. **公式迁移>从零开始**：双点医院→校园→博物馆，每次都用同一核心公式适配新产业，但每次都保持机械创新。这种"80%熟悉的+20%新鲜的"配方降低了设计风险。
3. **英式幽默=品牌壁垒**：双点系列的广播系统、荒诞疾病/展品设计形成了一种独特的声音——这是竞争对手很难复制的"软实力"。
4. **办公室的价值**：疫情远程开发《校园》的体验让团队明确回到办公室开发《博物馆》——对于需要频繁协作的创意工作，面对面交流仍有不可替代的价值。

### 对商业经营品类的参考价值

- 双点系列是"现代工业化经营模拟"的标杆——证明了经营品类的市场规模可以支撑3A级别(相对而言)的制作水准
- 博物馆的探险循环为经营品类打开了一条新路："获取端可以有不确定性"
- "多个独立场所并行经营"的架构（5个博物馆切换管理）为经营游戏的后期内容延伸提供了新范式
