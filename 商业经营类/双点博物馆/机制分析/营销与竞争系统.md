# 营销与竞争系统 — 双点博物馆

> 游戏版本: Two Point Museum v1.0 (Steam, 2025.3.4)
> 数据来源: [TheGamer Sponsorship Guide](https://www.thegamer.com/two-point-museum-sponsors-products-deals/), [Two Point Museum Fandom Wiki](https://two-point-museum.fandom.com/wiki/), [Steam Community Discussion on Marketing](https://steamcommunity.com/app/2185060)

---

## 一、营销手段

### 营销办公室系统

| 要素 | 说明 |
|------|------|
| **设施** | 营销办公室（2×2格，$11,100） |
| **人员** | 1名助理（需营销技能） |
| **容量** | 1张营销桌=1个营销活动位，可扩建多桌 |

### 营销活动类型

| 活动类型 | 目标效果 | 适用场景 |
|------|------|------|
| **一般宣传** | 增加基础客流 | 新博物馆开张/客流低谷期 |
| **教授招募** | 吸引教授类游客 | 知识型展品多/需要提高捐赠收入时 |
| **儿童专场** | 吸引儿童+家庭游客 | 互动展品多/礼品店儿童商品促销 |
| **游客吸引** | 吸引游客类游客 | 导览团完备/Buzz型展品多 |
| **爱好者定向** | 吸引特定主题爱好者 | 某主题展品集中时 |
| **活动营销** | 特殊活动/展览开幕 | 限时活动期间最大化客流 |

### 营销效果评估
- 营销活动有持续时间和费用（每项活动运行期间持续扣费）
- 在财务面板可查看各营销渠道的客流转化
- 停掉无效果的营销活动以控制成本

## 二、赞助商系统

### 赞助协议

| 要素 | 说明 |
|------|------|
| **获取方式** | 财务面板→赞助标签页（握手图标） |
| **收入模式** | 每月固定奖金 + 每位参观赞助展品游客的额外付费 |
| **义务** | 在博物馆中展示赞助商展品 |
| **负面效果** | 部分赞助品有负面效果（产生垃圾/降低游客心情/广告干扰） |

### 赞助策略

| 策略 | 说明 |
|------|------|
| **隔离放置** | 将含负面效果的赞助品放在偏僻位置 |
| **破产保险** | 资金耗尽时出现特殊赞助Jumbo McNally人形立牌（$4,500/月） |
| **优先接受** | 无负面效果的赞助品（如普通广告牌） |
| **及时终止** | 博物馆资金充裕后取消高负面影响的赞助 |

### 赞助品负面效果表

| 赞助品类型 | 负面效果 | 应对措施 |
|------|------|------|
| 垃圾食品广告 | 产生垃圾 | 增加清洁工覆盖 |
| 无聊广告牌 | 降低路过游客快乐 | 放在偏远角落 |
| 巨型人形立牌 | 视觉污染 | 放在员工区 |
| 噪音广告 | 不适感 | 远离主要展区 |

## 三、品牌与声誉机制

### 博物馆知名度

- **博物馆星级**：1-5星，由完成章节目标和游客评价综合决定
- **博物馆评分%**：游客评价的综合百分比
- **口碑传播**：完成梦想之旅的游客成为推广大使

### 知名度→客流映射

| 知名度等级 | 基础客流量 | 高端游客占比 | 政府拨款 |
|:--:|:--:|:--:|:--:|
| 1星 | 低 | 极少 | 低额 |
| 2星 | 中低 | 少 | 中低额 |
| 3星 | 中 | 中等 | 中额 |
| 4星 | 中高 | 多 | 中高额 |
| 5星 | 高 | 很多 | 高额 |

## 四、竞争系统

### 游戏内竞争分析

- 双点博物馆**没有直接的AI竞争者**（不同于Game Dev Tycoon的竞争对手公司）
- 取而代之的是"内在竞争"——不同博物馆之间的进度比拼
- 玩家需要多博物馆协同发展（不能只刷一个博物馆的星级）
- 各博物馆之间的资源（资金/员工）是共享的，形成间接竞争

### 排行榜与对比

- 游戏无全局排行榜
- 部分章节有评分目标要求
- 沙盒模式自由配置竞争压力

## 五、评价/媒体系统

### 游客评价

| 评价维度 | 影响因子 | 低分后果 |
|------|------|------|
| **Buzz充足度** | 展品品质+装饰+加成 | 游客不捐赠/提前离馆 |
| **知识获取** | 信息牌+已分析展品 | 教授类游客差评 |
| **设施服务** | 洗手间/餐饮/座位 | 快乐度下降 |
| **价格感知** | 门票/商品价格 | 感觉"不值"→差评→客流下降 |
| **清洁度** | 垃圾/展品脏污 | 大幅降低Buzz和快乐 |
| **安全** | 盗窃/幽灵逃脱事件 | 恐惧情绪→游客逃离 |

### 差评处理

- 差评直接降低博物馆评分%（影响拨款）
- 严重负面事件（盗窃/火灾）可能导致连锁客流下降
- 修复问题后评分缓慢恢复（非立即反弹）

## 六、设计特点分析

**以内在驱动力替代外部竞争**：与Game Dev Tycoon的AI对手竞争、过山车之星的模拟城市排名不同，双点博物馆几乎没有"对手"的概念。竞争压力来自玩家自己的扩张欲望——"我想解锁下一个博物馆""我想凑齐这套恐龙骨架""我想把Buzz冲到下一阶段"。这种"内在驱动"式设计更接近沙盒经营的本质，而非零和竞争。

**赞助商=带负面的收入**：赞助品在获得收入的同时可能产生垃圾/降低心情/视觉污染。这种"有代价的收入"在游戏设计中是一种优秀的风险权衡机制——它给低水平玩家一个安全网（没钱就靠赞助续命），同时给高水平玩家一个优化目标（完全不依赖赞助品）。类似设计的还有暗黑地牢的"负特质"系统和卡牌游戏中的"诅咒卡"。

**口碑的缓慢恢复机制**：评分不是立即回调，而需要经过游客重新体验→好评累积→评分回升的过程。这种"惯性"设计防止了"出问题→立刻修复→评分瞬间恢复"的廉价操作，让每次危机都有真实后果。

---

> 跨品类对比: 与双点医院的"声誉直接影响病人数量"不同，博物馆的知名度更多影响政府拨款和高端游客类型，而非基础客流——这减少了"恶性循环"的风险（评分低→没人来→没钱提升→评分更低）。
