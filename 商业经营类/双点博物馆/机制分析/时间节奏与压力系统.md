# 时间节奏与压力系统 — 双点博物馆

> 游戏版本: Two Point Museum v1.0 (Steam, 2025.3.4)
> 数据来源: [TheGamer Beginner's Guide](https://www.thegamer.com/two-point-museum-beginner-tips-exhibit-placement-security-sponsors/), [Two Point Museum Fandom Wiki](https://two-point-museum.fandom.com/wiki/), [Dexerto Buzz Guide](https://www.dexerto.com/gaming/how-to-generate-buzz-in-two-point-museum-3144372/)

---

## 一、时间结构

### 时间流速

| 时间单位 | 对应内容 |
|------|------|
| **游戏日** | 约30-60秒（可暂停/加速） |
| **游戏月** | 约30天 |
| **游戏年** | 12个月（无明确季节） |
| **速度控制** | 暂停/1×/2×/3×速度（与双点系列一致） |

### 时间循环特征

- **无固定结束日期**——开放式经营，可无限玩下去
- **每月结算**：拨款+薪资+赞助收入在月初/月末结算
- **探险时间**：独立于日常运营，探险有固定天数（5-40天不等）
- **培训时间**：12-30天，培训期间员工不可用

---

## 二、经营节奏

### 日/月节奏

| 时段 | 活动 |
|------|------|
| **月初** | 收到政府拨款+赞助商付款 |
| **月中** | 客流平稳，日常运营+派遣探险队 |
| **月末** | 员工薪资结算（关注现金流） |
| **全月** | 持续运行营销活动+管理员工+处理突发事件 |

### 节奏循环

```
派遣探险(等待N天) → 展品到达 → 布置展品+装饰 → Buzz提升 → 捐赠/门票收入增加 → 再投资(新探险/培训/扩建) → 循环
```

### 旺季/淡季

- 无自然季节概念（与星露谷明确四季不同）
- "旺季"由玩家主动创造：营销活动/展览开幕/新展品到馆→客流高峰
- "淡季"出现在：展品大量脏污/盗窃事件后/长期无新展品

---

## 三、压力/挑战机制

### 常规压力

| 压力类型 | 触发条件 | 后果 | 应对方式 |
|------|------|------|------|
| **财务压力** | 支出>收入持续数月 | 资金见底→破产警告 | 提高门票+接受赞助+贷款 |
| **展品劣化** | 展品随时间自然劣化 | Buzz下降→捐赠减少 | 专家定期维护 |
| **员工不满** | 拒绝加薪/休息不足 | 效率下降→罢工→离职 | 批准关键加薪+完善休息室 |
| **空间压力** | 展品越来越多 | 空间不足无法展示新品 | 扩建新建筑($100/块) |

### 突发事件（危机管理）

| 事件类型 | 触发条件 | 后果 | 需要 |
|------|------|------|------|
| **火灾** | 设备故障/随机事件 | 展品损毁+Buzz大跌 | 清洁工(防火技能)+灭火器 |
| **盗窃** | 高价值展品+安保不足 | 展品丢失(不可替代!) | 保安(监控室技能)+安保贴纸Perk |
| **幽灵逃脱** | 幽灵客房舒适度不足 | 游客恐慌→大量离馆→差评 | 清洁工(幽灵捕获技能) |
| **温度故障** | 加热/冷却器故障 | 展品加速劣化 | 清洁工(机械师技能)快速维修 |
| **鱼吃鱼** | 水族箱内掠食关系 | 小鱼被吃 | 分开饲养掠食鱼和非掠食鱼 |
| **尸体腐烂** | 冰冻展品解冻 | 恶臭→全馆快乐下降 | 修复冷却设备 |

### 危机设计特征

- 每种危机都有对应的"硬技能"需求（没有该技能的员工无法处理）
- 危机具有连锁效应（盗窃→差评→客流下降→收入下降→没钱应对下一次危机）
- 危机的出现频率随博物馆规模和展品价值上升而增加

## 四、目标系统

### 章节目标（故事模式）

每个博物馆有章节式任务目标：
- "达到Buzz 200" / "分析5件展品" / "完成一次5星导览"
- 完成章节目标→获得星星→提升馆长等级
- 馆长等级提升→解锁新博物馆或提高已有博物馆的星级上限

### 长期目标

| 目标类型 | 说明 |
|------|------|
| **馆长等级XV+** | 终极目标，需全部5个博物馆高星级 |
| **全展品收集** | 收集每一件展品的完美品质版本 |
| **全POI解锁** | 探索所有探险目的地 |
| **全成就** | Steam成就系统 |

### 沙盒模式

- 无章节目标限制
- 所有博物馆和房间从一开始解锁
- 可自由调整拨款幅度和挑战参数

## 五、失败条件

| 失败条件 | 说明 |
|------|------|
| **破产** | 资金归零且持续亏损（可通过破产赞助短暂维持） |
| **核心展品全部丢失** | 被偷光/烧光不可替代展品→无法吸引游客 |
| **全员离职** | 极端情况下全部员工离职→运营瘫痪 |

> 游戏整体偏宽容——破产后有不死鸟赞助续命，员工不满有缓冲期。设计意图是"施加压力而非阻断游戏"。

---

## 六、设计特点分析

**探险时间的异步设计**：探险进行期间，日常经营照常运转——这是一个关键的异步设计，防止了"探险=什么都没发生=无聊"的体验空洞。玩家在等探险队回来时可以继续装饰展品、调整布局、处理突发事件，两条时间线并行不悖。

**危机的技能门槛=隐性进度门**：火灾需要防火技能、盗窃需要监控室技能、幽灵需要幽灵捕获技能——如果玩家没有提前培训这些技能，危机发生时只能眼睁睁看着事态恶化。这种"软性进度门"比"等级不够打不开门"的硬性门更有机。

**破产不死鸟=优雅的失败设计**：资金耗尽后出现Jumbo McNally赞助（$4,500/月），相当于给了玩家最后一次翻身机会——但需要接受一个影响游客体验的人形立牌。这种"有代价的救援"比直接Game Over更有设计智慧。

---

> 跨品类对比: 与双点医院的"病人死亡=永久惩罚"不同，博物馆的失败更温和——展品丢了可以再探险找回，幽灵跑了可以再捕获。游戏整体从医院的"生死压力"转向了博物馆的"品味与效率"压力。也与温泉物语开罗系列的"年度评分"不同——博物馆没有年度结算的硬性Deadline。
