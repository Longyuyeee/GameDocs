## 一、展品/产品体系

### 五大博物馆×展品类型矩阵

| 博物馆 | 展品类型 | 代表展品 | 独有机制 |
|------|------|------|------|
| **纪念物之路** (史前) | 恐龙骨骼/化石/冰冻发现/古代发明/食人植物 | 波达迪洛骨架、冰冻冰箱、原始电脑 | 恐龙骨骼需多部件拼装 |
| **哀嚎旅馆** (超自然) | 被诅咒物品/幽灵(各时代)/著名灵魂 | 闹鬼人偶、地狱洞打手、玛格丽特·菲茨帕克 | 幽灵客房自定义(时代匹配) |
| **帕斯沃特湾** (海洋) | 温带鱼/热带鱼/寒带鱼/海怪/湿地星文物 | 小丑鱼、老虎鲨、水母 | 水族箱生态(鱼吃鱼/温度/水质) |
| **邦果废土** (科学) | 经典机械/工坊项目/科学材料 | 飞行器、印刷机、机器人清洁工 | 蓝图蓝图→工坊制作 |
| **佩伯利高地** (太空) | 外星文物/星体异常/外星科技 | 芝士-哞星文物、蛙人族神器 | 不明展品需分析后才能展示 |

### 展品品质等级

| 品质 | 基础Buzz | Perk插槽数 | 获取途径 |
|:--:|:--:|:--:|------|
| **普通** (Average) | 基础值 | 0 | 常规探险 |
| **优秀** (Great) | 1.3× | 1 | 常规探险（高调查等级提升几率） |
| **史诗** (Epic) | 1.6× | 2 | 调查等级高+详细探险/运气 |
| **完美** (Pristine) | 2.0× | 3 | 调查等级极高+详细探险+低概率 |

## 二、探险系统（核心研发机制）

### 探险流程

```
建造直升机停机坪 → 选择探险目标(POI) → 派遣员工团队 → 等待探险时间 → 获得展品/资源/事件
```

### 关键探险参数

| 参数 | 说明 |
|------|------|
| **调查等级** | 每个POI的累积探索进度，等级越高品质越好 |
| **探险时长** | 常规(标准时长) vs 详细(更长但品质更高) |
| **员工需求** | 部分POI需要特定技能（生存技能/飞行员之翼/快速修复等） |
| **事件风险** | 探险中可能出现攀爬事故/直升机损坏/毒害等，需要对应技能规避 |

### POI解锁逻辑

- 新POI需要满足条件：该主题/类型的总Buzz达阈值、调查等级、或博物馆星级
- 每个POI有特定的可发现展品列表
- 部分展品需多部件收集（如恐龙骨骼各部位→拼装完整骨架）

## 三、分析系统（展品知识研发）

### 分析室机制

| 操作 | 输入 | 产出 | 说明 |
|------|------|------|------|
| **分解展品** | 重复/低品质展品 | 知识(该类型永久+1) + 解锁Perk | 核心循环 |
| **解锁Perk** | 分解一定数量同类型展品 | 新Perk配方 | 合成到展品上 |
| **提升启蒙** | 累积分析操作 | 启蒙等级提升→解锁新POI | 跨博物馆的进度系统 |

### 知识的作用
- 信息牌将知识"传递"给游客→提升游客的知识评分
- 教授类游客按知识评分捐赠
- 高知识解锁该主题的新POI

---

## 四、Perk系统（展品定制化增强）

### Perk类型

| Perk | 效果 | 推荐用途 |
|------|------|------|
| **Buzz增强** | +X% Buzz | 高基础Buzz展品上效果最大化 |
| **装饰加成** | 锁定装饰评级为5 | 空间紧张无法放装饰时使用 |
| **知识增强** | +X% 知识传递 | 通用推荐，提升整体知识评分 |
| **安保贴纸** | 降低被盗几率 | 完美品质/不可替代展品必备 |
| **防尘涂层** | 降低维护频率 | 节省专家维护时间 |
| **停留时长增强** | 游客观看该展品时间+X% | 用于导览路线上的关键展品 |

- 仅优秀及以上品质展品可使用Perk
- 品质越高→Perk插槽越多（最多3个）
- Perk配方通过分析同类型展品解锁

## 五、工坊系统（科学博物馆专属）

### 工坊项目

| 项目类型 | 获取方式 | 制作要求 |
|------|------|------|
| **互动科技展品** | 探险获取蓝图 | 清洁工（工坊技能）+ 材料 |
| **诅咒消除器** | 超自然探险获取 | 清洁工（工坊技能） |
| **灵药** | 探险/事件获取配方 | 清洁工（工坊技能） |
| **机器人清洁工** | 科学探险蓝图 | 清洁工（工坊技能）+ 材料 |
| **机器人保安** | 科学探险蓝图 | 清洁工（工坊技能）+ 材料 |

- 工坊项目的品质可以升级（消耗材料重复制作）
- 互动科技展品是吸引儿童和太空长臂猿的关键

## 六、设计特点分析

**探险=产品研发，而非制造**：与商业经营品类常见的"投入材料→生产产品"模式（如开罗游戏开发物语的"组合类型+题材"）不同，双点博物馆的产品获取从"确定性制造"变为"概率性探险"——你不是在生产展品，而是在野外发现展品。这种"获取端的不确定性"创造了持续探险的动力。

**分析→知识→启蒙的三级研发体系**：分解重复展品→获取知识→传递知识给游客→提升启蒙等级→解锁新区域→发现新展品类型。这个三级循环（展品→知识→启蒙→新展品）将"刷重复掉落"这个常见的玩家抱怨转化为了一个有意义的养成系统。

**品质+Perk=展品定制化**：普通/优秀/史诗/完美四级品质+3个Perk插槽的组合，让每个展品有20+种可能的定制方案。这种轻量级但有效的"装备定制"系统在经营游戏中较少见——它让"哪个展品放C位"成为一个有意义的策略决策。

---

> 跨品类对比: 与游戏开发物语的"类型×题材组合→游戏评分"的确定性产品研发不同，双点博物馆的探险系统引入了"获取不确定性"（RNG元素），这在经营品类中是一个有趣的进化方向——在策略深度之上增加了一部分"运气"因素，让每次探险都有一定的开箱体验。
