## 一、扩张阶段划分

### 博物馆解锁顺序

| 阶段 | 博物馆 | 类型 | 解锁条件 |
|:--:|------|------|------|
| **初期** | 纪念物之路 | 史前 | 游戏开始即有 |
| **初期** | 哀嚎旅馆 | 超自然 | 馆长等级I + 纪念物之路1星 |
| **初期** | 帕斯沃特湾 | 海洋 | 馆长等级I + 纪念物之路1星 |
| **中期** | 邦果废土 | 科学 | 馆长等级III + 纪念物之路2星 + 哀嚎旅馆1星 + 帕斯沃特湾1星 |
| **后期** | 佩伯利高地 | 太空 | 馆长等级III + 纪念物之路2星 + 哀嚎旅馆1星 + 帕斯沃特湾1星 |

### 馆长等级解锁

| 馆长等级 | 每博物馆最高星数 | 说明 |
|:--:|:--:|------|
| I-II | 1-2星 | 初期阶段 |
| III-IV | 3星 | 解锁科学+太空博物馆 |
| V | 4星 | — |
| VI | 5星 | 单馆满星 |
| VII-XV+ | 6-14+星 | 持续积累 |

> 必须**轮换经营**多个博物馆——不能只刷一个馆的星级。每个馆获得的星星都贡献给全局馆长等级。

### POI探险解锁逻辑

| 解锁条件类型 | 示例 |
|------|------|
| **主题Buzz阈值** | "展示150 Buzz的化石"→解锁新化石POI |
| **Buzz加成比率** | "激活X%展品的Buzz加成"→解锁高级POI |
| **启蒙等级** | "启蒙等级达到5"→解锁该主题最终POI |
| **多博物馆联动** | 某POI需要其他博物馆达到一定星级 |

## 三、规模效应

### 正面规模效应

| 效应 | 说明 |
|------|------|
| **品牌效应** | 高星级博物馆→更多的自然客流 |
| **员工复用** | 高水平员工可在多馆之间调配 |
| **资金池共享** | 所有博物馆共享资金，盈利馆可补贴新馆 |
| **知识继承** | 分析获得的知识/Perk在所有博物馆通用 |
| **装饰品共享** | 解锁的装饰品在所有博物馆可用 |

### 负面规模效应（管理复杂度）

| 效应 | 说明 |
|------|------|
| **多线运营** | 同时管理3-5个博物馆→注意力分散 |
| **危机频率** | 博物馆越多→同时发生的突发事件越多 |
| **员工成本** | 每个馆需要独立员工团队→总薪资飙升 |
| **探险冲突** | 同一批高级专家要在多个馆之间分配探险任务 |

---

## 四、多博物馆管理

### 切换机制

- 通过世界地图随时切换博物馆
- 未活跃的博物馆进入"待机"状态（客流减少但问题也会累积）
- 长时间不理某个博物馆→展品劣化→幽灵逃脱→评分下降

### 多馆策略

| 策略 | 说明 |
|------|------|
| **专注法** | 先集中资源把一个馆冲到3-4星→再开新馆 |
| **轮换法** | 均衡发展多个馆（馆长等级提升更快） |
| **补丁法** | 主力馆稳定后偶尔切换到副馆灭火+派遣探险 |

## 五、科技/启蒙树

### 启蒙系统

| 启蒙要素 | 说明 |
|------|------|
| **获取方式** | 分析展品（分解重复品） |
| **作用** | 解锁新POI、解锁高级Perk配方 |
| **作用范围** | 按主题累积，跨博物馆生效 |
| **最高级** | 每个主题有独立启蒙等级上限 |

### Perk解锁树

```
分析同主题展品×1 → 基础Perk（Buzz增强/知识增强）
分析同主题展品×5 → 进阶Perk（防尘涂层）
分析同主题展品×10 → 高级Perk（安保贴纸/装饰加成）
分析同主题展品×20 → 终极Perk（停留时长增强）
```

## 六、设计特点分析

**强制轮换的扩张哲学**：不能只经营一个博物馆——馆长等级要求多馆都有星级。这与双点医院"一个医院玩到底"的设计截然不同。强制轮换解决了经营游戏的常见问题：后期内容枯竭（单馆已经完美了无事可做）。通过让玩家在5个主题之间跳转，始终保持新鲜感。

**知识/Perk的跨馆继承=软性Rogue-lite元素**：分析获得的知识和Perk在所有博物馆通用——这相当于一个跨局继承的元进度系统。玩家即使在B博物馆投资资源分析展品，A博物馆也会受益。这种设计降低了"开新馆=从零开始"的挫败感。

**负面规模效应=防止滚雪球**：随着博物馆数量增加，突发事件频率、员工成本、多线管理复杂度同步上升。这种设计避免了"有钱=无敌"的滚雪球效应——即使资金充裕，同时管理5个馆仍然有挑战性。

---

> 跨品类对比: 与Game Dev Tycoon的"同类型更高评级游戏"的纵向升级不同，博物馆的扩张路径是横向的（新主题博物馆）。与温泉物语的"扩建地块→更大地图"的渐进式扩张也不同——博物馆是多馆并行的网络化扩张。
