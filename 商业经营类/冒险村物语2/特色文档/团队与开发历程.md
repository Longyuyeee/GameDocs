## 2. 开发历程

### 2.1 从 1 代到 2 代: 开罗最重要的系统进化

初代《冒险村物语》（Dungeon Village）于 2012 年发布，内容相对简单：
- 玩家建设村庄设施，吸引冒险者前来访问
- 冒险者自动外出战斗，带回金币和物品
- 玩家无法直接控制冒险者，只能通过设施和环境间接影响

尽管 1 代玩法轻松有趣，但其系统深度有限——玩家实际上是"村庄管理 AI"而非"游戏参与者"。开罗社在 9 年后推出的续作进行了全面重构。

### 2.2 2 代的设计目标

据开罗社开发者自述，冒险村物语 2 的设计目标非常明确：

1. **解决"纯经营的被动感"** — 让玩家从"看着冒险者打"变成"参与冒险者的成长"
2. **增加 RPG 深度** — 引入转职系统、炼金系统、宠物系统
3. **扩展终局内容** — 7 张地图替代 1 代的一张小地图
4. **经济系统复杂化** — 三货币分离打破"只有金币"的单一维度

### 2.3 时间线

| 年份 | 事件 |
|:----:|------|
| 2012 | 初代《冒险村物语》手机版发布 — 简单的村庄建设+冒险者来访 |
| 2013-2020 | 开罗持续推出 30+ 款各主题经营游戏，积累系统设计经验 |
| ~2020 | 冒险村物语 2 立项 — 开罗决定为这个 9 年前的系列推出真正的续作 |
| **2021** | **手机版《冒险村物语 2》发布** — 大幅扩展：7 张地图、职业转职、炼金釜、城镇特征 100+ |
| 2022 | 手机版持续更新，收集玩家反馈 |

| 2023 | Switch 版同步发布 |

### 2.4 Steam 版的改进

冒险村物语 2 的 Steam 版不是简单的手机移植——它包含了针对 PC 平台的专门优化：

| 改进项 | 说明 |
|:------:|------|
| **高清画面** | 像素精灵适配高分辨率显示器 |
| **UI 重制** | 从手机触控 UI 改造为鼠标键盘操作的 PC UI |
| **加载优化** | 减少手机版的地图切片加载延迟 |
| **内容同步** | 包含手机版所有后续更新内容 |

## 3. "经营×RPG 融合"的设计创新

### 3.1 经营 × RPG 双循环

冒险村物语 2 的核心创新是建立了两个相互喂食的游戏循环：

```
经营循环: 建设设施 → 吸引冒险者 → 冒险者消费 → 获得金币 → 升级设施
                                 ↕
RPG 循环:  战斗获得物品 → 炼金合成 → 装备升级 → 冒险者变强 → 挑战更强怪物
```

这两个循环通过"冒险者"这一角色紧密结合——冒险者既是经营循环的消费者（使用设施、支付金币），也是 RPG 循环的主角（战斗、升级、转职）。

### 3.2 三货币经济系统

此前的开罗游戏多数只有"金币"一种货币。冒险村物语 2 引入了三货币分离的经济模型：

| 货币类型 | 来源 | 用途 |
|:--------:|:----:|:----|
| **金币** | 冒险者消费、任务报酬 | 建设村庄设施、购买基础物品 |
| **城镇点** | 战斗胜利、Boss 击杀 | 解锁高级设施与城镇特征 |
| **钻石** | 探索发现（宝箱、隐藏区域） | 购买稀有物品、特殊升级 |

三货币分离的核心设计目标是：**防止"赚钱即通关"**。即使玩家有很多金币，也需要通过战斗（城镇点）和探索（钻石）来解锁高级内容。

### 3.3 Roguelite 元进度

7 张地图（草原→洞窟→遗迹→雪山→火山→魔界→最终）构成了类似 Roguelite 的"赛季制"元进度结构。每一张新地图重置部分进度（冒险者等级回归、设施重造），但保留核心永久升级（炼金配方、城镇特征、宠物图鉴）。

这种结构在模拟经营品类中极为罕见——通常模拟经营是"建了拆、拆了建"的无限循环，而冒险村物语 2 引入了"征服→解锁→重来→更强"的赛季节奏。

## 5. 从 1 代到 2 代的进化幅度

| 系统 | 1 代 | 2 代 | 进化幅度 |
|:----:|:----:|:----:|:--------:|
| 地图 | 1 张 | 7 张（草原→魔界渐进）| ★★★★★ |
| 职业 | 少量 | **20+ 职业转职叠加** — 冒险者可以从战士→骑士→圣骑士等方向成长 | ★★★★★ |
| 炼金釜 | 无 | **物品→元素→装备升级** | ★★★★★ |
| 城镇特征 | — | 100+ 种特征组合解锁 | ★★★★★ |
| 宠物 | 简单 | 捕捉 + 养成 + 无限喂食 | ★★★★ |
| 货币 | 单一 | **三货币分离** | ★★★★★ |

进化幅度最大的三个维度（地图、职业、炼金釜）分别解决了 1 代的三大问题：终局内容不足、角色发展单薄、跨系统连接缺失。

## 6. 设计启示

1. **Roguelite 结构 × 模拟经营的融合** — 7 张地图 = 7 个赛季，展示了模拟经营游戏如何利用 Roguelite 的元进度设计解决"中后期无聊"的问题。这一设计在后续的 Against the Storm (2023) 中得到独立游戏界的进一步验证

2. **三货币分离 = 打破"有钱就无敌"** — 金币/城镇点/钻石的分离让"赚钱"不再是唯一目标 → 城镇点来源于战斗 → 钻石来源于探索。这在商业经营品类中是一次重要的经济系统创新

3. **炼金釜 = 跨系统胶合剂** — 一个机制将"食材购买"(经济系统)、"元素转化"(炼金系统)、"武器升级"(装备系统)、"战斗输出"(RPG 系统)串成一条完整的跨系统链条

4. **9 年续作间隔的利与弊** — 利: 积累了足够的技术和设计经验，续作能够实现质的飞跃；弊: 初代的粉丝可能已经流失，需要重新建立受众群体

5. **"经营 × RPG"是一种可复用的品类配方** — 冒险村物语 2 验证了"商业经营的游戏循环 + RPG 的角色成长"这种配方的可行性，为开罗后续的"经营 × 战斗"类游戏（如《冒险迷宫村》《星之开罗君》）提供了模板
