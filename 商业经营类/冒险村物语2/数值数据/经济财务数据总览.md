## 一、三货币体系

| 货币 | 获取方式 | 核心用途 | 稀缺度 |
|------|------|------|:--:|
| **金币(G)** | 建筑消费/住宿费 | 建造/道具/装备 | 低→中后期充裕 |
| **城镇点(TP)** | 每月按怪物击杀结算 | **转职**(核心)/特殊建筑/活动 | ★★★★ 永远稀缺 |
| **钻石(DP)** | 迷宫探索宝箱 | **稀有装备/水晶转职** | ★★★★★ 最稀缺 |

| 来源 | 权重 | 提升方式 |

| 装备店消费 | 40% | 武器/防具/饰品店属性喂满+数量多 |

## 三、城镇点经济学

### 每月结算公式(约)

```
月城镇点 = Σ(每个冒险者当月击杀怪物数 × 怪物城镇点值)
         × 城镇星级加成
```

### 城镇点最大化策略

| 策略 | 效果 |
|------|------|
| 喂满4大装备店→最强装备→冒险者伤害暴增 | 击杀效率暴增 |
| 多转职→累积战斗属性→一回合秒怪 | 击杀速度暴增 |
| 城镇高星级→加成倍率 | 间接提升 |
| 厨师职业→999努力值→光环触发 | 全面提升 |

## 四、钻石经济学

### 钻石最优使用

| 使用方式 | 消耗 | 优先级 |
|------|:--:|:--:|
| **王者之剑**(免费) | 0 | — |
| 黄金骷髅杖 | 18水晶 | ★★★★ |
| 其他水晶武器 | 15-30水晶 | ★★★ |
| **水晶转职** | — | **★(绝对不要!)** |
| 稀有道具 | — | ★★ |

> **核心策略**: 钻石**绝对不要**用于转职! 转职用城镇点。钻石只用于购买水晶武器/装备。

## 六、换村前清算

| 操作 | 原因 |
|------|------|
| 花光所有金币 | 新村庄继承极少 |
| 花光所有城镇点(转职/特殊建筑) | 同上 |
| 留100钻石给转职(核心5人可能需要) | 避免新村庄转职卡住 |
| 拆掉唯一建筑(回收资源) | 最大化换村前回收 |
