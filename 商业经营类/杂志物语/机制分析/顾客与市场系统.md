# 顾客与市场系统 — 杂志物语

> 游戏版本: Magazine Mogul (PC/Steam, Kairosoft, 2022)
> 数据来源: [Kairosoft Wiki (Fandom)](https://kairosoft.fandom.com/wiki/Magazine_Mogul), [Kairosoft Wiki Manual](https://kairosoft.fandom.com/wiki/Transcript:Manual_(Magazine_Mogul)), [Pocket Gamer Review](https://www.pocketgamer.com/magazine-mogul/review/)

## 一、读者体系：订阅者(Subscribers)为核心KPI

杂志物语中，**订阅者(Subscribers)** 是所有经营活动的核心目标指标——它既是杂志品质的验证，也是收入的基础。

### 订阅者增长模型

```
杂志品质 → 街头口碑 → 新订阅者 → 杂志销量提升 → 更多口碑
                        ↑
               推广活动(直接驱动)
                        ↑
               城镇人口(基础池)
```

| 增长方式 | 效果 | 说明 |
|------|:--:|------|
| **自然增长** | 缓慢持续 | 每期杂志发行后，品质驱动的小幅增长 |
| **推广活动(Publicity)** | 大幅增长 | 传单/海报/广播/电视——成本越高效果越强 |
| **杂志大赛获奖** | 爆发式增长 | 获奖后全国/全球关注→订阅者短期内暴增 |
| **城镇人口增长** | 间接增长 | 人口增加→潜在读者池扩大→订阅者天花板提升 |

### 订阅者与推广的失效机制

当订阅者数量达到一定规模后，部分推广方式会失效：

| 推广方式 | 失效条件 | 设计意图 |
|------|:--:|------|
| 传单(Leaflets) | 订阅者达到约1,000人后效果大幅下降 | 推动玩家升级到更高效的推广方式 |
| 海报(Posters) | 订阅者达到约5,000人后效果递减 | 逐步加码——推广投入与回报的动态平衡 |
| 广播(Radio) | 后期效果下降 | 电视(TV)成为终极推广手段 |
| 电视(TV) | 始终保持有效 | 最高成本、最高回报、无上限的推广方式 |

> **设计特点**: 推广方式的"阶段性失效"是一个精妙的节奏控制——它迫使玩家不断升级推广策略，而不是一招鲜吃遍天。这与真实世界中的营销逻辑一致：当你的受众从小众群体扩张到大众市场时，传播方式必须从"点对点"(传单)进化到"点对面"(电视)。这种设计也创造了"推广支出不断攀升"的自然经济压力。

## 二、城镇人口：受众基础池

城镇人口是订阅者的**基础池**。订阅者无法超越城镇人口的上限(大约比例)，因此城镇发展直接制约杂志的读者增长。

### 人口增长路径

| 方式 | 描述 | 阶段 |
|------|------|:--:|
| **公共事业** | 市议会推动公共项目→提升城镇文化→吸引居民迁入 | 中期解锁 |
| **店铺邀请** | 邀请各类店铺(生活/美食/服饰/娱乐)→完善城镇功能→更多居民定居 | 中后期 |
| **城镇合并** | 与周边城镇建立友好关系→合并→人口大幅增加 | 后期 |
| **自然增长** | 杂志宣传城镇→城镇知名度提升→缓慢人口增长 | 全周期 |

## 三、四种文化与读者分层

城镇有四种文化类型，不同文化吸引不同读者群体：

| 文化类型 | 提升方式 | 吸引读者特征 |
|------|------|------|
| **生活(Lifestyle)** | 生活类店铺+相关公共事业 | 关注家居/健康/日常内容的读者 |
| **美食(Food)** | 餐饮类店铺+美食活动 | 对美食内容有偏好的读者 |
| **服饰(Clothing)** | 服装店铺+时尚活动 | 对时尚/穿搭感兴趣的读者 |
| **娱乐(Entertainment)** | 娱乐场所+文化活动 | 追求娱乐/休闲内容的读者 |

### 文化与杂志内容的联动

四种文化不仅是城镇建设的数据维度，还会影响杂志内容的效果：

- 高生活文化 → 生活类主题的杂志特辑获得额外加成
- 高美食文化 → 美食相关切入角度的匹配度提升
- 高服饰文化 → 时尚类主题更受欢迎
- 高娱乐文化 → 娱乐类内容传播力更强

> **跨游戏对比 - 与冒险村物语2的"城镇特征"**: 冒险村物语2有100+种城镇特征解锁不同内容，而杂志物语的四种文化是更简洁的四维系统。两者的共同点是"城镇属性影响游戏核心玩法"——冒险村物语2的特征决定冒险者行为和掉落物，杂志物语的文化影响杂志内容效果。差异在于：冒险村物语2的特征是离散的(100+种独立特征)，杂志物语的文化是连续的(四维数值可同时提升)。连续维度更容易让玩家理解"全面发展"的目标，而离散特征更适合挖掘"特定组合"的深度。

## 四、杂志大赛(Town Magazine Awards)

### 赛事机制

| 项目 | 说明 |
|------|------|
| **举办方** | Town Magazine Awards(城镇杂志大赛) |
| **参赛条件** | 出版杂志即可自动参赛 |
| **评奖标准** | 杂志品质(写作/设计/趣味度)+当期销量+订阅者满意度 |
| **奖项等级** | 参与奖 → 优秀奖 → 亚军 → 冠军(大奖) |
| **大奖效果** | 全国/全球关注→订阅者暴增→城镇知名度大幅提升 |

### 获奖后的连锁反应

```
获大奖 → 全国媒体报道 → 订阅者暴增(爆发式增长)
    ↓
城镇知名度提升 → 更多居民愿意迁入
    ↓
店铺销售额提升 → 更多店铺愿意入驻
    ↓
城镇文化提升 → 公共事业效果增强
    ↓
下一期杂志品质更高 → 再次参赛有望卫冕
```

> **设计特点**: 杂志大赛是一个"全系统耦合点"——它将杂志品质、订阅者、城镇发展、经济收入全部串联起来。获奖不是终点，而是一个"正循环加速器"。这是开罗经典的"大奖引爆增长"设计(与游戏开发物语的GGA、漫画物语的漫画大奖同构)。

## 五、无竞争AI：开罗标配的"隐形竞争"

与大多数开罗游戏一致，杂志物语中**没有真实的竞争AI**：

| 竞争维度 | 实际情况 |
|------|------|
| **其他杂志社** | 不存在——玩家是城镇唯一的杂志社 |
| **营业额排名** | 显示"与其他杂志社的排名"，但本质是NPC分数脚本 |
| **杂志大赛对手** | 参赛对手是虚拟角色，没有独立的经营行为 |
| **读者争夺** | 不存在——读者的唯一选择就是玩家的杂志 |

> **设计分析**: 无竞争AI是开罗的一贯选择。去掉竞争能让玩家专注于"生产建设"而非"对抗压力"。但这也带来了一个问题：**没有竞争对手意味着"做得差"的后果很轻**——无论杂志多差，总有人买。因此开罗用"时间限制"(20年)和"成就目标"(大奖/订阅数里程碑)来替代竞争压力。这是一种"自我挑战"而非"击败对手"的动机模型。

## 六、口碑传播模型

| 传播阶段 | 机制 | 效果 |
|------|------|------|
| **直接购买** | 城镇人口→购买杂志→成为读者 | 基础消费 |
| **口碑推荐** | 满意的读者→推荐给朋友→新读者 | 自然增长主力 |
| **推广驱动** | 广告/活动→直接触达潜在读者 | 爆发式增长 |
| **获奖传播** | 媒体报导→全国关注→远距离订阅 | 最大规模增长 |

> **设计总结**: 杂志物语的顾客与市场系统本质是一个**城镇经营×杂志品质的二元增长模型**——城镇提供"读者池"，杂志品质决定"转化率"。推广活动是短期的"流量注入"，而城镇建设是长期的"池子扩容"。两者缺一不可。
