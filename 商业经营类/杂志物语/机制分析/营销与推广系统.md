## 一、推广活动(Publicity)

### 核心功能

推广活动是增加**订阅者(Subscribers)**的最直接手段，也是杂志物语营销体系的核心。

### 推广方式递进

| 推广方式 | 成本 | 效果范围 | 失效条件 |
|:------:|:--:|:--:|:--:|
| **传单(Leaflets)** | 低 | 街道/社区级 | 订阅者约1,000人后效果锐减 |
| **海报(Posters)** | 中低 | 城镇级 | 订阅者约5,000人后效果递减 |
| **广播(Radio)** | 中 | 区域级 | 中期效果下降(具体阈值待补充) |
| **电视(TV)** | 高 | 全国级 | 始终有效，无上限 |

### 推广活动的阶段性失效——深度分析

| 推广等级 | 受众定位 | 传播方式 | 触达上限 |
|:--:|------|------|:--:|
| 传单 | 路过行人 | 点对点物理分发 | 约1,000人 |
| 海报 | 公共空间人群 | 视觉曝光 | 约5,000人 |
| 广播 | 通勤/居家人群 | 听觉传播 | 数万人 |
| 电视 | 全体居民 | 视听传播 | 无上限 |

> **设计特点**: 推广方式的"阶段性失效"机制本质上是**市场分层理论**的简化模拟——不同类型(和成本)的推广手段触达不同规模的受众。当订阅者从小众(传单可覆盖)扩张到大众(需要电视覆盖)时，推广手段必须升级。这个设计在不增加复杂度的前提下，制造了"推广支出必须随规模递增"的自然经济压力——这是开罗游戏中少见的"营销预算规划"策略。

## 二、广告业务(Advertising)

### 机制

广告业务是杂志物语的特色收入来源——为城镇店铺印刷广告：

| 项目 | 说明 |
|------|------|
| **广告客户** | 城镇中的各类店铺 |
| **广告制作** | 分配给广告岗位的员工负责设计/撰写广告 |
| **广告质量** | 由员工的**设计(Design)**和**写作(Writing)**属性决定 |
| **收入** | 杂志发行后，按广告质量结算广告奖金 |
| **团队要求** | 广告岗位的员工组成广告团队→团队质量越高→奖金越多 |

### 广告质量的影响因素

| 因素 | 影响程度 | 说明 |
|------|:--:|------|
| **员工设计属性** | ★★★★★ | 广告视觉质量的核心决定因素 |
| **员工写作属性** | ★★★★ | 广告文案质量，影响广告效果 |
| **广告岗位人数** | ★★★ | 团队越大→可承接的广告业务越多 |
| **店铺类型** | ★★ | 不同店铺的广告预算不同 |

### 广告业务的经济价值

```
广告团队 → 制作高质量广告 → 杂志发行后结算广告奖金
    ↓
店铺满意 → 持续合作 → 稳定广告收入
    ↓
更多店铺入驻 → 更多广告客户 → 广告收入增长
    ↓
(良性循环)城镇繁荣 → 更多店铺入驻
```

> **设计分析**: 广告业务是将"城镇发展"与"杂志收入"直接挂钩的核心纽带——店铺越多→广告客户越多→广告收入越高。这个设计使"邀请店铺"不仅仅是为了城镇美观和人口增长，还有直接的经济收益。相比游戏开发物语(收入只来自游戏销售)和温泉物语(收入只来自游客消费)，杂志物语的**二元收入结构**(杂志销售+广告收入)使得经营策略更加丰富——你可以选择"内容优先"(专注杂志品质)或"广告优先"(专注广告团队建设)的不同路线。

## 三、杂志大赛——最大规模的营销事件

详见[评分与成就系统](./评分与成就系统.md)的详细分析。

### 大赛的营销价值

| 奖项 | 营销效果 |
|:--:|------|
| **参与奖** | 无特别效果 |
| **优秀奖** | 小范围知名度提升，订阅者小幅增长 |
| **亚军** | 全国范围关注，订阅者中幅暴增 |
| **冠军(大奖)** | 全球关注，订阅者大幅暴增 |

### 大赛获奖后的连锁营销效应

```
杂志大赛获奖 → 全国媒体报道 → 订阅者暴增
    ↓
广告客户信心提升 → 更多广告合作 → 广告收入增长
    ↓
城镇知名度提升 → 吸引新居民 → 潜在读者池扩大
    ↓
下一期杂志可获得更大关注 → 良性循环
```

## 四、公共事业——城镇级的"营销"

成为市议会成员后，推动公共事业项目相当于**城镇级的营销活动**：

| 公共事业类型 | 营销效果 | 间接影响 |
|------|------|------|
| 生活文化项目 | 提升生活文化→吸引生活类读者 | 增加杂志生活类内容的读者基础 |
| 美食文化项目 | 提升美食文化→吸引美食爱好者 | 吸引美食类店铺入驻→增加广告客户 |
| 服饰文化项目 | 提升服饰文化→吸引时尚人群 | 时尚人群的订阅转化率高 |
| 娱乐文化项目 | 提升娱乐文化→吸引休闲人群 | 扩大杂志的娱乐版块读者群 |

### 公共事业的"软营销"特性

公共事业不直接增加订阅者(不像推广活动)，但通过以下路径间接提升订阅基础：

```
公共事业 → 文化提升 → 居民满意度提升 → 人口增长
    ↓
人口增长 → 潜在读者增加 → 推广活动效果提升 → 订阅者增加
```

> **设计特点**: 公共事业是"软营销"——效果间接但长期。推广活动是"硬营销"——效果直接但需要持续投入。两者的结合构成了一个"短期+长期"的营销组合策略，与真实的营销理论(品牌建设+效果广告)高度吻合。

## 五、推广效果汇总

| 推广方式 | 直接目标 | 间接目标 | 成本类型 | 效果周期 |
|:------:|------|------|:--:|:--:|
| 传单 | 订阅者 | 无 | 一次性低额 | 单期 |
| 海报 | 订阅者 | 城镇知名度 | 一次性中低额 | 数期 |
| 广播 | 订阅者 | 品牌认知 | 一次性中额 | 单期 |
| 电视 | 订阅者 | 全国知名度 | 一次性高额 | 单期 |
| 广告业务 | 广告收入 | 店铺关系 | 持续(员工人力) | 每期 |
| 杂志大赛 | 订阅者(暴增) | 全国/全球知名度 | 无直接成本(需品质投入) | 事件性 |
| 公共事业 | 人口/文化(间接) | 订阅者基础池 | 城镇预算 | 长期持续 |

## 六、与同品类对比

| 对比维度 | 杂志物语 | 游戏开发物语 | 漫画物语 |
|------|:--:|:--:|:--:|
| 核心营销方式 | 推广活动(传单→电视) | 游戏媒体广告费投放 | 漫画杂志宣传 |
| 营销目标 | 增加订阅者 | 提升游戏知名度(Hype) | 提升漫画人气 |
| 二级收入营销 | 广告业务(为店铺服务) | 无 | 无 |
| 事件营销 | 杂志大赛 | GGA参展+获奖 | 漫画大奖 |
| 营销失效机制 | 推广方式阶段性失效 | 无(广告费始终有效) | 无 |
| 城镇营销 | 公共事业(间接) | 无 | 无 |

> **设计总结**: 杂志物语的营销系统是开罗游戏中最丰富的之一——它融合了"硬推广"(传单到电视)、"内容营销"(杂志大赛)、"服务营销"(广告业务)和"城镇营销"(公共事业)四个层次。推广方式的阶段性失效机制是开罗游戏中的独有设计，迫使玩家不断升级营销手段。广告业务作为二级收入来源的存在，使营销不仅是"花钱"(推广活动)，还可以是"赚钱"(广告业务)，这种"营销即收入"的设计在其他开罗游戏中很少见。
