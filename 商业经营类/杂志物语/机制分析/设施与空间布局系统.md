## 一、编辑部布局

### 工位配置

编辑部的核心设施是**工位(Desks)**——每个员工需要一个工位。工位的数量直接决定团队规模上限。

| 空间要素 | 说明 |
|------|------|
| **工位** | 每个员工占用一个工位。初始办公室工位有限 |
| **员工位置** | 体力低的员工放在靠近门口的位置更优——节省往返时间 |
| **布局效率** | 相邻工位的员工之间会产生"交谈"事件→获得知识点(Research Data) |

> **设计特点**: 员工交谈获得知识点是一个将"空间布局"与"员工成长"耦合的设计——相邻的员工越优秀，交谈产生的知识点越多。这让空间布局不仅是物理限制，还是员工培养策略的一部分。"体力低的员工放门口"是开罗游戏中常见的微操技巧(在游戏开发物语/漫画物语中也有类似设计)。

## 二、办公室升级与搬迁

### 升级路径

初始编辑部较小，随着业务扩张需要搬迁到更大的空间：

| 阶段 | 空间大小 | 工位数(约) | 解锁内容 |
|------|:--:|:--:|------|
| **初始办公室** | 小 | 4-5个 | 基础运营 |
| **第一次搬迁** | 中 | 6-8个 | 新培训方式、新广告选项、新招募方式 |
| **后续搬迁** | 大 | 8-10个 | 更多员工、更高效率的生产力 |

### 搬迁的影响

| 影响维度 | 具体变化 |
|------|------|
| **空间** | 更大空间→更多工位→可招聘更多员工 |
| **南瓜商店** | 搬迁后南瓜商店(Pumpkin Products)物价上涨 |
| **解锁内容** | 每次升级解锁新的培训方式、推广方式、招募方式 |
| **费用** | 逐次递增，成为中期重要的金钱Sink |

> **设计分析**: 每次办公室搬迁都解锁新内容(培训/推广/招募)，这让升级不仅是"空间变大"，更是"功能解锁"。这种设计消除了"搬家只是为了放更多人"的枯燥感——每次搬家都给玩家带来新的可玩内容。南瓜商店物价上涨与搬迁挂钩是一个负反馈设计——防止玩家无限搬迁而不受惩罚。

## 三、城镇建设(中后期核心)

成为市议会成员后，玩家进入**城镇建设**层面——这是杂志物语相较于早期开罗游戏(如游戏开发物语)的最大扩展。

### 公共事业

市议会可以推动公共事业项目，每种项目提升特定文化类型：

| 公共事业项目 | 文化类型 | 效果 |
|------|------|------|
| 打招呼运动 | 生活 | 提升生活文化，促进邻里关系 |
| 乘凉浴衣选美 | 服饰 | 提升服饰文化，吸引时尚人群 |
| 美食节 | 美食 | 提升美食文化 |
| 夏日祭典 | 娱乐 | 提升娱乐文化 |
| 其他项目 | 四种文化之一 | 根据项目类型决定文化增长方向 |

> 注：具体项目名称和效果可能因版本有所不同，部分项目待补充确认。

### 公共事业的资源消耗

| 资源 | 说明 |
|------|------|
| **城镇预算** | 推动公共事业消耗预算，与城镇税收挂钩 |
| **维护费用** | 大型项目需要持续维护，维护费用不足时项目可能自动终止 |
| **规模选择** | 玩家可对同一项目选择不同规模——规模越大→效果越强→维护费越高 |

## 四、店铺邀请

邀请各类店铺入驻城镇，完善城镇功能：

| 店铺类型 | 对应文化 | 对读者的吸引力 |
|------|:--:|------|
| **生活类店铺** | 生活 | 吸引关注日常生活的读者 |
| **餐饮类店铺** | 美食 | 吸引美食爱好者 |
| **服装类店铺** | 服饰 | 吸引时尚消费群体 |
| **娱乐类店铺** | 娱乐 | 吸引休闲娱乐人群 |

### 店铺邀请的策略意义

```
邀请店铺 → 完善城镇功能 → 吸引更多居民定居
    ↓
居民增加 → 潜在读者增加 → 订阅者天花板提升
    ↓
店铺繁荣 → 广告需求增加 → 广告收入提升
    ↓
城镇税收增加 → 公共事业预算增加 → 进一步城镇建设
```

> **设计特点**: 店铺邀请是城镇建设中的"杠杆"——投入一次(邀请店铺)产生多重效果(人口+文化+广告+税收)，是典型的"一举多得"设计。四种店铺类型对应四种文化，玩家需要根据当前杂志的内容偏向(生活/美食/服饰/娱乐)来优先邀请相关店铺——形成了一个"杂志内容→吸引读者→读者需要设施→邀请店铺→城镇发展"的反馈链。

## 五、周边城镇合并

### 合并机制

| 条件 | 说明 |
|------|------|
| **建立关系** | 与周边城镇互动，提升友好度 |
| **文化重叠** | 玩家城镇的文化与目标城镇的文化重叠度越高→关系增长越快 |
| **合并条件** | 友好度达到阈值后可合并 |
| **合并效果** | 土地扩大、人口增加、新取材地点解锁 |

### 合并的战略意义

```
合并城镇 → 土地面积扩大 → 可邀请更多店铺
    ↓
人口大幅增加 → 订阅者天花板提升 → 潜在收入增长
    ↓
新地点解锁 → 新取材机会 → 新主题/作家/内容库
    ↓
文化融合 → 四种文化获得新增长 → 杂志内容多样性提升
```

> **跨游戏对比 - 与都市大亨物语的"合并"**: 都市大亨物语中"合并"是通过占领地块扩展城市版图，核心是"空间扩张+地价提升"。杂志物语的城镇合并核心是"人口增长+文化叠加+新地点解锁"。两者的区别在于：都市大亨物语的合并是**物理空间导向**(更多土地→更高地价)，杂志物语的合并是**内容资源导向**(更多人口+新地点→更丰富的杂志内容)。这反映了两个游戏的本质差异——一个做"地产生意"，一个做"内容生意"。

## 六、空间布局策略总结

```
初级阶段(1-3年)
  初始办公室 → 有限工位 → 精简团队 → 优先取材和内容质量
        ↓
中级阶段(4-8年)
  首次搬迁 → 更多工位 → 扩招员工 → 解锁新培训/推广方式
        ↓
高级阶段(9-15年)
  成为市议会成员 → 城镇建设 → 店铺邀请 → 公共事业
        ↓
终局阶段(16-20年)
  城镇合并 → 最大化人口/文化 → 杂志大赛大奖冲刺
```

> **设计总结**: 杂志物语的设施与空间布局系统呈现**明显的三阶段结构**——从"编辑部内部布局"(工位配置)到"城镇设施建设"(店铺/公共事业)再到"区域扩张"(城镇合并)。每一阶段解锁全新的空间维度和玩法内容，让"扩张"不仅仅是数字增长，而是真正的新系统开放。这种"空间升级=玩法解锁"的设计是开罗游戏"易上手、有深度"哲学的具体体现。
