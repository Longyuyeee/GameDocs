# 经济与定价系统 — 杂志物语

> 游戏版本: Magazine Mogul (PC/Steam, Kairosoft, 2022)
> 数据来源: [Kairosoft Wiki (Fandom)](https://kairosoft.fandom.com/wiki/Magazine_Mogul), [Steam商店](https://store.steampowered.com/app/2054800/), [TouchArcade Review](https://toucharcade.com/2015/02/06/magazine-mogul-review/)

## 一、货币体系

游戏使用单一货币——**金币(G)**，无代币、无内购系统。资金来源多样但始终处于"勉强够用→大额开销→再积累"的循环中。

### 收入来源

| 收入渠道 | 描述 | 占收入比重（各阶段） |
|------|------|:--:|
| **杂志销售收入** | 核心收入来源。每期杂志发行后按销量结算，品质+定价决定销售额 | 初期70% → 中期60% → 后期50% |
| **广告收入** | 为城镇店铺印刷广告，杂志发行后按广告质量结算奖金。广告团队属性影响质量 | 初期15% → 中期25% → 后期30% |
| **订阅费** | 订阅者(Subscribers)每月自动付费，是稳定现金流来源 | 初期5% → 中期10% → 后期15% |
| **银行贷款** | 可从Alpaca Bank获得贷款，贷款额度与营业额排名挂钩 | 应急使用，非主力收入 |
| **取材发现金** | 外出取材时偶然发现的宝箱/事件可获得小額现金 | 初期补充，占比<5% |

> **设计特点**: 收入结构随阶段动态变化——杂志销售虽然始终占比最高，但广告收入和订阅费的比重随城镇发展稳步上升。这与"杂志社→城镇文化中心"的扩张路径完全吻合——杂志做得好→城镇繁荣→更多店铺→更多广告业务→收入结构多元化。

### 成本结构

| 成本类型 | 固定/可变 | 描述 |
|------|:--:|------|
| **员工薪资** | 固定/月 | 每月自动支出，随员工数量和等级递增。初期3-4人团队压力小，后期8-10人团队薪资成为主要负担 |
| **取材/外出费用** | 可变 | 派遣员工外出取材消耗金币，距离越远/时间越长→费用越高 |
| **印刷成本** | 可变/每期 | 每期杂志制作完成后需支付印刷费，与杂志页数/质量相关 |
| **作家稿费** | 可变/每期 | 签约连载作家后，每期杂志需支付稿费。作家知名度越高→稿费越高 |
| **办公室升级费** | 一次性 | 搬迁/升级办公室的大额支出，随升级次数递增 |
| **推广费用** | 单次/每期 | 进行推广活动(Publicity)消耗金币，传单/海报/广播/电视成本递增 |
| **公共事业费** | 定期 | 成为市议会成员后，公共事业项目消耗城镇预算，部分事业需持续维护 |

### 金钱Sinks(大额消费出口)

| 消费出口 | 金额(约) | 设计意图 |
|------|:--:|------|
| 首次办公室搬迁 | 数万G | 第2-3年的第一个大额门坎——迫使玩家积累，检验初期经营效率 |
| 后续办公室升级 | 逐次递增 | 中期门槛，在"扩招员工"与"储备现金"间做权衡 |
| 高级推广活动(电视) | 数万G/次 | 订阅者增长遇到瓶颈时的高额投入 |
| 城镇公共事业 | 数万G/项目 | 提升文化/人口的中后期投入 |
| 员工培训(高级方式) | 数千G/次 | 长期投资员工能力的持续消耗 |

> **设计特点**: 杂志物语的金钱Sinks不像游戏开发物语那样有"主机研发2,000万G"的终极目标，而是多条中小型支出线并行——办公室升级、城镇建设、员工培训、推广活动。这种"多线小额"的消费结构让玩家始终面临"这笔钱花在哪"的决策，而不是"存钱等大件"。

## 二、定价机制

### 杂志定价

玩家可以在每期杂志发行前设定售价。定价直接决定单本利润，但影响销量：

| 定价策略 | 效果 |
|:------:|------|
| **低价** | 销量更高，但单本利润低。适合追求订阅者增长的初期 |
| **中等价位** | 平衡销量与利润的标准策略 |
| **高价** | 单本利润高，但销量受限。适合杂志品质极高时 |

### 影响定价弹性的因素

| 因素 | 机制 |
|------|------|
| **杂志品质** | 写作(Writing)属性高的杂志能支撑更高的定价——读者愿意为好内容付更多钱 |
| **设计属性** | 设计(Design)属性影响初期销售爆发力——设计好的杂志在高价下销量下降更缓 |
| **订阅者基础** | 订阅者数量越大，定价弹性越大——忠诚读者对价格不那么敏感 |
| **城镇人口** | 人口越多→潜在读者越多→同一定价下可售出更多份 |

> **设计分析**: 与游戏开发物语("玩家不能定价，评分决定售价")不同，杂志物语给了玩家直接的定价权——这是一个有趣的差异。原因在于：杂志的"单位售价"是读者可以感知的(便利店一本杂志多少钱)，而游戏的售价在开罗世界观中是批发给渠道商的(玩家作为开发商不参与零售定价)。这种"是否给定价权"的设计选择反映了两个行业的不同商业逻辑。
>
> 但需要注意的是，杂志物语的定价权其实是**半自主**的——定价过高会直接体现为销量下降，而品质不足的杂志根本无法支撑高价。本质上，定价仍由产品质量(员工属性)间接决定，但多了一层"玩家选择价格带"的操作感。

## 三、贷款系统

### Alpaca Bank

| 项目 | 说明 |
|------|------|
| **贷款机构** | Alpaca Bank(羊驼银行) |
| **贷款额度** | 与营业额排名挂钩——排名越高→可贷额度越高 |
| **用途** | 应急资金、大额投资(如办公室搬迁) |
| **还款** | 需按时偿还，逾期可能影响信用或产生额外费用 |

> **设计特点**: 贷款额度与营业额排名挂钩，意味着"经营越好→融资能力越强"。这是一个正反馈设计——与现实中信用评级越高贷款利率越低的逻辑一致，但在游戏中被简化为"排名即信用"。

## 四、各阶段经济策略

| 阶段 | 年份(约) | 核心收入 | 核心支出 | 策略要点 |
|------|:--:|------|------|------|
| **起步期** | 1-3年 | 杂志销售+初期广告 | 员工薪资+印刷成本 | 控制团队规模，优先投资取材发现新主题/作家 |
| **成长期** | 4-8年 | 杂志销售+广告收入+订阅费 | 办公室升级+推广费用 | 扩大工位数，招聘更多员工，开始城镇建设投资 |
| **扩张期** | 9-15年 | 广告占比提升+订阅稳定 | 城镇公共事业+高级培训 | 成为市议会成员，推动公共事业提升文化，合并周边城镇 |
| **巅峰期** | 16-20年 | 多元收入+广告奖金高峰 | 顶级员工薪资+大额公共项目 | 冲击杂志大赛大奖，最大化订阅者数量和营业额 |

## 五、经济循环图

```
取材发现新主题/作家 → 提升杂志内容库
        ↓
策划高品质杂志特辑 → 定价销售 → 收入
        ↓                          ↓
广告业务(为店铺印刷) → 广告奖金    员工薪资+印刷成本等支出
        ↓                          ↓
城镇繁荣 → 更多店铺 → 更多广告业务 剩余利润 → 投资办公室/培训/推广
        ↓
城镇人口增长 → 更多订阅者 → 稳定订阅收入
        ↓
营业额排名提升 → 更高贷款额度 → 更大的投资能力
```

## 六、与同品类对比

| 对比维度 | 杂志物语 | 游戏开发物语 | 漫画物语 |
|------|:--:|:--:|:--:|
| 定价权 | 玩家自主定价 | 无定价权(评分决定) | 无定价权 |
| 核心收入 | 杂志销售+广告+订阅 | 游戏销售+合同 | 漫画销售 |
| 大额消费 | 办公室升级+城镇建设 | 办公室升级+主机研发 | 办公室升级 |
| 经济循环周期 | 每月(杂志出刊周期) | 数周(游戏开发周期) | 每月(漫画连载周期) |
| 贷款系统 | 有(Alpaca Bank) | 无 | 无 |
| 广告收入 | 有(为店铺印刷广告) | 无 | 无 |

> **设计特点总结**: 杂志物语的经济系统是开罗经营公式中**收入结构最丰富**的之一——杂志销售+广告收入+订阅费的三元收入结构使其比游戏开发物语(仅游戏销售+合同)更加多元。定价权是开罗游戏中的罕见设计，虽然实际操作中仍需遵循"质优价高"的经济规律。"多线小额"的消费结构让玩家始终有分散投资的压力，而不是聚焦于单一的大额目标。
