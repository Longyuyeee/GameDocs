## 一、杂志大赛(Town Magazine Awards)

### 赛事机制

杂志大赛是游戏中最核心的评分与奖励事件——每期杂志自动参赛，每年评选一次(或每个赛季?具体周期待确认)：

| 项目 | 说明 |
|------|------|
| **参赛条件** | 出版杂志自动参赛 |
| **评奖标准** | 杂志品质(写作+设计+趣味度)+当期销量+订阅者满意度 |
| **奖项等级** | 参与奖 → 优秀奖 → 亚军 → 冠军(大奖) |
| **大奖名称** | Grand Prize(大奖)/Town Magazine Awards冠军 |

### 奖项等级与效果

| 奖项 | 获取难度 | 效果 |
|:--:|:--:|------|
| **参与奖** | 自动获得 | 无特殊效果 |
| **优秀奖** | 低 | 小范围知名度提升 |
| **亚军(Runner-Up)** | 中 | 全国范围关注，订阅者中幅增长 |
| **冠军(Grand Prize)** | 高 | 全球关注，订阅者大幅暴增 |

### 大奖的深远影响

```
获大奖 → 订阅者暴增 → 收入大幅提升
    ↓
Steam成就解锁("Magazines that shine the town." 解锁率30.3%)
    ↓
城镇知名度飙升 → 人口加速增长
    ↓
广告客户信心增加 → 广告收入提升
    ↓
后续杂志的基础销量大幅提升
```

> **设计分析**: 大奖不仅是"荣誉",更是游戏中最强的增长加速器。Steam成就数据显示约30.3%的玩家解锁了大奖成就，意味着约1/3的玩家体验到了游戏的最高潮内容——这是一个健康的解锁率，说明难度设计合理，既不是太简单(人人都能拿)也不是太难(没人拿得到)。

## 二、订阅者里程碑

### 里程碑等级

订阅者数量达到特定数值时会触发里程碑事件：

| 订阅者数量(约) | 里程碑 | 效果 |
|:--:|------|------|
| 1,000 | 小有名气 | 城镇认可度提升 |
| 5,000 | 城镇之星 | 解锁某些内容(待补充) |
| 10,000 | 区域影响力 | 新推广方式解锁(如果未解锁) |
| 50,000 | 全国关注 | 触发特殊事件 |
| 100,000+ | 全球影响力 | 巅峰成就 |

注：具体的里程碑数值需要更精确的游戏内数据确认。

### 订阅者作为评分指标

在结局评分中，订阅者是重要的计分项：

```
结局评分 = 知识点 × 30 + 订阅者 × 8 + 奖杯 × 10,000 + 主题 × 550 + 地点 × 2,000 + 店铺总等级 × 3,500
```

| 评分项 | 权重系数 | 说明 |
|:--:|:--:|------|
| 知识点(Research Data) | ×30 | 累计获得的知识点总量 |
| 订阅者(Subscribers) | ×8 | 当前订阅者数量 |
| 奖杯(Trophies) | ×10,000 | 获得的大赛奖项数量 |
| 主题(Themes) | ×550 | 已解锁的主题数量 |
| 地点(Locations) | ×2,000 | 已解锁的地点数量 |
| 店铺总等级(Store Levels) | ×3,500 | 所有店铺等级之和 |

> **设计特点**: 结局评分公式是一个**多维累积指标**——它衡量玩家在游戏中的所有成就累积，而非仅仅是最终状态。知识点×30的权重表明"研发投入"在评价体系中占有重要地位；奖杯×10,000的高倍数说明获奖是拉高评分的捷径；店铺总等级×3,500的高权重说明城镇建设的重要性不亚于杂志经营本身。

## 三、营业额排名

### 排名机制

游戏中有"与其他杂志社的营业额排名"，但其他杂志社并非真实AI对手：

| 排名维度 | 说明 |
|------|------|
| **排名对象** | NPC脚本分数——非真实竞争AI |
| **排名依据** | 杂志营业额累计 |
| **排名变化** | 随玩家经营表现浮动 |
| **影响** | 高排名→Alpaca Bank贷款额度提升 |

> **设计分析**: 营业额排名本质是一个**自我挑战的进度条**——它告诉玩家你的经营表现处于哪个"段位"，但不涉及实质性的竞争压力。与Alpaca Bank贷款额度的挂钩使它有了实用价值。

## 四、Steam成就系统

### 成就列表

| # | 成就名称 | 描述 | 解锁率 | 难度 |
|:--:|------|------|:--:|:--:|
| 1 | **First issue!** | 证明出版了第一期杂志 | 85.2% | 极低 |
| 2 | **Any kind of seed can be a wonderful flower.** | 证明获得了隐藏职业 | 37.0% | 中 |
| 3 | **The soil and seeds that keep spreading.** | 证明合并了5个城镇 | 35.9% | 中 |
| 4 | **Invited by the sound of conch shell.** | 证明雇佣了Kairobot | 31.7% | 中高 |
| 5 | **Magazines that shine the town.** | 证明赢得了大奖 | 30.3% | 高 |
| 6 | **Stay tuned for the next issue!** | 证明经营了编辑部很多年 | 29.8% | 高 |

### 成就设计分析

| 成就 | 设计意图 |
|------|------|
| First issue! — 85.2% | 入门门槛——几乎所有玩家都能解锁 |
| 隐藏职业 — 37.0% | 激励玩家探索职业系统，"种子与花"的隐喻暗示了转职/隐藏内容 |
| 合并5城镇 — 35.9% | 推动玩家进行城镇扩张，探索合并机制 |
| 雇佣Kairobot — 31.7% | 奖励发现隐藏内容(Conch Shell的使用)，开罗彩蛋 |
| 大奖 — 30.3% | 核心挑战——约1/3的玩家能达到，平衡的难度设计 |
| 多年经营 — 29.8% | 需要坚持到结局(或至少接近结局)——最低解锁率 |

### 成就解锁率反映的玩家行为

```
85% → 至少试过游戏 (绝大多数)
37% → 深入体验了职业系统 (约1/3)
36% → 探索了城镇合并 (约1/3)
32% → 发现了Kairobot彩蛋 (约1/3)
30% → 实现了游戏的核心目标 (约1/3)
30% → 坚持到了终局 (约1/3)
```

> **设计特点**: 85%→30%的"漏斗"是标准的开罗成就曲线——大多数玩家浅尝辄止，约1/3的玩家体验了大部分内容。成就设计聚焦于"探索"(隐藏职业、Kairobot、城镇合并)和"完成"(大奖、坚持多年)，而非"效率"(如"在X年内获得大奖")。这种"重探索轻速度"的成就设计更符合开罗游戏的休闲定位。

## 五、多周目评分继承

### 继承计分

新游戏(+)中，已解锁内容提供更高的起点：

| 继承项 | 对新周目的影响 |
|------|------|
| **主题等级** | 开局即可制作高等级主题的杂志 |
| **角度等级** | 开局即有更多角度选择 |
| **Conch Shell** | 可直接招募Kairobot |
| **广告商/店铺类型** | 城镇建设开局更丰富 |

### 多周目的评分潜力

多周目允许玩家：

1. 基于已有内容库，更快达到高品质输出
2. 尝试不同的城镇建设路线(四种文化的不同发展策略)
3. 追求更高的最终评分(上限高于一周目)
4. 尝试"极限挑战"(如最短时间内获大奖)

## 六、与同品类评分系统对比

| 对比维度 | 杂志物语 | 游戏开发物语 | 漫画物语 |
|------|:--:|:--:|:--:|
| 核心评分 | 杂志大赛奖项 | 四媒体评分(1-10分) | 读者人气投票 |
| 评分维度 | 杂志品质+销量+订阅者 | 趣味/独创/画面/音乐 | 漫画品质 |
| 高分条件 | 大奖(Grand Prize) | 殿堂(HoF≥32)/年度最佳(GOTY≥36) | 漫画大奖 |
| 终局评分 | 多维公式(知识/订阅/奖杯等) | 无明确公式(年度流水) | 无明确公式 |
| 成就数量 | 6个(Steam) | 30个(Steam) | 约10个(Steam) |
| 成就风格 | 探索+完成 | 效率+完成 | 探索+完成 |

> **设计特点总结**: 杂志物语的评分与成就系统呈现"轻成就、重自我挑战"的特征——6个Steam成就在开罗游戏中属于较少的一档(游戏开发物语有30个)，但评分维度却相当丰富(6项指标的结局评分公式)。这与杂志物语的定位相符：它不追求"完成清单"(大量成就)，而是鼓励玩家"体验过程"(多维评分激励全面发展)。
