## 一、推广活动（Publicity）系统

推广活动是直接增加订阅者的主要手段。随着订阅者基数增长，低级别推广活动效果递减直至完全失效。

| 推广方式 | 费用(G)（推测） | 订阅者增量（推测） | 持续效果（期数） | 失效阈值（订阅者数） | 解锁条件 |
|------|:--:|:--:|:--:|:--:|------|
| 传单派发(Flyer Distribution) | 约10,000-20,000 | +30-80 | 1-2 | ~500 | 初始可用 |
| 街头宣传(Street Promotion) | 约30,000-50,000 | +80-200 | 2-3 | ~1,000 | 初始可用 |
| 社区活动(Community Event) | 约50,000-100,000 | +150-400 | 2-4 | ~2,000 | 中型办公室 |
| 报纸广告(Newspaper Ad) | 约80,000-150,000 | +200-500 | 2-3 | ~5,000 | 中型办公室 |
| 广播广告(Radio Ad) | 约150,000-250,000 | +300-800 | 3-4 | ~10,000 | 大型办公室 |
| 电视广告(TV Ad) | 约300,000-500,000 | +500-1,500 | 3-5 | ~50,000 | 大型办公室 |
| 名人代言(Celebrity Endorsement) | 约500,000-1,000,000 | +1,000-3,000 | 4-6 | ~100,000 | 豪华办公室 |

### 推广活动衰减机制

- 同一推广方式连续使用，效果递减。
- 切换不同推广方式后，之前方式的衰减效果逐渐恢复。
- **失效阈值**：当订阅者数超过该方式的失效阈值后，该方式完全不再有效——必须使用更高级的推广方式。

### 推广活动ROI分析

| 推广方式 | 每次成本/预计增量(G/订阅者) | 性价比评价 |
|:--:|:--:|:--:|
| 传单派发 | ~200-300 | ★★★ 初期最佳 |
| 街头宣传 | ~200-400 | ★★★ 初期好选择 |
| 社区活动 | ~250-400 | ★★★★ 中期性价比高 |
| 报纸广告 | ~300-400 | ★★★ 中期平稳 |
| 广播广告 | ~350-500 | ★★★ 稳定选择 |
| 电视广告 | ~300-500 | ★★★★ 效果好 |
| 名人代言 | ~300-500 | ★★★ 后期唯一选择 |

## 二、广告业务系统

广告业务是杂志的第二个核心收入来源。玩家为城镇中的店铺印刷广告，广告品质决定广告收入。

### 广告业务流程

1. 城镇中有店铺入驻后，可承接广告业务。
2. 分配员工到"广告岗位"制作广告。
3. 广告品质取决于员工的**设计属性为主（推测100%效率）+写作属性为辅（推测60%效率）**。
4. 广告在杂志中刊登，店铺向杂志社支付广告费。
5. 广告收入在杂志售出后结算。

### 广告品质等级与收入

| 广告品质等级 | 品质评分范围（推测） | 收入倍数（推测） | 条件 |
|:--:|:--:|:--:|------|
| 普通(Normal) | 0-30 | 1.0x | 设计属性较低的员工制作 |
| 良好(Good) | 30-60 | 1.2x-1.5x | 有一定设计能力的员工 |
| 优秀(Great) | 60-80 | 1.5x-2.0x | 高设计属性员工 |
| 完美(Excellent) | 80-100 | 2.0x-3.0x | 顶级设计师+良好要素匹配 |

### 广告收入计算公式（推测）

```
单笔广告收入 = 基础广告费 × 广告品质系数 × (1 + 杂志销量系数)
其中：
  基础广告费 = 固定值（与店铺等级相关）
  广告品质系数 = 广告品质评分 / 50（0-2.0x范围）
  杂志销量系数 = 杂志发行量 / 10,000（上限0.5x）
```

### 广告疲劳递减（推测机制）

| 同一店铺连续刊登期数 | 广告效果衰减 |
|:--:|:--:|
| 第1期 | 100% |
| 第2期（连续） | ~80% |
| 第3期（连续） | ~60% |
| 第4期+（连续） | ~40% |

> 切换不同店铺刊登可重置衰减。

## 三、杂志大赛营销效果

杂志大赛不仅带来荣誉，也是重要的营销手段。

| 奖项等级 | 参赛费（推测） | 订阅者增长（推测） | 知名度提升（推测） | 间接效果 |
|:--:|:--:|:--:|:--:|------|
| 未获奖 | 约10,000-20,000G | 0 | 少量 | 参与本身可能带来少量关注 |
| 单项奖（写作/设计/趣味） | 同上 | +200-500 | +5-10% | 当期杂志销量额外+20-50% |
| 专题奖（美食/时尚/娱乐） | 同上 | +300-600 | +10-15% | 解锁相关主题/角度的额外加成 |
| 综合大奖(Grand Prize) | 同上 | +500-2,000 | +20-30% | 当期杂志销量额外+50-100%，解锁特殊内容 |

## 四、公共事业推广效果

公共事业投资通过提升城镇文化水平，间接推动订阅者增长。这是一种长期、间接但可持续的推广方式。

| 公共事业项目 | 费用(G) | 对订阅者的间接效果（社区估算） | 生效周期 |
|------|:--:|:--:|:--:|
| 文化提升类项目 | 50,000-500,000 | 文化提升→人口吸引→潜在读者+5-20% | 长期（3-5期杂志后显效） |
| 人口增长类项目 | 100,000-1,000,000 | 人口增加→读者基数扩大+10-30% | 中-长期 |
| 店铺邀请 | 各店铺不同 | 店铺→广告客户+文化提升 | 持续 |
| 城镇合并 | 数百万G | 大人口增长+新地点解锁 | 永久性大提升 |

### 文化水平→订阅者增长间接转化路径

```
公共事业投资 → 文化值(四项)提升 → 城镇吸引力提升
    → 人口迁入 → 城镇人口增加
    → 更多潜在读者 → 转化率提升
    → 订阅者增长（间接但稳定）
```

## 五、连载作家对营销的贡献

签约作家不仅消耗稿费，也提供持续的营销效果。

| 贡献维度 | 效果 |
|------|------|
| 订阅者稳定增长 | 每期+5-100+（取决于作家等级和连载时间） |
| 杂志知名度 | 连载专栏增加杂志品牌认知 |
| 读者黏性 | 连载培养读者"追更"习惯，降低流失率 |
| 口碑传播 | 受欢迎连载引发读者推荐 |

## 七、推广有效性影响因素

| 因素 | 影响方式 |
|------|------|
| 杂志品质 | 高品质杂志→推广转化率更高 |
| 当期特辑主题 | 与推广内容契合的主题→加成 |
| 员工取材属性 | 取材员工影响推广活动效果（推测） |
| 城镇文化水平 | 文化越高→推广效果越好（推测） |
