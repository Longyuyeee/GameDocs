## 一、游戏经济系统概述

杂志物语的经济系统由多收入来源和多支出项目构成，随游戏阶段推进，各收入/支出项目的权重发生显著变化。核心经济循环：

```
杂志销售收入 → 现金流 → 员工薪资/取材费/印刷费/推广费
                              ↓
                         城镇建设投资 → 人口增加 → 更多订阅者
                              ↓
                         银行贷款（可选的资金杠杆）
```

## 二、全收入来源

| 收入来源 | 类型 | 计算公式（推测） | 各阶段占比 |
|:--:|:--:|------|:--:|
| 杂志销售收入 | 核心 | 定价×销量×品质加成 | 初期50% → 后期30% |
| 广告收入 | 核心 | 广告质量×店铺数量×广告费率 | 初期20% → 后期40% |
| 订阅费收入 | 稳定 | 订阅者数×每期订阅费 | 初期10% → 后期20% |
| 贷款(银行) | 债务 | 固定金额 | 仅初期使用 |
| 城镇税收 | 后期 | 城镇人口×税率 | 后期10%（推测） |

### 杂志销售收入详解

| 影响因素 | 效果系数（社区估算） | 说明 |
|:--:|:--:|------|
| 基础定价 | 1.0x | 玩家设定的销售价格 |
| 写作属性加成 | 0.8x-1.5x | 高写作延长销售周期 |
| 设计属性加成 | 0.8x-2.0x（短期） | 高设计提升首周爆发 |
| 趣味度加成 | 0.8x-1.5x | 全局影响力 |
| 错别字惩罚 | 0.5x-1.0x | 错别字越多惩罚越大 |
| 三要素匹配加成 | 1.0x-2.0x | 最佳匹配最高品质 |
| 知名度加成 | 1.0x-3.0x | 获奖和推广累积 |

### 广告收入模型

广告收入来源于为城镇店铺在杂志中印刷广告。

| 因素 | 影响 |
|:--:|------|
| 广告质量 | 由负责广告岗位的员工设计属性决定 |
| 店铺数量 | 城镇中入驻的店铺越多，潜在广告客户越多 |
| 杂志销量 | 销量越大，广告效果越好，广告费越高 |
| 广告品质→收入公式（推测） | 广告收入 = 基础广告费 × 广告质量系数 × 杂志销量系数 |

## 三、全支出项目

| 支出项目 | 频率 | 金额范围（推测） | 阶段特征 |
|:--:|:--:|:--:|:--:|
| 员工薪资 | 每年第4月 | 200-10,000G/人 | 前期占比大，后期比例下降 |
| 取材费用 | 每期 | 1,000-10,000G | 固定支出 |
| 印刷成本 | 每期 | 销量×单位成本 | 随销量递增 |
| 作家稿费 | 每期 | 500-50,000G/人 | 随作家等级递增 |
| 办公室搬迁 | 一次性 | 500,000-5,000,000G+ | 关键消费节点 |
| 推广活动费用 | 不定期 | 10,000-500,000G | 视推广方式而定 |
| 公共事业投资 | 不定期 | 50,000-1,000,000G+ | 后期主要支出 |
| 城镇合并费用 | 一次性 | 数百万G（推测） | 后期重大支出 |
| 道具购买 | 每年5月 | 1,500-数十万G | 按需购买 |
| 贷款利息 | 每月 | 贷款额×利率 | 仅在有贷款时 |
| 招募相关费用 | 不定期 | 签约金+入场费 | 扩张时支出 |

### 印刷成本模型

| 发行量级 | 每期印刷成本（推测） | 说明 |
|:--:|:--:|------|
| 初期（0-1,000册） | 约5,000-20,000G | 小批量印刷 |
| 中期（1,000-10,000册） | 约20,000-100,000G | 大批量印刷 |
| 后期（10,000+册） | 约100,000-500,000G+ | 规模效应，单位成本下降 |

## 六、经济效率指标

| 指标 | 初期（社区估算） | 中期（社区估算） | 后期（社区估算） |

### 资金使用优先级建议

| 优先级 | 投资方向 | 理由 |
|:--:|------|------|
| 1 | 员工薪资和培训 | 员工是生产力核心，培训ROI最高 |
| 2 | 办公室搬迁 | 解锁更多工位和功能 |
| 3 | 推广活动 | 增加订阅者最直接的方式 |
| 4 | 城镇公共事业 | 长期回报，扩大市场 |
| 5 | 道具购买（转职书优先） | 特定时间窗口（每年5月） |
| 6 | 城镇合并 | 后期重大扩张 |

## 七、止损与救助机制

| 机制 | 说明 |
|------|------|
| 自动救助 | 若资金过低，游戏可能触发救助机制（社区提及） |
