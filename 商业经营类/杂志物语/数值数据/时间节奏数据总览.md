## 三、每月出刊循环

每期杂志的制作遵循固定的流水线流程，通常每1-2个月出版一期（视员工效率和杂志规模而定）。

### 标准出版循环

```
第1-2周：企划会议(Planning Meeting)
    → 选择地点×主题×角度组合
    → 企划属性高的员工产生更好的企划方案（趣味度基础值）

第3-6周：取材(Scouting)与文章写作
    → 派遣员工取材（解锁新要素/搜集素材）
    → 写作属性高的员工负责执笔
    → 设计属性高的员工负责版面

第7-8周：校对(Proofreading)
    → 检查并修正错别字
    → 消耗Research Data修正错别字
    → 未被修正的错别字降低杂志品质

第9-10周：印刷(Printing)与发行
    → 确定最终版面（含广告和连载）
    → 印刷并发行
    → 杂志在报摊销售4-9周（取决于写作属性）
```

### 各阶段耗时影响因素

| 阶段 | 最短时间 | 最长时间 | 主要影响因素 |
|:--:|:--:|:--:|------|
| 企划会议 | 1周 | 2周 | 员工企划属性、要素组合复杂度 |
| 取材与写作 | 2周 | 4周 | 员工写作属性、取材属性、文章数量 |
| 校对 | 1周 | 2周 | 错别字数量、员工写作属性 |
| 印刷与发行 | 1周 | 1周 | （固定时间） |

## 四、杂志销售周期

每期杂志在报摊上销售4-9周，销售周期受杂志"写作"属性影响。

| 写作属性评分 | 推定销售周数 | 首周销量占比（推测） |
|:--:|:--:|:--:|
| 低（0-30） | 4-5周 | 40-50% |
| 中（30-60） | 5-7周 | 30-40% |
| 高（60-80） | 7-8周 | 25-35% |
| 极高（80-100） | 8-9周 | 20-30% |

> 设计属性影响首周销量峰值（爆发力），写作属性影响销售周数（持久力）。

## 五、取材（Scouting）节奏

取材是解锁新内容的核心手段，同时也是杂志制作流程的一部分。

| 取材类型 | 耗时 | 消耗 | 收益 |
|------|:--:|:--:|------|
| 城镇探索 | 1-2周 | 员工取材属性 | 新地点/新主题/新角度/新作家 |
| 特定地点取材 | 1周 | 无特别消耗 | 搜集特定素材 |
| 障碍突破 | 额外1周 | 高取材属性需求 | 解锁高级内容 |

### 取材机制说明

- 取材类似于"迷你地下城探索"（类似开罗的《冒险村物语》风格）。
- 途中会遇到"墙壁"（需要打破）、"问号"（需要调查）、"宝箱"（含有Research Data或金钱）。
- 员工的取材属性越高，突破障碍的成功率越高。
- 取材可以发现新的作家（签约连载）。

## 七、关键时间节点里程碑

| 时间节点 | 建议目标 | 延迟后果 |
|:--:|------|------|
| 第1年末 | 积累50,000G+、出版3-4期杂志 | 发展速度落后 |
| 第2年5月 | 准备至少4,500G购买南瓜商店道具 | 错过早期转职机会 |
| 第4年前后 | 首次办公室搬迁 | 工位不足限制员工数量 |
| 第8-10年 | 加入城镇议会、公共事业开始 | 城镇发展滞后 |
| 第12-15年 | 首次城镇合并 | 人口增长受限 |
| 第15-18年 | 冲击综合大奖(Grand Prize) | 成就达成困难 |
| **第20年3月** | **游戏结算/结局** | 最终得分定格 |

## 八、多周目继承（New Game+）

| 继承项目 | 是否继承 | 说明 |
|:--:|:------|------|
| 主题发现记录 | 是 | 已发现的主题在新周目中保留 |
| 地点发现记录 | 是 | 已发现的地点在新周目中保留 |
| 角度发现记录 | 是 | 已发现的角度在新周目中保留 |
| 员工等级 | 否 | **重置至初始等级** |
| 订阅者数 | 否 | 从零开始 |
| 资金 | 否 | 初始资金重新开始 |
| 部分广告客户 | 是 | 部分已合作的广告客户保留 |
| 部分店铺 | 是 | 部分已邀请的店铺保留 |
| 海螺壳道具(Conch Shell) | 是 | 特殊继承道具，用于招募Kairo君（推测） |
| 城镇文化等级 | 部分保留（推测） | 初始文化值有一定基础 |
| Steam成就 | 独立跟踪 | 不受多周目影响 |

### 多周目新增内容

- 使用海螺壳道具可以招募隐藏角色**Kairo君**（开罗系列吉祥物，全能型员工）
- 可能解锁新的初始要素组合
- 更高难度或更快的发展节奏（推测）
