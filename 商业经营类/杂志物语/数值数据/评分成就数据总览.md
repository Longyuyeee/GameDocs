## 一、杂志大赛（Town Magazine Awards）评分体系

杂志大赛每年举办一次，是衡量杂志品质的核心竞争机制。

### 评分维度与权重

| 评分维度 | 权重（推测） | 数据来源 |
|:--:|:--:|------|
| 写作(Writing) | 25% | 当期杂志的写作属性评分 |
| 设计(Design) | 25% | 当期杂志的设计属性评分 |
| 趣味度(Interest) | 25% | 当期杂志的趣味度评分 |
| 错别字率(Typos Penalty) | -25% ~ 0% | 错别字越多扣分越多 |
| 三要素匹配加成 | 10-20%（额外） | 地点×主题×角度匹配度 |

### 奖项等级与评分标准

| 奖项 | 所需评分（推测） | 奖金（推测） | 订阅者增长（推测） |
|:--:|:--:|:--:|:--:|
| 最佳写作奖 | 写作单项评分最高 | 约50,000-100,000G | +200-500 |
| 最佳设计奖 | 设计单项评分最高 | 约50,000-100,000G | +200-500 |
| 最佳趣味奖 | 趣味度单项评分最高 | 约50,000-100,000G | +200-500 |
| 美食奖(Gourmet) | 最高美食相关主题杂志 | 约80,000-150,000G | +300-600 |
| 时尚奖(Beauty) | 最高服饰相关主题杂志 | 约80,000-150,000G | +300-600 |
| 娱乐奖(Sports/Fun) | 最高娱乐相关主题杂志 | 约80,000-150,000G | +300-600 |
| **综合大奖** | **总分排名第1** | **约200,000-500,000G** | **+500-2,000** |

### 获奖策略

- **早期（第1-5年）**：专项奖为目标，集中提升单一属性（如堆高设计属性拿最佳设计奖）
- **中期（第6-12年）**：追求综合大奖，四属性均衡发展
- **后期（第13-20年）**：稳定获得多项奖项，确保综合大奖

## 五、营业额排名体系

游戏会统计杂志社的营业额，在全游戏中进行排名。

| 排名区间 | 推定月营业额门槛（社区估算） | 说明 |
|:--:|:--:|:--:|
| 未上榜 | <10,000G/月 | 起步阶段 |
| 15-20名 | ~10,000-50,000G/月 | 早期 |
| 10-15名 | ~50,000-200,000G/月 | 发展中 |
| 5-10名 | ~200,000-500,000G/月 | 中期 |
| 3-5名 | ~500,000-1,000,000G/月 | 后期 |
| 第2名 | ~1,000,000-3,000,000G/月 | 巅峰 |
| **第1名** | **~3,000,000G+/月** | **终极目标** |

### 提升营业额排名的关键

| 因素 | 影响 |
|------|------|
| 杂志定价与销量平衡 | 定价过高减少销量，定价过低降低收入 |
| 广告收入 | 后期广告收入占比可达40%，是超越排名的关键 |
| 订阅费 | 订阅者基数的持续贡献 |
| 成本控制 | 印刷成本随销量递增，需要规模效应对冲 |

## 六、终局评分计算

游戏在第20年3月（或第4个月）进行终局结算，评分基于多个维度。

### 评分维度（推测，基于开罗系列通用模式）

| 评分项 | 权重 | 说明 |
|:--:|:--:|------|
| 总资金(Funds) | 较高 | 累积财富的体现 |
| 知识点(Research Data) | 中等 | 研究积累 |
| 订阅者数(Subscribers) | 较高 | 核心经营成果 |
| 奖杯数量(Trophies) | 中等 | 获奖次数 |
| 发现主题数(Themes) | 每个×550分 | 内容丰富度 |
| 发现地点数(Locations) | 每个×2,000分 | 探索成果 |
| 城镇人口(Population) | 中等 | 城镇发展成果 |
| 商店总等级(Store Levels) | 较低 | 商业配套丰富度 |

### 终局评分等级（推测）

| 评分等级 | 总分范围（推测） | 评价 |
|:--:|:--:|------|
| S级 | 极高 | 完美经营 |
| A级 | 高 | 优秀 |
| B级 | 中等 | 良好 |
| C级 | 低 | 及格 |
| D级 | 极低 | 不及格（但游戏很难不及格） |
