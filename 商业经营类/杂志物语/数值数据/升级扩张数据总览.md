## 一、升级扩张体系概述

杂志物语的升级扩张分为三个维度：
1. **办公室升级** — 增加工位、解锁新功能（核心维度）
2. **城镇扩张** — 提升四种文化值、解锁公共事业和店铺（经营延伸）
3. **周边城镇合并** — 扩大城镇规模、增加人口和新地点（终局扩张）

三条线相互交织：办公室升级→解锁城镇议会→公共事业→文化提升→解锁合并条件→合并→人口增加→订阅者增长→收入增加→办公室再升级。

## 三、城镇扩张——四种文化系统

城镇的发展通过四种文化（Cultural Values）来衡量：**生活、美食、服饰、娱乐**。

### 文化等级阈值与解锁内容

| 文化等级 | 数值范围（推测） | 解锁内容 |
|:--:|:--:|------|
| Lv.1（初始） | 0-100 | 基础公共事业选项、基础店铺邀请 |
| Lv.2 | 100-200 | 中级公共事业、更多店铺类型 |
| Lv.3 | 200-350 | 高级公共事业、高级店铺类型 |
| Lv.4 | 350-500 | 特殊公共事业、豪华店铺 |
| Lv.5（最高） | 500+ | 终极公共事业、全店铺解锁 |

### 四种文化提升途径

| 提升途径 | 生活 | 美食 | 服饰 | 娱乐 | 效率 |
|:--:|:--:|:--:|:--:|:--:|:--:|

| 对应类型的店铺邀请 | O | O | O | O | ★★★（缓慢但持续） |
| 对应主题的杂志特辑 | O | O | O | O | ★★（间接影响） |

## 四、周边城镇合并

当城镇发展达到一定水平后，可以开始与周边城镇合并，大幅扩张城镇规模。

### 已知周边城镇（名称推测）

| 城镇名称 | 初始文化侧重 | 合并条件 | 合并费用（推测） | 合并后人口增量（推测） |
|------|:--:|------|:--:|:--:|
| 东克鲁顿(East Crouton) | 美食3星 | 友好度达标+支付费用 | 数百万G | +5,000-10,000 |
| 忍者镇(Ninjaville) | 服饰3星 | 友好度达标+支付费用 | 数百万G | +5,000-10,000 |
| 其他城镇 | 多种 | 友好度达标+支付费用 | 数百万G | +5,000-20,000 |

### 合并条件推测

| 条件 | 说明 |
|------|------|
| 友好度(Friendship) | 通过杂志报道该城镇+赠送文化投资=友好度提升 |
| 合并费用 | 一次性支付数百万G |
| 文化前置要求 | 特定文化值达到阈值（如生活文化300+） |
| 人口门槛 | 本镇人口达到一定规模后才可发起合并 |

### 合并效果

| 效果维度 | 说明 |
|:--:|------|
| 人口增加 | 直接增加数万人口，大幅扩大读者基数 |
| 新地点解锁 | 合并城镇的独特地点成为新的取材地 |
| 新主题/角度解锁 | 合并城镇的独特文化带来新的主题和角度 |
| 文化值提升 | 合并城镇的文化值合并计算，整体提升 |
| 商店种类增加 | 合并城镇带来的新店铺类型 |
| 广告客户增加 | 更多店铺=更多广告收入来源 |

## 六、扩张路线推荐

### 最优扩张路线

```
阶段1（第1-5年）：聚焦办公室升级
  → 目标：从Lv.1升级到Lv.2
  → 关键：积攒资金、训练员工、稳定出版

阶段2（第5-10年）：解锁城镇建设
  → 目标：从Lv.2升级到Lv.3，加入城镇议会
  → 关键：平衡杂志品质与城镇投资

阶段3（第10-15年）：全面发展文化
  → 目标：四种文化达到Lv.3-Lv.4
  → 关键：针对性投资公共事业，邀请互补型店铺

阶段4（第15-20年）：城镇合并冲刺
  → 目标：合并3-5个周边城镇（对应Steam成就FIVETOWN）
  → 关键：积累足够资金，提升友好度
```

### 扩张效率指标

| 投资方向 | 回收周期（推测） | 长期ROI |
|:--:|:--:|:--:|
| 办公室升级 | 中等（2-4年） | ★★★★★ 功能解锁的乘数效应 |
| 公共事业投资 | 较长（3-5年） | ★★★★ 间接但可持续 |
| 店铺邀请 | 较短（1-2年） | ★★★ 直接+间接收益 |
| 城镇合并 | 很长（5年+） | ★★★★ 终局大规模扩张 |
