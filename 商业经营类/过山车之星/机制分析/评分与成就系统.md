# 评分与成就系统 — 过山车之星

> 游戏版本: Planet Coaster v1.13+ (Steam, 2016)
> 数据来源: [Frontier官方公园管理指南](https://www.planetcoaster.com/), [Planet Coaster Wiki](https://planetcoaster.fandom.com/)

---

## 一、公园评分(Park Rating)——四维度木桶模型

公园评分是过山车之星的**全局评价指标**，由四个维度组成——**最低的维度决定整体评分**（木桶效应）：

| 维度 | 评估内容 | 提升方式 |
|------|------|------|
| **设施声望(Attraction Prestige)** | 所有设施Prestige的综合水平 | 建造高Prestige设施+提升景观 |
| **景观评分(Scenery Rating)** | 全园景观覆盖度和质量 | 在游客可见处放置装饰品 |
| **独特设施(Unique Attractions)** | 不同类型设施的数量 | 建造多样化的设施类型 |
| **公园声誉(Park Reputation)** | 游客总体满意度(心情+评价) | 满足游客需求+保持设施高品质 |

### 评分→客流量映射

| 公园评分 | 客流量级别 | 可承受票价 |
|:--:|:--:|:--:|
| 0-500 | 极低 | 低 |
| 500-1000 | 低 | 中低 |
| 1000-2000 | 中 | 中 |
| 2000-3000 | 高 | 中高 |
| 3000+ | 极高 | 高 |

## 二、设施Prestige三因素模型

### Prestige计算公式（概念）

```
Prestige = 乘坐体验分(Excitement/Fear/Nausea) + 景观奖励分(Scenery Bonus) + 声誉修正(Ride Reputation)
```

| 因素 | 权重 | 提升难度 |
|------|:--:|:--:|
| 乘坐体验(EFN) | ~50-60% | 高（需要轨道的工程设计） |
| 景观奖励 | ~30-40% | 中（放置装饰品） |
| 声誉老化 | ~10% | 低（保持高Prestige即可） |

### 声誉老化曲线

```
高Prestige: New → Established → Classic（永恒吸引力）
低Prestige: New → Established → Dated → Aging → Old（没人坐）
```

## 三、成就系统

### 生涯模式星级

| 星级 | 条件 | 奖励 |
|:--:|------|------|
| 铜(Bronze) | 基础指标达到 | 解锁下一关 |
| 银(Silver) | 中等指标 | 额外资金/物品 |
| 金(Gold) | 高指标 | 稀有解锁 |
| 铂(Platinum) | 全指标完美 | 终极奖励 |

### Steam成就精选

| 类型 | 示例 | 难度 |
|------|------|:--:|
| 建造 | 完成首个过山车/首个蓝皮书 | 低 |
| 经营 | 公园月利润10万+/游客数5000+ | 中 |
| 景观 | 景观评分达1000+ | 中 |
| 挑战 | 生涯全金/沙盒全解锁 | 高 |

## 四、游客评价反馈

### 游客"想法"系统

游客在公园中会产生"想法"(Thoughts)——实时反馈他们的状态：
- "这个队列太长了！"
- "这里的风景真美！"
- "这太贵了！"
- "我还想再坐一次！"

这些想法聚合在公园管理面板中→玩家可看到最频繁的正面/负面反馈。

## 五、设计特点分析

**木桶模型=强制全面发展**：公园评分四维度的"最低分决定总分"法则迫使玩家不能偏科——你有最好的过山车但全园都是垃圾？公园评分被毁灭。你景观极美但没有一个吸引人的设施？评分同样被毁灭。这种"短板决定一切"的设计在经营游戏中极为有效——它防止了单一维度的滚雪球。

**Prestige=复合评分的典范**：将"乘坐体验"（客观的物理模拟数据）、"视觉呈现"（主观的景观密度）、"时间因素"（声誉老化）三者合成为一个数字——这个设计将三个完全不同性质的评价维度统一到了一个可优化的指标上。比双点博物馆的Buzz/Knowledge双维度更进一步——Prestige是三因素融合。

**Classic=永恒吸引力的浪漫**：一个长期保持高Prestige的过山车会变成"Classic"——它会吸引"远方的游客"专程来坐。这是游戏对真实"经典游乐设施"的致敬——某些老过山车因为历史悠久和声誉卓著，会成为公园的核心吸引力数十年。

---

> 跨品类对比: PC的Prestige与双点博物馆的Buzz有相似的"核心评价指标"角色，但Prestige融合了更多维度（物理模拟+视觉+时间）。与温泉物语的"客层评分"不同——Prestige是设施级别的而非全域的。
