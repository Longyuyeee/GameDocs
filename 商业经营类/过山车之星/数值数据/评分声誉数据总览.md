# 评分声誉数据总览 — 过山车之星

> 游戏版本: Planet Coaster v1.13+ (Steam, 2016)
> 数据来源: [Frontier官方公园管理指南](https://www.planetcoaster.com/)

---

## 一、公园评分四维度

| 维度 | 评估方式 | 满分参考 |
|------|------|:--:|
| 设施声望 | 全设施Prestige总值/均值 | 无上限 |
| 景观评分 | 全域景观物品数量×品质 | ~5000-10000 |
| 独特设施 | 不同类型设施的种类数 | 取决于解锁总量 |
| 公园声誉 | 游客满意度调查均值 | 0-100% |

> 总分 = 最低分维度的分数——**木桶效应**(最短的木板决定水位)

## 二、评分→影响

| 公园评分 | 基础客流量 | 游客消费意愿 |
|:--:|:--:|:--:|
| <500 | 极低 | 低 |
| 500-1000 | 低 | 中低 |
| 1000-2000 | 中 | 中 |
| 2000-3000 | 高 | 高 |
| 3000+ | 极高 | 极高 |

## 三、设施声誉老化对Prestige的修正

| 阶段 | Prestige修正 | 游客吸引力 |
|:--|:--:|:--:|
| New/Established | 0% | 基准 |
| Dated | -15% | 略降 |
| Aging | -30% | 明显下降 |
| Old | -50-70% | 几乎无人乘坐 |
| Resurging | +15% | 回升 |
| **Classic** | **+30%** | **额外+远方游客** |

---

> 注: 生涯模式的星级评分与沙盒模式有不同标准。上表为挑战/沙盒模式的通用数据。

## 四、四维度快速诊断

| 症状 | 最可能低的维度 | 对策 |
|------|:--:|------|
| 客流增长缓慢 | 公园声誉(满意度低) | 检查游客反馈→修复最大痛点 |
| 票价无法提升 | 设施声望(Prestige低) | 提升景观/翻新老化设施 |
| 评分停滞不前 | 独特设施(种类单一) | 建设不同类型的设施 |
| 新游客不来 | 景观评分(公园太丑) | 大规模景观投入 |

## 五、Classic设施的培育

| 条件 | 要求 |
|------|------|
| Prestige维持 | 长期≥该类型设施平均水平 |
| 持续时间 | 至少1-2年(游戏内) |
| 景观维护 | 保持高景观评分不下降 |
| 翻新投入 | 定期机修工翻新(防止滑入Aging) |
| Classic收益 | +30% Prestige + 远方游客——相当于"免维护收入"
