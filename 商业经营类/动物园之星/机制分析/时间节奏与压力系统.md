# 时间节奏与压力系统 — 动物园之星

> 游戏版本: Planet Zoo v1.16+ (Steam, 2019)
> 数据来源: [Frontier官方指南](https://www.planetzoogame.com/), [Planet Zoo Fandom Wiki](https://planetzoo.fandom.com/)

---

## 一、时间结构

| 速度 | 用途 |
|:--:|------|
| 暂停 | 建造栖地/布置景观/规划布局 |
| 1× | 观察动物行为/游客流/运营节奏 |
| 2-3× | 加速等待(繁殖/研究/月末结算) |

## 二、动物的时间维度

| 阶段 | 持续时间 | 管理需求 |
|------|:--:|------|
| 幼年期 | 物种决定 | 生长中/不繁殖 |
| 青年期 | — | 开始可繁殖 |
| 壮年期 | 最长 | **最佳繁殖窗口/放归高峰期** |
| 老年期 | 渐短 | 生育力下降/需准备替代个体 |
| 死亡 | — | 移除尸体/基因传承检查 |

## 三、经营节奏

| 阶段 | 活动 |
|------|------|
| 建设期(暂停) | 设计栖地/放置围栏/配置地形植被 |
| 观察期(1×) | 看动物行为/检查福利/诊断问题 |
| 月末 | 财务结算/调整定价/基因审计 |
| 物种关键事件 | 新物种引进/濒危物种繁殖/白化动物诞生=客流峰值 |

## 四、压力系统

| 压力 | 来源 | 后果 |
|------|------|------|
| 财务 | 肉食动物饲料成本/过度建设 | 破产 |
| 种群 | 快速繁殖物种失控 | 栖地超载→福利下降 |
| 疾病 | 未检疫新动物/脏栖地 | 传染→种群崩溃 |
| 基因衰退 | 近亲繁殖持续多代 | 免疫力/生育力归零 |
| 时间老化 | 种畜变老 | 繁殖窗口关闭 |

## 五、设计特点分析

**动物生命周期=不可逆的时间压力**：与过山车之星的"翻新=重置老化"不同，动物的衰老和死亡是**不可逆的**。一只20岁的壮年种虎将在5-8年内进入老年期——你需要在它衰老前确保有下一代来延续基因。这种"代际规划"给动物园经营增加了一条PC没有的时间维度。

**快速繁殖物种=隐性的种群危机**：孔雀/疣猪等快速繁殖物种如果不避孕——会在短时间内填满栖地→超过最大群体上限→福利崩溃。PZ没有"自动停止繁殖"——玩家必须主动管理种群数量。这是对真实动物园管理挑战的诚实模拟。

**疾病检疫=风险管理的"软教程"**：新动物进入动物园时如果不经过检疫隔离——可能引入疾病→全栖地感染→种群崩塌。这个机制无声地教授了生物安全的重要性——玩家经历过一次"全园感染"后，再也不会跳过去检疫。

---

> 跨品类对比: PZ的"代际规划"是过山车之星完全不具备的时间维度——在PC中过山车是"永恒的"(可翻新/可重建蓝图)，在PZ中动物是"有限的"(出生→繁殖→死亡)。

## 六、季节性与特殊事件

| 事件 | 时机 | 影响 |
|------|------|------|
| 新物种引进 | 交易/购买后 | 开幕客流峰值+50-100% |
| 濒危物种繁殖成功 | 随机 | 保育评分大幅提升 |
| 白化/变异动物诞生 | 极低概率 | 动物园吸引力急剧上升 |
| 动物死亡(老龄) | 寿命终结 | 种群减少→检查是否有替代个体 |
| 动物死亡(意外) | 福利崩溃/疾病 | 检查根因+防止扩散+声誉损失 |
| 社区挑战(Frontier发布) | 定期 | 全社区目标→参与获得保育点数奖励 |

## 七、代际规划的具体策略

| 动物年龄 | 行动 |
|------|------|
| 幼年(0-20%寿命) | 不要繁殖——等待成熟 |
| 青年(20-40%) | 开始繁殖——最佳窗口开启 |
| **壮年(40-70%)** | **核心繁殖期——最大化繁殖次数/放归窗口** |
| 中年(70-90%) | 繁殖末班车——准备替代个体 |
| 老年(90-100%) | 停止繁殖——准备动物护理/临终关怀 |
