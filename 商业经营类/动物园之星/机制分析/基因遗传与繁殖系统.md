## 一、四基因体系

每只动物有四组基因，每组独立遗传：

| 基因 | 类型 | 遗传机制 | 影响 |
|------|:--:|------|------|
| **体型(Size)** | 加性(Additive) | 两等位基因值相加(如+42%+33%=75%) | 动物外观大小+动物评分+放归价值 |
| **寿命(Longevity)** | 加性(Additive) | 同上 | 动物能活的年数 |
| **生育力(Fertility)** | 多样性(Diversity) | 两序列的差异度越大→值越高 | 繁殖成功率 |
| **免疫力(Immunity)** | 多样性(Diversity) | 同上——突变率高 | 抗病能力 |

### 遗传类型对比

| 特性 | 加性基因(体型/寿命) | 多样性基因(生育力/免疫力) |
|------|------|------|
| 最优值来源 | 两个高值等位基因相加 | **两序列差异越大越好** |
| 近亲繁殖后果 | 无直接影响 | **双字母→0%值（致命）** |
| 突变率 | 低(~1/10) | 高(免疫力尤其) |
| 优化策略 | 选择性育高值个体 | **避免近亲繁殖/引入新基因** |

## 二、近亲繁殖与基因多样性

### 近亲繁殖惩罚

- 生育力和免疫力的**多样性基因**使用A/B/C/D序列
- 近亲交配→后代序列重复字母→差异度降低→值大幅下降
- 极端近亲→**0%生育力/0%免疫力**→无法繁殖/极易生病

### F1杂交优势

两个基因序列完全不同的亲本（即使各自免疫力都是0%）→后代继承两个不同的序列→**后代免疫力达到100%**。这是动物园育种的核心策略——利用基因差异"刷新"种群的免疫力和生育力。

## 三、毛色/外观遗传

| 特性 | 机制 |
|------|------|
| 毛色变异 | 白化(Albino)/黑化(Melanistic)/其他变异——**隐性基因** |
| 表达条件 | 需要**两个隐性等位基因同时存在**才能表达毛色变异 |
| 获取方式 | 近亲繁殖→增大隐性基因"double up"的概率 |

> 毛色变异动物极其罕见且极具吸引力——一只白化大猩猩或白虎能吸引大量游客。

## 四、繁殖条件

| 条件 | 要求 |
|------|------|
| 双亲福利 | **≥66%(理想>90%)** |
| 群体空间 | 不超最大群体上限 |
| 性别比例 | 符合物种推荐 |
| 食物品质 | 研究后提升食物等级→增加繁殖率 |
| 物种生育力 | 基础参数+基因加成 |

### 繁殖控制

| 方式 | 适用场景 |
|------|------|
| **避孕** | 控制种群不超上限但不移除动物 |
| **物理分离** | 将雄性/雌性分到不同栖地 |
| **放归野外** | 减少数量的同时获取保育点数 |
| **出售/交易** | 经营模式下的种群管理 |

## 五、育种策略

### 社区育种的"动物标签"系统

高级育种玩家会在动物名称中编码基因值：
```
"Anya 100 83 67 50" = 体型100/寿命83/生育力67/免疫力50
"Tiger 63 66" = 标记体型/寿命的序列步骤
```

### "隔代遗传"效应

```
父(100%体型) × 母(0%体型) = 子代(全部50%体型)
子代1(50%体型) × 子代2(50%体型) = 孙代(25%概率100%), (25%概率0%), (50%概率50%)
```

## 六、设计特点分析

**真实孟德尔遗传学的游戏化**：PZ的基因系统不是简单的"好/坏"二元分——它模拟了加性遗传(多个基因的累加效果)和多样性遗传(基因池的健康度)。这种真实的遗传学建模在主流经营游戏中是独一无二的。

**近亲繁殖=保护教育的隐性课程**：游戏中"近亲繁殖→种群崩溃"的机制无声地教授了基因多样性的重要性。玩家在经历几次"种群退化"后会自然地学会"引入新基因"——这就是真正的保护生物学教育。

**白化动物=稀有收藏品的经济学**：隐性基因的低表达率让白化/黑化动物成为极稀有的"收藏品"——它们不仅是视觉上的特别，更是动物园吸引力的巨大加成。这种设计将遗传学变成了一个"寻宝"游戏。

---

> 跨品类对比: PZ的基因系统让"动物"从过山车之星的"设施"（静态的/固定的/可替换的）变为"生命体"（可成长的/可遗传的/不可完全替代的）。一只基因完美的种马是不可复制的资产——这与PC中"蓝皮书可以无限复制"的逻辑完全不同。
