# 员工管理系统 — 动物园之星

> 游戏版本: Planet Zoo v1.16+ (Steam, 2019)
> 数据来源: [Frontier官方员工指南](https://www.planetzoogame.com/), [Planet Zoo Fandom Wiki](https://planetzoo.fandom.com/), [Steam社区](https://steamcommunity.com/app/703080)

## 一、七大员工类型

| 类型 | 核心职责 | 必需设施 | 推荐配比 |
|------|------|------|------|
| **饲养员(Keeper)** | 清洁栖地/喂食/补水/报告疾病 | 饲养员小屋 | 1人/3栖地 |
| **兽医(Vet)** | 治疗疾病/手术/抓捕逃脱动物/运输动物/**研究** | 兽医手术室+研究中心 | 1-2人 |
| **清洁工(Caretaker)** | 清扫路面/倒垃圾/清洁厕所/运输装箱动物 | 无 | 1人/主要区域 |
| **机修工(Mechanic)** | 修理围栏/水电设施/破坏物品/研究 | 工坊 | 1人/3-4围栏 |
| **保安(Security Guard)** | 阻止破坏/偷窃/游客投喂/恐吓动物 | 无 | 1人/主要区域 |
| **教育员(Educator)** | 主持动物讲座/游走互动/提升游客教育 | 动物讲座点 | 1-2人 |
| **商贩(Vendor)** | 运营商店/餐饮 | (商店自行附带) | 1人/1-2商店 |

### 与过山车之星的关键差异

| 差异 | 过山车之星 | 动物园之星 |
|------|------|------|
| 饲养员 | 无 | **核心角色**——动物依赖饲养员生存 |
| 兽医 | 无 | 治疗/手术/研究/运输 |
| 教育员 | **无** | **全新角色**——动物讲座+游走教育 |
| 演员(过山车之星) | 娱乐游客 | 替换为教育员（教育+娱乐融合） |

## 二、工作区域(Work Zones)

| 分区策略 | 说明 |
|------|------|
| 按区域分区 | 将动物园分为几个大区→每区配置完整的员工团队 |
| 功能分区 | 饲养员+兽医分区；清洁工/机修工/保安分区(更大范围) |
| 危动物种专任 | 高价值/高需求动物→专属饲养员+兽医 |

## 三、培训与等级

| 等级 | 效率 | 薪资涨幅 | 风险 |
|:--:|:--:|:--:|------|
| 1星 | 基准 | 基准 | — |
| 2星 | +20% | +15% | — |
| 3星 | +40% | +30% | — |
| 4星 | +60% | +50% | — |
| **5星** | **+100%** | **+80-100%** | 薪资压力显著 |

### 培训优先级

1. **兽医**(最高)——研究速度直接决定食物品质和丰容解锁速度
2. **饲养员**(高)——喂食/清洁效率直接影响动物福利
3. **教育员**(中)——更高星的讲座带来更大教育提升
4. **商贩**(不推荐)——与PC一致，过度培训=离职风险

## 四、设计特点分析

**饲养员=动物生命保障系统的唯一支柱**：在PC中，如果机修工旷工，过山车故障=退票。在PZ中，如果饲养员旷工，动物可能**饿死**。这种生死依赖关系让饲养员成为PZ中最关键的角色——你需要为每个栖地做好"饲养员覆盖"的冗余规划（不能只有1个饲养员负责所有栖地）。

**兽医=研究者+医生的双重身份**：兽医同时负责动物治疗和研究——这两个职能之间存在资源竞争。派兽医去研究=解锁更好的食物和丰容，但研究期间该兽医不能治疗动物。这种"研究vs治疗"的权衡与PC机修工的"维护vs研发"一脉相承。

**教育员=游乐场中没有的"知识传播者"**：教育员是PZ独有的创新——他们在游戏中的角色是将"动物学知识"从系统数据转化为游客的教育评分。一个5星教育员主持的动物讲座可以一次性大幅提升数十名游客的教育水平。

---

> 跨品类对比: PZ增加了兽医和教育员两个PC不具备的角色——这反映了"动物经营"比"设施经营"多出的两个维度：生命健康(兽医)和知识传播(教育员)。
