# 经济财务数据总览 — 动物园之星

> 游戏版本: Planet Zoo v1.16+ (Steam, 2019)
> 数据来源: [Frontier官方财务指南](https://cms.zaonce.net/en-GB/node/5478), [ScreenRant](https://screenrant.com/)

---

## 一、双货币体系

| 货币 | 获取 | 用途 | 跨园共享 |
|------|------|------|:--:|
| 现金(Cash) | 门票+捐赠+商店+出售动物 | 建造/工资/饲料/运营 | 否 |
| 保育点数(CC) | 放归+社区挑战+玩家交易 | **购买稀有/高基因动物** | 是 |

## 二、收入来源占比

| 来源 | 占比 | 优化杠杆 |
|------|:--:|------|
| 捐赠箱 | 30-40% | 教育板+动物视野+愿望清单 |
| 门票 | 20-30% | 动物园评分+逐步提价 |
| 商店/餐饮 | 20-30% | 食品2×/纪念品4×/领养包100%利润 |
| 展示动物自动出售 | 5-10% | 蝴蝶/蝎子/蛙→高速繁殖 |
| 动物交易 | 浮动 | 基因优化副产物 |

## 三、各阶段财务指标

| 指标 | 初期(2-5物种) | 中期(10-15) | 后期(20-50+) |
|------|:--:|:--:|:--:|
| 月收入 | $3K-10K | $20K-60K | $80K-300K+ |
| 月支出 | $2K-7K | $12K-40K | $50K-200K+ |
| 饲料% | 15-20% | 25-35% | 25-40% |
| 推荐储备金 | $5K+ | $20K+ | $50K+ |

## 四、定价策略

| 项目 | 定价 | 触发警告 |
|------|:--:|------|
| 门票 | 逐步提高 | 游客说"太贵"→降价一格 |
| 食品/饮料 | **成本×2** | — |
| 纪念品 | **成本×4** | — |
| 领养包 | 100%利润 | 零进货成本 |
| 捐赠箱 | 不可定价 | 靠教育和动物吸引力提升 |

## 五、初期经济路线

```
展示动物盒(蝴蝶/蝎子)×3 → 自动出售=被动收入
→ 鸵鸟/疣猪栖地(低成本) → 捐赠箱+教育板 → 稳定现金流
→ 积累$10K+储备 → 第一个中型食草动物(斑马/长颈鹿)
→ 游客量提升→商店收入→再投资→第一个肉食动物
```

> 核心原则: 避免早期肉食动物——单只狮子日饲料费可抵整个展示动物区的月收入。
