> 数据来源: [Big Ambitions Wiki](https://big-ambitions.fandom.com), [Steam社区指南](https://steamcommunity.com/app/1331550/guides), [官方论坛](https://forum.bigambitionsgame.com)

## 一、经济体系总览

Big Ambitions拥有商业经营品类中最**复杂且最具现实感的经济系统**之一——玩家不仅要管理公司财务，还要维持个人生计。这种"双重经济"结构是该游戏最显著的设计特征。

| 经济层面 | 货币 | 核心循环 | 层级 |
|:--:|:--:|------|:--:|
| **公司经济** | USD | 进货→销售→利润→扩张 | 主循环 |
| **人物经济** | USD | 工作收入→个人消费→生活维持 | 副循环 |
| **投资经济** | USD | 闲置资金→基金/股票→被动收入 | 终局循环 |

> **设计特点分析**: 传统商业经营游戏（如《游戏开发物语》《温泉物语》）通常将玩家与公司视为一体——公司赚钱=玩家有钱。Big Ambitions的"双重经济"将玩家角色与公司财务剥离：玩家需要自己租房、买车、吃饭，如果个人账户没钱，即使公司账户充裕也会"饿死"。这一设计迫使玩家在扩张公司规模的同时保持对个人财务的关注，增加了代入感和策略深度。

## 二、收入来源

### 2.1 公司收入

| 收入类型 | 占比权重 | 机制说明 | 解锁阶段 |
|------|:--:|------|:--:|
| **零售收入** | 主力(60-70%) | 店铺直接销售商品给顾客，按定价×销量结算 | 初期 |
| **批发转售** | 辅助(10-20%) | 从批发商/进口商采购→加价转售给其他商家 | 中期 |
| **工厂制造利润** | 高级(15-25%) | 自建工厂生产商品→供应自有店铺或批发 | 中后期 |
| **股票/基金投资收益** | 被动(5-15%) | 在Vantander银行开设投资账户，享受~8%年均回报 | 中期后 |
| **房地产租金** | 被动(待补充) | 购买商业地产后出租给其他商户 | 后期(待验证) |

### 2.2 人物个人收入

| 收入类型 | 说明 | 阶段 |
|------|------|:--:|
| **公司分红** | 从公司账户提取资金至个人账户 | 全程 |
| **初始工作** | 游戏开局时玩家有一份初始工作提供基本收入 | 初期 |
| **兼职收入** | 在开设公司前可从事兼职维持生计 | 初期过渡 |

### 2.3 收入汇总对比

```
初期(第1-7天):    初始工作收入 + 首店少量利润 → 勉强维持个人开支
中期(第1-4周):    多店零售利润 → 公司盈余增长 → 开始偿还贷款
中后期(第2-3月):  零售+批发+工厂+投资 → 多管道收入 → 大量现金累积
终局(第3月+):     全产业链控制 → 被动收入覆盖所有成本 → 财务自由
```

## 三、成本结构

### 3.1 公司运营成本

| 成本类型 | 支付频率 | 数额范围 | 说明 |
|------|:--:|:--:|------|
| **进货成本** | 按订单 | 可变(最大变量) | 供应链各层级价格不同——批发商最贵→批发仓库中等→进口商最便宜 |
| **店铺租金** | 每周/月 | $500-$5,000+/月 | 曼哈顿各区差异巨大，核心区域租金高昂 |
| **员工工资** | 每周 | $300-$2,000+/周/人 | 9种角色薪资不同，管理层(如HR经理、采购经理)薪资更高 |
| **贷款利息** | 每周 | 取决于贷款额 | 小银行利率较高，大银行贷款额度大但审批条件不同 |
| **税收** | 每周/月 | 利润的百分比 | **不注意会翻车的主要原因之一**——税率较高且累进计算 |
| **装修与设备** | 一次性 | $1,000-$50,000 | 含货架、收银台、展示柜、装修材料、电脑、服务器等 |
| **水电费** | 每月 | 待补充 | 店铺运营的固定杂费 |

### 3.2 个人生活成本

| 成本类型 | 数额范围 | 说明 |
|------|:--:|------|
| **房租** | $500-$3,000/月 | 人物需要租房，价格取决于位置和大小 |
| **车辆费用** | $200-$1,000/月 | 购车或养车费用，影响通勤效率 |
| **餐饮/生活** | $100-$500/周 | 人物需要吃饭，不同餐厅价格不同 |
| **服装/娱乐** | 待补充 | 人物有一定社交和个人需求 |

> **设计特点分析**: 成本结构中最具"特色"的是**税收系统**——许多商业游戏要么没有税收（如开罗系列），要么税收仅作为背景数值。Big Ambitions的税收是**实打实的支出项**，且不注意规划会严重影响利润。玩家需要像真实企业主一样考虑"税前利润vs税后利润"、"哪些支出可以抵税"、"如何通过合法途径降低税负"。这在商业经营品类中是一个**硬核特色**。

## 四、定价策略

### 4.1 定价模式

| 定价策略 | 操作方式 | 利润效果 | 风险 |
|------|------|:--:|:--:|
| **市价策略(Market Price)** | 系统自动设定为当前市场均价 | 稳定但平庸 | 低——懒人模式 |
| **高价策略(+X%)** | 手动上浮一定百分比 | 利润显著提升 | 中——可能流失价格敏感顾客 |
| **低价策略(-X%)** | 手动下浮 | 薄利多销/抢占市场 | 中——压制竞争对手利润空间 |
| **动态调价** | 根据客流/竞争/时段实时调整 | 最大化收益 | 高——需要持续监控 |

### 4.2 定价弹性

Big Ambitions并不要求玩家死板地按市价销售。玩家可以将价格设置在**市价+10%~20%**的水平而仍然能够卖出商品——这种"定价弹性"是游戏经济系统的深度设计：

| 商品类型 | 加价容忍度 | 说明 |
|------|:--:|------|
| **普通消费品** | +5%~10% | 顾客对价格敏感，加价空间有限 |
| **高端/奢侈品** | +15%~25% | 珠宝、电子产品等高价商品有更大加价空间 |
| **热门商品** | +15%~20% | 需求旺盛时顾客愿意支付更高价格 |
| **日用必需品** | +0%~5% | 基本没有加价空间 |

### 4.3 定价策略与竞争AI的互动

定价策略不是孤立决策——它直接触发竞争AI的反应。详见[营销与竞争系统](./营销与竞争系统.md)。

> **跨游戏对比 — 经济系统复杂度**

| 维度 | Big Ambitions | 双点医院 | 冒险村物语2 | 游戏开发物语 |
|------|:--:|:--:|:--:|:--:|
| **货币种类** | 1种(USD)但有双重账户 | 1种 | 3种 | 1种 |
| **个人财务独立** | **是(核心设计)** | 否 | 否 | 否 |
| **定价自由度** | **全自由(市价±)** | 固定 | 固定 | 固定 |
| **税收系统** | **是(实缴，有影响)** | 无 | 无 | 无 |
| **贷款系统** | **多级银行** | 简单贷款 | 无 | 无 |
| **供应链复杂度** | **5阶段(最深)** | 无 | 无 | 无 |
| **投资系统** | 基金~8%年回报 | 无 | 无 | 无 |

## 五、贷款系统

| 银行 | 贷款额度 | 利率 | 门槛 | 特点 |
|:--:|:--:|:--:|:--:|------|
| **小型银行(初期)** | ~$15,000 | 较高 | 无(开局可用) | 入门级，解决启动资金 |
| **中型银行** | ~$50,000-$100,000 | 中等 | 中等信誉 | 扩张期过渡 |
| **Vantander银行(大额)** | ~$1,000,000 | 较低 | **中期无门槛** | 大幅扩张的引擎，但利息压力大 |
| **投资银行** | 待补充 | 待补充 | 待补充 | 待验证 |

贷款技巧：
- 优先用小银行贷款起步——金额小/审批快
- 中期换Vantander银行申请大额贷款——一次性拿下多个店铺
- **注意**: 贷款利息每周扣款，如果店铺利润不足覆盖利息+运营成本会陷入债务螺旋

## 六、投资基金

Vantander银行同时提供投资基金功能——玩家可将闲置资金投入基金获取被动收益：

| 指标 | 数值 |
|------|:--:|
| 平均年回报率 | ~8% |
| 风险等级 | 低-中 |
| 存取灵活性 | 高(随时赎回) |
| 最低投资额 | 待补充 |

> 投资基金的策略意义：当公司现金流充裕且暂时没有扩张计划时，将资金投入基金比闲置在账户中更高效。相当于"资金效率最大化"手段。

## 七、税收系统详解

税收是Big Ambitions中最容易被新手忽视的"利润杀手"：

| 税务类型 | 计税方式 | 应对策略 |
|------|------|------|
| **公司所得税** | 按净利润百分比征收 | 合理规划支出降低应税利润 |
| **个人所得税** | 从个人账户中扣除 | 控制个人提取金额 |
| **销售税** | 待补充 | 待验证 |

> 提示: 游戏内税收是**累进税率**——利润越高税率越高。这意味着盲目扩张→高利润→高税率，可能反而降低净利润率。合理控制规模有时比无限制扩张更有效率。

## 八、各阶段经济策略

| 阶段 | 核心目标 | 资金来源 | 关键动作 |
|:--:|------|------|------|
| **生存期(Day1-7)** | 开第一家店、维持个人生计 | 初始贷款+少量零售 | 租小店→进货→设定市价→保证现金流正 |
| **扩张期(Week2-4)** | 开2-3家店、建立稳定现金流 | 零售利润+中型贷款 | 跨品类开店→利用需求报表选业态→开始关注税收 |
| **增长期(Month2-3)** | 供应链升级(批发仓库→进口商) | 多店利润+大额贷款 | 建仓库→签采购合同→降低进货成本→还清之前贷款 |
| **规模化(Month3+)** | 工厂制造+Hub&Spoke网络 | 全管道收入+基金收益 | 建工厂→自产自销→投资→被动收入覆盖成本 |

## 九、与同类游戏的跨品对比

| 对比维度 | Big Ambitions | 双点医院(TPH) | 冒险村物语2 |
|------|:--:|:--:|:--:|
| **经济系统核心理念** | 真实商业模拟(双重经济+供应链) | 轻量经营(平衡收支即可) | 三货币分离战略经济 |
| **定价自由度** | 完全自主定价(弹性空间) | 无定价权(服务价格固定) | 无定价权(固定售价) |
| **个人财务管理** | 强制(吃饭/租房/买车) | 不存在 | 不存在 |
| **税收复杂度** | 高(累进税率/税务筹划) | 无 | 无 |
| **供应链深度** | 极深(5阶段进化) | 无 | 无(直接购买设施) |
| **玩家身份** | 创业者兼CEO | 医院院长 | 村长 |

> **设计特点分析**: Big Ambitions的经济系统选择了"真实商业模拟"方向——它不像开罗系列那样用抽象化的单一货币简化经济，也不像双点医院那样将经济作为"解锁关卡的门槛"。它将**现实企业经营的财务复杂性**（定价弹性、税务规划、贷款杠杆、供应链成本优化、个人财务独立）系统性地转化为玩法机制。这种选择提升了游戏的学习门槛，但也为追求深度的玩家提供了其他商业经营游戏无法比拟的"当老板"体验。特别是"个人财务独立于公司"的设计——玩家不能像其他游戏那样"公司即我"，而是必须同时管理两套账本，这在商业经营品类中是一个**颠覆性的创新**。
