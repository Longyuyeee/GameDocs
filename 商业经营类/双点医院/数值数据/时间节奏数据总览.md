# 时间节奏数据总览 — 双点医院

> 游戏版本: Two Point Hospital Steam版 v2018.8.29 (基础版)
> 数据来源: [Fandom Wiki](https://two-point-hospital.fandom.com), [Steam社区指南](https://steamcommunity.com/sharedfiles/filedetails/?id=1519429114)

## 一、时间模型

| 特性 | 说明 |
|------|------|
| 模型 | **实时制**(连续流逝) |
| 控制 | **可暂停**+可加速+可减速 |
| 最小单位 | 秒(现实时间) |
| 月 | ~数分钟(取决于速度设置) |

## 二、频率事件

| 频率 | 事件 | 操作 |
|:--:|------|------|
| 每月末 | 财务报告 | 检查利润/退款/薪资 |
| **随机** | **急诊(Emergency)** | 暂停→优先处理→加价100% |
| **随机** | **疫情(Epidemic)** | 暂停→全员疫苗 |
| **随机** | 竞争对手CEO | 检查设备/幸福感 |
| 每年末 | 年度评奖 | — |
