## 三、员工特质速查

### 最佳特质(必招)

| 特质 | 效果 | 对所有员工 |
|------|------|:--:|
| **"我很便宜"(I'm Cheap)** | 薪资减半 | ✓ |
| 工作狂 | 工作速度+ | ✓ |
| 正能量 | 周围幸福感+ | ✓ |
| 教学能力 | 培训速度+50% | ✓(在培训室) |

### 最差特质(必拒)

| 特质 | 效果 |
|------|------|
| **"我很贵"(I'm Expensive)** | 薪资翻倍 |
| 懒惰 | 工作速度- |
| 恶心 | 周围幸福感- |
| 如厕需求高 | 频繁离岗 |
| 口渴 | 频繁离岗 |

## 四、技能效率临界值

| 组合 | 达到治疗能力上限 |
|------|:--:|
| Lv3机器 + Lv3员工+2个治疗技能 | **100%治疗能力**(溢出无效果) |
| Lv3机器 + Lv4员工+3个治疗技能 | 同上(浪费了技能) |
| Lv1机器 + Lv5员工 | 约70%治疗效果(机器拖后腿) |
