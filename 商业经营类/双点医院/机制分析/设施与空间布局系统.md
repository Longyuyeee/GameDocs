## 一、房间分类

双点医院的建筑逻辑是**房间(Room)制**——每个房间是一个独立的功能单元，由玩家自定义内部布局。

### 1.1 诊断房间

| 房间 | 功能 | 员工 | 必须设备 | 推荐数量/规模 |
|------|------|------|------|:--:|
| **GP全科(GP's Office)** | 首诊+确诊确认 | 医生 | 档案柜(File Cabinet×多个) | **最多**(5-8间) |
| 心脏病(Cardiology) | 中期诊断 | 护士 | 跑步机(Heart Racer)+档案柜 | 2-3间 |
| 体液分析(Fluid Analysis) | 中后期诊断 | 护士 | 体液分析仪 | 1-2间 |
| X光(X-Ray) | 中期诊断(最快22秒) | 医生 | X光机 | **优先**2-3间 |
| 核磁共振(MEGA Scan) | 高级诊断 | 医生(放射学) | 核磁共振扫描仪 | 1间(大) |
| DNA实验室(DNA Lab) | 遗传病诊断+治疗 | 医生(遗传学) | DNA分析仪 | 1-2间(兼治疗) |
| 病房(Ward) | 住院诊断+治疗 | **1-5名护士** | 病床×多+屏风 | 1-2间(大) |
| 精神病(Psychiatry) | 精神疾病诊断+治疗 | 医生(精神病学) | 沙发 | 1-2间 |

### 1.2 治疗房间

| 房间 | 治疗的疾病 | 员工 | 关键设备 |
|------|------|------|------|
| **药房(Pharmacy)** | 多种基础疾病 | 护士 | 药剂混合器(Drug Mixer) |
| **注射室(Injection Room)** | 情绪中毒/石肩症 | 护士 | 注射椅 |
| **手术室(Surgery)** | 多重重症 | 医生(外科学)+护士 | 手术台(最高收益!) |
| **De-Lux Clinic** | 头灯症 | 医生 | 去灯泡装置 |
| **小丑诊所(Clown Clinic)** | 小丑症 | 护士 | 小丑治疗椅 |
| **颅疗室(Head Office)** | 龟头症 | 医生 | 龟头治疗仪 |
| **染色体治疗(Chromatherapy)** | 灰度症 | 护士 | 彩色灯 |
| **分辨率实验室(Resolution Lab)** | 8位元症 | 医生 | 分辨率治疗仪 |
| **Pans Lab** | 锅盔症 | 护士 | 平底锅祛除器 |
| **骨折病房(Fracture Ward)** | 骨折类 | 护士 | 牵引床+石膏机 |

### 1.3 辅助/公用房间

| 房间 | 功能 | 员工 | 重要程度 |
|------|------|------|:--:|
| **前台(Reception)** | 病人登记→排队入口 | 助理(Assistant) | ★★★★★ |
| **全科(GP's Office)** | 见诊断 | (医生) | ★★★★★ |
| **员工休息室(Staff Room)** | 员工恢复体力/幸福感 | 无 | ★★★★ |
| **厕所(Toilet)** | 病人+员工如厕 | 无 | ★★★★ |
| **培训室(Training Room)** | 员工培训资格 | 外部讲师(自动) | ★★★★ |
| **研究室(Research Room)** | 研发升级设备/解锁新房间 | 医生(研究学) | ★★★ |
| **营销室(Marketing Room)** | 招募特定疾病病人 | 助理 | ★★(按需) |
| **咖啡厅(Café)** | 病人饮食 | 助理 | ★★(后期) |

## 二、房间声望/吸引力(Prestige)系统

每个房间有**声望等级(1-5级)**——受房间内装饰物品数量和品质决定。

| 声望等级 | 效果 |
|:--:|------|
| Lv1 | 基础——病人/员工无额外满意度 |
| Lv2 | 小幅满意提升 |
| Lv3 | 中幅满意提升——解锁部分目标 |
| Lv4 | 大幅满意提升——回复加快 |
| **Lv5** | **最高满意加成+最快回复**——多个三星任务目标 |

### 快速提升声望

| 方法 | 消耗 | 声望增幅 |
|------|------|:--:|
| 挂画/海报 | 低(100-500G) | 少量 |
| 植物/盆栽 | 低-中 | 少量 |
| 地毯 | 中 | 中量 |
| 奖杯/证书 | 中(需解锁) | 中-大量 |
| 高级墙纸/地板 | 中 | 中量 |
| **金马桶** | 高 | **巨量声望** |

## 三、空间布局的黄金法则

### 3.1 分区原则

```
┌─────────────┐   ┌──────────────────┐   ┌─────────────┐
│  前台+全科区 │   │    诊断区         │   │   治疗区      │
│  Reception  │→→→│ Cardiology       │→→→│ Pharmacy    │
│  + GP×5-8   │   │ X-Ray            │   │ Surgery     │
│  (入口附近)  │   │ Fluid Analysis   │   │ Injection   │
│              │   │ MEGA Scan        │   │ 各专科诊所   │
└─────────────┘   └──────────────────┘   └─────────────┘
```

**布局核心原则**:
1. **GP全科紧邻前台**——病人登记后第一站就是GP
2. **诊断室环绕GP**——病人从GP出来后去诊断室再回来(除非开启"快速治疗判定")
3. **治疗室可稍远但紧凑**——病人确诊后去治疗室是单程的(不返回)
4. **厕所/休息室/贩卖机均匀分布**——不要让病人和员工走太远

### 3.2 分散还是集中的取舍

| 策略 | 优点 | 缺点 | 适用 |
|------|------|------|------|
| **集中式**(一栋大建筑) | 病人步行距离短、暖气/空调统一 | 空间紧张→房间不够 | 小地图 |
| **分散式**(多栋建筑) | 空间充足→可分区 | 病人跨楼步行时间长 | 大地图(≥3栋) |
| **混合式** | 兼顾 | 需要规划 | 大多地图 |

### 3.3 不要放长椅!(特殊建议)

> 长椅(Benches)会让排队病人**坐下**——坐下后起身再走进诊室的时间反而延长了总的等待时间。如果必须放长椅，必须保证**每个排队的病人都有对应座位**，否则坐下的病人占位不动。

## 四、与开罗空间系统的对比

| 维度 | 双点医院 | 温泉物语 | 游戏开发物语 |
|------|:--:|:--:|:--:|
| 空间模型 | **房间制**(自定义内部) | 网格制(建筑占1格) | 纯工位 |
| 布局深度 | ★★★★(房间内+房间间) | ★★★★★(2格辐射) | ★ |
| 核心空间约束 | 地皮面积+房间大小 | 2格辐射范围 | 工位数量 |
| 装饰系统 | 有(声望等级) | 有(景观风景值) | 无 |
