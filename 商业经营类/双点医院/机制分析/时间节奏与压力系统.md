# 时间节奏与压力系统 — 双点医院

> 游戏版本: Two Point Hospital Steam版 v2018.8.29 (基础版)

## 一、时间结构

| 单位 | 内容 |
|------|------|
| **实时制** | 游戏时间连续流逝(非回合制)——可暂停/快进 |
| **1天** | 病人到达/进出诊室/治愈或死亡。日夜循环 |
| **1月** | 约30天。**月末结算**——收入/支出/利润报告 |
| **1年** | 12月。**年度评奖**(最佳医院/最佳员工) |
| **游戏全程** | 15个医院地图(基础版)+多DLC→每个地图独立三星成就 |

## 二、压力机制

### 2.1 运营压力

| 压力源 | 强度 | 触发条件 | 后果 |
|------|:--:|------|------|
| **GP全科排队暴增** | ★★★★★ | 诊断室不够/GP技能低 | 病人从门口排到厕所→满意度崩溃→退款→死亡 |
| **急诊(Emergency)** | ★★★★ | 限时到达的VIP病人 | 跳过诊断直接治疗→有时间限制→治愈=高额奖金 |
| **Epidemic(疫情)** | ★★★★★ | 传染性疾病暴发 | 必须在时限内完成全员疫苗接种——否则声誉暴跌 |
| **机器故障** | ★★★ | 维护不足→机器损坏 | 相关诊室停工→诊断链断裂 |
| **人员离职** | ★★★ | 幸福度低→威胁辞职 | 关键岗位空缺→病人堆积 |
| 竞争对手CEO破坏 | ★★★ | 随机 | 设备被毁/马桶堵塞/士气↓ |

### 2.2 财务压力

| 压力源 | 强度 | 说明 |
|------|:--:|------|
| **月薪支出** | ★★★★(前期) | 前期收入不稳时薪资是最大开支→用"我很便宜"特质对冲 |
| 贷款利息 | ★★★ | 高利率→不要轻易贷款 |
| 过度扩张 | ★★★★★ | 建了不用的诊室=维护费+护士/医生闲置→双倍浪费 |

### 2.3 节奏压力

| 压力类型 | 说明 |
|------|------|
| **病人潮汐** | 不像温泉物语的黄金周那样有明确峰值——但声誉高时病人会持续涌入 |
| **急诊打断** | 正在调布局时突然来急诊→需要暂停布局优先处理 |
| **多地图并行** | 后期可同时在多个医院地图间切换→相当于管理多个连锁医院 |

## 三、关键事件日历

| 频率 | 事件 | 策略 |
|:--:|------|------|
| **每月末** | 财务结算 | 检查利润+退款率 |
| **随机** | 急诊(Emergency) | 暂停一切→优先处理→加价100%←赚大钱 |
| **随机** | 疫情(Epidemic) | 暂停一切→全员疫苗→失败=声誉暴跌 |
| **随机** | 竞争对手CEO来访 | 勤杂工警戒 |
| **每年末** | 年度评奖 | 影响声誉 |
| **地图内** | 三星成就目标 | 完成即可去下一个地图(非强制) |

## 四、难度节奏设计

每个医院地图有一个清晰的三星节奏：

```
进入新地图
    ↓
★ 目标(基本): 建基础设施→治愈N个病人→达成声誉
    ↓
★★ 目标(中级): 培训员工资格→升级机器→扩大规模
    ↓
★★★ 目标(终极): 最高治愈率+高利润+高声誉+特定条件
    ↓
完成→去下一个地图(可选是否继续经营该医院)
```

## 五、与开罗系列时间系统的核心差异

| 维度 | 双点医院 | 开罗系列(所有) |
|------|:--:|:--:|
| 时序 | **实时制(可暂停可加速)** | 回合/周制 |
| 周期 | 无强制结束年月 | **有固定结束**(15年/21年) |
| 压力触发 | **随机事件**(急诊/疫情/CEO) | 周期性确定事件 |
| 终局 | 三星全达成后自由选择 | 年终结算(有限制) |

> 实时制让双点医院的压力感比开罗系列更"即时"——你不能在面临急诊时"再想一周"。
