## 一、"产品"即医疗服务

双点医院的"产品"是**从诊断到治愈的完整医疗服务流程**。不是卖一件东西，而是一条服务链：

```
GP首诊 → 专科诊断 → GP确诊 → 专科治疗 → 治愈
```

每个环节的品质由三种因素决定：
- **员工品质**：医生/护士的技能等级+相关资格
- **设备品质**：机器等级(研究升级)
- **流程效率**：诊断链环节数量+等待时间

## 二、机器升级系统——研究驱动

### 2.1 研究室(Research Room)

| 维度 | 数据 |
|------|------|
| 所需员工 | 医生(需**研究学Research**资格) |
| 研究项目 | 机器升级、新房间/新设备解锁、现金转换、营销活动解锁 |
| 研究时间 | 随项目复杂度和研究员技能变化 |
| 多研究室 | 可并行多个研究项目 |

### 2.2 机器升级的效果

| 机器等级 | 治疗/诊断能力加成 | 维护需求 |
|:--:|:--:|:--:|
| Lv1(基础) | 100% | 基准 |
| Lv2 | +20% | 适中 |
| **Lv3** | **+50%** | 更高 |

> Lv3机器 + 2个治疗技能(Lv3员工) = 达到治疗能力上限(100%)——再多升级和治疗技能都无效(溢出)。

---

## 三、治愈率公式(约)

```
治愈率 = 机器等级加成 × 员工技能(资格+等级) × 诊断确定性
```

| 因子 | 对治愈率的贡献 | 优化方式 |
|------|:--:|------|
| 机器等级 | Lv1=1.0 / Lv2=1.2 / Lv3=1.5 | 研究室研究升级 |
| 员工技能 | 基础+资格等级加成 | 培训室培训 |
| 诊断确定性 | 90%+ → 几乎无惩罚 | 充足诊断室 |

## 四、服务品质优化策略

### 4.1 全科(GP)——最重要的环节

全科是诊断链的**唯一入口和确认出口**。如果GP品质差，整个流程都会堵塞：

| 优化方式 | 效果 |
|------|------|
| 雇佣**全科资格(GP Qualification)**的医生 | GP诊断能力+ |
| GP诊室放多个**档案柜(File Cabinet)** | 每个+1%诊断能力 |
| 培训GP医生(最多5级全科资格) | 最佳成本方案(培训>高薪外聘) |

### 4.2 治疗端——最容易优化的环节

治疗只需要两个因素达标：
- Lv3机器(+50%)
- 员工Lv3 + 2个对应治疗技能

> 治疗端比诊断端容易优化——因为治疗是一对一的(确诊疾病→对应治疗室)，而诊断是一对多的(GP→多种可能疾病→需要多种诊断能力)。

### 4.3 快速通道策略

| 设置 | 位置 | 效果 |
|------|------|------|
| **加快治疗判定流程** | 概览→政策→勾选 | 病人诊断确定性达标后**直接去治疗**，不返回GP——大幅减少GP排队 |

## 五、研究→现金转化

研究室可以将研究项目转化为一次性现金收入。这是中后期的重要收入补充——让研究员在研究新型机器/升级的空闲周期中"印钱"。
