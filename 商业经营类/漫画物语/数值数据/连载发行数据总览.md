## 一、连载流程概述

漫画物语的核心玩法循环：

```
选题(主题×舞台) → 投入知识点 → 绘制(消耗体力) → 完稿 → 连载/投稿/参赛
```

连载是持续性的——选定一家出版社后，每周/每月需要交付一话，直到达到合同约定的话数或被迫腰斩。

## 二、特殊话组合系统

特殊话组合是漫画物语最独特的机制之一。在连载过程中，如果连续数话使用特定类型的主题组合，会触发特殊效果。

### 2.1 正面序列（金色序列）

连续使用特定顺序的主题组合可获得人气加成。

| 序列类型 | 话序列 | 人气加成 | 特殊效果 |
|:------|:------|:--:|------|
| 跃动连锁 | 跃动→连接→激烈→调和 | 大幅+↑↑↑ | 金色触发，读者评价"神展开！" |
| 热血连锁 | 热血→激烈→跃动 | 大幅+↑↑ | 战斗高潮效果 |
| 情感连锁 | 浪漫→治愈→日常 | 中幅+↑↑ | 情感转折加分 |
| 紧张连锁 | 悬疑→恐怖→黑暗 | 中幅+↑↑ | 悬疑气氛加成 |
| 成长连锁 | 日常→青春→热血 | 中幅+↑ | 成长故事加分 |
| SF连锁 | SF→冒险→未来 | 中幅+↑ | 科幻展开加分 |
| 和风连锁 | 和风→古代→自然 | 小幅+↑ | 传统叙事加分 |

> **注**: 序列名称为社区翻译，实际游戏中可能不同。序列触发条件为"连续话数"而非"必须连续天"——只要连载中按顺序使用这些主题即可。

### 2.2 负面序列（惩罚序列）

连续使用相同或过于相似的主题/舞台组合会触发负面效果。

| 序列类型 | 话序列 | 人气惩罚 | 触发条件 |
|:------|:------|:--:|------|
| 单一重复 | 同一主题连续3话以上 | 小幅-↓↓ | 连续3话使用完全相同主题 |
| 审美疲劳 | 同类型主题连续4话 | 中幅-↓↓↓ | 连续使用相近主题（如日常→日常→日常→日常） |
| 格式塔疲劳 | 相同主题×舞台连续2话 | 大幅-↓↓↓↓ | 连续使用完全相同的主题×舞台组合 |

### 2.3 特殊话策略分析

| 策略 | 说明 | 推荐度 |
|:------|------|:--:|
| 正面序列规划 | 提前计划好连载期间的序列，最大化人气加成 | ★★★★★ |
| 避免负面重复 | 确保不连续3话使用同一主题 | ★★★★★ |
| 序列实验 | 尝试不同顺序的正面序列，探索隐藏combo | ★★★★ |
| 序列打断 | 如果需要用同一主题，中间插入1话其他主题打断计数 | ★★★★★ |

### 3.1 单行本发行规则

## 四、休刊与腰斩系统

### 4.1 休刊（休载）

休刊允许玩家暂停连载1周（跳过一期），但会带来人气下降和出版社好感度降低。

| 休刊次数 | 人气下降 | 出版社好感度下降 | 备注 |
|:--:|:--:|:--:|------|
| 第1次休刊 | 小幅-10% | 小幅下降 | 可被理解 |
| 第2次休刊 | 中幅-20% | 中幅下降 | 读者开始不满 |
| 第3次休刊 | 大幅-35% | 大幅下降 | 出版社警告 |
| 第4次+休刊 | 极大-50%+ | 急剧下降 | 可能触发腰斩 |

### 4.2 腰斩

腰斩是连载被出版社强制终止。腰斩的后果极其严重。

| 腰斩后果 | 详细影响 |
|:------|------|
| 人气急剧下降 | 当前漫画人气降为0 |
| 出版社好感度崩溃 | 与该出版社的关系回到最低点 |
| 收入损失 | 剩余话数的稿费和单行本收入全部丢失 |
| 粉丝流失 | 大量粉丝流失 |
| 成就阻塞 | 部分成就需要完整连载才能获得 |

### 4.3 腰斩触发条件

| 条件 | 触发概率 |
|:------|:--:|
| 连续休刊3次以上 | 高 |
| 人气连续多期排50名以下 | 中 |
| 出版社好感度极低 | 中 |
| 截稿连续延期 | 高 |

## 六、知识点递增机制

这是漫画物语最核心的连载机制。

### 6.1 递增规则

```
第N话知识点投入 = 初始投入 + (N-1)
```

即：每多画一话，自动获得+1知识点投入到当前话。

### 6.2 递增效果

| 话数 | 额外知识点 | 品质提升 | 人气趋势 |
|:--:|:--:|:--:|:--:|
| 第1话 | 0 | 基准 | — |
| 第5话 | +4 | 中幅 | 缓慢上升 |
| 第10话 | +9 | 大幅 | 持续上升 |
| 第20话 | +19 | 大幅 | 高人气 |
| 第35话 | +34 | 极大 | 巅峰人气 |

### 6.3 知识点递增的设计意义

- **保证下限**：只要持续创作，品质自动提升，人气永不自然下降。
- **鼓励长连载**：越长连载，后期话数品质越高，人气也会越高。
- **降低焦虑**：玩家不需要担心"江郎才尽"——系统保证了稳定进步。
- **隐藏的教学**：告诉玩家"勤奋比天赋重要"——这是开罗游戏一贯的正能量设计哲学。
