## 一、10级漫画家等级

漫画家等级是衡量玩家成就的核心指标。从"涂鸦新手"到"漫画之神"，每级解锁新的功能和内容。

### 1.1 等级全览

| 等级 | 称号 | 累计销量门槛(约) | 解锁内容 |
|:--:|------|:--:|------|
| Lv.1 | 涂鸦新手 | 0 | 初始，可投稿月刊漫画馆 |
| Lv.2 | 漫画初学者 | 10,000 | 解锁商店购买器材 |
| Lv.3 | 新人漫画家 | 50,000 | 解锁周刊少年志，雇佣Lv.2助手 |
| Lv.4 | 职业漫画家 | 200,000 | 解锁第3-4次扩建 |
| Lv.5 | 人气漫画家 | 500,000 | 解锁青年漫画志，雇佣Lv.5助手 |
| Lv.6 | 畅销漫画家 | 1,000,000 | 解锁第5次扩建 |
| Lv.7 | 明星漫画家 | 2,000,000 | 解锁大型漫画志，Lv.7助手 |
| Lv.8 | 传说漫画家 | 5,000,000 | 解锁第6次扩建（助手专用房） |
| Lv.9 | 殿堂级漫画家 | 10,000,000 | 解锁业界顶级刊，雇佣Lv.9助手 |
| Lv.10 | **漫画之神** | **100,000,000** (1亿) | 终极称号，最强助手解锁 |

### 1.2 销量门槛分析

| 等级区间 | 销量跨度 | 难度评价 | 建议完成时间 |
|:--:|:--:|:--:|:--:|
| Lv.1→Lv.4 | 0→200,000 | 简单 | 第1年 |
| Lv.4→Lv.6 | 200K→1,000K | 中等 | 第2年 |
| Lv.6→Lv.8 | 1M→5,000K | 困难 | 第3-4年 |
| Lv.8→Lv.9 | 5M→10,000K | 很难 | 第4-5年 |
| Lv.9→Lv.10 | 10M→100,000K | **极难** | 5年全程+多周目继承 |

> **难度说明**: Lv.10（漫画之神）需要1亿累计销量，单周目几乎不可能达成。需要通过多周目继承策略，在3-5周目后才有望冲击。

## 四、多周目继承系统

漫画物语的一个核心特色是多周目继承——通关后选择"重新开始"可继承部分数据，让下一周目起点更高。

### 4.1 继承项目

| 继承项目 | 继承比例 | 重要性 |
|:------|:--:|:--:|
| **器材等级** | 100% | ★★★★★ 核心继承项 |
| **已解锁主题** | 100% | ★★★★★ 核心继承项 |
| **已解锁舞台** | 100% | ★★★★★ 核心继承项 |
| **已解锁助手** | 100% | ★★★★★ 核心继承项 |
| **助手雇佣次数** | 100% | ★★★★ (薪资递增的双刃剑) |
| **金钱** | 部分(约50%) | ★★★★ |
| **器材库存** | 100% | ★★★ |
| **漫画家等级** | 重置为Lv.1 | 需要重新升级 |
| **四属性值** | 重置 | 需要重新培养 |
| **出版社好感度** | 重置 | 需要重新建立 |
| **连载数据** | 不继承 | 每周目重新开始 |

### 4.2 多周目策略

| 周目 | 目标 | 策略重点 |
|:--:|:------|:------|
| **1周目** | 熟悉系统、积累器材和主题 | 尽可能解锁更多主题/舞台/器材 |
| **2周目** | 冲击更高销量和连载 | 利用继承的高等级器材快速创作 |
| **3周目** | 挑战高难度成就 | 完善助手配置，冲击长期连载 |
| **4周目+** | 极限冲刺1亿销量 | 全力冲击漫画之神 |

### 4.3 继承决策建议

| 需要积累的 | 不需要在意的 | 双刃剑 |
|:------|:------|:------|
| 器材尽量升满级 | 属性能量力而行 | 助手雇佣次数（省钱vs高起点） |
| 解锁尽可能多的主题/舞台 | 单次连载销量 | — |
| 金钱通关前多攒一些 | — | — |

## 五、通关评价

游戏在5年（60个月）结束后进行结算。通关评价基于以下指标：

| 评价指标 | 权重 | 说明 |
|:------|:--:|------|
| 累计销量 | 高 | 最主要的评价依据 |
| 连载完成数量 | 中 | 体现生产力 |
| 获奖次数 | 中 | 体现作品质量 |
| 粉丝数 | 低 | 人气佐证 |
| 总稿费收入 | 低 | 经济指标 |
