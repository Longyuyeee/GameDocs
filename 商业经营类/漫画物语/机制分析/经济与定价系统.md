# 经济与定价系统 — 漫画物语

> 游戏版本: The Manga Works (PC/Steam, Kairosoft, 2022)
> 数据来源: [Kairosoft Fandom Wiki](https://kairosoft.fandom.com/wiki/The_Manga_Works), [Steam商店](https://store.steampowered.com/app/1921770/), [萌娘百科](https://zh.moegirl.org.cn), [百度百科](https://baike.baidu.com)

---

## 一、货币体系

游戏使用单一货币**金钱(G)**，无内购、无代币、无氪金系统。收入结构围绕漫画家职业生涯的四个阶段展开：短篇投稿 → 连载出道 → 单行本出版 → 漫画之神。

### 收入来源

| 收入渠道 | 触发条件 | 计算方式 | 占收入比重 |
|------|------|------|:--:|
| **稿费(单次投稿)** | 完成短篇漫画并向出版社投稿 | 基础稿费 × 出版社倍率 × 篇幅长度 | 初期80% → 中期20% |
| **连载稿费(每话)** | 连载获批后每周/每月产出新话 | 基础话费 × 出版社等级 × 人气倍率 | 中期50% → 后期30% |
| **单行本版税** | 连载每满7话自动出版1卷 | 累计销量 × 版税率 × 卷数 | 中期25% → 后期50%+ |
| **参赛奖金** | 投稿至漫画比赛并获得名次 | 固定奖金(随比赛等级递增) | 初期<10%(波动) |
| **短篇委托** | 接受出版社的短篇委托任务 | 固定报酬(高于普通投稿稿费) | 中期间歇收入 |

> **注**: 游戏没有"定价"环节——玩家不能设定漫画售价。稿费和版税由游戏内置公式计算，玩家的控制点在于**提升漫画品质**（通过更好的主题×舞台组合、更多的知识点投入）和**维持连载人气**（通过知识点递增策略）。

## 二、出版社等级差异与收入倍率

出版社不仅是"发表渠道"，其等级直接决定了每话的稿费基准和单行本卷数上限——选对出版社是收入最大化的第一步。

| 出版社 | 最大连载话数 | 最大卷数 | 稿费倍率 | 解锁条件 |
|------|:--:|:--:|:--:|------|
| **第一出版社** | 18话 | 2卷 | 1.0x | 初始可用 |
| **第二出版社** | 30话 | 4卷 | 1.2x | Lv3后解锁 |
| **热血出版社** | — | — | 1.5x | Lv5后解锁 |
| **第四出版社** | — | — | 2.0x | Lv6(300万销量) |
| **第五出版社** | — | — | 2.5x | Lv7(1000万销量) |
| **开罗出版社** | 50话 | 7卷 | 3.0x | Lv10(1亿销量) |

**卷数 = 总版税的核心杠杆**。开罗出版社的7卷上限是第一出版社2卷的3.5倍，配合3.0x稿费倍率，后期一话的稿费收入可达初期的10倍以上。

## 三、成本结构

| 成本类型 | 固定/可变 | 金额范围 | 策略影响 |
|------|:--:|:--:|------|
| **取材费用(美术馆)** | 单次 | $X/次(初期最便宜) | 初期首选取材地，每次花费较少但知识点收益稳定 |
| **取材费用(咖啡厅)** | 单次 | $X/次 | 中等花费，知识点收益中等 |
| **取材费用(图书馆)** | 单次 | $X/次 | 花费较高，知识点收益高 |
| **取材费用(卡拉OK)** | 单次 | $X/次 | 高花费，知识点收益高+特殊事件 |
| **助手月薪(基础)** | 月/人 | 初始约$2,000/月 | 见员工管理系统详细分析 |
| **助手薪资递增** | 每续雇一次 | 薪资递增约50%/次 | 连续雇佣同一助手→薪资指数级增长 |
| **器材购买** | 一次性 | $500-$5,000/件 | 画板/书桌/陈列柜等各加不同属性 |
| **房间扩建** | 一次性 | $10,000-$100,000 | 增加可用格子+解锁仓库+助手房 |
| **短篇委托成本** | 可变 | 按篇幅和复杂度变动 | 委托回报通常高于投入 |

### 财务压力的关键节点

```
初期(第1年)          中期(第2-3年)          后期(第4-5年)
├─ 取材费用为主       ├─ 助手薪资膨胀期       ├─ 器材升级+扩建
├─ 无助手             ├─ 需稳定连载收入       ├─ 多出版社运作
├─ 稿费≈够用          ├─ 薪资增速>收入增速    ├─ 收入>支出(健康期)
└─ 最脆弱阶段         └─ 最危险阶段           └─ 冲刺漫画之神
```

## 四、金钱Sinks(大额消费出口)

| 消费出口 | 金额范围 | 设计意图 |
|------|:--:|------|
| **房间扩建(第一次)** | 约$10,000 | 初期门槛——迫使玩家在"存钱扩建"和"购买器材"之间抉择 |
| **房间扩建(后续)** | $30,000-$100,000 | 中期压力——需要连载稳定后才能负担 |
| **高级器材(奖杯/陈列柜)** | $3,000-$5,000 | 注视升级后的收益远高于初始价格 |
| **多助手薪资** | 月$6,000+ | 2-3个助手的叠加薪资可压垮不健康的现金流 |
| **多周目继承** | 一次性消费 | 继承优质器材需要预留充足资金 |

> **设计特点**: 漫画物语的经济Sinks不像游戏开发物语那样有"办公室升级"的巨型门槛，而是**阶梯式递增**——助手薪资每续聘一次涨50%，器材价格适中但注视升级需要时间。整体经济曲线更平滑，但"薪资递增"的指数性质是一个隐藏的财务陷阱：玩家在初期没意识到一名$2,000/月的助手，5次续聘后会变成$15,000+/月。

## 五、经济与经营循环

```
短篇投稿/委托(获取启动资金+解锁主题/舞台)
    ↓
连载出道(稳定话费收入→扩大再生产)
    ↓
连载维持(知识点递增→人气第一→单行本销量增长)
    ↓
单行本出版(每7话1卷→版税滚雪球)
    ↓
器材升级+房间扩建(生产力提升)
    ↓
多出版社运作(更高稿费倍率+更多连载)
    ↓
漫画之神(1亿销量+开罗出版社)
```

## 六、设计特点分析

1. **"薪资递增"作为自然通胀** — 开罗不设传统意义上的"物价通胀"，但用助手薪资每次续聘递增50%来制造支出压力。这不是数值错误，而是**刻意的经济压缩机制**——逼迫玩家不断提升收入水平，如果连载中断或人气下跌，薪资负担会迅速反噬。

2. **无定价权+品质导向** — 与游戏开发物语一样，玩家不能定价。稿费和版税由漫画品质和出版社等级间接决定。这种设计将注意力集中在"如何画出更好的漫画"而不是"如何定更高的价"——对于轻量级模拟游戏是正确取舍。

3. **卷数经济** — 单行本卷数是收入的核心乘数，而卷数又受限于出版社的连载话数上限。这种设计创造了一条清晰的升级路径：换更好的出版社 → 连载更多话 → 出版更多卷 → 更高收入。这是一个正反馈循环，驱动玩家不断追求升级到更高级出版社。

4. **多周目经济的重置与继承** — 游戏结束时玩家可选择继承器材和助手属性开始新一周目。继承不继承金钱，但器材是"固定资产"——这迫使玩家在一周目末期合理安排资金购买高价值器材。

## 七、跨游戏对比：与游戏开发物语的经济系统

| 对比维度 | 漫画物语 | 游戏开发物语 |
|------|------|------|
| **定价权** | 无(稿费由出版社公式决定) | 无(售价由评分间接决定) |
| **核心收入** | 连载稿费+单行本版税 | 游戏销售收入+合同外包 |
| **大额消费** | 房间扩建+器材购买 | 办公室升级+主机研发 |
| **成本通胀** | 助手薪资递增50%/次 | 员工年薪固定(无通胀) |
| **收入周期** | 每话/每卷(连载周期) | 每款游戏(开发周期) |
| **资金Sinks密度** | 阶梯式(渐进压力) | 阶段门式(巨型门槛) |
| **经济陷阱** | 薪资指数增长 | 精英签约金+大额研发失败 |
| **多周目继承** | 器材+助手属性 | 无(完全重开) |

> **核心差异**: 游戏开发物语的经济压力来自"阶段门"(积累足够资金跨过办公室/主机门槛)，而漫画物语的经济压力来自"指数成本"(助手薪资的50%递增率)。前者是"攒够钱就自由了"的间断压力，后者是"永远在追赶薪资曲线"的持续压力——这更贴合现实漫画家的生存状态：连载不能停，停了就付不起助手工资。
