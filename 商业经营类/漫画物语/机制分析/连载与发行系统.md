# 连载与发行系统 — 漫画物语

> 游戏版本: The Manga Works (PC/Steam, Kairosoft, 2022)
> 数据来源: [Kairosoft Fandom Wiki](https://kairosoft.fandom.com/wiki/The_Manga_Works), [Steam商店](https://store.steampowered.com/app/1921770/), [萌娘百科](https://zh.moegirl.org.cn), [百度百科](https://baike.baidu.com)

---

> **本篇是漫画物语品类的核心差异文档**。连载与发行系统是漫画物语区别于游戏开发物语等开罗游戏的核心特色——它不是"开发→发布→收入"的简单循环，而是引入了"连载周期管理"、"知识点递增"、"特殊话连锁"、"单行本出版"等一系列独特机制。

---

## 一、连载启动流程

```
完成漫画 → 投稿(OK) → 出版社批准 → 开始连载
               ↓ (NG)
         好感度降低/重投
```

| 步骤 | 详细说明 |
|------|------|
| **完成第1话** | 使用主题×舞台+知识点投入完成第1话 |
| **选择出版社** | 从已解锁的出版社中选择(受漫画家等级限制) |
| **自荐投稿** | 只有"自荐"可以开启连载(存稿和参赛不能) |
| **出版社审批** | 出版社根据当前好感度和漫画品质决定是否批准 |
| **连载开始** | 批准后以月刊/周刊节奏产出后续话数 |

### 连载的节奏

| 节奏类型 | 每话耗时 | 适合阶段 | 策略建议 |
|:--:|:--:|:--:|------|
| **周刊** | 较短 | 后期(工作量/属性高) | 产出快但知识点消耗大 |
| **月刊** | 较长 | 初期-中期 | 产出稳定，知识点消耗可控 |
| **具体时间** | 待补充 | — | 具体每话所需游戏内天数待确认 |

## 二、知识点递增——人气维持的"铁律"

这是漫画物语**最核心的连载规则**，也是最具特色的设计。

### 规则表述

```
连载第N话投入的知识点 ≥ 第(N-1)话投入的知识点 + 1
→ 人气排名绝对不会下降(甚至可能上升)

连载第N话投入的知识点 ≤ 第(N-1)话投入的知识点
→ 人气排名很可能下降
```

### 策略含义

| 知识点投入策略 | 人气趋势 | 可行性 |
|------|:--:|:--:|
| 每话递增1点 | 稳定#1(不下降) | 中期后可行(知识点来源充足) |
| 每话维持等量 | 可能下降 | 初期知识点有限时的妥协 |
| 每话递减 | 人气下降→粉丝流失 | 不推荐(除非特殊策略) |
| 大幅递增2-3点 | 人气上升+粉丝增长 | 知识点充裕时的最优选择 |

### 递增策略的资金基础

知识点获取需要外出取材(花费金钱)或多周目技能(消耗体力)。维持递增意味着玩家需要持续投入资源——这形成了一个"资源循环"：

```
连载收入 → 取材/技能 → 知识点 → 投入连载 → 维持#1 → 更高收入
```

## 三、特殊话连锁系统

这是连载期间的"Combo机制"——通过特定的话数类型组合触发额外效果。

### 特殊话类型

| 类型 | 效果 | 出现方式 |
|------|------|------|
| **跃动(Leap)** | 正面 | 待补充 |
| **连接(Connection)** | 中性 | 待补充 |
| **激烈(Intense)** | 正面 | 待补充 |
| **调和(Harmony)** | 正面 | 待补充 |
| **伏笔(Foreshadowing)** | 中性 | 待补充 |
| **寂静(Quiet)** | 中性 | 待补充 |
| **书归正传(Return to Basics)** | 中性 | 待补充 |

### 负面连锁——连续3次同一类型

| 连锁 | 触发条件 | 效果 |
|------|:--:|------|
| **懊悔(Regret)** | 连续3次"伏笔" | 人气下降？ |
| **寂寞(Loneliness)** | 连续3次"寂静" | 人气下降？ |
| — | 连续3次"书归正传" | 负面效果？ |
| — | 连续3次"连接" | 负面效果？ |

### 金色正面连锁——特定3话序列

| 序列 | 效果 |
|:--:|------|
| **跃动 → 连接 → 激烈 → 调和** | 大幅人气冲击，进入"金色"状态 |
| 其他序列 | 待补充 |

> **设计特点**: 特殊话连锁系统将连载从"重复产出"升级为"序列编排"——玩家不仅要考虑每话的知识点投入量，还要安排不同话数类型的顺序。这赋予了连载一种"节奏感"：有起有伏，有伏笔有高潮。负面连锁的存在要求玩家不能无脑重复同一类型——必须多样化话数安排，在"不同类型的组合"中寻找最优编排。

## 四、单行本系统

单行本(Tankobon)是漫画连载的"精华集结版"——每7话自动出版1卷。

### 出版规则

| 规则 | 说明 |
|------|------|
| **出版条件** | 连载话数达到7的倍数 |
| **自动出版** | 无需玩家操作，到话数自动出版 |
| **版税收入** | 按销量给玩家分账 |
| **销量公式** | 粉丝数 × 当前人气 × 品质系数(待补充) |

### 卷数的战略意义

| 卷数 | 出版社限制 | 累计版税 | 等级贡献 |
|:--:|:--:|:--:|:--:|
| 1-2卷 | 第一出版社的上限 | 低 | 少量 |
| 3-4卷 | 第二出版社的上限 | 中等 | 中等 |
| 5-6卷 | 热血出版社上限(？) | 高 | 高 |
| 7卷 | 开罗出版社上限 | 最高 | 1亿销量的核心基础 |

> **卷数 = 出版社的"最大长度"**。换出版社不只是换稿费倍率，更重要的是获得更多的卷数上限——7卷的累计收入远高于2卷。因此游戏的核心策略之一是：**尽快升级到可以连载更多话/卷的出版社**。

## 五、休刊与腰斩——连载的风险管理

| 风险事件 | 触发条件 | 后果 |
|------|------|------|
| **延期** | 未在截止日内完成当期话数 | 最多延期1天/话。人气轻微下降 |
| **休刊(第一次)** | 连续延期或主动选择休刊 | 人气下降 + 出版社好感度降低 + 不发售新卷 |
| **腰斩(强制终止)** | 2次休刊(或连续严重延期) | 连载强制终止 + 出版社好感度严重受损 |
| **出版社关系恶化** | 腰斩多次或频繁拒稿 | 可能无法在该出版社开设新连载 |

### 休刊的连锁反应

```
延期 → 休刊 → 无新话 → 人气下降 → 粉丝流失
              ↓
        出版社不悦 → 好感度降低
              ↓
        再次休刊 → 腰斩 → 关系严重受损
```

> **设计分析**: 休刊/腰斩系统是漫画物语区别于其他开罗游戏最明显的"压力设计"——游戏开发物语的合同失败只是损失报酬，没有"剧情中断"的后果。而漫画物语中连载中断意味着**故事断层**和**人设浪费**——玩家投资的每一话知识点、每一段剧情都可能因为腰斩而失去价值。这种风险让连载不只是"生产"，更是"维护"。

## 六、连载收益公式

```
连载收益 = 单话稿费(出版社等级 × 人气倍率)
         + 单行本版税(卷数 × 粉丝数 × 品质系数)
         + 特殊话触发奖励(如有)
```

| 变量 | 玩家控制方式 |
|------|------|
| **出版社等级** | 升级漫画家等级 + 维护出版社好感度 |
| **人气倍率** | 知识点递增策略 + 特殊话连锁 |
| **卷数** | 选择高连载上限的出版社 |
| **粉丝数** | 长期人气积累 |
| **品质系数** | 主题×舞台匹配度 + 属性加成 |

## 七、设计特点分析

1. **知识点递增 = "进步必须可见"** — 这是漫画物语中最具情感冲击力的设计规则。它传递了一个强烈的信号："读者能感受到你在进步"。与现实漫画产业中"新人必须每话进步才能生存"的逻辑高度吻合，但游戏化后变成了一个清晰的数值规则。

2. **特殊话连锁 = 连载的"节奏游戏"** — 特殊话系统将"每个话数"变成了一个策略决策节点：这画是伏笔还是高潮？玩家需要像编排一部真正的漫画那样思考剧情节奏。负面连锁(连续3次同类)的存在迫使玩家多样化，而金色连锁则奖励精心编排的结构。

3. **单行本 = "存量资产的累积"** — 单行本系统将连载从"即时收入"(每话稿费)扩展为"存量收入"(版税回流)。每多一卷，过去的劳动就产生一次新的回报。这是典型的"IP经济"模拟——连载期间的产出在单行本阶段持续变现。

4. **休刊/腰斩 = 无法存盘的进度** — 在开罗游戏中，休刊/腰斩是相对严厉的惩罚。它不像游戏开发物语那样可以"做不好就下一款"——连载中断意味着之前所有投入的画技、知识点、剧情编排都白费了。这种"进度不可逆"的设计增加了每话的心理权重。

## 八、跨游戏对比：与游戏开发物语的"发行"系统

| 对比维度 | 漫画物语 | 游戏开发物语 |
|------|------|------|
| **产品形式** | 连载漫画(系列内容) | 单款游戏(独立产品) |
| **发行流程** | 投稿→审批→连载 | 研发→发布(无审批) |
| **持续维护** | 每话产出+知识点递增 | 无需持续维护 |
| **收入模式** | 稿费+版税(持续收入) | 销售收入(一次性) |
| **"续作"机制** | 单行本新卷(同作品) | 续作链(新游戏) |
| **失败风险** | 腰斩(剧情中断) | 低评分(销量差) |
| **品质维护** | 知识点递增策略 | 提高各项属性 |
| **特殊事件** | 特殊话连锁(Combo) | 灵感迸发(Burst) |
| **内容周期** | 连载→完结(剧本式) | 开发→发布(项目式) |

> **核心差异**: 游戏开发物语的"产品"是一次性的（每款游戏独立），而漫画物语的"产品"是持续的（连载+单行本）。这意味着漫画物语需要**持续维护同一产品**的品质和人气——这在开罗游戏体系中是稀缺的设计。其他开罗游戏(拉面店/温泉物语)也有持续运营的元素，但"知识点递增"和"特殊话连锁"是漫画物语独有的连载深度策略。
