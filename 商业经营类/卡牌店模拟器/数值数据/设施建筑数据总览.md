## 四、设施布局建议

本游戏**无"相性加成"或"套装加成"机制**（与温泉物语/双点医院不同），纯自由布局。

### 推荐布局原则

| 原则 | 说明 |
|------|------|
| **游戏桌集中** | 靠近收银台放置，便于顾客快速找到并坐下，方便管理 |
| **除臭机入口放置** | 在店门口放置除臭机以第一时间处理臭气顾客，防止驱散其他顾客 |
| **高价值商品靠结账处** | 减少盗窃风险，增加曝光率 |
| **拆包机靠仓库** | 拆包机放在B区或靠近仓库，不影响顾客动线 |
| **畅销品靠入口** | Basic Card Pack等畅销品放门口吸引顾客 |
| **货架间隙合理** | 留出足够通道避免顾客拥堵，提升浏览效率 |
