## 一、全收入来源表

| 收入来源 | 类型 | 基础收入公式 | 加成因素 | 在总收入中的权重估算 |
|---------|:---:|:-----------:|:--------:|:----------------:|
| **货架零售** | 被动 | 商品定价 × 销售数量 | 定价%(市价±%)、市场波动、顾客钱包 | ~55% (核心收入) |
| **卡牌展示销售** | 被动 | 单卡定价(玩家自定) | 卡牌稀有度、边框等级、品相、市场价 | ~10% (中期后重要) |
| **游戏桌收费** | 被动+主动 | 每小时收费 × 使用时长 | 定价设定、游戏桌数量、顾客需求 | ~5% (稳定但不高) |
| **顾客卖卡差价** | 被动 | 市场价 - 收购价 | 红叹号顾客频率、稀有卡出现率 | ~15% (高利润) |
| **批量卡盒(Bulk Box)** | 主动(工作台) | 卡盒售价 - 材料成本 | 制作效率、库存管理 | ~10% (后期重要) |
| **赛事活动收入** | 主动(APP) | 门票/参与费 | 活动类型、参与人数 | ~5% (补充收入) |

## 三、定价弹性数据

|:-------:|:-------:|:-------:|:-----:|---------|
| 激进清仓 | 市价-20% | ~100% | 低 | 冲XP/快速销库存 |
| 平价策略 | 市价 | ~90% | 中 | 平衡型 |
| **推荐利润** | **市价+10%** | **~75%** | **高** | **日常最推荐** |

| 极高利润 | 市价+30% | ~30% | 极高 | 稀缺产品/单卡 |

## 四、进价成本公式

### 4.1 核心公式

| 项目 | 公式 | 说明 |
|:----|:----|------|
| **卡包进价** | `基础成本 × Random(1, 1.3)` | 每包独立随机，最低1倍，最高1.3倍 |
| **卡盒进价** | `卡包进价 × 8 × Random(1, 1.15)` | 卡盒=8包，额外乘1-1.15倍 |
| **进价下限** | `≥ 基础成本` | 随机结果不低于基础成本 |

### 4.2 实例计算

| 产品 | 基础成本($) | 进价随机因子 | 最终进价($) | 说明 |
|:----|:----------:|:----------:|:----------:|------|
| Basic Card Pack | 1.50 | ×1.15 | 1.73 | 最低档进价 |
| Basic Card Pack | 1.50 | ×1.30 | 1.95 | 最高档进价 |
| Basic Card Box | 12.00 | ×1.10(卡包)×1.10(盒) | 14.52 | 中等随机 |
| Legendary Card Pack | 4.95 | ×1.30 | 6.44 | 最高档进价 |
| Legendary Card Box | 38.50 | ×1.30(包)×1.15(盒) | 57.56 | 最高档进价 |

## 五、每日价格波动规则总结表

### 5.1 基本波动参数

| 参数 | 数值 | 说明 |
|:----|:---:|------|
| 每日波动触发概率 | **40%** | 每件商品每日独立判定 |
| 涨跌方向 | 50%/50% | 各一半概率 |
| 标准波动幅度(非卡牌) | ±0.08% ~ ±2% | 日用品/配件等 |
| 标准波动幅度(卡牌) | ±0.8% ~ ±2% | 卡牌类波动更大 |
| 价格硬上限 | 初始市价 **+200%** | 无论如何不会突破此上限 |
| 价格硬下限 | 初始市价 **-80%** | 无论如何不会跌破此下限 |
| 价格基准 | 初始市价(存档生成时) | 所有百分比基于初始市价，非当前市价 |

### 5.2 市场崩溃(Crash)机制

| 条件 | 触发规则 |
|:----|---------|
| 触发前提 | 当前价格 ≥ 初始市价 +25% |
| 触发概率 | 等于当前溢价百分比（如溢价50%则有50%概率触发） |
| 下跌幅度 | `Random(5%, 溢价百分比/2.1)` |
| 触发时段 | 每日开始，在常规波动判定之后 |

### 5.3 市场暴涨(Explosion)机制

| 条件 | 触发规则 |
|:----|---------|
| 触发前提 | 当前价格 ≤ 初始市价 -25% |
| 触发概率 | 等于当前折价百分比 |
| 上涨幅度 | 待补充(机制与Crash对称) |
| 触发时段 | 每日开始，在常规波动判定之后 |

### 5.4 赛事活动对价格的影响

| 参数 | 数值 |
|:----|:---:|
| 活动触发价格变化 | +0.2% ~ +5%(正面)/ -0.2% ~ -5%(负面) |
| 触发时机 | 活动日开始时，在常规波动之前 |
| 影响范围 | 特定类型卡牌 |

## 八、现金流管理

| 参数 | 数值 | 说明 |
|:----|:---:|------|
| **账单周期** | **7天** | 租金+电费+工资每7天自动扣除 |
| **首次账单** | 第7天营业结束时 | 游戏内首个账单日 |
| **破产条件** | 第7天无法支付账单 | 游戏结束 |
| **推荐现金储备(前期)** | ≥$500 | Lv1-15，覆盖首张账单+进货 |
| **推荐现金储备(中期)** | ≥$5,000 | Lv16-35，覆盖扩张+高额账单 |
| **推荐现金储备(后期)** | ≥$20,000 | Lv36-60，随时进货+扩张 |
| **终局现金储备** | ≥$50,000 | Lv60+，应对大额扩张+高日租金 |
| **账单自动扣除** | 资金不足时 | 会从收入中优先扣除 |
