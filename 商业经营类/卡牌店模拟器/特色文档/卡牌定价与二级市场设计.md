# 卡牌定价与二级市场设计 — 卡牌店模拟器

> 游戏版本: TCG Card Shop Simulator v0.60 (PC/Steam EA, 2024+)
> 数据来源: [TCG Card Shop Simulator Wiki (Fandom)](https://tcg-card-shop.fandom.com), [Steam定价市场指南 (LoddZee)](https://steamcommunity.com/sharedfiles/filedetails/?id=3342675597), [Reddit r/TCGCardShopSimulator](https://www.reddit.com/r/TCGCardShopSimulator/)

## 一、动态市场价 vs 固定价格

### 为什么不是固定价格？

如果所有卡牌都是固定价格，卡牌店模拟器会变成什么？
1. 玩家只需要找到"进价最低/售价最高"的卡包，反复进货即可
2. 收集卡牌只剩下"收集"意义，没有经济策略
3. 两周后玩家对经营失去兴趣

### 动态价格的三大功能

| 功能 | 市场价固定 | 市场价动态 | 设计目的 |
|:----|:---------|:----------|:--------|
| **投机空间** | 无 | 低价买入→高价卖出 | 创造"抄底"满足感 |
| **决策深度** | 一种正确策略 | 需要判断何时买/卖 | 防止最优解 |
| **沉浸感** | 无 | 模拟真实卡牌市场波动 | 增强经营感 |

### 动态价格的核心规则

| 规则项 | 数据 |
|:------|:----|
| 价格变动频率 | 每天结算后~40%概率波动 |
| 单次波动幅度 | ±0.08%~±2%（相对当前市价） |
| 波动方向 | 上涨/下跌随机，无长期趋势偏向 |
| 不同卡牌的相关性 | 各自独立波动 |
| 边框等级的影响 | 高稀有度边框波动幅度更大（风险更高，投机空间更大） |
| 特殊事件 | 活动举办时，受影响的边框价格整体偏移（+0.5%~5%） |

### 动态价格创造的行为模式

玩家面对动态价格时的典型行为链：

```
1. 查看今日市价 → 发现某种卡价格偏低
2. 决策：现在买入"抄底" vs 等待更低
3. 买入持有 → 期待第二天涨价
4. 次日查看 → 如果涨价了：卖出获利（成就感！）
                      如果跌了：继续持有（"总会涨回来的"）
```

这个行为链模拟了真实金融市场的"持有→期待→兑现/继续持有"循环。每次价格波动都在问玩家一个问题：**"现在是卖出的好时机吗？"**

## 二、活动操纵市场——你不是被动接受者

### 从价格接受者到价格操纵者

传统模拟经营游戏中，玩家是"价格接受者"——市场价由系统决定，玩家只能调整自己的售价。但卡牌店模拟器引入了一个关键机制：**通过举办活动主动影响特定卡牌的市价。**

| 玩家角色 | 传统经营游戏 | 卡牌店模拟器 |
|:--------|:-----------|:------------|
| 与市场的关系 | 被动适应 | **主动影响+被动接受** |
| 决策 | "价格涨了，我多屯点" | "我举办XX活动→价格涨→我出货" |
| 操纵手段 | 无 | 活动APP（6种+活动类型） |
| 成就感来源 | 适应市场 | 创造市场走势 |

### 操纵市场的策略深度

```
"我不再是被动接受市场的玩家——我在操纵市场。"

Step 1: 持有大量Silver边框卡（日常收购积累）
Step 2: 举办Silver边框活动（影响Silver市价+0.5%~2%）
Step 3: Silver卡价格上涨 → 卖出获利
Step 4: 赚到的钱再去收购下一批目标边框卡
Step 5: 循环
```

这引入了**自我实现预言**的玩法：活动之所以能影响市价，是因为设计师告诉玩家它能。但玩家的行为——在活动前囤货、活动中抛售——已经形成了真实市场中"内幕交易"的模拟。

### 活动与库存管理的交响

| 策略类型 | 操作 | 效果 |
|---------|------|------|
| **主动囤积** | 活动前3天收购目标边框卡 | 持仓成本最低 |
| **按需囤积** | 根据手头库存最多的边框选择活动 | 零额外成本 |
| **鉴定联动** | 将高评分卡留到相关活动期间出售 | 双重溢价（评级+活动） |
| **对冲策略** | 同时持有多种边框，办A活动时卖A，B活动时卖B | 分散风险 |

## 三、崩溃/暴涨机制

### 机制的数学规则

| 机制 | 触发条件 | 价格变化 | 持续时间 | 回归机制 |
|:----|:--------|:--------|:-------|:--------|
| **市场崩溃** | 单卡市价偏离基准价≥-25% | **反向暴涨**（价格强力反弹） | 1-3天 | 价格回归至接近基准价 |
| **市场暴涨** | 单卡市价偏离基准价≥+25% | **反向崩溃**（价格急跌） | 1-3天 | 价格回归至接近基准价 |

### 崩溃/暴涨的设计意图

| 设计意图 | 说明 |
|---------|------|
| **防止经济崩坏** | 如果价格无限下跌，卡牌会变得一文不值，整个经济循环断裂 |
| **防止通胀失控** | 如果价格无限上涨，玩家会变得"钱多到没处花"，失去经营动力 |
| **制造戏剧性** | 价格突然暴涨/崩溃是"好故事"——玩家会在社群中分享"我昨天$10买入的卡今天涨到$50了！" |
| **模拟真实市场** | 真实金融市场也有熔断机制和价格回归 |

### "回归"的不确定性

> 但崩溃/暴涨的"回归"也可能**反向**——崩溃后价格不是回归正常，而是暴涨；暴涨后不是回归，而是暴跌。

这意味着：
- 崩溃时抄底 = 可能赚大钱（回归暴涨）
- 暴涨时追高 = 可能亏大钱（回归暴跌）
- 但崩溃后继续跌/暴涨后继续涨也是可能的——直到触发极限

> 设计师引入了真实金融市场的核心特征：**不可预测性**。这阻止了"在崩溃时无脑买入"成为无敌策略。

### 崩溃/暴涨的玩家行为建议

| 市场状态 | 理性行为 | 高风险行为 |
|---------|---------|:----------:|
| 崩溃（-25%） | 谨慎买入，分批建仓 | 全仓抄底 |
| 暴跌中（-10%~-24%） | 观望，等待企稳 | 恐慌抛售 |
| 正常波动（±10%） | 正常经营，不必特殊应对 | 频繁交易浪费手续费 |
| 上涨中（+10%~+24%） | 分批卖出获利了结 | 贪婪持有等更高 |
| 暴涨（+25%） | 立即卖出套现 | 继续持有期待更高 |

## 四、顾客交易差价

### 交易机制

当顾客头顶出现**红色感叹号**时，表示他们有卡牌要卖给你或想与你交换。这个机制的核心是：

> **顾客的报价通常低于市场价格。** 玩家可以低价买入，然后按市场价卖出——赚取差价。

### "信息不对称"获利的模拟

| 角色 | 拥有的信息 | 行为 |
|:----|:----------|:----|
| **玩家** | 知道当前市场价（手机APP可查） | 低价收购→高价卖出 |
| **顾客** | 不知道当前市场价 | 报出低于市价的价格 |
| **差价** | = 玩家获利空间 | 10-40%的利润 |

这个机制模拟了真实世界中的**信息不对称**——卡牌店老板天天看价目表，普通顾客只有"我这张卡大概值多少"的模糊概念。店老板利用信息优势赚取差价，完全合乎商业逻辑，让玩家在道德上毫无负担地"占便宜"。

### 顾客交易ROI

| 交易类型 | 典型差价 | 单次交易利润（以$100市价卡为例） | 频率 |
|:-------:|:-------:|:-----------------------------:|:---:|
| 顾客卖卡（现金收购） | 市价的60-80% | $20-40 | 每天3-8次 |
| 顾客换卡（等价交换） | 无直接差价 | 增加收藏多样性 | 每天1-3次 |
| 顾客买卡（出价） | 市价的90-110% | 波动，通常略低 | 每天2-5次 |

## 五、定价策略（+10%/30%规则）

### 基础定价规则

| 商品类型 | 建议加价幅度 | 顾客反应 | 销量影响 |
|:--------|:----------:|:--------|:-------:|
| 架上商品（卡包/配件） | +10% | 顾客几乎感觉不到 | 销量几乎不受影响 |
| 单张卡牌（普通） | +10%~20% | 较易接受 | 轻微下降 |
| 单张卡牌（稀有） | +20%~30% | 收藏家愿意支付溢价 | 中等下降但利润更高 |
| 卡牌（Ghost/高稀有度） | +30%~50% | 目标客户明确，溢价空间大 | 销量显著降但单笔利润极高 |

### 为什么+10%几乎不影响销量？

| 解释 | 心理学原理 |
|:----|:----------|
| 卡包本身价格低（$2-5/包），+10%只有$0.2-0.5 | **绝对金额感知**——顾客关注的是"多少钱"而非"贵了多少" |
| 市场价在波动，顾客对"正确价格"没有固定锚点 | **锚定效应缺失**——没有参考价就不知道贵了 |
| 顾客对卡牌店的价格敏感度低于超市 | **品类特性**——爱好类消费的价格弹性低于日常消费 |

### 定价策略的最优解

通过社区测试和玩家经验总结，最有效的定价策略如下：

```
卡包类：基准价 + 10%（几乎无脑加，顾客不察觉）
配件类：基准价 + 10-15%（卡套/骰子等低价配件）
Battle Deck：基准价 + 10%（高价商品，加多影响销量）
普通单卡(Normal/1st)：基准价 + 10%（流动性最快）
稀有单卡(Silver+)：基准价 + 20-30%（高利润空间）
Ghost/高稀有度：基准价 + 30-50%（爱买不买，买就赚大）
```

## 六、整数定价与结账效率

### 一个微小的QoL设计，直接影响经营效率

在卡牌店模拟器中，如果商品定价带小数（例如$2.49），收银员在找零时会有明显的停顿动画。如果所有商品都是整数（$2、$3、$5），收银员的处理速度会显著提升。

| 定价方式 | 每单处理时间 | 每小时处理单数 | 每日总营收（估算） |
|:-------|:----------:|:------------:|:--------------:|
| 小数定价（$2.49） | ~3秒/单 | ~20单/小时 | ~$60/小时 |
| 整数定价（$2） | ~2秒/单 | ~30单/小时 | ~$90/小时 |
| **效率提升** | **-33%** | **+50%** | **+50%** |

### 为什么设计师加入这个小细节？

| 解释 | 设计意图 |
|:----|---------|
| 真实感 | 现实中收银员找零确实比收整钱慢 |
| 策略深度 | 将"定价策略"从单纯的"赚最多"延伸到了"运营效率"维度 |
| 奖励细心玩家 | 只有注意到这个细节的玩家才会调整定价，形成认知优势 |
| 惩罚"贪心" | 想多赚$0.49而设小数价的玩家，实际上因为效率损失而赚得更少 |

> **在实际经营中**：设整数价不仅提高收银效率，还能降低顾客的决策时间。顾客不需要盯着价格标签思考"这点零钱行不行"——直接递整钱，拿货走人。

## 七、与真实TCG二级市场的对比

### 核心差异

| 维度 | 真实TCG市场（Pokémon/Magic） | 卡牌店模拟器 |
|:----|:---------------------------|:------------|
| **参与者** | 收藏家、玩家、炒家、庄家 | 玩家（一人分饰上述所有角色） |
| **评级机构** | PSA/BGS/CGC（专业第三方） | 游戏内置鉴定系统（简化版1-10分） |
| **交易平台** | TCGplayer/eBay/卡店实体店 | 店内顾客交易（简化B2C） |
| **价格发现** | 供需关系+炒作+赛事成绩 | 系统随机波动+活动影响 |
| **信息透明度** | 半透明（TCGplayer有历史价格） | 全透明（手机APP实时市价） |
| **做市商** | 大型卡店和在线平台 | 玩家自己（买断所有顾客的卡） |

### 保留的核心定价逻辑

尽管做了大量简化，游戏保留了真实TCG定价的核心逻辑：

> **稀缺性 + 品相 = 价值**

| 公式变量 | 真实TCG | 卡牌店模拟器 |
|:--------|:-------|:-----------|
| **稀缺性(S)** | 印刷量+版本+年代 | 边框等级（6层） |
| **品相(C)** | PSA/BGS评分（1-10） | 游戏内鉴定评分（1-10） |
| **价值(V)** | V = S x C（非线性） | 市价 x 边框倍率 x 鉴定倍率 |

### 游戏简化了但没丢失的东西

1. **市场情绪模拟**：游戏通过"活动→边框影响"模拟了真实市场中"某卡在比赛中夺冠→价格暴涨"的现象
2. **信息不对称**：顾客交易报价低于市价，模拟了真实世界中"普通卖家vs专业卡店"的信息差
3. **持有成本**：租金+电费+工资 = 真实卡店的运营成本，迫使玩家不能只囤卡不卖卡
4. **流动性需求**：稀有卡虽然贵但难卖（GameStop效应）——游戏中的顾客交易系统也模拟了这一点

## 总结：二级市场设计的成功密码

| 设计亮点 | 核心机制 | 玩家体验 |
|:--------|---------|---------|
| 动态市价 | 每天~40%概率±0.08%-2%波动 | 投机空间+抄底快感 |
| 活动操纵 | 选择性影响特定边框价格 | 从被动接受到主动操纵 |
| 崩溃/暴涨 | 偏离≥25%触发反向回归 | 戏剧性事件+社交话题 |
| 顾客交易 | 信息不对称的差价获利 | "占便宜"的满足感 |
| +10%-30%定价 | 弹性定价模型 | 策略深度+运营效率 |
| 整数定价加速 | 定价影响收银效率 | 奖励细心观察的玩家 |
| 鉴定增值 | 品相决定价值的二层赌博 | 将卡牌从"商品"提升为"资产" |

卡牌店模拟器的定价与二级市场系统，本质上是将**真实TCG市场的核心经济逻辑**简化后封装在一个单人游戏中。它没有追求"完全模拟真实市场"的复杂度，而是提取了最有趣的几个元素——投机、操纵、信息差、品相定价——让玩家在单人游戏中体验到"做一个卡牌店老板兼市场操盘手"的乐趣。
