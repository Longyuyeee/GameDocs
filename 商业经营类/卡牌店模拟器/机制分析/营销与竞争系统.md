## 一、营销核心载体：活动管理APP

玩家手机中的"活动管理"(Event Management)功能是唯一的主动营销手段。店等级越高，可选活动越多。

### 1.1 活动类型与解锁

| 活动名称 | 解锁等级 | 举办费用($) | 对卡牌市价影响 | 影响边框类型 | 持续时间(天) |
|:---------|:--------:|:-----------:|:--------------:|:------------:|:------------:|
| Standard Tournament | Lv1 | 待补充 | +0.2%~+1% | Common | 1 |
| Rare Tournament | Lv3 | 待补充 | +0.5%~+2% | Rare | 1 |
| Foil Tournament | Lv6 | 待补充 | +1%~+3% | Foil | 1 |
| Borderless Tournament | Lv10 | 待补充 | +1.5%~+3.5% | Borderless | 1 |
| Holo Tournament | Lv15 | 待补充 | +2%~+4% | Holo | 1 |
| Reverse Holo Tournament | Lv20 | 待补充 | +2%~+4.5% | Reverse Holo | 1 |
| Destiny Tournament | Lv30 | 待补充 | +2.5%~+5% | Destiny | 1 |
| Legendary Tournament | Lv50 | 待补充 | +3%~+5% | Legendary | 1 |

> 说明：每次活动需支付前置举办费，但可通过向参赛顾客收取入场费回收成本甚至盈利。活动收益=入场费收入-举办费，入场费由玩家自行设定。

### 1.2 活动对市场价的动态影响机制

| 机制维度 | 描述 |
|---------|------|
| **影响范围** | 特定边框类别的所有卡牌，非单张卡 |
| **影响方向** | 仅正向上涨(无负面影响活动——负面影响由市场崩溃/暴涨机制单独处理) |
| **影响幅度** | +0.2%~+5%浮动，取决于活动等级和边框稀有度 |
| **影响持续** | 活动举办当日有效，次日恢复 |
| **策略意义** | 选择此时出售受影响边框的高价卡→利润最大化 |
| **叠加规则** | 每日仅可举办一个活动，无叠加效果 |

## 二、游戏桌收费系统

游戏桌(Game Table)是店内固定设施，顾客付费使用。活动管理APP可设置游戏桌的比赛格式和入场费。

### 2.1 游戏桌收费设定

| 设定项 | 可选值 | 说明 |
|-------|-------|------|
| **比赛格式** | Standard / Draft / Commander(待确认) | 不同格式影响顾客参与数量 |
| **入场费** | $0~$50+(由玩家手动设定) | 价格越高→顾客参与意愿降低 |
| **收益模式** | 一次收费=一次比赛参与权 | 顾客打完后即离桌，下一位顾客入座 |
| **被动收入占比** | 占总收入约5%~15% | 前期占比低，后期多张游戏桌摊薄单桌收益 |

### 2.2 游戏桌属性

| 属性 | 数值 |
|------|:---:|
| **解锁等级** | Lv1(可购买) |
| **购买价格** | $200/张 |
| **占用空间** | 2格×2格 |
| **上限数量** | 取决于店面空间(约8-12张) |
| **维护成本** | 无(一次性购买) |

## 三、口碑/评价机制

卡牌店模拟器**没有显式的顾客评价系统或星级评分**。顾客满意度通过以下方式间接体现：

| 间接指标 | 正面行为 | 负面行为 |
|---------|---------|---------|
| **客流量增长** | 商品充足+定价合理+环境干净 | 长期缺货+定价过高+臭气蔓延 |
| **顾客停留时间** | 货架排列整齐、动线通畅 | 货架杂乱、空间拥挤 |
| **员工效率** | 顾客结账等待时间短 | 排队过长(员工不足) |
| **红叹号频率** | 顾客更愿意主动找玩家交易卡牌 | 顾客不愿停留 |

> 核心逻辑：顾客体验越好→口碑越好(隐式)→更多顾客到访→销售额增长→形成正反馈循环。

## 四、竞争系统——无直接竞争对手

卡牌店模拟器**没有任何直接竞争对手AI**。玩家是城市中唯一的卡牌店经营者。这是该品类中商业经营模拟游戏的常见设计模式。

### 4.1 竞争元素对比

| 游戏 | 竞争对手 | 竞争形式 | 压力来源 |
|------|---------|---------|---------|
| **卡牌店模拟器** | **无** | 纯单人经营 | 账单/库存/市场波动 |
| 双点医院 | 无(但有星级评比) | 与自我历史对比 | 医院评级/关卡目标 |
| 温泉物语(开罗) | 无 | 纯单人经营 | 客流/设施评级 |
| 游戏开发物语(开罗) | 无 | 纯单人经营 | 销量/评分排名 |
| 冒险村物语2 | 怪物(非商业对手) | PvE战斗 | 怪物强度跃升 |
| 双点博物馆 | 同行对手(部分) | 展品竞争 | 盗贼/评级 |

### 4.2 "无竞争"设计意图分析

| 设计维度 | 分析 |
|---------|------|
| **核心体验聚焦** | 无竞争对手=玩家的全部注意力集中在"经营自己的店"而非"打败别人的店" |
| **减压设计** | 商业经营品类中很多玩家享受的是"渐进式积累"而非"零和竞争" |
| **单人沙盒感** | 没有AI对手使游戏更接近"沙盒"体验——玩家按自己节奏经营 |
| **隐性竞争替代** | 用"市场价波动"替代"竞争对手"作为经济压力源——市场本身成为你的对手 |
| **同品类惯例** | 开罗全系列、双点系列均无直接AI竞争——这表明"无竞争"是商业经营品类的舒适区 |

## 五、设计特点分析

### 5.1 营销系统的局限性

卡牌店模拟器的营销系统相较于同品类游戏显得较为单薄：

1. **维度单一**：唯一的营销手段是"办活动→抬高某类卡价"。没有广告投放、没有品牌建设、没有忠诚度计划。
2. **被动响应**：活动是被动响应市场波动的工具，而非主动塑造市场的工具——你只能在现有边框类型中选择，不能创造新需求。
3. **无差异化定价促销**：不能做"限时折扣"、"买二送一"等零售业常见的促销策略。
4. **活动影响短暂**：仅持续一天的效果，缺乏长期营销规划的空间。

**与开罗系列的对比**：开罗游戏(如游戏开发物语、温泉物语)设有"广告投放"、"媒体宣传"、"邀请评委"等多维营销手段。卡牌店模拟器的营销系统相比之下显得功能单一——但这可能是有意为之的简约设计，让玩家注意力集中在"卡牌交易+"拆包"的核心循环上。

### 5.2 竞争缺失的双刃剑效应

| 正面效应 | 负面效应 |
|---------|---------|
| 玩家可以完全按照自己的节奏经营 | 缺乏外部竞争压力→中期可能产生经营动力衰减 |
| 没有"被AI超越"的挫败感 | 无对比参照物→缺乏"我在进步"的外部验证 |
| 适合放松型玩家 | 适合想要"硬核竞争"的玩家可能会感到无聊 |
| 市场波动提供了替代性压力源 | 市场波动的随机性≠竞争对手的策略性——前者是运气，后者需要应对 |
| 沙盒感更强——更像"我的卡牌店" | 缺少社区/排行榜等社交证明机制 |

### 5.3 从"营销"到"投机"的范式转换

卡牌店模拟器的营销系统本质上不是传统意义上的"营销"——它更像是一种**市场操纵工具**：

- 传统商业经营游戏的营销 = 提高产品知名度→增加销量
- 卡牌店模拟器的营销 = 影响特定资产类别价格→增加投机利润

这种差异反映了游戏的深层设计哲学：你不是在"经营一家零售店"，而是在**经营一个微型金融市场**。活动管理APP的作用不是"让更多人知道你的店"，而是"让某类卡牌涨价"——这是零售经营×金融投机的跨界创新。

### 5.4 无竞争场景下的玩家动机维持

在没有竞争对手的情况下，卡牌店模拟器通过以下方式维持玩家动机：

| 动机类型 | 维持机制 | 示例 |
|---------|---------|------|
| **收藏动机** | 卡牌收集完成度 | 2984张卡牌的全收集追求 |
| **财务动机** | 被动收入+资产增值 | 7天账单→现金流的压力→"再赚一点" |
| **成长动机** | 店等级解锁 | 每升一级=新内容解锁(新卡包/新设施) |
| **投机动机** | 市场波动+交易套利 | 低买高卖→"聪明交易者"的成就感 |
| **赌博动机** | 拆包/鉴定 | God Pack/全图闪的惊喜时刻 |

> **设计洞察**: 卡牌店模拟器用**五种动机替代竞争对手**——每种动机都是"玩家vs系统"的内在竞争，取代了"玩家vsAI"的外在竞争。这让游戏在无竞争状态下依然驱动玩家持续投入数十甚至数百小时。
