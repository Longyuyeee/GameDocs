| 区域 | 解锁条件 | 初始大小 | 可扩张次数 | 功能定位 |
|:--|:--|:--:|:--:|:--|
| A 区（主店面） | 初始可用 | 小 | 30 次扩张 | 零售区、展示区、收银区 |

### 3.2 区域功能规划

| 方案 | B 区用途 | 优点 | 缺点 |
|:--|:--|:--|:--|

## 四、动线与布局设计

### 4.1 顾客动线原则

| 原则 | 说明 | 布局建议 |
|:--|:--|:--|
| 通道宽度 | 通道太窄会导致顾客拥堵 | 至少留 1 格宽的通道，主干道建议 2 格 |
| 入口 → 货架 | 顾客进店后应自然看到商品 | 入口正对主货架区域，避免视线遮挡 |
| 货架 → 收银台 | 从选购到结账的路径应简洁 | 收银台靠近出口，动线末端 |
| 高价值商品位置 | 吸引顾客深入店内 | 展示柜放在店铺深处，引导顾客走过更多货架 |

### 4.2 常见布局模式

| 布局模式 | 描述 | 适合阶段 | 优点 |
|:--|:--|:--|:--|
| 沿墙式 | 货架靠墙放置，中间留空 | 初期（空间小） | 最大化利用边缘空间 |
| 岛屿式 | 中央放置双面货架，四周沿墙 | 中期 | 利用双面货架展示更多商品 |
| 网格状 | 规整的行列排列 | 中后期 | 动线清晰，空间利用率高 |
| 自由式 | 不规则布局，配合角落货架 | 后期（空间大） | 视觉丰富，可设置主题区域 |

### 4.3 空间优化贴士

| 场景 | 问题 | 解决方案 |
|:--|:--|:--|
| 通道拥堵 | 顾客频繁卡住 | 扩宽主通道至 2~3 格，避免死胡同 |
| 收银台排队过长 | 顾客放弃购买 | 增加收银台或提高收银速度 |
| 货架补货困难 | 员工效率低 | 将仓库（B 区）靠近货架区 |
| 展示柜闲置 | 高价值卡卖不出 | 放在显眼位置或靠近入口 |
| 游戏桌无人用 | 空间浪费 | 放在动线经过处，不要放在角落 |

## 六、设计特点分析

### 6.1 设计意图

1. **渐进式解锁驱动持续动力** — 每个等级都有新设施解锁，从 Lv2 的 Small Cabinet 到 Lv95 的 Tetramon Shelf，玩家始终有"下一个目标"。这种渐进式的设施解锁与店等级直接挂钩，形成了等级提升→新设施→经营效率提升→更快升级的正反馈循环。

2. **空间管理的策略深度** — A 区/B 区双区域设计（Lv15 解锁 B 区）让玩家面临战略性选择：B 区是作为纯仓库提高供应链效率，还是作为额外零售区直接创收？这种"空间用途权衡"是模拟经营游戏中最经典也最有效的策略决策之一。

3. **自动化投资线** — 除臭机（Lv5/17/40）和自动拆包机（Lv16/32/55）构成了两条清晰的自动化升级路径：从手动（除臭喷雾/手动拆包）到半自动（基础机器）到全自动（高级机器）。每一次升级都是"一次性投资换取长期效率提升"的决策。

4. **顾客动线即隐性游戏机制** — 传统模拟经营游戏中，布局主要影响美观；在本游戏中，布局直接影响经营效率（通道太窄 = 顾客拥堵 = 销售损失）。这迫使玩家在实际运营中不断调整布局——将"室内设计"变成了有实际游戏机制反馈的决策维度。

### 6.2 跨游戏对比

与 **《Supermarket Simulator》** 的设施系统对比：

| 对比维度 | TCG Card Shop Simulator | Supermarket Simulator |
|:--|:--|:--|
| 货架种类 | 9 种零售 + 2 种仓库（共 11 种） | 3~5 种基本货架 |
| 功能性设施 | 多（展示柜/除臭机/拆包机/游戏桌/工作台） | 少（收银台/冷藏柜） |
| 双区域设计 | 有（A区+B区） | 无（单一店面） |
| 自动拆包 | 有（3 级自动拆包机） | 无对应机制 |
| 设施与等级绑定 | 强——每个等级解锁 1~2 种新设施 | 较弱——有限设施 |
| 动线影响 | 有——拥堵直接导致销售损失 | 有限——顾客可穿插 |

**本游戏独特之处**：自动拆包机是一种"生产型设备"——它将卡包转换为单卡，使玩家能够在不投入手动时间的情况下持续获得可销售的卡牌。这在模拟经营游戏中是不常见的设施类型（大多数游戏只有"进货→上架→销售"的线性链，而本游戏增加了"进货→加工→销售"的增值链）。

### 6.3 策略建议

| 阶段 | 优先投资的设施 | 原因 |
|:--|:--:|:--|
| Lv1~5 | Small Cabinet + Play Table | 性价比最高的初始组合 |
| Lv6~15 | 除臭机 + 解锁 B 区 | 环境管理 + 空间扩展 |
| Lv16~30 | 自动拆包机 S/M + 展示柜 | 解放手动时间 + 单卡销售 |
| Lv31~55 | 自动拆包机 L + 大型货架 | 全面自动化 + 最大化陈列 |
| Lv56+ | 主题货架 + 顶级除臭 | 终极配置 |
