## 一、店等级系统

店等级(Shop Level)是卡牌店模拟器的**核心推进器**——几乎一切内容都通过升级解锁。等级通过销售、拆包、游戏桌获得EXP。

### 1.1 等级推进概览

| 等级范围 | 解锁内容 | 主要里程碑 |
|:--------:|---------|-----------|
| Lv1 | 初始小店 | 基础货架、Standard Pack、Common卡 |
| Lv2-Lv4 | 基础扩张 | A区首次扩张($300)、Basic Pack |
| Lv5 | Rare Pack + 除臭机 | 卡牌稀有度跃升、店内环境管理 |
| Lv10 | **第1名员工 + 工作台** | 自动化补货、卡盒制作 |
| Lv15 | **B区解锁 + 收银台 + 第2名员工** | 双区域经营、双收银通道 |
| Lv20 | Legendary Pack + 第3名员工 | 顶级卡包解锁 |
| Lv25 | Basic Destiny Pack + 第4名员工 | Destiny系列第一弹 |
| Lv30 | Destiny Pack + 第5名员工 | 更高级Destiny包 |
| Lv35 | 待补充 | 待补充 |
| Lv40 | 待补充 | 待补充 |
| Lv45 | 第8名员工(全部员工满) | 人员结构完整 |
| Lv50 | **Legendary Destiny Pack** | 最强卡包解锁 |
| Lv73 | A区最后一次扩张($26,400) | A区空间最大化 |
| Lv100 | **Legendary Shop(Legendary成就)** | 顶级店称号 |
| Lv115 | B区最后一次扩张($34,150) | B区空间最大化 |

### 1.2 EXP获取来源

| 来源 | EXP效率 | 说明 |
|------|:------:|------|
| **销售卡包/商品** | ★★★★ | 每售出一件商品获得固定EXP，大额销售=快速升级 |
| **拆包** | ★★★ | 拆卡包获得的EXP相对较少，但可同时获得卡牌收藏 |
| **游戏桌收入** | ★★ | 每场游戏获得小额EXP，被动积累 |
| **卡牌鉴定** | ★ | 鉴定卡牌获得微量EXP(该功能侧重财务收益而非升级) |

### 2.3 扩张策略

| 策略 | 说明 | 推荐场景 |
|------|------|---------|
| **激进扩张** | 达到等级要求立刻扩张→最大化空间→更多货架更多收入 | 现金充裕、追求快速升级 |
| **稳健扩张** | 保留现金应对账单，有余钱再扩→减少破产风险 | 前期现金紧张、新手玩家 |
| **B区优先** | 解锁B区后优先扩B区(费用相对低+新区域新顾客) | Lv15~Lv30阶段 |
| **A区优先** | 先扩A区(主区域)再到B区 | 高客流需要更多主区空间 |

| 设计维度 | 分析 |
|---------|------|
| **节奏控制** | 许可证=等级之外的第二道门禁——即使等级够了，没钱买许可证也无法解锁 |
| **经济漏斗** | 前期→花$50解锁基础产品→赚到$200→解锁Rare Pack→...形成阶梯式经济推进 |

| **挫败感平衡** | 许可证费用设定在"可承受但不轻松"的水平——确保玩家需要认真考虑而非无脑购买 |

## 五、规模效应与正反馈循环

### 5.1 扩张的自我强化逻辑

```
店面扩张 → 更多空间
    ↓
更多货架(容纳更多卡牌种类)
    ↓
更多商品品类 → 吸引更广泛顾客群
    ↓
提升客单价 + 增加客流量
    ↓
更高的日收入
    ↓
更快积累 → 购买更多许可证 → 解锁更高级卡包
    ↓
更高利润的商品 → 更高收入
    ↓
更快扩张 → (回到顶部)
```

### 5.2 规模效应的效率评估

| 扩张阶段 | 投资回报率 | 风险 |
|:--------:|:---------:|:----:|
| 前期(1-5次扩张) | **极高**——少量投资换大量空间 | 低——费用低 |
| 中期(6-15次扩张) | 中——空间增加明显但租金也涨 | 中——租金开始成为负担 |
| 后期(16-30次扩张) | 递减——边际空间价值下降 | **高**——租金涨幅大 |
| B区解锁 | 高——全新区域吸引新顾客 | 中——解锁费$5,000 |

> **设计洞察**: 卡牌店模拟器的扩张系统在设计上呈现**先快后慢**的节奏——前期扩张费用低、解锁快，给予玩家"快速增长"的正反馈；后期费用高、等待等级门槛长，迫使玩家放慢节奏，将注意力从"扩张"转移到"精细化运营"(定价策略、市场投机、卡牌收集)。这种节奏设计有效延长了游戏生命周期。

### 6.2 卡牌店模拟器vs开罗系列的核心差异

| 差异点 | 卡牌店模拟器 | 开罗系列 |
|--------|------------|---------|
| **扩张方向** | 同店内**水平扩张**(面积变大) | **垂直扩张**(设施星级+功能升级) |
| **空间语义** | 空间是**资源**(决定了能放多少货架/顾客) | 空间是**载体**(重点是单建筑的品质) |
| **扩张节奏** | 按等级线性解锁 | 按年份/星级阶段性解锁 |
| **重置机制** | **无**——单店越来越好 | 开罗某些系列有重置/新游戏+机制 |
| **终极状态** | "我的店又大又全"——空间+商品品类全解锁 | "我的城镇评分最高"——建筑品质+组合最优化 |

## 七、设计特点分析

### 7.1 "双锁"解锁哲学——等级+资金的双重约束

卡牌店模拟器的所有可解锁内容都设置了**双重约束**：

| 解锁项目 | 第一道锁 | 第二道锁 |
|---------|---------|---------|
| 新卡包 | 等级达到 | 许可证费用 |
| 空间扩张 | 等级达到 | 扩张费用+租金增长 |
| 员工 | 等级达到 | 日薪支出 |
| B区 | Lv15 | $5,000解锁费 |
| 装修/设施 | 等级达到 | 购买费用 |

这种"先证明你的等级(投入时间)，再花你的钱(经济实力)"的双锁机制，确保玩家不会通过"单纯攒钱"跳过内容——即使有钱，等级不够也解锁不了。避免了"人民币玩家"式的跳过，保持了渐进式的游戏体验。

### 7.2 与双点医院的扩张哲学对比

双点医院和卡牌店模拟器采用了截然不同的扩张哲学：

| 维度 | 卡牌店模拟器 | 双点医院 |
|------|------------|---------|
| **扩张边界** | **无边界**——A+B区可无限扩大到整个空间 | **有边界**——每个关卡有固定空间限制 |
| **区域独立性** | A区(主区)+B区(副区)独立扩张 | 无分区概念，整体规划 |
| **扩张节奏** | 速度由玩家控制(想扩就扩) | **关卡制**——完成当前关→进下一关 |
| **重置** | 永不重置——同一家店从头到尾 | 每关重置——从零开始建新医院 |
| **目标** | "把我的店做到最大"——沙盒式 | "把医院开到最顶级"——关卡制 |

双点医院"关卡制+重置"的设计让每次进入新关卡都有"从零开始的兴奋感"；而卡牌店模拟器的"单店持续扩张"设计则让玩家对自家店铺产生**情感依恋**——"这是我亲手打造的店"。两种哲学各有优劣，卡牌店模拟器的选择更偏向"养成感"。

### 7.3 无连锁经营的战略意义

卡牌店模拟器明确选择了**单店经营**，摒弃了"连锁扩张"这个常见的商业经营题材元素：

- 玩家只需要管理**一家店**——所有资源、精力、注意力集中于此
- 没有"开分店"的扩张维度——这意味着玩家不会陷入多店管理的"行政疲劳"
- 单店的深度(等级/卡牌收集/全解锁)替代了多店的广度(开连锁店)
- 游戏的文件名"TCG Card Shop Simulator"强调"A Card Shop"(一家店)——单店是游戏的核心设定

> **与开罗《温泉物语》的对比**: 温泉物语虽然也是单场地经营，但其"建筑搭配→组合效应→全图5星"的目标链更为复杂，需要玩家考虑建筑之间的位置关系。卡牌店模拟器则没有这类"组合搭配"机制——扩张只是"买空间→放东西"，少了一层布局策略的深度，但也降低了学习门槛。
