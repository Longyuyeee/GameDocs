## 一、市场定价机制完整数据

### 1.1 每日价格波动机制

| 参数 | 数值 | 说明 |
|------|:----:|------|
| 触发时机 | 每日营业开始时自动刷新 | 每日凌晨(游戏内)系统对所有商品进行一次批量价格调整 |
| 波动对象 | 每个商品独立波动 | 商品之间的价格涨跌互不关联 |
| 显示方式 | 绿色↑(涨价) / 红色↓(降价) / 灰色—(持平) | 在市场界面中清楚显示 |
| 波动幅度(社区估算) | **约±5%~±15%** | 游戏未公示精确幅度，社区通过长期记录估算 |
| 进价联动机制 | **进价=市价(完全跟随)** | 独特的双轨制——进货价格直接等于当日的市场价格，无独立进价波动 |
| 玩家定价自由度 | 完全自由(可设任意金额) | 可高于或低于市场价，低于市价会亏损 |
| 极端波动风险 | **无崩溃/暴涨机制** | 与《卡牌店模拟器》不同，市场价格不会出现极端波动或经济崩溃 |

### 1.2 定价机制核心特征

| 特征 | 说明 | 与同类游戏对比 |
|------|------|--------------|
| 进价=市价 | 进货成本与市场价格完全绑定，不存在"低价进货高价卖出"的批发模式 | 卡牌店模拟器：固定进价+浮动市价；双点医院：无定价系统 |
| 无囤货套利 | 因进价=市价，无法通过"低价时囤货、高价时卖出"获利 | 与现实超市不同，简化了供应链金融 |
| 每日强制关注 | 因价格每日波动，玩家需要每日检查并调整定价 | 增加了日常经营仪式感和操作频次 |
| 退货无惩罚 | 商品未售出不会变质(目前版本)，可以长期持有 | 降低了定价错误的代价 |

### 1.3 社区15次独立测试结果（最优定价验证）

Steam社区多位玩家（Dragonfly等人）通过15次独立测试验证了不同定价策略的利润表现。以下是代表性3天测试结果：

| 定价策略 | 日均收入 | 日均毛利润 | 毛利率 | 日均投诉数 | 顾客流失率估 |
|:-------:|:-------:|:---------:|:-----:|:---------:|:----------:|
| -5%(低于市价) | ~$3,416 | ~$400 | ~11.7% | **0** | ~0% |
| 0%(市价) | ~$4,427 | ~$1,252 | ~28.3% | **0** | ~0% |
| +5% | ~$4,395 | ~$1,602 | ~36.5% | ~5 | <5% |
| **+10%** | **~$4,402** | **~$1,652** | **~37.5%** | **~12** | **~10%** |
| +15% | ~$4,103 | ~$1,395 | ~34.0% | ~45 | ~25% |
| +20% | ~$3,967 | ~$1,574 | ~39.7% | ~64 | ~30%+ |

**结论**：+10%定价在毛利和净利润之间取得最佳平衡，是社区公认的最优策略。

## 二、定价策略数据对比表

| 策略名称 | 加价幅度 | 日均投诉数 | 顾客流失率(估) | 日均净利润(估) | 推荐场景 |
|---------|:-------:|:---------:|:------------:|:-------------:|---------|
| 零投诉策略 (Zero Complaint) | +3%或+$0.10~0.20 | **0** | **0%** | 最低(~$1,300~1,400) | 冲等级/新手期 |
| 安全利润策略 (Safe Profit) | +5~7% | 0~2 | <5% | 中等(~$1,500~1,600) | 平衡发展/不想处理投诉 |
| **最优利润策略 (Optimal)** | **+8~10%** | **3~12** | **~10%** | **最高(~$1,600~1,750)** | **主流推荐/大多数场景** |
| 激进策略 (Aggressive) | +15~20% | 15~25+ | ~25% | 略低(~$1,400~1,600) | 短期急需现金(不推荐长期) |
| 不可持续 (Unsustainable) | >+30% | 大量(30+) | >50% | 大幅下降(~$1,000以下) | **不推荐** |

### 定价策略进阶技巧

| 技巧 | 说明 | 推荐度 |
|------|------|:-----:|
| 四舍五入到$0.25/$0.50 | 简化找零(尤其早期亲自收银时)，顾客投诉不会因此增加 | 高 |
| 酸奶单独降价 | 酸奶是投诉率最高的品类，即使+5%也可能触发投诉，建议仅+3%或保持市价 | 中 |
| 高单价商品灵活加价 | 威士忌等售价~$60的商品，+10%=+$6，可能导致极端投诉，建议降至+5~7% | 中 |
| 折扣叠加策略 | 设+10%售价 + 1%折扣 → 实际+9%，享受客流加成同时投诉减少 | 高 |
| 分品类差异化定价 | 敏感品类(酸奶/面包)设+5%，高利润品类(牛排/寿司)设+10% | 中(需精细管理) |

### 3.1 收入结构总表

| 在线配送 (Online Delivery) | v1.0 ~Lv50+ | **~5~10%** | ~$200~500 | 线上订单配送收入，无需玩家到店 |
| 其他收入 | N/A | **~0~5%** | ~$0~200 | 散装称重损耗差等零散收入 |

### 3.2 店内零售收入公式

```
单笔零售收入 = Σ(商品售价_i × 顾客购买数量_i)
日零售总收入 = Σ(所有顾客的单笔零售收入)
日净利润 = 日零售总收入 + 日配送收入 - 日总成本
```

## 五、批发 vs 送货成本对比

| 对比维度 | 批发市场自提 (Wholesale) | 送货到店 (Delivery) |
|---------|:----------------------:|:-----------------:|
| 商品价格 | **基准价(约-50%)** | **+~50%溢价** |
| 每箱有效成本 | ~$X(基准价) | ~$X × 1.5 |
| 送货附加费 | 无 | $2~8/次(视箱数) |
| 时间成本(现实时间) | **~5~10分钟/次** | ~30秒(下单操作) |
| 单次可购数量 | 不限(随车容量) | 不限 |
| 利润率影响 | **翻倍(批发节省=利润)** | 基准确认 |
| 推荐时机 | 资金充裕、时间充足 | 紧急补货、时间紧张 |

### 经济模型对比：单次采购$1,000商品

| 项目 | 批发自提 | 送货到店 |
|------|:-------:|:-------:|
| 商品总进价 | ~$500(约半价) | $1,000 |
| 送货费 | $0 | $8(7+箱) |
| 总成本 | **~$500** | **~$1,008** |
| 按+10%定价的售价 | ~$1,100 | ~$1,100 |
| 毛利 | **~$600** | **~$92** |
| 毛利率 | **~120%** | **~9%** |

> 警示：仅用送货到店模式下，+10%定价的毛利率极低(~9%)，大部分利润被进货成本吞噬。**批发自提是游戏中后期利润的核心来源**——社区报告指出，从仅用送货切换到批发自提可使净利润翻2~4倍。

## 六、折扣ROI分析

### 折扣力度 vs 关键经济指标

| 折扣力度 | 客流加成 | 日均收入变化(估) | 日均成本变化(估) | 日均利润变化(估) | 投资回报率 |
|:-------:|:-------:|:--------------:|:--------------:|:--------------:|:---------:|
| 0%(基准) | 基准 | 基准($4,400) | 基准($2,750) | 基准($1,650) | 基准 |
| 1%折扣(1件商品) | **+50%(达上限)** | +~5~10% | +~8~15%(补货成本增) | 略增或持平 | **高(利润≈基准)** |
| 5%折扣(1件商品) | +50%(已达上限) | +~3~8% | +~8~15% | 略降 | 中 |
| 10%折扣(多件商品) | +50%(已达上限) | +~0~5% | +~10~20% | 下降明显 | 低 |
| 20%+折扣 | +50%(已达上限) | 下降(让利过多) | +~15~25% | **大幅下降** | **负** |

### 折扣策略推荐

| 策略 | 操作方法 | 效果 | 推荐度 |
|------|---------|------|:-----:|
| 最小折扣引流 | 对1件商品设1%折扣 | +50%客流，利润几乎无损 | **强烈推荐** |
| 促销区布局 | 设5件商品各-40%折扣(标为"促销区") | 同样+50%客流，打造促销感 | 推荐(适合需要引流但想打造"促销感"的玩家) |
| 折扣叠加定价 | 先+10%定价再设1~5%折扣(实质+5~9%) | 享受折扣客流，减少投诉 | 推荐(兼顾流量和利润) |
| 全场打折 | 全部商品设折扣 | 客流已达上限(+50%)，无额外增益 | 不推荐 |

## 九、关键经济公式汇总

| 公式 | 表达式 |
|------|-------|
| 日总收入 | = Σ(每件商品售价 × 当日销量) + 配送收入 |
| 日总成本 | = 进货成本 + 员工工资 + 房租 + 电费 + 贷款利息 |
| 日净利润 | = 日总收入 - 日总成本 |
| 单件毛利 | = 售价 - 进价(进价=当日市价) |
| 毛利率 | = (售价 - 进价) / 售价 × 100% |
| 推荐售价 | = 市价 × (1 + 0.08~0.10) → 四舍五入到$0.25 |
| 折扣后实际售价 | = 推荐售价 × (1 - 折扣率) |
| 投资回收期 | = 设备购置费 / (日均节省 - 日均新增成本) |
| 批发节省金额 | = 送货价 - 批发价 ≈ 送货价 × 50% |
