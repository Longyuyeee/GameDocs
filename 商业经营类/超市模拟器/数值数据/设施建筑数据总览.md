## 三、设施推荐优先级

### 初期 (Lv1~15)

| 优先级 | 设施 | 理由 |
|:-----:|------|------|
| 1 | Basic Shelf ×2~4 | 初始商品必须上架 |
| 2 | Basic Fridge ×1~2 | 许可证1(Lv3)的乳制品/饮料需冷藏 |
| 3 | Basic Register ×1 | 收银必需，初期可手动操作 |
| 4 | Trash Can ×1 | 处理进货纸箱 |
| 5 | Mop ×1 | 清洁地面污渍 |
| 暂缓 | Storage Room | $1,000解锁费较高，可通过箱堆地面替代 |

### 中期 (Lv16~42)

| 优先级 | 设施 | 理由 |
|:-----:|------|------|
| 1 | Large Shelf / More Shelves | 许可证8~12带来大量新品，需更多货架空间 |
| 2 | Basic Freezer ×1~2 | Lv13解锁冷冻品，需冰柜 |
| 3 | Large Fridge ×1~2 | 更多冷藏品(肉类/寿司/奶制品)需要容量 |
| 4 | More Register / Self-Checkout | 客流量增大，排队问题凸显 |
| 5 | Storage Section 2 ($800) | 商品种类超过~60种后，仓库空间紧张 |
| 6 | Advanced Register (~$500) | 提升收银效率 |

### 后期 (Lv43~70)

| 优先级 | 设施 | 理由 |
|:-----:|------|------|
| 1 | Self-Checkout Kiosk ×1~2 | 减少排队，降低人工依赖 |
| 2 | Security Camera ($500) | 高价值商品增多，小偷损失加大 |
| 3 | Alarm Gate ($350) | 配合摄像头构成完整安防体系(注意Bug风险) |
| 4 | Packing Station + Delivery Vehicle | 解锁线上配送收入(v1.0) |
| 5 | Storage Expansion | 商品种类超过~100种后仓库必扩 |
| 6 | Security Guard (招聘) | 自动拦截小偷，减少手动操作 |

### 终局 (Lv70~90+)

| 优先级 | 设施 | 理由 |
|:-----:|------|------|
| 1 | 全店High Capacity设施替换 | 最大化单位面积效率 |
| 2 | 自助结账比例提升 | 降低员工工资成本 |
| 3 | 储藏室满配(32架) | 支持~156种商品的完整库存 |
| 4 | 装修装饰 | 无可量化收益，纯视觉优化 |
