### 从手游到PC的转型

Nokta Games并非横空出世的新团队。2017年成立后，团队最初专注于移动游戏市场：

| 时间 | 项目 | 类型 | 表现 |
|:--:|------|:--:|:--:|
| 2017 | Nokta Games成立 | —— | Yusuf Toptan创立，初始定位手游开发 |
| 2019 | Toy of War | 策略手游（Google Play） | IAP收入不到$5,000，商业上不成功 |
| 2020-2023 | 多款超休闲手游 | 超休闲 | 收入有限，但积累了Unity开发经验和快节奏设计感 |
| 2023 | 转向PC市场 | —— | 意识到PC独立游戏市场机会大于竞争激烈的手游市场 |
| 2024 | Supermarket Simulator EA | PC模拟经营 | 现象级爆款，峰值38K CCU |

> **转型启示**：手游时代磨练了Nokta对"快节奏上手体验"和"低门槛核心循环"的理解，这些经验被成功嫁接到了PC模拟经营品类中。超市模拟器的"即开即玩"特性，很大程度上继承了手游的易上手设计哲学。

## 3. 技术与工具

| 技术栈 | 用途 |
|------|------|
| Unity（引擎） | 游戏开发核心引擎 |
| Steamworks SDK | Steam成就+云存档+创意工坊（未来） |
| Unity Asset Store | 部分美术资源来源（推测） |
| Blender | 3D建模（内部资源） |
| Git | 版本控制 |

**技术亮点**：

- 第一人称视角的物理交互系统（抓取、搬运、扫描条码）
- 顾客AI寻路+排队系统（用Unity NavMesh实现）
- 实时市场价格波动系统（每日自动计算涨跌）
- 冷链变质计时系统

**技术局限**：

- 大型店面+多顾客时帧率下降明显（1.0优化后有所改善）
- 多人协作模式仍在开发中，技术挑战未完全解决

## 5. 设计启示

### 5.1 从手游到PC的转型路径

Nokta Games的转型之路提供了清晰的路线图：

1. **手游积累期**（2017-2023）：6年手游开发经验，虽未产生爆款，但积累了Unity技术栈和游戏设计sense
2. **切换赛道**（2023）：识别到PC独立游戏市场的增长红利，决定从竞争激烈的手游红海转向蓝海
3. **主题选择**：选择一个全球性、生活化的主题（超市）——无需文化背景解释，任何人都有超市购物经验
4. **差异化**：在抽象管理模拟泛滥的市场中，用第一人称手动操作制造差异化

### 5.2 第一人称的沉浸感溢价

- **竞品分析**：传统经营模拟（如《双点医院》《监狱建筑师》）均采用上帝视角+点击菜单
- **超市模拟器的选择**：第一人称+亲手操作——看到自己的手拿起商品、扫描条码、放入购物袋
- **效果**：这种身体参与感产生了远超预期的沉浸感，是游戏病毒传播的核心驱动力

### 5.3 Steam Next Fest的杠杆效应

| 事件 | 效果 |
|------|:--:|
| Next Fest Demo | 进入前50名→获得Steam算法推荐 |
| 33K愿望单 | Next Fest期间积累 |
| EA上线 | 首日3K CCU（正常表现） |
| 口碑扩散 | 2周后→38K CCU（12.7x增长） |

**关键洞察**：Next Fest的价值不是上线当天的暴发，而是为口碑扩散提供了"初始种子用户"——这33K愿望单玩家在EA首周扮演了传播节点的角色。

### 5.4 对小团队的启示

| 启示 | 说明 |
|:--:|------|
| 选一个生活化主题 | 超市→人人有共鸣→容易理解→自然传播 |
| 做到真实可感 | 第一人称搬运+条码扫描→"看起来很好玩"→冲动购买 |
| 利用直播效应 | "搬箱子"的视觉反差天然适合直播/短视频 |
| 价格策略合理 | $12.99的低门槛价格降低决策成本 |
| 持续更新 | EA期间的持续更新维持社区活力和评价 |

## 6. 竞品对比

| 维度 | 超市模拟器 | 卡牌店模拟器 | 传统经营模拟 |
|:--:|------|------|------|
| 视角 | **第一人称** | 第一人称 | 上帝视角 |
| 操作方式 | 手动搬货+扫描 | 手动拆包+上架 | 点击菜单 |
| 核心体验 | 身体劳动感 | 收集惊喜感 | 策略规划 |
| 团队规模 | 4人 | 3人 | 10-100+人 |
| 爆发路径 | Next Fest→Twitch→口碑 | 同类路径 | 传统宣发 |
| Steam评价 | 91%（65K+） | 类似好评率 | 不等 |

> 超市模拟器与卡牌店模拟器代表了"第一人称经营模拟"这一新兴子品类的双璧——它们证明了"亲手操作"比"上帝视角的抽象管理"更能引发玩家共鸣和社交传播。
