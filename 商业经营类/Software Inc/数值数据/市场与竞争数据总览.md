## 二、消费者触达(Reach)系统

### 2.1 基础Reach公式(社区估算)

```
总Reach = 基础Reach x 渠道系数 x 营销系数 x 市场份额系数
```

| 变量 | 说明 | 典型范围 |
|------|------|:--:|
| **基础Reach** | 由软件品类和市场总容量决定 | 1,000-10,000,000 (社区估算) |
| **渠道系数** | 发行渠道的覆盖效率 | 0.1x - 1.0x |
| **营销系数** | 营销活动的叠加效果 | 1.0x - 5.0x (社区估算) |
| **市场份额系数** | 当前品类中的品牌份额 | 0.01x - 1.0x |

### 2.2 Reach转化漏斗(社区估算)

```
Reach(触达)
  ↓ 转化率约1-10%
Trial(试用)
  ↓ 转化率约20-60%
Purchase(购买)
  ↓
Repeat Purchase(续购/升级)
```

> 注：上述转化率受产品评分(Review Score)、定价、品牌知名度综合影响。具体数值游戏未公开。

## 三、粉丝/追随者(Follower)系统

### 3.1 粉丝增长来源

| 方式 | 粉丝增长量(日) | 费用($) |解锁条件 | 备注 |
|------|:--:|:--:|------|------|
| **新闻稿(Newsletter)** | ±X(约50-500/天)(社区估算) | $100-500/次 | 无 | 基础营销 |
| **媒体图(Screenshot)** | ±Y(约100-1,000/天)(社区估算) | $200-1,000/次 | 有设计师 | 图文并茂效果更好 |
| **媒体片(Trailer/Video)** | ±Z(约500-5,000/天)(社区估算) | $500-5,000/次 | 有艺术家 | 最佳营销效果 |
| **试用版(Demo)** | 转化率约5-15%(社区估算) | 免费(开发投入) | 产品Beta后 | 将Reach转化为用户 |
| **产品发布(Launch)** | 大量一次性增长 | — | 产品发布日 | 高峰期 |
| **产品评分高** | 持续有机增长 | — | 评分8+ | 长期利好 |

### 3.2 粉丝转化率参考

| 动作 | 预期效果 | 备注 |
|------|------|------|
| 发送新闻稿 | ±50-500 followers | 基础操作，效果稳定 |
| 发布媒体截图 | ±100-1,000 followers | 比新闻稿效果好 |
| 发布宣传视频 | ±500-5,000 followers | 效果最好，费用最高 |
| 发布试用版 | 5-15%的触达用户转化为试用 | 试用后约20-60%购买 |
| 产品评分下降 | -50~-500 followers/天 | 负面口碑流失 |

## 七、股价影响因素

| 因素 | 影响方向 | 权重(估算) | 说明 |
|------|:--:|:--:|------|
| **营收增长率** | 正相关 | 高 | 季度环比增长 |
| **利润率** | 正相关 | 高 | 高利润→投资者青睐 |
| **市场份额** | 正相关 | 中 | 品类主导地位 |
| **产品评分** | 正相关 | 中-高 | 8分以上显著推高 |
| **行业趋势** | 波动 | 中 | 所在品类的兴衰周期 |
| **粉丝/追随者数量** | 正相关 | 中 | 品牌价值体现 |
| **负债率** | 负相关 | 低-中 | 高负债压低股价 |
| **负面新闻(产品召回等)** | 负相关 | 高(短期) | 事件性影响 |

### IPO估值公式(社区估算)

```
IPO估值 ≈ 年利润 × 市盈率(PE) × 增长系数

市盈率参考:
  - 成熟品类(OS/Office): PE 10-15x
  - 增长品类(Game/Video): PE 15-25x
  - 热门品类(AI/VR等): PE 25-40x

增长系数:
  - 年增长>50%: 1.5-2.0x
  - 年增长20-50%: 1.2-1.5x
  - 年增长<20%: 0.8-1.0x
```

## 八、收购成本估算

| 目标公司状态 | 溢价系数 | 说明 |
|------|:--:|------|
| 小型初创 | 市值 + 20-50% | 低价收购技术/IP |
| 中型公司 | 市值 + 30-60% | 有稳定产品线 |
| 大型竞争对手 | 市值 + 50-100% | 战略收购 |
| 陷入困境的公司 | 市值 - 10% ~ +20% | 可低价抄底 |
| 触发50%+持股 | 强制收购剩余股份 | 持股超50%可吸收全部IP |

> 当持有某公司>50%股份时，可吸收其全部IP和产品线。这是快速扩张技术能力的关键策略。

## 九、股票交易参数

| 参数 | 说明 | 备注 |
|------|------|------|
| 交易市场 | 游戏内模拟股票市场 | 所有上市公司可交易 |
| 可交易对象 | 玩家公司 + AI竞争公司 | 观察产品生命周期预测涨跌 |
| 出售本公司股份 | 融资但不给投票权 | 避免贷款利息但稀释控制权 |
| 强制收购触发条件 | 某公司丢失100%股份 | 最大股东获取其产品和IP |
| 分红政策 | 可设定分红比例 | 影响股价和投资者关系 |

---

> **总结**：市场竞争系统的核心是——选择合适的品类(顺应市场趋势) + 质量过硬的产品(Review评分8+) + 有效的营销(视频>截图>新闻稿) + 合理的发行渠道(自有商店0抽成但需流量)。合同Deal提供早期稳定现金流，后期通过IPO+收购实现规模化扩张。所有数据标注"社区估算"的部分均因游戏未公开精确数值，建议游戏中实测验证。
