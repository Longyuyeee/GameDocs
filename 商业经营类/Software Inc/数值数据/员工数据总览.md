## 三、团队兼容性矩阵

### 3.1 兼容性机制

| 概念 | 说明 |
|------|------|
| **兼容度计算** | 每个团队成员的性格特质互相加权计算 |
| **兼容度等级** | 兼容(+) / 中性 / 不兼容(-) / 严重不兼容(--) |

## 九、招聘费用

| 步骤 | 费用($)(社区估算) | 说明 |
|:--:|:------|------|
| **搜索(Search)** | $100-1,000/次 | 基础招聘广告 |
| **筛选(Remove Worst)** | 额外$500-2,000 | 移除10个最差候选人 |
| **面试(Interview)** | 首次$100，每次翻倍(上限$8,000) | 每次面试翻倍，建议有限使用 |
| **签约奖金** | 月薪×1-3 | 高价值人才需签约奖金吸引 |

> 面试费用递增机制：首次$100 → 第二次$200 → 第三次$400 → ... → 上限$8,000。建议每次只面试必要数量。

---

> **总结**：员工管理是Software Inc.的核心系统。关键策略包括：(1)招聘时优先选择Fast Learner/Cheap/Nonchalance特质的员工；(2)通过Bookcase加速培训；(3)控制福利成本在$200-500/人/月；(4)Lead角色至关重要，尽早培养高级Lead；(5)面试费用递增，每次搜索只面1-2人即可；(6)团队兼容性可在面试窗口右下角查看，目标100%+。
