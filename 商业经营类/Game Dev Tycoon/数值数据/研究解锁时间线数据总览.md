# 研究解锁时间线数据总览 — Game Dev Tycoon

> 游戏版本: Game Dev Tycoon v1.8+ (Steam, 2013) | 来源: [Fandom Wiki](https://gamedevtycoon.fandom.com/)

## 一、基于年份的解锁

| 年份 | 解锁 | 战略意义 |
|:--:|------|------|
| 第3年 | 目标受众(Target Audience) | 定向客群→游戏评分提升 |
| 第3年末 | 休闲游戏(Casual Games) | 新游戏类型→更多组合可能 |
| 第4年 | 营销(Marketing) | 初售销量拉升 |
| 第8年 | 续作(Sequels) | 成功IP的系列化 |
| 第12年 | 多类型(Multi-Genre) | 混合类型→指数级组合增长 |
| 第23年 | 自有展会(Own Convention) | 最大粉丝触达(或100万粉丝提前解锁) |

## 二、基于条件的解锁

| 条件 | 解锁 | 说明 |
|------|------|------|
| 互联网机遇事件 | MMO | 互联网→大型多人在线 |
| 大型游戏评分10 | AAA游戏 | 最高制作规模 |
| 现金200M+ | 自定义主机 | 从开发商→平台持有者 |
| 粉丝100万+ | 自有展会 | 提前解锁(原本第23年) |

## 三、研发优先级排序

| 优先级 | 项目 | 理由 |
|:--:|------|------|
| 1 | Custom Engine + Copy Protection V1 | 盗版保护——生存级 |
| 2 | 2D V3→V5 | 解锁中型/大型游戏 |
| 3 | Target Audience | 评分提升 |
| 4 | 3D V1→V5 | 解锁3D动作/冒险/RPG |
| 5 | Multi-Platform | 一个游戏→多个平台→收入翻倍 |

## 四、每个产业年代的技术基线

| 年代 | 最低预期引擎 | 否则 |
|:--:|------|------|
| 80年代 | 2D V1(基础) | — |
| 90年代 | 2D V3+ | 低于V3→大型游戏不可 |
| 2000年代 | 3D V3+ | 无法开发3D动作/RPG |
| 2010年代+ | 3D V5+ | 无法AAA |

## 五、遗漏关键研发的连锁惩罚

遗漏Copy Protection V1 → 所有游戏的70%+销售被盗版侵蚀 → 现金流永远不足 → 无法投资高级引擎 → 无法解锁更大规模游戏 → **恶性循环**。这就是为什么社区共识将Copy Protection V1列为研发优先级的**绝对第一**——它不是"nice to have"，而是"不研发=游戏最终会破产"的生存条件。
