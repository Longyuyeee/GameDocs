# 营销与粉丝系统 — Game Dev Tycoon

> 数据来源: [Game Dev Tycoon Fandom Wiki](https://gamedevtycoon.fandom.com/), [Steam社区](https://steamcommunity.com/app/239820)

## 一、粉丝(Fans)系统

粉丝是GD Tycoon的核心客户资产——每发布一款游戏根据评分获得新粉丝：

| 评分 | 粉丝增长 | 长期效果 |
|:--:|:--:|------|
| 9-10(杰作) | 大量增长 | 粉丝基础×下一款游戏的保底销量 |
| 7-8(优秀) | 中等增长 | 稳定积累 |
| 5-6(一般) | 少量增长 | 缓慢积累 |
| <5(差评) | 几乎不增长 | 停滞 |
| 重复同类型组合 | 负增长(粉丝流失) | "又是同样的东西" |

## 二、营销活动

| 类型 | 效果 | 适用阶段 |
|------|------|------|
| 小型推广 | 初售销量小幅提升 | 小型游戏 |
| 中型推广 | 中幅提升 | 中型游戏 |
| **大型宣传(Hype)** | 大幅初期销量提升 | **AAA游戏必备** |
| 无营销 | 完全靠口碑和粉丝基础 | 小型/实验性游戏 |

营销不能拯救烂游戏——高营销+差评分="加速死亡"(更多人知道这是烂游戏)。

## 三、设计特点分析

**粉丝=复合资产**：粉丝既是"销售保底"(每款游戏的最低销量=粉丝基础)，也是"口碑放大器"(高评分→粉丝传播→新粉丝)。这种滚雪球效应让"持续做好游戏"成为最优策略——一次9分游戏带来的粉丝增长会影响接下来3-5款游戏的销量。

## 四、粉丝基准销量

| 粉丝数 | 每款游戏保底销量(估) | 说明 |
|:--:|:--:|------|
| <1万 | 极低 | 几乎全靠评分拉动 |
| 1-10万 | 低-中 | 开始有品牌效应 |
| 10-50万 | 中 | 中型游戏的稳定基础 |
| 50-200万 | 中-高 | AAA的粉丝保底 |
| 200万+ | 高 | 任何游戏都至少卖百万份 |

## 五、营销的"加速死亡"效应

营销放大的是**游戏本身的品质**——不是魔法。高营销+低评分=更多人看到→更多人知道它烂→更快的口碑崩塌。这是一个优雅的"反泡沫"机制——你不能通过营销预算来掩盖劣质产品。

## 六、自建展会(Own Convention)

第23年(或100万粉丝时)解锁的自建展会是终极营销工具——你可以举办自己的游戏展会(类似E3)，一次性触达最大量的粉丝群体。这代表你的品牌已强大到不需要第三方平台——你已经是平台本身。

## 七、粉丝系统的设计隐喻

GD Tycoon的粉丝系统是一个关于"品牌价值"的优雅模拟。在真实游戏产业中，像暴雪或FromSoftware这样的工作室不需要为每款游戏"重新证明自己"——他们的粉丝基础意味着任何新游戏都有一个销售保底。GD Tycoon将这个抽象概念量化为"粉丝数=保底销量"，让玩家直接感受到"建立品牌"的长期价值。一次杰作(9-10分)带来的粉丝增长=接下来N款游戏的保底基础——这就是为什么"品质为王"在GD Tycoon中不仅是一种道德选择，而是最优的经济策略。
