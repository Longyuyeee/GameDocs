# 团队与开发历程 — Game Dev Tycoon

> Greenheart Games | 成立: 2012年7月2日 | 总部: 布里斯班/远程 | 创始人: Patrick & Daniel Klug (兄弟)

---

## 一、工作室背景

| 信息 | 详情 |
|------|------|
| **创始人** | Patrick Klug(CEO/主程) + Daniel Klug(开发) |
| **团队规模** | 2人(创始)→约7人(现在) |
| **地点** | 布里斯班(澳大利亚)+全球远程 |
| **理念** | "游戏不应该是薄薄伪装的老虎机" |

## 二、开发历程

| 时间 | 事件 |
|------|------|
| 2011年10月 | 受开罗Game Dev Story启发开始开发(原计划为回合制手游) |
| 2011年12月 | 转向实时等距模拟(目标Windows Store) |
| 2012年12月 | Windows Store首发(平台第一款模拟游戏) |
| 2013年4月 | PC/Mac/Linux发布+**"盗版实验"病毒式传播** |
| 2013年8月 | Steam发布——好评如潮 |
| 2014-2024 | 58+次更新/MOD API/社区翻译/主机版 |
| 2025年 | 第二作**Tavern Keeper**(酒馆经营)即将发布 |

## 三、市场表现

| 指标 | 数据 |
|------|:--:|
| Steam销量 | **100万+** |
| Steam评价 | 95%+(Overwhelmingly Positive) |
| 评价数 | ~39,500+ |
| 定价 | $9.99(PC)/$14.99(Switch)/$4.99(移动) |
| "盗版实验" | 病毒式全球传播——免费营销价值远超广告 |

## 四、设计启示

1. **2人可以做百万销量**——只需一个强大的核心概念+完善的执行
2. **"特技"可以替代营销预算**——盗版实验获得了全球媒体覆盖，价值远超任何付费广告
3. **受启发≠抄袭**——对开罗Game Dev Story的灵感是出发点上，GD Tycoon的深度进化是属于自己的
4. **一个游戏可以养一家工作室11年+**——持续的更新和移植让GD Tycoon从2013年活到了2025年
