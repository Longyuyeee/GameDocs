## 一、游戏评分系统

每款游戏完成后，四家虚拟媒体给出1-10分的评分（满分40）：

| 评分维度 | 对应游戏属性 | 评分来源 |
|------|:--:|------|
| 趣味性评分 | 趣味性(Fun) | 四家媒体 |
| 独创性评分 | 独创性(Creativity) | 四家媒体 |
| 画面评分 | 画面(Graphics) | 四家媒体 |
| 音乐评分 | 音乐(Sound) | 四家媒体 |
| **总评分** | 四项加权总和 | 四家媒体合计 |

### 殿堂级(Hall of Fame)

| 条件 | 效果 |
|------|------|
| 总评分 ≥ 32/40 | 游戏进入殿堂，**解锁续作制作权限** |
| 续作再入殿堂 | 系列可延续(第三代、第四代...) |
| 续作未入殿堂 | 系列终结 |

## 二、GGA 全球游戏大奖

每年12月举办。各奖项条件：

### 优秀奖(单项)

| 奖项 | 条件 |
|------|------|
| **优秀音乐奖** | 音乐属性 ≥ 50 |
| **优秀美术奖** | 画面属性 ≥ 50 |

### 金奖(最佳游戏)

| 条件 | 成功率 |
|------|:--:|
| 一项属性 ≥ 300，其余三项 ≥ 200 | **稳拿** |
| 一项属性 ≥ 200，其余 ≥ 100 | 有概率，不一定 |
| 属性全100以下 | 几乎不可能 |

### 年度最佳游戏(GOTY)

| 条件 | 成功率 |
|------|:--:|
| 四项属性总分 > 800 + 编辑总评分 ≥ 36/40 | 高概率 |
| 总分 600-800 | 看运气 |

### 获奖的连锁效应

```
获奖
    ↓
知名度飙升 → 公司品牌价值提升
    ↓
后续所有游戏首周销量增加 → 更容易再获奖
    ↓
隐藏角色解锁(金奖1次→森野大熊; 金奖3次→猩猩; 金奖5次→Kairo君)
    ↓
自社主机开发→独占平台→更高利润
```

## 三、成就系统(Steam版)

Steam版共有6项成就：

| 成就 | 条件 | 难度 |
|------|------|:--:|
| 游戏开发初体验 | 完成第一款游戏 | ★ |
| 殿堂级作品 | 第一款游戏进入殿堂(评分≥32) | ★★ |
| 金奖得主 | 获得GGA金奖 | ★★★ |
| 主机开发者 | 完成自社硬件研发 | ★★★★ |
| 年度最佳游戏 | 获得GOTY | ★★★★★ |
| 游戏帝国 | 第21年结算时达成行业霸主 | ★★★★★ |

> 6项成就实际上构成了游戏自然流程的里程碑路径——从入门到精通，每项成就解锁大约5-15小时的游戏深度。

## 四、内部评分 vs 外部认可

游戏开发物语的评分体系分为两层：

| 层级 | 系统 | 性质 |
|------|------|------|
| **内部评分** | 四属性数值(趣味/独创/画面/音乐) | 玩家可直接影响(员工培养×类型组合×培训方法) |
| **外部认可** | 编辑评分+奖项+成就 | 内部评分的"结果"——你无法直接控制编辑打分 |
| **终极外部认可** | 殿堂级+GOTY+隐藏角色解锁 | 最稀有、最满意 |

> **设计分析**: 这种"内部/外部"双层评价体系是激励设计的经典模式——玩家直接操控内部评分(通过员工培养)，而外部认可是内部评分的"奖励"(而不是直接目标)。玩家心理上在"培养员工"而非"刷评分"，这比直接给个分数进度条更有沉浸感。

## 五、评分的经济转化

评分如何转化为经济结果：

```
内部评分(四属性)
    ↓
编辑评分(1-10 × 4)
    ↓
总评分
    ↓
    ├→ 销量(直接影响)
    ├→ 能否进入殿堂(续作权限)
    └→ 能否获奖(知名度+角色解锁)
         ↓
    知名度 → 销量(间接影响)
```

## 六、与同品类对比

| 游戏 | 评分机制 | 评分维度 | 评分对经济的影响 |
|------|------|------|------|
| **游戏开发物语** | 四属性→编辑评分→销量 | 趣味/独创/画面/音乐 | 直接影响(销量=评分×知名度×平台) |
| **Game Dev Tycoon** | 多属性加权→评分→销量 | 引擎/玩法/剧情/画面/音乐/关卡 | 直接影响 |
| **双点医院** | 声誉+治愈率+价格→患者满意度 | 多维度 | 间接影响(满意度→口碑→客流) |

> **跨品类对比**: 商业经营类的"评分→收入"与星露谷物语的"品质等级→售价"概念相通——都是"投入资源提升品质→市场给予更高回报"。但游戏开发物语的评分是不可逆的(游戏开发完评分就固定了)，星露谷物语的作物品质可以通过肥料/技能持续改良下一批。
