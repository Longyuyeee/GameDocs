## 一、货币体系

游戏使用单一货币——**金币(G)**，无代币、无氪金系统。资金来源多样但每一分钱的取舍都有策略意义。

### 收入来源

| 收入渠道 | 描述 | 占收入比重（各阶段） |
|------|------|:--:|
| **游戏销售收入** | 核心收入来源，游戏评分×营销×市场体量决定销量 | 初期60% → 中期80% → 后期90%+ |
| **委托开发(合同)** | 接外包项目，固定报酬+研究数据奖励 | 初期40% → 中期10% → 后期<5% |
| **续作加成** | 殿堂级游戏的续作自带前作知名度加成，销量大幅提升 | 中期后成为主要收入引擎 |
| **奖项奖金**(间接) | 获金奖/年度最佳后游戏知名度飙升，销售收入暴增 | 间接驱动 |

### 成本结构

| 成本类型 | 固定/可变 | 描述 |
|------|:--:|------|
| **员工薪资** | 固定/年 | 每年3月(或4月)支付。第1年政府补贴免薪。后期8人团队年薪可达5-10万G |
| **签约金** | 一次性 | 招聘时一次性支付，随招募方式升级而递增(500→65535G) |
| **游戏开发费** | 可变 | 开发期间按周消耗，包含外包费用。体感游戏/网络SLG等大制作消耗更高 |
| **主机授权费** | 一次性/游戏 | 每开发一款第三方平台的游戏需支付一次授权费 |
| **自社硬件研发费** | 一次性 | 自主开发游戏主机(Ultimate Console研发费约2000万G+) |
| **广告费** | 单次/游戏 | $30K(Magazine)到$9,900K(Lunar Writing)，发布前+发布后两次投放 |
| **Gamedex参展费** | 年/次 | $150K(基础展位)到$7,000K(Kairo君嘉宾) |
| **办公室租金/升级** | 一次性升级费 | $600K(第2办公室) / $2,500K(第3办公室) |
| **培训/道具费** | 可变 | 培训费用随员工等级和培训方式递增；道具(如转职书)从商人处购买 |

### 金钱Sinks(大额消费出口)

| 消费出口 | 金额 | 设计意图 |
|------|:--:|------|
| 办公室升级(第2间) | $600K | 第4年门槛——迫使前期积累，区分新手/熟手 |
| 办公室升级(第3间) | $2,500K | 第7-10年门槛——将省下的钱用于扩张还是投资人才 |
| 自社主机研发 | $2,000万+ | 终极大额消费——验证"是否建立了赚钱机器" |
| 好莱坞招募 | $35,000入场+约$20,000/人 | 高端人才的高试用成本 |
| 精英员工签约金 | $45,000-$88,888 | 令狐澈/弗朗索瓦花的签约金足以压垮不健康的现金流 |

> **设计特点**: 金钱sinks不是均匀分布的，而是几个巨大的"门槛"——办公室升级、主机研发、高端人才——将玩家分为不同的发展阶段。这种"阶段门"式消费结构是开罗经营公式的核心要素：短期内的匮乏制造张力，跨越门槛后的丰裕释放满足感。

## 二、定价机制

### 游戏定价（无直接定价权）

玩家**不能直接设定游戏售价**。售价由以下因素间接决定：

| 影响因素 | 机制 |
|------|------|
| **游戏评分** | 编辑评分(1-10分)×四维属性，高评分→高售价→高销量 |
| **主机平台** | 不同平台的装机量决定市场容量，高端主机上的游戏售价更高 |
| **游戏类型/题材组合** | "绝佳!"组合的基础销售额倍率高于普通组合 |
| **知名度(Hype)** | 公司知名度→首周销量→总销量 |

> **设计分析**: 开罗选择了**不让玩家定价**的设计——与夜勤人(Moonlighter)和露塞提娅(Recettear)的"玩家自主定价"形成鲜明对比。这种设计将玩家的注意力从"试探最优价格"转向"提升产品质量"(更高评分→自然获得更高售价)。对于2-3小时的游戏周期，定价博弈的复杂度不是核心卖点，**简化决策**是正确的选择。

### 续作定价加成

殿堂级(Hall of Fame, 评分≥32)游戏的续作自带前作知名度加成，同品质下销售收入比全新IP高。如果续作再次进入殿堂，续作系列可延续；如果续作未进殿堂，系列终结。

## 三、各阶段经济策略

| 阶段 | 年份 | 核心收入 | 核心支出 | 策略要点 |
|------|:--:|------|------|------|
| **起步期** | 第1-2年 | 合同外包为主，低成本游戏辅助 | 招聘初始员工(1200G)，购买转职书(1500G×3) | 益智+黑白棋稳定赚钱；反复创建取消企划刷方针点数 |
| **成长期** | 第3-4年 | 游戏销售开始超越合同 | GBA授权(5500G)、办公室升级(600K→2500K) | 第4年1月搬6人办公室(需18,000G储备)；第4年10月搬8人 |
| **壮大期** | 第5-10年 | 殿堂级续作+金奖游戏成为主要收入 | 好莱坞招募(35,000G入场)、精英转会书 | 冲击金奖(一项属性≥300)；每年5月商人囤转职书 |
| **霸业期** | 第11-20年 | 高档平台(自社主机)上的大制作续作 | 自社主机研发(2000万G+)、顶级员工薪资 | 研发终极主机需6名硬件工程师；年度最佳评分≥38分 |

## 四、经济与经营循环

```
合同外包(获取启动资金+研究数据)
    ↓
研发低成本游戏→销售收入
    ↓
招聘更多员工→培训升级→组建硬件工程师团队
    ↓
开发自社主机→独占平台→更高售价+更高销量
    ↓
续作链(殿堂→续作→再殿堂→再续作)→商业帝国
```

## 五、与同品类对比

| 对比维度 | 游戏开发物语 | 温泉物语(开罗) | Game Dev Tycoon |
|------|:--:|:--:|:--:|
| 定价权 | 玩家无直接定价权 | 有价格设定 | 无直接定价权 |
| 收入来源 | 游戏销售+合同 | 入场费+消费 | 游戏销售 |
| 大额消费 | 办公室升级+主机研发 | 扩建+装修 | 办公室升级+R&D |
| 经济循环周期 | 数周(游戏开发周期) | 数天(游客住宿周期) | 数月(游戏开发周期) |

> **设计特点总结**: 游戏开发物语的经济设计核心是"阶段门"——用几个大额消费门槛将游戏切分为不同的发展阶段。不让玩家定价是一个刻意简化，将注意力集中在产品研发和员工培养上。续作链(Sequel Chain)是经济系统的隐藏引擎——一个殿堂级作品的续作可以"滚雪球"般产生稳定高收入，这比其他开罗游戏更依赖"爆款IP"的逻辑。

---

*文档与 Game Dev Tycoon 的关键差异：Game Dev Tycoon 引入了平台生命周期、行业趋势、技术迭代三层外部变量，而游戏开发物语的外部环境更静态。详见在特色文档《类型×题材组合公式》中的对比分析。*
