## 一、制面——轮盘机制下的手工技艺

### 制面四参数

| 参数 | 作用 | 高值影响 |
|------|------|------|
| **小麦粉(Flour)** | 提升中奖率(Win Rate)和弹性(Firmness) | 更容易在轮盘上命中高数值 |
| **碱水(Kansui)** | 提升弹性和变幅上限(Variability) | 轮盘能出现的最大数值更高 |
| **含水量(Moisture)** | 面条与不同汤底的兼容性 | 影响可搭配的汤底类型 |
| **形状(Shape)** | 直面/卷面+粗细 | 决定兼容的汤底类型和口感 |

### 轮盘机制

制面采用**类似赌博轮盘**的随机数值生成：
- **中奖率**: 轮盘上"中奖格"(高数值)出现的概率——由小麦粉决定
- **变幅**: 轮盘上每个格子的数值范围——由碱水决定
- 中奖率高+变幅大 = 更大概率出现高数值面条
- 中奖率低+变幅大 = "高波动"——可能很低也可能很高

### 四种汤底的黄金面条配比

| 汤底 | 粗细 | 卷曲 | 加水 |
|------|:--:|:--:|:--:|
| 酱油(Shoyu) | 极细 | 微卷/强卷 | 一般/多加水 |
| 味噌(Miso) | 普通→极粗 | 强卷/微卷 | 低加水 |
| 豚骨(Tonkotsu) | 极细/细 | 直/微卷 | 低/一般加水 |
| 盐味(Shio) | 极细/细/普通 | 直 | 普通/多加水 |

## 二、熬汤——Combo驱动的风味叠加

### 材料兼容与Combo

依次将食材投入汤锅中——每个食材与当前汤底的兼容性决定是否触发**Combo**。Combo越多次→汤底风味倍率越高。部分食材组合还会**自动产生高汤(broth)**，进一步强化风味。

### 经济汤底配方

| 配方 | 材料 |
|------|------|
| 基底1 | 生姜+蒜+黑胡椒+辣椒+鸡肉高汤×3 |
| 基底2 | 生姜+蒜+苹果+洋葱+糖+麻油+鸡肉高汤 |
| 基底3 | 苹果+蒜+生姜+蒜+生姜+黑胡椒+砂糖 |

### 四种汤底的关键搭配

| 汤底 | 关键追加食材 |
|------|------|
| 酱油 | +鸡肉高汤 |
| 味噌 | +红味噌/特产味噌 |
| 豚骨 | +豚肉/腔骨 |
| 盐味 | +盐/盐汁 |

## 四、设计特点分析

**制面轮盘=手工艺的"不完美"被游戏化**：在大多数经营游戏中，"产品制造"是确定性的——投入X=产出Y。开罗拉面店的制面轮盘引入了真实手工制作中的"不完美"因素——同一配方两次制作可能产生不同品质的面条。这不是"随机=不公平"——它是"手工艺的固有偶然性"被量化为可接受的波动范围。

**熬汤Combo=食材知识=玩家技能**：不是"选择配方→自动完成"——你需要**知道哪些食材互相兼容**，在正确的顺序投入锅中触发Combo。这种"食材知识"是玩家技能的一部分——经验丰富的玩家能持续触发高倍率Combo。

**面条+汤底=可拆解的产品创新**：与传统经营游戏"一个产品=一个整体"不同，开罗拉面店拆分了面条和汤底——你可以独立优化二者，然后组合出最佳搭配。这种"模块化产品设计"让产品创新有了乘法效应(5种面×4种汤=20种组合)。
