## 1. 开发团队

### 核心团队

符文工房龙之天地（日文名：龍の国 ルーンファクトリー，英文名：Rune Factory: Guardians of Azuma）由 **Marvelous Inc.** 内部开发，是符文工房系列首部采用 **Unreal Engine 5** 的作品。

| 角色 | 人员 | 说明 |
|------|------|------|
| **总制作人** | 佃健一郎 (Kenichiro Tsukuda) | 系列总制作人，主导项目方向 |
| **制作人** | 藤井宙 (Hiroshi Fujii) | 负责具体制作管理 |
| **总监/导演** | 前川司郎 (Shiro Maekawa) | 曾参与《符文工房4豪华版》《符文工房5》 |
| **音乐** | 朝仓纪行 (Noriyuki Asakura) | 负责本作配乐 |

### 开发背景

龙之天地被定位为符文工房系列的"衍生外传"，总监前川司郎在多次采访中将其描述为**"开发《符文工房6》之前的实验场"**。这意味着本作承担着测试新机制、新方向的重任，同时与正统续作《符文工房6》并行开发。

前川司郎在开发《符文工房5》后，意识到系列需要突破以避免创作停滞。他的团队选择通过这部衍生作品尝试创新，而不是在正统续作中冒险。

## 2. 开发历程

### 时间线

| 时间 | 里程碑 |
|------|--------|
| 2023年5月 | MARVELOUS GAME SHOWCASE 2023上首次公布，与《符文工房6》同时宣布 |
| 2023年下半年 | 全面进入开发阶段，首次采用UE5引擎 |
| 2025年3月 | 公布详细游戏内容：和风东国设定、12位候补、村庄建造系统 |
| 2025年5月30日 | 原定发行日（后因配合Switch 2上市延期） |
| 2025年6月5日 | 正式发售（Switch / Switch 2 / PC） |

### 关键创新

本作是系列历史上首次将舞台从西方奇幻转向**东方和风**设定。游戏发生在名为"东国"的幻想世界，这是符文工房系列15年来最大的世界观转变。

核心新机制包括：
- **村庄建造系统**：系列首次引入自由建造元素
- **可切换视角**：传统俯视角与第三人称视角自由切换
- **16位候补结婚对象**：历代最多可恋爱角色
- **全新战斗系统**：结合符文工房特色的实时ARPG战斗
- **东国主题**：和风建筑、服饰、音乐与怪物设计

## 3. 系列发展脉络

符文工房系列始于2006年，作为《牧场物语》的衍生ARPG分支：

| 作品 | 年份 | 平台 | 开发商 |
|------|------|------|--------|
| 符文工房 -新牧场物语- | 2006 | DS | Neverland |
| 符文工房2 | 2008 | DS | Neverland |
| 符文工房3 | 2009 | DS | Neverland |
| 符文工房：海洋 | 2011 | PS3/Wii | Neverland |
| 符文工房4 | 2012 | 3DS | Neverland |
| 符文工房4 豪华版 | 2019 | Switch/PC | HAKAMA |
| 符文工房5 | 2021 | Switch/PC | HAKAMA |
| **符文工房龙之天地** | **2025** | **Switch/PC** | **Marvelous** |
| 符文工房6 | TBA | TBA | TBA |

龙之天地是系列首次脱离Neverland/HAKAMA开发体系、由Marvelous内部工作室直接操刀的作品，也是首次使用UE5开发的系列作品。

## 4. 市场表现与评价

### 销量

截至2025年8月，全球销量已突破 **50万份**，在系列作品中表现稳健。Steam好评率达 **91%**（特别好评）。

### 评价

- **IGN Japan**: 8/10 —— "勇敢的创新方向，和风设定令人耳目一新"
- **Famitsu**: 9/9/8/8 (34/40) —— 进入金殿堂
- **玩家反馈**: 村庄建造系统获得高度评价，但部分老玩家认为战斗系统简化过度

### 设计遗产

龙之天地被定位为"承前启后"之作——它验证了系列在3D开放世界方向的可能性，同时为《符文工房6》提供了宝贵的开发经验和技术积累。UE5引擎的成功应用为系列未来的画面表现力奠定了基础。

## 5. 关键访谈摘要

### 前川司郎总监谈"实验场"定位

> "我们不想在《符文工房6》中冒险尝试太多新东西，因此选择通过这部外传来验证想法。龙之天地中的村庄建造系统和和风设定，如果获得玩家认可，将会影响未来系列的发展方向。"

### 佃健一郎总制作人谈系列未来

> "符文工房系列经历了Neverland破产、HAKAMA重建等多个困难时期。走到今天这一步很不容易。龙之天地是我们展示系列新可能性的作品，而《符文工房6》将是集大成者。"
