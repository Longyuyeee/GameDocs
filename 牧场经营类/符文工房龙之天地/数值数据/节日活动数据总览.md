# 节日活动数据总览 - 符文工房 龙之天地
![龙之天地节日](https://runefactory-gaz.com/img/system/ss_festival.jpg)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

## 一、四季节固定节日

### 1.1 节日日程表

| 节日 | 日期 | 时间 | 地点 | 解锁条件 |
|:---:|:---:|:---:|:---:|:--------:|
| **初诣(First Pilgrimage)** | 春1日 | 10:00 | 春之里神社 | **第二年春季开始** |
| **七夕(Tanabata)** | 夏7日 | 19:00 | 夏之里 | 彼方(Kanata)已解锁 |
| **花火大会(Fireworks Festival)** | 夏20日 | 19:00 | 夏之里海滩 | 祭(Matsuri)已解锁 |
| **灯笼节(Lantern Festival)** | 秋10日 | 19:00 | 秋之里神社 | — |
| **雪祭(Snow Festival)** | 冬24日 | 19:00 | 冬之里 | — |

### 1.2 节日详细说明

#### 初诣(Spring 1)

| 项目 | 内容 |
|:---|:------|
| **类型** | 约会型节日 |
| **参加条件** | 邀请牵绊等级Lv.5以上的恋爱候补 |
| **邀请时间** | 至少**提前3天**邀请 |
| **活动内容** | 与约会对象一同前往春之里神社参拜、许愿 |
| **错过惩罚** | 角色会生气 |
| **备注** | 仅在**第二年之后**出现 |

#### 七夕(Summer 7)

| 项目 | 内容 |
|:---|:------|
| **类型** | 全村活动 |
| **参加条件** | 彼方(Kanata)已解锁 |
| **活动时间** | 19:00-23:00 |
| **活动内容** | 帮助彼方实现村民们的愿望 |
| **奖励** | 成功实现所有愿望后**全角色好感度提升** |
| **其他** | 解锁**浴衣(Yukata)**服装 |

#### 花火大会(Summer 20)

| 项目 | 内容 |
|:---|:------|
| **类型** | 约会型节日 |
| **参加条件** | 祭(Matsuri)已解锁 |
| **特殊机制** | 中午找祭**自定义烟花**(选择颜色和大小) |
| **约会环节** | 19:00在海滩与约会对象一起看烟花 |
| **配色策略** | 不同角色有偏好的颜色/大小组合，匹配喜好获得额外好感 |
| **奖励** | Woolby赠送浴衣，之后可获得所有恋爱候补的浴衣 |

#### 灯笼节(Autumn 10)

| 项目 | 内容 |
|:---|:------|
| **类型** | 约会型节日 |
| **活动内容** | 邀请约会对象在秋之里神社放河灯 |
| **演出** | 温馨的剧情演出 |
| **好感度** | 大幅提升牵绊值 |

#### 雪祭(Winter 24)

| 项目 | 内容 |
|:---|:------|
| **类型** | 约会型节日 |
| **活动内容** | 与约会对象在冬之里共享雪屋(igloo) |
| **演出** | 简短温馨的剧情演出 |
| **好感度** | 提升牵绊值 |

## 二、可预约比赛(Contests)——通过神明神社预约

### 2.1 比赛机制

| 项目 | 说明 |
|:---|:------|
| **预约方式** | 在各村庄的神社处与告示板互动 |
| **可预约日期** | 每月**1、7、13、19、25日**(假日) |
| **活动时间** | 8:00-19:00(报名截止9:00) |
| **建议** | 比赛当天8:00前存档，可反复尝试最优成绩 |

### 2.2 比赛列表

| 比赛名称 | 所属村庄 | 评审 | 内容 | 取胜关键 |
|:-------:|:--------:|:---:|:----|:--------|
| **收获祭(Harvest Contest)** | 春之里 | Sakaki & Ulalaka | 提交作物(按类型/生长时间/等级评分) | 高等级稀有作物 |
| **咖喱大赛(Curry Contest)** | 春之里 | Sakaki & Ulalaka | 提交咖喱料理 | 高等级咖喱 + 独特性 |
| **快手菜大赛(Quick Meal Contest)** | 夏之里 | Hisui & Matsuri | 提交快手菜 | 高等级快手菜 |
| **料理大赛(Cooking Contest)** | 秋之里 | Yachiyo & Kurama | 提交料理(按等级/独特性评分) | 高等级/稀有料理 |
| **淡水钓鱼赛(Freshwater Fishing Frenzy)** | 冬之里 | Zaza | 限时钓最多淡水鱼 | 高钓鱼技能 |
| **公鱼钓赛(Pond Smelt Catch Contest)** | 冬之里 | Zaza | 限时钓最多公鱼 | 高钓鱼技能 |
| **海水鱼大赛(Saltwater Fish Contest)** | 夏之里 | Hisui | 提交最大海水鱼 | 稀有大型海水鱼 |
| **鱿鱼钓赛(Squid Catch Contest)** | 秋之里 | Yachiyo | 提交最大鱿鱼 | 稀有大型鱿鱼 |

### 2.3 比赛奖励

| 名次 | 奖励 | 好感度影响 |
|:---:|:----|:---------:|
| **优胜** | 奖金 + 奖品 | 全村民+100好感度 |
| **準优胜** | 奖金 | 部分好感度 |
| **参加奖** | 少量奖金 | — |

> **注**：提交的作物/料理**不予返还**，请确保提交前已备份或提交可接受的物品。

## 三、龙之神社(Dragon Shrine)活动

### 3.1 龙之神社功能

| 功能 | 说明 |
|:---|:------|
| **存档** | 在龙之雕像前进行牵绊之舞激活保存点 |
| **快速旅行** | 激活后可传送至各村庄的龙之雕像 |
| **剧情推进** | "休息时间(Time to Rest)"任务引导在神社睡觉推进主线 |
| **梦见** | 在神社睡觉触发特殊剧情(获得鼓、遇见Ulalaka等) |
| **怪物驯服** | 通过牵绊之舞在雕像处驯服怪物 |

### 3.2 主线相关神社事件

| 事件 | 触发条件 | 内容 |
|:---:|:--------:|:----|
| 维修龙之神社 | 初期主线 | 收集材料请拓海(Takumi)修复神社 |
| 获得鼓 | 第一次在神社睡觉 | 梦见Ulalaka并获得神圣之鼓 |
| 神乐神社(天空) | 中期主线 | 乘白龙前往四座神乐神社——战斗→跳舞→解除屏障 |
| 黑龙BOSS战 | 四座神乐神社激活后 | 解除屏障后触发黑龙战斗 |

### 3.3 神乐神社四座

| 神社 | 位置 | 活动内容 |
|:---:|:----:|:--------|
| 第一神乐神社 | 天空 | 进入→战斗→中心跳舞→激活 |
| 第二神乐神社 | 天空 | 同上 |
| 第三神乐神社 | 天空 | 同上 |
| 第四神乐神社 | 天空 | 同上→全部激活后屏障解除 |

## 四、故乡祭(Hometown Festival Events)

### 4.1 村庄等级相关活动

| 村庄等级 | 解锁活动 |
|:-------:|:--------|
| Lv.1-2 | 基础节日、收获祭 |
| Lv.3 | 新商店开张庆典 |
| Lv.4 | 装饰品解锁活动 |
| Lv.5 | 黄金种子解锁、大型畜棚建设 |
| Lv.6+ | 高级活动开放 |

### 4.2 各村特有节日

| 村庄 | 特色活动 | 周期 |
|:---:|:--------|:---:|
| **春之里** | 赏樱会、收获祭、咖喱大赛 | 每月可选 |
| **夏之里** | 七夕、花火大会、海水鱼大赛 | 固定+可选 |
| **秋之里** | 神轿祭(Mikoshi Melee)、灯笼节、料理大赛 | 主线+可选 |
| **冬之里** | 雪祭、淡水钓鱼赛、公鱼钓赛 | 固定+可选 |

### 4.3 节日参与策略

| 阶段 | 推荐策略 |
|:---:|:--------|
| **前期** | 参加所有可选比赛积累资金和好感度 |
| **中期** | 专注与恋爱候补的约会节日(提前3天邀请) |
| **后期** | 追求全角色好感度最大化、全节日制霸 |
| **比赛准备** | 比赛当天8:00前存档，可SL大法获取优胜 |
