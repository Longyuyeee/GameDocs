# 剧情与故事系统 - 符文工房 龙之天地
![白龙](https://runefactory-gaz.com/img/character/chara_dragon.png)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

## 1. 主线剧情结构

### 1.1 整体架构: 龙星陨落,大地舞者的使命

| 幕 | 内容 | 占比 |
|:--:|------|:--:|
| **开幕** | "龙星陨落"灾难后,东国大地荒芜。主角作为"大地舞者(Earth Dancer)"与白龙签订契约,被赋予拯救大地的使命 | ~5% |
| **第一幕: 春之觉醒** | 收复春之里,恢复春秋村庄,学习基础农耕和神器使用 | ~20% |
| **第二幕: 四季巡回** | 逐一恢复夏之里/秋之里/冬之里,每收复一个村庄解锁对应季节的资源和村民 | ~35% |
| **第三幕: 龙星之谜** | 揭示龙星陨落的真相,面对真正的敌人 | ~25% |
| **终局** | 四个季节村庄全部繁荣发展,东国恢复生机,白龙的真正愿望被实现 | ~15% |

### 1.2 叙事驱动方式

| 方式 | 说明 |
|------|------|
| **村庄收复驱动** | 每收复一个季节村庄=推进主线的一个章节 |
| **故乡点数驱动** | 完成村庄任务,累积故乡点数→提升村庄等级→解锁新剧情节点 |
| **主线任务锁** | 部分区域/迷宫需要主线达到特定阶段解锁 |
| **自由探索** | 在主线之外完全自由,可以放缓主线专注经营和社交 |

## 3. 世界观设定

### 3.1 东国(Azuma)

- **和风东方幻想**世界观——不是中世纪欧洲的符文工房,而是日式东方
- **四季节不是时间轮换,而是空间分布**: 春夏秋冬四个"里"(村庄)各自保持永久性的不同季节
- **白龙(White Dragon)** = 这片土地的主神/守护者
- **龙星陨落** = 灾难事件——不是"邪恶Boss做的",而是一个更深层的悲剧

### 3.2 核心冲突

| 层面 | 冲突 |
|------|------|
| **世界层** | 龙星陨落后大地的荒芜 vs 复苏的需要 |
| **叙事层** | 四村庄的资源争夺→需要主角作为协调者 |
| **个人层** | 主角作为"大地舞者"=被选中的人——但"被选中"意味着责任,不是特权 |

### 3.3 四村庄的文化差异

| 村庄 | 文化色彩 | 价值观 |
|------|---------|------|
| 春之里 | 樱花/新生/希望 | 传统/温柔/耐心 |
| 夏之里 | 活力/商业/热闹 | 进取/财富/开放 |
| 秋之里 | 丰收/金黄/勤劳 | 农耕/汗水/脚踏实地 |
| 冬之里 | 寂静/白色/坚韧 | 内省/耐力/在荒芜中生存 |

四村庄之间的关系=不同生活方式之间的张力。

## 4. NPC个人剧情

### 4.1 牵绊等级(Bonds Level)

| 等级 | 解锁 |
|:--:|------|
| Lv1 | 基础对话,**可邀请组队**(重大改变——好感度1就能组队!) |
| Lv3 | 个人事件① |
| Lv5 | 个人事件② |
| Lv7 | **可告白** |
| Lv10 | **可求婚** |

### 4.2 世界转移法(World Transfer)——龙之天地最独特的叙事设计

这是一个革命性的恋爱系统设计:

- **单存档多条世界线**: 在一个存档中,你可以"转移"到不同的世界线,体验不同候补的恋爱路线
- **不强制重开**: 你想体验另一个候补的剧情?不需要删档重来。转移到那条世界线即可
- **最多30条世界线**: 理论上有极大的叙事分支空间
- **叙事意义**: 让玩家不需要在"我想看这个故事"和"我不想失去我的农场"之间做选择——这在叙事驱动型游戏中是开创性的
- **但保留核心进度**: 转移后农场/财产/某些进度保留,只有关系状态重置到该世界线

### 4.3 16候补=16种"如何与东国共存"的方式

龙之天地有系列最多的16名候补(含DLC 2名),加上**同性婚姻**(系列首次):

- 各村庄的候补各自体现其村庄的价值观
- 春之里的候补=传统/温柔型故事
- 夏之里的候补=商业/进取型故事
- 秋之里的候补=踏实/勤劳型故事
- 冬之里的候补=坚韧/内省型故事
- **共同主题**: "在这片被荒芜的大地上,我想和谁一起重建?"

## 5. 龙神像与牵绊之舞

- 散落各地的**龙神像(Dragon Shrine)**是存档/传送点
- 需要主角跳**"牵绊之舞(Bonds Dance)"**激活
- 牵绊之舞不只是机械操作——在叙事上,它意味着主角与这片土地建立**精神连接**
- 每一个激活的龙神像=大地被恢复了一个节点

## 6. 设计特点与启示

1. **空间化季节=取消"等待"**: 传统牧场游戏"冬天不能种地"=4/1时间闲置。龙之天地的四季节村庄=你想种春季作物就去春之里,任何时候。这是对牧场游戏"季节等待"机制的根本性反思
2. **村庄经营=世界建设权交还玩家**: 你不仅是农夫,也是城市规划者——道路/建筑/田地由你布局。叙事上这意味着"你不只是住在这里,你在创造这里"
3. **牵绊之舞=仪式感**: 不只是"点一下激活",而是一个舞蹈——给机械的游戏操作附加了叙事情感
4. **世界转移=尊重玩家的时间**: 多周目玩家的核心痛点是"再刷一遍农场"——世界转移解决了这个矛盾,让你专注于体验新故事
5. **Lv1可组队=角色互动前置**: 传统牧场需要好几个心才能"一起行动",这里1级就能带进队伍——角色和你的互动从第一天就开始
6. **村民满意度=叙事后效**: 食物不足/住处不够→村民离开。你的经营成果直接影响"镇上还有谁"——这是比"村庄等级提升"更直接的世界反馈
