## 一、双货币体系

本作采用 **金币(Gold)** 与 **故乡点数(Hometown Points)** 双轨并行货币系统，二者功能互补，共同驱动村庄发展与玩家成长。

### 1.1 金币(Gold) —— 流通货币

金币是日常交易的主要货币，用于购买种子、素材、装备、料理以及建设村庄建筑。

| 金币来源 | 说明 |
|---------|------|
| 作物出售 | 将收获的作物直接出售获得金币 |
| 店铺经营 | 在夏之里开设商店，由村民代为经营，每日自动产生收入 |
| 委托任务(Quests) | 完成村庄任务、居民委托获得金币报酬 |
| 怪物掉落 | 击败怪物后获得金币或可出售的掉落物 |
| 钓鱼出售 | 钓到的鱼类可直接出售 |
| 采矿/伐木 | 采集的矿石和木材可作为素材出售 |
| 加工品出售 | 料理、锻造装备、调合药品等加工品可出售获利 |

### 1.2 故乡点数(Hometown Points) —— 发展货币

故乡点数是衡量村庄繁荣度的特殊货币/经验值，不可用于购买物品，而是用于提升 **村庄等级(Village Level)**。

| 故乡点数获取方式 | 说明 |
|----------------|------|
| 完成村庄任务(Village Missions) | 主要来源，每个任务奖励固定点数 |
| 提升村庄繁荣度 | 建筑物、装饰物提升繁荣度，间接影响点数获取效率 |
| 村民满意度 | 满意度高时村民会自发贡献更多点数 |
| 主线剧情推进 | 关键剧情节点奖励大量故乡点数 |

| 村庄等级 | 解锁内容 |
|---------|---------|
| Lv1 | 基础建筑解锁，初始开发区域 |
| Lv2 | 高级建筑解锁，新开发区域 |
| Lv3 | 水车、风车等功能性建筑 |
| Lv4-Lv6 | 茶屋、花车、浴场等，人口上限提升 |
| Lv7-Lv10 | 异国风格建筑、宝物库、天堂塔等终极建筑 |

> 四个季节村庄各自拥有独立的村庄等级和故乡点数系统，分别提升。

## 二、村庄经济特色

每个季节村庄因其固定季节不同，拥有不同的经济优势。

### 2.1 春之里(Spring Village)

| 项目 | 内容 |
|------|------|
| 经济强项 | 春季作物种植、观光 |
| 推荐产业 | 草莓等春季高价作物批量种植 |
| 特色建筑 | 初期工坊、简易房屋 |
| 经济定位 | 农业基础，前期主要收入来源 |

### 2.2 夏之里(Summer Village)

| 项目 | 内容 |
|------|------|
| 经济强项 | **店铺经营/商业(Commercialization)** |
| 推荐产业 | 开设商店、餐馆，由村民代为经营 |
| 特色建筑 | 茶屋(Teahouse)、居酒屋(Izakaya)、商店 |
| 经济定位 | **本作商业核心区域**，被动收入主要来源 |

**店铺经营系统详解：**
- 在夏之里建造商店建筑后，可指派拥有 **交易/经商特质** 的村民经营
- 店铺每日自动产生金币收入，无需玩家亲自管理
- 收入量受以下因素影响：
  - 村民的 **交易技能等级**
  - 店铺周边的 **装饰品加成(Trading bonus)**
  - 村庄等级
  - 村民满意度

### 2.3 秋之里(Autumn Village)

| 项目 | 内容 |
|------|------|
| 经济强项 | **大规模农业 / 水稻种植** |
| 推荐产业 | 水稻、糯米等水田作物批量种植 |
| 特色建筑 | 水田(Rice Paddy)、水路(Waterway)、水车(Watermill) |
| 经济定位 | **农业生产核心区域**，高产量带来高收入 |

**产量差异示例：**
| 作物 | 秋之里产量 | 冬之里产量 |
|------|-----------|-----------|
| 水稻(Rice) | ~24-25单位 | ~16-17单位 |
| 糯米(Sticky Rice) | ~24-25单位 | ~16-17单位 |

### 2.4 冬之里(Winter Village)

| 项目 | 内容 |
|------|------|
| 经济强项 | **钓鱼 / 矿业** |
| 推荐产业 | 淡水鱼大量捕捞、矿石采集 |
| 特色建筑 | 水车(需与水路相邻) |
| 经济定位 | 渔业和矿业中心，稀有资源产地 |

## 三、收入与支出分析

### 3.1 主要收入途径对比

| 收入方式 | 前期(第1-10小时) | 中期(第10-30小时) | 后期(30小时+) | 稳定性 |
|---------|-----------------|------------------|--------------|-------|
| 作物种植 | 高(主要收入) | 中 | 低(转向加工) | 周期波动 |
| 店铺经营 | 不可用 | 中 | 高(被动收入) | 稳定 |
| 委托任务 | 中 | 中 | 低 | 一次性 |
| 怪物掉落 | 低 | 中 | 中 | 不稳定 |
| 钓鱼 | 低 | 中 | 中 | 依赖天气 |
| 加工品出售 | 低 | 高 | 高 | 稳定 |

### 3.2 推荐高收益作物(各村庄最佳)

| 村庄 | 推荐作物 | 售价 | 备注 |
|------|---------|------|------|
| 春之里 | 草莓(Strawberry) | 50G | 每3天再生，持续收获 |
| 夏之里 | 粉红蜜瓜(Pink Melon) | 67G | 巨大化后售价405G |
| 秋之里 | 糯米/水稻 | 62G/60G | 每块地最高可达~1440G |
| 冬之里 | 柑橘树(Mandarin) | 55G | 树木无需浇水，5-6天再生 |
| 全域 | 金色作物(Golden Crops) | 200G | 巨大化后可达1000G |

### 3.3 主要支出项目

| 支出类型 | 金额范围 | 说明 |
|---------|---------|------|
| 建筑建造 | 500G - 5000G+ | 村庄建筑、装饰物建造费用 |
| 工具升级 | 350G - 2000G+ | 锻造升级武器、工具 |
| 素材购买 | 950G(铁) - 3800G(金) | 铁匠铺购买矿石素材 |
| 种子购买 | 10G - 200G | 不同作物种子价格不同 |
| 食谱面包 | 100G - 500G | 传统RF系列中用于学习配方(本作部分保留) |
| 村民工资 | 每日自动扣除 | 指派村民工作会消耗资金 |

## 四、商业系统(Commercialization)

### 4.1 店铺经营机制

本作取消了传统的 **出货箱(Shipping Bin)** 系统，改为更加主动的商业经营模式：

- 作物和加工品可直接出售给店铺，或通过村民经营的商店自动贩卖
- 4个季节村庄各有不同的商品需求偏好
- 夏之里是商业核心区域，建造商店后可指派村民经营

> 注：**本作取消了系列传统的出货箱系统**，转为通过村民店铺经营和直接交易完成商品流通，更接近《符文工厂：风之集市》的商业模式但更加简化。

### 4.2 村民商业特质

| 特质 | 效果 |
|------|------|
| 经商(Trading) | 提升店铺收入 |
| 烹饪(Cooking) | 可经营餐馆类建筑 |
| 锻造(Forging) | 可经营铁匠铺 |
| 调合(Chemistry) | 可经营药铺 |

## 五、村庄满意度与经济

### 5.1 满意度影响

村庄满意度(幸福度)对经济有直接影响：

| 满意度状态 | 经济影响 |
|-----------|---------|
| 高满意度 | 店铺收入提升、村民自发贡献故乡点数 |
| 中等满意度 | 正常运转 |
| 低满意度 | 店铺收入下降、村民可能离开村庄 |

### 5.2 影响满意度的因素

| 因素 | 影响程度 |
|------|---------|
| 建筑数量与种类 | 高 |
| 装饰品数量(风景评分) | 高 |
| 村民工资支付情况 | 中(资金不足时严重) |
| 村庄等级 | 中 |
| 玩家与村民的互动 | 低-中 |

## 六、经济策略建议

### 前期(1-10小时)
1. 集中发展 **春之里**，种植草莓等快速回本作物
2. 完成村庄任务积累故乡点数，尽快提升村庄等级
3. 收集基础素材(木材、石材)减少开支

### 中期(10-30小时)
1. 解锁 **夏之里店铺系统**，建立被动收入来源
2. 在 **秋之里** 开展水稻种植，利用水田系统提高产量
3. 指派村民负责采矿/伐木，减少自己采集时间

### 后期(30小时+)
1. 发展全4个村庄经济，形成产业分工
2. 通过圣剑升级种子等级，最大化作物价值
3. 挑战终末之森获取顶级配方，制造高价值装备出售
