# 畜牧系统 - 米斯特里亚的田野(Fields of Mistria)

> 游戏版本: Fields of Mistria (PC, Early Access 2024-至今)
> 数据来源: fieldsofmistria.com, GameRant

## 1. 可饲养动物

| 动物 | 产出 | 装饰 |
|------|------|:--:|
| 鸡(Chicken) | 蛋 | ✓(玩具+装饰品) |
| 牛(Cow) | 牛奶 | ✓ |
| 羊(Sheep) | 羊毛 | ✓ |
| 鸭(Duck) | 鸭蛋 | ✓ |
| 羊驼(Alpaca) | 羊驼毛 | ✓ |
| 其他(EA待扩展) | — | — |

## 2. 动物特色

| 特色 | 说明 |
|------|------|
| **动物装饰品** | 可给动物穿衣服/戴帽子/配玩具——系列首创 |
| **稀有颜色变体** | 繁殖→解锁稀有颜色(类似Roots of Pacha) |
| 金品质产品 | 高好感度→金蛋/金牛奶/金奶酪等 |

## 3. 加工品

| 原料 | 加工品 | 制作站 |
|------|------|------|
| 蛋 | 蛋黄酱 | 蛋黄酱机 |
| 牛奶 | 奶酪/黄油 | 奶酪机/黄油机 |
| 羊毛 | 毛线→布料 | 纺车→织布机 |

## 4. 设计特点

1. **动物装饰品=最强QoL**: 给动物穿衣戴帽——视觉效果+情感连接
2. **繁殖=稀有颜色变体**: 类似Roots of Pacha的遗传育种系统
3. **8种动物(EA)**: 已超星露谷基础阵容
