## 1. 开发团队

### Pixel Sprout Studios

Pixel Sprout Studios 是一家独立游戏开发工作室，由两位联合创始人 **Aiden** 和 **Claire** 于 2019 年在洛杉矶创立。工作室从仅2-3人的微型团队起步，随着《日光之城》的开发推进逐步扩大，到2023年1.0正式版发布时已扩展至约 **13名全职成员**。

团队核心优势在于美术表现——在Kickstarter众筹中，美术是资金分配的最大部分，这直接体现在游戏精致生动的像素艺术风格上。Pixel Sprout 后续还发布了另一款名为《Sun Down》的作品。

### 团队分工

| 角色 | 说明 |
|------|------|
| **制作人/策划** | Aiden —— 负责整体项目管理和游戏设计 |
| **美术总监** | Claire —— 主导像素艺术风格和视觉方向 |
| **程序团队** | 约5-6人，负责Unity引擎实现 |
| **美术团队** | 约4-5人，负责角色、场景、UI等所有视觉资产 |
| **音效/音乐** | 外包合作 |

## 2. 开发历程

### 概念与起源

《日光之城》的灵感来源于《星露谷物语》的核心循环，但开发团队希望创造一个更偏向RPG的奇幻世界——一个包含魔法、龙族、多民族种族的农场模拟游戏。与《星露谷物语》的乡村田园风格不同，Sun Haven选择了色彩鲜明的高幻想路线。

### 众筹阶段

2020年10月，Pixel Sprout Studios在Kickstarter上发起了《日光之城》的众筹活动：

| 众筹数据 | 数值 |
|----------|------|
| **目标金额** | $45,000 USD |
| **实际筹款** | **$85,000+ USD** (191%) |
| **支持者** | 约1,900人 |
| **活动日期** | 2020年10月 |

这次成功的众筹为团队提供了启动资金，也验证了市场对"奇幻农场模拟RPG"这一概念的需求。

### 开发时间线

| 时间 | 里程碑 |
|------|--------|
| 2019年 | Pixel Sprout Studios成立，项目概念化 |
| 2020年10月 | Kickstarter众筹成功（$85K） |
| 2021年6月25日 | Steam Early Access 开启 |
| 2021-2022年 | EA期间持续更新，增加Nel'Vari和Withergate两大区域 |
| 2023年3月11日 | **1.0正式版发布**（Windows） |
| 2024年中 | v1.4大型更新：新增水下区域"Brinestone Deeps"和新候补角色 |
| 2024年末 | Nintendo Switch版发售 |

### Early Access 进化

在近2年的Early Access期间，游戏经历了显著的内容扩充：
- **首发EA**：仅包含Sun Haven主城镇区域
- **后续更新**：逐步加入精灵森林 Nel'Vari、亡灵都市 Withergate
- **最终版**：3座完整城镇、超过40位NPC、8个可玩种族、大量魔法和锻造系统

## 3. 设计特色

### 与星露谷物语的区别定位

| 维度 | 星露谷物语 | 日光之城 |
|------|-----------|---------|
| 题材 | 乡村田园 | 高幻想RPG |
| 种族 | 人类 | 8个可玩种族(人类/精灵/恶魔/元素等) |
| 战斗 | 简单 | 完整技能树+魔法系统 |
| 地图 | 1个镇 | 3个独立城镇区域 |
| 联机 | 4人 | **8人联机** |

### 多人在线

8人联机是《日光之城》的核心差异化优势之一。在同类农场模拟游戏中，8人联机的能力使其在社交体验上远超大多数竞品（通常限制在4人）。

## 4. 市场表现与评价

### 销量估算

根据Steam第三方数据平台估计，《日光之城》在Steam上的销量估计在 **80万至180万份** 之间。考虑到其独立游戏的定位和Pixel Sprout Studios是自发行，这个数字意味着相当可观的商业成功。

### 媒体评价

- **Steam好评率**：86%（特别好评）
- **2023年度最佳白金级**：获得多个独立游戏奖项提名
- 评论普遍称赞其美术风格、内容量和多人体验
- 被一些媒体称为"《星露谷物语》之后最出色的农场模拟RPG之一"

### 社区反响

游戏拥有活跃的Mod社区，Steam创意工坊中有大量自定义物品、角色和地图Mod。开发团队对社区反馈较为积极，EA期间根据玩家建议进行了多次平衡性调整和功能追加。

## 5. 团队理念与未来

Pixel Sprout Studios 一直保持独立运营、不依附于大型发行商。团队在访谈中强调：

> "我们相信好的游戏来自于对玩家体验的真诚关注。Sun Haven 是我们对农场模拟类型的热爱之作，能在这个类型中加入我们喜欢的奇幻元素。"

工作室目前仍在持续更新《日光之城》的内容，同时也在探索下一款作品的立项方向。
