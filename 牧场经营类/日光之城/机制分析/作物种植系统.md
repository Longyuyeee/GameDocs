# 作物种植系统 - 日光之城(Sun Haven)

> 游戏版本: Sun Haven v1.4+ (PC/Switch)
> 数据来源: sunhaven.wiki.gg, IGN Sun Haven Crop Guide, cozygamingpanda.com

## 1. 概述

Sun Haven 拥有**三套独立种植体系**，对应三个主城区域。每个区域的作物只能在对应农场上种植，种子也只能在对应商店购买。Nel'Vari 和 Withergate 区域**无季节限制**，作物全年可种。

## 3. 生长周期模型

- **单次收获作物**: 生长→成熟→收获，一次种植一次收获
- **多次收获作物**: 生长→成熟→首次收获→间隔N天→再次收获，可在一个季节内多次产出
- **跨季节规则**: Sun Haven区作物跨季节会枯萎(除标注双季的如辣椒)，精灵和恶魔区全年可种
- **枯萎条件**: 季节更替时非当季作物枯萎；季节性灾害(火/荆棘/冰)不直接导致枯萎但阻碍收获

## 4. 品质/星级机制

| 品质 | 售价倍率 | 出现条件 |
|:-----|:------:|---------|
| 普通(Common) | ×1.0 | 基础 |
| 稀有(Rare) | ×1.1 | 随机/技能加成 |
| 史诗(Epic) | ×1.2 | 随机/技能加成/肥料 |
| 传说(Legendary) | ×1.3 | 最高品质/技能最大化 |

- 品质通过**随机概率**出现，受技能树和肥料影响
- "传播(Propagation)"技能(Farming T6): 4/6/8%概率额外收获一个作物
- "肥沃土地(Fertile Land)"技能(Farming T8): 施肥作物4/8/12%概率额外收获

## 5. 肥料/加成系统

| 肥料类型 | 效果 | 来源 |
|---------|------|------|
| 普通肥料 | 提高品质概率 | 制作/购买 |
| 高级肥料 | 更高品质加成 | 制作(Alchemy Table) |
| 保湿肥料 | 概率保持浇水状态过夜 | 技能"保水土壤(Retaining Soil)" |
| 生长加速肥料 | 缩短生长天数 | 特定配方制作 |

- "保水土壤(Retaining Soil)"(Farming T3): 5/10/15%概率作物过夜保持浇水状态

## 6. 果树与特殊作物

| 类型 | 特点 |
|------|------|
| 果树(Apple/Peach/Orange/Mango等) | 长期投资，反复收获，无需重复种植 |
| 茶树(Tea Leaves) | 4天生长，每4天收获2片茶叶，冬季可种 |
| 啤酒花(Hops) | 12天生长，产出4份，用于酿造 |
| 大麦(Barley) | 6天生长，产出4份，用于酿造 |
| 棉花(Cotton) | 10天生长+3天复收，产出布料原料 |
| 葡萄(Grapes) | **全季节通用**，1天生长，15g/天净利润 |

## 7. 自动化设施

Sun Haven **没有洒水器(Sprinkler)**，但有替代方案：

| 设施/技能 | 效果 |
|----------|------|
| 雨云(Rain Cloud)法术 | Farming T7主动技能，召唤移动云朵浇灌5格宽×10格长区域，消耗60/50/40法力 |
| 地震(Earthquake)法术 | Farming T5主动技能，瞬间翻耕5×5区域，消耗50/40/30法力 |
| 保水土壤技能 | Farming T3，概率跳过浇水 |
| 自动喂食器(Auto-Feeder) | 动物自动喂食 |
| 自动收集器(Auto-Collector) | 1,000g，自动收集动物产品 |

## 8. 种子获取途径

| 途径 | 说明 |
|------|------|
| 普通商店(General Store) | Sun Haven基础种子 |
| 节日种子商 | 比平日便宜约10%，可买反季种子 |
| 精灵商店 | Nel'Vari精灵作物种子(用法力球) |
| 恶魔商店 | Withergate恶魔作物种子(用票券) |
| 种子制造机(Seed Maker) | Farming T4解锁，将作物转化为种子(+0/1/2额外种子) |
| 技能赠送 | "花田(Flower Farm)"技能每周五邮寄10/15/20蓝玫瑰种子 |

## 9. 季节性灾害(可选)

| 季节 | 灾害 | 处理方式 |
|:----:|------|---------|
| 春 | 幼龙(Baby Dragons)害虫 | 使用镰刀或Vacuumulus法术清除 |
| 夏 | 作物着火 | 用水壶或雨云法术扑灭 |
| 秋 | 荆棘缠绕 | 用斧头清除 |
| 冬 | 冰封 | 用镐子打破 |

- **所有季节性灾害可在设置中关闭**
- 精灵农场(Nel'Vari)和恶魔农场(Withergate)**无季节灾害**
