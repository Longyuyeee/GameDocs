# 战斗探索系统 - 日光之城(Sun Haven)

> 游戏版本: Sun Haven v1.4+ (PC/Switch)
> 数据来源: sunhaven.wiki.gg, Fandom Sun Haven Wiki, IGN, Nintendo eShop

## 1. 战斗定位

战斗在Sun Haven中属于**重要但非强制性系统**。主线Boss(Dynus)可用战斗或非战斗(祭坛供奉)两种方式通关。战斗系统有独立的技能树和等级，与经营系统有资源联动(怪物掉落材料可用于制作装备/烹饪)。

## 2. 战斗方式

### 2.1 四种战斗风格

| 风格 | 武器类型 | 特点 |
|------|---------|------|
| **近战** | 剑(Sword) | 中距离攻击，攻速适中，最高基础伤害 |
| **远程** | 弩(Crossbow) | 无需弹药，安全距离输出 |
| **魔法** | 法杖(Staff) + 法术(Spells) | 消耗法力，AOE/控制/爆发 |
| **混合** | 任意搭配 | 可同时装备任意战斗方式 |

### 3.2 战斗试炼地牢(Combat Trial Dungeon)
- **31层**波次挑战
- 每5层奖励宝箱钥匙→开启获得**永久战斗属性提升**
- Steam成就: 通关5层/15层/30层
- 击败King Slimius XVII后解锁

### 3.3 Boss竞技场(Boss Arena)
- 位置: Lynn家东侧(矿洞前)
- 开放时间: **每周二和周六**
- 与Raimi对话选择Boss难度，Boss随机轮换
- 每天只能挑战**1次**
- Boss掉落稀有材料和专属物品

## 6. 战斗与经营的联动

| 联动方式 | 说明 |
|---------|------|
| 怪物掉落→装备制作 | 怪物材料(粘液/羽毛/布料)制作防具 |
| 怪物掉落→烹饪 | 部分怪物掉落是烹饪原料 |
| 矿洞怪物→宝石掉落 | 宝石直接出售或制作珠宝 |
| 战斗技能→金币收益 | "赏金猎人(Bounty Hunter)"技能: 15/30/45%概率击杀怪物获得3倍金币 |
| 战斗试炼地牢→永久属性 | 每5层宝箱提供永久战斗属性提升 |
| Dynus祭坛→免战斗通关 | 完成7+祭坛供奉即可非暴力通关 |

## 7. 设计特点

1. **四种战斗风格自由切换**: 剑/弩/法杖/法术混合，非职业锁定
2. **非强制性**: 主线Boss可选择非战斗方式通关，经营派也能推进剧情
3. **31层波次地牢**: Roguelite式的战斗试炼，永久奖励驱动重复挑战
4. **Boss竞技场周期性**: 每周二/六开放，随机Boss轮换，维持新鲜感
5. **法术跨技能树分布**: 火球/闪电链在Combat，Vacuumulus在Mining，灵体斧在Exploration，地震在Farming——鼓励全面发展
6. **季节性世界Boss**: 农场春季出现Weedil/Krusty等，将战斗与季节经营融合
7. **平台动作元素**: 空跳(Air Skip)+高跳(High Jump)增加纵向维度，矿洞和世界探索更具动作性
