## 五、好感度影响因素

| 行为 | 好感度变化 | 说明 |
|------|:--------:|------|
| 每日喂食 | + | 必须，不喂食降好感度和健康值 |
| 抚摸 | + | 非必须但不抚摸不衰减 |
| 不喂食 | - | 每日遗漏降好感度+健康值 |
| 健康值过低 | 停止产出 | 恢复喂食后逐渐恢复 |
