# 时间季节与节日系统 - 橄榄镇(Story of Seasons: Pioneers of Olive Town)

> 游戏版本: Story of Seasons: Pioneers of Olive Town (Switch/PC/PS4, 2021年)
> 数据来源: Harvest Moon Wiki, GameWith, TheGamer

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 每天 | 6AM - 2AM |
| 每季节 | 28天 |
| 每年 | 4季×28天=112天 |
| 时间流速 | 标准(不可调) |
| 睡觉 | 2AM强制 |

## 2. 季节

| 季节 | 特色作物 | 天气影响 |
|:--:|------|------|
| 春 | 芜菁/土豆/黄瓜/卷心菜/花椰菜 | 偶尔台风 |
| 夏 | 洋葱/番茄/玉米/西瓜/菠萝 | 台风 |
| 秋 | 菠菜/红薯/茄子/胡萝卜/青椒 | 台风 |
| 冬 | 冬季花卉/部分耐寒作物 | 雪 |

- 冬季**不停止种植**——有专属冬季作物

## 3. 节日系统

| 季节 | 节日 | 活动 |
|:--:|------|------|
| 春 | 蛋节(Egg Festival) | 找蛋比赛 |
| 春 | 宠物节(Pet Festival) | 宠物评比 |
| 夏 | 赛马 | 赌马/赛马 |
| 夏 | 烟花大会(Fireworks) | 与恋爱对象看烟花 |
| 秋 | 丰收节 | 作物比赛 |
| 秋 | 万圣节 | 变装/鬼怪主题 |
| 冬 | 钓鱼大赛 | 钓鱼比赛 |
| 冬 | 星夜节 | 与恋人看星星 |
| 冬 | 新年 | 年终庆祝 |

## 4. 体力系统

| 要素 | 说明 |
|------|------|
| 体力(Stamina) | 工具使用消耗 |
| 食物恢复 | 吃料理→恢复体力 |
| 温泉 | 浸泡恢复 |
| 体力上限 | 通过食物永久提升 |

## 5. 设计特点

1. **冬季可种植**: 不像BTN/矿石镇完全停摆——有专属冬季作物
2. **拍照=新维度**: 支线拍照收集博物馆奖励——利用时间的额外活动
3. **精灵=省时机制**: 派出精灵收集资源→节省每日采集时间
