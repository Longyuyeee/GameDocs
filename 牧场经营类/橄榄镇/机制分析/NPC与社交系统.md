## 二、社交机制

### 2.1 好感度系统
- 心色等级(与传统一致)
- 送礼+对话+节日互动→提升好感度
- 5♥→可赠送表白信物→恋爱
- 10♥→可求婚(需房屋升级+订婚戒指)

### 2.2 恋爱流程
```
送礼+对话→好感度提升
  → 触发心色事件(特定时间/地点)
  → 5♥→赠送表白信物(Confession Pendant)
  → 恋爱关系(可约会)
  → 10♥→赠送订婚戒指→求婚
  → 3天后婚礼
```

## 三、居民与博物馆

- 大量NPC居民(商店/功能角色/普通村民)
- 博物馆: 捐赠文物/鱼/照片→奖励
- 拍照系统: 特定地点拍照→博物馆收藏→奖励

## 五、设计特点

1. **全3D角色+事件**: 全3D过场动画(系列首次)——成本更高但沉浸感更强
2. **制造机功能NPC**: 城镇NPC不仅是社交对象——部分有专属功能
3. **博物馆=收集驱动**: 捐赠系统提供长尾收集目标
4. **DLC扩展候补**: 通过付费DLC增加可攻略角色——系列商业化探索
5. **角色深度争议**: 部分玩家认为候补性格和心色事件不如Trio of Towns丰富——3D化的代价
