# 团队与开发历程 — 潜水员戴夫(Dave the Diver)

> 开发商: MINTROCKET (NEXON子工作室) | 韩国
> 发行年份: 2023年 | PC/Switch/PS | 团队规模: **不到30人**

## 1. 开发团队

| 成员 | 角色 |
|------|------|
| **Jaeho Hwang** | 总监/编剧 |
| Woo Chan-hee | 策划团队负责人 |
| Jung Gi-yup | 美术团队负责人 |
| Seo Bo-sung | 程序员团队负责人 |

MINTROCKET最初是NEXON韩国开发总部内部的一个**小团队**，专为小型低风险项目而设。团队之前**没有主机开发经验**，不得不从零学习Switch开发。2024年10月MINTROCKET正式剥离为NEXON全资子公司，Jaeho Hwang任总经理。

## 2. 开发历程

| 阶段 | 关键事件 |
|------|------|
| 立项 | NEXON内部小团队启动"小项目"实验 |
| 开发中 | 团队因游戏融合太多内容(潜水+寿司店)而面临内部怀疑 |
| 设计灵感 | 从世嘉《如龙》系列汲取多样化副业活动的设计思路 |
| 发布 | 2023年，全球爆红 |
| GDC 2024 | Jaeho Hwang发表演讲"融合幽默与游戏性" |

## 3. "独立游戏"争议

Dave the Diver在2023年TGA被提名"最佳独立游戏"，引发巨大争议——它背后是NEXON(韩国最大游戏公司之一)。总监Jaeho Hwang明确表示: "我们从未打算让它有独立游戏的感觉……我们在NEXON制作游戏，所以它没有任何独立游戏的成分。"

这个争议反而证明了：**小团队在大公司内部也可以做出"有独立精神"的游戏**。

## 4. 市场表现

全球爆红，Steam好评如潮。MINTROCKET已发展为约60名员工，同时推进三个项目。

## 5. 设计启示

1. **大公司内部的小团队 = 有独立精神的商业游戏** — 不到30人、无主机经验、被内部怀疑——但做出了全球爆款。关键是获得了"像独立团队一样运作"的自主权
2. **"融合太多内容"从弱点变为卖点** — 内部怀疑它融合了太多类型→结果这恰恰是玩家最爱的点
3. **NEXON的资源+独立团队的自由 = 最佳组合** — 有资金保障但没有大公司病
