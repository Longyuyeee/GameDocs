# 武器升级与装备数据 - 潜水员戴夫(Dave the Diver)

> 游戏版本: Dave the Diver (PC/Switch/PS, 2023年) | 开发商: MINTROCKET(NEXON)
> 数据来源: dave-the-diver.fandom.com, IGN Guide, GameSkinny, Pro Game Guides

## 一、武器升级树(Duff武器店)

### 基础步枪树
```
水下步枪 → 步枪II → III → IV (死亡步枪)
  ├→ 火焰步枪I → II → 爆炸步枪
  ├→ 麻醉步枪
  ├→ 毒步枪I → II → 地狱毒步枪
  ├→ 闪电步枪I → II → 雷霆步枪
  └→ 震击步枪I → II
```

### 三连轴树
```
三连轴 → 四连轴I → II → 五连轴
  ├→ 火焰三连轴I → II → 爆炸三连轴
  ├→ 麻醉三连轴
  ├→ 毒三连轴I → II → III → 地狱毒三连轴
  └→ 闪电三连轴 → 雷霆三连轴
```

### 狙击步枪树
```
红狙击步枪 → II → III → 死亡狙击
  ├→ 火焰狙击I → II → 爆炸狙击
  ├→ 麻醉狙击
  └→ 毒狙击I → II → 地狱毒狙击
```

### 榴弹发射器树
```
榴弹发射器 → II → III
  ├→ 麻醉气弹发射器
  ├→ 毒气发射器
  ├→ 黑洞发射器/重力发射器
  └→ 闪光弹发射器
```

### 其他线性武器
- 网枪: 小网→中网→**大网→钢网**
- 麻醉镖: 麻醉镖→增强I→增强II→改造麻醉镖
- 冰枪: 冰枪→增强冰枪→**超冰枪**
- 粘性炸弹: 粘性炸弹枪→II→III → 粘性地雷发射器

## 三、推荐升级顺序

| 优先级 | 装备 | 理由 |
|:--:|------|------|
| 1 | 气罐 | 更长的潜水时间=更多鱼 |
| 2 | 货箱 | 每潜携带更多 |
| 3 | 鱼叉枪 | 更好的捕获效率 |
| 4 | 潜水服 | 解锁更深区域 |
| 5 | 打捞无人机 | 回收重型战利品 |
| 6 | 蟹笼 | 被动收入 |
| 7 | 抗寒服 | 剧情解锁后 |
