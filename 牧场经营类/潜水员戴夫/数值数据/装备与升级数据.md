## 一、潜水装备升级系统(iDiver App)

通过手机中的iDiver App进行升级(Ch1后解锁Dr. Bacon即开启)，消耗金币，升级永久的潜水属性。

### 1.1 氧气瓶(Air Tank)——核心生存属性

| 等级 | O2容量(Bar) | 升级费用 | 累计费用 | 优先级 |
|:---:|:----------:|:--------:|:--------:|:-----:|
| 1(初始) | 90 | — | — | — |
| 2 | 115 | 65g | 65g | ★★★★★ |
| 3 | 145 | 225g | 290g | ★★★★★ |
| 4 | 180 | 400g | 690g | ★★★★★ |
| 5 | 220 | 750g | 1,440g | ★★★★ |
| 6 | 265 | 1,500g | 2,940g | ★★★★ |
| 7 | 315 | 3,000g | 5,940g | ★★★ |
| 8 | 370 | 6,000g | 11,940g | ★★★ |
| 9 | 425 | 11,300g | 23,240g | ★★ |
| **Max** | **480** | **17,000g** | **40,240g** | ★★ |

**设计分析**: 氧气=时间=金钱。O2容量直接影响可探索深度和可收集鱼获。从90→480Bar，线性增长但指数级费用。前6级性价比极高(+175Bar仅需2,940g)，最后3级(+215Bar需37,300g)为深度探索者优化。

### 1.2 负重背包(Cargo Box)——经济核心属性

| 2 | 13 | 55g | 55g | **最优先** |
| 3 | 19 | 110g | 165g | **最优先** |
| 4 | 28 | 325g | 490g | **最优先** |

**设计分析**: 负重直接决定单次潜水收益上限——每次多带1条鱼就是多一份夜间收入。9→185kg(20倍提升)是所有升级中经济回报最高的。每级增加约50%容量，保证玩家持续有升级动力。

### 1.3 潜水服(Diving Suit)——深度解锁

| 等级 | 深度上限 | 升级费用 | 累计费用 | 解锁内容 |
|:---:|:--------:|:--------:|:--------:|:--------:|

| 3 | 150m | 225g | 225g | 中层深处 |
| 4 | 250m | 550g | 775g | 深层区域 |
| 5 | 400m | 1,500g | 2,275g | 深层深处 |

| **Max** | **800m** | **7,500g** | **13,275g** | 热泉区域 |

**设计分析**: 潜水服是**剧情锁定的硬门槛**——深度不够=无法推进主线。每升一级解锁新的可探索区域和鱼种。与O2瓶不同，潜水服是"权限"而非"效率"升级。Lv5(400m)是通往冰川区的关键门槛，Lv7(800m)为终局内容所需。

| 等级 | 伤害 | 升级费用 | 累计费用 |

- 满级仅40伤害，远低于Duff制作的武器——设计上鱼叉枪为"保底工具"，主力输出靠武器系统

### 1.5 打捞无人机(Salvage Drone)

| 等级 | 使用次数 | 解锁条件 |
|:---:|:--------:|:--------:|
| 1 | 每次1次 | Tuna Party事件解锁 |
| 2 | 每次2次 | 材料升级 |
| **Max** | **每次3次** | 最高级 |

- 用于打捞超过负重上限的大型鱼类
- 每次潜水限次使用，规划好打捞目标
- 优先用于鲨鱼、金枪鱼等高价值大型鱼

### 1.6 潜水刀(Underwater Knife)

- **解锁**: 冰川区域剧情
- **功能**: 采集矿石、切割障碍物
- **满级伤害**: 17
- **重要性**: 某些矿脉和宝箱需要刀具采集

## 二、武器升级系统(Duff武器店)

Ch2解锁Duff的武器店后，可通过收集蓝图+材料制作和升级武器。**每次潜水只能携带1把武器**，选择决定了当天的捕鱼策略。

### 2.1 八种基础武器蓝图

| 武器 | 所需蓝图 | 基础特性 | 适用场景 |
|:---:|:--------:|:--------:|:--------:|
| Basic Underwater Rifle | 自动解锁 | 标准射击 | 通用/新手 |
| Triple Axel | 2张 | 散射(霰弹式) | 近距离群鱼 |
| Red Sniper Rifle | 3张 | 远程高伤 | 精准击杀/安全距离 |
| Grenade Launcher | 3张 | 范围爆炸 | Boss战/鱼群密集区 |
| Sticky Bomb Gun | 3张 | 粘性炸弹 | Boss战/布设陷阱 |
| Small Net Gun | 3张 | 捕获小型鱼群 | **3星活捉** |
| Hush Dart | 2张 | 麻醉睡眠 | **大型鱼3星活捉** |
| Ice Gun | 3张 | 冰冻控制 | 控制+群体效果 |

### 2.2 完整武器升级树

每种基础武器有多个升级分支——元素附魔路线(火焰/毒/电/麻痹/睡眠/冰冻)和纯伤害路线:

**Basic Underwater Rifle 分支树**:
```
                    ┌→ Rifle II → Rifle III → Death Rifle(纯伤害)
                    ├→ Flame Rifle I → II → Explosive Rifle(火焰)
Basic Rifle ────────├→ Tranquilizer Rifle(睡眠)
                    ├→ Poison Rifle I → II → Hell Poison Rifle(毒)
                    ├→ Lightning Rifle I → II → Thunderbolt Rifle(电)
                    └→ Shock Rifle I → II(麻痹)
```

**Triple Axel 分支树** (类似霰弹枪，中近距离):
```
                     ┌→ Quattro Axel I → II → Pentra Axel(散射强化)
                     ├→ Flame Triple Axel I → II → Explosive Triple Axel
Triple Axel ─────────├→ Tranquilizer Triple Axel
                     ├→ Poison Triple Axel I → II → Hell Poison Triple Axel
                     └→ Lightning Triple Axel → Shock Triple Axel I → II
```

**Red Sniper Rifle 分支树** (远程高精度):
```
                      ┌→ Red Sniper II → III → Death Sniper(最高单伤)
                      ├→ Flame Sniper I → II → Explosive Sniper
Red Sniper ───────────├→ Tranquilizer Sniper
                      ├→ Poison Sniper I → II → Hell Poison Sniper
                      ├→ Lightning Sniper I → II → Thunderbolt Sniper
                      └→ Shock Sniper I → II
```

**Grenade Launcher 分支树** (范围AOE):
```
                       ├→ Grenade Launcher II → III(爆炸强化)
                       ├→ Tranquilizer Gas Bomb(范围睡眠)
Grenade Launcher ──────├→ Poison Launcher → Blackhole Launcher(引力)
                       ├→ Gravity Launcher
                       └→ Flash Grenade Launcher
```

**Sticky Bomb Gun 分支树** (可布置式):
```
                        ├→ Sticky Bomb II → III
                        ├→ Sticky Mine Launcher I → II
Sticky Bomb Gun ────────├→ Sticky Tranquilizing Bomb Gun
                        ├→ Poison Mine Launcher I → II
                        ├→ Lightning Mine Launcher I → II
                        └→ Shock Mine Launcher I → II
```

**Small Net Gun 分支树** (纯捕获路线):
```
Small Net Gun → Medium Net Gun → Large Net Gun → Steel Net Gun
```

**Hush Dart 分支树** (纯麻醉路线):
```
Hush Dart → Enhanced Hush Dart I → II → Modified Hush Dart
```

**Ice Gun 分支树** (纯冰冻路线):
```
Ice Gun → Enhanced Ice Gun → Ultra Ice Gun
```

## 四、升级策略与优先级

### 阶段优先级

| 游戏阶段 | 第1优先 | 第2优先 | 第3优先 | 说明 |
|:-------:|:-------:|:-------:|:-------:|:----:|
| 前期(Ch1-2) | Cargo Box(Lv5) | Air Tank(Lv5) | Diving Suit(Lv3) | 先赚钱再保命 |
| 中期(Ch3-5) | Air Tank(Lv7) | Cargo Box(Lv7) | Net Gun | 深层探索需求 |
| 后期(Ch6-7) | Diving Suit(Max) | Cargo Box(Max) | 武器升级路 | 剧情深度门槛 |
| 终局 | 全Max | 特殊武器 | 元素武器全收集 | 追求完美 |

### 核心原则

1. **负重>氧气>潜水服>鱼叉**: 负重直接产生金币->金币升级其他->正向循环
2. **单武器深度优于多武器广度**: 优先把1-2把关键武器升到T3/T4
3. **Net Gun + Hush Dart是经济最优解**: 3星活捉的售价倍率(×1.6-2.0)远超纯击杀
4. **Boss战前必升级O2和潜水服**: 探索更深=更多Boss战机会

## 五、设计分析与跨游戏对比

### 设计亮点

1. **双升级系统(iDiver + Duff)**: 永久属性(App升级) vs 策略选择(武器系统)的分离设计，让玩家既有稳定的成长感又有每次潜水的策略决策。
2. **单武器限制**: 不让玩家"全带"是天才设计——迫使每次潜水前做战术决策，避免万能装备的无聊。
3. **元素路线选择**: 8种基础武器×6种元素分支=48种最终升级组合，提供丰富的build多样性——类似《死亡细胞》的武器系统密度。
4. **升级费用曲线**: 指数级递增的费用让前期快速提升、后期缓慢优化，符合多数玩家的"先爽后精"心理。

### 跨游戏对比

| 维度 | 潜水员戴夫 | 动物进行曲 | Stardew Valley(参考) |
|:---:|:----------:|:----------:|:-------------------:|
| 升级方式 | 金币+材料双轨 | 矿石+金币 | 金币直购 |
| 武器多样性 | 8基础×6元素=48种 | 有限升级 | 有限武器 |
| 单次携带限制 | **限定1把(策略选择)** | 无限制 | 无限制 |
| 装备与经营关联 | 直接(鱼获→菜单) | 间接 | 间接 |
| 升级深度 | 阶段式(Ch锁定+材料) | 线性升级 | 线性升级 |

---

> **设计意图分析**: Dave the Diver的装备系统服务于"每次潜水都是一次战术决策"的核心体验——你带的武器决定了今天能捕什么鱼，能捕什么鱼决定了今晚的菜单，今晚的菜单决定了能赚多少钱。这种因果链条让装备选择不再是单纯的数值堆砌，而是贯穿整个游戏循环的战略决策。
