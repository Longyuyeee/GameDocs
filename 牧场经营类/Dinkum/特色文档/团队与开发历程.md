# 团队与开发历程 — Dinkum

> 开发商: James Bendon (单人) | 澳大利亚
> 发行年份: 2022年(EA) / 2025年(正式) | PC/Steam | 团队规模: **1人**

## 1. 开发团队

| 成员 | 角色 | 背景 |
|------|------|------|
| **James Bendon** | 全部(编程/3D建模/设计/美术) | 澳大利亚单人开发者 |

没有团队——一个人做了所有事。

## 2. 开发历程

| 阶段 | 时间 | 关键事件 |
|------|:--:|------|
| 灵感 | 2017年 | 一次穿越昆士兰的公路旅行激发了澳洲内陆背景的灵感 |
| 开发开始 | 2017年 | 单人开始全职开发 |
| Early Access | 2022年7月 | Steam上线——原计划EA一年，实际持续了近三年 |
| 前5个月 | 2022年底 | 销量突破20万份 |
| 关键节点 | 2024年 | 生命周期销量突破**100万份** |
| 出版合作 | 2025年2月 | KRAFTON签署全球发行协议 |
| 正式发布 | 2025年 | 完整版发布 |

**总开发周期**: 约8年(2017-2025，单人)

## 3. 技术与工具

| 类别 | 工具 |
|------|------|
| 引擎 | Unity |
| 建模 | 3D建模(Bendon自学) |
| 灵感 | 《动物森友会》+《牧场物语》+ 澳洲内陆风情 |

Bendon表示: "我爱任天堂的一切……我想创作一款受《动物森友会》启发的游戏，包含《动物森友会》当时缺少的所有功能。"

当2017年开始开发后，任天堂宣布了《集合啦！动物森友会》，Bendon一度担心: "它设定在岛上，你可以按自己喜欢的方式建立城镇。"但他坚持了下来，找到了自己的市场。

## 4. 市场表现

| 指标 | 数据 |
|------|:--:|
| 前4-5月销量 | 20万份 |
| 总销量 | **100万+份** |
| Steam评价 | 特别好评 |
| 出版合作 | KRAFTON(2025年起) |

## 5. 开发者感悟

Bendon谈独自工作:
> "说实话，我真的喜欢独自工作。没有截止日期或会议是非常自由的。"

但他也经历了发布后抑郁——独自处理关键bug(如存档损坏)时没有团队支持，平衡更新质量与玩家期望的巨大压力。

## 6. 设计启示

1. **1人×8年 = 100万份** — 一个人用Unity做出了动物森友会×牧场物语的澳洲版本，证明了单人在生活模拟品类仍然可以成功
2. **"包含他们缺少的功能" = 差异化策略** — Bendon不是"复刻动物森友会"，而是"做一个有所有缺失功能的动物森友会"——这种差异化思路值得学习
3. **KRAFTON签约 = 单人开发者的终极验证** — 大厂认可了单人作品的质量
