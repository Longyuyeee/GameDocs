## 1. 地图与生物群系

Dinkum的地图是**程序生成的澳大利亚荒野大陆**, 每次开新游戏地图布局不同, 但都包含以下生物群系:

| 生物群系 | 地形特征 | 主要资源 | 危险等级 |
|:--------:|:--------:|:--------:|:--------:|
| 平原(Grassland) | 开阔草地 | 野花/草种/基础木材 | ★ |
| 森林(Forest) | 密集树木 | 硬木/蘑菇/野果 | ★★ |
| 沙漠(Desert) | 沙地/仙人掌 | 仙人掌果/沙石/远古遗迹 | ★★★ |
| 海滩(Beach) | 沙滩海岸线 | 贝壳/螃蟹/漂流木 | ★ |
| 红树林(Mangrove) | 水域+树木 | 红木/鳄鱼 | ★★★★★ |
| 河流(River) | 淡水水域 | 淡水鱼/蛇 | ★★★ |
| 海洋(Ocean) | 深水区 | 海水鱼/鲨鱼/珍珠 | ★★★★★ |

## 2. 危险野生动物

Dinkum与大多数牧场经营游戏的根本区别在于: **野外是危险的**。鳄鱼、鲨鱼、毒蛇等具有真实攻击性的野生动物遍布地图。

### 2.1 主动攻击型

| 动物 | 威胁等级 | 攻击方式 | 出现区域 | 狩猎掉落 | 对策 |
|:----:|:--------:|:--------:|:--------:|:--------:|------|
| Croco(鳄鱼) | ★★★★★ | 快速冲刺+撕咬(高伤害) | 水域/红树林附近 | 鳄鱼肉+鳄鱼牙 | **前期绕路!** 后期用矛远程攻击 |
| Shark(鲨鱼) | ★★★★★ | 水中突袭 | 海洋区域 | 鲨鱼鳍+鲨鱼肉 | 避免下水, 或乘船时注意 |
| Wild Dog(野狗) | ★★★ | 追逐撕咬 | 平原/森林 | 兽皮+肉 | 近战击杀 |
| Snake(蛇) | ★★★ | 毒液(持续伤害) | 森林/沙漠 | 蛇皮 | 保持距离远程攻击 |
| Goanna(巨蜥) | ★★★ | 爪击 | 沙漠/岩石区 | 巨蜥皮+肉 | 近战注意躲避 |

## 3. Deep Mine(深井采矿)

Deep Mine是Dinkum的**终局挑战内容**。

### 3.1 解锁流程

```
采矿Lv10 → 购买深井采矿许可证(3,500 PP) →
建造矿洞入口(250,000 Dinks + 10水泥+10旧齿轮+2旧钥匙+1旧装置) →
进入Deep Mine
```

### 3.2 三层结构

| 层 | 矿物分布 | 敌人 | 推荐准备 | 特殊机制 |
|:--:|:--------:|:----:|:--------:|:--------:|
| 1层 | 铜/铁/基础宝石 | 基础怪物(低密度) | 铜镐+基础武器+火把 | 矿脉较稀 |
| 2层 | **大量铁**/银/宝箱 | 增强怪物(中密度) | 铁镐+铜武器+矿工帽 | 宝箱数量多 |
| 3层 | 金/铂金/稀有宝石/岩浆 | **最强怪物(高密度)** | 金镐+铁武器+滑翔伞+充足食物 | 岩浆地形+深坑 |

### 3.3 Deep Mine关键机制

| 机制 | 说明 |
|:----:|------|
| 每日重置 | 每天矿洞布局完全刷新——不能"条条大路通罗马" |
| 电梯 | 玩家需自行建造——可从地面运输储物到矿洞 |
| 照明 | 矿工帽(头戴) >> 手电筒(手持) > 火把(放置) |
| 路线标记 | 火把可用来标记已探索的路线——避免迷路 |
| Hand Trolley | John's Goods 30,000 Dinks——搬运储物箱进出电梯 |

### 3.4 Treasure Room(宝物库)

| 项目 | 说明 |
|:----:|------|
| 外观 | 圆形灰色结构(小地图可见) |
| 解锁 | 每个需要**2把旧钥匙(Old Key)** |
| 内容 | 宝箱内含: 资源/工具/武器/油漆 |
| 稀有掉落 | Chrome油漆和Gold油漆(仅限Jimmy处购买) |
| 宝箱搬运 | 清空后的宝箱可以拾起并搬家 |

#### 旧钥匙获取途径

| 途径 | 效率 | 说明 |
|:----:|:----:|------|
| 旧桶/垃圾桶 | 低(随机) | 地图上杂乱的废墟中打破 |
| 金属探测 | 中(稳定) | 挖掘出的宝藏有一定几率 |
| 蟹笼 | 中(被动) | 放在河流中收获几率最高 |
| Undergrove/Island Reef | 中(探索) | 随机宝箱 |
| Key Cutter | 高(固定) | 红宝箱中获得——每7天产出1把 |
| **Key Cycler** | **极高(自动化)** | **Jimmy船上700,000 Dinks购买——放入Relic每10分钟产1把** |

> Key Cycler是终局获取旧钥匙的最稳定方式——它是自动化设备, 只需提供Relic作为燃料。Jimmy只在雨天早晨出现(需要银行存款1,000,000+ Dinks)。

## 4. 特殊天气探索事件

### 4.1 闪电雷暴(Thunderstorm)

| 项目 | 说明 |
|:----:|------|
| 发生季节 | 夏季(最多4次/季) |
| 核心产出 | 雷电蛋(Lightning Egg) |
| 探索策略 | 雷暴过后立即外出探索——地面会出现稀有掉落物 |
| 额外掉落 | 待补充 |

> 雷电蛋是Dinkum中价值最高的特殊物品之一——一次雷暴后如果探索足够仔细, 可能收获价值数万Dinks的掉落物。

### 4.2 流星雨(Meteor Shower)

| 项目 | 说明 |
|:----:|------|
| 发生季节 | 秋季(Sky Fest期间) |
| 机制 | 跟随最慢的流星——它坠落的位置即为陨石地点 |
| 陨石掉落 | 陨石块(Meteorite Chunk) |
| 稀有掉落 | 银河服装(Galaxy Set)——极低概率 |
| 采集 | 用镐采集陨石 |

> 流星是跨越天空的——需要抬头观察并追踪最慢的那一颗。陨石块是Star Sabre(星辰军刀)的制作材料。

## 6. 体力与生命系统

Dinkum在Breezy Autumn更新后对体力系统进行了重大修改。

### 6.1 体力系统

| 机制 | 旧版 | 新版(Breezy Autumn+) |
|:----:|:----:|:-------------------:|
| 体力耗尽 | 晕倒(faint) | **昏睡(sleep)** |
| 体力恢复 | 进食瞬间恢复 | **进食→再生率** |
| Well Rested buff | 起床后临时加成 | 体力恢复速度提升 |
| 蓝闪电buff | 无 | 电工具不耗体力 |
| 灰装备buff(蟹汤) | 无 | 工具无耐久消耗 |

> 体力耗尽可能导致"昏睡"——不会死亡但会浪费游戏时间, 相当于强制推进到第二天。

### 6.2 生命恢复

| 方式 | 效果 | 说明 |
|:----:|:----:|:------|
| 急救包(First Aid Kit) | 恢复大量HP | John's Goods 7,000 Dinks |
| 烹饪料理 | 持续恢复HP+体力 | 高级烹饪台制作 |
| 睡眠 | 完全恢复HP+体力 | 家中床上睡觉 |

## 8. 设计特点分析

### 8.1 危险不是装饰——是核心机制

大部分牧场经营游戏的危险是"点缀性的"——被蜜蜂蛰一下或者被矿洞蝙蝠撞一下。但Dinkum的野外危险是**真实的生存威胁**——初级装备时被鳄鱼咬3-4次就会死亡。

这种设计的意图是:
1. 创造"荒野拓荒"的叙事感——你不是在经营已有的农场, 而是在危险中开辟家园
2. 强制玩家循序渐进——不能一开始就跑地图所有区域
3. 使后期变强有明确的回报感——"我终于可以在红树林走动而不怕鳄鱼了"

### 8.2 Deep Mine = Dinkum的"终局副本"

Deep Mine的设计语言非常接近MMO的"副本"概念:
- 每日重置(不能刷同一条路线)
- 三层递增难度(一层比一层难)
- 需要特殊装备(滑翔伞是3层的"门票")
- 存在隐藏房间(Treasure Room需要钥匙解锁)
- 有经济门槛(入口250,000 Dinks)

### 8.3 跨游戏对比: 探索与危险

| 维度 | 星露谷物语 | Dinkum | 动物森友会 |
|:----:|:--------:|:------:|:--------:|
| 野外危险 | 低(矿洞怪物) | **高**(鳄鱼/鲨鱼) | 几乎无(黄蜂-无伤害) |
| 动物攻击 | 仅矿洞内 | **全地图**(野外+矿洞) | 无 |
| 死亡惩罚 | 扣钱+物品 | 昏睡+时间浪费 | 无 |
| 探索驱动力 | 矿洞层数+收集 | 采矿+稀有掉落+宝物库 | 资源+DIY |
| 快速旅行 | 矿洞电梯 | 机场(2.5M)+直升机(3M) | 机场 |
| 地图生成 | 固定 | **程序生成** | 固定 |
| 天气事件 | 随机 | 季节固定+许愿控制 | 实时天气 |

### 8.4 程序生成 vs 固定地图

Dinkum选择程序生成地图而非固定地图——这与大多数牧场经营游戏不同(星露谷/珊瑚岛/波西亚时光均为固定地图)。程序生成的优点是每局体验不同, 缺点是地图布局的质量参差不齐。

---

> **设计启示**: Dinkum通过将"危险"从点缀升级为核心机制, 成功在治愈系牧场经营品类中创造出了独特的"生存感"。Deep Mine的"终局副本"设计证明了生活模拟游戏也可以有明确的终局挑战内容。程序生成地图在牧场品类中的运用值得关注——虽然增加了变化性, 但也牺牲了手工设计地图的叙事连贯性。
