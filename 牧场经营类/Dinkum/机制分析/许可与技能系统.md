## 1. 许可系统(Licenses)——Dinkum的"科技树"

许可是Dinkum**最核心的进度控制系统**。与传统游戏的"技能树"不同, Dinkum使用"许可证"的概念——玩家需要向Fletch(基地管理员)申请许可, 才能解锁对应的工具和活动。这个设计不仅控制了玩家能力的成长速度, 也与"澳洲荒野拓荒→文明化"的叙事主题相呼应。

## 2. Permit Points(许可点数)

PP是Dinkum的第二货币——专门用于购买许可证。

### 2.1 PP获取途径

| 途径 | 效率 | 说明 |
|:----:|:----:|------|
| 里程碑(Milestones) | ★★★★★ | 主要来源——完成各种成就在每个等级给予PP |
| 博物馆捐赠 | ★★★ | 每捐一条新鱼/虫获得100 PP |
| 每日里程碑 | ★★★ | 每天刷新的小任务——100-250 PP |
| 公告板任务 | ★★★ | 日常任务——PP+Dinks |
| 银行存款兑换 | ★(不推荐) | 500 Dinks→1 PP, 极不划算 |
| 出售物品 | ★(微量) | 极少量PP |

## 3. 技能系统

Dinkum有**8种技能**, 每种通过使用该活动自然升级。技能升级的核心收益: **高级=减少该活动的体力消耗**。

### 3.1 技能总览

| 技能 | 需许可证? | 升级方式 | Lv10解锁 | Lv20解锁 |
|:----:|:---------:|---------|:--------:|:--------:|
| 采矿(Mining) | ✓ | 开采矿石 | 深井采矿许可证解锁 | 高级采矿工具 |
| 伐木(Logging) | ✓ | 砍伐树木 | 高级伐木 | 终极伐木工具 |
| 钓鱼(Fishing) | ✓ | 钓鱼 | 蟹笼解锁 | 终极钓鱼工具 |
| 狩猎(Hunting) | ✓ | 猎捕动物 | Ted Selly出现 | Alpha武器制作 |
| 农耕(Farming) | ✓ | 种植+收获 | 铜工具 | 灌溉许可证 |
| 畜牧(Handling) | ✓ | 饲养动物 | 袋熊解锁 | 鸭嘴羊解锁 |
| 捕虫(Bug Catching) | ✗ | 捕捉昆虫 | 蜂箱制作 | 蠕虫农场制作 |
| 挖掘(Excavation) | ✓ | 挖掘物品 | 高级挖掘 | 终极工具 |
| 采集(Foraging) | ✗ | 采集野生资源 | 无 | 无 |

### 3.2 技能经验机制

| 技能 | 每次经验 | 升级曲线 | 体力消耗减少 |
|:----:|:--------:|:--------:|:-----------:|
| 采矿 | 每块矿石~5-10xp | 渐进 | 升级→采矿每次消耗体力减少 |
| 伐木 | 每棵树~5-10xp | 渐进 | 升级→砍树消耗体力减少 |
| 钓鱼 | 每条鱼~10-20xp | 渐进 | 升级→钓鱼消耗体力减少 |
| 狩猎 | 每个敌人~10xp | 渐进 | 升级→攻击消耗体力减少 |
| 农耕 | 每个作物~5xp | 渐进 | 升级→农活消耗体力减少 |
| 畜牧 | 每天~2-5xp | 缓慢 | 升级→管理消耗体力减少 |
| 捕虫 | 每只虫~5-10xp | 渐进 | 升级→捕虫消耗体力减少 |
| 挖掘 | 每挖~5xp | 渐进 | 升级→挖掘消耗体力减少 |

> Dinkum的技能升级采取**做得多升级快**的原则——没有每日经验上限。如果玩家一天内持续做同一活动, 可以快速提升该技能等级。

### 3.3 技能等级带来的质变

| 技能 | 低等级(1-5) | 中等级(6-15) | 高等级(16+) |
|:----:|:----------:|:-----------:|:----------:|
| 采矿 | 只能挖基础矿 | 可挖铁/银矿石 | 可挖金/铂金矿石 |
| 伐木 | 只能砍小树 | 可砍硬木树 | 可砍所有树 |
| 钓鱼 | 只能钓基础鱼 | 铜竿看到鱼泡 | 铁竿+蟹笼 |
| 狩猎 | 基础武器狩猎 | 铜级武器 | 铁级+特殊武器 |
| 农耕 | 基础种植 | 铜工具 | 铁工具+灌溉证 |
| 畜牧 | 养鸡 | 养袋熊 | 养鸭嘴羊 |
| 捕虫 | 基础捕虫 | 可见虫子(?) | 蜂箱+蠕虫农场 |

### 3.4 捕虫特殊技能树

捕虫是唯一**不需要许可证**但仍有明确技能解锁的技能:

| 等级 | 解锁 |
|:----:|------|
| 1 | 基础捕虫网 |
| 5 | 铜捕虫网 |
| 10 | **蜂箱**(蜂蜜生产) |
| 15 | 铁捕虫网 |
| 20 | **蠕虫农场**(钓鱼饵料自动生产) |

> 蜂箱是重要的被动收入来源——放置后自动生产蜂蜜, 可用于烹饪或出售。蠕虫农场自动生产钓鱼用的虫子, 省去手动挖虫的时间。

## 4. 许可证与技能的关系

许可证和技能是Dinkum两条并行的进度线, 它们互相依赖:

```
许可证(购买) → 解锁工具使用权限
     ↓
使用工具 → 提升对应技能
     ↓
技能达标 → 解锁高级许可证(需要技能等级)
     ↓
购买高级许可证 → 解锁更高级的工具
     ↓
使用更高级的工具 → 继续提升技能
     ...
```

### 4.1 许可证对技能的依赖

| 高级许可证 | 依赖的技能等级 |
|:----------:|:-------------:|
| 采矿Lv2 | 采矿10 |
| 采矿Lv3 | 采矿20 |
| 钓鱼Lv2 | 钓鱼5 |
| 钓鱼Lv3 | 钓鱼10 |
| 狩猎Lv2 | 狩猎5 |
| 狩猎Lv3 | 狩猎10 |
| 农耕Lv2 | 农耕10 |
| 农耕Lv3 | 农耕20 |
| 深井采矿 | 采矿10 |
| 灌溉 | 农耕20 |

> 这意味着玩家无法"用PP跳级"——必须实际使用工具提升技能等级, 才能解锁更高级的许可证。

## 5. 设计特点分析

### 5.1 许可=渐进的复杂度

Dinkum的许可证系统解决了生活模拟游戏的一个核心问题: **信息过载**。如果所有玩法一开始就全部可用, 新手会感到不知所措。

星露谷的解决方案是"让玩家自己探索"——地图上所有内容一开始就存在, 玩家哪天发现了就可以参与。Dinkum的解决方案是"强制渐进"——没有采矿证就无法使用镐, 不买镐就无法挖矿, 不挖矿就无法升级采矿技能——每个阶段只呈现当前可玩的内容。

### 5.2 PP=行为驱动的进度

PP不是"时间到了自动获得", 而是"做了特定行为才获得"。这种设计的意图很明确:

- **玩得多=解锁快**: 你喜欢钓鱼? 多钓会获得PP→解锁钓鱼许可证→继续钓鱼
- **鼓励全面游戏**: 不同行为对应不同里程碑, 鼓励玩家尝试所有玩法
- **消灭"刷"的动机**: 不需要刻意刷——日常游戏行为自然积累PP

### 5.3 技能升级=体力管理工具

技能升级的核心收益不是"新的能力", 而是"做旧事情消耗更少体力"。这意味着:

- 低级时做任何事都很吃力(每次消耗大量体力)
- 随着技能提升, 体力效率提高
- 玩家自然倾向于"专业分工"(集中练一个技能)

### 5.4 跨游戏对比: 进度系统

| 维度 | 星露谷物语 | Dinkum | 珊瑚岛 |
|:----:|:--------:|:------:|:------:|
| 技能数量 | 5种 | **8种** | 5+种 |
| 许可证/技能树 | 无(直接使用工具) | **许可证制度**(需购买) | 技能树+等级 |
| 升级方式 | 使用升级 | 使用升级 | 使用升级 |
| 高级解锁 | 自动(等级够) | 许可证+技能双重要求 | 自动+技能树 |
| PP点数 | 无 | **专用货币**(里程碑获得) | 无 |
| 经济关联 | 金币万能 | **双货币**(Dinks+PP) | 金币万能 |

### 5.5 许可证叙事的文化意义

Dinkum选择"许可证"而不是"技能树"有其文化背景——澳大利亚历史上是英国流放犯人的殖民地, 后逐步发展为一个高度制度化的移民国家。"向当局申请许可才能做某事"的设定, 与澳洲的殖民历史和文化记忆有一定的呼应。这是Dinkum通过游戏机制表达主题的绝佳例子。

---

> **设计启示**: Dinkum的许可证+技能双轨制是非常成功的进度控制设计。许可证解决了"信息过载"问题, 技能系统解决了"体力管理"问题, PP系统解决了"行为激励"问题。三者构成的三角循环, 为生活模拟游戏提供了一个值得参考的进度系统范本。
