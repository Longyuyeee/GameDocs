## 一、游戏概述

Dinkum是一款以**澳大利亚荒野内陆**为背景的开放世界牧场经营游戏。玩家从零开始在一片未开发的澳洲大陆上拓荒——采矿、伐木、狩猎、钓鱼、农耕、建造城镇、吸引NPC定居。其核心特征是**"动森式社交 x 星露谷物语式经营 x 澳洲荒野生存"**的三元融合。

与传统的治愈系牧场游戏不同，Dinkum强调**"荒野拓荒的严肃性"**——玩家需要面对鳄鱼、鲨鱼、毒蛇等危险野生动物，在雷暴和酷热中生存，通过许可证系统逐步解锁文明化的进程。

## 二、核心特色

| 特色 | 说明 |
|------|------|
| 澳洲荒野主题 | 干燥内陆、铁皮屋顶、风车、香肠卷、澳洲坚果、袋鼠、鸸鹋——独一无二的Australian美学 |
| 许可证进度系统 | 非传统技能树——许可证(Permit)制度，玩家需向Fletch申请许可才能使用工具/开展活动 |
| 8种技能 | 农耕/钓鱼/采矿/伐木/狩猎/捕虫/挖掘/采集——各自独立升级，高级减少该活动体力消耗 |
| NPC来访→入居 | NPC需多次访问→完成任务→好感提升→申请永久定居——动森式入居机制 |
| 危险野生动物 | 鳄鱼(水域)/鲨鱼(海洋)/野狗/毒蛇/食火鸡——创造"治愈但不安全"的张力 |
| 自由地形改造 | 完全自由的地形编辑——铺路/造桥/挖掘/填土——玩家自定义城镇布局 |
| 4人合作 | 最多4人联机协作——共享建设进度/分工采集 |
| 许愿喷泉 | 5种模式切换(和平/危险/丰收/幸运/洁净)——影响全岛参数 |

## 三、核心系统架构

```
Dinkum核心循环:
  初始拓荒 → 采集基础资源 → 购买许可证 → 升级工具/技能
      ↓                                              |
  赚取Dinks ← 出售加工品/鱼/宝石/作物 ← 深入探索/采矿
      ↓
  购买建筑契约 → NPC入居 → 解锁新功能 → 继续拓荒
```
