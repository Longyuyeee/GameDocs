> 原版：牧场物语 欢迎来到风之集市（NDS，2008年）

---

## 一、系统概述

社交系统是牧场物语系列的灵魂之一。本作在保留原版角色阵容的基础上，新增了两位恋爱候补——**新太（Shinta）** 和 **黛安娜（Diana）**，使结婚候补总数达到12人。重制版还新增了**一键打招呼（Quick Greet）**、**部分语音配音**和**性别无关恋爱**等现代化改进。

## 三、一键打招呼系统（Quick Greet）——重制版新增

### 3.1 系统概述

| 项目 | 说明 |
|:---|:----|
| 操作方式 | 接近NPC时按L键（或指定快捷键） |
| 效果 | 自动向**附近所有NPC**打招呼 |
| 范围 | 以玩家为中心约5格半径 |
| 亲密度增加 | 每位被打招呼的NPC +1 |
| 单日上限 | 无限制 |
| 效率对比 | 原版需逐个对话 → 重制版一键群发 |

### 3.2 与传统对话对比

| 对比项 | 原版（逐个对话） | 重制版（一键招呼） |
|:------|:--------------|:--------------|
| 单次互动时间 | 约5秒/人 | 约1秒/群体 |
| 5人同时在场耗时 | 25秒 | 2秒 |
| 亲密度收益 | 对话 +2，打招呼 - | 打招呼 +1，对话 +2 |
| 操作疲劳度 | 高 | 低 |

## 四、委托任务系统（Quest System）

### 4.1 任务类型

本作有三种颜色区分的任务类型：

| 任务类型 | 颜色标识 | 内容 | 奖励 | 刷新频率 |
|:--------|:-------:|:----|:----|:--------|
| 日常委托 | 💙 蓝色 | 采集/送货/收获指定物品 | 金钱、亲密度 | 每日刷新 |
| 成长任务 | 🩷 粉色 | 解锁新功能/扩展牧场/提升技能 | 工具升级、设施解锁 | 阶段性 |
| 角色故事 | 🧡 橙色 | 深入了解某个NPC的背景故事 | 特殊事件、珍稀道具 | 每角色5-7章 |

### 4.2 蓝色委托（日常委托）

| 委托等级 | 解锁条件 | 任务要求 | 基础报酬 | 附加奖励 |
|:-------:|:--------|:--------|:-------:|:-------:|
| D级 | 初始 | 交付1-3个基础物品 | 100-300G | 亲密度+2 |
| C级 | 集市Lv.2 | 交付3-5个中等物品 | 300-800G | 亲密度+3 |
| B级 | 集市Lv.3 | 交付加工品或高星作物 | 800-2,000G | 亲密度+5，稀有物品 |
| A级 | 集市Lv.5 | 交付珠宝或高级料理 | 2,000-5,000G | 亲密度+8，特殊道具 |
| S级 | 集市Lv.7 | 交付传说级物品 | 5,000-10,000G | 亲密度+10，限定装饰 |

**委托获取位置：** 集会所（Guild Hall）的**委托板（Quest Board）**

### 4.3 粉色委托（成长任务）

| 任务名称 | 触发条件 | 任务目标 | 奖励 |
|:--------|:--------|:--------|:----|
| 初次耕作 | 第一天 | 开垦10格田地 | 基础肥料×5 |
| 第一次集市 | 第一个周六 | 在集市获得5,000G以上销售额 | 摊位装饰×1 |
| 畜牧起步 | 集市Lv.2 | 购买第一只鸡 | 优质饲料×10 |
| 风车入门 | 集市Lv.2 | 使用风车完成一次加工 | 风车指南（解锁配方+3） |
| 交友第一歩 | 春15日前 | 与5名NPC达到1❤以上 | 礼物包装纸×5 |
| 品牌之路 | 第一次作物祭前 | 获得品牌认证 | 品牌招牌×1 |
| 牧场大亨 | 集市Lv.5 | 单次集市收入50,000G以上 | 仓库扩大许可证 |
| 飞行之梦 | 春25日后 | 获得滑翔伞 | 滑翔伞获得 |

### 4.4 橙色委托（角色故事）

每位NPC拥有5-7章的独立故事线，通过特定条件触发：

| 章节 | 触发条件 | 故事内容 | 奖励 |
|:---:|:--------|:--------|:----|
| 第1章 | 角色亲密度≥1❤ | 初次深度对话 | 该角色背景介绍 |
| 第2章 | 角色亲密度≥3❤ | 协助角色解决问题 | 该角色特殊道具 |
| 第3章 | 角色亲密度≥5❤ | 参与角色的人生事件 | 该角色CG解锁 |
| 第4章 | 角色亲密度≥7❤ | 重大事件（恋爱角色则为恋爱事件） | 特殊料理/物品 |
| 第5章 | 角色亲密度≥9❤ | 结局事件 | 该角色限定装饰/称号 |
| 第6章（部分角色） | 特殊条件 | 深层故事 | 隐藏道具 |
| 第7章（部分角色） | 全结局后 | 真结局 | 限定家具 |

## 六、恋爱与结婚系统

### 6.1 恋爱流程

| 阶段 | 条件 | 事件 | 说明 |
|:----|:----|:----|:----|
| 相识 | 初次对话 | 自我介绍 | 自动触发 |
| 友善 | 亲密度≥3❤ | 蓝色爱心事件 | NPC会主动打招呼 |
| 友好 | 亲密度≥5❤ | 粉色爱心事件 | 可赠送礼物，NPC会回礼 |
| 心动 | 亲密度≥7❤ | 橙色爱心事件 | 触发恋爱事件，可选择告白 |
| 恋爱 | 告白成功 | 成为恋人 | 亲密度上限提升至10❤ |
| 订婚 | 亲密度≥10❤ + 双人床 + 订婚戒指 | 求婚事件 | 需购买"蓝色羽毛"（告白道具，50,000G） |
| 结婚 | 订婚后7天 | 婚礼仪式 | 全镇NPC参加，获得配偶 |

### 6.2 性别无关恋爱（Gender-Free Romance）

| 项目 | 说明 |
|:----|:----|
| 设计 | 重制版取消性别限制 |
| 男性主角 | 可追求任意6男6女 |
| 女性主角 | 可追求任意6男6女 |
| 同性配偶 | 支持，对话无差异 |
| 婚后家庭 | 孩子领养系统（无生物学差异） |

### 6.3 婚后生活

| 项目 | 说明 |
|:----|:----|
| 配偶行为 | 每天早晨为玩家准备便当/咖啡 |
| 配偶工作 | 协助农场工作（浇水/喂食等） |
| 孩子系统 | 婚后约1年（游戏内）迎来第一个孩子 |
| 家庭事件 | 孩子的成长事件（学步、入学等） |
| 结婚纪念日 | 每年结婚日配偶会有特殊对话和礼物 |

## 十、总结

本作的社交系统在继承系列经典设计的基础上，通过**一键打招呼**大幅减轻了日常社交负担，通过**性别无关恋爱**拓展了角色关系的可能性。新增的两位角色——新太和黛安娜——为剧情增添了新维度。三色委托系统（蓝/粉/橙）提供了清晰的社交行为引导，从日常互动到深度角色故事，形成了层次丰富的社交体验。部分语音配音的加入也增强了角色的生动性。
