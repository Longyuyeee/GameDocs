## 一、概述

《牧场物语 来吧！风之繁华集市》是2008年NDS游戏《牧场物语 欢迎来到风之集市》（英文名：Harvest Moon: Grand Bazaar / Story of Seasons: Grand Bazaar）的**完全重制版**。由Marvelous Inc.开发，2025年8月28日登陆Nintendo Switch / Switch 2 / Steam平台。

### 1.1 基本信息

| 项目 | 原版(NDS, 2008) | 重制版(2025) |
|:----|:---------------|:------------|
| 平台 | Nintendo DS | Switch / Switch 2 / Steam |
| 画面 | 2D像素+2头身Q版角色 | **全3D+4头身比例角色** |
| 语音 | 无 | **主要剧情全程语音** |
| 角色立绘 | 无 | **可开关立绘** |
| 语言 | 日文 | 简中/繁中/日/英/法/德/西 |
| 恋爱候补 | 10名(5男5女) | **12名(6男6女)** |
| 婚姻 | 仅异性 | **支持同性婚姻** |
| 价格(Steam) | — | 一般版218元/豪华版258元 |

### 1.2 M站评分
- Metacritic均分：**83分**
- IGN评价：Switch 2版本"优化流畅"

### 1.3 全球销量
- 截至2025年9月16日，全球累计出货量突破**50万套**

## 三、新增系统与机制

### 3.1 滑翔伞系统

| 项目 | 内容 |
|:----|:-----|
| 类型 | **重制版完全新增**（原版无此系统） |
| 功能 | 借助风力在空中滑翔 |
| 可到达区域 | 屋顶、悬崖、隐藏区域 |
| 发现物 | 稀有资源、隐藏通道、秘密区域 |
| 与风力的互动 | 强风滑翔更远更快，台风无法使用 |
| 获取方式 | 中期在NPC处购买/事件获得 |

滑翔伞大幅扩展了游戏的**探索维度**。原版NDS限于2D画面，地图相对平面化。重制版利用3D地图的垂直空间，通过滑翔伞系统让玩家可以到达之前无法前往的区域。

### 3.2 跳跃+工具组合技

| 项目 | 内容 |
|:----|:-----|
| 类型 | **重制版完全新增** |
| 基础机制 | 跳起时使用工具，影响面前3格范围 |
| 二段跳 | 购买技能后可在空中再次跳跃 |
| 覆盖范围 | 跳跃高度和工具等级共同影响范围 |
| 涉及工具 | 锄头、浇水壶、镰刀、施肥器 |
| 体验提升 | 大幅减少重复操作，效率翻倍 |

**与NDS原版的对比**：
- 原版：必须一格一格耕地、浇水，体力消耗大，耗时长
- 重制版：跳跃+工具组合技，一次处理3~5格，效率提升300%

### 3.3 工具不占背包空间

| 原版 | 重制版 |
|:----|:------|
| 锄头、浇水壶等工具占据6~8格背包 | 工具固定在快捷栏，**不占背包空间** |
| 背包12格，工具占一半 | 背包全部用于装物品 |
| 需要频繁整理背包腾出空间 | 无需为工具预留位置 |

### 3.4 快速打招呼/范围社交

| 项目 | 内容 |
|:----|:-----|
| 类型 | 重制版新增 |
| 功能 | 一次性向范围内的多个NPC打招呼 |
| 效果 | 同时增加多位NPC的好感度 |

## 七、被移除的内容

### 7.1 麦克风吹气机制（已移除）

| 原版NDS机制 | 移除原因 |
|:-----------|:---------|
| 对着NDS麦克风吹气可加速风车 | Switch/Steam无此硬件功能 |
| 吹气效果持续到次日 | 无替代（已用风力系统本身补偿） |
| 吹气+大风天可极限加速 | 调整平衡性，暴风/台风效果增强 |

### 7.2 其他可能移除的要素

| 要素 | 状态 | 说明 |
|:----|:----|:-----|
| 部分NDS专属迷你游戏 | 可能移除或重制 | 替换为3D版本 |

| 双屏界面 | 完全移除 | 统一为单屏UI |

## 九、结论：重制版的价值

### 9.1 保留了什么
- 原作的核心经济系统（集市摆摊）
- 风车+风力系统（核心特色）
- 三大风车的功能定位
- 每周六集市的节奏
- 四季作物与动物系统
- 恋爱结婚生子的社交线

### 9.2 改进了什么
- **画面**：从2D到全3D的质变
- **探索**：新增滑翔伞+二段跳
- **效率**：工具不占背包+跳跃组合技
- **社交**：新增角色+同性婚姻+语音
- **引导**：支线任务系统
- **操作**：现代化手柄/键鼠适配

### 9.3 重制版评价

重制版被誉为"**近十年里起点与门槛都最高的牧场物语**"，拥有系列迄今为止最精致的自然风光。IGN评价Switch 2版本"优化流畅"。作为一款2025年的重制作品，它在保留原作灵魂的同时，用现代化品质将这款2008年的经典带到了新平台上，是一次成功的重制。
