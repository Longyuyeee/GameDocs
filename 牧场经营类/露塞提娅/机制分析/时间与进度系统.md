# 时间与进度系统 - 露塞提娅:道具屋经营妙方(Recettear)

> 游戏版本: Recettear: An Item Shop's Tale v1.0+ (PC, 2007/2010年)
> 数据来源: recettear.fandom.com, GameFAQs

## 1. 时间结构

### 1.1 一天周期

| 时段 | 活动 |
|------|------|
| 白天 | 商店营业(开店→顾客进入→定价→出售) |
| 黄昏 | 关店/市场采购/行会合成 |
| 夜间 | 地牢探索(雇佣冒险者→战斗→收集物品) |

- 每天4个时段: 早晨→开店→黄昏→夜间
- 开店期间时间暂停(顾客模式)——非实时时钟
- 地牢探索中时间推进(每层消耗时间)

### 1.2 30天周期

| 周 | 天数 | 还款额 | 阶段 |
|:--:|:--:|:---:|------|
| 第1周 | 1-7天 | 10,000 | 教学/适应 |
| 第2周 | 8-14天 | 30,000 | 学会定价/地牢入门 |
| 第3周 | 15-21天 | 80,000 | 效率化/招募多个冒险者 |
| 第4周 | 22-30天 | **500,000** | **终极压力/最终boss前** |

## 2. 一天流程优化

### 2.1 高效日常

```
早晨 → 开店(优先安排高价物品在柜台)
  → 观察顾客反应→调整价格
  → 关店→清点收入
黄昏 → 去市场检查价格浮动(低价=大量买入)
  → 去行会合成新装备
  → 给自动售货机补货
夜间 → 选择合适地牢+雇佣冒险者
  → 地牢清怪+开宝箱→获取物品
  → 返回→整理背包→为明天开店备货
```

## 3. 进度门控

| 门控类型 | 示例 |
|------|------|
| 商人等级 | 合成/店面扩张/售货机/功能解锁(Lv1-50) |
| 还款额 | 主线进度(第7/14/21/30天) |
| 地牢通关 | 新地牢解锁(试炼→翡翠→琥珀→黑曜→拉碧斯) |
| 冒险者招募 | 特定条件(Charme=击败/Caillou=订单/Griff=黑暗氛围) |
| 顾客好感度 | 预算上限提升 |

## 4. 时间压力vs自由度

| 维度 | 压力 | 自由 |
|------|:--:|:--:|
| 30天还款 | 硬截止——不到=Bad End | Bad End=保留进度重来 |
| 每天活动 | 只能做有限事情(开店→采购→地牢) | 具体做什么自主决定 |
| 地牢探索 | 每层消耗时间 | 可随时返回 |
| 市场波动 | 错过高价窗口要等下次 | 波动周期快(数天) |

## 5. 进度加速策略

| 策略 | 效果 |
|------|------|
| 地牢→花瓶(21,000g基准) | 翡翠之路15层→宝箱→花瓶→早期暴富 |
| 蓝色市场大量采购 | 低价囤货→红色市场出售→套利 |
| Just Pin专注 | 加速好感→更快提升顾客预算 |
| 跳过铜/银装备 | 直接合成高级装备→省材料+时间 |
| Alouette关系优先 | 最快提升支付力→终局出售渠道 |

## 6. 设计分析

1. **30天硬截止=最激进的生存压力**: 不是"可以无限拖延"而是"不到500K就GG"——此后所有商店游戏都没有这种硬核度
2. **Bad End=保留进度重来**: 不是全部清零→减少挫败感的同时保留压力——聪明的Roguelite设计
3. **"4天=1周"的周期节奏**: 快速循环→7天还款→市场波动→策略不断调整——没有"无聊的重复天"
4. **各周还款额递增=难度曲线**: 1万→3万→8万→50万——每周的跳跃幅度迫使玩家不断优化策略
