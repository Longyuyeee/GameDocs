# 团队与开发历程 — 符文工房4(Rune Factory 4)

> 开发商: Neverland Co. (已破产) / Marvelous AQL | 日本
> 发行年份: 2012年(3DS) / 2019年(Switch豪华版) | 团队规模: **约36人**(全公司)

## 2. 开发历程

| 阶段 | 时间 | 关键事件 |
|------|:--:|------|
| 公司成立 | 1993年 | Neverland在东京成立 |
| RF4开发 | 2010-2012 | 3DS平台开发 |

| RF5立项 | 2012年8月 | 因RF4成功，Hashimoto批准RF5开发 |

| 员工救援 | 2014年2月 | Marvelous雇佣了大部分前Neverland员工 |
| RF4豪华版 | 2019年7月25日 | Switch HD移植，Marvelous新团队开发 |
| RF5 | 2021年 | 由HAKAMA Co.(桥本嘉史创立，Marvelous旗下)开发 |

## 3. 破产原因

Neverland的破产不是因为RF4失败——RF4恰恰是成功的(日本超15万份)。破产是因为**其他项目的销售乏力+过多贷款+融资失败**。RF4成了Neverland的**遗作**。

关键细节: RF4的**主程序员未被Marvelous雇佣**，直接导致欧洲本地化遇到技术障碍——欧洲版最终于2014年12月以eShop独占数字版形式发布，比日本晚了2年半。

## 4. 市场表现

| 指标 | 数据 |

| 日本销量 | 15万+份 |
| 全球估计 | 45-50万份 |
| 对Marvelous的影响 | RF4成功使Marvelous将利润预期**上调106.7%** |
| 系列延续 | 直接催生了RF5(虽因破产推迟了8年) |

## 5. 设计启示

1. **成功的游戏≠成功的公司** — RF4是系列最成功作品，但Neverland仍然破产了。单款游戏的成功不能弥补公司整体的财务问题
2. **IP救活了但团队散了** — Marvelous保留了符文工房IP和前员工，但主程序员的流失直接导致欧洲发布延迟2.5年
3. **HD移植 = 保留火种** — 2019年的RF4豪华版让新团队重新熟悉这个系列，为RF5铺路
4. **桥本嘉史 = IP的灵魂** — 他是RF4的制作人和编剧，Neverland破产后他创立HAKAMA继续开发RF5——关键人物的连续性保住了IP
