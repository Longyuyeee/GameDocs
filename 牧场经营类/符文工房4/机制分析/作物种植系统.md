![RF4 农场](https://www.runefactory4.com/img/system/ss_farm01.jpg)

> 游戏版本: 符文工房4 豪华版(Rune Factory 4 Special) (Switch 2019年 / PC 2021年)

## 1. 系统概述

符文工房4的种植系统在RF3基础上大幅扩展，新增**土壤品质五维属性**和**巨大化作物(Giant Crop)**机制，是系列中种植系统最深的一作。

## 3. 土壤品质系统(RF4独有)

使用放大镜(Magnifying Glass)可查看每格土壤的5项属性:

| 属性 | 影响 | 提升方式 |
|------|------|---------|
| **HP(健康度)** | 可种植次数上限 | 翻埋枯萎草恢复HP; HP归零=作物死亡 |
| **ATK(品质)** | 作物售价 | Greenifier(绿肥)提升 |
| **SPD(生长速度)** | 缩短成熟天数 | Formula B/C 加速, 效果累积 |
| **DEF(抗性)** | 抵抗台风/暴风雨 | Wettable Powder(湿润粉)提升 |
| **SIZE(体积)** | 能否巨大化 | Giantizer(巨大化剂)提升 |

### 3.1 土壤管理策略
- **翻埋枯萎草**: 自动恢复该格HP值
- **玉米/四叶草**: 种植后枯萎会在土壤留下营养(自动恢复HP)
- **不腐烂剂(No Rot α/β)**: 防止作物枯萎对土壤的损耗
- 每格土壤独立计算，不同格可有完全不同的属性

## 5. 巨大化作物(Giant Crop)

### 5.1 自然巨大化
- 条件: 相邻2×2格子种植**同一种**作物，且该4格全部成熟
- 概率: 取决于各格SIZE属性，SIZE越高概率越大
- 效果: 4格合并为1个巨大作物，**收获量×2**
- **帝王作物(King Crop)**: 3×3格(9格)=1帝王作物，**收获量×4**

### 5.2 强制巨大化
- 使用巨大化剂(Giantizer)可100%触发巨大化
- 不需要随机等待

## 6. 种子获取与品质循环

| 方式 | 说明 |
|------|------|
| 镰刀收种 | 对成熟作物使用镰刀，收获Lv=作物Lv的种子 |
| 商店购买 | Lv1基础种子 |
| 行商由(Yue) | 随机出售黄金作物种子 |
| 迷宫宝箱 | 随机掉落种子 |
| 反复种植 | Lv1→种植→镰刀→Lv1种子→再种植... 理论可无限循环提升 |

## 7. 设计特点

1. **五维土壤系统**: 远超同类牧场游戏的种植深度，"每格土壤都是独立个体"
2. **巨大化机制**: 2×2和3×3的空间策略增加种植布局的趣味性
3. **迷宫农场**: 将种植分散到战斗场景中，战斗探索与经营无缝连接
4. **镰刀收种循环**: 品质(等级)提升的正反馈循环驱动玩家持续种植
5. **黄金作物**: 行商随机出货制造稀缺性，高等级黄金作物是后期顶级收入
