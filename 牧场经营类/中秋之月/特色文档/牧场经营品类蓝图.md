# 牧场经营品类蓝图 - 中秋之月(Harvest Moon: Back to Nature)

> 游戏版本: Harvest Moon: Back to Nature (PS1, 1999年)

## 一、概述

中秋之月(BTN)是**现代牧场经营游戏品类的奠基之作**。虽然SFC上的初代Harvest Moon(1996)先出现，但BTN确立了几乎所有后续牧场游戏都沿用的核心模板。2003年的矿石镇(GBA版)是BTN的直接移植/扩展——2020年的重聚矿石镇是BTN的完全3D重制。

**几乎所有"牧场经营"品类的基础设计都在BTN中首次出现或首次完善。**

## 二、BTN首创/首次完善的品类模板

| 设计元素 | BTN中的实现 | 后续影响 |
|------|------|------|
| **心色等级** | 黑→紫→蓝→绿→黄→橙→红 7级 | 全品类标准 |
| **蓝色羽毛求婚** | 1,000G购买/橙心解锁 | 求婚石/吊坠/花环的前身 |
| **出货箱结算** | 下午5点统一结算 | 全品类标准 |
| **工具四级升级** | 铜→银→金→秘银(使用次数解锁) | 升级模板 |
| **冬季采矿转型** | 无种植→转向矿洞→矿石+宝石收入 | 季节节奏模板 |
| **加工机器** | 蛋黄酱/奶酪/毛线机(20,000G+金刚矿) | 加工增值链初心 |
| **力量果实** | 隐藏/永久体力上限提升 | 全品类沿用 |
| **旅行商人Won** | 限定种子+高价物品 | 限定商人设计初心 |
| **情敌系统** | 第3年情敌可抢先结婚 | 社会模拟深度 |
| **动物死亡机制** | 不喂食→生病→死亡 | 后作大多取消(太硬核) |
| **节日系统** | 每月1-2个节日(赛牛/赛狗/感恩节等) | 季节节奏节点 |
| **情敌配对** | 5候补→5情敌→独立配对剧情 | 社区叙事深度 |
| **房屋扩建系统** | 厨房→卧室→温室(木材+金币) | 渐进式成长 |

## 三、BTN→后续游戏的传承线

```
Harvest Moon SFC (1996, 初代——概念验证)
  ↓
Back to Nature PS1 (1999, 模板完善——品类蓝图)
  ↓ 移植+扩展
Friends of Mineral Town GBA (2003, 矿石镇——移动端经典)
  ↓ 3D重制
Story of Seasons: Friends of Mineral Town (2020, 重聚矿石镇)
  ↓ 同时影响了
Stardew Valley (2016) ← 直接灵感来源
Rune Factory系列 ← 牧场+战斗的变体
Coral Island / Sun Haven / Roots of Pacha ← 后星露谷时代
```

## 四、BTN中"太硬核"后来被废弃的设计

| 设计 | 废弃原因 |
|------|------|
| 动物死亡 | 太残酷——情绪波动大 |
| 情敌deadline | 时间压力过大——限制自由恋爱 |
| 工具使用次数限制升级 | 太隐藏——新手不理解 |
| 冬季完全无种植 | 太停摆——后作加入冬季作物/温室 |
| 无同性恋爱 | 时代局限——后作加入 |

## 五、BTN至今仍被称赞的设计

| 设计 | 原因 |
|------|------|
| 情敌配对系统的叙事深度 | 不是"你追的NPC坐着等你"——她们有自己的生活和恋爱对象 |
| 角色日程的真实感 | 每个NPC有独立的时间表——医院只在白天开门/酒馆晚上热闹 |
| 矿洞的探索深度 | 楼层递进+矿物层级分布+稀有矿深层——探索驱动 |
| 力量果实的隐藏设计 | 不是"升级系统"而是"探索奖励"——驱动全游戏探索 |
