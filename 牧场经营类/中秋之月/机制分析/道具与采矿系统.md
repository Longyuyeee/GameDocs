| 等级 | 效果示例（水壶） | Dva 需求 | 制作材料 | 说明 |

## 二、采矿系统

### 2.1 矿洞总览

Winds of Anthos 拥有 **3 个以上主要矿洞区域**，分布在不同的地图位置。

| 矿洞 | 位置 | 层数结构 | 独特矿产 |
|:----:|:----:|:--------:|:--------:|
| **东卡里森矿场** | Lectenbury 附近 | 不限层数（随机/无限往下） | 铁/青铜/水晶/玻璃宝石 |
| **帕斯蒂亚矿场** | 中部区域 | 不限层数 | 银/金/蛋白石/蓝宝石/坦桑石/钻石(20层+) |
| **莱布库亨矿场** | Herbstburg 区域 | **100层**（固定楼层） | 钛/铂(10+)/精金(60+)/翡翠/石榴石/红宝石/翠绿/紫翠玉(60+) |

> 注意：前两个矿洞为"不限层数"——这与传统牧场游戏的固定楼层矿洞不同。莱布库亨矿场为100层固定结构，有检查点机制。

### 2.2 采矿系统核心机制：雷达

本作采矿的最大创新是**雷达探测系统**，替代了传统"随机挖/敲"的模式。

| 雷达功能 | 说明 |
|:--------:|------|
| **探测宝物** | 使用雷达可定位周围隐藏宝物位置 |
| **探测梯子** | 可定位下层的梯子位置（不限层数矿洞关键） |
| **探测范围** | 随工具升级扩大 |
| **使用方法** | 装备雷达后按对应按钮激活探测波 |

> 雷达是否持续消耗能量/电池——社区对此缺乏统一记录。推测为不消耗或者消耗极低。

### 2.3 矿洞机制

| 机制 | 不限层数矿洞 | 莱布库亨(100层) |
|:----:|:----------:|:--------------:|
| **每层梯子** | 使用雷达定位 | 固定位置或随机（待确认） |
| **检查点** | 无（每层随机） | 每10层有检查点 |
| **陷阱** | 推测存在 | 掉落1-5层，损失1-5心 |
| **深层策略** | 无底洞式→越深层矿石越稀有 | 60+层出精金/紫翠玉 |
| **电力/体力消耗** | 每敲一下消耗体力 | 同左 |

> 不限层数矿洞的"深度"是否有上限——官方未确认。社区目前最深报告约数百层，怀疑有底层限制。

## 五、设计特点分析

### 5.1 与传统牧场游戏的对比

| 维度 | Winds of Anthos | 星露谷物语 | 牧场物语: 矿石镇 |
|:----:|:--------------:|:----------:|:----------------:|
| **工具升级材料** | Dva+矿物（冶炼体系） | 矿物+金币 | 矿石+金币 |
| **升级解锁方式** | 材料制作（非使用次数） | 使用次数解锁 | 使用次数解锁 |
| **采矿雷达** | **有（核心创新）** | 无 | 无 |
| **矿洞结构** | 不限层+固定层混合 | 固定层（120层） | 固定层（255层/65535层） |
| **Wisp系统** | 有（资源+能力双体系） | 无 | 无 |
| **矿洞数量** | 3+ | 2（矿井+骷髅洞） | 1（2个入口） |
| **钓竿获取** | NPC任务 | 任务 | 任务（海滩） |

### 5.2 核心创新

1. **雷达系统消除"挖矿随机性"**：传统牧场游戏中找楼梯是体力和运气的双重考验。雷达将采矿从"随机乱挖"变为"定位→精准挖掘"，降低了挫败感但增加了策略性。

2. **Dva 冶炼体系**：不再将矿石直接交给铁匠，而是先熔炼为 Dva（材料币）再用于制作。增加了一层加工环节——使采矿和锻造之间的链条更长。

3. **不限层数矿洞**：理论上玩家可以无限往下挖（越深越稀有），这改变了"到底层就毕业"的传统认知。

### 5.3 局限性

| 问题 | 说明 |
|:----:|------|
| 雷达系统的操作指引不足 | 新手可能不知道如何定位宝物 |
| Dva 换算表未整合进游戏 | 玩家需要外部资料/试错 |
| 不限层矿洞缺乏终止标志 | "到底要挖到多深"缺乏反馈闭环 |
| Power Wisp 收集缺乏追踪工具 | 部分 Wisp 极其隐蔽，DocPad 追踪功能有限 |
