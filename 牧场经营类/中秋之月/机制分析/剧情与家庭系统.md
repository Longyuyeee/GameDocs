## 一、世界观设定

Winds of Anthos 采用了 Harvest Moon 系列的经典设定框架——但将舞台从"一个小镇"扩展为"一个世界"。

| 设定要素 | 说明 |
|:--------:|------|
| **世界观** | Anthos 大陆——丰收女神守护的大地 |
| **主角背景** | 来到 Anthos 大陆开始新生活的牧场主 |
| **核心目标** | 恢复丰收女神的力量（主线）+ 经营自己的牧场 + 与各村庄建立联系 |
| **主线驱动** | 通过行动唤醒沉睡的丰收女神，解锁更多区域和能力 |
| **世界状态** | 开放世界——各村庄和文化区域逐步解锁 |

> 本作的剧情呈现方式不同于传统牧场游戏——主线任务（恢复女神力量）是推动玩家探索世界的主要动力，而非纯粹的"自由经营"。

## 二、主线剧情结构

```
序幕：主角来到 Anthos 大陆 → 在 Lectenbury 开始牧场生活
  ↓
第一阶段：经营初期的基本农场 + 认识各村庄
  ↓
第二阶段：通过 Harvest Wisp 和 Power Wisp 的收集，逐渐了解女神的故事
  ↓
第三阶段：解锁更多区域（火山/Zimagrad/Providence等）
  ↓
第四阶段：恢复丰收女神力量 → 世界焕发新生
  ↓
终局：自由经营 + 结婚/家庭 + 解锁所有内容
  ↓
DLC扩展：新角色/新区域/新内容
```

> 主线剧情的具体章节划分和通关时长，社区尚未有统一报告。推测完整主线约 30-50 小时（取决于探索速度）。

### 3.2 结婚条件

| 步骤 | 条件 | 说明 |
|:----:|:----:|------|
| 1 | **主线通关**（复活丰收女神） | 必须完成主线剧情 |
| 2 | **候补 5心 好感度** | 达到第五心的好感度阈值 |
| 3 | **房屋扩建至最大** | 约 420,000G + 材料 |
| 4 | **制作第二张床** | 在工作台制作 |
| 5 | **获得蓝色羽毛** | 复活女神后跟随蓝鸟获得 |
| 6 | **告白事件触发** | 约 3.5 心时触发告白事件 |

> 精确的 5 心好感度数值、房屋扩建材料清单和费用详见《候补好感度数据》和《全物品与商店数据》。

### 3.3 心事件系统

每位候补拥有 **5 个心事件**（Heart Events / Note Events）——在特定好感度阶段触发。

| 事件顺序 | 好感度条件（社区估算） | 触发内容 |
|:--------:|:--------------------:|:--------:|
| 第1事件 | 1心左右 | 初次结识+背景介绍 |
| 第2事件 | 2心左右 | 深入了解 |
| 第3事件 | 3心左右 | 关系升温/告白前奏 |
| 第4事件 | 4心左右 | 情感深化 |
| 第5事件 | 5心（满心） | 告白/最终事件 |

> 每个心事件的具体触发条件（时间/地点/天气/已完成前置任务）——社区数据尚不完整。部分玩家报告事件触发需要考虑"当前所在区域"和"一天中的时段"。

### 3.4 同性婚姻

| 维度 | 说明 |
|:----:|------|
| **支持同性婚姻** | 是——10位候补无性别限制，任意主角性别可追求任意候补 |
| **实现方式** | 系统不检查主角和候补的性别匹配 |
| **DLC候补** | 同样无性别限制 |

> Winds of Anthos 是否在游戏中有专门的同性婚姻对话/剧情变体——据社区报告，心事件和婚后对话与异性婚姻基本相同。

## 六、设计特点分析

### 6.1 与传统牧场游戏的对比

| 维度 | Winds of Anthos | 星露谷物语 | 牧场物语: 矿石镇 | SoS系列 |
|:----:|:--------------:|:----------:|:----------------:|:--------:|
| **候补数量** | 10+2 (DLC) | 12 | 5 | 6+ |
| **同性婚姻** | 是 | 是 | 否（原版） | 部分有 |
| **主线绑定婚姻** | 是（需通关主线） | 否 | 否 | 部分有 |
| **心事件数量/人** | 5 | 14（含小事件） | 4 | 5-7 |
| **情敌系统** | 推测无 | 无 | 有 | 部分有 |
| **子女系统** | 待确认 | 有（有限） | 有（完整） | 有 |
| **DLC候补** | 有（2名） | 无 | 无 | 无 |

### 6.2 核心创新

1. **主线绑定婚姻**：结婚不再是"可选项"而需要完成主线（复活女神）。这一设计将婚姻从"随时可做的事"转变为"值得追求的终局目标"。

2. **全性别婚姻**：Natsume 自研系列首次实现——10 名基础候补 + 2 名 DLC 候补均支持任意性别。

3. **DLC候补扩展**：通过 DLC 追加角色，延长了游戏的社交内容更新周期。但同时也分割了完整体验。

### 6.3 局限性

| 问题 | 说明 |
|:----:|------|
| 婚姻绑定主线使得结婚成为终局内容 | 想"早期结婚"的传统牧场玩家可能感到受限 |
| 心事件触发条件不够透明 | 部分玩家报告没有明确的引导 |
| 子女系统存在不确定性 | 社区对婚后是否可以生育子女没有统一确认 |
| 婚后内容深度有限 | 婚后对话/互动模式可能不够丰富 |
| DLC角色与基础角色的内容差异 | DLC候补的心事件是否与基础候补同等丰富待确认 |
