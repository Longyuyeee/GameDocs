# 体力与健康系统 - 中秋之月(Harvest Moon: Back to Nature)

> 游戏版本: Harvest Moon: Back to Nature (PS1, 1999年)
> 数据来源: harvestmoon.fandom.com, GameFAQs

## 1. 体力(Stamina)

| 要素 | 说明 |
|------|------|
| 体力条 | 使用工具→逐渐消耗→归零=无法继续工作 |
| 恢复方式 | 吃东西(作物/料理)/泡温泉/睡觉 |
| 上限提升 | **力量果实(Power Berries)**——永久+体力上限 |
| 初始体力 | 约可完成半片田的浇水+翻耕 |

## 2. 疲劳(Fatigue)

| 要素 | 说明 |
|------|------|
| 机制 | 过度工作→累积疲劳→第二天体力上限减少 |
| 恢复 | 泡温泉(唯一高效恢复方式) |
| 0疲劳 | 蓝脸→正常体力恢复 |
| 高疲劳 | 红脸→体力上限大幅减少→可能晕倒 |
| 极端疲劳 | 晕倒→强制送回家→第二天体力极低 |

## 3. 温泉

| 属性 | 说明 |
|------|------|
| 位置 | 母亲山(矿洞旁) |
| 效果 | 浸泡→恢复体力+消除疲劳 |
| 时间消耗 | 1小时浸泡≈完全恢复(初始体力) |
| 策略 | 每天收工后泡1小时→保持疲劳为0 |

## 4. 力量果实(永久成长)

| 获取方式 | 数量 | 难度 |
|------|:--:|:--:|
| 矿洞深层随机 | 1-2 | 低概率 |
| 钓鱼随机 | 1 | 低概率 |
| 赛马奖励兑换 | 1 | 中等(需要奖牌) |
| 其他隐藏地点 | 2-3 | 探索 |
| **总计** | **~5-7个** | — |

- 每果实≈+15-20%体力上限
- 全部收集→体力上限≈初始2倍+

## 5. 体力管理策略

| 阶段 | 策略 |
|------|------|
| 初期(体力低) | 每天只做1-2件事(要么种田→要么采矿，不能兼顾) |
| 中期(2-3果实) | 日常农场+半天采矿——泡温泉后仍有体力 |
| 后期(5+果实) | 全天农场+矿洞+钓鱼——体力储备充足 |

## 6. 与后续游戏的对比

| 游戏 | 体力恢复 | 疲劳系统 | 永久提升 |
|------|------|:--:|------|
| **BTN(1999)** | 食物+温泉 | **有(独立维度)** | 力量果实 |
| 星露谷 | 食物+温泉 | 无(仅体力条) | 星之果实 |
| 符文工房4 | RP系统 | 无 | 等级成长 |

## 7. 设计分析

1. **体力+疲劳双维度=更真实的劳作模拟**: 仅体力条→可以用食物一直补充→疲劳系统让"连续劳作"有累积代价
2. **温泉=强制休息节奏**: 每天泡1小时温泉→自然将一天分为"工作→休息→继续"的节奏
3. **力量果实=探索驱动**: 不是"升级加体力"而是"探索隐藏地点"→驱动全游戏探索
4. **后续游戏简化**: 大多数后续游戏移除了疲劳系统→只保留体力条→简化了但失去了"过度工作"的真实感
