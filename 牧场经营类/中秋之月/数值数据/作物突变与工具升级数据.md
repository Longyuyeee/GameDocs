# 作物突变与工具升级数据 - 中秋之月(Harvest Moon: One World)

> 游戏版本: Harvest Moon: One World (Switch/PS4/Xbox/PC, 2021年, Natsume)
> 数据来源: Neoseeker Guide, GameFAQs, DiamondLobby, OutsiderGaming

## 一、作物突变系统(核心机制)

### 1.1 突变五要素

| 因素 | 说明 |
|------|------|
| **季节** | 必须在正确季节种植 |
| **地形** | 必须种在正确地形(P=平原/B=沙滩/D=沙漠/M=山/A=高山/V=火山/W=湿地/F=森林/SP=雪地) |
| **品质点(QP)** | 用肥料提升, 典型阈值QP 70-130 |
| **突变概率** | 条件满足时约40%/生长阶段 |
| **出货解锁** | 出货10-15个突变作物→商店解锁种子 |
