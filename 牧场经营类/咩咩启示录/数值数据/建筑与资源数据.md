## 四、建筑优先度决策树

```
开局(0-5天):
  庇护所(防疲惫) → 农场(防饥饿) → 神庙(信仰收入)

前期(5-15天):
  厕所(防疾病) → 烹饪火(食物效率+25%) → 木材厂(资源自动化)

中期(15-30天):
  治疗湾(疾病管理) → 监狱(异见管理) → 精炼厂(高级材料)
  清洁站(自动清洁) → 恶魔召唤阵(地牢辅助)

后期(30天+):
  农夫站II+丰收图腾(食物系统完全自动化)
  会幕III+供品雕像(信仰被动收入最大化)
  地下墓穴+停尸房(尸体管理)
  DLC特有建筑(交配/孵化/牧场)
```

## 五、资源加工链汇总

```
木材 → [精炼厂] → 木板(价值×2, 高级建筑)
石材 → [精炼厂] → 石块(价值×2, 高级建筑)
金矿 → [精炼厂] → 金条(价值×2-3, 高级建筑/装饰)
作物 → [烹饪火/厨房] → 熟食(饱食值+25-50%)
粪便/草 → [堆肥箱] → 肥料(农场产量↑)
尸体 → [墓地/教义Natural Burial] → 肥料(DLC)
蜘蛛丝 → [裁缝台] → 服装/高级建筑
草+种子 → [种子仓] → 自动种植
葡萄/啤酒花 → [酒馆] → 酒+Sin(DLC)
```

> **设计洞察**: 精炼是整个经济系统的核心转折点——在拥有精炼厂前，木材和石材仅用于基础建筑；精炼后解锁所有高价值建筑。建议在中期(信徒人数10+、食物已自给)后优先建造精炼厂。
