# 教派管理与信仰系统 - 咩咩启示录(Cult of the Lamb)

> 游戏版本: Cult of the Lamb v1.0+ (PC/Switch/PS/Xbox, 2022年, 含Relics/Sins/Woolhaven DLC)
> 数据来源: cult-of-the-lamb.fandom.com, IGN, GameTruth, GamePressure, progameguides.com

## 1. 系统概述

咩咩启示录的教派管理系统是**教派经营×Roguelite地牢**双循环的"经营侧"。玩家需要管理信仰(Faith)、忠诚度(Loyalty)、饥饿(Hunger)、卫生(Hygiene)、秩序(Order)五个维度的子系统。与牧场游戏的"让动物快乐产奶"不同,咩咩启示录的教派管理是"控制一群会生病、造反、老死的信徒"——这是一个压力持续存在的管理模式。

## 2. 信仰(Faith)系统深度解析

### 2.1 信仰的核心定位

信仰是教派的**核心存活指标**——比金币更重要。信仰归零意味着全面的异见爆发和教派崩溃。

| 属性 | 说明 |
|------|------|
| 满值 | 100 |
| 危险线 | <30(异见者开始出现) |
| 崩溃线 | <10(全面异见+大量叛逃) |
| 每日自然消耗 | -1~-5/日(取决于教徒数量+满意度) |

**信仰增加:**

### 2.3 信仰危机级联

信仰管理的核心挑战在于异见的**传染性**:

```
信仰<30 → 第一个异见者出现
  ↓ 1-2天不处理
传播→2-3个异见者 → 信仰每日-5~-15%
  ↓ 3-4天不处理
异见者开始偷金币(约10%/次) + 开始离开教派
  ↓ 5天+
半数信徒异见/离开/死亡 → 教派进入不可逆崩溃
  除非: Holy Day(+80信仰) 或 Brainwashing(锁满2天)紧急救场
```

### 2.4 信仰维持策略

| 阶段 | 维持手段 | 频率 | 优先级 |
|:--:|------|:--:|:--:|
| 前期(0-15天) | 每日布道+基础装饰 | 每日 | ★★★★★ |
| 中期(15-30天) | 婚礼+False Idols装饰 | 一次性+持续 | ★★★★ |
| 后期(30天+) | Holy Day做保险+全自动装饰流 | 应急 | ★★★ |
| 终局(DLC) | 建筑流Sacral Architecture | 持续 | ★★★ |

### 3.3 食物危机管理

```
食物短缺检测: 存量 < 信徒数×2
  ↓
立即行动:
  1. 使用Ritual of Harvest(75骨,作物立即熟)
  2. 使用Ritual of Feasting(75骨,+25信仰+喂饱)
  3. 全体切换草餐(Grass Eater教义)
  4. 地牢探索→专注食物房
  ↓
长期方案:
  1. 扩建农场+专职农夫
  2. 升级厨房(自动烹饪)
  3. Farmer Station II(自动收获)
```

## 5. 异见(Dissent)管理系统

### 5.1 异见者生成机制

```
信仰<30 + 信徒忠诚<40 → 异见者判定(每日)
  ↓
判定通过 → 信徒变为异见者:
  - 停止工作
  - 开始向周围信徒传播(每日判定)
  - 偷窃金币(约10%库存/次)
  - 最终会离开教派
  ↓
传播判定:
  周围信徒忠诚<60? → 有概率被感染
```

### 5.2 异见者处理手段

| 手段 | 信仰影响 | 骨消耗 | 效果 | 推荐度 |
|------|:--:|:--:|------|:--:|
| 囚禁(Prison) | 0 | 0 | 停止异见传播 | ★★★★★(首选) |
| 再教育(Re-educate) | 0 | 0 | 恢复忠诚(需Disciples) | ★★★★★ |
| 处决(Execute) | +5-10 | 0 | 移除异见者 | ★★★(紧急) |
| 升天(Ascend) | +5-10 | 75 | 移除+忠诚增益 | ★★★★ |
| 牺牲(Sacrifice) | +20(有教义) | 75 | 移除+永久Buff | ★★★(有教义时) |
| 忽视 | **-5~-15/日** | 0 | **教派崩溃** | **绝不推荐** |

### 5.3 最佳异见管理流程

```
发现异见者
  ↓
第一步: 立即囚禁(防止传播)
  ↓
第二步: 检查信仰值
  信仰<30? → Holy Day或Brainwashing救场
  信仰≥30? → 布道+装饰恢复正常
  ↓
第三步: 处理异见者
  有Disciples? → 让他们自动再教育
  无Disciples? → 等异见者冷静→释放→祝福提升忠诚
  再次异见? → 处决/升天
```

## 7. 传教远征(Missionary)系统

|:--:|:--:|:--:|------|:--:|

| Missionary II | 2 | 1-3天 | 同上+中概率信徒+金币 | 中 |
| Missionary III | 3 | 1-3天 | 全部+高概率信徒+稀有材料 | 高(信徒可能受伤) |

## 8. 每日经营流程优化

### 8.1 标准日流程

```
清晨(返回营地时):
  □ 收集一夜产出(农场/木材厂/石矿/精炼厂)
  □ 喂食信徒(优先熟食)
  □ 清理营地(粪便/呕吐物/尸体)
  □ 检查新蛋/幼崽(DLC)

上午:
  □ 布道(+10-20信仰+忠诚+虔诚)
  □ 祝福高价值信徒(5-10忠诚/次)
  □ 检查疾病信徒+送入Healing Bay
  □ 检查长老信徒→决定处置方式

中午:
  □ 建造/升级建筑
  □ 分配工作(新建筑需要新工人)
  □ 规划布局(用Renovation Ritual移动建筑)

下午:
  □ 准备地牢探险
  □ 确认信仰>50(否则先处理)
  □ 确认食物存粮足够
  □ 携带高忠诚恶魔(如果召唤了)

夜晚:
  □ 地牢探险(Crusade)
  │  注意: 如果信仰<30,不要长时间探索!
  │  除非已有Brainwashing锁定信仰

返回后:
  □ 立即处理战利品
  □ 售卖不需要的材料
  □ 购买急需的材料
  □ 确认信仰没有在下滑
```

### 8.2 批量操作技巧

| 技巧 | 说明 |
|------|------|
| 批量祝福 | 不要逐个祝福——优先只祝福Disciples候选 |
| 批量喂食 | 将食物放在明显位置→信徒自动进食 |
| 批量建造 | 一次性规划3-5个建筑→一次分配工人 |
| 批量地牢 | 确保状态满→一次性探索至Boss→往返高效 |

## 10. 跨游戏对比

| 维度 | 咩咩启示录 | 双点医院(Two Point Hospital) | 环世界(RimWorld) |
|------|------|------|------|
| 核心管理 | 信仰+忠诚+饥饿+卫生 | 治愈率+满意度+金钱 | 心情+食物+防御+社交 |
| 资源风险 | 信仰=最脆弱,会传染性崩溃 | 声誉=缓慢积累/快速失去 | 心情=波动但可控 |
| "员工"管理 | 信徒有寿命限制+异见传播 | 员工有技能但无寿命 | 殖民者无寿命限制 |
| 危机模式 | 异见→传播→全教派崩溃 | 传染病爆发 | 袭击/心理崩溃/疾病 |
| 自动化程度 | 中(清洁站/农夫站/厨房) | 高(员工自动工作) | 极高(mod可全自动) |
| 惩罚力度 | 25%资源损失+教派崩溃风险 | 星级下降+收入减少 | 角色死亡+装备损失 |

> **设计洞察**: 咩咩启示录的教派管理系统本质上是**反向的牧场经营**——传统牧场让动物快乐以获取更多产出,而咩咩启示录需要持续投入注意力来**防止信徒不快乐导致的系统崩溃**。信仰系统是这个设计的核心: 它不是"赚得更多"的增益系统,而是"防止崩盘"的防御系统。这种"防御性管理"创造了持续的紧迫感和压力——与游戏的黑暗风格完美匹配。
