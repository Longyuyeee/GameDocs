# 信徒生命周期系统 - 咩咩启示录(Cult of the Lamb)

> 游戏版本: Cult of the Lamb v1.0+ (2022年, 含Sins of the Flesh DLC)
> 数据来源: cult-of-the-lamb.fandom.com, IGN, GamePressure, progameguides.com

## 1. 系统概述

信徒生命周期系统是咩咩启示录"教派经营"侧的核心机制。每位信徒从招募到死亡大约经历55天的完整生命周期，期间经历获取、培养、衰老、死亡(或转化)四个阶段。与牧场游戏中"动物是永久资产"的范式不同，咩咩启示录的信徒是**消耗性资源**——需要持续补充，且每位信徒都需要在生命周期结束前做出"处置决策"。

## 3. 一生阶段:完整生命周期

```
 招募(第0天)
   ↓
 幼年(0-2天)——仅限孵化信徒(DLC),需Nursery照顾
   ↓
 少年(2-3天)——仅限孵化信徒,需Nursery照顾
   ↓
 成年(3-44天)——核心劳动阶段
  │  ├─ 日常工作(采集/烹饪/清洁/建筑)
  │  ├─ 虔诚生成(0.5-2/日,受Faithful影响)
  │  └─ 社交互动(打架/传教/交配)
   ↓
 长老(45-55天)——停止工作!
  │  必须做出处置决策:
  │  ├─ 升天(Ascend)→全员忠诚↑+虔诚
  │  ├─ 牺牲(Sacrifice)→永久力量加成
  │  ├─ 复活(Resurrection)→重置年龄→回到成年
  │  └─ 自然死亡→-5~-20信仰(不推荐)
   ↓
 死亡(55天+)—遗体处理:
    ├─ 埋葬(Grave)→无惩罚
    ├─ 食用(Cannibal教义)→+5信仰
    └─ 腐烂→降低卫生→疾病风险
```

### 3.1 长老决策数据对比

| 决策 | 骨消耗 | 忠诚度影响 | 资源产出 | 长期影响 | 推荐场景 |
|:--:|:--:|:--:|:--:|------|:--:|
| 升天(Ascend) | 75 | 全员+5-10 | +虔诚 | 释放床位 | 低忠诚长老 |
| 牺牲(Sacrifice) | 75 | 按教义 | 永久力量+1级 | 永久Buff | 满忠诚高级信徒 |
| 复活(Resurrection) | 75 | 保留 | 保留所有特质+技能 | **最保值的75骨投入** | **所有高级信徒** |
| 自然死亡 | 0 | -5~-20 | 尸体 | 卫生风险 | 仅资源极度短缺时 |

## 4. 忠诚度深度系统

### 4.1 忠诚度判定逻辑

```
每日判定:
  基础需求满足?(有床+有食物+卫生可接受) → 是: +1-3 忠诚
  基础需求未满足? → 否: -2-5 忠诚
  异见者传播? → 否: -1-3 忠诚(周围信徒)
  目睹死亡/献祭? → -5-20 忠诚(取决于教义)

主动提升:
  祝福(Bless): +5-10 忠诚(每日1次/人)
  送礼(Gift): +10-20 忠诚(随礼物价值)
  Confession Booth: +75 忠诚(每日1次/人,极高价值)
  布道(Sermon): +1-3 忠诚(全体)
```

### 4.2 忠诚度阈值效果

| 忠诚度范围 | 工作效率 | 异见抗性 | 特殊行为 |
|:--:|:--:|:--:|------|
| 0-19 | -30% | 极低 | 可能拒绝工作+传播异见 |
| 20-39 | -15% | 低 | 易受异见影响 |
| 40-69 | 100% | 中 | 正常 |
| 70-89 | +15% | 高 | 额外虔诚+10% |
| 90-100(Disciples) | +25% | **免疫** | 自动再教育异见者 |

## 5. 信徒特质交互系统

### 5.1 特质冲突规则

| 冲突组合 | 效果 | 处理 |
|:--:|------|------|
| Lustful + Celibate | 繁殖概率归零 | 分离管理 |
| Coprophiliac + Germaphobe | 互相-10信仰 | 避免同室 |
| Jerk + 任何 | Jerk持续骚扰他人 | Jerk优先献祭 |
| Hot Tempered + Hot Tempered | 频繁打架 | 分开居住 |
| Snorer + 周围信徒 | 他人睡眠中断→疲惫 | 单独床位 |

### 5.2 特质优先级管理

**必须立即处理的负面特质:**
1. **Poet** (不工作) → 献祭/升天/驱逐
2. **Spy** (偷钱离开) → 发现后立即处理
3. **Jerk** (持续性破坏) → 管理成本远超价值
4. **Catatonic** (不工作) → 治疗或处理

**可以接受的负面特质:**
- Faithless(-15%虔诚): 不影响工作, 只是信仰产出低
- Cynical(-15%忠诚速度): 培养成本高但可管理
- Snorer: 只要有独立床位即可

**最优正面特质组合:**
- Industrious(+15%工作) + Faithful(+15%虔诚) + Gullible(+15%忠诚速度) = "完美工人"
- Naturally Obedient(招募+10信仰) = 开局神特质

## 6. 信徒死亡/处置链

```
死亡事件
  ↓
触发信仰损失: -5(普通) / -20(无教义)
  ↓
尸体出现:
  未处理→腐烂→卫生降低→疾病风险↑
  埋葬(有Grave)→无后续影响
  食用(Cannibal教义)→+5信仰+食物
  ↓
最终处置:
  复活(有Resurrection教义)→75骨→重生
  升天(仪式)→清理+忠诚增益
  永久消失
```

| 属性 | 数据 |
|------|------|
| 获取 | Demonic Summoning Circle |
| 作用 | 地牢辅助(跟随战斗) |
| 来源 | 选择1个信徒→临时转化为恶魔 |

| 等级提升 | 高忠诚信徒→更强大恶魔 |

## 8. 跨游戏对比

| 维度 | 咩咩启示录 | 环世界(RimWorld) | 守墓人(Graveyard Keeper) |
|------|------|------|------|
| 信徒寿命 | 55天(固定) | 数年(随机) | 不死(僵尸) |
| 忠诚度系统 | 有(4级) | 无(心情代替) | 无 |
| 特质系统 | 2-3个特质 | 复杂特质树 | 无 |
| 年龄管理 | 长老必须处置 | 老人可工作 | 无效 |
| 复活机制 | 75骨/人 | 有(科技要求高) | 无 |
| 繁殖 | DLC可有 | 核心系统 | 无 |
| 异见传播 | 有(传染性) | 有(社交影响) | 无 |

> **设计洞察**: 咩咩启示录的生命周期是**加速版的殖民管理**——55天的寿命在游戏节奏中相当于现实1-2小时。这意味着玩家必须不断招募新血、管理老人、做"道德"抉择。相比环世界的"长期可持续发展"，咩咩启示录更强调"短期的残酷决策"——与游戏整体黑色幽默风格一致。
