# 营地建设与资源系统 - 咩咩启示录(Cult of the Lamb)

> 游戏版本: Cult of the Lamb v1.0+ (2022年, 含Relics/Sins/Woolhaven DLC)
> 数据来源: cult-of-the-lamb.fandom.com, IGN, TechRaptor

## 1. 系统概述

营地建设是咩咩启示录教派经营侧的"空间维度"。玩家在固定大小的营地上规划建筑布局,管理资源生产与消耗的平衡。与地牢探险的动态战斗不同,营地建设是**长期的、静态的资源管理游戏**——涉及生产效率、空间利用和自动化程度三个核心维度。

## 2. 营地基础设施

### 2.1 营地基本参数

| 参数 | 数据 |
|------|:--:|
| 营地面积 | 约15×15网格(固定,不可扩展) |
| 初始可用面积 | 约50%(被杂物/树木/石头占据) |
| 最大建筑数 | 约30-40个(取决于建筑大小) |
| 营地位置 | 固定的,不可移动(仅可在营地内调整布局) |
| Renovation仪式 | 免费移动建筑(无冷却,重要!) |

### 2.2 营地清理

| 障碍物 | 清理工具 | 所需时间 | 产出 |
|:--:|:--:|:--:|:--:|
| 树木 | 斧/信徒 | 数秒 | 木材 |
| 石头 | 镐/信徒 | 数秒 | 石材 |
| 杂草 | 徒手 | 瞬间 | 草 |
| 装饰废墟 | — | — | 不可清理(属于固定装饰) |

> 初期投入10-15分钟清理营地是值得的——每棵树=木材,每块石=石材。清理面积越大意味着可建造空间越大。

### 3.3 装饰品(Decorations)系统

装饰品有两个核心功能: **提升信仰**和**产生虔诚**。

| 装饰品 | 材料成本 | 信仰效果 | 虔诚效果 | 范围 |
|------|:--:|:--:|:--:|:--:|
| 木灯笼(Wooden Lantern) | 2木 | +1 | — | 小范围 |
| 旗帜(Flag) | 1木+10草 | +1 | — | 小范围 |
| 骨雕(Bone Sculpture) | 4-5骨 | +2-3 | +1/日 | 中范围 |
| 骨拱门(Bone Arch) | 4骨 | +3 | +2/日 | 大范围 |
| 王冠喷泉(Crown Fountain) | 5石 | +3 | +1/日 | 中范围 |
| 主教战利品 | ~4金条+区域材料 | +5 | +3/日 | 大范围 |
| 怪物雕像(Monster Statue) | 特殊获取 | +4 | +2/日 | 中范围 |

> **关键策略**: 装饰品放置位置影响其效率。将它们放置在神庙周围(信徒聚集处)和高频通行路径上,可以获得最大覆盖率。更新版本中,装饰品有范围指示器,可帮助优化布局。

## 4. 资源经济系统

### 4.1 资源分级

| 等级 | 资源 | 获取难度 | 核心用途 |
|:--:|------|:--:|------|
| 基础 | 木材/石材/金币 | 低 | 前中期建筑 |
| 中级 | 木板/石块/金条/肥料 | 中 | 高级建筑 |
| 高级 | 蜘蛛丝/Crystal Shard | 高 | 终局建筑/DLC |
| DLC | Sin/羊毛/Rotburn/Charged Shard | 高 | DLC专属内容 |
| 特殊 | 骨(Bones) | 中 | 仪式(75/次) |
| 货币 | 虔诚(Devotion) | 持续 | 神学解锁 |

### 4.2 资源生产与消耗平衡

```
前期(0-15天):
  收入: 地牢采集(主动) + 少量被动(信徒手动)
  消耗: 基础建筑(木/石/金)
  瓶颈: 木材(几乎所有建筑都要)
  策略: 手动砍树+捡石+地牢优先拿木/石

中期(15-30天):
  收入: 木材厂+石矿(被动,1人/建筑)
  消耗: 精炼材料+高级建筑
  瓶颈: 木板/石块(精炼厂效率)
  策略: 升级精炼厂II+多派信徒到木材厂/石矿

后期(30天+):
  收入: 全自动(设施+传教+装饰供给)
  消耗: 终局建筑+DLC内容
  瓶颈: 蜘蛛丝/Crystal(需要特定地牢区域)
  策略: 地牢目标→特定区域刷材料
```

### 4.3 金币经济平衡

| 金币获取途径 | 单次量 | 频率 | 备注 |
|:--:|:--:|:--:|------|
| 地牢Boss | 100-300 | 每周目1次 | 最大单笔 |
| 地牢宝箱 | 20-80 | 每轮多次 | 稳定收入 |
| 供品箱出售 | 不定 | 持续 | 材料换金 |
| 钓鱼稀有鱼 | 20-50 | 随时 | 需钓鱼竿 |
| Ritual of Enrichment | 不定 | 7天CD | 不稳定 |
| 传教远征 | 10-40 | 1-3天 | 附带其他 |

> **金币经济学**: 金币最初看似紧缺(早期建筑都要金币),但随着地牢推进很快变得充裕。实际瓶颈从不是金币本身,而是**建筑材料的产出速度**。因此, 不要花时间专门"刷金币"——专注推进地牢和自动化资源生产才是正确策略。

## 5. DLC扩展内容

### 5.1 Sins of the Flesh DLC — 罪孽经济

| 新增建筑 | 功能 | 资源循环 |
|:--:|------|------|
| Drinkhouse(酒馆) | 信徒饮酒→产生Sin | 食物→酒→Sin |
| Mating Tent(交配帐篷) | 消耗Sin→产蛋 | Sin→新信徒 |
| Hatchery(孵化器) | 孵化蛋 | 2-4天→新信徒 |
| Nursery(托儿所) | 照顾幼崽 | 保障幼崽生存 |
| Drum Circle(鼓圈) | 产生Sin | 持续Sin收入 |

Sin经济创造了一个**可选但强力的次级资源循环**: 食物→酒→Sin→新信徒。这让信徒繁殖成为可能,但也带来了"Sin管理"的新挑战(高Sin信徒忠诚度降低)。

### 5.2 Woolhaven DLC — 牧场与季节

| 新增系统 | 说明 |
|:--:|------|
| 动物牧场 | 养动物(非信徒,真正的牧场动物) |
| 季节系统 | 冬天气温降低→需要供暖 |
| 羊毛经济 | 羊毛作为DLC专属货币 |
| 驱魔祭坛 | 可修改信徒特质(3级) |
| 腐化系统 | 腐化信徒+腐化材料 |

Woolhaven DLC为营地增加了**真正的"牧场"元素**——玩家终于可以养一些不会老死/异见/造反的动物了。同时, 季节系统为营地管理增加了新的维度: 冬季需要供暖规划。

## 6. 营地布局策略

### 6.1 功能区划分

```
[A区: 居住区]
  └─ 庇护所集群 + 厕所 + 清洁站
  → 信徒睡眠+卫生中心

[B区: 食物区]
  └─ 农场(6-10地块)+ 农夫站 + 厨房 + 堆肥箱
  → 食物生产+加工中心

[C区: 信仰区]
  └─ 神庙 + 祭坛 + 会幕 + 告解室 + 供品雕像
  → 信仰+虔诚核心区

[D区: 资源区]
  └─ 木材厂 + 石矿 + 精炼厂 + 仓库
  → 材料生产中心

[E区: 特殊区]
  └─ 治疗湾 + 监狱 + 恶魔召唤阵 + 传教所
  → 秩序+应急功能区

[F区: 装饰走廊]
  └─ 连接各区的通道→放置装饰品最大化覆盖
```

### 6.2 布局原则

| 原则 | 说明 |
|------|------|
| 密集布局 | 建筑之间紧贴放置(节省空间,最多可放35-40个建筑) |
| 装饰品走廊 | 信徒通行路径上放置装饰品—最大化信仰/虔诚覆盖 |
| 神庙居中 | 神庙放在营地中心—布道时的信徒聚集效应 |
| 农场边缘 | 农场放在营地边缘—不占核心通行区 |
| 升级时保留原位置 | 升级建筑在原基础上建造(不能移动后再升级) |
| 留1-2格通道 | 所有建筑之间至少留1格通道(信徒寻路AI需要) |

## 7. 跨游戏对比

| 维度 | 咩咩启示录 | Dinkum | 守墓人(Graveyard Keeper) |
|------|------|------|------|
| 营地面积 | 固定,不可扩展 | 可扩展(买地) | 固定+可解锁 |
| 建筑密度 | 高(40+/小区域) | 低(建筑大) | 中 |
| 自动化程度 | 中(工人建筑) | 低(手动为主) | 高(僵尸自动) |
| 资源链条深度 | 3-4级(木→板→建筑) | 3-4级 | 5-6级(非常深) |
| 装饰系统 | 有用(加信仰+虔诚) | 纯粹美观 | 有用(加声誉) |
| DLC扩展 | 3个DLC持续扩展 | 免费持续更新 | 2个DLC |

> **设计洞察**: 咩咩启示录的营地建设是"实用主义"设计——所有建筑和装饰都必须有明确的功能价值,不存在纯粹的美学建筑。这与《动物森友会》式的"随心所欲装修"完全不同。有限的空间意味着玩家必须做出取舍和优先级排序——这与游戏整体"资源管理"的核心主题一致。装饰品虽然提供信仰/虔诚收益,但优先级始终低于功能性建筑——这是一种刻意引导: 先确保教派存活,再考虑"美化"。
