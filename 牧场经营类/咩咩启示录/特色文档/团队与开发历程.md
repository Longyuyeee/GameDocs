# 团队与开发历程 — 咩咩启示录(Cult of the Lamb)

> 开发商: Massive Monster | 澳大利亚墨尔本
> 发行年份: 2022年 | PC/Switch/PS/Xbox | 团队规模: **6人**(发布时)

## 1. 开发团队

| 成员 | 角色 |
|------|------|
| **Julian Wilton** | 创意总监(联合创始人) |
| **Jay Armstrong** | 设计总监(联合创始人) |
| **James Pearmain** | 美术总监(联合创始人) |

三位联合创始人于2014年开始合作，之前在Newgrounds和Kongregate上共同制作了**50多款小型Flash游戏**。2016年正式成立Massive Monster。

发布时团队仅6人——每个部门约1人，后期扩展到3名程序员。发布后缓慢扩张，坚持"小团队文化"。

## 4. 设计启示

1. **6人×50万美元 = Devolver史上最畅销IP** — 不是预算最大的项目，但是回报率最高的。小团队+少量资金+精准定位=超额回报
2. **Devolver主动加预算 = 好的发行商会逼你拿更多钱** — Massive Monster申请30万，Devolver说你至少需要50万。这体现了好的发行伙伴不只是"给钱"，而是"帮你看清你需要什么"
3. **50款Flash游戏 = 10年练功** — 三位创始人在做Cult of the Lamb之前已经合作了50多款小游戏。这不是"一夜成名"，是"10年磨一剑"
4. **6人团队发布后缓慢扩张** — 不是"拿到钱就扩招"，而是"坚持小团队文化"
