## 四、冬季

冬季无法在户外种植常规作物。建议利用冬季专注于: 采矿（加农矿洞）、社交送礼（冲刺好感度）、主线推进、料理制作、裁缝制作。

## 五、优质作物（Good/High Quality Crops）

### 5.1 品质等级与售价倍率

| 品质等级 | 获取条件 | 售价倍率（基础售价×） |
|:-------:|---------|:-----------------:|
| 普通 (Normal) | 未施肥或未达标 | ×1.0 |
| 好 (Good) | 连续施肥一定次数 | ×1.5 |
| 优质 (High) | 长期高级肥料 | ×2.0 |

| 主线需求 | 作物 | 数量 | 用途 |

| 绿钟任务 | 优质玉米→制作优质玉米粉 | 1个 | 还原魔女材料之一 |
| 绿钟任务 | 待补充 | -- | -- |

**种植建议**: 如果不确定选哪种作物冲优质，推荐**荞麦**（秋季种植）+ **卷心菜**（春季种植）——生长周期适中的作物更容易通过施肥获得优质品质。

## 九、设计要点

1. **草莓 = 春季经济引擎**: 种子360G略贵但10天首熟+3天复收的模式让它成为春季投资回报比最高的作物。如果春1就种下草莓，到春28可以收获约7次。
2. **南瓜 = 夏季单次王者**: 种子400G/售价500G，看似利润不高（100G），但品质提升后利润飞跃——优质南瓜售价1,000G，利润率150%。
3. **肥料与优质作物的关系**: 每天施肥 + 至少连续使用7天以上 → 约50%概率产出"好"品质 → 继续施肥可提升至"优质"。建议在黄钟任务前2个月开始准备。
4. **数据待核实项**: 精确的种子价格和售价数值在不同攻略来源中有细微差异，建议以Fogu.com的HM10数据为准进行二次验证。
