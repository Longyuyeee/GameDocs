## 二、子女五阶段成长

### 2.1 成长时间线

```
阶段1: 出生 (Birth)
  第1天 → 婴儿诞生，在婴儿床中
    ↓ 约14天
阶段2: 爬行 (Crawling)
  可在房间内爬行移动 → 玩家可互动对话
    ↓ 约14天
阶段3: 走路 (Walking)
  可在房间内外走路 → 跟随NPC成人
    ↓ 约14天
阶段4: 说话 (Talking)
  开始说话 → 表达性格 → 有对话内容
    ↓ 约14天
阶段5: 儿童之日 (Child's Day)
  完全成长 → 外形变为儿童 → ✅ 可帮农活!
    = 总计约56天从婴儿到儿童
```

### 2.2 各阶段可用互动

| 阶段 | 可对话 | 可送礼 | 可帮农活 | 触发事件 | 好感度变化 |
|:---:|:-----:|:-----:|:-------:|:-------:|:---------:|
| 出生 (Birth) | 否 | 否 | 否 | 出生事件 | 初始0♥ |
| 爬行 (Crawling) | 有限 | 是 | 否 | 家庭互动 | 缓慢上升 |
| 走路 (Walking) | 是 | 是 | 否 | 外出事件 | 上升 |
| 说话 (Talking) | 是（丰富） | 是 | 否 | 对话事件 | 上升加速 |
| 儿童 (Child) | 是（完全） | 是 | **是** | 多种事件 | 可至10♥ |

### 2.3 可生2个孩子

| 项目 | 第一个孩子 | 第二个孩子 |
|:----:|:---------:|:---------:|
| 触发 | 条件满足后配偶询问 | 第一个完全成长后再次触发 |
| 成长 | 经历完整5阶段 | 同样的5阶段 |
| 最终 | 儿童——可帮农活 | 儿童——可帮农活 |
| 总计 | 约56天完成 | 从第二个出生起约56天 |

### 5.2 情敌子女与玩家子女的互动

当情敌子女和玩家的子女都成长到"儿童"阶段后，他们会在城镇中相遇并产生互动事件。这些事件包括:
- 一起玩耍
- 学校/学习相关
- 争夺玩具/和解
- 探险

### 5.3 多代社区规模

```
玩家: 结婚 → 2个子女
+
8对情敌: 每人结婚 → 各1个子女 = 8个情敌子女
=
城镇中总计 10个"下一代"NPC
```

这意味着游戏中期以后，卡斯塔尼岛的儿童人口显著增加——街道上不再只有成人NPC。

## 七、子女功能总结

| 功能 | 说明 | 与其他游戏的对比优势 |
|:----:|------|:----------------:|
| 帮农活 | 儿童可帮忙浇水/采集/钓鱼/采矿(取决于性格) | **星露谷: 无此功能; 矿石镇: 婴儿止** |
| 子女性格影响玩法 | 性格决定家务专长——提供了"选择配偶→决定子女能力"的策略 | **多数游戏: 子女无性格差异** |
| 情敌子女互动 | 子女可与情敌子女社交——社区在代际更替 | **系列唯一有此系统的作品** |
| 力量果实条件 | 子女10♥ = 1颗力量果实 | 终局奖励——激励玩家育儿 |
| 新游戏+关联 | 子女是NG+解锁条件之一 | 子代→新一轮 |
