## 二、浪漫节日约会详情

### 2.1 约会条件与流程

| 步骤 | 详情 |
|:---:|------|
| 条件 | 目标候补好感度 ≥ **3♥** |
| 邀请时间 | **节日前一天** 的下午3:50PM前（花节）或4:00PM前（萤火虫节） |
| 邀请方式 | 在节日前一天找到该NPC → 选择邀请选项 |
| 碰面时间 | 节日当天 **7:00PM** |
| 碰面地点 | 各节日指定地点 |
| 活动内容 | 一起参加节日 + 对话选择 |
| 效果 | 好感度上升 + 特殊对话 + 关系推进 |

### 2.2 各节日时间

| 节日 | 邀请截止时间 | 碰面地点 | 备注 |
|------|:----------:|---------|------|
| 花节 (春14) | 前日 3:50PM | 教堂广场樱花树 | 春季最浪漫场景 |
| 萤火虫节 (夏7) | 前日 4:00PM | 笛田大水车 | 夏夜萤火虫——画面最美 |
| 月节 (秋14) | 前日下午 | 月亮山丘 | 秋月高悬 |
| 星夜节 (冬11) | 前日下午 | 月亮山丘 | 冬季星空 |

## 三、动物节 (春28) 详情

### 3.1 动物节时间线

| 时间 | 活动 |
|:---:|------|
| 9:00AM | 动物节开始——所有商店关闭 |
| 10:00AM | 竞赛报名开始 |
| 10:00AM-5:30PM | 竞赛进行（家畜/宠物/赛跑同时） |
| 5:50PM | 动物节结束 |

### 3.2 家畜竞赛

| 项目 | 数据 |
|------|:----:|
| 报名NPC | Hamilton（市长） |
| 参与条件 | 携带成年家畜（牛/羊/山羊/等） |
| 评判标准 | **好感度（♥）**——几乎完全由好感度决定 |
| 10♥保证 | 好感度10♥ → **必胜** |

## 五、和谐日 (冬5日) 详情

| 项目 | 详情 |
|------|------|
| 日期 | 冬5日 |
| 主题 | 给村民送蛋糕——传递和谐 |
| 水车状态 | **锁定/停转**（和谐日水车不工作） |
| 重要提示 | **提前磨好面粉！** 如果和谐日需要做蛋糕但没有面粉，水车停转无法磨面 |
| 送礼 | 将蛋糕送给村民可获得好感度 |
| 商店 | 所有商店关闭 |

## 六、节日商店规则

| 规则 | 说明 | 影响 |
|------|------|------|
| 全部商店关门 | 节日当天所有商店不可进入 | **必须提前1-2天采购**种子/饲料/礼物 |
| 电话订购可用 | 可通过电话订购商品 | 但**价格为正常价×2**——仅紧急使用 |
| NPC节日对话不加好感 | 日常对话不加好感 | 只有节日特定互动（如约会/送礼/参赛）才加 |
| 约会节日NPC行程变动 | 约会对象节日当天在碰面点 | 注意：不要在节日当天才找对方邀请 |
