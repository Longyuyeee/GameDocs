# 20位候补与社交系统深度 - 动物进行曲(Harvest Moon: Animal Parade)

> 游戏版本: Harvest Moon: Animal Parade (Wii, 2008年)

## 一、概述

动物进行曲拥有牧场品类中**阵容最庞大的攻略角色群**——10位男性+10位女性(含2位特殊神祇)。配合7心告白关系锁定+8对情敌配对+情敌子女——社交系统的深度至今未被超越。

## 二、20位候补的构成

| 类别 | 人数 | 角色 |
|:--:|:--:|------|
| 普通男性 | 8 | Luke/Toby/Julius/Chase/Gill/Owen/Calvin/Jin |
| 特殊男性 | 2 | **丰收之王**(神)/**魔法师** |
| 普通女性 | 8 | Luna/Renee/Phoebe/Kathy/Anissa/Candace/Maya/Selena |
| 特殊女性 | 2 | **丰收女神**(神)/**魔女** |

## 三、社交系统的独特设计

### 3.1 7心告白=关系锁定

```
社交进度:
2♥→第一份礼物
3♥→可节日约会
4♥→第二份礼物
5♥→浪漫约会
6♥→情书
7♥→告白(灯塔4PM)→关系锁定!
    ⚠️ 其他候补的好感度会下降
9♥→蓝羽毛求婚
```

> 告白="确定关系"——不是可逆转的——你需要选择一个人

### 3.2 蓝羽毛=登山仪式

- 蓝羽毛获取: 加农山顶/下午5-7PM(仅在Mira告知蓝鸟后)
- 不是"去商店买"——而是"爬上山顶在黄昏获取"——仪式感

### 3.3 特殊候补的不同规则

| 规则 | 普通候补 | 丰收之王/女神 |
|------|:--:|:--:|
| 同居 | ✓ | **不同居** |
| 帮农活 | ✓ | **不帮**(他们是神!) |
| 婚礼信物 | 婚戒 | 大地之坠/太阳之戒 |
| 子女 | ✓ | ✓(特殊事件) |

## 四、社交系统的压力梯度

| 阶段 | 压力 |
|------|:--:|
| 0-5♥ | 低(可随意送礼/探索) |
| 5♥(约会) | 中(需要管理约会时间) |
| **7♥(告白)** | **高**(关系锁定→不可逆转→其他候补好感下降) |
| 9♥(求婚) | 中(满足条件=可求婚) |
| 婚后 | 低(维护+触发子女事件) |

## 七、设计启示与跨游戏对比

动物进行曲的情敌系统与20位候补设计,核心意图是"让社区真正活起来"。NPC不仅是可攻略对象,他们彼此之间也有情感、结婚、生子——玩家离开游戏后,小镇依然在运转。这与星露谷物语形成了鲜明对比:星露谷允许玩家离婚再娶,且离婚后前配偶仅降好感而无其他后果,情敌系统被完全移除;珊瑚岛和符文工房5同样取消了情敌事件。

情敌系统的消失并非技术原因,而是设计哲学的转变——现代牧场游戏倾向于"玩家中心主义"即所有NPC围绕玩家转,而动物进行曲代表着"社区中心主义"即NPC有自己的生活轨迹。在动森式社交大行其道的今天,动物进行曲可能是牧场品类中情敌系统的"最后传承者"。

| 维度 | 动物进行曲(2008) | 星露谷物语(2016) | 珊瑚岛(2023) |
|------|:---:|:---:|:---:|
| 情敌系统 | **8对情敌**→24情敌事件 | 无(已移除) | 无 |
| 离婚后果 | 不可离婚(锁定) | 降好感+清除记忆 | 降好感 |
| 情敌子女 | **8个情敌子女**→代际传承 | 无 | 无 |
| 社区中心主义 | **NPC有自己的生活** | 玩家是社区中心 | 玩家是社区中心 |
| 告白锁定 | 7心硬锁定(不可逆转) | 10心后可随时恋爱 | 8心后可恋爱 |
