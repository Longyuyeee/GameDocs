# 经济系统 - 动物进行曲 (Harvest Moon: Animal Parade)

> 版本: Harvest Moon: Animal Parade (Wii, 2009 NA) | 开发商: Marvelous
> 来源: [IGN Walkthrough](https://www.ign.com/articles/2010/01/20/harvest-moon-animal-parade-walkthroughguide-1045265) | [Fogu Guide](https://fogu.com/hm10/) | [RanchStory Wiki](https://wiki.ranchstory.farm/) | [Harvest Moon Wiki](https://harvestmoon.fandom.com)

## 一、系统概述

动物进行曲采用**单货币体系（金币G）**，以出货箱 (Shipping Bin) 为核心结算方式。经济系统的核心驱动力不在于"赚最多钱"，而在于**通过经营证明自己配得上"英雄"称号**——农场等级制度将经济积累直接关联到主线通关。多农田 + 多动物 + 加工链 + 采矿构成了四条并行的经济管线。

## 二、货币体系

| 项目 | 详情 |
|------|------|
| 货币单位 | 金币 (G) |
| 结算方式 | 出货箱——每日下午5点自动结算，收入存入账户 |
| 价格体系 | 固定价格——无市场波动——无供需影响 |
| 现金上限 | 9,999,999G（系列常规上限） |
| 账户 | 资金自动存入，无利息 |

### 3.2 各阶段最优经济策略

**初期（春1-夏末，启动资金 < 10,000G）**:
1. **卷心菜开局**: Marimba Farm赠送9颗卷心菜种子→春植14天→收获9颗×350G=3,150G → 第一桶金
2. **芜菁快速周转**: 种子120G/4天成熟/售价90G → 单块地4天周期可循环3-4次
3. **动物起步**: 免费奶牛（Hanna赠送）→ 每天牛奶120G → 稳定日收入

**中期（第一年秋-第二年春，资金 10,000-100,000G）**:
1. **草莓/南瓜高价值作物**: 春草莓复收利润极高；夏南瓜单次500G
2. **加工品增值**: 蛋黄酱机/奶酪机采购后，动物产品利润翻倍
3. **采矿**: 矿石出售（银/金/秘银）补充收入流
4. **畜牧扩展**: 购入羊群（羊毛→毛线→布料三级加工）

**后期（英雄称号后，不限资金）**:
1. **闪耀作物**: 黄钟解锁后，闪耀作物出货价数倍于普通作物
2. **鸵鸟蛋加工**: 最高价值加工品
3. **宝石出货**: 钻石/红宝石单价极高

## 五、农场等级制度 (Farm Degree)

农场等级是动物进行曲经济系统的**最终目标**——不是"赚最多钱"而是"通过经营获得英雄称号"。

### 5.1 等级评定

每月末由**市长Hamilton**根据玩家出货箱的总金额和各分类出货额进行评定。

| 称号等级 | 解锁条件 | 奖励/解锁内容 |
|---------|---------|:------------:|
| Beginner | 初始 | 基础功能 |
| 待补充 | 累积出货 | 新种子/新工具升级选项 |
| 待补充 | 继续累积 | -- |
| **Hero (英雄)** | **任一单项出货达标: 作物50,000G / 动物50,000G / 钓鱼20,000G / 采矿30,000G** | **主线通关必需 + 登加农山顶资格** |

### 5.2 "英雄"称号获取策略

最推荐的路径: **作物路线**（草莓复收 / 南瓜高价 / 闪耀作物）或**动物路线**（加工品增值）。

| 路径 | 难度 | 推荐理由 | 预计时间 |
|------|:---:|---------|:-------:|
| 作物出货50,000G | ★★★ | 春季草莓6-7次复收即可达标 | 第一年春末 |
| 动物出货50,000G | ★★★ | 多个动物产品加工后出货 | 第一年夏秋 |
| 钓鱼20,000G | ★★★★ | 钓鱼单价低，需要大量时间投入 | 第一年末 |
| 采矿30,000G | ★★★★★ | 宝石价值高但出现概率低 | 随缘 |

## 六、经济系统与主线的关系

```
农业(作物出货)
   ↓ 黄钟需要10优质作物
   ↓ 英雄称号需要50,000G出货
   ↓
经济 → 主线
   ↓
赚钱 → 买动物/种子/工具升级
   ↓ 扩建房屋 → 婚姻/子女
   ↓
繁荣度 → 解锁区域 → 更多农田
```

**核心设计**: 经济不是目的而是手段。赚钱→扩建房屋（结婚/子女）→升级工具（效率提升）→解锁新区域（主线推进）→获得英雄称号（通关）。经济活动与主线进度紧密绑定。

## 七、设计特点分析

### 7.1 正向设计

1. **"英雄"称号 = 经营验证**: 不是通过战斗或解谜获得英雄称号——而是通过经营证明自己。这是牧场物语系列的核心命题: "经营就是英雄主义"。
2. **四条并行经济管线**: 作物/动物/钓鱼/采矿——玩家可任选精通的路线。这提供了高度的玩法自由——想纯种植、纯畜牧或混合经营都可以。
3. **加工增值 = 经济深度**: 从"直接出货原料"到"加工后出货"的转变不仅仅是多赚了钱——它引入了"先投资再回报"的经济决策: 买加工机需要先花钱→之后赚更多。
4. **电话购物双倍价**: 应急可用但不划算——鼓励提前规划采购——节日商店关门时的备选方案。

### 7.2 跨游戏对比

| 维度 | 动物进行曲 (2009) | 矿石镇 (2003) | 星露谷物语 (2016) |
|------|:----------------:|:-------------:|:----------------:|
| 经济与主线关系 | **直接绑定**（英雄称号必需） | 弱关联 | 弱关联 |
| 收入管线数 | 4条主流 | 3条 | 5条+ |
| 加工增值 | 三级(羊毛→布料) | 二级 | 二级 |
| 农场等级 | 有（主线必需） | 有（无强制） | 无 |
| 固定价格 | 是 | 是 | 是 |
| 自动灌溉 | 无 | 无 | 有（洒水器） |

### 7.3 局限性

1. **无市场波动**: 价格完全固定——没有"屯货等涨价"的策略空间——降低了经济系统的决策深度。
2. **无投资系统**: 没有股票/房地产等被动收入手段——全部收入来自玩家手动操作。
3. **后期经济溢出**: 通关后资金大量积累但缺乏消费出口——这是系列通病。
