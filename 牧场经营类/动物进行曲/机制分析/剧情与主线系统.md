# 剧情与主线系统 - 动物进行曲(Harvest Moon: Animal Parade)

> 游戏版本: Harvest Moon: Animal Parade (Wii, 2008年)
> 数据来源: harvestmoon.fandom.com, fogu.com, IGN Walkthrough

## 1. 世界观设定

- **卡斯塔尼岛(Castanet Island)**——由丰收女神守护的岛屿
- **神圣女神之树(Divine Goddess Tree)**——岛的自然力量之源→正在枯萎
- **五钟+丰收之王**——唤醒丰收之王=恢复女神树=拯救岛屿

## 2. 主线剧情结构

```
序幕: 到达卡斯塔尼岛→遇到精灵Finn
  → 访问女神之泉→女神树枯萎→必须拯救
  → 寻找并敲响五钟(主线)
Act 1: 红钟(火/Alan) — 加农矿洞下层10F
Act 2: 黄钟(大地/Collin) — 海滩南侧
Act 3: 蓝钟(水/Ben) — 水洞
Act 4: 绿钟(风/Daren) — 赋格森林
Act 5: 紫钟(心/Edge) — 教堂(凌晨2AM)
高潮: 五钟齐鸣→获得"英雄"称号
  → 登加农矿山上层45层→山顶
  → 五精灵齐鸣→丰收之王降临
  → 女神树重生→岛屿自然恢复
结局: 丰收之王创造山顶捷径→无限后日谈
  → 可攻略丰收之王(9♥+蓝羽毛)
```

## 3. 五钟主线详解

### 红钟(火)——第1钟
- 位置: 加农矿洞下层10层
- BOSS: Owen碎石→发现钟座空→Ramsey修复钟
- 材料: 铁矿+铜矿+银矿+金矿各1→交Ramsey→获得修复的钟

### 黄钟(大地)——第2钟
- 位置: 海滩南侧(农场水域方向)
- 精灵: Collin(太虚弱)
- 门控: Ruth不开门→交付**10个优质作物**(优质莴苣/番茄/南瓜/荞麦之一)
- 解决: 取钟→女神说需月光沐浴→月石上放置

### 蓝钟(水)——第3钟
- 位置: 水洞(Watery Cave)
- 精灵: Ben(忘记了钟的旋律)
- 解决: Paolo给出旋律→制作"旋律重现器"
- 材料: **黑珍珠+鸭蛋+5条任意鱼**

### 绿钟(风)——第4钟
- 位置: 赋格森林
- 前置: 与Hanna/Cain/Dale对话→解锁森林
- 问题: **魔女被变成了青蛙!**
- 解决: 魔法师要求→**完美黄油+优质玉米粉+木槿**→还原魔女→魔女赠予绿钟
- 木槿来源: 巨嘴鸟岛(蓝钟后解锁)

### 紫钟(心)——第5钟
- 位置: 教堂(哈莫尼卡镇)
- 触发: 四钟完成后→村民提"教堂鬼魂"
- 进入: **凌晨2:00AM**站在教堂门口→剧情触发→门锁打开
- 精灵: Edge(哭泣——因为没人许愿了)
- 解决: 与Finn访问许愿神社→女神指导

## 4. 通关条件

| 条件 | 说明 |
|------|------|
| 五钟全部敲响 | 主线 |
| **"英雄(Hero)"称号** | 农场等级→单一类别(作物/动物/钓鱼/采矿)达到足够金币→月末评定 |
| 登加农矿山上层45层 | 下午5PM前到达 |
| 五精灵齐鸣 | 召唤丰收之王 |

## 5. 设计特点

1. **五钟=五元素=五个独立任务线**: 不是"收集五个同样东西"——每钟有独特的故事/谜题/NPC互动
2. **任务链相互依赖**: 绿钟需要木槿→木槿来自巨嘴鸟岛→巨嘴鸟岛在蓝钟后解锁→任务顺序有逻辑依赖
3. **凌晨2AM教堂事件**: 打破常规作息——需要刻意熬夜等待——意外的悬疑氛围
4. **"英雄"=经营成就门控**: 不是"打完BOSS就是英雄"——需要通过经营获得——经营=英雄主义的独特定义
5. **丰收之王=主线Boss+攻略对象**: 双重身份——既是你要唤醒的神→也是你可以恋爱结婚的对象
