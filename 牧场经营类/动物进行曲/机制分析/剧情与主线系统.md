# 剧情与主线系统 - 动物进行曲 (Harvest Moon: Animal Parade)

> 版本: Harvest Moon: Animal Parade (Wii, 2009 NA) | 开发商: Marvelous
> 来源: [IGN Walkthrough](https://www.ign.com/articles/2010/01/20/harvest-moon-animal-parade-walkthroughguide-1045265) | [Fogu Guide](https://fogu.com/hm10/) | [RanchStory Wiki](https://wiki.ranchstory.farm/) | [Harvest Moon Wiki](https://harvestmoon.fandom.com)

## 一、系统概述

动物进行曲拥有Harvest Moon系列中最具结构感的**主线任务设计**——"五钟使命"（Five Bell Quest）。与系列前作"做完XX天就通关"或"纯沙盒无结局"不同，动物进行曲提供了清晰的主线目标：敲响五钟 → 获得英雄称号 → 召唤丰收之王 → 拯救女神之树。这条主线不仅是剧情驱动——它也**直接控制着地图解锁、功能开放和终局内容**。

## 二、世界观设定

### 2.1 卡斯塔尼岛 (Castanet Island)

| 项目 | 详情 |
|------|------|
| 岛屿名称 | Castanet Island（卡斯塔尼岛）——名字来源于乐器"响板"（castanet） |
| 守护者 | **丰收女神 (Harvest Goddess)**——维系岛屿自然力量 |
| 核心地标 | **女神之树 (Divine Goddess Tree)**——岛屿生命力的源泉 |
| 危机 | 女神之树正在枯萎 → 岛屿的自然力量在衰退 → 五钟失去力量 |
| 解决方案 | 找到并敲响五钟 → 召唤丰收之王 (Harvest King) → 恢复女神之树 |

### 2.2 故事序幕

玩家扮演一位来到卡斯塔尼岛的年轻人/女性，继承了一座荒废的农场。在探索岛屿时遇到了**精灵 Finn**（丰收精灵 / Harvest Sprite），Finn告知玩家女神之树正在枯萎，只有敲响五钟、召唤丰收之王才能拯救岛屿。

> "女神之树正在失去力量……只有五钟齐鸣才能唤醒丰收之王。" —— Finn

从此开始了"拯救岛屿 + 经营农场 + 融入社区"的三线叙事。

## 三、主线剧情结构

### 3.1 五幕结构

```
序幕（春1-2日）: 到达卡斯塔尼岛 → 初见Finn → 女神之泉 → 发现女神树枯萎
  ↓
第1幕: 红钟（火 / 精灵Alan）
  加农矿洞下层10F → 收集4种矿石 → Ramsey修复 → 敲响红钟
  ↓
第2幕: 黄钟（大地 / 精灵Collin）
  海滩南侧 → Ruth守门 → 10个优质作物 → 月光沐浴 → 敲响黄钟
  ↓
第3幕: 蓝钟（水 / 精灵Ben）
  Paolo的钥匙 → 水洞 → Ben失忆 → 旋律重现器 → 敲响蓝钟
  ↓
第4幕: 绿钟（风 / 精灵Daren）
  赋格森林 → 魔女被变青蛙 → 完美黄油+优质玉米粉+木槿 → 还原魔女 → 获得绿钟
  ↓
第5幕: 紫钟（心 / 精灵Edge）
  四钟完成后 → "教堂鬼魂"传言 → 凌晨2AM进入教堂 → 许愿神社 → 敲响紫钟
  ↓
高潮: 五钟齐鸣 → 获得"英雄"称号 → 登加农上层45F → 下午5PM前到达山顶
  → 五精灵齐鸣 → 丰收之王降临 → 女神树重生
  ↓
结局: 丰收之王创造山顶捷径 → 岛屿恢复生机
  → 可继续攻略丰收之王（9♥+蓝羽毛）
```

### 3.2 五钟任务链详解

#### 红钟（火）—— 第1钟

| 属性 | 详情 |
|------|------|
| 精灵 | Alan（火之精灵） |
| 位置 | 加农矿洞下层10层 |
| 触发 | Owen帮玩家打碎矿洞入口碎石 |
| 阶段1 | 探索矿洞下层 → 到达10F → 发现钟座是空的 |
| 阶段2 | 找Ramsey → 得知钟需要修复 → 需要**铁+铜+银+金矿石各1个** |
| 阶段3 | 收集矿石 → 交Ramsey → 获得修复的钟 |
| 阶段4 | 回到10F → 放入钟座 → 敲响 |
| 解锁 | 加农矿洞下层自由通行、解锁黄钟任务线 |
| 提示 | 四种矿石在矿洞1-10F均可获得——铁最常见、金相对稀有所以优先收集 |

#### 黄钟（大地）—— 第2钟

| 属性 | 详情 |
|------|------|

| 位置 | 哈莫尼卡海滩（南侧） |

| 阶段1 | 遇到Collin——她太虚弱了，黄钟被Ruth家锁住 |
| 阶段2 | Ruth（Marimba农场的女主人）要求**10个优质作物**（Good品质蔬菜）才开门 |
| 阶段3 | 种植并使用肥料获得优质作物 → 交10个给Ruth → 进入钟所在洞穴 |

#### 蓝钟（水）—— 第3钟

| 属性 | 详情 |
|------|------|
| 精灵 | Ben（水之精灵） |
| 位置 | 水洞 (Watery Cave) |
| 触发 | 黄钟敲响后找Paolo（灯塔看守人） |
| 阶段1 | Paolo给钥匙 → 进入水洞 |
| 阶段2 | 找到Ben → 但他**忘记了蓝钟的旋律** |

| 阶段4 | 材料: **黑珍珠×1 + 鸭蛋×1 + 任意鱼类×5** |
| 阶段5 | 收集材料 → 制作旋律重现器 → 演奏旋律 → Ben记起蓝钟 → 敲响 |
| 解锁 | **巨嘴鸟岛 (Toucan Island)** —— Pascal开船送达 |
| **关键提示** | 鸭蛋需要养鸭；黑珍珠在水洞或海滩采集；鱼类=钓鱼 |

#### 绿钟（风）—— 第4钟

| 属性 | 详情 |

## 四、通关条件

### 4.1 条件清单

| 条件 | 要求 | 说明 |
|------|------|------|
| **五钟全部敲响** | 完成红/黄/蓝/绿/紫五钟任务 | 主线核心 |
| **获得"英雄"称号** | 任一经营类别出货达标 | 见下方农场等级表 |
| **登加农上层45层** | 从加农矿山深入上层矿洞 | 下午5PM前到达山顶 |
| **下午5PM前抵达山顶** | 必须在时间截止前 | 否则入口关闭 |
| **五精灵齐鸣** | 在丰收之王王座前触发 | 召唤丰收之王 |
| **女神之树重生** | 丰收之王降临后恢复 | **游戏通关** |

### 4.2 英雄称号条件（农场等级）

| 类别 | 估计出货额要求 | 推荐达成方式 |
|------|:------------:|-------------|
| 作物 (Crops) | 约50,000G | 草莓复收/南瓜/闪耀作物 |
| 动物 (Animals) | 约50,000G | 加工品（奶酪/蛋黄酱/布料） |
| 钓鱼 (Fishing) | 约20,000G | 高级鱼/传说鱼 |

## 六、设计特点分析

### 6.1 正向设计

1. **五钟 = 五元素 = 五个独立且不同的任务**: 红钟=收集+制作（矿洞探索）、黄钟=农业经营检查（优质作物）、蓝钟=探索+解谜（失忆旋律）、绿钟=跨岛任务+魔法主题、紫钟=氛围+时间触发（凌晨2AM）。每个钟的任务设计风格完全不同——避免了"五个同样的跑腿任务"的重复感。
2. **任务依赖链有逻辑性**: 绿钟需要木槿→木槿来自巨嘴鸟岛→巨嘴鸟岛在蓝钟后解锁。这不是"为了拖延而设的门"——而是有叙事逻辑的区域逐步开放。
3. **凌晨2AM教堂事件**: 这是全游戏唯一需要在凌晨执行的事件——刻意打破玩家建立的"早睡早起"作息——制造了一种特殊的神秘氛围。
4. **"英雄" = 经营验证**: 在牧场物语中获得"英雄"称号不是通过战斗——而是通过种植/畜牧/钓鱼/采矿来证明自己。这是极其核心的"经营即英雄主义"设计理念。
5. **丰收之王 = 双重身份**: 既是主线最终BOSS（被召唤的存在）-> 又是可恋爱结婚的攻略对象。这种身份反转——从"神"到"配偶"——是系列最具特色的设计之一。

|---------|:----------:|------------|
| 红钟 | 春1-14日 | 基础种植、畜牧起步 |
