# 经济系统
![三村全景](https://www.bokumono.com/series/tots/img/ss/ss01.jpg)

游戏版本: 牧场物语 三个村庄的珍贵朋友(Story of Seasons: Trio of Towns) (Nintendo 3DS, 2016年)

---

## 1. 货币体系

游戏采用单一货币 **G(金币)**，无任何二级货币或代币系统。所有交易、升级、建造均以 G 结算。

| 项目 | 说明 |
|------|------|
| 货币单位 | G(金币/Gold) |
| 携带上限 | 9,999,999G |
| 仓库上限 | 无独立限制(直接存入钱包) |
| 初始资金 | 500G(游戏开始) |

---

## 2. 出货系统(Shipping Bin)

出货箱位于牧场入口处，是游戏最主要的收入结算方式。

### 出货规则

| 规则 | 说明 |
|------|------|
| 结算时间 | 每日 **下午5:00(17:00)** 自动结算 |
| 出货地点 | 农场出货箱(蓝色箱子) |
| 可出货品 | 作物、畜牧产品、加工品、采集品、鱼类 |
| 不可出货 | 工具、家具、装饰品、种子(未种过的)、建材 |
| 出货记录 | 在菜单中可查看已出货物品清单 |
| 特殊机制 | 第一次出货某物品会记录在图鉴中 |

### 出货收益影响因素

- **基础价格**: 每种物品有固定基础价
- **星级加成**: 星级越高,价格越高
  - 0.5★ = 基础价 × 1.0
  - 1.0★ = 基础价 × 1.1
  - 1.5★ = 基础价 × 1.2
  - 2.0★ = 基础价 × 1.3
  - ...(每0.5★增加10%)
  - 5.0★ = 基础价 × 1.9(最大)

## 3. 收入三阶段

### 第一阶段: 初期(春~夏1年目)

**主要收入来源**: 作物出货 + 兼职工作(part-time jobs)

**初期推荐作物**(春):

| 作物 | 种子价格 | 出货价格(0.5★) | 生长天数 | 再生 |
|------|----------|-----------------|----------|------|
| 芜菁(Turnip) | 120G | 130G | 4天 | 否 |
| 马铃薯(Potato) | 150G | 180G | 7天 | 否 |
| 草莓(Strawberry) | 300G | 280G | 9天 | 每5天再生 |

**兼职收入**:

| 村庄 | 兼职类型 | 收入范围 | 解锁条件 |
|------|----------|----------|----------|
| 西部镇(Westown) | 牧场清洁 | 200~500G/次 | 春1日解锁 |
| 西部镇(Westown) | 货物搬运 | 300~600G/次 | 春1日解锁 |
| 露草里(Tsuyukusa) | 田地帮忙 | 250~550G/次 | 夏5日解锁 |
| 露露可可村(Lulukoko) | 海滩清理 | 200~500G/次 | 春15日解锁 |

兼职由中介人 **Edith(伊迪丝,兼职中介)** 在西部镇介绍，每天可做1次，每次约消耗半天时间(6:00~12:00)。

### 第二阶段: 中期(夏1年目~冬1年目)

**核心策略**: 料理加工(烹饪)赚取差价

**推荐料理利润链**:

| 料理 | 材料成本 | 售价(0.5★) | 利润 | 所需设备 |
|------|----------|-------------|------|----------|
| 玉米面包(Corn Bread) | 玉米×1(≈100G) | 380G | ≈280G | 厨房(烤箱) |
| 生鱼片(Sashimi) | 鱼×1(≈80~150G) | 350~500G | ≈270~350G | 厨房(菜刀) |
| 烤起司(Grilled Cheese) | 面包×1 + 起司×1 | 680G | ≈400G | 厨房(烤箱) |
| 奶油烤马铃薯 | 马铃薯×1 + 黄油×1 | 550G | ≈350G | 厨房(烤箱) |

**关键**: 第一个房屋扩建获得厨房后，料理利润远超直接出售原材料。

### 第三阶段: 后期(2年目以后)

**核心策略**: 高品质加工品 + 畜牧产品

## 4. 完整价格链: 种子→作物→加工品

以草莓(Strawberry)为例:

```
种子(300G)
  → 种植9天
  → 草莓果实(280G 0.5★ / 532G 5★)
    → 直接出货: 280G(0.5★)
    → 果酱制造机
      → 草莓果酱(≈420G 0.5★)
    → 葡萄酒制造机
      → 草莓酒(≈500G 0.5★)
    → 厨房(点心制作)
      → 草莓蛋糕(≈680G)
```

以小麦(Wheat)为例:

```
种子(150G)
  → 种植7天
  → 小麦(200G 0.5★)
    → 磨粉机
      → 小麦粉(面粉, 230G)
        → 厨房(面包制作)
          → 面包(350G)
            → 起司制造机结合的料理
```

## 7. 村庄等级经济

村庄等级升级与经济的双向关系:

| 升级方式 | 每次操作增加的村庄点数 |
|----------|----------------------|
| 从村庄商店购买物品 | +1~3点(依金额) |
| 在村庄出货 | +1~3点(依金额) |
| 与村民对话 | +1点 |
| 赠送礼物 | +2~5点 |
| 参加节日 | +10~30点 |
| 做兼职 | +5点 |

**村庄等级解锁内容**:

| 等级 | 解锁内容 |
|------|----------|
| E→D | 基础种子解锁 |
| D→C | 中级种子 + 新动物 |
| C→B | 高级种子 + 新加工机 |
| B→A | 稀有种子 + 特殊装饰品 |
| A→S | 最高级物品 + 特殊事件 |
