## 2. 升级系统概述

所有工具可在 **露露可可村(Lulukoko)** 的工坊由 **Ludus(卢德斯,木工/铁匠)** 升级。

### 三维升级体系

每把工具可从 **3个维度** 分别升级:

| 维度 | 等级(低→中→高) | 效果 |
|------|-----------------|------|
| 重量(Weight) | Light(轻)→Regular(普通)→Heavy(重) | **每级加工次数+1** |
| 容量(Capacity) | Low(低)→Medium(中)→High(高) | **效果范围扩大** |
| 效能(Effectiveness) | Low(低)→Medium(中)→High(高) | **效率提升** |

### 升级费用一览

| 升级项目 | 重量(轻→普) | 重量(普→重) | 容量(低→中) | 容量(中→高) | 效能(低→中) | 效能(中→高) |
|----------|-------------|-------------|-------------|-------------|-------------|-------------|
| 锄头 | 1,000G | 3,000G | 1,000G | 3,000G | 1,000G | 3,000G |
| 水壶 | 1,000G | 3,000G | 1,000G | 3,000G | 1,500G | 4,000G |
| 斧头 | 1,000G | 3,000G | 1,000G | 3,000G | 1,000G | 3,000G |
| 锤子 | 1,000G | 3,000G | 1,000G | 3,000G | 1,500G | 4,000G |
| 镰刀 | 1,000G | 3,000G | 1,000G | 3,000G | 1,000G | 3,000G |
| 钓竿 | 1,000G | 3,000G | - | - | 1,000G | 3,000G |

> **每把工具完全升级(满3维×2级)**: 约10,000~12,000G

## 3. 各工具升级效果详解

### 锄头(Hoe)

| 重量等级 | 效果 |
|----------|------|
| Light | 每次耕地1格 |
| Regular | 每次耕地3格(直线) |
| Heavy | 每次耕地5格(直线) |

| 容量等级 | 效果 |
|----------|------|
| Low | 可耕地范围: 1×1 |
| Medium | 可耕地范围: 1×3 |
| High | 可耕地范围: 1×5 |

### 水壶(Watering Can)

| 重量等级 | 效果 |
|----------|------|
| Light | 每次浇1格 |
| Regular | 每次浇3格(直线) |
| Heavy | 每次浇5格(直线) |

| 容量等级 | 效果 |
|----------|------|
| Low | 蓄水量: 20 |
| Medium | 蓄水量: 50 |
| High | 蓄水量: 100 |

| 效能等级 | 效果 |
|----------|------|
| Low | 标准浇水速度 |
| Medium | 浇水效率提升 |
| High | 大面积快速浇水 |

### 斧头(Axe)

| 重量等级 | 效果 |
|----------|------|
| Light | 每次砍1棵小树 |
| Regular | 可砍中型树木 |
| Heavy | 可砍大型树木 |

| 效能等级 | 效果 |
|----------|------|
| Low | 砍树3次倒下 |
| Medium | 砍树2次倒下 |
| High | 砍树1次倒下 |

**可砍伐的木材类型**:

| 木材类型 | 所需斧头等级 | 用途 |
|----------|-------------|------|
| 小木材(Small Lumber) | 任意等级 | 基础建造 |
| 木材(Lumber) | Regular+ | 房屋扩建、机器 |
| 坚硬木材(Hard Lumber) | Heavy | 高级建造 |

### 锤子(Hammer)

锤子是 **最优先升级** 的工具。其重量等级直接影响矿石采集量。

| 锤子等级 | 同时采集矿石点数 | 备注 |
|----------|-----------------|------|
| Lv1(Light) | 2个矿石点 | 初始状态 |
| Lv2(Light→Regular) | 3个矿石点 | **优先升级** |
| Lv3(Regular→Heavy) | **4个矿石点(最大)** | 关键节点 |

> **锤子升级策略**: 锤子Lv1只能敲2个矿点,每升1级增加1个矿点,最大4个。矿点是矿石、宝石、化石的重要来源,因此 **锤子→水壶→斧头(Lv3)→其他** 是最推荐的升级顺序。

## 4. 推荐升级顺序

| 优先级 | 工具 | 目标等级 | 理由 |
|--------|------|----------|------|
| ★★★ | 锤子(Hammer) | 重量→Heavy(Lv3) | 矿点4个最大化,矿石是一切升级和建造的基础 |
| ★★★ | 水壶(Watering Can) | 重量→Heavy + 容量→High | 大面积浇水节省大量每日时间 |
| ★★☆ | 斧头(Axe) | 重量→Regular(至少Lv2) | 砍中型树获取木材,建议到Lv3 |
| ★★☆ | 锄头(Hoe) | 重量→Regular | 一次耕多格地 |
| ★☆☆ | 镰刀(Sickle) | 重量→Regular | 割草效率提升 |
| ★☆☆ | 钓竿(Fishing Rod) | 效能→Medium | 稀有鱼料理利润高 |

### 推荐升级路径

```
第1步: 锤子 Light→Regular (1,000G)  [解锁3矿点]
第2步: 水壶 Light→Regular (1,000G)  [解锁3格浇水]
第3步: 锤子 Regular→Heavy (3,000G)  [解锁4矿点,最大]
第4步: 水壶 Regular→Heavy (3,000G)  [解锁5格浇水]
第5步: 斧头 Light→Regular (1,000G)  [解锁中型树木]
第6步: 斧头 Regular→Heavy (3,000G)  [解锁大型树木]
第7步起: 其他工具按需升级
```

**总花费(核心6步)**: 1,000+1,000+3,000+3,000+1,000+3,000 = **12,000G**

## 6. 工具与"武器"说明

**本游戏不存在战斗系统**,因此:

| 常见误解 | 实际情况 |
|----------|----------|
| 有没有剑/武器？ | 无。没有任何战斗工具 |
| 矿坑有没有怪物？ | 无。采矿仅为锤子敲矿石点 |
| 需不需要防具？ | 不需要 |
| 野外有没有敌人？ | 无。只有可互动的野生动物 |

所有工具均为 **生产/采集/照料** 用途:

| 工具 | 纯生产用途 |
|------|-----------|
| 锄头 | 耕地(农业生产) |
| 水壶 | 浇水(作物栽培) |
| 斧头 | 伐木获取木材(建造) |
| 锤子 | 敲碎岩石/挖矿(获取矿物) |
| 镰刀 | 割草/收获(采集) |
| 钓竿 | 钓鱼(料理原料) |
| 挤奶器 | 挤奶(畜牧生产) |
| 刷子 | 刷毛(动物照料) |
| 毛剪 | 剪毛(畜牧生产) |
