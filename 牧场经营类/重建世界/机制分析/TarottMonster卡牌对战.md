# Tarott Monster卡牌对战 - 重建世界(Littlewood)

> 游戏版本: Littlewood (PC/Switch, 2020年)
> 数据来源: littlewood.fandom.com

## 1. 概述

Littlewood内置了一个**完整的卡牌收集对战游戏**——Tarott Monster。这是牧场/建造品类中极少集成的完整CCG系统。前暗黑巫师的怪物被封印在卡牌中，你可以在镇民家中与他们对战。

## 2. 卡牌获取

| 方式 | 说明 |
|------|------|
| 击败暗黑巫师怪物 | →卡牌封印 |
| 与NPC对战 | →赢得新卡 |
| 探索/任务奖励 | →稀有卡 |
| 商店购买 | →基础卡包 |

## 3. NPC牌组主题

| NPC | 牌组主题 | 风格 |
|------|:--:|------|
| 各镇民 | 火/土/水/光/暗 | 元素主题 |
| Dudley | 特殊 | — |

## 4. 对战机制

| 要素 | 说明 |
|------|------|
| 场地 | Tarott Monster桌(放在NPC家中) |
| 规则 | 类似简化版TCG——攻/防/特殊效果 |
| 年度比赛 | Dirboros举办——年度Tarott Monster大赛 |

## 5. 设计特点

1. **游戏内游戏**: 完整CCG→独立于建造/经营的独立玩法维度
2. **NPC=牌组=性格**: 每个NPC的牌组风格反映其性格——类似宝可梦道馆
3. **年度比赛=终局目标**: 收集最强牌组→年度赛夺冠——额外长期动力
