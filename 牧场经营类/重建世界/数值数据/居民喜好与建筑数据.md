# 居民喜好与建筑数据 - 重建世界(Littlewood)

> 游戏版本: Littlewood (PC/Switch, 2020年)
> 数据来源: littlewood.fandom.com

## 一、居民位置偏好

| 居民 | 偏好 | 要求 |
|------|:--:|------|
| 各镇民 | 各异 | 高处/水边/树旁/远离/靠近特定建筑 |
| 全员 | 家具喜好 | 特定家具在屋内→+心EXP |

## 二、建筑一览

| 建筑 | 功能 | 解锁条件 |
|------|:--:|------|
| 酒馆(Tavern) | 烹饪(3站: Bubble Pot/Sizzle Pan/Chop Board) | Bubsy |
| 伐木场(Lumber Mill) | 木材加工 | 镇民 |
| 钓鱼小屋(Fishing Hut) | 钓鱼相关 | 镇民 |
| 魔法图书馆(Magical Library) | Tarott/虫 | 剧情 |
| 广场(Town Square) | 季节活动 | 剧情 |
| 杂货店(General Shop) | 日用品(Tue/Thu补货) | Dalton |
| 工坊(Crafting Workshop) | 制作 | 剧情 |
| 购物区(Shopping District) | 出售物品→Dewdrops | 剧情 |

## 三、居民请求机制

| 类型 | 说明 |
|------|------|
| 位置请求 | "我想住在高处/水边/树旁/远离XX/靠近XX" |
| 家具请求 | "我的房间需要XX家具" |
| 完成奖励 | **+心EXP→解锁约会→推进关系** |

## 四、捐赠系统

| 建筑 | 可捐赠 |
|------|------|
| 各建筑 | 物品/材料→稀有奖励+新制作 |
| 博物馆 | 虫/鱼/物品→收集展示 |

## 五、Dewdrops经济

| 获取 | 消耗 |
|------|------|
| 购物区出售物品 | 建筑/种子/材料购买 |
| 约会奖励(200/次) | 气球升级 |
| 完成任务 | 装饰/家具 |
