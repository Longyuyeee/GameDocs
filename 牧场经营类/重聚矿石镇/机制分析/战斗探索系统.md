# 战斗/探索系统 - 牧场物语 重聚矿石镇

> 矿石镇是无战斗系统的纯经营游戏。本章标注「无此系统」,但全面记录矿洞探索、钓鱼和采集的完整数据。

---

## 一、战斗系统

**无战斗系统(无此系统)**

矿石镇是纯粹的牧场经营游戏,无怪物、无战斗、无HP战斗损耗、无武器系统。所有"冒险"元素围绕矿洞探索、钓鱼和采集展开。

### 与符文工房的设计分支

| 系统 | 重聚矿石镇 | 符文工房系列 |
|:----|:--------:|:----------:|
| 战斗 | 无 | ARPG实时战斗 |
| 怪物 | 无 | 多种怪物+Boss战 |
| 武器 | 无 | 剑/杖/锤/双剑 |
| 矿洞 | 纯体力管理 | 怪物+体力双重管理 |
| 魔法 | 无 | 多种元素魔法 |
| BOSS | 无 | 迷宫BOSS |
| 定位 | 纯经营体验 | 经营+ARPG融合体验 |

> 符文工房(2006年起)本质上是"矿石镇+ARPG战斗"的融合产品——两个系列同属Marvelous,共享世界观元素(如女神、四季、节日等),但战斗系统的有无决定了完全不同的用户群。

### 2.2 矿洞探索核心机制: 体力管理

矿洞探索的本质不是打怪,而是**管理体力储备**:

| 动作 | 消耗HP | 建议 |
|:----|:-----:|:----|
| 锤子砸石头(普通) | 2HP/次 | 建议用S/L确保每次出矿石 |
| 蓄力砸大石头 | 5HP/次 | 仅当石头挡路时 |
| 锄头挖地 | 2HP/次 | 用于寻找隐藏道具 |
| 下楼梯 | 0HP | 无消耗 |
| 踩陷阱 | 20~70HP | 体力消耗取决于掉落层数 |

**体力补给策略**(按推荐度排序):

| 方案 | 成本 | 携带量 | 效果 | 适合场景 |
|:----|:---:|:-----:|:----|:--------|
| **乌冬面** | 极低 | 11~12组 | 每个+20HP | 255层冲刺(性价比最佳) |
| 饭团 | 低 | 多组 | 每个+10HP | 前期浅层探索 |
| **体力强化剂** | 中 | 5组+ | 每个+50HP | 深层矿洞(泉矿100F+) |
| 消除疲劳药 | 中 | 1~2组 | 疲劳大幅降低 | 搭配体力药使用 |
| 艾丽草 | 极高 | 1~2个 | HP全满+疲劳清零 | 终局奢侈方案 |

### 2.3 时间管理: 255层冲刺

矿洞内每下行1层消耗约1分钟游戏时间,因此:

| 进入时间 | 可到达最大层数 | 备注 |
|:--------|:------------:|:----|
| AM6:00 | ~255F(理论上) | 清晨进入→深夜=最充裕 |
| PM12:00 | ~200F | 下午进入→必须高效 |
| PM5:44 | ~255F(极限) | 最晚安全进入时间 |
| PM5:45+ | <255F | 可能走不完 |

**利用陷阱快速下层**(推荐):

1. 进入新层→**按下ZR即时存档**
2. 寻找陷阱→踩踏→随机掉落1~5层
3. 如掉落层数不满意→**读档重来**(陷阱位置不变)
4. 最优目标: 每次掉落**3层**(消耗40HP性价比最高)
5. 安全层(1~19F/249~255F)可以用S/L挖矿后再走

### 3.4 钓鱼技巧

1. **抛竿位置**: 不同抛竿距离可能影响钓到的鱼种(远近水深处不同)
2. **鱼上钩判定**: 浮标沉入水中的瞬间按A→进入"拉锯战"迷你游戏
3. **鱼的力量": 传说鱼需要更多次拉锯才能钓起
4. **钓鱼EXP**: 每成功钓起一条鱼获得EXP(满100%→钓竿可跨级升级)

## 六、设计意图分析

1. **矿洞作为"资源型地牢"**: 矿石镇用矿洞探讨了一个问题:如果一个游戏没有战斗,玩家如何获得"冒险"体验?答案是将体力管理和时间管理包装为冒险的核心机制。矿洞不是迷宫(没有岔路),而是垂直的生存挑战——这本质上是资源管理博弈的"纯化"形式。

2. **255层不是关卡设计,而是耐力测试**: 两座255层矿洞没有设计变化——每层的布局基本一致。区别只在于矿石种类和隐藏物品的出现层数。这意味着矿洞探索不是"内容驱动"(探索新区域),而是"目标驱动"(我需要某层的某样东西)。这种设计在GBA时代受限于硬件,但重制版保持了这一特性——纯粹为老玩家保留体验。

3. **钓鱼作为"辅助冒险"**: 钓鱼系统在矿石镇中不占主导地位(约占总游戏时间的5~10%),但传说鱼的设定提供了类似"稀有的掉落物"的RPG式满足感——虽然钓竿没有战斗作用,但钓到传说鱼的感觉类似于在RPG中击败了一只稀有怪物。

4. **采集系统的有限性**: 母亲山每日只刷新有限数量的采集物,且售价极低(10~30G)。采集在矿石镇中不是主要经济来源,而是为料理提供免费材料,以及为早期提供一点点"额外收入"。这种设计避免了"采集碾压耕种"的不平衡。

5. **与同类游戏的地图探索对比**:
   - 星露谷物语: 地图更加开放,有秘密森林、沙漠、姜岛等多样化区域
   - 符文工房: 每作有多个迷宫地图(每迷宫10~20层),含宝箱+Boss
   - 矿石镇: 地图相对封闭,主要区域固定,探索集中在两座矿洞的垂直深度上
