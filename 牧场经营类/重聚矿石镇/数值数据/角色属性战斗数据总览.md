# 角色属性战斗数据总览 - 重聚矿石镇

> 重聚矿石镇是纯经营游戏,无战斗系统。本文档聚焦体力/疲劳/恢复/属性成长等角色生存数据。

---

## 一、无战斗系统声明

矿石镇无怪物、无战斗、无武器、无HP战斗损耗、无攻击/防御/暴击等战斗属性。角色属性围绕**体力管理**和**时间管理**两大核心。

| 传统RPG属性 | 矿石镇对应 | 说明 |
|:----------|:----------|:----|
| HP | 体力(Stamina) | 作用于耕种/采矿/钓鱼等所有劳作 |
| MP | 疲劳(Fatigue) | 类似"反向MP",越高越危险 |
| 等级 | 无角色等级 | 成长=工具升级+隐藏牧场度 |
| 攻击力 | 工具等级 | 高等级工具效率更高 |
| 防御力 | 围栏/建筑 | 保护作物和动物不受天气影响 |
| 魔法 | 无 | 无魔法系统 |

### 2.4 体力/疲劳消耗叠加机制

矿石镇的体力消耗不仅取决于动作类型,还受以下因素影响:

| 因素 | 效果 | 备注 |
|:----|:----|:----|
| 疲劳积累 | 疲劳75%+(脸变蓝)→**消耗×2** | 最危险的恶性循环 |
| 雨天 | 浇水不消耗(但疲劳累积速度不变) | 雨天不出门工作可休整 |
| 温泉泡澡 | 每10秒疲劳-2,体力不直接恢复 | 但降低疲劳=间接减少消耗 |
| 食物Buff | 特定料理可暂时提升体力上限 | 如:体力强化剂 |
| 工具祝福化 | **所有消耗减半** | 祝福工具的重要价值 |

## 七、设计意图分析

1. **体力即节奏控制器**: 体力系统本质上是游戏节奏的控制器——初期100体力约相当于30~50格操作(浇水+锄地),限制了玩家每天的扩张量。随着工具升级和力量果实收集,扩张速度逐步加快,形成自然的难度曲线。

2. **疲劳作为第二资源**: 疲劳度的设计巧妙地创造了"第二层资源管理"——体力可以靠料理快速回复,但疲劳累积需要靠温泉/睡觉解决。这防止了"无限吃料理撑体力"的滥用,迫使玩家遵循作息节奏。

3. **无战斗属性=降低门槛**: 没有攻击/防御/暴击等战斗属性意味着游戏数值系统极其简单——任何玩家不需要理解"build"就能直接上手。这是矿石镇"全年龄段友好"定位的核心体现。

4. **与符文工房的商业设计分支**: 同样出自Marvelous的牧场物语系,符文工房选择在矿石镇的体力系统基础上叠加了完整的ARPG战斗属性,形成了完全不同的用户群——矿石镇吸引经营向玩家,符文工房吸引RPG向玩家。
