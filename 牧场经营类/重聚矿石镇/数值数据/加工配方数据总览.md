### 2.2 灵感系统详解

灵感是矿石镇料理系统的核心——它不是通过"买书"或"找NPC教"来学菜谱,而是通过**实践中积累经验自然领悟**:

| 触发条件 | 解锁料理示例 |
|:--------|:-----------|
| 使用鸡蛋类食材累计N次 | 玉子烧、蛋包饭、布丁等蛋料理 |
| 使用鱼类食材累计N次 | 生鱼片、烤鱼、寿司等鱼料理 |
| 使用牛奶类食材累计N次 | 奶油、黄油、布丁等奶料理 |
| 使用面粉类食材累计N次 | 面包、面条、披萨等小麦料理 |
| 使用水果类食材累计N次 | 果汁、果酱、水果沙拉 |
| 组合使用(混合食材) | 复杂复合料理 |

**速通灵感技巧**:
1. 前期大量种植芜菁——便宜、生长快,用来练习灵感
2. 批量买入面粉(100G/个)——万能材料,几乎所有料理都能用
3. 钓鱼→做成生鱼片(初始即会)→同时练习鱼力和菜刀灵感
4. 牛奶过剩→做奶酪→练习奶类灵感

## 四、料理售价公式与经济学

### 4.1 基础公式

```
料理售价 ≈ 各食材单独售价之和 × 1.2 ~ 2.0(因食谱而定)
```

- 简单料理(如生鱼片): ×1.5~2.0倍(利润极高)
- 复合料理(多食材): ×1.2~1.5倍(利润一般)
- 四季之阳: 远超普通公式(设计溢价)

### 4.2 加工路径选择

| 畜牧产物 | 最佳加工路径 | 最终售价 | 说明 |
|:--------|:-----------|:-------|:----|
| S蛋 | →蛋黄酱 | 100G | +100%增值 |
| X蛋 | →蛋黄酱 | 800G | +129%增值 |
| S羊毛 | →毛线 | 530G | +130G |
| G羊毛 | **直接售出** | 2,500G | 加工→毛线=亏本 |
| S兔毛 | →毛线 | 530G | +290G(+121%) |
| X兔毛 | →毛线 | 3,200G | +1,400G(+78%) |

### 4.3 加工链设计总结

1. **蛋/奶**: 全部品质都建议加工(蛋黄酱/奶酪),增值始终为正
2. **兔毛**: 全部品质都建议加工(毛线),始终增值
3. **羊毛/羊驼毛**: 仅S/M/L品质加工,G/P/X品质直接出售(加工降级)
4. **特殊牛奶**(咖啡/草莓/水果): 不可加工也不可料理,唯一选择=直接出售
5. **黄油**: 不可出售,只用于料理(制作蛋糕/布丁等)

## 五、设计意图分析

1. **灵感系统: 从"被动购买"到"主动探索"**: 传统牧场游戏的料理学习方式通常是购买食谱书或NPC教学。矿石镇的灵感系统让玩家通过**反复使用食材**来自然解锁料理——这更贴近真实的"烹饪学习"过程:做了很多次番茄炒蛋后,自然知道怎么加料做番茄蛋汤。这是朴素而巧妙的设计。

2. **加工设备的深度瓶颈**: 所有加工机都需要精金石(泉矿10F+)——这意味着加工链在第一天是不可用的。玩家必须先投入时间学习下矿、收集精金石,然后才能解锁加工利润。这不是"教程→解锁"的模式,而是"探索→发现→利用"的渐进式解锁。

3. **四季之阳作为终局内容**: 四种太阳料理构成了料理系统的金字塔顶——它们所需的材料跨越了畜牧(蛋/奶/毛)、钓鱼(鱼王)、采矿(粉红钻石)、种植(部分作物)。这些都是游戏不同维度的最高成就。将这些成就汇聚到"制作料理"这一行为中,为各类玩家提供了一个统一的终局目标。

4. **加工≠全自动赚钱**: 与后来许多牧场游戏不同,矿石镇的加工需要玩家**手动操作**(每次放一个产物到机器中)。这保留了"每一件产品都被亲手加工"的仪式感,而非无脑流水线。这也意味着加工需要时间成本——玩家需要在"出货直接变现"和"加工后再出货"之间权衡时间。

5. **与星露谷的加工对比**: 星露谷物语的加工由工匠设备(酒桶、腌菜罐等)实现,周期长但利润极高(水果→果酒×3倍)。矿石镇的加工更倾向于"每一件都立刻完成"(无需等待发酵),但增值幅度较小。两种设计反映了不同的游戏节奏理念:星露谷鼓励"计划性批量生产",矿石镇鼓励"每日手动管理"。
