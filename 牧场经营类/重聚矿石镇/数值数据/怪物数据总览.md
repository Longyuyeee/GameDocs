# 环境威胁与危险数据总览 - 重聚矿石镇

> 重聚矿石镇是纯经营游戏,无战斗系统、无怪物、无Boss。本文档聚焦游戏中的环境威胁、野生动物风险与生存挑战。

## 核心结论:无传统战斗系统

矿石镇的经营核心是无战斗、零暴力的牧场生活。以下是明确的"无此系统"清单:

| 项目 | 状态 | 说明 |
|------|:---:|------|
| 战斗系统 | 无 | 无攻击/防御/暴击/武器 |
| 怪物刷新 | 无 | 矿洞/地图均无怪物 |
| 怪物掉落 | 无 | 无怪物掉落物 |
| Boss战 | 无 | 无任何Boss |
| HP战斗损耗 | 无 | 仅有体力(Stamina)和疲劳(Fatigue)系统 |
| 武器系统 | 无 | 工具仅用于农牧/采集,无武器属性 |

---

## 一、环境威胁系统

矿石镇的"敌人"不是怪物,而是**自然环境**——天气灾害、体力枯竭、时间管理。

### 1.1 天气灾害

| 灾害类型 | 季节 | 概率 | 影响 | 防护措施 |
|---------|:---:|:---:|------|---------|
| 台风(Typhoon) | 夏/秋 | 随机 | 作物损坏(未用围栏);动物无法放牧;无法外出 | 围栏保护作物;留屋内 |
| 暴风雪(Blizzard) | 冬 | 随机 | 与台风等同;无法外出;动物友好度下降 | 留屋内;提前备足饲料 |
| 大雨(Heavy Rain) | 春/夏/秋 | 较常见 | 无法出门;动物不能放牧;+1体力恢复 | 无特殊措施 |
| 大雪(Heavy Snow) | 冬 | 较常见 | 与大雨等同 | 无特殊措施 |

**灾害后的修复**:
- 被台风破坏的作物直接消失,需重新种植
- 围栏(L Fence)可以有效保护作物区域,每格围栏成本50G+材料

### 1.2 疲劳系统(隐性"敌人")

疲劳度的累积是矿石镇最具威胁性的"生存挑战",类似于传统RPG的中毒/诅咒debuff:

| 疲劳阶段 | 视觉提示 | 效果 | 触发条件 |
|---------|:-------:|------|---------|
| 正常 | 无 | 体力消耗×1.0 | 0-49%疲劳 |
| 中度疲劳 | 无明显变化 | 行动略微变慢(游戏内) | 50-74%疲劳 |
| 重度疲劳 | **脸变蓝** | **体力消耗×2** | 75%+疲劳 |
| 晕倒 | 画面转黑 | 丢失物品+金钱,在医院醒来(扣诊金10%金钱) | 体力归零+继续工作 |

**疲劳度每日恢复**:
| 方式 | 恢复量 |
|------|:-----:|
| 睡觉(PM10:00前) | 疲劳完全清零 |
| 睡觉(PM10:00后~AM0:00) | 疲劳-85%~-95% |
| 睡觉(AM0:00后) | 疲劳-70%~-80% |
| 温泉泡澡 | 每10秒-2疲劳 |
| 安神茶(Relax Tea) | -50疲劳 |
| 力量果实 | 最高体力上限+10(非直接恢复) |

## 二、野生动物威胁

### 2.1 狐狸偷鸡

狐狸(Fox)是矿石镇唯一具有"攻击性"的野生动物:

| 项目 | 数据 |
|------|------|
| 出现条件 | 鸡舍门未关,鸡在外过夜 |
| 目标 | 散养在外的鸡 |
| 后果 | 鸡被偷走(永久消失) |
| 预防措施 | PM6:00前关鸡舍门;养狗驱赶 |
| 狗驱狐效果 | 高好感度狗可自动驱赶 |
| 报警铃声 | 如果装了铃铛,Saibara会来告知 |

### 2.2 狗的功能(生态守护)

| 功能 | 解锁条件 | 效果 |
|------|---------|------|
| 驱赶狐狸 | 初始 | 阻止狐狸偷鸡(需PM8:00后在外) |
| 飞盘比赛 | 沙滩节(夏1) | 赢→获得力量果实 |
| 好感度提升 | 每日对话/刷毛/玩球 | 提高驱狐成功率和飞盘胜率 |

## 三、矿洞环境威胁(替代地牢冒险)

矿洞中没有怪物,但有三种"环境威胁":

### 3.1 塌方陷阱(Falling Trap)

本作新增的矿洞陷阱系统,是快速下层的核心机制,同时也是体力消耗的来源:

| 陷阱掉落层数 | 体力消耗 | 性价比评价 |
|:----------:|:-------:|:--------:|
| 1层 | 20HP | 最低效 |
| 2层 | 30HP | 效率一般 |
| 3层 | 40HP | **最高效** |
| 4层 | 60HP | 一般 |
| 5层 | 70HP | 高消耗 |

**安全楼层(无陷阱)**:
- 泉矿: 1~19F, 249~255F
- 湖矿: 1~9F, 249~255F

### 3.2 体力消耗(矿洞版)

| 动作 | 消耗 | 备注 |
|------|:---:|------|
| 用锤子敲石头1次 | 2HP | 普通/铜锤 |
| 敲开大石头 | 5HP | 需秘银+锤 |
| 挖地面(锄头) | 2HP | 寻找道具 |
| 踩陷阱 | 20~70HP | 见上表 |
| 下楼梯 | 0HP | 无消耗 |

### 3.3 时间限制

- 必须在 **PM5:44前** 进入矿洞,才有足够时间到达255F(每下层消耗1分钟)
- 超过AM6:00强制晕倒→丢失物品

## 六、设计意图分析

1. **纯经营理念**: 矿石镇代表"零暴力"牧场游戏的最纯粹形态——所有冲突都来源于自然环境而非敌人。这种设计使得游戏对全年龄段友好。

2. **体力即HP**: 在传统RPG中,HP是战斗资源;在矿石镇中,体力(Stamina)和疲劳(Fatigue)共同构成生存资源。体力管理替代了战斗策略,本质上是"资源管理型冒险"。

3. **天气作为"Boss"**: 台风和暴风雪是游戏中唯一会"主动攻击"玩家的系统——它们无视玩家的操作,按照预定日程(Schedule)或随机发生。玩家需要提前准备(建造围栏、储备饲料)来应对。

4. **狐狸偷鸡作为稀有事件**: 唯一具有"动画死亡"效果的事件,概率极低(多数玩家几百小时都不会遇到),但起到了"提醒玩家管理牧场安全"的警示作用。

5. **与符文工房的设计分支**: 同样是牧场物语系,符文工房(2006年起)将"矿洞怪物"升级为可战斗的ARPG系统,代表完全不同的设计方向——两者都是牧场物语衍生,但矿石镇坚持了原版的纯粹性。
