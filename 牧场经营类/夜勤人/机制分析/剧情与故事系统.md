# 剧情与故事系统 - 夜勤人(Moonlighter)

> 游戏版本: Moonlighter v1.0+ (PC/Switch/PS4/Xbox, 2018年)
> 数据来源: moonlighter.fandom.com, Wikipedia, TrueAchievements

## 1. 世界观设定

| 要素 | 说明 |
|------|------|
| 背景 | 考古发掘中发现神秘**地牢大门**——通往不同维度的古代通道 |
| 小镇 | **莱诺卡村(Rynoka)**——因大门发现而兴起，曾繁荣但因地牢伤亡率上升逐渐衰落 |
| 两类探索者 | **英雄**(杀怪获荣)和**商人**(取宝箱获物出售) |
| 主角身份 | **商人**——不是"天选勇者"而是"继承了家族店面的年轻人" |

## 2. 主角

**威尔(Will)**——17岁孤儿，继承家族商店"夜勤人(The Moonlighter)"。虽是商人而非英雄，却因发现疯狂皮特的笔记而踏上击败四个守护者的旅程。

## 3. 主线剧情

### Act 1: 继承商店
- Will继承破败的家族商店
- 探索魔像地牢→发现**疯狂皮特(Crazy Pete)**的尸体
- 皮特的笔记揭示: 四个守护者持有钥匙能打开神秘的**第五门**

### Act 2: 四个守护者
- 击败魔像王(Golem King)→获得第一把钥匙
- 击败食人花(Carnivorous Mutae)→第二把钥匙
- 击败娜迦(Naja)→第三把钥匙
- 击败能量流(Energy Flux)→第四把钥匙

### Act 3: 第五门真相
- 收集四钥匙打开第五门
- 发现**维度海盗**基地——他们曾是最早实现跨维度旅行的文明
- 海盗掠夺各维度后藏匿于此→躲避ICT(跨维度商业条约)的追捕
- 随从被Crazy Pete杀死→困在此处直到Will开门
- 只剩**最后的维度海盗船长**

### Act 4: 结局
- Will击败海盗船长
- ICT认可Will→正式邀请该维度加入跨维度商业圈
- 地牢永久封印
- 从"普通商人"到"为维度打开了星际商业大门的人"

## 4. 叙事方式

| 机制 | 说明 |
|------|------|
| 地牢笔记 | 疯狂皮特的笔记散布在地牢中→逐步揭示秘密 |
| 营地日记 | 废弃营地中有剧情文本 |
| NPC对话 | 简单的剧情推进对话(无分支) |
| 环境叙事 | 地牢场景和Boss设计讲述维度海盗的历史 |

## 5. 设计特点

1. **商人主角**: 不是"去拯救世界"而是"继承家业"——动机是经营而非英雄主义
2. **科幻反转**: 奇幻地牢背后的真相是维度海盗(科幻)——出人意料的类型转折
3. **最少化叙事**: 无长篇对话/支线/分支结局——重点在玩法而非故事
4. **疯狂皮特=叙事催化剂**: 一个已死的冒险家留下的笔记是整个主线的驱动力
