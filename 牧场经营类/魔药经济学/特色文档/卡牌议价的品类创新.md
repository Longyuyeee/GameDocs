# 卡牌议价的品类创新 - 魔药经济学(Potionomics)

> 游戏版本: Potionomics (2022-2024年)

## 一、概述

魔药经济学将**讨价还价(Haggling)**这一商店经营的核心机制，从"数字博弈"(Recettear)或"表情观察"(Moonlighter)进化为**20张卡的Deck-Building战斗**。这是经营品类中议价系统最深度的实现。

## 二、三代议价进化

| 游戏 | 议价方式 | 深度 |
|------|:--:|:--:|
| Recettear(2007) | 数字博弈——出价→顾客还价→再出价 | ★★★★★ |
| Moonlighter(2018) | 情感观察——设价格→看表情→调整 | ★★★ |
| **Potionomics(2022)** | **卡牌Deck-Building——20张卡组×5种类型×多回合策略** | ★★★★★★ |

## 三、为什么卡牌议价是进化

| 维度 | 数字博弈(Recettear) | 卡牌Deck-Building(Potionomics) |
|------|:--:|:--:|
| 策略深度 | 记忆顾客的价格因子 | **卡组协同+出牌顺序+时机管理** |
| 成长系统 | 无(仅经验值) | **社交→卡牌→更强Deck→更难的谈判** |
| 可重玩性 | 中(16种顾客记住价格后重复) | **高(Deck组合无限+每局不同)** |
| 趣味性 | 数学优化 | **卡牌战斗的紧张感+多巴胺** |

## 四、卡牌来源=社交驱动

```
交友(NPC好感↑)
  → 解锁NPC特色卡牌
    → Deck更强→议价更成功→更多金币
      → 升级设备→更强药水→更高基础价格
        → 更难的比赛→需要更好的Deck
```

## 五、设计分析

1. **Deck-Building+经营=品类的终极进化**: 议价从"系统功能"变为"核心玩法"——不是"你定价"而是"你打牌谈判"
2. **社交=卡牌成长**: 交友不是为了"看剧情"而是为了"获得更强的议价卡牌"——社交有功利价值
3. **耐心/压力/兴趣三维**: 不同于大多数卡牌游戏的"HP+法力"二维→三维资源管理
4. **比赛=Deck检查**: 每10天的比赛是Deck强度的硬检查——逼你不断优化
