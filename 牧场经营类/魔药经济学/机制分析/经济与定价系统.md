# 经济与定价系统 - 魔药经济学(Potionomics)

> 游戏版本: Potionomics (2022-2024年)
> 数据来源: TheGamer, PSNProfiles

## 1. 货币体系

| 货币 | 获取 | 用途 |
|------|------|------|
| 金币(Gold) | 出售药水/比赛奖金 | 升级设备/购买原料/装饰 |
| 好感度(Friendship) | 与NPC交友 | **解锁卡牌**(Deck-Building核心!) |

## 2. 药水定价

### 2.1 基础价格公式

```
基础价格 = 药水品质等级 × 特性修正 × 议价结果
品质等级: Minor(×1.0)→Masterwork(×3.0+)
特性: 5维度×±5%叠加
议价: 卡牌战斗→最终成交价(可能远高于基础!)
```

### 2.2 收入来源

| 来源 | 效率 | 说明 |
|------|:--:|------|
| 手工出售(卡牌议价) | ★★★★★ | **核心收入**——Deck越好利润越高 |
| 自动售货机 | ★★★ | 无议价→15%溢价→懒人模式 |
| 比赛奖金 | ★★★★ | 一次性但数量可观 |
| 熟成药水 | ★★★★ | 延时→更高价格 |

## 3. 主要消费

| 消费项 | 说明 |
|------|------|
| 原料购买 | 药水制作基础 |
| 设备升级(大锅/货架) | MukTuk→更高级制作能力 |
| 装饰 | 提升售价+药水品质 |
| 熟成桶 | 额外投资→延时回报 |

## 4. 设计特点

1. **价格=制作×议价的双层**: 基础价格(制作品质)+最终价格(卡牌议价)→两个系统共同决定收益
2. **自动售货机=放弃议价利润**: 类似Recettear的自动售货机——QoL牺牲利润
3. **熟成=期货**: 现在放入→几天后取出→更高价格——时间投资
