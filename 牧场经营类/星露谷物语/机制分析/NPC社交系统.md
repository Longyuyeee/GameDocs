## 3. 好感度机制

### 3.1 好感度等级

| 状态 | 心级(Hearts)上限 | 好感度数值 |
|------|:----------------:|:----------:|
| 普通村民 | 0-10心 | 0-2,500 |
| 可结婚对象(单身) | 0-8心(送花束后解锁10心) | 0-2,000→解锁后→0-2,500 |
| 伴侣(婚后) | 0-14心 | 0-3,500 |
| 科罗布斯(室友) | 0-14心 | 0-3,500 |

> 每半心 = 100好感值，1心 = 200好感值。

### 3.2 好感值变化表

| 行动 | 好感值变化 | 备注 |
|:----:|:----------:|:----:|
| **每日第一次对话** | **+20** | 最稳定的好感获取方式 |
| 送礼(最爱/Loved) | **+80** | 最高单次增幅 |
| 送礼(喜欢/Liked) | **+45** | 次优选择 |
| 送礼(普通/Neutral) | +20 | 无加成也没惩罚 |
| 送礼(不喜欢/Disliked) | **-20** | 惩罚 |
| 送礼(讨厌/Hated) | **-40** | 最大惩罚 |
| **生日送礼(最爱)** | **+640** | 80×8倍！最大单次好感 |
| **生日送礼(喜欢)** | **+360** | 45×8倍 |
| **完成任务** | **+150** | 公告板/求助任务 |
| 不交谈(当天) | -2 | 仅限非满心时(每心扣2点) |
| 夏威夷宴会汤好评 | +120(全体村民) | 上优质食材 |
| 夏威夷宴会汤差评 | -100(全体村民) | 上垃圾/差食材 |
| 电影约会(最爱零食) | +200 | 1.6新增电影院功能 |
| 电影约会(喜欢零食) | +100 | — |

### 3.3 好感度衰减机制

| 条件 | 衰减值 | 说明 |
|:----:|:------:|------|
| 当日未与该NPC说话 | -2 | 仅限好感度未满心的NPC |
| 每周衰减上限 | -10 | 最多不会每周降太多 |
| 满心后是否衰减 | 否 | 达到当前心上限后不再衰减 |

> 使用花束将可结婚对象的关系从8心"提升"到10心上限后，10心上限内仍有衰减，但使用人鱼吊坠结婚后额外心数不再衰减。

### 4.1 基本规则

| 规则 | 内容 |
|:----|------|
| **每周送礼限制** | 每个NPC每周最多接受**2次**礼物 |
| **生日额外送礼** | 生日当天可额外送1次(不限在2次内) |

## 5. 恋爱/婚姻系统

### 5.1 恋爱发展流程

```
8心好感 → 送花束 → 确认恋爱关系 → 10心好感 → 人鱼吊坠 → 求婚 → 3天后结婚
```

| 阶段 | 条件 | 内容 |
|:----:|:----:|------|
| **好感8心** | 与可结婚对象好感达到8心 | 从皮埃尔处购买花束(Bouquet)配方(200g)并制作 |
| **送花束** | 好感≥8心时送出花束 | 确认恋爱关系，好感上限提升至10心 |
| **好感10心** | 恋爱状态下达到10心 | 可购买人鱼吊坠 |
| **人鱼吊坠** | 雨天去海滩(东侧)，找老水手(Old Mariner) | 花费5,000g购买，需携带 |
| **求婚** | 将人鱼吊坠送给恋爱对象 | 对方立即接受，约定3天后结婚 |
| **结婚** | 求婚3天后 | 自动婚礼(主角前往)，必定晴天，配偶搬入农场 |

### 5.2 老水手(Old Mariner)出现条件

| 条件 | 说明 |
|:----|:----|
| **天气** | 必须是**雨天**(非暴风雨也可) |
| **位置** | 海滩东侧(经过断桥后的区域) |
| **时间** | 全天(只要下雨且桥已修复) |
| **前置** | 必须在皮埃尔处买过花束并已送出 |
| **其他** | 海滩断桥需花费300木材修复 |

## 6. 婚后内容详解

### 6.1 伴侣搬入农场后的变化

| 项目 | 说明 |
|:----|:----|
| **配偶房间** | 每位配偶在农场房屋右侧增加独特风格的房间 |
| **户外区域** | 农场房屋后方增加配偶专属户外区域(风格化小花园/角落) |
| **帮助农活** | 随机：浇水作物、喂动物、修围栏、给宠物水碗添水 |
| **做早餐** | 早晨有时为玩家做早餐(提供增益效果的食物) |
| **星之果实** | 好感12.5心时赠送星之果实(+34能量上限，仅一次) |

### 6.2 子女系统

| 项目 | 说明 |
|:----|:----|
| **触发条件** | 房屋2级升级(育儿室，50,000g+150硬木)；结婚后约5%概率/晚触发 |
| **数量上限** | 最多**2个**孩子 |
| **性别** | 随机 |
| **生育方式** | 伴侣怀孕(女角色)或领养(男角色或同性伴侣) |
| **成长阶段** | 婴儿(Baby，14天) → 站立(Crawler，14天) → 爬行(28天) → 幼儿(Toddler，永久) |
| **幼儿状态** | 永久保持为幼儿，不会继续长大 |
| **功能** | 纯装饰，幼儿会跑动，可戴帽子 |

### 6.3 婚后好感维持

| 行动 | 好感变化 | 备注 |
|:----|:---------|:----|
| 每日对话 | +20 | 每天稳定来源 |
| 送礼(最爱) | +80 | 婚后仍可送礼 |
| 亲吻(好感≥10心) | +10 | 点击伴侣可亲吻，每日1次 |
| 给伴侣倒咖啡/茶 | 小额提升 | 有增益效果 |
| 忽视伴侣 | 不衰减 | 婚后好感不减(但每天对话仍有+20) |

## 7. Krobus室友路线

科罗布斯(Krobus)是下水道的暗影人(Shadow Person)，提供独特的**室友**机制。

### 7.1 室友路线流程

| 条件 | 说明 |
|:----|:----|
| **好感达到10心** | 与科罗布斯好感达到10心 |
| **获取幽灵吊坠(Void Ghost Pendant)** | 在女巫沼泽的暗黑神龛使用一个虚空蛋黄酱(Void Mayonnaise)获得 |
| **赠送幽灵吊坠** | 将幽灵吊坠送给科罗布斯 |
| **科罗布斯搬入** | 与结婚类似，科罗布斯入住农场房屋 |
| **好感上限** | 0-14心(同结婚系统) |

### 7.2 室友与结婚对比

| 项目 | 结婚 | 科罗布斯室友 |
|:----:|:----:|:------------:|
| 14心上限 | 是 | 是 |
| 搬入农场 | 是 | 是 |
| 配偶房间 | 是 | 是(暗影风格) |
| 户外区域 | 是 | 否 |
| 星之果实赠送 | 是(12.5心) | 是(12.5心) |
| 做早餐 | 是 | 是(提供随机黑暗料理) |
| 孩子 | 是(最多2个) | **否** |
| 亲吻 | 是 | **否** |
| 离婚 | 可行 | **不适用**(直接去暗黑神龛分开) |
| 结婚兼容 | — | 不能同时结婚 |

> **重要：** 一旦选择科罗布斯室友路线，就不能再结婚。同样，已婚玩家不能邀请科罗布斯为室友。

## 9. NPC日程系统

NPC的每日行走路线和行为受多种因素影响。

### 9.1 影响因素

| 因素 | 影响 |
|:----|:----|
| **季节(Season)** | 不同季节NPC的日程可能不同 |
| **天气(Weather)** | 雨天/雪天NPC通常留在室内或在有遮蔽地点 |
| **星期(Day of Week)** | 很多NPC有固定日程(如皮埃尔周三休息、罗宾周二健身) |
| **好感度(Heart Level)** | 高好感可能解锁新的日程路线和访问区域 |
| **节日(Festival)** | 节日日NPC全天在节日地点 |
| **社区中心完成** | 一些NPC的日程会变化(如克林特不再周五关门) |
| **特殊事件** | 婚后伴侣的日程完全改变(固定在农场) |

### 9.2 常见NPC日程模式

| NPC | 典型日程特点 | 最易找到的位置 |
|:----:|------------|:--------------:|
| 刘易斯(Lewis) | 上午在家，下午在镇上走动 | 镇长家/皮埃尔家 |
| 皮埃尔(Pierre) | 9:00-17:00在杂货店，周三休息 | 杂货店(周三找不到) |
| 罗宾(Robin) | 9:00-17:00在木匠店，周二上午健身 | 木匠店(周二上午在卤酒吧) |
| 克林特(Clint) | 9:00-16:00在铁匠铺，周五晚上去卤酒吧 | 铁匠铺 |
| 潘姆(Pam) | 中午前在家，下午在卤酒吧 | 拖车/卤酒吧 |
| 科罗布斯(Krobus) | **全天在下水道** | 下水道 |
| 矮人(Dwarf) | **全天在矿井** | 矿井入口右侧(需钢镐敲开障碍) |
| 桑迪(Sandy) | **全天在绿洲商店** | 卡利科沙漠 |
| 法师(Wizard) | **全天在法师塔** | 法师塔 |

### 9.3 婚后伴侣日程

| 行为 | 时间 | 说明 |
|:----|:----:|:----:|
| 在屋内活动 | 清晨/夜晚 | 在配偶房间或厨房 |
| 在户外区域 | 白天 | 在农场房屋后方专属区域 |
| 随机外出 | 某些天 | 回原住所探望亲友 |
| 帮农活 | 随机 | 浇水/喂动物/修围栏 |
