## 一、经济系统 (Economy System)

### 1.1 货币类型 (Currency Types)

| 货币名称 | 用途 | 获取方式 |
|---------|------|---------|
| **金币 Gold (g)** | 通用货币，几乎所有交易 | 出售物品、完成任务等 |
| **Qi 币 Qi Coins** | 赌场 (Casino) 消费 | 用金币兑换，10g = 1 Qi Coin，不可反向兑换 |
| **Qi 宝石 Qi Gems** | Qi 的核桃房商店 | 完成 Mr. Qi 的特殊挑战任务、出售多余金核桃 |
| **星币 Star Tokens** | 星露谷展览会 (Stardew Valley Fair) | 参加节日小游戏 |
| **五彩蛋 Calico Eggs** | 沙漠节 (Desert Festival) | 节日期间活动获取 |
| **金核桃 Golden Walnuts** | 姜岛鹦鹉解锁功能 | 全岛探索收集（共130个） |

### 1.2 出货箱机制 (Shipping Bin)

- **位置**：农场上默认有一个出货箱 (Shipping Bin)
- **结算时间**：放入物品后，每天**入睡后**自动结算，次日醒来时金币到账
- **不可撤回**：一旦入睡结算，物品无法取回
- **当日修正**：同一日内可取出最后放入的**1个物品**（右键点击出货箱）
- **价格说明**：出货价格 = 直接卖给商店的价格，无差异
- **额外出货箱**：可从木工店购买，250g + 150 木材/个
- **迷你出货箱 Mini-Shipping Bin**：Qi 宝石商店 60 Qi Gems，可放置在任何地方
- **全成就追踪**：出货记录用于"Full Shipment"成就和 100% Perfection

### 1.3 主要收入来源 (Income Sources)

星露谷物语没有官方的收入权重统计，但根据社区共识，各技能在游戏不同阶段的收入权重如下：

#### 前期（第1年春夏）：
1. **钓鱼 (Fishing)** — 前期收入主力，单日可达 2,000-5,000g
2. **采集 (Foraging)** — 春季野菜/葱等早收
3. **农耕 (Farming)** — 逐步扩大规模
4. **采矿 (Mining)** — 获取矿石和宝石

#### 中期（第1年秋冬 - 第2年）：
1. **农耕 (Farming)** — 大规模种植 + 工匠设备加工，占收入 60-70%
2. **畜牧 (Animal Products)** — 加工品（奶酪/蛋黄酱/松露油）
3. **钓鱼 (Fishing)** — 传说鱼和特定鱼种
4. **采矿/战斗 (Mining/Combat)** — 宝石、铱矿

#### 后期（第2年+）：
1. **工匠制品 (Artisan Goods)** — 上古水果酒/杨桃酒等，占收入 80%+
2. **畜牧 (Animal Products)** — 猪的松露油
3. **姜岛种植** — 全年不间断生产
4. **宝石复制机 (Crystalarium)** — 钻石被动收入

### 1.6 市场波动 (Market Dynamics)

- **星露谷物语没有供求机制或价格浮动系统**
- 所有物品的出售价格是**固定的**，不受以下因素影响：
  - 出售数量（多卖不会跌价）
  - 季节变化
  - 游戏进度
  - 村民好感度
- 唯一的价格差异来源：**品质等级（银/金/铱）** 和 **耕种职业加成（Tiller/Artisan）**

### 1.7 季节对经济的影响 (Season Impact)

| 季节 | 经济影响 |
|------|---------|
| **春季** | 野外采集收入高（葱、蒲公英、野山葵等），可种草莓高利润 |
| **夏季** | 农耕黄金期，蓝莓/杨桃/啤酒花利润极高 |
| **秋季** | 蔓越莓/南瓜利润高，采集物种类多（黑莓灌木丛） |
| **冬季** | **户外无法种植常规作物**，收入转向畜牧/温室/采矿/钓鱼/冬季种子 |
| **跨季作物** | 玉米(夏+秋)、上古水果(春夏秋)、咖啡豆(春夏)、小麦(夏+秋)在跨季时可存活 |

### 2.3 生长周期模型 (Growth Cycle)

- **生长阶段**：每种作物分为3-6个不等的生长阶段，每个阶段持续1天以上
- **每日检查**：每天清晨检查是否满足生长条件（是否浇水）
- **跨季规则**：
  - 多季节作物（玉米/上古水果/咖啡/小麦）可跨季存活
  - 单一季节作物在季节更替时**立即死亡**（肥料也会消失）
  - 温室和姜岛不受季节限制
- **枯萎条件**：乌鸦可吃掉作物（需要稻草人防护）、闪电可击中作物
- **生长激素**可缩短生长时间（按百分比）

### 2.4 品质机制 (Quality System)

#### 品质等级与售价倍数

|-----|---------|-------|
| 普通 (Regular) | x1.0 | 无 |

| 金星 (Gold) | x1.50 | 金色 |
| 铱星 (Iridium) | x2.0 | 紫色 |

#### 品质概率计算公式

品质判定顺序：**铱星 -> 金星 -> 银星 -> 普通**

```
金星概率 = 0.2 * (耕种等级/10) + 0.2 * 肥料等级 * ((耕种等级+2)/12) + 0.01
银星概率 = Min(0.75, 金星概率 * 2.0)
铱星概率 = 金星概率 / 2  （仅使用豪华肥料时有效）
```

#### 心情系统 (Mood System)

| 心情值范围 | 心情描述 | 修正系数 |
|-----------|---------|---------|
| 0-255 | — | — |
| ≥200 | "Looks really happy today!" | 心情 x **1.5** |
| 30-199 | "Looks fine." | 修正值 = **0** |
| 0-29 | "Looks sad." | 修正值 = **-100**（心情-100） |

#### 大型产品判定公式

前提条件：好感度 ≥ 200

```
判定分数 = (好感度 + 心情修正值) / 1200
随机数 = 0~1 的浮点数
判定分数 > 随机数 => 产出大产品
判定分数 ≥ 1.0 => 100%产出大产品
```

> 例：好感度600(3心) + 心情210 => 分数 = (600 + 210x1.5) / 1200 = 0.7625 => **~76% 大产品概率**

#### 产品品质判定公式

**品质判定顺序**：铱星 -> 金星 -> 银星 -> 普通

```
品质分数 = (好感度 / 1000) - (1 - (心情值 / 225))

职业加成：牧羊人/鸡舍大师 +0.333
```

**判定流程**：

| 步骤 | 条件 | 结果 |
|-----|------|------|
| 1 | 分数 > 0.95 AND (分数/2) > 随机数(0~1) | **铱星** |
| 2 | 未铱星 AND (分数/2) > 随机数(0~1) | **金星** |
| 3 | 未金星 AND 分数 > 随机数(0~1) | **银星** |
| 4 | 以上条件均不满足 | **普通** |

> 例：好感度600 + 心情150 => 分数 = 0.6 - (1 - 150/225) = 0.6 - 0.333 = **0.266**
>   金星概率 ≈ 13%，银星概率 ≈ 27%

| **功能** | 每天早上自动收集建筑内动物产物，存于内部36格箱子 |
| **缺点** | 不获得采集的友好度+5，不获得耕种经验 |

#### 自动抚摸机 (Auto-Petter)

| 属性 | 数值 |
|-----|------|
| **获取方式1** | 骷髅洞穴宝藏房（~3.6%概率）|
| **获取方式2** | Joja 路线购买 **50,000g** |
| **获取方式3** | 挑战之神庙激活后的危险矿井 |
| **功能** | 每天自动抚摸建筑内全部动物 |
| **友好度** | 自动抚摸每次+8（手动抚摸+15），**但可与手动叠加心情** |

#### 最高效畜牧自动化方案

```
豪华畜棚/鸡舍 + 自动喂食系统 + 自动采集机 + 自动抚摸机 + 放牧草地
= 无需日常交互、全自动畜牧
```
