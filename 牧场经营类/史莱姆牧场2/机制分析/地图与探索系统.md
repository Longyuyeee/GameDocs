## 1. 彩虹岛(Rainbow Island)世界概述

史莱姆牧场2发生在地球之外的未知星球——**彩虹岛(Rainbow Island)**，与前作"远远牧场(Far Far Range)"并非同一星球。彩虹岛是一个神秘的人工改造岛屿，充斥着古代科技、废弃无人机和史莱姆生态。岛屿分为6个主要生物群系，每个具有独特的地貌、气候和史莱姆种群。

## 3. 探索门控系统

SR2的探索门控通过多层机制实现，与SR1的"解锁牧场扩展"模式完全不同。

### 3.1 Gordo门控(主要门控)

```
彩虹田野(初始)
  |-> 粉色Gordo(30+任意) -> 余烬谷
  |-> 棉花Gordo(30+蔬菜) -> 星光滨
  |-> (后续)
余烬谷 -> 爆炸Gordo(50+肉类) -> 灰色迷宫入口(替代)
星光滨 -> 振翅Gordo(15个月露花蜜) -> 灰色迷宫入口
余烬谷(瀑布后) -> 3个Plort雕像 -> 粉雪崖
```

### 3.2 Plort门(Plort Gates)

Plort门是散布在关卡中的次级门控——需要特定种类和数量的Plort才能解锁:

| Plort门位置 | 所需Plort | 解锁内容 |
|-------------|-----------|----------|
| 余烬谷(洞穴入口) | 特定Plort | 水晶史莱姆区域 |
| 星光滨(多处) | 各类Plort | 隐藏宝箱/稀有资源 |
| 灰色迷宫 | 终局Plort | 核心区域 |

### 3.3 地图数据节点(Treasure Pods)

| 类型 | 功能 |
|------|------|
| 全息笑脸标记 | 绿色发光史莱姆笑脸 |
| 每区域数量 | 2-3个 |
| 激活效果 | 揭示该区域地图和资源位置 |
| 寻找提示 | 喷射包飞到高处更容易发现 |
| 解锁内容 | 新蓝图/装饰/升级 |

## 6. 设计特点分析

1. **Gordo门控=有机探索驱动**: 与传统的"到等级解锁区域"不同，SR2的探索门控是"喂饱Gordo"。这创造了"探索->收集食物->喂Gordo->探索更多区域"的正向循环。每个Gordo不仅是障碍，还暗示了所需食物类型——引导玩家建立对应的农业系统。

2. **FPS视角的探索优势**: 第一人称3D环境下的喷射包飞行+冲刺，使得探索本身具有动作游戏般的快感。对比《星露谷物语》的俯视角平面地图探索，SR2的3D垂直空间(洞穴、悬崖、瀑布后)提供了更多的发现惊喜。

3. **资源再生与探索动机**: 区域的资源节点每1-2游戏日刷新一次(非保证出现)。这意味着玩家需要定期回到已探索区域获取特定资源——而不是"探索一次就毕业"。这保持了各区域在整个游戏周期中的经济价值。

4. **灰色迷宫=终局探索范式转变**: 灰色迷宫的辉光投影器谜题(光线折射解谜)与之前的"收集+喂食"模式完全不同。这是游戏对玩家的终极考验——从"经营思维"转变为"解谜思维"。这种范式转变防止了终局体验变成"更大的牧场"的无趣扩展。

5. **与SR1的地图对比**: SR1的"远远牧场"是半开放沙漠地形，区域间联系松散; SR2的彩虹岛是紧密连接的垂直生态系统——每个区域有明确的物理联系(传送器/通道)，世界整体感更强。SR2的地图约为SR1的2倍大小。
