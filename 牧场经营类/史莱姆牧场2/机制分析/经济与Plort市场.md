## 1. 货币体系概述

史莱姆牧场2的货币体系比前作更复杂——除了新币(NB)之外，Plort和制造材料也构成了完整的经济循环。

| 货币类型 | 获取方式 | 主要用途 |
|----------|----------|----------|
| 新币(Newbucks / NB) | 出售Plort完成订单开宝箱 | 购买基础设施和升级 |
| Plort(史莱姆结晶) | 史莱姆进食后产出 | 制造材料区域解锁**经济核心** |
| 制造材料(矿石/木材等) | 资源采集器采集 | 制造台制作高级装备 |
| 奇异钻石(Strange Diamond) | 稀有节点宝箱 | 量子无人机等高级制作 |
| 棱柱代币(Prisma Token) | 夜市兑换(ProntoMart) | 终局兑换特殊物品 |

SR2与SR1最大的区别: 前作的经济是"Plort->新币->升级"的单向循环; SR2增加了"Plort+材料->制造升级"的制造维度，使Plort不仅是货币，还是制造材料。

## 3. 动态价格系统

### 3.1 市场机制

| 机制 | 说明 | 策略含义 |
|------|------|----------|
| 价格箭头 | 每个Plort显示🟢上涨/🔴下跌/⚪平稳 | 上涨时卖出，下跌时等待 |
| 过度出售惩罚 | 大量出售同一种Plort->价格持续下跌 | 防止单一策略 |
| 需求量恢复 | 一段时间不出售->价格缓慢回升 | 储存等待策略可行 |
| 饱和阈值 | 不同Plort的饱和阈值不同 | 高阈值=更抗跌 |
| 每日波动 | 每天市场有小幅随机波动 | 短期价格不可预测 |

### 3.2 市场饱和阈值解释

饱和阈值表示该Plort市场能承受的日销售量而不触发重度贬值:

- **阈值50**(粉色): 市场承受力最强——粉色Plort需求极高——但单价最低
- **阈值15**(终局/天气史莱姆): 市场承受力最弱——少量出售即触发跌价——需**多样化策略**
- **阈值20~25**(中档史莱姆): 平衡型——适合批量养殖

### 3.3 价格恢复机制

```
出售量 > 阈值 -> 价格下跌15-25%
  ↓
停止出售 2-3 游戏日 -> 价格回升至基准价
  ↓
出售量 < 阈值 -> 价格维持基准价±5%
```

## 7. 设计特点分析

1. **Plort=唯一经济基础**: 与绝大多数牧场经营游戏不同——SR2不卖作物/动物/加工品——唯一收入来源是Plort。经济链条极度聚焦: 史莱姆->Plort->新币。这使得每一只史莱姆都是"生产资料"，每一只Largo都是"效率升级"。

2. **动态市场=反单一策略**: 价格箭头和饱和阈值的设计，明确惩罚"只养一种史莱姆"的策略。这是对Largo系统的经济层补充——Largo在物理空间上鼓励杂交，动态市场在经济层面也鼓励多样化。两个系统相互强化。

3. **无通货膨胀压力**: 对比《星露谷物语》中后期赚钱太多无处可花，SR2通过制造台系统(需要大量材料+Plort)保持经济压力。玩家永远不会"钱多到不用思考"——因为高级升级需要的是罕见材料而非单纯的新币。

4. **黄金Plort=黑天鹅事件**: 遇到黄金史莱姆并成功投喂，一次获得~900+新币(3个x300)。这在游戏早期相当于半天的收入——创造了"野外探险可能暴富"的期待感。类似《暗黑破坏神》的稀有掉落机制。

5. **与SR1的经济对比**: SR1的经济是"养鸡生蛋"的线性模式; SR2的"Plort+材料"二元经济增加了制造深度。这种变革的代价是经济系统复杂性显著提升——新手需要同时管理新币、多种Plort和多种制造材料。
