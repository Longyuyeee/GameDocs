## 1. Gordo系统概述

Gordo是史莱姆牧场2中探索门控系统的核心机制——它们是巨大化的史莱姆(约为普通史莱姆的20-30倍)，静态地堵住关键路径/入口。玩家需要喂食Gordo足够的食物使其爆炸(释放出大量小史莱姆和奖励)，从而解锁新区域。

### 1.1 Gordo喂食机制

| 属性 | 说明 |
|:---:|------|
| 喂食方式 | 向Gordo发射食物(吸尘背包仍进Gordo嘴里) |
| 最爱食物效果 | 最爱食物的饱食度是普通食物的**2倍** |
| 膨胀阶段 | Gordo会随着喂食逐渐变大，颜色变深 |
| 爆炸前兆 | Gordo开始剧烈抖动，发出特殊音效 |
| 爆炸效果 | 释放大量对应种类小史莱姆(8-12只)+宝箱+新币+可能解锁新区域 |
| 爆炸冷却 | Gordo爆炸后，该位置变成空区域(不会刷新) |

### 1.2 最佳喂食策略

Gordo的最爱食物机制是最大化效率的关键:

- 使用最爱食物 = 仅需约30个
- 使用普通食物 = 需要约50-60个
- **振翅Gordo例外**: 最爱食物为月露花蜜，仅需15个即可喂饱(这是游戏中最高效的Gordo解锁)

## 5. Plort雕像系统

除了Gordo，部分区域使用**Plort雕像(Plort Statues)**作为解锁机制:

| 位置 | 所需Plort | 解锁内容 |
|------|-----------|----------|
| 余烬谷瀑布后通道 | 3个特定Plort | 通往粉雪崖 |
| 星光滨Plort门 | 各类Plort | 隐藏宝箱/秘密区域 |
| 灰色迷宫 | 终局Plort | 核心区域 |

Plort雕像与Gordo的区别:
- Gordo: 需要大量食物(30-60个) -> 一次性解锁
- Plort雕像: 需要特定Plort -> 每次通过消耗Plort -> 永久解锁

### 6.2 避坑提醒

| 常见错误 | 后果 | 正确做法 |
|----------|------|----------|
| 用普通食物喂振翅Gordo | 需要50-60个 | 夜间采月露——仅需15个——最划算 |
| 过早解锁剑齿Gordo | 粉雪崖区域难度高——缺乏喷气背包等级 | 至少解锁喷气背包II再去 |
| 忽略爆炸Gordo | 灰色迷宫只能通过振翅Gordo进入 | 两条路线都可行——留一个备用 |
| 一次性喂太多 | Gordo吃不下——浪费食物 | 每次喂食后等Gordo完成"吞咽"动作 |
| 喂Gordo时背包太满 | 爆炸释放的奖励可能丢失 | 清空背包再去喂Gordo |

## 7. 设计特点分析

1. **Gordo=有机的探索门控**: Gordo系统不是"等级到了才能去新区域"的传统模式，而是"收集食物->探索->喂食->解锁"的有机循环。每个Gordo堵在关键路径上，让玩家自然形成"先去探索->带回食物->喂饱Gordo"的行为模式。

2. **最爱食物=食物系统中的策略线索**: 每种Gordo的最爱食物其实是在暗示玩家该如何准备——棉花Gordo爱吃蔬菜->提示玩家建立花园; 猎人Gordo爱吃肉->提示玩家建立鸡舍。Gordo系统同时充当了"教学工具"的角色。

3. **振翅Gordo的"最优设计"**: 振翅Gordo仅需15个月露花蜜(而其他Gordo需要30-60个食物)，这是开发者有意为之的"最佳路径"设计。月露花蜜需要夜间去星光滨采集——这本身就是一种"任务"，完成这个任务后获得最便宜的灰色迷宫入口——奖励玩家"冒险精神"。

4. **Gordo作为经济控制阀**: Gordo需要的大量食物(30-60个)在早期是一笔不小的投资。玩家必须规划:"是先买围栏升级，还是先存食物解锁新区域?" 这个"投资vs扩张"的选择是SR2经济策略的一部分。

5. **与SR1的Gordo对比**: SR1的Gordo数量较少，爆炸后主要释放稀有资源。SR2的Gordo系统大幅扩展——数量翻倍+部分Gordo需要最爱食物才能高效解锁+与主线剧情(灰色迷宫入口)直接绑定。Gordo从SR1的"可选挑战"变成了SR2的"核心推进机制"。
