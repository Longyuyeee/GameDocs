## 1. 资源系统概述

SR2引入了一个比前作更复杂的资源系统——玩家不仅要管理Plort和新币，还需要采集多种制造材料用于制作升级和装置。这个"Plort+材料"的二元资源系统是SR2与SR1最大的设计差异之一。

### 1.1 资源类型全览

| 资源类型 | 获取方式 | 用途 | 稀有度 |
|----------|----------|------|:------:|
| 新币(NB) | 出售Plort/完成任务/开宝箱 | 购买基础设施 | 通用 |
| Plort | 史莱姆产出 | 制造材料/区域解锁/出售 | 按史莱姆价值分级 |
| 矿石(Ore) | 矿石节点(资源采集器) | 基础制造材料 | 常见 |
| 木材(Wood) | 木材节点(资源采集器) | 基础制造材料 | 常见 |
| 辉石矿(Radiant Ore) | 余烬谷/星光滨特定节点 | 高级升级(喷气背包等) | 稀有 |
| 粉砂(Pink Sand) | 余烬谷瀑布附近 | 灰色迷宫区域解锁 | 稀有 |

## 2. 资源采集器系统

玩家必须升级吸尘背包的**资源采集器(Resource Harvester)**才能采集矿石、木材等资源。

| 升级 | 成本 | 效果 | 解锁采集能力 |
|------|:----:|------|-------------|
| 基础资源采集器 | 10棉花Plort + 450 NB | 解锁资源采集 | 矿石、木材 |
| 中级采集器 | 20岩石Plort + 10辉石矿 | 采集效率+50% | 辉石矿、粉砂 |
| 高级采集器 | 30水晶Plort + 20辉石矿 | 采集效率+100% | 熔岩尘、野蜜 |
| 终极采集器 | 高额Plort+材料 | 采集效率+150% | 深盐水、冰晶 |

## 5. 科技树结构

SR2的科技树是非线性的——解锁顺序取决于玩家探索路径和优先选择。

### 5.1 科技树阶段划分

```
[阶段1: 基础生存]
  探索彩虹田野 -> 收集基础Plort -> 解锁资源采集器
  目标: 建立基础食物+Plort循环

[阶段2: 区域扩张]
  解锁余烬谷/星光滨 -> 获得新资源种类
  目标: 收集辉石矿+粉砂+野蜜 -> 升级喷气背包

[阶段3: 制造时代]
  解锁制造台 -> 开始制作蓝图装置
  目标: 制作传送器+链接炮 -> 提高探索效率

[阶段4: 自动化时代]
  收集奇异钻石 -> 制作量子无人机
  目标: 实现全自动Plort生产 -> 释放时间给探索

[阶段5: 终局]
  收集大量材料+Plort -> 制作4个辉光投影器
  目标: 进入灰色迷宫 -> 完成主线 -> 无限经营
```

## 7. 设计特点分析

1. **"Plort+材料"二元资源 = SR2的核心变革**: SR1的经济是单向的(Plort->新币->升级)，而SR2的制造系统使得Plort同时具有"货币"和"材料"双重属性。这带来了一个关键变化——玩家不能把所有Plort都卖掉，必须保留一部分用于制造。这个"自用vs出售"的决策增加了经济策略的深度。

2. **制造台替代7Zee公司的设计意义**: SR1的7Zee奖励系统本质上是"出售Plort->获得等级->被动奖励"的线性系统。SR2的制造台要求玩家主动选择"制造什么"和"保留哪些材料"——从被动接收奖励变为主动选择发展方向。

3. **奇异钻石=探索驱动力**: 作为游戏中最稀有的资源，奇异钻石不仅是无人机的必需材料，还迫使玩家探索全岛的每一个角落。这个设计类似于《旷野之息》的"呀哈哈"——通过稀缺资源驱动探索行为。

4. **资源检测器的"后期便利"设计**: 1.2版本加入的资源检测器解决了中后期"我知道我需要XX资源但不知道去哪找"的问题。这是典型的"降低中后期管理负担"设计——游戏不希望你花太多时间在"找资源"上，而希望你专注于高价值决策。

5. **月露花蜜=时间窗口驱动**: 月露花蜜是游戏中唯一一个必须夜间特定区域采集的食物。这个设计创造了"夜间出行"的独特游戏节奏——玩家需要调整作息表来配合月露花蜜的产出周期，否则振翅史莱姆的养殖效率会受到严重影响。
