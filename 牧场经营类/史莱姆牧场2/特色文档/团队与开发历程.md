## 1. 开发团队——Monomi Park

### 创始故事：从"课后项目"到百万销量

**Monomi Park** 由 **Nick Popovich** 和 **Mike Thomas** 于2014年共同创立。两人此前都曾在 **Three Rings Design**（一家位于旧金山的游戏工作室，后被SEGA收购）工作，参与开发了MMO游戏《Spiral Knights》。

| 角色 | 人员 | 说明 |
|------|------|------|
| **联合创始人/CEO/系列总监** | Nick Popovich | 前Three Rings设计师，自封"首席史莱姆官" |
| **联合创始人/技术总监** | Mike Thomas | 前Three Rings工程师 |
| **团队规模 (SR1时期)** | 2人创始 → 约12-15人 |
| **团队规模 (SR2时期)** | **约20-30人** |

### Nick Popovich 的传奇起步

Popovich 在2014年离开Three Rings后，花了两周时间学习Unity，然后制作了《史莱姆牧场》的首个原型。他在采访中回忆：

> "我完全没有正式写过程序。我只是有想法，有动力，有对任天堂式游戏体验的热爱。实际上，如果有人告诉我做游戏有多难，我可能根本不会开始。"

这个两周的原型最终演变成了一个销量超过600万份的全球热门游戏。

### 公司文化

Monomi Park以"健康工作"文化闻名：

- **朝九晚五，不加班**：严格遵守标准工作时间
- **无加班文化**：即使在EA发售后也保持正常工作节奏
- **远程友好**：部分团队成员远程工作
- **动物友好办公室**：办公室允许带宠物

Popovich明确表示：

> "游戏行业有一种把加班当作荣誉徽章的文化。我们拒绝这样做。好的游戏来自健康的团队，而不是疲惫的团队。"

## 2. 开发历程——从续作焦虑到成功

### SR1的巨大成功

《史莱姆牧场》(2017)是Monomi Park的首部作品，取得了超出预期的成功：

| 指标 | 数据 |
|------|------|
| **总销量** | **超过600万份** |
| **Steam好评率** | 97%（好评如潮） |
| **Xbox Game Pass** | 首发加入，覆盖亿万玩家 |
| **媒体评价** | 被评为"像任天堂做的PC游戏" |

### SR2的立项：从DLC到独立续作

SR2最初被构思为SR1的大型DLC，但团队很快意识到要做的东西太多了：

> "我们想把捕猎系统做得更好，把地图做得更大，把史莱姆种类加得更多……然后我们发现，这不是DLC能装下的。它需要成为续作。"

### 技术升级——HDRP

SR2相比SR1最重要的技术升级是 **从Unity标准渲染管线迁移到HDRP（High Definition Render Pipeline）**：
- 实时光照和阴影
- 更高质量的材质和反射
- 体积光效果
- 更高质量的粒子系统

这使得SR2的画面表现力远超SR1，但同时也意味着更长的加载时间和较高的硬件要求。

### 开发时间线

| 时间 | 里程碑 |
|------|--------|
| 2014年 | Monomi Park 成立，SR1原型完成 |
| 2017年8月 | SR1 1.0 正式发售 |
| 2018-2021年 | SR2 概念立项，早期开发 |
| 2022年6月 | SR2 正式公布 |
| **2022年9月22日** | **SR2 Early Access 发售** |
| 2022年9月 | 首6小时售出10万份，首周超40万份 |
| 2022-2025年 | EA期间持续3年，约8次大型内容更新 |
| **2025年9月23日** | **1.0 正式版 + PS5同步发售** |
| 2025年 | 电影改编版宣布开发中 |

### 奇迹般的首发数据

| 时间窗口 | 销量 |
|---------|------|
| **首6小时** | 100,000份 |
| **首24小时** | 超过Popovich的"白日梦"目标 |
| **首周** | 400,000+份 |
| **对比** | SR2首周销量 = SR1前9个月销量总和 |

Popovich在谈到这个数字时表示：

> "我们设定了一个'白日梦'目标——首24小时卖出10万份。结果首6小时就达到了。这完全超出了我们最疯狂的预期。"

## 3. Core Design Evolution

### SR1 → SR2 的进化

| 维度 | SR1 | SR2 |
|------|-----|-----|
| **地图** | 单一牧场区 + 2个野外区域 | 多生物群系开放世界 |
| **史莱姆** | 16种基础 + 变异 | 新增约10种新史莱姆 |
| **图形** | Unity标准渲染 | **Unity HDRP** |
| **自动化** | 基础管道系统 | 量子无人机等高级自动化 |
| **天气** | 无 | 完整天气系统（含史莱姆雨） |
| **剧情** | 简单的日志叙事 | 更深层的灰迷宫探索 |

### 1.0 正式版的重大更新

2025年9月的1.0正式版带来了：
- **天气系统**：包括独特的"史莱姆雨"事件
- **量子无人机**：高级自动化系统
- **灰迷宫 (Grey Labyrinth)**：终局内容的封闭区域，包含剧情高潮
- **扩展的无障碍选项**
- **PS5版本首发**

## 4. 市场表现与评价

### 商业成功

| 数据 | SR1 | SR2 |
|------|-----|-----|
| 总销量 | 600万+ | EA首周40万+ |
| Steam好评率 | 97% | 93%（EA期间） |
| 开发模式 | 2年完整开发 | 3年EA |

### 评价要点

- **画面升级**：HDRP带来的视觉提升广受好评
- **探索乐趣**：新的彩虹岛（Rainbow Island）提供了丰富的探索内容
- **EA策略**：被赞美为"如何做EA的教科书级示范"——永不清理玩家存档，每次更新都像礼物
- **家庭友好**：延续了SR1的全年龄向设计

### "任天堂式的PC游戏"

Nick Popovich 曾将《史莱姆牧场》系列描述为 **"like a Nintendo game on PC"**（像任天堂游戏一样的PC游戏）。这个评价体现在：
- 色彩丰富可爱的美术风格
- 没有暴力/死亡元素
- 上手容易但追求完美的深度设计
- 家庭友好的全年龄定位
