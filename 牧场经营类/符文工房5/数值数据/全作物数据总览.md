## 四、作物品质等级(Crop Level)机制

| 等级 | 售价倍率 | 获取方式 |
|:---:|:-------:|---------|
| Lv.1 | 1.0x | 基础种植 |
| Lv.2 | 1.1x | 使用镰刀收获高等级作物→升级种子 |
| Lv.3 | 1.2x | 继续升级循环 |
| Lv.4 | 1.3x | 升级循环 |
| Lv.5 | 1.5x | 升级循环 |
| Lv.10 | 2.0x | 持续升级 |

### 升级方式
1. **镰刀升级循环**: 用镰刀收获作物→获得高等级种子→播种→再次用镰刀收获
2. **肥料(Fertilizer)**: 使用化学台制作的肥料可提升作物品质
3. **耕种技能等级**: 技能越高→收获高等级作物的概率越高

> **关键机制**: 只有用镰刀收割才能获得升级种子, 手动收获只能获得基础Lv.1。这意味着"收割-升级种子循环"是最核心的作物利润提升手段。一个Lv.10的作物售价是基础售价的2倍, 足以将大部分亏损作物转为盈利。

## 五、巨型作物(Giant Crops)

极小概率种植出巨型作物, 用镰刀收获后可获得5-10个普通作物+巨型种子。

| 属性 | 说明 |
|------|------|
| 出现概率 | 约1%-3%, 受肥料和耕种技能影响 |
| 收获产物 | 5-10个对应作物 + 巨型种子(Giant Seed) |
| 巨型种子 | 可再次种出巨型作物 |
| 利润影响 | 一次巨型收获相当于5-10倍的作物产量 |

## 六、季节性种植策略总结

### 春季
- **快速启动**: 芜菁(3天/10G利润) → 积累资金
- **中期主力**: 韭葱(9天/920G, 102G/天) = **春季最佳单次作物**
- **复收选择**: 草莓(16天准备, 之后每3天收135G)
- **注意避开**: 卷心菜(基础亏损300G)、白菜(基础亏损580G) — 必须升级到Lv.5以上才有利润

### 夏季
- **最佳单次**: 玉米(7天/550G, 79G/天)
- **最佳复收**: 粉红甜瓜(7天准备, 之后每3天收850G, 长期利润最高)
- **避坑**: 洋葱(基础亏损680G)、热辣果(基础亏损300G)
- **备注**: 菠萝虽然单次利润高但28天生长期太长, 复收周期5天也较长

### 秋季
- **最佳单次**: 萝卜(5天/240G, 48G/天)
- **速生**: 菠菜(2天/5G利润, 适合土地空闲期)
- **复收选择**: 茄子(7天准备, 每3天收190G)
- **避坑**: 大部分秋季作物需升级才有利润

### 冬季
- **唯一利润源**: 铁叶(21天/511G, 24G/天)
- **最佳策略**: 专注畜牧产品加工 + 钓鱼 + 矿洞探索
- **四季土壤**: 解锁后可在冬季种植任何季节作物

## 七、跨游戏对比

| 维度 | RF5 | RF4 | 星露谷物语 |
|------|:---:|:---:|:--------:|
| 每季天数 | 30 | 30 | 28 |
| 作物品质等级 | Lv.1-10(售价比例提升) | Lv.1-10 | 无品质(只有铱星) |
| 复收作物占比 | ~30% | ~30% | ~25% |
| 基础作物亏损比例 | **高(~40%)** | 类似 | 几乎无亏损 |
| 种子升级方式 | 镰刀收割循环 | 镰刀收割循环 | 无(直接买高级种子) |
| 星陨作物 | 有(免费高利润) | 无 | 无 |
| 金色作物 | 有(高端) | 有 | 无 |

> **设计观察**: RF5延续了RF4的"基础作物部分亏损→依赖升级和加工"的设计哲学, 这与星露谷"播种即有正收益"的温和发展路线形成鲜明对比。RF系列将种植视为"需要技能投入的加工原料产业", 而非纯粹的现金流来源。
