## 一、工具五级升级体系

RF5农具和工具均采用五级升级路线: 廉价(Cheap) → 铜(Bronze) → 银(Silver) → 金(Gold) → 白金(Platinum/Legendary)。

### 工具升级表

| 等级 | 所需锻造Lv | 核心材料 | 售价 | 效果提升 |
|:---:|:---------:|---------|:---:|---------|
| 廉价(Cheap) | — | 铁/基础矿石 | 基础 | 基础使用 |
| 铜(Bronze) | 5+ | 铜矿 x5 | ~2,000G | 效率+范围↑ |
| 银(Silver) | 10+ | 银矿 x8 | ~5,000G | 效率+范围↑↑ |
| 金(Gold) | 20+ | 金矿 x10 | ~15,000G | 效率+范围↑↑↑ |
| 白金(Platinum) | 35+ | 白金矿 x15 | ~40,000G | 最高效率+最大范围 |

### 各工具升级效果

| 工具 | 廉价→铜→银→金→白金效果 |
|------|----------------------|
| 锄头(Hoe) | 翻耕格数: 1→1→3→5→9(范围递增) |
| 水壶(Watering Can) | 浇水格数: 1→3→5→7→9(范围递增) |
| 斧头(Axe) | 砍树次数: 5→4→3→2→1(一次砍倒) |
| 锤子(Hammer) | 碎石次数: 4→3→3→2→1(一次敲碎) |
| 镰刀(Sickle) | 收割范围: 1→3→3→5→7(范围递增) |
| 钓竿(Fishing Rod) | 可钓鱼种品质提升, 稀有鱼概率↑ |

> **升级优先级建议**: 水壶(影响每日效率) > 镰刀(影响作物升级循环) > 锄头 > 斧头/锤子 > 钓竿

### 工具购买/制作条件

1. 完成Eliza的"购买锻炉"委托(需200 SEED点)
2. 从Palmo处购买锻炉(3,000G + 25石材 + 30木材)
3. 学习锻造配方(吃食谱面包或在锻炉试制)
4. 升级需要对应材料+RP消耗

| 规则 | 说明 |
|------|------|
| 第1次同材料 | 100%效果 |
| 第2次同材料 | 50%效果 |
| 第3次同材料 | 25%效果 |
| 最佳策略 | 每次使用不同材料→获得全属性加成 |

## 六、设计特点分析

1. **配方面包系统是核心**: 不是等级到了自动学会——必须吃面包随机解锁, 增加了反复尝试的RNG要素
2. **材料品质决定武器质量**: 使用高等级材料锻造的武器基础属性更高, 且材料附加效果也是重要考量
3. **升级强化有边际递减**: 同种材料叠加效果下降, 鼓励多样化使用材料
4. **RP消耗与技能等级挂钩**: 锻造超过技能等级的配方会消耗大量RP, 失败率提高
5. **与RF4基本一致的制作体系**: 但3D化后制作界面更友好, 材料分类更清晰
6. **饰品优先级高于武器**: 心之吊坠(技能EXPx2)是所有玩家第一个该制作的饰品, 之后再考虑武器
