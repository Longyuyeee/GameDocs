## 1. 开发团队 — Pathea Games

### 1.1 公司背景

| 信息 | 内容 |
|:----:|------|
| **公司名称** | 帕斯亚科技 (Pathea Games) |
| **成立时间** | 2010年 |
| **总部** | 中国重庆市 |
| **创始人** | 吴志强 (Ziqiang Wu) 等 |
| **团队规模** | 约 15-20 人 (立项时) → 约 80 人 (2025年) |
| **代表作品** | 波西亚时光、沙石镇时光 |
| **企业定位** | 专注于模拟经营/RPG 融合类型的独立游戏开发商 |

Pathea Games 位于中国重庆，最初是一家小型外包工作室，参与过《天涯明月刀》等中国网游的美术外包工作。大约在 2015-2016 年，公司决定转型为原创游戏开发商，波西亚时光是他们的首款自研原创作品。

### 1.2 核心团队 (波西亚时光时期)

| 成员 | 角色 | 说明 |
|:----:|:----:|------|
| **吴志强 (Ziqiang Wu)** | CEO / 制作人 | 公司创始人，负责项目整体方向 |
| **邓永进 (Deng Yongjin)** | 主策划 | 核心玩法设计、系统架构 |
| **技术团队** | 主程 | Unity 引擎开发，团队早期规模极小 |
| **美术团队** | 3D 美术/场景设计 | 全 3D 场景 + Low Poly 风格 |

### 1.3 Pathea 的"中国独立游戏出海"模板

Pathea 是中国最早一批成功面向全球市场（尤其是 Steam 欧美用户）的开发商之一。他们的模式被后续的中国独立游戏开发者广泛研究：
- **Kickstarter 众筹获取初始用户**: 2017 年在 Kickstarter 上筹集超过 14 万美元，建立初始社区
- **Steam Early Access 验证玩法**: 2018 年 1 月进入 Steam EA，通过社区反馈迭代核心系统
- **本地化优先于国区**: 游戏先以英文版面向全球市场，中文版在后续更新中加入
- **跨平台同步开发**: 从立项起就考虑主机和移动端移植，而非后期适配

## 2. 开发历程

### 2.1 立项背景

波西亚时光的立项恰逢《星露谷物语》(2016) 全球成功后掀起的"牧场经营游戏热潮"。Pathea 团队观察到：
- 市场上的牧场经营游戏几乎全部为 2D 像素风格（星露谷、牧场物语、开花吧小橘子等）
- **3D 牧场经营游戏**是明显的市场空白——没有一款游戏提供"3D 开放世界 + 牧场经营 + RPG 战斗"的组合
- 团队自身具备 UE4 和 Unity 美术外包经验，3D 开发能力是他们的比较优势

于是 Pathea 决定做一款"有史以来最完整的 3D 牧场经营游戏"。

### 2.2 时间线

| 阶段 | 时间 | 关键事件 |
|:----:|:----:|---------|
| 公司成立 | 2010年 | 重庆成立，早期从事游戏美术外包 |
| 转型自研 | ~2015年 | 决定从外包转型为原创游戏开发商 |
| 波西亚立项 | 2016年 | 项目正式启动，当时团队约 15 人 |
| **Kickstarter 众筹** | **2017年8月** | 目标 $10 万 → 最终 **$14 万** (140% 达成) |
| Steam EA 上线 | 2018年1月 | Early Access 抢先体验，获得全球关注 |
| 正式版 1.0 | **2019年6月** | PC 正式版发布，约 3 年开发周期 |
| Switch 版 | 2019年9月 | 首批主机版，但初期优化问题严重 |
| PS4/Xbox 版 | 2019年12月 | 全平台覆盖 |
| 完整版 3.0 | 2021年 | 最终内容更新，包含所有 DLC 和扩展内容 |
| 移动版 | 2022年 | iOS/Android 版本发布 |
| 续作开发 | 2019-2023 | 同期启动《沙石镇时光》开发 |

**总开发周期**: 2016-2021 (约 5 年持续开发)

### 2.3 Kickstarter 众筹详情

| 维度 | 数据 |
|:----:|:----:|
| 众筹开始 | 2017年8月 |
| 目标金额 | $100,000 |
| 实际募集 | $141,743 (142%) |
| 支持人数 | 2,618 人 |
| 最高档位 | $1,500 (将赞助人做成游戏内 NPC) |
| 退还款项 | 部分主机版承诺因开发延期导致退款 |

Kickstarter 众筹对于波西亚时光的意义远超资金层面——它验证了"3D 牧场经营游戏"在全球范围内的市场需求，并建立了第一批核心社区成员。这些早期支持者在 EA 阶段的测试反馈对游戏系统迭代起到了至关重要的作用。

### 2.4 开发中的挑战

| 挑战 | 详细说明 |
|:----:|---------|
| **团队规模过小** | 早期约 15 人需要同时完成 3D 建模、程序、策划、QA 的全部工作 |
| **技术债务积累** | EA 阶段的快速迭代导致大量代码需要重构，正式版 1.0 发布后仍有大量 bug |
| **主机优化困难** | Switch 版的加载时间和帧率问题让开发团队花了数月进行性能调优 |
| **文化差异** | 面向全球市场时，游戏内对话和剧情的文化适配是一个持续的挑战 |
| **续作分流** | 2019 年后团队部分资源转向沙石镇时光的开发，波西亚的后期更新速度变慢 |

## 3. 技术架构

| 类别 | 技术选型 | 说明 |
|:----:|:--------:|------|
| **引擎** | Unity 3D | 全 3D 场景，Low Poly 美术风格 |
| **美术风格** | Low Poly 3D | 色彩明亮，柔和光照，性能友好的多边形数量 |
| **光照系统** | Unity 内置光照 + 烘焙光贴图 | 实现昼夜循环和季节变化 |
| **物理引擎** | Unity Physics | 用于角色运动、碰撞检测、物品物理 |
| **音频** | FMOD | 自适应音乐系统，根据场景和活动切换 |

### 4.3 对中国独立游戏行业的意义

波西亚时光是第一款真正意义上"走向全球"的中国独立游戏之一，它的成功对中国独立游戏行业产生了深远影响：
1. **证明了 3D 牧场经营类可以独立于星露谷的 2D 像素范式存在**
2. **验证了 Kickstarter + Steam EA 的中国出海模式**
3. **重庆团队做全球市场"不再是梦"** — Pathea 作为非北上广深的团队做出了全球爆款

## 5. 设计启示

1. **填补 3D 牧场经营的市场空白** — 在 2016 年星露谷成功后的市场中，Pathea 敏锐地发现了"3D 开放世界 + 牧场经营"的结合点，这是波西亚时光成功的核心定位
2. **工业制造 = 牧场经营的新方向** — 波西亚时光的"组装工坊"系统（将采集材料加工为订单产品）是牧场经营品类中的一个重要创新，后被沙石镇时光继承和扩展
3. **"后星露谷时代"的差异化竞争** — 波西亚时光证明：后发者不一定需要在核心系统上超越星露谷，可以通过"走不同的路"（3D 化、工业化、战斗深度化）找到自己的生态位
4. **中国开发者的全球化能力** — Pathea 从一开始就面向全球市场设计，从语言本地化到文化符号的选择都体现了高度的国际化意识
5. **Kickstarter 不只是筹款** — 它是一个"社区验证"机制，波西亚时光的 2,618 名支持者构成了游戏的初始社群，这些人在 EA 阶段提供了关键的测试反馈
