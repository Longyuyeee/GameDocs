# 经济系统 - 沙石镇时光(My Time at Sandrock)

> 游戏版本: My Time at Sandrock v1.0+ (PC/Switch/PS5/Xbox, 2023年正式版)
> 数据来源: mytimeatsandrock.fandom.com, Siliconera, Steam Community, GameRant

## 1. 货币体系

| 货币 | 获取途径 | 主要用途 |
|------|---------|------|
| 戈尔(Gols) | 完成委托/出售物品/任务奖励 | 购买材料/种子/装备/升级/婚礼 |
| 数据碟(Data Discs) | 采矿/废墟探索 | 研究中心(Qi)研究新蓝图 |
| 废墟代币(Ruins Tokens) | 危险废墟评分奖励 | 兑换稀有装备/材料 |
| 公会徽章(Commerce Guild Badges) | 委托完成/排名奖励 | 公会商店兑换 |

## 2. 收入来源

| 来源 | 前期 | 中后期 | 说明 |
|------|:--:|:---:|------|
| 商业公会委托(Commissions) | ★★★★★ | ★★★★★ | **核心收入引擎**，每日接取，100-2,000+戈尔/件 |
| 钓鱼(沙钓) | ★★ | ★★★★ | 沙钓之王(King Sandacuda)=3,125戈尔/条 |
| 成品出售(加工品) | ★★★ | ★★★★ | 原料加工后再出售(如6石=3戈尔→石槽=6戈尔) |
| 主线/支线任务 | ★★★★ | ★★★ | 一次性但金额可观 |
| 废墟采矿 | ★★★ | ★★★ | 铜/铁/铝矿石稳定收入 |
| 农作物出售 | ★ | ★★ | 非核心收入(种植定位为辅助) |
| 广告收入(Advertising) | ★ | ★★ | 知识技能解锁后被动收入 |

## 3. 委托(Commission)系统

| 要素 | 说明 |
|------|------|
| 接取地点 | 商业公会布告栏(Commerce Guild Board) |
| 刷新频率 | 每日刷新 |
| 委托价格 | 100-2,000+戈尔/件，随难度和稀有度浮动 |
| 同时接取数 | 初始1-2个，排名第1可接3个 |
| 城市委托 | 每周特别高额委托 |
| 同日交付 | "快速交付(Quick Delivery)"知识技能+5-10%戈尔 |
| 高品质加成 | "品质奖励(Quality Bonus)"技能+10%戈尔 |
| 取消惩罚 | 降低声望，比跳过不接更严重 |

## 4. 工坊排名(Workshop Ranking)

| 排名 | 奖励/解锁 |
|:----:|------|
| 第1名 | 最多同时接取3个委托、额外公会徽章 |
| 第2-3名 | "榜首领航员(Chart-topper)"技能+5-10%声望/经验 |
| 低排名 | 委托槽位少、收入低 |

- 排名每月根据完成委托数量+品质综合评定
- 增加排名手段: "广告(Advertising)"技能+委托完成量

## 5. 水资源经济

水是**所有机器运转的燃料**。缺水→机器停工→无法生产→无法完成委托→无收入。

| 获取方式 | 效率 | 成本 |
|---------|:--:|------|
| 水世界(Water World)购买 | 快速 | 每天少量低价水→之后涨价 |
| 露水收集器(Dew Collectors) | 持续 | 建造后免费产露水→10露水=1水 |
| 宠物派遣 | 自动 | 中期后"几乎无限水" |
| 采集圆球仙人掌(Ball Cactus) | 手动 | 免费但耗时 |

## 7. 设计特点

1. **委托驱动替代传统耕种**: 不是"种地卖钱"而是"完成委托赚钱"——工坊是核心，农耕是辅助
2. **水=能源而非体力**: 水是机器的燃料而非角色的体力(Sandrock仍有体力系统但水更稀缺)
3. **工坊排名竞争**: 引入了RPG式的排名机制，创造委托完成的紧迫感
4. **多货币体系**: 戈尔/数据碟/废墟代币/公会徽章——每种货币有独立获取和使用渠道
5. **加工增值明确**: 原料(6石=3戈尔)→加工品(石槽=6戈尔)→2倍增值
6. **知识技能深度影响经济**: Quick Delivery/Quality Bonus/Advertising等技能直接决定收入效率
