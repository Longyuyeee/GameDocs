# 废墟探索与战斗系统 - 沙石镇时光(My Time at Sandrock)

> 游戏版本: My Time at Sandrock v1.0+ (PC/Switch/PS5/Xbox)
> 数据来源: mytimeatsandrock.fandom.com, TechRaptor, TheGamer, Neoseeker

## 1. 概述

沙石镇时光的探索系统围绕**废墟(Ruins)**展开。废墟分为三类: 废弃废墟(采矿)、危险废墟(战斗地牢)、任务废墟(剧情限定)。战斗为**即时动作战斗**，使用多种武器类型。

## 2. 废墟类型

### 2.1 废弃废墟(Abandoned Ruins)

| 属性 | 详情 |
|------|------|
| 敌人 | **无**(纯采矿/遗物收集) |
| 机制 | 使用镐锤(Pickhammer)挖掘体素地形 |
| 探测器 | 宝物探测器(Treasure Detector)定位遗物和出口 |
| 楼层 | 每废墟**11层** |
| 体力消耗 | 消耗体力 |
| 时间流逝 | 正常流逝 |
| 通行费 | 200-500戈尔/周 |

### 2.2 危险废墟(Hazardous Ruins)

| 属性 | 详情 |
|------|------|
| 敌人 | **大量**(含精英和Boss) |
| **体力** | **冻结**——无限战斗/采矿 |
| **时间** | **冻结**——无需赶在睡觉前 |
| 评分 | 按通关时间/击杀数/治疗使用评级(S/A/B/C) |
| 奖励 | 废墟代币+稀有材料+装备 |
| HP=0 | 游戏结束(需带足治疗物品) |

### 2.3 任务废墟(Mission Ruins)
- 主线限定场景
- 完成后通常变为废弃/危险废墟

## 5. 战斗系统

### 5.1 武器类型

| 类型 | 特点 | 技能树 |
|------|------|:--:|
| 剑与盾(Sword & Shield) | 平衡攻防，+10%防御 | 有 |
| 重剑(Great Sword) | 高伤害慢速，+10%暴击伤害 | 有 |
| 长矛(Spear) | 中距快速，+5%攻击 | 有 |
| 匕首(Dagger) | 近身快攻，+5%暴击率 | 有 |
| 枪械(Firearm) | 远程，消耗弹药 | 有 |
| 投掷武器(Throwing) | 消耗品远程 | 有 |

### 5.2 战斗机制

| 机制 | 说明 |
|------|------|
| 体力(Stamina) | 非战斗消耗；危险废墟中冻结 |
| HP | 减至0=游戏结束(危险废墟)/晕倒(野外) |
| 韧性(Toughness) | 防御/破防机制，受Combat技能树影响 |
| 翻滚(Dodge Roll) | 消耗耐力，避伤核心动作 |
| BREAK | 破防值——强力攻击破防后造成额外伤害 |

### 5.3 关键战斗知识技能

| 技能 | 效果 |
|------|------|
| 生存知识(Survival) | 最大HP+10/20% |
| 坚韧(Toughness) | 韧性恢复加速50% |
| 强力攻击(Strong Attack) | 暴击率+5%, BREAK+1 |
| 耐力(Endurance) | 翻滚耐力消耗-5, 攻击50%概率恢复5耐力 |
| 各武器精通(Novice) | 解锁4-5连击combo |

## 7. 设计特点

1. **体素采矿**: 废弃废墟的体素地形挖掘——非传统"敲节点"，更互动的手动挖掘体验
2. **危险废墟时间冻结**: "Boss房间"外的时间/体力冻结→鼓励从容战斗而非赶时间
3. **评分系统**: 通关时间/击杀/治疗使用→追求S级→废墟代币效率最大化
4. **6种武器×4连击Combo**: 每种武器类型有独立的技能树和连击系统
5. **无永久死亡**: 危险废墟死亡→重新开始楼层；野外晕倒→失去少量戈尔
6. **Scrooge McMole**: 废弃废墟稀有随机生物——类似星露谷的"蘑菇层"设计
