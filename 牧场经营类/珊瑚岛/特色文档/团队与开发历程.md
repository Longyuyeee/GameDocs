## 一、开发团队：Stairway Games

### 1.1 公司概况

| 项目 | 数据 |
|:----|:------|
| 开发商 | Stairway Games |
| 总部 | 印度尼西亚，日惹(Yogyakarta) |
| 成立时间 | 2019 年 5 月 |
| 初创团队规模 | 约 15 人 |
| 当前团队规模 | 51-100 人(2024 年数据) |
| 发行商 | Humble Games (PC/主机平台发行) |
| 代表作品 | Coral Island (唯一作品至今) |
| 引擎 | Unreal Engine 4 → 计划升级 UE5 |

### 1.2 核心团队成员

| 姓名 | 职位 | 背景 |
|:----|:----|:------|
| **Jeremiah Bramble** | 联合创始人 / 总监 / 主关卡设计 | 游戏设计出身，负责整体方向与关卡架构 |
| **Soma Wisnu Putera** | 联合创始人 / 游戏制作人 | 项目管理+商务运营，负责团队协调 |
| **Frisca Ayuni** | 创意总监 / 主编剧 | 负责世界观设定、角色剧情和对话编写 |
| **David Ardinaryas Lojaya** | 美术总监 | 负责整体视觉风格、3D 建模指导 |
| **Yugo Pratomo** | 主作曲 / 音效总监 | 负责原声配乐——融合东南亚传统乐器+现代管弦乐 |

### 1.3 团队背景故事

Stairway Games 的创始故事具有典型的东南亚初创色彩：

- **2019 年**，两位创始人 Jeremiah Bramble 和 Soma Wisnu Putera 在雅加达的一个**不相关的项目中**相识。聊起"做游戏"的共同梦想后一拍即合。
- 两人搬到日惹(Yogyakarta)——印尼的游戏开发新兴城市(类似中国成都的定位，生活成本低+高校资源多+游戏开发氛围浓)。
- 最初通过 **Artstation 招聘**，吸引了一批有热情但缺乏 3A 经验的年轻开发者。
- 前 2 年靠**独立出资**维持开发，团队 15-20 人，工资低于市场水平——但团队对项目充满信念。

## 二、开发历程

### 2.1 完整时间线

| 时间 | 事件 | 里程碑意义 |
|:----|:-----|:----------|
| 2019 年 5 月 | Stairway Games 公司成立 | 创始人在日惹租房创业 |
| 2019-2020 | 预开发阶段(约 15-20 人) | 原型制作，确定 UE4 引擎 |
| 2021 年 2-3 月 | **Kickstarter 众筹** | **目标 7 万美元 → 实际 164 万美元 (超额 23 倍)** |
| 2021 年夏 | 获得 Epic MegaGrant | Epic 资助用于招聘扩大团队 |
| 2021-2022 | 全面开发阶段(30-50 人) | 内容填充、系统完善 |
| 2022 年 10 月 | **Steam Early Access 发布** | 首个公开可玩版本 |
| 2023 年 2 月 | EA 大更新(海底区域) | 海底探索系统上线 |
| 2023 年 7 月 | EA 大更新(结婚+孩子系统) | 社交系统完整上线 |
| **2023 年 11 月** | **v1.0 正式版发布(PC/PS5/Xbox)** | 全平台发布，标价 $29.99 |
| 2024 年 1 月 | v1.1 更新(性能优化+新内容) | 修复大量 EA 遗留 Bug |
| 2024 年 7 月 | **Humble Games 发行团队重组** | 发行商裁员重组→主机补丁和 Switch 版延迟 |
| 2025 年 8 月 | v1.2 **多人模式更新** | 本地+在线合作(最多 4 人) |
| 2026 年(预计) | Switch 版 | 多次延期，社区等待中 |

### 2.2 Kickstarter 传奇

Coral Island 的 Kickstarter 是农场模拟品类众筹史上的里程碑：

| 指标 | 数据 | 比较 |
|:----|:-----|:-----|
| 目标金额 | $70,000 | 极低目标(团队保守估计) |
| 实际募集 | **$1,639,368** | 超目标 **2,342%** 即 23 倍 |
| 支持者人数 | **36,374 人** | 平均每人约 $45 |
| 排名 | 2021 年游戏类 Kickstarter 第 3 名 | 仅次于一些知名 IP |
| 用时 | 35 天 | 第 1 天就达到目标($70K→$100K+) |

**成功因素分析**:
1. **"星露谷 3D 版"的定位**极其精准——Stardew Valley 长期占据 Steam 榜单前列但无官方 3D 版
2. **东南亚热带岛屿设定**填补了市场空白(同类游戏多以温带/欧洲田园为背景)
3. **Unreal Engine 4 的 3D 画面**在农场模拟品类中显得"高级"
4. **$7 万目标**看起来可信且易于达成——低目标降低了支持者的门槛感

### 2.3 Early Access 阶段

2022 年 10 月进入 EA 时，游戏约完成 **60%** 的内容：

| EA 阶段 | 已包含 | 缺失 |
|:--------|:-------|:-----|
| 核心玩法 | 种植/畜牧/钓鱼/挖矿 | 完整剧情线 |
| 矿洞 | 2 个矿洞(地/水) | 风矿洞/火矿洞 |
| 社交 | 基础 NPC 对话 | 结婚/孩子/心事件 |
| 海底 | 基础潜水 | 海底动物/完整人鱼线 |
| 博物馆 | 可捐赠 | 全收集奖励未实装 |
| 多人 | — | EA 全程单人 |

EA 期间积累了 **约 10 万玩家**，Steam 评价维持在 **88-92% 好评率**。

### 2.4 正式版发布 (v1.0)

**2023 年 11 月 14 日**正式发布，登陆 PC(GOG/Steam)、PS5、Xbox Series X|S。

| 版本亮点 | 内容 |
|:---------|:------|
| 4 矿洞全开 | 地/水/风/火 各 40F |
| 28 位候补全实装 | 含 3 位人鱼 |
| 完整博物馆 | 约 235 件收藏品 |
| 海底全区域 | 0m→深渊 |
| 剧情线 | 主线 + 角色个人线 |
| 城镇等级 S 级 | 终极目标可达成 |
| 性能优化 | EA 帧率问题大幅改善(但仍不如 2D 同类稳定) |

### 2.5 发行商重组与 Switch 版延期

2024 年 7 月，**Humble Games 进行了大规模重组**，解雇了大部分发行团队成员。这对 Coral Island 产生了连锁影响：

- **主机(PS5/Xbox)补丁发布减缓** —— 部分修复程序推迟数月
- **Switch 版延期** —— 从 2024 年推迟至 2025 年，再推迟至 2026 年
- **社区不满** —— Kickstarter 支持者中 Switch 版承诺的延迟引发了大量负面反馈
- **Stairway Games 接手发行** —— 团队转而自己处理部分发行工作，减少对 Humble 的依赖

这一事件暴露了**独立游戏依赖外部发行商的风险**。

## 三、技术与美术

### 3.1 技术栈

| 类别 | 选用技术 | 备注 |
|:----|:--------|:------|
| 引擎 | **Unreal Engine 4** | 大多数农场模拟游戏使用 Unity(Coral Island 是少数使用 UE 的) |
| 渲染 | 3D 第三人称固定视角 | 可自由旋转视角 |
| 光照 | 动态昼夜循环 | 真实感光照系统 |
| 音频 | Wwise | 交互式音频系统 |
| 网络(多人) | Steamworks P2P | v1.2 加入 |

### 3.2 为什么选择 UE4？

| 原因 | 说明 |
|:----|:------|
| **3D 差异化** | 同类游戏(星露谷/牧场物语)均为 2D/2.5D，UE4 可做出视觉差异化 |
| **团队偏好** | 创始人团队熟悉 UE4 工作流 |
| **海底场景** | UE4 的水体渲染效果优于 Unity(当时)——对海底探索系统至关重要 |
| **长线扩展** | 计划升级 UE5 以获得更高效能和 Lumen/Nanite 技术 |

### 3.3 美术风格

- **低多边形(Low Poly)写实风格** —— 介于《动物森友会》的可爱和《过山车之星》的写实之间
- **饱和色彩** —— 热带岛屿的强烈日光+珊瑚礁的鲜艳配色
- **东南亚建筑元素** —— 建筑风格融入印尼传统建筑(Balinese/Javanese)元素
- **角色设计** —— 多元化种族特征(反映印尼的多元文化)

### 3.4 音乐

主作曲 **Yugo Pratomo** 为游戏创作了超过 **60 首**原声曲目，特色：

- 使用 gamelan(甘美兰——印尼传统打击乐器)作为基调
- 融合现代管弦乐和 ambient 电子乐
- 每个区域(城镇/海滩/森林/海底/矿洞)有独立主题
- 昼夜和季节变化影响音乐风格

## 五、与 Eric Barone / ConcernedApe 的模式对比

### 5.1 发展模式对比

| 维度 | **ConcernedApe (星露谷)** | **Stairway Games (珊瑚岛)** |
|:----|:-------------------------|:---------------------------|
| 团队规模 | **1 人**(4 年独立开发) | **15→100 人**(工作室团队) |
| 开发周期 | 4 年(2012-2016) | 4 年(2019-2023) |
| 众筹 | 无(自筹资金) | Kickstarter **$164 万** |
| 发行方式 | 自发行 → Chucklefish(早期)→ 收回自发行 | Humble Games(发行商) |
| 引擎 | 自研引擎 → 移植 XNA/MonoGame | **Unreal Engine 4**(商用引擎) |
| 初始平台 | PC only | PC + PS5 + Xbox (多平台首发) |
| 开发成本估计 | 极低(单人生活费) | **数百万美元**(团队薪资+发行商投资) |
| 更新模式 | 长期免费大更新(1.4/1.5/1.6) | DLC 模式(含付费内容) |
| 社区关系 | 作者亲自管理，极深绑定 | 团队管理+社区经理 |

### 5.2 核心差异总结

| 差异维度 | ConcernedApe 模式 | Stairway Games 模式 |
|:--------|:-----------------|:-------------------|
| **优势** | 一人决策效率高，预算低风险小，完全掌控 | 大团队可做更多内容、3D 画面、多平台、多人模式 |
| **劣势** | 内容更新慢(1 人)，无法做 3D，无多人 | 开发成本高，依赖投资，发行商风险 |
| **对品类的影响** | 证明了单人也能做爆款 | 证明了团队化运作可以提升品类的制作规格 |

## 六、设计启示

### 6.1 对游戏开发者的启示

1. **Kickstarter 的"低目标高达成"策略**
   - 设定 $7 万极低目标 → 一旦超募就是正面新闻 → 吸引更多支持者
   - 信任建立: 低目标=团队务实，超募=社区认可

2. **差异化定位的力量**
   - "Stardew Valley 3D 版 + 热带海岛 + 海底探索" —— 每一个定语都在创造品类增量
   - 东南亚团队做东南亚主题游戏——真实性成为卖点

3. **UE 引擎穿越品类壁垒**
   - 证明 UE 不仅适合射击/动作游戏，也能做农场模拟
   - 3D 农场模拟的视觉天花板被拉起

4. **发行商风险**
   - Humble Games 的裁员直接影响了主机版和 Switch 版的发布节奏
   - 独立团队需要建立"去发行商依赖"的应急预案

5. **东南亚游戏开发的潜力**
   - Stairway Games 证明了东南亚(印尼)也能做出全球认可的农场模拟游戏
   - 印尼的游戏开发生态在崛起(Yogyakarta 和 Bandung 已成为开发 hubs)

### 6.2 对农场模拟品类的启示

| 启示 | 具体内容 |
|:----|:---------|
| 3D 化是可行的 | 以前认为农场模拟只能用 2D/2.5D，Coral Island 证明了 3D 的市场 |
| 海底是未被开发的领域 | 海底探索+海底农业+人鱼社交——一个全新的内容维度 |
| 环保主题有市场 | 不同于传统的"田园怀旧"，环保重建的叙事获得了当代玩家共鸣 |
| 大团队≠更短开发 | 即使 100 人的团队，从原型到 1.0 也需要 4 年——农场模拟的内容需求极大 |
| 多人模式可以后加 | v1.0 出后 2 年才加入多人——但单人体验足够支撑前期的口碑 |
