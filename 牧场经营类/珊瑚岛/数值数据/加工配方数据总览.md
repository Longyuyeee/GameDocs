## 一、加工系统总览

Coral Island 的加工系统分为四大类：**工匠设备(Artisan Equipment)**、**熔炉与冶炼(Smelting)**、**回收与转化(Recycling)**、**厨房烹饪(Cooking)**。加工品是游戏中后期的主要收入来源，也是全收集和博物馆捐赠的重要环节。

| 类别 | 设备数 | 核心用途 | 收入占比(中后期) |
|:----|:-----:|:--------|:---------------:|
| 工匠设备 | 12+ | 农产品→高级制品(果酒/奶酪/蛋黄酱等) | 约 50-60% |
| 熔炉冶炼 | 5 | 矿石→金属锭→工具/装备升级材料 | 必需(非直接收入) |
| 回收转化 | 3 | 垃圾→可用材料 | 环保增值 |
| 厨房烹饪 | 30+ 配方 | 食材→料理(恢复/Buff/送礼) | 功能性为主 |

## 七、设计特点分析

| 特点 | 说明 |
|:----|:------|
| **加工品为核心经济** | 中后期 50%+ 收入来自工匠品(类似星露谷)，Keg 的果酒和 Aging Barrel 的陈酿体系提供了长期收益 |
| **烘干机倍率最高** | ×5+25 的倍率是同类最高(星露谷无烘干机)，但加工时间也更长——平衡设计 |
| **陈酿体系简化** | 星露谷只有酒可以陈酿，Coral Island 增加了奶酪陈酿——但只有银/金/铱三阶(星露谷也是三阶) |
| **3D 设备模型** | 所有加工设备都有独立的 3D 模型和动画效果，视觉反馈优于星露谷的 2D 精灵 |
| **燃料系统更简单** | 熔炉只需要煤炭(不像某些游戏需要多种燃料)，降低冶炼门槛 |
| **厨房体量大** | 30+ 配方，但多数配方收益低于直接卖工匠品——烹饪主要服务于战斗/日常 Buff 而非经济 |
| **回收系统环保主题** | 回收机将垃圾变废为宝，契合游戏的环保主题——而不仅是简单的资源转化 |
