# 节日活动数据总览 - 珊瑚岛(Coral Island)

> **版本**: Coral Island v1.0 (PC/Steam, Stairway Games, 2023)
> **数据来源**: [Coral Island Wiki - Festivals](https://coralisland.fandom.com/wiki/Festivals) | [IGN - Festival Guide](https://www.ign.com/wikis/coral-island) | [TheGamer - Coral Island Events](https://www.thegamer.com/coral-island-events-guide/)

## 一、节日总览

Coral Island 每年有 **16 个节日/活动**，均匀分布在四个季节中(每季 4 个)。此外还有**城镇等级晋升庆祝**和**海底净化里程碑**等特殊事件。

| 季节 | 节日数量 | 核心主题 |
|:----:|:--------:|:--------|
| 春(Spring) | 4 | 新生/花卉/寻宝 |
| 夏(Summer) | 4 | 海洋/清凉/欢乐 |
| 秋(Fall) | 4 | 丰收/恐怖/收获 |
| 冬(Winter) | 4 | 庆典/寒冷/新年 |

## 四、秋季节日 (Fall Festivals)

### 4.1 收获祭 (Harvest Festival) — 秋 10 日

> 这是 Coral Island **最重要的节日**，相当于星露谷的 Stardew Valley Fair。

| 项目 | 内容 |
|:----|:------|
| 时间 | 09:00 - 18:00 |
| 地点 | 城镇广场 |
| 类型 | 展示/评分类 |
| 活动内容 | 玩家展示当季最高品质的作物/动物产品，由 NPC 评审团评分 |

| 评分类别 | 可提交物品 | 最高分 |
|:--------|:----------|:------:|
| 作物(Vegetable) | 所有蔬菜类 | 100分/个 |
| 水果(Fruit) | 所有水果类(含树木水果) | 100分/个 |
| 动物产品(Animal) | 蛋/奶/毛/松露等 | 100分/个 |
| 工匠品(Artisan) | 蛋黄酱/奶酪/果酒/布等 | 100分/个 |
| 海底产品(Ocean) | 海底动物产出 | 100分/个 |

| 评分公式 | 说明 |
|:--------|:------|
| 基础分 | 物品基础售价 ÷ 10(取整) |
| 品质加成 | 银星 ×1.25 / 金星 ×1.5 / 铱星 ×2.0 |
| 总分类别 | 综合得分 = (最高品作物 + 最高品水果 + 最高品动物 + 最高品工匠 + 最高品海底) × 系数 |

| 排名 | 奖励 | 备注 |
|:----:|:----|:-----|
| 第 1 名 | 金南瓜奖杯 + 5000g + 秋之星勋章 | 称号"丰收大师" |
| 第 2 名 | 银南瓜奖杯 + 2000g | — |
| 第 3 名 | 铜南瓜奖杯 + 1000g | — |
| 参与奖 | 南瓜灯(家具) | — |

## 七、设计特点分析

| 特点 | 说明 |
|:----|:------|
| **16 节/年 + 特殊事件** | 节日密度高(每季度 4 个)，几乎每周都有可预期事件——保持新鲜感 |
| **收获祭为核心** | 对应星露谷的城镇展览会——一年中最重要的事件，综合考验种/牧/工匠/海底四线产出 |
| **环保主题贯穿** | 海滩清理节+海底净化里程碑+回收系统——环保是珊瑚岛区别于其他农场游戏的独特标签 |
| **人鱼节独有** | 海洋祭(夏 28)——海底人鱼文化展示，利用 3D 优势呈现水下庆典视觉 |
| **与星露谷对比** | 星露谷约 12 个节日(含复活节/万圣节/冰雪节等)——Coral Island 多了环保类+人鱼文化类节日 |
| **许愿机制(除夕)** | 给玩家策略选择——财富/爱情/幸运/健康四选一，持续一季——轻度策略深度 |
| **节日奖励品质** | 大部分节日提供限定装饰品/家具/服装，全收集党有动力参加每个节日 |
