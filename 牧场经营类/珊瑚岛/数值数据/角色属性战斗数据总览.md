## 一、角色基础属性

Coral Island 的角色属性系统相对简洁，未采用传统 RPG 的 STR/DEX/INT 结构，而是通过**技能等级 + 装备 + 食物 Buff** 三项叠加来定义角色能力。

| 属性 | 初始值 | 增长方式 | 上限(约) | 说明 |
|:----:|:-----:|:--------:|:--------:|:------|
| HP(生命值) | 100 | 战斗技能每级 +5，防御装备加成 | ≈250(无装备) | 归零则晕倒，次日在家醒来丢失部分物品 |
| 体力(Stamina) | 270 | 特定道具+永久升级(星露) | ≈400+ | 所有工具操作消耗体力，归零则无法操作 |
| 攻击力(Attack) | 5(空手) | 武器基础攻击 + 战斗技能加成 | ≈30-60(依武器) | 决定对怪物伤害量 |
| 防御力(Defense) | 0 | 防具/饰品提供 + 部分技能 | ≈15-25 | 减少受到的伤害 |
| 速度(Speed) | 100% | 特定食物/饰品 | ≈130% | 影响移动和攻击速度 |
| 暴击率(Crit Rate) | 2% | 武器自带 + 战斗技能加成 | ≈15-25% | 暴击时造成 1.5x-2x 伤害 |
| 暴击倍率(Crit Dmg) | 1.5x | 武器/饰品/技能 | ≈3.0x | 暴击时额外伤害倍数 |

## 二、HP(生命值)系统

### 2.1 HP 构成

| 来源 | 增加量 | 备注 |
|:----|:-----:|:-----|
| 初始值 | 100 | 游戏开始时 |
| 战斗技能等级 | +5/级 | 共10级，最多+50 |
| 战斗技能 Lv5「防御者(Defender)」 | +25 | 分支职业选择 |
| 防具(头盔/胸甲/靴子) | +5~+30/件 | 品质越高加成越多 |
| 饰品(戒指/项链) | +5~+20 | 稀有掉落或制作 |
| 星露(Stardust)奖励 | +5~+25 | 通过博物馆捐赠等获得 |
| **理论最大HP** | **≈250-300** | 满级+顶级装备+全收集 |

### 2.2 受伤与恢复

| 机制 | 说明 |
|:----|:------|
| 怪物伤害 | 直接扣除 HP，无防御时全额承受 |
| 掉落伤害 | 矿洞边缘掉落扣 3-10 HP(取决于高度) |
| 饿晕 | 体力归零后继续操作会开始扣 HP (每分钟 2-3 点) |
| 自动恢复 | 睡觉后 HP 全满 |
| 食物恢复 | 大部分食物恢复 HP(具体见下表) |
| 温泉恢复 | 温泉(位于森林)可快速恢复 HP 和体力 |

### 2.3 常见恢复食物

| 食物 | HP恢复 | 体力恢复 | Buff |
|:----|:-----:|:--------:|:----:|
| 沙拉 | 30 | 45 | — |
| 面包 | 25 | 35 | — |
| 鱼汤 | 50 | 60 | +1 防御(5分钟) |
| 果酒 | — | 40 | — |
| 披萨 | 75 | 100 | +2 攻击(5分钟) |
| 巧克力蛋糕 | 100 | 120 | +2 速度(3分钟) |
| 金色综合餐 | 150 | 200 | 全属性 +2(10分钟) |

## 三、体力(Stamina)系统

### 3.1 体力消耗规则

| 工具/动作 | 基础消耗 | 蓄力加成消耗 | 升级减少 |
|:---------:|:-------:|:-----------:|:--------:|
| 锄头(Hoe) | 2 | 4(3x3范围) | -0.5/级 |
| 水壶(Watering Can) | 3 | 6(3x3范围) | -0.5/级 |
| 斧头(Axe) | 2(普通)/3(硬木) | — | -0.5/级 |
| 镐(Pickaxe) | 2 | — | -0.5/级 |
| 镰刀(Scythe) | 1 | — | 不变 |
| 钓鱼(Rod) | 8(每抛一次) | — | -1/级 |
| 虫网(Bug Net) | 4(每挥一次) | — | -1/级 |
| 剑(攻击) | 2 | 5(特殊技) | — |
| 翻滚闪避 | 3 | — | — |

### 3.2 体力恢复途径

| 途径 | 恢复量 | 每日可用次数 |
|:----|:-----:|:-----------:|
| 睡眠(到次日6:00) | 全满 | 1次 |
| 午睡(床) | 50% | 1次 |
| 温泉 | 5/秒 | 不限(6:00-24:00 开放) |
| 食物 | 依食物而定 | 不限 |
| 星露(Stardust) | +25 永久上限 | 一次性 |
| 体力药剂 | 100 | 商店购买/制作 |

### 3.3 特殊机制

- **体力归零**: 无法使用任何工具，但可以移动和对话
- **体力透支**: 归零后继续强行操作→开始扣 HP→HP 归零→晕倒
- **晕倒惩罚**: 次日 6:00 自动在家醒来，体力全满但会丢失部分物品(随机 3-5 件，可去失物招领处找回)

## 四、战斗属性详解

### 4.1 攻击力计算

```
最终攻击力 = 武器基础攻击 + 战斗技能加成 + 食物Buff + 饰品加成
```

| 武器等级 | 基础攻击 | 获取方式 |
|:--------:|:--------:|:--------:|
| 基础剑(Basic Sword) | 5-8 | 初始赠送 |
| 铜剑(Copper Sword) | 10-14 | 铁匠铺制作(铜锭×5) |
| 银剑(Silver Sword) | 18-24 | 铁匠铺制作(银锭×5) |
| 金剑(Gold Sword) | 28-35 | 铁匠铺制作(金锭×5) |
| 锇剑(Osmium Sword) | 40-50 | 铁匠铺制作(锇锭×5) |
| 稀有武器(传说级) | 45-65 | 宝箱/矿洞Boss掉落 |

### 4.2 防御力计算

```
最终防御力 = 防具总值 + 技能加成 + 食物Buff
```

| 防具部位 | 铜级 | 银级 | 金级 | 锇级 |
|:--------:|:----:|:----:|:----:|:----:|
| 头盔 | 2 | 4 | 6 | 8 |
| 胸甲 | 3 | 5 | 7 | 10 |
| 靴子 | 1 | 2 | 3 | 5 |

### 4.3 暴击系统

| 属性 | 说明 |
|:----|:------|
| 基础暴击率 | 2% |
| 暴击率每级战斗技能 | +0.5% |
| 战斗技能 Lv5「侦察兵(Scout)」 | 暴击率 +50%(乘法) |
| 暴击倍率基础 | 1.5x |
| 稀有武器附加 | 暴击率 +3~8%，暴击倍率 +0.5~1.0x |
| 饰品附加 | 暴击率 +2~5% |

## 五、属性与技能的关系

| 技能 | 直接影响 | 间接影响 |
|:----|:--------|:--------|
| 种植(Farming) | — | 获得更高品质作物(制作更好食物/恢复品) |
| 挖矿(Mining) | — | 获得更高级矿石→更高级装备 |
| 战斗(Combat) | HP、攻击、防御、暴击 | 解锁战斗职业分支 |
| 采集(Foraging) | — | 获得更多资源→更多制作材料 |
| 钓鱼(Fishing) | — | 获得鱼→高级料理→更多恢复品 |
| 捕虫(Bug Catching) | — | 获得昆虫→博物馆捐赠→属性奖励 |
| 畜牧(Animal) | — | 高品质动物产品→更好工匠品 |
| 烹饪(Cooking) | 食物Buff质量 | 更高级料理→更强Buff |

## 六、设计特点分析

| 特点 | 说明 |
|:----|:------|
| **简洁属性系统** | 无复杂加点系统，属性增长完全来自技能+装备，降低新手门槛 |
| **体力为核心资源** | 体力管理是每日规划的核心——做什么、做多少由体力预算决定 |
| **属性和装备绑定** | 战斗属性主要来自装备升级(武器→攻击、防具→防御)，装备升级和矿洞探索形成正循环 |
| **食物Buff策略** | 高级战斗前准备食物Buff→提升属性→挑战更难矿洞→获得更好材料→更好装备→更强战斗能力 |
| **惩罚机制温和** | 晕倒只丢失部分物品(可找回)，不损失进度——鼓励尝试而非惩罚冒险 |
| **与星露谷差异** | Coral Island 引入了装备槽(头盔/胸甲/靴子/饰品)超越了星露谷的简单系统；体力上限更高(270 vs 星露谷初始270但增长机制不同) |
