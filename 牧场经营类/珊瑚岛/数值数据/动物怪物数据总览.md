### 1.3 动物好感度机制

| 项目 | 说明 |
|:----|:------|
| **初始爱心** | 0❤ (刚购买) |
| **爱心上限** | 10❤ |
| **抚摸效果** | +5 好感度，每日限 1 次 |
| **喂食效果** | 少量好感度(≈2)，必须每天喂以保持产出 |
| **户外放牧** | 晴天放牧可自然增长好感度 |
| **动物收音机** | 放在畜棚/鸡舍内可提升好感度增长速度(10%) |
| **5❤ 解锁** | 产出变为大号(Large)版本，售价+50% |
| **10❤ 解锁** | 最大好感度，产出品质可能提升 |
| **不喂食后果** | 好感度每日-5，产出停止 |

### 3.2 海底动物特点

- 不需要喂食(自动吸收海洋营养)→减少日常维护
- 不需要放牧(海域即牧场)
- 产出可直接用于工匠加工或售卖
- 部分产出可制作特殊海底主题装饰品
- 海底动物好感度仅影响产出品质(不影响产出频率)

## 四、马匹坐骑

马匹通过建造马厩获得，是快速移动的工具，不属于畜牧动物。

| 特性 | 数据 |
|:----|:----:|
| 解锁方式 | 建造马厩(Stable) |
| 基础速度 | 玩家步行×1.4 |
| 速度升级 | 在马厩升级马鞍 |
| 互动 | 吹口哨召唤(任何地方) |
| 外观 | 可通过道具改变马匹颜色和装饰 |
| 情绪 | 抚摸可提升，影响奔跑速度表现 |

## 六、设计特点分析

| 特点 | 说明 |
|:----|:------|
| **分层解锁** | 动物随城镇等级渐进解锁(F→B)，形成长期养成目标 |
| **垂直产品线** | 蛋→蛋黄酱、奶→奶酪、毛→布，每条线都有明确加工价值提升 |
| **海底差异化** | 海底动物不需要喂食(与陆地完全不同)，降低海底牧场维护成本 |
| **高价值动物** | 孔雀(羽毛)和路瓦克(咖啡)属于奢侈品等级，后期主要收入来源 |
| **宠物功能弱化** | 宠物无产出、无实质辅助功能→纯陪伴型，与《牧场物语》系列类似 |
| **与星露谷差异** | 多了海底动物体系(独特特色)；动物种类更多(10+种 vs 星露谷7种)；有城镇等级门槛而非单纯金钱门槛 |
