## 一、系统总览

Coral Island 的社交系统以**好感度(爱心)** 为核心数值，通过对话、送礼、完成请求等方式提升。当好感达到特定阈值时触发**心事件(Heart Event)**，推进角色剧情。

| 指标 | 数值 |
|:----|:----:|
| 可结婚候补总数 | 28 位 |
| 男性候补 | 14 位 |
| 女性候补 | 13 位 |
| 非二元候补 | 1 位 (Raj) |
| 人鱼候补 | 3 位 (Denali, Semeru, Miranjani) |
| 好感度上限 | 10❤ |
| 恋爱门槛 | 8❤ (需购买吊坠) |
| 结婚门槛 | 10❤ (需送「永恒的吊坠」) |
| 可同时恋爱人数 | 无上限(可多线) |
| 可结婚人数 | 1 位(一夫一妻) |

## 四、恋爱与结婚系统

### 4.1 流程

```
8❤ → 购买「吊坠(Locket)」(4,000g, 在杂货店购买)
      → 赠送吊坠 → 确认恋爱关系
10❤ → 购买「永恒的吊坠(Eternal Locket)」(8,000g, 在杂货店购买)
      → 赠送 → 求婚成功
      → 3 天后举办婚礼 (或自行选择日期)
      → 配偶搬入农场
```

### 4.2 结婚后效果

| 效果 | 说明 |
|:----|:------|
| 配偶搬入 | 在农舍增加配偶房间(带专属装饰) |
| 配偶帮忙 | 每天有概率帮忙: 浇水/喂动物/修理篱笆(约 20-30% 概率) |
| 配偶礼物 | 每周赠送 1-2 件物品(料理/材料) |
| 亲密互动 | 可每天亲吻(增加好感度) |
| 孩子 | 结婚 14 天后触发孩子剧情 |

### 4.3 多线恋爱与后果

- 可以同时向多位候补赠送吊坠(同时建立恋爱关系)
- 但只能与其中 1 位结婚
- 结婚后将自动结束与其他人的恋爱关系(好感度不变但不能再继续约会)
- 结婚后不可离婚(目前版本无离婚机制)

## 五、育儿系统

### 5.1 孩子获得流程

| 阶段 | 触发条件 | 持续时间 | 表现 |
|:----|:--------|:--------:|:----:|
| 怀孕通知 | 结婚 14 天后配偶询问 | — | 配偶说想要孩子→玩家确认 |
| 怀孕期 | 确认后 | 14 天 | 配偶腹部隆起(外观变化) |
| 分娩 | 14 天后 | 1 天 | 玩家当天不能操作，自动触发剧情 |
| 婴儿 | 出生后 | 14 天 | 在婴儿床中，只能看/互动 |
| 幼儿阶段 | 出生 14 天后 | 28 天 | 可以在房间内爬行 |
| 儿童阶段 | 出生 42 天后 | 持续 | 可以在农场内走动(偶尔跟玩家出门) |

### 5.2 孩子相关数据

| 项目 | 数据 |
|:----|:-----|
| 最多孩子数 | 2 个 |
| 孩子性别 | 由配偶提议(玩家可选择同意) |
| 孩子外观 | 继承父母特征(发色/瞳色) |
| 孩子好感度 | 有独立的爱心系统(通过对话/礼物提升) |
| 孩子帮忙 | 儿童阶段后可帮忙(极少量, 收鸡舍蛋等) |
| 孩子房间 | 随房屋升级获得儿童房 |

### 5.3 当前版本限制 (v1.0)

- 孩子目前停留在儿童阶段(不会长到青少年/成人)
- 最多 2 个孩子后无法再孕
- 孩子没有独立日程(NPC 行为)——仅在农场内活动

## 七、设计特点分析

| 特点 | 说明 |
|:----|:------|
| **28 位候补(大池子)** | 超越同类所有游戏(星露谷12位/牧场物语系列12-16位/波西亚时光10位)，覆盖面广 |
| **三大人鱼候补** | 独特的海底浪漫线，将探索进度和社交深度绑定 |
| **通用最爱策略** | 粉红钻石/金珍珠/黑玫瑰 → 简化送礼决策，避免查表疲劳 |
| **8❤恋爱门槛低** | 对比星露谷需要购买道具+特定好感度，Coral Island 相对容易进入恋爱线 |
| **多线恋爱无惩罚** | 可同时和多人恋爱 → 降低试错成本，但结婚后自动切断 |
| **育儿停留在儿童** | v1.0 版本孩子不会长大到青少年——比星露谷(孩子长到幼儿停)更浅 |
| **心事件质量** | 3D 场景+配音(部分)→沉浸感优于像素风同类，但事件数量较少(每人5个 vs 星露谷每人10+个) |
