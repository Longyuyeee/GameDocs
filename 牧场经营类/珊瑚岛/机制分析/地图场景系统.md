## 一、地图总体布局

Coral Island 的地图采用开放式岛屿设计，总面积约为星露谷的2-3倍，分为**地面区域**和**海底区域**两个独立的可探索维度。地面以中部偏北的 Starlet Town（星莱镇）为核心，向四周辐射出森林、海滩、湖区、高地和温泉等自然区域。海底则由海滩入口向下延伸，包含4个深度层级的美人鱼王国和广阔的珊瑚礁生态系统。

地图整体采用东南亚热带岛屿风格——棕榈树、茅草屋顶建筑、色彩鲜艳的植物群，与星露谷的温带乡村风形成鲜明对比。

## 二、地面区域详解

### 2.1 农场区域 (The Farm)

| 属性 | 说明 |
|------|------|
| **位置** | 岛屿西南部，通过木桥与星莱镇相连 |
| **初始状态** | 杂草丛生、碎石遍布、有废弃建筑（祖父的旧屋） |
| **可清理面积** | 约400+可耕地格（初始约100格，逐步解锁） |
| **核心建筑** | 农舍（可升级）、初始仓库、信箱 |
| **扩建选项** | 木匠铺提供：畜棚/鸡舍/筒仓/作坊/鱼塘/马厩等 |

**农场特色**:
- 初期有大量树桩、石块和杂草需要清理，清理过程本身就是获取资源的方式
- 农场西侧有通往海滩的小路
- 农场地图高度自定义——可以放置建筑、装饰、围栏、道路
- 与星露谷不同，Coral Island 的农场不分割为"农场地图风格"选择，而是统一的绿地+海岸格局

### 2.2 星莱镇 (Starlet Town)

岛屿的社交与商业中心，包含以下关键地点：

| 地点 | NPC / 服务 | 营业时间 | 休息日 |
|------|-----------|---------|:-----:|
| Sam's General Store（杂货店） | Sam | 9AM-6PM | 周日 |
| Sanchez Brothers Blacksmith（铁匠铺） | Sanchez兄弟（Kira在其中） | 9AM-6PM | 周日 |
| Ling's Laboratory（实验室） | Ling | 9AM-5PM | 周六 |
| Museum（博物馆） | Scott | 10AM-5PM | 周一 |
| Town Hall（市政厅） | Connor (镇长) | 8AM-5PM | 周末 |
| The Inn（旅馆/酒馆） | 多NPC出没 | 傍晚-深夜 | 无 |
| Clinic（诊所） | 医生、护士 | 9AM-3PM | 周末 |

**设计特点**: 星莱镇的建筑布局紧凑但有层次感——商店集中在主街，实验室和博物馆在侧翼，市政厅位于广场中心。与星露谷的"乱中有序"不同，Coral Island 的城镇规划更接近现实小镇的"广场+放射状街道"格局。

### 2.3 海滩 (Beach)

| 属性 | 说明 |
|------|------|
| **位置** | 岛屿南端，连接农场东侧 |
| **主要功能** | 钓鱼、采集贝壳、海底潜水入口、海滨餐厅 |
| **关键地标** | 码头（潜水出发点）→ 通往海底 |
| **季节变化** | 夏季有季节性水果（椰子等），冬季可能有风暴 |
| **特殊区域** | 潮汐池（低潮时可采集特殊海产） |

**海滩小屋 (Beach Shack)**: Sunny 经营，提供鱼竿和虫网升级服务。

### 2.4 森林 (Forest)

| 属性 | 说明 |
|:----|------|
| **位置** | 岛屿中部偏东 |
| **主要功能** | 采集野果/蘑菇、伐木、通往矿洞 |
| **关键地标** | The Cavern 入口（矿洞） |
| **特殊居民** | Ben（流浪商人，住 caravan） |
| **生态特征** | 密林区 + 开阔林间空地，有猴子群 |

森林是连接多个区域的枢纽——向北通往湖区，向东通往 Woodlands，向南回到城镇。

### 2.5 湖区 (Lake) 与湖中神殿

| 属性 | 说明 |
|:----|------|
| **位置** | 森林以北，通过木桥到达 |
| **主要功能** | 钓鱼、祭坛供奉 |
| **核心建筑** | Lake Temple（湖中神殿）——整个主线推进的核心地点 |
| **环境** | 宁静的湖泊，周围环山，瀑布点缀 |
| **季节变化** | 冬季湖面会结冰（视觉效果，不影响钓鱼） |

此处是玩家遇见湖中女神并进行供奉的场所。湖中神殿的入口被藤蔓覆盖，需要完成"神话之梦"任务后才可进入。

### 2.6 Woodlands（林地）

| 属性 | 说明 |
|:----|------|
| **位置** | 森林东侧 |
| **主要功能** | 住宅区、学校、美食车 |
| **关键建筑** | 学校（School）、Taco 卡车 |
| **氛围** | 比森林更开阔、更适合居住的家庭区域 |

### 2.7 山坡 (Hillside) 与葡萄园 (Vineyard)

| 属性 | 说明 |
|:----|------|
| **位置** | 森林北侧，连接 Woodlands 和温泉 |
| **主要功能** | 葡萄种植、葡萄酒酿造 |
| **关键建筑** | Vineyard 庄园 |
| **NPC** | 葡萄园主人 |

### 2.8 花园巷 (Garden Lane)

位于森林和 Woodlands 之间的宁静通道，有一些特色住宅和花园。是部分 NPC 的居住区。

### 2.9 瞭望台/天文台 (Lookout / Observatory)

| 属性 | 说明 |
|:----|------|
| **位置** | 岛屿最北端高地 |
| **主要功能** | 观景点——可俯瞰整个岛屿和海洋 |
| **关键建筑** | 灯塔 (Lighthouse)、天文台 |
| **特殊用途** | 黑市入口（每周二/四/五晚，从 Ratih 处进入） |
| **景观价值** | 游戏中最美的日落观赏点之一 |

**黑市 (Black Market)**: 每周二、四、五晚在灯塔处开放，由 Ratih 经营。出售稀有物品、文物和高级装备，但价格昂贵。

### 2.10 温泉 (Hot Spring)

| 属性 | 说明 |
|:----|------|
| **位置** | 岛屿最北端，瞭望台西侧 |
| **初始状态** | 被藤蔓覆盖、干涸（需恢复） |
| **恢复条件** | 完成湖祭坛2个供奉包后自动触发过场动画 |
| **恢复效果** | 热泉开始流动，水体恢复 |
| **功能** | 免费恢复体力——约每秒恢复4点体力（约30分钟游戏内时间可恢复满） |
| **升级** | 完成更多供奉可提升恢复速度 |

> **与星露谷对比**: 星露谷的温泉位于地图北部铁路区域，同样提供免费体力恢复，但 Coral Island 的温泉恢复速度更快，且随供奉进度升级——这种"渐进式增强"比星露谷的"恒定恢复率"更能给玩家带来成就感。

### 2.11 Savanna（萨凡纳）—— 后期解锁区域

| 属性 | 说明 |
|:----|------|
| **解锁条件** | 完成稀有祭坛全部供奉包 |
| **位置** | 通过桥梁连接岛屿（原被封锁的区域） |
| **生态特点** | 热带草原地貌——开阔草地、稀疏树木、野生动物 |
| **新动物** | 狮子、斑马、鸵鸟等萨凡纳动物（畜牧系统扩展） |
| **新地牢** | Cave of Memories（记忆之洞）——51+层超高难度地牢 |
| **功能** | 高级农业区、新资源采集 |

### 2.12 巨人村落 (Giant Village)

| 属性 | 说明 |
|:----|------|
| **解锁条件** | 解放至少2位巨人 |
| **位置** | 地图特定区域（通往萨凡纳的方向） |
| **核心建筑** | 附魔神坛 (Enchantment Shrine)、巨人酋长屋、各元素巨人屋 |
| **功能** | 工具附魔、巨人对话、特殊任务 |

## 三、海底区域详解

### 3.1 深度层级

海底是 Coral Island 最具特色的地图维度——海洋面积约等于陆地的三分之二，包含4个深度层级：

| 深度 | 解锁条件 | 环境主题 | 关键发现 |
|:---:|---------|---------|---------|
| 10m | 初始可用（完成潜水设备任务） | 退化珊瑚礁、稀疏海藻 | 普通贝壳、海盐、小型海胆 |
| 20m | 完成10m区域主要清理 | 中密度珊瑚、海草床 | 露兜树、稀有贝壳、宝箱 |
| 40m | 完成20m区域主要清理 | 珊瑚群恢复区域、多彩鱼类 | 美人鱼遗址、文物宝箱 |
| 50m+ | 完成40m区域主要清理 | 深海珊瑚礁、荧光生物 | 美人鱼王国入口、Naga 宫殿 |

### 3.2 海底关键地点

| 地点 | 功能 |
|------|------|
| 珊瑚修复点（Coral Sites） | 各深度均有，清理垃圾后珊瑚恢复彩色 |
| 海底农场（Underwater Farm） | 全年无季节限制的种植区域 |
| 海底围栏（Underwater Fence） | 养殖海洋动物（蟹、龙虾等） |
| 太阳球阵列（Dua Pulu Orbs） | 激活后打通通往美人鱼王国的路径 |
| 美人鱼王国（Merfolk Kingdom） | 美人鱼的主要聚居地，王宫所在地 |
| Naga 宫殿（Naga Palace） | 王室所在地，"Unwanted Guest"任务目的地 |
| 美人鱼传送点 | 各深度清理珊瑚后解锁的快速移动点 |

### 3.3 美人鱼王国 (Merfolk Kingdom)

| 属性 | 说明 |
|:----|------|
| **统治结构** | 国王 Krakatoa + 女王 Nanda Devi |
| **可恋爱角色** | Semeru、Denali、Miru（共3位美人鱼候补） |
| **建筑风格** | 珊瑚宫殿 + 贝壳房屋 + 海中花园 |
| **主要功能** | 美人鱼社交、海底任务、特殊商品 |

| 瞭望台 (Lookout) | 灯塔下 | 初始可用 |

## 六、季节景观变化

| 区域 | 春季 | 夏季 | 秋季 | 冬季 |
|:---:|:----|:----|:----|:----|
| 农场 | 绿意盎然，野花盛开 | 作物繁茂，日照长 | 落叶开始，金黄色调 | 部分区域积雪（非全地图） |
| 星莱镇 | 樱花点缀 | 棕榈树更翠绿 | 暖色调落叶 | 温暖地区少雪 |
| 森林 | 蘑菇丛生，动物活跃 | 密林茂盛 | 落叶覆盖地面 | 稀疏雪景 |
| 湖区 | 湖面清澈 | 水鸟活跃 | 周围树木变黄 | 湖面半冰封效果 |
| 海滩 | 宜人 | 旅游旺季 | 偶尔风暴 | 海浪更大 |
| 海底 | 无变化（四季恒定） | 无变化 | 无变化 | 无变化 |

> **与星露谷对比**: Coral Island 位于热带地区，冬季降雪范围远小于星露谷（后者所有的地面区域都会积雪）。Coral Island 的冬季仅在高海拔区域（森林北部、瞭望台）可见薄雪，整体保持热带岛屿的绿色基调。这一设计更符合真实地理逻辑，但也减少了一部分"冬季转型"的仪式感。

## 七、地图设计特点分析

1. **热带岛屿的垂直层次**: 地图从海滩（海拔0m）延伸到瞭望台（海拔最高点），形成了自然的垂直层次。玩家在日常活动中会自然地经历"上山下海"的动线。

2. **陆海双维度的探索**: 海底不是一小块额外区域，而是占据了地图总面积约40%的第二维度。这种"双层地图"设计使探索深度远超同类游戏。

3. **流畅的区域过渡**: 森林作为地图的地理中心，连接了城镇、湖区、Woodlands 和矿洞——玩家几乎任何旅程都会经过森林，这种设计减少了"我该去哪里"的困惑。

4. **隔离与回报的循环**: 温泉（需要完成2个供奉）、萨凡纳（需要完成所有稀有供奉）、美人鱼王国（需要大量海底清理）——这些高价值区域都被设置在最远的位置，且需要完成特定条件才能进入。玩家需要先付出努力，然后获得"新区域"作为回报。

5. **密度控制的节奏感**: 城镇区域建筑密集、道路短、互相可达；自然区域（森林、湖区）则开阔、步行时间较长。这种"密集→开阔→密集"的节奏控制让玩家不会感到单调。

6. **海底与陆地的视觉对比**: 陆地的暖色调（绿/金/棕）与海底的冷色调（蓝/紫/荧光）形成强烈的视觉反差，每一次"潜水"都像从一世界进入另一世界。

7. **黑市的隐秘性设计**: 瞭望台在白天是一个纯粹的观景点，而在夜晚（特定日期）变为黑市入口。这种"双重用途"设计鼓励玩家在不同时间重新访问已探索的区域。

## 八、与星露谷物语地图对比

| 维度 | 珊瑚岛 (Coral Island) | 星露谷物语 (Stardew Valley) |
|------|-------------------|-------------------------|
| **总面积** | 约为星露谷2-3倍 | 标准 |
| **第二维度** | 海底（占地图40%） | 无 |
| **区域数量** | 约12个主要区域 | 约10个主要区域 |
| **城镇风格** | 东南亚热带小镇 | 欧美温带乡村 |
| **矿洞入口** | 森林Cavern（多矿洞合一入口） | 地图北部单一入口 |
| **温泉** | 北部高地，需解锁 | 北部火车站旁，初始可用 |
| **后期区域** | 萨凡纳+美人鱼王国 | 姜岛（需完成社区中心） |
| **传送密度** | 约11个传送点 | 约9个传送点 |
| **季节景观** | 冬季不积雪或少积雪 | 四季分明，冬季全覆盖积雪 |
| **快速旅行解锁** | 完成第1个供奉 | 完成社区中心或购买回程法杖 |
| **黑市** | 有（限时开放） | 无 |
| **室内地图** | 商店有独立室内场景 | 商店/建筑有独立室内场景 |

## 九、地图探索推荐路线

| 阶段 | 重点区域 | 目的 |
|:---:|---------|------|
| 春1-7 | 农场 + 星莱镇 + 海滩 | 熟悉核心商业和社交圈，采集基础资源 |
| 春8-14 | 森林 + 湖区 + 矿洞 | 推进"Mythical Dream"任务，开始供奉，解锁温泉 |
| 春15-夏 | 海滩潜水（10m）+ 森林矿洞 | 解锁海底和矿洞探索，两条线同时推进 |

| 秋-冬 | 40m海底 + 火矿洞 | 挑战最高难度，解放所有巨人 |
| 第二年 | 萨凡纳 + 美人鱼王国 | 终极目标：全区域解锁，全地图探索 |
