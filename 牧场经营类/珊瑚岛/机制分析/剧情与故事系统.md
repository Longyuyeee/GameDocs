# 剧情与故事系统 - 珊瑚岛(Coral Island)
> 数据来源: IGN, Fandom Wiki

## 一、主线
- 继承祖父的农场,搬到珊瑚岛
- 发现岛上存在严重的环境问题(垃圾污染)
- 通过清理垃圾+社区建设→恢复岛屿生态
- 最终目标: 城镇等级S + 博物馆全收集 + 海底完全探索
- 无"Boss战"结局,纯经营+环保叙事

## 二、关键节点
| 节点 | 触发 |
|------|------|
| 潜水装备 | 主线推进 |
| 海底深度解锁 | 清理海洋垃圾数量 |
| 城镇等级提升 | 清理+捐赠+任务 |
| 博物馆全捐赠 | 6大类全收集 |

## 三、候补剧情
- 每位候补约2-4个心色事件
- 8心后+吊坠=恋爱
- 10心+婚戒=结婚
- 3位人鱼(Merfolk)候补待后续更新
