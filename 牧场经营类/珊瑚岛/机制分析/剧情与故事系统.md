## 一、故事世界观设定

Coral Island 的故事发生在印度尼西亚风格的虚构热带岛屿上。主角原本在繁华都市 Pokyo 从事高压白领工作，身心俱疲后决定继承祖父留下的农场，来到 Coral Island 开启新生活。然而，岛屿在工业化浪潮下面临严重的生态危机——海洋被垃圾污染，珊瑚礁大面积白化，森林的古老精灵（Giants）陷入沉睡，湖中女神的神力被封印。

游戏的核心叙事融合了三个层面的冲突：**人与自然的和谐重建**（环保）、**工业化与传统的平衡**（城镇发展）、**古老秘密的揭晓**（巨人族+美人鱼王国）。

## 二、主线剧情流程

### 2.1 第一章：继承与觉醒（春1-春8）

| 事件 | 触发条件 | 核心内容 |
|------|---------|---------|
| 抵达岛屿 | 游戏开始 | 主角乘船到达 Coral Island，继承祖父农场，遇见镇长 Connor 和助手 |
| 城镇等级解锁 | 第一个任务 | 了解城镇等级系统（F→S级），各等级解锁新的商店物品和区域 |
| Mythical Dream（神话之梦） | 第一天睡觉后（约春6-8） | 梦中神秘声音指引主角前往岛上的四棵神树之一 |
| 会见巨人酋长 | Mythical Dream 完成后 | 巨人酋长（Chieftain Giant）为主角举行仪式，授予"第三只眼"——使主角能理解古老语言，并指引去见湖中女神 |

### 2.2 第二章：祭坛供奉与岛屿恢复

主角在湖中神殿（Lake Temple）遇见女神（Goddess of the Lake）。她因力量衰竭而陷入沉睡，需要主角通过供奉祭品来恢复神力。每完成一组供奉，女神的灵力就会增强，同时解锁新的区域、设施或奖励。

供奉系统分为四个祭坛（详见第四节），是游戏中期的主线推进器。

| 供奉进度 | 解锁内容 |
|---------|---------|
| 完成第1个供奉 | 快速旅行（Waypoints）解锁 |
| 完成作物祭坛全部 | 温室（Greenhouse） |
| 完成鱼类祭坛全部 | 挖掘场（Dig Site） |
| 完成高级祭坛全部 | Warp Sesajen（传送道具） |
| 完成稀有祭坛全部 | 萨凡纳（Savannah）区域 + 神话宠物种子 |

### 2.3 第三章：矿洞清理与解放巨人

四大元素矿洞（Earth/Water/Wind/Fire）的底层各封印着一位巨人（Giant）。清理矿洞至底部（40F）并击败/解除诅咒石碑，即可解放巨人。解放的巨人会加入巨人村落（Giant Village），提供独特奖励。

| 矿洞 | 解放的巨人 | 奖励 |
|:---:|----------|------|
| 土矿洞 (Earth) | Grog | 附魔神殿（Enchantment Shrine）解锁 |
| 水矿洞 (Water) | 未定 | 水域相关增益 |
| 风矿洞 (Wind) | 未定 | 风元素相关增益 |
| 火矿洞 (Fire) | 未定 | 火元素相关增益 |

解放至少两名巨人可解锁巨人村落传送点。

### 2.4 第四章：美人鱼王国与海底秘密

海底深处存在着美人鱼王国（Merfolk Kingdom），由国王 Krakatoa 和女王 Nanda Devi 统治。主线通过以下阶段推进：

| 阶段 | 条件 | 内容 |
|:---:|------|------|
| 潜水装备解锁 | 完成特定主线 | 获得从海滩下水的能力 |
| 珊瑚清理 | 10m/20m/40m/50m深度 | 清理污染珊瑚，解锁海底传送点 |
| Dua Pulu 太阳球 | 全海底收集 | 激活太阳球阵列，打通通往美人鱼王国的路径 |
| Unwanted Guest 任务 | 到达 Naga 宫殿 | 解决美人鱼王国内部危机 |
| 三位人鱼候补 | 主线推进后解锁 | Semeru、Denali、Miru 成为可恋爱对象 |

> **海底探索特点**: 海底区域是非战斗区，玩家通过清理垃圾、采集海产、养殖海洋动物来推进进度。每个深度区域都有独特的生态系统和收集品。

### 2.5 第五章：萨凡纳与最终篇章

完成稀有祭坛全部供奉后，解锁萨凡纳（Savannah）——一片遥远的大陆，包含新的生态环境、高级动物（如狮子、斑马）以及 Cave of Memories（记忆之洞，超高层战斗地牢）。

萨凡纳中包含最后一位巨人的线索，贯穿全部矿洞+祭坛供奉的完整循环。

## 三、关键 NPC 剧情线

### 3.1 湖中女神（Goddess of the Lake）

- 作为祭坛供奉的核心NPC，女神的恢复程度与岛屿生态恢复直接关联
- 她的对话随供奉进度变化——从虚弱低语到逐渐恢复神力
- 最终揭示她与巨人族和美人鱼的古老契约

### 3.2 博物馆馆长 Scott

- 玩家到达岛屿后触发"Empty No More"任务
- Scott 是博物馆的 curator，也是可恋爱角色之一
- 每捐赠一定数量藏品触发特殊对话和奖励
- 捐赠里程碑：10捐→土豆种子，50捐→洒水器I，100捐→鱼卷，200捐→体力果实，300捐→自动SFH，340捐→独角兽服装

### 3.3 巨人族（The Giants）

- 古代的岛屿守护者，曾与花之女神（Goddess of Flowers）和美人鱼共同维持生态平衡
- 因某种灾难而沉睡/被封印在矿洞底部
- 每个巨人都有独特个性和能力
- 解放后可在巨人村落找到他们

### 3.4 可恋爱NPC（部分关键剧情角色）

| 角色 | 位置 | 剧情特点 |
|------|------|---------|
| Scott | 博物馆 | 历史学者，与博物馆捐赠线绑定 |
| Kira | 铁匠铺 | 战斗训练员，最初赠送剑 |
| Theo | 农场周围 | 自然主义者，与植物/生态相关 |
| Alice | 城镇 | 热心环保主义者 |
| Surya | 城镇 | 艺术家，与社区美化相关 |
| Semeru | 海底 | 美人鱼族，需解锁海底后才可恋爱 |

每位候补有约2-4个心动事件（Heart Events），8心后可赠送吊坠确定恋爱关系，10心后结婚。

## 四、湖祭坛（Lake Temple）供奉系统

### 4.1 与星露谷社区中心的对比

这是 Coral Island 最明显的星露谷致敬/改进点：

| 维度 | 珊瑚岛湖祭坛 | 星露谷物语社区中心 |
|------|------------|-----------------|
| **核心机制** | 4个祭坛×每个含5-6个供奉包 | 6个房间×每个含3-7个收集包 |
| **供奉方式** | 放入物品后直接完成（即时结算） | 放入物品后需点击按钮完成 |
| **奖励形式** | 每完成一个供奉包立即获得奖励 | 每完成一个收集包立即获得奖励 |
| **全局奖励** | 祭坛全部完成解锁特定区域 | 全部房间完成解锁电影院 |
| **前置解锁** | 按顺序解锁后续祭坛 | 所有房间同时可见 |
| **环保主题** | 供奉包含海洋清理产物 | 无环保主题方向 |
| **品质要求** | 部分供奉要求铜/银/锇品质 | 无品质要求（但需数量） |

## 七、设计特点分析

1. **环保叙事游戏化**: 海洋清理、珊瑚修复、垃圾分类回收等环保行为被巧妙地转化为核心玩法循环，而非说教。玩家在"游玩中自然接受环保理念"。

2. **东南亚文化底蕴**: 祭坛称为"Sesajen"（印尼语：祭品）、传送道具叫"Warp Sesajen"、美食包括 Serabi 和 Es Cendol——大量东南亚文化元素嵌入叙事，区别于星露谷的西方乡村风格。

3. **三层叙事叠加**: 个人故事（逃离城市、继承农场）+ 社区故事（城镇复兴）+ 史诗故事（巨人/美人鱼），玩家可在三者之间自由切换节奏。

4. **祭坛供奉的递进驱动**: 与星露谷的"所有收集包同时可见"不同，Coral Island 的四个祭坛需要按顺序解锁——这种设计使玩家始终有明确的短期目标，但也降低了自由选择的空间。

5. **美人鱼线的时机问题**: 海底美人鱼王国线路需要较深的主线推进，导致部分玩家在游戏前期忽略了海底探索的重要性。这一设计在 v1.0 社区中引发了"海底内容应该更早解锁"的讨论。

6. **无最终 BOSS 战**: 与星露谷类似，Coral Island 没有传统意义上的最终 BOSS。游戏的高潮是生态完全恢复、城镇达到 S 级、所有巨人解放——是一种"递进满足"而非"终极对决"的叙事结构。

## 八、与星露谷社区中心对比总结

| 项目 | 珊瑚岛 | 星露谷物语 |
|------|--------|----------|
| 核心NPC | 湖中女神 | 祝尼魔（Junimos） |
| 祭坛/房间数 | 4个祭坛（22个供奉包） | 6个房间（约30个收集包） |
| 物品品质要求 | 有（部分包需要铜/银/锇级） | 无 |
| 全局完成奖励 | 萨凡纳区域 + 神话宠物 | 电影院 |
| 解锁顺序 | 线性递进（完成上位祭坛才能解锁下位） | 非线性的所有房间同时可见 |
| 生态主题 | 强（贯穿全剧情） | 弱（仅作为背景设定） |
| 文化融合 | 东南亚元素 | 欧美乡村/四季元素 |
| 传送系统 | 供奉解锁 Waypoint | 社区中心完成解锁矿车 |
