## 一、系统概述

Coral Island 的战斗探索系统分为三个完全不同的维度：**地面矿洞战斗**（4个元素矿洞，共160层）、**海底非战斗探索**（4个深度区域，潜水清理）、以及**记忆之洞**（Cave of Memories，后期超高难度战斗地牢）。与星露谷的"矿洞一次性战斗"不同，Coral Island 设计了分元素锁链、逐渐提升难度的多矿洞体系，且引入"镐也是武器"的双重用途设计。

## 二、四大元素矿洞 (The Cavern)

### 2.1 矿洞总览

矿洞入口位于森林区域的 The Cavern。四个矿洞按顺序解锁，通关前一个才能进入下一个：

| 矿洞 | 元素 | 层数 | 矿石产出 | 解锁条件 | 怪物等级 |
|:---:|:---:|:---:|---------|:-------:|:-------:|
| 土矿洞 (Earth Mine) | 土 | 40F | 铜矿/土元素晶石 | 初始可用 | ★☆☆☆ |
| 水矿洞 (Water Mine) | 水 | 40F | 银矿/水元素晶石 | 土矿洞通关 | ★★☆☆ |
| 风矿洞 (Wind Mine) | 风 | 40F | 金矿/风元素晶石 | 水矿洞通关 | ★★★☆ |
| 火矿洞 (Fire Mine) | 火 | 40F | 锇矿/火元素晶石 | 风矿洞通关 | ★★★★ |

### 2.2 矿洞内部结构

每个矿洞的结构一致：
- **40层**，每层随机生成
- **每5层设电梯**（5F/10F/15F/20F/25F/30F/35F），解锁后可快速传送
- **底层（40F）** 有诅咒石碑（Cursed Tablet），破坏后解放对应巨人
- 层数越深，怪物越强、掉落越好
- 矿洞内可钓到对应元素的**特殊鱼种**

**与星露谷对比**: 星露谷的矿洞共120层（普通矿洞）+ 骷髅洞穴（无限），电梯每5层保存，但仅有一个矿洞入口，不按元素分洞。Coral Island 的元素分洞设计增加了"攻略感"——每解锁一个新矿洞都像开启新关卡。

### 2.3 矿石与晶石分布

| 矿洞 | 1-10F | 11-20F | 21-30F | 31-40F |
|:---:|:-----|:------|:------|:------|
| 土矿洞 | 铜矿 铜晶石 | 铜矿 铜晶石 宝石 | 铜矿 银矿 土晶石 | 银矿 土晶石 稀有宝石 |
| 水矿洞 | 银矿 银晶石 | 银矿 银晶石 水母 | 银矿 金矿 水晶石 | 金矿 水晶石 宝石 |
| 风矿洞 | 金矿 金晶石 | 金矿 金晶石 幽灵 | 金矿 锇矿 风晶石 | 锇矿 风晶石 宝石 |
| 火矿洞 | 锇矿 火晶石 | 锇矿 火晶石 熔岩 | 锇矿 火晶石 宝石 | 锇矿 火晶石 稀有宝石 |

## 四、战斗系统详解

### 4.1 基本机制

Coral Island 采用即时战斗制（非回合制），玩家使用武器进行攻击：

| 维度 | 说明 |
|------|------|
| 攻击方式 | 鼠标左键/右键→基础攻击；蓄力→重击 |
| 主武器 | 剑（Sword，初始由 Kira 赠送，后可购买/升级） |
| 副武器 | 镐（Pickaxe，兼具采矿和攻击功能） |
| 防御 | 翻滚闪避（空格键）— 无敌帧约0.5s |
| HP系统 | 红色血条，归零则昏迷，损失部分金币和物品 |
| 回复方式 | 食物烹饪（非战斗状态）、温泉（免费） |

### 4.2 武器系统

与星露谷不同，Coral Island 的剑无法自行制作，只能通过以下方式获得：

| 武器名称 | 获取方式 | 攻击力参考 |
|---------|---------|:---------:|
| 基础剑 (Basic Sword) | Kira 赠送（第一次进入矿洞） | 低 |
| 铁匠铺购买的剑 | 铁匠 Sanchez 购买 | 中-高 |
| 特殊武器 | 任务/宝箱奖励 | 不定 |
| 高级剑 | 后期购买或特殊掉落 | 极高 |

> **与星露谷对比**: 星露谷的武器系统更丰富——通过矿洞宝箱获取、冒险家公会购买、或怪物掉落（如银河剑），且剑、匕首、锤等多武器类型。Coral Island 的武器选择相对有限，更强调"战斗是采矿的附属功能"而非独立体系。

### 4.3 镐作为武器的独特设计

Coral Island 中，镐（Pickaxe）不仅用于采矿，也作为有效的副武器使用。镐的升级等级直接影响对怪物的伤害。初期建议"剑+镐"组合使用——剑对付高速怪物，镐对付高甲怪物。

> **设计意图**: 这种"工具=武器"的设计降低了玩家在"战斗装备"和"采集装备"之间的切换成本，但也削弱了战斗深度。玩家只需升级镐就能兼顾采矿能力和战斗输出。

### 4.4 战斗技能与增益

v1.0 版本中无独立的"战斗技能树"，但有如下影响战斗的因素：

| 增益来源 | 效果 |
|---------|------|
| 食物 Buff | 攻击力/防御力/速度临时提升（烹饪所得） |
| 装备附魔（Enchantment） | 剑可附魔增加暴击率/伤害倍率 |
| 温泉休息 | 满HP恢复（免费，无冷却） |
| 睡眠回复 | 每天8AM自动满HP |

## 五、非战斗探索：海底世界

### 5.1 海底区域

海底探索完全不涉及战斗（海洋生物不会主动攻击玩家）。分为4个主要深度区域：

| 深度 | 解锁方式 | 特色产物 | 美景 |
|:---:|---------|---------|------|
| 10m (近海) | 初始可潜 | 海盐、普通贝壳、小型海胆 | 退化珊瑚礁 |
| 20m (浅海) | 清理10m区域 | 中型贝壳、海参、螃蟹 | 部分彩色珊瑚 |
| 40m (中海) | 清理20m区域 | 稀有贝壳、海星、大龙虾 | 恢复的珊瑚群 |
| 50m+ (深海) | 清理40m区域 | 珍珠贝、发光海产 | 美人鱼王国入口 |

### 5.2 海底玩法

| 活动 | 描述 |
|------|------|
| 垃圾清理 | 用镰刀清理海底垃圾（塑料/金属/玻璃），提升清洁度 |
| 海产采集 | 用镰刀收割海藻、海胆、贝类 |
| 海底农场 | 种植水下作物（全年无季节限制） |
| 海底围栏 | 养殖海洋动物（蟹、龙虾等） |
| 珊瑚修复 | 清理污染后珊瑚恢复彩色，可采集珊瑚产物 |

## 六、记忆之洞 (Cave of Memories)

位于萨凡纳（Savannah）区域的深层地牢，是游戏后期的高难度挑战：

| 维度 | 说明 |
|:---:|------|
| 解锁条件 | 稀有祭坛全部供奉完成，解锁萨凡纳 |
| 怪物等级 | 51F+ 出现 Ombuto、Advanced Slime、Amalo 等顶级怪物 |
| 核心价值 | 顶级装备、终极材料、特殊文物 |
| 刷新机制 | 每月刷新（存疑，需进一步验证） |

## 八、设计特点分析

1. **元素锁链挑战设计**: 四个矿洞按土→水→风→火的元素顺序排列，不仅对应了从铜→银→金→锇的矿石升级链条，也象征着玩家从"初级冒险者"到"终极矿工"的成长路径。每个新矿洞都带来新的视觉主题、怪物生态和资源层次。

2. **战斗简化但陪伴性高**: 与星露谷相比，Coral Island 的战斗系统明显简化——无技能树、武器种类少、无复杂攻防公式。但与此同时，"Kira赠送剑"、"铁匠铺买剑"等设定让战斗系统"可用不复杂"，降低了非战斗型玩家的进入门槛。

3. **矿洞的经济分层**: 每个矿洞承担特定的经济角色——土矿洞提供铜和基础宝石（早期发展），水矿洞提供银和水晶（中期），风矿洞提供金和高级宝石，火矿洞提供锇（终极材料）。玩家不可能跳过低级矿洞直接获取高级材料。

4. **海底的非战斗设计**: 将海底区域设置为纯探索/采集区（无战斗），是对"战斗疲劳"的一种调剂——玩家可以在战斗间隙潜入海底，进行放松型收集，形成"战斗→采集→加工→战斗"的节奏循环。

5. **怪物等级与矿洞的对应问题**: 部分玩家反馈风矿洞到火矿洞的难度跳跃较大（Sphekter/Amalo 速度快而难击中），而缺少战斗技能树意味着"如果操作不行，升级装备可能不足以弥补"——这在操作型玩家和非操作型玩家之间产生了一定分歧。
