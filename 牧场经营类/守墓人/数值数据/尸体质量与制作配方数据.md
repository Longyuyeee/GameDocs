## 二、墓地/教堂评分

### 2.1 评分公式

```
坟墓评分 = 尸体白骷髅 + 墓碑加成 + 围栏加成 (上限=白骷髅数)
总墓地 = 所有坟墓评分之和 + 教堂围栏 + 花 + 灯 + 骨灰龛
```

## 五、僵尸自动化(Breaking Dead DLC)

- 创建: 1僵尸汁+10信仰+1尸体
- 效率=尸体白骷髅数(基础最高12/BSS DLC最高26=65%效率)
- 可用工作站: 伐木/采矿/造纸/制陶/珠宝/写书/石工/农耕/运输
- 限制: 不产生科技点/不能炼金/不能设定阈值生产
