# 经济与贸易系统 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (2018年)
> 数据来源: Steam Community, TrueAchievements

## 1. 货币体系

| 货币 | 获取 | 用途 |
|------|------|------|
| 铜币(Copper) | 出售物品/布道/埋葬证明 | 基础购买 |
| 银币(Silver) | 大量铜币兑换(100铜=1银) | 贸易许可证/重要物品 |
| **金币(Gold)** | 大量银币兑换(100银=1金) | **Aristocrat Papers(12金!)** |
| 信仰(Faith) | 布道 | 研究/科技/僵尸复活 |
| 科学(Science) | 分解纸张 | 研究(与信仰共同消耗) |

## 2. 收入来源

| 来源 | 效率 | 说明 |
|------|:--:|------|
| 布道(Sermons) | ★★★★★ | 每Pride日→金币+信仰(墓地评分越高越多) |
| 埋葬证明(Burial Certificate) | ★★★★ | 每埋葬1具尸体→1.5银(Horadric处出售) |
| 蔬菜箱(Vegetable Crate) | ★★★★ | 贸易许可证后→商人→批量蔬菜出口 |
| 出售铁钉(Nails) | ★★★ | 僵尸采矿→自动生产钉→贸易站出售 |
| 出售煤炭(Coal) | ★★★ | 采石场→铁匠处出售 |
| 酒馆经营(Stranger Sins DLC) | ★★★★ | 被动收入→卖酒/啤酒/麦酒 |
| 任务奖励 | ★★ | 一次性 |

## 3. 主要消费出口

| 消费项 | 费用 |
|------|:--:|
| 种子/材料 | 铜-银 |
| 炼金配方(Clotho) | 5银/个 |
| 蓝书(Astrologer) | 5银/25蓝点 |
| 贸易许可证 | **50银** |
| Aristocrat Papers | **12金币(1,200银)!** |
| 传送石(Teleportstone) | 2银(永久使用!最划算) |

## 4. 通货膨胀与货币兑换

```
100铜(Copper) = 1银(Silver)
100银(Silver) = 1金(Gold)
12金(Gold) = 1,200银 = 120,000铜
```

- 12金=Aristocrat Papers=游戏最大的金币门槛

## 5. 贸易许可证后经济

- 50银→贸易许可证
- 商人每周出现→可批量出售蔬菜/商品
- 蔬菜箱经济→从"手到口"到"规模经济"

## 6. 设计特点

1. **布道=核心收入**: 不是"种地卖钱"而是"布道收捐款"——宗教作为经济引擎
2. **12金=终局检查**: Aristocrat Papers=1,200银——需要建立完整的收入流水线
3. **传送石=最划算投资**: 2银→永久使用→节省大量往返时间
4. **三币制=中世界感**: 铜→银→金的递进兑换→符合历史感
