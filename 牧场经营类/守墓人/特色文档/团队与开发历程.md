## 1. 开发团队

### Lazy Bear Games

Lazy Bear Games 成立于2014年，最初名为 Game Jam Studio，2015年4月更名为 Lazy Bear Games。工作室最初位于俄罗斯圣彼得堡，后迁至立陶宛维尔纽斯。团队以制作精良的像素风格模拟经营游戏而闻名。

| 角色 | 人员 |
|------|------|
| **联合创始人** | 两位核心创始人（具体姓名未完全公开） |
| **开发团队** | 约5-7人（《守墓人》开发期间） |

### 前作基础

在《守墓人》之前，Lazy Bear Games 于2016年推出了 **Punch Club**（拳击俱乐部）——一款像素风格的拳击手管理模拟游戏。Punch Club在商业上取得了相当的成功，由 tinyBuild 发行，Steam销量超过50万份。

Punch Club的成功为Lazy Bear Games带来了：
- 足够的资金储备，使团队能承担更大规模的项目
- 与发行商 tinyBuild 的稳定合作关系
- 像素风格模拟游戏开发的宝贵经验

## 2. 开发历程

### 概念起源

《守墓人》的创意来自于一个简单的想法："如果玩家不是经营农场，而是经营墓地，会怎样？" 开发团队希望创造一种颠覆传统农场模拟游戏期待的体验——充满黑色幽默、道德灰色地带和中世纪黑暗氛围。

游戏被官方描述为 **"the most inaccurate medieval cemetery management sim"**（最不准确的中世纪墓园管理模拟），这一定位准确传达了游戏的幽默和自嘲风格。

### 时间线

| 时间 | 里程碑 |
|------|--------|
| 2016年 | Punch Club 发售后，新项目立项 |
| 2016-2017年 | 核心开发期，Unity引擎搭建 |
| 2018年8月15日 | **PC版正式发售**（Steam/GOG） |
| 2018年 | Xbox One版发售 |
| 2019年 | Nintendo Switch / PS4 版发售 |
| 2019年 | DLC1: Breaking Dead（僵尸自动化） |
| 2020年 | DLC2: Stranger Sins（酒馆经营） |
| 2021年 | DLC3+4: Game of Crones / Better Save Soul |
| 2026年（预计） | **守墓人2** 发售 |

## 3. 核心设计——黑暗农场模拟

### 颠覆性主题

《守墓人》的核心创新在于它将传统农场模拟游戏的"积极建设"主题完全颠倒：

| 传统农场模拟 | 守墓人 |
|-------------|--------|
| 种植农作物 | **处理尸体** |
| 养殖动物 | **解剖并出售器官** |
| 社交交友 | **操纵村民** |
| 美化城镇 | **经营墓园** |
| 拯救世界 | **追求个人利益** |

### 黑色幽默与深度系统

游戏的黑暗幽默不仅体现在主题上，更渗透在系统的交互中：
- 教会捐赠系统与道德选择的讽刺
- 村民任务的荒谬和自私动机
- 对中世纪历史和宗教的戏谑性解读

尽管主题黑暗，游戏在系统深度上毫不含糊——包含超过10种技能树、复杂的制造链（从尸体处理到布匹生产的完整流程）、以及多个经济系统（墓地、教堂、酒馆、农场）的联动。

### tinyBuild 的发行角色

**tinyBuild** 是美国独立游戏发行商，以《Hello Neighbor》《SpeedRunners》《Punch Club》等热门独立游戏闻名。tinyBuild 在《守墓人》的成功中扮演了关键角色：
- 提供发行资金和开发支持
- 全球多平台发行策略
- 市场营销和社区运营

## 4. DLC 内容扩展

《守墓人》在正式发售后陆续推出了4款主要DLC，持续拓展游戏内容：

| DLC | 发售年 | 核心内容 |
|-----|--------|---------|
| **Breaking Dead** | 2019 | 僵尸自动化系统——利用僵尸进行采集和制造 |
| **Stranger Sins** | 2020 | 酒馆经营 + 主角来到此地的背景故事 |
| **Game of Crones** | 2020 | 全新区域和任务线，三位女巫的故事 |
| **Better Save Soul** | 2021 | 灵魂拯救机制，道德系统扩展 |

每个DLC都引入了全新的机制而非仅仅是内容追加，体现了开发团队对游戏深度的持续投入。

### 惊人的ROI

### 免费赠送奇迹

## 6. 续作展望

### 守墓人2 (2026)

2026年公布的《守墓人2》延续了前作的黑色幽默和中世纪墓园经营主题，但采用了全新的绘画风格和技术引擎。tinyBuild CEO Alex Nichiporchik 透露：

> "《守墓人》是我们最成功的独立游戏之一，它的粉丝群极其忠实。续作在公布之前就已经积累了45万个愿望单，这证明了Lazy Bear Games创造的世界的吸引力。"

### Swen Vincke 式的黑暗幽默

《守墓人》在社区中常与 **Swen Vincke**（《神界：原罪》系列、《博德之门3》开发商 Larian Studios 的CEO）的风格相提并论——两者都擅长在看似严肃的主题中植入令人会心一笑的黑色幽默。
