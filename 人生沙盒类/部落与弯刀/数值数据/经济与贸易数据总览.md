### 1.3 最优跑商路线

**初级阶段(资本<5000乌塔):**
```
红石城(买赫炎矿石) -> 驼铃集市/黄金集市(卖出) -> 买玉器/丝线 -> 三石堡/黑崖(卖出)
```
利润期望: 单趟 1500~3000 乌塔

**中级阶段(资本5000~20000乌塔):**
```
驼铃集市(买玉器+丝线+书卷) -> 三石堡/陶山镇(卖部分+买太阳木) -> 黑崖(卖出+买染料/角)
-> 双月山谷(卖出+买金器/银器) -> 冰谷(卖出+买钻石) -> 返回沙漠循环
```
利润期望: 单趟 8000~15000 乌塔

**高级阶段(资本>20000乌塔,黄金三角路线):**
```
沙漠(买玉器+丝线) -> 黑崖(卖出+买特产) -> 双月山谷(卖出+买金器) -> 循环
```
利润期望: 单趟 20000~50000 乌塔
