## 二、通用技能伤害公式

### 2.1 最终伤害结算公式

```
基础伤害 x (1 - 物理/巫术减伤比例) x (1 - 近战/远程减免比例) = 实际伤害
```

**实例计算:**
- 攻击方: 造成2000点物理伤害
- 防御方: 60%物理抗性 + 30%远程减免
- 实际伤害 = 2000 x (1 - 0.6) x (1 - 0.3) = **560点** (仅原始伤害的28%)

### 2.2 伤害类型体系

| 伤害类型 | 受物理抗性影响 | 受巫术抗性影响 | 受近战减免影响 | 受远程减免影响 | 示例 |
|:-------:|:------------:|:------------:|:------------:|:------------:|------|
| 物理伤害 | 是 | 否 | 是 | 否 | 剑术、弓矢 |
| 巫术伤害 | 否 | 是 | 否 | 是 | 魔弹、元素 |
| 范围伤害 | 是/否取决于类型 | 是/否取决于类型 | **否** | **否** | 爆炸、震荡波 |
| 真实伤害 | **否** | **否** | **否** | **否** | 燃烧、中毒、流血 |

### 2.3 抗性上限

| 抗性类型 | 上限值 | 说明 |
|:-------:|:----:|------|
| 物理抗性 | ~90% | 装备+技能叠加，上限约90% |
| 巫术抗性 | ~90% | 装备+技能叠加，上限约90% |
| 近战减免 | ~80% | 实测最高80%，全額生效 |
| 远程减免 | ~80% | 实测最高80%，全額生效 |

### 2.4 通用技能伤害系数模式

绝大多数技能的伤害公式遵循统一模板：

```
伤害 = 基础固定值 + 等级成长系数 x 技能等级 + 攻击系数 x 对应攻击力
```

| 系数类型 | 典型范围 | 说明 |
|---------|:-------:|------|
| 基础固定值 | 10~100 | 技能等级为1时的基础伤害 |
| 等级成长系数 | 10~100/级 | 每提升1级技能增加的伤害 |
| 攻击系数(物攻) | 0.2~1.5 | 物理攻击力乘数 |
| 攻击系数(巫攻) | 0.3~2.0 | 巫术攻击力乘数 |
