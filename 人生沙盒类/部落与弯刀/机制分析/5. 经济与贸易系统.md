## 一、经济系统总览

部落与弯刀的经济系统由**个人经济**和**领地经济**两个层次构成：

| 经济层次 | 核心内容 | 适用阶段 |
|---------|---------|---------|
| **个人经济** | 跑商/任务/战斗掉落/装备交易 | 前期-中期 |
| **领地经济** | 税收/资源产出/城镇建设/军队维护 | 中期-后期 |

---

## 二、跑商系统

### 跑商机制

跑商是部落与弯刀前期积累财富的主要手段。核心机制：

> **离原产地越远，商品售价越高**（参考《大航海时代2》的设计理念）

| 因素 | 对价格的影响 |
|------|:-----------:|
| 距离原产地 | 正相关——越远售价越高 |
| 区域物价差异 | 不同城镇的基础物价不同 |
| 部落关系 | 友好部落商品价格优惠 |
| 通商许可 | 非部落成员需额外成本 |

### 商品类型（待补充完整清单）

| 商品类型 | 产地 | 高价销售区域 |
|---------|------|-------------|
| 铁石制品 | 黑崖（蛮牛领地） | 雪狼/雄鹰领地 |
| 木材制品 | 大雪山（雪狼领地） | 沙漠区域 |
| 玉石制品 | 待补充 | 待补充 |
| 食物/药品 | 红石谷（雄鹰领地） | 沙漠/雪山区域 |

### 通商许可证

| 状态 | 效果 | 获取方式 |
|------|------|---------|
| 持有许可证 | 可在对应部落领地正常交易 | 部落好感达标后购买/任务获得 |
| 无许可证 | 交易受限或被巡逻队攻击 | — |
| 部落成员 | 可自由交易 | 加入对应部落 |

通商许可证系统施加了一个**贸易门槛**——在没有建立外交关系的情况下，你不能自由地在敌对部落的领地进行贸易。这种设计模拟了真实的中世纪/古代贸易关系：**商业活动与政治/外交关系深度绑定**。

## 三、城镇经济系统

### 占领城镇后的经济收益

玩家占领城镇后，可以获得以下经济收益：

| 收益类型 | 获取方式 | 影响因素 |
|---------|---------|---------|
| **金币税收** | 自动收取 | 城镇规模、税率设置、民心 |
| **资源产出** | 根据建筑类型产出 | 已建设施类型和等级 |
| **贸易收益** | 城镇商店交易 | 物价水平、供应量 |

### 城镇资源产出

| 资源 | 产出建筑 | 用途 |
|------|---------|------|
| **铁石** | 矿场/铁匠铺 | 兵种升级、装备锻造 |
| **木材** | 伐木场 | 城镇建设、部分兵种升级 |
| **玉石** | 玉石矿场 | 高级兵种升级、稀有物品 |
| **金币** | 市场/税收 | 通用货币 |

### 税收系统

| 税率设定 | 经济收益 | 民心影响 | 适用场景 |
|:--------:|:-------:|:--------:|---------|
| 低税率 | 低 | 正增长 | 和平发展期 |
| 中税率 | 中 | 稳定 | 常规运营 |
| 高税率 | 高 | 负增长（叛乱风险） | 急需资金/战时经济 |
| 极高税率 | 短期极高 | 快速下降 | 不建议长期使用 |

## 四、封地管理系统

### 封地建设

获得封地后，玩家可以投资建设各类建筑来提升城镇功能和产出：

| 建筑类型 | 功能 | 升级效果 |
|---------|------|---------|
| **图书馆** | 解锁领主技能等级上限 | Lv.1→2→3，每级提升技能上限 |
| **兵营** | 招募/升级兵种 | 高等级兵营解锁高阶兵种 |
| **巫师集会所** | 招募巫师 | 解锁法系部队招募 |
| **市场** | 贸易/税收 | 提升贸易效率和税收 |
| **城墙/防御** | 城镇防御 | 提升城防耐久度 |

### 封地建设的经济循环

```
税收收入 → 投资建设 → 建筑升级 → 产出提升 → 税收增加
```

这是一个经典的**正反馈经济循环**——但与所有正反馈循环一样，它需要一个**启动资本**。对于领主职业来说，开局就拥有城镇意味着启动资本天然存在；对于非领主职业来说，启动这个循环需要先通过跑商/战斗积累第一桶金。

### 封地维护成本

| 维护项 | 消耗 | 说明 |
|-------|------|------|
| 驻军军费 | 金币/天 | 城镇守军的维持费用 |
| 建筑维护 | 金币/资源 | 建筑破损后的修复 |
| 官员薪水 | 金币/天 | 封地管理人员的薪酬 |
| 定期事件 | 可变 | 天灾/叛乱/瘟疫等消耗 |

## 五、装备交易与经济

### 装备获取途径

| 途径 | 成本 | 品质范围 |
|------|:----:|:--------:|
| 城镇商店购买 | 金币 | 普通-稀有 |
| 战斗掉落 | 免费（战斗成本） | 普通-传说 |
| 任务奖励 | 免费 | 特定品质 |
| SL刷装备 | 时间成本 | 稀有-传说 |

### SL刷装备机制

玩家可以通过存档/读档（SL）在特定商店或BOSS掉落中反复尝试获取稀有装备：

| 知名稀有装备 | 类型 | 获取方式 |
|------------|:----:|---------|
| **初雪魔戒** | 饰品 | 特定BOSS掉落/SL刷取 |
| 其他稀有装备 | 待补充 | 待补充 |

SL机制的存在引发了关于"设计意图"的讨论——它究竟是一个漏洞，还是一种隐性的"玩家自助"机制？

## 六、物价系统

### 区域物价差异

| 区域 | 物价水平 | 经济特征 |
|------|:-------:|---------|
| 红石谷（雄鹰） | 中等 | 物产丰富，物价均衡 |
| 风哭岩（毒蝎） | 较高 | 沙漠资源稀缺，进口依赖度高 |
| 西多喀那沙漠（野马） | 较高 | 游牧经济，物资短缺 |
| 大雪山（雪狼） | 高 | 极端气候，农业几乎为零 |
| 黑崖（蛮牛） | 中等 | 矿产资源丰富，食物依赖进口 |

### 跑商利润计算

```
跑商利润 = 销售价 - 进货价 - 运输成本 - 风险成本（被劫掠概率）
```

| 商品路线 | 进货价 | 销售价 | 预估利润率 |
|---------|:-----:|:-----:|:---------:|
| 黑崖铁石→大雪山 | 低 | 高 | 高（但路程危险） |
| 红石谷粮食→风哭岩 | 低 | 中高 | 中（路程较安全） |
| 大雪山木材→沙漠 | 低 | 高 | 高（路程较远） |
| 其他路线 | 待补充 | 待补充 | 待补充 |

## 七、与骑马与砍杀2的经济系统对比

| 对比维度 | 部落与弯刀 | 骑马与砍杀2 |
|----------|-----------|-------------|
| **跑商机制** | 固定区域差价，越远越贵 | 动态供需驱动，实时价格波动 |
| **物价驱动** | 区域预设差价 + 部落关系 | 供需动态模型 + 战争消耗 |
| **通商许可** | 需要许可证（非部落成员） | 无此概念，但存在关税 |
| **领地经济** | 建筑升级 + 资源产出 | 村庄/城堡/城镇各有特色产出 |
| **工厂/企业** | 无此系统 | 可投资开设工厂（酿酒/制革等） |
| **商队系统** | AI自主跑商 | 玩家可派遣商队 + AI商队 |
| **经济深度** | 中——作为辅助系统 | 高——可作为核心玩法之一 |
| **SL刷装备** | 常见（玩家自主选择） | 不可（自动存档） |

**核心差异**：
- 骑砍2的经济系统是一个**动态供需模型**——战争会影响某区域的食物价格，商队被劫会影响商品供应。理论上你可以作为"纯粹的经济玩家"不参与战斗，专注于跑商和产业投资。
- 部落与弯刀的经济系统相对简化——区域差价是固定的，跑商行为基于"已知的利润地图"。经济更多是**支持战斗和征服的辅助手段**，而非独立的玩法维度。
- 两者的差异本质上是**模拟取向 vs 游戏取向**的差异——骑砍2试图模拟一个"真实经济"，部落与弯刀则是设计了一个"够用的经济系统"。

## 八、设计特点分析

**"大航海时代式"的跑商**: "离原产地越远越贵"的定价逻辑直接继承了《大航海时代2》的经典设计。这种简单直观的利润模型让玩家不需要复杂的计算就能理解跑商的本质——"低价买入，长途运输，高价卖出"。其设计优势在于：
1. **直觉驱动**——玩家不需要查攻略就知道"把黑崖的铁石运到大雪山能赚钱"
2. **地图学习的副产品**——跑商过程本身就是玩家熟悉世界地图的过程
3. **冒险动机**——高利润路线往往经过危险区域，制造了"高风险高回报"的决策点

**通商许可证 = 政治经济学的入门设计**: 将"贸易权"与"政治关系"绑定是非同寻常的游戏设计——大多数游戏允许你在任何地方自由买卖（只有价格差异）。通商许可证迫使玩家思考：**我想和哪里做生意 → 我需要和那里的部落搞好关系 → 我该如何提升好感度**。这一设计将"经济活动"与"社交活动"两个系统连接起来：

```
贸易需求 → 外交投入 → 关系提升 → 获得许可 → 扩大贸易 → 更多收益 → 更强影响力
```

**领地经济的"循环启动"问题**: 领主职业开局拥有城镇，可以直接进入"税收→建设→产出→更多税收"的正循环。而非领主职业需要通过跑商/战斗积累到足够资本，才能"买下第一座城镇"进入这个循环。这意味着：
- 领主职业的**核心玩法是"管理"**——如何优化已有的资源
- 非领主职业的**核心玩法是"积累"**——如何从零到一

这种**经济层次的设计分化**与职业系统形成了良好的互补。

**经济系统的"够用即好"哲学**: 与骑砍2的动态经济模型不同，部落与弯刀的经济系统没有追求"真实感"——它选择了**简单但有效**的方案：
- 固定差价 → 玩家可以规划"最优跑商路线"
- 四种资源 → 清晰简洁，不需要复杂的产业链
- 建筑升级 → 直观的正反馈循环

这种"减法设计"的合理性在于：**部落与弯刀的核心体验是"英雄冒险 + 部队征战"，经济系统只需要"够用"即可**——它不需要像骑砍2那样成为一个可独立玩耍的经济模拟器。

**SL刷装备的灰色地带**: SL刷稀有装备的行为在玩家社群中非常普遍，但开发团队没有选择"修复"它（比如像骑砍2那样强制自动存档）。这可以被理解为一种**默许的玩家自由**——"如果你愿意花时间SL，那是你的选择"。这种设计态度与Elona的"无限重掷角色属性"有相似的哲学基础：**规范玩家的行为没有意义，给予自由才是沙盒的终极目标**。

---

> 跨品类对比: 与太吾绘卷的"经济与资源系统"对比——太吾绘卷的经济系统围绕"资源采集+制作+交易"展开，更加"生存导向"（需要关注食物/木材/药材等生存资源）。部落与弯刀的经济系统则更加"军事导向"——金币用于招募军队，铁石/木材/玉石用于兵种升级。太吾绘卷的经济是"让自己活下去"的手段，部落与弯刀的经济是"让自己变强"的手段。
