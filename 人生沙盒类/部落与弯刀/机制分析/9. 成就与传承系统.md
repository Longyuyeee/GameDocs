# 9. 成就与传承系统

> 游戏版本: 部落与弯刀 v1.0 (PC/Steam, 汉家松鼠, 2020+)
> 数据来源: [灰机Wiki](https://wandao.huijiwiki.com/wiki/%E9%83%A8%E8%90%BD%E4%B8%8E%E5%BC%AF%E5%88%80), [Steam](https://store.steampowered.com/app/1094520/), [GameRes](https://www.gameres.com/910339.html), [游戏陀螺](https://youxituoluo.com/)

## 一、Steam 成就系统

| 成就类别 | 示例成就 | 解锁条件 | 备注 |
|:--------:|:--------|:--------|:----|
| 职业通关 | "心灵巫师通关" / "剑豪通关"等 | 使用对应职业完成统一 | 每种职业有独立成就 |
| 统一大漠 | "大漠霸主" | 统一全部势力版图 | 核心通关成就 |
| 全英雄招募 | "百川归海" | 招募全部可招募英雄 | 收集向成就 |
| 特定挑战 | "以少胜多" / "赤手空拳"等 | 在特定限制条件下获胜 | 挑战向成就 |
| 流程成就 | "初出茅庐" / "小有名气"等 | 游戏中达到特定里程碑 | 成长向成就 |
| 隐藏成就 | 待补充 | 待补充 | 探索向成就 |

> 注：详细的Steam成就列表（成就ID、解锁百分比、隐藏成就条件）待补充。

### 1.1 成就对游戏进程的影响

| 影响维度 | 说明 |
|:--------:|:----|
| 传承点数计算 | 已解锁的成就在周目结算时影响传承点数总量 |
| 解锁提示 | 成就解锁时游戏内弹出提示 |
| 额外奖励 | 部分成就解锁后可能解锁游戏内特殊内容（待确认） |

> 设计特点分析：
> 部落与弯刀的成就系统采用了"多层次"结构：基础流程成就面向所有玩家（通关就有基础成就），职业成就驱动多周目（换职业再打一次），挑战成就面向硬核玩家（特定限制条件）。这种分层设计让各类玩家都能在成就中找到目标——休闲玩家追求通关成就，收集癖玩家追求全英雄，挑战玩家追求极限。成就与传承点数的绑定，则让成就收集有了实质性的游戏内价值（而非仅仅是Steam展示）。

## 二、周目结算与传承点数

### 2.1 结算流程

| 步骤 | 操作 | 影响因素 | 产出 |
|:----:|:----|:--------:|:----:|
| 1. 通关判定 | 达成统一/特定结局条件 | 各势力状态、任务完成度 | 通关判定结果 |
| 2. 数据统计 | 统计本局数据 | 击杀数/城镇数/招募英雄数/任务完成数等 | 统计数据汇总 |
| 3. 成就核算 | 核算已解锁成就 | Steam成就解锁情况 | 成就加成 |
| 4. 点数计算 | 综合上述数据→计算传承点数 | 统计数据×权重+成就加成 | 最终传承点数 |
| 5. 点数结算 | 显示本局所得+累积总数 | — | 可用于下周目的传承点数 |

### 2.2 传承点数计算公式（推测）

> 待补充：精准的传承点数计算公式需从游戏代码或官方攻略确认。以下为基于玩家经验的推测结构：

```
传承点数 ≈ 基础值(通关难度调整)
  + 击杀BOSS数量 × 系数
  + 招募英雄数量 × 系数
  + 占领城镇数量 × 系数
  + 完成任务数量 × 系数
  + 成就解锁加成
  + 剩余天数加成（速通奖励，待确认）
```

> 设计特点分析：
> 周目结算的设计巧妙地将"多周目"从"被迫重新开始"转化为"有回报的新征程"。结算时的各项统计让玩家直观看到自己这个周目的"成绩单"——这本身就提供了一种成就感。而传承点数作为"货币"，将这份成就感转化为下周目的"起始资本"，形成了一个良性的循环：完成一个周目→获得点数→下周目更强→完成更多内容→获得更多点数。

## 三、传承点数用途

| 用途 | 消耗点数 | 效果 | 解锁限制 |
|:----:|:--------:|:----|:--------:|
| 解锁新职业 | 50~120 点不等 | 解锁新的初始职业供下周目选择 | 无名剑主需120点（最高） |
| 提升初始属性 | 10~50 点/项 | 下周目起始属性更高 | 单属性有上限 |
| 解锁初始装备 | 20~80 点/件 | 下周目携带特定装备开局 | 部分装备需条件 |
| 解锁初始随从 | 30~60 点/位 | 下周目开局自带一名英雄随从 | 随从类型有限制 |
| 解锁特殊天赋 | 40~100 点/项 | 下周目获得特殊天赋加成 | 天赋树逐步解锁 |
| 初始金钱 | 5~20 点/档 | 下周目起始金钱更多 | 多档位 |

### 3.1 关键传承项目一览

| 项目名称 | 消耗点数 | 效果描述 | 优先级评价 |
|:--------:|:--------:|:--------|:---------:|
| 无名剑主职业 | 120 | 可学习其他所有职业技能 | ★★★★★（终极目标） |
| 初始等级提升 | 50 | 开局等级+5，技能点+5 | ★★★★☆ |
| 传说武器开局 | 80 | 携带一把传说级武器 | ★★★★☆ |
| 英雄随从开局 | 60 | 开局自带一名已招募英雄 | ★★★☆☆ |
| 部落好感初始 | 40 | 开局与某部落好感+30 | ★★★☆☆ |
| 金钱×3开局 | 30 | 开局金钱为默认3倍 | ★★☆☆☆ |

> 设计特点分析：
> 传承点数可解锁的项目的设计遵循了"重要 > 便利"的优先级排序：核心产出是"新职业"（尤其是无名剑主）和"初始属性/装备"——这些直接影响游戏玩法和强度；次要产出是"金钱/好感"——这些只是降低开局难度但不改变玩法。这种设计确保了传承点数对游戏体验的实质性影响——你积累的点数不是买一堆锦上添花的装饰品，而是解锁真正改变玩法的新内容。

## 四、多周目设计

### 4.1 解锁路径建议（从开发者视角）

| 周目 | 建议职业 | 重点目标 | 传承点投入 | 体验目标 |
|:----:|:--------:|:--------|:----------:|:--------|
| 一周目 | 心灵巫师 / 剑豪 | 熟悉机制、探索世界、完成主线 | 不投入（先积累） | 了解游戏基础系统 |
| 二周目 | 领主 / 圣职者 | 体验不同职业玩法、尝试其他势力路线 | 初始属性+装备 | 感受不同职业差异化 |
| 三周目 | 工程师 / 逐光者 | 挑战更高难度、尝试自立 | 随从+天赋 | 深度策略体验 |
| 四周目+ | 无名剑主 | 自由搭配技能、挑战英魂副本 | 全部可用点数 | 自由+挑战 |

### 4.2 多周目价值分析

| 价值维度 | 一周目 | 多周目 |
|:--------:|:------:|:------:|
| 世界认知 | 不知道势力分布→逐步探索 | 了解地图→策略性选择初始区域 |
| 职业理解 | 未体验过→边玩边学 | 深入了解→优化技能搭配 |
| 策略深度 | 保守玩法→避免犯错 | 激进玩法→追求效率/速通 |
| 挑战等级 | 默认难度 | 可通过传承点数降低/提高难度 |
| 内容覆盖 | 只能体验一条势力路线 | 多条势力路线+不同结局 |

> 设计特点分析：
> 部落与弯刀的多周目设计不是简单的"二周目继承等级"——而是通过解锁新职业来"解锁新玩法"。每个职业有独立的技能树和玩法逻辑（心灵巫师=法术远程、剑豪=近战爆发、领主=部队指挥），实际上构成了一种"全新游戏"的体验。加上规则驱动的世界演化保证每周目势力走向不同，多周目体验的差异化程度远超传统"多周目=更高难度+更强敌人"的设计模式。

## 五、无名剑主——传承的终极目标

| 属性 | 说明 |
|:----:|:----|
| 解锁条件 | 消耗 120 传承点数 |
| 定位 | 传承职业、终极自由角色 |
| 核心特色 | 可以学习其他所有职业的技能 |
| 初始强度 | 较弱（无专属技能开局） |
| 成长曲线 | 前期弱→中期收集技能→后期极强（全技能） |
| 适合玩家 | 已体验3+职业的老手、追求自由build的玩家 |

### 5.1 无名剑主的技能池获取

| 技能来源 | 获取方式 | 数量估算 |
|:--------:|:--------:|:--------:|
| 心灵巫师技能 | 招募心灵巫师英雄/完成特定任务 | 6~8 个技能 |
| 剑豪技能 | 招募剑豪英雄/完成任务 | 6~8 个技能 |
| 领主技能 | 加入势力/参与攻城 | 6~8 个技能 |
| 其他职业 | 各职业对应方式 | 待补充 |

> 设计特点分析：
> 无名剑主是传承系统设计中最精巧的一环——它同时实现了三个目标：第一，给追求自由度的玩家一个"终极目标"（收集所有技能）；第二，给多周目玩家一个"奖励"（120点的投入有回报）；第三，给游戏增加一个"隐藏内容"（需要用多个周目解锁才能玩到）。无名剑主的存在让"传承点数积累"这条主线有了终点——它不是无止境的投入，而是一个明确的、可达成的目标。

### 6.1 MOD 对传承系统的影响

| 维度 | 说明 |
|:----:|:----|
| 职业扩展 | MOD新增职业→理论上无名剑主可学习（需MOD兼容支持） |
| 传承点数扩展 | 部分MOD可能增加新的传承兑换项目 |
| 多周目内容 | MOD剧本/地图→增加了多周目的内容量 |
| 平衡性影响 | 强力MOD可能降低传承系统的重要性 |

> 设计特点分析：
> 部落与弯刀的MOD生态扩展了传承系统的外围——原版传承系统提供的是"职业×周目"的循环，而MOD社区则在这个循环中不断注入新内容（新职业、新剧本、新兵种）。这种"官方搭台+社区唱戏"的模式，使得传承系统的重玩价值不再局限于官方内容——即使你解锁了无名剑主、体验了所有职业，MOD社区仍然能源源不断地给你新东西玩。

## 七、跨游戏对比

### 7.1 与【太吾绘卷】传承系统对比

| 对比维度 | 部落与弯刀 | 太吾绘卷 |
|:--------:|:----------:|:--------:|
| 传承机制 | 通关→结算点数→下周目使用（周目叠加） | 死后传剑→下一代继承（世代接力） |
| 传承对象 | 同一个人在不同世界线 | 不同世代的不同角色 |
| 传承感来源 | "我变强了"（点数和属性叠加） | "家族延续"（后代继承遗志） |
| 失败惩罚 | 无（可以重新开始） | 角色死亡→进度损失 |
| 传承方式 | 主动选择（消费点数） | 被动触发（角色死亡时） |
| 叙事意义 | 多周目世界线 | 世代家族史诗 |

| 两种传承设计哲学 |
|:-----------------:|
| **部落与弯刀 = "轮回传承"**：同一灵魂在不同的平行世界中轮回，每次轮回都保留之前的经验和力量。更像"转世修行"——每一世都在积累，越轮回越强大。 |
| **太吾绘卷 = "血脉传承"**：先辈的意志通过血脉传递给后人，后人站在前人的肩膀上继续前行。每一代都有不同的命运，传承的是"可能性"而非"确定性"。 |

> 跨游戏对比分析：
> 两种传承设计各有适用场景。部落与弯刀的"周目叠加"更适合"以玩法为核心"的游戏——玩家想体验的是不同职业+不同策略下的世界线走向，传承点数保证了玩家不会永远从零开始。太吾绘卷的"世代接力"更适合"以叙事为核心"的游戏——玩家想体验的是一个家族的兴衰史，传承不仅传递能力，更传递情感（对上一代角色的怀念）。
>
> 部落与弯刀的传承系统本质上是一个"成长系统"——你把多个周目的成长累积在一起，每一个周目都比上一个周目更强大。太吾绘卷的传承系统本质上是一个"延续系统"——你不会越来越强，但你可以让后代替你走得更远。

### 7.2 与【黑帝斯】（Hades）的多周目传承对比

| 对比维度 | 部落与弯刀 | 黑帝斯 |
|:--------:|:----------:|:-----:|
| 传承形式 | 传承点数→解锁新职业/属性 | 黑暗宝石→解锁天赋/武器 |
| 周目长度 | 10~50小时/周目 | 30~60分钟/局 |
| 周目重置 | 完全重置（世界重新生成） | 半重置（剧情推进保留） |
| 叙事绑定 | 弱（传承与叙事关系松散） | 强（每局死亡都有剧情对话） |
| 失败惩罚 | 无（可选择继续游玩或重新开始） | 需要重新开始（死亡重置） |

## 八、设计特点总结

1. **"成就→点数→解锁→更强→更多成就"的正循环**：传承系统的核心是构建了一个自我强化的飞轮——玩家的每一次努力（成就、通关）都转化为下一次起跑的优势，形成了一个"越玩越强、越强越想玩"的心理模型。

2. **无名剑主作为终极目标**：120点的门槛让无名剑主成为了一个"需要玩家证明自己"才能解锁的内容。它不仅是一个职业，更是一种"你已经玩懂了游戏"的身份象征。

3. **多周目的差异化体验**：不是简单的"数值膨胀"，而是通过不同职业+不同势力路线+规则驱动世界的组合，让每个周目都有全新的体验。传承点数只是"助推器"——真正驱动多周目的是每周目世界和玩法的差异。

4. **MOD生态的无限扩展性**：Steam创意工坊使传承系统的内容上限从"官方内容数量"扩展到了"社区创造力"。这是国产游戏中较为成功的MOD运营案例——不是简单的"开放编辑器"，而是形成了活跃的MOD创作者+使用者社区。

5. **与太吾绘卷的传承设计哲学差异**：部落与弯刀的"轮回传承"鼓励玩家在同一角色框架下不断优化和挑战；太吾绘卷的"血脉传承"鼓励玩家在不同的角色命运中感受家族史诗。两种设计哲学对应了两种不同的"人生沙盒"叙事——个人英雄史诗 vs 世代家族传奇。
