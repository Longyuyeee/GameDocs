## 一、战斗系统总览

部落与弯刀的战斗是**2.5D俯视角的实时战术战斗（RTS式）**——玩家控制英雄单位，同时指挥部队进行协同作战。

### 战斗基本要素

| 要素 | 说明 |
|------|------|
| **视角** | 2.5D俯视角，可缩放 |
| **操作模式** | 鼠标点击移动/攻击 + 键盘快捷键施放技能 |
| **单位控制** | 可单独控制英雄，可框选部队下达指令 |
| **战斗规模** | 英雄 + 部队（常规战斗中约10-50个单位） |
| **实时性** | 完全实时，无回合制要素 |

### 英雄战斗能力构成

| 能力模块 | 子项 | 说明 |
|---------|------|------|
| **主动技能** | 伤害/控制/治疗/召唤 | 通过快捷键施放，有冷却时间 |
| **被动技能** | 属性强化/特殊效果 | 永久生效，无需操作 |
| **普通攻击** | 近战/远程 | 自动攻击范围内的敌人 |
| **装备** | 武器/防具/饰品/坐骑 | 提供属性加成和特殊效果 |
| **巫术** | 大师/基础/无法使用 | 决定能否施放法术 |

## 二、部队系统

### 部队构成

英雄可以带领士兵进入战场，部队是部落与弯刀战斗力的重要组成部分：

| 部队类型 | 来源 | 特点 |
|---------|------|------|
| **常规兵种** | 城镇兵营招募 | 人类兵种为主，可升级至6阶 |
| **野兽兵种** | 地图商人/野性之魂职业 | 野兽单位，特殊获取方式 |
| **机械兵种** | 黑崖地区/旧科技 | 机械单位，高防御 |
| **雇佣兵** | 酒馆佣兵队长招募 | 特殊能力单位，需花费金币 |
| **英雄随从** | NPC英雄招募 | 独立技能树，可独立成长 |

### 兵种进阶体系（6阶）

部落与弯刀的兵种拥有1-6阶的进阶体系：

| 阶位 | 升级消耗 | 战力水平 |
|:----:|---------|---------|
| 1阶 | 初始招募 | 基础单位 |
| 2阶 | 金币 + 铁石 | 低级进阶 |
| 3阶 | 金币 + 铁石 + 木材 | 中级单位 |
| 4阶 | 金币 + 铁石 + 木材 + 玉石 | 高级单位 |
| 5阶 | 大量金币 + 资源 | 精英单位 |
| 6阶 | 大量金币 + 稀有资源 | 顶级单位（终极形态） |

**升级资源需求**：
- **金币**：通用货币，所有升级都需要
- **铁石**：兵种升级和装备锻造的核心材料
- **木材**：建筑材料和部分兵种升级需求
- **玉石**：高级兵种升级和稀有物品所需

## 三、三大兵种系

### 人类兵种

人类兵种是最常见的部队来源，通过各城镇的兵营招募。每个部落拥**有独特的特色兵种**：

| 部落 | 特色兵种 | 兵种类型 | 特点 |
|------|---------|:--------:|------|
| **雄鹰** | 雕弩手 | 远程 | 高射速远程单位 |
| **毒蝎** | 暗夜使徒 | 刺客 | 高爆发、潜行、近战刺杀 |
| **野马** | 先锋骑士 | 骑兵 | 高机动、冲锋伤害 |
| **雪狼** | 狂野兽王 | 野兽 | 高生命、近战肉搏 |
| **蛮牛** | 破天战神 | 重装 | 极高防御、攻城特化 |

### 野兽兵种

| 野兽单位 | 获取方式 | 特点 |
|---------|---------|------|
| 狼 | 地图上随机遭遇/驯服 | 高攻速，脆皮 |
| 熊 | 地图特定区域/野性之魂职业 | 高血量，近战 |
| 豹 | 地图特定区域 | 高敏捷，爆发 |
| 特殊野兽 | BOSS级单位 | 待补充具体数据 |

野性之魂职业与野兽兵种有深度绑定——该职业可以更容易地统领和强化野兽单位。

### 机械兵种

机械兵种主要出现在黑崖地区的"旧科技"相关剧情线中：

| 机械单位 | 特点 | 获取方式 |
|---------|------|---------|
| 机械守卫 | 高防御、低速度 | 特定区域招募 |
| 古代机械 | 特殊攻击方式 | 剧情解锁 |
| 其他机械单位 | 待补充 | 待补充 |

## 四、雇佣兵系统

在酒馆的佣兵队长处可以雇佣特殊单位：

| 雇佣兵种 | 特色能力 | 适用场景 |
|---------|---------|---------|
| **赫炎步兵** | 城甲（减伤）、高生命 | 攻城战前排 |
| **赫炎工兵** | 器械伤害加成 | 攻城战/破坏设施 |
| **沙蜥探险家** | 反巫师特化 | 对抗法系敌人 |

雇佣兵不占用常规兵种的升级路线，但需要持续支付维护费用——**是一种"花钱买战斗力"的经济玩法**。

## 五、兵种克制关系

| 攻击方 \ 防御方 | 步兵 | 骑兵 | 远程 | 飞行 |
|:--------------:|:----:|:----:|:----:|:----:|
| **步兵** | 均衡 | 劣势 | 优势 | 劣势 |
| **骑兵** | 优势 | 均衡 | **优势** | 劣势 |
| **远程** | 劣势 | 劣势 | 均衡 | **优势** |
| **飞行** | 优势 | 优势 | 劣势 | 均衡 |

**克制机制说明**：
- 骑兵 → 冲锋切入远程阵线，远程单位无法反击

- 步兵 → 密集阵型克制分散的骑兵冲锋

## 六、攻城战系统

### 攻城流程

```
准备阶段 → 建造攻城器械 → 攻破城门 → 巷战阶段 → 占领城镇
```

| 阶段 | 操作 | 关键要素 |
|------|------|---------|
| **准备** | 集结部队、准备资源 | 兵力充足性、资源储备 |
| **攻城器械** | 建造攻城塔/投石车等 | 器械类型影响破门效率 |
| **破城** | 摧毁城门防御 | 城门耐久度、守军远程火力 |
| **巷战** | 城内逐街战斗 | 地形狭窄、远程单位受限 |
| **占领** | 清除所有守军 | 占领后获得城镇控制权 |

### 攻城战与野战的区别

| 维度 | 野战 | 攻城战 |
|------|:----:|:------:|
| 地形 | 开阔 | 狭窄街道 |
| 防御工事 | 无 | 城墙 + 城门 + 箭塔 |
| 远程单位 | 开阔优势 | 城墙优势 |
| 持续时间 | 较短 | 较长 |
| 战后收益 | 经验/掉落 | 城镇控制权 |

## 七、与骑马与砍杀2的战斗对比

| 对比维度 | 部落与弯刀 | 骑马与砍杀2 |
|----------|-----------|-------------|
| **视角** | 2.5D俯视角RTS式 | 第一/第三人称动作 |
| **操作模式** | 点击移动+技能快捷键 | WASD移动+鼠标攻击方向 |
| **战斗规模** | 10-50单位（常规） | 100-400+单位同屏 |
| **英雄定位** | 技能核心，决定战局走向 | 武将/指挥官，个人战力有限 |
| **部队控制** | 框选+右键指令 | F1-F3阵型指令系统 |
| **技能华丽度** | Diablo式华丽技能特效 | 写实风格，技能有限 |
| **攻城战** | 建造器械+破门+巷战 | 建造攻城塔+云梯+城墙战 |
| **兵种差异** | 部落特色兵种丰富 | 文化特色兵种但差异较小 |
| **战斗节奏** | 技能CD循环为主 | 走位+格挡+攻击时机 |

**核心差异**：
- 骑砍2追求的是**沉浸式第一人称战场模拟**——你能听到箭矢从耳边飞过，能看到骑兵冲锋时的尘土飞扬。400单位同屏的战场规模带来的是真实的"战争感"。
- 部落与弯刀追求的是**英雄视角的RTS战斗**——你以上帝视角指挥部队，同时操作英雄施放华丽技能。战场规模较小，但**英雄的个人能力对战局的影响更大**——一个高等级心灵巫师的大范围法术可能瞬间改变战局。

## 八、设计特点分析

**"英雄中心主义"的战斗设计**：与骑砍2的"指挥官"定位不同（骑砍中你的个人战力再强也架不住被20个土匪包围），部落与弯刀更接近"割草"类游戏——英雄的技能在战斗中占主导地位。部队更像是"会移动的buff/debuff"，而非独立战斗力。这种设计降低了微操门槛，让玩家能将更多注意力放在**英雄的技能循环和走位**上。

**6阶兵种的资源消耗曲线**：兵种升级需要四种资源（金币/铁石/木材/玉石），且越到高阶消耗越大。这种设计有两个目的：
1. **防止玩家过早拥有顶级部队**——即使有足够的金币，也可能因为缺少玉石而无法升到6阶
2. **创造经济策略点**——玩家需要权衡"把资源用于兵种升级"还是"用于城镇建设"还是"用于装备锻造"
3. 与Kenshi的"装备品质"升级曲线有相似的经济思维——**好装备/好兵种需要时间和资源的沉淀**

**部落特色兵种的身份标识**：五大部落各有独特兵种的设计，不仅仅是玩法上的差异化——它同时也是**叙事的辅助工具**。雄鹰的雕弩手暗示了他们的远程战术传统，蛮牛的破天战神暗示了他们的重装文化。兵种成为了"可玩的部落身份"——当你招募一个破天战神时，你也在"扮演"蛮牛部落的军事传统。

**攻城战的阶段性设计**：将攻城战拆分为"准备→破城→巷战→占领"四阶段，每个阶段有不同的战术要求：
- 准备阶段考验的是玩家的**战略规划能力**
- 破城阶段考验的是**器械使用和对防御火力的规避**
- 巷战阶段考验的是**微操和技能施放时机**
- 占领阶段考验的是**战后管理能力**

这种设计让攻城战不仅仅是一场"更大规模的战斗"，而是一个**跨越多个游戏系统的复合型挑战**。

---

> 跨品类对比: 与Kenshi的"战斗"对比——Kenshi的战斗是自动进行的（玩家只制定战术策略，不直接操作），且胜负不仅取决于单次战斗，还受到"断肢""失血""被俘虏"等长期后果的影响。部落与弯刀的战斗则是**完全可控的实时操作**，胜负在战斗中即刻决定。两者的差异反映了"操作型沙盒"与"模拟型沙盒"的两种战斗哲学。
