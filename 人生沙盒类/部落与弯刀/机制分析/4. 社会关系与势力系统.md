## 一、势力总览

### 五大部落

部落与弯刀的世界由五大部落主导，每个部落拥有自己的领地、文化特色和军事传统：

| 部落 | 领地 | 首领 | 特色兵种 | 文化特征 |
|------|------|:----:|---------|---------|
| **雄鹰** | 红石谷、骆驼城 | 霍驹 | 雕弩手（远程） | 平原文明，远程战术传统 |
| **毒蝎** | 风哭岩 | 阿曼莎 | 暗夜使徒（刺客） | 沙漠中的暗杀文化 |
| **野马** | 西多喀那沙漠 | 苍骐 | 先锋骑士（骑兵） | 游牧骑兵传统，机动战 |
| **雪狼** | 大雪山 | 封凛 | 狂野兽王（野兽） | 雪山部落，与野兽共生 |
| **蛮牛** | 黑崖 | 汗卢达 | 破天战神（重装） | 重装文化，矿山经济 |

### 五大部落的领地分布

```
[大雪山] ← 雪狼
    |
[红石谷] ← 雄鹰 — [骆驼城]
    |
[风哭岩] ← 毒蝎
    |
[西多喀那沙漠] ← 野马
    |
[黑崖] ← 蛮牛
```

各部落之间并非平等——**雄鹰势力最强，蛮牛资源最丰富，雪狼最封闭**。这种不对称设计模拟了真实的政治地理格局。

### 野外/敌对势力

| 势力 | 所在区域 | 性质 | 特点 |
|------|---------|------|------|
| **光煜教会** | 风哭岩北部 | 宗教势力 | 崇拜火魔人，敌视主流社会 |
| **沙漠兄弟会** | 红石谷 | 强盗集团 | 沙漠中的土匪组织 |
| **野火组织** | 黑崖 | 雇佣兵团体 | 亦正亦邪的雇佣兵/无赖组织 |
| **雪山游击军** | 大雪山 | 反抗军 | 雪山地区的饥民武装 |

## 二、势力加入与晋升体系

### 加入条件

| 条件 | 要求 | 说明 |
|------|:----:|------|
| 部落首领好感度 | ≥特定值 | 通过送礼/完成任务提升 |
| 部落声望 | ≥10 | 积累声望的方式包括：完成部落任务、击败敌对势力、捐献资源 |
| 特殊事件 | 可变 | 部分部落有专属加入剧情 |

### 势力内晋升体系

加入势力后，玩家可以通过积累贡献来提升地位：

| 地位 | 贡献要求 | 权利 |
|------|:--------:|------|
| 成员 | 基础 | 可接取势力任务、使用势力设施 |
| 核心成员 | 中等 | 可参与势力决策 |
| 封臣 | 高 | **可获得封地**（下属城镇） |
| 叛变自立 | 获得封地后 | 可宣布独立建国 |

### 势力贡献获取方式

| 方式 | 贡献量 | 频率 |
|------|:-----:|:----:|
| 完成势力任务 | 中-高 | 常规 |
| 捐献资源（金币/铁石/木材/玉石） | 中 | 随时 |
| 击败敌对势力部队 | 低-中 | 常规 |
| 攻城占领 | 高 | 低频率 |
| 护送商队 | 低 | 低频率 |

## 三、自立为王系统

### 建国条件

```
途径A: 攻下一座城镇 → 宣布建国
途径B: 获得封地（加入势力后） → 叛变 → 宣布独立
```

### 建国后的管理要素

| 管理维度 | 内容 | 说明 |
|---------|------|------|
| **外交** | 与其他部落的关系管理 | 宣战/结盟/中立/贸易协定 |
| **税收** | 城镇税率设定 | 影响民心和经济收入 |
| **建设** | 城镇建筑升级 | 图书馆/兵营/巫师集会所等 |
| **军事** | 部队招募与部署 | 城防/野战部队/远征军 |
| **人才** | 英雄招募与管理 | 封臣管理/任务分配 |

### 自立王国的潜在风险

- **被多部落围攻**：自立后，周边部落可能组成联军讨伐
- **内部叛乱**：民心过低时，城镇可能发生叛乱
- **经济崩溃**：战争状态下，税收锐减+军费暴增
- **孤立无援**：没有盟友时，难以应对多线作战

## 四、英雄招募系统

### 英雄招募流程

```
提升好感度 → 完成特定任务 → 好感达标 → 邀请入队
```

| 阶段 | 方式 | 说明 |
|------|------|------|
| **好感度提升** | 送礼 | 不同英雄喜欢不同的礼物类型 |
| **好感度提升** | 共同战斗 | 并肩作战可提升好感 |
| **好感度提升** | 完成任务 | 为英雄解决个人事务 |
| **条件触发** | 好感达标 | 达到特定阈值触发招募对话 |
| **邀请入队** | 确认 | 英雄加入队伍成为随从 |

### 英雄属性与定位

每位可招募英雄有：

| 英雄信息 | 说明 |
|---------|------|
| **独立技能树** | 与玩家职业不同的专属技能配置 |
| **背景故事** | 每位英雄有独立的背景设定 |
| **初始等级** | 招募时的等级，随世界时间推移可能变化 |
| **装备配置** | 初始装备配置，可后续更换 |
| **性格倾向** | 影响对话选项和好感度变化 |

### 英雄招募策略

| 招募方向 | 推荐英雄类型 | 用途 |
|---------|-------------|------|
| 战斗型 | 剑术/法术专精英雄 | 增强部队战斗力 |
| 辅助型 | 治疗/增益专精英雄 | 团队续航 |
| 管理型 | 待补充 | 城镇管理效率提升 |

## 五、巫师盟会系统

### 巫师集会所

在占领的城镇中建设**巫师集会所**后，可以招募巫师加入队伍：

| 建筑等级 | 可招募巫师等级 | 功能解锁 |
|:--------:|:-------------:|---------|
| Lv.1 | 低级巫师 | 基础招募功能 |
| Lv.2 | 中级巫师 | 更丰富的巫师选择 |
| Lv.3 | 高级巫师 | 可招募精英巫师 |

巫师盟会系统是**非领主职业获得法系兵种的主要途径**——没有领主的玩家也可以通过建设巫师集会所来组建法系部队。

## 六、势力间的规则驱动演化

### 世界沙盒的运行机制

部落与弯刀最核心的沙盒特性之一是：**世界在玩家不参与时依然自主运转**。

| 演化维度 | 机制 | 效果 |
|---------|------|------|
| **部落攻伐** | AI部落间自主宣战/攻防 | 势力版图持续变化 |
| **势力消长** | 战败部落可能灭亡/分裂 | 世界格局动态调整 |
| **英雄流动** | NPC英雄可被其他势力招募 | 人才市场动态变化 |
| **商队活动** | AI商队自主跑商 | 经济系统自主运转 |

这种设计意味着：**玩家不是世界的中心**——你只是这个运转中的世界的一个参与者。即使你完全不去干预，五大部落之间的战争和联盟也会持续进行。

### 与骑马与砍杀2的规则驱动对比

| 对比维度 | 部落与弯刀 | 骑马与砍杀2 |
|----------|-----------|-------------|
| **世界自主性** | 高——部落自主攻伐，势力消长 | 高——王国AI自主宣战/议和 |
| **玩家影响力** | 中——玩家可改变局部但非全局 | 中——战力积累到一定程度可左右战局 |
| **势力灭亡机制** | 存在——部落版图可能被完全吞并 | 存在——王国可被完全征服 |
| **英雄流动** | 送喜好礼物→招募入队 | 说服/家族联姻→招募家族 |
| **继承系统** | 无多代继承 | 角色会老死，后代继承 |

## 七、设计特点分析

**"骑砍式势力 + RPG式英雄招募"的融合**：部落与弯刀的势力系统本质上继承了骑马与砍杀的"加入势力→积累贡献→获得封地→自立为王"的经典路径。但英雄招募系统的加入改变了传统的势力关系——在骑砍中，同伴（companion）只是功能性NPC，你无法通过"送礼物"来拉拢敌对势力的核心将领。部落与弯刀则允许玩家通过社交系统**直接招募英雄**，这一设计大幅增加了"挖墙脚"的玩法深度——你可以通过送礼把一个敌对部落的核心成员变成自己的队友。

**五大部落的不对称设计**：雄鹰（最强）/ 蛮牛（最富）/ 雪狼（最封闭）的不对称格局不是随意的——它制造了天然的"势力选择"策略：加入雄鹰意味着抱大腿但贡献难出头；加入雪狼意味着安全但发展受限；加入蛮牛意味着资源丰富但容易成为众矢之的。这种**不对称势力设计**是沙盒游戏的经典手法——让每个势力的"优劣势"成为玩家决策的依据。

**规则驱动世界的沉浸感**：部落自主攻伐的设计看似是AI基础功能，但其实是一个关键的**沉浸感机制**——当玩家离开一片区域一段时间后回来发现"雄鹰把野马灭了"，这种世界在玩家不参与时依然变化的感觉，是沙盒游戏区别于"预设叙事"游戏的根本所在。与Kenshi的"世界状态"系统有相似之处——在Kenshi中，如果你不干预，某些势力会逐渐扩张或衰落。

**自立为王的"惩罚曲线"**：自立后的多部落围攻、内部叛乱、经济崩溃等风险形成了一个**逐步加大的压力曲线**：
- 初期：宣布独立→周边部落不满→可能遭遇1-2个部落的讨伐
- 中期：持续扩张→树敌增多→多线作战压力
- 后期：版图过大→民心管理困难→内部叛乱风险上升

这种压力曲线的设计目的是防止"优势滚雪球"——**征服世界不会越来越容易，反而越来越难**。

**好感度 = 社交货币**：在部落与弯刀中，好感度不仅是招募工具，也是一种"社交货币"——你可以通过送礼来换取英雄的忠诚，也可以通过交涉来用好感度换取其他利益（如通商许可/情报/资源）。这种设计将社会关系量化为一种可交换的资源，与太吾绘卷的"人情系统"有类似的设计思路。

---

> 跨品类对比: 与太吾绘卷的"门派与社会关系"对比——太吾绘卷的门派系统更加"中国化"：门派不仅是势力组织，还包含"功法传承""辈分秩序""门派立场（正/邪/中立）"等文化维度。部落与弯刀的势力系统则更接近"西方奇幻"的政治格局——部落 = 军事-经济实体，玩家可以选择加入、叛变或征服。两者的共同点是都将"社会治理"纳入了游戏玩法而非仅仅是背景设定。
