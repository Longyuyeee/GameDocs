# 密谋与谍报数据总览 — 十字军之王3

> 游戏版本: CK3 v1.12+ (Steam, 2020) | 来源: [CK3 Paradox Wiki](https://ck3.paradoxwikis.com/)

---

## 一、密谋成功率因素

| 因素 | 加成 | 说明 |
|------|:--:|------|
| 密谋技能 | 每10点+5-10% | 核心 |
| 代理人数量 | 每个+5-15% | 在目标宫廷中收买NPC |
| 配偶协助 | +10-20% | 配偶密谋技能高=更多加成 |
| 目标密谋防御 | 负加成 | 目标的密谋技能+间谍总管防御 |
| 性格特质干扰 | 浮动 | 诚实者参与密谋=自身压力+ |

## 二、秘密类型与勒索价值

| 秘密 | 钩子强度 | 公开后果 |
|------|:--:|------|
| 私生子(是某人的秘密父亲) | 强 | 丑闻/继承权争议 |
| 谋杀秘密 | **极强** | **犯罪/可合法逮捕** |
| 假信(秘密改宗) | 中 | 宗教领袖敌对 |
| 乱伦/通奸 | 强 | 丑闻/离婚 |
| 异端信仰 | 中 | 可要求改宗→拒绝=合法监禁 |

## 三、恐惧值(Dread)获取

| 方式 | Dread增加 | 风险 |
|------|:--:|------|
| 处决囚犯 | +5-15 | 被处决者的家族仇视 |
| 折磨囚犯 | +3-10 | 角色可能死亡 |
| 公开处决(事件) | +20-30 | 大量家族仇恨 |
| 制造恐怖(perk) | 加成 | — |

## 四、密谋发现与防御

| 防御层 | 效果 |
|------|------|
| 高密谋技能 | 基础防御 |
| 间谍总管(防御任务) | +密谋防御(技能值加成) |
| 忠诚的配偶 | 防止配偶参与谋杀你的密谋 |
| 食物品尝官 | 防止毒杀(宫廷职位) |
