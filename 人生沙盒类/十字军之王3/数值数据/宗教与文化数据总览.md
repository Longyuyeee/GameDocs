# 宗教与文化数据总览 — 十字军之王3

> 游戏版本: CK3 v1.12+ (Steam, 2020) | 来源: [CK3 Paradox Wiki](https://ck3.paradoxwikis.com/)

---

## 一、核心教义(Tenets)——定义宗教的"宪法"

| 教义 | 效果 | 推荐 |
|------|------|:--:|
| 修道主义(Monasticism) | 可要求子女发誓→退出继承 | ★★★★★(继承管理) |
| 圣战(Armed Pilgrimages) | 解锁大圣战 | ★★★★(军事扩张) |
| 共融圣事(Communion) | **教皇=无限赎罪券金钱** | ★★★★★(经济) |
| 追求力量(Pursuit of Power) | 无限征服宣战理由 | ★★★★★(扩张) |
| 食人(Cannibalism) | 处决囚犯=获得食物 | —(特殊玩法) |

## 二、教条(Doctrines)——宗教的内部规则

| 类别 | 可选范围 |
|------|------|
| 神职人员性别 | 仅男性/仅女性/两性皆可 |
| 神职人员婚姻 | 可婚/不可婚/纳妾 |
| 宗教领袖 | 无/属灵(如教皇)/世俗(国王=宗教领袖) |
| 圣战目标 | 仅异教徒/仅不同分支/所有人 |
| 一夫多妻/纳妾 | 允许/禁止 |

## 三、文化传统精选

| 传统 | 效果 | 民族 |
|------|------|------|
| 海盗(Raider) | 可劫掠 | 北欧 |
| 以剑为生(By the Sword) | **无限圣战**(极强) | 特定军事文化 |
| 选侯制(Elective) | 特殊继承法 | 撒克逊 |
| 骑士精神(Chivalry) | 骑士效+ | 法兰西/英格兰 |
| 城堡守护者(Castle Keepers) | 城堡税收+防御+ | 日耳曼 |

## 四、文化融合条件

| 要求 | 说明 |
|------|------|
| 两种文化的领地相邻 | 通常是征服异文化领地后的情况 |
| 文化接受度(Cultural Acceptance)足够 | 时间积累+内阁任务加速 |
| 消耗威望 | 大量 |
| 结果 | 新文化=融合双方传统+可选新增传统 |

## 五、宗教改革成本

| 因素 | 虔诚(Piety)消耗 |
|------|:--:|
| 改变教义 | 500-2000/个 |
| 改变教条 | 1000-5000/项 |
| 创建新信仰 | 全部成本+一次性 |
| 学术生活方式perks | 减少30-50%成本 |
