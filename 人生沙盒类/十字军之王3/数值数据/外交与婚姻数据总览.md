# 外交与婚姻数据总览 — 十字军之王3

> 游戏版本: CK3 v1.12+ (Steam, 2020) | 来源: [CK3 Paradox Wiki](https://ck3.paradoxwikis.com/)

---

## 一、婚姻联盟价值

| 联姻类型 | 同盟强度 | 宣称权价值 | 遗传价值 |
|------|:--:|:--:|:--:|
| 国王的女儿 | 极高(强军事同盟) | 下一代获得王国宣称 | 取决于特质 |
| 公爵的女儿 | 高 | 下一代获得公国宣称 | 取决于特质 |
| 天才平民 | 无(平民=无军事同盟) | 无 | **极高**(天才遗传) |
| 年长寡妇(高能力) | 无 | — | —(已过生育年龄/但技能加成) |

## 二、同盟触发

| 同盟类型 | 触发 | 可召唤参战 |
|------|------|:--:|
| 婚姻同盟 | 家族间联姻 | 进攻+防御 |
| 防御同盟(perk) | 外交perk解锁 | **仅防御** |
| 宗族同盟 | 同宗族成员 | 自动 |

## 三、好感度构成

| 来源 | 范围 |
|------|:--:|
| 相同宗教 | +0~+20 |
| 相同文化 | +0~+15 |
| 送礼 | +0~+100(递减) |
| 拉拢(Sway) | 逐渐+直到+100 |
| 联姻 | +15~+30 |
| 恐惧/恐怖 | 强力封臣不敢行动(但好感不会增加) |

## 四、宣称权获取方式

| 方式 | 耗时 | 成本 | 适用 |
|------|:--:|------|------|
| 婚姻 | 1代(30年) | 婚姻机会成本 | 长期策略 |
| 伪造(内阁任务) | 1-5年 | 免费 | 短期扩张 |
| 宗族宣称 | 即时(已存在) | 战争 | 为家族成员而战 |
| 邀请宣称者 | 即时 | 金币 | 收买有宣称权的人→使其成为你的封臣→为他而战 |
