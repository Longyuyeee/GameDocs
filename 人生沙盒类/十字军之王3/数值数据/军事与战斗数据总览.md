# 军事与战斗数据总览 — 十字军之王3

## 一、兵种克制

| 兵种 | 强克 | 被克 |
|------|:--:|:--:|
| 重骑兵 | 弓兵 | 长枪兵 |
| 弓兵 | 散兵 | 重骑兵 |
| 长枪兵 | 重骑兵 | 重步兵 |
| 重步兵 | 长枪兵 | 散兵 |
| 散兵 | 重步兵 | 弓兵 |
| 攻城武器 | —(攻城专用) | 所有近战 |

## 二、军队构成数据

| 类型 | 数量上限 | 战斗力/单位 | 维护成本 |
|------|:--:|:--:|:--:|
| 征召兵(Levies) | 根据领地+封臣 | 10(低) | 低 |
| 兵士(Men-at-Arms) | 根据建筑+文化 | 30-45(中高) | 中高 |
| 骑士(Knights) | 根据声望+等级 | 100×勇武(极高) | **零** |
| 雇佣兵(Mercenaries) | 根据金钱 | 浮动 | 极高(短期) |

## 三、战斗阶段

| 阶段 | 说明 |
|:--|------|
| 初始(布阵) | 双方对峙——决定初始优势骰 |
| 战斗 | 每3天一轮——击杀计算 |
| 追击 | 溃败方被追击——额外损失 |

## 四、指挥官的影响

| 指挥官军事技能 | 战斗优势骰加成 | 特殊能力 |
|:--|:--|------|
| 0-8 | 低 | 无 |
| 9-14 | 中 | 攻守地形适应 |
| 15-22 | 高 | 兵种特化加成 |
| 23-30+ | 极高 | **可改变战斗结果的决定性指挥官** |
