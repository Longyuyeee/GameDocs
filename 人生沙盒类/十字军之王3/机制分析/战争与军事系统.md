# 战争与军事系统 — 十字军之王3

> 游戏版本: CK3 v1.12+ (Steam, 2020)
> 数据来源: [CK3 Paradox Wiki](https://ck3.paradoxwikis.com/)

---

## 一、宣战理由(Casus Belli)

CK3中不能"无理由宣战"——必须有一个合法的宣称权:

| 宣战理由 | 获取方式 | 战争目标 |
|------|------|------|
| 宣称权战争 | 继承/伪造/婚姻 | 夺取特定领土 |
| 圣战 | 宗教教义解锁 | 夺取异教徒的整个公国/王国 |
| 征服战争 | 部落统治者/特定文化传统 | 夺取邻接领土 |
| 独立战争 | 封臣发起 | 脱离宗主国 |
| 宗族宣称 | 家族成员有宣称权 | 为家族成员夺取领土 |

## 二、军队构成

| 类型 | 来源 | 成本 |
|------|------|:--:|
| **征召兵(Levies)** | 领地+封臣 | 低质量/低维护 |
| **兵士(Men-at-Arms)** | 建筑+文化传统 | 高质量/高维护 |
| **骑士(Knights)** | 宫廷成员+高勇武 | 极高战斗力/零维护 |
| **雇佣兵(Mercenaries)** | 金钱购买 | 昂贵/短期使用 |

## 三、战斗系统

| 要素 | 影响 |
|------|------|
| 指挥官军事技能 | 决定战斗优势骰 |
| 兵种克制 | 重骑兵>弓兵>长枪兵>重骑兵 |
| 地形/天气 | 影响战斗结果 |
| 补给系统 | 在敌对领土过久=损耗 |

## 四、设计特点分析

**"必须先有理由才能开战"=战争的合法性博弈**：与大多数策略游戏中"想打就打"不同——CK3的宣战理由(Casus Belli)系统要求你必须先有合法的战争理由。这意味着"战争"不仅是军事问题——它也是外交/继承/婚姻的**结果**。你需要先通过联姻获取宣称权、或通过宗教获得圣战权——然后才能开战。这种设计让"战争"成为了"长期外交策略的终点"而非"冲动的按钮"。

**骑士(Knights)=中世纪个人英雄主义的体现**：一个勇武(Prowess)30+的骑士可以在战场上击杀数十个敌人——骑士不是"一个战斗单位"，而是"一个可以改变战局的个体英雄"。这种设计体现了中世纪战争的真实特征——个人勇武可以产生决定性影响。
