# 角色系统：特质、技能与压力 — 十字军之王3

> 游戏版本: CK3 v1.12+ (Steam, 2020)
> 数据来源: [CK3 Paradox Wiki](https://ck3.paradoxwikis.com/), [TheGamer Guide](https://www.thegamer.com/)

---

## 一、性格特质——定义"他是谁"

每个CK3角色有一套性格特质，这些特质**不只是属性修正**——它们定义了角色的AI行为：

| 特质 | 行为倾向 | 压力来源 |
|------|------|------|
| 仁慈(Compassionate) | 避免暴力/释放囚犯 | **处决/折磨囚犯→压力暴增** |
| 虐待狂(Sadistic) | 享受暴力/乐于处决 | **释放囚犯/仁慈行为→压力** |
| 诚实(Honest) | 拒绝密谋 | **参与谋杀计划→压力** |
| 狡诈(Deceitful) | 乐于密谋 | **诚实行为→压力** |
| 勇敢(Brave) | 不惧战斗/领导军队 | **逃避战斗→压力** |
| 懦弱(Craven) | 回避危险 | **上战场→压力** |
| 虔诚(Zealous) | 热衷宗教/敌视异教徒 | **与异教徒交好→压力** |

## 二、五维技能

| 技能 | 核心影响 | 关键角色 |
|------|------|------|
| **外交(Diplomacy)** | NPC好感/同盟谈判/送礼效果 | 外交型君主 |
| **军事(Martial)** | 军队指挥/战斗力/征召兵数量 | 战争型君主 |
| **管理(Stewardship)** | 直辖领上限/税收/建筑效率 | 经济建设型 |
| **密谋(Intrigue)** | 谋杀成功率/绑架/发现秘密 | 幕后操控型 |
| **学术(Learning)** | 文化发展/宗教改革/健康寿命 | 学者型君主 |

### 教育特质

角色16岁时根据教育经历获得教育特质(1-4星)。教育特质直接影响对应生活方式的**月度经验获取**——4星教育=更快解锁技能树perks。

## 三、压力系统——CK3的核心创新

### 什么是压力

压力是CK3将"角色扮演"从"你可以做任何事"变为"你的角色有自己的性格"的核心机制。当角色做出**与其性格特质相悖**的行为时，压力增加。压力过高→精神崩溃(Mental Break)→可能触发负面事件或死亡。

### 压力来源

| 来源 | 示例 |
|------|------|
| **违背特质的行为** | 仁慈者处决囚犯/诚实者参与谋杀 |
| **家庭悲剧** | 子女/配偶/挚友死亡 |
| **外交屈辱** | 被迫接受不公平条约 |
| **疾病/伤残** | 长期患病或受伤 |

### 压力释放

| 方式 | 效果 |
|------|------|
| 狩猎/宴会 | 临时降低压力 |
| 朝圣 | 虔诚角色大幅减压 |
| 折磨/处决囚犯 | 虐待狂角色的减压方式 |
| 喝酒(酒鬼特质) | 用酒精麻痹压力 |

## 四、健康与死亡

- **健康值**：隐藏数值——随年龄下降/受疾病/受伤/生活方式影响
- **死亡**：健康耗尽→角色死亡→触发继承法→新角色继位
- "知道什么时候会死"的不确定性是CK3核心张力之一——你的70岁国王可能明天就死，也可能活到85岁

---

## 五、设计特点分析

**压力=角色扮演不是"你决定"而是"他决定"**：在大多数游戏中，玩家完全控制角色的行为。CK3的压力系统打破了这一点——你的角色有"性格"，做违背性格的事会产生压力。这是一个优雅的设计——它将"角色扮演"从"主动选择"变为"内在约束"。

**性格特质=最优策略的阻碍器**：你可能是玩家，知道"谋杀这个封臣是最优的"，但你的角色是"仁慈"的——如果你执行这个最优策略，压力会飙升。所以在CK3中"最优策略"并不总是"最优的"——你必须在"游戏最优"和"角色性格"之间做出妥协。这就是CK3作为"角色驱动策略游戏"的核心张力。

---

> 跨品类对比: CK3的压力系统与Elona的"善恶值影响商店服务"有相似之处——都是"行为=后果"的软约束。但CK3更进一步——约束不是来自"世界规则"，而是来自"角色自己的性格"。
