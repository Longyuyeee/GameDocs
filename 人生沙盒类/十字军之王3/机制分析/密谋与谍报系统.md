# 密谋与谍报系统 — 十字军之王3

> 游戏版本: CK3 v1.12+ (Steam, 2020)
> 数据来源: [CK3 Paradox Wiki](https://ck3.paradoxwikis.com/)

---

## 一、密谋类型

| 密谋 | 目标 | 成功率因素 | 被发现后果 |
|------|------|------|------|
| **谋杀(Murder)** | 消灭目标 | 密谋技能/代理人数量 | 弑亲者/凶手/罪行暴露 |
| **绑架(Abduct)** | 将目标抓到你的宫廷 | 高密谋/高勇武的代理人 | 犯罪 |
| **勾引(Seduce)** | 成为目标的秘密情人 | 魅力/密谋 | 私生子/丑闻 |
| **伪造宣称** | 获得一个头衔的宣称权 | 牧师技能 | — |
| **揭露秘密** | 发现NPC的隐藏秘密 | 间谍总管任务 | — |

## 二、谋杀计划

| 要素 | 说明 |
|------|------|
| **成功率** | 基础值+代理人(Agents)的数量+技能加成-目标的密谋防御 |
| **代理人** | 在目标的宫廷中收买NPC→共同参与谋杀→大幅提升成功率 |
| **耗时** | 数月到数年——不是即时执行的 |

## 三、秘密(Secrets)

| 秘密类型 | 发现后可用 |
|------|------|
| 私生子秘密 | **勒索**(获得对方的"钩子Hook") |
| 谋杀秘密 | 勒索/公开→对方犯罪 |
| 信仰秘密(假信) | 勒索 |
| 乱伦/通奸 | 最大勒索价值 |

### 钩子(Hook)

握有秘密=拥有对该角色的"钩子"——可以用来: 强迫入内阁/强迫接受婚姻/投票支持/收取赎金。

## 四、设计特点分析

**密谋=非军事的"影子战争"**：CK3中你不必通过军事征服来获取权力——一个成功的谋杀计划可以让你的儿子成为邻国王国的继承人(如果你安排对了婚姻)。密谋是CK3中"不用一兵一卒就可以征服王国"的路径。

**秘密=政治货币**：在一个中世纪世界中，信息就是权力。知道别人的"秘密"等于拥有一张可兑现的政治支票——可以用它来勒索、控制、或摧毁一个对手。这种"信息不对称=政治权力"的设计在策略游戏中是独特的。
