## 一、宗教系统——自定义信仰

### 核心教义(Tenets)

| 教义 | 效果 |
|------|------|
| 修道主义(Monasticism) | 可要求子女发誓→退出继承 |
| 圣战(Armed Pilgrimages) | 解锁大圣战(十字军) |
| 共融圣事(Communion) | **教皇=无限金钱来源**(赎罪券) |
| 追求力量(Pursuit of Power) | 征服宣战理由无限 |

### 宗教改革——创建你自己的教派

| 可更改内容 | 说明 |
|------|------|
| 教义(3个核心) | 定义宗教的核心规则 |
| 教条(Doctrines) | 神职人员性别/婚姻制度/神职制度 |
| 宗教领袖 | 谁是这个宗教的"教皇"？ |
| 圣战目标 | 对谁可以发动圣战 |

## 二、文化系统——传统与融合

### 文化传统(Traditions)

每个文化有5-8个传统——类似于"民族天赋"：

| 传统 | 效果 |
|------|------|
| 海盗(Raider) | 可劫掠邻国(北欧特色) |
| 选侯制(Elective) | 特殊继承法 |
| 城堡守护者(Castle Keepers) | 城堡税收+防御+ |
| 骑士精神(Chivalry) | 骑士效率+浪漫事件+ |
| 以剑为生(By the Sword) | **无限圣战**(极强) |

### 文化融合

当你的文化与其他文化接触足够长的时间后(通常是征服异文化领地)→可以**融合两种文化**→创建一个新文化(继承双方的传统+可以新增传统)。这是CK3后期最强大的策略工具之一——你可以创造一个"拥有维京海盗的劫掠传统+希腊文化的科技发展速度"的融合文化。

## 三、设计特点分析

**"自定义宗教"=打破历史剧本**：历史策略游戏通常被"历史事实"限制。CK3的宗教改革系统让你可以**改变历史**——你可以把天主教改成"允许女性神职人员+支持一夫多妻+认为吃人是神圣的"。这种"在历史设定中创造非历史可能"的自由是CK3最吸引人的特征之一。

**文化融合=征服者的"软统治"工具**：征服一个异文化的领地→分离主义风险高(文化不同→好感惩罚)。将你的文化与当地文化融合→减少文化冲突→让征服更加可持续。这不是"军事"的，而是"文化"的——CK3让"软实力"变得与"硬实力"一样重要。
