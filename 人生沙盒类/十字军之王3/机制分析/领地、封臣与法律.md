# 领地、封臣与法律 — 十字军之王3

> 游戏版本: CK3 v1.12+ (Steam, 2020)
> 数据来源: [CK3 Paradox Wiki](https://ck3.paradoxwikis.com/), [GameWatcher](https://www.gamewatcher.com/)

---

## 一、直辖领(Domain)

| 概念 | 说明 |
|------|------|
| **直辖上限** | 你能直接管理的伯爵领数量(由管理技能决定) |
| **超出惩罚** | 每个超出上限的伯爵领=税收和征召兵大幅降低 |
| **首府** | 你的"家"——通常是你最好的伯爵领 |
| **建筑** | 每个伯爵领有建筑槽——经济/军事/防御 |

## 二、封臣(Vassals)系统

### 封臣等级

| 等级 | 直辖上限 | 头衔 |
|:--|:--:|------|
| 伯爵(Count) | 1 | 伯爵领 |
| 公爵(Duke) | 2-3 | 公爵领(多个伯爵领) |
| 国王(King) | 3-4 | 王国 |
| 皇帝(Emperor) | 4-5 | 帝国 |

### 封臣意见(Vassal Opinion)

- 封臣好感<0→无法获得该封臣的税收和征召兵
- **强力封臣**(军队最强的3-5人)必须在内阁中——否则-40好感
- 低好感→形成派系(Factions)→可能爆发内战

### 恐惧值(Dread)

| 恐惧值 | 效果 |
|:--:|------|
| 0-19 | 无效果 |
| 20-49 | 被恐吓(Intimidated)——不太可能加入派系 |
| 50-79 | 被威慑 |
| 80-100 | **被惧怕(Terrified)**——绝不会加入反对你的派系 |

## 三、王权法(Crown Authority)

| 等级 | 核心权力 | 对封臣的影响 |
|:--:|------|------|
| 1(最低) | 基础税收 | 封臣可自由宣战 |
| 2 | 可以剥夺头衔 | 封臣间战争受限 |
| 3 | 可以监禁封臣 | 封臣间战争禁止 |
| **4(绝对)** | **指定继承人**/封臣不可战争 | **税收+35%/征召兵+35%** |

## 四、内阁(Council)

| 职位 | 对应技能 | 任务效果 |
|------|:--:|------|
| Chancellor(宰相) | 外交 | 改善封臣好感/整合头衔 |
| Marshal(元帅) | 军事 | 训练军队/增加控制力 |
| Steward(财政官) | 管理 | 收税/发展文化 |
| Spymaster(间谍总管) | 密谋 | 发现秘密/防止谋杀 |
| Court Chaplain(宫廷牧师) | 学术 | 改善宗教关系/促进文化传播 |

> 强力封臣不在内阁=-40好感——这是CK3最核心的政治博弈之一: 你必须把最强大的封臣放在内阁中(即使他们能力平庸)。

---

## 五、设计特点分析

**内阁=政治妥协的机制化**：在大多数策略游戏中，你任命"能力最强的人"当顾问。在CK3中，你必须任命"军队最强的人"(强力封臣)进内阁——即使他能力平庸。这种"政治制度>技术理想"的设计是CK3区别于所有"纯效率优化"策略游戏的关键。

**恐惧值=暴君统治的可行路线**：你不需要所有封臣都"喜欢你"——恐惧值达到80+的封臣不会被恐吓/威慑/惧怕而不会加入派系。这为"暴君统治者"提供了一条可行的路线——通过处决囚犯、折磨敌人来积累恐惧。这与"好人统治"(通过外交/送礼积累好感)形成两种不同的统治风格。
