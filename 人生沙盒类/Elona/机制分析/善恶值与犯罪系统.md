## 一、善恶值(Karma)基础

### 数值范围

| 指标 | 数值 |
|------|:--:|
| 初始值 | 0 |
| 上限(正常) | 20 |
| 上限(善人之秘宝) | 40 |
| 下限 | -100 |
| 犯罪门槛 | **-31** |

### 善恶值影响

| 范围 | 身份 | 世界反应 |
|:--:|------|------|
| -31~+20(+40) | 正常 | 一切正常 |
| **-31以下** | **罪犯(Criminal)** | 商店拒绝服务/守卫敌对/不能接任务 |

## 四、罪犯生存策略

### 伪装与绕过

| 方式 | 效果 |
|------|------|
| **隐形术/卷轴** | 守卫暂时看不见你 |
| **伪装套件(Disguise Kit)** | 暂时伪装——守卫变为中立 |
| **德尔菲** | **唯一无法律限制的城镇**——罪犯可在此自由交易 |
| 家门贸易 | 在家中设置商店与NPC交易 |

### 守卫机制

- 守卫会主动攻击罪犯
- 每次击杀守卫后，**复活的守卫更强**
- 守卫有特殊能力：**将受伤罪犯传送进监狱**
- 守卫掉落的武器不能捡（他们会改用更强的格斗）

## 五、监狱系统

### 入狱方式

1. 被守卫攻击（受伤+负善值）→直接传送到监狱
2. 使用诅咒回城卷轴→随机传送到监狱

### 狱中生活

| 特征 | 说明 |
|------|------|
| 传送禁止 | "魔法场阻止了传送" |
| 时间流速 | 加速（快速服刑） |
| 饥饿 | 需自带食物；饿死→回家(善值+10) |
| 以太风 | **不受影响**(室内) |
| 释放条件 | 善值达到**-30** |

### 越狱

| 方式 | 需要 |
|------|------|
| 挖墙 | 高采矿技能(墙壁是**钢**制的) |
| 撬锁 | 开锁技能 |
| 暴力破门 | 可能造成**永久属性损伤** |
| 召唤怪物+影步转移 | 召唤怪物在门外→换位 |
| 上吊 | 自杀→在家复活(善值+10) |

### 监狱宝藏

西北出口附近有**2个保险柜**——每次入狱都会刷新，是可刷取的隐藏奖励。

## 六、特殊特性

| 特性 | 来源 | 效果 |
|------|------|------|
| **罪恶之秘宝**(You don't feel guilty) | 帕尔米亚"红之花"任务(Lv14) | 善值惩罚-25%；善值上限-20(变为0)；核弹惩罚降至-20 |
| **善人之秘宝**(You are a good man) | 尤因`<Gwen>`稀有掉落 | 善值惩罚+50%；善值上限+20(升至40)；税率减免25%；许愿概率提升 |

> 两者**互斥**——获得一方会消除另一方。

## 七、设计特点分析

**-31的门槛=精确的容错空间**：不是-1就犯罪，而是-31。这给玩家留出了约30次的"小恶"空间——你可以偶尔偷个东西、不小心打中友好NPC，仍然能保持正常社会身份。但当超过-31后，游戏毫不手软——守卫敌对、商店拒绝服务、整个文明世界对你关上大门。

**赎罪的方式多种多样=给回头的路**：不是"作恶了就永远洗不白"——你可以花钱买赎罪券、可以蹲监狱、可以做善事慢慢恢复。这种设计防止了玩家"既然已经恶了，不如恶到底"的心理陷阱——回头总有路，只是有代价。

**犯罪镇德尔菲=故意设计的安全阀**：游戏中故意保留了一个无法律限制的城镇——这不是疏忽，而是给犯罪玩家一个"还能继续玩游戏"的出口。如果所有城镇都拒绝服务，罪犯玩家就彻底玩不下去了——但德尔菲的存在意味着"犯罪"仍然是一个可行的玩法路线。

---

> 跨品类对比: 与十字军之王3的"暴政/恐怖值"系统不同——Elona的善恶值有更细粒度的行为惩罚列表（甚至"叫醒睡觉的NPC"都会扣1点）。与RimWorld的"心情惩罚"不同——Elona的惩罚是物理性的（守卫攻击/商店关闭），而非心理性的。与GTA的"通缉星级"不同——Elona的善恶值不会随时间自然消退，需要主动行为来恢复。
