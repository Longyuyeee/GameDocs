# 战斗与魔法系统 — Elona

> 游戏版本: Elona+ v1.97+ (2007原版/2012+分支)
> 数据来源: [Elona Fandom Wiki](https://elona.fandom.com/wiki/), [Elona+ Wiki](https://wikiwiki.jp/elonaplus/), [Elona灰机Wiki](https://elona.huijiwiki.com/)

## 一、战斗方式

### 回合制异步行动

Elona的战斗是**基于速度的异步回合制**——每个角色有一个速度值决定行动频率。速度越高，单位时间内行动次数越多。这类似于传统Roguelike（NetHack/ADOM）的机制，但加入了更多JRPG元素。

| 战斗特征 | 说明 |
|------|------|
| **行动模式** | 基于速度的回合制——高速度角色可在一回合内多次行动 |
| **战斗空间** | 2D网格地图，地形影响视线和移动 |
| **目标锁定** | 近战自动攻击相邻目标，远程/魔法可指定目标 |
| **友军伤害** | **存在**——范围魔法会误伤宠物和NPC |
| **逃跑** | 可以通过移动离开战斗区域 |

### 距离与位置

- **近战**：相邻1格，武器范围受重量影响（重型武器范围更大）
- **远程**：弓/弩/火器/投掷，命中受距离衰减
- **魔法**：范围法术有AoE（火球、冰风暴等），**会伤害友军**

## 二、物理战斗系统

### 伤害计算

物理伤害的核心公式涉及：

| 参数 | 来源 | 说明 |
|------|------|------|
| 基础伤害 | 武器基础值×(力量/10) | 力量越高伤害越高 |
| 技能加成 | 武器技能等级 | 每级武器技能提升命中+伤害 |
| 战术技能 | 战术(Tactics)技能 | Elona+独有，提升近战伤害倍率 |
| 双手/双持 | 双手/双持技能 | 不同握持方式的伤害修正 |
| PV/DV | 装备护甲值/回避值 | 目标防御的减伤计算 |

### 武器重量机制

| 重量 | 系统 | 说明 |
|:--:|------|------|
| <4.0s | 可单手/可配盾 | 标准用法 |
| ≥4.0s | **必须双手技能** | 双持/配盾→命中/伤害大幅惩罚 |

这个机制简单但有效——**重型武器的高伤害被"不能配盾/不能双持"的成本平衡了**。

### 暴击与特殊攻击

- **心眼(Eye of Mind)**技能增加暴击率
- 武器可附魔特殊攻击效果（麻痹/毒/出血/混乱等）
- Elona+中装备特定武器类型可使用**特殊动作**（如镰刀的AoE收割）

## 三、魔法系统

### 施法机制

| 要素 | 机制 |
|------|------|
| **法术获取** | 从法术书阅读学习（需要读写Literacy技能） |
| **法术书** | 一次性消耗品——读后消失；可投资魔法店获取更多法术书 |
| **MP消耗** | 魔力属性决定MP池大小 |
| **成功率** | 受魔力(MAG)属性和施法(Casting)技能影响 |
| **反冲伤害** | 法术施放时有一定几率"反冲"——对施法者造成法力伤害 |

### 法术类型

| 类型 | 代表法术 | 用途 |
|------|------|------|
| 伤害法术 | 魔法箭/冰箭/火球/虚空箭 | 直接伤害 |
| 辅助法术 | 加速/治疗/隐身/传送/回归 | Buff/功能 |
| 控制法术 | 睡眠/混乱/麻痹/减速 | 削弱敌人 |
| 召唤法术 | 召唤怪物/召唤元素 | 战术召唤 |
| 特殊法术 | 支配/复活/许愿 | 稀有高级效果 |

### 魔法装置

与直接施法不同，魔法装置(Magic Device)技能允许使用**杖/卷轴/魔棒**等消耗品施放法术效果——不消耗MP，但消耗物品本身。适合非法系角色。

## 四、远程战斗

| 武器类型 | 弹药 | 特征 |
|------|------|------|
| 弓(Bow) | 箭 | 中距离，高射速 |
| 弩(Crossbow) | 矢 | 高伤害，低射速 |
| 火器(Firearm) | 子弹 | 极高伤害，最低射速，耶尔雷斯种族加成 |
| 投掷(Throwing) | 投掷物 | 距离最近，但附带特殊效果 |

## 五、宠物战斗AI

宠物在战斗中自主行动，受以下因素影响：
- 宠物的AI类型（肉盾型/远程型/法师型/治疗型）
- 装备的武器和防具
- 基因工程继承的技能
- 与主人的距离（倾向跟随还是主动出击）

宠物会**捡拾战场上的掉落物**——这既是优势（自动收集），也是劣势（可能捡走你想要的装备）。

## 六、死亡惩罚

| 等级 | 死亡惩罚 |
|:--:|------|
| ≤5级 | 仅损失金币（最小惩罚） |
| ≥6级 | **随机降低属性值+技能等级**+损失金币 |

属性/技能的降低是随机的且可能不可逆——这是Roguelike传统的标志。但不同于NetHack的永久死亡，Elona允许你"带着伤疤继续"。

## 七、设计特点分析

**基于速度的异步回合制=公平的深度**：速度值高的角色能在一回合多动，这让"堆速度"成为一个有效的build方向。与纯回合制（人人一回合动一次）相比，速度系统创造了更多的策略维度——你可以在慢速敌人行动之前完成2-3次操作。

**魔法书的消耗性=知识即资源**：法术书读后消失——这意味着"学习一个法术"本身是一次性资源消耗行为。这比"学习后永远拥有的技能树"创造了更强的资源管理压力——你需要谨慎选择学习哪些法术，以及何时使用珍贵的高级法术书。

**友军伤害=战术深度**：范围魔法会误伤宠物和NPC——这个看似"不方便"的设计实际上创造了丰富的战术考量。你需要考虑宠物的位置、是否值得为了一个AoE法术冒误伤风险、以及如何配置宠物装备来抵御你使用的法术属性。

---

> 跨品类对比: 与太吾绘卷的"功法格子+距离+架势"的复杂战斗系统相比，Elona的战斗更简洁直接——它更接近传统Roguelike，但加入了宠物AI和魔法装置等JRPG元素。与RimWorld的实时战斗不同——Elona的回合制让每步决策都有重大后果。
