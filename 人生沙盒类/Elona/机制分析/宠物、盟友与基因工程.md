## 二、宠物管理

### 宠物装备

- 宠物可以装备与玩家相同的装备（取决于身体部位）
- 人形宠物（小女孩/奴隶/神仆）=主力选择，可全装备
- 非人形宠物受限于身体部位数量（如鸡只有翅膀和脚部槽位）

### 宠物训练

| 训练方式 | 说明 |
|------|------|
| **城镇自动训练** | 宠物在城镇中自动使用金币/矿石消费训练 |
| **道场(Dojo)** | $10,000训练一只宠物，不占上限 |
| **废弃牧场** | 后期替代道场，$500,000 |
| **料理早餐** | 宠物有烹饪技能+伙伴好感→起床时做早餐，全队吃饱+提升潜力 |

### 宠物进化

使用**心型道具**（爱心/心之石等）可以让特定宠物进化成更强形态。例如：
- 火幼龙→火龙→古龙（三阶段进化）
- 贝尔(铃铛)→快速贝尔→终结贝尔

### 推荐宠物类型

| 类型 | 代表 | 优势 |
|------|------|------|
| 坦克型 | 火幼龙、巨人 | 极高HP，吸收伤害 |
| 极速型 | 贝尔(Quickling)、金属贝尔 | 90%物理减伤+极速 |
| 多手型 | 阿修罗/迦梨 | 4手武器=恐怖近战输出 |
| 协助型 | 银眼魔女 | 减速+目盲+闪避光环 |
| 神仆 | 放逐者(Itzpalt)/黑猫(Ehekatl) | 各有特色，基础强大 |

## 三、基因工程系统（核心特色）

### 系统概述

基因工程是Elona最具特色的系统之一——允许将一只宠物**融合**到另一只宠物身上，传递技能、属性和身体部位。被融合的宠物会永久消失（消耗）。

### 融合机制

| 可传递内容 | 机制 |
|------|------|
| **技能** | 随机从捐献者学习最多2个技能 |
| **经验/等级** | 捐献者等级更高时，接收者等级提升 |
| **属性** | 比较捐献者最高的3个属性——接收者低于捐献者的属性获得提升(每次每属性最多+10) |
| **身体部位** | 每融合最多传递1个身体部位——捐献者该部位数量必须>接收者 |

### 限制条件

- 捐献者等级 ≤ 基因工程技能 + 5
- "身体复杂"特性：超过13个身体部位后，每个额外部位-5%速度

### 实用捐献来源（Elona+）

| 目标技能 | 捐献怪物 | 获取地点 |
|------|------|------|
| 战术(Tactics) | 艾莱战士 | 小狗洞窟/奴隶商 |
| 心眼(Eye of Mind) | 哥布林 | 小狗洞窟 |
| 高等闪避 | 蝙蝠 | 野外 |
| 施法(Casting) | 艾莱法师 | 小狗洞窟/奴隶商 |

| 演奏(Performer) | 飞蛙 | 野外 |

## 四、繁殖系统（牧场）

- 购买牧场后将宠物留在牧场作为**种畜**
- 后代可带走当宠物/宰杀取材料/卖给德尔菲奴隶商
- 混沌体(Mutant)每3级随机增加身体部位（最多16个），是极佳的基因捐献者
- 触手蠕虫在巢穴类地牢中培育产蛋→蛋包饭料理→天价商品

## 五、婚姻与遗传

与宠物达到"Soul Mate"(灵魂伴侣)好感后：
- 可**结婚**→宠物成为配偶
- 可**基因合成**→创建"下一代"的初始基因，用于**新游戏+的继承角色**

## 六、设计特点分析

**基因工程=宠物系统的元游戏**：宠物不只是"跟你一起打架"的跟班——它们是可以分解、重组、优化的一整套"生物装备系统"。你需要考虑哪只宠物的哪些技能值得提取、哪些属性值得融合、以及牺牲哪只宠物来强化另一只。这种"消耗性优化"创造了持续的决策深度。

**"从人到宠物再回到人"的遗传闭环**：你可以将一只精心培育的宠物基因融合到另一只身上，不断迭代优化，最终通过婚姻→基因合成→新游戏继承——**将宠物培育的成果转化为下一局角色的初始优势**。这个闭环在游戏设计中极为罕见：它将"这一局的努力"和"下一局的起点"连接了起来。

**15只宠物上限=队伍构建的空间经济学**：不是无限的宠物槽位——你需要像构建卡组一样构建宠物队伍。坦克+输出+辅助+特殊用途的搭配考验玩家的队伍构建能力。这个设计比"抓一个最强的就够了"的宠物系统多了一层元策略。

---

> 跨品类对比: 与宝可梦的"捕捉→训练→对战"不同，Elona的宠物系统是"获取→装备→融合→遗传"的四阶段循环。基因工程本质上是让宠物变为"可消耗的资源"——这是一种比传统怪物收集更深层的资源管理设计。与RimWorld的"殖民者=不能融合"不同——Elona允许将两个生物"物理融合"的特性是品类中独一无二的。
