## 三、善值恢复方式

| 行为 | 善值变化 | 可重复 |
|------|:--:|:--:|
| 完成任意任务 | +1 | 是 |
| 归还失物(未开)给守卫 | +5 | 是 |
| 罪犯身份死亡(6级+) | +10 | 是 |
| 服刑(监狱) | +1/时间单位 | 是 |
| 复活死者(卷轴/魔法) | +2 | 是 |
| 给乞讨者金币(家中访客) | +2 | 是 |
| 赎罪券(诺伊埃尔) | **设为-29** | 仅罪犯可用 |
| 许愿"赎罪" | 负值减半 | 仅1次 |
| 额外给抗体(以太风任务) | +20 | 是 |
| 给非人形怪物给找老婆NPC | +2 | 是 |

## 四、罪犯身份效果对照

| 系统 | 正常(≥-30) | 罪犯(≤-31) |
|------|------|------|
| 商店交易 | 正常 | 拒绝(除德尔菲) |
| 守卫态度 | 友好/中立 | 敌对攻击 |
| 任务接取 | 正常 | 大部分不可 |
| 城镇通行 | 正常 | 被追捕 |
| 德尔菲 | 正常 | **正常——无法律限制** |
| 善值上限 | 20(40) | -100(下限) |

## 六、特殊特性效果

| 特性 | 善值惩罚倍率 | 善值上限 | 额外效果 |
|------|:--:|:--:|------|
| **罪恶之秘宝** | ×0.75(减少25%) | 0(降低20) | 核弹城中惩罚-20 |
| **善人之秘宝** | ×1.50(增加50%) | 40(提升20) | 税率减免25%/许愿↑ |
| 无特性 | ×1.0 | 20 | — |

> 两特性互斥，获取一方消除另一方。

## 七、守卫数据

| 参数 | 说明 |
|------|------|
| 攻击条件 | 目标善值≤-31且可见 |
| 特殊能力 | 将受伤罪犯传送至监狱 |
| 守卫复活 | **守卫被杀后会以更强形态复活** |
| 守卫武器掉落 | 不能捡(它们会改用格斗→更危险) |
