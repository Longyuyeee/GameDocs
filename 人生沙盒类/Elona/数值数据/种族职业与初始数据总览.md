## 三、种族×职业推荐组合

| 组合 | 类型 | 推荐理由 |

| 耶尔雷斯×农民 | 新手全能 | 最高学习+额外技能点=快速成长 |
| 魔像×战士 | 新手坦克 | 150HP+免疫昏睡+高力量=推土机 |

| 艾莱×法师 | 施法专家 | 以太病减半+低法力反冲=安全施法 |
| 蜗牛×观光客 | 终极挑战 | —(仅推荐给想证明自己的玩家) |
| 混沌体×任意 | 不可预测 | 每局都不同的随机体验 |

## 四、身体部位与装备槽

### 标准身体部位(13个)

| 部位 | 可装备 |
|------|------|
| 头部 | 头盔/帽子 |
| 颈部 | 护符/项链 |
| 背部 | 披风/翅膀 |
| 躯干 | 铠甲/衣服 |
| 腕部×2 | 护腕×2 |
| 手部×2 | 手套/武器×2 |
| 戒指×2 | 戒指×2 |
| 腰部 | 腰带 |
| 脚部 | 鞋子 |
| 远程 | 弓/弩/火器 |
| 弹药 | 箭/子弹 |

### 额外身体部位(混沌体/以太病/基因工程)

| 来源 | 最多额外部位 | 说明 |
|------|:--:|------|
| 混沌体种族 | +16 | 每3级随机获得1个(最多36级) |
| 以太病(双头) | +1头 | 意志-25 |
| 以太病(多指) | +1戒指 | 力量-25% |
| 以太病(触手) | +4触手 | 意志-10每层 |
| 基因工程 | 每融合+1 | 最多传递1部位/次 |

**"身体复杂"特性**: 超过13部位后每额外1个-5%速度。
