## 五、基因工程限制

| 参数 | 限制 |
|------|------|
| 捐献者等级要求 | ≤ 基因工程技能 + 5 |
| 传递技能数/次 | 最多2个(随机) |
| 传递属性/次 | 最多3项(捐献者最高的3属性) |
| 传递部位/次 | 最多1个 |
| 每属性提升上限 | +10/次 |
| 身体部位>13后惩罚 | 每额外1部位-5%速度 |
