# 自由度与涌现性系统 — Elin

> 游戏版本: Elin Steam EA (2025.6)
> 数据来源: [Ylvapedia Wiki](https://ylvapedia.wiki/), [AUTOMATON Noa访谈](https://automaton-media.com/en/interviews/)

---

## 一、Elin保留了Elona的自由度DNA

Elin的核心交互自由**几乎完全保留**——你可以做Elona中几乎所有的"非常规行为"。区别在于Elin在此基础上增加了新维度的自由。

### 保留的极端自由度

- 攻击任何NPC(包括友好)
- 偷窃任何角色
- 开店/定价/成为商人
- 演奏/街头卖艺
- 吃尸体/下毒/下药
- 犯罪/成为罪犯
- 杀害商人/NPC(永久消失)

---

## 二、Elin新增的自由维度

### 空间自由——你可以建造世界

| Elona | Elin |
|------|------|
| 只能使用预设的房屋内部 | **逐格建造墙壁/地板/房间** |
| 不能改变世界外观 | **你的家园改变世界地图** |
| 单人独居 | **建设城镇/招募居民** |

### 职业自由——你可以加入多个公会

| Elona | Elin |
|------|------|
| 只能加入1个公会 | **可以加入多个公会** |
| 职业身份固定 | 商人/冒险者/法师可以同时进行 |

### 经济自由——你可以选择销售模式

- 运输箱(批量/低摩擦)
- 销售标签(零售/高利润)
- 不同类型的玩家可以选择不同的经济活动方式

## 三、EA阶段的限制

### 尚未实装的内容（相对于Elona）

| 缺失内容 | Elona中的状态 | Elin预计 |
|------|:--:|:--:|
| 婚姻/结婚 | 完整 | 待实装 |
| 竞技场 | 完整 | 待实装 |
| 宠物竞技场 | 完整 | 待实装 |
| 外星人怀孕 | 有 | 待确认 |
| 混沌要塞 | 终局地城 | 待实装 |
| 拉尔纳城 | 可访问 | 地图可见但未开放 |

EA状态的Elin在某些维度上相对Elona是"不完整的"——但新增的基地建设和制作维度远超Elona。

---

## 四、涌现性的新来源

### 家园涌现

| 涌现类型 | 说明 |
|------|------|
| 居民自发行为 | 居民自主工作/消费/互动 |
| 怪物入侵 | 怪物可能杀死你的居民→形成"悲剧故事" |
| 经济涌现 | 你的家园吸引游客→游客消费→经济繁荣→吸引更多居民 |

### 制作链涌现

复杂制作链的材质转化创造了不可预测的优化空间——发现一个新的"锤子炼金术"路径是Elin特有的探索形式。

---

## 五、设计特点分析

**自由从"可做什么"扩展到"可建造什么"**：Elona的自由是**破坏性**的（你可以摧毁/盗窃/杀害），Elin在保留破坏性自由的同时增加了**建设性**的自由（你可以建造/招募/培养）。这两种自由的共存创造了一种全新的体验——你既是毁灭者也是创造者。

**EA的限制=开发者与玩家的边界测试**：noa选择在EA阶段就发布Elin——这意味着游戏在"不完整"的状态下就被玩家测试。但这种"不完整"实际上创造了另一种涌现——玩家围绕"缺失的内容"形成了社区预期和MOD开发生态。Steam Workshop的1,500个MOD已经填补了许多原本由开发者负责的空白。

---

> 跨品类对比: Elin的"破坏+建设"双重自由与Minecraft的"生存+创造"模式类似——玩家可以在同一个世界中既是毁灭者(杀怪/炸矿)也是创造者(建造/美化)。
