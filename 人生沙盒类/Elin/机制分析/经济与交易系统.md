# 经济与交易系统 — Elin

> 游戏版本: Elin Steam EA (2025.6)
> 数据来源: [Steam New Player Guide](https://steamcommunity.com/sharedfiles/filedetails/?id=3400762983), [Ylvapedia - Housing](https://ylvapedia.wiki/wiki/Elin:Housing)

## 一、与Elona经济系统的核心差异

| 维度 | Elona | Elin |
|------|------|------|
| 核心货币 | 金币+白金币 | **金币+金条(Oren/Gold Bars)** |
| 开店 | 家中设柜台 | **运输箱(批量)/销售标签(零售)**两种模式 |
| 最佳早期收入 | 老鼠牧场→肉干 | **蘑菇→酿酒** |
| 商店投资 | 投资提高商人品质 | 保留+家园折扣政策 |
| 税收 | 等级税 | **家园物品价值税**(按月评估) |
| 物品运输 | 手动搬 | **快递箱跨城镇运输** |
| NPC收购 | 极低价(0.01×) | 改进但仍不建议 |

## 二、蘑菇酿酒经济——Elin的"入门经济引擎"

### 完整价值链

```
Step 1: 工作台制作镰刀/锄头/铲子
Step 2: 种植蘑菇（任何土壤/室内外/无需阳光/1天长）
Step 3: 收获蘑菇
Step 4: 蘑菇放入酿造桶 → 等待 → 葡萄酒
Step 5: 葡萄酒放入运输箱 → 自动出售 → 金条收入
```

| 指标 | 数值 |
|------|:--:|
| 单个蘑菇→葡萄酒价值 | 数倍增值 |
| 启动成本 | 接近零（基础工具） |
| 生长周期 | **1天**（最快） |
| 全年性 | 可在室内种植（度过以太风/冬季） |

### 进阶：柴鱼片(Bonito Flakes)

钓鱼→锯木厂加工→鱼干→酿造桶→酒→更高价值

## 三、金矿雕像——中期经济

### 流程

```
1. 采矿6-7+淤泥镐→德尔菲盗贼公会下方→挖金墙→金切石
2. 金切石→雕塑台→金雕像
3. 金雕像→运输箱→大量金条
```

| 效率 | 数据 |
|------|:--:|
| 6个金雕像 | 足够付一整月税金 |
| 消耗 | 每个金块善值-1(挖金墙) |
| 进阶技 | 锤子拆除金切石→50%几率获金石→石匠台→2金切石(翻倍) |

## 四、运输箱 vs 销售标签

| 维度 | 运输箱(Shipping Crate) | 销售标签(Sales Tag) |
|------|------|------|
| 速度 | 即时(放入即售) | **慢**(等顾客购买) |
| 利润 | 中等 | **最高**（可自由定价） |
| 便利性 | 极高 | 低(需要商店+客流) |
| 适用 | 日常批量出售 | 高价值单品/特殊物品 |
| 输出 | 金条 | 金币(需手动转化为金条) |

> 推荐策略: 蘑菇酿酒等批量产品走运输箱，高价值装备/稀有道具走销售标签。

## 五、税收系统

### 家园税

| 参数 | 说明 |
|------|------|
| 计税基础 | **家园中所有物品的总价值** |
| 税率 | 基于游戏进度浮动 |
| 缴纳周期 | **每月1日** |
| 滞纳惩罚 | 善值下降(类似Elona) |
| 逃税 | 家园技能"逃税"可降低税率 |
| 居民税(政策) | 向居民收税补贴家园税 |

### 税收管理策略

- **不要把高价值物品囤积在家园中**——放在城镇银行的保险柜中
- 优先升级"逃税"家园技能
- 启用"居民税"政策创造被动收入

## 六、物流网络

### 快递箱(Delivery Box)

- 各主要城镇设有快递箱
- 存入物品→自动运送到**你的下一个基地**
- 付费服务(费用基于距离+物品量)
- 创建了**跨城镇的物品流通网络**

### 传送石(Waystone)

- 石匠台制作
- 放置后实现多点传送
- 是物流网络的空间基础设施

## 七、设计特点分析

**运输箱=刻意设计的"低摩擦变现"**：运输箱的即时出售免去了开店/等待顾客/管理库存的麻烦。这是为"想专注于冒险而非经营"的玩家设计的出口——你的蘑菇不需要等待，扔进运输箱就拿钱。

**金条=货币的分层设计**：金币用于日常消费（食物/工具/消耗品），金条用于家园升级（技能/政策/建筑）。这可防止玩家用冒险收入"跳过"家园发展——你需要通过经济运营（运输箱/销售标签）获取金条，而不仅仅是杀怪掉金币。

**蘑菇酿酒=设计者有意的经济安全网**：一个1天生长、不需要阳光、可以在室内种植的作物，配上简单一步的酿造，产生稳定现金流的完整产业链。这个设计太过完美以至于不可能是偶然的——noa刻意为新玩家设计了一条低门槛经济路线，让他们在理解游戏的其他复杂系统之前先解决"怎么活下去"的问题。

---

> 跨品类对比: Elin的运输箱+销售标签的两种销售模式与星露谷物语的"出货箱(低摩擦) vs 直接卖给商人(需要去城镇)"类似——都是给不同类型玩家提供不同摩擦级别的变现方式。金条/金币的双货币设计与Elona的白金币/金币双轨制一脉相承但用途不同。
