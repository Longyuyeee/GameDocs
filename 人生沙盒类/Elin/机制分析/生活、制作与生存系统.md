## 一、与Elona生活系统的核心差异

| 维度 | Elona | Elin |
|------|------|------|
| 制作链深度 | 中等(材料→物品) | **深(多层中间产物)** |
| 食物系统 | 烹饪等级决定产出量 | **加入腐败机制**(新鲜→微陈→陈→腐坏) |
| 同伴需求 | 不需要喂食 | **必须喂食**(不会饿死但影响状态) |
| 睡眠 | 提升潜力 | 仅恢复体力(梦想事件仅限于法术梦) |
| 工具体系 | 便携工具为主 | 固定的制作台+手持工具 |

## 二、制作系统

### 制作台网络

| 制作台 | 产出类型 | 关键配方 |
|------|------|------|
| **工作台(Workbench)** | 基础工具/设施 | 镰刀/锄头/铲子/酿造桶 |
| **铁砧(Anvil)** | 金属武器/防具/工具 | 镐子升级/金属防具 |
| **锯木厂(Sawmill)** | 木材加工 | 木板/柴鱼片(Bonito Flakes) |
| **石匠台(Mason's Table)** | 石材加工/雕像 | 传送石/金石雕像 |
| **雕塑台(Sculpting Table)** | 雕像制作 | 材质转化 |
| **地板工坊(Floor Workshop)** | 地板制作 | 土壤地板 |
| **酿造桶(Brewery Barrel)** | 酒类酿造 | 蘑菇→葡萄酒 |
| **纺织机(Loom)** | 布料/织物 | 防火/防寒毯 |
| **裁缝台(Sewing Table)** | 布甲/衣物 | — |
| **珠宝台(Jeweler's Table)** | 戒指/项链 | — |
| **炼金工具(Alchemy Kit)** | 药水/药剂 | 以太抗体/潜力药水 |

### 材质转化链——"锤子炼金术"

Elin独特的多步材质转化系统：

```
矿石 → 铸锭 → 雕塑台→雕像 → 锤子拆除→原木/木板/树枝
金属 → 塑像 → 拆除 → 木材质(材质"变性")
金块 → 金切石 → 金雕像 → 高价值商品
```

这种"材质变性"让制作链有了一层元策略——你可以通过正确的操作序列将低价值材料转化为高价值产出。

## 三、食物与腐败系统（全新）

### 腐败阶段

| 阶段 | 可食用 | 效果 |
|:--:|:--:|------|
| 新鲜(Fresh) | 是 | 全效果 |
| 微陈(Slightly Stale) | 是 | 效果略降 |
| 陈(Stale) | **可食用但风险** | 效果大幅降低 |
| 腐坏(Rotten) | 否 | 不可食用/可能致病 |

### 食物储存策略
- 腐败系统驱动了"及时加工"的经济循环
- 蘑菇→酒=将易腐的新鲜食材转化为长保质期的酒类
- 柴鱼片(Bonito Flakes)=鱼类的长期保存形式

## 四、蘑菇酿酒经济（Elin标志性玩法）

### 价值链

```
种植蘑菇(1天) → 收获 → 放入酿造桶 → 等待数小时 → 葡萄酒 → 运输箱/销售标签出售
```

| 步骤 | 所需工具 | 耗时 |
|------|------|:--:|
| 制作工具 | 工作台→镰刀/锄头/铲子 | 即时 |
| 种植蘑菇 | 任何土壤(室内外均可/无需阳光) | 1天 |
| 酿造 | 蘑菇+酿造桶 | 数小时 |
| 出售 | 运输箱(快速批量)或销售标签(高价零售) | — |

### 为什么蘑菇是基础食材之王
- **1天生长周期**(最快)
- **不需要阳光**(可以在室内/冬季种植)
- **可在任何土壤种植**
- **转换为酒=高价值保存品**

## 五、采矿与工具升级

### 镐子升级阶梯

| 镐子材质 | 所需采矿等级 | 可开采材质 |
|------|:--:|------|
| 花岗岩(Granite) | 1 | 基础岩石 |
| 黑曜石(Obsidian) | 3-5 | 基础矿石 |
| 淤泥(Silt) | 6-7 | 金块 |
| 精金/以太 | 15+ | 顶级矿石 |

> **核心原则**: 永远优先升级镐子——更高级的镐子解锁更高级的材料，驱动整个制作链。

## 六、运输与物流网络

### 运输箱(Shipping Crate)

- 放入物品→物品被运输到家园→**自动出售**(换金条)
- 最快的变现方式（但利润非最高）
- 金条是升级家园技能的货币

### 快递箱(Delivery Box)

- 每个城镇都有快递箱
- 存入物品→自动运到你的下一个基地（付费）
- 创建了跨城镇的物品流通网络

### 销售标签(Sales Tag)

- 在自己商店中使用标签定价出售
- **最高利润但最慢**
- 需要顾客流量支持

## 七、生存元素

### 同伴喂养

Elin中同伴**必须被喂食**——不会饿死但饥饿状态影响战斗表现：
- 需要为旅行的同伴携带食物
- 同伴自主从共享容器中取食
- 蘑菇的高产量使其成为同伴食物的首选

### 工具耐久度
- 工具(镰刀/镐子/斧头)有耐久度
- 需要维修或更换
- 高级材质工具耐久度更高

## 八、设计特点分析

**制作链的纵深=从"做什么"到"怎么转化"**：Elona的制作是"材料→成品"的简单两步。Elin引入了中间产物和材质转化——矿石→铸锭→雕像→拆除→另一种材质。这种多步转化让"制作"从一次性的配方执行变成了需要策略规划的**产业链优化游戏**。

**蘑菇酿酒=刻意设计的入门经济引擎**：1天生长的蘑菇+酿造桶=稳定现金流。这显然不是巧合——noa刻意设计了这条低门槛经济路线作为新玩家的安全网。它同时教授了制作系统(制作工具/酿造桶)、农业系统(种植+收获)、和经济系统(运输箱/定价)。

**腐败系统=反囤积机制**：食物腐败为制作链增加了时间压力——你不能无限囤积原材料，必须及时加工。这驱动了"收获→立即加工→出售"的节奏循环，防止了"仓库堆满食材"的静态囤积玩法。

---

> 跨品类对比: Elin的蘑菇酿酒的入门路径与星露谷物语的"春季防风草→留种→扩大"惊人地相似——都是刻意设计的低门槛经济引擎。Elin的腐败系统比Don't Starve的食物腐败更宽松(只有4个阶段，没有立即惩罚)，设计重心是"驱动加工"而非"增加生存难度"。
