# 属性与技能成长系统 — Elin

> 游戏版本: Elin Steam EA (2025.6)
> 数据来源: [Ylvapedia Wiki](https://ylvapedia.wiki/), [Steam社区指南](https://steamcommunity.com/)

---

## 一、属性体系

Elin保留了Elona的九大主属性体系，但引入了重要调整：

| 属性 | Elona核心影响 | Elin变化 |
|------|------|------|
| 力量(STR) | 近战伤害/负重 | 切割领域法术也受力量影响 |
| 体质(CON) | HP/耐力 | 基本不变 |
| 敏捷(DEX) | 回避/远程命中 | 基本不变 |
| 感知(PER) | 远程伤害/鉴定 | 基本不变 |
| 学习(LER) | 技能点/级 | **特性经验也受学习影响** |
| 意志(WIL) | HP/MP/信仰 | 基本不变 |
| 魔力(MAG) | 法术成功率/MP | 元素领域法术仍受魔力影响 |
| 魅力(CHR) | 宠物数/价格 | **心灵领域法术受魅力影响**(召唤术) |
| 速度(SPD) | 回合频率 | 基本不变 |

### 核心变化：法术领域×属性绑定

Elona中"所有法术用魔力"的单一依赖被打破——不同法术领域关联不同属性：
- 切割(Cut)领域→力量
- 心灵(Mind)领域→魅力
- 元素(Element)领域→魔力
- 法师需要更均衡的属性配置

## 二、潜力(Potential)系统

Elin保留了Elona的潜力机制核心——属性/技能潜力决定经验转化效率——但调整了提升方式：

| 提升方式 | Elona | Elin |
|------|------|------|
| 白金训练 | 核心方式 | 保留 |
| 睡眠 | 做梦提升随机潜能 | **仅法术梦**(不再有宝藏梦) |
| 城镇自动训练 | 宠物消耗金币自动训练 | 保留(但机制有调整) |
| 料理早餐 | 宠物烹饪→全体潜力+ | 调整后 |
| 训练假人 | **无** | **新增被动训练**(见战斗系统) |

## 三、特性(Feat)经验系统

### 特性 vs 技能的区分

Elin引入了独立于技能的**特性(Feat)经验**系统：
- 每个种族有专属特性（如耶尔雷斯的"快速学习者"）
- 特性通过**特定行为**获取经验（而非通用经验值）
- 特性升级=特性效果增强
- 耶尔雷斯的"快速学习者"=特性经验加速

### 特性 vs 技能的成长协同

```
技能使用 → 技能经验+潜能衰减 → 技能等级提升 → 更强角色
                                              ↓
特性行为 → 特性经验 → 特性等级提升 → 特性效果增强
```

两个平行的成长线互不替代但相互增强。

## 四、培训系统变化

### Elin的改进

| 维度 | Elona | Elin |
|------|------|------|
| 训练师位置 | 各城镇分散 | 集中+家园内可雇佣 |
| 训练假人 | 无 | **沙袋/木人/吊桩**→被动训练技能 |
| 开锁练习 | 需实际盗窃 | **可重置练习箱** |
| 同伴训练 | 城镇自动 | 可使用训练假人绑同伴训练 |

## 五、成长阶段

Elin的成长曲线与Elona相似但加入了家园维度的扩展：

| 阶段 | 主要活动 | 新增的Elin元素 |
|:--:|------|------|
| 初期(1-15) | 基础属性+技能+生存 | **蘑菇酿酒启动经济** |
| 中期(16-40) | 技能多样化+装备迭代 | **基础建设+居民招募** |
| 后期(41-80) | 极限优化+材料追求 | **高级制作链+政策优化** |
| 终局(80+) | 全系统完美化 | **多基地网络+全居民高效率** |

## 六、设计特点分析

**法术领域=属性的解放而非限制**：将不同法术领域绑定到不同属性看似"限制"了法师，实则是解放——它让非法力属性（力量/魅力）也有了施法的"出口"。一个高力量的战士可以投资切割领域法术（受力量影响），实现"剑与魔法"的混合Build，而不需要分点给魔力。

**特性经验=种族的长期相关性**：在大多数RPG中，种族的初始优势在5-10级后就消失了。Elin的特性经验系统确保种族特性在整个游戏过程中持续成长——一个50级的耶尔雷斯和一个50级的魔像不仅在属性上有差异，在特性强度上也有差异。

**训练假人=消除"不敢尝试新技能"的心理障碍**：在传统Roguelike中，尝试新武器类型需要冒风险（用低技能武器=打不过敌人）。Elin的训练假人让玩家可以在安全环境中将新技能训练到可用水平再出门冒险——这是一个重要的"降低试错成本"设计。

---

> 跨品类对比: Elin的特性经验系统类似于RimWorld的" passions"(热情)——每个角色的成长速度和方向不仅由通用属性决定，还有个体差异。训练假人是Kenshi式的"挂机训练"机制在Roguelike中的移植。
