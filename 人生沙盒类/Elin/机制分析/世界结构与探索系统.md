# 世界结构与探索系统 — Elin

> 游戏版本: Elin Steam EA (2025.6)
> 数据来源: [Ylvapedia Wiki](https://ylvapedia.wiki/), [Steam Tips Guide](https://steamcommunity.com/sharedfiles/filedetails/?id=3401513288)

---

## 一、与Elona世界结构的差异

| 维度 | Elona | Elin |
|------|------|------|
| 地图风格 | 固定骨架+随机地城 | **更多手工制作+随机生成融合** |
| 图形视角 | 俯视2D | **等距2D(Isometric)** |
| 操作方式 | 小键盘为核心 | **完整鼠标支持**(左键移动/右键交互/中键菜单) |
| 控制选项 | 单一方案 | WASD/鼠标/键盘多方案可配置 |
| CTRL+F搜索 | 无 | **可搜索物品和NPC** |
| 同伴面板 | 文本列表 | **头像图标可直接点击交易** |
| 运输 | 手动搬运 | **快递箱网络+传送石** |
| 固定地城 | 34个 | 重构中(EA——部分未实装) |

| 地点 | Elin变化 |

## 三、探索方式

### 移动与导航

| 方式 | 说明 |
|------|------|
| 步行 | 传统探索 |
| 骑乘 | 马/宠物骑乘 |
| **快速旅行** | 新增——消耗旅行口粮(fast travel rations) |
| 传送石(Waystone) | 石匠台制作→放置→形成传送网络 |
| 回城 | 保留 |
| 炉石(Hearthstone) | 跨基地传送 |

### 快速旅行的代价

Elin的快速旅行比Elona的步行+传送慢（步行在高旅行技能时很快），但提供了便利性——是一种"时间换便利"的权衡。

## 四、地城(Nefia)系统

### Elin的地城特点

- 地城面积比Elona**更小/更紧凑**
- 固定地城回归但数量可能减少(EA)
- 随机Nefia保留刷新机制
- **无地城排行榜**（替代为战士公会排名任务）

### 采集地点

| 地点类型 | 采集内容 |
|------|------|
| 海滩 | 蘑菇/针叶/漂流物 |
| 山脉地图 | 采矿/(黑曜石/金属) |
| 森林 | 药草/木材/猎物 |
| 地城内部 | 矿石/宝箱/怪物掉落 |

## 五、城镇功能性设施

### 井水刷新

Elin中城镇的**井水(Well)会随时间重新注满**——许愿需要**幸运神像(Lucky God Statue)**，且每年仅一次。

### 快递箱网络

每个主要城镇都有快递箱——存入物品→付费→自动运到你的下一个基地。这是贯穿整个世界的物流基础设施。

---

## 六、设计特点分析

**等距视角+鼠标操作=从Roguelike到现代CRPG**：Elin的等距2D视角和完整的鼠标操作标志着从传统Roguelike(键盘驱动/ASCII式UI)到现代CRPG(鼠标驱动/图形化UI)的转变。这不是"降低深度"，而是**降低进入摩擦**——同样的系统深度，更直观的操作方式。

**传送石+快递箱=玩家创建的基础设施网络**：在Elona中，你只能依赖游戏预设的传送点和手动搬运。Elin的传送石和快递箱将物流网络的建设权交给了玩家——你可以选择在哪里放置传送石、哪些物品走快递路线。这种"玩家创建基础设施"的设计为探索增加了一层策略性。

**CTRL+F搜索=QoL即设计**：在Elona中找一个特定NPC或物品需要遍历整个地图。Elin的CTRL+F搜索功能似乎是"纯QoL改进"，但它实际上改变了一个核心设计假设——**信息不对称不再是难度来源**。在Elona中"找不到人"是一种有意的摩擦，在Elin中这个摩擦被移除了。

---

> 跨品类对比: Elin的等距视角+鼠标操作让人联想到Ultima Online或Divinity: Original Sin的视角——与传统Roguelike的俯视键盘操作的差距类似于Windows GUI与DOS命令行的差距。快递箱网络类似于Terraria的传送塔网络——都是玩家自己构建的快速移动基础设施。
