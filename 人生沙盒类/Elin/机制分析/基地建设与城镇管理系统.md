# 基地建设与城镇管理系统 — Elin

> 游戏版本: Elin Steam EA (2025.6)
> 数据来源: [Ylvapedia - Housing](https://ylvapedia.wiki/wiki/Elin:Housing), [Steam Tips Guide](https://steamcommunity.com/sharedfiles/filedetails/?id=3401513288)

---

## 一、系统概述

这是Elin相比Elona**最大的新增系统**——从"有一个可以放家具的家"到"建造和管理一座城镇"。基地建设为Roguelike沙盒增加了一个全新的空间维度。

---

## 二、家园(My Home)核心机制

### 基本结构

| 要素 | 说明 |
|------|------|
| **地域** | 在世界地图选择一个地点建立家园 |
| **建筑** | "四面墙+一扇门"=一个房间 |
| **地板/墙壁** | 可逐格编辑，支持多层结构 |
| **搬迁** | 可通过搬迁契约迁移到新的世界地图位置 |

### 家园等级(Building Level)

| 等级 | 解锁功能 |
|:--:|------|
| Lv1 | 基础建造 |
| Lv2 | **编辑地块**(铲雪/平地/等) |
| Lv3+ | 更多建造选项/高级建筑 |

---

## 三、居民(Residents)系统

### 居民职业——40+种工作

每个居民有一个**主职业**和一个**爱好**。效率取决于居民技能等级、床品质、好感度、家园特性。

| 职业 | 所需设施 | 产出/效果 |
|------|------|------|
| **伐木工** | 伐木标志 | 原木 |
| **铁匠** | 铁砧 | 矿石加工 |
| **厨师** | 烹饪设施 | 料理/调味料 |
| **渔夫** | 钓鱼标志 | 鱼(稀有几率小奖章) |
| **农夫** | 农场标志 | 土壤/应急口粮 |
| **园丁** | 农场标志 | 土壤/应急口粮 |
| **猎人** | 无 | 动物尸体/霜降肉 |
| **历史学家** | 书架 | 古书 |
| **歌手** | 麦克风 | 提升地方爱国心 |
| **探索者** | 无 | **宣传+金条** |
| **清洁工** | 无 | 清扫污渍/垃圾 |
| **护士** | 无 | 提升公共道德/牛奶/自动复活居民 |
| **研究员** | 书架 | 提升逃税 |
| **购物者** | 无 | 提升家园折扣 |
| **宠物** | 猫/狗屋 | 行政/安保 |
| **画家** | 画架 | 书籍/牛奶 |
| **祈祷者** | 无 | 公共道德/赎罪信 |
| **博主** | 电脑 | 宣传 |
| **锻炼/冥想** | 无 | 自然发电 |
| **奢侈者** | 无 | 缴纳大量税金 |

### 居民管理

| 操作 | 方式 |
|------|------|
| 招聘居民 | 家园吸引游客/特定方式招募 |
| 指定职业 | 通过家园面板分配 |
| 更改职业 | **爱之鞭(Whip of Love)**(主线任务获取) |
| 迁移居民 | 带为同伴旅行/使用炉石 |
| 设为家畜 | 消耗草产奶/蛋，不占人口配额 |

## 四、家园技能(Home Skills)

通过**房屋看板**升级，消耗金条(Gold Bars)：

| 技能 | 效果 | 升级成本 |
|------|------|:--:|
| **建筑(Construction)** | 解锁更多建造功能 | 技能Lv+5金条/Lv |
| **自然发电** | 为电器/工具供电 | 同上 |
| **逃税(Tax Evasion)** | 减少月度税金 | 同上 |
| **勤学(Studious)** | 提升居民工作效率 | 同上 |
| **行政(Administration)** | 提供行政点数(用于启用政策) | 同上 |
| **宣传(Publicity)** | 提升游客数量 | 同上 |
| **幸运(Luck)** | 改善运气相关结果 | 同上 |
| **地方爱国心** | 提升居民好感增长 | 同上 |
| **土壤(Soil)** | 提升土地肥力 | 同上 |
| **食品供应** | 增加人口上限/减少超载惩罚 | 同上 |

## 五、政策(Policies)系统

通过消耗**行政点数**启用政策，政策随使用升级且有独立潜能：

| 部门 | 政策 | 效果 | 行政成本 |
|------|------|------|:--:|
| 经济 | **居民税** | 向居民征税(被动收入) | 4 |
| 经济 | 家园折扣 | 降低家园内商人价格 | 6 |
| 治安 | 公共安全 | 降低家园内怪物等级 | 3 |
| 福利 | 严格禁扔垃圾 | 减少垃圾生成量 | 2 |
| 福利 | 除草运动 | 减少杂草生成量 | 2 |
| 经济 | 开放营业 | 允许游客消费 | — |

## 六、家园防御

### 怪物入侵

Elin中的家园会受到**怪物刷新**的威胁——怪物会杀死居民：

| 应对方式 | 说明 | 成本 |
|------|------|:--:|
| **武装居民** | 给居民高品质装备(龙鳞/精金/以太) | 高(需要大量优秀装备) |
| **围墙封锁** | 用墙完全围住基地防止怪物进入 | 中(建造时间) |
| **柱子卡位** | 用地形工具创建高柱卡住怪物 | 低(最便宜但取巧) |
| **刷满怪物上限** | 达到刷新上限后新怪物不再出现 | 间接方式 |

## 七、搬迁与多基地

- **搬迁契约(Base Relocation Deed)**：将家园整体搬迁到世界地图新位置
- **炉石(Hearthstone)**：多基地之间传送/居民迁移
- 每个基地独立管理(独立的家园技能/政策)

## 八、设计特点分析

**基地建设=空间化的"为什么战斗"**：Elona中你战斗是为了变强——而变强是为了什么？Elin的回答是：**为了建设你的家园**。基地建设为整个游戏提供了终极目标——你打地城获取稀有材料→制作更高级的家具/设施→提升家园等级→吸引更多居民→解锁更多功能。这是一种"从力量到创造"的价值转化链。

**居民系统=从独行侠到社区领袖**：Elona是独行侠式的——只有你、你的宠物、和你偶尔互动的NPC。Elin的居民系统将玩家从冒险者转变为社区管理者——你需要招聘、分配工作、提升好感、解决居民的问题。这种"管理者"的角色是Elona完全没有的维度。

**政策系统=城镇的Build定制**：不同的政策组合创造出不同的城镇风格——高税收工业城/低税文化旅游城/自给自足农业镇。政策的升级+潜能机制（每个政策有独立等级和潜能）将"城镇管理"也变成了一个有深度的Build游戏。

---

> 跨品类对比: 与RimWorld的基地建设不同——RimWorld是生存驱动的（你必须建造否则会死），Elin的基地建设是表达驱动的（你建造来表达你的世界）。与Stardew Valley的"固定农场地块"不同——Elin的建造是网格自由的，更接近Terraria。与Elona的对比见特色文档《基地建设：Roguelike沙盒的空间化转折》。
