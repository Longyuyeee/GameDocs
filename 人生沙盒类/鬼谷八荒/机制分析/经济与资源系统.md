# 经济与资源系统 — 鬼谷八荒

> 游戏版本: 鬼谷八荒 v1.0 (Steam, 2023)
> 数据来源: [3DM攻略站](https://www.3dmgame.com/gl/3824864.html)

---

## 一、货币体系

| 货币 | 获取 | 用途 |
|------|------|------|
| **灵石** | 任务/贩卖/拍卖/宗门贡献兑换 | 购买物品/拍卖竞价/宗门贡献 |
| **宗门贡献** | 宗门任务/降妖 | 藏经阁兑换功法/聚宝仙楼换材料 |
| **道点** | 论道/战斗 | 学习/参悟新功法的"知识货币" |

| 来源 | 效率 | 门槛 |
|------|:--:|------|
| 拍卖会出售 | 极高(高品级物品) | 需要有好东西卖 |
| 资源点占领 | 中(持续被动) | 战斗力+宗门支持 |

| 聚宝仙楼兑换→贩卖 | 中-高 | 宗门贡献兑换材料→出售 |
| 打怪掉落 | 低-中 | 持续战斗 |

## 四、拍卖会——经济体系的核心枢纽

| 参数 | 数据 |
|------|:--:|
| 周期 | **每12月1次**(各主城) |
| 可竞拍物品 | 高品级功法/橙色装备/稀有材料/太古残卷 |
| 可出售物品 | 玩家持有的任何高价值物品 |
| 首次宗主后 | 太古残卷可能出现(可改逆天改命) |

## 五、设计特点分析

**灵石不是"通用资源"**：很多修仙游戏把灵石设为万能货币。鬼谷通过宗门贡献/道点的并行货币体系，将不同类型的成长绑定到不同的行为上——你不可能只靠"有钱"就学会宗门最顶级的功法(需要宗门贡献)或参悟出完美词条(需要参悟材料+道点)。

**拍卖会=12月一次的经济高潮**：每12月一次的拍卖会是整个游戏经济的"发条"——玩家会积累12个月的资源，然后在拍卖会上集中释放(购买/出售)。这种"储蓄→释放"的节奏创造了自然的"经济期待感"。
