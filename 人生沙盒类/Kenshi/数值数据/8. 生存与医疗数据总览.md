### 1.2 最佳食物推荐

| 场景 | 最佳食物 | 理由 |
|------|---------|------|
| 长途旅行（轻便优先） | 脱水口粮 (Dried Food) | 营养/重量比 = 100.0，重量仅 0.3kg，保质期 180 天 |
| 居家 / 基地（经济优先） | 蜂人面包 / 烤生肉 | 原材料易得，制作技能要求低 |
| 探险（综合考量） | 干肉 (Dried Meat) | 营养/重量比 66.7，可大批量从肉干制，保质期极长 |
| 小团队初期 | 生蔬菜 + 面包 | 最便宜的城镇购入选择 |
| 培养烹饪技能 | 从生肉→烤生肉→脱水口粮 | 循序渐进的技能树 |

## 二、饥饿速率

### 2.1 各基础种族饥饿衰减速度

| 种族 | 饥饿衰减倍率 | 说明 | 满饥饿值 (100%) 耗尽时间 |
|:----:|:----------:|------|:----------------------:|
| 人类 (绿原人/焦土人) | 1.0x（基准） | 标准代谢 | ~72 小时（游戏内 3 天） |
| 沙克族 (Shek) | **1.25x** | 更快饥饿，需要更多食物 | ~58 小时（约 2.4 天） |
| 蜂人 (Hive) | **0.5x** | 一半饥饿速度 | ~144 小时（约 6 天） |
| 骨人 (Skeleton) | **0.0x** | **不需要进食** | 永不饥饿 |
| 蜂人王子 (Hive Prince) | 0.5x | 同蜂人标准 |
| 蜂人士兵 (Hive Soldier) | 0.5x | 同蜂人标准，但可食用生肉 |
| 蜂人女皇 (Hive Queen, MOD) | 0.6x | 略快于普通蜂人 |

### 2.2 饥饿状态效果

| 饥饿百分比 | 效果 | 视觉提示 |
|:---------:|------|---------|
| 100%~51% | 正常 | 无 |
| 50%~26% | 饥饿 — 开始减少力量/敏捷 | 角色开始出现肋骨可见 |
| 25%~1% | 极度饥饿 — 各项属性大幅降低（-20%~-40%），体力恢复减慢 | 肋骨明显突出 |
| 0% (空腹) | **开始掉血（每分钟 -1~3 HP）**，到达 0 HP 则死亡 | 角色倒下/趴地 |

> **策略提示**: 骨人是 Kensing 中唯一不需要进食的种族，这使得大后期组建骨人军团时后勤压力最小。蜂人的低饥饿速率非常适合长途探险队。沙克族虽然战斗属性优秀，但食物消耗是最快的——带沙克队需要准备更多干粮。

### 3.2 医疗效果详解

| 伤口类型 | 严重程度 | 可用医疗物品 | 效果 | 自然愈合速度 |
|---------|:-------:|-------------|------|:-----------:|
| 切割伤口 (Cut) | 轻~重 | 所有绷带 | 止血 + 治疗 | 慢（若有绷带止血） |
| 钝器伤 (Blunt) | 轻~重 | 所有绷带 | 治疗瘀伤 | 一般 |
| 骨折 (Fracture) | 轻~重 | 急救包 (Splint Kit) | 减少行动/攻击惩罚 | 极慢 |
| 流血 (Bleeding) | x0.1~x2.0 | 所有绷带 | 止血 | 不会自愈，必须用绷带 |
| 骨人电路损伤 | 轻~重 | 维修包 | 恢复 HP，修复电路 | 不会自愈 |

### 3.3 医疗技能影响

- **医疗技能 < 伤口等级**: 治疗失败，或效果大幅降低。伤口等级 1~5，对应绷带类型如上。
- **医疗技能 50+**: 可以处理所有类型伤口，且治疗效果有加成（+20%~40% 额外治疗量）。
- **医疗技能 80+**: 治疗时几乎总是"完美包扎"，恢复量达到最大值。

### 4.1 假肢品质等级

## 五、天气危害

### 5.1 各区域天气危害

| 天气类型 | 发生区域 | 伤害速率 | 防护要求 | 影响 |
|---------|---------|:-------:|---------|------|
| 酸雨 (Acid Rain) | 洪水地、斯托布的果园、斯托比的赌注、灰烬之地、死亡之地 | 1~5 HP/秒（基于浓度） | 全身防酸装备（防酸外套 + 防酸面具 + 防酸裤）或骨人（免疫） | 对有机生物持续伤害 |
| 沙尘暴 (Sandstorm) | 大沙漠、骨场 | 0（不直接扣血） | 面罩/防尘面具（可堆叠） | 降低可见度和远程精度 |
| 毒气 (Poison Gas) | 沼泽（部分地区） | 1~3 HP/秒 | 防毒面具（Gas Mask） | 吸入后持续扣血 |
| 太阳光束 (Sunbeam) | 赎罪之谷 | 5~15 HP/秒（日间） | 阴影遮蔽/室内 | 白天在户外暴露即被灼烧 |
| 大雾 (Fog) | 雾岛、薄雾山 | 0 | 无 | 极低可见度（约 5m），容易迷路 |
| 飓风 (Hurricane) | 绿色之地 | 0 | 无 | 轻微减速，偶尔将物品吹飞 |

|---------|---------|:----------:|---------|:-----:|

## 六、奴隶镣铐系统

### 6.1 镣铐类型与参数

| 镣铐名称 | 枷锁级别 | 锁撬难度 | HP | 价格 (Cats) | 获得方式 |
|---------|:-------:|:-------:|:--:|:----------:|---------|
| 铁镣 (Iron Shackles) | 1 | 20~40 | 50 | — | 奴隶营固定装备 |
| 铁项圈 (Iron Collar) | 1 | 15~30 | 30 | — | 奴隶标记 |
| 锁链 (Chain) | 2 | 40~60 | 80 | — | 高级奴隶/囚犯 |
| 强化镣铐 (Reinforced Shackles) | 3 | 60~90 | 120 | — | 重要囚犯专用 |
| 远古镣铐 (Ancient Shackles) | 4 | 80~120 | 200 | — | 天龙要塞囚徒 |

### 6.2 逃脱方式

| 逃脱方法 | 成功条件 | 风险 | 说明 |
|---------|---------|:----:|------|
| 撬锁逃脱 (Lockpick) | 撬锁技能 ≥ 镣铐难度 | 被抓到=增加惩罚（锁链升级） | 最常用方法，需持续提升撬锁技能 |
| 暴力挣脱 (Strength Escape) | 力量 > 镣铐 HP 恢复速度 | 可能被守卫攻击 | 成功率较低，仅适合高力量角色 |
| 外力解救 | 其他队友 / NPC 帮忙 | 无直接风险 | 需要外部队友 |
| 被购买/解救 | 支付金币 | 无 | 特定城镇奴隶商人可用金钱购买囚犯 |
| 叛变/夺权 | 杀死/打晕所有守卫 | 极高（可能被全部守卫群殴） | 大后期玩法 |

### 6.3 被奴役后的游戏机制

| 机制 | 说明 |
|------|------|
| 强制劳动 | 被奴役后将被派往矿山强制采矿 |
| 逃跑惩罚 | 被抓回后镣铐升级，增加撬锁难度 |
| 喂食 | 奴隶主会提供最低限度的食物（饥饿条不会到 0） |
| 属性训练 | 在奴役期间可偷偷训练撬锁、潜行和力量 |
| 逃脱时机 | 夜间守卫注意力降低时是最佳逃脱时机 |
| 自由后再遇 | 逃脱后再次进入同一区域可能被重新抓捕 |

> **奴隶系统最有意思的设计**: 被奴役在 Kenshi 中**不一定是坏事**。许多玩家故意被 Rebirth 营地抓去当奴隶，利用"强制劳动"的时间安全训练力量和运动技能，同时在夜间偷学撬锁——"在奴役中变强"是 Kenshi 最独特的游戏体验之一。
