## 一、核心原则：做即学习 (Use-to-Level)

Kenshi 的角色成长系统没有"升级→加点"的抽象层。所有技能的提升完全基于实际操作。这是 Kenshi 最核心的设计哲学：

- **你做了什么事，你就提升什么技能**
- **没有技能上限**（理论值 100，但 90+ 后边际效益极高）
- **越级面对更强敌人 = 更快成长**
- **100% 的训练效率 = 你做这件事本身** 而不是"读一本书加 5 点"

## 二、全技能最佳训练方法

### 2.1 战斗技能

| 技能 | 最佳训练方式 | 推荐地点 | 预期升级速度(0→50) | 注意事项 |
|:----:|------------|---------|:-----------------:|---------|
| **近战攻击 (Melee Attack)** | 与敌对 NPC 实战（技能越高→需要越强对手） | Squin 城外土匪 → 雾岛雾民 → 沙克掠夺者 | 20~30 小时实战 | 使用"重武器"同时练力量和近战攻击 |
| **近战防御 (Melee Defense)** | 被多人围攻（防御经验 = 被攻击次数×敌人技能差） | 需要 2~4 个低等级敌人训练 | 15~25 小时 | **训练塔 (Training Tower)** 可练到 Lv15~20 |
| **闪避 (Dodge)** | 同一训练塔，或在实际战斗中被攻击时自动闪避 | 武术训练塔 | 15~20 小时 | 穿重甲无法闪避！必须轻甲/无甲 |
| **武术 (Martial Arts)** | 徒手战斗，拳击敌人 | 武术训练塔（可练到 15~20）→ 低威胁敌人实战 | 25~35 小时 | 武术伤害与力量+武术技能挂钩，初期伤害极低 |
| **精准射击 (Precision Shot)** | 使用远程武器射击 | 炮塔打路过的饥饿土匪 | 20~30 小时 | 友伤极危险！建议在空旷区域训练 |
| **炮塔操作 (Turret)** | 使用炮塔自动射击来袭敌人 | 基地城墙 | 随敌袭来自动成长 | 需要先建炮塔和弹药 |

### 2.2 身体素质技能

| 技能 | 最佳训练方式 | 推荐地点 | 预期升级速度(0→50) | 效率倍率说明 |
|:----:|------------|---------|:-----------------:|:-----------:|
| **力量 (Strength)** | **背包装满铁矿石 → 在安全区域来回走**（负重 70%+ 经验最快） | The Hub 到 Squin 往返路（安全） | 10~15 小时 | 负重 0%=0 经验；50%=1x；70%+=1.5x；100%=2x |
| **敏捷 (Dexterity)** | 使用切割伤害武器（武士刀/军刀）与敌人战斗 | 实战 | 20~30 小时 | 武器伤害类型中切割比例越高→敏捷经验越多 |
| **坚韧 (Toughness)** | **找到沙克战士/高伤害敌人 → 故意被打倒 → 站起来 → 再被打倒 → 重复** | Admag 城外沙克巡逻队 | 8~12 小时 | **被打倒后站起有大额坚韧经验加成**；被打磨血时经验更多 |
| **运动 (Athletics)** | **无负重 → 在安全区域持续跑步** | The Hub → Stack 往返（最直的路） | 12~18 小时 | 负重越低经验越多；无负重时效率最高 |
| **潜行 (Stealth)** | 在城镇中潜行并保持"未发现"状态绕圈 | The Hub （晚上最佳） | 15~20 小时 | 在敌人旁边潜行额外经验；被发现后经验减少 |

### 2.3 非战斗技能

| 技能 | 最佳训练方式 | 推荐地点 | 预期升级速度(0→50) | 注意事项 |
|:----:|------------|---------|:-----------------:|---------|
| **暗杀 (Assassination)** | **找到昏迷NPC → 反复暗杀（失败不产生仇恨）** | 任意城镇酒吧（夜间睡觉的 NPC） | 5~8 小时 | 暗杀失败 NPC 会醒来但不会攻击（无仇恨），可继续尝试 |
| **偷窃 (Thievery)** | **偷物品 → 放回去 → 再偷 → 重复** | 任意商店 | 8~12 小时 | 偷窃成功→放回→再偷同一物品可循环练。被发现 = 罚款/入狱 |
| **撬锁 (Lockpick)** | 反复撬锁（自己的笼子/箱子或城镇中的锁） | 家里的箱子或监狱笼子 | 6~10 小时 | 越难的锁经验越多。从简单锁开始逐步升级 |
| **医疗 (Medicine)** | 给受伤队友反复包扎 | 战斗后或故意让队友受轻伤 | 10~15 小时 | 包扎已愈合的伤口不给经验 |
| **工程 (Engineering)** | 建造/维修建筑、制造电子元件 | 基地 | 15~25 小时 | 建造城墙是最快的工程训练方式 |
| **武器制造 (Weapon Smithing)** | 大量制造低级武器（如铁棒） | 锻造台 | 20~30 小时 | 前期亏损 → 后期盈利。用低级武器练技能最省材料 |
| **盔甲制造 (Armor Smithing)** | 大量制造低级盔甲（布料绑腿） | 盔甲台 | 20~30 小时 | 布料绑腿是最省材料的训练对象 |
| **烹饪 (Cooking)** | 大量做饭 | 厨房/篝火 | 10~15 小时 | 从烤生肉开始 → 进阶到脱水口粮 |
| **农耕 (Farming)** | 种田、收割 | 农场 | 15~20 小时 | 蔬菜是最快的训练对象 |

## 三、战斗经验倍率系统

### 3.1 对手强度与经验倍率

Kenshi 的战斗经验系统鼓励"越级打怪"。经验倍率基于**你的技能 vs 对手技能**的对比：

| 你的技能 - 对手技能 | 经验倍率 | 评价 |
|:-----------------:|:-------:|------|
| ≥ +50（碾压） | 0.1x~0.3x | 几乎不涨经验，太弱了 |
| +20~+49（大幅领先） | 0.3x~0.6x | 效率低 |
| -5~+19（势均力敌） | 1.0x（基准） | 正常成长 |
| -10~-6（略逊） | 1.2x~1.5x | 高效 |
| **-20~-11（劣势）** | **1.5x~2.0x** | **最高效训练区间** |
| -30~-21（大幅劣势） | 2.0x~2.5x | 风险很高 |
| ≤ -31（绝望级） | 2.5x~3.0x | 可能被秒杀 |

### 3.2 最佳战斗训练策略

| 阶段 | 推荐对手 | 技能区间 | 安全措施 |
|------|---------|:-------:|---------|
| 初期 (1~15) | 饥饿土匪 | 双方技能均低 | 组队 2~3 人围殴 |
| 前期 (15~30) | 沙漠土匪 / 黑砂忍者 | 从势均力敌到劣势 | 队友在旁救援，防止失血过多 |
| 中期 (30~50) | 沙克掠夺者 / 联合城市武士 | 劣势 → 大幅劣势 | 穿好甲，准备好医疗物资 |
| 中期 (40~60) | 雾民 / 食人族 | 势均力敌 → 劣势 | 注意不要被雾民拖走 |
| 后期 (50~75) | 天龙军团 / 铁蜘蛛 | 劣势 → 大幅劣势 | 全队顶级装备，大量医疗包 |
| 终极 (75+) | Cat-Lon / 巨型机器人 | 绝望级 | 全队大师装备 + 义肢 + 极限属性 |

## 四、训练设备

### 4.1 训练假人 (Training Dummy)

| 属性 | 轻型假人 | 重型假人 (Weighted) |
|------|:-------:|:-----------------:|
| 训练技能 | 近战攻击 (Melee Attack) | 近战攻击 + 少量力量 |
| 技能上限 | **Lv 15**（超过 Lv15 后无效） | **Lv 15** |
| 经验效率 | 低（约等于 Lv5 对手） | 中（约等于 Lv8 对手） |
| 建造费用 | 建筑材料 ×3、铁件 ×2 | 铁板 ×6、建筑材料 ×4、铁件 ×8 |
| 是否消耗 | 否，永续使用 | 否，永续使用 |

### 4.2 武术训练塔 (Martial Arts Training Pole)

| 属性 | 数值 |
|------|:----:|
| 训练技能 | 武术 (Martial Arts) + 闪避 (Dodge) |
| 技能上限 | 武术 Lv 20 / 闪避 Lv 20 |
| 经验效率 | 中低（武术约 Lv10 对手水平，闪避约 Lv8） |
| 建造费用 | 铁板 ×2、建筑材料 ×3 |
| 是否消耗 | 否，永续使用 |

### 4.3 训练假人的局限性

- 训练设备只能将技能提升到 Lv15~20，此后必须通过真实战斗继续提升
- 训练假人完全不提供坚韧 (Toughness) 经验——这是必须通过受伤获得的技能
- 训练设备不消耗材料，但建造需要一定的科技等级

## 五、角色属性极限

### 5.1 各属性边际收益递减

| 技能等级 | 相对于 Lv1 的效率倍数 | 备注 |
|:-------:|:------------------:|------|
| 1~20 | 线性增长（每级约 +5% 效果） | 新手期，成长最快 |
| 21~50 | 每级约 +2%~3% 效果 | 稳定成长 |
| 51~75 | 每级约 +1%~1.5% 效果 | 技能熟练期 |
| 76~89 | 每级约 +0.5%~1% 效果 | 高手区间 |
| **90~100** | **每级约 +0.1%~0.3% 效果** | **极度过剩** |

### 5.2 技能 100 = 理论极限

- Kenshi 的每个技能理论最大值是 **100**
- 达到 100 需要极端大量的训练时间（通常 200~500+ 小时的单技能训练）
- **99→100 一点需要的经验约等于 1→50 全部经验的总和**
- 少数 NPC（如 Cat-Lon 的近战攻击）拥有超过 100 的技能（通过 FCS 编辑）
- **100 后继续训练不会继续增长**（硬上限）

### 5.3 实际训练时间估算

| 技能 | 0→50 | 50→75 | 75→90 | 90→100 |
|:----:|:----:|:-----:|:-----:|:------:|
| 力量 | 15h | 30h | 60h | 150h+ |
| 敏捷 | 25h | 40h | 80h | 200h+ |
| 坚韧 | 10h | 20h | 40h | 100h+ |
| 运动 | 15h | 25h | 50h | 120h+ |
| 近战攻击 | 25h | 45h | 90h | 220h+ |
| 暗杀 | 6h | 15h | 35h | 90h+ |
| 撬锁 | 8h | 18h | 40h | 100h+ |
| 武器制造 | 25h | 40h | 80h | 180h+ |

> **注**: 以上为"纯目的性训练"的估算时间。正常游玩过程中由于穿插探索、战斗、管理基地等，实际到达 90+ 的时间通常是上述数字的 2~3 倍。**大部分玩家在正常通关时主力技能约在 50~75**，极少将单技能练到 90+。

### 5.4 关于"被动成长"

Kenshi 中有部分技能即使你不在"刻意训练"也在缓慢成长：

| 技能 | 被动成长条件 | 成长速率 |
|:----:|------------|:-------:|
| 力量 | 负重行走（即使不战斗） | 慢但持续 |
| 运动 | 任何移动行为 | 非常慢但持续 |
| 坚韧 | 每次被击中（即使你不"训练"） | 随战斗频率 |
| 近战防御 | 每次被攻击成功闪避/格挡 | 随战斗频率 |

## 六、高效训练路线图

### 6.1 新手开局最强训练流程 (0→30小时)

```
第1阶段 (0~5小时): 基础生存
  ├─ The Hub 区挖铜矿 → 赚钱
  ├─ 买背包 → 装铁矿石 → 在 Hub 区往返走（练力量→Lv20）
  └─ 买基础装备+食物

第2阶段 (5~15小时): 战斗入门
  ├─ 找饥饿土匪 → 打（练近战攻击+近战防御→Lv15~20）
  ├─ 故意被打倒 → 站起来（练坚韧→Lv30）
  └─ 招募 3~5 个队友

第3阶段 (15~30小时): 技能全面发展
  ├─ 在 Stack/Hub 夜间潜行 + 暗杀（潜行→Lv40、暗杀→Lv50）
  ├─ 去 Squin 城外挑战沙克巡逻队（坚韧+战斗技能→Lv35~40）
  ├─ 偷窃+撬锁（Lv40+）
  └─ 组建 8~10 人队 → 开始建设基地
```

### 6.2 高效率训练的"邪道"技巧

| 技巧 | 方法 | 用途 |
|------|------|------|
| **囚犯训练法** | 抓捕一个敌对 NPC → 关在笼子里 → 打晕 → 治好 → 再打晕（循环） | 医疗 + 近战攻击 + 坚韧同时训练 |
| **重物行走法** | 背包塞满铁矿石（负重 100%+）→ 在城镇内自动奔跑 | 力量 + 运动同时训练 |
| **撬锁自练法** | 建一个笼子把自己关进去 → 反复撬锁 → 出来 → 再关进去 | 撬锁技能快速提升 |
| **暗杀循环法** | 在骨头人技师处买一个昏迷床 → 把队友打晕 → 反复暗杀 | 暗杀技能快速提升（无仇恨） |
| **自残医术法** | 在蜂人/沙克族的角色上使用陷阱 → 受伤 → 包扎 → 重复 | 医疗技能（注意不要致死） |

## 七、技能成长效率的核心公式

Kenshi 技能经验获取的核心逻辑可用以下简化公式表示（社区估算）：

```
经验获得 = 基础值 × 效率系数 × (1 + 随机浮动 0.8~1.2)

效率系数 = 难度系数 × 状态系数 × 装备系数

难度系数 = max(0.1, (对手技能 - 自身技能 + 50) / 100)
  - 对手技能 < 自身技能 - 50 → 难度系数 ≈ 0.1（极低效率）
  - 对手技能 ≈ 自身技能 → 难度系数 ≈ 0.5（正常效率）
  - 对手技能 > 自身技能 + 50 → 难度系数 ≈ 1.0（最高效率）

状态系数 = 饥饿状态 × 疲劳状态 × 受伤状态
  - 正常状态 = 1.0
  - 饥饿 (饥饿条 < 50%) = 0.6~0.8
  - 极度饥饿 (饥饿条 < 25%) = 0.3~0.5
  - 受伤 (HP < 50%) = 0.7~0.9
```

> **关键推论**: 战斗训练时，保持良好状态（不饥饿、不疲劳）是最高效率的前提。而坚韧技能是唯一在"受伤状态"下效率反而更高的技能。
