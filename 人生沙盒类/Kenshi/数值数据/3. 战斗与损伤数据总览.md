## 二、断肢阈值

| 种族 | 断肢阈值 | 说明 |
|:--|:--:|:--|
| 绿原人 | **-100** | 四肢HP ≤ -100 时断肢 |
| 焦土人 | **-100** | 同绿原人 |
| 沙克 | **-100** | 同绿原人(沙克HP面板虽高但断肢阈值同) |
| 蜂人工人 | **-75** | 更易断肢 |
| 蜂人士兵 | **-100** | 标准 |
| 蜂人王子 | **-80** | 较易断肢 |
| 骨人 | **-200** | 最难断肢 |

**断肢公式总结**:
```
断肢条件 = 该部位HP ≤ -(部位初始HP)
即: 肢体的"负血量"超过其最大HP的绝对值
```

**断肢影响**:
- 断肢造成**大量失血**(立即损失大量血液)
- 断肢后需要**义肢**替换
- 骨人断肢同样可使用义肢
- 义肢不可逆: 装上义肢后无法恢复为原装肢体

### 流血机制详述

```
每秒失血量 = 切割伤害 × 部位流血倍率 × 种族流血倍率
```

| 种族 | 种族流血倍率 | 血量 | 说明 |

### 治疗机制

| 治疗方式 | 恢复效果 | 适用对象 |
|:--|:--|:--|
| 急救包(基础/标准/特级) | 治疗切割伤害(红色血条) | 所有有机种族 |
| 救援包 | 治疗角色至可移动状态 | 所有有机种族 |
| 骨骼维修包 | 修复骨人/义肢损伤 | 骨人+义肢 |
| 骨骼修复床 | 完全修复骨人HP/消除磨损 | 骨人+义肢 |
| 自然恢复(睡袋/床) | 缓慢恢复所有伤害(含钝击灰血) | 所有种族(骨人仅限修复床) |

> **关键战术**: 头部受伤流血极快，战斗中应优先保护头部护甲。切割伤害可通过急救包快速治疗，钝击伤害只能自然恢复(极慢)。断肢后需尽快止血(使用急救包于断肢部位)，否则角色会因失血而昏迷/死亡。
