## 五、移动速度系统

### 5.1 无负重跑步速度（运动技能影响）

| 运动技能等级 | 无负重速度 (km/h) | 备注 |
|:----------:|:----------------:|------|
| 1 | ~15 km/h | 初始水平，慢跑 |
| 10 | ~19 km/h | 常速 |
| 20 | ~22 km/h | 可以开始跑赢大部分敌人 |
| 30 | ~24 km/h | 快跑 |
| 40 | ~26 km/h | |
| 50 | ~28 km/h | 中等水平 |
| 60 | ~29 km/h | |
| 70 | ~30 km/h | 高级跑者 |
| 80 | ~31 km/h | |
| 90 | ~32 km/h | 极限训练水平 |
| 100 (理论极限) | ~33 km/h | 理论上可达到，但需要极端训练 |

**速度计算公式（社区估算）**:
- 基础速度 = 15 + (运动技能 × 0.15) + (敏捷 × 0.03) km/h
- 实际速度 = 基础速度 × 负重系数 × 装备修正 × 肢体状态修正

### 5.2 负重减速曲线

| 负重百分比 | 速度衰减 | 描述 |
|:---------:|:-------:|------|
| 0% (无负重) | 0% | 全速 |
| 25% | -10% | 轻度负重 |
| 50% | -25% | 明显减速 |
| 75% | -40% | 严重影响行动 |
| 100% (等于力量上限) | -55% | 极慢，几乎无法奔跑 |
| >100% | -70%+ | 无法奔跑，只能步行 |

**负重上限 = 力量 × 25 kg**。超出上限的角色会被"压垮"，行动极度受限。

### 5.3 装备对速度的影响

| 装备类型 | 速度影响 | 备注 |
|---------|:-------:|------|
| 轻甲 (布甲、忍者装) | 0%~-5% | 几乎不影响速度 |
| 中甲 (链甲、皮甲) | -5%~-15% | 平衡型 |
| 重甲 (圣国甲、沙克甲) | -15%~-30% | 高防护，显著减速 |
| 木制义肢 | -5% | 基础义肢 |
| 蒸汽义肢 | -5%~+10% | 高级义肢可能有速度加成 |
| 偷来的重型背包 | -10% | 额外惩罚 |
| 木乃伊裹布 | 0% | 不影响 |

## 六、旅行策略建议

| 阶段 | 推荐旅行范围 | 交通方式 | 主要目标 |
|------|------------|---------|---------|
| 初期 (Lv 1~10) | The Hub → Squin → The Border Zone | 步行 | 熟悉地图，收集基础资源 |
| 中期 (Lv 10~25) | 大沙漠 → 沼泽 → 谢姆 | 步行 + 背包 | 积累财富，建立关系 |
| 中后期 (Lv 25~50) | 雾岛 → 洪水地 → 骨场 | 步行 + 骆驼 | 探索遗迹，获取科技书 |
| 后期 (Lv 50+) | 死亡之地 → 斯托比的赌注 → 灰烬之地 | 全员重装步行 | 终极挑战，AI 核心，Meitou 武器 |

> **注**: Kenshi 没有坐骑/载具系统（不计算 MOD），所有旅行完全依靠步行。因此运动技能（Athletics）是所有角色必须训练的"废土生存第一技能"。负重管理对于长途旅行至关重要。

> **参考数据**: 移动速度公式为社区大量测试估算值（社区估算），游戏内不直接显示 km/h 单位，以上值为对照游戏内时间+距离计算所得。城镇数据基于 v1.0 原生地图。部分遗迹的战利品池可能因游戏更新有细微变化，建议参考 [Kenshi Wiki](https://kenshi.fandom.com/wiki/Map) 获取最新数据。
