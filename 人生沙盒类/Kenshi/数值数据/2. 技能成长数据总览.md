## 一、经验获取公式

### 基础公式

```
获得经验 = 基础经验 × 全局倍率 × 种族倍率 × 等级倍率 × 强敌逻辑(SOL)
```

| 参数 | 说明 | 默认值 |
|:--|:--|:--:|
| **基础经验** | 等级0时一次行动获得的经验量(约等于等级提升量) | 因技能而异 |
| **全局倍率** | FCS中的"GLOBAL CONSTANT"设定值 | **3.0x** |
| **种族倍率** | 各技能的种族加成/减成 | 0.8x-1.2x |
| **等级倍率** | (1 - 当前等级/101)²，若结果≥20则不再获得经验 | 等级越高越慢 |
| **强敌逻辑(SOL)** | 仅用于战斗技能，对方等级越高经验越多 | 见下文 |

### 强敌逻辑(SOL)公式

```
SOL = 1 + (对手等级 - 自身等级) × 系数

当对手等级 > 自身等级: 每级差 +0.1x，上限 +5.0x (总 6.0x)
当对手等级 < 自身等级: 每级差 -0.04x，下限 -0.9x (总 0.1x)
被围攻时: 每额外+1敌人 +0.1x，上限叠加至约 1.667x (1v8时)
```

**关键结论**:
- 打比自己强50级的对手: 获得 **6.0x** 经验
- 打比自己弱22.5级的对手: 仅获得 **0.1x** 经验(几乎无成长)
- 1v8极限: 最高可获得 **10.0x**(6.0x × 1.667x) 经验

## 五、快速冲技能策略

| 技能 | 最快升级方法 | 效率(社区估算) |
|:--|:--|:--:|
| **力量** | 背满石头(负重90%+)在城镇内反复跑 | 1级→50级约 2h 游戏内时间 |
| **坚韧** | 故意找野兽/低级土匪群殴，倒地后反复站起 | 1级→50级约 1h |
| **敏捷** | 使用生锈/劣质打刀攻击高护甲敌人(高切割比例) | 1级→50级约 3-4h |
| **运动** | 脱光装备无负重跑图 | 1级→50级约 3h |
| **近战攻击/防御** | 抓高等俘虏反复实战(打不死练到昏) | 1级→50级约 5h+ |
| **暗杀** | 反复暗杀昏迷/睡眠目标 | 1级→50级约 30min(最快) |
| **撬锁** | 反复撬城镇里任意锁(被抓也不掉级) | 1级→50级约 1h |
| **偷窃** | 反复偷→放回→再偷 | 1级→50级约 1-2h(但会降好感) |
| **锻造** | 反复制造大量铁板/钢板/低级武器 | 1级→50级约大量材料投入 |
| **科学** | 多人同时研究(多人叠加少量加成) | 1级→50级约大量研究点数 |

> **核心原则**: Kenshi 没有技能上限(经社区测试部分技能可达9000+级)，但 >100级后升级速度极其缓慢。正常游戏流程中，绝大多数角色的核心技能在30-70级范围内发挥作用。
