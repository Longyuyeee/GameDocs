## 一、全势力好感等级体系

Kenshi 好感度范围为 **-100 至 +100**，分为以下等级:

| 等级 | 阈值 | 行为表现 |
|:--:|:--:|:--|
| **同盟 (Allied)** | **≥ +50** | 不会攻击你; 治疗你的伤员; 协助防守你的基地; 免费提供资源(部分势力); 送你礼物(部分势力); 不再收税 |
| **友好 (Friendly)** | 0 至 +49 | 允许进城; 可交易; 正常对话; 部分势力会治疗你的昏迷成员 |
| **中立 (Neutral)** | -1 至 -29(注意范围!) | 允许进城; 可交易; 可能对你冷嘲热讽 |
| **敌对 (Hostile)** | **-30 至 -100** | 见即攻击; 禁止进城; 无法交易; 攻击你的基地 |

> **重要**: 并非所有势力都遵循完全相同的阈值。部分势力在-10就开始攻击(如圣国对非人类)，而有些在-49仍可交易。以下为通用规则。

- 每次调停后需等待约**2小时**游戏内时间才能再次使用同一Pacifier

## 六、神圣王国 Prayer Day 详情

| 参数 | 数值/说明 |
|:--|:--|
| **触发范围** | 基地距圣国据点 **≤2000m** 内 |
| **最大概率** | 距圣国据点 ≤1500m 时每日最大 **50%** 概率 |
| **触发递减** | 1500m-2000m 随距离增加概率递减至 **0%** |
| **再次触发间隔** | 每次Prayer Day完成/失败后 **100-160小时**再次触发 |
| **来使队伍** | 5圣国高骑士 + 4圣仆 + 1圣祭司 |

### 缺席/违规惩罚

| 玩家行为 | 后果 | 袭击时间 |
|:--|:--|:--|
| 不在基地(缺席) | 圣国突袭(Minor) | 36-48小时后 |
| 对话选择错误/攻击祭司 | 立即开战 + 圣国突袭(Minor) | 12-24小时后 |
| 全员女性角色与祭司对话 | 祭司离开 + 圣国突袭 | 12-36小时后 |
| 非人类角色与祭司对话 | 立即敌对 | 10-20小时后 |
| 骨人与祭司对话 | 立即攻击(无需对话) | 立即 |
| **正确完成仪式** | **和平结束，得圣火书一本** | — |

### 正确应对方法

```
必要条件:
1. 推荐一名人类男性(绿原人最佳)与祭司对话
2. 身上携带一本《圣火》(The Holy Flame)书
3. 在对话中选择正确的祈祷选项

结果: 祭司满意离开，下一次Prayer Day在100-160小时后。
如果玩家与圣国好感达+50同盟，Prayer Day永久停止。
```

### 永久停止Prayer Day的条件

| 条件 | 方式 |
|:--|:--|
| 与圣国同盟(好感≥+50) | 积累好感或Pacifier调停 |
| 与Flotsam Ninjas同盟 | 自动与圣国敌对 |
| 与沙克王国同盟 | 自动与圣国敌对 |
| Holy Lord Phoenix 死亡或囚禁 | 圣国势力瓦解 |
| 基地搬出2000m范围 | 移出圣国势力范围 |
