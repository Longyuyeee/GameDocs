# "你不是天选之人"：反传统 RPG 叙事与涌现叙事设计分析

> 游戏版本: Kenshi v1.0 (PC/Steam, Lo-Fi Games, 2018)
> 引用来源: [Kenshi Wiki (Fandom)](https://kenshi.fandom.com/wiki/Lore) | [Steam 社区](https://steamcommunity.com/app/233860) | [游研社](https://www.yystv.net/p/6008) | [Game World Observer](https://gameworldobserver.com/)

---

## 一、什么"你不是天选之人"？

### 1.1 传统 RPG 的"天选之人"叙事

几乎所有主流 RPG 共享一个潜在叙事前提：**你是特殊的**。

| 游戏 | 你的特殊身份 | 叙事前提 |
|------|------------|---------|
| 上古卷轴 V：天际 | 龙裔（Dovahkiin） | 你是唯一能屠龙的人 |
| 巫师 3 | 杰洛特 | 你是最强的猎魔人之一 |
| 质量效应 | 谢泼德指挥官 | 你是人类最强战士，唯一能拯救银河系的人 |
| 辐射：新维加斯 | 邮差（但迅速成为关键人物） | 你被卷入一场决定莫哈维命运的斗争 |
| 黑暗之魂 | 被诅咒的不死人 | 你有能力挑战薪王 |

这些游戏的设计逻辑是："因为你是天选之人，所以你有能力改变世界。"**玩家角色的特殊身份是叙事的原动力。**

### 1.2 Kenshi 的"反天选之人"设计

Kenshi 彻底颠覆了这个模式。进入游戏后你面对的不是史诗命运，而是：

> **你什么都不是。这个世界不在乎你。你死了也没人记得你。**

具体体现在以下设计：

**没有角色背景**：
- 游戏不给你安排任何身份
- 开局选项（奴隶、流浪者、商人、自由人）只是经济状态，不是身份标签
- 没有人认识你，没有人在等你，没有使命召唤你去完成

**世界不因你而改变规则**：
- 野兽不会因为你"Lv1"就降低攻击欲望
- 沙漠土匪不会因为你"还在新手村"就手下留情
- **没有等级缩放**——The Great Desert 的沙尘暴不会因为你只有 5 级就变得更温和

**没有"任务"的概念**：
- Kenshi 中没有传统 RPG 的"任务日志"
- 没有 NPC 头顶感叹号
- 没有"你被选中了，去打败邪恶"的指引
- 你做什么完全由你自己决定（或由生存需求驱动）

## 二、这种设计的情感影响：一条独特的心路历程

### 2.1 初期：纯粹的挫败（0~10 小时）

```
体验：
  - "我被一只野狗咬死了。"
  - "我在沙漠里迷路了，渴死了。"
  - "我偷了一个面包，被守卫砍死了。"
  - "我被奴隶贩子抓走了，成了矿工。"
  - "我加入了商队当保镖，然后全员被土匪全灭。"
```

新手期的 Kenshi 体验充满了死亡、失败和对"这游戏到底该怎么玩"的困惑。这不是"开放世界自由探索"的浪漫——这是**生存的残酷试炼**。

Kenshi 不解释任何东西。没有新手教程。没有引导。你只能自己摸索——就像废土上的任何活物一样。

### 2.2 中期：从"被打"到"能打"（10~50 小时）

这是 Kenshi 的魅力真正显现的阶段。

关键转折：**当你意识到自己变强的那一刻，不是看到升级动画，而是突然发现自己能打赢曾经秒杀你的对手。**

```
玩家自述示例：
  "我在 Squin 城外混了 20 小时。一开始沙漠土匪来的时候我必须立刻跑。
  现在我可以用一根生锈的废铁剑打翻 3 个土匪了。
  我不记得自己是什么时候变强的——但我确实变了。"
```

这种成长感之所以如此强烈，是因为**没有系统告诉你你变强了**。你不能打开面板看到"力量 +5"的提示。你只能在战斗中自己感受到——挥剑更快了，跑得更远了，挨打后站起来的次数更多了。

这就是 Kenshi 的"做即学习"系统的情感效果：**你不会被告知你在成长，但你清楚地知道自己成长了。**

### 2.3 后期：从奴隶到世界改变者（50 小时+）

当你的角色技能达到 50+，有了 10+ 人小队，拥有自己的基地时，Kenshi 真正的叙事深度才完全展开。

**从"世界不在乎你"到"世界不得不关注你"**：

```
玩家可能的后期体验：
  - 你曾经被神圣王国巡逻队追捕 → 现在你攻击了 Blister Hill
  - 你曾经是 Rebirth 的奴隶 → 现在你释放了 Rebirth 的所有囚犯
  - 你曾经被饥饿土匪抢劫 → 现在你建立了一个自给自足的堡垒
  - 你曾经被沙漠土匪绑架 → 现在你成了赏金猎人，在抓捕沙漠土匪头目

  你没有 "任务"——但这些行动世界确实有反应。
  城镇归属改变。势力领袖死亡或入狱。NPC 会讨论你的事迹。
```

Kenshi 的 **世界状态系统** 让"你改变了世界"成为可验证的事实。当你杀死神圣王国的 Phoenix 并看到世界地图上 Stack 和 Blister Hill 的旗帜变为"叛军控制"时——这种成就感是任何预设任务系统都无法提供的。

## 三、涌现叙事 vs 固定脚本

### 3.1 什么是涌现叙事？

| 概念 | 定义 |
|------|------|
| **固定脚本叙事 (Scripted Narrative)** | 设计师预写好的故事。玩家按顺序经历预设的情节（如《最后生还者》《巫师 3》） |
| **涌现叙事 (Emergent Narrative)** | 游戏系统规则碰撞产生的、不可预测的故事。玩家自己"生成"了独特的经历 |

Kenshi 属于**极端的涌现叙事**——它甚至比大部分沙盒游戏更依赖涌现。

### 3.2 Kenshi 的涌现叙事是如何产生的？

Kenshi 有四个系统层级的交互产生了涌现叙事：

**第一层：生存系统**
- 饥饿、受伤、天气 → 被迫做出决策 → 决策产生故事
- 例："我在雾岛被雾民追捕，食物耗尽，队友断了一条腿，我必须在天黑前跑到 Mongrel——否则全队都会死。"

**第二层：战斗与损伤系统**
- 身体部位破坏、断肢、流血 → 每次战斗的结果都不同 → 产生不可预测的后续
- 例："我的主力战士在战斗中失去了右臂。我以为他废了。但我在废品商那里买到了一个高级机器义肢——装上后他比断臂前更强了。"

**第三层：世界状态系统**
- 势力领袖死亡、城镇归属变化、外交关系改变 → 世界因果动态变化
- 例："我杀死了联合城市的 Emperor Tengu，结果 UC 的城镇陷入了混乱。曾经繁荣的商业中心 Heft 变成了没有管理的自由城。"

**第四层：奴隶系统**
- 被捕获、囚禁、逃脱 → 产生了 RPG 中极为罕见的"从最底层崛起"叙事
- 例："我在 Rebirth 被奴役了 30 小时。我装作顺从，但每晚都在练习撬锁和潜行。第 31 天凌晨我逃了出去——偷了守卫的剑，放走了 20 个奴隶。我从一个奴隶变成了废土上最危险的逃犯。"

### 3.3 涌现叙事的叙事优势

| 优势 | Kenshi 的实现 |
|------|--------------|
| **唯一性** | 每个玩家的 Kenshi 故事完全不同。没有人有相同的经历 |
| **情感强度** | "我断了一条腿然后活下来了"的故事比"打败了 Boss 然后看了一段过场动画"更动人——因为是你自己经历了这个挣扎 |
| **重新语境化** | 10 年前被奴隶贩子抓走 → 10 年后亲手摧毁奴隶制——这种回环式叙事只有涌现系统才能自然产生 |
| **可分享性** | Kenshi 玩家社区最活跃的内容就是分享"自己的故事"——这些故事天然适合社交媒体传播 |

### 3.4 涌现叙事的代价

| 代价 | 表现 |
|------|------|
| **缺乏叙事焦点** | 没有"主线剧情"让部分玩家感到迷茫 |
| **情感空白期** | 在"有故事的时刻"之间可能有长时间的无聊（挖矿、走路） |
| **叙事不稳定** | Bug 或系统漏洞可能破坏一个本来很好的涌现叙事 |
| **学习曲线** | 没有精心设计的"教程关卡"，玩家必须自己从失败中学习 |

## 四、与同类游戏的涌现叙事对比

### 4.1 与《环世界》(RimWorld) 的 AI 叙事对比

| 维度 | Kenshi | RimWorld |
|:----:|:------:|:--------:|
| **叙事驱动者** | 系统规则自然碰撞 | **AI Storyteller**（刻意制造戏剧性） |
| **事件来源** | 世界模拟（巡逻队、商人、袭击） | AI 叙事者选择事件（随机但受控） |
| **频率** | 基于世界模拟结果 | AI 叙事者调整节奏（"平淡期"和"高潮期"交替） |
| **情感策略** | 更"真实"——好事坏事完全随机 | 更"戏剧性"——AI 会在你刚好恢复时给你新的打击 |
| **玩家评价** | "这个世界是真实的" | "这个 AI 太懂我了" |

**核心差异**: RimWorld 的 AI Storyteller（Cassandra Classic、Phoebe Chillax、Randy Random）会**主动控制叙事节奏**——当你恢复得差不多时给你新的打击，当你太惨时给你喘息的机会。Kenshi **没有任何类似的设计**——世界就是按照自己的规则运行，不管你是不是刚被打残。

结果：
- RimWorld 的叙事更"工整"——像一个好的故事讲述者在控制节奏
- Kenshi 的叙事更"真实"——但有时候会显得平淡（"走了 20 分钟路，什么也没发生"）

**谁更好？取决于你更喜欢"好的故事"还是"真实的世界"。**

### 4.2 与《骑马与砍杀 2：霸主》(Mount & Blade II: Bannerlord) 的对比

| 维度 | Kenshi | 骑砍 2 |
|:----:|:------:|:------:|
| **初始身份** | 无名小卒（0） | 无名小卒（但有家族背景） |
| **成长路径** | 从废土求生到争夺权力 | 从雇佣兵到贵族/国王 |
| **叙事系统** | 无系统驱动的纯粹涌现 | **王国管理 + 家族系统驱动** |
| **叙事焦点** | 个体生存史诗 | 家族/国家争霸史诗 |
| **决策影响** | 杀死领袖 = 永久改变世界 | 征服城市 = 改变地图颜色 |
| **细节深度** | 断腿装义肢 = 永久改变角色 | 战死 = 角色永久死亡（但可继承） |

**核心差异**: 骑砍 2 虽然也是涌现叙事，但它的叙事被"王国管理"系统引导到了一个清晰的通道：**从流浪者到国王**。你有一个明确的"阶级上升通道"，虽然游戏不强迫你走，但系统的奖励结构指向了这个方向。

Kenshi 没有这个"通道"——你不知道"从弱到强"应该做什么、能做什么、会走到哪里。部分玩家因此感到迷茫（"我变强了，然后呢？"），但这也意味着 Kenshi 的叙事开放性是更彻底的。

**关键区别**:
- 骑砍 2 的涌现叙事 = "我可以成为国王"（有预设终点，但路径自由）
- Kenshi 的涌现叙事 = "我可以成为任何东西，也可以什么都不成为"（既无预设路径，也无预设终点）

### 4.3 与《矮人要塞》(Dwarf Fortress) 的对比

| 维度 | Kenshi | Dwarf Fortress |
|:----:|:------:|:--------------:|
| **叙事尺度** | 个体/小队史诗 | 文明史诗 |
| **世界生成** | 固定世界 | 程序生成世界（含完整历史） |
| **历史模拟** | 实时世界状态变化 | 世界生成时已有 1000+ 年模拟历史 |
| **叙事粒度** | 角色断腿等身体细节 | 从传奇武器到矮人关系的全面传奇生成 |
| **叙事抽象层** | 3D 实时画面 + 文字 | 纯 ASCII / 简单图形 + 大量文字描述 |

Kenshi 可以被理解为"3D 实时版矮人要塞"的一个子集——它专注于角色尺度的涌现叙事，而 DF 涵盖文明尺度的更深层模拟。

## 五、"不是天选之人"的设计意义

### 5.1 为什么这种设计有效？

"你不是天选之人"之所以在情感上如此有效，是因为它**反转了 RPG 中成就感的来源**：

```
传统 RPG:
  成就 = 你完成了天命（预设任务）

Kenshi:
  成就 = 你在一个不在乎你的世界里，硬是靠自己的意志活了下来
```

后者的成就感来源更"原始"——它触及了人类最底层的满足感：**在极端不利条件下的幸存**。

这正是为什么 Kenshi 玩家在描述游戏体验时，使用的语言更接近"真实回忆"而非"游戏经历"：
- "我当时断了一条腿..."
- "我在沙漠里走了 3 天，食物吃完了..."
- "当我逃出 Rebirth 时，外面正在下酸雨..."

### 5.2 对其他游戏的影响

Kenshi 的"反天选之人"设计理念已经开始影响其他独立游戏：

| 游戏 | 受影响的设计 |
|------|------------|
| Outward | 类 Kenshi 的无等级缩放 + 生存导向 RPG |
| Colony Ship | 非"天选之人"的硬核 RPG 叙事 |
| 各种 Kenshi-like（开发中） | 直接模仿"做即学习"和无主角设定 |

### 5.3 局限性与争议

| 争议点 | 支持者观点 | 批评者观点 |
|--------|-----------|-----------|
| "没有目标" | 自由就是最大的目标 | 没有方向导致大量玩家弃坑 |
| "太残酷" | 废土不应该温柔 | 很多玩家被劝退了 |
| "没有故事" | 每个玩家有自己的故事 | 缺乏精心设计的叙事体验 |
| "孤独感" | 这就是废土的感觉 | 缺乏与世界的深度连接 |

## 六、结语

Kenshi 的"你不是天选之人"不仅仅是一个设计选择——它是整个游戏的哲学核心。从"做即学习"到"无等级缩放"，从"没有任务"到"世界状态系统"，所有的设计都在服务同一个理念：

> **这个世界不欠你什么。没有人等待你去拯救。但正因为如此，你的每一点成就——哪怕只是活着——都是完全属于你自己的。**

在传统 RPG 日益流水线化、每个玩家都经历相同"英雄之旅"的年代，Kenshi 用最笨拙但最真诚的方式提醒我们：**真正的 RPG 不是让你扮演天选之人——而是让你在残酷的世界里找到属于自己的位置。**

这不是一个英雄的故事。这是一个**幸存者**的故事——而且是你自己书写的故事。
