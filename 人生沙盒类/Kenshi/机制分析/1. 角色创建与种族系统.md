# Kenshi — 角色创建与种族系统

> 游戏版本: Kenshi v1.0 (PC/Steam, Lo-Fi Games, 2018)
> 数据来源: [Kenshi Wiki (Fandom)](https://kenshi.fandom.com/wiki/Races), [Kenshi Wiki (独立)](https://kenshi.wiki/races), [Steam](https://store.steampowered.com/app/233860/), [游研社](https://www.yystv.net/p/6008)
> 所属品类: [人生沙盒类](../_index.md)

## 一、系统概述

Kenshi 的角色创建系统与传统 RPG 截然不同——**开局只决定起点和初始装备，不决定"你是谁"或"你能做什么"**。游戏没有职业系统、没有天赋树、没有"天选之人"的预设；每个角色从一穷二白开始（属性低至 1），通过行动逐步定义自身。种族选择是唯一不可更改的先天差异，直接影响角色的生存策略、战斗定位和势力关系。

### 2.2 各族详细分析

#### 绿原人 (Greenlander)

| 项目 | 内容 |
|------|------|
| 种族标签 | 人类 |
| 属性加成 | 无显著加成 |
| 属性惩罚 | 无显著惩罚 |
| 势力态度 | 神圣王国视为"纯种"——完全接纳 |
| 推荐定位 | 万能型——任何玩法路线均可 |
| 备注 | 最"标准"的种族选择。适合新手首次游玩 |

绿原人是Kenshi的基准人类种族，没有显著的优势也没有致命的短板。神圣王国认为绿原人是"Okran的选民"，给予最高待遇等级。对于不确定选什么种族的玩家，绿原人是安全的选择。

#### 焦土人 (Scorchlander)

| 项目 | 内容 |
|------|------|
| 种族标签 | 人类 |
| 属性加成 | 敏捷+10%，劳动-10%，武器锻造加成 |
| 属性惩罚 | 力量成长略慢（-10%劳动效率） |
| 势力态度 | 神圣王国容忍但视为次等人类 |
| 推荐定位 | 盗贼/潜行/工匠/弩手 |
| 备注 | 肤色较深，适合沙漠区域生存（天然伪装） |

焦土人的敏捷加成使其在潜行、偷窃、开锁、远程攻击方面有天然优势。配合天赋的劳动惩罚（-10%），焦土人更适合"巧活"而非"苦力"。锻造加成使其成为优秀的武器工匠。神圣王国虽然接受焦土人但态度冷淡——他们仍被视为次等人类。

#### 沙克人 (Shek)

| 项目 | 内容 |
|------|------|
| 种族标签 | 沙克 |
| 属性加成 | 全部位HP+25，钝器伤害+10%，徒手伤害+10% |
| 属性惩罚 | 饥饿速度+50%（吃得更多），劳动-10% |
| 势力态度 | 神圣王国视为"非人"——立即敌对 |
| 推荐定位 | 前排战士/坦克/武术家 |
| 备注 | 游戏中最强的近战种族之一 |

沙克人的125HP（所有部位）使其成为天然的肉盾。更高的HP阈值意味着更不容易被击倒（坚韧判定更有利），配合钝器和徒手伤害加成，沙克人是正面战场的统治者。代价是食量比人类大50%——在食物紧缺的废土上是沉重的后勤负担。劳动惩罚使其不适合早期挖矿维生。

> 设计特点分析: 沙克人的食量惩罚(1.5x)是一个精妙的设计——"更强的战力=更高的维持成本"。这让玩家在选择种族时面临"性能 vs 后勤"的权衡，而不是单纯追求数值最大化。这一设计理念在《环世界》中也有体现（例如不同种族/基因的代谢率差异），但Kenshi将其简化为一个饥饿倍率参数，理解成本更低。

#### 蜂人工人 (Hive Worker Drone)

| 项目 | 内容 |
|------|------|
| 种族标签 | 蜂人 |
| 属性加成 | 劳动+20%，武术+10%，跑速+10% |
| 属性惩罚 | 全部位HP-25（仅75HP），无法戴头盔，商店翻倍溢价 |
| 势力态度 | 西蜂巢视为"逃兵"——敌对；商店价格x2 |
| 推荐定位 | 采矿/搬运/劳动/探路/跑商护卫 |
| 备注 | 最脆弱的种族——但最优秀的劳动者+免疫酸雨 |

蜂人工人是Kenshi中"玻璃大炮"的极致——75HP + 无法戴头盔 = 战斗中极易被一击致命。然而其+20%的劳动效率使其成为游戏中最强的采矿/搬运/农业人口。免疫酸雨的特性使其可以在酸雨区域（如沼泽/溺没之城）安全作业，这是其他种族难以替代的优势。跑速+10%意味着蜂人工人可以更快逃离危险——Kenshi中"跑得快"本身就是生存策略。

> 跨游戏对比: 蜂人工人的设计类似《Elona》中的"妖精"种族——极低HP但高速度、高回避。两者都遵循"脆皮快跑"的设计哲学。但与Elona妖精的魔法亲和力不同，Kenshi蜂人工人更注重劳动/采矿方向，体现了Kenshi"挖矿是基础生存手段"的玩法重心。

#### 蜂人士兵 (Hive Soldier Drone)

| 项目 | 内容 |
|------|------|
| 种族标签 | 蜂人 |
| 属性加成 | 战斗属性成长+20%，力量成长+20% |
| 属性惩罚 | 无法戴头盔，商店翻倍溢价，劳动惩罚 |
| 势力态度 | 西蜂巢视为"逃兵"——敌对；商店价格x2 |
| 推荐定位 | 纯战斗角色/二阶输出手 |
| 备注 | 比工人更强——但仍需注意头部防护问题 |

蜂人士兵是蜂巢的"战士种姓"，有更强的战斗成长（+20%力量/战斗技能成长率），但依然无法戴头盔——这是蜂人的种族硬伤。在后期战斗中，头部受伤导致昏迷/死亡是蜂人最大的威胁。不过其100HP的基准生命值（与人类相同）使其比蜂人工人更能承受打击。

#### 骨人 (Skeleton)

| 项目 | 内容 |
|------|------|
| 种族标签 | 骨人 |
| 属性加成 | 全部位HP**200**（全种族最高），免疫饥饿，免疫酸雨/沙尘暴/毒气/血液流失，战斗属性加成 |
| 属性惩罚 | **必须使用维修包**（绷带无效），**沉没**（不能游泳，过深水域必死），被所有非机械势力恐惧/敌对 |
| 势力态度 | 神圣王国"恨之入骨"——立即攻击；联合城市容忍但恐惧；沙克王国中立但冷淡 |
| 推荐定位 | 终极战士/独立生存者/无需后勤的探索者 |
| 备注 | 最强也是最难的种族——新人慎选 |

骨人是Kenshi中最极端的种族。200HP + 免疫流血 + 免疫饥饿 + 免疫天气 = 理论上最强生存力。但代价同样沉重：维修包需要工业补给（铁板+电子元件）制作，价格昂贵且不易获得；不能游泳意味着骨人无法穿越深水区域（地图上某些关键路径受限）；被神圣王国追杀意味着地图西北部的大部分区域对骨人是禁区。

> 设计特点分析: 骨人的"不需要进食"听起来是纯粹的优势，但在Kenshi的生存体系中，"不需要进食"意味着"不需要进食=不需要去城镇买食物=与文明世界的互动减少"。这看似优势的设计实际上进一步孤立了骨人玩家，强化了Kenshi的"废土孤独感"。此外，骨人不能用绷带只能用维修包的设定增加了后期经济压力——维修包需要特定工业原料，迫使骨人玩家必须建立基地或维持高收入。

## 四、无职业系统设计

Kenshi**没有职业系统**——这是其与几乎所有RPG最核心的区别。

| 对比项目 | Kenshi | 传统RPG (如《上古卷轴》) | 骑马与砍杀2 |
|---------|--------|------------------------|------------|
| 开局职业 | 无——只有起点场景 | 选择职业=锁定发展方向 | 选择出身=初始技能方向 |
| 技能发展 | 全自由——想练什么就做什么 | 职业限制某些技能无法/低效学习 | 技能有上限和效率惩罚 |
| 成长边界 | 无等级上限 | 硬等级上限 | 技能等级上限+专精点 |
| 角色定义 | 由"你做了什么"定义 | 由"你选了哪个职业"定义 | 由"你点了哪些专精"定义 |
| 重置成本 | 0成本——随时改变方向 | 需洗点/重新开档 | 属性固定/技能上限固定 |

> 设计特点分析: Kenshi取消职业系统的设计意图非常明确——**"你不是天选之人，你只是废土上的一块白板"**。这让Kenshi的叙事产生了质变——如果你成为了剑术大师，不是因为你"选了战士职业"，而是因为你在废土上经历了无数场死斗；如果你成为了开锁大师，不是因为你"点了盗贼天赋"，而是因为你在奴隶营中反复练习逃跑。这种"涌现叙事"（Emergent Narrative）是Kenshi区别于所有传统RPG的核心设计哲学。

## 五、"不是天选之人"——初始属性设计

Kenshi的初始属性设计坚决贯彻"你什么都不是"的理念：

| 属性 | 初始值 | 意义 |
|------|:------:|------|
| 力量 (Strength) | 1 | 挥不动重武器，负重走几步就累 |
| 敏捷 (Dexterity) | 1 | 攻速极慢，闪避几乎为零 |
| 坚韧 (Toughness) | 1 | 被一拳打倒在地，半天爬不起来 |
| 近战攻击 | 1 | 挥砍软弱无力 |
| 近战防御 | 1 | 几乎格挡不住任何攻击 |
| 运动 (Athletics) | 1 | 跑得比乌龟还慢 |
| 潜行 | 1 | 走两步就被发现 |
| 撬锁 | 1 | 大部分锁都撬不开 |

> 与《骑马与砍杀2》对比: 骑砍2虽然也是"从零开始"，但初始角色有约10-15点的各项基础属性，且出身选择直接影响技能初始等级。Kenshi更为极端——所有属性从1开始，初始场景只决定"起点位置"和"第一波装备"，不决定任何属性加点。这是Kenshi"反RPG"设计哲学的集中体现：**游戏不给你任何预设身份，你的身份由你在废土上的每一个选择共同定义。**

## 六、设计特点分析总结

| 维度 | 设计特点 | 设计意图 |
|------|---------|---------|
| 种族差异 | 极端差异化（HP 75~200，多维度惩罚/加成） | 让种族选择成为重要的生存策略决策 |
| 种族歧视 | 不同势力对不同种族态度完全不同 | 种族选择本质上是"地图选择"——影响活动范围 |
| 开局设计 | 仅决定初始装备+起点位置 | 抹去"预设身份"——让游戏体验从零开始 |
| 无职业系统 | 全自由发展 | 强化"做即学习"——你做什么就成为什么 |
| 初始属性 | 所有属性=1 | "你不是天选之人"——成长才有意义 |
| 种族惩罚 | 每个种族都有明确的短板 | 避免出现"最佳种族"——所有选择都是权衡 |

| 跨游戏对比 | 相似点 | 差异点 |
|-----------|--------|--------|
| 骑马与砍杀2 | 开局选择影响初始难度 | 骑砍有出身技能加点→Kenshi只有装备区别 |
| 环世界 | 角色有先天特质影响效率 | 环世界特质可后天修改（基因/仿生）→Kenshi种族不可改 |
| Elona | 种族差异极端（如妖精脆皮/蜗牛极慢） | Elona有职业系统+神祇信仰→Kenshi只有种族和开局 |

## 七、补充信息

| 项目 | 内容 |
|------|------|
| 是否可中途更改种族 | 不可（开局锁定）——骨人不可变血肉，血肉不可变机械 |
| 种族对对话文本的影响 | 是——不同种族+势力身份触发不同对话（数百行差异性文本） |
| 开局对剧情的影响 | 无——Kenshi没有主线剧情。开局只是"从哪里开始" |
| 多人开局差异 | 单人 vs 5人开局（流浪者 vs 自由人）——影响初期生存曲线 |
| 动物角色 | 可招募动物（狗/牛/山羊/驮兽）——有自己的种族标签和属性 |
| MOD种族 | Steam Workshop有大量自定义种族MOD（如机器人/精灵/矮人） |

> 待补充: 各族对特定天气的详细耐受数据（如骨人对"血雾"的反应机制）、各开局场景的NPC特殊对话列表、种族对战斗AI行为的影响数据。
