# 社会关系与门派势力 - 觅长生(Immortal Way of Life)

> 游戏版本: 觅长生 v1.0 (PC/Steam, Chalcedony Network, 2023)
> 数据来源: [觅长生BWIKI](https://wiki.biligame.com/mcs/) | [Steam商店页](https://store.steampowered.com/app/1189490/) | [机核网评测](https://www.gcores.com/articles/160854) | [GameRes分析](https://www.gameres.com/863556.html)

## 1. 五大门派体系

觅长生的修仙世界由五大宗门统治，分别对应五行属性，形成了一套完整的势力格局。

### 1.1 门派基础信息

| 门派 | 主城 | 五行 | 掌门 | 入门灵根偏好 | 门派特色 |
|:---:|:---:|:---:|:----:|:----------:|---------|
| **金虹剑派** | 武陵城 | 金 | 掌门待补充 | 金灵根 | 剑道正宗——攻伐第一 |
| **竹山宗** | 广陵城 | 木 | 掌门待补充 | 木灵根 | 炼丹大派——毒术与医术并存 |
| **星河剑派** | 宁州(水乡) | 水 | 掌门待补充 | 水灵根 | 剑法飘逸——控制与多段攻击 |
| **离火门** | 天星城 | 火 | 掌门待补充 | 火灵根 | 暴烈刚猛——火系极致输出 |
| **化尘教** | 沙州 | 土 | 掌门待补充 | 土灵根 | 神秘隐世——阵法与诅咒 |

> 注：五大门派的地理分布覆盖宁州全境，玩家在游戏开始时可以根据出生地决定初始活动的区域范围。

### 1.2 门派晋升体系

| 职位 | 晋升条件 | 权限/收益 |
|:---:|:--------:|---------|
| **外门弟子** | 入门即得 | 基础功法+藏经阁第1层+每月微薄俸禄 |
| **外门执事** | 炼气后期+完成门派任务 | 藏经阁第2层+贡献任务+中级俸禄 |
| **内门弟子** | 筑基+外门大比冠军(或特殊贡献) | 核心功法+藏经阁第3层+洞府使用权 |
| **内门执事** | 筑基后期+门派贡献达标 | 藏经阁第4层+收徒资格+高级俸禄 |
| **挂名长老** | 金丹期+掌门认可 | 门派高层权限+镇派功法权限 |
| **炼丹长老** | 炼丹造诣达标+门派贡献 | 专属炼丹房+材料配额+丹方库权限 |
| **炼器长老** | 炼器造诣达标+门派贡献 | 专属炼器室+材料配额+器方库权限 |
| **外务长老** | 声望达标+外交贡献 | 对外事务决策权+外交资源 |
| **授业长老** | 特定功法造诣达标 | 收徒+传功+功法传承 |

## 2. NPC沙盒系统

觅长生的NPC系统采用**时间同步演化**机制——NPC独立于玩家运行，拥有完整的修炼、突破、社交生命周期。

### 2.1 NPC生命周期

| 阶段 | 触发条件 | 对玩家的影响 |
|:---:|:--------:|------------|
| **修炼** | 世界时间推进 | NPC境界随时间提升——你可能被曾经的小辈超越 |
| **突破** | NPC修为达标 | NPC可能突破失败(损落)或成功(变强) |
| **结婚** | 好感度达标+适龄 | 可追求NPC结为道侣——或仅作为世界事件旁观 |
| **生子** | 结婚后 | 子代NPC出生→数年后成长为可互动角色 |
| **死亡** | 寿元耗尽/战斗死亡 | **你认识的人会老死**——世界真实感的核心来源 |

### 2.2 NPC关系网维度

| 关系类型 | 建立方式 | 影响 |
|:-------:|---------|------|
| **师徒** | 拜师/收徒 | 传授功法+门派关系绑定 |
| **道侣** | 追求+好感达标+双修 | 双修加速修炼+情感结局分支 |
| **好友** | 送礼/共同任务/性格相合 | 互相帮助+战斗支援 |
| **仇敌** | 截杀/背叛/竞争 | 被寻仇+声望下降 |
| **同门** | 同一门派 | 基础关系+门派事件共享 |

### 2.3 好感度系统

| 好感度区间 | 等级 | 互动权限 |
|:---------:|:----:|---------|
| 0-20 | 陌路 | 基础对话 |
| 20-40 | 相识 | 基础交易+赠送礼物 |
| 40-60 | 朋友 | 请求指点+组队 |
| 60-80 | 挚友 | 请求功法+邀请同行 |
| 80-100 | 知己/道侣 | 双修+托付遗物+特殊事件 |

| 好感度变化方式 | 变化量 | 说明 |
|:------------:|:-----:|------|
| 赠送合对方胃口的礼物 | +5-30/次 | 越稀有/对口的礼物加越多 |
| 共同战斗胜利 | +3-10/次 | 考验越难加越多 |
| 完成对方请求 | +10-20/次 | 请求难度与境界差相关 |
| 选择对话中的友好选项 | +1-5/次 | 需了解对方性格 |
| 威胁/攻击 | -20-50/次 | 大幅度降低好感 |
| 截杀对方亲友 | -50-100/次 | 几乎不可逆转 |

## 3. 声望系统

| 声望区间 | 等级称号 | 影响 |
|:-------:|:-------:|------|
| 0-500 | 默默无闻 | 商店无折扣——NPC互动无加成 |
| 500-2000 | 小有名气 | 商店折扣5%——部分高阶NPC愿意对话 |
| 2000-5000 | 名动一方 | 商店折扣10%——可参与高阶拍卖会 |
| 5000-10000 | 威震宁州 | 商店折扣15%——NPC见面有概率主动送礼 |
| 10000+ | 盖世天骄 | 商店折扣20%——NPC可能主动投靠 |

| 声望获取 | 增加量 | 备注 |
|:-------:|:-----:|------|
| 击败高阶敌人 | 50-300/次 | 敌人境界越高声望越多 |
| 门派大比获胜 | 100-500/届 | 名次越高越多 |
| 拍卖会竞价成功 | 20-100/次 | 拍品越贵声望越多 |
| 完成大型事件 | 200-1000/次 | 主线/支线大型剧情 |
| 炼丹/炼器极品 | 100-300/次 | 产出极品成色时触发 |

| 声望降低 | 减少量 | 备注 |
|:-------:|:-----:|------|
| 截杀正道NPC | -50-200/次 | 正道势力声望下降 |
| 背叛门派 | -500-1000 | 同门关系降至冰点 |
| 当众行凶 | -100-300/次 | 目击者越多损失越大 |

## 4. 道侣与双修系统

| 要素 | 说明 |
|:---:|------|
| **追求条件** | 好感度80+、境界不低于对方、单身状态 |
| **双修效果** | 双方同时获得修为增长+心境恢复(效率取决于双方契合度) |
| **契合度** | 灵根互补→双修效率更高(如金灵根+水灵根相生) |
| **道侣专属** | 道侣间可互相请求功法、丹药——高阶NPC道侣是高性价比"供货商" |
| **风险** | 道侣可能被杀/老死——造成心境大幅下降 |

## 5. 世界时间线与NPC演化

| 时间节点 | 世界事件 | NPC行为 |
|:-------:|---------|---------|
| 第1-20年 | 玩家炼气→筑基期 | NPC也在修炼——同龄竞争者陆续筑基 |
| 第20-50年 | 玩家筑基→金丹期 | 第一批NPC因寿元耗尽开始自然死亡 |
| 第50-100年 | 玩家金丹→元婴期 | 大量NPC更替——新一代修士崛起 |
| 第100-200年 | 玩家元婴→化神 | 旧识几乎全部老死——"百年孤独"叙事高潮 |
| 200年+ | 化神期 | 世界对你的认知已更新数代——真正的"老怪物" |

## 6. 设计特点分析

1. **NPC沙盒的"物是人非"叙事**：觅长生最具感染力的设计不是战斗或养成，而是时间流逝带来的NPC自然更替。游戏中期，玩家会发现曾经帮助自己的长辈已寿元耗尽；游戏后期，玩家认识的所有NPC都已不在人世。这种"长生者的孤独"叙事无法通过脚本事件实现——它是由系统规则自然涌现的。这种设计的说服力远超任何预设剧情。

2. **五大门派的差异化不足**：虽然五行属性区分清晰，但五大门派的晋升路线几乎完全相同（都是外门→内门→长老），差异仅体现在功法库上。对比太吾绘卷的门派系统——每个门派有独立的社交结构、武学倾向、立场倾向(善/恶/中庸)、地区影响力——觅长生的五派更像是"五个颜色的同一个模子"。

3. **与鬼谷八荒的NPC系统对比**：鬼谷八荒的NPC以**数量**和**互动频率**取胜——每月自动演化，NPC会主动找玩家互动(送礼/挑战/求助/结仇)，制造了大量"意外事件"。觅长生NPC的互动频率更低，但每个NPC的**生命周期完整**——你有足够时间看着一个NPC从炼气到金丹再到寿尽。鬼谷是"密密麻麻的社会网络"，觅长生是"精雕细琢的少数真实人物"。

4. **声望系统的实用价值明确**：觅长生的声望不是单纯的"成就计数器"，而是直接绑定商店折扣和NPC互动权限——高声望=更低购物成本+更广社交圈。这种设计让"刷声望"有了明确的功利驱动力。

5. **道侣系统的功能性大于情感性**：觅长生的道侣系统更接近"双修加速器"而非"恋爱模拟器"。双修带来的修为增长是实质性的数值收益，这使得"找道侣"成为优化修炼效率的策略选择而非情感选择——与《火山的女儿》那种以情感叙事为核心的社交系统形成鲜明对比。

6. **时间线设计的结构性叙事力量**：NPC在特定时间段的自然死亡节奏被精确设计——第20-50年第一批死亡（给玩家带来第一次冲击），第50-100年大量死亡（让玩家意识到自己也终将如此），100年后几乎全部更替（"我已非我"）。这不是偶然的数值设定，而是有意识的叙事节奏编排。
