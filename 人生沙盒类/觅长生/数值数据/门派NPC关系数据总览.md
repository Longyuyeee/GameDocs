## 二、门派晋升体系

### 2.1 晋升条件

| 弟子等级 | 境界要求 | 贡献要求（社区估算） | 宗门大比要求 | 解锁福利 |
|:-------:|:-------:|:-----------------:|:-----------:|---------|
| **外门弟子** | 炼气期 | — | — | 基础功法兑换权，宗门执法保护 |
| **内门弟子**（执事） | 筑基期 | ≥500贡献 | 无需 | 藏经阁地阶功法，师父传授，洞府使用权 |
| **核心弟子** | 金丹期 | ≥2,000贡献 | 宗门大比前4 | 天阶功法兑换权，宗门派遣任务 |
| **长老** | 元婴期 | ≥5,000贡献 | — | 收徒权，宗门决策权，永久洞府 |
| **掌门/太上长老** | 化神期 | — | 上代指定或公认 | 门派最高权限，资源调配权 |

### 2.2 贡献获取途径

| 途径 | 贡献获得量 | 限制 |
|:----|:---------:|:----|

| 击败入侵妖兽 | 100~300/次 | 随机事件 |
| 外门讲法 | +30道行经验 | 每年7月 |

## 三、NPC沙盒行为参数

### 3.1 自主行为系统

| 行为 | 参数（社区估算） | 说明 |
|:----|:-------------:|------|
| 自主修炼速度 | 玩家速度的30%~50% | NPC按设定效率自动增长修为 |
| 自主突破概率 | 基础~15%，有丹药可达~40% | NPC会自行收集突破材料尝试突破 |
| 自主采集/炼丹 | 低~中 | NPC会采集资源、炼丹，但效率远低于玩家 |
| 结婚条件 | 好感度≥80，相性匹配 | 玩家与NPC或NPC之间自动匹配 |
| 生子概率 | ~10%/年（婚后） | 有伴侣后每年概率|
| 子代继承 | 资质=父母平均值±随机 | 子代会继承部分天赋/属性 |
| 自然死亡年龄 | 炼气~150岁，筑基~250岁 | 超过寿元上限后死亡（延寿丹可延长） |

### 3.2 NPC动态事件

| 事件类型 | 触发概率 | 影响 |
|:--------|:-------:|------|
| NPC突破 | 低~中 | 突破成功则境界提升，改变NPC地位 |
| NPC寻仇 | 低 | 玩家与NPC有仇怨时可能触发 |
| NPC双修 | 低 | 好感度高的NPC结成道侣 |
| NPC炼丹/炼器 | 低 | NPC自行炼丹，成功后可能售卖 |
| NPC游历 | 中 | NPC在不同城市间移动 |
| NPC死亡 | 依寿元 | 寿元耗尽自然死亡，游戏中会发讣告 |

## 四、好感度系统

### 4.1 好感等级阈值

| 等级 | 阈值 | 解锁内容 |
|:---:|:---:|---------|
| 冷漠 | 0~19 | 基本对话 |
| 友善 | 20~39 | 可以交易，基础请教 |
| 亲近 | 40~59 | 可请求指点，获得小额赠礼 |
| 信任 | 60~79 | **可请教功法/神通**，获得中额赠礼 |
| 亲密 | 80~99 | 可双修，获得大量赠礼/灵石，**可邀请同行** |
| 道侣 | 100+ | 永久双修伴侣，共享部分属性加成 |

### 4.2 好感度增减行为

| 行为 | 好感变化 | 限制 |
|:----|:-------:|:----|
| 送礼（喜好物品） | +10~+30 | 不同NPC喜好不同，标"喜好"则翻倍 |
| 送礼（普通物品） | +1~+5 | 价值越低加成越少 |
| 论道 | +5~+15 | 依论道质量（悟性/悟道等级） |
| 双修 | +5~+10/次 | 需亲密等级以上 |
| 完成委托 | +10~+20 | 限任务相关NPC |
| 赠送灵石 | 每1,000灵石+1 | 性价比极低 |
| 切磋（胜利） | +2 | 每日限1次 |
| 切磋（失败） | +5 | 看似输了实则涨好感 |
| 杀害亲友 | -30~-50 | 严重降好感 |
| 拒绝请求 | -1~-5 | 多次拒绝可降至冷漠 |

### 4.3 双修系统

| 项目 | 数据 |
|:----|:----|
| 解锁条件 | 好感度≥80（亲密等级） |
| 修为增益 | 双方各获得相当于~1个月闭关的修为量 |
| 冷却时间 | 约30天（游戏内） |
| 额外收益 | 概率获得悟道感悟，双倍修为（特定天赋） |
| 风险 | 无，但需注意道侣寿元 |

## 五、声望系统

### 5.1 声望等级

| 声望等级 | 声望值范围 | 称号 | 影响 |
|:-------:|:---------:|:----|:-----|
| -5 | ≤-500 | 魔头 | 正道门派敌视，商店拒绝交易，NPC恐惧 |
| -4 | -499~-200 | 恶名昭彰 | 部分商店涨价50%，低好感开局 |
| -3 | -199~-50 | 声名狼藉 | 部分NPC态度冷漠 |
| -2 | -49~-10 | 风评不佳 | 轻微负面影响 |
| -1 | -9~0 | 默默无闻 | 无明显影响 |
| 0 | 0 | 初出茅庐 | 默认状态 |
| +1 | 1~99 | 小有名气 | 部分NPC初始好感+5 |
| +2 | 100~499 | 小有声望 | **商店折扣5%**，NPC好感初始+10 |
| +3 | 500~999 | 名震一方 | 商店折扣10%，可解锁特殊对话 |
| +4 | 1,000~2,999 | 威名远扬 | **商店折扣15%**，可请教"秘"字功法 |
| +5 | 3,000+ | 名满天下 | 商店折扣20%，门派直接以礼相待 |

### 5.2 声望增减途径

| 行为 | 声望变化 | 备注 |
|:----|:-------:|------|
| 完成宗门任务 | +5~+50 | 取决于任务难度 |
| 宗门大比夺冠 | +20 | 每30年 |
| 击杀妖兽 | +1~+10 | 视妖兽强度 |
| 拍卖会高价成交 | +2~+5 | 身价象征 |
| **截杀天机阁执事** | **-10/次** | 虽然收益高但降声望 |
| 杀害正道NPC | -20~-50 | 严重降声望 |
| 灭门派（拜山） | -100~-300 | 最大降声望行为 |
| 完成特殊剧情 | +50~+200 | 主线/重要支线奖励 |

### 5.3 声望对游戏的影响汇总

| 影响维度 | 低声望（负） | 高声望（正） |
|---------|------------|------------|
| 商店价格 | 涨价20%~50% | **折扣5%~20%** |
| NPC初始态度 | 冷漠/恐惧 | 友善/尊敬 |
| 功法获取 | 无法请教"秘"字功法 | 可请教所有功法 |
| 事件触发 | 正道事件关闭，魔道事件开启 | 正道事件优先 |
| 门派关系 | 可能被驱逐 | 晋升更容易 |
| 拍卖会 | NPC可能抵制 | NPC更愿意竞价 |

---

> 核心提示：觅长生的NPC沙盒系统是游戏"修仙小说生成器"体验的关键。NPC会自主修炼、突破、结婚、生子乃至老死，玩家出关后发现故人已逝的"沧海桑田"感正是此系统的核心设计目标。建议玩家在闭关之余多与关键NPC互动，不要错过限时剧情事件。

> 注：部分NPC名称、境界数据及详细行为参数为社区估算或待补充数据，游戏版本更新可能导致个别NPC设定调整。
