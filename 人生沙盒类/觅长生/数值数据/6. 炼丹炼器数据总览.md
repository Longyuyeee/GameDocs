## 一、炼丹系统

### 1.1 药力值换算

药材品级与药力值的关系：

| 品级 | 药力值 | 相对一品倍数 | 常见药材举例 |
|:---:|:------:|:-----------:|------------|
| 一品 | 1 | 1× | 红绫草、夏枯草、银月花、宁心草、蛇涎果 |
| 二品 | 3 | 3× | 补天芝、轻灵草、天元果、龙葵、罗犀草 |
| 三品 | 9 | 9× | 五柳根、天灵果、幻心草、乌稠木、龙鳞果 |
| 四品 | 36 | 36× | 鸡冠草、银精芝、白沉脂、绝魂草、鬼面花 |
| 五品 | 180 | 180× | 阴凝草、三叶青芝、地火心芝、血玉骨参、狼桃 |
| 六品 | 1080 | 1080× | 苦蔓藤、地心淬灵乳、天麻翡石精、八角玄冰草 |

**核心规律**：低品级可向上替代（如1个三品=3个二品=9个一品），高品级药材放入低品丹炉会浪费药力产生药渣。

### 1.3 丹毒系统

服用丹药会产生丹毒，丹毒累积到一定值会产生惩罚效果：

| 丹毒值区间 | 惩罚效果 |
|:---------:|----------|
| 0~30 | 无显著惩罚 |
| 31~60 | 修为获取速度开始降低（约-10%~-25%） |
| 61~90 | 修为获取速度大幅降低（约-30%~-50%），生命回复受抑制 |
| 91~110 | 修为获取速度锐减（约-60%~-80%），战斗中可能触发"丹毒发作"负面状态 |
| 111~119 | 修为几乎停滞，战斗中每回合概率触发"丹毒反噬"，损失生命 |
| 120 | **毒发身亡**——角色立即死亡 |

**清脉丹**（丹毒-10）：

| 项目 | 内容 |
|------|------|
| 丹方名 | 清脉丹 |
| 主药 | 红绫草×3 |
| 辅药 | 凝血草×2 |
| 药引 | 恒心草×1 |
| 效果 | 移除丹毒-10 |
| 品级 | 一品 |
| 推荐度 | ★★★★★（必备丹药，定期服用） |
| 单次成本 | 约100~200灵石 |

**丹毒管理策略**：建议每服用4~5颗丹药后服用1颗清脉丹，保持丹毒在30以下。高品级丹药产生的丹毒更多，元婴期后服用五品丹药单次可能产生5~10点丹毒。

### 1.4 耐药性机制

同种丹药连续使用时，效果会递减：

| 连续使用次数 | 效果保留率 |
|:-----------:|:---------:|
| 第1次 | 100% |
| 第2次 | 80% |
| 第3次 | 60% |
| 第4次 | 40% |
| 第5次 | 20% |
| 第6次及以后 | 10% |

**耐药性清除**：停止使用该丹药一定时间（约3~6个月）后耐药性逐步恢复。不同种类的丹药耐药性独立计算，因此建议轮换使用多种丹药。

## 二、炼器系统

### 2.1 材料五维属性（5类×8属性×阴阳交叉表）

炼器材料共分**5种类型**，每种有**五维属性**（韧性、坚固、亲和、灵性、操控），上限均为500：

| 材料类型 | 韧性 | 坚固 | 亲和 | 灵性 | 操控 | 特性 |
|---------|:---:|:---:|:---:|:---:|:---:|------|
| **金属** | 100 | 0 | 75 | 25 | 50 | 高韧性、无坚固 |
| **矿石/石** | 75 | **100** | 50 | 0 | 25 | 高坚固、无灵性 |
| **牙骨** | 25 | 50 | 0 | 75 | **100** | 高操控、无亲和 |
| **皮鳞** | 0 | 25 | **100** | 50 | 75 | 高亲和、无韧性 |
| **灵武** | 50 | 75 | 25 | **100** | 0 | 高灵性、无操控 |

**五维总和标准**：≥2200为合格（不浪费材料灵力），标准配方（2-2-2-2-2型）总和2500。

**材料属性交叉表（8属性×阴阳）**：

| 属性类型 | 天阳(阳) | 地阴(阴) |
|:--------:|:--------:|:--------:|
| 金 | 天阳金 | 地阴金 |
| 木 | 天阳木 | 地阴木 |
| 水 | 天阳水 | 地阴水 |
| 火 | 天阳火 | 地阴火 |
| 土 | 天阳土 | 地阴土 |
| 混元(气) | 天阳混元 | 地阴混元 |
| 神念(神) | 天阳神念 | 地阴神念 |
| 剑 | 天阳剑 | 地阴剑 |

**重要规则**：天阳与地阴材料不能混合使用（五行相克材料不能同时成器），除非点了器道相合天赋。

### 2.3 标准炼器配方（灵纹前）

**标准配方（2-2-2-2-2型）**：金属×2 + 矿石×2 + 牙骨×2 + 皮鳞×2 + 灵武×2 = 五维总和2500（满值）

**常用简化配方（仅用3种材料，五维≥2200）**：

- 4金属 + 4矿石 + 2牙骨 = 2200

- 6金属 + 2牙骨 + 2灵武 = 2250
- 4金属 + 4牙骨 + 2皮鳞 = 2250

### 2.5 器道灵纹

灵纹是炼器的进阶功能——为已有装备附加额外效果，消耗材料灵力：

| 灵纹名 | 副作用 | 增加灵力 | 材料需求 |
|:------:|:------:|:--------:|---------|
| **白虎灵纹** | 对自己造成真实伤害（6~30） | +30~150 | 金属性材料+高品妖兽血液 |
| **玄武灵纹** | 添加debuff（中毒/缠绕/霜冻/灼烧） | +30~120 | 水/土属性材料+龟甲类材料 |
| **青龙灵纹** | 技能冷却时间乘以倍率(1.5×~3×) | 灵力×倍率 | 木属性材料+龙族材料 |
| **朱雀灵纹** | 生命值低于阈值时触发(50%/25%/10%) | 灵力×1.5/×2/×3 | 火属性材料+羽翼类材料 |

**灵纹策略**：灵纹可通过副作用换取额外灵力，使装备达到更高品质。白虎纹适合爆发流派（不怕自伤），玄武纹适合控制流派，青龙纹适合慢速高爆发build，朱雀纹适合残血反杀打法。

### 2.6 赚钱炼器推荐

**炼器赚钱的核心思路**：使用中品材料制作"纯阳法宝"级装备，利用器道灵纹提升品质，在拍卖会或南崖城高价出售。
