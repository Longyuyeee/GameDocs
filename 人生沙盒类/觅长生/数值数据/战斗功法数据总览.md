## 一、全属性灵气生成机制

觅长生的战斗核心是 **五行灵气卡牌系统**。玩家每回合抽取一定数量的灵气牌，每张灵气牌具有金/木/水/火/土五种属性之一，以此释放神通。

### 1.1 灵根权重→灵气概率分布公式

灵气出牌概率完全由 **角色灵根权重** 决定：

```
某属性灵气出现率(%) = (该属性灵根权重 / 全属性灵根权重之和) × 100%
```

| 灵根配置 | 金 | 木 | 水 | 火 | 土 | 概率分布说明 |
|---------|:--:|:--:|:--:|:--:|:--:|------------|
| 天灵根（金） | 100 | 0 | 0 | 0 | 0 | 几乎100%出金灵气 |
| 天灵根（木） | 0 | 100 | 0 | 0 | 0 | 几乎100%出木灵气 |
| 双灵根（金木） | 50 | 50 | 0 | 0 | 0 | 金木各~50% |
| 三灵根（金木水） | 34 | 33 | 33 | 0 | 0 | 各~33% |
| 四灵根 | 25 | 25 | 25 | 25 | 0 | 各~25% |
| 五灵根（伪灵根） | 20 | 20 | 20 | 20 | 20 | 各~20% |

### 1.2 每回合灵气获取数

| 境界 | 基础每回合灵气数 | 上限（基础） | 备注 |
|:---:|:--------------:|:----------:|------|
| 炼气期 | 3~4张 | 10 | 部分功法/神通可增加抽牌数 |
| 筑基期 | 4~5张 | 15 | 境界提升增加灵气上限 |
| 金丹期 | 5~6张 | 20 | 神通可增加临时灵气 |
| 元婴期 | 6~7张 | 25 | 元婴后灵气管理更加灵活 |
| 化神期 | 7~8张 | 30(社区估算) | 高阶战斗需大量灵气支撑 |

> 抽牌数可通过功法特效（如"每回合多抽一张XX属性牌"）、悟道天赋获得额外加成。

## 二、五行相生相克

五行顺序为：**金 → 水 → 木 → 火 → 土 → 金**

### 2.1 五行生克规则

| 关系 | 规则 | 具体对应 |
|:---:|------|---------|
| **相生** | 相邻相生 | 金生水 → 水生木 → 木生火 → 火生土 → 土生金 |
| **相克** | 隔一相克 | 金克木 → 木克土 → 土克水 → 水克火 → 火克金 |

### 2.2 战斗中触发生克的效果

| 机制词缀 | 触发条件 | 效果说明 |
|---------|---------|---------|
| **【生】** | 释放技能时使用的灵气属性满足相生关系（如金+水） | 触发额外增益：额外抽牌/伤害加成/灵气返还 |
| **【同】** | 释放技能时使用的灵气属性相同（如金+金） | 触发额外效果：伤害增加/层数叠加 |
| **【克】** | 释放技能时使用的灵气属性满足相克关系（如火+金） | 触发额外控制效果：易伤/滞气/灼烧等 |
| **【连击】** | 上一个释放的技能满足特定属性条件 | 触发连击特效：额外伤害/叠加状态 |

### 2.3 五行增益循环

以战斗中的五行循环为例：
```
木（释放木系神通）→ 触发【生】→ 获得火灵气 → 火系神通增强
火（释放火系神通）→ 触发【生】→ 获得土灵气 → 土系神通增强
土（释放土系神通）→ 触发【生】→ 获得金灵气 → 金系神通增强
金（释放金系神通）→ 触发【生】→ 获得水灵气 → 水系神通增强
水（释放水系神通）→ 触发【生】→ 获得木灵气 → 木系神通增强
```

> 熟练运用相生关系，可以在一回合内打出多段高伤连击。五行剑流的核心逻辑即基于此循环。

## 三、功法等级与突破系统

### 3.1 功法阶位

| 功法等级 | 解锁境界 | 基础属性加成范围 | 神通最大等级 |
|:-------:|:-------:|:--------------:|:----------:|
| 人阶 | 炼气期 | 血量+20~80，神识+1~3 | Lv.3 |
| 地阶 | 筑基期 | 血量+80~300，神识+3~8 | Lv.4 |
| 天阶 | 金丹期 | 血量+300~1000+，神识+8~15 | Lv.5 |

### 3.2 功法突破（5层）

每个功法均有 **5层突破** 等级，突破层数可提升功法效果：

| 功法层数 | 突破需求（境界） | 效果增幅（社区估算） |
|:-------:|:-------------:|:----------------:|
| Lv.1→Lv.2 | 炼气中期 | 基础属性+20% |
| Lv.2→Lv.3 | 炼气后期 | 基础属性+30% |
| Lv.3→Lv.4 | 筑基中期 | 基础属性+40%，解锁隐藏效果 |
| Lv.4→Lv.5 | 金丹期 | 基础属性+50%，特效质变 |

> 突破功法层数需消耗悟道值、灵石、特定材料（突破丹）。

### 3.3 功法修炼方式

| 修炼方式 | 效率 | 限制 |
|---------|:---:|------|
| 闭关修炼 | 基础效率 | 无特殊限制 |
| 洞府闭关 | **高（1.5x~2x）** | 需拥有洞府 |
| 双修 | **极高** | 需双修道侣，受好感度限制 |
| 丹药辅助 | 倍率加成 | 消耗对应修炼丹药 |
| 悟道加持 | 持续加成 | 需要对应悟道等级 |
