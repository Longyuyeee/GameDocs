## 一、境界与寿元对照

### 1.1 角色境界寿元表

《觅长生》中寿元是核心压力机制。每次突破大境界增加寿元上限：

| 境界 | 基础寿元(年) | 突破增加的寿元 | 累计寿元 | 备注 |
|:----:|:----------:|:-------------:|:--------:|------|
| **炼气期** | 50~75 | — | 50~75 | 初始寿元由天赋决定 |
| **筑基期** | — | +100 | 150~175 | 筑基后寿元大幅提升 |
| **金丹期** | — | +300 | 450~475 | 金丹期正式迈入"长生"门槛 |
| **元婴期** | — | +500 | 950~975 | 元婴期寿元近千年 |
| **化神期** | — | +1000 | 1950~1975 | 化神飞升，寿元接近2000年 |

**总可用寿元**：若不依赖延寿手段，最大寿元约**1950~1975年**（化神期）。加上延寿丹等可突破2000年，但理论寿命仍有上限（社区估算约2500~3000年为游戏机制软上限）。

### 1.2 NPC标准寿命

NPC寿命是《觅长生》"物是人非"情感设计的基础：

| NPC境界 | 标准寿命(社区估算) | 与玩家关系 |
|:-------:|:-----------------:|-----------|
| 炼气期 | ~60年 | 多数相遇的NPC为炼气期，60年内就会寿终正寝 |
| 筑基期 | ~160年 | 筑基期NPC可存活较久，但仍是"中期"寿命 |
| 金丹期 | ~460年 | 金丹期NPC是游戏中的中流砥柱，如门派长老 |
| 元婴期 | ~960年 | 元婴期NPC极为罕见，各派掌门/太上长老级别 |
| 化神期 | ~1960年 | 化神期NPC基本站在世界顶端，已飞升或闭关 |

**NPC寿命机制**：
- NPC寿元耗尽后会陨落（死亡），无法复活
- 玩家可以给NPC赠送寿元丹延长其寿命（延寿丹+10、青冥丹+20、长生丹+30、六阳长生丹+50）
- NPC全套餐具寿元丹可延长约+500年
- NPC突破大境界也会增加寿元，与玩家机制相同
- "物是人非"：50年内未见的炼气期好友可能已经老死

### 1.3 延寿手段汇总

| 延寿手段 | 增加寿元 | 获取方式 | 成本/难度 |
|---------|:--------:|---------|:---------:|
| **延寿丹** | +10 | 炼丹（阴凝草×2+玉龙参×4+七星草×1） | 低（三品丹药，材料易得） |
| **青冥丹** | +20 | 炼丹（乌稠木×6+白沉脂×6+幻心草×1） | 中（四品丹药，需拍卖行材料） |
| **长生丹** | +30 | 炼丹（阴凝草×8+血玉骨参×2+苦蔓藤×1） | 高（五品丹药） |
| **六阳长生丹** | +50 | 炼丹（阴凝草×5+天麻翡石精×1+苦蔓藤×5） | 极高（六品丹药） |
| **续命丹** | +5 | 炼丹（天元果×3+玉龙参×3+百草露×1） | 极低（二品丹药） |
| 特定功法 | 不定 | 特殊功法/天阶功法附带寿元加成 | 不定 |
| 奇遇奖励 | 不定 | 部分任务/秘境事件奖励寿元 | 不定 |

**理论最大寿元**（社区估算）：
```
基础化神寿元(~1975) + 延寿丹(+10) + 青冥丹(+20) + 长生丹(+30) + 六阳长生丹(+50)
+ 续命丹(+5) + 未知奇遇/功法加成(约+100~200)
≈ 2200~2300年
```
注意：丹药存在耐药性，同种延寿丹连续服用效果递减，建议分散服用不同品类。

## 三、"出关"时间代价

### 3.1 标准时间消耗

游戏内时间流动是连续的，任何行动都会消耗时间：

| 行动 | 标准时间消耗 | 影响因素 |
|:----:|:-----------:|---------|
| **一次标准闭关** | 3~12个月 | 功法品阶、资质、悟性、心境 |
| **一次领悟功法** | 1~6个月 | 悟性、功法品阶 |
| **炼丹(一炉)** | 1~3个月 | 丹药品级、丹道等级 |
| **炼器(一件)** | 1~4个月 | 器道等级、材料品级 |
| **大地图移动** | 1~15天/格 | 灵舟/遁速、距离 |
| **海上移动** | 3~30天/格 | 灵舟等级、海域 |
| **论道一次** | 1~3个月 | NPC等级、主题匹配度 |
| **城市间传送** | 1~3天 | 传送阵使用 |
| **坊市购物** | 1~7天 | 购物数量 |
| **奇遇事件** | 1~30天/次 | 事件类型 |

### 3.2 默认时间流速

- 1年 = 12个月
- 1个月 ≈ 30天（游戏内时间单位）
- 角色出生：第1年1月
- 标准建议：40年内筑基 / 150年内金丹 / 400年内元婴 / 1000年内化神

## 五、时间设计哲学

《觅长生》的时间系统是其核心设计差异点：

1. **寿元倒计时**：与大多数开放世界RPG不同，时间是稀缺资源而非免费资源
2. **机会成本**：花1年探索 = 少1年修炼，每步决策都需要权衡
3. **不可逆的NPC死亡**：NPC寿尽真实死亡，无法复活（区别于大多数RPG的"任务NPC永生"）
4. **延寿丹稀缺性**：不能无限延寿，耐药性限制了"靠吃药永生"的可能性
5. **柔性时间压力**：不同于硬性deadline，觅长生的时间压力是"越晚突破越危险"而非"到时间就失败"
