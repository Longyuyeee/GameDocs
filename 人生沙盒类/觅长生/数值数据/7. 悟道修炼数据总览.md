## 一、悟道系统

### 1.1 11大悟道方向概述

悟道是《觅长生》的核心成长系统之一，共11个方向（5行+6特）：

| 悟道方向 | 分类 | 核心效果侧重 |
|:--------:|:----:|------------|
| **金** | 五行·金 | 金属性功法/神通强化、金属性灵气效率 |
| **木** | 五行·木 | 木属性功法/神通强化、回血/毒系增强 |
| **水** | 五行·水 | 水属性功法/神通强化、冰冻/灵气回复 |
| **火** | 五行·火 | 火属性功法/神通强化、灼烧/爆发伤害 |
| **土** | 五行·土 | 土属性功法/神通强化、护盾/减伤 |
| **神** | 特殊·神 | 神识相关（探查距离、神念攻击、识破隐身） |
| **体** | 特殊·体 | 肉身强度（生命上限、遁速、物理防御） |
| **剑** | 特殊·剑 | 剑系功法/神通专精、剑气、剑意 |
| **气** | 特殊·气 | 灵气管理（最大灵气、灵气回复效率） |
| **丹** | 特殊·丹 | 炼丹相关（成丹率、丹药效果、丹毒减免） |
| **器** | 特殊·器 | 炼器相关（装备品质、灵纹效果、材料利用率） |

### 1.3 功法境界对悟道境界的需求

| 功法品级 | 所需悟道境界 |

### 1.4 各方向关键悟道技能示例

以下为验证有效的部分悟道技能效果，按方向列出代表性技能：

**金之道**：金系伤害加成、金灵根权重提升、金属性神通消耗减少、金属性暴击率
**木之道**：木系伤害加成、回血效果提升、中毒层数叠加、木灵气转化效率
**水之道**：水系伤害加成、冰冻概率提升、水灵气回复、每回合额外水灵气
**火之道**：火系伤害加成、灼烧伤害倍率、火系神通抽牌率、爆发期伤害增幅
**神之道**：神识范围扩大（每级+1）、神念攻击力、识破概率、论道先手权
**体之道**：生命上限%（每级约+5%）、遁速基础值、物理减伤率、伤势恢复速度
**剑之道**：剑气层数上限（每级+1）、剑意伤害倍率、剑系神通减耗、剑系暴击伤害
**气之道**：最大灵气上限（每级+1~2）、每回合灵气回复量、灵气转化效率、同属性灵气储存上限
**丹之道**：成丹率提升（每级约+3%）、丹药效果增强（每级约+2%）、丹毒减免、炼丹时间缩短
**器之道**：装备灵力上限提升、灵纹效果增强、材料利用率提升、可混合阴阳材料

**获取门槛**：每个方向的技能树共5~10个节点，需要以悟道境界为前置（如"初窥门径"可学基础节点，"融会贯通"可学高级节点）。

### 1.5 论道系统

论道是获取悟道经验的重要手段：

| 项目 | 说明 |
|------|------|
| **参与条件** | 与NPC好感度达到一定程度（通常为点头之交以上） |
| **论道收益** | 根据NPC的悟道等级和方向产生思绪（100~5000悟道经验） |
| **单次论道时间** | 约1~3个月（游戏中） |
| **论道上限** | 最多获得**30悟道点** |
| **同NPC冷却** | 约5~10年（视NPC悟道等级而定） |
| **收益影响因素** | ① 论道主题方向匹配度 ② NPC与玩家的悟道等级差 ③ NPC好感度 ④ 玩家悟道等级 |

**NPC论道收益参考**：

| NPC悟道境界 | 产出思绪品级 | 悟道经验参考值 |
|:-----------:|:-----------:|:--------------:|
| 初窥门径 | 二品 | 100~300 |
| 略有小成 | 三品 | 300~800 |
| 融会贯通 | 四品 | 1000~2000 |
| 道之真境 | 五品 | 3000~5000 |
| 大道已成 | 五品+ | 5000~10000 |

**论道技巧**：找比自己高1~2个境界的NPC论道收益最高。找"大道已成"级NPC论道可获得高质量思绪，是目前后期最有效的悟道经验获取方式。

## 二、修炼系统

### 2.1 资质/悟性/心境三角效率

修炼速度由三大基础属性共同决定：

#### 资质 — 修炼速度加成（线性/阶梯）

| 资质值 | 修炼速度加成(社区估算) | 对应灵根品质 |
|:------:|:-------------------:|:-----------:|
| 1~30 | -30%~-10% | 废灵根 |
| 31~60 | -10%~+0% | 低品质灵根 |
| 61~100 | +0%~+30% | 中等灵根 |
| 101~140 | +30%~+60% | 高品质灵根/单灵根 |
| 141~180 | +60%~+100% | 天灵根/先天道体 |
| 181~200 | +100%~+150% | 极限资质（丹药+奇遇堆叠） |

**资质影响**：资质影响闭关时修为获取速度，同时影响神通/功法的修炼成功率。天灵根（单灵根150）筑基后修炼速度约为废灵根的2~3倍。

#### 悟性 — 感悟速度倍率

| 悟性值 | 感悟速度倍率(社区估算) | 效果描述 |
|:------:|:-------------------:|---------|
| 1~30 | 0.5×~0.7× | 感悟极慢，功法升级困难 |
| 31~60 | 0.7×~0.9× | 感悟慢 |
| 61~100 | 0.9×~1.1× | 正常感悟速度 |
| 101~140 | 1.1×~1.4× | 明显加速 |
| 141~180 | 1.4×~1.8× | 快速领悟功法/神通 |
| 181~200 | 1.8×~2.5× | 极速感悟（丹药堆积） |

**悟性效果**：悟性决定"领悟"功法/神通的时间长短，高悟性可以大幅减少闭关次数。

#### 心境 — 闭关效率修正

| 心境值 | 闭关效率修正(社区估算) |
|:------:|:--------------------:|
| 0~20 | -40%~-25%（极低效率，易走火入魔） |
| 21~40 | -25%~-10%（低效） |
| 41~60 | -10%~+0%（略低） |
| 61~80 | +0%~+5%（正常） |
| 81~100 | +5%~+10%（良好） |
| 101~120 | +10%~+15%（高效） |
| 121~140 | +15%~+20%（极高效） |

**心境维持**：心境会随时间缓慢下降（每月约-0.1~-0.5），突破境界时心境重置/提升。可通过服用冰心丹(+1)、鬼面炼心丹(+7)、特殊奇遇等方式提升。

### 2.2 闭关修炼机制

**一次标准闭关流程**：
1. 选择闭关地点（洞府/灵脉/普通场所）
2. 设定目标（修炼功法/神通/修为）
3. 消耗时间
4. 获得修为/进度

**每次闭关时间**：根据功法和资质变化，通常为3~12个月一次标准闭关周期。

**闭关速度公式（玩家，社区推算）**：
```
每月修为获取 = 功法基础修为 × (1 + 资质加成%) × 心境修正 × (1 + 地点灵气加成%) × (1 + 悟道加成%)
```

### 2.3 主修功法修炼速度倍率

功法是修炼速度的核心决定因素：

| 功法品阶 | 典型功法名 | 基础修炼速度(社区估算) | 适用境界 | 速度倍率(相对人阶下) |
|:-------:|:---------:|:--------------------:|:--------:|:-----------------:|
| 人阶下 | 练气诀 | ~60/月 | 炼气 | 1× |
| 人阶中 | 混元诀 | ~120/月 | 炼气~筑基 | ~2× |
| 人阶上 | 三阳诀 | ~200/月 | 炼气~筑基 | ~3.3× |
| 地阶下 | 玄冰诀 | ~360/月 | 筑基~金丹 | ~6× |
| 地阶中 | 紫阳诀 | ~540/月 | 筑基~金丹 | ~9× |
| 地阶上 | 五行纳气法 | ~720/月 | 筑基~金丹 | ~12× |
| 天阶下 | 太和真气 | ~1080/月 | 金丹~元婴 | ~18× |
| 天阶中 | 混元天诀 | ~2160/月 | 金丹~化神 | ~36× |
| **天阶上** | **玄清天衍录** | **~6480/月**(满层) | 元婴~化神 | **~108×** |

**功法搭配策略**：
- **炼气期**：人阶功法足够，重点在快速筑基
- **筑基期**：尽快更换地阶中/上功法
- **金丹期**：天阶下起步，推荐在宁州古迹获得玄清天衍录
- **元婴期**：天阶中/上功法为核心，配合洞府灵脉+满资质可达6~7万/月

### 2.4 洞府与灵脉加成

| 洞府等级 | 灵脉加成(灵气浓度) | 获取方式 |
|:--------:|:-----------------:|---------|
| 逸风城普通洞府 | +20%~+50% | 逸风城购买(约5万灵石) |
| 逸风城上品洞府 | +50%~+100% | 逸风城购买(约20万灵石) |
| 上品灵眼 | +50%~+80% | 门派贡献兑换/奇遇 |
| 五品聚灵阵 | +80%~+120% | 炼器/阵法制作 |
| 六品聚灵阵 | +150%~+230% | 炼器/阵法制作（极难） |
| 门派专属洞府 | +30%~+150%(依门派等级) | 门派晋升获得 |

**洞府建议**：逸风城洞府是性价比最高的选择，后期升级灵脉+聚灵阵可大幅提升闭关效率。

## 三、修炼效率优化策略

### 3.1 各阶段核心目标

| 阶段 | 核心目标 | 关键手段 |
|:----:|---------|---------|
| 炼气期(~40年) | 尽快筑基 | 嗑药（天尘丹等修为丹）、完成英杰会获取筑基丹 |
| 筑基期(~150年) | 提升资质/悟性 | 服用草还丹(+资质)、启灵丹(+悟性)、更换地阶功法 |
| 金丹期(~400年) | 获取天阶功法 | 天魔眼探索、宁州古迹准备、炼丹赚钱买丹药 |
| 元婴期(~800年) | 满配修炼体系 | 玄清天衍录+满资质+满洞府+六品丹药 |
| 化神期前(~1000年) | 悟道+寿元兜底 | 吃满寿元丹、悟道大道已成、准备化神材料 |

### 3.2 快速修炼的核心循环

1. **赚钱**：炼丹（蚀月强魂丹/天尘丹）→ 出售赚灵石
2. **买药**：购买高品药材 → 炼制修为丹/资质丹
3. **闭关**：服用修为丹后闭关获得大量修为
4. **悟道**：穿插论道 + 学习新功法/神通提升悟道等级
5. **突破**：满足条件后尝试突破大境界
