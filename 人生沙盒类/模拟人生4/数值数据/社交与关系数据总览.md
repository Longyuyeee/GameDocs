## 一、友谊与浪漫双轨关系系统

### 1.1 关系等级阈值

| 关系等级(友谊/绿条) | 数值范围 | 对应关系标签 | 核心互动解锁 |
|:-----------------:|:-------:|------------|------------|
| 陌生人(Stranger) | 0 | - | 基础介绍/打招呼 |
| 相识(Acquaintance) | 1-29 | 点头之交 | 闲聊/了解 |
| 朋友(Friend) | 30-49 | 朋友 | 邀请社交/一起去... |
| 好友(Good Friend) | 50-69 | 老友 | 要求帮助/请求物品 |
| 挚友(Best Friend) | 70-99 | 至交好友 | 搬入/要求同居(儿童/青少年间) |
| 灵魂挚友(Soul Mate) | 100 | 灵魂之友 | 特殊互动"分享心事" |

| 关系等级(浪漫/粉条) | 数值范围 | 对应关系标签 | 核心互动解锁 |
|:-----------------:|:-------:|------------|------------|
| 陌生人 | 0 | - | 基础调情需魅力≥2 |
| 有点兴趣 | 1-19 | 轻微吸引力 | 害羞调情 |
| 有感觉 | 20-39 | 吸引力 | 浪漫互动基础套 |
| 恋人(Lover) | 40-59 | 在恋爱 | 接吻/拥抱/正式约会 |
| 激情恋人 | 60-79 | 热恋 | 更亲密互动/公开关系 |
| 订婚 | 80-89 | 已订婚 | 计划婚礼/搬入同住 |
| 已婚 | 90-100 | 已婚 | 全部浪漫互动解锁/婚后加成 |

### 1.2 关系条的叠加机制

```
关系总状态 = 友谊值(Friendship) × 权重 + 浪漫值(Romance) × 权重
```

| 状态 | 友谊要求 | 浪漫要求 | 示例 |
|------|:-------:|:-------:|------|
| 纯好友 | ≥50 | <20 | 男女纯友谊 |
| 暧昧 | ≥30 | ≥30 | 友情以上恋人未满 |
| 恋爱中 | ≥50 | ≥60 | 正式恋爱 |
| 订婚/已婚 | ≥70 | ≥80 | 家庭关系确立 |
| 单相思 | <30 | ≥50 | 一方有感觉一方无 |
| 分手后 | 波动 | <0 | 绿色正/粉色负 |

> 关系值范围: 0~100(正向) 或 -100~0(负向,敌人/仇人)。负关系可通过道歉/友善互动修复。

### 2.2 影响成功率的因素

| 影响因素 | 成功率修正 | 说明 |
|---------|:---------:|------|
| **魅力等级** | +2-3%/级 | 每级魅力约+2~3%成功率 |

| **自信情绪** | +10-15% | 自信状态下成功率提升 |

| **特质:善良** | +20%(友善) | 友善互动专属加成 |

| **先前拒绝(同一天)** | -20~-50% | 连续多次相同请求递减 |

## 三、浪漫边界系统

### 3.1 四种边界类型

| 边界类型 | 预设值(默认) | 可调范围 | 说明 |
|---------|:----------:|:-------:|------|
| **浪漫边界(Romantic)** | 不限制 | 不限制/排他/只对主控 | 控制市民可以与其他谁发生浪漫互动 |
| **性向边界(Physical Intimacy)** | 不限制 | 不限制/排他/只对主控 | 控制市民可以与其他谁有亲密行为 |
| **嫉妒边界(Jealousy)** | 开启 | 开启/关闭(自定义) | 伴侣与他人调情是否触发嫉妒 |
| **探索边界(Casual)** | 开放可探索 | 是/否 | 允许市民自主探索浪漫互动 |

### 3.2 边界自定义范围详解

| 边界设置 | 效果 | 适用范围 |
|---------|------|---------|
| **不限制(Allow)** | 市民可与任意他人发生该类型互动 | 自由恋爱/开放关系 |
| **排他(Exclusive)** | 只与伴侣一人发生该类型互动 | 传统关系 |
| **只对主控(Only My Sim)** | 只与指定主控市民互动(非主控NPC不可) | 控制型关系 |
| **嫉妒开关(Jealousy: On)** | 伴侣越界→触发"生气"情绪+关系值下降 | 默认设置 |
| **嫉妒开关(Jealousy: Off)** | 伴侣越界→无情绪反应 | 开放关系(需沟通) |

### 3.3 嫉妒触发阈值

| 互动类型 | 嫉妒阈值 | 关系下降值 |
|---------|:-------:|:---------:|
| 轻度调情(赞美外貌) | 浅 | -10 浪漫 |
| 中度调情(暧昧) | 中 | -20 浪漫 |
| 接吻 | 深 | -35 浪漫 |
| 更亲密互动 | 极深 | -50 浪漫 + 关系等级下降 |
| 结婚(出轨) | 背叛 | -80 浪漫 + 自动分手可能 |

> "善妒(Jealous)"特质→嫉妒阈值降为原来的50%,更容易吃醋
> "不受约束(Non-Committal)"特质→嫉妒阈值+50%

## 四、关系衰减系统

### 4.1 基础衰减速率

| 关系类型 | 每天衰减值(不互动) | 7天不互动 | 14天不互动 | 30天不互动 |
|---------|:--------------:|:--------:|:---------:|:---------:|
| 陌生人(0-29) | -1~-2/天 | -10~-15 | -20~-25 | -40~-50* |
| 朋友(30-49) | -3~-5/天 | -25~-35 | -50~-70 | -100~-150* |
| 好友(50-69) | -5~-8/天 | -40~-55 | -80~-110 | -200~-240 |
| 挚友(70-99) | -8~-12/天 | -60~-85 | -130~-170 | -300~-360 |

| 已婚(90-100) | -3~-5/天 | -20~-35 | -40~-70 | -90~-150 |
| 家人(同住) | -0~-1/天 | -5~-8 | -10~-15 | -30~-45 |

> 注: *陌生人/朋友接近0后不再衰减(归零)。

### 4.2 特质对衰减的影响

| 特质 | 衰减速度变化 | 说明 |
|------|:----------:|------|
| 外向(Outgoing) | 衰减-25% | 社交需求更强,维持关系更容易 |
| 不善交际(Loner) | 衰减+15% | 不主动社交导致朋友值下降更快 |
| 善良(Good) | 衰减-10% | 更珍视关系 |
| 邪恶(Evil) | 衰减+20% | 不珍视他人 |
| 以家庭为重(Family-Oriented) | 衰减-50%(家人) | 仅限家人关系维护 |
| 浪漫风(Romantic) | 衰减-25%(浪漫) | 仅限浪漫关系 |
| 善妒(Jealous) | 衰减+30%(浪漫) | 嫉妒导致主动疏远 |
| 野心家(Ambitious) | 衰减+10% | 更关注事业 |

### 4.3 维持关系的最小互动频率

| 关系等级 | 维持当前值的互动频率建议 |
|---------|----------------------|
| 好友/挚友 | 每3-5天一次高品质社交互动 |
| 恋人/已婚 | 每2-3天一次浪漫互动+日常交流 |
| 同住家人 | 每天简单交流即可(同住衰减极慢) |
| 长期异地 | 每天至少一次电话/短信 |
| 敌人(负值) | 持续对立关系不会自动修复 |
