## 一、生命周期阶段

Sims 4 在经历多次更新后，现行版本（2023年3月婴幼儿更新后）共有 **8个生命阶段**（原7个，新增Infant阶段）：

### 各阶段天数表

| 生命阶段 | 中文名 | 短(天) | 普通(天) | 长(天) |
|:-------:|:-----:|:------:|:--------:|:------:|
| Newborn | 新生儿 | 0.5 | 1 | 4 |
| Infant | 婴幼儿 | 2.5 | 5 | 20 |
| Toddler | 幼儿 | 3.5 | 7 | 28 |
| Child | 儿童 | 7 | 14 | 56 |
| Teen | 青少年 | 10.5 | 21 | 84 |
| Young Adult | 青年 | 14 | 28 | 112 |
| Adult | 成年 | 21 | 42 | 168 |
| Elder | 老年 | 7 | 14 | 56 |
| **合计总天数** | | **66** | **132** | **528** |

### 阶段特征

| 阶段 | 可控制 | 上学 | 生育 | 工作 | 技能上限 |
|:----:|:------:|:---:|:---:|:---:|:-------:|
| 新生儿 | 否(仅照顾) | 否 | 否 | 否 | 无技能 |
| 婴幼儿 | 是 | 否 | 否 | 否 | 5级(5项技能) |
| 幼儿 | 是 | 否 | 否 | 否 | 5级(5项技能) |
| 儿童 | 是 | 是(小学) | 否 | 否 | 10级(儿童专属) |
| 青少年 | 是 | 是(中学) | 否 | 可兼职 | 10级 |
| 青年 | 是 | 可上大学 | 是 | 是 | 10级 |
| 成年 | 是 | 可上大学 | 是(女性) | 是 | 10级 |
| 老年 | 是 | 否 | 否 | 可退休 | 10级 |

### 生命周期设置

- 可在 **游戏选项 > 游戏 > 市民寿命** 随时更改（短/普通/长/关闭自动衰老）
- 关闭自动衰老后，市民不会因年龄增长而自然死亡
- 孕妇在怀孕期间**暂停衰老**
- 可以使用"青春药水"(1,500 满意度点数)重置当前阶段年龄至第一天

## 二、死亡系统

### 死亡类型总览

| 排名 | 死亡类型 | 触发条件 | 致死概率 | 可避免性 |
|:---:|:--------:|----------|:--------:|:--------:|
| 1 | **老年寿终** | 老年阶段天数结束 | 100%(自然) | 使用青春药水/关闭衰老 |
| 2 | **火灾** | 烹饪技能低时炉灶着火/壁炉未清理 | 中~高(视火势) | 安装烟雾报警器/灭火器 |
| 3 | **触电** | 低灵巧技能修理电子设备失败 | 低~中 | 技能等级高/使用升级零件 |
| 4 | **溺水** | 在泳池中耗尽体力无法上岸 | 高(若玩家不控制) | 修建泳池梯子/监控体力 |
| 5 | **饥饿** | 需求条长期归零 | 中(需较长时间) | 保持冰箱有食物 |
| 6 | **暴怒致死(心脏爆炸)** | 愤怒情绪达到"暴怒"等级(+7愤怒叠加) | 中 | 冷静(镜子/冷水淋浴) |
| 7 | **笑死(歇斯底里)** | 嬉闹情绪达到"歇斯底里"等级(+15嬉闹叠加) | 中 | 冷静(镜子/睡觉) |
| 8 | **尴尬致死(羞愧)** | 尴尬情绪达到"极度羞愧"等级(+6尴尬叠加) | 中 | 自我贬低/躲藏/冷静 |
| 9 | **陨石** | 使用望远镜(白天，低逻辑技能) | 极低(~1%) | 高逻辑技能/室内使用 |
| 10 | **牛头摇摇树** | 被牛头摇摇树吃掉 | 中(~50%被吃即死) | 不与之互动 |
| 11 | **猝死(老年人)** | 老年市民过度运动/热/冷 | 中 | 避免剧烈活动 |
| 12 | **火箭事故** | 火箭发射失败 | 低 | 高火箭科学技能 |
| 13 | **吸血鬼日晒** | 吸血鬼在阳光下 | 中~高(依等级) | 带伞/避免白天出门 |

### 情绪致死详细条件

#### 1. 暴怒致死 (Cardiac Explosion)
- **触发情绪等级**: 暴怒(Enraged)
- **最低叠加**: +7 愤怒值（约7个愤怒情绪叠加）
- **典型触发场景**:
  - 暴躁特质市民被反复激怒
  - 发现伴侣出轨
  - 食用愤怒火焰意大利面
  - 与气味的市民互动(狼人变身等)
- **警告信号**: 情绪图标边框闪烁(淡入淡出)
- **致死计时**: 随机(从数秒到数小时不等)
- **避免方法**: 对镜子冷静/冷水淋浴/打沙袋/向他人倾诉

#### 2. 笑死 (Death by Laughter / Hysteria)
- **触发情绪等级**: 歇斯底里(Hysterical)
- **最低叠加**: +15 嬉闹值
- **典型触发场景**:
  - 连续讲笑话+看喜剧频道
  - 吃傻笑熊软糖煎饼
  - 泡泡浴(+2嬉闹)
  - 与歇斯底里幽灵互动
- **避免方法**: 对镜子冷静/睡觉/停止嬉闹活动

#### 3. 尴尬致死 (Death by Embarrassment / Mortification)
- **触发情绪等级**: 极度羞愧(Mortified)
- **最低叠加**: +6 尴尬值
- **典型触发场景**:
  - 在公共场合尿裤子
  - 嘿咻被抓
  - 撞见他人如厕/洗澡
  - 被吸血鬼"情绪操控"(+8尴尬)
- **避免方法**: 自我贬低笑话/躲进房间/对镜子鼓励自己

### 死亡概率补充说明

| 死亡类型 | 青年 | 成年 | 老年 | 特殊免疫 |
|:-------:|:----:|:----:|:----:|:--------:|
| 老年 | - | - | 100% | 任何非老年市民 |
| 火灾 | 是 | 是 | 是 | - |
| 情绪致死 | 是 | 是 | 是 | 儿童/青少年/吸血鬼免疫 |
| 触电 | 是 | 是 | 是 | 灵巧等级10免疫 |
| 陨石 | 是 | 是 | 是 | 逻辑等级高/儿童免疫 |
| 牛头摇摇树 | 是 | 是 | 是 | 满足植物需求可避免 |

| 方式 | 来源 | 成功率 | 说明 |
|:----|:----|:------:|------|

## 三、生育系统

### 怀孕流程

| 阶段 | 持续时间 | 事件 |
|:----:|:--------:|------|
| 怀孕确认 | 互动后立即 | "尝试要宝宝"或"嘿咻"后有概率怀孕 |
| 第1阶段 | 第1天 | 获得"恶心"情绪/孕吐 |
| 第2阶段 | 第2天 | 肚子开始隆起/可以听胎教音乐 |
| 第3阶段 | 第3天 | 强烈孕吐/情绪波动/分娩 |
| 分娩 | 第3天末 | 自动前往医院或在家分娩 |

> 注: 使用"On Ley Line"地段特征或"Fertile"奖励特质可缩短怀孕时间（待补充精确数值）

### 多胞胎概率

| 条件 | 单胎 | 双胞胎 | 三胞胎 |
|:----:|:---:|:------:|:------:|
| 基础概率 | **89%** | **10%** | **1%** |
| 拥有"Fertile"奖励特质(3,000 SP) | **78%** | **20%** | **2%** |
| "On Ley Line"地段特征 | **~7.5%** | **~84%** | **~8.5%** |
| Fertile + On Ley Line | 待补充 | 待补充 | 待补充 |
| 生育按摩(Spa Day) | 叠加效果 | 待补充 | 待补充 |
| 生育药水(Outdoor Retreat) | 叠加效果 | 待补充 | 待补充 |

### 提升多胞胎概率方法

| 方法 | 来源 | 效果 |
|:----|:----|:----|
| Fertile 奖励特质 | 满意度商店(3,000 SP) | 双倍多胞胎概率 |
| On Ley Line 地段特征 | City Living 资料片 | 大幅提升双胞胎概率 |
| 生育按摩 | Spa Day 游戏包 | 24小时内提升概率 |
| 生育药水 | Outdoor Retreat | 配方: 万寿菊+蛇根草 |
| 愿望井-生育许愿 | Romantic Garden Stuff | 随机结果 |
| 听儿童音乐/看儿童电视 | 基础版 | 社区传闻(概率待确认) |

### 婴儿性别

- 基础概率: **~50% 男 / ~50% 女**
- 无官方方法可以100%选择性别的机制
- 可在创建模拟市民(CAS)中更改婴儿性别

## 四、遗传系统

### 外貌遗传

| 特征 | 遗传方式 | 突变概率 |
|:----|:--------|:--------:|
| 肤色 | 父母混合(中间色) | 低(可能在父母区间内偏移) |
| 发色 | 50%父/50%母 | 低(可能产生父母均无的随机天然色) |
| 发型 | 不遗传(随机) | 100%随机 |
| 瞳色 | 50%父/50%母 | 低(可能产生混合色或随机色) |
| 脸型/五官 | 父母基因混合 | 中(可产生父母特征的重组) |
| 体形 | 遗传父母倾向 | 中(受活动量影响) |
| 耳型/鼻型/嘴型 | 50%父/50%母 | 低~中 |

> 注: Sims 4 的遗传系统较 Sims 2 简化，没有"隐性基因"系统。孙辈不会跳过父辈直接遗传祖辈特征（除非父辈表现该特征）。

### 技能与特质遗传

| 遗传项 | 是否遗传 | 说明 |
|:------|:--------:|------|
| 性格特质 | **否** | 儿童阶段通过"特质里程碑"事件随机获得，与父母特质无关 |
| 技能等级 | **否** | 新生市民技能为0 |
| 抱负 | **否** | 独立选择 |
| 职业 | **否** | 独立选择 |
| 奖励特质 | **否** | 需独立获取 |
| 超自然类型 | **部分** | 吸血鬼父母→子女可能为吸血鬼；人鱼/狼人同理 |

### 超自然遗传概率

| 父母类型 | 子女类型概率 |
|:--------|:-----------:|
| 人类+人类 | 100%人类 |
| 吸血鬼+吸血鬼 | 高概率吸血鬼(具体概率待补充) |
| 人类+吸血鬼 | 中概率吸血鬼 |
| 外星人+人类 | 中概率外星人(男性被外星人绑架后怀孕) |
| 人鱼+人类 | 中概率人鱼 |
| 狼人+人类 | 中概率狼人 |
| 法师+人类 | 中概率法师 |

## 五、家庭关系系统

### 家庭关系情绪加成

| 关系类型 | 日常互动加成 | 特殊互动情绪 |
|:--------|:-----------:|:------------|
| 父母↔子女 | 快乐/自信增强 | "家庭成员"快乐情绪(+1~+2) |
| 祖孙 | 快乐增强 | "祖孙亲情"快乐情绪(+1~+2, Growing Together) |
| 兄弟姐妹 | 嬉闹/快乐增强 | "手足情"嬉闹情绪(+1) |
| 伴侣(已婚) | 挑逗/快乐增强 | "已婚幸福"快乐情绪(+2) |

| 订婚 | 挑逗增强 | "订婚快乐"快乐情绪(+2) |
| 远亲(叔伯/表亲) | 轻微友好增强 | 无特殊加成 |

### Sentiment 系统（情绪记忆，2020年11月更新）

| Sentiment | 触发条件 | 持续时间 | 效果 |
|:----------|:---------|:--------:|:-----|
| 深厚友情 | 多次成功友好互动 | 数天~数周 | 见面时快乐+2 |
| 信任 | 保守秘密/支持对方 | 数天 | 社交需求下降减慢 |

### 家庭关系值公式

| 关系维度 | 数值范围 | 初始值 | 说明 |

## 六、最大家庭成员

| 项目 | 限制 | 扩展方式 |
|:----|:----:|:--------|

## 七、实用秘籍

| 秘籍 | 效果 |
|:----|:------|
| `cas.fulleditmode` | 在CAS中完全编辑所有特征(需前置testingcheats) |
| `traits.equip_trait [trait]` | 添加指定特质 |
| `traits.remove_trait [trait]` | 移除指定特质 |
| `sims.add_buff buff_Pregnancy_Trimester1/2/3` | 立刻怀孕(1/2/3阶段) |
| `sims.remove_all_buffs` | 移除所有情绪效果 |
| `death.toggle true/false` | 开启/关闭死亡 |
| `aspirations.complete_current_milestone` | 完成当前抱负里程碑 |
