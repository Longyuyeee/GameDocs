## 二、情绪强度等级系统

### 2.1 强度分级与效果

| 等级 | 数值范围 | 名称 | 效果差异 |
|:---:|:--------:|------|---------|
| 0级 | 0 | 无情绪 | 默认"良好"状态，无任何加成 |
| 1-3级 | +1~+3 | 微弱 | 技能加速约+10~20%，可触发最基础的情绪互动 |
| 4-6级 | +4~+6 | 中等 | 技能加速约+30~40%，解锁中级情绪互动，Moodlets可叠加 |
| 7-8级 | +7~+8 | 强烈(进阶) | 技能加速约+50~65%，解锁高级互动(如自信演讲/灵感杰作) |
| 9+级 | +9~+15 | 极端 | 技能加速约+75~100%，可能触发**情绪致死**(生气/嬉闹/尴尬) |

### 2.2 强度叠加规则

| 叠加方式 | 上限 | 示例 |
|---------|:---:|------|
| 基础Moodlets(单个) | +1~+5 | "看了好电影"+2快乐 |
| Moodlets叠加 | 无明确上限 | 3个快乐Moodlets→+2+3+4=+9极端快乐 |
| 环境装饰光环 | +1~+3(可浮动) | 高品质画作+3受启发 |
| 情绪光环物品 | +1~+3 | 情绪光环喷泉+2快乐 |
| 地段特征 | +1~+2 | "浪漫之域"地段+1挑逗 |
| 特质加成 | +1~+3 | 乐天派特质快乐+2 |
| **总强度上限** | **约+15** | 社区实测 |

### 2.3 负面情绪压制规则

| 压制规则 | 说明 |
|---------|------|
| **极度情绪压制** | 当正面情绪达到9+级时，所有低于5级的负面Moodlets被压制(不显示) |
| 反之亦然 | 负面9+级压制正面5级以下 |
| 同级冲突 | 正负同级(均为7+)时，数量多的一方胜出 |
| 情绪平衡特质 | 部分特质会影响压制阈值 |

## 四、六维需求系统

### 4.1 需求基本数据

| 需求 | 图标 | 满值→空值时间(普通) | 衰减影响因素 | 低需求红色区域Moodlets |
|------|:---:|:------------------:|------------|---------------------|
| **膀胱(Bladder)** | 马桶 | 约12小时 | 喝水量+咖啡加速 | 尿急→不适-3→尿裤子→尴尬-5 |
| **饥饿(Hunger)** | 盘子 | 约24-36小时(不进食) | 运动消耗加速 | 饿了→不适-2→空腹→饿死危机-5 |
| **精力(Energy)** | 床 | 约16-18小时(不睡) | 运动/工作消耗加速 | 困了→眩晕-2→睡眠不足-4 |
| **娱乐(Fun)** | 游戏手柄 | 约10-12小时 | 无娱乐活动加速 | 无聊→无聊-2→极度无聊-4 |
| **社交(Social)** | 对话气泡 | 约12-15小时 | 独处特质减速 | 孤独→悲伤-2→极度孤独-4 |
| **卫生(Hygiene)** | 肥皂 | 约20-24小时 | 运动/脏活加速消耗 | 体臭→不适-2→恶臭-3 |

### 4.2 需求衰减速度差异(不同特质)

| 特质 | 受影响需求 | 衰减速度变化 |
|------|----------|:----------:|
| 懒散(Lazy) | 精力 | 衰减+50%(更快变困) |
| 热爱运动(Active) | 精力 | 衰减-25%(更慢变困) |
| 贪吃(Glutton) | 饥饿 | 衰减-50%(更慢饿) |
| 整洁(Neat) | 卫生 | 衰减+25%(更快变脏,感知更敏感) |
| 邋遢(Slob) | 卫生 | 衰减-50%(更慢变脏) |
| 外向(Outgoing) | 社交 | 衰减-50%(更慢孤独) |
| 不善交际(Loner) | 社交 | 衰减+25%(不社交更舒适) |
| 爱家(Domestic) | 卫生/饥饿 | 各衰减-15%(做完家务更满足) |
| 美食家(Foodie) | 饥饿 | 衰减-25%+高品质食物需求降低 |

### 4.3 需求对情绪的影响权重

| 需求状态 | 情绪影响 | 权重(压制力) |
|---------|---------|:----------:|
| 全绿(80-100%) | 无独立影响 | - |
| 黄区(40-80%) | 无独立影响 | - |
| 红区(<40%) | 对应负面Moodlet触发 | ★★(中等) |
| 极端红区(<15%) | 强烈负面Moodlet触发 | ★★★★(强压制) |
| 全满(100%) | 可能触发正面Moodlet | ★(弱) |
| 多个需求同时红区 | 负面叠加，优先级最高的需求占据 | ★★★★★(完全压制正面) |

> 需求优先级(负面压制权重从高到低): **饥饿 > 精力 > 膀胱 > 社交 > 娱乐 > 卫生**

## 六、情绪致死阈值

### 6.1 三种情绪致死

| 致死类型 | 触发条件 | 情绪累积值 | 致死动画 | 触发后能否自救 |
|---------|---------|:---------:|---------|:------------:|
| **生气致死(心脏骤停)** | 生气强度连续9+持续>2小时 | 生气累积≥+9 | 捂胸倒地 | 可(高级医疗或使用"情绪重置"秘籍) |
| **嬉闹致死(笑死)** | 嬉闹强度连续9+持续>2小时 | 嬉闹累积≥+9 | 大笑倒地 | 可(冷静互动或使用情绪控制物品) |
| **尴尬致死(羞愧而死)** | 尴尬强度连续9+持续>2小时 | 尴尬累积≥+9 | 消失遁走 | 可(极低概率自动解除) |

### 6.2 致死风险因素

| 风险因素 | 致死概率提升 | 说明 |
|---------|:----------:|------|
| 基础情绪爆发(9+) | 基准×1.0 | 任何9+级情绪持续超过2小时 |
| 多个同向Moodlets叠加 | ×1.3/每多一个 | 叠加越多风险翻倍 |
| 极端强度Moodlet(单个≥+7) | ×1.5 | 稀有事件触发的极值Moodlet |
| 情绪强化特质(古怪/忧郁) | ×1.2 | 情绪波动更大的特质 |
| 地段特征加成 | ×1.1 | 地段环境持续供应情绪值 |
| 使用情绪解除方法(冷静/冥想) | ×0.3 | 可主动解除致死状态 |

### 6.3 情绪致死经验值(死神互动)

| 互动类型 | 经验/收益 |
|---------|:--------:|
| 哀求死神(乞求活命) | 50%概率成功(魅力相关) |
| 与死神调情 | 30%成功+解锁死神互动成就 |
| 与死神争论 | 40%成功+独特对话 |
| 献上礼物(需有特定物品) | 60%成功 |

## 七、需求管理策略

| 需求 | 最优补充方式 | 补充效率(满分100%) | 额外收益 |
