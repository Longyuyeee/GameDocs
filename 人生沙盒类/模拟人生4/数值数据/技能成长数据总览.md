## 四、技能加速乘区

### 4.1 技能加速公式(估算)

```
技能速度 = 基础速度 × (1 + 理想情绪加成 + 情绪光环加成 + 地段特征加成 + 特质加成 + 奖励特质加成)
```

### 4.2 各加速因子倍率

| 加速因子 | 最大倍率 | 说明 |
|---------|:-------:|------|
| **理想情绪(中等+4~6)** | +35-40% | 对应技能的理想情绪等级中等 |
| **理想情绪(强烈+7~8)** | +50-60% | 强烈情绪状态下 |
| **理想情绪(极端+9+)** | +65-75% | 极端情绪下(注意致死风险) |
| **情绪光环物品** | +15-25% | 情绪光环对应技能类型 |
| **地段特征** | +15-20% | 如"天才之地"专注+1→逻辑/编程+15% |
| **书虫特质(阅读时)** | +100% | 仅限阅读技能书时 |
| **野心家特质** | +25% | 仅限工作技能 |
| **快速学习(奖励特质)** | +15% | 完成脑力大师抱负获得 |
| **通才(文艺复兴奖励)** | +10% | 全技能加速 |
| **早起鸟/夜猫子** | +25% | 对应时段(早6-9/晚10-2) |
| **图书馆/博物馆地段** | +15-25% | 地段自带加成 |

### 4.3 最大理论加速(社区估算)

| 配置 | 加速倍率 |
|------|:-------:|
| 无加速 | ×1.0 |
| 理想情绪(中等)+地段特征 | ×1.5-1.6 |
| 理想情绪(强烈)+地段+特质 | ×1.8-2.0 |
| 理想情绪(极端)+地段+特质+奖励特质+时段 | ×2.2-2.5 |
| 阅读技能书:书虫特质+专注情绪+图书馆 | ×2.5-3.0 |
