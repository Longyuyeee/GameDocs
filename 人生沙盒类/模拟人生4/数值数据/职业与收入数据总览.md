## 二、主动职业(Active Career)

### 2.1 医生(Doctor)

| 内容 | 详情 |
|------|------|
| 工作模式 | 每日去医院上班，**手动治病**（非兔子洞） |
| 日常任务 | 诊断病人(使用诊断仪)→化验→确诊→治疗/手术 |
| 每日奖励 | 完成任务=工作表现+，时薪随等级提升 |
| 特殊事件 | 流行病/急诊/难产接生 |
| 10级专属 | "主刀医生"或"医学科研员"路线 |
| 专属物品 | 诊断仪/手术台/CT机/医院病床 |

### 2.2 侦探(Police)

| 内容 | 详情 |
|------|------|
| 工作模式 | 每日去警察局上班，**手动破案** |
| 日常任务 | 出警→调查现场→询问证人→审问嫌犯→结案 |
| 破案步骤 | 收集证据→比对指纹→分析线索→锁定嫌犯 |
| 每日奖励 | 完成破案=大幅工作表现+ |
| 10级专属 | "秘密行动"或"法医调查"分支 |
| 专属物品 | 警车/手铐/证据板/审讯椅/警服 |

### 2.3 科学家(Scientist)

| 内容 | 详情 |
|------|------|
| 工作模式 | 每日去实验室上班，**手动研究** |
| 日常任务 | 做实验→分析样本→发明创造→撰写报告 |
| 特殊能力 | 可制造"克隆机"(克隆NPC)/"传送机"/"模拟光线" |
| 每日奖励 | 完成研究=工作表现+ |
| 10级专属 | "疯狂科学家"路线→解锁外星人互动 |
| 专属物品 | 分析仪/制造机/传送机/克隆机/模拟光线枪 |

## 六、财富获取效率排名(社区估算)

| 排名 | 方式 | 日均收入(§) | 前期投入 | 被动/主动 |
|:---:|------|:----------:|:-------:|:--------:|
| 1 | 10级绘画(杰作巨大画) | 5,000-20,000 | 画架§300+ | 主动 |
| 2 | 10级园艺(超级植物) | 3,000-10,000 | 种子+土地 | 半被动 |
| 3 | 10级作家(畅销书版税) | 2,000-8,000 | 电脑§500+ | 被动 |
| 4 | 10级编程(游戏版税叠加) | 1,000-5,000 | 电脑+ | 被动 |
| 5 | 10级科技职业 | 800-1,200 | 无 | 主动(上班) |
| 6 | 10级特务职业 | 800-1,100 | 无 | 主动 |
| 7 | 街头表演(10级吉他) | 500-1,500 | 吉他§200+ | 主动 |
| 8 | 普通职业(10级) | 600-1,000 | 无 | 主动(上班) |

> **结论**: 绘画和园艺是基础版最赚钱的方式，而版税(书+编程)是最强的被动收入。主动职业(上班)收入最稳定但上限低于创造性职业。

> **数据说明**: 本表涵盖模拟人生4基础版全部11种职业及主要收入来源的数值数据。时薪/版税/售价等收入数据为社区通过大量测试获得的估算值，实际游戏内受多重因素(情绪加成/技能等级/人际关系/市场波动)影响。DLC职业(如《来去上班》的3个主动职业、《星梦起航》的演员等)未包含。
