## 一、世界总览

截至2025年，Sims 4 共有 **30+** 个世界（含基础版3个 + 各资料片/游戏包新增）。以下为基础版世界详细数据：

### 基础版世界

#### 1. 柳溪 (Willow Creek)
- **风格:** 新奥尔良/南方沼泽风情
- **解锁条件:** 默认解锁
- **街区数:** 6个（Foundry Cove / Crawdad Quarter / Sage Estates / Pendula View / Courtyard Lane / Magnolia Blossom Park）
- **总地段数:** 21
- **住宅:** 16
- **社区:** 5
- **隐藏地段:** Sylvan Glade × 1
- **内置家庭:** Goth / Pancakes / Spencer-Kim-Lewis / Bheeda 等

#### 2. 绿洲之泉 (Oasis Springs)
- **风格:** 沙漠/西南部风情
- **解锁条件:** 默认解锁
- **街区数:** 6个（Skyward Palms / Mirage Canyon / Acquisition Butte / Bedrock Strait / Parched Prospect / Desert Bloom Park）
- **总地段数:** 21
- **住宅:** 16
- **社区:** 5
- **隐藏地段:** Forgotten Grotto × 1
- **内置家庭:** Landgraab / Caliente / Bjergsen 等

#### 3. 新新月 (Newcrest)
- **风格:** 空白画布(无预设建筑)
- **解锁条件:** 2015年6月免费更新
- **街区数:** 3个（Bridgeview / Ridgeback / Llama Lagoon）
- **总地段数:** 15（全为空地段，由玩家自行建造）
- **住宅:** 0（全部空白）
- **社区:** 0（全部空白）
- **隐藏地段:** 无

## 四、隐藏地段

### 1. 幽幕林 (Sylvan Glade)

| 项目 | 详情 |
|:----|:------|
| **所在世界** | 柳溪(Willow Creek) |
| **入口地点** | 柳溪 → Crick Cabana 空地后的**大树** |
| **解锁方法** | 与大树互动(按顺序): ①浇水 ②赞美树叶 ③对话 → 出现传送门 → 进入后选择: 跟随声音→顺流而下→进入迷雾→前往林地 |
| **环境** | 永恒黄昏(吸血鬼友好) |
| **独家收集品** | 树鱼(Treefish) / 特殊青蛙 / 稀有植物(樱桃树/梨树/金鱼草/百合/蓝铃花/草莓/罗勒/蘑菇) |
| **独家奖励** | 树鱼(鱼类收集必需) / 高概率稀有矿石 / 平静氛围 |

### 2. 遗忘石窟 (Forgotten Grotto)

|:----|:------|

| **解锁条件** | **灵巧技能等级10**方可撬开入口 |
