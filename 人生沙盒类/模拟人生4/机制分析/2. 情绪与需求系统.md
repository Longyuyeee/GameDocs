## 一、系统概述

情绪系统是模拟人生4**最具创新性的核心机制**，也是该系列"从数值驱动到情感驱动"的关键进化。不同于前代（Sims 3 的"心情条"——一个简单的-100到+100的数值），Sims 4 引入了 **16 种离散情绪状态 + 上百种 Moodlets（临时情绪修正符）+ 情绪光环 + 负面压制正面规则**，构成一个极其丰富的动态情绪网络。

六维需求系统则提供了情绪系统的基础输入——当需求未满足时，市民产生负面情绪；当需求满足时，市民获得正面情绪基础。**需求是情绪的基础，情绪是行为的驱动**——这一因果链是 Sims 4 整个 AI 自主行为系统的核心。

## 二、16种情绪状态详解

### 2.1 情绪总览表

| 情绪 | 类型 | 触发方式 | 影响效果 |
|:----:|:----:|---------|---------|
| **良好(Good)** | 中性基础 | 默认状态，无特殊触发源 | 无特殊加成，基础行为 |
| **快乐(Happy)** | 正面基础 | 需求满足/社交成功/环境舒适 | 所有行动速度+25%，社交互动成功率提升 |
| **专注(Focused)** | 正面技能 | 洗澡沉思/逻辑技能/喝咖啡/看星星 | 逻辑+5、灵巧+5、编程+5、园艺+3技能加速 |
| **受启发(Inspired)** | 正面技能 | 看美丽画作/淋雨/吃高品质食物/烹饪 | 绘画+5、写作+5、烹饪+5、音乐+3技能加速 |
| **精力充沛(Energized)** | 正面技能 | 喝咖啡/淋冷水澡/听动感音乐/健身 | 健身+5技能加速，运动类互动成功率提升 |
| **自信(Confident)** | 正面技能 | 照镜子/社交成功/穿特定服装/完成挑战 | 魅力+5、表演+5技能加速，社交互动成功率提升 |
| **挑逗(Flirty)** | 正面浪漫 | 浪漫互动/喷香水/喝浪漫饮料/看浪漫片 | 浪漫互动成功率大幅提升 |
| **嬉闹(Playful)** | 正面社交 | 看喜剧/恶作剧/嬉闹对话/喝起泡酒 | 喜剧+5技能加速，恶作剧类互动成功率提升 |
| **尴尬(Embarrassed)** | 负面轻度 | 被拒绝/当众出糗/尿裤子/调情失败 | 社交互动成功率降低，倾向逃离社交场景 |
| **不舒服(Uncomfortable)** | 负面轻度 | 需求变黄/饥饿/脏乱/疼痛 | 行动速度降低，倾向解决需求问题 |
| **紧张(Tense)** | 负面轻度 | 工作压力/技能瓶颈/未完成工作/见房东 | 技能学习速度降低，倾向放松活动 |
| **生气(Angry)** | 负面重度 | 被挑衅/被背叛/被刻薄对待/做讨厌的事 | 力量类互动成功率提升，但社交成功率大幅降低 |
| **悲伤(Sad)** | 负面重度 | 亲友死亡/失恋/被拒绝/伤感回忆 | 创意技能加速(悲伤时写作=优秀作品)，社交意愿降低 |
| **无聊(Bored)** | 负面轻度 | 无事可做/重复活动/环境单调 | 娱乐需求加速衰减，倾向寻找新活动 |
| **眩晕(Dazed)** | 特殊 | 被殴打/醉酒/药物/精神冲击 | 所有技能学习大幅降低，行动速度降低 |
| **沉睡(Sleeping)** | 特殊 | 睡觉中 | 无法操作，需求恢复中 |

### 2.2 情绪强度分级

| 等级 | 名称 | 效果叠加层数 | 条件 |
|:----:|:----:|:-----------:|------|
| **+1** | 快乐/专注/受启发/... | 1层正面 | 单个轻度Moodlet触发 |
| **+3** | 非常快乐/非常专注/... | 2-3层正面 | 2-3个同类型Moodlet叠加 |
| **+5** | 极度快乐/极度专注/... | 4-6层正面 | 4个以上同类型Moodlet叠加 |
| **特殊** | 暴怒/极度悲伤/... | 负面+3以上 | 高强度负面事件触发(配偶出轨/至亲死亡) |

**情绪强度叠加规则：**
1. **同类型叠加**：多个同类型的Moodlet，其强度相加。例如：装饰品+2受启发 + 吃高质量食物+3受启发 + 看美丽画作+2受启发 = 总计+7→"极度受启发"
2. **负面压制正面**：任何负面情绪Moodlet都会压制正面情绪→市民陷入该负面情绪
3. **负面叠加**：负面情绪也能叠加(如被多次刻薄对待→从"生气"升级到"暴怒")
4. **快乐强化**：当最高正面情绪≥+2时，快乐状态与之融合，形成"快乐+专注"混合状态

## 三、Moodlets 系统

Moodlets（情绪修正符）是情绪状态的具体"来源"，每个Moodlet带有情绪类型+强度值+时效。Moodlet是Sims 4中最精细的情绪控制粒度。

### 3.1 Moodlets 来源分类

| 来源类别 | 说明 | 典型Moodlet示例 |
|:--------:|------|----------------|
| **需求状态** | 需求值变化自动触发 | "饥饿!"(-5不舒服)、"忍住!"(-3紧张/膀胱) |
| **环境** | 所处环境的装饰品/家具/地段特征 | "精心装饰的房间"(+2快乐)、"肮脏环境"(-2不舒服) |
| **社交互动** | 与其他市民互动后的结果 | "尴尬的调情"(-3尴尬)、"美妙的约会"(+3快乐) |
| **事件** | 人生大事件触发 | "升职了!"(+5快乐)、"亲友去世"(-7悲伤) |
| **食物/饮品** | 摄入特定食物或饮品 | "高品质餐点"(+2受启发)、"咖啡因提神"(+2精力充沛) |
| **特质相关** | 由市民特质触发的自动Moodlet | "书呆子的乐趣"(+2快乐/阅读时) |
| **技能/工作** | 技能升级或工作表现 | "技能升级!"(+2快乐)、"工作出色"(+3自信) |
| **天气(资料片)** | 四季天气影响 | "晒太阳"(+2快乐)、"被淋湿"(-2不舒服) |

### 3.2 Moodlets 关键属性

| 属性 | 说明 |
|:----:|------|
| **强度值** | 1-10，决定情绪状态的叠加等级 |
| **时效** | 固定时长(如4小时)或触发消失(如"饥饿"在进食后消失) |
| **叠加层数** | 同一来源Moodlet不重复叠加，不同来源可叠加 |
| **优先级** | 负面Moodlet优先级>正面Moodlet |
| **隐藏Moodlet** | 自动消失(如需求从红变绿)，或完成目标后消失 |

### 3.3 情绪光环 (Emotional Aura)

| 光环类型 | 来源 | 范围 | 强度 |
|:--------:|------|:---:|:----:|
| 装饰品光环 | 特定画作/装饰/家具 | 房间范围内 | +1至+3 |
| 市民光环 | 拥有特定特质或技能等级的市民 | 附近其他市民 | +1至+2 |
| 地段特征光环 | 某些地段特征(如"浪漫之地") | 全地段 | +1 |
| 可开关光环 | 特定物品可切换光环开启/关闭 | 物品所在房间 | +1至+3 |

## 四、六维需求系统

### 4.1 需求总览

| 需求 | 图标 | 初始值 | 衰减速率(正常) | 解决方式 |
|:----:|:---:|:-----:|:-------------:|---------|
| **膀胱(Bladder)** | 马桶 | 100% | 中速 | 上厕所 |
| **饥饿(Hunger)** | 餐叉 | 100% | 中速(约8小时从满到空) | 进食(冰箱/外卖/餐厅) |
| **精力(Energy)** | 电池 | 100% | 中速(清醒时持续衰减) | 睡觉/打盹/咖啡(暂时) |
| **娱乐(Fun)** | 游戏手柄 | 100% | 慢速(取决于活动) | 游戏/电视/书籍/乐器/社交 |
| **社交(Social)** | 对话气泡 | 80% | 中速(独处时更快) | 与任何人互动/打电话/网络社交 |
| **卫生(Hygiene)** | 肥皂 | 100% | 慢速(取决于活动) | 淋浴/泡澡/洗手 |

### 4.2 需求等级对情绪的影响

| 需求值 | 状态 | 对情绪的影响 |
|:-----:|:----:|:-----------:|
| **81-100%** | 绿色(已满足) | 无负面，全绿=+1快乐Moodlet |
| **41-80%** | 黄色(一般) | 无直接情绪影响 |
| **11-40%** | 橙色(需要注意) | 较弱负面Moodlet开始出现 |
| **0-10%** | 红色(紧急) | 强烈负面Moodlet→触发对应情绪。如饥饿→"饿死了!"(-5不舒服) |
| **全绿(六维皆满)** | 巅峰状态 | +3快乐+"满足"Moodlet，持续时间短(需求快速衰减) |

### 4.3 各需求衰减速率影响因素

| 因素 | 影响 | 说明 |
|:----:|:----:|------|
| **懒惰特质** | 精力衰减+25% | 懒惰市民更容易疲劳 |
| **活跃特质** | 精力衰减-20%，饥饿衰减+15% | 活跃市民更易饿 |
| **独行侠特质** | 社交衰减-40% | 独行侠不需要频繁社交 |
| **科技达人特质** | 娱乐衰减-20%(电脑活动时) | 玩电脑娱乐消耗更慢 |
| **素食者特质** | 饥饿衰减+10%(植物性食物恢复更多) | 肉类对素食者效率低 |
| **年龄(幼儿)** | 所有需求衰减+30% | 幼儿需求更快变红 |
| **年龄(老年)** | 精力衰减+20%，卫生衰减-15% | 老年人更易疲劳但不轻易脏 |
| **怀孕** | 饥饿衰减+40%，精力衰减+30% | 孕妇更需进食和休息 |
| **疾病(资料片)** | 随机需求加速衰减 | 视疾病类型而定 |
| **地段特征** | 特定需求衰减调整 | 如"快车道"加快娱乐衰减 |

## 五、情绪×需求的交互循环

```
需求满足 ──→ +正面Moodlet ──→ 正面情绪 ──→ 技能加速/社交成功 ──→ 更好的生活品质
        ↕                                          ↕
需求未满足 ──→ +负面Moodlet ──→ 负面情绪 ──→ 技能降低/社交失败 ──→ 生活品质下降
```

**正循环示例：**
1. 市民保持需求全绿 → 获得"满足"+1快乐Moodlet
2. 市民在良好环境中 → 装饰品+2受启发
3. 市民以受启发情绪画画 → 画面质量提升(杰出/大师级)
4. 杰出画作 → 出售获得高收入 → 购买环境更好的家具
5. 更好家具 → 更强情绪光环 → 下一幅画质量更高

**负循环示例：**
1. 市民忙于技能不睡觉 → 精力黄色 → 开始不舒服
2. 不舒服 → 行动速度降低 → 烹饪质量下降
3. 低质食物 → 吃后不舒服+1 → 叠加不舒服
4. 不舒服叠加 → 社交互动失败 → 被拒绝→尴尬Moodlet
5. 尴尬+不舒服叠加 → 情绪崩溃 → 无法进行有效活动

**情绪级联效应（情感觉醒/情绪崩溃）：**

Sims 4 中不存在"情绪锁死"但存在"情绪螺旋"——如果一个市民连续遭遇同类事件，情绪强度会不断叠加。正面事件的叠加(如"升职+结婚+约会成功")会产生持续多日的"极度快乐"状态，而负面事件的叠加(如"失业+失恋+亲友离世")可能导致市民连续数日处于"极度悲伤"甚至触发"情绪崩溃"(一段时间的强制情绪状态)。

## 六、设计特点分析

### 6.1 情绪系统的核心设计优势

1. **离散情绪×技能绑定：** 每种正面情绪绑定特定技能类别的加速——这创造了"优化情绪→优化技能"的元游戏策略。玩家需要主动为特定目标营造情绪环境(如为逻辑训练布置"专注"环境)，这是一个很强的深层次规划动机。

2. **负面情绪的正面作用：** 悲伤时创意技能加速——这使得"负面情绪≠完全负面"。玩家有时故意不解决悲伤来源以便创作高水平作品，这是非常人性化的设计洞见。

3. **负面压制正面"第一性原理"：** 一个负面Moodlet压制所有正面Moodlet——这模拟了"一个坏消息毁了一整天"的真实心理。但也因此产生了一个设计脆弱点：一旦陷入负面循环很难自行恢复。

4. **Moodlet作为"生活事件的量化反馈"：** 任何有意义的交互→Moodlet反馈→情绪变化。玩家能感知到"我做的事有意义的变化"，而不是抽象的数值增减。

### 6.2 环世界心情系统对比

| 对比维度 | 模拟人生4 | 环世界(RimWorld) |
|:--------:|:---------:|:----------------:|
| **情绪模型** | 16种离散情绪×强度等级 | 单一心情条(-100到+100) |
| **心情来源** | 上百种Moodlet×6维需求×环境×社交×事件 | 约50种心情修正(环境/社交/事件/思想) |
| **情绪对行为的影响** | 每种情绪解锁/强化特定互动 | 心情条仅决定崩溃阈值和精神状态 |
| **情绪系统深度** | 极高——离散×叠加×压制×级联 | 中等——心情条×精神状态 |
| **设计哲学** | 情绪是行为**驱动者** | 心情是行为**后果** |
| **负面反馈机制** | 负面压制正面，级联可能 | 心情低→崩溃→短暂昏迷，之后恢复 |
| **正面强化** | 理想情绪+技能加速，叙事+收益双驱动 | 心情高→工作效率提升15% |

### 6.3 CK3 压力系统对比

CK3的压力是单一维度积累条(0-100%)。积累到阈值→触发压力崩溃(暴食/酗酒/遁世/发怒)。压力管理是CK3的核心循环之一。

| 对比维度 | 模拟人生4 | 十字军之王3(CK3) |
|:--------:|:---------:|:----------------:|
| **模型复杂度** | 16情绪×上百Moodlets | 单压力条×四类崩溃 |
| **对游戏的影响** | 直接影响所有行为(技能/社交/需求) | 仅影响压力崩溃判定 |
| **正面/负面平衡** | 双向：正面强化+负面压制 | 单向：只有负面积累 |
| **玩家对情绪的掌控力** | 中——可主动营造情绪环境 | 低——决策本身积累压力 |

## 七、跨游戏对比深度分析

### 7.1 环世界 "心情" vs Sims 4 "情绪"：从模拟到叙事的进化

环世界的心情系统本质上是**殖民地管理工具**——你用心情条判断殖民地是否稳定、是否需要干预。心情条就是管理面板上的一列行数据。

Sims 4 的情绪系统本质上是**叙事生成器**——每种情绪解锁特定行为(如嬉闹时自动恶作剧)，驱动市民做出符合性格的行动。情绪不是状态监测，而是**AI行为选择的权重参数**。

**关键区别：** 在环世界中，你看到"心情-20"→去解决房间太挤的问题。在Sims 4中，你看到市民"悲伤"→她走到窗前看雨→触发更多悲伤回忆→持续悲伤→写作一本悲伤主题的小说。Sims 4让情绪生成了故事，而环世界只让情绪生成了工作效率变化。

### 7.2 情绪作为"品类差异化核心竞争力"

在人生沙盒品类中，情绪系统的深度直接决定了"角色像个真实的人"的程度：
- **Sims 4(品类标杆)：** 16种情绪×离散状态×技能绑定×叙事驱动
- **环世界：** 单心情条×精神状态(极端的情绪简化——因为核心不是"人"而是"殖民地")
- **CK3：** 压力条×崩溃(情绪只作为游戏机制的约束——避免玩家无脑扩张)
- **太吾绘卷：** 几乎无情绪系统(只有心情影响好感度，情绪不是核心系统)

Sims 4 在情绪系统上的投入使其在"角色真实感"维度上大幅领先所有同类游戏，这也是它无可替代的品类价值。
