# 建造与房屋系统 — 模拟人生4

> 游戏版本: The Sims 4 (PC/Steam, Maxis/EA, 2014, 基础版2022免费)

## 一、建造模式概览

《模拟人生4》的建造系统是该品类中最深入、最自由的建筑工具之一。它将建造分为 Build（建筑结构）和 Buy（家具装饰）两种模式，配合网格放置、房间工具、环境值系统和自由放置秘籍，形成了一个从实用主义到艺术创作的完整建造生态。

## 二、Build 模式（建筑结构）

### 建筑结构元素一览

| 元素 | 功能 | 进阶操作 |
|------|------|---------|
| 墙壁(Wall) | 基础围护结构 | 半墙/全墙/斜墙/自定义高度 |
| 房间(Room) | 一键生成预设房间形状 | 拖拽生成→自动建造墙壁+地板+天花板 |
| 屋顶(Roof) | 覆盖建筑顶部 | 多段屋顶/玻璃屋顶/天窗/坡度调整 |
| 地基(Foundation) | 抬升建筑离地高度 | 高度可调/柱式地基/封闭式地基 |
| 地下室(Basement) | 地下空间挖掘 | 多层地下室/楼梯连接/采光井 |
| 泳池(Pool) | 室外/室内水景 | 异形泳池/泳池平台/瀑布特效(DLC) |
| 楼梯(Stair) | 楼层垂直连接 | L型/U型/螺旋/直梯/宽度调整 |
| 围栏(Fence) | 边界/阳台/平台围护 | 柱式/矮墙/栅栏/玻璃 |
| 柱(Pillar) | 结构支撑与装饰 | 方形/圆柱/壁柱 |
| 拱门(Arch) | 开放式房间过渡 | 各种宽度/高度/风格 |

### 特殊建造技术

| 技术 | 说明 | 操作方法 |
|------|------|---------|
| **房间工具** | 快速建造预设形状的房间 | 点击并拖拽→自动生成四面墙壁+地板+天花板 |
| **多层建造** | 地上4层+地下1层(基础)；资料片可扩展(公寓上限不同) | 直接叠放楼层/使用楼梯连接 |
| **半墙建造** | 建造半人高矮墙 | 选取半墙模式→拖拽 |
| **平台(Platform)** | 同一楼层内的抬升平台 | DLC功能/高度可自定义/多层平台叠加 |
| **屋顶改造** | 修改屋顶形状/材质/天窗 | 选取屋顶段→拖拽顶点调整坡度 |
| **地形工具** | 修改地面高度/挖洞/堆坡 | 需特定DLC支持 |

| 属性 | 说明 | 影响 |

## 四、网格放置系统

### 放置模式

| 模式 | 启用方式 | 效果 |
|:----:|---------|------|
| **标准网格** | 默认 | 物品对齐1×1网格/物品之间有最小碰撞间距 |
| **半格移动** | 按住Alt键 | 可在半格范围内微调位置(非完全自由) |
| **bb.moveobjects** | 输入秘籍 `bb.moveobjects on` | **完全自由放置**：无视网格限制+无视碰撞检测 |
| **bb.ignoregameplayunlocksentitlement** | 输入秘籍 | 解锁所有职业/技能/抱负奖励物品 |
| **bb.showhiddenobjects** | 输入秘籍 | 显示隐藏的调试物品 |
| **bb.enablefreebuild** | 输入秘籍 | 允许在任何地段进行建造(包括不可建造区域) |

### 秘籍详解：bb.moveobjects

`bb.moveobjects on` 是《模拟人生4》建造系统最核心的秘籍，它从根本上改变了放置逻辑：

```
标准模式:  物品必须对齐网格 + 物品之间不能重叠 + 物品不能"浮空"
bb.moveobjects:  无视网格 + 允许重叠 + 允许悬空(需配合其他操作)
```

该秘籍允许：
- 将一个物品完全覆盖在另一个物品之上（叠加装饰）
- 将物品放置在标准的"不可放置"位置（如半墙之上）
- 创建非对称布局（不依赖网格对齐的美学设计）
- 物品之间零间隙排列（打造"定制家具"的视觉效果）

## 五、环境值与情绪光环系统

### 环境评分(Environment Score)

每个房间的**环境评分**由室内所有物品的环境值累计计算：

| 房间环境等级 | 所需分数 | 效果 |
|:----------:|:-------:|------|
| 简陋(Shabby) | 0-1 | 负面情绪Mooldets可能触发/觉得"这个地方好乱" |
| 一般(Average) | 2-3 | 中性/无特殊情绪影响 |
| 不错(Nice) | 4-5 | 轻微正面情绪加值 |
| 很好(Great) | 6-7 | 正面情绪Mooldets(装饰好的房间=快乐+) |
| 极好(Fabulous) | 8+ | 强大的正面情绪Mooldets |

> 环境值公式: 房间环境评分 = 所有物品环境值之和 ÷ 房间内物品数量(有上限阈值)

### 情绪光环物品(Emotion Aura Items)

部分装饰品/画作/奖励物品具有**情绪光环**属性：

| 物品类型 | 情绪光环 | 触发条件 | 影响范围 |
|---------|---------|---------|---------|
| 灵感画作 | 受启发(Inspired) | 灯光开启+在房间内 | 整房间 |
| 专注装饰 | 专注(Focused) | 灯光开启+在房间内 | 整房间 |
| 自信奖励 | 自信(Confident) | 特定互动触发 | 数个格子 |
| 挑逗光环 | 挑逗(Flirty) | 灯光开启+特定装饰 | 整房间 |
| 快乐物品 | 快乐(Happy / Very Happy) | 灯光开启+房间内有市民 | 整房间 |
| 悲伤装饰 | 悲伤(Sad) | 灯光开启+在房间内 | 整房间 |
| 精力充沛 | 精力充沛(Energized) | 特定健身装饰 | 整房间 |
| 特定情绪画作 | 由画作主题决定 | 绘画完成+高品质 | 整房间 |

> 情绪光环物品是建造系统与情绪系统的**直接连接点**——玩家可以通过精心布置房间来主动调节市民的情绪状态，从而影响技能学习效率（受启发→绘画/写作加速，专注→编程/逻辑加速）。

## 九、设计特点分析

**建造不是背景，而是"情绪引擎"**：《模拟人生4》的建造系统与大多数游戏的核心差异在于——你建的房子不只是"好看的场景"，它是市民情绪的直接调节器。一面墙的颜色、一幅画的主题、一盏灯的位置，都会通过环境值和情绪光环系统影响市民的心理状态。这意味着"装修"不是一个装饰性副玩法，而是与角色成长（技能学习速度）、关系发展（浪漫成功率）、甚至生存（快乐Moodlet压制负面情绪）深度绑定的核心机制。从设计哲学上，Sims 4将"家"重新定义为"情绪驱动的生存空间"——这是其他任何品类都未尝试过的设计方向。

**bb.moveobjects = 从"编辑器"到"画布"的跃迁**：这个看似简单的秘籍实际上将建造模式从"游戏内工具"升级为"数字创作工具"。当碰撞检测和网格限制被移除后，玩家可以像使用3D建模软件一样进行自由创作——物品叠加、零间隙排列、非对齐设计。这是《模拟人生4》建造玩法能够持续10年保持活力的关键原因：它给了玩家超越"游戏玩法"的创作自由度。

**房间工具的"杠杆效应"**：房间工具降低建造门槛的同时保持了专业建造的可能性。新手只需拖拽出房间形状就能得到一个可居住空间，而高手可以通过半墙、平台、屋顶改造等进阶技术建造出惊人的建筑。这种渐进式的难度曲线——简单到拖拽生成，复杂到可以建造真实世界的建筑——是Sims 4建造系统成为品类标杆的核心设计策略。

---

> **跨游戏对比**:
> - **与《环世界》(RimWorld)对比**: 环世界的建造是完全的功能驱动——每个墙壁和家具的唯一价值是生存效用（温度调节/防御/生产加速）。Sims 4的建造则加入了"美学价值"和"情绪价值"两个新维度。环世界的基地设计是为了"更高效地生存"，Sims 4的房屋设计是为了"让市民更快乐地生活"。环世界=实用主义最优解，Sims 4=美学+情绪最优解。
> - **与《集合啦！动物森友会》对比**: 动物森友会也有房屋装饰和风水系统(红/黄/绿家具摆放会影响幸运值/HHA评分)。动物森友会的建造受限于"平坦的岛屿"和"户外放置上限"，Sims 4的建造则提供多层地板、地下室、屋顶改造、半墙、平台等更丰富的垂直建造工具。两者都强调装饰的可分享性——动物森友会的梦境门牌号和Sims 4的Gallery都是社区展示平台，但Sims 4的Gallery支持一键完整房屋下载而动物森友会不能直接复制他人的岛屿布局。
