## 一、系统概述

Create-A-Sim (CAS) 是模拟人生系列标志性的角色创建工具，也是玩家与游戏建立情感连接的第一道桥梁。CAS 提供极细致的外貌定制、8 类服装分类、50+ 性格特质和 10+ 抱负类别——玩家可以在几分钟内创造出一个在外貌和性格上都独一无二的虚拟化身。CAS 的核心设计哲学是 **"表达自我，讲述故事"**——让每个市民拥有鲜明的个体特征，从而驱动后续的叙事涌现。

**核心维度：** 外貌定制 × 服装体系 × 特质系统 × 抱负系统 × 生命周期 × 遗传系统

## 二、外貌定制系统

### 2.1 面部与体型定制

| 部位 | 可调节参数 | 说明 |
|:----:|-----------|------|
| **面部整体** | 脸型(预设10+)、肌肉/脂肪滑块、面部比例 | 滑块式调节(1-100)，非数值显示 |
| **眼部** | 眼型(20+预设)、眼距/眼深/眼倾斜/眼大小 | 可独立调节左右眼不对称 |
| **鼻部** | 鼻型(15+预设)、鼻高/鼻宽/鼻尖/鼻梁 | 高精度的局部塑形 |
| **嘴部** | 嘴型(15+预设)、嘴宽/上唇/下唇/微笑程度 | 影响说话和表情时的嘴部动画 |
| **下巴/下颌** | 下巴型/下巴突出/下颌宽度/下颌角 | 决定面部轮廓的关键参数 |
| **耳朵** | 耳型(10+预设)、耳大小/耳突出/耳倾斜 | 细节定制 |
| **肤色** | 肤色(100+色板, 含奇幻色: 蓝/绿/紫/灰/红) | 基础版即可使用全部肤色 |
| **发型** | 发型(50+)、发色(色板+挑染渐变可选) | 部分资料片额外赠送发型 |
| **眉毛** | 眉型(30+预设)、眉色/眉浓淡 | 匹配发色或独立设置 |
| **胡须/面部毛发** | 胡型(20+，仅限男性或中性体型) | 覆盖范围/浓淡可调 |
| **化妆** | 眼影/眼线/腮红/唇彩(各50+选项) | 分日常/派对/炎热等服装类别独立设置 |
| **疤痕/纹身/细节** | 疤痕(10+)、雀斑/皱纹(各10+)、纹身(50+，覆盖全身) | 纹身可按身体区域分别选择 |
| **体型** | 肌肉/脂肪滑块(×上身/下身独立) | 0=极瘦, 100=极壮/极胖 |
| **身高** | 基础版固定高度（资料片"高中时代"增加青少年身高差异） | 待补充 |

> 外观保存/预设功能: CAS 支持保存完整外观到 Gallery，也可保存单个部位的预设(如特定鼻子/嘴型组合)以供后续使用。

### 2.2 服装系统

| 服装类别 | 用途 | 套装/上下分件 |
|:--------:|------|:------------:|
| **日常** | 市民日常活动穿着(默认) | 支持上下分件 |
| **正装** | 正式场合/婚礼/葬礼/颁奖 | 支持上下分件 |
| **运动** | 健身/慢跑/运动相关活动 | 上下分件(上衣+裤子+鞋) |
| **睡衣** | 睡觉前后的穿着 | 一件式或分件 |
| **派对** | 社交聚会/夜店/派对场合 | 支持上下分件 |
| **泳装** | 游泳/海滩活动 | 一件式或分件 |
| **炎热天气** | 夏季/炎热地段自动切换 | 支持上下分件 |
| **寒冷天气** | 冬季/寒冷地段自动切换 | 含外套层 |

- **服装分层系统:** 部分服装支持开启/关闭外套、夹克层，模拟"脱外套"效果
- **风格标签:** 每件服装有"风格标签"(如"时髦""摇滚""波西米亚""运动风")，影响NPC穿着的匹配逻辑
- **颜色/花纹自定义:** 每件服装可独立选色+花纹(含纯色/条纹/格子/图案等)
- **配件:** 帽子/围巾/手套/袜子/眼镜/耳环/项链/手镯/戒指——各服装类别可独立配置
- **鞋子:** 高跟鞋/平底鞋/靴子/运动鞋——按上/下身搭配自动适配

### 2.3 声音与走路风格

| 特征 | 选项 | 说明 |
|:----:|------|------|
| **语音音高** | 高/中/低三档 | 影响市民说话时的音调 |
| **语音类型** | Simlish(模拟语) — 全部市民 | 所有市民说同一种虚构语言，感染力极强 |
| **走路风格** | 8种: 正常/自信/时髦/慵懒/僵硬/活泼/趾高气扬/鬼鬼祟祟 | 走路风格影响第一印象和社交互动 |
| **走路风格对情绪的影响** | 自信走路→自信情绪增强 | 部分走路风格与情绪系统联动 |

## 七、设计特点分析

### 7.1 CAS 的核心设计优势

1. **"低门槛高天花板"的外貌定制：** 预设滑块+数值调节的双层设计——新手用预设快速创建，高手用微调打造高度个性化的面容。100+肤色包含奇幻色，直接允许非人类的角色创建。
2. **特质的"行为驱动"而非"数值驱动"：** 特质不提供直接属性加成(如"+5智力")，而是改变AI自主行为的权重(如"懒惰"特质让市民更大概率选择坐下而非运动)。这使特质系统成为**叙事引擎**而非养成计算器。
3. **抱负系统的叙事引导：** 抱负提供长期目标框架但完全可选→玩家可以忽略抱负只过日常，也可以围绕抱负展开剧情。这种"存在但不强制"的设计尊重了每个人的游戏风格。
4. **服装类别×自动切换：** 8类服装+天气自动切换=角色在特定场合自动"变身"——强化了角色在不同生活场景下的身份感，而不需要玩家手动换装。

### 7.2 与同类游戏对比

| 对比维度 | 模拟人生4 | 环世界(RimWorld) | 十字军之王3(CK3) | 太吾绘卷 |
|:--------:|:---------:|:----------------:|:----------------:|:--------:|
| **角色创建深度** | 极高(面部数十参数+服装8类+3特质+1抱负) | 低(随机生成+背景故事选择) | 中(选择出身/文化/宗教/特质) | 低(随机或预设+姓名/性别) |
| **特质设计理念** | 行为驱动——改变AI权重 | 属性驱动——正/负面数值加成 | 属性+行为混合——有性格权重+数值修正 | 属性驱动——影响功法/战斗属性 |
| **角色定制自由度** | 极高——CAS创造任何外貌的人 | 低——只能选择背景故事 | 中——可自定义纹章/服饰/文化 | 低——预设立绘+简单参数 |
| **生命周期** | 7阶段，约95天(默认) | 无固定——直到死亡 | 出生→成年→死亡(≈70年) | "传剑"——可多代传承 |
| **传承机制** | 外貌遗传+特质概率遗传 | 无(殖民地不是家族) | 宗族传承——血统+头衔+宣称 | 传剑——直接继承能力 |

### 7.3 设计亮点与局限

**亮点：**
- 3特质系统×50+特质=约20,000+种特质组合，每个市民理论上独一无二
- CAS到游戏的无缝过渡——创建完直接进入游戏，无需漫长的教程
- 服装类别系统=最简单的角色身份切换，无需玩家主动存档/读档

**局限：**
- 基础版外貌参数不如资料片(如"生儿育女"增加遗传细节)
- 身高在基础版不可调(直到2022年"高中时代"资料片才加入青少年身高差异)
- 特质的"行为权重"对部分玩家来说不够直观——不如"+X技能"那样有明确数字反馈
- 部分特质(如"刻薄""偏执")叙事性强但实用性差，很少被"强度党"选择

## 八、跨游戏对比深度分析

### 8.1 环世界 vs 模拟人生4 —— "角色个性"的两种实现

环世界没有CAS，角色是随机生成的——但从"背景故事系统"中产生了独特的角色个性。每个殖民者有童年/成年两个背景故事阶段(如"童年：城市少年→成年：厨师")，组合出约几千种背景故事→影响可用技能和属性。

区别在于：
- **环世界**：个性是**生成的**——你接受随机结果，从随机中创作故事
- **模拟人生4**：个性是**设计的**——你主动选择每个特质，从选择中驱动故事

### 8.2 CK3 vs 模拟人生4 —— "角色驱动"的两种方向

CK3的角色创建同样涉及特质选择(3个特质：教育+性格+生活方式)，但设计逻辑完全不同：

- **CK3的特质=政治工具**：情欲特质提升生育率，暴怒特质提升战斗能力，狡猾特质提升计谋成功率
- **Sims 4的特质=生活风格**：浪漫特质让市民更浪漫，整洁特质让市民爱打扫

CK3是"特质的数值属性×政治场景"，Sims 4是"特质的AI行为×日常生活"——同一个设计理念(特质系统)，服务于两种完全不同的游戏类型。

### 8.3 太吾绘卷 vs 模拟人生4 —— "传承"的哲学差异

太吾绘卷的"传剑"系统是Sims 4遗传系统的极端反面：
- **太吾绘卷**：继承的是能力/功法/装备——"你变强了"(但你的外貌变了，因为这是新的人)
- **Sims 4**：继承的是外貌/遗传特质——"你的孩子像你"(但孩子是全新的个体，不是你)

太吾用"灵魂传承"打破血缘限制(传剑给任何人)，Sims 4用"外貌遗传"强化血缘连接(子女像父母)——两种传承哲学反映了中西方游戏文化对"血脉"和"灵魂"的不同重视程度。
