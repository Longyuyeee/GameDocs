## 一、系统概述

模拟人生4的职业系统是市民"社会化生存"的核心支柱——它提供三个功能：**收入来源**（维持生活）、**叙事驱动**（职业事件/同事关系/晋升剧情）和**成就感反馈**（等级提升/奖励）。职业系统由10个等级组成，玩家通过满足升迁条件（技能达标+工作表现+社交关系）逐步晋升，部分职业在Lv5-6出现分支选择。

收入系统则更为多元化——工资只是基础，版税、绘画出售、园艺、自由职业和房地产构成了多层次的收入版图。这一多元收入系统允许玩家走不同的人生路线：**打工族、艺术家、企业家、地主**，每种路线都有不同的游戏体验。

## 三、升迁机制

### 3.1 升迁条件

| 条件 | 说明 | 权重 |
|:----:|------|:----:|
| **技能达标** | 当前等级要求的技能达到某等级 | 必要条件(硬性要求) |
| **工作表现** | 表现条从0%积累到100% | 必要条件(硬性要求) |
| **情绪状态** | 特定情绪下工作表现增长更快 | 加速条件(非必需) |
| **日常任务** | 每日可选的"今日工作重点" | 加速条件(非必需) |
| **社交关系** | 拥有某等级的同事/老板好感 | 部分等级需要(非全部) |

### 3.2 工作表现积累

| 影响因素 | 效果量 | 说明 |
|:--------:|:------:|------|
| **正面情绪(快乐)** | 基准速度 | 正常表现积累速度 |
| **理想情绪(如专注/自信)** | +50% | 与职业性质相关的情绪有额外加成 |
| **负面情绪** | -25%至-50% | 生气/悲伤/不舒服→表现累计减慢 |
| **完成日常任务** | +30%(当日) | 每日选择一个"今日重点"(如"今天多编程") |
| **技能达标** | 浮动 | 远超要求=加速，刚达标=正常 |
| **迟到/早退** | -50%(当日) | 未按时上班/提前离开 |
| **旷工** | -100%(当日) | 完全不去上班→表现无增长或下降 |
| **与同事社交** | +10%(每次成功互动) | 上班期间的社交互动有正面影响 |

### 3.3 职业分支机制

| 阶段 | 说明 | 示例 |
|:----:|------|------|
| **Lv1-Lv5** | 单一晋升路径 | 画家Lv1"落魄画家"→Lv5"有经验的画家" |
| **Lv5-Lv6(分叉点)** | 选择分支(不可逆) | 写实大师 vs 抽象大师 |
| **Lv6-Lv10** | 分支路径独立升级 | 各分支有不同的头衔、奖励和技能要求 |

**分支选择的影响：**
- 不同的技能要求（写实大师需要绘画Lv8，抽象大师需要绘画+逻辑Lv7）
- 不同的日常任务和奖励
- 不同的工资水平（通常差异5-10%）
- 不同的工作时间和模式
- 不同的升迁奖励物品（如写实大师获得"画架"，抽象大师获得"抽象雕塑"）

### 4.3 自由职业系统

| 自由职业 | 工作方式 | 收入构成 | 特点 |
|:--------:|---------|:--------:|------|
| **自由作家** | 接写作任务→提交稿子 | 任务基础稿费+品质加成 | 无需上班，在家完成 |
| **自由程序员** | 接编程任务→开发软件 | 任务基础开发费+品质加成 | 需要电脑，升级更快 |
| **自由艺术家** | 接绘画任务→完成作品 | 任务基础佣金+品质加成 | 不同风格作品收入不同 |

**自由职业 vs 常规职业的权衡：**

| 维度 | 自由职业 | 常规职业 |
|:----:|:--------:|:--------:|
| **时间自由度** | 完全自由——何时工作何时休息 | 固定上班时间 |
| **收入稳定性** | 不稳定——有任务才有收入 | 稳定——每周固定工资 |
| **收入上限** | 中——任务有上限(约3000-5000/天) | 高——Lv10可达10000/周 |
| **社交需求** | 低——可以完全不社交 | 中——同事关系影响升迁 |
| **前期门槛** | 高——需要Lv5以上关键技能 | 低——Lv1即可开始 |
| **适合玩法** | "独行侠"式、在家式玩法 | 传统"上班族"式玩法 |

## 六、设计特点分析

### 6.1 核心设计优势

1. **职业=叙事框架而非唯一目标**：Sims 4 的职业系统不像太吾绘卷那样"变强是核心目的"，也不像环世界那样"工作是殖民地生存的基础"。Sims 4 的职业是**生活叙事的一部分**——你可以完全忽略职业，靠绘画/园艺/遗产生活，游戏体验同样完整。

2. **职业分支提供重玩性**：Lv5-6的分支选择是轻度但有效的决策——选择写实大师还是抽象大师，职业选手还是健美教练——每个分支解锁不同的奖励物品和日常任务，鼓励玩家尝试多个角色。

3. **多元收入路径的"人生自由"**：工资/版税/绘画/园艺/自由职业——多个收入路径对应不同的人生理想。这不只是"更多赚钱方式"，而是给了玩家"你想过什么样的生活"的选择权。

4. **主动职业的玩法突破**：Get to Work资料片的3个主动职业(医生/侦探/科学家)展示了Sims 4的"职业模式切换"潜力——从日常模拟切换到迷你游戏模式。虽然只在资料片中实现，但这证明了Sims 4的框架可以承载多种玩法。

### 6.2 太吾绘卷经济系统对比

| 对比维度 | 模拟人生4 | 太吾绘卷 |
|:--------:|:---------:|:--------:|
| **经济模式** | 现代市场经济(工资/版税/投资) | 封建农业经济(战斗/贸易/制造) |
| **收入主渠道** | 工资+技能产出+被动收入 | 战斗掉落+制造出售+资源采集 |
| **支出方向** | 生活消费(食物/账单/家具) | 修炼突破+装备强化+建筑升级 |
| **收入可预测性** | 高——工资稳定+被动收入可规划 | 低——战斗收益随机+市场波动 |
| **经济自由度** | 极高——可选择过任何生活 | 中——需要资源支撑修炼循环 |
| **贫富差距** | 大——初期~500到后期~50000+ | 中——收益与实力正相关 |

### 6.3 环世界经济系统对比

| 对比维度 | 模拟人生4 | 环世界(RimWorld) |
|:--------:|:---------:|:----------------:|
| **经济基础** | 个人收入——市民自力更生 | 殖民地集体产出——所有人贡献 |
| **货币** | 模拟币——到处可用 | 银币——贸易货币 |
| **交易** | 买卖(购买/出售/账单) | 贸易商队/轨道贸易/定居点交易 |
| **支出大头** | 房屋/食物/账单/家具升级 | 武器/医药/机械体/建筑材料 |
| **自给自足度** | 中(可自种食物但很多需购买) | 极高(可建立自循环殖民地) |
| **压迫感** | 低——账单不高，不会饿死 | 中-高——冬季/袭击可能导致崩溃 |

## 七、跨游戏对比深度分析

### 7.1 "个人经济"vs"集体经济"的设计差异

Sims 4 的个人经济系统和环世界的集体经济系统反映了完全不同的游戏目标：

- **Sims 4**：你的钱是你自己的。你想怎么花就怎么花——你可以整天躺着吃外卖，也可以省吃俭用买豪宅。**经济系统支持"人生选择"的多样性。**
- **环世界**：钱是大家的。你管理殖民地的共同财产——没有"个人收入"只有"集体产出一交易"循环。**经济系统支持"殖民地管理"的复杂度。**

这个差异直接决定了玩家对经济的感知：Sims 4的经济压力远小于环世界，因为Sims 4中"没钱"只是"不能买新家具"，而环世界中"没钱"意味着"冬天前买不到足够的药"。

### 7.2 CK3 vs 模拟人生4——"赚钱"vs"统治"

CK3的经济系统是封建经济——收入来自领地、封臣和掠夺。玩家不需要"工作"，而是管理领地的开发度、建设建筑、处理封臣关系。

Sims 4的经济比CK3简单得多，但Sims 4的优势在于**"可感知的消费升级"**——你用赚来的钱买新房子、新家具、新衣服，每一步都有直观的反馈。CK3的金币更多是抽象的战略资源——"存够钱造建筑"或"招募雇佣兵"——没有Sims 4那种"新沙发的颜色和客厅更配"的生活感。

这就是Sims 4的品类优势：**微观经济学——每一分钱都花在生活质量的改善上。**
