## 四、各技艺的造诣上限与用途

| 技艺 | 可制作最高物品 | 最佳配套技艺 |

| 锻造 | 神一品武器 | 治金(材料精炼) |
| 炼药 | 神一品丹药 | 医术(鉴别药材) |
| 织锦 | 神一品护甲 | — |
| 烹饪 | 神一品食物 | — |
| 诗书 | (元技艺——阅读加速) | **所有其他技艺** |

## 五、技艺学习的优先级路线

```
第一优先: 诗书(加速所有阅读) → 第二优先: 锻造/炼药(产出装备/丹药)
→ 第三优先: 门派关联技艺(道法/佛学/医术/毒术)
→ 第四优先: 其他文化技艺(音律/品鉴/弈棋/绘画)
→ 第五优先: 剩余生产技艺(织锦/制木/烹饪/治金)
```
