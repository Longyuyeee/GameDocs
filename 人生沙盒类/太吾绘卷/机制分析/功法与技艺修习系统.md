## 一、功法获取

### 获取途径

| 途径 | 条件 | 说明 |
|------|------|------|
| **门派请教** | 好感≥热忱+对方会该功法+门派支持度达标 | 最正统的获取方式 |
| 秘籍购买/交换 | 好感≥喜爱 | NPC可能愿意出售或交换秘籍 |
| 奇遇掉落 | 探索地格事件 | 随机获取 |
| 古冢遗刻 | 特殊奇遇 | 可获得高品级功法 |
| 传剑继承 | 死亡传剑 | 所有功法自动传给下一代太吾 |

### 门派支持度与功法学习

| 功法品级 | 所需支持度 | 备注 |
|:--:|:--:|------|
| 九品(下·九阶) | 30% | 入门门槛 |
| 八品 | 40% | — |
| ... | +10%/品级 | — |
| 一品(神·一品) | **100%** | 该门派完全认可的标识 |

## 二、正练 vs 逆练——武学的二元路线

### 双路线的本质

| 路线 | 阅读方式 | 效果 | 适用 |
|------|------|------|------|
| **正练** | 读**真传**秘籍 | 提高功法**发挥效率上限** | 资质够高→完全发挥功法威力 |
| **逆练** | 读**手抄**秘籍 | 降低功法**使用需求** | 资质不够→降低门槛来使用高品级功法 |

### 心法等级≥5时的分化

当功法修炼到心法等级5以上时，正练和逆练会产生**不同的特殊效果**——同样的招式，正练可能是"伤害+穿透"，逆练可能是"速度+连击"。

## 三、技艺系统

### 十六种技艺

| 类别 | 技艺 | 主要产出/用途 |
|------|------|------|
| 武学 | 内功/身法/绝技等 | 战斗 |
| 生产 | 锻造/织锦/制木/炼药/烹饪/治金 | 制作装备/药品/食物 |
| 文化 | 诗书/绘画/音律/品鉴/弈棋 | 与NPC交流/书籍阅读辅助 |
| 杂学 | 道法/佛学/医术/毒术/杂学 | 门派特殊修习/治疗/下毒 |
| 特色 | **促织**(斗蟋蟀) | 收集+对战+与NPC互动 |

### 技艺造诣获取

| 方式 | 说明 |
|------|------|
| **读书** | 阅读技艺书籍→每章增加对应造诣(九品每章+1，一品每章+9) |
| 请教 | 向NPC请教→直接提升造诣(好感≥喜爱可请教3次) |
| 实践 | 制作物品/使用技艺→少量经验 |

## 四、书籍研读系统

### 读书机制

| 元素 | 说明 |
|------|------|
| **耐心** | 读书消耗耐心——耐心耗尽→阅读中断 |
| **残页** | 部分书页损毁——需要使用"融融惟"技巧攻克 |
| **参考书** | 使用同类型书籍作为参考→提高阅读效率 |
| **"温"+"独"** | 组合技巧→刷取历练(用于突破) |

### 读书策略

```
优先读低品级书→积累造诣→造诣高了→高品级书阅读难度降低
→ 用"迁"保持耐心 → 用"融融惟"攻克残页
```

## 五、设计特点分析

**正练/逆练="正道"与"捷径"的哲学选择**：双路线不仅是机制选择，也是武侠哲学的选择。正练="脚踏实地/厚积薄发"，需要高资质才能完全发挥——这是"正道"。逆练=降低门槛/走捷径——这是"邪道"。这种二元路线将武侠世界观中的"正邪之辨"编码为了游戏机制——不需要文字告诉你"正练是好的"，系统本身就说明了两种路线的代价和回报。

**读书系统=将"阅读"游戏化**：太吾的读书不是"点击→读完"——它是一个有策略的小游戏。耐心管理、残页攻克、参考书选择——这些将"读书"从一个枯燥的被动行为变成了一个需要策略的主动行为。这种"将学习过程本身游戏化"的设计在RPG中极为罕见。

**十六种技艺=非战斗的完整人生**：你不必是武林高手——你可以是一个擅长锻造的铁匠、一个精通医术的大夫、一个养促织的闲人。这16种技艺为"非战斗人生"提供了完整的支撑——你可以在太吾世界中过一种完全不打架的"生活"。

---

> 跨品类对比: 太吾的正练/逆练二元路线与Elona的"善恶值"有微妙的相似——都是"两条路径都能成功，但代价和回报不同"的设计。读书系统的深度在RPG中与《王国风云3》的"生活方式"系统类似——都是将"角色成长"变成一个策略游戏。
