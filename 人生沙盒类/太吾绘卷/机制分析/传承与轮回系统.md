# 传承与轮回系统 — 太吾绘卷

> 游戏版本: 太吾绘卷 正式版 v1.0+ (Steam, 2022)
> 数据来源: [灰机Wiki](https://taiwu.huijiwiki.com/), [贴吧](https://tieba.baidu.com/f?kw=太吾绘卷)

## 一、传剑——太吾核心机制

### 传剑的触发与流程

| 条件 | 说明 |
|------|------|
| **死亡触发** | 太吾角色死亡→自动进入传剑界面 |
| **主动传剑** | 可在太吾祠堂主动选择继承人 |
| **必要条件** | **必须已建造太吾祠堂**——否则死亡=游戏失败 |

### 传剑继承内容

| 继承 | 不继承 | 说明 |
|------|------|------|
| **所有功法**(含内力+心法等级) | **资质**(智力/悟性等) | 功力全部传递 |
| **所有技艺造诣**(16种) | **个人特性** | 技能知识全部传递 |
| **太吾村所有建筑** | — | 基业永存 |
| **仓库所有道具+装备** | — | 物质遗产 |
| — | 初始属性(出生时掷骰) | 新太吾是独立的新角色 |

### 传剑遗惠

通过消耗**生涯遗惠点数**，可在传剑时获得额外加成：
- 临危顿悟：继承时强制读完一本未读完的功法秘籍
- 精纯传承：新太吾获得额外精纯等级
- 其他遗惠：可根据玩家偏好定制

## 二、轮回——世界中的重生

### 轮回机制（独立于传剑）

| 维度 | 说明 |
|------|------|
| **触发** | 任何角色死亡后→灵魂进入轮回池→在世界某地转世为婴儿 |
| **继承内容** | 前世**一部分**主属性和资质 |
| **寻找方式** | 需要玩家在世界中主动寻找转世者(查看NPC履历中的"前世"条目) |
| **NPC也轮回** | 所有NPC死后都进入轮回——前代掌门/高手的转世属性极好 |

### 轮回→传剑的组合策略

```
第一代太吾培养高属性角色 → 死亡后进入轮回 → 找到其转世(婴儿) →
收为弟子/同道 → 培养成人 → 传剑给这个转世者 →
此时该角色同时拥有: 前世的部分属性 + 继承的全部功法/技艺 + 新太吾的潜能
```

## 三、子女与遗传

| 遗传项目 | 机制 |
|------|------|
| **特性遗传** | 子女随机获得父母特性——甚至可能获得比父母等级更高的特性 |
| **资质遗传** | 父母的资质影响子女基础资质 |
| **天赋遗传** | 早熟/晚熟/均衡特性部分可遗传 |
| **立场遗传** | 可能受父母影响 |

### 子女培养的价值

子女是可被培养的**最优继承人**——因为他们：
1. 从幼年开始可塑性最强
2. 继承了父母的优质基因
3. 与太吾关系天然亲近(好感度高)

## 四、设计特点分析

**传剑=死亡不是终点，而是策略计画的一部分**：在其他RPG中，死亡=失败=读档。在太吾中，死亡是**计划中的战略节点**——你可能主动选择一个属性/资质更好的年轻NPC来完成未竟之业。这让"死亡"从令人沮丧的惩罚变成了一个需要规划的**传承策略**。

**功法继承="功力不会消失"的武侠浪漫**：传剑时所有功法/技艺/内力全部传给下一代——这模拟了武侠小说中"将毕生功力传给弟子"的经典桥段。这种"功力累进"机制让每一次太吾世代的更替都让角色变得更强——一代比一代强，这是真实的武侠叙事弧线。

**轮回系统=世界的"永恒记忆"**：前一代掌门死亡后不只"消失"——他转世为一个婴儿，保留了一部分属性和资质。如果你能找到他并培养他，等于是"复活了前一代的精英"。这让世界的NPC有了除"死亡=永久消失"外的另一条存在路径。

---

> 跨品类对比: 与十字军之王3(CK3)的"继承"不同——CK3继承的是头衔和领土(政治权力)，太吾继承的是功法和技艺(个人能力)。与RimWorld的殖民者死亡不同——RimWorld死亡=永久失去技能，太吾死亡=功力的"转移"而非"消失"。
