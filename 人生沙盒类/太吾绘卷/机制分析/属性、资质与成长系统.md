# 属性、资质与成长系统 — 太吾绘卷

> 游戏版本: 太吾绘卷 正式版 v1.0+

---

## 一、主属性

| 属性 | 影响 |
|------|------|
| 膂力 | 外伤伤害/负重 |
| 体质 | HP/耐力/防御 |
| 灵敏 | 命中/闪避/移动 |
| 根骨 | 内功修炼效率/内力上限 |
| 悟性 | 读书效率/技艺修炼速度 |
| 定力 | 内息稳定/内伤抵抗 |

## 二、资质——影响功法修习效率

| 资质 | 对应功法类型 | 数值决定 |

| 刀法资质 | 刀法 | 同上 |

| 身法资质 | 身法 | 移动/闪避 |

## 三、技艺造诣

16种技艺各有独立造诣值——通过读书和实践累积。每品级书籍每章增加对应品级的造诣(九品+1/八品+2...一品+9)。

## 四、内力与精纯

- **内力**：修炼内功→内力上限提升→可施展更强功法
- **精纯**：太吾独有——通过主线推进(清除剑冢)获得——决定角色的战斗力天花板
- **混元**：五种内力全修到顶点→融合为混元→内力最高形态

## 五、设计特点分析

**资质≠属性**：太吾将"资质"（学习潜力）和"属性"（当前战斗力）分离——你的角色可能属性很高（能打）但资质平庸（学不会高品级功法）。这种分离让传剑时"功力传递但资质不传递"的规则有了意义——你继承了一身功力，但能不能学会新的神一品功法取决于你自己的资质。

**精纯=不可刷的主线进度门**：精纯只能通过清除剑冢获得——不能通过重复性劳动来刷取。这种设计确保太吾的战力天花板是**主线进度的直接函数**——你不可能通过"刷"来超越剧情设定的强度上限。

**属性上限的"三重叠加"**：太吾的最终属性可以来自三个来源的叠加：(1)出生骰子(初始属性)+(2)丹药/练习(游戏内成长)+(3)轮回属性(前世继承)。这种三重叠加让"最优角色"的培养成为一个需要规划多世代的元游戏。

**悟性=元属性的设计智慧**：悟性不是"法术伤害"——它决定你读书的速度，而读书决定你所有技艺的造诣，造诣决定你的成长效率。一个高悟性的太吾可以比低悟性太吾快3-5倍地提升所有技艺——这就是"聪明人学什么都快"的游戏化表达。

**16种技艺造诣=完整的非战斗成长体系**：技艺不仅是辅助战斗的工具——它们是独立的成长维度。你的太吾可以完全不修炼任何战斗功法，而是成为一个精通锻造/医术/诗书的"文化太吾"。这种"战斗/非战斗"的双轨成长在RPG中极为罕见——大多数RPG的非战斗技能是战斗能力的附属品。

**精纯=不能刷的进度门**：精纯只能通过主线推进获得——不能通过"刷"来获取。这确保太吾的终局强度不是"花时间就有的"，而是"推进主线才有的"。
