# 自由度与涌现性系统 — 太吾绘卷

> 游戏版本: 太吾绘卷 正式版 v1.0+ (Steam, 2022)
> 数据来源: [灰机Wiki](https://taiwu.huijiwiki.com/), [贴吧](https://tieba.baidu.com/f?kw=太吾绘卷)

---

## 一、"不被预设的太吾人生"

太吾绘卷没有一个线性的"主角应该做什么"的预设路径。游戏的核心开放性是：**你可以选择成为什么样的太吾**。

| 太吾类型 | 核心活动 | 门派倾向 | 主要产出 |
|------|------|------|------|
| 战斗型太吾 | 修习功法/挑战剑冢/比武 | 少林/武当/金刚宗 | 武力/声望 |
| 生产型太吾 | 锻造/炼药/织锦/太吾村经营 | 铸剑山庄/百花谷 | 装备/丹药/银两 |
| 社交型太吾 | 结交NPC/联姻/建立关系网 | 然山派/璇女派 | 人脉/威望 |
| 学者型太吾 | 读书/修习技艺/著书 | 峨眉/然山 | 造诣/秘籍 |
| 邪道太吾 | 偷学功法/暗杀/绑架/勒索 | 界青门/血犼教 | 快速获取功法/大量银两 |
| 促织太吾 | 收集促织/斗蟋蟀/与NPC赌斗 | 不限 | 乐趣/银两/NPC好感 |

## 二、涌现性的来源

### 系统交互的涌现

太吾世界中，故事不是脚本写的——是系统碰撞产生的：

| 涌现类型 | 示例 |
|------|------|
| 爱恨情仇 | NPC与你的关系好→帮你对抗敌人→在剑冢战斗中被杀→你誓死复仇 |
| 师门恩怨 | 被逐出师门的弟子→加入敌对门派→回头挑战掌门→夺回地位 |
| 家族兴衰 | 太吾家族多代经营→太吾后人继承产业→遭遇门派征伐→家族毁灭→新太吾重建 |
| 轮回奇缘 | 老太吾死亡→转世为婴儿→被下一代太吾收养→培养成人→重新执掌太吾使命 |
| 促织风云 | 一只绝世促织→与多个NPC赌斗→赢来无数银两→用来扩建太吾村 |

### NPC行为的涌现

所有NPC在无玩家干预时：
- 自主修炼功法/提升属性/学习技艺
- 结婚/生育/建立家庭/离婚/丧偶
- 与好友结拜/与敌人结仇/向仇人复仇
- 在门派内部晋升或被排挤
- 死亡后进入轮回/转世重生

## 三、开放世界中的"可做之事"

| 行为 | 说明 | 后果 |
|------|------|------|
| 偷学功法 | 在门派外偷看他人练功 | 被发现→门派敌对 |
| 绑架/勒索 | 绑走NPC索要赎金 | 善值下降/门派通缉 |
| 暗杀 | 偷偷击杀NPC | 被其亲友复仇 |
| 结拜/联姻 | 建立深厚关系 | 获取政治/军事同盟 |
| 乞讨 | 向NPC乞讨 | 获得少量银两/降低声望 |
| 收养 | 收养孤儿作为弟子 | 培养下一代太吾候选人 |
| 比武招亲 | 参加/主持比武招亲 | 获得伴侣/声望 |

## 四、自由度的边界

太吾的"可做任何事"不是无限制的:

| 行为 | 限制 | 原因 |
|------|------|------|
| 学习神一品功法 | 需要资质≥100 + 门派支持度100% | 最高的武功不是谁都能学的 |
| 暗杀掌门 | 需要足够高的战斗力 | 掌门是门派最强者 |
| 摧毁一个门派 | 不可——门派是永久机构 | 世界结构不会因为玩家行为而毁灭 |
| 改变世界格局 | 剑冢清除=太吾的核心使命 | 有一条主线牵引玩家行为 |

## 五、设计特点分析

**太吾的"自由"是"角色自由"而非"世界自由"**：在Elona中你可以炸毁整个城镇——世界本身可以被你改变。在太吾中，门派、地图、基本结构是固定的——但**你在世界中的角色是完全自由的**。你可以是任何类型的太吾：战斗者、工匠、学者、社交家、促织大师。这种"角色自由"vs Elona的"世界自由"是两种不同的设计哲学。

**NPC自主人生=世界在你不在的时候也在呼吸**：太吾的所有NPC都有独立的人生轨迹——他们不依赖玩家存在。闭关修炼3年→出来发现世界局势已变——这不是"事件"触发的，而是NPC自主行为累积的结果。这种"世界在呼吸"是太吾涌现性的核心来源。

**主线=不是约束，是"终极意义"**：太吾有一条主线——清除7个剑冢。你可以在100小时内完全不推进主线（专心经营/培养/恋爱/斗促织），但剑冢始终在那里——它不催促你，但它提醒你："你终究是太吾传人，有使命要完成"。这种"可推迟但不消失"的主线设计是最优雅的"自由vs引导"平衡。

---

> 跨品类对比: 太吾的"角色自由"与Elona的"世界自由"形成对比——Elona允许你毁灭世界但角色某种程度上是"混沌的流浪者"，太吾不允许你毁灭世界但你的角色有"清晰的使命和传承"。
