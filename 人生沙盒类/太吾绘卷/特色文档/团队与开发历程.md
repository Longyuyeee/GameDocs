# 团队与开发历程 — 太吾绘卷

> 螺舟工作室(ConchShip Games) | 2015年成立 | 云南昆明 | 制作人: 茄子

---

## 一、"绿皮代码"传奇

| 阶段 | 细节 |
|------|------|
| **创始人背景** | 茄子——中文系毕业，原建筑设计师，用CAD画设计图 |
| **起步** | 最初用RPG Maker做了西方玄幻游戏《末日文书》 |
| **组建团队** | 3-5人远程团队，分布全国各地，早期素未谋面 |
| **第一次危机** | 开发一年后主程序退出→茄子自学编程→用"不专业的绿皮代码"继续开发 |

## 二、爆红与重做

| 时间 | 事件 |
|------|------|
| 2018年9月 | EA上线→首日3万份/首周30万份/3个月100万份→**爆红** |
| 2019年9月 | 销量200万→茄子决定**推倒重来**→昆明组建18人团队封闭开发 |
| 2020-2022 | 疫情/版号停发263天/主程序再次离开→重创 |
| 2022年9月 | **正式版发布**→首日同时在线6.4万→但BUG多→口碑暴跌→持续修复 |

## 三、市场表现

| 指标 | 数据 |
|------|:--:|
| **总销量** | **200万+** |
| Steam评价 | 褒贬不一(正式版初期)→逐步回升 |
| EA版评价 | 多半好评(一度是国产独立游戏评分最高) |

## 四、设计启示

1. **"绿皮代码"也能产出杰作**——茄子用不专业的代码实现了专业级的设计深度
2. **EA=双刃剑**——2018年的爆红给了团队资源，但4年封闭开发让玩家社区与开发者失去了联系
3. **正式版=信任的重建**——从2022年"褒贬不一"到持续修复和更新，螺舟在重建与玩家的信任
4. **一个人的设计愿景可以承载一个品类**——茄子负责策划/程序/文案/数值的几乎全部设计
