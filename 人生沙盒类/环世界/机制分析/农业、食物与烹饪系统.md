# 农业、食物与烹饪系统 — 环世界

> 游戏版本: RimWorld v1.5+ (Steam, 2018)
> 数据来源: [RimWorld Wiki](https://rimworldwiki.com/)

---

## 一、作物选择策略

| 阶段 | 推荐作物 | 原因 |
|:--|------|------|
| **开荒(第1周)** | **水稻**(3天) | 最快解决饥饿——必须在食物耗尽前收获 |
| 第2-3周 | 水稻+土豆 | 多样化的粮食来源 |
| 第1-3月 | 水稻→转玉米 | 玉米产量最高(22/格)但生长18天 |
| 后期 | 玉米(主食)+棉花(制衣)+药草(医疗) | 完整的农业产业链 |

## 二、冷藏与食品安全

| 概念 | 说明 |
|------|------|
| 冷藏室(-1°C~5°C) | 食材保存——延缓腐败 |
| **冷冻室(<-10°C)** | **永久防腐**——肉类不腐败 |
| 双层墙隔热 | 减少温度流失=节省电力 |
| 气锁(Airlock) | 双门+1格间隔——开门时减少冷气流失 |

## 三、烹饪管理

| 关键决策 | 说明 |
|------|------|
| 烹饪最低技能 | 技能0→简餐=高食物中毒风险。找有烹饪热情的殖民者。 |
| 精致餐(+5心情) | 需要蔬菜+肉——稳定的心情Buff来源 |
| 奢侈餐(+12心情) | 浪费资源但拯救处于精神崩溃边缘的殖民者 |
| 食人肉 | -20心情(所有人) / +15心情(食人者特性) |

## 四、设计特点分析

**水稻=刻意设计的"新手安全网"**：3天的生长周期是游戏中最短的——Tynan刻意设计了水稻作为初期"快速解决饥饿"的方案。新手不可能在第一周饿死——如果他们知道种水稻。这种"快速反馈"的农业入门降低了游戏的初期挫败感。

**食人=资源策略+道德选择**：在极端环境下(全殖民地断粮/冬季)，食人是唯一的生存选择——但它会带来沉重的心情惩罚。你必须权衡: 吃人肉→暂时解决饥饿→但全员心情暴跌→精神崩溃风险→殖民地可能从内部瓦解。这就是RimWorld——每一个生存选择都有代价。
