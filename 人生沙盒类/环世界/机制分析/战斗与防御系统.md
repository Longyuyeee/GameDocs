## 三、阵地设计

### 阵地(Killbox)设计原则

| 原则 | 说明 |
|------|------|
| 单入口 | 强迫所有敌人从同一个方向进攻 |
| 长走廊 | 敌人在走廊中无掩体——你的殖民者从末端射击 |
| 交叉火力 | 殖民者分布在多个角度——敌人无法同时掩蔽 |
| 陷阱配合 | 走廊中放置陷阱——敌人进入前已被削弱 |
| 炮塔辅助 | 自动炮塔提供额外火力 |

## 五、设计特点分析

**掩体系统="位置>装备"的战术哲学**：在RimWorld中，一个在75%掩体后方用短弓的部落战士可能击败一个在开阔地带用突击步枪的海盗。掩体是战斗中最强大的单一因素——这种"位置>装备"的设计让战术比装备更重要。

**阵地(Killbox)=玩家自发创造的"防御建筑学"**：阵地不是游戏设计的"推荐防御方式"——它是玩家社区通过系统交互自发发现的最优防御策略。这种"玩家创造策略"的涌现性是沙盒设计深度的标志。
