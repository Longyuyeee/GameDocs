# 经济与交易系统 — 骑马与砍杀2

> 游戏版本: Bannerlord v1.2+ (Steam, 2022)
> 数据来源: [Mount & Blade Wiki](https://mountandblade.fandom.com/)

---

## 一、收入来源

| 来源 | 稳定性 | 规模 |
|------|:--:|:--:|
| 封地税收 | 稳定(繁荣度决定) | 中-高 |
| 工坊(Workshop) | 稳定(城市资源决定) | 低-中(每工坊100-300/日) |
| 商队(Caravan) | 不稳定(可能被劫) | 中-高 |
| 战斗缴获/战利品 | 爆发性 | 极高(大型战斗) |
| 锻造出售 | 极高(需高锻造) | 10,000-50,000/件 |
| 雇佣兵工资 | 稳定 | 低-中(按影响力日结) |
| 竞技场奖金 | 一次性 | 低-中 |

## 二、工坊系统

| 要素 | 说明 |
|------|------|
| 购买价格 | 15,000-25,000第纳尔 |
| 产出 | 取决于城市周边村庄的原材料 |
| 利润 | 100-300/日(取决于当地供需) |
| 上限 | 家族等级决定最大工坊数量 |

## 三、商队系统

| 要素 | 说明 |
|------|------|
| 创建成本 | 15,000-25,000第纳尔 |
| 队长 | 需要同伴(高交易技能=更高利润) |
| 利润 | 浮动(可能亏)——取决于贸易路线安全 |
| 风险 | 被敌人/盗匪劫掠→商队消失 |

## 四、城市供需经济

每个城市的物价受**实际库存量**影响——大量买入→价格上升/大量卖出→价格下跌。玩家可以通过在不同城市之间倒卖商品(低价城买入→高价城卖出)来赚取差价。

| 商品 | 低价区 | 高价区 |
|------|------|------|
| 谷物 | 产粮村 | 饥荒城/围城中的城市 |
| 马匹 | 草原文化 | 缺马文化 |
| 铁矿石 | 矿村 | 缺铁城市 |

## 五、设计特点分析

**锻造=游戏的经济输出引擎**：一把传奇双手剑可以卖出数万第纳尔——这让锻造超越了"装备制造"，成为Bannerlord中GDP最高的产业之一。锻造暴利改变了游戏的经济基础——你不再需要靠"劫掠村庄"或"收税"来获取财富。

**工坊+商队=从"战士"到"资本家的转型**：初期经济靠"拿剑砍人→卖战利品"——中后期靠"工坊+商队的被动收入"。这种从"劳动力挣钱"到"资本挣钱"的经济转型是Bannerlord中从流浪者到国王叙事弧线的经济基础。
