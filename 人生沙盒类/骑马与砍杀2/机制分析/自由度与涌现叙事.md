# 自由度与涌现叙事 — 骑马与砍杀2

> 游戏版本: Bannerlord v1.2+ (Steam, 2022)

---

## 一、"从流浪者到国王"的完整弧线

Bannerlord没有主线剧情——它提供了一个**完全开放的沙盒**，玩家可以自由定义自己的"故事"：

| 人生路线 | 核心活动 | 终点 |
|------|------|------|
| 佣兵队长 | 为不同的王国当佣兵/打竞技场 | 积累财富→退休/开始自己的王国 |
| 商人工匠 | 锻造武器/经营工坊和商队 | 成为卡拉迪亚最富有的平民 |
| 封臣领主 | 宣誓效忠→积累封地→参与王国政治 | 成为王国中最有影响力的公爵 |
| 征服者 | 建立自己的王国→征服整个大陆 | **统一卡拉迪亚** |

## 二、涌现叙事的来源

Bannerlord的涌现故事来自系统交互而非脚本事件：
- 在竞技场中偶然击败一个领主→他成为你的宿敌→数年后在战场上相遇
- 你的工坊所在的城市被敌人攻占→你的投资化为乌有→你发誓夺回
- 你救了一个被俘虏的领主→他日后在投票中支持你获得封地

## 三、自由度边界

| 可以做的 | 不能做的 |
|------|------|
| 攻击任何NPC/领主 | 谋杀(不能像CK3那样暗杀) |
| 背叛你宣誓效忠的国王 | —(但代价是所有领主的好感暴跌) |
| 处决战败的领主 | —(但所有人都会恨你) |
| 抢劫村庄 | 毁灭城市(不可以) |

## 四、设计特点分析

**"无脚本=完全涌现"的沙盒哲学**：Bannerlord从不告诉你"应该做什么"——它只是给你一个世界、一些系统、然后放手。你的故事不是"固定剧情+玩家选择"——它是"你的行为×系统反应=独一无二的故事"。这是与CK3的"事件驱动叙事"不同的涌现哲学——Bannerlord更偏向"无脚本涌现"。
