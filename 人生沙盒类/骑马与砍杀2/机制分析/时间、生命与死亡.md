# 时间、生命与死亡 — 骑马与砍杀2

> 游戏版本: Bannerlord v1.2+ (Steam, 2022)
> 数据来源: [Mount & Blade Wiki](https://mountandblade.fandom.com/)

---

## 一、时间结构

| 单位 | 说明 |
|------|------|
| 1季(3天) | 季节循环——春季/夏季/秋季/冬季 |
| 1年(12天) | 年度结算 |
| 速度 | 暂停/1×/2×/4×(大地图) |

## 二、角色生命周期

| 阶段 | 年龄 | 游戏影响 |
|:--|:--|------|
| 幼年 | 0-16 | 不可操作/在家族中成长 |
| 青年 | 16-20 | 可开始带领部队/参战(属性低) |
| 壮年 | 20-40 | **巅峰期**——属性最高/技能增长最快 |
| 中年 | 40-55 | 属性持平/开始轻微下降 |
| 老年 | 55-65 | 属性下降 |
| 高龄死亡 | 65+ | 自然死亡概率逐年增加 |

## 三、死亡方式

| 方式 | 说明 | 可继承？ |
|------|------|:--:|
| 战斗阵亡 | 在战斗中被击杀(概率取决于医疗技能+战场情况) | 是(切换到继承人) |
| 处决 | 被敌方领主处决(如果你被俘虏) | 是 |
| 自然死亡(老死) | 年龄过高→健康耗尽 | 是 |
| 疾病 | 长期患病→死亡(罕见) | 是 |
| **家族灭绝** | 所有成年家族成员死亡→无继承人 | **游戏结束** |

## 四、战斗死亡的风险管理

| 方式 | 效果 |
|------|------|
| 医疗技能(高) | 降低角色在战斗后被判定为"阵亡"的概率 |
| 重甲/高防御 | 减少战斗中受到的伤害→不容易被击杀 |
| 避免亲自冲锋 | 在后方指挥→减少暴露于危险的时间 |
| 撤退时机 | 战斗不利时果断撤退→保存角色生命 |

## 五、季节系统

| 季节 | 地图影响 | 经济影响 |
|------|------|------|
| 春季 | 正常移动/无惩罚 | 粮食产量正常 |
| 夏季 | 正常 | 粮食产量最高 |
| 秋季 | 正常 | 粮食产量高 |
| **冬季** | **移动速度降低/雪地额外惩罚** | **粮食减产/城市可能缺粮** |

季节影响军事行动——冬季围城=补给困难，部队更容易因饥饿而损耗。

## 六、设计特点分析

**"角色会死"=沙盒的终极赌注**：在大多数游戏中你是"不死的英雄"。在Bannerlord中你可能会在30岁时被一个无名小卒一箭射死→游戏切换到你还未成年的继承人→你的王国可能在继承危机中崩溃。这种"角色会死"的风险让每一次战斗都带有真实的紧张感——你不仅是在为"胜利"而战，而是在为"不死"而战。

**继承人年龄窗口=最戏剧性的策略挑战**：安排继承人需要精确的时间计算——你必须确保在角色死亡前至少有一个18岁以上的继承人准备好。如果你40岁才开始生育→继承人在你60岁时才20岁→但你可能在55岁时战死→继承人尚未成年→家族崩溃。这种"年龄=时间窗口"的设计是所有家族策略的核心。
