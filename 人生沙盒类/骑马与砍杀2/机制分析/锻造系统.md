# 锻造系统 — 骑马与砍杀2

> 游戏版本: Bannerlord v1.2+ (Steam, 2022)
> 数据来源: [Mount & Blade Wiki](https://mountandblade.fandom.com/)

---

## 一、锻造循环

```
收集材料(矿石/木炭/熔炼缴获武器) → 精炼材料
→ 选择武器类型→自定部件(刃/护手/握柄/配重)
→ 锻造→根据技能和部件品质产出武器
→ 自由命名→使用/出售
```

## 二、部件解锁

每种武器由4个部件组成(刃/护手/握柄/配重)——不同部件通过**锻造**或**熔炼**对应类型的武器来解锁。高等级部件需要高锻造技能+运气。

| 部件品质 | 对武器的影响 |
|:--|------|
| 刃部 | 伤害/速度/长度 |
| 护手 | 防御/重量 |
| 握柄 | 速度/操作性 |
| 配重 | 平衡/伤害修正 |

## 三、传奇锻造

高锻造技能(200+)→小概率锻造出**传奇武器**——名字前有"Legendary"词缀——属性大幅超越同配方的普通版本。传奇武器是Bannerlord中价值最高(可达数万第纳尔)的可制造物品。

## 四、锻造经济

锻造是Bannerlord中最暴利的产业——一把高锻造技能+稀有部件+传奇品质的双手剑可卖出10,000-50,000第纳尔。锻造经济的完整循环:

```
战斗缴获武器→熔炼→获取材料+解锁部件→锻造高价武器→出售→
用利润买更多材料/更好的装备→更强的战斗力→缴获更好的武器→循环
```

## 五、设计特点分析

**锻造="从战士到工匠"的完整转型**：你不需要是战斗高手就能在Bannerlord中致富。一个高锻造技能的"铁匠"角色可以通过在城镇中锻造+出售武器来积累数百万第纳尔——足以雇佣最强大的雇佣军来为你征战。这是"非战斗人生"在Bannerlord中的最佳体现。

**传奇随机=锻造的"抽卡"快感**：即使两个完全相同的武器配方——一个是普通一个是传奇——属性差距巨大。这种随机性让锻造有了"万一这次出传奇"的期待感(类似于Diablo的传奇掉落)。
