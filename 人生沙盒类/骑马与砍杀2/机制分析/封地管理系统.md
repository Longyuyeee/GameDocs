# 封地管理系统 — 骑马与砍杀2

> 游戏版本: Bannerlord v1.2+ (Steam, 2022)
> 数据来源: [Mount & Blade Wiki](https://mountandblade.fandom.com/), [blink5.org](https://blink5.org/)

---

## 一、封地三维度

| 维度 | 说明 | 核心影响因素 |
|------|------|------|
| **繁荣度(Prosperity)** | 封地的经济规模——决定税收 | 粮食盈余/市场商品/建筑/政策 |
| **粮食(Food)** | 封地的生存基础 | 村庄产量(每村+6×Hearth等级)/围城=停产 |
| **忠诚度(Loyalty)** | 封地稳定——低忠诚=叛乱风险 | 总督文化匹配/政策/驻军规模 |
| **民兵(Militia)** | 本地防御力量 | 繁荣度/建筑/总督perk/政策 |

## 二、繁荣度循环

```
高繁荣→高税收(每天获得更多第纳尔)
但→高繁荣消耗更多粮食→如果粮食不足→繁荣下降
→ 必须投资粮食生产(建筑/村庄保护)来支持更高的繁荣
```

## 三、总督系统

指派同伴/家族成员作为封地总督——总督的技能和perk(管理/医疗/工程)直接影响封地的繁荣/忠诚/防御。

| 关键总督perk | 技能(等级) | 效果 |
|------|:--|------|
| 整洁街道 | 医疗(150) | +1繁荣 |
| 清洁基建 | 医疗(200) | +2繁荣 |
| 城墙 | 工程(175) | +100粮食储存 |
| 军火商 | 流氓(225) | +2民兵(围城时) |

## 四、建筑系统

每个封地有3-5个建筑槽——建设永久性建筑来增强封地。建筑依次升级(Lv1→Lv2→Lv3)，每级需要大量时间和金钱。

| 建筑 | 效果 |
|------|------|
| 果园/花园 | 粮食产量+ |
| 谷仓 | 粮食储存上限+ |
| 民兵场地 | 民兵生成+ |
| 水渠 | 繁荣度+ |
| 城墙 | 防御+粮食储存+ |

## 五、设计特点分析

**繁荣度=双刃剑**：高繁荣产生高税收(好事)，但也消耗更多粮食(坏事)。如果你的封地村庄被敌人掠夺——粮食供应中断——繁荣度会暴跌。这创造了一种动态的张力——你需要保护村庄来维持繁荣，而繁荣本身让你成为更有吸引力的攻击目标。

**总督=封地的"缩小版统治者"**：你不需要亲自管理每一个封地——指派总督来代理。总督的选择(谁有最好的管理/医疗/工程perk组合)定义了封地的管理质量。这与CK3的"内阁成员"有相似之处——都是"代理管理"的政治设计。
