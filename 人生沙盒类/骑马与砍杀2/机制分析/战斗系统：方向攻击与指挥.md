## 一、四方向战斗——骑砍的标志性系统

### 攻击与格挡

| 方向 | 攻击动作 | 格挡动作 | 优势 |
|:--:|------|------|------|
| 左 | 从左横劈 | 左挡 | 范围攻击/可命中多个目标 |
| 右 | 从右横劈 | 右挡 | 范围攻击/骑马时右侧攻击 |
| 上 | 下劈 | 上挡 | 高伤害/破盾率高 |
| 下 | 刺击 | 下挡 | 最远距离/破甲高/盾后刺击 |

### 操作方式

鼠标滑动决定攻击方向——向左滑=左劈，向前推=刺击，向后拉→前推=下劈。这种"鼠标手势=剑法"的设计让中世纪武术在PC上有了直觉式的操作方式。

### 战斗深度

| 要素 | 影响 |
|------|------|
| 速度加成 | 马速/角色移速→攻击伤害加成 |
| 蓄力 | 按住攻击键=蓄力→更高伤害 |
| 格挡时机 | 完美格挡=无伤害+反制窗口 |
| 破防 | 脚踢(按E)=破盾→连击窗口 |
| 部位伤害 | 头/躯干/四肢——不同护甲覆盖率 |

## 二、大规模战场指挥

### 部队命令

| 命令类型 | 作用 |
|------|------|
| F1-移动 | 指定位置/跟随我/冲锋 |
| F2-阵型 | 线阵/盾墙/方阵/散兵/圆阵 |
| F3-交战 | 射击/停火/近战/冲锋 |
| F4-面向 | 指定部队面对方向 |
| F5-坐骑 | 上马/下马 |
| F6-转移 | 部队编组间转移士兵 |

| 兵种 | 战场角色 | 最佳阵型 |
|------|------|------|
| 步兵(盾) | 前线坦克/吸收伤害 | 盾墙 |
| 步兵(双手) | 破阵/高伤害 | 线阵 |
| 弓兵 | 远程火力覆盖 | 散兵(高地) |
| 弩兵 | 破甲远程/慢射速 | 线阵 |

## 四、设计特点分析

**方向战斗="简单规则，无限深度"**：四个方向的攻击/格挡似乎简单——但组合了移动速度、蓄力、脚踢、完美格挡、部位伤害后，产生了远比大多数ARPG更深的战斗系统。骑砍的战斗不是"按技能键放招"——它是**物理驱动的武术模拟**。

**指挥系统="既是将军又是士兵"的独特视角**：你可以站在山顶上指挥你的300人部队——然后骑马冲下山加入冲锋。这种"战略视角↔第一人称战斗"的即时切换是骑砍独有的——在Total War中你只是指挥官，在For Honor中你只是战士，在Bannerlord中你是两者。
