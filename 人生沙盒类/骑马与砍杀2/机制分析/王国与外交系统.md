## 一、建国之路

| 阶段 | 条件 | 核心活动 |
|:--|------|------|
| 流浪者 | — | 打竞技场/做任务/招募小部队 |
| 雇佣兵 | 加入王国当佣兵 | 随王国军队参战/获取军饷 |
| 封臣(Vassal) | 宣誓效忠一个国王 | 获得第一块封地/参与王国政治 |

## 二、影响力(Influence)——政治货币

| 获取方式 | 影响力 |
|------|:--:|
| 战斗胜利(大规模) | 高 |
| 捐赠囚犯给封地 | 中 |
| 城镇建筑 | 少量被动 |
| 王国政策(神圣威严) | +3/日 |
| 王国政策(皇家卫队) | 统治者的部队上限+60 |

### 影响力用途

| 用途 | 成本 |
|------|:--:|
| 提议新政策 | 100-200 |
| 召唤封臣军队 | 根据军队规模 |
| 推翻不喜欢的投票 | 200-500 |
| 将同伴晋升为封臣 | 500 |

## 三、战争与和平

| 机制 | 说明 |
|------|------|
| **宣战** | 王国投票通过→自动进入战争 |
| **战争分数** | 击杀/破城/掠村/俘虏贵族=增加战争分数 |
| **议和** | 双方同意→输方支付贡金(Tribute) |
| **贵族囚禁** | 俘虏敌方贵族→阻止其招募新军队 |
| **处决贵族** | 极恶关系——**所有人都会恨你**(不推荐) |

## 五、设计特点分析

**影响力=花钱投票=中世纪民主的雏形**：你不是"国王=独裁者"。你需要影响力来通过政策——影响力来自战斗(不是来自"头衔")。所以即使你是国王，你仍然必须通过战斗来获得影响力→推动你的政治议程。这种"政治权力=军事贡献"的等式是对中世纪政治的忠实模拟。

**"招募战败领主"=软征服路线**：与大多数策略游戏中"征服=消灭敌人"不同——Bannerlord允许你劝说战败的敌方家族转投你的王国。这不是"杀死敌人"——而是"说服他们加入你"。这种设计更接近真实的中世纪政治——贵族家族之间的联盟和背叛是常态。
