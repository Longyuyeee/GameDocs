## 一、婚姻系统

在Bannerlord中，婚姻不是恋爱支线——它是**政治扩张工具**和**家族延续保障**。

### 求婚流程

| 步骤 | 内容 |
|------|------|
| 1. 寻找目标 | 寻找未婚的贵族女性/男性(通常18-30岁) |
| 2. 说服 | 通过对话小游戏说服对方(高魅力=更容易) |
| 3. 支付聘礼 | 向对方家族支付第纳尔(数额取决于对方家族等级) |
| 4. 结婚 | 配偶加入你的家族→成为可控制的家族成员 |

### 婚姻的战略价值

| 价值 | 说明 |
|------|------|
| **军事同盟** | 配偶的家族成为你的永久盟友——在战争中可以召唤他们 |
| **家族延续** | 配偶生育子女→子女成年后可带领部队/成为继承人 |
| **技能互补** | 配偶的技能可以用于你的家族(管理/医疗/工程等) |
| **政治联姻** | 与其他强大家族联姻→巩固政治地位 |
| **继承权** | 配偶的家族领地可能在继承中传给你的子女 |

## 二、子女与家族延续

| 阶段 | 年龄 | 可做的事情 |
|:--|:--|------|
| 幼年 | 0-12 | 在家族中成长(不可操作) |
| 少年 | 12-16 | 开始学习技能/接受教育 |
| 青年 | 16-17 | 可开始带领部队/参与战斗 |
| **成年** | **18+** | **完全可操作——带领部队/担任总督/继承家族** |

### 子女培养

| 培养方式 | 影响 |
|------|------|
| 父母教导 | 子女继承父母的部分技能和特质 |
| 战斗经验 | 带领子女参战→加速战斗技能成长 |
| 分配角色 | 指派子女担任军需官/斥候/外科医生→专注发展对应技能 |

## 三、家族继承

### 继承人指定

| 条件 | 说明 |
|------|------|
| 家族成员死亡/老死 | 玩家自动切换到指定的继承人 |
| 继承内容 | 所有封地/部队/工坊/商队/第纳尔=全部保留 |
| 继承人不成熟 | 如果继承人过于年轻(属性低/技能低)→可能导致封地叛乱/内外危机 |

## 四、设计特点分析

**婚姻="用一代人的时间换取永久同盟"**：你今天娶了瓦兰迪亚国王的女儿→你今天获得瓦兰迪亚的永久军事同盟。这个同盟在你妻子死亡前一直有效——可能是30年。这种"以婚姻换取长期军事安全"的机制是Bannerlord对中世纪政治的忠实模拟。

**家族延续=沙盒游戏中的"软终局"**：Bannerlord没有"胜利条件"——但有一个隐性的"失败条件"：你的家族灭绝。如果你的角色死了且没有成年继承人→游戏结束。这让"安排婚姻→生育子女→培养继承人"从"可选的RPG要素"变成了**生存必需的长期策略**。

**继承人年龄的"窗口期"风险**：如果你的角色在继承人才12岁时突然死亡→你切换到一个12岁的孩子→无法战斗/无法指挥/封臣可能叛变。这种"青黄不接"的危机是Bannerlord最戏剧性的家族叙事之一。
