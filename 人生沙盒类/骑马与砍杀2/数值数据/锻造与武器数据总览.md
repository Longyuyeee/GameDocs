# 锻造与武器数据总览 — 骑马与砍杀2

> 游戏版本: Bannerlord v1.2+ (Steam, 2022)

---

## 一、锻造基础

| 要素 | 说明 |
|------|------|
| 所需材料 | 铁矿石→精炼→粗铁/熟铁/钢/精钢/Thamaskene钢 |
| 燃料 | 木炭(木材→精炼→木炭) |
| 体力 | 每次锻造/熔炼/精炼消耗体力(体力耗尽需休息) |
| 技能需求 | 锻造技能决定可使用的部件等级和传奇概率 |

## 二、材料等级

| 材料 | 技能需求 | 武器伤害加成 | 获取 |
|------|:--:|:--:|------|
| 粗铁 | 0 | 基准 | 精炼铁矿石 |
| 熟铁 | 25 | +10-20% | 精炼粗铁 |
| 钢 | 75 | +20-40% | 精炼熟铁 |
| 精钢 | 150 | +40-70% | 精炼钢 |
| **Thamaskene钢** | **225** | **+70-100%** | **精炼精钢**(稀有) |

## 三、武器部件

| 部件位置 | 影响属性 | 可解锁数量 |
|------|------|:--:|
| 刃(Blade) | 伤害/速度/长度 | 20-50+(每种武器) |
| 护手(Guard) | 防御/重量 | 5-15 |
| 握柄(Grip) | 速度/操作性 | 5-15 |
| 配重(Pommel) | 平衡/伤害修正 | 5-10 |

## 四、部件解锁方式

| 方式 | 说明 |
|------|------|
| 熔炼武器 | 熔炼对应类型的武器→概率解锁该武器的部件 |
| 锻造武器 | 锻造某种武器→概率解锁该武器的新部件 |
| 高锻造技能 | 提升解锁概率 |

## 五、传奇武器概率

| 锻造技能 | 传奇概率 |
|:--|:--:|
| 100 | ~1% |
| 200 | ~5% |
| 300(上限) | ~10-15% |

传奇武器=名字前加"Legendary"→全属性+20-50%。
