| 政策 | 效果 | 推荐度 |

| 债务豁免 | +2忠诚(所有封地) | ★★★★★ |
| 神圣威严 | +3影响力/日(统治者) | ★★★★★ |
| 皇家卫队 | +60部队上限(统治者) | ★★★★ |

| 战争税 | -1繁荣/+税收 | ★★(有副作用) |
| 路桥费 | -0.2繁荣/日 | ★(副作用>收益) |

## 二、影响力来源

| 来源 | 影响力 |
|------|:--:|
| 大型战斗胜利 | 20-50+ |
| 攻破城镇/城堡 | 30-100+ |
| 捐赠囚犯 | 5-20 |
| 城镇建筑 | 少量/日 |
| 神圣威严政策 | +3/日(统治者的固定收入) |

## 三、宣战与议和

| 条件 | 说明 |
|------|------|
| 宣战 | 王国投票通过→立即进入战争 |
| 战争分数 | 击杀+掠村+破城+俘虏=累计 |
| 议和 | 双方投票通过→输方支付贡金 |
| 每日贡金 | 数百-数千第纳尔(取决于战争结果) |
| 停战期 | 和平后短时间内不能重新宣战 |

## 四、领主招募

| 方式 | 成本 | 忠诚度 |
|------|:--|:--:|
| 说服敌方家族转投 | 数十万-百万第纳尔 | 低(可能再次背叛) |
| 同伴晋升封臣 | 500影响力+一块封地 | **高**(你提拔的=更忠诚) |
| 婚姻联姻 | 无直接成本(外交) | 中(家族联盟) |
