## 二、关键结局详解

### 2.1 天下第一

**达成路径：**
1. 名声≥6级前往风云擂
2. 挑战天阶风云擂并获胜(多场战斗)
3. 找三教掌门(道玄宗/释法寺/儒圣馆)对话
4. 选择"挑战天下第一"
5. 前往论道峰进行四场车轮战(各派掌门)
6. 全部战胜后达成

**难度评估**：极高。需要顶级武功+装备+队友配置。

### 2.2 武林盟主

**达成路径：**
1. 六大门派(道玄宗/释法寺/儒圣馆/琅琊剑阁/九江水寨/神捕门)关系≥40
2. 六大门派掌门好感≥50~56
3. 说书人存活(不能死亡)
4. 前往轩辕台依次单挑六位掌门并获胜
5. 全部战胜后达成

**关键条件**：不能剿灭任何门派，需维持正道声望。

### 2.3 一代高僧

**达成路径：**
1. 加入释法寺
2. 完成释法寺所有门派任务
3. 品德值达到特定标准(高仁德/高礼节)
4. 通过铜人阵试炼
5. 触发佛法领悟事件

### 2.4 封官赐爵

**达成路径：**
1. 加入将军府或叶家军
2. 完成勤王救驾(皇帝存活)
3. 接受皇帝封赏
4. 达成异姓封王条件

### 2.5 登基称帝(君临天下)

**达成路径(全部满足)：**
1. 持有传国玉玺(不可装备，放背包)
2. 习得甲级功法
3. 完成叶家军事件并重返叶家军
4. 满声望
5. 剿灭九流门
6. 杀康王+杀齐王
7. 战胜柴王爷

### 2.6 归隐山林

**达成路径：**
1. 达到一定江湖成就(名声/武功/财富等)
2. 前往无名小村找村长
3. 选择退隐江湖
4. 根据品德值有不同变体(小村隐士/田园风光等)

## 四、多周目继承

大侠立志传的多周目系统是鼓励玩家反复体验不同路线的核心设计。

### 继承内容

| 继承项目 | 详细说明 | 是否绑定成就 |
|:--------:|:--------|:-----------:|
| 生活技能满级 | 任一能力达到5级(登峰造极)后，新周目可直接继承 | 是 |
| 成就点数 | 成就完成越多，开局可分配的五维属性点数越多 | 是 |
| 开局特质(先天天赋) | 更多成就解锁更多可选的先天天赋 | 是 |
| 开局装备 | 部分成就解锁开局装备(如功德牌、七宝佛珠) | 是 |
| 成长特性 | 任一该类能力5级后，下轮可选+30%成长特性 | 是 |
| 配方 | 已获得的配方可以继承 | 否(社区估算) |

### 不继承内容

| 不继承项目 | 说明 |
|:----------:|:----|
| 五维属性(基础值) | 但成就点数可在开局时重新加点 |
| 武道点数 | 总共100点，点满一条需40点(需重新获取) |
| 医术/毒术等级 | 面板技能等级不继承 |
| 门派关系 | 各门派好感/声望清零 |
| 背包物品 | 所有道具/装备不继承 |
| 金钱 | 铜钱清零 |
| 武功秘籍 | 已习得武功不继承 |
| NPC关系 | 所有人际关系清零 |

### 成就→初始点数递增关系

| 成就完成数量(约) | 开局可分配点数 | 解锁先天天赋数 |
|:--------------:|:-------------:|:-------------:|
| 0-10 | 10点 | 基础(3个) |
| 10-20 | 12点 | 5个 |
| 20-30 | 14点 | 8个 |
| 30-50 | 16点 | 12个 |
| 50-70 | 18点 | 15个 |
| 70-90 | 20点 | 18个 |
| 90+ | 22点 | 20+个 |

### 多周目规划建议

| 周目 | 目标 | 难度选择 | 重点 |
|:---:|:----|:--------|:----|
| 1周目 | 熟悉系统+扫荡基础成就 | 普通/困难 | 加门派+练生活技能+达成一个真结局 |
| 1.5周目 | 属性成就(悟性/福缘等) | 普通 | 对2周目开局很重要 |
| 2周目 | 清剩余成就+不同路线 | 困难 | 完成兵器200等大型成就 |
| 3周目+ | 挑战修罗+全收集 | 修罗 | 全结局+全成就+DLC内容 |

### 真结局继承特殊说明

- 必须达成至少一个**真结局**(武林盟主/武林霸主/天下第一/君临天下等)才能开启多周目继承功能
- 二周目最多继承**4个能力/特质**
- 前期的重要继承推荐：心有灵犀(外功资质翻倍)、草木皆兵(无视武器限制)
