## 二、核心门派详解

### 2.1 道玄宗

> 类型: 正道 | 地点: 霖安城 | 核心属性: 仁德>=50, 义气>=50

| 项目 | 详细数据 |
|------|---------|
| **加入条件** | 仁德>=50 + 义气>=50 |
| **核心武功** | 七星剑法、五行拳、太极剑法、太极拳 |
| **核心内功** | 无极洞心法、无极混元功、黄帝阴符经 |
| **绝招** | 太极十三势(乙)、天地无极(甲)、一剑破苍穹(甲,三教线) |
| **门派特色** | 摘星阁观星悟道、星宿大阵挑战 |
| **晋升路线** | 入门→弟子→核心→高层→掌门候选人 |
| **贡献获取** | 日常任务+捐献+藏经阁 |

**关键NPC：** 重阳子(掌门)、邋遢张(酒仙)、长春子(道友)

**特殊事件：** 无极洞心法获取需仁德-10以下触发禁闭（条件极苛刻：悟性=1且内力<=5才能领悟）

### 2.2 琅琊剑阁

> 类型: 中立 | 地点: 霖安城 | 核心属性: 御剑>=50

| 项目 | 详细数据 |
|------|---------|
| **加入条件** | 御剑>=50, 声望>=6级；或完成欧冶恒任务；或硬闯 |
| **核心武功** | 九宫剑法、无锋剑法、无名剑法 |
| **核心内功** | 九霄真经 |
| **绝招** | 万剑归宗(甲,轩辕剑任务+少阁主) |
| **门派特色** | 锻造加成(打造神兵利器)、铸剑任务(轩辕剑) |
| **贡献获取** | 铸剑+任务+捐献 |

**关键NPC：** 剑痴(掌门)、独孤剑一(大师兄)、欧冶恒(铸剑师)

**好感60事件：** 抓内奸任务→战斗后获得甲级内功【九霄真经】

**出师/高层：** 挑战大师兄(单挑/组队)→获得甲级金剑(干将/寒光/鱼肠之一)；版本更新后可铸造【轩辕剑】

### 2.3 释法寺

> 类型: 正道 | 地点: 大梁城 | 核心属性: 仁德>0

| 项目 | 详细数据 |
|------|---------|
| **加入条件** | 仁德>0（可出家/俗家两条路线） |
| **核心武功** | 如来神掌(佛法感悟400+拳掌>=50)、大力金刚掌、般若掌 |
| **核心内功** | 大日如来经、梵天圣功 |
| **绝招** | 菩提金身(乙)、六字大明咒(甲,三教线) |
| **门派特色** | 佛法感悟提升武功威力、僧侣路线限制(出家不可还俗) |
| **贡献获取** | 诵经+任务+捐献 |

**关键NPC：** 慧元(方丈)、释愚(武僧)、释厄(戒律)

**如来神掌获取流程：**
佛法感悟100→石碑1→战斗→佛法感悟200→石碑2→战斗→佛法感悟300→石碑3→战斗→佛法感悟400→石碑4→战斗(拳掌>=50)→获得

### 2.4 神捕门

> 类型: 正道 | 地点: 大梁城 | 核心属性: 仁德>=40

| 项目 | 详细数据 |
|------|---------|
| **加入条件** | 仁德>=40, 声望>=6级；或开局帮助洛千雪3日后入门；或硬闯交好 |
| **核心武功** | 贪狼剑法 |
| **绝招** | 天罗地网(乙,好感60捣毁势力任务) |
| **门派特色** | 悬赏任务系统、执法者路线 |
| **好感30** | 可接悬赏任务 |
| **好感60** | 捣毁燕子窝/玄衣会等势力→获得绝招 |
| **好感80(敬重)** | 洛天雄请教→乙级刀法/剑法 |
| **好感100** | 自动出师任务(洛千雪带队灭门战) |

**关键NPC：** 洛天雄(捕头)、洛千雪(搭档)、裘之墨(同事)

### 2.5 燕子窝

> 类型: 黑道 | 地点: 楚襄城 | 核心属性: 仁德<-39, 礼节<-39

| 项目 | 详细数据 |
|------|---------|
| **加入条件** | 仁德<-39, 礼节<-39, 声望>=4级；或开局帮助黑衣人；或硬闯 |
| **核心武功** | 云龙三现(乙) |
| **绝招** | 待补充 |
| **门派特色** | 暗取技能、妙手空空天赋、盗宝任务 |
| **好感30** | 学习【暗取】技能，获得乙级偷窃道具 |
| **好感60** | 能力考验(楚襄城找3人)→天赋【妙手空空】 |
| **暗取达标** | 触发盗宝任务(偷夜明珠)→燕子窝绝学 |
| **好感100** | 出师任务(洛千雪带队灭门战，步绝尘死亡则直接退门) |

**关键NPC：** 步绝尘(首领)、华青青(初始接触)

**推荐策略：** 优先入门(负道德要求)，完成后再刷仁德去正派

### 2.6 清风寨

## 五、门派叛出系统

### 5.1 叛门后果

| 后果类型 | 详细说明 |
|---------|---------|
| 声望惩罚 | 叛出门派后声望大幅下降（具体数值：约-50~-100声望，视门派地位而定） |
| 门派好感清空 | 该门派好感度归零，无法再接该门派任务 |
| 武功保留 | 已学武功和技能**保留**，不会被回收 |
| 门派特征回收 | 门派专属特征被回收 |
| 被追杀 | 部分门派（如神捕门、燕子窝）叛出后触发追杀事件 |
| 武功被废 | 道玄宗叛出时若仁德降到-80则触发"武功尽废"事件（内力降到1） |

### 5.2 叛门策略

| 策略 | 适用门派 | 说明 |
|:---:|---------|------|
| 拿完核心武功即退 | 老头门、清风寨 | 贡献要求低，核心武功拿到后立即退门 |
| 满好感叛出 | 神捕门、燕子窝 | 好感100出师后叛出，收益最大化 |
| 事件触发叛出 | 道玄宗 | 利用无极洞心法获取流程自动触发 |
