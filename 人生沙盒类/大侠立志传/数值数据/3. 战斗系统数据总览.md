## 一、集气系统（行动顺序）

大侠立志传采用**半即时制战棋**战斗系统。角色通过集气条积累决定行动顺序。

### 1.1 集气公式

```
集气速度 = 基础集气速度(14) + 敏捷加成(每点+1) + 轻功加成(每50点+1) + 特征/装备加成
```

| 影响因素 | 具体数值 | 说明 |
|---------|:-------:|------|
| 基础集气速度 | 14 | 1敏捷开局 |
| 敏捷加成 | 每点+1 | 10敏捷=24基础集气速度 |
| 轻功加成 | 每50点+1 | 轻功200额外+4 |
| 10敏捷天赋【先发制人】 | 初始集气+500 | 战斗开始集气条直接过半 |
| 25敏捷天赋【翩若惊鸿】 | 闪避集气+150 | 闪避后快速再行动 |
| 身如鬼魅(紫特征) | +20 | 紫色顶级集气特征 |
| 动如脱兔(紫特征) | +10 | 另+2格移动距离 |
| 逆流而上(紫特征) | +20 | 但移动距离-1格 |
| 万里独行(金特征) | +25~50 | 无队友时，每多1队友-5 |

### 1.2 集气行动规则

| 机制 | 说明 |
|------|------|
| 集气条上限 | 约1000点（基础值） |
| 行动触发 | 集气进度达到上限即行动 |
| 休息 | 待命可更快集气 |
| 战意 | 攻击/受击累积，100点释放绝招 |
| 会心一击 | 战意200点必出，释放后清空 |

## 二、伤害计算公式

### 2.1 完整伤害公式

```
伤害 = ((基础伤害基数 + 体力buff基数) × 属性修正 × 攻击种类修正
         × 武功系数 × 伤害加成 + 额外伤害) × 最终伤害加成 × 最终伤害减少
```

### 2.2 基础伤害基数

| 条件 | 公式 |
|:---:|------|
| A > D | (外功威力 + 0.4 × (A-D)) × A ÷ (A+D) |
| A <= D | 外功威力 × A ÷ (A+D) |
| 普攻 | 视为威力50的特殊外功，无攻击种类加成 |

> A = 攻方攻击力，D = 守方防御力

### 2.3 属性修正（三项加算）

| 修正项 | 计算方式 |
|-------|---------|
| 等级修正 | 每差1级 ±0.5% |
| 实战能力修正 | 每差10点 ±0.5%（上限12.5%） |
| 内功属性修正 | 无/调/阴/阳 互有 ±5%克制 |

### 2.4 伤害加成与减免

**加算类加成：**

| 加成来源 | 数值 |
|---------|:---:|
| 万夫之勇 | +15% |
| 烧香 | +15%~+35% |
| 破甲劲 | +25% |
| 会心一击 | +100% |
| 佛法加持(减) | -25% |
| 柔情似水(减) | -10% |
| 金钟罩(减) | -20% |

**最终伤害加成（乘算）：**

| 加成来源 | 数值 | 条件 |
|---------|:---:|------|
| 天煞孤星 | ×200% | 独自一人 |
| 稳扎稳打 | 每差1臂力+8% | 臂力高于对手 |
| 孔武有力 | 每1臂力+1.5% | 火系武道 |
| 跬步千里 | ×125% | 雷系武道 |
| 九凤武器 | ×115% | 装备九凤武器 |

**最终伤害减少（乘算）：**

| 减少来源 | 数值 |
|---------|:---:|
| 积甲如山 | ×50% |
| 坚若磐石 | 每点体质-1.5% |
| 金刚护体 | ×50% |
| 困难难度 | ×90%（受击乘算） |

### 2.5 暴击/连击/格挡/闪避率计算

| 属性 | 基础公式 | 上限 |
|:---:|---------|:---:|
| 暴击率 | 臂力×0.5% + 福缘×0.4% + 装备加成 | 约40%~60%+ |
| 闪避率 | 敏捷×0.5% + 福缘×0.4% + 轻功加成 | 约30%~50%+ |
| 格挡率 | 福缘×0.4% + 体质加成 | 约20%~40%+ |
| 连击率 | 悟性负相关（1悟+0.2%/级，10悟+0.02%/级） | ≥11悟性时为0 |

## 七、战斗系统特殊机制

### 7.1 武功种类与攻击范围

| 武功种类 | 典型武功 | 攻击范围形状 | 范围说明 |
|---------|---------|:----------:|---------|
| 拳法 | 如来神掌 | 单体 | 1格单体目标 |
| 剑法 | 万剑归宗 | 大范围 | 以自身为中心多格范围 |
| 刀法 | 无双刀法 | 扇形 | 前方扇形范围 |
| 枪法 | 沥泉枪法 | 直线贯穿 | 前方直线多格 |
| 棍法 | 达摩杖法 | 十字/扇形 | 范围攻击 |
| 音律 | 广陵散 | 范围 | 以自身为中心范围攻击 |

### 7.2 战斗地形影响

| 地形要素 | 影响 |
|---------|------|
| 障碍物 | 阻挡移动和直线攻击 |
| 高低差 | 影响攻击范围（有高低差加成设定） |
| 草丛 | 可潜行隐藏 |

### 7.3 异常状态

| 状态名称 | 效果 | 持续时间 |
|---------|------|:-------:|
| 流血 | 每回合损失生命 | 3~5回合 |
| 内伤 | 减少内力上限 | 3~5回合 |
| 中毒 | 每回合损失生命+属性降低 | 3~5回合 |
| 封穴 | 无法行动 | 1~2回合 |
| 致盲 | 命中率降低 | 2~3回合 |
