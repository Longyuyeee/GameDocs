## 一、事件触发条件一览

大侠立志传采用"无主线、多事件链"架构，200+动态事件通过角色等级、名声、天数、地点互动等条件触发。以下为主要事件线的触发条件。

### 事件触发通用规则

- **等级门槛**：大量事件需要角色达到特定等级(如50级触发勤王救驾)
- **名声门槛**：名声等级影响飞鸽传书和事件触发(如名声≥4级收珠光宝气楼请柬)
- **时间门槛**：部分事件在特定天数后触发(如楚襄兵祸在勤王救驾5天后)
- **前置事件**：大部分事件需要前置事件完成才可触发(网状事件链)
- **飞鸽传书**：达到条件后通过飞鸽传书通知玩家

## 二、三大事件线流程

### 2.1 九流门主线

#### 触发与推进流程

```
各主城小事件 → 直捣黄龙 → 浮屠寺 → 剿灭九流门 → 九流归一
```

**阶段一：各主城小事件**
在各主城(霖安/大梁/楚襄)触发与九流门相关的支线，包括：
- 街头暴徒事件
- 江湖人士失踪案
- 九流门渗透线索

**阶段二：竞宝大会(珠光宝气楼)**
- 名声≥4级触发
- 获得家园后解锁后续事件

**阶段三：浮屠寺大战**
- 位置：霖安城右侧
- 逐层释放被关押武林人士(每层均有)
- 顶层获取混沌图纸(枪)
- 出寺与九流门(铁面人)决战
- 释放的武林人士会参战，战死则永久死亡

**阶段四：剿灭九流门老巢**
- 大理寺风云3日后收飞鸽获得地址
- 多场战斗
- 燕歌行与师兄对决
- 楚狂生被秦会之偷袭
- 最终战后可选择当九流门主，获「九流归一」功法

### 2.2 叶家军事件线

#### 触发与推进流程

```
加入叶家军 → 收复失地 → 抗击燕军 → 勤王救驾 → 大理寺风云 → 行宫伸冤 → 重返叶家军 → 君临天下
```

**阶段一：加入叶家军**
- 前往叶家军中军帐
- 送包子给叶云提升好感至60
- 完成叶家军入门任务

**阶段二：收复与抗击**
- 参与叶家军收复失地战斗
- 抗击燕云铁军入侵
- 积累军功提升地位

**阶段三：勤王救驾**
- 与叶家军一同前往囚龙之地救宣和皇帝
- 选择"舍命保护陛下逃离"
- 吟诗四首获文武状元(有条件)

**阶段四：大理寺风云**
- 叶鹏举被诬陷
- 叶云离队救父
- 大理寺公堂选择：
  - 「确实属实」→ 仁德-100/义气-100/信用-100 → 风波亭行刑
  - 「放屁，纯属诬陷」→ 勇气+100/生命-90% → 风波亭救人或金牌令箭

**阶段五：重返叶家军**
- 前往吴越行宫为叶鹏举伸冤
- 连战两场+单挑秦桧之
- 成功后返回叶家军中军帐
- 获结局「尽忠报国(真)」

### 2.3 皇帝线(登基)

#### 触发与推进流程

```
勤王救驾 → 楚襄兵祸 → 浮屠寺 → 双王登基 → 大理寺风云 → 重返叶家军 → 合成玉玺 → 杀双王 → 战柴王 → 君临天下
```

**核心条件(必须全部满足)：**
1. 持有传国玉玺(放在背包中，不可装备)
2. 习得甲级功法
3. 完成叶家军事件并重返叶家军(尽忠报国后回中军帐触发)
4. 满声望
5. 剿灭九流门

**登基结局流程：**
1. 勤王救驾获得金牌令箭
2. 完成叶家军线重返叶家军
3. 合成传国玉玺
4. 先杀康王(吴越行宫)→掉落北冥(拳套)+狻猊(防具)
5. 再杀齐王(中州行宫)→掉落太虚(笔)+紫薇(防具)
6. 大地图行走被柴王爷攻击→强制战斗
7. 叶家军帮忙拦截第一波小兵
8. 战胜后达成结局「君临天下(真)」

**注意**：若不满足重返叶家军条件，杀双王后只会打出「游戏江湖(真)」结局。「君临天下」和「游戏江湖」都是强制结束游戏，不想结束就别去见柴王。

## 四、江湖历练

江湖历练是大侠立志传的核心串联机制，将零散的事件连接成有意义的江湖生涯。

### 触发逻辑

```
探索地图 → 遇到事件 → 做出选择 → 影响属性/关系 → 解锁新事件 → 改变世界状态
```

### 历练类型与奖励

| 历练类型 | 触发方式 | 典型奖励 | 备注 |
|:--------:|:--------|:--------|:----|
| 路遇不平 | 大地图随机 | 品德+声望+铜钱 | 可选择帮或不帮 |
| 江湖传闻 | 酒馆打听 | 线索+事件触发 | 获得隐藏地点信息 |
| 飞鸽传书 | 名声达标自动 | 主线推进 | 主要串联方式 |
| 门派任务 | 门派内接取 | 贡献+武功 | 门派晋升必经 |
| 偶遇高人 | 特定地点 | 武功秘籍/能力提升 | 一次性事件 |
| 英雄救美 | 特定条件 | 好感+结缘机会 | 可结缘角色相关 |
| 镖局押镖 | 虎威镖局 | 金钱+声望 | 韩红玉相关 |

## 五、事件时效

大侠立志传最核心的特色之一就是**取消时限，自由探索**。游戏中绝大多数事件没有时间限制，玩家可以自由选择何时推进。

### 唯一有时限的事件列表

大侠立志传中几乎没有硬性时限事件。以下为少数与时间相关的软性约束：

| 事件 | 时效说明 | 类型 |
|:----|:--------|:----|
| 浮屠寺→双王登基 | 浮屠寺完成后，去双王登基前可做大量支线 | 触发型(登基后部分事件关闭) |
| 登基前支线 | 双王登基后开化塔/郡主结缘/九江水寨等事件关闭 | 触发型(一经过期无法挽回) |
| 楚襄兵祸 | 兵祸期间楚襄城无法正常进出(可用脏街密道) | 状态型(非时限) |
| 勤王救驾皇帝存活 | 皇帝若中途死亡则跳过直接楚襄兵祸 | 事件分支(非时限) |
| 新年结局 | 新年期间通关剧情获得(限时活动) | 现实时间限时 |
| 拍卖会 | 各拍卖会在特定时辰举办 | 周期性(可反复参加) |

### 软性推进机制

虽然没有硬性时限，但游戏通过以下方式"推动"玩家推进事件：

1. **飞鸽传书**：条件满足后发送，不会过期，但不推进则不会解锁后续
2. **世界状态变化**：如双王登基后城市状态变化(部分NPC位置/对话改变)
3. **蝴蝶效应**：一个事件的选择影响后续可触发的事件
4. **结局触发**：达到条件后可选择触发结局，也可选择继续游历

> **设计解读**：大侠立志传的"无时限"设计是其"江湖生存模拟器"定位的核心——玩家不需要被时间追着跑，可以按照自己的节奏探索江湖。这与同类游戏《太吾绘卷》的季节性时间压力形成鲜明对比。
