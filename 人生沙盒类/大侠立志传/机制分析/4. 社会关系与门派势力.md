## 1. 门派总览

大侠立志传拥有**9个以上可加入门派**，分为正道、官府/军旅、黑道、其他四大阵营。不同门派有独立的武功体系、剧情线和晋升路径。

## 2. 善恶系统

### 2.1 善恶值的运作机制

| 项目 | 说明 |
|:----:|------|
| **初始值** | 0（中立） |
| **善行增加** | 助人/不杀/捐赠/宽恕敌人/完成正道任务 |
| **恶行增加** | 滥杀无辜/偷窃/抢劫/背叛/加入黑道 |
| **善恶等级** | 大善人(≥100)→善人(50-99)→中立(±49)→恶人(-50~-99)→大恶人(≤-100) |
| **善恶值不决定一切** | 部分正道门派也接受"改邪归正"的玩家 |

### 2.2 善恶值的影响范围

| 影响维度 | 善值高 | 恶值高 | 中立 |
|:-------:|:-----:|:-----:|:---:|
| **正道门派加入** | 容易 | 极难或不可 | 需要额外条件 |
| **黑道门派加入** | 不可 | 容易 | 需要额外条件 |
| **NPC初始好感** | +10-20 | -10-30 | 正常 |
| **商店折扣** | 部分商店优惠 | 黑市优惠 | 正常 |
| **队友招募** | 正道角色更易招募 | 黑道角色更易招募 | 正常 |
| **事件选择** | 更多"帮助他人"选项 | 更多"暴力/胁迫"选项 | 正常 |
| **结局方向** | 正道结局/善终 | 黑道结局/恶报 | 中立结局 |

### 2.3 善恶值对门派路线的锁定效应

大道侠立志传的善恶系统不是简单的"奖励好人，惩罚坏人"——它是**门派选择的隐形门槛**：

```
善值路线限制：
  大善人 → 可加入道玄宗/琅琊剑阁/释法寺/儒圣馆/神捕门/叶家军
         → 不可加入清风寨/九流门（除非叛门）
         → 燕子窝需特定事件（非纯恶）

恶值路线限制：
  大恶人 → 可加入清风寨/九流门/燕子窝
         → 不可加入道玄宗/琅琊剑阁/释法寺/儒圣馆
         → 神捕门/叶家军可能被通缉

中立路线：
  自由切换——但善恶值偏移到极端后难以逆转
```

> 设计特点：善恶值不是"锁死"的——通过特定事件和大量捐赠可以"洗白"或"堕落"。但洗白/堕落的成本很高（需要大量资源投入），这意味着玩家的善恶选择具有**长期承诺性**——你选择了恶路，就要承担相应的后果。

## 3. NPC全交互系统

### 3.1 交互维度

大侠立志传的NPC交互系统非常深度——几乎**所有NPC都可交互**，交互维度远多于同类游戏：

| 交互类型 | 消耗 | 好感度影响 | 效果 | 解锁条件 |
|:-------:|:----:|:---------:|------|:-------:|
| **送礼** | 道具（礼物） | 大幅提升 | 好感度增加 | 初始可用 |
| **切磋** | 无 | 小幅度提升/降低（取决于胜负） | 实战经验+好感 | 好感度≥30 |
| **结拜** | 无 | 锁定最高档 | 获得结拜兄弟/姐妹——可召唤助战 | 好感度≥80+特定条件 |
| **结仇** | 无 | 大幅降低 | 进入敌对状态——可攻击 | 交互失败或主动选择 |
| **交易** | 铜钱 | 无直接影响 | 购买/出售物品 | 初始可用（好感度高则价格优惠） |
| **交换物品** | 自身物品 | 小幅度提升 | 以物易物——获得稀有物品 | 好感度≥50 |
| **请教武功** | 好感度消耗 | 小幅度降低（为了平衡） | 学习NPC的武功 | 好感度≥60+NPC会该武功 |
| **送信/传话** | 时间 | 小幅度提升 | 任务相关 | 特定任务触发 |
| **邀请入队** | 无 | 达到阈值 | NPC成为队友 | 好感度≥60+特定条件 |
| **偷窃** | 无 | 被发现后大幅降低/结仇 | 获取NPC物品 | 需要潜行技能 |

### 3.2 好感度机制

| 好感度区间 | 等级描述 | 解锁交互 | 效果 |
|:---------:|:-------:|:--------:|------|
| 0-19 | 陌生人 | 送礼/交易 | 基础交互 |
| 20-39 | 认识 | +切磋 | NPC会主动打招呼 |
| 40-59 | 朋友 | +交换物品/+请教 | 交易折扣5% |
| 60-79 | 好友 | +邀请入队 | 交易折扣10%+特殊对话 |
| 80-99 | 至交 | +结拜 | 交易折扣15%+专属事件 |
| 100 | 生死之交 | 全解锁 | 交易折扣20%+专属奖励 |

### 3.3 礼物偏好系统

不同NPC有不同礼物偏好——送礼不是"什么贵送什么"，而是"送对人才能涨好感"：

| NPC类型 | 偏好礼物类型 | 厌恶礼物类型 | 高效送礼策略 |
|:-------:|:-----------:|:-----------:|:----------:|
| **武人** | 武器/武功秘籍/酒 | 书籍/文玩 | 送低级武功秘籍即可 |
| **文人** | 书籍/字画/酒 | 武器/防具 | 送书籍性价比最高 |
| **佛门** | 佛经/素斋 | 酒肉/武器 | 佛经最优先 |
| **女性NPC** | 饰品/化妆品/花 | 武器/粗鄙之物 | 送花（价格低效果好） |
| **商人** | 钱/值钱物品 | 无特殊厌恶 | 送价值高的物品 |
| **孩童** | 玩具/小吃 | 武器/书籍 | 送小吃最经济 |

## 6. 与太吾绘卷门派对比

### 6.1 门派系统维度对比

| 维度 | 大侠立志传 | 太吾绘卷 |
|:----:|:---------:|:--------:|
| **门派数量** | 9+个（含隐藏门） | 15个（正派×7+邪派×5+中立×3） |
| **武功数量** | 百余种（四品级×六大类×子类） | 1000+种（各门派独立功法体系） |
| **加入方式** | 属性阈值/事件触发/善恶门槛 | 地区声望+门派好感+推荐信 |
| **晋升体系** | 4阶（入门→弟子→核心→高层） | 9阶（从九品→正一品） |
| **门派关系** | 有（友好/敌对/中立） | 有（正/邪/中立阵营+独立关系） |
| **叛门惩罚** | 追杀/武功禁用 | 叛门后功法威力减半 |
| **门派专属剧情** | 几乎所有门派有独立剧情线 | 部分门派有独立事件 |

### 6.2 核心差异分析

1. **"RPG式" vs "模拟式"门派**：
   - 大侠立志传的门派是**RPG式的**：每个门派有独立的剧情线、专属NPC、独特的晋升事件。加入一个门派就像开始了一条新的任务线。游戏的叙事重心在"门派故事"上。
   - 太吾绘卷的门派是**模拟式的**：门派是武功和资源的提供者——你加入门派主要是为了学习功法，门派本身的故事性较弱。游戏的叙事重心在"太吾传人的个人冒险"上。

2. **善恶路线的"软硬兼施"**：
   - 大侠立志传的善恶系统对门派加入有"软限制"（善值低→正道难以加入）和"硬限制"（某些门派完全拒绝特定善恶值的玩家）。这种设计鼓励玩家在创建角色时就规划好门派路线。
   - 太吾绘卷的立场系统（刚正/仁善/中庸/叛逆/唯我）对门派加入也有影响，但更多是"效率影响"而非"门槛限制"——立场不匹配时仍然可以加入门派，只是获取支持的方式不同。

3. **门派晋升的可视化奖励**：大侠立志传的每个门派晋升阶段都提供明确的可视化奖励（新武功/新装备/新称号），晋升感很强。太吾绘卷的门派晋升更像"解锁借阅权限"——晋升后可以学习更高级的功法，但并不提供直接的战力提升。

4. **NPC交互的"全交互"设计**：大侠立志传最显著的设计特色是"几乎所有NPC都可交互"——这不只是"能对话"，而是"能送礼/请教/结拜/结仇/偷窃"。相比太吾绘卷，大侠立志传的NPC更像一个有独立需求的"人"，而不是"功法发放机"。

## 7. 设计特点分析

1. **善恶系统作为"软性道德锁"**：大侠立志传的善恶系统不像某些游戏那样"善恶决定一切"——它更接近一种"软性引导"：你可以选择无视善恶，但你的选择会影响你能加入的门派、能招募的队友和能走向的结局。这种设计避免了"强制道德"的僵硬感，同时让"道德选择"有了实际的分量。

2. **门派即"职业系统"**：大侠立志传没有传统的"职业"选择（战士/法师/盗贼）——门派起到了职业系统的作用：加入道玄宗=成为"太极拳师"，加入叶家军=成为"枪术将军"。门派的定位比职业选择更宽泛——它不仅决定你的战斗风格，还决定你能接触的剧情、遇到的角色和走向的结局。

3. **NPC交互的"网格化"设计**：所有NPC共享同一套交互框架（送礼/切磋/结拜/结仇/交易/偷窃），这让交互逻辑变得可预测——你不需要对每个NPC"重新学习"交互方式。这种"一致性"设计降低了认知负担，让玩家可以更专注于"选择"而不是"怎么操作"。

4. **阵营对立的"非对称设计"**：正道门派数量多于黑道（4 vs 3+九流门），但黑道门派的武功强度往往更高（九流门的禁术、清风寨的暴力刀法）。这种"数量劣势×质量优势"的平衡设计，让走黑道路线的玩家不会感到"被惩罚"。

5. **与部落与弯刀的对比**：部落与弯刀的势力系统更接近"战略地图"——不同势力占据不同城市，玩家可以加入/背叛/征服。大侠立志传的门派系统更接近"RPG职业选择"——加入的意义是获得该门派的武功和剧情，而不是参与势力间的领土争夺。大侠立志传的门派是"叙事驱动"的，部落与弯刀的势力是"战略驱动"的。
