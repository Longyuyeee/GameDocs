## 一、队友招募系统

### 1.1 招募机制

大侠立志传的队友招募采用"好感度门槛+特定任务触发"的双层机制：

```
送礼/对话/共同事件 → 好感度提升 → 好感达标 → 触发角色专属任务
→ 完成任务 → 好感再度提升 → 好感达成邀请阈值 → 选择"邀请入队"
```

### 1.2 好感度影响因素

| 因素 | 影响幅度 | 说明 |
|:--:|:--:|:--:|
| **礼物品质** | +5~+50/次 | 高品质礼物（甲级）提升幅度远大于低品质 |
| **礼物喜好匹配** | 1.5x~3x | 每个角色有喜好的礼物类型（如花四娘喜好花卉） |
| **"吉人天相"特征** | 最多+90% | 角色创建时选择该特征可大幅提升好感增长速度 |
| **声望差距** | ±10%~±30% | 玩家声望低于NPC时好感增速降低；高于时提升 |
| **"彬彬有礼"特质** | +20% | 该特质提升送礼好感收益 |
| **共同战斗** | +1~+3/场 | 带NPC共同战斗，每场微量提升 |
| **完成任务** | +10~+50/次 | 完成与NPC相关的支线任务 |

### 1.3 好感度等级体系

| 好感等级 | 好感值范围 | 解锁效果 |
|:--:|:--:|:--:|
| **冷淡** | 0-19 | 基本对话，交易价格不利 |
| **中立** | 20-39 | 正常对话，普通交易 |
| **友善** | 40-59 | 解锁部分角色的专属对话，交易折扣 |
| **亲近** | 60-79 | 解锁专属任务线起点 |
| **信赖** | 80-99 | 可邀请入队/结缘前置 |
| **亲密** | 100 | 可结缘（特定角色）/结拜（男性角色） |

## 三、战斗与家园中的队友

### 3.1 战斗出战

| 参数 | 说明 |
|:--:|------|
| **最大出战数** | 5名队友（含宠物）同时出战 |
| **出战位** | 战斗中可与玩家角色协同，每个队友独立行动 |
| **AI控制** | 队友自动行动（可设置战术倾向：进攻/防守/辅助） |
| **阵亡机制** | 队友血量归零→倒地可复活（战斗结束后自动恢复） |
| **经验分配** | 队友与主角共享战斗经验，但成长速度为主角的70% |

### 3.2 家园进驻

拥有家园后，可邀请伙伴进驻，提供被动属性加成：

| 进驻角色类型 | 加成类型 | 加成幅度 |
|:--:|:--:|:--:|
| **战斗型** | 主角攻击/防御 | +5%~+15% |
| **生产型** | 制造技能经验 | +10%~+20% |
| **辅助型** | 生活技能效率 | +10%~+15% |
| **特殊型** | 特殊效果（如金币掉落增加） | 视角色而定 |

### 3.3 宠物系统

宠物（通过驯兽获得）在战斗系统中承担特殊角色：

| 宠物类型 | 战斗定位 | 核心技能 |
|:--:|:--:|:--:|
| **狼** | 敏捷输出 | 撕咬（流血）、狼嚎（队友增益） |
| **虎** | 高爆输出 | 猛扑（击倒）、虎啸（减益敌方） |
| **熊** | 坦克/控制 | 拍击（眩晕）、防御姿态 |
| **鹿** | 治疗/辅助 | 治愈（回血）、灵敏光环 |
| **鹰** | 侦察/骚扰 | 俯冲（打断）、视野扩展 |

## 四、特殊交互系统

### 4.1 结仇系统

与NPC交恶（攻击/偷窃被抓住/拒绝请求/选择敌对选项）会触发结仇系统：

| 结仇等级 | 效果 | 消除方式 |
|:--:|:--:|:--:|
| **不满** | NPC对你冷淡，交易价格+10% | 送礼道歉 |
| **敌对** | NPC拒绝互动，可能主动攻击 | 完成特殊任务+送礼 |
| **死仇** | NPC永不原谅，见面即战斗 | **无法消除**（必须击杀或避开） |

### 4.2 月老祠灭失警告

> **重要设计提醒**：如果玩家选择灭掉白云观（游戏中的一种极端路线选择），则**月老祠功能永久失效**——所有需要月老祠结缘的角色将无法结缘。这是一个不可逆的操作，象征着"毁灭了江湖中的姻缘纽带"。这一设计意图是让玩家在选择极端路线时意识到后果——你不仅改变了自己的命运，也关闭了整个世界中的某些可能性。

## 五、设计特点分析

**"好感度驱动的RPG队友"而非"关系模拟器"**：大侠立志传的队友系统本质上是传统的CRPG式队友招募——好感度是升级机制，队友任务是剧情载体，结缘是终局奖励。这与太吾绘卷的CK3式复杂关系网（婚姻、仇恨、同盟、师承形成的动态社会网络）有本质差异。太吾的关系系统是"世界的运行规则"——NPC会自行结婚生子、结仇结盟，玩家只是这个社会中的一员。大侠的关系系统是"玩家的工具"——好感度是玩家需要刷的数值，队友是玩家可以使用的资源。

**"男性结拜+女性结缘"的二元设计是优点也是局限**：优点在于它提供了清晰的社交目标导向——玩家知道对女性角色刷好感到100是"攻略"，对男性角色刷好感到100是"兄弟"。局限在于这种性别二元划分在2023年的游戏设计中显得略显传统（相比于觅长生中道侣不分性别、鬼谷八荒的双修道侣也无性别限制）。不过从武侠文化角度看，这种设计确实更贴近传统武侠小说的叙事模式。

**"不可逆的姻缘关闭"体现了武侠的"选择重量"**：月老祠因灭白云观而永久失效的设计，是大侠立志传中"选择有重量"理念的典型体现。在大多数RPG中，支线选择的后果通常是可逆的（读档即可）。但在大侠的设计中，"灭白云观"不是一个支线决定——它是一个改变世界状态的重大选择，其后果包括但不限于：失去月老祠、某些角色无法结缘、某些路线永久关闭。这种设计理念与Majora's Mask中"时间不可逆"的设计有异曲同工之妙——选择不只是"改变故事"而是"改变世界"。

**与觅长生道侣系统的对比**：觅长生的道侣系统更注重"双修陪伴"的长期关系——道侣可以一起修炼、切磋、探索，关系中没有一个明确的"终点"（结婚只是关系的一个状态）。大侠的结缘系统则有一条清晰的"攻略路径"和"终局奖励"——好感收集→结缘→家园加成。如果说觅长生的道侣是"旅途中的伴侣"，大侠的结缘更像是"奖杯式的成就"。

## 六、跨游戏对比

| 维度 | 大侠立志传 | 太吾绘卷 | 觅长生 | 神界原罪2 |
|:--:|:--:|:--:|:--:|:--:|
| 关系系统本质 | 好感度+RPG任务 | 社会关系网模拟 | 道侣+双修 | 队友个人任务线 |
| 可招募数量 | 15+角色+宠物 | 无固定队友（村民可拉拢） | 6个道侣位 | 6个起源角色 |
| 结缘/结婚 | 月老祠结缘/结拜 | 自由婚姻+子嗣传承 | 双修道侣 | 可恋爱（结局才有实质） |
| 性别限制 | 男性结拜/女性结缘 | 无限制 | 无限制 | 无限制 |
| 专属剧情深度 | 每个角色有独立支线 | NPC有随机事件链 | 道侣有单任务线 | 每个起源角色有完整主线 |
| 关系不可逆后果 | 灭白云观→月老祠失效 | 仇恨会自我演化 | 背叛道侣→永久敌对 | 杀死队友→永久失去 |
| 家园互动 | 伙伴进驻提供加成 | 太吾村工作安排 | 洞府双修 | 无 |

---

> 跨品类对比: 大侠立志传的"好感度→结缘/结拜→家园加成"的三段式设计在武侠品类中属于轻量级社交系统——它比太吾绘卷的CK3式关系网更易上手，但比鬼谷八荒的"点赞就双修"更有仪式感。结缘系统的核心价值不在于关系模拟的深度，而在于它为玩家的江湖旅程提供了"人"的锚点——那些被你招募的队友构成了你在这个世界的情感记忆。
