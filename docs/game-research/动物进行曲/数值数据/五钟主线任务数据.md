## 二、每钟任务详细步骤

### 红钟
1. Owen碎石→进入矿洞下层
2. 10F→钟座空→钟在Ramsey处
3. 铁+铜+银+金矿各1→Ramsey→获得修复的钟
4. 放回10F钟座→敲响

### 黄钟
1. 海滩南→遇到Collin(虚弱)
2. Ruth守门→10优质作物→开门
3. 取钟→女神说月光浴→月石上放置
4. 敲响

### 蓝钟
1. Paolo→钥匙→水洞
2. Ben忘记旋律→Paolo知城镇旋律
3. 制作"旋律重现器": 黑珍珠+鸭蛋+5鱼
4. 奏旋律→Ben记起→敲响

### 绿钟
1. Hanna/Cain/Dale→解锁赋格森林
2. 发现魔女被变蛙→魔法师要求
3. 完美黄油+优质玉米粉+木槿→魔女还原
4. 魔女赠绿钟→敲响

### 紫钟
1. 四钟后→村民提"鬼魂"
2. **凌晨2AM**→教堂门口→剧情→门开
3. Edge哭泣(没人许愿)→许愿神社
4. 女神指导→敲响紫钟
