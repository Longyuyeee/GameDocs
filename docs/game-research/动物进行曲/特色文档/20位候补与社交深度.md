# 20位候补与社交系统深度 - 动物进行曲(Harvest Moon: Animal Parade)

> 游戏版本: Harvest Moon: Animal Parade (Wii, 2008年)

## 一、概述

动物进行曲拥有牧场品类中**阵容最庞大的攻略角色群**——10位男性+10位女性(含2位特殊神祇)。配合7心告白关系锁定+8对情敌配对+情敌子女——社交系统的深度至今未被超越。

## 二、20位候补的构成

| 类别 | 人数 | 角色 |
|:--:|:--:|------|
| 普通男性 | 8 | Luke/Toby/Julius/Chase/Gill/Owen/Calvin/Jin |
| 特殊男性 | 2 | **丰收之王**(神)/**魔法师** |
| 普通女性 | 8 | Luna/Renee/Phoebe/Kathy/Anissa/Candace/Maya/Selena |
| 特殊女性 | 2 | **丰收女神**(神)/**魔女** |

## 三、社交系统的独特设计

### 3.1 7心告白=关系锁定

```
社交进度:
2♥→第一份礼物
3♥→可节日约会
4♥→第二份礼物
5♥→浪漫约会
6♥→情书
7♥→告白(灯塔4PM)→关系锁定!
    ⚠️ 其他候补的好感度会下降
9♥→蓝羽毛求婚
```

> 告白="确定关系"——不是可逆转的——你需要选择一个人

### 3.2 蓝羽毛=登山仪式

- 蓝羽毛获取: 加农山顶/下午5-7PM(仅在Mira告知蓝鸟后)
- 不是"去商店买"——而是"爬上山顶在黄昏获取"——仪式感

### 3.3 特殊候补的不同规则

| 规则 | 普通候补 | 丰收之王/女神 |
|------|:--:|:--:|
| 同居 | ✓ | **不同居** |
| 帮农活 | ✓ | **不帮**(他们是神!) |
| 婚礼信物 | 婚戒 | 大地之坠/太阳之戒 |
| 子女 | ✓ | ✓(特殊事件) |

## 四、社交系统的压力梯度

| 阶段 | 压力 |
|------|:--:|
| 0-5♥ | 低(可随意送礼/探索) |
| 5♥(约会) | 中(需要管理约会时间) |
| **7♥(告白)** | **高**(关系锁定→不可逆转→其他候补好感下降) |
| 9♥(求婚) | 中(满足条件=可求婚) |
| 婚后 | 低(维护+触发子女事件) |

## 五、20人攻略量级的意义

| 维度 | 数值 |
|------|:--:|
| 总心色事件 | 20人×4事件=**80+独立事件** |
| 情敌事件 | 8对×3事件=**24事件** |
| 约会对话变体 | 20人×多个节日 |
| 配偶日常变体 | 20人各自独立 |
| 子女对话变体 | 4性格×2子女×不同配偶 |
| **总独特文本量** | **极高(估计数万字)** |
