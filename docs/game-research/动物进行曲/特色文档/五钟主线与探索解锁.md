# 五钟主线与探索解锁设计 - 动物进行曲(Harvest Moon: Animal Parade)

> 游戏版本: Harvest Moon: Animal Parade (Wii, 2008年)

## 一、概述

动物进行曲的五钟主线是牧场品类中**最有结构感的主线任务设计**。五个钟分布在地图各角落→每个钟有独立的任务链和NPC互动→完成顺序有逻辑依赖(绿钟需要蓝钟后的巨嘴鸟岛)→形成完整的探索解锁梯度。

## 二、五钟=五元素=五种任务设计

| 钟 | 任务类型 | 设计特色 |
|------|:--:|------|
| 红(火) | **收集+制作** | 矿洞探索→找材料→NPC修复→最直接的冒险感 |
| 黄(大地) | **农业考验** | 10个优质作物→**经营能力检查**→不是打怪是种田 |
| 蓝(水) | **解谜+制作** | 失忆的旋律→探索记忆→制作机械→智力和创造力 |
| 绿(风) | **魔法+跨岛探索** | 魔女变蛙→需要三样物品(其中木槿需解锁新岛)→跨区域任务 |
| 紫(心) | **氛围+时间触发** | 凌晨2AM鬼魂→许愿神社→打破常规时间→情感核心 |

## 三、探索解锁梯度

```
初始: 农场+哈莫尼卡镇+女神之泉
  ↓ 红钟
解锁: 加农矿洞下层(10层)
  ↓ 黄钟
解锁: 海滩南侧/月石
  ↓ 蓝钟
解锁: 水洞+巨嘴鸟岛(Pascal的船)
  ↓ 绿钟(需巨嘴鸟岛的木槿)
解锁: 赋格森林
  ↓ 紫钟
解锁: 教堂/许愿神社
  ↓ 五钟齐鸣+"英雄"称号
解锁: 加农矿山上层(45层)→丰收之王山顶
```

## 四、任务设计对探索的驱动

| 设计 | 效果 |
|------|------|
| 每钟分布在岛的不同方向 | 驱散→全岛探索 |
| 钟之间需前钟解锁的新区域 | 顺序无法跳过 |
| 黄钟=农业检查(10优质作物) | 确保经营进度跟上 |
| 绿钟=跨岛(木槿需巨嘴鸟岛) | 确认玩家已探索蓝钟解锁的内容 |
| 紫钟=凌晨2AM | 刻意打破正常游戏节奏→制造记忆点 |

## 五、设计的精妙之处

1. **任务类型混合**: 收集/制作/解谜/经营检查/时间触发——不是"五个收集任务"
2. **依赖链自然**: 绿钟木槿→巨嘴鸟→蓝钟解锁→顺序无法跳过→自然的任务排序
3. **经营检查(黄钟)**: 10个优质作物→在冒险任务中插入农业检查→确保玩家没有忽略经营
4. **凌晨2AM(紫钟)**: 四钟完成后→最后一个钟是情感核心+氛围转变→不是另一个冒险而是"鬼魂和许愿"
5. **"英雄"=经营定义**: 不是"打完所有钟就是英雄"→而是"通过经营证明自己"→牧场游戏的核心命题
