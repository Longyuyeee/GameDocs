## 4. 设计特点

1. **主线驱动探索**: 五钟分布地图各个角落→必须探索全岛→主线=地图引导
2. **加农矿山=双层**: 下层(10层/矿洞)+上层(45层/Boss)——双矿洞设计
3. **凌晨2AM事件**: 紫钟需要凌晨进入教堂——特殊时间触发→打破常规节奏
4. **NPC分布反映社交网络**: 裁缝店(Candace+Luna姐妹)/酒吧(Owen+Kathy+Selena)/诊所(Jin+Anissa)——NPC的居住地点反映关系
