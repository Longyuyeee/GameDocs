## 2. 烹饪系统

### 2.1 厨具

| 厨具 | 功能 | 代表食谱 |
|------|------|------|
| 锅(Pot) | 煮/炖/汤 | 炖菜/汤类 |
| 煎锅(Frying Pan) | 煎/炒 | 煎蛋/炒菜/咖喱面包 |
| 烤箱(Oven) | 烘焙 | 蛋糕/派/披萨/香蕉冰淇淋 |
| 搅拌机(Mixer) | 混合/饮品 | 果汁/鸡尾酒/苹果鸡尾酒 |
| 蒸锅(Steamer) | 蒸 | 蒸菜/布丁 |

### 2.2 料理用途

| 用途 | 说明 |
|------|------|
| 送礼 | 特定NPC最爱(如Luke=香蕉冰淇淋/咖喱面包) |
| 体力恢复 | 高品质料理恢复更多体力 |
| 任务需求 | 完美黄油/优质玉米粉(绿钟主线) |
| 节日参赛 | 料理节/丰收节 |
| 出售 | 高价值出货 |

## 5. 设计特点

1. **加工链完整**: 原料→中间品→成品——羊毛→毛线→布料→衣物(三级加工)
2. **烹饪=社交核心**: 许多NPC最爱是特定料理(而非简单原料)→驱动烹饪系统使用
3. **裁缝=功能性+送礼**: 衣物不仅可穿也可作为特定NPC的最爱礼物
4. **黄油=主线材料**: 完美黄油是绿钟任务必需品——加工品直接关联主线进度
