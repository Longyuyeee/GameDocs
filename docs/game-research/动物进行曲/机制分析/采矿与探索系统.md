# 采矿与探索系统 - 动物进行曲(Harvest Moon: Animal Parade)

> 游戏版本: Harvest Moon: Animal Parade (Wii, 2008年)
> 数据来源: harvestmoon.fandom.com, fogu.com, IGN

## 1. 战斗定位

**无战斗系统。** 动物进行曲是纯和平的牧场经营游戏。洞穴探索仅涉及采矿，无怪物/无HP/无武器。

## 6. 设计特点

1. **双层矿洞=主线双阶段**: 下层(10层/早期)→上层(45层/终局)——与五钟主线同步推进
2. **5PM截止=时间管理**: 登山顶需下午5点前到达——太晚=错过=等明天
3. **矿洞无敌人**: 纯和平→无战斗压力→聚焦于采矿效率
4. **宝石=社交货币**: 矿石不仅升级工具——特定宝石是特定NPC的最爱礼物(红宝石→丰收之王/紫水晶→Julius)
