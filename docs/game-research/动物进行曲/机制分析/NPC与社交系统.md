## 二、社交机制

### 2.1 好感度阶段与事件

| ♥数 | 事件 | 细节 |
|:--:|------|------|
| 2♥ | 第一份礼物 | 角色早6AM来访→送礼物→接受=好感 |
| 3♥ | **可约会**(特定节日) | 花节/萤火虫/月节/星夜 |
| 4♥ | 第二份礼物 | 早6AM→更好的礼物 |
| 5♥ | **浪漫约会** | 中午前对话→下午4PM见面 |
| 6♥ | **情书(Love Letter)** | 邮递到邮箱 |
| 7♥ | **告白(Confession)** | 中午前对话→下午4PM灯塔见面→**确定关系!** |
| 9♥ | **可求婚** | 蓝色羽毛(加农山顶5-7PM)→展示 |
| 婚后 | 配偶功能 | 做饭/帮农活/散步/生日/纪念日 |

### 2.2 7心告白=关系锁定

**关键**: 7心告白后→其他候补的心会下降。建议存档→看完所有告白→选择一人推进。

## 五、设计特点

1. **20位候补=系列最多**: 10男+10女——包括2位特殊神祇——空前的攻略选择量
2. **7心告白=关系分岔点**: 告白后其他人好感下降→"确定关系"的仪式感——增强社交决策重量
3. **8对情敌→8个情敌子女**: 情敌系统产生了额外的"下一代NPC"——社区真正在成长
4. **子女=完全成长→帮农活**: 不是停留在婴儿阶段的装饰→儿童可以帮做农活——子女有功能性
5. **丰收之王/女神可婚**: 攻略神祇——特殊规则(不同居/不帮农活/独特信物)——最高难度的社交成就
