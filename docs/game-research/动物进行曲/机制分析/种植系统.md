## 2. 肥料系统

| 肥料 | 效果 |
|------|------|
| 普通肥料 | 轻微提升作物品质 |
| 高级肥料 | 显著提升品质 |
| 加速肥料 | 缩短生长时间 |

## 3. 多农田管理

| 农田位置 | 适合种植 |
|------|------|
| 主农场(家旁) | 主力高价值作物 |
| 水车田 | 复收作物(减少往返) |
| 额外地块 | 季节性辅助 |

## 4. 设计特点

1. **多农田=需要规划**: 不同农田适合不同作物类型——复收作物放远处/高价值放近处
2. **优质作物=任务关键**: 多个主线任务需要特定优质作物(解锁大地的黄钟等)
3. **荞麦=秋季任务核心**: 解锁黄钟需要的10个优质荞麦之一
4. **冬季可种植**: 不像BTN完全停摆——有少量冬季专属作物
