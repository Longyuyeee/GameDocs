## 6. 设计特点

1. **6种产出动物**: 鸡/牛/羊/鸭/山羊/鸵鸟——Wii牧场中最丰富的动物阵容
2. **鸵鸟=终局动物**: 特殊解锁条件→最高价值产出→终局畜牧目标
3. **猪=功能性宠物**: 遛猪→找蘑菇——动物不仅是资源产出还有探索辅助功能
4. **动物赛跑=押注系统**: 类似BTN的赛马——可参赛或押注
