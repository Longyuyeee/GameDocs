# 物品与服装系统 - 美少女梦工厂3 梦幻妖精(Princess Maker 3)

> 游戏版本: Princess Maker 3: Fairy Tales Come True (PC, 1997/2017年)
> 数据来源: princessmaker.fandom.com, SteamSolo

## 1. 服装系统

| 服装类型 | 代表 | 效果 | 获取 |
|:--:|------|------|:--:|
| 日常服 | 各种款式 | 基础/无修正 | 购买 |
| 正式服 | 礼服/舞会装 | 魅力+/气品+(社交事件) | 购买 |
| 特殊服 | **暗黑服装(Dark Dress)** | **触发Imp事件**(度假时!) | 特定条件 |
| 季节服 | 夏装/冬装 | 适应季节 | 购买 |

## 2. 服装对心情的交互

| 服装 | 心情限制 |
|------|:--:|
| 特殊服装 | 骄傲(Sassy)心情→**拒绝穿!** |
| 华丽服装 | 需要一定气品→否则女儿不穿 |
| 暗黑服装 | 无限制(暗黑路线专用) |

## 3. 道具系统

| 类型 | 代表 | 效果 |
|:--:|------|------|
| 体力恢复 | 营养品/食物 | 体力+/体重+ |
| 压力恢复 | 玩具/书 | 压力-/感性+ |
| 属性书 | 各类书籍 | 对应属性+ |
| 礼物 | 花/宝石/香水 | 送女儿→信任+/魅力+ |
| 武器/防具 | — | **PM3无战斗系统→无武器装备** |

## 4. 饮食系统

| 食物 | 效果 | 副作用 |
|:--:|------|:--:|
| 普通餐 | 体力+ | 体重+ |
| 营养餐 | 体力++ | 体重++ |
| 减肥餐 | 体力— | **体重—/魅力+** |
| 甜点 | 压力—/体力+ | 体重++ |

## 5. 设计特点

1. **暗黑服装=打开隐藏路线的钥匙**: 不是纯装饰——穿特定服装+度假→Imp事件——服装与事件联动
2. **心情限制服装使用**: 骄傲的女儿拒绝特殊服装→女儿的性格影响可用的装备
3. **无战斗=无装备**: 相比PM2→PM3移除战斗→装备系统简化为服装+道具
4. **饮食=体重管理**: 吃太多→体重增加→减肥中心→循环——更真实的健康管理
