## 2. 竞争对手系统(PM3独有!)

每个课程有一个**竞争对手(NPC少女)**——她和你女儿在同一课程中竞争。竞争对手有自己的属性成长曲线——如果你女儿的属性被超过→可能在比赛中输给她。

| 竞争对手 | 对应课程 | 性格 |
|:--:|:--:|------|
| Chia | 学問 | 学霸型 |
| Ariel | 武术 | 运动型 |
| Chamie | 舞蹈 | 活泼型 |
| Haita | 烹饪 | 温暖型 |
| Haming | 音乐 | 艺术型 |
| Arkron | 绘画 | 奇妙型 |
| Natera | 礼仪 | 优雅型 |

## 3. 竞争对手的影响

| 影响 | 说明 |
|------|------|
| **比赛竞争** | 年度比赛→需要属性超过竞争对手→否则输 |
| **课程效率** | 属性超过竞争对手→课程效率提升 |
| **社交/友情** | 与竞争对手可发展友情→特殊事件 |
| **结局影响** | 部分结局需要与特定竞争对手的关系 |

## 5. 设计特点

1. **竞争对手=PM3最独特的社交维度**: PM2没有→火山的女儿简化(只有3领主)→PM3的竞争对手系统是独特的
2. **宫廷侍女=低薪高回报**: 4G/月是最低薪——但解锁王子邂逅+提升气品/魅力——长期投资
3. **矿工=最高薪但有限制**: 20G/月——但不能处于"漂亮"心情——心情系统限制打工
4. **减肥中心=魅力的捷径**: 5G→直接提升魅力——但减少体力(有代价)
