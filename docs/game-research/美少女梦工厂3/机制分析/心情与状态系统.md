# 心情与状态系统 - 美少女梦工厂3 梦幻妖精(Princess Maker 3)

> 游戏版本: Princess Maker 3: Fairy Tales Come True (PC, 1997/2017年)
> 数据来源: princessmaker.fandom.com, SteamSolo

## 1. 六大心情(Mood)——PM3独有系统

PM3最独特的创新是**心情系统**——女儿的属性组合会决定她的心情状态，而不只是被动的数值。不同心情下，课程/打工的效果不同。

| 心情 | 触发条件 | 关键效果 |
|:--:|------|------|
| **正常(Normal)** | 默认 | 最优状态——所有行动正常 |
| 骄傲(Sassy) | (气品+骄傲) > (态度+道德)×2 | 所有课程→态度下降; 拒绝多数特殊服装 |
| 叛逆(Rebellious) | 道德<300 且满足公式 | **不能做农场/家教/保姆/市场工作** |
| 漂亮(Pretty) | 魅力 > 道德×2 | **不能去矿工工作**; 社交事件触发+ |
| 贫穷(Poor) | 接近负债 | 限制效果 |
| **生病(Sick)** | **压力 > 体力** | **不能做任何事! 所有属性持续下降!** |

## 2. 心情系统的策略影响

| 心情 | 应对策略 |
|:--:|------|
| 骄傲(Sassy) | 降低气品/骄傲→或利用(提升态度的工作) |
| 叛逆(Rebellious) | 提升道德≥300→恢复正常→否则大量工作不可用 |
| 漂亮(Pretty) | 提升道德→或利用(专注非矿工路线) |
| 生病(Sick) | **紧急休息**→体力恢复>压力→恢复 |
| 贫穷(Poor) | 立即打工→增加收入 |

## 3. 压力(Stress)管理

| 来源 | 压力增加 |
|:--:|:--:|
| 课程 | +中(每课固定) |
| 打工 | +低-中(因打工类型而异) |
| 休息 | **减少**(主动恢复) |
| 度假 | **大量减少+信任增加** |
| 生病 | 自动触发→恢复期不能安排活动 |

## 4. 体重(Weight)系统

| 因素 | 效果 |
|:--:|------|
| 饮食 | 增加体重 |
| 运动(武术/舞蹈) | 减少体重 |
| 减肥中心(课程) | **主动减重**(消耗金币/减少体力) |
| 体重过高 | 魅力下降/部分结局受影响 |
| 体重过低 | 体力下降 |

## 5. 设计特点

1. **心情=属性→行为反馈**: PM2的属性是纯数值——PM3的心情让属性变得"可见"——女儿会因为属性高而变得骄傲
2. **生病=最严重惩罚**: 不只是"休息一回合"——而是所有属性持续下降——必须立即处理
3. **体重=第三健康维度**: PM2只有体力+压力——PM3增加了体重——更真实的"健康管理"
4. **心情系统未被火山的女儿继承**: 火山的女儿简化了状态管理——没有心情维度——PM3独特遗产
