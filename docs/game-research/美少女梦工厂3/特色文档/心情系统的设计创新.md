# 心情系统(Mood)的设计创新 - 美少女梦工厂3 梦幻妖精(Princess Maker 3)

> 游戏版本: Princess Maker 3: Fairy Tales Come True (PC, 1997/2017年)

## 一、概述

PM3的**心情系统(Mood)**是养成品类中最独特的创新之一。与PM2(纯数值)和火山的女儿(简化状态)不同，PM3的女儿会根据属性组合进入6种不同的心情状态——每种心情影响可用的行动和属性收益。这相当于给养成对象赋予了一个"性格引擎"。

## 二、心情=属性→行为反馈

| 心情 | 属性条件 | 行为后果 |
|:--:|------|------|
| 骄傲(Sassy) | (气品+骄傲) > (态度+道德)×2 | "我太优秀了不想穿普通衣服"→**拒绝特殊服装** |
| 叛逆(Rebellious) | 道德<300 | "我不想听你的"→**大量打工不可用** |
| 漂亮(Pretty) | 魅力 > 道德×2 | "我太漂亮了不能干粗活"→**矿工不可用** |
| 生病(Sick) | 压力>体力 | "我动不了"→**全部行动不可用** |
| 贫穷(Poor) | 接近负债 | — |
| 正常(Normal) | 默认 | 全部可用 |

## 三、心情=隐藏的"性格"

| 心情 | 女儿的性格画像 |
|:--:|------|
| 正常 | 平衡/健康/听话 |
| 骄傲 | "公主病"——被宠坏了/认为自己比别人好 |
| 叛逆 | "青春期叛逆"——不听话/拒绝权威 |
| 漂亮 | "太在意自己的外表"——看不起粗活 |
| 生病 | "过度劳累"——需要休息 |

## 四、为什么心情系统是创新

| 传统养成 | PM3心情系统 |
|------|------|
| 属性=数字 | 属性→心情→**女儿的表情/对话/行为改变** |
| 玩家看数字做决策 | 玩家看**女儿的状态**做决策——更直观 |
| "智力80"=抽象 | "女儿现在很骄傲所以不穿这件衣服"=**具体的故事** |

## 五、心情系统为什么没被继承

| 原因 | 说明 |
|------|------|
| 复杂度高 | 6心情×多种行动限制→玩家需要同时追踪心情+属性+压力 |
| 开发成本 | 每种心情需要不同的对话/动画/行为——成本是纯数值系统的数倍 |
| 火山的女儿简化 | 火山的女儿只有"正常/生病"两种状态——简化了心情维度 |
| PM4/PM5 | 部分继承但更简化 |

## 六、设计分析

1. **心情=养成游戏的"性格模拟"**: PM2的女儿是"空白画布"→PM3的女儿有独立性格→她会因为你的培养方式而"变坏"
2. **骄傲=过好的副作用**: 气品太高→骄傲心情→拒绝服装→需要额外管理——"培养得太好也有代价"
3. **漂亮心情=性别化的策略**: 魅力太高→"太在意外表"→限制粗活——性别化的设计在1997年的前瞻性
4. **未被继承的遗产**: 至今没有任何养成游戏完整复制PM3的心情系统——是品类中"被遗忘的宝石"
