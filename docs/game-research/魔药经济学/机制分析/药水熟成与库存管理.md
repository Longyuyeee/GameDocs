# 药水熟成与库存管理 - 魔药经济学(Potionomics)

> 游戏版本: Potionomics (2022-2024年)
> 数据来源: TheGamer, PSNProfiles

## 1. 熟成(Aging)系统

| 参数 | 说明 |
|------|------|
| 设备 | 熟成桶(Barrel) |
| 机制 | 药水放入→等待天数→取出→**价格提升** |
| 时间 | 数天(取决于药水品质+桶等级) |
| 策略 | 高价值药水→放入熟成→比赛前取出→最大利润 |

## 2. 库存管理

| 管理维度 | 策略 |
|------|------|
| 原料库存 | 常用原料保持充足→避免酿造中断 |
| 药水库存 | 高需求药水常备→快速响应顾客 |
| 熟成桶规划 | 比赛前→高价值药水入桶→比赛日取出 |
| 自动售货机 | 过量库存→放入售货机(15%溢价)→清库存 |

## 3. 生产链

```
原料采购(Muktuk/采集)
  ↓
酿造(匹配魔素→品质)
  ↓ 分叉:
  ├→ 立即出售(卡牌议价)→高利润(需Deck)
  ├→ 熟成桶(等待)→更高价格→之后出售
  └→ 自动售货机(15%溢价)→即时有保证
```

## 4. 设计特点

1. **熟成=期货投资**: 现在的药水→几天后更值钱——类似守墓人的炼金
2. **库存=生产规划**: 不是"有多少卖多少"——需根据比赛日+市场需求预判
3. **三出口策略**: 议价出售(最大利润)/熟成(延时增值)/售货机(清库存)——三条路径各有适用场景
