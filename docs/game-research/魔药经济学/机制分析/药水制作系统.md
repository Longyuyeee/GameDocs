# 药水制作系统 - 魔药经济学(Potionomics)

> 游戏版本: Potionomics (PC/Switch/PS5, 2022-2024年)
> 数据来源: PSNProfiles, TheGamer

## 2. 药水品质等级

| 品质 | 条件 | 价格倍率 |
|:--:|------|:--:|
| Minor | 基础匹配 | ×1.0 |
| Common | 较好匹配 | ×1.3 |
| Greater | 良好匹配 | ×1.6 |
| Grand | 优秀匹配 | ×2.0 |
| Superior | 完美匹配 | ×2.5 |
| **Masterwork** | **终极匹配**(Day41+才可靠) | **×3.0+** |

## 3. 特性(Traits)

| 特性 | 影响 |
|:--:|------|
| 味道(Taste) | 价格±5% |
| 触感(Touch) | 价格±5% |
| 嗅觉(Smell) | 价格±5% |
| 视觉(Sight) | 价格±5% |
| 听觉(Sound) | 价格±5% |

## 4. 熟成(Aging)

- 药水放入木桶→熟成→**提升价格**
- 时间+魔素匹配→更高品质

## 5. 设计特点

1. **五行魔素+品质=策略深度**: 不是"选配方→制作"——需匹配魔素比例→品质决定基础价格
2. **特性=微调**: 5个维度×±5%→追求完美特性搭配
3. **熟成=延时投资**: 不是立即出售→放入桶中等待→类似守墓人的炼金
4. **大锅升级=更多魔素**: 更高级的大锅→更多原料→更高等品质
