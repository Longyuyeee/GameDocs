# 卡牌议价系统 - 魔药经济学(Potionomics)

> 游戏版本: Potionomics (PC/Switch/PS5, 2022年, Masterwork Edition 2024年)
> 数据来源: TheGamer, PSNProfiles, GamerJournalist

## 1. 概述

魔药经济学将**议价(Haggling)**做成**卡牌对战**——经营系统中最创新的讨价还价机制。不是Recettear的"出价→顾客还价→你再出价"的数字博弈，而是**20张卡的Deck-Building战斗**。

## 2. 议价=卡牌战斗

| 卡牌游戏要素 | 魔药经济学中的实现 |
|------|------|
| 生命值 | **兴趣(Interest)**——顾客的购买意愿→降到0=离开 |
| 法力 | **耐心(Patience)**——出牌消耗→0=顾客离开 |
| Boss攻击 | 顾客反击→增加Sylvia的**压力(Stress)**→100%=晕倒 |
| 胜利 | 兴趣条填满→**以当前出价成交** |
| 失败 | 兴趣0或耐心0→顾客空手离开 |

## 3. 卡牌类型

| 类型 | 效果 | 使用时机 |
|:--:|------|:--:|
| 开场牌(Opener) | 首打出牌→额外效果 | 回合开始 |
| 主力牌(Work Horse) | 主要兴趣提升 | 中段 |
| 终结牌(Closer) | **结束交易→额外奖励** | 最后一击 |
| 屏障牌(Barrier) | 护盾/减少压力 | 防御时刻 |
| 状态牌(Status) | 持续效果 | 早期(持续多回合) |

## 4. NPC卡牌特色

| NPC | 特色 | 提供卡牌风格 |
|------|------|------|
| MukTuk | 铁匠 | **暴力兴趣牌**(大量伤害) |
| Mint | 植物学家 | 防御牌/护盾 |
| Saffron | 女巫 | 减压牌 |
| Baptiste | 商人 | 增加耐心牌 |
| Quinn | 发明家 | 多回合持续兴趣 |

## 5. 设计特点

1. **议价=Deck-Building**: 不是"选择对话选项"——而是"构建卡组+出牌策略"——赋予议价策略深度
2. **耐心=时间限制**: 每顾客的耐心有限→不能无限出牌→需效率策略
3. **压力=反击系统**: 顾客会反击→增加你的压力→需要防御/减压牌
4. **卡牌来源=社交**: 与NPC交友→获得他们的特色卡牌——社交驱动Deck成长
