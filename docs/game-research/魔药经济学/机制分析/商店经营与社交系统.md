# 商店经营与社交系统 - 魔药经济学(Potionomics)

> 游戏版本: Potionomics (PC/Switch/PS5, 2022-2024年)
> 数据来源: PSNProfiles, TheGamer

## 1. 经营机制

| 要素 | 说明 |
|------|------|
| 货架 | 摆放药水→出售 |
| 装饰 | 提升价格+药水品质 |
| 自动售货机(Quinn) | 无需议价→15%溢价自动出售 |
| 每日时间 | **6个时间槽**(酿造/访友/出售均消耗) |

## 2. 社交系统

| NPC | 职务 | 提供 |
|------|:--:|------|
| MukTuk | 铁匠 | 货架升级/大锅升级/暴力卡牌 |
| Saffron | 女巫 | 减压卡牌 |
| Mint | 植物学家 | 防御卡牌 |
| Baptiste | 商人 | 耐心卡牌 |
| Quinn | 发明家 | 自动售货机/持续卡牌 |
| Roxanne | 竞争对手 | 第10天比赛 |
| Corsac | 竞争对手 | 第20天比赛 |
| Boss Finn | 最终对手 | 第30天比赛+可攻略(Masterwork) |

## 3. 10位可攻略角色

- 提升好感(Rank 1-10)→**解锁新卡牌+更好服务**
- Rank 10→需要特定药水→最终事件
- Masterwork Edition: **"Free Love"/多元恋爱**(同时约会全部/无嫉妒)

## 4. 每10天比赛

| 日期 | 对手 | 所需药水 |
|:--:|------|------|
| Day 10 | Roxanne | 生命药水/魔力药水/火补剂 |
| Day 20 | Corsac | 冰补剂/视力增强剂/速度药水 |
| Day 30 | Boss Finn | 解毒剂/耐力药水/雷补剂 |
| Day 40 | Anubia | 沉默解药/耐性药水/洞察增强剂 |
| Day 50 | **Robin(最终)** | 最难——零压力进入! |

## 5. 设计特点

1. **卡牌议价=核心创新**: 经营+卡牌Deck-Building→品类中最独特的议价机制
2. **社交=卡牌来源**: 交友→获得卡牌→更强的议价Deck——社交驱动战斗成长
3. **每10天比赛=进度检查**: 不是"无限天经营"——而是定期Boss战——压力+目标
4. **多元恋爱=包容性**: Masterwork Edition加入——同时约会全部角色无惩罚
