# 经营与Deck-Building的融合设计 - 魔药经济学(Potionomics)

> 游戏版本: Potionomics (2022-2024年)

## 一、概述

魔药经济学将两类看似无关的游戏类型——**商店经营**和**卡牌Deck-Building**——通过"议价"这个共同点融合在一起。议价既是经营的核心(决定售价/利润)也是卡牌战斗的载体(Deck策略)。

## 二、融合机制

```
经营面:
  药水制作(魔素匹配→品质等级→基础价格)
  顾客管理(不同顾客→不同预算→不同耐心)
  设备升级(大锅/货架/装饰→更强制作能力)
    ↓ 交汇点: 议价
卡牌面:
  Deck构建(20张卡/5种类型/多NPC卡牌来源)
  回合制博弈(兴趣/耐心/压力三维资源)
  NPC友谊=卡牌解锁(社交驱动Deck成长)
```

## 三、与纯卡牌游戏的关键差异

| 维度 | 纯CCG(杀戮尖塔等) | 魔药经济学 |
|------|:--:|:--:|
| Deck目标 | 杀死敌人 | **最大化成交价(利润)** |
| Deck成长 | 战斗中选卡 | **社交(NPC好感→解锁新卡)** |
| 战斗结果 | 赢/死 | **成交价(利润高低)** |
| 战前准备 | 无 | **制作药水(品质决定基础价→议价起点)** |
| 失败后果 | 重来 | 顾客空手离开→损失利润→金币压力 |

## 四、设计分析

1. **议价=天然的交汇点**: "谈判"既是经营行为也是策略博弈——卡牌化是最自然的进化
2. **社交=Deck打造**: 交友不是为了"看剧情"而是为了"解锁更好的议价卡牌"——赋予社交功利目的
3. **药水品质=议价起点**: 制作与Deck不是独立的——品质越高→议价空间越大→Deck越能发挥
4. **比赛=定期Deck检查**: 每10天的比赛是Deck强度的硬检查——确保玩家持续优化
