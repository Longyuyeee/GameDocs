## 二、弓(远程物理)伤害公式

### 2.1 基础伤害公式(估算)

```
物理伤害 = (ATK × 技能倍率) - (敌方DEF × 0.5)
```

### 2.2 弓特殊机制——头部判定(Headshot)

弓具有**弱点瞄准机制**，命中敌人特定部位产生额外伤害：

| 命中部位 | 伤害倍率 | 说明 |
|:-------:|:-------:|:----|
| 身体 | ×1.0 | 普通命中 |
| **头部(Headshot)** | **×1.5-2.0** | 额外伤害，部分敌人可打断行动 |
| 弱点部位 | ×2.0-3.0 | 部分BOSS特定部位 |

### 2.3 弓属性推荐

| 弓属性 | 适用场景 |
|:-----:|:--------|
| 物理高伤弓 | 通用战斗 |
| 属性弓(火/水/风/地) | 克制特定属性敌人 |
| 暴击弓 | 高DEX配装 |

## 三、符咒(Talisman)自动追踪机制

符咒是本作的远程魔法武器，操作简化但策略深度不减：

| 机制 | 说明 |
|:---|:----|
| **自动追踪** | 魔法弹自动追踪附近敌人，无需精确瞄准 |
| **充能攻击(Charge Attack)** | 充能后同时发射多枚符咒，覆盖范围目标 |
| **终极攻击(Ultimate Attack)** | 4连击后发动高伤终结技 |
| **弱点放大(Weakness Magnifier)** | 利用属性克制时伤害大幅提升(可继承至其他武器) |
| **魔防穿透(M.DEF Piercer)** | 部分无视敌人魔法防御(可继承至其他武器) |
| **回避反击(Evasive Attack)** | 闪避后接续反击攻击 |

## 五、即时闪避(Instant Dodge)与子弹时间

### 5.1 即时闪避机制

| 项目 | 说明 |
|:---|:----|
| **触发方式** | 敌人攻击命中前瞬间按闪避键 |
| **子弹时间效果** | 成功闪避后进入短暂慢动作(约1-2秒) |
| **RP消耗** | 每次闪避消耗少量RP |
| **无敌帧** | 闪避动作期间全程无敌 |
| **反击窗口** | 子弹时间内攻击力提升(约×1.2-1.5) |

### 5.2 子弹时间持续时间

| 情况 | 持续时间(约) |
|:---|:----------:|
| 普通敌人攻击闪避 | 1.0-1.5秒 |
| BOSS攻击闪避 | 1.5-2.0秒 |
| 多段攻击最后一段闪避 | 1.0秒 |
| 连续闪避 | 第二次开始缩短至0.5-0.8秒 |

    光 ←→ 暗
