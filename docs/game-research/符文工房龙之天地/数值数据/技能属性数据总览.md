# 技能属性数据总览 - 符文工房 龙之天地

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

## 一、角色基础属性

| 属性 | 英文 | 效果 |
|:---:|:---:|------|
| **HP** | Hit Points | 生命值，归零则战斗不能 |
| **RP** | Rune Points | 符文点，使用神器技能消耗 |
| **STR** | Strength | 力量，影响物理攻击力 |
| **VIT** | Vitality | 体力，影响物理防御力 |
| **INT** | Intelligence | 智力，影响魔法攻击力(M.ATK) |
| **MND** | Mind | 精神，影响魔法防御力(M.DEF) |
| **DEX** | Dexterity | 灵巧，影响暴击率 |

## 二、技能树系统

本作共有**17个技能树**，分为三大类：武器技能、神器技能、实用技能。

### 2.1 武器技能树(Weapon Skill Trees)

| 技能树 | 对应武器 | 核心提升方向 |
|:-----:|:--------:|:----------:|
| **短剑(Sword)** | 短剑/长剑 | 物理攻击力、连击伤害 |
| **双刃(Dual Blades)** | 双刃武器 | 暴击率、攻速 |
| **弓(Bow)** | 弓 | 远程伤害、头部射击倍率 |
| **符咒(Talisman)** | 符咒 | 魔法攻击力、自动追踪、充能强化 |

### 2.2 神器技能树(Sacred Treasure Skill Trees)

| 技能树 | 关联神器 | 核心提升方向 |
|:-----:|:--------:|:----------:|
| **鼓之舞(Drum Dance)** | 鼓 | 地属性攻击、范围眩晕、HP恢复 |
| **剑之舞(Sword Dance)** | 剑 | 火属性攻击、采种等级 |
| **扇之舞(Fan Dance)** | 扇 | 风属性攻击、批量收获范围 |
| **伞之舞(Parasol Dance)** | 阳伞 | 水属性攻击、灌溉范围 |
| **鬼面之舞(Mask Dance)** | 鬼面 | 暗属性攻击、破石能力 |
| **锡杖之舞(Staff Dance)** | 锡杖 | 光属性攻击、光之蝶数量 |

### 2.3 实用技能树(Practical Abilities)

| 技能树 | 解锁条件 | 核心技能 |
|:------:|:--------:|:--------:|
| **社交活动(Social Activities)** | 初始可用 | Skilled Merchant(高价售出)、Bargain Hunter(低价买入) |
| **自我提升(Self-Improvement)** | 初始可用 | Max RP Up、Skilled Miner(额外采矿)、Skilled Gatherer(额外木材) |
| **料理(Cooking)** | 初期 | Skilled Cook(料理等级+1)、Materialistic(返还材料)、Max HP Up |
| **锻造(Forging)** | "铁匠风波2"任务后 | STR Up、Max HP Up、Penny-Pincher(减费)、Materialistic |
| **建造/农业(Construction/Farming)** | "村长之道"任务后 | Max RP Up、DEX Up、Penny-Pincher |
| **化学(Chemistry)** | "冬之里复兴2"任务后 | Max RP Up、Skilled Healer(回复量提升)、Materialistic |

| 技能 | 所属技能树 | 效果 |
|:---:|:---------:|:----|

| **Weakness Magnifier** | 符咒(可继承) | 克制属性伤害加成 |
| **M.DEF Piercer** | 符咒(可继承) | 部分无视敌方魔法防御 |

## 三、Azuma特有技能

本作相较传统RF新增的技能方向：

| 特有机制 | 说明 |
|:--------|:----|
| **神器熟练度** | 每件神器独立经验系统，使用即成长 |
| **牵绊之舞(Dance of Bonds)** | 用于提升NPC好感、驯服怪物、激活存档点 |
| **村庄建造** | 通过大地手杖进行自由建造/布局 |
| **村民委派** | 管理村民工作岗位的技能类系统 |

## 四、故乡点数(Hometown Points)与村庄等级

### 4.1 核心机制

**故乡点数(Hometown Points)** 是每个村庄独立的发展货币，通过完成各类任务获得：
- 村长委托(Village Chief Quests)
- 村民请求
- 主线推进
- 参与节日

### 4.2 村庄等级阈值

| 村庄等级 | 所需故乡点数(约) | 核心解锁内容 |
|:-------:|:--------------:|:-----------|
| **Lv.1** | 初始 | 基础建筑、人口上限4人 |
| **Lv.2** | 500 | 新建筑解锁、人口+2 |
| **Lv.3** | 1,200 | 商店解锁、人口+2 |
| **Lv.4** | 2,500 | 装饰品解锁、人口+2 |
| **Lv.5** | 5,000 | **大型畜棚解锁**、人口+2、苗圃可购黄金种子 |
| **Lv.6** | 8,000 | 高级建筑解锁、人口+2 |
| **Lv.7** | 12,000 | 终极装饰解锁、人口+2 |
| **Lv.8+** | 20,000+ | 全内容解锁、最高人口 |

> **注**：具体阈值可能因版本更新有所调整，以上为参考数值范围。

### 4.3 村庄等级收益表

| 等级收益 | 说明 |
|:--------|:----|
| **人口上限提升** | 每升1-2级增加人口上限 |
| **新建筑配方** | 解锁更多可建造建筑 |
| **商店功能** | 新商店/新商品解锁 |
| **黄金种子** | Lv.5后可在苗圃购买 |
| **装饰品** | 解锁更多村庄装饰(提升风景评分) |
| **村民特质** | 高级村民带有更好的初始特质 |
| **自动招募** | 达到一定等级后，每天睡眠时有最多3名村民自动加入 |

### 4.4 村民满意度

| 影响因素 | 正向 | 负向 |
|:--------|:----|:----|
| **住房** | 足够住房+ | 住房不足--(最大原因) |
| **食物供应** | 食品店运营+ | 食物短缺- |
| **工资支付** | 每日支付+ | 资金不足-- |
| **风景评分** | 高风景评分+ | 低风景评分- |
| **商店运营** | 店铺正常营业+ | 无人店铺- |
| **节日参与** | 举办节日+ | — |

> 满意度过低会导致**村民离开村庄**——但拥有"Hometown Proud"特质的村民除外。
