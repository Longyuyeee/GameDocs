# 四季节村庄与村民委派系统
![四季节村庄](https://runefactory-gaz.com/img/system/ss_village.jpg)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

## 核心创新：阿祖玛的村庄哲学

《符文工房 龙之天地》最激进的系统革新，是用**四个永久季节村庄**彻底取代了系列延续20年的四季轮换制。这不是一个增量改进，而是对整个RF农业循环逻辑的重新设计——从一个"随时间适应环境"的被动模式，转变为一个"选择在哪里种植"的主动战略模式。

---

## 一、四季节村庄：打破季节的枷锁

### 1.1 村庄概览

在阿祖玛(東の国)大地上，春、夏、秋、冬四个村庄因龙神的力量而**永远停留在各自季节**。玩家不是跟随时间推移被动改变种植策略，而是主动选择在哪个村庄发展。

| 村庄 | 日文名 | 核心神祇 | 气候特征 | 适宜作物 | 经济定位 |
|:---:|:------:|:--------:|:--------:|:--------:|:--------:|
| **春之里** | 春の里 | Ulalaka(乌拉拉卡) | 樱花飘落的温暖春天 | 芜菁、土豆、卷心菜、草莓 | 农业开局、新手友好 |
| **夏之里** | 夏の里 | Matsuri(祭) | 阳光海滩的热带夏天 | 黄瓜、番茄、玉米、粉红蜜瓜、菠萝 | **商业中心**、交易枢纽 |
| **秋之里** | 秋の里 | Kurama(鞍马) | 红叶遍野的丰收之秋 | 茄子、南瓜、**水稻/糯米** | **农业主力**、最高收益 |
| **冬之里** | 冬の里 | Fubuki(吹雪) | 银装素裹的宁静之冬 | 胡萝卜、菠菜、韭葱、白萝卜、柑橘 | 稳扎稳打的辅助村 |

### 1.2 村庄间移动

| 移动方式 | 解锁条件 | 说明 |
|:--------:|:--------:|:----|
| **徒步/跑图** | 初期 | 通过大地图区域连接移动 |
| **龙之雕像传送** | 激活雕像后 | 快速旅行，瞬间到达已激活村庄 |
| **白龙飞行** | 中期主线 | 乘白龙在村庄间高速移动 |

### 1.3 改变的核心：种植策略的质变

传统RF的季节系统：
```
春种草莓 → 夏种菠萝 → 秋种红薯 → 冬种白萝卜 → 下一年循环
```
——所有计划都是围绕"这个月该种什么"展开的被动适应。

Azuma的四季村庄系统：
```
春之里=草莓(全年可种)  |  夏之里=蜜瓜(全年可种)
秋之里=水稻(全年可种)  |  冬之里=胡萝卜(全年可种)
```
——玩家从"时间管理"转向**"空间管理"**：不是决定什么时候种，而是决定**在哪块地上种什么**。

这一转变的意义在于：
1. **取消季节错过焦虑**：不再有"错过了春季就没法种草莓"的压力
2. **全年稳定产出**：每个村庄都是一个小型经济生态系统
3. **并行发展**：可同时在四个村庄开展农业活动
4. **专业化战略**：可以将一个村打造成水稻专精区，另一个村发展为商业中心

## 二、村庄建设与自定义

### 2.1 自由建造系统

本作采用类似《动物森友会》的**自由放置建造模式**，通过大地手杖(Terra Tiller)进入建造模式：

#### 可建造内容

| 类别 | 内容 |
|:---:|:----|
| **田地** | 普通田地、水田、不同地形砖块 |
| **道路** | 石板路、土路、装饰性路径 |
| **建筑** | 住宅、商店、工坊、畜棚、仓库 |
| **装饰** | 灯笼、雕像、花坛、栅栏、树木 |
| **设施** | 出货棚、储物箱、烹饪台 |

#### 建造流程

```
大地手杖 → 建造模式 → 选择地块 → 铺设地形 → 放置建筑/装饰 → 布置功能设施
```

> **建造材料**：各类地形砖块通过Woolby制作，建筑需要木材和材料石。

### 2.2 村庄等级系统

每个村庄独立拥有等级系统，通过积累**故乡点数(Hometown Points)**提升。

| 等级 | 所需故乡点数(约) | 解锁内容 |
|:---:|:--------------:|:--------|
| Lv.1 | 初始 | 基础建筑、人口上限4人 |
| Lv.2 | 500 | 新建筑、人口+2 |
| Lv.3 | 1,200 | 商店解锁、人口+2 |
| Lv.4 | 2,500 | 装饰品解锁、人口+2 |
| Lv.5 | 5,000 | 大型畜棚、黄金种子购买权、人口+2 |
| Lv.6 | 8,000 | 高级建筑、人口+2 |
| Lv.7 | 12,000 | 终极装饰、人口+2 |
| Lv.8+ | 20,000+ | 全内容解锁 |

#### 故乡点数获取方式

| 途径 | 点数 | 频率 |
|:---|:---:|:----|
| 村长委托(Village Chief Quests) | 50-200 | 日常/周常 |
| 村民请求 | 20-100 | 随机 |
| 主线推进 | 100-500 | 阶段性 |
| 节日参与/优胜 | 50-200 | 节日时 |
| 村庄发展里程碑 | 100-300 | 阶段性 |

## 三、村民委派系统

### 3.1 村民招募

| 方式 | 说明 |
|:---:|:----|
| **主线解锁** | 通过主线推进解放被魔物束缚的村民 |
| **自动招募** | 达到一定条件后睡眠时有最多3名村民自动加入 |
| **流浪者** | 野外发现的流浪者可以邀请定居 |

### 3.2 委派岗位

| 岗位 | 效果 | 效率影响因素 |
|:---:|:----|:-----------:|
| **农业(Farming)** | 自动播种、浇水、收获 | Crop Whisperer特质、牵绊等级 |
| **商店/工业(Shop/Industry)** | 店铺经营，被动日收入 | 对应职业特质(美食家/木工/化学家) |
| **畜棚(Monster Care)** | 生产类怪物产出翻倍 | Animal Lover特质 |
| **资源收集(Gathering)** | 自动采集木材/矿石 | — |

### 3.3 委派机制详解

#### 农业委派

农业委派是游戏后期自动化的核心：

| 操作 | 委派行为 | 玩家仍需要做的 |
|:---:|:--------|:-------------:|
| 播种 | 村民自动使用库存种子播种 | 补充种子库存 |
| 浇水 | 村民每天自动浇水 | 检查是否有遗漏地块 |
| 收获 | 村民自动收获成熟作物 | 使用鼓之舞催熟、剑之舞采种 |

> **切换开关**：在Options菜单中可以独立开关村民的播种、浇水、收获行为（1.0.5版本后支持）。

#### 商店委派

| 商店类型 | 所需特质 | 收入模式 |
|:--------:|:--------:|:--------:|
| 锻造店 | Gourmand(美食家) | 每日自动金币收入 |
| 木工店 | Hobby Woodworker(木工爱好者) | 每日自动金币收入 |
| 化学店 | Amateur Chemist(化学爱好者) | 每日自动金币收入 |
| 食品店 | Gourmand(美食家) | 每日自动金币收入 |

### 3.4 村民特质系统

每个村民拥有**正面和负面特质**，影响其工作效率和村庄生活：

| 正面特质 | 效果 |
|:--------:|:----|
| **Crop Whisperer(作物低语者)** | 农业工作效率提升 |
| **Animal Lover(动物爱好者)** | 畜棚产出提升 |
| **Hometown Proud(家乡自豪)** | 不会因满意度低离开 |
| **Gourmand(美食家)** | 食品/锻造店经营加成 |
| **Hobby Woodworker(木工爱好者)** | 木工店经营加成 |
| **Amateur Chemist(化学爱好者)** | 化学店经营加成 |

| 负面特质 | 效果 |
|:--------:|:----|
| **Big Eater(大食客)** | 村庄支出增加 |
| **Slacker(懒鬼)** | 工作效率降低 |
| **Timid(胆小)** | 战斗相关委派效率降低 |

> **村民管理**：可以通过村民菜单驱逐表现不佳的村民。

### 3.5 牵绊等级对委派效率的影响

| 牵绊等级 | 委派效率修正 |
|:-------:|:----------:|
| Lv.1-2 | 基础效率(100%) |
| Lv.3-4 | 效率提升(约110%) |
| Lv.5-6 | 效率提升(约125%) |
| Lv.7-8 | 效率提升(约140%) |
| Lv.9-10 | 最大效率(约150%) |

## 四、村镇满意度系统

### 4.1 满意度影响因素

| 因素 | 影响幅度 | 说明 |
|:---:|:-------:|:----|
| **住房充足** | 大幅正向 | **最关键因素**——基础房只能住4人 |
| **工资支付** | 大幅正向 | 每日从出货收益中自动扣除 |
| **食物供应** | 中幅正向 | 食品店运营可维持 |
| **风景评分(Scenery Score)** | 中幅正向 | 通过装饰品提升 |
| **商店运营** | 小幅正向 | 所有店铺有村民值守 |
| **住房不足** | **大幅负向** | 满意度快速下降 |
| **资金不足** | 大幅负向 | 无法支付工资时 |
| **无人商店** | 小幅负向 | 长期无人值守 |

### 4.2 村民离开机制

当村庄满意度持续低迷时，村民会选择离开：
- **唯一例外**：拥有"Hometown Proud"特质的村民绝不会离开
- **离开后果**：劳动力减少、收入下降，但支出也减少
- **应对**：优先确保住房和工资支付

### 4.3 满意度管理策略

| 阶段 | 策略重点 |
|:---:|:--------|
| **初期** | 控制人口不超过住房上限 |
| **中期** | 每个村庄建造至少2栋住宅 |
| **后期** | 通过装饰品最大化风景评分 |
| **紧急** | 检查Woolby的夜间报告，针对问题快速解决 |

## 五、与前代的对比分析

### VS RF4 指令系统(Order System)

| 对比项 | RF4 指令系统 | RF 龙之天地 村庄系统 |
|:-----:|:-----------:|:------------------:|
| **核心货币** | PP(王子点数) | 故乡点数(Hometown Points) |
| **发展方向** | 玩家单方决定 | 村庄等级+玩家选择 |
| **NPC参与** | NPC不参与生产 | **NPC全流程参与**(委派) |
| **物理空间** | 固定农场布局 | **完全自由建造** |
| **节奏** | 等待季节轮换 | **四个季节并行发展** |
| **失败代价** | 无 | **村民可能离开** |

### VS RF3 公告板系统

| 对比项 | RF3 公告板 | RF 龙之天地 委派系统 |
|:-----:|:---------:|:------------------:|
| **任务来源** | 公告板固定刷新 | 村长委托 + 村民请求 |
| **自动化** | 无 | **全流程委派自动化** |
| **村庄发展** | 线性小镇评价 | 多维度村庄等级 |
| **NPC深度** | 简单委托 | 特质系统+满意度管理 |

### 设计分析

龙之天地的村庄系统代表了RF系列在"农场经营"维度上的重大突破：

1. **从时间管理到空间管理**：四个永久季节村庄将策略重心从"何时做"转向"何地做"
2. **从个人到社群**：委派系统将NPC从"装饰性居民"转变为"生产参与者"，村庄真正有了社区感
3. **从固定到自由**：完全自由的建造模式允许每个玩家的村庄独一无二
4. **有代价的失败**：村民可能离开的设定让村庄管理有了真正的紧迫感——这是系列首次让"经营不善"产生实质后果
