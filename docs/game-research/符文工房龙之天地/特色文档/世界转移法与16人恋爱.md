# 世界转移法(World Transfer)与16人恋爱系统
![牵绊](https://runefactory-gaz.com/img/system/ss_bonds.jpg)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

## 概述

《符文工房 龙之天地》在恋爱系统上做出了系列史上最大胆的革新：**16位恋爱候补**（系列最多）、**首个支持同性结婚的主线作品**，以及最关键的系统创新——**世界转移法(World Transfer)**——允许玩家在**同一个存档文件中体验全部恋爱路线**，而无需反复开新档。

本作是**RF系列正传中明确支持同性结婚的作品**（RF5的后继）。无论玩家选择昴(男)还是卡古亚(女)，都可以追求任意性别的恋爱候补：
- 昴可以追求男性或女性候补
- 卡古亚也可以追求男性或女性候补
- 选择不同主人公仅影响"未被选中的主人公"（卡古亚或昴）成为可攻略对象

## 二、牵绊等级系统(Bonds Level)

### 2.1 等级机制

牵绊等级取代了传统的友好度/恋爱度分离系统，将**朋友、恋人和伴侣关系**统一在同一个等级体系中。

| 等级 | 所需经验 | 关键解锁 | 意义 |
|:---:|:--------:|:--------|:----|
| **Lv.1** | 5 | **可邀请为同伴**(队伍成员) | 从普通朋友到冒险伙伴 |
| **Lv.2** | 15 | — | 关系深化 |
| **Lv.3** | 40 | 浴室对话、生日礼物、**牵绊任务**开启 | 共享私密空间 |
| **Lv.4** | 75 | **可共进晚餐** | 日常互动升级 |
| **Lv.5** | 120 | **可邀请参加节日** | 公开的约会关系 |
| **Lv.6** | 175 | **可一起做饭**(解锁特殊料理) | 更深层次的合作 |
| **Lv.7** | 240 | **告白许可**("我爱你") | 确定恋爱关系 |
| **Lv.8** | 315 | — | 稳定交往期 |
| **Lv.9** | 400 | — | 准结婚阶段 |
| **Lv.10** | 495 | **结婚解锁** | 终成眷属 |

## 三、世界转移法(World Transfer)——跨周目的革命

### 3.1 核心概念

**世界转移法(世界转移法 / Technique to Travel Between Worlds)** 是《龙之天地》最独特的设计创新。它允许玩家在**同一个存档文件**中创建最多**30个平行世界线(timelines)**，每条世界线可独立发展恋爱关系和部分剧情，而核心进度（等级、装备、物品、已解锁内容）共享。

### 3.2 与传统模式的对比

| 对比项 | 传统RF(开新档) | 世界转移法 |
|:-----:|:-------------:|:---------:|
| **体验全部路线的方式** | 创建全新存档，从头开始 | **同一存档内创建平行世界线** |
| **保留内容** | 无(完全从头开始) | 等级、装备、物品、技能、成就 |
| **重玩内容** | 全部流程 | 仅恋爱相关 + 部分可重置内容 |
| **最大周目数** | 无限制(但每次从头) | 30条世界线 |
| **切换成本** | 极高(几十小时) | 极低(即时切换) |
| **反悔成本** | 极高(需重新攻略) | 低(回退到分叉点) |
| **动力** | 真爱驱动 | **探索驱动**——没有惩罚，鼓励全收集 |

### 3.3 工作机制

```
主世界线(主线推进)
    │
    ├── 选择与A结婚 → 世界线A(与A的婚后生活)
    │
    ├── 选择与B结婚 → 世界线B(与B的婚后生活)
    │
    ├── 选择与C结婚 → 世界线C(与C的婚后生活)
    │
    └── ...(最多30条)
```

1. **分叉点**：在向某人求婚前存档，创建新的世界线
2. **共享内容**：所有世界线共享等级、装备、物品、技能、村庄等级
3. **独立内容**：牵绊等级、恋爱事件、婚后生活、部分剧情选择
4. **自由切换**：随时可以在世界线之间切换
5. **最大30条**：每条世界线独立保存

### 3.4 保留 vs 重置

| 项目 | 世界转移时保留 | 世界转移时重置/独立 |
|:---:|:-------------:|:-----------------:|
| 角色等级 | 保留 | — |
| 装备/物品 | 保留 | — |
| 技能/技能树 | 保留 | — |
| 村庄等级 | 保留 | — |
| 建造内容 | 保留 | — |
| 地图探索 | 保留 | — |
| 牵绊等级(Bonds) | — | **独立** |
| 恋爱事件 | — | **独立** |
| 婚姻状态 | — | **独立** |
| 部分剧情选择 | — | **独立** |
| 子嗣 | — | **独立** |

### 3.5 设计意义

世界转移法的引入解决了RF系列长期存在的"多周目悖论"：

**旧模式问题**：
```
想攻略全角色 → 必须开新档 → 重新打几十小时主线 → 疲劳 → 放弃
```

**世界转移法解决方案**：
```
想攻略全角色 → 创建新世界线 → 跳过重复内容 → 专注恋爱 → 无缝体验全路线
```

**更深层的设计智慧**：
1. **尊重玩家时间**：不要求重复体验相同的主线内容
2. **降低情感负担**：不再需要"背叛"已结婚的角色来体验其他路线
3. **全收集友好**：完美主义者可以在一个存档中完成所有恋爱成就
4. **平行宇宙叙事**：呼应了本作的"龙神/平行世界"主题设定

## 五、婚姻与子嗣系统

### 5.1 结婚流程

```
牵绊Lv.10 + 全部牵绊任务完成
    → 获得梅枝(Plum Tree Branch)
    → 制作神酒(Divine Spirits)：水稻×1 + 紫藤×1
    → 邀请对方前往命运纺织神社(Fateweaver Shrine)
    → 结婚仪式
    → 婚后生活开始
```

### 5.2 婚后生活

| 婚后内容 | 说明 |
|:--------|:----|
| **同居** | 伴侣搬入玩家的住所 |
| **日常互动** | 新的对话和互动选项 |
| **特殊料理** | 伴侣会烹饪特殊料理 |
| **协助农活** | 伴侣有时会帮忙农场工作 |
| **节日新对话** | 婚后节日有专属对话 |

### 5.3 子嗣系统

| 项目 | 说明 |
|:---:|:----|
| **最大子嗣数** | 2人 |
| **子嗣可入队** | 可以编入冒险队伍 |
| **外观** | 受NPC伴侣影响 |
| **性格** | 受NPC伴侣性格影响 |
| **成长** | 子嗣会随游戏进程成长 |

### 5.4 婚后继续恋爱(世界转移法)

结婚后，玩家可通过世界转移法创建新的世界线来追求其他角色：
- 主世界线：保持与当前伴侣的婚姻
- 新世界线：从分叉点开始追求其他角色
- 所有世界线并行存在，随时可切换

## 六、系统设计分析

### 6.1 从"专一"到"全收集"

传统RF的恋爱系统设计基于"选择与代价"的哲学——选择一人意味着放弃其他人。这一设计在叙事上有其合理性（浪漫关系的排他性），但在游戏性上导致了：
- 必须多次通关才能体验全部内容
- 每次通关需重复大量相同内容
- 事后后悔无法挽回

龙之天地通过世界转移法，将恋爱系统从"专一路线"转变为"全收集"，在不破坏浪漫叙事的完整性前提下，消除了多周目的重复劳动成本。

### 6.2 平行宇宙叙事

世界转移法并非只是一个"偷懒"的便利功能——它在叙事层面也契合了龙之天地的世界设定：
- 龙神的力量贯穿整个阿祖玛大陆
- 时间与空间的掌控是龙神的核心能力
- 多条世界线的存在本身就符合游戏的东方神话/平行宇宙主题

### 6.3 对系列未来的影响

世界转移法可能成为RF系列的分水岭设计：
- 如果成功，未来RF作品可能会继承这一机制
- 它为"如何平衡多周目内容和玩家时间"提供了RF独有的解决方案
- 在这个"时间越来越宝贵"的游戏市场中，尊重玩家时间的机制创新将越来越重要
