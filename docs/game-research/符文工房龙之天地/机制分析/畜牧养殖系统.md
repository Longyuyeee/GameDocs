# 畜牧养殖系统
![龙之天地魔物](https://runefactory-gaz.com/img/system/ss_monster.jpg)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

---

## 一、系统总览

本作的畜牧系统继承了《符文工房》系列的经典"魔物驯服+资源产出"模式，同时引入了与村庄开发系统的深度联动。畜牧核心区域位于 **秋之里(Autumn Village)**，驯服魔物需要先建造牧场。

---

## 二、牧场解锁与建设

### 2.1 解锁条件

| 步骤 | 内容 |
|------|------|
| 触发条件 | 推进主线至**秋之里**(第三个解锁的村庄) |
| 初始任务 | NPC Yachiyo(八千代)和 Kurama(鞍马)委派与 Buffamoo 交朋友的任务 |
| 初始奖励 | 免费获得**牧场(4栏位)** 一座 |

### 2.2 牧场建造

| 牧场类型 | 栏位数 | 建造材料 | 解锁条件 |
|---------|-------|---------|---------|
| 牧场(Barn) | 4格 | Lumber x35 + Material Stone x15 | 秋之里初始解锁 |
| 大型牧场(Large Barn) | 8格 | Lumber x100 + Material Stone x40 | 秋之里等级5 |

> 小体型魔物占 **1格**，大体型魔物(如 Buffamoo)占 **2格**。
> 牧场可在任意季节村庄的开发区建造。

## 三、魔物驯服系统

### 3.1 驯服流程

```
步骤1: 牵绊之舞(Dance of Bonds)
  └─ 使用 Plum Branch(梅枝)对目标魔物跳舞
  └─ 直到魔物头顶出现微笑图标

步骤2: 击败魔物
  └─ 将魔物打到残血(不要杀死)
  └─ 魔物进入可驯服状态

步骤3: 赠送喜爱物品
  └─ 3次赠送机会
  └─ 根据喜好程度出现1-3颗心
  └─ 即使3颗心也有失败概率，可重试
```

### 3.2 驯服成功率提升技巧

| 技巧 | 说明 |
|------|------|
| 赠送自身掉落物 | 魔物自身掉落的物品通常是喜爱物 |
| 高牵绊等级 | 玩家牵绊技能等级影响成功率 |
| Plum Branch 技能升级 | 舞蹈技能等级提升增加成功率 |

### 3.3 是否可驯服判断

- 打开菜单 → **冒险日志(Adventure Log)** → **魔物图鉴(Beastiary)**
- 可查看每种魔物是否可驯服
- BOSS魔物也可驯服(需收集其喜好物品)

## 四、魔物资源产出系统

### 4.1 日常产出机制

| 机制 | 说明 |
|------|------|
| 放置条件 | 魔物必须放置在牧场的栏位中 |
| 产出频率 | **每日自动产出** |
| 产出品质 | 受魔物等级影响 |
| 收取方式 | 地面拾取 |

### 4.2 魔物等级与产出品质

魔物等级提升的关键节点：

| 等级区间 | 产出品质 |
|---------|---------|
| Lv1 | 基础品质 |
| Lv15 | 产出提升 |
| Lv35 | 产出大幅提升 |
| Lv65 | 高级产出 |
| Lv80 | 最高品质产出 |

> 魔物升级方式：**必须加入玩家队伍**才能获得经验值，无法像前作那样在家中被动升级。

### 4.3 经典产出魔物(Producer Monsters)

本作沿袭了系列的经典产出魔物设定：

| 魔物 | 产出物 | 栏位占用 | 备注 |
|------|-------|---------|------|
| Buffamoo(牛) | 牛奶(Milk) | 2格 | 经典乳品产出 |
| Cluckadoodle(鸡) | 蛋(Egg) | 1格 | 经典蛋品产出 |
| Wooly(羊) | 毛(Material) | 1格 | 毛料产出 |
| Bee(蜂) | 蜂蜜(Honey) | 1格 | 甜味素材产出 |
| Chipsqueak | 食物素材 | 1格 | 杂项食物产出 |
| Veggiesqueak | 蔬菜类 | 1格 | 蔬菜产出 |
| 其他小体型魔物 | 各自掉落物 | 1格 | 视种类而定 |

### 4.4 BOSS驯服与产出

| 项目 | 内容 |
|------|------|
| 可驯服BOSS | 主线中的部分BOSS |
| 驯服方式 | 比普通魔物更复杂，需收集特定喜好物品 |
| 产出 | 稀有宝石、高级素材 |
| 战斗能力 | 高基础属性，可作为强力队伍成员 |

## 五、骑行系统(Mounts)

### 5.1 骑行解锁条件

| 步骤 | 内容 |
|------|------|
| 1 | 驯服可骑乘的魔物(查看魔物图鉴确认可骑乘) |
| 2 | 将魔物放置在牧场中 |
| 3 | 将魔物编入**队伍(Party)** |
| 4 | 前往村庄外的**开阔地带**，靠近魔物上骑 |

### 5.2 骑行限制

| 限制 | 说明 |
|------|------|
| 村庄内 | **不可骑乘** |
| 地形 | 部分狭窄区域不可骑行 |
| 飞行 | **可骑乘魔物不能飞行**(仅地面上移动) |
| 速度 | **不如圣扇冲刺速度** |

### 5.3 骑行战斗

| 项目 | 内容 |
|------|------|
| 攻击 | 轻攻击/重攻击 |
| 采集 | 骑行中可采集资源 |
| 评价 | 骑行速度被玩家社区评价为不如圣扇冲刺实用 |

### 5.4 骑行 vs 圣扇(玩家社区评价)

| 移动方式 | 速度评价 | 实用性 |
|---------|---------|-------|
| 圣扇冲刺 | **最快** | 日常探索推荐 |
| 骑乘魔物 | 中等 | 战斗辅助价值 > 移动价值 |
| 步行 | 最慢 | 仅限初期 |

> 魔物骑乘最重要的价值其实是作为**战斗队员**，因为魔物不依赖装备，属性成长曲线比人类队员更高。

## 六、魔物战斗系统

### 6.1 队伍编成

| 项目 | 内容 |
|------|------|
| 队伍人数 | 最多6人(3主力 + 3后备) |
| 魔物定位 | 可作为独立队员编入 |
| 战斗AI | 自动战斗 |

### 6.2 魔物战斗优势

| 优势 | 说明 |
|------|------|
| 高属性成长 | 不依赖装备，等级即实力 |
| 稀有掉落 | 战斗中获得稀有素材 |
| BOSS战力 | 驯服BOSS后拥有极高基础属性 |

---

## 七、畜牧与村庄发展联动

### 7.1 畜牧相关建筑

| 建筑 | 效果 |
|------|------|
| 牧场(Barn) | 基础魔物收纳 |
| 大型牧场(Large Barn) | 更多栏位 |
| 水车(Watermill) | 农业产出+1%(需旁水路) |

### 7.2 村民委任与畜牧

| 委任类型 | 效果 |
|---------|------|
| 照顾魔物 | 村民可协助喂养/照顾牧场魔物 |
| 畜牧特质村民 | 效率提升 |
