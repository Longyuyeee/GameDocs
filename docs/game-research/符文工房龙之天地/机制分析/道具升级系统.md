# 道具升级系统
![神器](https://runefactory-gaz.com/img/system/ss_treasure.jpg)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

---

## 一、系统总览

本作的道具系统进行了重大改革，将传统的"农具+武器"分离体系整合为 **六大神器(Sacred Treasures)多功能体系**。神器既是农业工具，也是战斗武器，同时还可用于地图解谜。此外，独立武器系统(单手剑/双手剑/双剑/弓/符咒)仍保留并可通过锻造升级。

---

## 二、六大神器(Sacred Treasures)详解

### 2.1 神器总览

| 神器 | 属性 | 农业功能 | 战斗功能 | 解谜功能 | 解锁时机 |
|------|------|---------|---------|---------|---------|
| **圣剑(Sacred Sword)** | 火 | 成熟作物→种子(概率升级) | 高伤害输出 | 破坏邪祟 | 初期 |
| **圣鼓(Sacred Drum)** | 地 | 加速生长1天、复活枯萎 | 治疗队友 | 震碎障碍 | 初期 |
| **圣伞(Sacred Parasol)** | 水 | 大范围灌溉 | 近战范围伤害 | 空中滑翔 | 初期 |
| **圣扇(Sacred Fan)** | 风 | 批量直线收获 | 穿透远程攻击 | 吹走毒气、冲刺 | 初期 |
| **鬼神面具(Sacred Oni Mask)** | 暗 | 粉碎巨石(清理耕地) | 连续快攻 | 破坏发光的巨石 | 中期 |
| **锡杖(Sacred Staff)** | 光 | 无直接农业用途 | 自动追踪射击 | 看穿幻影/隐形地板 | 中期 |

### 2.2 神器获取与升级

**获取方式：**
- 主线剧情推进过程中自动获得

**升级方式：**

| 升级要素 | 说明 |
|---------|------|
| 经验获取 | **使用神器即获得经验值**(如使用圣鼓获得 Drum XP) |
| 技能树 | 消耗经验值在**技能树(Skill Tree)** 中解锁新能力和强化 |
| 普适经验值 | 每日睡眠时将未分配的其他经验值转化为"All-Purpose XP" |

**技能树结构：**
- 17个技能树分为3大类：
  1. **武器技能**(Swords/Long Swords/Dual Blades/Bows/Talismans)
  2. **神器舞蹈**(Drum Dance/Sword Dance/Fan Dance/Parasol Dance/Staff Dance/Oni Mask Dance)
  3. **实用技能**(Practical Abilities)

| 实用技能树 | 解锁条件 |
|-----------|---------|
| 烹饪(Cooking) | 初始可用 |
| 建造(Constructing) | 完成"村长的职责"主线任务 |
| 锻造(Forging) | 完成"铁匠铺风波2"主线任务 |
| 调合(Chemistry) | 完成"冬之里复兴2"主线任务 |
| 社交活动(Social Activities) | 初始可用 |
| 自我提升(Self Improvement) | 初始可用 |

## 三、武器升级系统

### 3.1 武器种类

| 武器类型 | 特点 | 升级地点 |
|---------|------|---------|
| 单手剑(Short Sword) | 快速、均衡 | 铁匠铺(Blacksmith) |
| 双手剑(Long Sword) | 高伤害、慢速 | 铁匠铺 |
| 双剑(Dual Blades) | 多段攻击、风属性 | 铁匠铺 |
| 弓(Bow) | **新增**，远程连射 | 铁匠铺 |
| 符咒(Talisman) | **新增**，魔法追踪 | **Hisui**(夏之里) |

### 3.2 铁匠铺升级

**解锁方式：**
- 完成"铁匠铺风波2"(Blacksmith Brouhaha 2)任务
- 建造铁匠铺建筑(材料: 800G + Lumber x40 + Material Stone x20 + Iron x1)
- 需要指派拥有 **Journeyman Smith(工匠)** 特质的村民经营

**升级分类：**

| 分类 | 包含项目 |
|------|---------|
| 短剑(Short Swords) | 单手剑系列 |
| 长剑(Long Swords) | 双手剑系列 |
| 双剑(Dual Blades) | 双持剑系列 |
| 弓(Bows) | 远程弓系列 |
| 肩甲(Pauldrons) | 防具 |
| 头盔(Headgear) | 防具 |

**升级所需素材示例：**

| 素材 | 价格(铁匠铺购买) | 获取方式 |
|------|----------------|---------|
| 铁(Iron) | 950G | 采矿/购买 |
| 青铜(Bronze) | 1900G | 采矿/购买 |
| 银(Silver) | 2850G | 采矿/购买 |
| 金(Gold) | 3800G | 采矿/购买 |
| 羽毛(Fletch) | — | 魔物掉落 |
| 布(Cloth) | — | 魔物掉落/购买 |

## 四、勾玉系统(Magatama Slots) — 终局强化

### 4.1 解锁条件

| 条件 | 说明 |
|------|------|
| 主线进度 | 完成"冬之里复兴"主线任务后 |
| NPC | Zaza(冬之里) / Hisui(夏之里) |
| 材料 | **红宝石(Crimson Jewel)** 用于添加插槽 |

### 4.2 勾玉效果

勾玉是镶嵌在武器插槽中的强化道具，可提供各种属性加成：

| 勾玉类型 | 效果 |
|---------|------|
| 属性勾玉 | 添加元素属性(火/水/地/风/光/暗) |
| 能力勾玉 | 提升攻击力、防御力等基础属性 |
| 特殊勾玉 | 提供特殊效果(如吸血、暴击等) |

**获取方式：**
- Jewel Squeeks(宝石鼠)魔物掉落 — 分布于遗忘森林(Forgotten Grove)

## 六、工具升级

### 6.1 基础采集工具

| 工具 | 用途 |
|------|------|
| Cheap Hammer(廉价锤) | 破碎矿石、采集石材 |
| Common Hatchet(普通斧) | 采伐木材 |

### 6.2 工具升级方式

与传统Rune Factory不同，本作工具升级方式为：

| 步骤 | 内容 |
|------|------|
| 1 | **提升村庄等级(Village Level)** — 完成村庄任务积累故乡点数 |
| 2 | 访问**龙神像/神社**接收新配方 |
| 3 | 使用配方在**木匠处**制作升级版工具 |
| 4 | 无需铁匠铺参与 |

> 基础采集工具的升级与村庄发展进度绑定，是推动玩家参与村庄建设的激励机制之一。

## 七、背包扩容系统

### 7.1 存储分类

| 存储类型 | 初始容量 | 最大容量(1.1.0更新后) |
|---------|---------|---------------------|
| 个人背包 | 有限 | 通过物品/任务扩展 |
| 存储箱(Storage Box) | 基础容量 | **1200** |
| 种子箱(Seeds & Such) | 基础容量 | **400** |
| 素材箱 | 基础容量 | 随升级扩展 |

### 7.2 容量扩展方式

| 扩展方式 | 说明 |
|---------|------|
| 购买/制作扩容道具 | 商店购买或任务奖励 |
| 村庄等级提升 | 高等级解锁更多存储空间 |
| 1.1.0版本更新 | 最大设施数+5，装饰上限256，存储箱扩至1200 |

## 八、技能树实用技能推荐

### 8.1 优先解锁技能

| 优先级 | 技能 | 理由 |
|-------|------|------|
| ★★★ | 锻造(材料)成本降低 | 节省大量金币 |
| ★★★ | 建造(材料)成本降低 | 节省大量素材 |
| ★★★ | 调合(材料)成本降低 | 节省药品素材 |

| ★★☆ | RP上限提升 | 更多神器使用次数 |
