# 地图场景系统
![龙之天地地图](https://runefactory-gaz.com/img/system/ss_village.jpg)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

---

## 一、世界背景设定

### 1.1 世界观

故事发生在东方国度 **"东国(Azuma)"**，因 **"天崩(Celestial Collapse)"** 灾难导致大地破碎，地形碎片漂浮在天空与海洋之间，形成了**垂直分层**的开放世界结构。

### 1.2 移动方式

| 移动手段 | 说明 |
|---------|------|
| **白龙骑乘(White Dragon)** | 在浮空岛屿间飞行移动，通过龙神像或地图选择"飞向天空" |
| 步行 | 地域内移动 |
| 骑乘魔物 | 野外移动(不可在村庄内) |
| 圣扇冲刺 | 地面高速移动 |

## 二、四季村庄 —— 核心枢纽

### 2.1 村庄总览

| 村庄 | 固定季节 | 地形特征 | 核心功能 | 解锁顺序 |
|------|---------|---------|---------|---------|
| **春之里(Spring Village)** | 春季 | 高地/樱花/绿叶 | **初期据点**、基础农业、工坊 | 第1个 |
| **夏之里(Summer Village)** | 夏季 | 海岸/阳光/沙滩 | **商业中心**、商店经营 | 第2个 |
| **秋之里(Autumn Village)** | 秋季 | 红叶/森林/山地 | **农业/畜牧核心**、水田、牧场 | 第3个 |
| **冬之里(Winter Village)** | 冬季 | 雪山/冰原/温泉 | **钓鱼/矿业中心** | 第4个 |

### 2.2 村庄功能分布

| 功能建筑 | 所在村庄 |
|---------|---------|
| 木匠(Carpenter) | 所有村庄(独立建造) |
| 铁匠铺(Blacksmith) | 各村庄可建造 |
| 茶屋(Teahouse) | 夏之里 |
| 居酒屋(Izakaya) | 秋之里(由Yachiyo经营) |
| 温泉(Hot Spring) | 冬之里 |
| 神社(Shrine) | 各村庄(剧情/节日相关) |

## 三、村庄建造系统(Village Builder Mode)

### 3.1 建造模式概述

本作最大创新之一：玩家不仅是农场主，更是**村庄规划者**。

### 3.2 进入方式

| 步骤 | 内容 |
|------|------|
| 1 | 获得 **Terra Tiller(神圣耕具)** — 复活 Ulalaka 的圣树后 |
| 2 | 前往各村庄的 **"开发区域"** (大块空地，可能有障碍物) |
| 3 | 装备 Terra Tiller，在开发区域按右扳机(手柄)/Z键(键盘)进入建造模式 |

### 3.3 可建造内容

| 类别 | 内容 | 效果 |
|------|------|------|
| **建筑(Buildings)** | 房屋、商店、工坊、浴场、风车、水车等 | 提供功能性服务 |
| **装饰(Decorations)** | 花车、灯笼、雕像、园艺等 | 提升**风景评分(Scenic Score)** |
| **地形地板(Terrain Floors)** | 农田(Field)、水田(Rice Paddy)、水路(Waterway)、道路(Road) | 农业/通行基础 |
| **基础设施** | 桥梁、楼梯、扩展区域 | 扩大村庄范围 |

### 3.4 建造限制

| 项目 | 限制 |
|------|------|
| 建筑/装饰 | **有数量上限**，可通过提升村庄等级增加 |
| 地形地板 | **仅受开发区域空间限制** |
| 1.1.0更新后 | 最大设施数+5(村庄等级10时)，最大装饰数256 |

### 3.5 俯瞰模式(Overhead View)

| 项目 | 内容 |
|------|------|
| 进入方式 | 建造模式下按右摇杆(手柄)/T键(键盘) |
| 功能 | 鸟瞰视角规划村庄布局、建筑摆放、农田规划 |
| 优势 | 时间暂停+全局视角，便于大规模规划 |
| Switch2 | 支持鼠标模式操作 |

### 3.6 建筑功能加成

建筑和装饰提供各类加成：

| 加成类型 | 效果 |
|---------|------|
| Farming(农耕) | 提高作物预估产量 |
| Trading(交易) | 提高店铺收入 |
| Mining(采矿) | 提高采矿效率 |
| Scenic Score(风景) | 影响村民满意度 |
| Max HP | 提升角色生命上限 |

## 四、龙神像系统(Dragon Shrines)

### 4.1 功能总览

龙神像是遍布东国的重要互动点，功能如下：

| 功能 | 说明 |
|------|------|
| **保存游戏** | 手动存档点 |
| **快速传送(Fast Travel)** | 在已激活的龙神像间传送 |
| **睡眠恢复** | 可睡觉推进时间/恢复状态 |
| **白龙召唤** | 骑乘白龙飞向天空 |
| **牵绊之舞(Bonds Dance)** | **需使用Plum Branch跳舞激活** |

### 4.2 激活方式

| 步骤 | 内容 |
|------|------|
| 1 | 找到未激活的龙神像 |
| 2 | 使用Plum Branch(梅枝)执行**牵绊之舞(Bonds Dance)** |
| 3 | 龙神像激活，成为传送/保存点 |

> 部分龙神像位于山顶或洞穴深处，需通过滑翔点(frog statue)或神器解谜到达。

### 4.3 龙神像修复

初期剧情任务涉及修复春之里的龙神像，需要材料：

| 材料 | 数量 |
|------|------|
| Guardian Timber(守护者木材) | 若干 |
| Lumber(木材) | 若干 |
| Material Stone(石材) | 若干 |

## 五、迷宫系统(Dungeons)

### 5.1 主线迷宫一览

| 迷宫名称 | 所在区域 | BOSS | BOSS弱点 |
|---------|---------|------|---------|
| **废弃神社(Abandoned Shrine)** | 春之里 | **Mezuki**(马形) | 地属性(圣鼓) |
| **遗忘森林(Forgotten Grove)** | 春之里 | **Corpse Tree**(腐树) | 弓 + 地属性 |
| **守护者森林(Guardian's Grove)** | 春之里 | **Cursed Tree**(诅咒树) | 弓(瞄准额头) |
| **黄金花矿山(Goldblossom Mines)** | 夏之里 | **The Hound**(猎犬) | 短剑 + 光属性 |
| **神乐神社(Kagura Shrines)** | 秋之里 | **黑龙(Black Dragon)** | 弓(骑白龙空中战) |

### 5.2 迷宫机制

| 机制 | 说明 |
|------|------|
| **枯萎花(Blightblooms)** | 阻塞道路的障碍，需使用神器清除(圣剑砍、圣扇吹等) |
| **枯萎树(Blighted Trees)** | 使用圣鼓舞蹈恢复，显示为小地图上的黑圈 |
| **怪物屏障(Monster Barriers)** | 击败区域内所有敌人后解锁通路 |
| **神器解谜** | 使用对应神器(火烧、风吹、伞滑翔等)解开机关 |

## 六、开放世界探索要素

### 6.1 野外互动点

| 互动点类型 | 功能 |
|-----------|------|
| **蛙石(Frog Statues)** | 跳舞获得**料理食谱**，向Croakie汇报获得更多配方 |
| **路边神社(Wayside Shrines)** | 红色 altar，跳舞获得**装备锻造配方** |
| **地藏像(Jizo Statues)** | 清扫叶子后有概率在房间获得料理配方 |
| **弓靶(Yoichi's Bow Targets)** | 射中红白靶获得配方 |
| **篝火(Bonfires)** | 烹饪料理、恢复体力 |
| **滑翔点(Gliding Spots)** | 使用圣伞滑翔到达高处/远处平台 |

### 6.2 资源采集

| 资源类型 | 采集工具 | 刷新周期 |
|---------|---------|---------|
| 木材 | Common Hatchet(普通斧) | 每日刷新 |
| 矿石 | Cheap Hammer(廉价锤) | 每日刷新 |
| 药草 | 手动采集 | 每日刷新 |
| 鱼类 | 钓鱼 | 随时 |

### 6.3 浮空岛屿

通过骑乘白龙访问各种浮空岛屿，包含：
- 隐藏宝箱
- 稀有素材采集点
- 解谜机关
- 额外迷宫入口
