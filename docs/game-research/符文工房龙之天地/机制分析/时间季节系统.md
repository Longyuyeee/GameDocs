# 时间季节系统
![四季节](https://runefactory-gaz.com/img/system/ss_season.jpg)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

---

## 一、系统总览 —— 本作最大变革

本作彻底改革了系列延续20年的季节系统。**传统的一年四季旋转被取消**，取而代之的是 **4个固定季节的独立村庄**。这是《符文工房》系列历史上最具颠覆性的时间系统改革。

| 对比项 | 前作(RF4/RF5) | 本作 |
|-------|--------------|------|
| 季节 | 全地图统一旋转(30天/季) | **4个固定季节村庄**，无季节更替 |
| 日历 | 每月1-30日，四季循环 | 存在日历，但不受季节约束 |
| 作物限制 | 季节不对应则枯萎/无法种植 | **全年可种**，推荐村庄产量更高 |
| 事件解锁 | 日历/季节依赖 | **村庄等级+牵绊等级**驱动 |
| RP消耗 | 农活消耗RP | 农活**不消耗**RP |

## 二、四季村庄详细

### 2.1 四个固定季节区域

| 村庄 | 固定季节 | 环境特征 | 气候特点 |
|------|---------|---------|---------|
| 春之里(Spring) | 春季 | 樱花常开、绿意盎然 | 温暖、多春雨 |
| 夏之里(Summer) | 夏季 | 阳光明媚、海岸沙滩 | 炎热、多晴天 |
| 秋之里(Autumn) | 秋季 | 红叶覆盖、山林气息 | 凉爽、多秋风 |
| 冬之里(Winter) | 冬季 | 常积雪、冰原景观 | 寒冷、多雪 |

### 2.2 季节固定化的设计意图

| 设计目标 | 实现方式 |
|---------|---------|
| 消除季节等待 | 四季作物**同时可种**，不再需要等待特定季节 |
| 增加策略深度 | 玩家需要决定在哪个村庄发展哪种产业 |
| 世界多样性 | 每个村庄拥有独特的视觉风格和氛围 |
| 叙事可能性 | 各村庄独立故事线，与季节主题深度绑定 |

## 三、日循环系统

### 3.1 时间流速

| 项目 | 内容 |
|------|------|
| 游戏内1分钟 | 约等于现实3秒 |
| 1小时(游戏内) | 约现实3分钟 |
| 完整1天 | 约现实72分钟 |

### 3.2 时间暂停场景

| 场景 | 时间状态 |
|------|---------|
| 对话中 | 暂停 |
| 建造模式 | **暂停** |
| 菜单/背包操作 | 暂停 |
| 战斗状态 | 正常流动 |
| 野外探索 | 正常流动 |
| 钓鱼 | 正常流动 |

### 3.3 作息与活动安排

| 时间段 | 游戏内时间 | 推荐活动 |
|-------|-----------|---------|
| 清晨 | 6:00-8:00 | 起床、检查作物、安排村民任务 |
| 上午 | 8:00-12:00 | 种植、浇水、收获、比赛报名(8:00) |
| 中午 | 12:00-18:00 | 探索、战斗、采集、钓鱼、支线任务 |
| 傍晚 | 18:00-19:00 | 回村整理、送货、提交任务 |
| 夜晚 | 19:00- | 节日/约会、自由活动、深度探索 |
| 深夜 | 24:00-6:00 | NPC休息、适合独自活动 |

> 本作**没有强制睡眠机制**，玩家可以通宵活动，但NPC会在特定时间休息(店铺关门等)。

## 四、天气系统

### 4.1 天气类型

每个村庄的天气**每日独立变化**，出门前需要注意当前地区的天气状况。

| 天气类型 | 效果 |
|---------|------|
| 晴天 | 无特殊效果 |
| 阴天 | 正常 |
| 雨天(春/夏) | 自动灌溉部分农田，特定鱼类出现概率提升 |
| 雪天(冬) | **自动灌溉水田和农田**，特定鱼类出现 |
| 大风(秋) | 影响风属性神器效果 |
| 特殊天气(季节限定) | 节日相关天气效果 |

### 4.2 天气对游戏的影响

| 系统 | 天气影响 |
|------|---------|
| 农作物 | 推荐村庄的固定天气有助于作物生长 |
| 钓鱼 | 雨天钓鱿鱼概率更高、雪天特定鱼种出现 |
| 采集 | 部分素材在特定天气下出现概率变化 |
| 战斗(微弱) | 某些天气下特定属性伤害略有变化 |

> **关键**：由于村庄季节固定，天气对农业的整体影响相比前作大幅降低。作物不会因天气恶劣而枯萎死亡。

## 五、节日系统

### 5.1 节日一览

#### 浪漫剧情节日(需牵绊等级5+，提前3天邀请伴侣)

| 节日名称 | 所在村庄 | 日期 | 时间 | 内容 |
|---------|---------|------|------|------|
| **初诣(First Pilgrimage)** | 春之里 | 春1日 | 10:00 | 与伴侣一起参拜神社 |
| **七夕(Tanabata)** | 夏之里 | 夏7日 | 19:00-23:00 | 帮助实现愿望(需解锁Kanata) |
| **花火大会(Fireworks Festival)** | 夏之里 | 夏20日 | 19:00 | 海滩烟花，可自定义颜色(中午找Matsuri定制) |
| **灯笼节(Lantern Festival)** | 秋之里 | 秋10日 | 19:00 | 与伴侣放河灯 |
| **雪祭(Snow Festival)** | 冬之里 | 冬24日 | 19:00 | 与伴侣在雪穹下观景 |

#### 比赛类活动(手动安排，每月固定日期)

比赛安排在每月的 **1、7、13、19、25日**(假日)，可在各村庄的神社桌子上安排。比赛日 **8:00开始报名，9:00截止**，持续至 **19:00**。

| 比赛名称 | 所在村庄 | 内容 |
|---------|---------|------|
| **收获祭(Harvest Contest)** | 春之里 | 提交作物，评判等级、生长时间与大小 |
| **咖喱比赛(Curry Contest)** | 春之里 | 提交咖喱类料理 |
| **快手菜比赛(Quick Meal Contest)** | 夏之里 | 提交快手菜(Quick Meals) |
| **咸水鱼比赛(Saltwater Fish Contest)** | 夏之里 | 钓最大咸水鱼 |
| **料理比赛(Cooking Contest)** | 秋之里 | 提交一般料理 |
| **鱿鱼钓比赛(Squid Fishing Contest)** | 秋之里 | 钓最大鱿鱼 |
| **淡水鱼大作战(Freshwater Fish Contest)** | 冬之里 | 钓最多淡水鱼 |
| **公鱼钓比赛(Smelt Fishing Contest)** | 冬之里 | 钓最多公鱼 |

### 5.2 比赛参与流程

```
1. 在节日前(指定日期前)到神社桌子上安排比赛
2. 比赛当天 8:00 到场地报名
3. 9:00 前未报名则取消资格
4. 提交参赛物品
5. 19:00 公布结果，获得奖励
```

### 5.3 比赛奖励

| 名次 | 奖励内容 |
|------|---------|
| 第1名 | 高级素材、稀有物品、大量金币 |
| 第2-3名 | 中级奖励 |
| 参与奖 | 基础奖励 |

## 六、村庄等级系统 —— 替代日历进展

### 6.1 核心设计

本作用 **村庄等级(Village Level)** 替代了传统Rune Factory中的日历驱动的进展系统。关键剧情、建筑解锁、功能开放不再依赖"某个日期"，而是依赖玩家在村庄中的贡献度。

| 系统 | 前作 | 本作 |
|------|------|------|
| 功能解锁 | 主线进度/日期 | **村庄等级+主线进度** |
| 作物解锁 | 季节轮换 | 购买即可，推荐村庄产量更高 |
| 事件触发 | 特定日期 | 牵绊等级(随时可触发) |
| 建筑解锁 | 主线进度 | **村庄等级+故乡点数** |

### 6.2 村庄等级提升路径

```
[完成村庄任务] → [获得故乡点数] → [村庄等级提升] → [解锁新内容]
        ↑                                               ↓
    [村民满意度] ← [建造建筑/装饰] ← [新建筑/功能可用]
```

### 6.3 各等级解锁内容

| 村庄等级 | 解锁内容 |
|---------|---------|
| Lv1 | 基础建筑、初始开发区域 |
| Lv2 | 高级建筑、新开发区域 |
| Lv3 | 水车、风车等 |
| Lv4-Lv5 | 茶屋、新装饰、大型牧场(秋) |
| Lv6-Lv7 | 浴场、花车、人口上限增加 |
| Lv8-Lv10 | 终极建筑、异国风格建筑、宝物库、天堂塔 |

## 七、体力/RP系统改革

### 7.1 RP消耗规则

| 操作类别 | 具体操作 | 是否消耗RP |
|---------|---------|-----------|
| **基础农活** | 锄地、播种、浇水、收获 | **不消耗** |
| **基础采集** | 砍树、采矿、钓鱼 | **不消耗** |
| **神器技能** | 圣鼓催熟、圣剑转种子、圣扇批量收、圣伞范围浇 | **消耗** |
| **战斗** | 武器攻击、翻滚闪避 | **不消耗** |
| **神器战斗技能** | 神器舞蹈攻击技 | **消耗** |
| **烹饪/锻造/调合** | 制造物品 | **不消耗** |

### 7.2 RP恢复方式

| 方式 | 效率 |
|------|------|
| 睡眠(床上) | 完全恢复 |
| 料理食用 | 中-高(视料理品质) |
| 休息 | 少量缓慢恢复 |
| 龙神像祈祷 | 恢复 |
| 药品使用 | 中 |

## 八、时间综合策略建议

### 8.1 每日最佳流程

```
早晨(6:00-8:00):
  ├─ 起床检查所有村庄作物
  ├─ 使用圣鼓加速高价值作物
  ├─ 安排村民当日工作委任
  └─ 收取牧场产出

上午(8:00-12:00):
  ├─ 比赛日 → 8:00报名参赛
  ├─ 种植/浇水/收获(圣扇批量)
  ├─ 与NPC对话提升牵绊等级
  └─ 完成简单的村庄任务

中午-下午(12:00-18:00):
  ├─ 探索迷宫/野外区域
  ├─ 挑战BOSS
  ├─ 采集素材、钓鱼
  └─ 完成委托任务

傍晚-夜晚(18:00-):
  ├─ 节日/约会(如有)
  ├─ 整理背包、存储物品
  ├─ 烹饪/锻造/调合
  └─ 自由活动
```

### 8.2 跨村庄时间管理

- 四个村庄的时间**同步流动**，在春之里过了一小时，夏之里也过了一小时
- 通过龙神像快速传送可高效跨村庄活动
- 每个村庄的天气独立变化，出行前可查看各村庄天气预报
