# 战斗探索系统
![神器战斗](https://runefactory-gaz.com/img/system/ss_battle.jpg)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

---

## 一、系统总览

本作的战斗系统在《符文工房》系列基础上进行了**大幅强化**，引入了两种**全新武器类型**(弓/符咒)、**即时闪避**反击机制、**6人队伍系统**和**神器舞蹈**战斗体系。战斗的节奏感和动作性达到系列最高水平。

| 项目 | 内容 |
|------|------|
| 战斗类型 | 即时动作制(Action RPG) |
| 武器切换 | 可随身携带**2把武器**自由切换 |
| 队伍人数 | 最多**6人**(3主力+3后备) |
| 核心新机制 | 即时闪避→子弹时间→强力反击 |
| 元素系统 | 6属性互相克制 |

## 二、武器系统

### 2.1 武器类型一览

| 武器类型 | 特点 | 难度 | 推荐场景 |
|---------|------|------|---------|
| **单手剑(Short Sword)** | 快速、均衡、易上手 | ★☆☆ | 通用 |
| **双手剑(Long Sword)** | 高伤、慢速、范围大 | ★★☆ | BOSS战 |
| **双剑(Dual Blades)** | 多段连击、风属性 | ★★☆ | 群体战 |
| **弓(Bow)【新增】** | **远程连射**、可攻击飞行敌人、头部暴击加成 | ★★☆ | 飞行敌人/风筝 |
| **符咒(Talisman)【新增】** | **自动追踪**、魔法伤害(M.ATK)、对飞行敌人最优 | ★★★ | 魔法输出/飞行敌人 |

### 2.2 新武器详解

#### 弓(Bow)

| 特性 | 说明 |
|------|------|
| 攻击方式 | 远程连续射击 |
| 特殊效果 | 可命中**飞行敌人** |
| 暴击机制 | **爆头(Headshot)** 造成额外伤害 |
| 蓄力攻击 | 可进行强力蓄力射击 |
| 适用场景 | 风筝打法、空中敌人、远程安全输出 |

高级弓示例：

| 弓名 | 攻击力(ATK) | 属性 |
|------|------------|------|
| Kingfisher Bow | +264 | 水 |
| Falcon Bow | +1054 | 风 |
| Yatagarasu Bow | +2449 | 火 |

#### 符咒(Talisman)

| 特性 | 说明 |
|------|------|
| 攻击方式 | 魔法投射物，**自动追踪**敌人 |
| 伤害类型 | **魔法攻击(M.ATK)** |
| 最佳目标 | **飞行敌人**(弓箭难以命中的高速目标) |
| 升级地点 | **Hisui**(夏之里)——不同于铁匠铺 |

高级符咒示例：

| 符咒名 | 魔法攻击(M.ATK) | 属性 |
|--------|---------------|------|
| Talismans of Fissures | +260 | 地 |
| Arcane Light Talismans | +746 | 光 |
| Holy Split Sea Talismans | +2258 | 水 |

### 2.3 武器切换

玩家可同时装备**2把武器**，战斗中通过快捷键即时切换，实现不同战术组合：

```
推荐组合:
  单手剑 + 弓    = 近战+远程全能型
  双剑 + 符咒    = 高速近战+魔法追踪型
  双手剑 + 弓    = 高伤害+远程安全型
  符咒 + 弓      = 双远程魔法型
```

### 2.4 武器升级

详见[道具升级系统.md]文档，简而言之：
- 弓/剑/双剑/双手剑 → **铁匠铺**升级
- 符咒 → **Hisui**(夏之里)升级
- 终局强化 → **勾玉(Magatama)** 镶嵌(需红宝石添加插槽)

## 三、神器舞蹈战斗系统

### 3.1 神器与属性对应

| 神器 | 属性 | 战斗技能类型 |
|------|------|-------------|
| 圣剑(Sacred Sword) | 火 | 高伤害输出技 |
| 圣鼓(Sacred Drum) | 地 | 治疗/辅助技 |
| 圣伞(Sacred Parasol) | 水 | 范围近战技 |
| 圣扇(Sacred Fan) | 风 | 穿透远程技 |
| 鬼神面具(Sacred Oni Mask) | 暗 | 连续快攻技 |
| 锡杖(Sacred Staff) | 光 | 自动追踪射击、幻影透视 |

### 3.2 神器舞蹈技能

| 舞蹈技能 | 所属神器 | 效果 |
|---------|---------|------|
| 鼓之舞(Drum Dance) | 圣鼓 | 回复HP |
| 剑之舞(Sword Dance) | 圣剑 | 火属性高伤害 |
| 扇之舞(Fan Dance) | 圣扇 | 风属性远程穿透 |
| 伞之舞(Parasol Dance) | 圣伞 | 水属性范围伤害 |
| 鬼面之舞(Oni Mask Dance) | 鬼神面具 | 暗属性连续攻击 |
| 锡杖之舞(Staff Dance) | 锡杖 | 光属性追踪弹 |

> 神器舞蹈技能通过**技能树(Skill Tree)** 解锁和强化，消耗对应神器的使用经验值。

### 3.3 神器在战斗中的战略价值

- 神器攻击附带对应**元素属性**，用于克制敌人
- 舞蹈技能消耗**RP**，需合理分配
- 装备圣剑时，普通武器蓄力攻击也会附加火属性
- 神器技能与武器攻击可形成连招

## 四、即时闪避与反击系统

### 4.1 即时闪避(Instant Dodge)

| 项目 | 内容 |
|------|------|
| 操作 | 受到攻击瞬间按下闪避键 |
| 成功效果 | 触发 **"子弹时间(Bullet Time)"** → 敌人动作变慢 |
| 反击时机 | 画面出现 **"好机会"** 提示 |
| 反击伤害 | 大幅提升，BOSS通常撑不过两轮反击 |

### 4.2 闪避机制对比

| 闪避类型 | 操作 | 效果 |
|---------|------|------|
| 普通垫步 | 按闪避键 | 快速位移躲避 |
| 即时闪避(成功) | 攻击命中瞬间闪避 | 子弹时间+反击机会 |
| 即时闪避(失败) | 时机不对 | 普通闪避动作 |

## 五、元素系统

### 5.1 7种属性

| 属性 | 对应神器 | 对应高级武器 |
|------|---------|-------------|
| 地(Earth) | 圣鼓 | 特定武器 |
| 火(Fire) | 圣剑 | 火属性武器 |
| 风(Wind) | 圣扇 | Falcon Bow等 |
| 水(Water) | 圣伞 | Kingfisher Bow等 |
| 光(Light) | 锡杖 | Aureate Cutlass等 |
| 暗(Dark) | 鬼神面具 | 特定武器 |
| 爱(Love) | 梅枝(Plum Branch) | 非战斗属性，用于驯服 |

### 5.2 克制关系

本作采用**双向互相克制**而非单向克制：

| 互克对 | 关系 |
|-------|------|
| **火(Fire) ↔ 水(Water)** | 互相克制，伤害对等 |
| **风(Wind) ↔ 地(Earth)** | 互相克制 |
| **光(Light) ↔ 暗(Dark)** | 互相克制 |

### 5.3 伤害颜色指示

| 伤害数字颜色 | 含义 |
|------------|------|
| **黄色** | **属性克制有效** → 高伤害 |
| **白色** | 普通伤害(无属性/属性无效) |
| **灰色** | **属性被抵抗** → 低伤害 |

> **注意**：如果用与敌人相同属性的攻击去打它，可能造成极低甚至无伤害。

### 5.4 克制查询

战斗中可通过观察敌人身上的光环颜色或查看冒险日志中的怪物图鉴来确认弱点。

## 六、队伍系统

### 6.1 编队规则

| 项目 | 内容 |
|------|------|
| 解锁邀请 | 牵绊等级Lv1 |
| 最大人数 | **6人**(3主力 + 3后备) |
| 编成对象 | NPC村民(人类) + 驯服魔物(怪物) |

### 6.2 战斗AI定位

| AI类型 | 行为模式 | 推荐人选 |
|-------|---------|---------|
| **攻击型(Attack)** | 高伤害输出，主动进攻 | 高攻击角色 |
| **防御型(Defense)** | 保护玩家，吸引仇恨 | 高防御/高HP角色 |
| **支持型(Support)** | 提供增益效果 | 辅助特质角色 |
| **恢复型(Heal)** | 回复队伍HP | 治疗特质角色 |
| **扰乱型(Disrupt)** | 干扰/削弱敌人 | 特殊技能角色 |

### 6.3 队员切换

| 操作 | 效果 |
|------|------|
| 战斗切换 | 随时在3名主力队员间切换控制 |
| 自动替换 | 主力队员战败时，后备队员自动补上 |

### 6.4 魔物队伍优势

| 优势 | 说明 |
|------|------|
| 属性成长 | 不依赖装备，等级即实力(人类队员依赖装备) |
| 稀有掉落 | 战斗中可获得稀有素材(如Jewel Squeek掉落勾玉) |
| BOSS战力 | 驯服的BOSS拥有极高属性 |

## 七、BOSS战系统

### 7.1 主线BOSS一览

| BOSS名称 | 所在迷宫 | 推荐武器 | 属性弱点 | 备注 |
|---------|---------|---------|---------|------|
| **Mezuki**(马形) | 废弃神社 | 短剑 | 地(圣鼓) | 春之里 |
| **Corpse Tree**(腐树) | 遗忘森林 | 弓 | 地 | 春之里 |
| **Cursed Tree**(诅咒树) | 守护者森林 | 弓(瞄准额头) | 地 | 春之里 |
| **The Hound**(猎犬) | 黄金花矿山 | 短剑 | 光 | 夏之里 |
| **Black Dragon**(黑龙) | 神乐神社 | 弓(骑白龙空中战) | — | 秋之里 |
| **Kamurosaki** | 终末之森F9 | 长剑 | 光 | 通关后最终BOSS |

### 7.2 BOSS战通用策略

```
1. 确认阶段
   ├─ 使用冒险日志确认BOSS的武器弱点和属性弱点
   ├─ 根据弱点选择武器和神器
   └─ 检查装备耐久度和RP

2. 战斗阶段
   ├─ 观察BOSS攻击模式，寻找闪避时机
   ├─ 成功即时闪避 → 进入子弹时间 → 发动反击
   ├─ 利用元素克制输出黄色伤害
   └─ 切换队员应对不同阶段

3. 终结阶段
   ├─ 画面出现"好机会"提示 → 全力输出
   ├─ 利用神器舞蹈技能发动终结技
   └─ BOSS通常撑不过两轮反击
```

战斗中可观察BOSS身上的**光环颜色**判断属性：
- 也可通过系统指示直接查看推荐武器和属性

## 八、探索系统

### 8.1 探索区域

| 区域类型 | 数量 | 内容 |
|---------|------|------|
| 季节村庄野外 | 4 | 每个村庄周边有独立探索区域 |
| 主线迷宫 | 5+ | 线性箱庭风格，包含神器解谜 |
| 浮空岛屿 | 多个 | 骑白龙可达，含隐藏宝箱 |
| 终末之森 | 1(10层) | 通关后最高难度迷宫 |
| 魔界 | 1 | 通关后追加区域 |

### 8.2 探索工具与神器解谜

| 神器 | 探索/解谜用途 |
|------|-------------|
| 圣伞 | **空中滑翔**——到达高处/远处平台 |
| 圣扇 | **吹散毒气**、冲刺快速移动 |
| 圣剑 | **破坏邪祟/枯萎花**——清理解谜路线 |
| 圣鼓 | **震碎障碍**、复活枯萎的树木 |
| 鬼神面具 | **粉碎发光巨石** |
| 锡杖 | **看穿幻影和隐形地板** |

### 8.3 枯萎花(Blightbloom)清理

| 类型 | 清除方式 |
|------|---------|
| 普通枯萎花 | 使用对应神器攻击 |
| 枯萎树(Blighted Tree) | 使用圣鼓舞蹈恢复(小地图显示为黑圈) |
| 怪物屏障 | 击败区域内全部敌人 |

### 8.4 白龙飞行

- 通过龙神像或地图选择"飞向天空"
- 在浮空岛屿间自由移动
- 可发现隐藏区域和高空宝箱
- 部分BOSS战需要骑白龙进行(如黑龙)
