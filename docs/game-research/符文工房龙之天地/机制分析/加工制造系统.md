# 加工制造系统
![龙之天地料理](https://runefactory-gaz.com/img/system/ss_cooking.jpg)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

---

## 一、系统总览

本作的加工制造系统继承了《符文工房》系列的四大经典工种——**锻造(Forging)**、**烹饪(Cooking)**、**调合(Chemistry)**、**手工艺(Crafting)**——同时引入了与村庄建造系统的深度联动。配方获取方式相比前作有了显著变化。

| 工种 | 对应建筑 | 核心功能 |
|------|---------|---------|
| 锻造(Forging) | 铁匠铺(Blacksmith) | 武器、防具制造与升级 |
| 烹饪(Cooking) | 茶屋/居酒屋(Eatery) | 料理制作、特殊料理变体 |
| 调合(Chemistry) | 调合台(Chemistry Set) | 药品、药剂制作 |
| 手工艺(Crafting) | 工作台(Crafting Table) | 装饰品、杂项物品制作 |
| 建造(Constructing) | 木匠(Carpenter) | 村庄建筑、装饰物建造 |

## 二、配方获取系统

### 2.1 配方获取方式汇总

本作改变了传统的"吃面包学配方"机制(部分保留)，配方获取渠道更加多样化：

| 获取方式 | 对应配方类型 | 说明 |
|---------|-------------|------|
| **蛙石(Frog Statues)** | 料理食谱 | 探索地图发现蛙石，跳舞获得食谱；向Croakie汇报增加累计获得数 |
| **路边神社(Wayside Shrines)** | 装备锻造配方 | 红色 altar，跳舞获得武器/防具配方 |
| **地藏像(Jizo Statues)** | 料理食谱 | 清扫雕像上的叶子，有概率在房间中获得 |
| **弓靶(Yoichi's Bow Targets)** | 多种配方 | 射中红白靶获得配方 |
| **村庄等级提升** | 建筑/装饰配方 | 完成村庄任务提升等级自动解锁 |
| **牵绊等级提升** | 角色专属装备配方 | 例如 Mauro Lv3 解锁帽子配方 |
| **任务奖励** | 各类配方 | 完成村庄任务/委托 |
| **Recipe Bread(食谱面包)** | 传统方式(部分保留) | 从商店购买食用学习 |

### 2.2 蛙石系统详解

| 项目 | 内容 |
|------|------|
| 分布 | 东国世界各地 |
| 使用方式 | 在蛙石前跳舞(Dance of Bonds) |
| 奖励 | 随机料理食谱 |
| 累计汇报 | 向春之里的 **Croakie** 汇报蛙石发现数，获得额外食谱 |

### 2.3 路边神社系统详解

| 项目 | 内容 |
|------|------|
| 外观 | 红色鸟居/altar |
| 分布 | 各个区域 |
| 使用方式 | 跳舞解封 |
| 奖励 | **武器/防具锻造配方** |

## 三、锻造系统(Forging)

### 3.1 解锁条件

| 步骤 | 内容 |
|------|------|
| 1 | 完成"铁匠铺风波2"(Blacksmith Brouhaha 2)任务 |
| 2 | 获得铁匠铺建筑配方 |
| 3 | 建造铁匠铺(材料: 800G + Lumber x40 + Material Stone x20 + Iron x1) |
| 4 | 指派拥有 **Journeyman Smith(工匠)** 特质的村民经营 |

### 3.2 可制作物品

| 分类 | 包含项目 |
|------|---------|
| 短剑(Short Swords) | 单手剑系列 |
| 长剑(Long Swords) | 双手剑系列 |
| 双剑(Dual Blades) | 双持剑系列 |
| 弓(Bows) | 远程弓系列 |
| 肩甲(Pauldrons) | 身体防具 |
| 头盔(Headgear) | 头部防具 |

### 3.3 素材需求

| 素材 | 购买价 | 获取方式 |
|------|-------|---------|
| Iron(铁) | 950G | 采矿 |
| Bronze(青铜) | 1900G | 采矿/合成 |
| Silver(银) | 2850G | 采矿 |
| Gold(金) | 3800G | 采矿 |
| Cloth(布) | — | 魔物掉落 |
| Fletch(羽毛) | — | 魔物掉落 |
| Bamboo(竹) | — | 野外采集 |

### 3.4 铁匠铺经营

| 项目 | 内容 |
|------|------|
| 指派条件 | 村民拥有 Journeyman Smith 特质 |
| 收入机制 | 每日自动产生金币收入 |
| 收入加成 | 周边 Trading 加成的建筑/装饰提升收入 |
| 玩家使用 | 可在铁匠铺制作和升级装备 |

## 四、烹饪系统(Cooking)

### 4.1 烹饪场所

| 场所 | 说明 |
|------|------|
| **茶屋(Teahouse)** | 夏之里建造，可制作各类料理 |
| **居酒屋(Izakaya)** | 秋之里，由Yachiyo经营，可制作日式料理 |
| **野外篝火(Bonfire)** | 在野外发现，可进行基础烹饪 |

### 4.2 料理分类

| 分类 | 说明 |
|------|------|
| 快手菜(Quick Meals) | 快速制作的基础料理 |
| 一般料理(Regular Dishes) | 需要多种食材的正式料理 |
| 零食(Snacks) | 小食类 |
| 甜品(Desserts) | 甜点类 |
| 汤品(Soups) | 汤类 |
| 饮品(Drinks) | 饮料类 |
| 咖喱(Curry) | 咖喱类(参加咖喱比赛) |

### 4.3 共同烹饪系统(Lv6+)

| 项目 | 内容 |
|------|------|
| 解锁条件 | 与特定NPC牵绊等级达到**Lv6** |
| 操作方式 | 在烹饪场所按 -(Switch)/对应键 邀请NPC |
| 特殊料理 | 拥有**厨师帽图标**的NPC拥有**料理变体** |
| 永久解锁 | 共同烹饪一次后，该变体永久记录在食谱中 |
| 注意 | 若玩家选择Kaguya则不能与Kaguya共同烹饪 |

### 4.4 食材等级与替代规则

| 规则 | 说明 |
|------|------|
| 等级替代 | 可使用**任意等级**的蛋、奶等(等级不影响成品) |
| 巨大作物替代 | **巨大蔬菜**可替代普通蔬菜 |
| 不可制作食材 | 部分食材只能购买或掉落，**等级固定为Lv1** |
| 高等级食材奖励 | Lv2 面包等可通过收获祭奖励获得 |

## 六、建造系统(Constructing)

### 6.1 木匠(Carpenter)

| 项目 | 内容 |
|------|------|
| 功能 | 建造村庄建筑、装饰物 |
| 配方获取 | 村庄等级提升解锁 |
| 材料 | Lumber(木材)、Material Stone(石材)、金币 |

### 6.2 建筑分类

| 建筑类型 | 功能 | 效果 |
|---------|------|------|
| 居住建筑 | 增加村民入住空间 | 人口上限提升 |
| 商业建筑 | 店铺、茶屋、居酒屋 | 产生被动收入 |
| 生产建筑 | 铁匠铺、调合台、工坊 | 加工制造 |
| 农业建筑 | 风车、水车 | 农业加成 |
| 娱乐建筑 | 浴场、花车 | 提升满意度 |
| 装饰物 | 灯笼、雕像、园艺 | 提升风景评分 |

### 6.3 建筑加成系统

建筑和装饰提供多种加成效果：

| 加成类型 | 效果 | 来源建筑示例 |
|---------|------|-------------|
| Farming +1% | 农业产出提升 | 水车(需邻水路) |
| Trading +1% | 店铺收入提升 | 市场类建筑 |
| Mining +1% | 采矿效率提升 | 铁匠铺 |
| Max HP +10 | 生命上限提升 | 铁匠铺等 |
| Scenic Score | 风景评分提升 | 装饰类 |

## 七、神器相关制造

### 7.1 神器升级

神器本身通过**技能树**系统升级(而非铁匠铺)，但以下制造系统与神器相关：

| 神器 | 相关制造系统 |
|------|-------------|
| 圣剑(Sacred Sword) | 锻造系统可制作火属性武器，与圣剑协同 |
| 圣鼓(Sacred Drum) | 调合系统可制作增益药品配合 |
| 圣伞(Sacred Parasol) | 无直接制造关联 |
| 圣扇(Sacred Fan) | 无直接制造关联 |
| 鬼神面具 | 无直接制造关联 |
| 锡杖 | 无直接制造关联 |

### 7.2 勾玉(Magatama)制造

| 项目 | 内容 |
|------|------|
| 解锁 | 冬之里复兴主线完成后 |
| NPC | Zaza(冬之里) / Hisui(夏之里) |

## 九、制造系统技能树

### 9.1 相关技能树

| 技能树 | 解锁条件 | 推荐优先节点 |
|-------|---------|-------------|
| 烹饪(Cooking) | 初始可用 | 食材效率提升 |
| 建造(Constructing) | 完成"村长职责" | **建筑成本降低**(高优先) |
| 锻造(Forging) | "铁匠铺风波2" | **锻造成本降低**(高优先) |
| 调合(Chemistry) | "冬之里复兴2" | **调合成本降低**(高优先) |

### 9.2 成本降低技能的价值

在1.1.0版本中，制造相关技能树的"材料成本降低"被玩家社区公认为**最高优先级的技能**，原因：
- 铁匠铺矿石价格高昂(金: 3800G)
- 建筑建造需要大量素材
- 节省的成本在游戏全程累计效益巨大
