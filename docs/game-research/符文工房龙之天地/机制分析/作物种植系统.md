# 作物种植系统
![龙之天地农耕](https://runefactory-gaz.com/img/system/ss_farm.jpg)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

---

## 一、系统总览 —— 本作最大变革

《龙之天地》的种植系统是系列历史上**变革幅度最大的系统**。核心变化如下：

| 变化项目 | 前作(RF4/RF5) | 本作 |
|---------|--------------|------|
| 季节 | 世界整体旋转四季 | **4个固定季节村庄**，无季节更替 |
| RP消耗 | 锄地/浇水/收获均消耗RP | **基础农活不消耗RP**，仅神器技能消耗 |
| 肥料系统 | 存在(肥料/巨化剂) | **已取消**，无传统肥料系统 |
| 出货箱 | 统一出货箱 | **已取消**，改为商店经营+直接交易 |
| 农场管理 | 玩家个人管理 | **村庄开发制**，可指派村民协助 |
| 水田 | 无 | **新增水田+水路系统** |
| 种植视角 | 第三人称 | 新增**俯瞰模式**支持大规模规划 |

## 二、四个固定季节村庄

### 2.1 四季村庄定位

| 村庄 | 固定季节 | 种植特点 |
|------|---------|---------|
| 春之里(Spring) | 春季(常年) | 春作物最佳，樱花环境 |
| 夏之里(Summer) | 夏季(常年) | 夏作物最佳，高温环境 |
| 秋之里(Autumn) | 秋季(常年) | **秋作物/水田最佳**，产量最高 |
| 冬之里(Winter) | 冬季(常年) | 冬作物最佳，雪自动灌溉 |

### 2.2 季节对作物的影响

- 作物有 **推荐种植村庄**，种在非推荐村庄时**产量显著降低**
- 作物**不会死亡**，即使在非推荐村庄也能生长，只是产量减少
- 四个村庄同时可种植不同季节作物，实现全年多季节产出

**产量差异示例：**

| 作物 | 推荐村庄 | 推荐村产量 | 非推荐村产量 |
|------|---------|-----------|-------------|
| 水稻(Rice) | 秋之里 | ~24-25单位/块 | ~16-17单位/块 |
| 草莓(Strawberry) | 春之里 | 高 | 中 |
| 粉红蜜瓜 | 夏之里 | 高 | 中 |
| 柑橘 | 冬之里 | 高 | 中 |

## 三、水田系统(Rice Paddy System)

本作**全新加入**的系统，是日式农业的核心特色。

### 3.1 解锁条件

1. 推进主线至 **秋之里复兴第2阶段**
2. 完成"怪兽羁绊"任务
3. NPC **Yachiyo(八千代)** 赠送水田配方、水路配方和水稻苗

### 3.2 水田与普通农田的区别

| 项目 | 普通农田(Field) | 水田(Rice Paddy) |
|------|---------------|-----------------|
| 适用作物 | 所有普通作物 | **仅水稻/糯米** |
| 浇水需求 | 每日需要浇水 | 相邻水路时自动灌溉 |
| 解锁时间 | 初期可用 | 中期解锁 |
| 产量 | 中等 | **较高** |
| 特殊需求 | 无 | 需要规划和放置水路 |

### 3.3 水路系统(Waterway System)

水路是水田的配套基础设施，核心机制：

- 将**水路地块与水田地块相邻放置**，水田自动灌溉，无需手动浇水
- **必须直接相邻**才生效(水路 tile 紧贴水田 tile)
- 无水路时：玩家或村民必须每天手动浇水，否则水稻会枯萎
- 可用 **圣伞(Sacred Parasol)** 进行批量浇水作为替代

> 冬之里因常年下雪，会自动灌溉田地和部分水田，但不建议在冬之里种植水稻(产量极低)。

### 3.4 水车(Watermill)

| 项目 | 内容 |
|------|------|
| 解锁条件 | 冬之里等级2(冬之里复兴第3阶段任务) |
| 效果 | 农业产出 **Farming +1%** |
| 放置要求 | **必须放置在水路旁边** |
| 建材需求 | Lumber x20 + Material Stone x5 + 500G |
| 扩展 | 可在多个村庄建造，效果叠加 |

## 四、神器与种植联动

六大神器中的四种直接用于农业操作，是本作种植系统的核心创新。

### 4.1 神器农业功能一览

| 神器 | 农业功能 | RP消耗 | 解锁时机 |
|------|---------|--------|---------|
| **圣鼓(Sacred Drum)** | 加速作物生长1天/株/日、复活枯萎作物 | 消耗 | 初期 |
| **圣剑(Sacred Sword)** | 成熟作物→种子转换、**概率提升种子等级** | 消耗 | 初期 |
| **圣扇(Sacred Fan)** | 直线范围一次性批量收获 | 消耗 | 初期 |
| **圣伞(Sacred Parasol)** | 大范围灌溉(水田/农田) | 消耗 | 初期 |
| 鬼神面具 | 粉碎巨石(清理耕地) | 消耗 | 中期 |
| 锡杖 | 解谜相关(无直接农业用途) | 消耗 | 中期 |

### 4.2 圣鼓详解 —— 催熟机制

| 机制 | 说明 |
|------|------|
| 效果 | 对一株作物使用，生长周期**缩短1天** |
| 限制 | **每株作物每天限用1次** |
| 复活 | 可复活枯萎/枯萎中的作物 |
| 战略 | 用于加速高价值作物，或赶在节日比赛前成熟 |

### 4.3 圣剑详解 —— 种子升级循环

圣剑取代了前作的肥料系统，成为提升作物品质的**唯一手段**。

```
收获流程：
  [成熟作物] → 使用圣剑 → [获得种子(等级可能+1)] → [播种高级种子] → [收获高级作物]
                                                                        ↓
                                                                 循环提升至Lv9
```

| 种子等级(Lv) | 效果 |
|-------------|------|
| Lv1 | 基础品质 |
| Lv2-Lv4 | 中等品质，产量和售价提升 |
| Lv5-Lv8 | 高品质，显著提升 |
| Lv9 | **最高品质** |

> 注：本作**取消了肥料和巨化剂系统**，种子等级提升只能依靠圣剑循环。

### 4.4 圣扇详解 —— 批量收获

| 机制 | 说明 |
|------|------|
| 范围 | 直线方向的**多个作物** |
| 效率 | 一次性收获范围内所有成熟作物 |
| 搭配 | 配合俯瞰模式可快速规划大面积收获 |

### 4.5 圣伞详解 —— 范围灌溉

| 机制 | 说明 |
|------|------|
| 范围 | 大范围圆形区域 |
| 效果 | 一次灌溉范围内所有需要浇水的作物 |
| 无水路替代 | 在没有铺设水路的农田，可作为主要灌溉手段 |

## 五、基础农活与RP机制改革

### 5.1 RP(符文点数)消耗规则

本作对RP系统进行了根本性改革：

| 操作 | 是否消耗RP | 说明 |
|------|-----------|------|
| 锄地(Tilling) | **不消耗** | 自由开垦耕地 |
| 播种(Planting) | **不消耗** | 直接种植 |
| 浇水(Watering) | **不消耗** | 使用圣伞时也不消耗 |
| 收获(Harvesting) | **不消耗** | 手动收获或圣扇收获 |
| 圣鼓催熟 | **消耗RP** | 加速生长需消耗 |
| 圣剑转种子 | **消耗RP** | 种子升级需消耗 |
| 圣扇批量收获 | **消耗RP** | 范围收获技能消耗 |
| 圣伞灌溉 | **消耗RP** | 范围灌溉技能消耗 |

> 核心逻辑：**基础农业劳动完全免费**，只有使用神器的"技能"部分消耗RP。这大大降低了农业的门槛，鼓励玩家自由耕种。

### 5.2 农具改革

本作取消了传统独立农具(锄头、水壶、镰刀、斧头、锤子)的设定：

| 传统农具 | 本作替代方案 |
|---------|-------------|
| 锄头(Hoe) | **Terra Tiller(神圣耕具)** —— 村庄建造模式中使用 |
| 水壶(Watering Can) | **圣伞(Sacred Parasol)** + 水路自动灌溉 |
| 镰刀(Sickle) | **圣剑(Sacred Sword)** 生成种子 + **圣扇(Sacred Fan)** 批量收获 |
| 斧头(Axe) | **简易锤子(Cheap Hammer)** 采集木材 |
| 锤子(Hammer) | **Common Hatchet** 采集矿石 |

## 六、作物品质与等级系统

### 6.1 作物等级(Crop Level)

| 等级范围 | 获取方式 | 效果 |
|---------|---------|------|
| Lv1 | 商店购买种子 | 基础品质 |
| Lv2-Lv9 | 圣剑升级循环 | 更高的预估产量(Estimated Yield)和售价 |

### 6.2 巨大作物(Giant Crops)

| 机制 | 说明 |
|------|------|
| 触发条件 | **随机**，在作物收获当天判定 |
| 培育技巧 | 在作物还剩1天收获时，**不要使用圣鼓加速**，让其自然过夜 |
| 存档技巧 | 睡前存档，第二天早上查看是否巨大化，否则读档重试 |
| 售价 | 巨大作物售价大幅提升(如粉红蜜瓜→征服者蜜瓜67G→405G) |
| 金色作物 | 全域可种，200G基础，巨大化可达1000G |

> 与系列前作不同，本作**无法通过巨化剂(Giantizer)催生巨大作物**。

### 6.3 预估属性

| 属性 | 说明 |
|------|------|
| 预估产量(Estimated Yield) | 收获时能获得的作物数量，受村庄加成影响 |
| 预估价值(Estimated Value) | 出售价格，受作物等级影响 |

## 七、俯瞰模式(Overhead View)

### 7.1 功能说明

| 项目 | 内容 |
|------|------|
| 进入方式 | 村庄建造模式下按右摇杆(手柄)/T键(键盘) |
| 视角 | 鸟瞰俯视 |
| 功能 | 大规模种植/浇水/收获规划 |
| 优势 | 一次性掌握整片农田状况 |
| Switch2 | 支持鼠标模式操作 |

### 7.2 实用技巧

- 进入建造模式后**时间暂停**，可从容规划
- 俯视模式下可快速切换建筑和农田布局
- 俯瞰模式+圣扇组合实现最高效批量收获

## 八、村民委任系统

### 8.1 委任内容

| 农业任务 | 说明 |
|---------|------|
| 播种(Sowing) | 村民自动播种(1.0.5版起可单独开关) |
| 浇水(Watering) | 村民自动浇水(1.0.5版起可单独开关) |
| 收获(Harvesting) | 村民自动收获(1.0.5版起可单独开关) |

### 8.2 委任效率

| 村民特质 | 农业效率 |
|---------|---------|
| 无农业特质 | 基础效率(较慢) |
| 有农业特质 | 高效操作 |
| 高牵绊等级 | 操作范围更大 |

> 注意：农业委任村民效率相对较低，高价值作物建议玩家自行管理。

### 9.2 种植全流程

```
1. 准备阶段
   ├─ 选择种植村庄(考虑推荐季节)
   ├─ 在开发区域放置 Farm(农田) 地块
   ├─ (水稻)放置 Rice Paddy + Waterway 水路
   └─ 购买种子

2. 播种阶段
   ├─ 手持种子点击农田
   ├─ 可委任村民自动播种
   └─ 不消耗RP

3. 成长期
   ├─ 每日浇水(圣伞或水路自动灌溉)
   ├─ 可多次使用圣鼓加速生长(每株每天1次)
   ├─ 复活枯萎作物(圣鼓)
   └─ 不消耗RP(基础农活)

4. 收获阶段
   ├─ 成熟时手动收获或圣扇批量收获
   ├─ 使用圣剑生成种子(概率升级)
   ├─ 随机判定是否巨大化
   └─ 不消耗RP(基础收获)
```
