| 评价 | 计算方式 | 对应结局类型 |

## 3. 结局判定算法(七步优先级)

| 步骤 | 判定 | 条件 |
|:--:|------|------|
| **1** | **罪孽型** | 道德=0且罪孽≥100 **或** 道德<30且罪孽≥250 |
| 2 | 艺术型 | 艺术修养≥80且最高 |
| 3 | 基本型(职业) | **四项评价最高-最低≤50**(均衡) |
| 4 | 战士型 | 战士评价最高(>其他+50) |
| 5 | 魔法型 | 魔法评价最高(>其他+50) |
| 6 | 社交型 | 社交评价最高(>其他+50) |
| 7 | 家事型 | 家务评价最高(>其他+50) |

## 4. 关键阈值

| 结局 | 关键阈值 |
|------|:--:|
| **女王(Ruling Queen)** | 气品≥**800** + 四项≥421 + 均衡(+50内) + 道德≥500 |
| 勇者(Hero) | 战士评价≥**421** + 道德≥600 + 感性>智力 |
| 魔法勇者 | 魔法评价≥**421** + 道德≥50 |
| 暗黑结局 | 罪孽≥100(或≥250) |

## 5. 设计特点(品类模板奠基)

1. **七步优先级判定=结局系统模板**: 不是"选结局"而是"属性+评价自动判定"——火山的女儿的58结局系统由此衍生
2. **气品800=品类中第一个"超高门槛"**: 女王结局需要气品800——驱动玩家专门规划8年路线
3. **评价=属性组合**: 不是单属性判定——而是属性加权组合→更真实的能力评估
4. **罪孽=隐藏道德维度**: 不直接显示为"黑暗值"——通过打工/事件间接积累
