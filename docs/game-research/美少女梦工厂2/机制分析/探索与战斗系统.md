# 探索与战斗系统 - 美少女梦工厂2(Princess Maker 2)

> 游戏版本: Princess Maker 2 Refine (PC, 1993/2016年)
> 数据来源: princessmaker.fandom.com

## 1. 概述

PM2是Princess Maker系列中**唯一有完整战斗系统**的作品。女儿可以独自外出探险，进行回合制战斗，获得宝物和金币。这个系统后来被PM3/PM5放弃——使PM2的战斗系统成为系列中独特的遗产。

## 2. 探索机制

| 参数 | 说明 |
|------|------|
| 类型 | 回合制RPG战斗 |
| 触发 | 安排"自由行动"→选择"探险" |
| 消耗 | 体力(Stamina) |
| 区域 | 多个区域(森林/洞窟/山脉等)→难度递增 |
| 奖励 | 金币+宝物+战士/魔法经验 |

## 3. 战斗属性

| 属性 | 影响 |
|:--:|------|
| 战士经验(Combat) | 物理攻击力/命中 |
| 魔法经验(Magic) | 魔法攻击力/MP |
| 体力(Stamina) | HP |
| 装备(武器/防具) | 攻击/防御加成 |

## 4. 装备系统

| 类型 | 获取 | 效果 |
|:--:|------|------|
| 武器(剑/弓/法杖等) | 购买/探索/奖励 | 攻击力+ |
| 防具(铠甲/袍等) | 购买/探索 | 防御力+ |
| 饰品 | 购买/事件 | 特殊效果 |

## 5. 设计特点

1. **战斗=PM2独有的系统**: PM3/PM5移除了战斗——使PM2在系列中更接近"RPG+养成"
2. **战斗→战士评价→结局**: 战斗经验直接影响战士评价→战士系结局——战斗系统与结局联动
3. **比武大会=战斗的社交化**: 王子嫁需要比武大会优胜——战斗不仅是属性成长也是婚姻条件
4. **影响火山的女儿**: 火山的女儿的冥滩战斗系统是PM2战斗的现代复兴
