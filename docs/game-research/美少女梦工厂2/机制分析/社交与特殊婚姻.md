# 社交与特殊婚姻系统 - 美少女梦工厂2(Princess Maker 2)

> 游戏版本: Princess Maker 2 Refine (PC, 1993/2016年)
> 数据来源: princessmaker.fandom.com, 贴吧

## 一、社交系统

### 1.1 社交方式

| 方式 | 频率 | 效果 |
|:--:|:--:|------|
| 父女对话 | **每月1次** | 道德+/亲密度+(选择影响) |
| 城镇社交 | 自由行动 | 遇到NPC/事件触发 |
| 比赛 | 年度活动 | 声望+/遇到特定角色 |
| 礼物 | 获取→赠送 | NPC好感+ |

### 1.2 关键NPC

| NPC | 身份 | 好感影响 |
|------|:--:|------|
| 吉普(Cube) | 管家 | 吉普嫁结局(亲密度最高) |
| 王子(Prince) | 王国继承人 | **王子嫁结局** |
| 魔王(Demon Lord) | 魔界统治者 | 魔王嫁结局 |
| 小龙(Dragon) | 幼龙 | 龙嫁结局 |
| 父亲(Father) | **玩家角色** | **父嫁结局**(亲密度最高) |

## 二、特殊婚姻结局(5种)

### 王子嫁(最高难度)

| 条件 | 要求 |
|------|:--:|
| 气品 | 极高(700+) |
| 魅力 | 极高 |
| 道德 | 高(500+) |
| 比武大会 | **优胜**(战士评价需极高) |
| 触发事件 | 与王子多次相遇→好感累计 |

### 父嫁(最特殊)

| 条件 | 要求 |
|------|:--:|
| 亲密度 | **最高**(与父亲关系) |
| 道德 | 偏低(不是极低) |
| 部分评价 | 达到一定标准 |
| 触发 | 特定事件+亲密度阈值 |

### 吉普嫁

| 条件 | 要求 |
|------|:--:|
| 与吉普亲密度 | **最高** |
| 日常互动 | 长期与管家互动 |

### 魔王嫁

| 条件 | 要求 |
|------|:--:|
| 罪孽 | 高(100+) |
| 魔法评价 | 高 |
| 特定事件 | 触发魔界相关事件 |

### 龙嫁

| 条件 | 要求 |
|------|:--:|
| 特定事件 | 龙相关事件 |
| 魔法相关 | 魔法属性/事件 |

## 三、设计特点(品类模板奠基)

1. **父嫁=品类最具争议的结局**: "父亲娶养女"——在当时(1993年)引起巨大争议和话题——也是最被记住的结局
2. **特殊婚姻=独立于职业结局**: 婚姻和职业是两条线——火山的女儿继承了这个设计
3. **比武大会=战斗×社交**: 王子嫁需要比武大会优胜——将战斗系统和社交系统连接
4. **5种婚姻=最早的"恋爱游戏": PM2比大多数恋爱游戏更早——是GalGame的早期先驱之一
