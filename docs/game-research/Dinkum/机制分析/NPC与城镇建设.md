# NPC与城镇建设系统 - Dinkum

> 游戏版本: Dinkum (PC/Switch, 2025年)
> 数据来源: dinkum.fandom.com, AsiaGameMaster

## 1. NPC入驻机制

| 阶段 | 说明 |
|:--:|------|
| 访问 | NPC首次访问→在岛上停留→接到任务 |
| 复访 | 多次来访→完成更多任务→好感度提升 |
| **入居** | 好感度足够→NPC申请永久入居→成为居民 |

- 每日任务有惩罚: 未完成→扣好感度
- 放弃任务→不扣分(优于失败)

## 2. 关键NPC

| NPC | 功能 | 解锁后 |
|------|------|------|
| Fletch | 许可证商人/基地管理员 | 全部许可证 |
| (其他NPC待补充) | 动物商店/博物馆/工具升级等 | 对应商店+功能 |

## 3. 城镇建设

### 3.1 布局策略

| 区域 | 说明 |
|:--:|------|
| **城镇中心** | 帐篷放地图中央→减少跑图时间 |
| 玩家帐篷 | 不要离城镇太远 |
| 住宅区 | NPC房屋 |
| 商业区 | 商店集中 |
| 农业区 | 种植+养殖 |
| 工业区 | 加工设施 |

### 3.2 建设许可证

| 许可证 | 功能 |
|------|------|
| 景观Lv1 | 路径/台阶/坡道 |
| 景观Lv2 | 围栏/大门 |
| 建筑Lv1 | 桥梁 |
| 建筑Lv2 | **风车(加速生产!)** |
| 建筑Lv3 | 高级工作台 |

### 3.3 装饰

- 围栏/花坛/路灯/蜂箱/风车/喷泉
- 城镇开心度↑→NPC更容易入居

## 4. 设计特点

1. **动森式NPC入居**: NPC不是"一开始就住在这里"——需多次访问+好感
2. **自由地形改造**: 可完全自定义城镇布局——不是"固定格子"
3. **惩罚机制**: 任务失败扣好感→与动森的宽容不同
4. **多人协作**: 最多4人联机→分工完成NPC任务+建设
