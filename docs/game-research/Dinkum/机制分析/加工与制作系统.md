# 加工与制作系统 - Dinkum

> 游戏版本: Dinkum (PC/Switch, 2025年)
> 数据来源: dinkum.fandom.com, AsiaGameMaster

## 1. 加工增值

| 原料 | 直接出售 | 加工品 | 加工出售 | 增值 |
|------|:----:|------|:----:|:--:|
| 铜矿石 | ~100 Dinks | 铜锭 | ~400 Dinks | ×4 |
| 硬木 | ~30 Dinks | 硬木板 | ~120 Dinks | ×4 |
| 香蕉 | ~50 Dinks | 烤香蕉 | ~100 Dinks | ×2 |

## 2. 工作站

| 工作站 | 功能 | 解锁 |
|------|------|:--:|
| 基础工作台 | 基础制作 | 初始 |
| 精炼厂 | 原料→高级材料 | 中期 |
| 风车(Windmill) | **加速生产** | 建筑Lv2 |
| 高级工作台 | 高级制作 | 建筑Lv3 |

## 3. 制作类别

| 类别 | 代表 |
|------|------|
| 工具 | 镐/斧/钓竿/虫网 |
| 武器 | 基础→高级 |
| 建筑 | 桥梁/风车/围栏 |
| 家具 | 房屋+城镇装饰 |
| 消耗品 | 料理/药水 |

## 4. 设计特点

1. **加工=×4价值**: 与守墓人相同的加工增值逻辑
2. **风车=加速器**: 类似守墓人的僵尸——加快生产速度
3. **制作=消耗许可证**: 高级制作需先买建筑许可证
