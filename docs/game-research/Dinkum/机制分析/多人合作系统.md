# 多人合作系统 - Dinkum

> 游戏版本: Dinkum (PC/Switch, 2025年)
> 数据来源: dinkum.fandom.com, AsiaGameMaster

## 1. 联机参数

| 参数 | 设定 |
|------|:--:|
| 最大玩家数 | **4人** |
| 模式 | 本地+在线 |
| 访客限制 | **无法完成自己的任务**(仅协助) |

## 2. 协作内容

| 活动 | 协作效果 |
|------|------|
| 建造 | 建材收集速度翻倍+ |
| 采矿 | 矿洞探险→多人安全 |
| 打猎 | 合作打鳄鱼→更容易 |
| NPC任务 | 分工完成 |
| 购物 | 可互相交易 |
| 赛跑 | 骑鸸鹋赛跑 |

## 3. 设计特点

1. **访客=帮手**: 不是"每个人经营自己的农场"——而是"大家一起经营同一个城镇"
2. **访客任务不保存**: 访客做的是主机世界的任务→自己的进度不受影响
3. **建材加倍**: 多人收集→城镇建设速度大幅提升
