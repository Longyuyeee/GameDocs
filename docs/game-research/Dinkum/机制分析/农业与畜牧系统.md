# 农业与畜牧系统 - Dinkum

> 游戏版本: Dinkum (PC/Switch, 2025年)
> 数据来源: dinkum.fandom.com, AsiaGameMaster

## 1. 农业

### 1.1 定位: 辅助(非核心!)

**新手最大误区**: Dinkum不是星露谷——采矿/探索是核心，农业是后期辅助。

### 1.2 作物

| 作物 | 价值 | 季节 |
|------|:--:|:--:|
| 基础作物 | 低 | 多季 |
| 西瓜(Watermelon) | 高 | 夏 |
| 辣椒(Chili) | 高 | 夏 |
| 香蕉(Banana) | 中(加工后×2) | 多季 |

### 1.3 灌溉

- 洒水器 = **灌溉许可证解锁后才有效**
- 无洒水器前→手动浇水→极度耗时→不建议大面积种植
- 等灌溉许可后再规模化

## 2. 畜牧

| 动物 | 解锁 | 产出 |
|------|:--:|------|
| 鸡(Chooks) | Handling Lv1 | 蛋 |
| 袋熊(Vombats) | Handling Lv2 | 毛皮 |
| Pleep | Handling Lv3 | 特殊 |
| 袋牛/朋克鸟/猪狸 | 2025更新 | 新增 |

## 3. 设计特点

1. **农业后置**: 明确的设计——采矿→赚钱→再务农
2. **灌溉=农业门槛**: 不是"开始就能高效种地"——必须等灌溉许可
3. **加工增值**: 香蕉→烤香蕉×2——简单加工链
