# 探索与危险系统 - Dinkum

> 游戏版本: Dinkum (PC/Switch, 2025年)
> 数据来源: dinkum.fandom.com, AsiaGameMaster

## 1. 澳洲荒野主题

Dinkum的独特之处在于它的"荒野生存"基调——不是温暖的田园而是危险的澳洲内陆。

## 2. 危险动物

| 动物 | 威胁 | 对策 |
|------|:--:|------|
| **鳄鱼(Crocodile)** | ★★★★★ | **前期绕路!** 攻击强/速度快 |
| 鲨鱼(Shark) | ★★★★★ | 水域避开 |
| 鹤/袋鼠(Crane/Kangaroo) | ★★★ | 会冲撞→侧闪 |
| 食火鸡(Cassowary) | ★★★ | 类似鸸鹋的危险鸟 |

## 3. 天气事件

| 天气 | 效果 |
|------|------|
| 雨天 | 作物免浇水 |
| 雷暴 | 落雷→**雷电蛋**(极高价值!) |
| 流星雨 | 跟随最慢流星→陨石→可能银河服装 |

## 4. 移动方式

| 方式 | 解锁 | 功能 |
|------|------|------|
| 步行 | 初始 | — |
| 滑翔伞(Glider) | 后期 | 空中移动/三层矿洞必需 |
| 车辆(Vehicles) | 后期 | 快速移动(推测) |

## 5. 设计特点

1. **危险=生存感**: 不是"牧场经营"的安逸——而是"荒野开荒"的警觉
2. **雷电蛋=天气暴富机会**: 雷暴→落雷产特殊蛋→极高价值
3. **鳄鱼=新手第一课**: 看到鳄鱼绕路——不是所有东西都要打
