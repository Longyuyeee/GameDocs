# 许可与技能系统 - Dinkum

> 游戏版本: Dinkum (PC/Switch, EA 2022-2025, 正式版2025年4月)
> 数据来源: dinkum.fandom.com, NeoSeeker, AsiaGameMaster

## 1. 许可系统(Licenses)——核心进度控制

许可是Dinkum的"科技树"——所有进阶行为必须先购买许可证。

| 优先级 | 许可证 | 费用(PP) | 功能 |
|:--:|------|:-----:|------|
| ★★★★★ | 采矿(Mining) | 250 | 镐子→矿石→**中后期最强赚钱** |
| ★★★★★ | 伐木(Logging) | 250 | 斧头→木材→建材基础 |
| ★★★★ | 钓鱼(Fishing) | 250 | 钓竿→**前期最稳定收入** |
| ★★★★ | 狩猎(Hunting) | 250 | 武器→防身+任务 |
| ★★★ | 挖掘(Excavation) | 500 | 铲子→种植/挖掘 |
| ★★★ | 金属探测(Metal Detecting) | 500 | 寻宝→化石/高价值物品 |
| ★★ | 农业(Farming) | 250+ | **前期不要!** 等灌溉许可后再规模 |
| ★★ | 畜牧(Handling) | 递进 | Lv1鸡/Lv2袋熊/Lv3 Pleep |

> 新手最常犯错: 过早解锁农业——没有洒水器时浇水极度耗时

## 2. 许可购买NPC

- Fletch(基地NPC)——出售所有许可证
- 花费PP(Permit Points/许可点数)
- PP通过完成任务/出售物品获得

## 3. 技能系统(7种)

| 技能 | 需许可? | 升级方式 | 10级解锁 |
|------|:--:|------|------|
| 采矿(Mining) | ✓ | 开采矿石 | — |
| 伐木(Logging) | ✓ | 砍树 | — |
| 钓鱼(Fishing) | ✓ | 钓鱼 | — |
| 狩猎(Hunting) | ✓ | 猎捕动物 | — |
| 农业(Farming) | ✓ | 种植收获 | — |
| 觅食(Foraging) | ✗ | 采集野生资源 | — |
| 捕虫(Bug Catching) | ✗ | 捉虫 | Lv10→蜂箱/Lv20→蠕虫农场 |

## 4. 设计特点

1. **许可=渐进的复杂度**: 不是"开始就能做所有事"——而是逐步解锁→避免新手被选项淹没
2. **PP=行为驱动**: 不是"时间到了自动获得"——而是"完成游戏行为获得"→做得多解锁得快
3. **农业最后解锁**: 明确的设计信息——"这不是星露谷, 采矿/探索才是核心"
