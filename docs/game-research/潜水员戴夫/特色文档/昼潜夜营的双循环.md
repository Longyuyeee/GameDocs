# 昼潜夜营的双循环设计 - 潜水员戴夫(Dave the Diver)

> 游戏版本: Dave the Diver v1.0+ (PC/Switch, 2023年)

## 一、概述

Dave the Diver的"昼潜夜营"双循环是服务端经营品类中最优雅的设计之一。两个阶段通过"食材→菜单→金币→装备→更好食材"的正向螺旋无缝连接。

## 二、双循环结构

```
[白天] 潜水(Blue Hole):
  捕鱼→食材
  采集→资源
  探索→剧情推进
    ↓ 食材+资源
[夜晚] 寿司店(Bancho Sushi):
  设计菜单(基于白天捕获)
  服务顾客(速度+品质)
  赚取金币+Artisan's Flame
    ↓ 金币+研究
[升级] 装备/员工/菜单强化
    ↓ 更强的潜水装备+更好的员工
[明天] 更深的潜水→更好的鱼→更贵的寿司→更多金币
```

## 三、两个系统的资源联动

| 潜水产出 | 寿司店使用 |
|------|------|
| 鱼(各种品质) | 菜单食材(3星=最高价格) |
| 海藻/海产 | 特殊食谱材料 |
| 金币(宝箱) | 直接收入 |
| 剧情物品 | 推进主线→解锁新内容 |

| 寿司店产出 | 潜水使用 |
|------|------|
| 金币 | 装备升级/员工培训/招聘 |
| Artisan's Flame | 研究新食谱→更大的菜单 |
| Cooksta评级 | 解锁新内容/吸引VIP |
| 员工 | 派遣采集调味料→减少潜水负担 |

## 四、与咩咩启示录双循环的对比

| 维度 | 咩咩启示录 | 潜水员戴夫 |
|------|:--:|:--:|
| 白天 | 地牢探险 | **潜水捕鱼** |
| 夜晚 | 管理教派 | **经营寿司店** |
| 时间流逝 | 持续(不管你在哪) | **分阶段**(潜水/营业各自独立) |
| 惩罚 | 营地可能崩溃 | 潜水死=几乎全丢 |
| 核心资源 | 信徒+信仰 | **食材+Cooksta评级** |
| 基调 | 黑暗/讽刺 | 轻松/幽默 |

## 五、设计分析

1. **食材=终极纽带**: 不是"去地牢找东西卖"——而是"今天我捕了什么鱼=今晚菜单上有什么"——无缝的叙事+机制连接
2. **完美捕获=双重收益**: 3星鱼→寿司店更高价格+烹饪更多肉→激励技巧>暴力
3. **Cooksta=品质驱动**: 不是"赚最多的钱"而是"获得最好的评价"——寿司店的"味道>价格"策略
4. **夜间=白天成果的展示**: 白天捕的好鱼→晚上的寿司卖出高价——即时正向反馈
