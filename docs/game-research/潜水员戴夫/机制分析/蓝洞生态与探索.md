# 蓝洞生态与探索 - 潜水员戴夫(Dave the Diver)

> 游戏版本: Dave the Diver v1.0+ (PC/Switch, 2023年)
> 数据来源: Casual Game Guides, IGN

## 1. 蓝洞(Blue Hole)——每日变化的地形

| 属性 | 说明 |
|------|------|
| 地形 | **每天随机变化**(Roguelite式的每日新地图) |
| 深度 | 浅水(0-50m)→中层(50-150m)→深层(150m+) |
| 生态 | 不同深度=不同鱼种+不同危险 |

## 2. 探索策略

| 深度 | 风险 | 鱼种价值 | 推荐装备 |
|:--:|:--:|:--:|------|
| 浅水 | 低 | 低(蓝唐鱼/水母) | 基础 |
| 中层 | 中 | 中(大型鱼/鲨鱼) | 麻醉枪 |
| 深层 | **高** | **高**(Boss/稀有鱼/海人) | 满O2+最佳武器 |

## 3. 夜间潜水

- 剧情解锁后→不同鱼类
- 独特遭遇(白天没有)

## 4. 资源节点

| 类型 | 获取 |
|------|------|
| 鱼类(各种) | 捕鱼 |
| O2罐 | 随机节点→恢复O2 |
| 蛤壳(Clamshells) | 含空气→恢复O2 |
| 宝箱 | 金币+稀有物品 |
| 海藻/珊瑚 | 特殊食谱材料 |

## 5. 设计特点

1. **每日随机地形=无限重玩性**: 不是"固定地图"——每天潜水都是新的
2. **三层深度=自然的难度梯度**: 不是"升级了才能去深层"而是"装备够好才敢去"
3. **O2罐散布=风险缓冲**: 节点不是无限的→必须规划路线
