# 装备与升级系统 - 潜水员戴夫(Dave the Diver)

> 游戏版本: Dave the Diver v1.0+ (PC/Switch, 2023年)
> 数据来源: Casual Game Guides, IGN, NoobFeed

## 1. 潜水装备

| 装备 | 功能 | 升级效果 |
|------|------|------|
| 潜水服(Diving Suit) | 基础防护 | 减少伤害 |
| O2瓶(O2 Tank) | **核心!** 氧气容量 | 更长时间+更多缓冲 |
| 负重背包(Weight Capacity) | 更多鱼→更多金币 | **最先升!** |
| 鱼叉(Harpoon) | 基础捕鱼 | 伤害/射程/可靠性 |
| 潜水无人机(Drone) | 拖最大鱼 | 容量升级 |
| 速度鞋(Fins) | 游速 | 效率 |

## 2. 武器(Duff制作)

| 武器 | 制作材料 | 适合场景 |
|:--:|------|------|
| 基础鱼叉 | — | 通用 |
| 麻醉枪 | 贝壳+矿石 | **大型鱼/鲨鱼活捉** |
| 网枪 | 纤维+材料 | **小型鱼群活捉** |
| 榴弹发射器 | 稀有材料 | Boss/最高伤害 |
| 每次潜水**仅1把武器** | — | **根据目标选择!** |

## 3. 寿司店装备

| 升级 | 效果 |
|------|------|
| 芥末台(Wasabi) | 补芥末(耗尽=不能做菜) |
| 厨房设备 | 做菜速度 |
| 餐桌 | 客容量 |

## 4. 设计特点

1. **O2=HP=时间=三重资源**: 战斗/探索/货币——一切来自O2管理
2. **单武器=策略选择**: 每次潜水前决定——今天是捕大鱼(麻醉枪)还是扫小鱼群(网枪)
3. **负重=终极优先**: 更多鱼=更多金币——经济与探险的直接关联
