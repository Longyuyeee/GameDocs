# 寿司店经营系统 - 潜水员戴夫(Dave the Diver)

> 游戏版本: Dave the Diver v1.0+ (PC/Switch, 2023年)
> 数据来源: NoobFeed Restaurant Guide, Casual Game Guides, IGN

## 1. 菜单设计

| 要素 | 说明 |
|------|------|
| 设置时间 | 每晚营业前 |
| 核心策略 | **味道>价格**→好味道=更多顾客=更高Cooksta评级 |
| 多样化 | 尽可能供应多种菜品(利用库存) |
| 实际库存 | 只上菜单有足够食材的菜品 |

## 2. 营业操作

| 操作 | 效果 |
|------|------|
| 送餐 | 快速送到对应桌子 |
| 倒茶(Green Tea) | **完美倒茶=最高金币** |
| 清理桌子 | 客人离开后→腾出空位 |
| **补充芥末(Wasabi)** | **芥末耗尽=不能做任何菜!**——最优先! |
| 红条顾客 | 等太久→红条满→离开→**降低Cooksta评级** |

## 3. Cooksta评级系统

| 等级 | 解锁 |
|:--:|------|
| 铜(Bronze) | 基础 |
| 银(Silver) | 更多菜单位/新食谱 |
| 金(Gold) | 更多员工位/新食谱 |
| 白金(Platinum) | VIP客人 |
| **钻石(Diamond)** | **全部解锁** |

- 获取: 高味道评分+快速服务
- Artisan's Flame: 每夜评星→用于**研究新食谱**

## 4. 菜品强化(Enhance)

- 永久提升**味道+价格**(不影响原料成本)
- 优先: 基础高销量菜品(蓝唐鱼/水母/鹦鹉鱼)
- 策略: 专注强化1-2道主力菜品→不要太分散

## 5. 食谱获取

| 类型 | 解锁方式 |
|:--:|------|
| 基础 | 捕获对应鱼→自动解锁(1种食材) |
| 剧情 | 主线推进 |
| Cooksta | 新评级+Artisan's Flame研究 |
| Boss | 击败Boss(已满级→无需研究) |
| VIP | 限时VIP客人服务 |
| 员工 | 员工Lv5/10/15→解锁特色食谱 |
| 活动 | 限时活动(黄瓜派对/金枪鱼派对等) |

## 6. 金币策略

| 策略 | 说明 |
|------|------|
| **紧缩菜单** | 1-2道高价值菜→稳定供应 |
| **强化优先** | 先强化主力菜→再扩展菜单 |
| 活捉=更高利润 | 3星完美捕获→更多肉+更贵价格 |
| 复盘数据 | 查看每夜统计→最佳菜品→针对性捕鱼 |

## 7. 设计特点

1. **菜单设计=库存管理**: 不是"有多少卖多少"——而是"只上今晚供应得起的菜"——库存不足=菜单失效
2. **芥末=最核心的资源**: 芥末=能做菜的能力——芥末耗尽=全厨房停摆=顾客愤怒
3. **Cooksta=社交评级**: 类似餐厅的Yelp评分——不是"赚多少钱"而是"好评多少"
4. **强化=长期投资**: 永久提升——不是"换更好的菜单"而是"让基础菜越来越强"
