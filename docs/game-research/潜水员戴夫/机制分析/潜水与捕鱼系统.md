# 潜水与捕鱼系统 - 潜水员戴夫(Dave the Diver)

> 游戏版本: Dave the Diver v1.0+ (PC/Switch, 2023年)
> 数据来源: dave-the-diver.fandom.com, NoobFeed, Casual Game Guides, IGN

## 1. 核心循环: 昼潜夜营

```
[白天] 潜入蓝洞(Blue Hole)
  → 捕鱼/采集资源/推进剧情/探索
    ↓
[夜晚] 经营番长寿司(Bancho Sushi)
  → 设计菜单/服务顾客/赚取金币
    ↓
[升级] 金币→装备/员工/菜单研究
  → 第二天更深的潜水/更好的寿司
```

## 2. 氧气系统(O2=HP)

| 参数 | 说明 |
|------|------|
| O2 | 潜水员的生命值——被鱼攻击=消耗氧气 |
| O2耗尽 | "死亡"=被救援→**仅保留1件物品**(其余全丢!) |
| 恢复 | O2罐/含空气蛤壳(潜水途中收集) |
| 容量升级 | 优先升级O2容量! |

## 3. 捕鱼方式

| 工具 | 功能 | 品质 |
|:--:|------|:--:|
| 鱼叉(Harpoon) | 基础捕鱼→射杀 | 1-2星 |
| 网枪(Net Gun) | 捕获小型鱼→**活捉** | **3星(完美!)** |
| 麻醉枪(Tranquilizer Gun) | 麻醉大型鱼/鲨鱼→活捉 | **3星** |
| 水下无人机(Drone) | 拖运最大型麻醉鱼 | — |

### 3.1 完美捕获(3星)
- 不伤害鱼→活捉→更多鱼肉+额外材料
- 活捉>杀死=更高品质=更高寿司价格

## 4. 武器系统(Duff制作)

| 武器类型 | 特点 | 获取 |
|:--:|------|------|
| 鱼叉(基础→升级) | 伤害/可靠性递增 | Duff分析→制作 |
| 麻醉枪 | 非致命捕获 | Duff |
| 网枪 | 小型鱼群捕 | Duff |
| 榴弹发射器(终局) | **最高伤害** | 后期解锁 |

- 每次潜水**只能携带1件武器**→需根据目标选择

## 5. 潜水策略

| 策略 | 说明 |
|------|------|
| 浅水优先(早期) | 深层有致命生物 |
| 优先升级负重 | 更多鱼=更多金币 |
| 检查任务目标 | 每次潜水前确认目标→减少无效潜水 |
| **提前撤退** | O2低+背包满→不要贪→保命撤退 |

## 6. 夜间潜水

- 剧情推进后解锁
- 不同鱼类/独特遭遇(白天没有)

## 7. 设计特点

1. **O2=HP=时间限制**: 双重含义——既是战斗资源也是潜水时间限制——创造自然的"回家"时刻
2. **完美捕获=品质驱动**: 不是"杀更多鱼"而是"更好捕获鱼"——技巧>暴力
3. **单武器限制=策略选择**: 每次潜水只带1把武器→必须根据目标(大鱼/小鱼群/Boss)选择
4. **死亡=几乎全丢**: "仅保留1件"——严厉的死亡惩罚→增强紧张感
