# 员工管理系统 - 潜水员戴夫(Dave the Diver)

> 游戏版本: Dave the Diver v1.0+ (PC/Switch, 2023年)
> 数据来源: NoobFeed, Casual Game Guides

## 1. 招聘

| 方式 | 费用 | 效果 |
|:--:|:--:|------|
| 传单(Flyer) | 50g | 低Cooksta→基础员工 |
| TV广告 | 150g | 中Cooksta |
| 网络广告 | 400g | **高Cooksta→高级员工** |
| 等待 | 1天 | 应聘者出现 |

## 2. 员工角色

| 位置 | 最多人数 | 职能 |
|:--:|:--:|------|
| 厨房(Kitchen) | 2人 | 做菜速度 |
| 服务(Serving) | 2人 | 送餐/倒茶/清理 |
| 等待室(Waiting Room) | 额外 | 可随时替换进厨房/服务 |

## 3. 员工属性

| 属性 | 影响 |
|:--:|------|
| 烹饪(Cooking) | 做菜速度(最重要!) |
| 服务(Serving) | 送餐+倒茶速度(最重要!) |
| 采集(Procure) | 小费加成/Cooksta评级 |
| 魅力(Appeal) | 客流量 |

## 4. 培训

- 花费金币→**永久提升属性**
- 特定等级解锁**员工技能**: 芥末补充/倒茶/保证小费等
- Lv5/10/15→**解锁特色食谱**(员工专属!)

## 5. 派遣任务

- 未使用的员工→派去**采集食材**(酱油/味噌等)
- 时间: 1-数天→返回带资源

## 6. 设计特点

1. **招聘=投资**: 400g网络广告→更高级员工→更快服务→更高评级→更多金币→回收投资
2. **培训=员工成长**: 类似RPG的角色培养——不是"换更好的员工"而是"培养现有员工"
3. **派遣=闲置员工利用**: 不用的员工不是浪费——派出去采集调味料
4. **员工食谱=独特价值**: 特定食谱仅特定员工解锁→每个员工有不可替代性
