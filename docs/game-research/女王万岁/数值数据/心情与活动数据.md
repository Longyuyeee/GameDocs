## 三、技能成长公式

```
每周技能获得 = (2 + 心情Buff + 该技能组×0.01 + 该大类×0.001) × 5天
```

| 术语 | 说明 | 数值范围 |
|:--:|------|:--:|
| 2 | 基础值 | 固定 |
| 心情Buff | 四轴各自修正不同类别 | -3~+1 |
| 技能组×0.01 | 已有点数/100 | 0~0.5+ |
| 大类×0.001 | 大类总点数/1000 | 0~0.2+ |
| ×5天 | 每周5训练日 | — |
