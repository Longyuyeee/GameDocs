# 心情与情绪系统 - 女王万岁(Long Live the Queen)

> 游戏版本: Long Live the Queen (PC, 2012年)
> 数据来源: longlivethequeen.fandom.com, NeoSeeker

## 1. 四轴八极心情系统

LLTQ的心情不是"一个状态"而是**4个独立轴**——每轴两端各有Buff/Penalty。

| 轴 | 左极(-5~0) | 右极(0~+5) | 左Buff | 右Buff |
|:--:|:--:|:--:|------|------|
| 1 | **Afraid(恐惧)** | Angry(愤怒) | Agility+1/Faith+1 | Royal Demeanor-1 |
| 2 | Depressed(沮丧) | Cheerful(开朗) | Expression+1/Animal+1 | Royal Demeanor-1/Conversation-2 |
| 3 | Yielding(顺从) | **Willful(任性)** | Royal Demeanor+1/History+1/Faith+1 | **Lumen-3(Yielding)** |
| 4 | Lonely(孤独) | Pressured(压力) | Conversation+1/Medicine+1 | Intrigue-1(Lonely) |

## 2. 心情调整(周末活动)

| 活动 | 心情变化 | 效果 |
|:--:|:--:|------|
| Play with Toys | Afraid↑ | 减轻恐惧 |
| Attend Service | Yielding↑/Angry↓ | 宗教/顺从 |
| Sports | Angry↑/Cheerful↑ | 运动/积极 |
| Visit Treasury | Pressured↑ | 经济/关注 |
| Walk in Gardens | Cheerful↑ | 减压/自然 |
| Sneak Out | Willful↑/Lonely↑ | 独立/反叛 |
| Visit Tomb | Depressed↑ | 沉思/哀思 |
| Visit Julianna | Willful↑ | **Lumen训练关键!** |

## 3. 心情→技能成长

| 心情 | 技能加成 |
|:--:|------|
| Angry | Agility+1(武术路线) |
| Afraid | Faith+1 |
| Cheerful | Expression+1/Animal+1 |
| Willful | **Lumen+1**(魔法必需!)/Intrigue+1/Military+1 |
| Yielding | Royal Demeanor+1 |
| Pressured | Conversation+1/Medicine+1 |

## 4. Lumen路线的关键心情

| 步骤 | 所需心情 | 原因 |
|:--:|:--:|------|
| 成为Lumen后 | **Willful** | **唯一给Lumen+1的心情!** |
| 训练Sense/Resist/Wield | Willful | +1技能成长 |
| Yielding | 避开 | Lumen-3(致命!) |

## 5. 设计特点

1. **四轴心情=PM3心情的进化版**: PM3是6种离散心情→LLTQ是4连续轴——更精细
2. **心情=技能成长修正**: 不是"心情影响事件"——而是"心情影响技能成长效率"——机制层而非叙事层
3. **Willful=魔法路线的必需**: 唯一给Lumen加成的心情——将心情选择绑定到Build路线
4. **周末活动=主动心情控制**: 不是"被动进入心情"——而是"每周主动选择调整"——策略性
