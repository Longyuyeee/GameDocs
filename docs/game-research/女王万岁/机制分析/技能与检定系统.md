# 技能与检定系统 - 女王万岁(Long Live the Queen)

> 游戏版本: Long Live the Queen (PC, 2012年)
> 数据来源: longlivethequeen.fandom.com, GameFAQs, NeoSeeker

## 1. 核心机制: 技能检定(Skill Check)

LLTQ的核心是**不是纯数值养成**——而是**技能检定驱动的生死叙事**。每周事件会检查Elodie的特定技能→通过=存活/获得好处→失败=可能死亡!

```
事件触发 → 检查[技能X ≥ 阈值?]
  → 通过 → 好的结局分支(存续/好感/权力)
  → 失败 → 坏的结局分支(受伤/失去支持/死亡!)
```

## 2. 42技能(4大类×多技能组)

## 3. 技能成长公式

```
每周技能获得 = (2 + 心情加成 + 技能组×0.01 + 大类×0.001) × 5天
```

| 影响因素 | 说明 |
|------|------|
| 心情加成 | 心情轴影响不同类别技能 |
| 技能组×0.01 | 已有点数越高→成长越快(雪球效应) |
| 大类×0.001 | 大类整体越高→成长越快 |
| ×5天 | 每门课每周5个训练日 |

## 4. 服装系统

| 条件 | 效果 |
|------|------|
| 技能组3项全部≥25 | 解锁该组服装→**+10全部3项!** |
| 穿戴时机 | **检定前换对应服装**——大幅提升通过率 |

## 5. 设计特点

1. **技能检定=生死门**: 不是"属性不够结局差"——而是"技能不够=死亡"——最极端的技能→后果映射
2. **42技能=极细化的能力维度**: 火山的女儿10属性/PM2约10属性→LLTQ=42技能——最细的养成颗粒度
3. **服装=装备**: 解锁+穿戴=临时属性提升——类似RPG的装备系统
4. **雪球效应**: 已有技能越高→成长越快→早期投资的选择极为重要
