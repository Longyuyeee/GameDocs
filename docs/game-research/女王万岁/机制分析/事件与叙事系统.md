# 事件与叙事系统 - 女王万岁(Long Live the Queen)

> 游戏版本: Long Live the Queen (PC, 2012年)
> 数据来源: longlivethequeen.fandom.com, GameFAQs

## 1. 事件结构

| 参数 | 说明 |
|------|------|
| 总周数 | **40周**(14岁→加冕) |
| 每周事件 | 1-3个(基于之前选择/技能状态) |
| 事件类型 | 技能检定/社交/政治/暗杀/魔法 |
| 分支 | 每个事件有1-4个选项→不同后果 |

## 2. 关键事件链(按周)

| 周 | 事件 | 关键技能 | 失败后果 |
|:--:|------|:--:|------|
| 2 | Julianna到访 | — | 失去Lumen路线(让她离开) |
| 2-3 | 蛇咬 | Reflexes≥15或Composure≥10 | **死亡** |
| 8-10 | 成为Lumen(魔法路线) | Presence≥70或Accounting≥60 | 失去魔法能力 |
| 12 | Talarist求婚 | Court Manners≥40 | 外交失败 |
| 16 | 游行 | Elegance≥70/Public Speaking≥50 | 声望大跌 |

| 28 | 精神试炼 | Composure≥50或Meditation≥50 | 精神崩溃 |
| 30 | 演说 | Public Speaking≥70 | 政治失败 |

| 40 | **加冕** | 综合(所有技能+政治状态) | 多结局 |

## 3. 事件选择→分支

| 选择类型 | 示例 | 影响 |
|:--:|------|------|
| 外交 | 接受/拒绝求婚 | 联盟/敌对 |
| 内政 | 处决/宽恕 | 民望/贵族支持 |
| 魔法 | 使用/隐藏魔法 | Lumen路线/传统路线 |
| 军事 | 进攻/防御/谈判 | 战争结果 |

## 4. 设计特点

1. **线性事件+分支选择**: 事件按周固定——但选择决定分支和结局
2. **技能检定=叙事门控**: 不是"属性到了看好结局"——而是"属性不够=死"——极端的叙事门控
3. **多线叙事**: Lumen路线/非魔法路线/外交路线/战争路线——不同Build导致完全不同的故事
4. **一周目必死体验**: 设计意图——通过死亡学习→二周目应用知识→最终存活加冕
