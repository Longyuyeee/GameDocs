# 死亡与生存压力 - 女王万岁(Long Live the Queen)

> 游戏版本: Long Live the Queen (PC, 2012年)
> 数据来源: longlivethequeen.fandom.com, GameFAQs

## 1. 概述

LLTQ是养成品类中**最极端的生存压力设计**——不是"培养不好=结局差"，而是"技能检定失败=**立即死亡**"。Elodie可以在40周中因数十种原因而死。这赋予了技能训练一种其他养成游戏无法比拟的**紧迫感**。

## 2. 全死亡方式(部分)

| 死亡方式 | 周数 | 技能检定 | 条件 |
|------|:--:|:--:|------|
| 蛇咬 | 2-3 | Reflexes或Composure~10-15 | 失败=被蛇咬死 |
| 毒巧克力 | 早期 | Intrigue或Medicine | 未识破毒杀 |
| 暗杀 | 中期 | Reflexes/Flexibility/Composure | 多种暗杀事件 |
| 决斗 | 35 | Wield Magic≥100(魔法)或Swords≥X(物理) | Togami决斗 |
| 魔法决斗 | 后期 | Lumen类总计≥200 | 终局魔法对抗 |
| 海战 | 后期 | Military/相关 | 海军对抗 |
| 献祭(Kraken) | 中后期 | — | 特定事件链 |
| 内战 | 终局 | 综合(多种) | 政权崩溃 |
| 处决 | 多种 | — | 被政敌捕获 |

## 3. 死亡的设计功能

| 功能 | 说明 |
|------|------|
| **驱动技能训练** | 知道"下周可能有毒巧克力事件"→必须提前训练Intrigue/Medicine |
| **鼓励多周目** | 死亡→重试→这次提前训练了对应技能→活下来了 |
| **知识积累** | 玩家通过死亡学习事件链→NG+不是"继承属性"而是"继承知识" |
| **情感投资** | Elodie的脆弱性→玩家更在乎她的生死 |

## 4. 生存策略

| 策略 | 说明 |
|------|------|
| **平衡训练** | 不要专精一门——多种死亡方式需要多种技能 |
| **事件前存档** | 每周存档——事件失败→回档调整训练 |
| **服装切换** | 事件前穿对应技能服装→临时提升检定通过率 |
| **Lumen路线** | 魔法路线提供最强的生存能力(Wield Magic可应对战斗+决斗) |

## 5. 设计特点

1. **死亡=最直接的反馈**: 不是"结局不好"——而是"你死了, 游戏结束"——养成品类中最强的后果
2. **40周=不断收紧的绞索**: 越接近加冕→事件越危险→技能检定阈值越高
3. **技能训练的"预判"** : 必须提前数周预测未来事件所需技能→不是"想学什么学什么"
4. **多周目=多死亡**: 一周目几乎必然会死多次——这是设计的意图而非缺陷
