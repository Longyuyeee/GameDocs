## 二、无传统节日系统

天穗之咲稻姬**没有传统节日/活动系统**。季节性稻作节点取代了节日的节奏功能:

| 稻作节点 | 类似传统节日的节奏功能 |
|------|------|
| 插秧(春1) | 春季祭典——新一年农耕开始 |
| 收割(秋2-3) | 丰收祭典——一年劳动成果 |
| 脱壳完成 | 年终结算——属性增长+新米品尝 |
| 冬末预施肥 | 新年准备——为下一年打基础 |

## 三、剧情事件

| 事件 | 触发条件 | 说明 |
|------|------|------|
| 第一次成功丰收 | 第1次白米产出 | 佐久名骄傲感+成就感高峰 |
| 农场被毁 | 主线后期 | 心轮到访+帮助重建+水车解锁 |
| 祭典(秋) | 主线后期(秋收时) | 大蛟复活触发节点 |
| 天雨神社解锁 | 中期 | 200层Boss Rush挑战 |
| 结局 | 击败大蛟后 | 部分NPC选择返回人间 |
