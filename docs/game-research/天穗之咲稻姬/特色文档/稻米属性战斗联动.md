# 稻米属性与战斗属性联动系统 - 天穗之咲稻姬(Sakuna: Of Rice and Ruin)

> 游戏版本: Sakuna: Of Rice and Ruin v1.0+ (PC/Switch/PS4, 2020年)

## 一、概述

天穗之咲稻姬的核心创新在于**稻米品质直接决定战斗属性**。这不是"种田赚钱买装备提升战斗"的间接关系，而是每粒米的产量/味道/硬度/香气/美观/粘性→HP/力量/体力/魔力/运气/满足度的**直接数学转换**。

## 二、六维转换表

| 稻米属性 | 战斗属性 | 转换公式 | 每1点稻米属性的战斗收益 |
|------|:------:|------|------|
| 产量(Yield) | **HP** | (产量-150)÷11+10 | 每增产11→+1HP |
| 味道(Taste) | **力量(Strength)** | 味道÷10 | 每+10味道→+1力量 |
| 硬度(Hardness) | **体力(Vitality)** | 硬度÷10 | 每+10硬度→+1体力 |
| 香气(Aroma) | **魔力(Magic)** | 香气÷10 | 每+10香气→+1魔力 |
| 美观(Aesthetic) | **运气(Luck)** | 美观÷10 | 每+10美观→+1运气 |
| 粘性(Stickiness) | **满足度(Gusto)** | (粘性÷100)+10 | 每+100粘性→+1满足度 |

## 三、正向循环

```
更好的肥料+更精确的水分管理+更优的选种插秧策略
  ↓
更高的稻米品质(六维属性全面提升)
  ↓(每年1次收获后)
战斗属性大幅增长(HP/力量/体力/魔力/运气)
  ↓
能挑战更强的敌人/更深的天雨神社层数
  ↓
获得更好的材料(稀有矿石/怪物掉落/琥珀)
  ↓
制作更优的肥料/锻造更强的武器/纺织更好的防具
  ↓
(循环回到第一步)
```

## 四、策略深度: 农业Build→战斗Build

| 想打造的战斗风格 | 稻作策略 |
|------|------|
| **坦克型**(高HP+体力) | 密播/密植/深水成熟/大量根部肥/腐败食物 |
| **力量型**(高力量) | 专注穗部肥+肉/鱼材料→味道最大化 |
| **法师型**(高魔力) | 专注叶面肥+香草/蘑菇→香气最大化 |
| **暴击型**(高运气) | 重选种/稀播/稀植/全程精细管理→美观最大化 |
| **Buff持久型**(高满足度) | 穗部肥+水分平衡→粘性最大化→食物Buff更长 |

## 五、设计分析

1. **机制统一**: 种田=角色养成——不是"两个独立系统"而是"一个系统的两个表达"
2. **每年一次大成长**: 不像传统RPG的"每次打怪升级"，而是"每年收获时属性大幅跃升"——节奏完全不同
3. **农业策略=Build选择**: 你不是"选职业"而是"选择种稻方式"来决定战斗风格
4. **不可逆转的糙米vs白米抉择**: 糙米=强Buff弱成长/白米=强成长弱Buff——永久影响角色发展方向
5. **"种田就能变强"的终极表达**: 很多牧场游戏说"种田很重要"，但天穗是唯一通过数学公式直接证明"种田=变强"的
