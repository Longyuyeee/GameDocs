# 稻米种植的真实模拟 - 天穗之咲稻姬(Sakuna: Of Rice and Ruin)

> 游戏版本: Sakuna: Of Rice and Ruin v1.0+ (PC/Switch/PS4, 2020年)

## 一、概述

天穗之咲稻姬拥有**游戏史上最详尽的水稻种植模拟**。不是"翻地→播种→浇水→收获"的农场游戏简化版，而是包含真实日本稻作的完整12步流程——从选种到脱壳。其详尽程度足以作为稻作农业的入门教材。

## 二、12步稻作完整流程

| # | 步骤 | 现实对应 | 游戏内操作 | 决策点 |
|:--:|------|------|------|------|
| 1 | 选种 | 泥水/盐水选种 | 泥水桶→浮选不良种→留沉种 | 重选=高质低量/轻选=高量低质 |
| 2 | 育苗 | 育苗箱育苗 | 冬季播种→密播=高产/薄播=高质 | 播种密度 |
| 3 | 整地 | 翻耕+施肥 | 锄头翻耕→去除石块→预施肥 | 冬末预施肥=白送1级 |
| 4 | 插秧 | 大田插秧 | 将秧苗植入大田→密植/稀植 | **种植间距影响品质维度** |
| 5 | 水分管理 | 灌溉水位控制 | 调节水门→浅/深/排干 | **不同阶段不同水位** |
| 6 | 根部施肥 | 追肥(根部) | 根部养料→影响产量/硬度 | 每2天1次(勿过度) |
| 7 | 除草 | 手工除草 | 每日拔除杂草 | 鸭子可辅助 |
| 8 | 防虫 | 生物/化学防虫 | 鸭子/青蛙/蜗牛/蜘蛛/杀虫药 | 分蘖3期必须移除鸭子 |
| 9 | 叶面施肥 | 追肥(叶面) | 叶面养料→影响美观/香气 | 抽穗期施用 |
| 10 | 排水干燥 | 中干(夏季落水) | **夏2排干全部水!** | **最关键步骤!** |
| 11 | 收割→晾晒 | 收割+架干 | 镰刀收割→晾晒架约1天→防雨 | 晴天收割=高质 |
| 12 | 脱粒→脱壳 | 脱粒+精米 | 脱粒小游戏→选择碾磨度0-100% | **糙米vs白米的永久抉择** |

## 三、真实模拟的农业知识点

| 知识点 | 游戏内体现 |
|------|------|
| 盐水选种 | 坏种子浮/好种子沉→重选=留下高质种 |
| 育苗密度 | 密=高产/薄=高质+大粒 |
| 插秧间距 | 密=高产低质/稀=高质低量 |
| 中干(Midsummer Drainage) | 分蘖3期排水=避弱稻+促深根——省略=全年白费 |
| 水位与品质 | 深水=产量/浅水=硬度 |
| 氮素管理 | 过度施肥=徒长/纤弱→需要节制 |
| 生物链防虫 | 鸭子→普通虫/青蛙→迅捷幼虫→蜘蛛→各类——完整的稻田生物链 |
| 盐害 | 少量盐=催质/过量=盐害→需泉水中和 |
| 腐败物质=优质肥 | 腐败食物(有机物分解)=最高效全属性肥料——真正的农业循环 |
| 晾晒防雨 | 收割后必须晒干→雨=灾难→需祈祷晴天 |

## 四、稻作品质六维与种植策略

| 目标属性 | 策略 |
|------|------|
| 产量↑(HP) | 轻选种/厚播/密植/深水成熟/大量根部肥 |
| 味道↑(力量) | 穗部肥+腐败食物+特定材料 |
| 硬度↑(体力) | **重选种/稀播/稀植/浅水成熟/排干彻底** |
| 香气↑(魔力) | 叶面肥+香料/草药材料 |
| 美观↑(运气) | 重选种+稀植+精心管理全程 |
| 粘性↑(满足度) | 穗部肥+水分平衡 |

## 五、设计分析

1. **游戏作为农业教育**: 非日本玩家通过此游戏学会了真正的水稻种植流程——这在游戏史上是罕见的
2. **复杂度=沉浸感**: 12步×每步决策→种田不是"每日任务"而是"全年策略规划"
3. **失败有后果**: 分蘖3期不排水→弱稻→全年品质低→战斗力增长慢→打不过Boss——不是"随便种种也行"
4. **腐败食物=最优肥**: 游戏用机制告诉你——农业的本质是有机物循环，腐烂不是"坏"的
5. **冬末预施肥=免费升级**: 巧妙地奖励玩家"提前规划"而非"临时抱佛脚"
