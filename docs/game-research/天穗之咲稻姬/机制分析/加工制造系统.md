# 加工制造系统 - 天穗之咲稻姬(Sakuna: Of Rice and Ruin)

> 游戏版本: Sakuna: Of Rice and Ruin v1.0+ (PC/Switch/PS4, 2020年)
> 数据来源: sakuna-of-rice-and-ruin.fandom.com, StrategyWiki

## 1. 概述

天穗之咲稻姬的加工制造围绕**稻米收获后处理**和**角色制作**两条线展开。没有"作坊→原料→加工品→出售"的传统制造链——而是"自给自足以提升战斗力+探索能力"。

## 2. 稻米收获后处理(核心加工链)

### 2.1 收割→入仓完整流程

```
收割(镰刀) → 晾晒(挂架~1天) → 脱粒(小游戏) → 脱壳(选择碾磨度) → 白米/混合米/糙米
```

| 步骤 | 设备 | 关键因素 |
|------|------|------|
| 收割 | 镰刀(Sickle) | 晴天收割→高品质 |
| 晾晒 | 晾晒架(Drying Rack) | 约1天，**必须防雨**(祈祷晴天) |
| 入库 | 谷仓 | **冬1前必须入库**→否则全部损失 |
| 脱粒 | —(小游戏) | 分离谷粒与秸秆→秸秆可作纺织/肥料 |
| 脱壳 | —(选择碾磨度0-100%) | 0%=糙米/50%混合/100%=白米 |

### 2.2 稻谷副产品利用

| 副产品 | 用途 |
|------|------|
| 稻壳(Husk) | 肥料原料 |
| 稻秆(Straw) | 纺织材料/肥料/农场建材 |
| 米糠(Bran) | 肥料/食物补充 |
| 腐败食物 | **最优全属性肥料**(需加强害虫防治) |

## 3. 烹饪系统

### 3.1 晚餐机制

| 属性 | 详情 |
|------|------|
| 厨师 | Myrthe(固定) |
| 菜品数 | 最多5道菜套餐 |
| 食材来源 | 探索采集+稻谷副产品 |
| 效果 | 食物Buff(HP/力量/体力/魔力/运气暂时提升) |
| 持续时间 | 满足度(Fullness)消耗完毕为止 |

### 3.2 食谱获取

- 通过探索获得食谱
- 不同食材组合=不同菜品+不同Buff
- 5种属性Buff卡(HP/攻击/防御/魔法/运气)→每餐尽量凑满5种

## 4. 锻造系统(金太)

| 产出 | 材料来源 | 功能 |
|------|------|------|
| 单手武器(1H) | 矿石+怪物掉落 | 快速连击 |
| 双手武器(2H) | 矿石+怪物掉落 | 高伤害慢速 |
| 升级农具 | 矿石+材料 | 更快整地/收割/更多功能 |

| 产出 | 材料来源 | 功能 |
|------|------|------|
| 衣物(Clothing) | 纤维+稻秆+怪物材料 | 防御力+属性加成 |
| 帽子(Hat) | 纤维+特殊材料 | 特殊效果/属性 |
| 面具(Mask) | 纤维+特殊材料 | 特殊效果/属性 |

## 6. 肥料制作

| 成分类型 | 对应稻米属性 | 主要材料来源 |
|------|:------:|------|
| 根部养料(Root) | 产量/硬度 | 探索材料+腐败食物 |
| 穗部养料(Kernel) | 味道/粘性 | 探索材料+特定怪物掉落 |
| 叶面养料(Leaf) | 美观/香气 | 探索材料+植物类 |
| 三种药剂 | 治疗/减量/复活 | 特殊材料 |

- 肥料在施肥箱中混合制作(无专门"制作站")
- 成分三角可视化→实时显示每种成分百分比

## 7. 设计特点

1. **稻米处理=核心制造链**: 不是"收获→出货"，而是"收割→晾晒→脱粒→脱壳→白米/糙米"——每个决策都影响最终品质
2. **金太+结=装备双线**: 武器/农具(锻造)+衣物/防具(纺织)——两个NPC各自专精
3. **肥料=最复杂的"制造"**: 不是"选配方→制作"，而是调整成分三角→平衡属性→避免疾病→需要策略
4. **晚餐=Buff制造**: 5道菜=5种Buff→每天出发前策略性选择
5. **无"出货"概念**: 所有产品(大米/装备/衣物/食物)都是自我消耗→自给自足封闭循环
