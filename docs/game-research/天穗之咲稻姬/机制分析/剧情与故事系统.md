# 剧情与故事系统 - 天穗之咲稻姬(Sakuna: Of Rice and Ruin)

> 游戏版本: Sakuna: Of Rice and Ruin v1.0+ (PC/Switch/PS4, 2020年)
> 数据来源: sakuna-of-rice-and-ruin.fandom.com, StrategyWiki, TV Tropes, sakunaofriceandruin.com

## 1. 世界观设定

### 宇宙结构

| 领域 | 说明 |
|------|------|
| 高天原(Lofty Realm) | 神之领域，由神光公主统治的都城所在地 |
| 苇原中国(Lowly Realm) | 人类世界("人子们") |
| 创造之树(Tree of Creation) | 连接两界的宇宙树，产出琥珀——众神的生命之源 |
| 天浮桥(Floating Bridge of Heaven) | 瞬现瞬逝的两界通道——连神光公主也无法预知何时出现 |
| 日之惠岛(Hinoe Island) | 鬼岛——佐久名被贬黜之地，也是她父母曾经的家园 |

### 双界共生

高天原和苇原中国互为镜像——一方的繁荣/苦难会在另一方显现。当高天原繁荣→人间和平；高天原受难→人间饥荒/战争。

## 2. 主线剧情结构

### Act 1: 堕落→贬黜

```
佐久名: 高天原被宠坏的丰收女神
  → 一群人类(田右卫门等)通过天浮桥闯入都城求食
  → 佐久名试图阻止→意外烧毁整个都城粮仓
  → 神光公主定罪: 贬黜至日之惠岛(鬼岛)
  → 任务: 清除岛上怪物+照看人类
```

### Act 2: 稻作生活

```
到达鬼岛→遇到芦云(最后的镰鼬族)
  → 发现父母(Takeribi+Toyohana)曾在此生活的废弃农场
  → 开始种稻+打鬼的双重生活
  → 金太建锻造铺/结建纺织屋
  → 每晚集体晚餐→角色背景逐一揭示
  → 第1次成功丰收→骄傲+成就感
```

### Act 3: 过去的秘密

```
玉(Tama)揭示父母真相:
  └── Toyohana原本是人类→进入高天原→升格为神
  └── Takeribi爱上Toyohana+与镰鼬族交友→被都城放逐
  └── 两人牺牲生命封印大蛟(Omizuchi)
  └── 玉的刀刃缺片=岛上邪气来源→创造全部Specter

心轮(Kokorowa)被石丸利用→出于嫉妒破坏佐久名的米产品
  └── 神光公主饶恕心轮(因为她的才能太宝贵)
  └── 心轮到访日之惠岛→帮助重建农场+水车
```

### Act 4: 结的真相(鹤的报恩)

```
结原本是鹤→被"访客"异界女神化为人形
  └── 有禁忌誓言(不能被人知道真相)
  └── 佐久名无意中让她打破誓言→变回鹤
  └── 佐久名+金太祈求访客恢复结→新的代价
  └── 日本古典"鹤的报恩"全新演绎
```

### Act 5: 最终决战

```
Homusubi Specter(火山)→回收玉的残缺刀刃碎片→金太锻成新武器
  → 岛上祭典日→大蛟完全复活
  → 三头龙战斗(火/土/水三属性头)
  → 神衣使用代价几乎拖佐久名+玉入黄泉
  → Takeribi+Toyohana残存灵魂+海童介入→拯救
  → 佐久名击败大蛟→龙承认她→"只要你在种稻，我就不会回来"
  → 结局: 部分人类回归苇原中国/部分留守
  → 佐久名从被宠坏的懒女神成长为真正的丰收女神
```

## 3. 叙事方式

| 机制 | 说明 |
|------|------|
| **晚餐对话驱动**: | 不是"走到NPC面前对话"，而是每天傍晚自动集体围坐→自然推进 |
| **稻米品质影响剧情**: | 第一次真正成功的丰收→佐久名的骄傲→剧情情绪高峰 |
| **主线关卡解锁**: | 清除关卡→解锁新区域→推进故事 |
| **角色背景通过关系自然揭示**: | 田右卫门的过去/海童的失语/结的秘密→全在日常对话中展开 |

## 4. 核心情感弧线

```
被宠坏的女儿 → 被迫承担责任 → 抱怨和不满
  → 第一次丰收的骄傲 → 理解父母的牺牲
  → 接受"丰收女神"的真正含义
  → "稻米不只是食物——它是生命、成长、和爱"
```

## 5. 结局

- **单一主线结局**: 击败大蛟/父母灵魂拯救/和平回归
- **开放后日谈**: 继续种稻/天雨神社200层/完美化稻米品质
- 主题: "只要你继续种稻，世界就不会陷入黑暗"

## 6. 设计特点

1. **成长弧线=种稻能力**: 佐久名从"被宠坏的富家女"到"真正丰收女神"的转变，通过种稻能力的逐步提升来体现——叙事和机制完美统一
2. **父母遗产**: 不是"继承财产"而是"继承使命和价值观"——Takeribi+Toyohana是佐久名永远不会见到的导师
3. **没有真正的反派**: 大蛟是"混沌邪恶的具象化"——不是有动机的恶人；石丸被利用后被杀；心轮出于嫉妒但被原谅——复杂的人性而非简单的善恶二分
4. **日本古典文学演绎**: "鹤的报恩"、镰鼬、创造之树、天浮桥——全来自神道教/民间传说
5. **稻米=希望=抵抗黑暗**: "只要你种稻，大蛟就不会复活"——将农业劳动提升为宇宙级的抵抗黑暗行为
