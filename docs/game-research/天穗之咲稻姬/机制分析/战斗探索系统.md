# 战斗探索系统 - 天穗之咲稻姬(Sakuna: Of Rice and Ruin)

> 游戏版本: Sakuna: Of Rice and Ruin v1.0+ (PC/Switch/PS4, 2020年)
> 数据来源: sakuna-of-rice-and-ruin.fandom.com, StrategyWiki, NeoSeeker, sakunaofriceandruin.com

## 1. 战斗定位

战斗在天穗之咲稻姬中**与种田同等重要**。两个系统形成完整正向循环: 种田→提升属性/解锁技能→打更强敌人→获得更好材料→制作更好肥料→种出更好米。战斗是**横版卷轴动作**风格。

## 2. 战斗属性体系

| 属性 | 英文 | 成长方式 | 影响 |
|------|------|------|------|
| HP | 生命值 | 稻米产量→HP | 归零=战斗失败 |
| 力量(Strength) | 物理攻击 | 稻米味道→力量 | 武器+技能物理伤害 |
| 体力(Vitality) | 物理防御 | 稻米硬度→体力 | 承受伤害减免 |
| 魔力(Magic) | 魔法攻击 | 稻米香气→魔力 | 技能魔法伤害 |
| 运气(Luck) | 暴击率 | 稻米美观→运气 | 暴击概率+其他增益 |
| 满足度(Gusto) | Buff持续 | 稻米粘性→满足度 | 食物Buff持续时间 |
| SP | 技能点 | — | 使用Fighting Skills消耗 |

## 3. 攻击属性克制

| 颜色 | 意义 | 说明 |
|:--:|------|------|
| 🟠橙色 | 有效 | 攻击效果正常/偏高 |
| 🔴红色 | **超级有效** | 最大化伤害 |
| 🔵蓝色 | 效果差 | 减伤/抵抗 |

四种攻击属性: **打击(Strike)/斩击(Slash)/穿刺(Pierce)/魔法(Magic)**

- 不同敌人有不同属性弱点
- 魂枝(Spirit Boughs)可改变武器攻击属性

## 4. 神衣(Divine Raiment)战斗应用

| 操作 | 效果 |
|------|------|
| 抓取敌人 | 勾住→荡到敌人背后→穿插攻击 |
| 投掷敌人 | 抓起敌人→投向另一个敌人→**碰撞伤害**(核心输出机制!) |
| 空中机动 | 类似钩爪→跨越地形/躲避攻击 |
| Raiment Skills | 对倒地/反击敌人→Debuff/削弱攻防/吸血 |

## 8. 设计特点

1. **种田→战斗正向循环**: 不是"两套独立系统"而是"种田质量决定战斗能力→战斗获取材料提升种田"
2. **投掷碰撞=核心伤害机制**: 不是"技能输出"而是"抓起敌人投向敌人"——神衣操作是区分高玩的标志
3. **熟练度=技能深度**: 每技能15级×99,999经验→鼓励长期使用和精通
4. **敌人攻击属性克制**: 四种攻击属性+敌人弱点→魂枝配置是build核心
5. **200层终局挑战**: 天雨神社提供无限可重玩终局内容
