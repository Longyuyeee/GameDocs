# NPC与社交系统 - 天穗之咲稻姬(Sakuna: Of Rice and Ruin)

> 游戏版本: Sakuna: Of Rice and Ruin v1.0+ (PC/Switch/PS4, 2020年)
> 数据来源: sakuna-of-rice-and-ruin.fandom.com, StrategyWiki, TV Tropes

## 一、固定同伴(7位，无恋爱系统)

天穗之咲稻姬**没有恋爱/婚姻系统**。7个角色是共同生活在日之惠岛的"米饭家庭"——通过每日晚餐聚集在一起。

| 角色 | 身份 | 职能 | 剧情弧线 |
|------|------|:--:|------|
| **佐久名(Sakuna)** | 丰收女神(被贬) | 主角/种田/战斗 | 被宠坏的懒女神→真正的丰收女神 |
| **玉(Tama)** | 刀之精灵 | 导师/守护者/道德指南 | 缺失的刀刃碎片=主线关键 |
| **田右卫门(Tauemon)** | 前武士→前强盗 | 农业顾问 | 逃避强盗过往→找到人生价值 |
| **米尔特(Myrthe)** | 西方传教士 | 厨师/照料者 | 异文化视角→神道世界的"外人" |
| **金太(Kinta)** | 战争孤儿/铁匠 | 锻造武器+农具 | 失散姐姐→复仇→成长 |
| **结(Yui)** | 少女/织工 | 纺织衣物 | **鹤的报恩**改编→仙鹤变人→禁忌→变形 |
| **海童(Kaimaru)** | 幼童 | 动物沟通/招募益虫 | 创伤性失语→逐渐恢复 |

## 四、社交机制

### 4.1 每日晚餐(核心社交节点)

- 每天傍晚集体围坐吃饭
- Myrthe烹饪5道菜套餐→食物Buff
- 晚餐时触发角色对话/剧情
- 新角色背景揭示几乎全部在晚餐时

### 4.2 无传统社交系统

| 缺失的传统要素 | 替代机制 |
|------|------|
| 无好感度柱/心 | 剧情驱动角色关系 |
| 无送礼系统 | 集体晚餐=社交 |
| 无恋爱/婚姻 | 佐久名的成长才是主角弧线 |
| 无生日 | — |
| 无嫉妒/多角恋 | — |

## 五、角色剧情线精选

### 佐久名(Sakuna)
```
被宠坏的懒女神(高天原)
  → 犯大错(烧毁都城粮仓)→被贬黜
  → 被迫种田+打鬼→最初抱怨
  → 逐渐学会责任和劳动的价值
  → 第1次成功丰收→骄傲和成就感
  → 了解父母真相→接受遗产
  → 击败Omizuchi→真正的丰收女神
```

### 田右卫门(Tauemon)
```
武士→强盗→逃跑(不认同杀戮)
  → 救人(Myrthe/Kinta/Yui/Kaimaru)
  → 到达高天原(寻求庇护)
  → 在日之惠岛找到价值——他的农业知识救了所有人
  → 从"没用的前强盗"变成"不可或缺的农业导师"
```

### 结(Yui)
```
鹤(被异界女神化为人形)
  → 有禁忌誓言(不能被人知道真相)
  → 佐久名无意中让她打破誓言→变回鹤
  → 佐久名+金太必须祈求异界女神恢复她→新的代价
  = 日本古典"鹤的报恩"传说的全新演绎
```

## 六、设计特点

1. **无恋爱但有深度的角色关系**: 7个角色不是"攻略对象"而是"家人"——通过共同劳动+共同吃饭建立联系
2. **晚餐=剧情节点**: 不靠"走到NPC面前按A触发对话"，而是每晚自动围坐→自然推进
3. **小阵容大深度**: 7个角色(而不是20+可攻略NPC)→但有完整的个人剧情弧线
4. **佐久名=唯一主角**: 不是"自定义角色"而是固定女主角——性格/成长/配音全部预设
5. **日本神话角色群**: 神/精灵/镰鼬/仙鹤——全部来自日本传统文化而非西方奇幻
