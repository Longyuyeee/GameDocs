# 道具与装备升级系统 - 天穗之咲稻姬(Sakuna: Of Rice and Ruin)

> 游戏版本: Sakuna: Of Rice and Ruin v1.0+ (PC/Switch/PS4, 2020年)
> 数据来源: sakuna-of-rice-and-ruin.fandom.com, StrategyWiki, NeoSeeker

## 1. 农具系统

### 1.1 农具种类

| 农具 | 功能 | 升级方式 |
|------|------|------|
| 锄头(Hoe) | 整地/翻耕 | Kinta锻造升级 |
| 镰刀(Sickle) | 收割稻谷 | Kinta锻造升级 |
| 水桶/水门(Water Gate) | 调节水位 | 基础工具 |
| 育苗箱 | 冬季育苗 | 基础设施 |

- 农具升级提升效率(更快整地/更大范围收割)
- 所有农具升级由Kinta在锻造铺完成

## 2. 武器系统

### 2.1 单手武器(1H)

| 特点 | 说明 |
|------|------|
| 攻击速度 | 快 |
| 攻击范围 | 短 |
| 连击数 | 多段combo |
| 操作 | 可推动左摇杆上下改变方向 |
| 代表技能 | 旋转打击(Spinning Strike)、狂野陀螺(Wild Top) |

### 2.2 双手武器(2H)

| 特点 | 说明 |
|------|------|
| 攻击速度 | 慢 |
| 攻击范围 | 长 |
| 连击数 | 少段但高伤害 |
| 特殊效果 | 连击末可击倒敌人 |
| 代表技能 | 正面粉碎(Frontal Smash)、凶猛犁沟(Fierce Furrow)、沸怒(Boiling Rage) |

### 2.3 神衣(Divine Raiment)

| 功能 | 说明 |
|------|------|
| 抓取 | 勾住敌人→空中机动 |
| 投掷 | 抓起敌人→投向其他敌人→碰撞伤害 |
| 移动 | 类似钩爪，跨越地形间隙 |
| 技能 | Raiment Skills: 对倒地/反击敌人施加Debuff/削弱攻防/吸血 |

### 2.4 隐藏力量(Hidden Powers)

- 装备潜藏着"隐藏力量"——达成特定条件后解锁
- 提供额外属性或特殊效果

### 2.5 魂枝(Spirit Boughs)

- 可镶嵌在装备上的特殊道具
- 效果如: 改变武器属性类型(打击/斩击/穿刺/魔法)
- 关键魂枝: 吸血(Life Drain)、报应(Retribution)、复活(Revival)、双刃剑(Double-Edge Sword)

## 4. 战斗技能(Fighting Skills)

| 属性 | 详情 |
|------|------|
| 获取方式 | **主要通过种田学习**(稻米品质提升解锁新技能) |
| 提升方式 | 战斗中使用→熟练度(Proficiency)成长 |
| 熟练度上限 | 15级(需99,999经验满级) |
| 同时装备数 | 最多4个Fighting Skills + 4个Raiment Skills |
| 资源消耗 | SP(技能点) |

### 4.1 技能熟练度成长效果

- 攻击次数+1/+2/...
- 承受伤害减免(如凶猛犁沟最高-80%)
- 碰撞伤害倍率提升
- 新增效果(如狂野陀螺多段使用)

## 5. 设计特点

1. **种田→技能成长主线**: 不是"打怪升级学技能"，而是"种出好米→解锁新技能"——种田是成长的唯一路径
2. **神衣=核心操作装置**: 既是武器(投掷碰撞)也是移动(钩爪)也是技能触发——一个核心机制服务三个设计目的
3. **熟练度系统**: 每种技能有独立熟练度→鼓励使用不同技能而非"一招打天下"
4. **魂枝=装备定制**: 不是简单的"换更好的装备"，而是"选择适合当前挑战的魂枝搭配"
5. **Kinta+Yui=装备双线**: 武器装备(Kinta锻造)+纺织(Yui衣物)——两个跟随NPC各有专属制作功能
