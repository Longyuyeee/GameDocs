# 脑洞小游戏的设计创新 - 中国式家长(Chinese Parents)

> 游戏版本: Chinese Parents (PC, 2018年)

## 一、概述

中国式家长的"脑洞"(Brain Hole)是养成品类中**最独特的活动执行系统**。不是传统的"选课程→数值增加"——而是每回合进入一个策略小游戏——用有限AP点击方块获取属性/天赋/悟性。这个设计将"养成"从一个"决策"系统变成了一个"策略+随机"系统。

## 二、与传统养成活动系统的对比

| 传统养成(PM系列/火山的女儿) | 中国式家长(脑洞) |
|------|------|
| 选课程→自动获得属性 | **策略点击→每步选择** |
| 结果确定(属性固定增加) | **结果不确定(随机方块分布)** |
| 没有"操作"只有"决策" | **有操作(点击顺序/优先级)** |
| 每回合=做出选择 | **每回合=玩一个小游戏** |

## 三、脑洞=养成×Roguelite的地牢探索

| 维度 | Roguelite地牢 | 脑洞 |
|:--:|------|------|
| 空间 | 房间→下楼 | **层→下楼** |
| 资源 | HP/金币 | **AP(行动力)** |
| 收益 | 装备/宝物 | **属性/天赋/悟性** |
| 策略 | 清房vs下楼 | **收集vs下楼(更深=更高收益)** |
| 永久成长 | 装备保留 | **天赋50%遗传(跨"游戏")** |

## 四、为什么脑洞有效

| 原因 | 说明 |
|------|------|
| **策略+随机=可重玩性** | 每回合方块分布随机→同样的培养路线≠同样的脑洞体验 |
| **下楼vs收集=核心张力** | 每层+3%全局收益→早点下楼=长期收益更高 vs 清当前层=即时收益 |
| **天赋=长线vs短线** | 天赋收益低(15AP换被动+1/回合)但50%遗传——长期投资者vs短视者 |
| **小游戏=减轻"安排课程"的枯燥** | 48回合×如果只是"选课"=无聊——脑洞=每回合都有操作乐趣 |

## 五、设计分析

1. **脑洞=养成游戏的"操作维度"** : 不是"决策→等待→看结果"——而是"决策→操作→看结果"——玩家有更多事做
2. **下楼优先=长线思维的教育**: 下楼的3%全局加成在早期不明显——但48回合累积下来=海量差异——教育中的"基础决定上限"
3. **天赋=为下一代投资**: 15AP换取50%遗传——类似现实中的"父母的教育投资遗传给孩子"——孩子受益于父母的选择
4. **脑洞未被任何其他养成游戏继承**: 火山的女儿回到传统"选课程"——PM系列也从未引入类似系统——中国式家长的独特遗产
