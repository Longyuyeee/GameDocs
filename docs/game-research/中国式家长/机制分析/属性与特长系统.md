# 属性与特长系统 - 中国式家长(Chinese Parents)

> 游戏版本: Chinese Parents (PC, 2018年)
> 数据来源: 萌娘百科, SteamAH, 3loumao

## 1. 六维属性

| 属性 | 图标 | 加成课程 | 关键阈值 |
|:--:|:--:|------|:--:|
| **智商(IQ)** | 绿三角 | 理科/编程/数学 | >2000(大文豪) |
| **情商(EQ)** | 红圆 | 语文/文学/沟通 | >12000(大文豪) |
| **体魄(Constitution)** | 蓝滴 | 体育/运动 | 过低→无法开脑洞 |
| **记忆力(Memory)** | 黄菱 | 英语/文科 | >14500(大文豪) |
| **想象力(Imagination)** | 紫方 | 艺术/创作 | >16000(大文豪) |
| **魅力(Charm)** | 米色 | 社交/演艺 | >1000(面子对决) |

- 魅力**不能从脑洞获得**(仅课程+社交)——独特属性

## 2. 面子(Face)——中国特色的"声望"

| 要素 | 说明 |
|------|------|
| 含义 | 父母的"面子"——社会地位/骄傲 |
| 获取 | 面子对决胜利/比赛获奖/好成绩 |
| 消耗 | 向父母索要物品(黄冈密卷等) |
| 太低 | 索要失败→无法获得关键道具 |

## 3. 特长(Trait)系统——四稀有度

| 稀有度 | 面子对决伤害 | 选秀成功率 | 获取概率 |
|:--:|:--:|:--:|:--:|
| 普通(Common) | 7伤害 | 低 | 高 |
| 稀有(Rare) | 14伤害 | 中 | 中 |
| 史诗(Epic) | 21伤害 | **高** | 低 |
| **传说(Legendary)** | **28+伤害** | **绝对选上** | **极低** |

- 获取方式: 日程活动随机获得(非脑洞!)
- 父母类型"忽视放养"→**+10%特长获取概率**(最优!)

## 5. 设计特点

1. **六维=中国教育评价体系的游戏化**: IQ/EQ/体魄/记忆力/想象力/魅力——不是西方RPG的"力量/敏捷/智力"
2. **面子=中国特有的"社会资本"** : 不是"声望(Fame)"——而是"父母的面子"——精妙的社会观察
3. **特长随机获取=现实的不确定性**: 不是"选了课就一定能得到特长"——而是"可能得到也可能得不到"
4. **传说特长=中国家长的"别人家的孩子"** : 极低概率出传说——但出了就是碾压——中国家长的"神童梦"
