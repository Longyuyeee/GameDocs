# 经济系统 - 中国式家长(Chinese Parents)

> 游戏版本: Chinese Parents (PC, 2018年)
> 数据来源: SteamAH, 3loumao

## 1. 货币体系

| 货币 | 获取 | 用途 |
|:--:|------|------|
| 悟性(Insight) | 脑洞方块(25/个)+父母奖励 | **解锁所有课程**(核心!) |
| 零花钱(Pocket Money) | 小学起/父母给/打工/红包 | 商店购物/社交/五年高考 |
| 行动力(AP) | 每回合自动恢复 | 脑洞小游戏/社交/打工/考试 |

## 2. 悟性经济(最核心)

| 用途 | 消耗范围 |
|:--:|:--:|
| 基础课程(拼音/算术等) | 低(10-50) |
| 中级课程 | 中(100-300) |
| 高级课程(黄冈密卷相关) | **高(500-999!)** |
| 策略 | 早期积攒→后期集中消耗 |

## 3. 零花钱经济

| 来源 | 金额 |
|:--:|:--:|
| 父母给(基础) | 低(基础职业) |
| 父母给(巨星后代) | **130¥/回合!** |
| 社会实践(剪纸/编程等) | 中-高(因技能而异) |
| 过年红包 | 不定(取决于小游戏表现) |
| 传家宝加成 | +10¥/回合 |

## 4. 商店消费

| 商品 | 价格 | 效果 |
|:--:|:--:|------|
| 小猪储钱罐 | — | →世界首富/投资之神路线 |
| 独立游戏 | — | →**游戏制作人路线**(遗传最强!) |
| 演员报名表 | — | →影帝路线 |
| 《跟马云学说话》 | — | →销售总监路线 |

| 减压物品 | 不等 | 降低压力(黄冈密卷冲刺期必需) |

## 5. 社会实践(打工)

| 打工 | 收入 | 技能要求 |
|:--:|:--:|:--:|
| 手工剪纸 | 中 | 理科4级(早期可学) |
| 编程相关 | 高 | 编程课程 |
| 演艺相关 | 高 | 演艺课程 |

## 6. 设计特点

1. **悟性=最稀缺的经济资源**: 不是"有钱就学"——而是"有悟性才能学"——有限的总悟性=有限的学习选择
2. **零花钱=上代职业的遗产**: 巨星后代130¥/回合vs科学家后代30¥——职业选择影响下一代经济基础
3. **五年高考=一次性机会**: 50¥/仅第7轮可买——错过=高考少一个准备工具
4. **商店物品=职业路线解锁**: 不是"买装备"——而是"买职业方向"——一个商品决定一条路线
