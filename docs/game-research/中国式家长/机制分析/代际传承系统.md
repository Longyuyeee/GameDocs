# 代际传承系统 - 中国式家长(Chinese Parents)

> 游戏版本: Chinese Parents (PC, 2018年)
> 数据来源: 萌娘百科, SteamAH, 3loumao

## 1. 概述

中国式家长的核心创新是**代际传承(Generational Inheritance)**——不是"一周目结束"而是"这一代的孩子成为下一代的家长"。每代的终点不是"游戏结束"——而是"新生命的开始"。

## 2. 代际遗传机制

| 遗传内容 | 比例 | 说明 |
|:--:|:--:|------|
| 天赋属性(Talent) | **50%** | 脑洞中收集的天赋→半数遗传下一代 |
| 职业→零花钱 | 100% | 上代职业决定下代零花钱(巨星=130¥/回合;科学家=30¥/回合) |
| 配偶→基础属性 | — | 与谁结婚影响下代初始属性 |
| 传家宝(Heirloom) | 100%(第2代起) | 每回合压力-10/+特长获取/考试保底等 |
| 特长(非继承) | — | 特长不继承(每代重新获取) |

## 3. 代际属性累积曲线

```
第1代: 基础属性(父母普通→初始低)
  ↓ 遗传50%天赋
第2代: 中等属性(开始有传家宝)
  ↓ 遗传50%天赋+累积
第3代: 较高属性
  ↓
第5代: 显著属性优势
  ↓
第10代: **"超人"级别**——三项属性接近20/高考属性8000+
```

## 4. 传家宝系统(第2代起)

| 传家宝 | 效果 | 推荐度 |
|:--:|------|:--:|
| 每回合压力-10 | 核心——减少心理阴影风险 | ★★★★★ |
| 获得史诗特长 | 加速特长收集 | ★★★★ |
| 获得稀有特长 | 加速特长收集 | ★★★★ |
| 每回合零花钱+10 | 经济保障 | ★★★ |
| 考试保底50幸福值 | 无视考试成绩 | ★★★ |

## 5. 职业→下代影响

| 上代职业 | 下代起始零花钱 | 其他影响 |
|:--:|:--:|------|
| 巨星(Idol) | **130¥/回合** | 最高 |
| 游戏制作人 | 高 | 遗传属性极强 |
| 世界首富 | 极高 | — |
| 科学家 | 30¥/回合 | 较低 |
| 普通人 | 基础 | — |

## 6. 推荐多代路线

| 代 | 目标 | 原因 |
|:--:|------|------|
| 第1代 | 游戏制作人 | **遗传属性最强**——为后代打基础 |
| 第2-5代 | 继续刷属性 | 累积天赋→后代越来越强 |
| 第5代+ | 尝试各种结局 | 属性已"超人"→可自由尝试 |
| 第10代+ | 全成就收集 | 属性碾压→什么结局都能达成 |

## 7. 设计特点

1. **代际=中国家庭的"望子成龙"循环**: 你赚不到的钱→期望下一代赚→下一代继续期望下下一代——精准的社会讽刺
2. **遗传50%=现实中的"家庭背景"** : 不是1:1继承特权——但优势确实会累积——公平的不完美
3. **传家宝=中国家庭的"家底"** : 不是RPG装备——而是"家族的积累"——父母给子女留下的东西
4. **多代=养成游戏中唯一的"时间深度"** : 火山的女儿12年→结束/中国式家长12年×N代→无限——唯一能体验"家族史"的养成游戏
