## 4. 设计特点

1. **16技能树=100+职业的路径**: 不是"几个属性"——而是16条完整技能树——每个选择都通向不同职业
2. **课程=渐进式解锁**: 不是"一开始就能学所有"——而是需要前置课程→类似科技树
3. **999悟性=终极课程的门槛**: 后期课程需要大量悟性→早期必须积累→资源规划的终极考验
4. **忍术=隐藏/恶搞路线**: 忍者文化→上忍——与其他"现实"课程形成反差——游戏中的幽默
