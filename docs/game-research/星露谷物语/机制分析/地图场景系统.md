## 1. 完整地图区域总览

星露谷物语的世界由多个相互连接的区域组成，每个区域具有独特的生态、资源和用途。

## 2. 矿洞系统详解

### 2.1 基础信息

| 属性 | 数值 |
|------|------|
| **解锁条件** | 游戏第2天，乔家(Joja)公司来信后自动解锁 |
| **位置** | 山区(罗宾家东北方向) |
| **层数** | 120层(固定) |
| **电梯** | 每5层一个电梯点，首次到达后永久解锁 |
| **重置** | 每天自动重置(除特殊层外) |
| **时间流逝** | 矿洞内时间与外界同步 |

### 2.2 三大区域结构

矿洞分为三个主题区域，每个区域有独特的矿石、怪物和环境特征：

| 区域 | 层数范围 | 主题 | 主要矿石 | 主要怪物 |
|------|---------|------|---------|---------|
| 棕色泥土层(Brown Earth) | 1-39 | 棕色泥土环境(31-39层暗影变体) | 铜矿(大量31-39层出现) | 绿史莱姆(Green Slime)、臭虫(Bug)、杜伊鬼(Duggy)、岩石蟹(Rock Crab)、蛆虫(Grub)、洞穴蝇(Cave Fly) |
| 冰冻层(Frozen) | 40-79 | 冰雪环境(70-79层冰城堡变体) | 铁矿(大量41+层出现)，钻石从50层开始出现(1/500石头) | 冰霜史莱姆(Frost Slime)、冰蝙蝠(Frost Bat)、灰尘精灵(Dust Sprite)、幽灵(Ghost) |
| 熔岩层(Lava) | 80-120 | 熔岩环境(110-119层深红变体) | 金矿(81+层出现)，铱矿极少量出现 | 红色史莱姆(Red Slime)、熔岩蝙蝠(Lava Bat)、熔岩蟹(Lava Crab)、暗影蛮族(Shadow Brute)、暗影萨满(Shadow Shaman)、金属大头(Metal Head)、鱿鱼娃(Squid Kid) |

### 2.4 矿洞特殊层

| 特殊层类型 | 特征 | 出现层数 |
|-----------|------|---------|
| **怪物层(Infested Floor)** | 大量怪物集中出现，必须击杀全部怪物或到达楼梯才能继续 | 随机出现(非多重) |
| **蘑菇层(Mushroom Floor)** | 大量红蘑菇和紫蘑菇生长，采集后次日重新刷新 | 随机出现(非多重) |
| **地牢层(Dungeon Floor)** | 更大面积，更多怪物和资源 | 随机出现 |

> 注：每天每个特殊层类型最多出现1个，且不会在电梯层(5的倍数)出现。

## 3. 骷髅洞穴系统

### 3.1 基础信息

| 属性 | 数值 |
|------|------|
| **位置** | 卡利科沙漠(Calico Desert)北部 |
| **解锁条件** | 获得头骨钥匙(矿洞120层宝箱) |
| **楼层机制** | **无限层**，随机生成，**无电梯**，每次从头开始 |
| **时间流逝** | 比外界慢(每游戏小时54秒 vs 外界43秒) |
| **下降方式** | 楼梯(Staircase，99石头合成1个)或找竖井(Shaft) |
| **竖井机制** | 打破石头20%概率出现，一次跳3-15层 |
| **竖井伤害** | 每跳过1层受到3倍伤害(最少保留1HP) |

### 3.2 重要里程碑

| 层数 | 奖励/事件 | 条件 |
|------|----------|------|
| 100层 | **铱蛇奶(Iridium Snake Milk)**，永久+25最大生命值 | 阅读秘密笔记#10后到达，遇到齐先生(Mr. Qi) |
| 100层 | 齐先生的赞扬 | 使用不超过10个楼梯到达 |

### 3.3 关键产出公式

**铱矿产出公式：**
```
基础铱矿出现概率 = 0.15% × 当前层数
例如：层 10 = 1.5%，层 100 = 15%，层 500 = 75%
```
铱矿节点的实际出现概率随深度线性增加，每层增加约0.15%。在深层(200+)铱矿密集分布。

**宝藏房间概率公式：**
```
宝藏房间基础概率 = 0.01 + (每日运气 / 10) + (运气增益 / 100)
- 基础概率：1%
- 每日运气范围：-0.1 ~ +0.1（影响±1%）
- 运气增益：每级+1% (如香辣鳗鱼+1运气=+1%)
- 最佳情况下可达约3%
```
宝藏房间出现在第10层之后，从石头中打开(非挖掘)，独立于普通楼层生成。

**史前楼层(Prehistoric Floor)概率：**
```
出现概率 ≈ 2.2%
- 特殊楼层，长满野草和树木
- 出现霸王龙(Pepper Rex)等史前怪物
- 可采集特有的史前植物
```
