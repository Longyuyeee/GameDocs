## 1. 战斗定位与概览

战斗在星露谷物语中约占 **15-20%** 的游戏内容，主要集中在矿井/骷髅洞穴/火山地牢的探索和战斗中。

### 1.1 战斗核心机制

| 项目 | 说明 |
|:----|:----|
| **生命值(HP)** | 初始100HP，每级战斗技能+5HP，理论最大约245HP |
| **能量(Energy/Stamina)** | 不同于HP，用于工具活动，武器战斗不消耗能量 |
| **攻击伤害** | 取决于武器基础伤害、锻造强化、附魔、食物增益 |
| **暴击率(Critical Hit Rate)** | 基础暴击率+武器暴击率+海蓝宝石锻造+侦察兵职业 |
| **暴击威力(Critical Power)** | 暴击时额外伤害倍数，基础x3，可强化 |
| **防御(Defense)** | 来自靴子和部分戒指，减少受到的伤害 |
| **免疫(Immunity)** | 减少负面状态(debuff)持续时间和概率 |

### 1.2 战斗区域总览

| 区域 | 难度 | 推荐等级 | 主要战利品 |
|:----:|:----:|:--------:|-----------|
| 矿井1-39层 | 低 | Lv1-30 | 铜矿、绿宝石、初期装备 |
| 矿井40-79层 | 中 | Lv31-60 | 铁矿、钻石、冰封晶球 |
| 矿井80-120层 | 中高 | Lv61-90 | 金矿、岩浆晶球、黑曜石之刃 |
| 骷髅洞穴 | 高 | Lv90+ | 铱矿、银河之魂、宝藏 |
| 火山地牢 | 高 | Lv90+ | 火山晶石、龙牙、矮人武器 |
| 秘密森林 | 低 | 任意 | 桃花心木、史莱姆 |
| 下水道/突变虫穴 | 中 | Lv30+ | 特殊鱼类 |
| 危险矿井(齐先生挑战) | 极高 | 满级+顶级装备 | 放射性矿石、铱针 |

## 3. 骷髅洞穴系统

### 3.1 基础信息

| 属性 | 数值 |
|:----|:----|
| **位置** | 卡利科沙漠(Calico Desert)北部 |
| **解锁条件** | 获得头骨钥匙(矿井120层)后进入沙漠 |
| **楼层机制** | **无限层**，随机生成，**无电梯**，每次从头开始 |
| **时间流逝** | 比外界慢(每游戏小时54秒 vs 外界43秒) |
| **下降方式** | 楼梯(Staircase，99石头合成1个)或竖井(Shaft) |
| **竖井概率** | 打破石头20%概率出现，一次跳3-15层 |
| **竖井伤害** | 每跳过1层受到3倍伤害(最少保留1HP) |

### 3.2 铱矿产出公式

```
铱矿节点出现概率 = 0.15% × 当前层数
```

| 深度 | 铱矿出现概率 | 实际感受 |
|:----:|:-----------:|---------|
| 10层 | 1.5% | 几乎看不见 |
| 50层 | 7.5% | 偶尔出现 |
| 100层 | 15% | 开始常见 |
| 200层 | 30% | 大量铱矿 |
| 500层 | 75% | 遍地铱矿 |

### 3.3 宝藏房间公式

| 参数 | 公式/数值 |
|:----|:----------|
| **基础概率** | 0.01(1%) |
| **每日运气影响** | +每日运气(范围-0.1~+0.1)÷10 |
| **运气增益影响** | +运气增益等级÷100 |
| **最终概率公式** | P = 0.01 + (dailyLuck÷10) + (luckLevel÷100) |
| **最佳情况** | 约0.01+0.01+0.06=0.03(3%) |
| **出现条件** | 10层以上，从石头中打开(非挖地) |

## 4. 火山地牢与锻造系统

### 4.1 基础信息

| 属性 | 数值 |
|:----|:----|
| **位置** | 姜岛北部火山内 |
| **层数** | 10层(随机生成，第5层和第10层固定) |
| **解锁** | 到达姜岛北部 |
| **通关后** | 解锁电梯+第10层到入口的快捷通道 |
| **重置** | 可反复探索 |

### 4.2 各层功能

| 楼层 | 功能 |
|:----:|:----|
| 1-4 | 随机生成层：怪物、宝箱、矿石、熔岩阻挡 |
| **5** | **固定层：矮人商店(Dwarf Shop)+水池(可装水)** |
| 6-9 | 随机生成层：怪物密集，含铱矿及火山晶石(Cinder Shard) |
| **10** | **固定层：锻造台(Forge)** |

### 4.3 锻造台系统

#### 4.3.1 武器锻造强化

在火山锻造台使用火山晶石(Cinder Shard)强化武器，最多3次：

| 锻造等级 | 所需火山晶石 |
|:--------:|:-----------:|
| 第1次锻造 | 10 |
| 第2次锻造 | 15 |
| 第3次锻造 | 20 |

每次锻造需附加一颗宝石，效果如下：

| 宝石 | 英文名 | 每级效果 |
|:----:|:------:|:---------|
| **红宝石** | Ruby | +10%伤害(最小/最大值) |
| **祖母绿** | Emerald | 第1/3级+2速度，第2级+3速度 |
| **海蓝宝石** | Aquamarine | +4.6%暴击率 |
| **翡翠** | Jade | +10%暴击威力 |
| **紫水晶** | Amethyst | +1击退 |
| **黄玉** | Topaz | +1防御 |
| **钻石** | Diamond | 随机提升剩余空格 |
