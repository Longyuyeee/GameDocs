## 1. 时间流速与昼夜机制

### 1.1 时间流速

| 项目 | 数据 |
|------|------|
| 1游戏分钟(1 Game Minute) | ≈ 0.7现实秒(Real Seconds) |
| 1游戏小时(1 Game Hour) | ≈ 42现实秒 |
| 全天(6:00 AM ~ 2:00 AM) | ≈ 13.5~14现实分钟 |
| 游戏内每日开始时间 | 6:00 AM |
| 强制昏倒时间 | 2:00 AM |
| 每日总时长 | 20游戏小时(1,200游戏分钟) |
| 存档时机 | 仅睡觉后 |

### 1.2 时间消耗参考

| 活动 | 时间消耗 |
|------|---------|
| 使用工具(锄/镐/斧/水壶) | 约2-3秒/次 |
| 钓鱼(上钩-完成) | 约15秒~2分钟 |
| 进入/离开建筑 | 约2-3秒 |
| 购买物品(商店界面) | 时间暂停 |
| 菜单操作 | 时间暂停 |
| 对话/剧情 | 时间暂停 |
| 节日活动 | 时间停止流逝 |
| 矿洞/骷髅洞穴 | 时间与外界同步 |
| 火山地牢 | 时间与外界同步 |

### 1.3 昏倒机制

| 条件 | 处罚 |
|------|------|
| 2:00 AM未睡觉 | 强制昏倒 |
| 能量降至-15 | 体力耗尽昏倒 |
| 2:00 AM昏倒(在家) | 损失10%金币(最多1,000g) |
| 2:00 AM昏倒(在矿洞) | 损失10%金币(最多1,000g)+可能丢失物品 |
| 体力耗尽昏倒(非2:00AM) | 无金币损失，次日能量恢复50% |
| 矿洞内昏倒 | 被林纳斯(Linus)或冒险者公会救起，可能丢失物品 |

## 2. 能量/体力系统

### 2.1 基础数据

| 项目 | 数值 |
|------|------|
| 初始最大能量(Energy) | 270 |
| 各星之果实(Stardrop)加成 | 每个+34(共7个) |
| 星之果实累计加成 | +238 |
| **理论最大能量** | **508** |
| 昏倒条件 | 能量降至-15 |
| 2:00AM昏倒处罚 | 损失10%金币(最多1,000g) |

### 2.2 体力消耗行动

| 行动 | 消耗体力 |
|------|---------|
| 锄地 | 2-4点/次(取决于工具等级) |
| 浇水 | 2点/次(非雨天) |
| 用镐敲石头 | 2-4点/次 |
| 用斧砍树 | 2-4点/次 |
| 钓鱼(上钩+捕获) | 8点/次 |
| 使用工具战斗 | 2-4点/次 |

### 2.3 不消耗体力的行动

割草(镰刀)、使用武器战斗(近战)、走路/跑步、收获作物、淘金、挤奶、剪毛、骑马、放置机器、食用大多数食物、与NPC对话、进入建筑。

### 2.4 睡眠恢复规则

| 睡觉时间 | 能量恢复比例 | 说明 |
|---------|------------|------|
| 午夜12:00(凌晨0:00)前 | **100%满恢复** | 最佳 |
| 凌晨1:00 | ~75%恢复 | 中等 |
| 凌晨2:00前 | ~50%恢复 | 最低 |
| 体力耗尽(疲惫状态)时睡觉 | 次日仅恢复50% | 惩罚效果 |

**恢复公式参考：**
```
能量恢复 = 最大能量 × 恢复系数(0.5~1.0)
- 12:00AM前睡觉：恢复系数 ≈ 1.0
- 1:00AM睡觉：恢复系数 ≈ 0.75
- 2:00AM睡觉：恢复系数 ≈ 0.5
```

### 2.5 体力恢复方式总汇

| 方式 | 恢复效率 | 备注 |
|------|---------|------|
| **睡觉** | 50%~100% | 取决于睡觉时间 |
| **吃食物/料理** | 不等 | 最常用，推荐野外零食(Field Snack)和沙拉(Salad) |
| **泡温泉(Spa)** | 缓慢恢复 | 铁路区域温泉，期间时间流逝 |
| **星之果实茶(Stardrop Tea)** | +250能量 | 奖励物品 |
| **泡菜/果酱** | 中等 | 罐头瓶加工品 |
| **奶酪/蛋黄酱** | 中等 | 动物产品加工 |
| **采矿/战斗食物** | 不等 | 如香辣鳗鱼(+115能量)、南瓜汤(+200能量) |

### 2.6 全部7个星之果实(Stardrop)获取方法

星之果实是永久增加最大能量的稀有物品，每个+34。

| 编号 | 获取方式 | 具体条件 | 阶段 |
|:----:|---------|---------|:----:|
| 1 | **星露谷展览会(Stardew Valley Fair)** | 秋季16日，用2,000星星币(Star Tokens)购买(仅一次) | 中期 |
| 2 | **矿井100层宝箱** | 首次到达矿井100层 | 中期 |
| 3 | **配偶/室友赠送** | 婚后好感度达到12.5心时伴侣赠送 | 后期 |
| 4 | **科罗布斯商店(Krobus Shop)** | 下水道20,000g购买 | 中期 |
| 5 | **秘密森林雕像(Old Master Cannoli)** | 将一颗甜宝石莓(Sweet Gem Berry)供奉 | 中期 |
| 6 | **钓鱼大师成就(Master Angler)** | 钓到所有鱼类后威利(Willy)邮寄 | 后期 |
| 7 | **博物馆全捐赠(Museum Complete)** | 捐赠全部95件(53种矿物+42种古物) | 后期 |

## 4. 年份机制

### 4.1 各年份新增内容

| 年份 | 新增内容 | 说明 |
|:----:|---------|------|
| **第1年** | 基础体验 | 春季第3天固定下雨，逐步解锁所有区域，社区中心建设 |
| **第2年** | **肯特(Kent)回归** | 春季第4天出现在镇上(战后回家) |
| | 作物价格变化 | 部分作物种子价格调整，皮埃尔出售更多种类 |
| | 木料/石材涨价 | 木材10g→50g，石材20g→100g(罗宾处购买) |
| | 新的作物种子 | 大蒜(Garlic)、红叶卷心菜(Red Cabbage)、洋蓟(Artichoke)等解锁 |
| | 混合种子(Mixed Seeds)可产出更多品种 | — |
| **第3年** | **爷爷的评分(Grandpa's Evaluation)** | 春季第1天鬼魂出现进行评估 |
| **无年份上限** | 游戏可无限进行 | 第3年后无额外新增内容 |

### 4.2 游戏阶段划分

| 阶段 | 时间范围 | 特征 |
|:----:|---------|------|
| **前期** | 第1年春季~夏季 | 熟悉游戏，基础工具升级，初期资金积累 |
| **中期** | 第1年秋季~第2年夏季 | 社区中心完成，工匠设备投产，温室解锁，矿井探索 |
| **后期** | 第2年秋季之后 | 骷髅洞穴深入，姜岛探索，大规模自动化生产，完美追求 |
| **终局** | 100%完美(Perfection) | 全部技能10级+精通，完全体农场，所有成就 |

## 7. 天气系统

### 7.1 天气类型总表

| 天气类型 | 英文名 | 出现季节 | 出现概率 | 效果 |
|---------|--------|:--------:|:--------:|------|
| **晴天** | Sun | 全年 | 基础天气 | 无特殊效果，默认天气 |
| **雨天** | Rain | 春/夏/秋 | 春约18.3%/夏约13%/秋约18.3% | 地面湿润(自动浇灌)，可钓雨天才有的鱼(鲶鱼/鳗鱼等)，触发人鱼吊坠购买条件 |
| **暴风雨** | Storm/Thunder | 春夏为主 | 低于雨天概率 | 同雨天效果+闪电(可被避雷针收集→**电池组**)，**夏13和26必定暴风雨** |
| **雪天** | Snow | **冬季** | 约63% | 地面覆盖积雪，冬季特有采集物出现 |
| **风天** | Wind | 春/秋 | 春约16.6%/秋约49.8% | "部分多云，微风"，可触发特殊采集事件 |
| **绿色雨天** (1.6新增) | Green Rain | **夏季特定日期** | 低 | 绿色天空，生成绿色雨树(Green Rain Tree)和苔藓(Moss) |
| **节日** | Festival | 按节日日期 | 必定晴天 | 时间停止流逝，动物自动喂食 |
| **婚礼** | Wedding | 婚后第3天 | 必定晴天 | 全天无特殊天气效果 |

### 7.3 天气对游戏的影响

| 天气 | 游戏影响 | 策略建议 |
|:----:|---------|---------|
| **雨天** | 自动浇灌，节省浇水时间；特定鱼类出现(鲶鱼Catfish/鳗鱼Eel/西鲱Shad/红鲷鱼Red Snapper/狗鱼Pike) | 最适合矿井探索或社交送礼(村民更多在室内) |
| **暴风雨** | 闪电击中作物/树木(概率)，避雷针收集→**电池组(Battery Pack)**；同时有雨天钓鱼效果 | 确保有足够避雷针(建议10-20个)，电池组是高级制作的核心材料 |
| **雪天** | 冬季特有采集物出现，户外行走速度略微下降 | 适合采矿和钓鱼 |
| **风天** | 秋季49.8%高概率发生 | 不影响正常活动 |
| **绿色雨天** | 生成苔藓(Moss)和绿色雨树，1.6新增 | 特定采集机会 |
