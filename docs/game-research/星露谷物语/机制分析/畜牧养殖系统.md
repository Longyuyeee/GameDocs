## 3. 好感度系统

### 3.1 好感度机制

- **范围**：0 - **1,000** 点（对应 0 - 5 心）
- **每心**：200 点
- **显示**：右键点击动物查看

### 3.2 好感度变化

| 行为 | 好感度变化 |
|------|:---------:|
| 每日抚摸(第一次) | +15 |
| 每日抚摸(宠物大师职业) | +30 |
| 喂食(有干草或吃草) | +0(喂食本身不加好感) |
| 挤奶/剪毛 | +5 |
| 吃草在户外 | +8(仅晴天，加心情) |
| 不喂食(饥饿) | **-20** |
| 淋雨(无家可归) | -5 |
| 未抚摸 | -5 到 -10(取决于心情) |
| 被关在室内一整夜 | **-20** |

### 3.3 好感度影响

| 好感度 | 心数 | 效果 |
|:-----:|:----:|------|
| 0-199 | 0 | 基本产品（普通品质） |
| **200-399** | **1** | **可产出大产品(大蛋/大壶牛奶)** |
| 400-599 | 2 | 高品质概率提升 |
| 600-799 | 3 | 更高品质概率 |
| 800-999 | 4 | 高品质产品为主 |
| **1,000** | **5** | **最高品质概率** |

## 4. 心情系统

### 4.1 心情机制

- **范围**：0 - **255**
- **影响**：x1.5 倍售价加成（心情>200时）
- **初始**：每天开始随机设定（受吃饭、户外等因素影响）

### 4.2 心情变化因素

| 行为 | 心情变化 |
|------|:--------:|
| 吃草(户外) | +8 |
| 抚摸(好感高时) | 最高+32 |
| 抚摸(好感低时) | 最低0 |
| 晴天在户外 | 提升 |
| 淋雨/冬季在户外 | 降低 |
| 不喂食 | 大幅降低 |
| 未关入畜棚/被攻击 | 降低 |

### 4.3 心情对产出的影响

- **心情 > 200**：所有产品价值**x1.5**（最终售价乘以1.5）
- **心情 100-200**：正常产出
- **心情 < 100**：产出数量可能减少（如鸡可能不下蛋）

## 5. 品质决定机制

### 5.1 品质判定流程（4步法）

动物产品的品质通过以下**4步判定**决定：

```
第1步: 判定是否铱星(Iridium)品质
  ↓ 否
第2步: 判定是否金星(Gold)品质
  ↓ 否
第3步: 判定是否银星(Silver)品质
  ↓ 否
第4步: 输出普通(Normal)品质
```

### 5.2 品质概率公式

```
// 基础分数 = 好感度 + 心情修正
baseScore = ((friendship / 1000) - (1 - (mood / 255)))

// 铱星概率
iridiumChance = mood / 255 / 2

// 金星概率（未铱星时）
goldChance = min(0.5, baseScore)

// 银星概率（未金/铱时）
silverChance = min(0.5, baseScore)
```

### 5.3 品质概率示例

**场景1：高好感+高心情（5心，心情255）**

| 品质 | 概率 |
|:----:|:----:|
| 铱星(Iridium) | **50%** |
| 金星(Gold) | **50%** |
| 银星(Silver) | 0% |
| 普通(Normal) | 0% |

**场景2：中等好感+中等心情（3心，心情150）**

| 品质 | 概率 |
|:----:|:----:|
| 铱星(Iridium) | ~29% |
| 金星(Gold) | ~40% |
| 银星(Silver) | ~25% |
| 普通(Normal) | ~6% |

**场景3：初获动物（0心，心情100）**

| 品质 | 概率 |
|:----:|:----:|
| 铱星(Iridium) | ~20% |
| 金星(Gold) | 0% |
| 银星(Silver) | 0% |
| 普通(Normal) | **~80%** |

### 5.4 品质与价格倍率

| 品质 | 价格倍率 | 示例：大壶牛奶(190G) |
|:----:|:-------:|:-----------------:|
| 普通(Normal) | x1.0 | 190 G |
| 银星(Silver) | x1.25 | 237 G |
| 金星(Gold) | x1.5 | 285 G |
| 铱星(Iridium) | x2.0 | 380 G |

**心情 > 200 时的额外加成**：最终售价再乘以 **1.5**

## 7. 饲料系统

### 7.1 干草(Hay)

| 属性 | 数据 |
|:----:|:----:|
| 来源 | 用镰刀割草获得 |
| 干草产出率 | 割草 **50%概率** 得干草 |
| 金镰刀 | **75%概率** 获得干草 |
| 干草售价 | 玛妮处购买 **50 G /个** |
| 冬季节省 | 冬季必须喂干草或购买 |

### 7.2 筒仓(Silo)

| 属性 | 数据 |
|:----:|:----:|
| 建造费用 | **100 G** |
| 材料 | 石头x100 + 粘土x10 + 铜锭x5 |
| 干草容量 | **240 个** |
| 存储查看 | 右键点击筒仓查看储量 |

**筒仓机制**：
- 割草时，干草自动存入筒仓
- 筒仓满后割草不再获得干草
- 从筒仓取干草：进入畜棚/鸡舍，从饲料槽处的干草堆取出
- 建议：建造2个以上筒仓（合计容量480）

### 7.3 牧草(Grass)

| 属性 | 数据 |
|:----:|:----:|
| 购买 | 皮埃尔杂货店 **100 G / 10个** |
| 种植 | 播撒牧草种子到地面 |
| 生长 | 春季开始生长，会在周围蔓延 |
| 吃草 | 动物在户外**直接吃牧草**（心情+8） |
| 冬季 | 牧草冬季枯萎消失 |

**牧草种植建议**：
- 春季第一天大量种植（可用栅栏保护避免动物一次吃光）
- 每只动物至少需要约1-2格牧草维持

### 7.4 饲料消耗

| 季节 | 饲料需求 |
|:----:|---------|
| 春季(户外晴) | 动物吃草不消耗干草 |
| 春季(雨天) | 每只动物1干草/天 |
| 夏季(户外晴) | 动物吃草不消耗干草 |
| 夏季(雨天/暴风雨) | 每只动物1干草/天 |
| 秋季(户外) | 整体同春夏 |
| **冬季** | **每只动物1干草/天（必须室内喂养）** |

**一只动物冬季饲料成本**：28天 x 50 G = **1,400 G**

## 9. 自动化系统

### 9.1 自动喂食系统

| 建筑等级 | 是否有自动喂食 |
|:--------:|:-------------:|
| 鸡舍/畜棚(基础) | 无（需手动放干草到饲料槽） |
| 大型(升级后) | 无（仍需手动） |
| **高级(最终)** | **有**（自动从筒仓取干草饲喂） |

### 9.2 自动采集器(Auto-Grabber)

| 属性 | 数据 |
|:----:|:----:|
| 价格 | **25,000 G** |
| 购买地点 | 玛妮的牧场(Marnie's Ranch) |
| 功能 | 自动收集动物产品（奶、蛋、毛等） |
| 容量 | 可储存**多日产品** |
| 范围 | 所在建筑的全部动物 |

**工作原理**：
- 每天早上自动收集建筑内所有动物的产品
- 无需手动挤奶/剪毛/拾蛋
- 不收集：松露（猪的户外产物）

### 9.3 自动抚摸机(Auto-Petter)

| 属性 | 数据 |
|:----:|:----:|
| 价格 | **50,000 G** |
| 获取方式 | Joja路线购买 / 骷髅洞穴稀有掉落 |
| 功能 | 每日自动抚摸动物（代替手动抚摸） |
| 好感效果 | 每日自动+15好感（同手动抚摸） |
| 叠加性 | 可与手动抚摸叠加 |

**注意**：
- Joja路线会员可直接购买
- 社区中心路线仅能通过骷髅洞穴宝箱/特殊怪物掉落
- 建议：后期批量生产该设备

### 9.4 自动化效果对比

| 设备 | 功能 | 手动耗时 | 自动耗时 |
|:----:|------|:--------:|:--------:|
| 自动采集器 | 收集产品 | 10-20秒/动物 | **0秒** |
| 自动抚摸机 | 增加好感 | 5秒/动物 | **0秒**(自动) |
| 高级建筑 | 自动喂食 | 30秒/建筑 | **0秒** |
| 封闭系统 | 全部自动化 | 每日5-10分钟 | **完全免维护** |

### 10.4 养猪流（主流赚钱方案）

