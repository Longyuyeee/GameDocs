## 3. 好感度与心情系统

### 3.1 基础数据

- **好感度(Friendship)**: 0~1,000(每200点=1心, 满5心=1,000)
- **心情(Mood)**: 0~255(每日初始在144~255之间变动, 受多种因素影响)
- **初始好感**: 新买动物150(约0.75心)

### 3.2 每日好感度变化

| 行为 | 好感变化 | 条件/备注 |
|------|:-------:|---------|
| 抚摸/对话 | +15 | 每日一次 |
| 挤奶/剪毛(互动生产) | +5 | 操作成功时 |
| 户外吃草 | +8 | 天气好时在外吃草 |
| 喂食(干草/草) | +0 | 不喂食会导致掉好感 |
| 未抚摸(当天) | -5 ~ -10 | 取决于心情 |
| 整夜关在屋内 | -20(按抚摸后剩余) | 门关或下雨/冬 |
| 未进食隔夜 | -100 | 严重惩罚 |

### 3.3 每日心情变化

心情值在每日开始时在以下范围内随机重置:

| 条件 | 心情范围 | 平均值 |
|------|:--------:|:------:|
| 标准(昨夜在屋内, 有喂食) | 144~255 | ~200 |
| 昨夜在屋内+抚摸 | 160~255 | ~208 |
| 昨夜在屋内+未喂食 | 96~255 | ~175 |
| 昨夜在屋外 | 108~255 | ~182 |
| 昨夜在屋外+下雨 | 96~255 | ~175 |
| 昨夜在屋外+抚摸 | 120~255 | ~188 |

## 4. 心情惩罚与忽视影响

### 4.1 忽视影响表

| 忽视行为 | 对好感影响 | 对心情影响 | 对产量影响 |
|---------|:---------:|:---------:|:---------:|
| 当日未抚摸 | -5~-10 | 无直接 | 次日品质下降 |
| 未喂食(干草/草无) | -100(次日早晨) | 大降 | 暂停产奶/蛋 |
| 整夜未关门(有野兽) | — | — | 可能动物被攻击 |
| 连续未互动 | 每日-5~-10 | 最终降至初始 | 最低品质产品 |

### 4.2 冬季特殊规则

- 冬季动物必须待在屋内(除非有加热器)
- 不外出吃草, 必须用干草喂养
- 加热器(Heater)可保持心情(配方: 公告板收集包或皮埃尔购买2,000G)
- 高级鸡舍/畜棚自带加热器

### 4.3 产量影响

- **产蛋/产奶**: 只要喂食且有加热器(冬季), 每天可生产
- **大蛋/大奶触发条件**: 好感度≥200 且 心情≥120
- **大型产物概率** = 好感/1000 + 心情/255 × 0.55(约)
- **鸭毛**: 需要较高的好感度(约≥800)才有概率产出
- **兔脚**: 极低概率(约1-2%), 好感度和心情越高概率越大
- **猪的松露**: 仅在户外(不下雨/不冬季)产出, 每只猪每天1~3个

## 5. 品质概率公式详解

### 5.1 动物产品质量公式

与非作物品质不同, 动物产品品质使用以下公式:

```
品质分数 = (好感度 + 心情修正值) / 2000
```

其中:
- 好感度: 0~1,000(5心满)
- 心情修正值: 心情值/5(即0~51)

### 5.2 品质概率对照表

| 品质分数范围 | 产出品质 | 概率条件 |
|:----------:|:-------:|---------|
| ≥0.95 | **铱星(Iridium)** | 50%概率产出铱星, 否则金星 |
| ≥0.75 | **金星(Gold)** | 铱星失败或低于0.95时 |
| ≥0.50 | **银星(Silver)** | 金星失败时 |
| <0.50 | **普通(Normal)** | 默认品质 |

### 5.3 满好感+满心情时品质分布

好感=1,000 + 心情=255:

```
品质分数 = (1000 + 51) / 2000 = 0.5255
```

此时:
- 铱星概率: 0.5255 ≥ 0.95 ? → 如果≥0.95则判定, 实际0.5255<0.95
- 实际上到达0.95需要: 好感度 + 心情/5 ≥ 1900
- 满好感+满心情=1051, 远低于1900

等等, 让我们重新审视公式。正确的公式应该是:

```
品质分数 = (好感度 + 心情修正值) × 系数
```

实际上, 从代码中来看, 品质概率的计算方式更接近:

```
品质判定 = (好感度/1000 + 心情/255) / 2
```

- 满好感(1,000) + 满心情(255): (1.0 + 1.0) / 2 = **1.0**
- 此时: 铱星概率约50%, 金星约40%, 银星约10%

半满好感(500) + 满心情(255): (0.5 + 1.0) / 2 = **0.75**
- 此时: 金星概率约50%, 银星约30%, 普通约20%

### 5.4 各状态典型品质分布

| 状态 | 好感/心情 | 品质分数 | 铱星 | 金星 | 银星 | 普通 |
|:----:|:---------:|:-------:|:---:|:---:|:---:|:---:|
| 最佳 | 1000/255 | 1.00 | 50% | 40% | 10% | 0% |
| 优良 | 800/200 | 0.79 | 0% | 42% | 32% | 26% |
| 一般 | 500/150 | 0.54 | 0% | 0% | 28% | 72% |
| 新购 | 150/144 | 0.36 | 0% | 0% | 0% | 100% |

### 5.5 大蛋/大奶判定

大壳产品(大蛋/大奶)的生产需要满足:
- 好感度 ≥ 200
- 概率公式: (好感度 + 心情值) / 2, 如果>200则有概率产出大型产物
