## 1. 基础属性系统

### 1.1 角色属性总览

| 属性 | 说明 | 初始值 | 最大值 |
|------|------|:-----:|:-----:|
| 生命值 (Health/HP) | 角色承受伤害的能力 | 100 | 215 |
| 精力值 (Energy/Stamina) | 执行劳作消耗的资源 | 270 | 508 |
| 攻击 (Attack) | 影响对敌人造成的伤害 | 0 | 视装备而定 |
| 防御 (Defense) | 减免受到的伤害 | 0 | 视装备而定 |
| 免疫 (Immunity) | 减少负面状态效果概率 | 0 | 视装备而定 |
| 暴击率 (Crit. Chance) | 造成暴击的概率 | 武器基础 | 视装备而定 |
| 暴击威力 (Crit. Power) | 暴击时的额外伤害倍率 | 武器基础 | 视装备而定 |
| 速度/攻速 (Speed) | 武器挥动速度 | 武器基础 | 视装备而定 |
| 运气 (Luck) | 影响多种随机事件 | 0 | 视增益而定 |

### 1.2 属性效果说明

| 属性 | 效果描述 |
|------|---------|
| **攻击 (Attack)** | 每点攻击力增加基础伤害，伤害公式中的固定加成部分 |
| **防御 (Defense)** | 每点防御减少1点受到的伤害（伤害最低为1） |
| **免疫 (Immunity)** | 降低被施加负面状态（燃烧、冰冻、暗影等）的概率 |
| **暴击率 (Crit. Chance)** | 武器基础暴击率(约2%/点)，受海蓝宝石锻造和技能加成 |
| **暴击威力 (Crit. Power)** | 暴击时额外伤害加成，公式为 `3 + 暴击威力/50` 的倍率 |
| **速度/攻速 (Speed)** | 武器挥动速度，+4为最快档，-1为最慢档 |
| **运气 (Luck)** | 影响掉落率、开晶球质量、矿井竖井出现率等多种随机事件 |

## 6. 伤害计算公式

### 6.1 基础伤害公式

```
原始伤害 = (武器最小伤害 ~ 武器最大伤害) 之间的随机值
```

### 6.2 最终伤害公式

```
最终伤害 = [ 原始伤害 × 暴击倍率 + 攻击力(ATK) × 3 ] × 战士加成 × 蛮力加成 × 其他修正
```

**各参数说明：**

| 参数 | 说明 |
|------|------|
| 原始伤害 | 武器伤害范围内的随机值 |
| 暴击倍率 | 非暴击=1.0，暴击时=(3 + 暴击威力/50) |
| ATK × 3 | 角色的攻击力属性×3，固定加成部分 |
| 战士加成 | 选择"战士"职业时 ×1.1 |
| 蛮力加成 | 选择"蛮力"职业时 ×1.15 |
| 亡命徒加成 | 暴击时伤害×2(选择亡命徒职业) |
| 十字军附魔 | 对特定怪物种类+50%伤害 |
| 虫族杀手附魔 | 对虫族/蜘蛛/岩石蟹伤害翻倍 |

### 6.3 暴击率公式

```
实际暴击率 = [ 武器基础暴击率 + 海蓝宝石锻造加成 + 戒指加成 ] × 侦察兵1.5倍 + 每日运气/40
```

| 影响因素 | 数值 |
|---------|------|
| 武器基础暴击率 | 剑类2%，匕首3-10%，铱针10% |
| 海蓝宝石锻造 | 每级+4.6% |
| 幸运戒指(Lucky Ring) | 每枚+1运气(相当于+2.5%暴击率) |
| 侦察兵职业(Scout) | 暴击率×1.5 |
| 每日运气 | 每日运气值/40(约-0.1~+0.1) |

### 6.4 暴击倍率公式

```
暴击倍率 = (3 + 暴击威力/50) × (1 + 暴击威力加成百分比)
```

**亡命徒(Desperado)**：暴击伤害额外×2

| 武器 | 基础暴击威力 | 暴击倍率(非亡命徒) | 暴击倍率(亡命徒) |
|------|:----------:|:----------------:|:--------------:|
| 银河剑 | 0 | ×3.0 | ×6.0 |
| 铱针 | +200 | ×7.0 | ×14.0 |
| 无限匕首(翡翠锻造) | 每级+10% | 公式计算 | 公式计算 |

### 6.5 格挡减伤

剑类武器右键格挡时：
```
格挡后伤害 = 原伤害 × 50%
```

### 6.6 护甲减伤

```
实际受到伤害 = 怪物攻击力 - 角色防御力(最低为1)
```

## 8. 极限暴击流配装示例

### 目标：追求最高暴击率

| 装备/技能 | 暴击率加成 | 说明 |
|----------|:---------:|------|
| 铱针 (Iridium Needle) | +10% | 基础暴击率最高的武器 |
| 海蓝宝石锻造×3 | +13.8% | 每级+4.6%暴击率 |
| 侦察兵职业(Scout) | ×1.5倍 | 暴击率变为1.5倍 |
| 幸运戒指×2 | ~+5% | 每枚+1运气(约+2.5%暴击率) |
| 每日运气(最佳) | ~+2.5% | 最佳运气日(每日运气/40) |
| 魔法糖冰棍 | ~+12.5% | +5运气相当于约+12.5%暴击率 |

**计算过程：**
```
基础暴击率 = 10%(铱针) + 13.8%(锻造) + 2.5%(戒指×2) = 26.3%
侦察兵加成后 = 26.3% × 1.5 = 39.45%
运气加成 = 5(魔法糖) + 1(每日最佳) = 6运气 → 相当于约15%
最终暴击率 ≈ 39.45% + 15% ≈ 54.45%
```

> **极限暴击流组合**：铱针(海蓝宝石×3) + 幸运戒指×2 + 侦察兵→亡命徒 + 魔法糖冰棍 + 姜汁汽水
> 暴击率可达 **~54%**，配合亡命徒暴击伤害×2，每次暴击造成约14倍伤害(铱针基础暴击威力+200)
