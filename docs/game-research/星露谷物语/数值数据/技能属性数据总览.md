## 1. 等级经验值表(通用)

五种技能使用**统一经验值表**, 每级上限10级, 总计需15,000 XP。

### 1.1 等级对应经验值

| 等级 | 本级需XP | 累计XP | 升级差值 |
|:---:|:-------:|:------:|:-------:|
| 1 | 100 | 100 | 100 |
| 2 | 280 | 380 | 180 |
| 3 | 390 | 770 | 110 |
| 4 | 530 | 1,300 | 140 |
| 5 | 850 | 2,150 | 320 |
| 6 | 1,150 | 3,300 | 300 |
| 7 | 1,500 | 4,800 | 350 |
| 8 | 2,100 | 6,900 | 600 |
| 9 | 3,100 | 10,000 | 1,000 |
| 10 | 5,000 | **15,000** | 1,900 |

### 1.2 各阶段累计

| 阶段 | 累计XP |
|:----:|:------:|
| Lv1~Lv5 | 2,150 |
| Lv6~Lv10 | 12,850 |
| 全部Lv10(5技能) | 75,000 |
| 全部Lv10 + 精通(5×10,000) | 125,000 |

### 1.3 经验与等级关系公式

```
经验值 = (等级 - 1) × 100 + 等级 × (等级 - 1) × 20
```

注: 等级Lv5和Lv10可选择**职业(Profession)**。可到下水道的**不确定雕像(Statue of Uncertainty)** 花费10,000G重置职业选择。

## 2. 耕种(Farming)技能

### 2.1 经验获取方式

| 行为 | 经验值 |
|:----:|:------:|
| 收获作物(每作物) | 取决于作物类型, 基础价低->经验少, 基础价高->经验多。公式: XP = 作物基础价/20 + 种植天数/农业等级加成 |
| 抚摸动物(每日) | 5 XP(每天仅限一次) |
| 收集动物产品(挤奶/剪毛) | 5 XP(每次) |
| 普通作物经验参考 | 防风草8 XP / 花椰菜23 XP / 蓝莓10 XP(每颗) |

| 3 | 蜂房(Bee House)、生长激素(Speed-Gro)配方、农夫午餐(Farmer's Lunch)食谱 |
| 4 | **罐头瓶(Preserves Jar)**、初级保湿土壤(Basic Retaining Soil)、铁栅栏配方 |
| **5** | **职业选择(农耕人/牧场主)** |

### 2.4 职业数值详解

| 职业 | 具体数值 | 适用场景 |
|:----:|---------|---------|
| 牧场主(Rancher) | 动物产品(蛋/奶/毛)直接售价+20% | 纯动物流 |

## 5. 钓鱼(Fishing)技能

### 5.1 经验获取方式

| 行为 | 经验值 |
|:----:|:------:|
| 钓上鱼(取决于鱼品质/难度) | 0~100+(鱼的基础经验+距离加成) |
| 采集蟹笼 | 5 XP/个 |
| 收集鱼塘产物 | 10 XP/次 |
| 完美钓鱼(鱼从未离开绿条) | 额外+50%经验 |

**单次钓鱼经验公式(近似)**:

```
XP = (鱼难度 × 3 + 完美加成 + 品质加成) / 4
```

- 小型普通鱼(鲢鱼 ~15XP)
- 大型困难鱼(传说鱼 ~70XP)
- 完美上钩额外加成 ~50%

## 7. 精通系统(1.6新增)

### 7.1 解锁条件

**五项技能全部达到Lv10后**, 自动解锁**精通山洞(Mastery Cave)**。位于秘密森林(Secret Woods)附近, 用钢斧劈开大圆木进入秘密森林后可见入口。

### 7.2 各精通领域经验需求

每个精通领域需要10,000 XP, 以总经验(跨技能)积累方式获取。每获得一个精通, 下一精通所需XP递增。

| 精通顺序 | 所需XP | 解锁技能 |
|:-------:|:-----:|---------|
| 第1个精通 | 10,000 | 取决于你投入的顺序 |
| 第2个精通 | 20,000 | 下一个 |
| 第3个精通 | 30,000 | 下一个 |
| 第4个精通 | 40,000 | 下一个 |
| 第5个精通 | 50,000 | 下一个 |
| **合计** | **150,000** | 全部5个 |

### 7.3 各精通奖励

| 精通领域 | 核心奖励 | 额外效果 |
|---------|---------|---------|
| **农耕精通(Farming Mastery)** | **铱镰刀(Iridium Scythe)** — 可收获任意作物 | 收割更大范围, 割草必产干草 |
| **采矿精通(Mining Mastery)** | **重型熔炉(Heavy Furnace)** 配方 — 一次可熔炼25个矿石 | 有概率产出额外金属锭 |
| **采集精通(Foraging Mastery)** | **神秘树种子(Mystic Tree Seed)** 配方 | 可种出神秘树(产神秘树液) |
| **钓鱼精通(Fishing Mastery)** | **高级铱金鱼竿(Advanced Iridium Rod)** — 支持双鱼饵+渔具 | 钓鱼等级+1 |
| **战斗精通(Combat Mastery)** | **饰品(Trinket)** 配方 | 解锁6种饰品(见装备数据), 战斗有概率掉落饰品 |

### 7.4 推荐精通顺序

| 优先级 | 精通 | 理由 |
|:-----:|:----:|------|
| 1 | 钓鱼精通 | 高级铱金鱼竿大幅提升钓鱼效率 |
| 2 | 农耕精通 | 铱镰刀可收获一切作物, 解放生产力 |
| 3 | 战斗精通 | 饰品大幅提升战斗能力 |
| 4 | 采矿精通 | 重型熔炉批量处理矿石 |
| 5 | 采集精通 | 神秘树种子锦上添花 |
