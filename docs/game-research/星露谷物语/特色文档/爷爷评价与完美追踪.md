## 1. 爷爷评价系统概述

### 1.1 基本机制

爷爷的鬼魂在第**3年春季第1天**出现，对玩家前两年的表现进行评价。

| 项目 | 内容 |
|------|------|
| 触发时间 | 第3年春1 |
| 评价形式 | 鬼魂出现，点燃蜡烛(1-4根) |
| 计分方式 | 21分制 |
| **4根蜡烛门槛** | **≥12分** |
| 满4根奖励 | **完美雕像 (Statue of Perfection)** |
| 未满4根 | 可在之后用钻石重新评估 |

### 1.2 评分等级

| 蜡烛数 | 所需分数 | 奖励 |
|:-----:|:-------:|------|
| 1根 | 0-3分 | 无奖励 |
| 2根 | 4-7分 | 无奖励 |
| 3根 | 8-11分 | 无奖励 |
| **4根** | **≥12分** | **完美雕像** |

> **要点：** 凑满12分即可获得4根蜡烛。21分满分并不需要全部达成。

## 3. 完美雕像奖励

### 3.1 完美雕像 (Statue of Perfection)

| 项目 | 内容 |
|------|------|
| 获取方式 | 爷爷评分获得4根蜡烛后自动获得 |
| 位置 | 放置在农场任何地方 |
| **产出** | **每天早晨产出2-8个铱矿石(Iridium Ore)** |
| 额外产出 | 有概率产出铱锭(低概率) |
| 年产出(估算) | 约730~2,920个铱矿石/年 |

> **价值评估：** 完美雕像是游戏中最稳定的铱矿来源之一，每天的免费铱矿石对于升级铱工具和制作铱锭非常有价值。

## 4. 重新评估机制

### 4.1 重新评分条件

未拿到4根蜡烛，或想重新查看评价：

| 项目 | 内容 |
|------|------|
| 祭品 | **钻石 (Diamond x1)** |
| 地点 | 爷爷的神社(农场上角) |
| 操作 | 手持钻石，右键点击神社 |
| 次数 | 无限次，随时可以重新评估 |
| 结果 | 根据当前进度重新评分 |

> **提示：** 即使第3年春1没有拿到4根蜡烛，之后随时可以带钻石去神社重新评分。完美雕像晚拿也比不拿好。

## 5. 完美追踪系统

### 5.1 基本介绍

完美追踪系统(Perfection Tracker)在 **v1.5版本** 加入游戏，是对爷爷评分的"终极升级版"，要求100%完成所有游戏内容。

| 项目 | 内容 |
|------|------|
| 解锁条件 | 收集**100个金色核桃**，进入姜岛齐先生的核桃房 |
| 追踪器位置 | 核桃房内左侧的完美追踪器雕像 |
| 评分方式 | 点击雕像查看每项完成度百分比 |
| 总完成度 | 各项完成度的平均值 |
| **100%奖励** | **完美雕像 (Statue of True Perfection)** |

### 5.2 完美追踪九大类别

| 编号 | 类别 | 占总分比例 | 最高难度 |

| 2 | 全村民满好感 | **11%** | 高 |
| 3 | 黄金钟 | **10%** | 极高(1,000万g) |
| 4 | 怪物猎杀全目标 | **10%** | 中 |

## 7. 完美追踪实用技巧

### 7.1 完美追踪各部分完成建议顺序

| 优先级 | 类别 | 建议 |
|:-----:|:----|------|
| 1 | 出货15% | 日常累积，注意不要遗漏任何季节性作物 |
| 2 | 全技能10级 | 尽早完成，自然累积 |
| 3 | 金色核桃5% | 姜岛搜索全部核桃 |
| 4 | 全鱼类10% | 分季节完成，注意传说鱼类的季节限制 |
| 5 | 怪物猎杀10% | 在刷矿井/骷髅洞穴时顺便完成 |
| 6 | 全星之果实10% | 逐步完成各项前置 |

| 类别 | 难度评级 | 主要原因 |
