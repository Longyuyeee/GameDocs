## 1. 齐先生概述

| 项目 | 内容 |
|------|------|
| 中文名 | 齐先生 (Mr. Qi) |
| 英文名 | Mr. Qi |
| 身份 | 神秘商人/挑战提供者 |
| 首次出现 | 秘密笔记#10→骷髅洞穴100层 |
| 主要据点 | 姜岛核桃房 |
| 角色定位 | 提供高难度挑战和稀有物品兑换 |

### 1.1 齐先生接触流程

```
第1步：获得放大镜 → 接到冬季收集任务(冬1矿井入口)
第2步：收集秘密纸条 → 找到#10号纸条(骷髅洞穴100层信息)
第3步：骷髅洞穴100层 → 遇到齐先生，获得铱蛇奶(+25HP)
第4步：修复威利的船 → 前往姜岛
第5步：收集100个金色核桃 → 打开核桃房
第6步：进入核桃房 → 接齐先生挑战任务 → 兑换齐钻物品
```

## 2. 金色核桃收集系统

### 2.1 基本介绍

金色核桃(Golden Walnut)是姜岛的核心货币，用于解锁姜岛的各种设施。

| 项目 | 数值 |
|------|:----:|
| 姜岛金色核桃总数 | **130个** |
| 解锁核桃房所需 | 100个 |
| 剩余可兑换 | 30个(继续升级岛屿/兑换齐钻) |

### 2.2 金色核桃主要来源

| 来源类别 | 数量(约) | 获取方式 |
|:--------:|:-------:|---------|
| **姜岛西部** | | |

| 岛屿丛林 | ~20 | 砍树/挖掘/解谜 |
| 岛屿农场 | ~20 | 收获作物/砍树/锄地 |

## 3. 齐先生的核桃房

### 3.1 基本介绍

| 项目 | 内容 |
|------|------|
| 位置 | 姜岛北部上方(需通过丛林) |
| **解锁条件** | **收集100个金色核桃** |
| 设施 | 挑战任务板 + 齐钻商店 + 完美追踪器 |

### 3.2 核桃房内部设施

| 设施 | 功能 |
|:----|------|
| **齐先生挑战任务板** | 左侧墙壁上，每周一刷新一个挑战任务 |
| **齐钻商店** | 右侧售卖机，使用齐钻兑换物品 |
| **完美追踪器** | 房间中间的雕像，查看完美追踪进度 |
| **回城图腾** | 可传送回农场 |

### 4.2 各任务详细攻略

#### 任务1：齐先生的作物 (Qi's Crop)

| 项目 | 内容 |
|:----|------|
| 目标 | 出货500个齐瓜(Qi Fruit) |
| 时限 | 28天 |
| 奖励 | **100齐钻** |
| 难度 | 中等 |

**攻略要点：**
- 齐先生会邮寄**齐瓜种子(Qi Bean)**×1作为起始
- 齐瓜种子种下后约4天成熟，收获时获得1个齐瓜+1-4个齐豆种子
- 需要用收获的齐豆继续种植扩张规模
- 建议使用大量洒水器和生长激素
- 28天足够完成500个目标

#### 任务2：我们来玩游戏 (Let's Play A Game)

| 项目 | 内容 |
|:----|------|
| 目标 | 星露谷小游戏(Junior Kart)无限模式达到50,000分 |
| 时限 | 7天 |
| 奖励 | 20齐钻 |
| 难度 | **极高** |

**攻略要点：**
- 游戏可以在星之果实酒吧的街机上玩
- Junior Kart操作难度很高，对很多玩家来说是**最难任务**
- 无捷径，只能靠练习
- 如果实在无法完成，可以跳过此任务等下周刷新

#### 任务3：四种宝石 (Four Precious Gems)

| 项目 | 内容 |
|:----|------|
| 目标 | 收集4个五彩碎片(Prismatic Shard) |
| 时限 | 28天 |
| 奖励 | 40齐钻 |
| 难度 | 容易 |

**攻略要点：**
- 28天收集4个五彩碎片非常充裕
- 推荐去骷髅洞穴刷铱矿层(大量炸矿)
- 配合窃贼戒指刷怪也有掉落可能
- 万象晶球也有小概率开出

#### 任务4：齐先生的饥饿挑战 (Qi's Hungry Challenge)

| 项目 | 内容 |
|:----|------|
| 目标 | 不进食/不喝饮料到达骷髅洞穴100层 |
| 时限 | 7天 |
| 奖励 | 25齐钻 |
| 难度 | **高** |

**攻略要点：**
- 任务期间不能吃任何食物或饮料(可以喝生命药水/蒜油等药品)
- **关键是提前准备大量楼梯(99石头/个)**
- 推荐使用翡翠复制机大量积攒翡翠，周日去沙漠商人换楼梯
- 带炸弹和楼梯，走到就下楼梯，别探索
- 可以穿高防御装备减少伤害

#### 任务5：齐先生的料理 (Qi's Cuisine)

| 项目 | 内容 |
|:----|------|
| 目标 | 出货价值100,000g的"新鲜"烹饪品 |
| 时限 | 14天 |
| 奖励 | 25齐钻 |
| 难度 | 中等 |

**攻略要点：**
- "新鲜"是指任务期间制作的烹饪品
- 提前准备大量高品质食材
- 推荐制作高价料理：**泰式酸辣汤**(600g+)、**鱼子酱**(500g)、**生鱼片**(220g)
- 配合工匠/农耕职业加成

#### 任务6：齐先生的善意 (Qi's Kindness)

| 项目 | 内容 |
|:----|------|
| 目标 | 一周内送出50个最爱礼物(最爱=+80好感) |
| 时限 | 7天 |
| 奖励 | 40齐钻 |
| 难度 | 高 |

**攻略要点：**
- 需要在一周内对NPC送出50次最爱礼物
- 同一NPC每天限送1次(含生日额外1次)
- **建议：** 使用"星之果实茶"——不占送礼次数，+250好感，一次可以有多个NPC
- 备足NPC喜爱的最爱礼物，如五彩碎片、兔子的脚
- 参与冬日星盛宴或生日可额外多送

#### 任务7：大家族 (Extended Family)

| 项目 | 内容 |
|:----|------|
| 目标 | 钓到5种传说鱼类的"二代"(Extended Family variants) |
| 时限 | **3天** |
| 奖励 | 20齐钻 |
| 难度 | **极高** |

**攻略要点：**
- 5种传奇鱼二代：
  - **传奇二代 (Son of Legend)**：山区湖，雨天，春季(难度最高)
  - **深红鱼二代 (Ms. Angler)**：海滩东码头，任意天气，夏季
  - **垂钓者二代 (Mr. Angler)**：巴士站北，任意天气，秋季
  - **冰河鱼二代 (Glacierfish Jr.)**：森林南部，任意天气，冬季
  - **变异鲤鱼二代 (Mutant Carp)**：下水道突变虫穴，全年
- 时限仅3天，时间非常紧迫
- 使用海泡布丁(+4钓鱼)、渔夫职业、诱饵大师、陷阱浮标等
- 需要极高钓鱼等级

#### 任务8：深处的危险 (Danger In The Deep)

| 项目 | 内容 |
|:----|------|
| 目标 | 到达矿井120层(怪物为增强版"危险"模式) |
| 时限 | 7天 |
| 奖励 | 50齐钻 |
| 难度 | 高 |

**攻略要点：**
- 矿井怪物替换为危险版本(腐臭幽灵、蓝色鱿鱼、暗影狙击手等)
- 需要从第1层重新打到120层(电梯保留)
- 推荐带高等级武器(Galaxy系列)、高防御装备
- 大量炸弹+楼梯可加速
- 或选择不靠楼梯硬打

#### 任务9：骷髅洞穴入侵 (Skull Cavern Invasion)

| 项目 | 内容 |
|:----|------|
| 目标 | 到达骷髅洞穴100层(怪物增强版) |
| 时限 | 7天 |
| 奖励 | 40齐钻 |
| 难度 | **极高** |

**攻略要点：**
- 骷髅洞穴怪物替换为危险版本(皇家飞蛇、危险木乃伊等)
- 皇家飞蛇HP 300-1,050(取决于尾巴长度)，极为难缠
- 必备：大量楼梯、炸弹、高DPS武器(无限系列)
- 配合三倍浓缩咖啡+香辣鳗鱼加速+加运气
- 穿上高防御靴子和铱环

#### 任务10：齐先生的五彩农场 (Qi's Prismatic Grange)

| 项目 | 内容 |
|:----|------|
| 目标 | 收集6种颜色各100件物品(红/橙/黄/绿/蓝/紫) |
| 时限 | 14天 |
| 奖励 | 35齐钻 |
| 难度 | 中等 |

**颜色分类建议：**

| 颜色 | 建议物品 | 获取方式 |
|:---:|---------|---------|
| 红色 | 红蘑菇、草莓、蔓越莓、红甘蓝 | 采集/种植 |
| 橙色 | 南瓜、铜矿石、胡萝卜、橙子 | 种植/采矿 |
| 黄色 | 玉米、金矿石、向日葵、枫糖浆 | 种植/采集 |
| 绿色 | 海藻、绿豆、纤维、蕨菜 | 采集/种植 |
| 蓝色 | 蓝莓、泪晶、沙鱼、蓝调爵士 | 种植/矿井/钓鱼 |
| 紫色 | 紫蘑菇、葡萄、大海参、甜菜 | 采集/种植/钓鱼 |

> 提前准备大量各种颜色的物品，接任务后直接提交即可快速完成。

## 6. 姜岛区域总览

### 6.1 姜岛开放条件

| 条件 | 说明 |
|:----|------|
| 前置 | 完成社区中心全部收集包(或Joja路线全部购买完成) |
| 威利的船 | 触发剧情后，修复威利鱼店后屋的船 |
| 船修复材料 | 硬木×200、电池组×5、铱锭×5 |
| 费用 | 免费(但材料要求高) |
| 船修复后 | 随时可前往姜岛(船在威利鱼店后屋) |

### 6.2 姜岛四大区域

#### 区域1：姜岛海滩 (Ginger Island Beach)

| 特征 | 内容 |
|:----|------|
| 初始到达点 | 船停靠的海滩 |
| 鹦鹉 | 用金色核桃支付解锁功能 |
| 岛民帐篷 | 里奥(Leo)住在这里，直到他搬去大陆 |
| 贝壳/采集 | 海滩采集物、珊瑚、海胆等 |

#### 区域2：姜岛丛林 (Ginger Island Jungle)

| 特征 | 内容 |
|:----|------|
| 位置 | 海滩向北 |
| 金色核桃 | 多处隐藏核桃(砍树/挖土/采集) |
| 宝石鸟谜题 | 东部丛林内的鸟，按顺序触摸获得核桃 |
| 通往 | 核桃房(北部)、姜岛农场(西部) |

#### 区域3：姜岛农场 (Ginger Island Farm)

| 特征 | 内容 |
|:----|------|
| 位置 | 丛林向西 |
| **可耕种土地** | **516+块** (极大种植空间) |
| 气候 | **全年夏季** (可种任何季节作物) |
| 水源 | 池塘取水(需浇水壶)，有喷灌点 |
| 设施 | 初始帐篷→可升级为小屋 |

> **姜岛农场可以种任何季节的作物**，不受姜岛"全年夏季"天气的限制。春季/秋季/冬季种子也能在这里生长。

#### 区域4：火山地牢 (Volcano Dungeon)

| 特征 | 内容 |
|:----|------|
| 位置 | 姜岛北部 |
| 层数 | 10层(随机生成) |
| 第5层 | 矮人商店 |
| 第10层 | **锻造台 (Forge)** |
| 通关后 | 解锁电梯 |
| 快捷通道 | 第10层可打开到入口的快捷通道 |

## 11. 火山地牢

### 11.1 基本信息

| 项目 | 内容 |
|:----|------|
| 位置 | 姜岛北部(经过熔岩河) |
| 层数 | **10层** (随机生成) |
| 进入方式 | 用浇水壶浇灭岩浆→过河→进入火山 |
| 时间流逝 | 与外界同步 |
| 保存点 | 无(死亡回到入口) |
| 通关奖励 | 解锁电梯、第10层到入口的快捷通道 |

### 11.2 层数机制

| 层数 | 内容 |
|:---:|------|
| 1-4层 | 随机生成，有怪物/宝箱/矿石 |
| **第5层** | **固定：矮人商店 + 水池** (可补充水壶) |
| 6-9层 | 随机生成，难度递增 |
| **第10层** | **固定：锻造台 (Forge)** |

### 11.3 第5层矮人商店

需要**矮人翻译指南(Dwarvish Translation Guide)** 才能交易。

| 商品 | 价格 | 说明 |
|:----|:----:|------|
| 灰烬小丑鞋 (Cinderclown Shoes) | 100火山晶石 | 靴子(+5防+5免) |
| 樱桃炸弹×10 | 450g | — |
| 炸弹×10 | 1,600g | — |
| 传送图腾:姜岛配方 | 10,000g | 永久传送姜岛 |
| 姜汁汽水配方 | 1,000g | 饮料配方 |

### 11.4 第10层锻造台 (Forge)

锻造台是火山地牢的核心功能，用于：

| 功能 | 费用 | 说明 |
|:----|:----:|------|
| **武器强化** | 火山晶石+宝石 | 提升武器属性(最多3级) |
| **武器附魔** | 1五彩碎片+20火山晶石 | 获得特殊附魔效果 |
| **银河→无限升级** | 3银河之魂+60火山晶石 | 将银河武器升级为无限系列 |
| **戒指合并** | 20火山晶石 | 将2个戒指合并为1个(释放一个戒指位) |
| **工具附魔** | 1五彩碎片+20火山晶石 | 为工具添加特殊附魔 |
| **移除附魔** | 10火山晶石 | 移除已有附魔 |

**详见 [角色属性战斗数据总览.md] 的武器锻造和附魔章节。**

### 11.5 火山地牢资源

| 资源 | 用途 |
|:----|------|

| 龙牙 (Dragon Tooth) | 制作传送图腾:姜岛、姜岛方尖塔 |

- 击杀熔岩大头(Lava Lurk)
