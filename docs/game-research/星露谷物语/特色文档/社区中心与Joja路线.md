## 1. 社区中心路线总览

### 1.1 基本介绍

社区中心位于鹈鹕镇中心，初始状态为破败。触发剧情后，在进入社区中心时看到祝尼魔(Junimos)动画，阅读金色卷轴后即可开始收集包任务。

| 项目 | 内容 |
|------|------|
| 解锁方式 | 第1年春5日左右，进入鹈鹕镇触发剧情 |
| 触发条件 | 雨天进入社区中心 |
| 完成6个房间 | 获得社区中心完工剧情 |
| 总收集包数 | 6个房间共20+个独立收集包 |
| 房间主奖励 | 每个房间完成给予一个主线奖励 |

### 1.2 六个房间总览

| 房间 | 中文名 | 主线奖励 | 收集包数量 |
|:---:|:------:|---------|:---------:|
| Crafts Room | 工艺室 | **修复桥梁（解锁采石场）** | 6个 |
| Pantry | 茶水间 | **解锁温室 (Greenhouse)** | 6个 |
| Fish Tank | 鱼缸 | **移除闪亮的卵石（解锁淘金）** | 6个 |
| Boiler Room | 锅炉房 | **修复矿车（快速移动）** | 3个 |
| Bulletin Board | 公告板 | **所有非恋爱NPC+2心** | 5个 |
| Vault | 金库 | **修复巴士（解锁沙漠）** | 4个(支付) |

| 电影院建造 | **500,000g** | 社区中心完成后的遗失收集包 |

## 9. 两条路线对比分析

| 对比项目 | 社区中心路线 | Joja路线 |
|:--------:|:-----------:|:--------:|
| **入门方式** | 完成收集包(无需花钱) | 购买Joja会员(5,000g) |
| **解锁采石场** | 工艺室收集包(免费) | 25,000g |
| **解锁温室** | 茶水间收集包(免费) | 35,000g |
| **解锁矿车** | 锅炉房收集包(免费) | 15,000g |
| **解锁巴士** | 42,500g(金库包支付) | 40,000g |
| **解锁淘金** | 鱼缸收集包(免费) | 20,000g |
| **总花费** | ~42,500g(仅金库包) | 135,000g(全部购买) + 5,000g(会员) |
| **额外奖励** | 各收集包的独立奖励物品 | 可在Joja购买**自动抚摸机** |
| **公告板奖励** | **全非恋爱NPC+2心** | 无法获得此奖励 |
| **时间成本** | 需等待季节变化，耗时较长 | 支付即完成，无需等待 |
| **电影院** | 遗失收集包(需5种稀有物品) | 500,000g |
| **商店营业** | 皮埃尔周三休息，克林特周五下午锻炼 | Joja每天营业，但皮埃尔周三关门 |
| **社区中心** | 修复为美丽建筑 | 变为Joja仓库 |

### 9.1 社区中心路线优势

1. **免费完成大部分项目**：除金库包外，其他房间只需收集物品无需支付金币
2. **公告板+2心奖励**：所有非恋爱NPC自动+2心，省去大量送礼时间
3. **收集包附赠奖励**：每个收集包完成都赠送实用道具(种子、工具、配方等)
4. **社区中心修复**：建筑变美，可进入
5. **电影院建造免费**：只需完成遗失收集包(5件稀有品)

### 9.2 Joja路线优势

1. **节省时间**：无需等待季节，第1年内即可解锁所有区域
2. **自动抚摸机可购买**：这是Joja路线最大优势(社区中心路线极难获得)
3. **总体费用更低**：巴士修理比社区中心便宜2,500g
4. **Joja每天营业**：不会因为周三休息而无法购物

### 9.3 推荐选择

| 推荐人群 | 推荐路线 | 理由 |
|---------|:-------:|------|
| 新手/休闲玩家 | **社区中心** | 引导全面体验游戏内容 |
| 完美主义者 | **社区中心** | 公告板+2心对完美追踪有帮助 |
| 速通/效率玩家 | **Joja路线** | 最快速度解锁所有区域 |
| 自动抚摸机需求 | **Joja路线** | 唯一可靠的自动抚摸机获取方式 |
| 收集控 | **社区中心** | 收集包体验更有成就感 |

## 10. 遗失收集包与电影院

### 10.1 解锁条件

完成社区中心全部6个房间后，下一个雨夜闪电击中废弃的Joja超市，内部出现**遗失收集包 (Missing Bundle)**。

### 10.2 遗失收集包内容

| 需要物品(任选5种) | 获取方式 |
|-----------------|---------|
| **银星品质酒 (Silver-quality Wine)** | 水果+小桶后，放入地窖木桶陈酿至少14天达到银星 |
| **恐龙蛋黄酱 (Dinosaur Mayonnaise)** | 恐龙蛋→蛋黄酱机(800g/个) |
| **五彩碎片 (Prismatic Shard)** | 铱矿节点/万象晶球/怪物稀有掉落 |
| **金星上古水果 (Gold-quality Ancient Fruit)** | 种植上古水果(需上古种子) |
| **金星/铱星虚空鲑鱼 (Gold/Iridium Void Salmon)** | 女巫沼泽钓鱼 |
| **鱼子酱 (Caviar)** | 鲟鱼子→罐头瓶 |

### 10.3 电影院 (Movie Theater)

| 项目 | 内容 |
|------|------|
| 位置(社区中心路线) | 原Joja超市位置 |
| 位置(Joja路线) | 原社区中心位置 |
