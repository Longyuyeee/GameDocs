# 种植与制作系统 - 月石岛(Moonstone Island)

> 游戏版本: Moonstone Island (PC/Switch, 2023年)
> 数据来源: moonstoneisland.wiki.gg, TheGamer, QuestRated

## 1. 种植系统

### 1.1 定位
- 种植是**辅助系统**——不是核心经济来源
- 主要用途: 精灵食物/体力恢复/炼金材料

### 1.2 作物

| 作物 | 用途 | 季节 |
|------|------|:--:|
| 基础作物(Applum/Beach Peach等) | 食物/体力 | 多季 |
| 花类(Black Pearl/Bloodroot/Yarrow等) | 送礼(全员喜欢!) | 多季 |
| 辣椒(Hot Pepper) | Quill最爱/料理 | 夏 |
| Grumpkin | Gaiana最爱 | 秋 |
| Prickly植物 | Gaiana最爱 | 多季 |

### 1.3 种植机制

| 要素 | 说明 |
|------|------|
| 树木/岩石/草 | **不重生**(砍了就没了)——但可购买树苗/岩石/草来放置 |
| 体力(Stamina) | 工具消耗→温泉恢复(散布在世界各地) |
| 作物Buff | 喂给精灵→提升精灵属性 |

## 2. 炼金(Alchemy)

| 产物 | 用途 |
|------|------|
| 治疗药水 | 恢复HP |
| Buff药水 | 临时属性提升 |
| 精灵强化药水 | 永久提升精灵属性 |
| 速度药水(Speed Potion) | Ossono最爱 |
| 时间膨胀药水(Time Dilation) | Ossono最爱 |

## 3. 制作(Crafting)

| 类别 | 代表 |
|------|------|
| 工具 | 镐/斧/锄/钓竿 |
| 飞行器 | 气球/扫帚/滑翔翼 |
| 家具 | 200+款(可自定义设计) |
| 精灵相关 | 精灵球/精灵食物/月石手环 |
| 信标(Beacon) | 标记地点/快速旅行 |

## 4. 设计特点

1. **树木不重生=有限资源**: 需要购买树苗补充→与大多数牧场的"无限重生"不同
2. **精灵食物=种植主要用途**: 喂精灵→提升属性/驯服野外精灵
3. **体力=温泉**: 不是"回家睡觉"而是"找附近温泉"——开放世界中的体力管理
4. **200+家具+自定义**: 无预设墙纸/地板——可自己设计或复制粘贴
