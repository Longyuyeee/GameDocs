# 精灵与卡牌战斗系统 - 月石岛(Moonstone Island)

> 游戏版本: Moonstone Island (PC/Switch, 2023年)
> 数据来源: moonstoneisland.wiki.gg, TheGamer, Prima Games, Siliconera

## 1. 概述

月石岛是**精灵收集+卡牌战斗+牧场经营**的三合一融合。战斗不是动作/回合制选菜单——而是**卡牌Deck-Building**。70+精灵各有独立卡组，3只出战精灵的卡组混合为共享战斗Deck。

## 2. 精灵系统(70+种)

### 2.1 八大元素类型

| 元素 | 代表精灵 | 特性 | 出现区域 |
|:--:|------|------|------|
| 大地(Earth) | Ankylo/Brickhouse/Punchypot | 愤怒(Rage)/眩晕(Dizzy) | 全岛 |
| 水(Water) | Fishbo/Octopup/Snorcko | 治疗/探索辅助 | 水岛 |
| 火(Fire) | Flambit/Lucerna/Sinjin | 燃烧(Burn)/高伤 | 火岛 |
| 毒(Poison) | Marsha/Ribbite/Snaplant | DOT毒/转移 | 毒岛 |
| 电(Electric) | Sparky/Capacibee/Breadmaus | 充能(Charge)/AOE | 电岛 |
| 冰(Ice) | Bonguin/Eyesee/Whistler | 冻结(Freeze)/护甲 | 冬岛(仅冬季) |
| 超能(Psychic) | Magicat/Mystislime/Lookachoo | 无形(Intangible)/能量 | 超能风暴(8PM) |
| 暗(Dark) | Catsket/Arkeya/Wolfbane | 吸血/强力 | 外环暗岛 |

### 2.2 精灵获取方式

| 方式 | 说明 |
|------|------|
| 野外驯服 | 战斗→削弱→投喂→捕获 |
| 精灵蛋孵化 | 特定精灵仅可从蛋获取 |
| Boss(不可捕获) | 各元素有1个Boss——仅战斗 |

### 2.3 精灵能力(被动)

每只精灵有独特的**探索/战斗外被动能力**:

| 能力类型 | 代表 |
|------|------|
| 探索辅助 | Womr(走路割草)/Armbot(拾取范围+2)/Toxitoe(传送哨可用2次) |
| 经济加成 | Burgloon(战后金币)/Sandcrashle(岩石掉金币) |
| 社交加成 | Fluffox(+5-10%社交成功率)/Starsee(野生精灵易驯服) |
| 战斗加成 | Pappus(减少风阻)/Carkey(移速提升) |
| XP加成 | Baebaeu(晴天+10%XP)/Whistler(雪天+20%XP) |

## 3. 卡牌战斗系统

### 3.1 机制

| 参数 | 说明 |
|------|------|
| 共享Deck | 3只出战精灵的卡组合并→共享卡组 |
| 能量(Energy) | 出牌消耗→每回合恢复 |
| 卡牌稀有度 | 1-5★——4-5★仅精灵Lv11+出现 |
| 升级选卡 | 精灵升级→3选1新卡加入卡组 |

### 3.2 核心卡牌类型

| 类型 | 代表卡 | 效果 |
|:--:|------|------|
| 攻击 | Tackle/Fireball/Earthquake | 造成伤害 |
| 抽牌 | Perception/Premonition | 补充手牌 |
| 增益 | Charge/Sharp | 提升属性 |
| 减益 | Burn/Freeze/Poison | DOT/控制 |
| 防御 | Armor/Intangible | 减免/闪避 |
| 特殊 | Retreat(100%逃跑)/Refresh(重抽手牌) |

## 4. 设计特点

1. **卡牌Deck-Building=不是传统RPG**: 不是"等级碾压"而是"卡组协同"——3只精灵的卡组需要协同
2. **精灵被动=探索工具**: 不只战斗——走路割草/捡物品/社交加成——精灵选择影响经营效率
3. **70+精灵=收集深度**: 接近宝可梦级别的收集量——在牧场品类中是最大的
4. **八元素=克制系统**: 火→冰→水→电→...元素克制增加策略层
