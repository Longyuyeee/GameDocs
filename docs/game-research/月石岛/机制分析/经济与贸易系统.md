# 经济与贸易系统 - 月石岛(Moonstone Island)

> 游戏版本: Moonstone Island (PC/Switch, 2023年)
> 数据来源: moonstoneisland.wiki.gg, QuestRated

## 1. 货币体系

| 货币 | 获取 | 用途 |
|------|------|------|
| 金币(Coins) | 战斗/精灵棚产出/任务 | 购买种子/树苗/升级/礼物 |
| 月石(Moonstones) | 矿洞/特定区域 | 制作月石手环(求婚!)/特殊物品 |

## 2. 收入来源

| 来源 | 效率 | 说明 |
|------|:--:|------|
| 战斗奖励 | ★★★★★ | **主要收入**——击败野外精灵+地牢Boss |
| 精灵棚产出 | ★★★★ | 精灵在棚中→产出物品→出售(被动收入) |
| 任务奖励 | ★★★ | NPC任务 |
| 作物出售 | ★★ | 辅助(非核心) |
| 钓鱼 | ★★ | 补充 |

## 3. 精灵棚经济

| 精灵 | 产出 | 用途 |
|------|------|------|
| 各类精灵 | 精灵特定掉落物 | 出售→金币/制作材料 |

- 精灵不喂食→会逃跑(警告后)
- 定期喂食→持续产出

## 4. 主要消费出口

| 消费项 | 费用 |
|------|:--:|
| 种子/树苗/岩石 | 基础(树木不重生→需购买补充) |
| 精灵球 | 递进(高级球更贵) |
| 工具升级 | 金币+材料 |
| 气球/扫帚/滑翔翼 | 飞行器制作 |
| 200+家具 | 房屋定制 |
| 月石手环 | 3月石+1铁锭(求婚) |
