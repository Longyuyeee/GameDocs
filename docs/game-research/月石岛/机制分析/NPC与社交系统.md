## 二、社交机制

### 2.1 每日互动

| 方式 | 风险 | 奖励 |
|:--:|:--:|:--:|
| Chat(聊天) | 低 | 低好感 |
| Joke(开玩笑) | 中 | 中好感 |
| **Flirt(调情)** | **高** | **高好感** |
| 送礼 | 无(1次/周) | 最爱+大量好感 |

- 成功率**百分比随机(RNG)**——非对话选择
- Fluffox精灵: +5-10%社交成功率

### 2.2 关系进展

```
每日互动→好感度提升
  → 约会(Date): 海滩/酒馆/温泉——选正确对话
  → 制作月石手环(Moonstone Bracelet): 3月石+1铁锭→月石附魔台
  → 赠送手环→正式交往
  → 婚姻(更新中添加)
```

### 2.3 精灵匹配(Matchmaker)

每个NPC有一个"想要的精灵"——送出对应精灵→特殊奖励:

| NPC | 想要精灵 | 奖励 |
|------|------|------|
| Cleo | Wolfbane | 金币 |
| Ferra | Beardini | 强化熔炉配方 |
| Gaiana | Emoshroom | 新植物 |
| Zed | Armbot | 信标配方 |
| Quill | Sheemp | 每周五免费啤酒 |

## 三、设计特点

1. **RNG社交=高风险高回报**: Flirt成功+大量好感/失败→扣好感——不是"送礼物必然加好感"
2. **11位多样化候补**: 非二元/神经多样性/多语种——品类中最多样化的社交阵容
3. **精灵匹配=社交+收集联动**: 社交不仅靠礼物——精灵也是社交工具
4. **16个独特约会/角色**: 虽短但甜——量不够但质可接受
