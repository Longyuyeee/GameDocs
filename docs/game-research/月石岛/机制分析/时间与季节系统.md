# 时间与季节系统 - 月石岛(Moonstone Island)

> 游戏版本: Moonstone Island (PC/Switch, 2023年)
> 数据来源: moonstoneisland.wiki.gg

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 一天 | 白天+夜晚循环 |
| 每季节 | ~28天 |
| 每年 | 4季(春夏秋冬) |

## 2. 季节

| 季节 | 特色 |
|:--:|------|
| 春 | 基础/春庙解锁(Longlog精灵能力) |
| 夏 | 夏庙解锁(Dropple精灵能力) |
| 秋 | 秋庙解锁(Snaplant精灵能力) |
| 冬 | 冬庙解锁(Sparky精灵能力)/**冰精灵仅在冬季出现!** |

## 3. 昼夜

| 时段 | 特色 |
|:--:|------|
| 白天 | 正常探索/晴天Baebaeu+10%XP |
| 8PM | **超能风暴(Psychic Storm)**→超能精灵出现 |
| 夜晚 | 暗精灵更活跃 |

## 4. 天气

| 天气 | 影响 |
|:--:|------|
| 晴天 | Claudio+20%移速 |
| 雨天/雪天 | 部分精灵能力触发 |
| 超能风暴(8PM) | 超能精灵限定窗口 |

## 5. 无节日系统

- 仅1个非常基础的聚会
- 无传统节日日历

## 6. 设计特点

1. **季节=神庙解锁**: 不是"种什么作物"而是"解锁什么精灵能力"
2. **8PM超能风暴=每日限定窗口**: 必须晚上才能遇到超能精灵
3. **冬=冰精灵限定季**: 仅冬季在冰岛出现——错过等一年
