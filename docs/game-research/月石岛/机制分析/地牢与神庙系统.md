# 地牢与神庙系统 - 月石岛(Moonstone Island)

> 游戏版本: Moonstone Island (PC/Switch, 2023年)
> 数据来源: moonstoneisland.wiki.gg

## 1. 地牢

| 属性 | 详情 |
|------|------|
| 标准地牢 | 30个(分布各生物群系) |
| 每地牢Boss | ✓ |
| 宝箱数 | UI显示各地牢剩余宝箱数 |
| 无限地牢 | 1个——无限层→终局奖励 |

## 2. 四神庙

| 神庙 | 季节 | 解锁精灵能力 |
|:--:|:--:|------|
| 春庙 | 春 | Longlog(体力消耗-10%) |
| 夏庙 | 夏 | Dropple(每回合减1护甲) |
| 秋庙 | 秋 | Snaplant(满HP→毒) |
| 冬庙 | 冬 | Sparky(过载→Exhaust敌卡) |

## 3. 设计特点

1. **30地牢=适中体量**: 不是哈迪斯式的无限重玩——探索完30个=全收集
2. **无限地牢=终局**: 30标准后→无限层→持续奖励
3. **神庙=季节解锁**: 驱动跨越季节→每年至少去每个神庙一次
