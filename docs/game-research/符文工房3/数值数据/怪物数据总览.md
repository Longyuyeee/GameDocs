## 三、怪物刷新与怪物门

| 刷新机制 | 说明 |
|---------|------|
| 怪物门(Monster Gate) | 迷宫各层固定位置,持续刷新怪物直到被破坏 |
| 破坏门后 | 该区域不再自然刷新,需切换楼层后再生 |
| 切换楼层 | 所有门和怪物重置 |
| Boss房间 | Boss不刷新,击败后永久消失(主线) |

## 四、怪物掉落与经营联动

| 掉落类别 | 用途 | 示例 |
