# 道具与升级系统 - 符文工房3 豪华版
![RF3 锻造](https://manuals.marvelousgames.com/rf3sp/09/img/top.jpg)

> 游戏版本: 符文工房3 豪华版(Rune Factory 3 Special) (Switch / Steam, 2023年)

## 1. 工具种类

### 1.1 农具(6种核心工具)

| 工具 | 主要用途 | 次要用途 |
|------|---------|---------|
| **锄头(Hoe)** | 翻耕土地 | 挖掘竹笋,战斗击退 |
| **水壶(Waterpot)** | 给作物浇水 | — |
| **镰刀(Sickle)** | 除草 | 收割作物获得同等级种子 |
| **锤子(Hammer)** | 砸碎石头/采矿 | 将已耕地还原为未耕地 |
| **斧头(Axe)** | 砍树枝 | 砍树桩获得木材 |
| **钓竿(Fishing Pole)** | 钓鱼 | — |

### 1.2 特殊工具(不可升级)

| 工具 | 用途 | 获取 |
|------|------|------|
| **刷子(Brush)** | 与怪物建立好感 | 铁匠铺购买 |
| **羊毛剪(Clippers)** | 从 沃利(Wooly) 怪物剪取毛绒 | 铁匠铺购买 |
| **放大镜(Magnifying Glass)** | 查看土壤品质 | 商店购买 |

## 2. 工具升级体系

### 2.1 升级机制
- 工具在**锻造台(Forge)**升级
- 锻造台解锁条件: 通过 春之森林(Privera Forest) 迷宫 + 到达 尤尼维尔(Univir) 聚落 + 支付 5,000G + 100木材
- 继承机制: 升级时以上一级工具为材料

### 2.2 锄头(Hoe)升级路径(示例)

| 等级 | 工具名 | 材料 | 所需技能等级 |
|------|--------|------|------------|
| Lv1 | 生锈锄头(Rusty Hoe) | 废铁(Scrap Metal) + 铁(Iron) + 铁(Iron) | 5 |
| Lv2 | 坚固锄头(Sturdy Hoe) | 生锈锄头(Rusty Hoe) + 铜(Bronze) + 铜(Bronze) | 15 |
| Lv3 | 老练锄头(Seasoned Hoe) | 坚固锄头(Sturdy Hoe) + 银(Silver) + 银(Silver) | 30 |
| Lv4 | 闪亮锄头(Shiny Hoe) | 老练锄头(Seasoned Hoe) + 金(Gold) + 金(Gold) + 金(Gold) | 60 |
| Lv5 | 祝福锄头(Blessed Hoe) | 闪亮锄头(Shiny Hoe) + 白金(Platinum) ×4 | 80 (光属性) |

### 2.3 升级效果
- **范围扩大**: 高等级锄头/水壶可一次性处理3x1或更大范围
- **效率提升**: 减少RP(Rune Points/符文点数)消耗
- **蓄力功能**: 蓄力后范围更大

## 3. 武器种类与系统

### 3.1 七种武器类型

| 武器类型 | 特点 | 对应技能 |
|---------|------|---------|
| **短剑(Short Sword)** | 单手,攻速快,易连击 | 短剑(Short Sword) |
| **长剑(Long Sword)** | 伤害高,范围大,速度慢 | 长剑(Long Sword) |
| **枪(Spear)** | 极长攻击距离,突刺型 | 枪(Spear) |
| **斧/锤(Axe/Hammer)** | 斧暴击高,锤眩晕高 | 斧与锤(Axe & Hammer) |
| **双剑(Dual Blades)** | 极速连击,放弃盾牌防御 | 双剑(Dual Blade) |
| **法杖(Staff)** | 释放魔法,蓄力扩大范围 | 各元素魔法 |
| **农具(Farm Tools)** | 极低攻击,仅应急 | — |

### 3.2 武器升级
- 在锻造台合成,使用 武器面包(Weapon Bread) 学习配方
- 可附加元素属性(火/水/土/风/光/暗)
- 升级继承: 以低等级武器为材料合成高等级武器

## 5. 装备/饰品系统

### 5.1 防具
- **盾牌(Shield)**: 使用双剑时无法装备盾牌
- **头盔(Helmet)**: 防御头部
- **铠甲(Armor)**: 防御身体
- **鞋子(Boots)**: 移动速度/防御
- **饰品(Accessory)**: 戒指/项链/护符,提供各种加成

### 5.2 饰品制作
- 在**制作台(Crafting Table)**制作
- 使用 饰品面包(Accessory Bread) 学习配方
- 饰品可提供属性加成、元素抗性、特殊效果(如自动恢复)

## 6. 背包与仓库

### 6.1 背包
- 基础容量有限(约20-30格)
- 背包内可携带: 工具、武器、食物、药品、种子、道具等
- 豪华版仓库扩容: 原版 60 格 → **240 格**(v1.03更新)

### 6.2 仓库
- 位于农场小屋(希雅蕾斯树(Sharance Tree)内)
- 存储非携带物品

## 8. 设计特点

1. **工具升级=锻造合成**: 不是交给NPC升级,而是自己收集材料在锻造台制作,将"升级"转化为"制作"
2. **技能数量庞大(20+)**: 从战斗到走路到睡觉都有技能,无事不做无技能,鼓励玩家尝试各种行为
3. **技能升级直接关联属性**: 战斗技能提升STR(力量),魔法技能提升INT(智力),防御提升HP(生命值)——用技能驱动属性成长,而非等级
4. **抗性技能通过"受难"升级**: 被中毒多了就获得毒抗,设计符合直觉
5. **失败无惩罚(锻造/Forging, 制作/Crafting)**: 锻造合成失败不损失材料,鼓励试错和刷技能等级
6. **农具和武器共用锻造台**: 将农耕和战斗放在同一升级维度,绑定二者
