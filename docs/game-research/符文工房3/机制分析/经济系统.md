# 经济系统 - 符文工房3 豪华版
![RF3 希雅蕾斯镇](https://manuals.marvelousgames.com/rf3sp/assets/images/top.jpg)

> 游戏版本: 符文工房3 豪华版(Rune Factory 3 Special) (Switch / Steam, 2023年)
> 原版: 符文工房3 (Rune Factory 3) (NDS, 2009年)

## 1. 货币类型

游戏中只有一种基础货币:**金币(Gold)**。单位符号为 G。

无代币、无特殊点数、无付费货币(豪华版为买断制单机游戏)。

## 2. 收入来源

### 2.1 农作物出售(主要来源)
- 通过农场旁的出货箱(Shipping Bin)出售，**每日下午 5:00 结算**
- 出货后不可撤回
- 高等级作物售价显著增长(Lv 10 作物售价约为 Lv 1 的 1.2-5 倍)

## 3. 物价体系

### 3.1 完整价格链(以春季为例)

```
种子(80-620G) → 种地+浇水 → 收获作物(200-4200G) → 出货
                                            ↕
                                    高品质种子收获
                                            ↓
                              Lv10 作物(240-6300G) → 出货
```

- **种子的成本占售价比例**: 约 8-40%,视作物而定
- **无加工增值链**: 符文工房 3 的烹饪主要用于自用/送礼,非主要盈利手段
- **作物品质升级**: 通过反复"种植→镰刀收种→再种植",提升作物等级

### 3.2 商店定价

- 种子和基础商品价格固定,无市场波动
- **由(Yue,游商)**: 随机出售稀有种子,价格较高
- **跳蚤市场节(Flea Market)**: 夏季19日,以物易物,可买到稀有物品

## 4. 主要消费出口(金钱 Sinks)

| 消费项 | 价格 | 说明 |
|--------|------|------|
| **锻造台(Forge/Furnace)** | 5,000G + 100木材(Lumber) | 解锁武器/工具升级,故事推进必须 |
| **双人床(Double Bed)** | 15,000G + 80木材(Lumber) | 结婚必须 |
| **婚戒(Engagement Ring)材料** | 1 红宝石(Ruby) + 2 银(Silver) | 锻造技能 Lv 20 制作 |
| **怪物小屋(Monster Barn)** | 建设费用(木材+金币) | 驯服怪物放牧 |
| **房屋扩建** | 通过诺伊曼农场(Neumann's Farm) | 解锁更多制作设施 |
| **种子批量采购** | 每季初一次性投入 | 视种植规模而定(数百至数千 G) |
| **配方面包(Recipe Bread)** | 350-450G/个 | 学习新配方,持续消耗 |
| **装备/武器购买** | 不等 | 铁匠铺可直接购买基础装备 |

## 5. 市场波动机制

- **无市场波动**: 所有物品价格固定,不受供求影响
- **无季节差价**: 非当季作物种子不可购买,但作物售价不变

## 6. 季节对经济的影响

| 因素 | 影响 |
|------|------|
| 季节更替(30天) | 当季最后一天未收获的作物会枯萎(跨季作物除外) |
| 天气 | 雨天无需手动浇水,节省体力(RP/符文点数)和时间 |
| 台风/暴风雨 | 可能破坏作物,造成经济损失 |
| 节日 | 部分节日可赢得稀有奖品(配方面包、稀有种子等) |

## 7. 经济设计特点

1. **单货币经济**: 只有金币,设计简洁但缺少分层消费驱动
2. **农业是核心收入**: 战斗/钓鱼/采矿是补充,无法替代农耕
3. **缺乏通胀控制**: 固定价格+有限消费出口,后期金币会大量溢出
4. **无自动化加工经济**: 不像星露谷物语有酿酒/腌制的加价链条
5. **复收作物(草莓/茄子等)效率极高**: 是中期最稳定的收入来源
