# 作物/种植系统 - 符文工房3 豪华版
![RF3 农场](https://manuals.marvelousgames.com/rf3sp/08/img/top.jpg)

> 游戏版本: 符文工房3 豪华版(Rune Factory 3 Special) (Switch / Steam, 2023年)

## 1. 作物种类与分类

游戏共约 40+ 种可种植作物,分为三大类:

### 1.1 食物作物(钻石杂货店出售种子)
四季各 5-6 种,包括蔬菜、水果类

### 1.2 花卉/药草(奇幻花店出售种子)
四季各 4-5 种,包括花卉、药草、水晶花

### 1.3 金作物(由 Yue 游商随机出售)
四季各 1 种,超级稀有,售价极高

## 3. 生长周期模型

### 3.1 基本规则
- 每天必须浇水一次(雨天自动浇水)
- 未浇水的作物当天不生长
- 季节最后一天(30日)结束时,非当季作物全部枯萎
- 无生长阶段可视化(只有"幼苗→成熟"两态)

### 3.2 生长天数范围
- **极短**: 2-3天(菠菜、土豆、粉色芜菁)
- **短**: 4-7天(粉猫花、卷心菜、粉色蜜瓜)
- **中**: 8-14天(洋葱、胡萝卜、南瓜、番茄)
- **长**: 15-20天(灯草、水晶系列、烫烫果)
- **极长**: 22-27天(绒球草、铁叶草、金刚砂花)

### 3.3 复收机制
部分作物收获后不消失,继续生长并定期产出:
- **每天复收**: 番茄(Tomato)、山药(Yam)、青椒(Green Pepper) —— 效率最高!
- **每2天复收**: 草莓(Strawberry)、黄瓜(Cucumber)、玉米(Corn)、茄子(Eggplant)
- **不可复收**: 其余所有作物

复收作物的日平均收益远超单次收获作物,是中后期最经济的种植选择。

## 4. 品质/星级机制

### 4.1 作物等级(Lv 1 ~ Lv 10)
- 所有作物从 Lv 1 开始
- 出售 Lv N 的作物可解锁对应种子在商店销售(部分)
- 使用**镰刀(Sickle)**收割成熟作物可获得**同等级的种子**
- 反复"种→镰刀收种→再种"可逐步提升种子等级

### 4.2 等级对售价的影响
- Lv 10 作物售价通常是 Lv 1 的 **1.2~5 倍**(视作物而定)
- 花卉/药草类等级倍率更高(如玩具草: Lv1=180G, Lv10=900G, 5倍)
- 基础作物等级倍率较低(如粉色芜菁: Lv1=200G, Lv10=240G, 1.2倍)

### 4.3 土壤品质
- 使用**放大镜(Magnifying Glass)**可以查看每格土壤的品质数值
- 土壤品质影响作物生长速度和最终品质
- 可使用肥料改善土壤

## 5. 肥料/加成系统

### 5.1 肥料种类
肥料通过**药房(Pharmacy)**合成,使用 配方面包-药学类(Pharmacy Bread) 学习配方:

| 肥料类型 | 效果 |
|---------|------|
| 生长加速肥 | 缩短作物生长天数 |
| 品质提升肥 | 提高收获作物的等级 |
| 保湿肥 | 减少浇水频率 |
| 免耕肥 | 收获后土壤保持已耕状态 |

### 5.2 使用方式
- 将肥料撒在已耕地上
- 效果持续一定天数
- 可叠加使用多种肥料

## 6. 种子获取途径

| 途径 | 说明 |
|------|------|
| **钻石杂货店(Diamond General Store)** | 基础食物作物种子,每季固定供应 |
| **奇幻花店(Fantastic Flowers)** | 花卉/药草种子 |
| **由(Yue,游商)** | 随机稀有种子(包括金作物系列) |
| **镰刀收种** | 用镰刀收割成熟作物,获得同等级种子 |
| **怪物掉落** | 部分怪物掉落特殊种子 |
| **任务奖励** | NPC 委托完成后偶尔获得 |

## 7. 自动化设施

**无工业化自动化设施**(无洒水器、收割机等)。

自动化依赖:**驯服的怪物**。与怪物建立好感后,它们可以帮忙:
- 浇水
- 收割作物
- 割草
- 播种(需支付金币)

这是符文工房 3 独有的"生物自动化"设计,与星露谷物语的"机械自动化"形成对比。

## 8. 设计特点

1. **复收作物是最优收入策略**: 一次种植,长期收割,减少劳动重复
2. **作物等级系统推动长期投入**: 需要多个种植周期才能达到 Lv 10
3. **镰刀收种机制巧妙**: 用工具使用换种子保留,避免了"优质作物=更好种子"的认知断裂
4. **花卉/药草经济价值高于食物**: 奇幻花店的作物售价普遍高于钻石杂货店作物,但种植时间也更长
5. **金作物系列是终极追求**: 由(Yue)随机出售,种子价格高,但回报极其丰厚
6. **怪物帮工替代机械自动化**: 驱动玩家与怪物系统互动,是"社交+效率"的双重设计
