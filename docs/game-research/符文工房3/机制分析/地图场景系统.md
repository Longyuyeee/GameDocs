## 2. 玩家基地: 希雅蕾斯树(Sharance Tree)

一棵巨大空心古树的内部,是玩家的生活+工作中心:

| 楼层 | 功能区域 |
|------|---------|
| **B1(地下)** | 农田(可扩展) + 出货箱(每日下午5:00结算) + 怪物小屋 |
| **1F** | 公共区域: 各类商店摊位、NPC互动区 |
| **2F** | 玩家私宅: 卧室(存档/睡觉)、厨房、锻造台、制作台、药房 |
| **3F(右)** | 奖杯室(Trophy Room)(成就展示) + 特殊挑战迷宫入口 |
| **左室** | 挑战迷宫(通关后解锁更多层次) |

### 2.1 2F 制作设施布局
- **厨房(Kitchen)**: 烹饪台
- **锻造台(Forge)**: 武器/工具制作升级
- **制作台(Crafting Table)**: 饰品/配件制作
- **药房(Pharmacy)**: 药品/肥料合成

## 3. 希雅蕾斯镇(Sharance Town)

### 3.1 商店一览

| 商店 | 店主 | 出售 |
|------|------|------|
| **钻石杂货店(Diamond General Store)** | Hazel | 种子、基础工具、日用品 |
| **奇幻花店(Fantastic Flowers)** | Shara | 花卉/药草种子、魔法种子(Magic Seeds) |
| **布莱斯餐厅(Blaise's Diner)** | Blaise | 配方面包(Recipe Bread)(各类)、食物 |
| **伊芙琳服饰店(Evelyn's Fashion Shop)** | Evelyn | 家具、装饰品、服饰 |
| **爱德华诊所(Edward Clinic)** | Edward | 药品、状态恢复 |
| **宫古旅馆(Miyako Inn)** | Shino | 住宿、Yue(游商)出现于二楼 |
| **铁匠铺(Blacksmith)** | Gaius | 基础武器、工具、剪刀(Clippers)、刷子(Brush) |

### 3.2 NPC 住宅
- 每个 NPC 有自己的家/房间
- NPC 按日常时间表行动(9:00-18:00 在自家商店,之后回家)

## 4. 尤尼维尔聚落(Univir Settlement)

- 独角兽族(Univir)的聚居地
- 通过春之森林(Privera Forest)后解锁
- 特殊商店和委托(Owl 公告板)
- Kuruna(可攻略候补)居住于此
- 主线剧情核心: 连接两个种族

## 5. 迷宫/战斗区域

### 5.1 四大季节迷宫

| 迷宫 | 季节主题 | 解锁时间 | Boss | 特点 |
|------|---------|---------|------|------|
| **春之森林(Privera Forest)** | 春 | 初始 | 春季Boss | 花草/昆虫类怪物,入门级 |
| **沙漠日暮(Sol Terrano)** | 夏 | 击败春之森林(Privera Forest) Boss | 夏季Boss | 沙漠/火焰主题 |
| **秋之谷(Oddward Valley)** | 秋 | 击败沙漠日暮(Sol Terrano) Boss | 冬季Boss(双胞胎) | 红叶/岩石主题 |
| **雪之遗迹(Snow Ruins)** | 冬 | 击败秋之谷(Oddward Valley) Boss | 冬季Boss | 冰雪/遗迹主题 |

### 5.2 最终迷宫
- **龙之洞穴(Dragon Cave)**: 击败四个季节迷宫 Boss 后解锁
- 最终 Boss: **水龙(Aquaticus)**
- 通关后可进入 New Game+ 内容

### 5.3 迷宫结构特点
- 每个迷宫多层,从入口到 Boss 房间逐步推进
- 每层由多个房间+走廊构成
- **怪物门(Monster Gates)**: 不断刷新怪物,需优先破坏
- 可耕种区域: 迷宫中也有可耕种的土地

## 6. 资源点分布

| 资源类型 | 主要分布区域 |
|---------|------------|
| 矿石: 铁(Iron)/银(Silver)/金(Gold)/铂(Platinum)等 | 各迷宫矿区 |
| 木材(Lumber)/木头(Wood) | 各迷宫树木/树桩 |
| 钓鱼点 | 城镇河流、迷宫水域 |
| 采集物(药草/果实等) | 各迷宫地面、野外 |
| 怪物掉落物 | 全迷宫 |
| 隐藏物品 | 地图隐蔽角落(搜寻(Searching)技能越高越容易发现) |

## 7. 可交互场景元素

| 元素 | 交互方式 | 产出 |
|------|---------|------|
| 石头 | 锤子砸 | 矿石/石材 |
| 树桩 | 斧头砍 | 木材 |
| 杂草 | 镰刀割 | 道具/种子 |
| 怪物门(Monster Gate) | 攻击破坏 | 停止该区域怪物刷新 |
| 宝箱 | 打开 | 道具/武器/金币 |
| 土壤 | 锄头翻耕 | 可种植区域 |
| 水源 | 水壶装水 | 恢复水壶水量 |

## 8. 快速移动/传送系统

- **传送魔法(Teleport)**: 可传送回 希雅蕾斯树(Sharance Tree)(基地)
- **迷宫回程**: 各迷宫有回程捷径(击败 Boss 后解锁)
- **无马/坐骑传送点**: 相比后来的符文工房,地图传送点较少

## 9. 季节/天气对场景的影响

| 因素 | 视觉效果 | 可获取资源变化 |
|------|---------|-------------|
| 春季 | 樱花/绿色 | 春季作物、春季采集物 |
| 夏季 | 阳光/沙滩 | 夏季作物、泳装 NPC |
| 秋季 | 红叶/落叶 | 秋季作物、蘑菇类 |
| 冬季 | 白雪覆盖 | 冬季作物、冰属性采集物 |
| 雨天 | 阴雨 | NPC 集中在室内,图书馆人多 |
| 暴风雨 | 狂风暴雨 | 可能破坏作物 |

## 10. 设计特点

1. **"家"在树中**: 希雅蕾斯树(Sharance Tree) 既是家也是社区中心,将私人空间和公共空间垂直叠合
2. **迷宫季节化**: 四个迷宫对应四个季节,将季节系统融入关卡设计
3. **迷宫内可耕种**: 打破农场与迷宫的物理边界,鼓励探索性种植
4. **独角兽族(Univir)双聚落设计**: 两个种族(人类+独角兽族(Univir))分居两地,驱动主线推进
5. **挑战迷宫独立入口**: 在家即可进入,奖励探索型玩家
6. **无法自由改造地形**: 相比星露谷物语,RF3 可放置装饰但无法大规模改造地貌
