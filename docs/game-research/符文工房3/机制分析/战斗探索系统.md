# 战斗/探索系统 - 符文工房3 豪华版
![RF3 战斗](https://manuals.marvelousgames.com/rf3sp/10/img/top.jpg)

> 游戏版本: 符文工房3 豪华版(Rune Factory 3 Special) (Switch / Steam, 2023年)

## 1. 战斗定位

| 维度 | 定位 |
|------|------|
| **是否为游戏核心系统** | 是,与经营并列,主线推进必须通过战斗 |
| **战斗占游戏比重** | 约 40-50%(新手期更多,后期可偏向经营) |
| **战斗类型** | ARPG(动作角色扮演),即时制 |
| **视角** | 等距俯视角(类似传统2D塞尔达) |

## 2. 武器系统

### 2.1 七种武器类型

| 武器 | 攻击速度 | 攻击范围 | 特殊属性 | 防御能力 |
|------|---------|---------|---------|---------|
| **短剑(Short Sword)** | 快 | 短 | 连击流畅,易上手 | 可配盾 |
| **长剑(Long Sword)** | 慢 | 较长 | 高伤害 | 可配盾 |
| **枪(Spear)** | 中 | 最长(突刺) | 穿透攻击 | 可配盾 |
| **斧(Axe)** | 慢 | 中 | **暴击率高** | 可配盾 |
| **锤(Hammer)** | 慢 | 中 | **眩晕率高** | 可配盾 |
| **双剑(Dual Blades)** | 极快 | 极短 | 极速连击 | **不能配盾** |
| **法杖(Staff)** | 慢(蓄力) | 远(蓄力后) | 魔法属性攻击 | 不能配盾 |

### 2.2 武器制作与升级
- 在锻造台(Forge)制作,使用武器面包(Weapon Bread)学习配方
- 可附加元素属性(火(Fire)/水(Water)/土(Earth)/风(Wind)/光(Light)/暗(Dark))
- 高级武器以低级武器为材料(继承系统)

## 3. 魔法系统

### 3.1 六大元素魔法

| 元素 | 技能 | 升级方式 | 效果 |
|------|------|---------|------|
| **火(Fire)** | Fire Magic | 使用火属性法杖/武器 | 火焰伤害 |
| **水(Water)** | Water Magic | 使用水属性法杖/武器 | 水流伤害 |
| **土(Earth)** | Earth Magic | 使用土属性法杖/武器 | 地震/岩石伤害 |
| **风(Wind)** | Wind Magic | 使用风属性法杖/武器 | 切割/击退 |
| **光(Light)** | Light Magic | 使用光属性法杖/武器 | 神圣伤害 |
| **暗(Dark)** | Dark Magic | 使用暗属性法杖/武器 | 暗影伤害 |

### 3.2 第七魔法

| 魔法 | 效果 | 特别之处 |
|------|------|---------|
| **爱(Love)** | 治疗魔法/播种 | 摩可摩可/毛绒怪(Wooly)形态下减少符文点数(RP/Rune Points)消耗 |

### 3.3 魔法使用
- 通过法杖释放(对应元素),蓄力扩大范围
- 封印(Seal)状态禁止使用魔法和符文能力(逃脱和变身形除外)
- 各元素魔法升级同时提升 符文点数(RP/Rune Points)、智力(INT) 和魔法防御

## 4. 战斗动作系统

### 4.1 基本动作
- **普通攻击**: B键,连按形成连击
- **冲刺攻击**: 奔跑中攻击
- **蓄力攻击**: 长按蓄力后释放
- **终极攻击**: 随武器技能等级解锁

### 4.2 特殊动作
- **回避/闪避**: 移动躲避
- **防御**: 装备盾牌时可用

## 5. 变身战斗(特色)

### 5.1 怪物形态(金色摩可摩可/毛绒怪(Wooly))
主角变身成金色摩可摩可/毛绒怪(Wooly)后的专属战斗操作:

| 操作 | 按键 | 效果 |
|------|------|------|
| 攻击(击退) | B | 击中敌人使其后仰 |
| 抓取 | A | 抓住后仰的敌人 |
| **摔投(Slam)** | A(抓取后) | 跳起将敌人摔向地面,伤害周围 |
| **连打(Pound)** | B(抓取后) | 将敌人反复砸向地面,伤害周围 |
| **旋转投(Swing)** | X/Y(抓取后) | 抓起敌人旋转,移动中伤害周围 |

- 变身下 NPC 对话台词也会变化
- 变身是剧情核心——主角是半人类半摩可摩可混血

## 6. 迷宫/地下城

### 6.1 主线迷宫(5个)

| 迷宫 | 主题 | Boss(头目) |
|------|------|-----------|
| 春之森林(Privera Forest) | 春季/森林 | 春季Boss(头目)(植物/昆虫系) |
| 沙漠日暮(Sol Terrano) | 夏季/沙漠 | 夏季Boss(头目)(火焰/岩石系) |
| 秋之谷(Oddward Valley) | 秋季/山谷 | 双胞胎Boss(头目) |
| 雪之遗迹(Snow Ruins) | 冬季/遗迹 | 冬季Boss(头目)(冰系) |
| 龙之洞穴(Dragon Cave) | 最终迷宫 | 水龙(Aquaticus) |

### 6.2 迷宫结构
- 多层结构,每层多个房间+走廊
- **怪物门(Monster Gates)**: 持续刷新怪物的传送门,必须优先破坏
- 可耕种区域: 迷宫内也有可耕种的土地
- 传送点: 击败 Boss(头目) 后解锁回程捷径

### 6.3 挑战迷宫(奖杯室(Trophy Room))
- 位于希雅蕾斯树(Sharance Tree)家中左侧
- 完成度/推荐等级递增
- 没有剧情绑定,纯挑战向
- 掉落更稀有的材料

## 7. 怪物系统

### 7.1 刷新机制
- **怪物门(Monster Gates)**: 只要门还在,怪物会无限刷新
- 破坏门后该区域不再刷新
- 优先破坏门是探索核心策略

### 7.2 怪物掉落
- 战斗掉落物与农场经营联动:
  - 怪物掉落物可直接出货
  - 部分掉落物是合成材料
  - 部分掉落物是 NPC 最爱的礼物(如 Sakuya 喜欢 Boss(头目) 掉落物)

### 7.3 驯服与伙伴
- 可在战斗中喂食+刷子驯服怪物
- 驯服后作为战斗伙伴(最多带1只)

## 8. 状态异常

| 异常 | 效果 | 解除方式 |
|------|------|---------|
| **中毒(Poison)** | 按百分比持续扣除生命值(HP)(不会致死) | 解毒药/Edward诊所 |
| **麻痹(Paralysis)** | 移动速度大幅降低 | 解麻痹药 |
| **封印(Seal)** | 禁止使用魔法和符文能力(逃脱和变身形除外) | 解封印药 |
| **疲劳(Fatigue)** | 所有行动符文点数(RP/Rune Points)消耗增加 | 睡觉 |
| **感冒(Cold)** | 按百分比持续扣除符文点数(RP/Rune Points) | 感冒药 |
| **睡眠(Sleep)** | 无法行动 | 被攻击后解除/药品 |
| **即死(Death)** | 生命值(HP)瞬间归零 | — |

- **增益(Buff)**: 食用料理获得临时属性加成(正面效果)

## 9. 难度系统(豪华版新增)

| 难度 | 变化 |
|------|------|
| **普通(Normal)** | 标准 |
| **困难(Hard)** | Boss(头目) 生命值(HP) ×1.5 |
| **地狱(Hell)** | Boss(头目) 生命值(HP) ×1.7,敌人攻击伤害 +20% |

- 可在奖杯室(Trophy Room)的书中随时切换难度

## 10. 战斗与经营的联动

| 联动机制 | 说明 |
|---------|------|
| **怪物掉落→出货** | 战斗所得材料可直接出售 |
| **怪物掉落→合成** | 材料用于锻造/制药/饰品制作 |
| **驯服怪物→帮农** | 怪物帮忙浇水/收割/除杂草 |
| **驯服怪物→产出** | 摩可摩可/毛绒怪(Wooly) 剪毛产出 Fur |
| **迷宫矿产→工具升级** | 矿石用于农具和武器升级 |
| **Boss(头目)掉落→社交** | Boss(头目) 掉落物是特定 NPC 最爱礼物 |
| **战斗系统驱动主线** | 必须通关迷宫才能解锁剧情和新城镇 |

## 11. 设计特点

1. **七种武器=七种打法**: 每种武器有独立的技能和手感,不是换皮数值
2. **变身战斗=第二套操作**: 与人类形态完全不同,不是"换皮",而是增加玩法维度
3. **迷宫可耕种**: 将经营玩法嵌入战斗地图,不愿回家的玩家可在野外也种地
4. **怪物门设计**: 优先破坏门 vs 刷怪练级,是探索中的战略选择
5. **战斗产出全面反哺经营**: 没一个掉落物是纯"战利品",全都有经营用途
6. **状态异常种类多(7种)**: 比大多数牧场游戏更RPG,每种需要对应的药品/策略
7. **挑战迷宫延展终局**: 主线通关后的挑战迷宫提供长期游玩动力
