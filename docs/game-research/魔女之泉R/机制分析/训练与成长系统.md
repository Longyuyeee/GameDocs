# 训练与成长系统 - 魔女之泉R(Witch Spring R)

> 游戏版本: Witch Spring R (PC/Switch/PS5, 2023年——魔女之泉1的全面重制)
> 数据来源: witch-spring.fandom.com, GameFAQs, Steam社区指南

## 1. 核心设定: 被人类恐惧的魔女

玩家扮演**Pieberry(派莓)**——一个独自在森林中生活的魔女。人类恐惧并猎杀魔女，Pieberry必须通过训练变得更强才能生存。这是一个**养成×生存×RPG**的融合。

## 2. 时间结构

| 参数 | 设定 |
|------|------|
| 培养周期 | **100天** |
| 每天 | 可进行多次活动(消耗时间) |
| 活动类型 | 训练/探索/战斗/收集/对话 |
| 时间消耗 | 不同活动消耗不同天数 |

## 3. 训练系统(培养核心)

| 训练 | 消耗时间 | 主要属性 | 副属性 |
|:--:|:--:|:--:|:--:|
| 力量训练 | ~3天 | **力量(STR)+** | HP+ |
| 魔法训练 | ~3天 | **魔法(MAG)+** | MP+ |
| 敏捷训练 | ~3天 | **敏捷(AGI)+** | — |
| 防御训练 | ~3天 | **防御(DEF)+** | HP+ |
| 冥想 | ~2天 | HP/MP恢复 | — |
| 休息 | ~1天 | 恢复 | — |

## 4. 核心属性

| 属性 | 影响 | 提升方式 |
|:--:|------|------|
| HP | 生命值 | 力量训练/防御训练/道具 |
| MP | 魔力值 | 魔法训练/冥想/道具 |
| **力量(STR)** | 物理攻击力 | 力量训练 |
| **魔法(MAG)** | 魔法攻击力 | 魔法训练 |
| **敏捷(AGI)** | 速度/暴击率 | 敏捷训练 |
| **防御(DEF)** | 物理减伤 | 防御训练 |

## 5. 训练策略

| 阶段 | 天数 | 策略 |
|:--:|:--:|------|
| 早期(1-30天) | ~30 | 均衡训练→建立基础 |
| 中期(31-60天) | ~30 | 根据Build倾向偏重魔法或物理 |
| 后期(61-100天) | ~40 | 冲刺核心属性→准备终局战斗 |

## 6. 设计特点

1. **训练=时间投资**: 每次训练消耗数天——100天有限→必须规划训练分配→类似火山的女儿的432回合
2. **物理vs魔法Build**: 力量训练vs魔法训练→不同的战斗风格→多周目可尝试不同Build
3. **训练+探索=时间竞争**: 训练消耗天数→探索也消耗天数→必须平衡
4. **R版=原版重制**: R版在原WS1基础上大幅扩展了训练选项+剧情内容
