# 事件与剧情分支 - 魔女之泉R(Witch Spring R)

> 游戏版本: Witch Spring R (PC/Switch/PS5, 2023年)
> 数据来源: witch-spring.fandom.com, GameFAQs, Steam社区

## 1. 事件类型

| 类型 | 触发 | 示例 |
|:--:|------|------|
| 主线事件 | 剧情推进 | 人类猎魔女者首次遭遇 |
| 属性事件 | 属性阈值 | STR>100→可推开某块巨石→新区域 |
| 时间事件 | 特定天数 | Day 50→某角色到访 |
| 选择事件 | 对话选择 | 帮助人类/拒绝→关系变化 |
| 战斗事件 | Boss战 | 区域Boss→击败=解锁新区域/剧情 |
| 隐藏事件 | 特殊条件 | 全收集特定物品→隐藏结局 |

## 2. 关键选择事件

| 事件 | 选项 | 后果 |
|:--:|------|------|
| 首次遇到猎魔女者 | 战斗/逃避/交谈 | 与人类关系的起点 |
| 帮助受伤人类 | 帮助/不帮 | 人类好感+/-/隐藏路线 |
| 是否与人类合作 | 合作/拒绝 | 结局分支 |
| 对同类魔女的态度 | 友好/敌对 | 盟友/孤独路线 |

## 3. 属性→事件门控

| 属性 | 阈值 | 解锁 |
|:--:|:--:|------|
| STR | 100+ | 推开巨石→新区域 |
| MAG | 150+ | 破解魔法屏障→隐藏洞穴 |
| AGI | 80+ | 跨越裂隙→隐藏区域 |
| 综合 | 200+ | 终局Boss区域 |

## 4. 剧情分支树

```
早期(1-30天): Pieberry独自在森林→遇到人类→选择态度
  ↓
中期(31-60天): 与人类关系深化(友好/中立/敌对)
  → 友好路线: 解锁人类村庄/商店
  → 敌对路线: 持续被猎杀/独自训练
  ↓
后期(61-100天): 最终抉择
  → 真结局: 和解共存(特定条件)
  → 普通结局: 保护自己
  → 暗黑结局: 报复人类
```

## 5. 设计特点

1. **属性=剧情门控**: 不是"属性到了好结局"——而是"属性不够无法触发事件"——错过后不可重来
2. **选择=不可逆**: 关键选择影响后续所有剧情——类似LLTQ的"选错=不同路线"
3. **100天=紧迫**: 在100天中错过关键事件→结局受影响——必须平衡训练+探索+事件触发
