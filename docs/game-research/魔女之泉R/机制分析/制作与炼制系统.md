# 制作与炼制系统 - 魔女之泉R(Witch Spring R)

> 游戏版本: Witch Spring R (PC/Switch/PS5, 2023年)
> 数据来源: witch-spring.fandom.com, GameFAQs

## 1. 概述

魔女之泉R的制作系统是**养成×RPG融合的核心纽带**——通过收集材料→制作道具/装备/药水→提升战斗力和训练效率。不是独立的"副系统"——而是连接训练和战斗的桥梁。

## 2. 炼制类型

| 炼制 | 材料 | 产物 | 用途 |
|:--:|------|------|------|
| 药水(Potion) | 草药+蘑菇等 | HP/MP恢复/属性提升 | 战斗恢复+训练加速 |
| 装备(Equipment) | 矿石+皮革等 | 武器/防具/饰品 | 战斗属性加成 |
| 魔法道具 | 魔法材料 | 攻击/辅助道具 | 战斗中消耗 |
| 宠物饲料 | 食材等 | 宠物食品 | 宠物成长 |

## 3. 材料来源

| 来源 | 方式 | 材料类型 |
|:--:|------|:--:|
| 森林采集 | 探索 | 草药/蘑菇/植物 |
| 战斗掉落 | 击败怪物 | 皮革/爪子/魔法材料 |
| 矿山 | 探索 | 矿石/宝石 |
| 商店购买 | 金币 | 稀有材料 |

## 4. 制作→训练效率提升

| 制作物 | 效果 | 训练应用 |
|:--:|------|------|
| 力量药水 | +STR(临时/永久) | 力量训练前使用→效率更高 |
| 魔法药水 | +MAG(临时/永久) | 魔法训练前使用 |
| 敏捷药水 | +AGI(临时/永久) | 敏捷训练前使用 |
| HP/MP恢复 | 减少休息时间 | 更多时间用于训练而非恢复 |

## 5. 设计特点

1. **制作=训练效率的乘数**: 制作好的药水→训练效果×2-3→同样的100天→更高的属性
2. **材料=探索的驱动力**: 不是"探索为了主线"——而是"探索为了找材料→制作更好东西"
3. **炼制=魔女的身份表达**: 魔女制作药水和魔法道具——与"战士买装备"的身份差异
4. **宠物饲料=制作×养成的交叉**: 制作饲料→喂养宠物→宠物成长→战斗更强
