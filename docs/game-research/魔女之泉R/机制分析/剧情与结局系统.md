# 剧情与结局系统 - 魔女之泉R(Witch Spring R)

> 游戏版本: Witch Spring R (PC/Switch/PS5, 2023年)
> 数据来源: witch-spring.fandom.com, GameFAQs, Steam社区

## 1. 世界观

- **魔女(Witch)**——天生拥有魔法力量的存在，被人类恐惧和猎杀
- **人类(Humans)**——普遍视魔女为威胁，组织猎魔女行动
- **Pieberry(派莓)**——年幼的魔女，独自在森林中生活，必须决定自己与人类的关系

## 2. 主线剧情

```
Pieberry在森林中独自生活
  ↓
人类猎魔女者进入森林→冲突
  ↓
Pieberry必须选择: 与人类为敌/寻求共存/独自生存
  ↓
探索世界→遇到更多角色→了解魔女与人类的复杂历史
  ↓
最终抉择→结局分支
```

## 3. 结局分支(多结局)

| 结局 | 条件 | 核心选择 |
|:--:|------|:--:|
| 真结局(True) | 特定条件达成 | 和解/共存 |
| 普通结局 | 基础完成 | 保护自己 |
| 暗黑结局 | 特定选择 | 与人类为敌 |
| 隐藏结局 | 特殊条件 | — |

## 4. 角色系统

| 角色 | 身份 | 与Pieberry关系 |
|:--:|------|:--:|
| Pieberry | 主角魔女 | — |
| 人类角色(多名) | 猎魔女者/朋友 | 取决于选择 |
| 其他魔女 | 同类 | 盟友/导师 |
| 精灵/怪物 | 森林生物 | 中立/敌对/宠物 |

## 5. 设计特点

1. **养成→剧情→养成循环**: 训练→变强→探索新区域→遇到新剧情→需要更强→继续训练
2. **100天=紧迫的叙事**: 不是"无限天随便玩"——100天内必须做出最终选择
3. **选择=养成×叙事的交叉**: 你的属性(战斗能力)影响你能触发哪些剧情→属性不够可能错过关键事件
4. **多结局=多周目驱动**: 不同Build(物理/魔法)→不同结局路线→类似火山的女儿的多周目
