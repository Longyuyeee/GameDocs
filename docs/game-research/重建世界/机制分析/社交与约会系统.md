> 游戏版本: Littlewood (PC/Switch, 2020年)
> 数据来源: littlewood.fandom.com

## 2. 关系阶段

| ♥等级 | 解锁 | 行为 |
|:--:|------|------|
| 1-19 | — | 每日夸赞(Compliment) |
| 20+ | — | 夸赞→有时变**调情(Flirt)** |
| **30+** | **约会(需♥Lv6/8/10)** | 安排→1-2天后→广场见→Port City Deluca旅行 |
| 约会×3 | 3个独特剧情 | 每角色3个渐进约会 |
| 满♥ | **求婚** | 婚戒+选日期(1年内)+选主题 |
| 婚后 | 配偶搬入 | 每日问候+特殊蓝图 |

## 4. 设计特点

1. **约会=旅行**: 不是"看对话框"——而是一起去Port City Deluca——仪式感
2. **婚礼=演出**: 自定义日期+主题→过场动画→字幕滚动→不同CP组合展示
3. **多人同时约会**: 无嫉妒→自由社交
4. **婚后=Game+继续**: 配偶搬入+特殊蓝图——但不是"游戏结束"
