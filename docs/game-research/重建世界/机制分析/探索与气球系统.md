# 探索与气球系统 - 重建世界(Littlewood)

> 游戏版本: Littlewood (PC/Switch, 2020年)
> 数据来源: littlewood.fandom.com

## 1. 气球平台(Air Balloon)

| 等级 | 解锁 |
|:--:|------|
| Lv1 | Endless Forest(无尽森林) |
| Lv2 | Dark Mining Caves(暗矿洞) |
| Lv3-5 | 更多森林区域 |
| Lv6 | **Port City Deluca**(港口城市/约会地) |
| 6星+ | Grand Library(大图书馆/Catacomb Tree) |

## 2. 探索地点

| 地点 | 主要活动 | 稀有资源 |
|------|------|:--:|
| Endless Forest | 伐木/采集/钓鱼 | 魔法树/金树/生命树 |
| Dark Mining Caves | 采矿/捕虫 | 宝石/稀有矿 |
| Port City Deluca | **购物/约会**/见旅行者 | 特殊商品 |
| Grand Library | 捕虫(Catacomb Tree) | **最稀有虫种** |

## 3. 移动

| 方式 | 说明 |
|------|------|
| 气球远行 | 选择目的地→前往探索 |
| 菜单→回家 | **任何地点→瞬间回家**(不限次数!) |
| 与NPC同乘(终极升级) | **每日1次→大幅+心EXP** |

## 4. 设计特点

1. **气球=可升级的探索中枢**: 类似星露谷的巴士——升级=更多目的地
2. **瞬间回家=无惩罚**: 探索不需要"走回去"——随时菜单回城
3. **与NPC同游=社交+探索结合**: 一起坐气球→探索+大幅好感→多重效率
