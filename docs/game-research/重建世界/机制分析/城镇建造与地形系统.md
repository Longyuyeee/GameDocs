# 城镇建造与地形系统 - 重建世界(Littlewood)

> 游戏版本: Littlewood (PC/Switch, 2020年)
> 数据来源: littlewood.fandom.com, Steam Community

## 1. 核心设定: 你已经拯救了世界

Littlewood的独特叙事: **你是已经击败暗黑巫师的英雄**——游戏发生在"大故事结束之后"。你失忆了，需要重建一个叫Littlewood的小镇。没有战斗，没有怪物，没有时间压力。

## 2. 城镇建造

| 要素 | 说明 |
|------|------|
| 蓝图 | 与镇民交好→解锁新建筑蓝图 |
| 建筑类型 | 酒馆/伐木场/钓鱼小屋/魔法图书馆/广场/商店/工坊等 |
| 房屋 | 每个新居民需要一间房屋→**玩家完全自定义内部家具** |
| 搬迁 | **任何东西随时免费移动**(零费用/零等待!) |
| 100%材料回收 | 拆除任何东西→100%退还材料 |

## 3. 地形改造(从游戏一开始就可使用!)

| 功能 | 说明 |
|------|------|
| 升高/降低地形 | 任意室外地块改变高度 |
| 添加/移除水域 | 池塘/河流/水砖任意放置 |
| 土路 | 自由铺设 |
| 楼梯 | 连接不同高度(仅垂直方向) |
| 移动一切 | 树木/作物/花/建筑/房屋/装饰全可重定位 |

## 4. 设计特点

1. **"后英雄时代"叙事**: 不是"去冒险"而是"冒险结束后回家建造"——最独特的牧场游戏前提
2. **无限自由的城镇设计**: 动森式自由搬迁+100%回收——可无限次调整
3. **无战斗纯建造**: 最纯粹的"建造+社交"体验
4. **10种技能(RuneScape式)** : 采集/伐木/采矿/钓鱼/捕虫/农耕/烹饪/制作/经商/卡牌——独立等级最高99
