# 能量与时间系统 - 重建世界(Littlewood)

> 游戏版本: Littlewood (PC/Switch, 2020年)
> 数据来源: littlewood.fandom.com

## 1. 无实时时钟!

| 传统牧场 | Littlewood |
|------|------|
| 游戏内时间持续流逝 | **仅技能动作消耗能量→推进"一天"** |
| 2AM强制睡觉 | **无强制睡觉** |
| 季节/年份限制 | **无季节/无限天** |
| 任务有截止日 | **无任何截止日** |

## 2. 能量系统

| 动作 | 消耗能量 |
|:--:|:--:|
| 伐木/采矿/农耕/烹饪等技能 | ✓消耗 |
| 走路/对话/NPC互动 | **不消耗!** |
| 建造/搬迁/装饰 | **不消耗!** |
| 自定义城镇布局 | **无限时间** |

## 3. 设计特点

1. **零时间压力**: 是牧场品类中最"放松"的设计——没有必须几点睡觉/几天完成任务
2. **建造=无限时间**: 可花无限时间在建造模式中调整城镇→不消耗能量
3. **可随时存档**: 不需要回家睡觉
4. **RuneScape式技能**: 10种技能→每技能独立等级1-99→使用→升级
