# 后英雄时代的治愈叙事 - 重建世界(Littlewood)

> 游戏版本: Littlewood (PC/Switch, 2020年)

## 一、概述

Littlewood的叙事前提是牧场品类中**最独特的**: 你已经击败了暗黑巫师——游戏发生在RPG的"结束后"。你失忆了，不记得自己的英雄事迹，需要在和平的世界中重建一个叫Littlewood的小镇。

## 二、"后冒险"=品类中唯一

| 传统RPG/牧场 | Littlewood |
|------|------|
| "去冒险/种地/赚钱" | **"冒险已经结束——现在过好余生"** |
| 战斗/怪物/地牢 | **零战斗** |
| 时间压力/截止日 | **零压力** |
| 成为英雄 | **曾是英雄——现在成为建造者** |

## 三、叙事设计分析

1. **失忆=重新开始**: 不是"继承农场"而是"忘记自己是英雄"——发现自己的过去是隐藏剧情
2. **重建=治愈**: 不是"赚钱发达"而是"在战后世界中恢复和平"——更深的情感共鸣
3. **无限自由建造=和平的奖赏**: 因为你已经拯救了世界——所以现在你可以随心所欲地建造
4. **约会=在和平中寻找爱**: 不是"冒险中的浪漫"而是"日常生活中的陪伴"
