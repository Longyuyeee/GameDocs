# 时间季节与节日系统 - 中秋之月(Harvest Moon: Back to Nature)

> 游戏版本: Harvest Moon: Back to Nature (PS1, 1999年)
> 数据来源: harvestmoon.fandom.com, GameFAQs

## 1. 时间流速

| 参数 | 设定 |
|------|------|
| 游戏内一天 | 6:00 AM - 5:50 AM(约24小时) |
| 1游戏分钟 | ~1现实秒 |
| 商店营业 | 各店不同(一般9AM-5PM) |
| 强制睡觉 | 5:50 AM(通宵)→次日体力惩罚 |
| 每季节 | 30天 |
| 每年 | 4季×30天=120天 |

## 2. 季节

| 季节 | 天数 | 特色 |
|:--:|:--:|------|
| 春(Spring) | 30天 | 种植季/卷心菜+草莓(利润最高) |
| 夏(Summer) | 30天 | 菠萝(夏季王者)/Kai出现/台风 |
| 秋(Autumn) | 30天 | 红薯(推荐)/蘑菇采集 |
| 冬(Winter) | 30天 | **无种植**→采矿/钓鱼/畜牧/社交 |

## 4. 体力系统

| 要素 | 说明 |
|------|------|
| 体力(Stamina) | 工具使用消耗 |
| 疲劳(Fatigue) | 过度工作→累积→第二天体力减少 |
| 温泉恢复 | 浸泡温泉→恢复体力+减轻疲劳 |
| 力量果实 | 永久+体力上限(隐藏于矿洞/钓鱼/节日) |
| 食物恢复 | 食用作物/料理→恢复体力 |

## 5. 设计特点

1. **30天/季=更多种植窗口**: 比星露谷的28天多2天——更充裕
2. **冬季=经济转型期**: 无种植→采矿为主——确立了"不同季节不同核心活动"的节奏
3. **16个节日**: 几乎每季4个——密集的社交/经济节点
4. **赛马=经济事件**: 赌马可赚大量金币——"节日=经济机会"而不仅是社交
5. **体力+疲劳双维度**: 不是单一体力条——疲劳累积是隐性的"过度工作惩罚"
