# 剧情与家庭系统 - 中秋之月(Harvest Moon: Back to Nature)

> 游戏版本: Harvest Moon: Back to Nature (PS1, 1999年)
> 数据来源: harvestmoon.fandom.com, GameFAQs

## 1. 世界观设定

- 矿物镇(Mineral Town)——一个靠海的小农村
- 主角小时候在祖父的农场度过夏天
- 祖父去世后→农场荒废→主角继承→回到矿物镇

## 2. 主线剧情(简约)

```
序幕: 收到祖父的信→返回矿物镇
  → 农场已荒废→镇长和村民帮助
  → 开始在祖父的农场生活
核心循环: 种植/畜牧/采矿/社交
  → 第1年: 建立农场基础
  → 第2-3年: 结婚/家庭/农场繁荣
结局: 无强制结局→无限经营
  → 第3年未结婚=情敌抢先="隐性失败"
```

## 3. 婚姻→家庭系统

| 阶段 | 条件 | 时间线 |
|------|------|:--:|
| 恋爱 | 心色事件(黑→紫→蓝→黄) | 第1年 |
| 求婚 | 红心+蓝色羽毛(1,000G)+房屋2级 | 第1-2年 |
| 婚礼 | 求婚1周后 | — |
| 怀孕 | 结婚后随机(约1季后) | 婚后1季+ |
| 出生 | 怀孕2季→婴儿 | 婚后~9个月 |
| 幼儿期 | 婴儿成长→能爬行 | 出生后1季 |
| 儿童期 | 能走路说话 | 出生后1年+ |

## 4. 家庭互动

| 阶段 | 互动内容 |
|------|------|
| 配偶 | 做饭/对话/节日互动 |
| 婴儿 | 喂奶/哄睡 |
| 幼儿 | 对话/玩耍/节日互动 |
| 儿童 | 对话/送礼/日常互动 |

## 5. 剧情与经营的关联

| 经营→剧情 | 剧情→经营 |
|------|------|
| 房屋扩建→可求婚 | 结婚→配偶偶尔帮农场 |
| 满足结婚条件→触发心色事件 | — |
| 第3年未结婚→情敌抢先→候补消失 | — |

## 6. 设计特点(模板奠基)

1. **祖父母农场=继承叙事**: "回到祖父的农场开始新生活"——成为后续几乎所有牧场游戏的开场模板
2. **结婚→生育→子女成长**: 最早实现完整婚姻→生育→育儿的生命周期
3. **第3年情敌deadline=隐性压力**: 不是"游戏结束"而是"永久失去候补"——制造婚姻的紧迫感
4. **无主线结局**: 无限年经营——后续游戏在此基础上增加主线目标
5. **子女系统=极早期**: 1999年实现→婴儿→幼儿→儿童的三阶段成长——比星露谷的"停留在婴儿"更完整
