## 4. 设计特点(模板奠基)

1. **基础作物+特殊作物**: 超市日常种子+Won限定种子——双种子来源设计的鼻祖
2. **复收作物机制**: 黄瓜/草莓/番茄/玉米/菠萝——复收概念由BTN确立
3. **动物健康=死亡机制**: 不喂食→生病→死亡——后续牧场游戏大多移除了死亡(BTN是少数保留的)
4. **加工机器=终局投资**: 20,000G+金刚矿——每台都是重大投资，决策重量大
5. **马=免费但需维护**: 春Y1免费获得→冬天前需维持8♥→否则被带走——早期的"时间压力"设计
