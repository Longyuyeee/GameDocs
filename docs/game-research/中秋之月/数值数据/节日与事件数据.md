## 二、节日分类

| 类型 | 节日 | 作用 |
|:--:|------|------|
| **经济型** | 赛马(春18) | 赌马→大量金币——最重要的赚钱节日 |
| 竞赛型 | 料理/鸡/牛/羊/丰收/赛狗 | 展示养殖/种植成果 |
| 社交/浪漫型 | 烟花/月见/星夜 | 与候补互动(好感度加成) |
| 送礼型 | 春感恩/冬感恩 | 送/收礼物(好感度) |

## 三、赛马节(春18)详解

| 要素 | 说明 |
|------|------|
| 机制 | 购买马票→赌马→胜率随机+马好感度影响 |
| 收益 | 最高可达数万G(早期巨大资金来源) |
| 策略 | 用SL大法→刷高回报马票 |
| 马好感度 | 马好感度越高→比赛表现越好 |
| 饰品兑换 | 奖牌可兑换特殊饰品(高价值出售!) |
