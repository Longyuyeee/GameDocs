## 三、动物好感度影响

| 好感度 | 产出品质 |
|:----:|------|
| 0-3♥ | S(小)品质 |
| 4-7♥ | M(中)品质 |
| 8-9♥ | L(大)品质 |
| 10♥ | G(金)品质 → 加工品更高价值 |
