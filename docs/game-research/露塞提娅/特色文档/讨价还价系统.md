# 讨价还价系统 - 露塞提娅:道具屋经营妙方(Recettear)

> 游戏版本: Recettear: An Item Shop's Tale v1.0+ (PC, 2007/2010年)

## 一、概述

露塞提娅是**商店RPG鼻祖**——它首创了"经营商店+地牢冒险"的跨界融合。而它的讨价还价(Haggling)系统是整个商店经营品类中最复杂的谈判模拟——每个顾客有独立的价格因子、还价偏移、预算、和目标价位(Pin Price)。

## 二、谈判流程

```
你出价 → 顾客判断:
  ├── 在最佳价范围内 → 直接接受(无还价)→ "Just Combo"奖励
  ├── 在价格上限内但超出最佳价 → 顾客还价
  │   └── 你: 接受/拒绝→再次出价 → 重复(最多2轮还价)
  └── 超出价格上限 → 顾客拒绝还价→直接离开
```

## 三、16种顾客=16个不同的小游戏

| 维度 | 范围 | 最极端值 |
|------|:--:|------|
| 售价因子 | 105-150% | Alouette=150% |
| 还价偏移 | 0-20% | Alouette=+20%/+20% |
| 预算上限 | 20,000-4,000,000 | Griff=4,000,000 |
| 收购价因子 | 15-60% | Alouette=15%(收购)/Prime=60% |
| 随机性 | ±1-8% | 老人=±8%(最高) |

## 四、"Tear的建议是陷阱"设计

教程中Tear建议: "定价130%基准价+积极还价"

**实际最优策略**:
- 定价104-110%(接近顾客最佳价)
- 尽快达成"Just Pin/Near Pin"→加速好感度+预算成长
- 避免还价——还价虽成功但减缓好感度增长

> Tear是贷款公司的妖精——她的建议偏向于"最大化单次利润"而非"长期经营健康"。这个设计巧妙地将叙事与机制统一。

## 五、目标价(Pin Price)系统

目标价是顾客的"隐藏最高愿付价"——命中有额外经验:

| 命中 | 条件 | 经验奖励 | 好感度影响 |
|:--:|:--:|:--:|:--:|
| Just Pin | ≤0.5%偏差 | +30 XP | 大幅增加 |
| Near Pin | ≤5%偏差 | +15 XP | 中等增加 |
| 未命中 | >5%偏差 | 0 | 正常/减少 |

> Pin Price系统将"定价准确度"从"赚钱效率"提升为"角色成长速度"——精准定价=更快升级=更多功能。

## 六、还价的策略深度

### Alouette(最佳还价对象)
- 还价偏移+20%/+20%→每轮还价她的接受价**上升20%**
- 策略: 首次出价较低→她拒绝→你还价→她接受!→她的心理价位被推高了

### Caillou/Nagi/Tielle/Griff(不可还价!)
- 还价偏移0%/0%→**从不改价**
- 策略: 首次出价必须在最佳区间内→否则永远不接受

### Woman/母亲(第二最佳还价对象)
- 还价偏移+10%/+10%
- 预算150,000→高价值长期客户

## 七、设计分析

1. **谈判=角色性格**: 每个顾客的价格因子/还价偏移反映性格——Alouette=任性但大方(150%/+20%)，Prime=理性(105%/0%)——不是随机数而是角色刻画
2. **隐性惩罚>>显性惩罚**: 还价成功但减缓好感→很多玩家不知道为什么"生意看着不错但顾客预算不涨"
3. **Tear的陷阱=叙事机制统一**: 罕见的设计——教程角色的建议是游戏内的"贷款公司政策"而非设计者建议
4. **Pin Price=准确度奖励**: 不是"价格越高越好"而是"越准越好"——鼓励玩家了解每个顾客而非"一刀切130%"
5. **Just Combo系统**: 首次出价直接成交→经验翻倍——惩罚反复还价
