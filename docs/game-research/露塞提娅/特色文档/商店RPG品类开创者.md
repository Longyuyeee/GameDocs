# 商店RPG品类开创者 - 露塞提娅:道具屋经营妙方(Recettear)

> 游戏版本: Recettear: An Item Shop's Tale v1.0+ (PC, 2007/2010年)

## 一、历史地位

露塞提娅是**商店经营×地牢冒险**这一品类的**开创者**。2007年在日本Comiket 73以同人游戏形式发布，2010年由Carpe Fulgur英文本地化后在Steam上引发轰动。它证明了"经营商店+RPG冒险"的跨界融合不仅可以成立，而且极具魅力。

## 二、品类影响树

```
Recettear (2007, 同人游戏/2010 Steam)
  ↓ 启发
Moonlighter (2018) — 商店经营+Roguelite地牢
  ↓
Potionomics (2022) — 药水店+卡牌议价
  ↓
Dave the Diver (2023) — 寿司店+海底探索
  ↓
Cult of the Lamb (2022) — 教派经营+Roguelite地牢
```

可以说，**所有后续的商店经营+战斗冒险混合游戏都直接或间接受到Recettear的影响。**

## 三、Recettear首创的核心机制

| 机制 | 说明 | 后继者 |
|------|------|------|
| **讨价还价(Haggling)** | 16种顾客×独立价格因子×还价偏移 | Moonlighter(简化版)/Potionomics(卡牌版) |
| **Pin Price系统** | 隐藏目标价位→命中=额外经验+好感 | 无直接后继(遗失了) |
| **顾客好感度=预算成长** | 好感度×100-400倍预算提升 | Moonlighter(简化) |
| **市场状态浮动** | 4状态独立浮动×每物品 | Potionomics(扩展) |
| **间接战斗(雇佣冒险者)** | 8个冒险者AI控制 | Dave the Diver(员工) |
| **债务驱动主线** | 30天62万还款 | 无直接后继(太硬核) |
| **Tear的陷阱建议** | 教程角色的建议实际上是游戏内的坏建议 | 无后继(独特的叙事机制统一) |

## 四、Recettear vs Moonlighter对比

| 维度 | Recettear(2007) | Moonlighter(2018) |
|------|:-----:|:-----:|
| 定价方式 | **主动出价→顾客还价** | 设定价格→观察反应 |
| 顾客多样性 | 16种×独立价格因子 | 3种(普通/英雄/富人) |
| 市场动态 | 4状态独立浮动 | 供需(隐藏) |
| 时间压力 | **30天硬截止** | 无限 |
| 战斗 | 间接(雇佣AI冒险者) | 直接(玩家操作) |
| 失败后果 | Bad End(保留进度) | 死亡=丢部分背包 |
| 好感度影响 | 预算×100-400倍 | 无好感度系统 |
| 合成系统 | 5级×22种装备×+15品质 | 装备4级升级 |
| 商店装饰 | 氛围影响顾客类型 | 纯数值加成 |

## 五、Recettear中被遗忘的宝藏设计

1. **Tear的陷阱**: 教程角色给出"坏建议"——叙事(她是贷款公司的)与机制(她的建议不利于长期经营)完美统一——后无来者
2. **Pin Price的精确度奖励**: 不是"价格越高越好"而是"越准越好"——鼓励了解每个顾客——Moonlighter完全丢掉了这个
3. **Alouette的双重性**: 最佳客户(150%支付因子)+最差卖家(15%收购因子)+竞争对手(叙事)——一角色三功能
4. **Euria的诈骗**: 专门设了一个"以500%价格诈骗新手"的NPC——对市场信息的次级教育

## 六、设计启示

1. **同人游戏能改变一个品类**: Recettear的预算(Comiket同人)远低于Moonlighter的预算(商业发行)，但设计深度远超后者
2. **"硬核"≠"不成功"**: 30天硬截止债务——大多数商业游戏不敢做的设计——反而成为Recettear记忆点
3. **叙事机制统一最难也最珍贵**: Tear的建议是陷阱——角色身份、教程功能、机制引导三合一——这种设计在商业游戏中几乎不可能通过委员会审批
4. **复杂度未被继承**: 16种顾客×独立价格因子×还价偏移×市场浮动——Moonlighter简化成3种顾客×固定定价——品类"后继者"反而退化了
