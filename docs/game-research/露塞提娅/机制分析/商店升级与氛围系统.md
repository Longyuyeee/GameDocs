## 3. 店铺氛围(Atmosphere)

| 氛围 | 吸引顾客 | 调整方式 |
|:--:|------|------|
| 明亮(Light) | Elan/普通顾客 | 装饰选择 |
| 黑暗(Dark) | **Griff(解锁必需!)**/Caillou | 装饰选择 |
| 朴素(Plain) | Louie/Elan | 装饰选择 |
| 华丽(Gaudy) | Charme | 装饰选择 |

- 通过更换壁纸/地板/地毯/柜台样式调整氛围
- **Griff解锁**: 店铺氛围持续黑暗→触发Griff夜间到访事件→解锁黑曜塔50层

## 5. 自动售货机(Vending Machines)

| 属性 | 说明 |
|------|------|
| 售价 | **固定100%基准价**(无溢价) |
| 用途 | 处理低价滞销品/低价市场时买入的物品 |
| 限制 | 初始1台→Lv22无限 |
| 关键 | **绝不放高价稀有物品!**(只能卖基准价) |

## 6. 设计特点

1. **50级商人等级+29个阶段解锁**: 比Moonlighter的5级商店升级更细致——将持续成长感延长
2. **氛围=角色解锁门控**: Griff需要黑暗氛围才能触发——氛围不仅是装饰而是游戏进度要素
3. **自动售货机=策略工具而非利润工具**: 100%基准价→用于处理滞销品→释放柜台空间给高价品
4. **每次扩张=更大经营空间**: 3次店面扩张(4→6→10柜台)匹配债务增长的节奏
