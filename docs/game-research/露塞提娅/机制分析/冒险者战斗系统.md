# 冒险者战斗系统 - 露塞提娅:道具屋经营妙方(Recettear)

> 游戏版本: Recettear: An Item Shop's Tale v1.0+ (PC, 2007/2010年)
> 数据来源: recettear.fandom.com, GameFAQs, Steam Community

## 1. 概述

露塞提娅的战斗是**间接控制**——玩家雇佣冒险者进入地牢，冒险者为AI操控。玩家仅可进行少量辅助操作(药水/切换装备)。8位冒险者各有独立的战斗风格、成长曲线和地牢适应度。

## 3. 冒险者属性与装备

### 3.1 装备槽位

每个冒险者有独立的:
- 武器(Weapon)——决定攻击力和攻击类型
- 防具(Armor)——决定防御力
- 饰品(Accessory)——额外属性/效果

### 3.2 装备来源

| 来源 | 方式 |
|------|------|
| 商店道具 | Recettear出售→冒险者可直接装备 |
| 合成(Fusion) | 行会合成→最高品质装备(可达+15) |
| 地牢宝箱 | 特定地牢掉落特定类型装备 |
| Boss掉落 | 限定装备(仅从特定Boss获取) |

## 4. 冒险者雇佣

| 要素 | 机制 |
|------|------|
| 雇佣费 | 固定Pix/次(随冒险者而异) |
| 半价(Lv21) | 商人等级21→雇佣费减半 |
| 背包归属 | 地牢中所有物品归Recette(店主)所有 |
| 死亡 | 失去地牢中部分物品(保装备)/Lv16=保留2件/Lv50=保留3件 |

## 5. 地牢战斗机制

| 机制 | 说明 |
|------|------|
| 楼层推进 | 每层清除敌人→找楼梯→下楼 |
| Boss层 | 每N层有Boss(免疫常规攻击的部分机制) |
| 宝箱 | 普通宝箱+上锁宝箱(Charme可开) |
| 陷阱 | 地牢中有各类陷阱(伤害/状态异常) |
| 返回 | 可随时使用返回道具→保留所有已获物品 |
| 全灭 | 冒险者HP归零→失去部分物品→强制返回 |

## 6. 冒险者适配地牢推荐

| 地牢 | 推荐冒险者 | 原因 |
|------|------|------|
| 试炼大厅 | Louie | 新手教学 |
| 翡翠之路 | Charme | 可开锁箱→额外宝箱→花瓶(21,000g) |
| 琥珀花园 | Caillou/Tielle | 远程清怪→避免近战毒/藤蔓 |
| 黑曜塔(50层) | Griff | 最强近战→生存+输出兼需 |
| 拉碧斯遗迹(100层) | Arma | 全能型→应对所有敌人类型 |
| 水晶噩梦(Boss Rush) | Griff/Arma | 终极输出+生存 |

## 7. 设计分析

1. **间接战斗=经营游戏的身份保持**: 你是"店主"不是"英雄"——战斗委托冒险者→保持经营主题
2. **冒险者差异化=收集动力**: 8个角色×不同解锁条件→驱动探索全部内容
3. **Charme开锁=功能性差异化**: 不是纯粹战斗差异→额外宝箱→额外收入→经济与战斗的联动
4. **Arma=全解锁奖励**: 需要全部角色+还清债务+拉碧斯遗迹——终极收集成就的顶点
5. **Griff=氛围门控**: 黑暗氛围→触发Griff事件→解锁最强近战——氛围系统不仅是装饰
