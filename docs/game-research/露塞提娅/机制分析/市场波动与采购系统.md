# 市场波动与采购系统 - 露塞提娅:道具屋经营妙方(Recettear)

> 游戏版本: Recettear: An Item Shop's Tale v1.0+ (PC, 2007/2010年)
> 数据来源: recettear.fandom.com, GameFAQs, Steam Community

## 1. 市场价格浮动机制

Recettear的市场不是固定价格——每件物品的**市场价格状态**独立浮动。

| 状态 | 出售倍率 | 收购倍率 | 含义 |
|:--:|:-----:|:-----:|------|
| 正常(Normal) | 100% | 100% | 基准 |
| **高价/红色(High Price)** | **200-250%** | 140% | 需求旺盛——**出售窗口!** |
| 低价/蓝色(Low Price) | 45-55% | 35% | 供过于求——**采购窗口!** |
| 崩盘(Crash) | 35-45% | — | 持续下跌——最大采购 |

## 2. 采购(Procurement)策略

### 2.1 四种采购来源

| 来源 | 价格 | 特点 |
|------|:--:|------|
| 市场(Market) | 随市场状态波动 | 最大物品选择/受价格浮动影响 |
| 顾客出售(Customer Selling) | 取决于顾客类型 | 商人Lv2解锁/Louie/Prime=好卖家 |
| 行会(Guild) | 固定100%基准价 | 安全网/无利润 |
| 地牢宝箱 | 免费(时间+风险成本) | **最佳利润来源** |

### 2.2 "买低卖高"策略

```
市场状态: 低价/蓝色(45-55%基准)
  → 大量买入囤积该物品
  → 等到市场转为高价/红色(200-250%基准)
  → 以2-2.5倍基准价出售
  → 利润: 100-400%投资回报
```

### 2.3 不要在市场高价时采购

- 高价=200-250%基准价→进货成本2-2.5倍
- 此时应**大量出售**而非买入

## 3. 顾客收购(Customer Selling)策略

| 顾客 | 收购价因子 | 策略 |
|------|:-----:|------|
| **Prime** | 60% | **最佳卖家**——收购价高达基准的60% |
| Girl/Louie | 45-50% | 好卖家 |
| Man/行会会长 | 35% | 中等 |
| Caillou/Griff等 | 30% | 较差(尽量不从他们收购) |
| **Alouette** | **15%** | 极差——**绝对不向她买东西** |
| Euria | 200%(向你卖价) | 诈骗者——**绝对不买** |

## 4. 地牢采购=零成本库存

| 地牢 | 推荐物品 | 基准价范围 |
|------|------|:-----:|
| 翡翠之路 | 花瓶(Vase) | **21,000+基准价**——最佳早期暴富来源 |
| 琥珀花园 | 中级武器/防具 | 数千基准价 |
| 黑曜塔 | 高级装备 | 数万基准价 |
| 拉碧斯遗迹 | 终局Boss掉落 | **50,000+基准价** |

## 5. 库存轮换策略

| 物品类型 | 持有策略 |
|------|------|
| 高价值/稀有品 | 等待市场高价(红)→出售给Alouette(150%因子)→**最高410%收益** |
| 中等价位 | 正常价出售→快速周转→积累商人经验 |
| 低价滞销品 | 自动售货机清库存(100%基准价) |
| 合成材料 | 保留(不出售)→合成更高级物品→大幅增值 |
| 地牢消耗品(药水等) | 保留→冒险者在途 |

## 6. 特殊物品的定价策略

| 策略 | 实现方式 |
|------|------|
| 高级预购订单 | 市场高价(红)→slot1放高价物品→**整个订单包溢价** |
| 低价物品伪装 | slot1放低价/崩盘物品→抵消其他物品的高价负面影响 |

## 7. 设计分析

1. **价格浮动=耐心回报**: 不急售→等市场高价→2.5倍利润——奖励长期持有
2. **四种采购来源各有策略深度**: 市场(浮动)/顾客收购(因人而异)/行会(安全网)/地牢(零成本)——丰富的供应链管理
3. **Prime=最佳卖家叙事**: 她是机器人→"理性"→慷慨收购价——机制与角色性格一致
4. **Alouette=双重身份**: 最佳买家(150%出售因子)+最差卖家(15%收购因子)——名副其实的"大小姐"
5. **Euria=新手陷阱**: 以500%价出售物品→看似有稀有品→实际上是诈骗——"如果看起来好得难以置信，那确实是假的"
