## 3. 物品价格体系

### 3.1 市场价格浮动

| 状态 | 售价倍率 | 说明 |
|:--:|:-----:|------|
| 正常(Normal) | 100% | 基准 |
| **高价/红色(High Price)** | **200-250%** | 每次交易随机重设 |
| 低价/蓝色(Low Price) | 45-55% | 每次交易随机重设 |
| 崩盘(Price Crash) | 35-45% | 低价持续下跌 |

### 3.2 核心概念

| 概念 | 说明 |
|------|------|
| 市场价(Market Price) | 基准价×当前市场状态乘数 |
| 目标价(Pin Price) | 顾客"期望"的隐藏价格——命中有额外经验 |
| 价格上限(Price Limit) | 超过此价顾客拒绝还价直接离开 |
| 最佳价(Best Price) | 首次出价在此范围内→顾客直接接受无还价 |

## 5. 还价(Haggling)机制

- 首次出价超出最佳价→顾客还价(价格偏移)
- **Caillou/Tielle/Elan/Nagi/Griff/Arma**: 偏移0%——从不改价！还价=必须降到原最佳区间
- **Alouette**: 偏移+20%/+20%——每次还价她的接受价位大幅上升(最佳还价对象!)
- **Woman**: 偏移+10%/+10%——第二个最佳还价对象

## 6. 顾客购买(你从顾客处收购)

| 顾客 | 购买价因子 | 说明 |
|------|:-----:|------|
| Prime | 60% | 最慷慨——59-65%收购价 |
| Girl | 50% | 友好——51-55% |
| Louie/Elan/Tielle/Woman | 45% | 标准——59-70%出价(他们接受高价) |
| **Alouette** | **15%** | 极低收购价——29-35% |
| **Euria(诈骗者)** | **200%** | 以500%价出售给你——绝对不要从她买! |

## 7. 设计特点

1. **讨价还价=核心玩法**: 不是Moonlighter的"设定价格观反应"，而是"主动出价→顾客还价→你调整"的回合制谈判
2. **目标价(Pin Price)系统**: 隐藏的最优价位——命中=near pin/just pin=额外经验+好感度加速
3. **每个顾客=不同的小游戏**: 16种顾客各有独立的价格因子/还价偏移/预算——需要记住每个人的"性格"
4. **Tear的建议是陷阱**: 教程中Tear建议130%出售+积极还价——实际这会损害好感度+降低预算增长——叙事与机制统一(Tear是贷款公司的)
5. **债务=生存压力**: 不是"慢慢赚"而是"30天内必须凑够62万"——时间紧迫感驱动效率优化
6. **自动售货机=基准价出售**: 不能溢价——策略工具而非利润工具
