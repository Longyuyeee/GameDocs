# 地牢冒险系统 - 露塞提娅:道具屋经营妙方(Recettear)

> 游戏版本: Recettear: An Item Shop's Tale v1.0+ (PC, 2007/2010年)
> 数据来源: recettear.fandom.com, GameFAQs, NeoSeeker

## 1. 概述

露塞提娅的冒险者系统是**间接战斗**——你不是自己打，而是**雇佣冒险者**进入地牢。冒险者是AI控制的，你只需跟随+偶尔辅助。每个冒险者有独立等级/装备。

## 4. 冒险者雇佣

| 要素 | 说明 |
|------|------|
| 雇佣费 | 每个冒险者有固定雇佣费(Pix) |
| 半价(Lv21+) | 商人等级21→冒险者雇佣费减半 |
| 物品带回 | 地牢中获取的所有物品归你所有 |
| 冒险者死亡 | 失去本次地牢中部分物品(保留装备) |
| 带回保护 | 商人Lv16→死亡可保留2件/Lv50→保留3件 |

## 6. 设计特点

1. **间接战斗**: 不是玩家操控而是雇佣AI冒险者——你扮演"经理"而非"战士"
2. **冒险者差异化**: 8个冒险者各有独立职业/操作风格/解锁条件——类似"收集角色"玩法
3. **地牢深度跨越巨大**: 5层(试炼)→100层(拉碧斯遗迹)——终局内容体量庞大
4. **黑曜塔50层+Griff解锁**: 需要特定店铺氛围(黑暗)才能触发——系统间联动
5. **Boss=冒险者**: 多个Boss(Charme/Tielle/Griff)实际上是可招募的冒险者——"击败后成为队友"的经典设计
