# 顾客关系与好感度系统 - 露塞提娅:道具屋经营妙方(Recettear)

> 游戏版本: Recettear: An Item Shop's Tale v1.0+ (PC, 2007/2010年)
> 数据来源: recettear.fandom.com, GameFAQs, Steam Community

## 1. 概述

露塞提娅的顾客系统比任何商店游戏都更深——每个顾客有**独立的好感度(Reputation)**系统，通过出售物品逐步提升。好感度直接决定顾客的**预算上限**和**出现频率**。

## 2. 好感度机制

### 2.1 好感度提升方式

| 方式 | 效果 | 说明 |
|------|:--:|------|
| 直接出售(柜台) | +好感 | **唯一可提升的方式** |
| Just Pin出售 | **大幅+好感** | ≤0.5%偏差→+30XP+好感加速 |
| Near Pin出售 | 中等+好感 | ≤5%偏差→+15XP+好感提升 |
| 普通出售(还价后) | 少量+好感 | 还价成功但好感增长慢 |
| 自动售货机出售 | **0好感** | 售货机出货不给好感度！ |
| 愤怒离开 | **减好感** | 价格过高→顾客走→可能几天不来 |

### 2.2 好感度等级(隐藏)

| 等级 | 效果 | 视觉提示 |
|:--:|------|:--:|
| Lv0(初始) | 基础预算 | — |
| Lv1 | 预算轻微增长 | — |
| Lv2 | 预算明显增长 | 偶尔出现♥图标 |
| Lv3 | **预算大幅增长** | 经常♥图标 |
| Lv4(最高) | **预算最大化**(初始的~100倍) | ♥频繁出现 |

### 2.3 预算增长示例

## 3. 好感度策略

### 3.1 早期(前2周)
- **优先Just Pin/Near Pin**→加速好感度→快速提升预算
- 宁可少赚单次利润→换取顾客预算增长
- 重点培养: Alouette(最高支付力)+Old Man(最高预算上限)+Griff(终局)

### 3.2 中期(第3周)
- 预算已提升→开始出售中高价物品
- 维持Pin精度→持续好感增长
- 自动售货机处理低价滞销品(不给好感度无所谓)

### 3.3 后期(第4周+)
- 最大化: Alouette买最贵物品(410%高价市场)
- Griff预算4,000,000+→出售终局Boss掉落物
- 还价策略: 已满好感度的顾客→可适度还价(好感已满不降)

## 4. 顾客出现机制

| 因素 | 影响 |
|------|------|
| 好感度 | 好感越高→出现越频繁 |
| 店铺氛围 | 吸引特定类型(明亮=Elan/黑暗=Griff) |
| 愤怒离开 | 可能几天不来 |
| 时段 | 部分顾客偏好特定时间 |

## 5. 好感度与Tear建议的矛盾

- Tear建议: "定价130%+积极还价"→最大化单次利润
- 实际最优: "定价104-110%+追求Just Pin"→最大化长期好感→更高预算→未来更多利润
- **这个设计矛盾是刻意的**——Tear是贷款公司的→她的建议偏向短期现金流→不利于长期经营

## 6. 设计分析

1. **好感度=长期投资**: 前期少赚→好感度→预算翻10-400倍→终局暴利——精妙的远期回报设计
2. **自动售货机=无好感**: 再次强调直接交易的不可替代性——售货机是清库存工具而非核心
3. **愤怒离开=沉默惩罚**: 不像Moonlighter的即时反馈(客人说"太贵了")→Recettear的惩罚是"客人几天不来"→更隐蔽但更严重
4. **Griff=4,000,000预算=终局客户**: 招募最强冒险者的代价是培养最富有的客户——系统间呼应
5. **Alouette的双重身份**: 既是竞争对手(叙事)也是最佳客户(机制)——角色与系统的巧妙统一
