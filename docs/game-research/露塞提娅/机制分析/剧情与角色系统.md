# 剧情与角色系统 - 露塞提娅:道具屋经营妙方(Recettear)

> 游戏版本: Recettear: An Item Shop's Tale v1.0+ (PC, 2007/2010年)
> 数据来源: recettear.fandom.com, GameFAQs, TV Tropes, NeoSeeker

## 1. 世界观设定

| 要素 | 说明 |
|------|------|
| 背景 | 经典JRPG世界——有冒险者行会/地牢/怪物/魔法 |
| 主角身份 | **非英雄**——是一个父亲离家"去当英雄"留下巨债的普通女孩 |
| 核心冲突 | **债务**而非魔王的威胁——30天内还清62万Pix |

## 2. 主要角色

| 角色 | 身份 | 性格/关系 |
|------|------|------|
| **Recette(露塞提娅)** | 主角/道具店主 | 乐观/天真/被迫经营→逐渐成长为精明的商人 |
| **Tear(蒂亚)** | 妖精/讨债人 | 严谨/冷漠外表→逐渐对Recette产生感情——她的还价建议是**陷阱**(她为贷款公司工作!) |
| **Alouette(阿露埃特)** | 竞争对手/大小姐 | 大商店主的女儿→Recette的对手→有自己的妖精Prime |
| **Prime(普莱姆)** | 妖精(Alouette的) | 比Tear更热情——会告诉Alouette她太过分了 |
| **Recette的父亲** | 失踪/冒险者 | 离家"当英雄"→留下巨债→从未出现 |

## 3. 主线剧情

### Act 1: 绝望开局(第1周)
- Recette的父亲失踪→留下巨债
- Tear代表贷款公司Terme Finance上门讨债
- Recette必须将家改造成道具店→"Recettear"
- 第7天首次还款10,000 Pix

### Act 2: 竞争对手(第2-3周)
- Alouette(大商店主的女儿)出现→视Recette为竞争对手
- 第14天还款30,000→第21天还款80,000
- 逐一招募冒险者(Louie→Charme→Caillou→Tielle→Nagi)
- Tear逐渐从"冷漠讨债妖精"变为"关心Recette的朋友"

### Act 3: 真结局路线(第4周)
- 第30天最终还款500,000 Pix
- 成功=保留商店+进入无尽模式
- 若在3周内完成全部还款→触发**真结局条件**
- 招募Griff(黑曜塔50层+黑暗氛围)→招募Arma(全员解锁+拉碧斯遗迹)

### Act 4: 真结局
- 全部债务还清+全部冒险者招募+拉碧斯遗迹通关
- Recette和Tear的友谊最终超越了债务人与讨债人的关系
- "道具屋"从被迫营生变为Recette热爱的职业

## 4. 叙事方式

| 机制 | 说明 |
|------|------|
| 债务倒计时 | 每7天的还款日是剧情节点的自然触发器 |
| 角色互动事件 | 特定时间/地点触发(广场/酒馆/行会) |
| 冒险者个人剧情 | 每个冒险者有独立小故事线 |
| Tear的情感变化 | 从冷漠到关心——通过日常对话+还款日反应逐步展现 |

## 5. 设计特点

1. **债务=叙事引擎**: 不是"打败魔王"而是"还清贷款"——经营游戏的终极目标自然转化为故事驱动力
2. **Tear=双重身份**: 既是教程引导者(但给的建议是陷阱!)又是讨债人——角色的两个身份制造了自然的矛盾和张
3. **父亲消失=反JRPG套路**: 不是"父亲是传说中的英雄你去继承"而是"父亲是个不负责的冒险者留下巨债"
4. **Alouette不是反派**: 竞争对手——有她的位置和动机→不是"打倒她"而是"证明自己"
5. **还债=游戏胜利**: "赢"不是打Boss而是还清所有贷款——资本主义现实主义
