# 合成与物品管理系统 - 露塞提娅:道具屋经营妙方(Recettear)

> 游戏版本: Recettear: An Item Shop's Tale v1.0+ (PC, 2007/2010年)
> 数据来源: recettear.fandom.com, GameFAQs, Steam Community

## 1. 合成(Fusion)系统

### 1.1 解锁等级

| 合成等级 | 商人等级 | 可合成物品品质 |
|:--:|:------:|------|
| Rank 1 | Lv5 | 基础装备 |
| Rank 2 | Lv13 | 中级装备 |
| Rank 3 | Lv19 | 高级装备 |
| Rank 4 | Lv25 | 顶级装备 |
| Rank 5 | Lv30 | 最终装备 |

### 1.2 合成机制

- 在**行会(Merchant's Guild)**进行
- 消耗: 地牢材料(粘液/蝠翼/焦蜥蜴等)+Pix
- 材料品质: 最高+4
- 成品品质: 最高+15(通过多次合成+高品质材料)

## 2. 物品来源

| 来源 | 特点 |
|------|------|
| 地牢宝箱 | 主要来源——品质随机 |
| 地牢怪物掉落 | 材料类(粘液/蝠翼等) |
| 顾客出售 | Lv2解锁——低价收购→高价出售 |
| 市场(Market) | 行会旁——价格随市场状态波动 |
| 行会(Guild) | 固定100%基准价购买——安全网 |
| 自动售货机 | 基准价出售——清库存工具 |

## 3. 物品管理策略

### 3.1 采购策略

| 市场状态 | 策略 |
|:--:|------|
| 高价(红) | **不买**——进货成本过高 |
| 正常 | 可买——留空间给地牢物品 |
| 低价(蓝) | **大量买入!**——等待市场反弹后出售 |
| 崩盘 | **最大买入**——最低进货成本 |

### 3.2 存货管理

| 物品类型 | 策略 |
|------|------|
| 高价稀有品(如21,000g花瓶) | 等待市场高价(红)+卖给Alouette(150%因子)→最高可达**410%售价** |
| 基础材料(粘液等) | 保留用于合成——出售价低，合成后价值倍增 |
| 中等价位物品 | 基准定价出售→积累商人经验 |
| 滞销品 | 放入自动售货机(100%基准价)→清库存释放柜台 |

## 4. 自动售货机策略

- 售价=**固定100%基准价**(无溢价，无经验加成)
- 用途: 处理低价/滞销/市场崩盘时买入过剩的物品
- **绝不放高价稀有品**
- 初始限1台→商人Lv22解锁无限

## 5. 设计特点

1. **合成=材料保值**: 低价值材料(粘液6g)→合成→高价值装备(数千g)→材料不是"卖不掉就扔"
2. **"买低卖高"真实市场逻辑**: 蓝色市场大量囤货→等红色市场出售——真实商业套利
3. **行会=安全网**: 100%基准价→紧急需要Pix时→快速变现(但无利润)
4. **物品品质+15上限**: 多次合成+高品质材料→追极品质→终局追求
