## 4. 暗杀者事件(属性缩放)

每周属性要求随周数增加:
| 属性 | 每周增长公式 |
|:--:|:--:|
| 魅力(Charm) | ≥ 周数 × 2 |
| 气质(Refinement) | ≥ 周数 × 1.8 |
| 其他 | 递增 |

## 5. 设计特点

1. **13技能=PM系列最复杂的技能体系**: PM2有10属性→PM5有13技能+属性——颗粒度最细
2. **战斗技能≠生活技能**: 剑术/武术/魔法影响战斗→烹饪/时尚影响生活和社交——技能有各自的应用领域
3. **御宅=PM5的幽默**: "御宅"作为正式技能——反映了2007年日本的御宅文化现象
4. **五界战斗=逐步升级的Boss挑战**: 人→妖→灵→地狱→天——由近及远、由低到高的战斗梯度
