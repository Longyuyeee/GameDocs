# 墓地与教堂系统 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (PC/Switch/PS4/Xbox, 2018年)
> 数据来源: Steam Community, TrueAchievements, NeoSeeker

## 1. 墓地评分(Graveyard Quality)

### 1.1 计算方式

```
墓地评分 = Σ(每个坟墓评分)
坟墓评分 = 尸体净白骷髅 + 墓碑加成 + 围栏加成
         (墓碑+围栏≤尸体白骷髅上限)
空坟墓 = -2评分
```

### 1.2 里程碑

| 评分 | 解锁 | 重要性 |
|:--:|------|:--:|
| **5** | **教堂开放**——布道+信仰+科学 | ★★★★★ |
| 30 | 早期Bishop任务 | ★★★ |
| 200 | 大教堂升级+终局Bishop任务 | ★★★★ |
| 400+ | 最大化布道收入 | ★★★★ |

## 2. 墓碑与围栏

| 等级 | 墓碑 | 围栏 | 加成 |
|:--:|------|------|:--:|
| T1 | 木制 | 木制 | +1/+1 |
| T2 | 石制 | 石制 | +2/+2 |
| T3 | 大理石 | 大理石 | +3/+3 |
| T4+ | 高级 | 高级 | +4+/+4+ |

- 墓碑和围栏的加成**不能超过尸体白骷髅数**

## 3. 墓地扩建

- 与Inquisitor对话→完成他的任务(给他柴火)→西半区解锁
- 扩建后可容纳**最多88个坟墓**

## 4. 教堂系统

### 4.1 教堂品质(Church Quality)

| 装饰 | 加成 | 解锁科技 |
|------|:--:|------|
| 教堂长椅(Church Bench) | +? | 信仰的舒适 |
| 烛台I(Candelabra I) | +? | 信仰之光 |
| 香(Incense) | +? | 炼金 |

- 教堂品质10→早期里程碑
- 教堂品质50→大教堂升级

### 4.2 布道(Sermons)

| 参数 | 说明 |
|------|------|
| 时间 | 每**Pride日**(周日) |
| 收入 | **金币**+**信仰(Faith)** |
| 收入公式 | 墓地评分越高→捐赠越多 |
| 信仰用途 | 研究物品(蓝点!)/任务/高级制作 |

## 5. 火化系统

- 解锁: Theology科技树
- 位置: 墓地下方
- 产出: 灰烬(肥料/瓮/墨水)+盐(调味料)
- 策略: 低品质尸体→火化; 高品质→埋葬

## 6. 骨灰龛(Columbariums)

- 材料: 12灰烬+12壶
- 加成: **+20评分/个**
- 无需对应尸体→填充剩余空间的最佳方案

## 7. 设计特点

1. **墓地评分=核心进度**: 不是"金币"而是"墓地品质"驱动游戏进度
2. **教堂=信仰工厂**: 每周布道→产生信仰→驱动科技树——"宗教经济"
3. **火化=废物利用**: 低品质尸体不是扔掉——火化→灰烬→骨灰龛/肥料
4. **88坟墓=规模目标**: 终局需要填充整个墓地的完美(或接近完美)坟墓
