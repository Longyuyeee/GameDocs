# 僵尸自动化系统 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (Breaking Dead DLC/免费更新)
> 数据来源: Steam Community, NeoSeeker Zombies 101, TrueAchievements

## 1. 概述

守墓人的**僵尸工人**是品类中最独特的"自动化"系统——不是洒水器/传送带，而是**复活尸体成为劳动力**。僵尸可用于采矿/伐木/搬运/农耕/炼金——将游戏从手动劳动转变为自动资源流水线。

## 2. 僵尸复活流程

```
与Gerry(会说话的头骨)对话
  → 建造复活台(Resurrection Table): 4木板+4复杂铁件
  → 制作僵尸汁(Zombie Juice): 生命粉+血液(炼金台I)
  → 复活尸体: 1僵尸汁+10信仰→复活台
```

## 3. 僵尸效率

| 尸体白骷髅 | 效率 |
|:--:|:--:|
| 1白骷髅 | ~15-20% |
| 5白骷髅 | ~25-35% |
| 10-12白骷髅 | **~40-65%** |

- **白骷髅少的尸体=用作僵尸**(不是埋葬!)
- **高白骷髅尸体=最高效的僵尸工人**

## 4. 僵尸工作站点

| 站点 | 位置 | 功能 | 材料 |
|------|------|------|------|
| 僵尸铁矿(左/右) | 采石场(西北) | 自动采铁矿 | 8木板+4钉+镐 |
| 僵尸采石场 | 采石场 | 自动采石 | — |
| 僵尸伐木 | 森林 | 自动伐木 | — |
| 僵尸农场/葡萄园 | 花园/葡萄园 | 自动种植 | 种子+肥料(首次) |
| 僵尸搬运工(Porter) | 采石场 | **搬运→家** | — |
| 僵尸炼金 | 教堂地下室 | 自动制作粉/溶液/提取物 | — |
| 僵尸酿酒 | 酒窖 | 自动酿酒 | — |

## 5. 采石场自动化(推荐设置)

```
最佳配置:
  2×僵尸铁矿 + 1×僵尸采石场
  + 铁储存器 + 石储存器
  + 1×僵尸搬运工(把所有东西搬回家)
  → "设置后忘记"的完全自动矿物流水线
```

## 6. 僵尸的局限性

| 限制 | 说明 |
|------|------|
| 僵尸炼金**不享受科学家Perk** | 手动炼金有+1加成→手动做高端/僵尸做批量 |
| 效率与白骷髅直接相关 | 低品质尸体=慢速僵尸 |
| 需持续提供僵尸汁+信仰 | 每次复活成本 |

## 7. 设计分析

1. **尸体→劳动力**: 不是"雇工人"而是"复活尸体"——黑暗幽默贯穿系统设计
2. **效率梯度**: 1白骷髅=15%到12白骷髅=65%——驱动寻找更好的"僵尸原料"
3. **搬运工=物流革命**: 采石场资源→自动运送家中——无需手动往返
4. **僵尸=终局自动化**: 手动时代→僵尸时代——游戏体验从"劳动"变为"管理"
