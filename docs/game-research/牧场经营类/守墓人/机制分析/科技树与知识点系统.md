# 科技树与知识点系统 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (PC/Switch/PS4/Xbox, 2018年)
> 数据来源: Steam Community, GamePressure, TrueAchievements

## 1. 三色知识点

| 颜色 | 获取方式 | 解锁领域 | 全满所需 |
|:--:|------|------|:-----:|
| 🔴红点(Red) | 制作物理物品(木工/石工/金属) | 锻造/建筑/工具 | **5,665** |
| 🟢绿点(Green) | 采集/种植/烹饪/解剖 | 农业/烹饪/采集/解剖炼金 | **3,252** |
| 🔵蓝点(Blue) | **仅教堂开放后**——研究物品/制作玻璃/绳/书 | 神学/书籍/高级制作 | **3,665** |

## 2. 蓝点获取(最关键!)

| 方法 | 效率 | 条件 |
|------|:--:|------|
| **研究身体器官**(研究桌) | ★★★★★ | 消耗信仰+科学→大量蓝点 |
| **制作麻绳(Hemp Rope)** | ★★★★ | 教堂工作台→每根+1蓝点 |
| 制作玻璃→烧瓶→高级烧瓶 | ★★★ | 锻造科技解锁后 |
| 购买蓝书(占星师) | ★★★ | 每本25蓝点/5银 |
| 研究墓地装饰 | ★★★ | 研究桌 |
| 制作墓碑/围栏T2+ | ★★ | 石制/大理石制品 |

## 3. 信仰(Faith)与科学(Science)

| 资源 | 获取 | 用途 |
|------|------|------|
| **信仰(Faith)** | 每Pride日布道 | **研究物品(蓝点!)**/任务/高级制作 |
| **科学(Science)** | 分解纸张(研究桌) | 研究物品(与信仰共同消耗) |

## 4. 科技树结构

| 类别 | 关键科技 | 点数 |
|------|------|:--:|
| 建筑(Building) | 锯木/木工/石工 | 🔴 |
| 锻造(Smithing) | 原始锻造/工具/高级金属 | 🔴 |
| 农业(Farming) | 花园/肥料/葡萄园 | 🟢 |
| 解剖炼金(Anatomy & Alchemy) | 软备件/硬备件/防腐注射 | 🟢 |
| 神学(Theology) | 火化/基础墓碑/蜡烛/香 | 🔵 |
| 书籍(Writing) | 纸/笔记/章节/书/祈祷 | 🔵 |
| 高级制作 | 大理石制品/高级教堂装饰 | 🔵 |

## 5. 研究系统

- 研究桌(Study Table)——教堂地下室
- 消耗: 信仰+科学→获得蓝点
- 研究物品: 身体器官/墓地装饰/特殊物品
- 所有可研究物品→总蓝点收益: **~1,940蓝点**

## 6. 推荐科技路线

```
1. 建筑→锯木/木工(处理木材)
2. 锻造→原始锻造/工具(铁制品)
3. 神学→火化+基础墓碑围栏
4. 解剖炼金→软备件/硬备件
5. 书籍→纸/笔记/章节(蓝点农场!)
6. 农业→花园/肥料
```

## 7. 设计特点

1. **三色科技点=三种玩法方向**: 红=工匠/绿=农夫/蓝=学者——不是单一经验值
2. **蓝点=教堂门控**: 教堂开放前无法获取蓝点→强制墓地品质5→驱动核心循环
3. **信仰=研究货币**: "信仰"(宗教资源)→驱动"科学"(技术进步)——讽刺性的系统设计
4. **科学=纸张分解**: 不是"实验室设备"而是"分解纸张"→中世纪的"科学"
