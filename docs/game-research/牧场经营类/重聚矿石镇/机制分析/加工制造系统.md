# 加工制造系统 - 牧场物语 重聚矿石镇
![矿石镇厨房料理](https://www.bokumono.com/series/fomt/img/ss/screenshot07.jpg)

> 数据来源: ranchstory.farm, irvangen.com, fogu.com

## 1. 核心特征: 无加工机械

矿石镇是**纯出货型经济**——产品从农场直接进入出货箱,无中间加工环节。这与星露谷物语等现代牧场游戏形成鲜明对比。

## 2. 唯一加工方式: 料理系统

### 2.1 解锁条件
- **厨房(Kitchen)**: 房屋扩建第1次,3,000G + 木材×200
- 扩建后获得: 菜刀/平底锅/锅/烤箱/搅拌机/蒸笼 6种基础厨具

### 2.2 食谱获取
| 方式 | 说明 |
|------|------|
| 电视(TV Tuesday) | 每周二"料理节目",观看习得新食谱 |
| NPC好感度 | 与特定NPC达到一定好感后赠予食谱 |
| 试错实验 | 在厨房手动组合食材尝试新料理 |
| 全120种食谱 | 学习所有食谱=解锁美食家(Bon Vivant)结婚候补条件之一 |

### 2.3 料理经济
- 料理售价 = 食材总价值 × 1.2 ~ 2.0
- 料理主要用途: 送礼(+FP/LP) > 体力恢复 > 出货赚钱
- 料理出货利润有限(增殖倍率不如星露谷的加工机械)

## 6. 设计特点

1. **极简加工系统**: 唯一加工方式是料理,无其他增值机械
2. **"出货箱=终点"**: 所有产品直接出货,保留"纯种田"的质朴感
3. **料理=社交工具**: 送礼价值>出货价值
4. **这是矿石镇的特色而非缺陷**: 作为2003年GBA游戏,极简设计就是它的定位。2020年重制保留了这一特色,没有借机添加加工机械
5. **与星露谷最核心的差异**: 星露谷经济核心是"原料→加工→高价值商品"的增值链;矿石镇经济核心是"种出更高品质的作物直接出货"
