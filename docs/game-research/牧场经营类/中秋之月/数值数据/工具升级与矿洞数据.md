# 工具升级与矿洞数据 - 中秋之月(Harvest Moon: Back to Nature)

> 游戏版本: Harvest Moon: Back to Nature (PS1, 1999年)
> 数据来源: harvestmoon.fandom.com, GameFAQs

## 一、工具四级升级

| 等级 | 矿石 | 费用 | 效果(锄头/水壶/斧/锤) |
|:--:|------|:--:|------|
| 基础(Basic) | — | — | 1格范围 |
| 铜(Copper) | 铜矿 | 1,000G | 2格范围(蓄力) |
| 银(Silver) | 银矿 | 2,000G | 3格范围 |
| 金(Gold) | 金矿 | 3,000G | 5格范围 |
| **秘银(Mystrile)** | 秘银矿 | 5,000G | **最大范围+最高效率** |

- 升级条件: 工具使用到特定程度(菜单中工具名旁出现蓝石)
- 升级时间: 交给铁匠铺→等待1-2天

## 二、五种工具效果

| 工具 | 升级效果 |
|------|------|
| 锄头(Hoe) | 翻耕范围扩大(1→2→3→5→最大格) |
| 水壶(Watering Can) | 浇灌范围扩大+蓄水量增加 |
| 斧头(Axe) | 砍树次数减少+可砍硬木 |
| 锤子(Hammer) | 碎石次数减少+可碎大石 |
| 镰刀(Sickle) | 除草/收割范围扩大 |

## 三、矿洞深度与矿物

| 深度 | 常见矿物 | 稀有矿物 |
|:--:|------|------|
| 1-10层 | 铜/银 | — |
| 11-20层 | 银/金 | 秘银(罕见) |
| 21-30层 | 金/秘银 | 金刚矿(罕见) |
| 31-50层 | 秘银 | 金刚矿/奥利哈钢 |
| 50+层 | 秘银/金刚矿 | 奥利哈钢(极稀有) |
| 100+层 | 全部 | 力量果实(极低概率) |

| 矿物 | 工具 | 特殊制作 | 出售价 |
|------|:--:|------|:---:|

## 五、力量果实位置

| 来源 | 获取方式 |
|------|------|
| 矿洞深层 | 随机层数/极低概率 |
| 钓鱼 | 特定钓点/低概率 |
| 节日 | 赛马奖励/其他 |
| 采集 | 母亲山特定季节 |

- 每次获得→永久提升体力上限
