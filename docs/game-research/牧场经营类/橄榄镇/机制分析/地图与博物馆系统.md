# 地图与博物馆系统 - 橄榄镇(Story of Seasons: Pioneers of Olive Town)

> 游戏版本: Story of Seasons: Pioneers of Olive Town (Switch/PC/PS4, 2021年)
> 数据来源: Harvest Moon Wiki, TheGamer

## 1. 地图结构

```
橄榄镇(Olive Town)
├── 玩家农场(Farm) — 可自定义布局/铺设道路/围栏
│   ├── 初始区域
│   ├── 第2区域(剧情解锁)
│   └── 第3区域(后期解锁)
├── 城镇中心
│   ├── 商业公会
│   ├── 博物馆
│   ├── 各类商店(种子/动物/工具)
│   └── 广场(节日场地)
├── 森林/山丘
│   ├── 采集点(草药/野果/花)
│   ├── 钓鱼点
│   └── 矿洞
├── 海滩
│   ├── 钓鱼点
│   └── 采集点
└── 精灵之森(Sprite Forest)
    └── 精灵派遣管理
```

## 2. 农场自定义

| 元素 | 说明 |
|------|------|
| 道路(Paths) | 可铺设不同材质路面→移动速度加成 |
| 围栏(Fences) | 圈出种植区/畜牧区 |
| 装饰(Decorations) | 美观+可能功能加成 |
| 机器布局 | 制造机可任意摆放 |

## 3. 博物馆系统

| 捐赠类别 | 物品来源 |
|------|------|
| 文物(Artifacts) | 挖掘/矿洞 |
| 鱼类(Fish) | 钓鱼 |
| 宝石(Gems) | 矿洞 |
| 照片(Photos) | **拍照系统**(特定地点拍摄) |
| 作物(Crops) | 种植(高品质要求) |

- 捐赠奖励: 道具/装备/家具/稀有物品
- 全收集=额外奖励

## 4. 拍照系统

- 完成特定拍摄任务→博物馆收藏
- 支线剧情: 部分NPC有拍照相关任务
- 解锁特殊滤镜/功能

## 5. 设计特点

1. **农场=可自定义空间**: 不同于BTN的"固定布局"——玩家可自由摆放道路/围栏/装饰
2. **博物馆=拍照维度**: 除了传统捐赠(文物/鱼)，新增加拍照收藏——利用Switch的截屏文化
3. **三阶段农场扩展**: 渐进式解锁→避免初期空间过大导致无所适从
4. **精灵=省力化**: 派遣精灵自动采集→缓解日常重复劳动(尤其是机器管理之外)
