# 时间与日常系统 - 沙石镇时光(My Time at Sandrock)

> 游戏版本: My Time at Sandrock v1.0+ (PC/Switch/PS5/Xbox)
> 数据来源: mytimeatsandrock.fandom.com, Siliconera, Gamer Guides

## 1. 时间系统

| 参数 | 设定 |
|------|------|
| 游戏内一天 | 6:00 AM - 2:00 AM(20小时) |
| 每周 | 7天 |
| 每季节 | 4周(~28天) |
| 每年 | 4季 |
| 强制睡觉 | 2:00 AM(晕倒→次日体力惩罚) |

## 2. 一天结构

| 时段 | 时间 | 特点 |
|------|:--:|------|
| 清晨 | 6:00 AM | 新一天开始 |
| 白天 | 6:00 AM - 8:00 PM | 全部商店/NPC活跃 |
| 晚上 | 8:00 PM - 12:00 AM | 部分NPC仍在外(酒馆等) |
| 深夜 | 12:00 AM - 2:00 AM | 大部分NPC回家/睡觉 |
| 强制睡觉 | 2:00 AM | 眩晕→体力惩罚 |

## 3. 季节系统(弱化)

沙石镇的季节系统**远不如传统牧场游戏重要**。沙漠环境季节差异小——以"主线推进"而非"季节事件"驱动游戏节奏。

| 季节 | 特色作物 | 备注 |
|:----:|------|------|
| 春 | 小麦/马铃薯果/甜薯椒 | 游戏起始季 |
| 夏 | 黄麻甜瓜/甜薯椒 | — |
| 秋 | 小麦/马铃薯果/咖啡茶树 | — |
| 冬 | 山地玫瑰 | 部分作物偏好冬/春 |

- 大量作物**全年可种**(沙米/沙韭/仙人掌/玫瑰柳/犀牛角/栗子树)
- 没有"冬季停摆"——沙漠本来就荒

## 4. 体力系统

| 项目 | 说明 |
|------|------|
| 体力(Stamina) | 采集/采矿/战斗等消耗 |
| 0体力 | 无法继续工作(但危险废墟中冻结) |
| 恢复方式 | 食物/睡觉 |
| "充分休息" | 3AM前睡觉触发(Stamina Knowledge Lv3) |
| 体力上限 | 通过食物/知识技能提升 |

## 5. 机器运作(独立时间)

| 参数 | 说明 |
|------|------|
| 机器工作时间 | 与现实游戏时间同步，不因睡觉跳过 |
| 燃料(Fuel) | 机器运行时消耗 |
| 水(Water) | 机器运行时消耗，缺水=停机 |
| 排队生产 | 离开前排队任务→晚上睡觉→早上领取 |
| Feather Duster | 加速机器生产+50% |

## 6. 知识技能的时间相关加成

| 技能 | 效果 |
|------|------|
| Stamina Knowledge Lv3 | 3AM前睡觉→"充分休息"效果 |
| Time Management Lv2-3 | 机器生产速度+10/20% |
| Fuel Saving Lv1-2 | 燃料使用-10%(机器运转更久) |
| Water Conservation Lv1 | 机器水消耗-10% |
| Quick Delivery Lv1-2 | 同日交付委托+5-10%戈尔/经验 |

## 7. 设计特点

1. **季节弱化**: 沙漠设定让季节变化没有田园感——把节奏驱动交给主线而非季节
2. **机器时间独立**: 不同与星露谷的"睡觉=跳过所有加工"，Sandrock的机器在时间线中连续运作
3. **危险废墟时间冻结**: 战斗地牢内时间停止——玩家可以从容攻略
4. **3AM弹性截止**: Stamina Knowledge→3AM前睡觉奖励——比2AM硬截止更灵活
5. **无节日密集日历**: 节日数量少于传统牧场(见节日数据文档)——游戏重心放在委托和探索
