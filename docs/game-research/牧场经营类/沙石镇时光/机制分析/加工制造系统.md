# 加工制造系统 - 沙石镇时光(My Time at Sandrock)

> 游戏版本: My Time at Sandrock v1.0+ (PC/Switch/PS5/Xbox)
> 数据来源: mytimeatsandrock.fandom.com, Fandom Wiki, Steam, Gamer Guides

## 1. 概述

沙石镇时光的加工制造系统以**组装站(Assembly Station)**为核心，配合一系列专业工作站进行物品制作、精炼和烹饪。与工坊建造系统(侧重建筑/机器框架)不同，本系统侧重**物品制作链和精炼升级**。

## 2. 核心工作站

### 2.1 基础工作站

| 工作站 | 功能 | 主要配方类型 |
|--------|------|------------|
| 工作台(Worktable) | 基础物品制作 | 工具/武器部件/基础材料 |
| 高级工作台(Advanced Worktable) | 复杂组件 | 高级武器/装备/机器组件 |
| 熔炉(Furnace) | 矿石→金属锭 | 铜/铁/铬/铝/钛锭 |
| 回收机(Recycler) | 废料→可用材料 | 旧零件/铜棒/木棒/塑料 |
| 研磨机(Grinder) | 原料加工 | 铜线/铁棒/铝板等 |
| 处理器(Processor) | 组件精炼 | 轴承/管道/弹簧/齿轮 |

### 2.2 专业化工作站

| 工作站 | 功能 | 关键产品 |
|--------|------|------|
| 裁缝机(Tailoring Machine) | 制作衣装/布料 | 防具上衣/裤子/布料/皮革 |
| 珠宝机(Jewelry Machine) | 制作饰品 | 戒指/项链/护身符 |
| 锻造机(Forging Machine) | 金属精加工 | 高品质武器/工具 |
| 精炼机(Refiner) | 装备属性强化 | 武器/防具强化(消耗材料) |
| 厨师烹饪台(Chef's Cooking Station) | 烹饪食物 | HP/体力恢复+Buff |

### 2.3 素材处理

| 工作站 | 功能 | 原材料 |
|--------|------|------|
| 矿物过滤器(Mineral Filtrator) | 矿物提纯 | 矿石→高品质金属 |
| 露水收集器(Dew Collectors) | 水资源 | 露水→水(10:1) |
| 堆肥器(可选) | 肥料制作 | 生物废料→肥料 |

## 3. 加工链条(完整)

### 3.1 金属加工链

```
矿石(Ore) → 熔炉 → 金属锭(Bars) → 工作台/研磨机/处理器
  → 铜棒/铁板/轴承/齿轮/弹簧/管道
    → 组装站 → 机器/工具/武器/装备
```

### 3.2 废料回收链

```
废料堆(Scrap Piles) → 回收机 →
  ├── 旧零件(Old Parts)
  ├── 铜/铁棒
  ├── 塑料/橡胶
  └── 微芯片/电子元件
```

### 3.3 装备精炼链

```
基础装备 → 精炼机(Refiner) → 属性提升装备
  → 再次精炼 → 更高属性/稀有品质
    (Advanced Refining Knowledge: 5%免费+5%双倍)
```

### 3.4 食物加工链

```
原料(作物/肉/鱼) → 厨师烹饪台(Chef's Cooking Station)
  → 恢复食品(HP/体力)
  → Buff食品(攻击/防御/速度)
  → 送礼食品(NPC最爱)
```

## 4. 品质系统

| 品质 | 获得方式 |
|------|------|
| 普通(Normal) | 基础制作 |
| 优秀(Outstanding) | 随机概率/技能加成 |
| 完美(Perfect) | 低概率/高品质技能最大化 |
| 稀有(Rare) | 极低概率/特定条件 |

- "高品质组装(High Quality Assembly)"知识: 组装站产出更高品质概率
- "高品质工作台(High Quality Worktable)"知识: 工作台/机器产出更高品质概率
- "高级建造知识(Advanced Building Knowledge)": 进一步提高所有机器高品质概率

## 5. 工厂(Factory)自动化

| 属性 | 详情 |
|------|------|
| 容量 | 20制造批次+20回收任务同时 |
| 供电 | 需火力发电机(Fire Powered Generator) |
| 限制 | 不享受品质加成(机器放在外面手动运行可享受) |
| 物流 | 产出→自动送入交付箱 |

## 6. 关键知识技能

| 技能 | 效果 |
|------|------|
| 学习者(Learner Knowledge) | 组装站+10/20%经验, 质检时间+20秒 |
| 量产(Mass Production) | 工作台/机器+10/20%经验 |
| 高品质组装(High Quality Assembly) | 组装站高品质概率提升×2级 |
| 高品质工作台(High Quality Worktable) | 工作台/机器高品质概率提升×2级 |
| 高级建造(Advanced Building) | 机器产品高品质概率再提升 |
| 高效过滤(Efficient Filtration) | 矿物过滤器资源消耗-1, 回收/精炼20%额外产出 |
| 高级精炼(Advanced Refining) | 5%免费精炼, 5%双倍精炼效果 |
| 节水(Water Conservation) | 机器水消耗-10/20%, 露水收集器+50%效率 |
| 时间管理(Time Management) | 机器生产速度+10/20% |
| 省燃料(Fuel Saving) | 燃料消耗-10/20% |

## 7. 设计特点

1. **组装站+工作站的二级体系**: 大件→组装站(即时完成)，小件→工作台(队列生产)——区分"建造"和"制作"
2. **废料回收=资源核心**: 不是采矿→冶炼，而是废料→回收→材料——契合废土设定
3. **精炼替代更换**: 不是"造新装备替换旧装备"，而是"精炼强化现有装备"→减少材料浪费
4. **工厂=终局自动化但牺牲品质**: 策略选择——效率(QoL)vs品质(收益)
5. **知识深度绑定制造效率**: 15+个制造相关技能——经营方向的skill build有意义
6. **水=燃料**: 机器消耗水而非电力→沙漠设定中的水资源稀缺叙事与机制一致
