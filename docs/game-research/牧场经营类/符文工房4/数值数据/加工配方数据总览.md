## 四、制作规则

| 规则 | 说明 |
|------|------|
| 等级上限 | 可尝试制作技能等级+9以内的配方 |
| 超限成功率 | 仅10%，但材料**不损失**(锻造/制作/药学) |
| 烹饪失败 | 材料和成品**都损失** |
| 材料继承 | 素材自带属性可部分继承到成品 |
