# 经济系统 - 符文工房4 豪华版
![RF4 塞尔菲亚](https://www.runefactory4.com/img/top/mainvisual.jpg)

> 游戏版本: 符文工房4 豪华版(Rune Factory 4 Special) (Switch 2019年 / PC/PS/Xbox 2021年)
> 原版: 符文工房4 (Rune Factory 4) (3DS, 2012年)

## 1. 货币类型

| 货币 | 获取途径 | 用途 |
|------|---------|------|
| **金币(Gold/G)** | 出货、任务奖励、怪物掉落 | 购买种子/装备/食谱面包、建造、升级 |
| **王子/公主点数(Prince/Princess Points/PP)** | 完成伊莱莎(Eliza)委托、节日获胜 | 解锁设施、背包扩展、节日许可证、指令系统 |

**RF4 独有**: PP 点数系统是 RF4 的核心创新——作为第二货币驱动城镇建设，将"玩家行为→世界变化"具象化。

## 2. 收入来源

### 2.1 各阶段收入权重

| 来源 | 前期(1-2季) | 中期(1年) | 后期(2年+) | 说明 |
|------|:---------:|:--------:|:--------:|------|
| 农作物 | ★★★★★ | ★★★★ | ★★★ | 稳定的基础收入 |
| 烹饪品出货 | ★★ | ★★★★ | ★★★★★ | 加工增值链是后期核心 |
| 怪物掉落 | ★★ | ★★★ | ★★★ | 高级迷宫掉落物价值高 |
| 委托奖励 | ★★★ | ★★ | ★ | PP点数才是主要收益 |
| 钓鱼 | ★★★ | ★★ | ★ | 季节性补充 |
| 节日奖品 | ★ | ★★ | ★ | 作物祭9个食谱面包很值 |
| 锻造品出货 | — | ★ | ★★★ | 高级装备出货利润极高 |

### 2.2 出货机制
- 出货箱(Shipping Bin)在农场旁边
- **每日 8:00 AM 结算**（RF3 为下午5:00，RF4 改为早晨——更合理的设计）
- 出货后不可撤回
- 可出货物品: 所有作物、怪物掉落、武器、装备、食物等

## 3. 物价体系

### 3.1 完整价格链(以春季为例)

```
种子(80-620G)
  → 种植+浇水
    → 收获作物(Lv1: 200-4200G)
      → 出货(直接出售)
      → 镰刀收种 → 再种植 → 提升等级 → Lv10作物
      → 烹饪加工 → 成品出货(价值×1.5-3+)
      → 锻造材料 → 武器/防具出货
```

- 种子成本约占售价 8-40%
- **烹饪加工**是中后期核心增值手段
- 高等级作物提升出售价格但对烹饪品配方无加成（配方只看基础食材类型）

### 3.2 商店定价
- 所有商品价格固定，无市场波动
- 游商由(Yue)随机出售稀有种子和物品
- 跳蚤市场节(Flea Market, 夏19日): 以物易物，NPC出售稀有品

## 4. 主要消费出口(金钱 Sinks)

| 消费项 | 价格 | 说明 |
|--------|------|------|
| 锻造台(Forge/Furnace) | 5,000G + 100木材(Lumber) | 武器/工具升级必须 |
| 双人床(Double Bed) | 15,000G + 80木材 | 结婚必须 |
| 婚戒(Engagement Ring) | 1红宝石(Ruby) + 2银(Silver) | 锻造Lv20 |
| 怪物小屋(Monster Barn) | 约2,000G+100木材 | 可建7间 |
| 房屋扩建 | 多次升级(金币+木材) | 解锁制作台/药学台 |
| 料理台升级 | 各厨具2,000-3,000G | Frying Pan/Pot/Mixer/Steamer/Oven/Knife |
| 食谱面包(Recipe Bread) | 350-750G/个 | 持续消耗 |
| 背包扩展 | PP点数购买 | 20→25→30格 |
| 节日许可证 | 500-1,000PP | 每个节日需要购买 |

## 5. PP点数系统(第二货币)

| 获得方式 | PP数量 |
|---------|--------|
| 完成伊莱莎委托 | 10-50 PP |
| 节日获胜 | 50-200 PP |
| 主线推进 | 大量PP奖励 |

| 消费项 | PP消耗 |
|--------|--------|
| 背包扩展 | 500→1000→1500 PP |
| 节日许可证 | 500-1000 PP |
| 新设施解锁 | 1000-3000 PP |
| 怪物小屋许可证 | 500 PP/间 |
| 锻造/药学许可证 | 各500 PP |

## 6. 设计特点

1. **双货币驱动**: 金币驱动个人成长，PP驱动城镇建设，两条线并行推进
2. **烹饪加工是后期核心经济**: 不像RF3缺乏加工链，RF4的烹饪系统提供了明确的增值路径
3. **PP点数="玩家主动塑造世界"**: 不再是固定解锁，而是玩家用行动赚PP选择先开发什么
4. **固定价格+有限sinks**: 长期游玩金币仍会溢出，但PP始终有消费出口
5. **食谱面包是持续消耗**: 350-750G/个，驱动玩家持续赚金币
