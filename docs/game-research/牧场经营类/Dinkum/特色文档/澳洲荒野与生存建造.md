# 澳洲荒野与生存建造主题 - Dinkum

> 游戏版本: Dinkum (PC/Switch, 2025年)

## 一、概述

Dinkum是**唯一以澳洲内陆荒野为主题**的牧场/建造游戏。不是绿色田园(星露谷)/热带海岛(珊瑚岛)/史前森林(帕夏时代)——而是干燥、危险、需要"拓荒"的澳洲大陆。这个主题选择带来了独特的系统设计。

## 二、荒野主题对系统的影响

| 系统 | 传统牧场 | Dinkum(澳洲荒野) |
|------|:--:|:--:|
| 核心活动 | 种植 | **采矿+探索** |
| 动物 | 鸡/牛/羊(可爱) | **鳄鱼/鲨鱼(危险!)** |
| 起步 | 继承农场 | **一无所有→拓荒** |
| 节奏 | 渐进经营 | **许可解锁→激烈探索** |
| 基调 | 温馨/治愈 | **荒野求生的警觉** |

## 三、与动森/星露谷的对比

| 维度 | 动森 | 星露谷 | Dinkum |
|------|:--:|:--:|:--:|
| NPC入居 | 邀请→盖房→立即来 | 一开始就都在 | **多次访问→好感→永久入居** |
| 地形 | 固定 | 固定 | **完全自由改造** |
| 战斗 | 无 | 矿洞怪物 | **野外+矿洞+危险动物** |
| 多人 | 联机 | 联机 | **4人协作建设** |
| 危险 | 无(黄蜂) | 低(矿洞) | **高(鳄鱼/鲨鱼)** |
| 开发阶段 | 完成 | 完成 | 持续更新 |

## 四、设计分析

1. **澳洲=独特的荒野幻想**: 不是每个人想当"日本农民"或"美国牧场主"——澳洲荒野提供了一个新的幻想空间
2. **危险动物=生存感**: 看到鳄鱼绕路——不是所有游戏内容都是"可征服"的——尊重荒野
3. **自由地形改造=城镇建造**: 不是"在固定土地上种地"——而是"从零建造整个小镇"
4. **许可=渐进的拓荒**: "我要先学采矿才能下矿洞"——不是技能树而是"许可证"——更符合"在荒野中逐步文明化"的叙事
