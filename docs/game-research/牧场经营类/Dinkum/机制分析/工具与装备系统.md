# 工具与装备系统 - Dinkum

> 游戏版本: Dinkum (PC/Switch, 2025年)
> 数据来源: dinkum.fandom.com

## 1. 工具

| 工具 | 需许可 | 功能 | 升级 |
|------|:--:|------|:--:|
| 镐子(Pickaxe) | 采矿 | 开采矿石 | 铜→铁→银→金 |
| 斧头(Axe) | 伐木 | 砍树 | 铜→铁→银→金 |
| 钓竿(Fishing Rod) | 钓鱼 | 钓鱼 | 基础→升级 |
| 武器(Weapon) | 狩猎 | 防身/狩猎 | 基础→高级 |
| 铲子(Shovel) | 挖掘 | 挖掘/种植 | 基础→升级 |
| 金属探测器 | 金属探测 | 寻宝 | 基础 |
| 虫网(Bug Net) | 无 | 捕虫 | 基础→升级 |

## 2. 装备

| 装备 | 功能 |
|------|------|
| 滑翔伞(Glider) | 空中移动/三层矿洞必需 |
| 矿工帽(Miner's Hat) | 照明(矿洞) |
| 手电筒(Flashlight) | 照明 |
| 火把(Torch) | 标记路线(矿洞) |

## 3. 矿洞装备策略

| 必备 | 数量 | 原因 |
|------|:--:|------|
| 武器 | ×2 | 备用(防止损坏) |
| 镐子 | ×2 | 备用 |
| 食物 | 充足 | 体力恢复 |
| 矿工帽 | 1 | 照明 |
| 火把 | 多 | 标记路线 |

## 4. 设计特点

1. **工具需许可证**: 不是"有钱就买"——需先买许可证
2. **双工具策略**: 矿洞中工具会损坏→需携带备用——增加准备深度
3. **滑翔伞=终局移动**: 类似塞尔达的滑翔伞——不仅是移动工具也是矿洞3层必需品
