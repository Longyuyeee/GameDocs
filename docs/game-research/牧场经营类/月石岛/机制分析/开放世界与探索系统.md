# 开放世界与探索系统 - 月石岛(Moonstone Island)

> 游戏版本: Moonstone Island (PC/Switch, 2023年)
> 数据来源: moonstoneisland.wiki.gg, TheGamer

## 1. 世界结构

| 参数 | 说明 |
|------|------|
| 岛屿数量 | **100座程序化生成的浮空岛** |
| 生物群系 | 大地/水/火/毒/电/冰/超能/暗——对应精灵元素 |
| 地牢 | 30个标准地牢+**无限地牢(Infini-dungeon)** |
| 神庙 | 春/夏/秋/冬四神庙——解锁对应精灵能力 |

## 2. 移动方式

| 方式 | 获取 | 功能 |
|------|------|------|
| 气球(Balloon) | 早期 | 基础飞行→跨岛 |
| 扫帚(Broom) | 中期 | 更快飞行 |
| 滑翔翼(Glider) | 早期 | 滑翔/下降 |
| 传送哨(Warp Whistle) | — | 传送回基地(Toxitoe精灵可每天使用2次) |

## 3. 资源分布

| 生物群系 | 矿物 | 植物 | 精灵 |
|:--:|------|------|------|
| 大地岛 | 铁/铜/煤 | 基础作物/花 | 大地精灵 |
| 水岛 | 宝石 | 海草/海玻璃 | 水精灵 |
| 火岛 | 稀有矿 | 火植物 | 火精灵 |
| 毒岛 | 毒材料 | 毒蘑菇 | 毒精灵 |
| 冰岛(冬季) | 冰晶 | 冬季植物 | 冰精灵 |
| 暗岛(外环) | 暗物质 | 暗植物 | 暗精灵 |

## 4. 地牢

| 地牢 | 数量 | 特色 |
|------|:--:|------|
| 标准地牢 | 30 | 每地牢有Boss/宝箱(UI显示剩余宝箱数) |
| 无限地牢 | 1 | 无限层→终局挑战→最高奖励 |

## 5. 设计特点

1. **100座浮空岛=开放世界的牧场**: 不是"一个农场+一个城镇"——而是"100个岛组成的天空世界"
2. **程序化生成=可重玩性**: 每次新游戏→岛屿分布不同
3. **飞行=核心移动**: 气球/扫帚/滑翔翼——3D空间探索
4. **无限地牢=终局挑战**: 30标准地牢后→无限地牢持续提供挑战
