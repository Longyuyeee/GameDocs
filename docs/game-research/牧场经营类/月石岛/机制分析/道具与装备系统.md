# 道具与装备系统 - 月石岛(Moonstone Island)

> 游戏版本: Moonstone Island (PC/Switch, 2023年)
> 数据来源: moonstoneisland.wiki.gg

## 1. 工具

| 工具 | 功能 | 升级路线 |
|------|------|:--:|
| 镐(Pickaxe) | 采矿 | 基础→铜→铁→银→金 |
| 斧(Axe) | 砍树 | 基础→铜→铁→银→金 |
| 锄(Hoe) | 翻耕 | 基础→铜→铁→银→金 |
| 钓竿(Fishing Rod) | 钓鱼 | 基础→升级 |
| 精灵球(Spirit Ball) | 捕获精灵 | 基础→高级→终极 |

## 2. 飞行器

| 装备 | 速度 | 获取 |
|:--:|:--:|------|
| 滑翔翼(Glider) | 慢 | 早期 |
| 气球(Balloon) | 中 | 早期制作 |
| 扫帚(Broom) | 快 | 中期制作 |

## 3. 体力(Stamina)

| 要素 | 说明 |
|------|------|
| 消耗 | 工具使用/飞行 |
| 恢复 | **温泉**(散布世界各岛)→不是回家睡觉 |
| 上限 | 通过食物/升级提升 |

## 4. 设计特点

1. **温泉=开放世界的体力管理**: 不需要回到基地→找最近的温泉即可
2. **三飞行器=速度梯度**: 滑翔翼(初始)→气球(升级)→扫帚(最快)
3. **精灵球=捕获消耗品**: 类似宝可梦——需要持续制作/购买
