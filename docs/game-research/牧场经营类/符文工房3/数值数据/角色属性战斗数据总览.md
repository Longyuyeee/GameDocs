# 角色属性与战斗数据总览 - 符文工房3 豪华版
![RF3 战斗](https://manuals.marvelousgames.com/rf3sp/10/img/top.jpg)

> 游戏版本: 符文工房3 豪华版(Rune Factory 3 Special) (Switch / Steam, 2023年)
> 注: 符文工房系列没有传统"角色等级",属性完全由技能等级决定。精确公式需数据挖掘验证。

## 一、基础属性一览

| 属性 | 缩写 | 含义 | 影响的行为 |
|------|------|------|----------|
| 生命值(Hit Points) | HP | 生命值 | 归零=晕倒,丢失部分金币 |
| 符文点数(Rune Points) | RP | 行动体力 | 一切工具/武器/魔法使用的消耗资源,归零后消耗HP替代 |
| 力量(Strength) | STR | 物理攻击力 | 影响物理伤害(所有近战武器) |
| 智力(Intelligence) | INT | 魔法攻击力 | 影响魔法伤害(法杖/属性攻击) |
| 体力(Vitality) | VIT | 物理防御力 | 减少受到的物理伤害 |
| 魔法防御(Magic Defense) | M.Def | 魔法防御力 | 减少受到的魔法伤害 |
| 攻击力(Attack) | ATK | 武器+STR补正后的最终攻击力 | 决定造成的物理伤害 |
| 魔法攻击力(Magic Attack) | M.ATK | 法杖/INT补正后的魔法攻击力 | 决定造成的魔法伤害 |
| 暴击率(Critical Rate) | 无缩写 | 普通攻击的暴击概率 | 斧类武器暴击率高 |

## 三、物理伤害计算公式(推测)

```
基础伤害 = 武器攻击力 + (STR × 系数)
实际伤害 = 基础伤害 × (1 - 防御减免) × 暴击倍率 × 元素修正

其中:
- 武器攻击力: 由武器等级决定
- STR系数: 约 0.5-1.0 (不同类型武器有差异)
- 防御减免: VIT / (VIT + 常数)
- 暴击倍率: 1.0(普通) / 约1.5-2.0(暴击)
- 元素修正: 相克 ×1.5 / 同属性 ×0.5 / 其他 ×1.0
```

> 注: 精确公式需数据挖掘验证。符文工房系列通常使用分段函数,低防御时每点VIT减伤效果更明显。

## 四、魔法伤害计算公式(推测)

```
基础魔法伤害 = 法杖基础攻击 + (INT × 系数)
实际魔法伤害 = 基础魔法伤害 × (1 - 魔法防御减免) × 元素修正

其中:
- 法杖基础攻击: 由法杖等级决定
- INT系数: 约 1.0-1.5
- 魔法防御减免: M.Def / (M.Def + 常数)
```
