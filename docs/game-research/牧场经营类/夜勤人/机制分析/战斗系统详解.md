# 战斗系统详解 - 夜勤人(Moonlighter)

> 游戏版本: Moonlighter v1.0+ (PC/Switch/PS4/Xbox, 2018年)
> 数据来源: moonlighter.fandom.com, TrueAchievements

## 1. 战斗定位

夜勤人的战斗是**俯视角动作Roguelite**风格。不是回合制/横版卷轴——而是类似塞尔达传说(2D)的房间制战斗。每个房间清除敌人→解锁门→移动到下一个房间。

## 2. 操作机制

| 操作 | 效果 |
|------|------|
| 普通攻击 | 武器基本攻击(连击) |
| 蓄力攻击 | 长按→蓄力→释放强力攻击 |
| 闪避翻滚(Dodge Roll) | 短无敌帧→穿越敌人和弹幕 |
| 切换武器 | 快捷切换主/副武器 |
| 使用药水 | 即时恢复HP(快捷键) |

## 3. 六种武器操作差异

| 武器 | 操作手感 | 蓄力效果 | 适合应对 |
|------|------|------|------|
| 剑与盾 | 中速3连击，可格挡 | 盾反 | 新手/均衡 |
| 大剑 | 慢速弧形挥砍 | 蓄力大范围 | 群体小怪 |
| 长矛 | 中速直线穿刺 | 穿透攻击 | **安全输出**(穿墙) |
| 弓 | 远程单发，可蓄力 | **穿透+越障碍** | **Boss战首选** |
| 拳套 | 极速多段连击 | 蓄力升龙拳 | 近身速杀/单体DPS |
| 扫帚 | 极低伤害 | — | 成就挑战 |

## 4. 敌人攻击模式分类

| 类型 | 代表敌人 | 应对 |
|------|------|------|
| 直线冲锋 | 野猪/石兵 | 侧闪避→反击 |
| 圆形AOE | 史莱姆/孢子 | 拉开距离 |
| 弹幕(Bullet Hell) | 火焰小恶魔/科技炮台 | 弓远程清/走位 |
| 召唤型 | 播种者/母魔像 | 优先击杀召唤者 |
| 护盾/格挡 | 熊(格挡姿态)/龟(缩壳) | **等待破绽**(不要在格挡时攻击!) |
| 飞行型 | 蝙蝠/飞行魔像 | 弓最佳→近战跳攻 |
| 宝箱怪(Mimic) | 伪装宝箱 | 接近前先攻击试探 |

## 5. Boss战设计

| Boss | 阶段 | 核心机制 |
|------|:--:|------|
| 魔像王 | 单阶段 | 中间固定→浮石臂扫荡→打史莱姆弱点 |
| 食人花 | 多阶段 | 召唤孢子植物/根刺AOE/毒藤——弓远程最佳 |
| 娜迦 | 3段身体 | **随机攻击模式**——最快节奏Boss |
| 能量流 | 多AOE | 电浆球/电能场——持续移动 |
| 维度海盗船长 | 多阶段 | 剑盾格挡+冲刺+能量爆发 |

## 6. 战斗技巧

| 技巧 | 说明 |
|------|------|
| **投掷碰撞** | 抓起小怪→投向其他敌人→碰撞伤害 |
| **房间门控** | 清完房间才开门→优先击杀远程/召唤型敌人 |
| **陷阱利用** | 部分房间有地形伤害(尖刺/岩浆)→引敌人撞陷阱 |
| **药水时机** | 不要等HP见底才喝→Boss战HP<50%就补 |
| **武器切换** | 近战清小怪/弓打Boss——双武器配置 |

## 7. 设计特点

1. **俯视角房间制**: 非开放地图/横版卷轴——类似以撒/塞尔达的房间清版
2. **六种武器=六种操作体验**: 大剑的蓄力/拳套的连击/弓的远程——不是"数值差异"而是"玩法差异"
3. **Boss不可重复刷**: 每Boss一局一次→鼓励推进但无法刷Boss掉落
4. **扫帚=极限挑战**: 极低伤害的扫帚武器+成就(用扫帚击杀全部Boss)——游戏内置的硬核挑战模式
