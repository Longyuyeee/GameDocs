# 海底探索与环保系统 - 珊瑚岛(Coral Island)

> 珊瑚岛最大的差异化特色: 海底探索+环保清理+海底农场三位一体

## 1. 海底探索(潜水系统)

### 1.1 解锁
- 主线推进后获得潜水装备(Diving Suit)
- 可在海洋区域自由探索
- 海底地图分多个区域,按深度解锁(Ocean Depth)

### 1.2 海底活动

| 活动 | 说明 |
|------|------|
| 清理海洋垃圾 | 用镰刀清除海底垃圾(塑料/渔网等) |
| 采集海产 | 海带/珊瑚/海葡萄/海胆/珍珠 |
| 海底农场 | 种植海底作物(全年无季节限制) |
| 海洋生物互动 | 海龟/鱼群/鲨鱼(不可战斗) |
| 潮汐池 | 低潮时搜集贝壳/海星 |

### 1.3 海底深度等级

| 深度 | 解锁条件 | 特点 |
|------|------|------|
| 10m | 初始 | 基础海产 |
| 20m | 清理一定量垃圾 | Mermoo/Carapoultry解锁 |
| 30-40m | 继续清理 | Seashroud/Ink Hog解锁 |
| 50m | 深度清理 | Shell Cluck(珍珠产出) |

## 2. 环保清理系统

### 2.1 地面垃圾
- 岛上散落垃圾(可捡可回收)
- 清理=城镇等级经验

### 2.2 海洋垃圾
- 潜水后用镰刀清除
- 包括: 塑料瓶/渔网/绳索/油桶
- 部分垃圾可回收为材料

### 2.3 环保=核心主线
- 清理进度直接驱动城镇等级(E→D→C→B→A→S)
- 每级解锁:
  - 新商店/新种子/新动物/新功能
  - 海底新区域

## 3. 海底动物养殖

| 动物 | 价格 | 深度 | 产物 | 周期 |
|------|:---:|:--:|------|:--:|
| Mermoo(海牛) | 14,000g | 20m | Mermoo Milk | 1天 |
| Carapoultry(海鸡) | 8,000g | 20m | Carapoultry Egg | 2天 |
| Seashroud(海兽) | 18,000g | 40m | Seashroud Fiber | 3天 |
| Ink Hog(墨猪) | 9,000g | 40m | Ink Hog Truffle | 1天 |

- 海底动物需要海底围栏(Underwater Pen)
- 吃海藻(Seaweed)而非牧草

## 4. 博物馆捐赠

| 类别 | 种类数 |
|------|:---:|
| 文物(Artifacts) | 30+ |
| 矿石(Gems/Minerals) | 50+ |
| 鱼类(Fish) | 60+ |
| 昆虫(Bugs) | 40+ |
| 海底生物(Sea Creatures) | 30+ |
| 化石(Fossils) | 20+ |

- 全套捐赠=角色能力永久提升
- 与城镇等级联动

## 5. 设计分析

1. **环保主题=游戏核心驱动力**: 不是附加内容,而是主线
2. **"清理垃圾"=有意义的重复劳动**: 把"捡东西"和城镇发展挂钩,不是纯负累
3. **海底农场=第二经济区**: 全年无季节限制,与陆地农场互补
4. **深度解锁=阶段性目标**: 10m→20m→40m→50m的层级给玩家持续的探索动力
5. **海底动物=差异化畜牧**: 不是"换皮牛鸡",而是有不同产物和机制
