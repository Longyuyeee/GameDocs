# 时间与季节系统 - 天穗之咲稻姬(Sakuna: Of Rice and Ruin)

> 游戏版本: Sakuna: Of Rice and Ruin v1.0+ (PC/Switch/PS4, 2020年)
> 数据来源: sakuna-of-rice-and-ruin.fandom.com, StrategyWiki, GameFAQs

## 1. 季节结构

| 参数 | 设定 |
|------|------|
| 每年 | 4季(春夏秋冬) |
| 每季 | **仅3天** |
| 全年 | 4季×3天=**12天** |
| 水稻种植 | 每年1次(冬末→秋收) |

> 这是所有牧场游戏中**最短的季节周期**——1年=12天意味着资源节奏和决策频率极高。

## 2. 一天结构

| 时段 | 特点 |
|------|------|
| 清晨 | 新一天开始，施肥/农活 |
| 白天 | 探索关卡(最佳时间) |
| 傍晚 | 返回农场/收尾 |
| 夜晚 | 敌人变强+需要光源，通常回家 |
| 吃饭时间 | 晚餐=集体聚餐=剧情触发节点 |

## 3. 季节与农事对应

| 季节 | 天数 | 稻米阶段 | 关键操作 |
|:--:|:--:|------|------|
| 冬 | 1-3 | 整地→育苗 | 冬3整地/选种/播种/预施肥(关键!) |
| 春 | 1-3 | 插秧→分蘖1期 | 春1插秧/追肥 |
| 夏 | 1-3 | 分蘖2期→排水→抽穗 | **夏2排干(最关键!)** |
| 秋 | 1-3 | 成熟→收割→晾晒→脱粒→脱壳 | 秋2-3收割(不下雨)/冬1前入库 |

## 4. 昼夜对探索影响

| 时间 | 敌人强度 | 视野 | 适合行动 |
|------|:------:|:--:|------|
| 白天 | 基准 | 正常 | 探索/战斗 |
| 黄昏 | 轻微增强 | 减弱 | 收尾/返回 |
| 夜晚 | **明显增强** | **需光源** | 高风险/少做 |

## 5. 天气系统

| 天气 | 对稻田影响 | 对探索影响 |
|------|------|------|
| 晴天 | 最佳(正常蒸发+吸收) | 正常 |
| 雨天 | 水位上升/晾晒不能 | 正常(室内关卡) |
| 风暴 | **晾晒灾难**→需祈祷 | 可能受限 |

- **祈祷晴天**: 消耗琥珀→向Tauemon祈祷→晴天保障(收割/晾晒期关键)
- 天气预测: 与Tauemon对话获取

## 6. 满足度/食物系统

| 属性 | 说明 |
|------|------|
| 满足度(Fullness) | 随时间逐渐减少→0时食物Buff失效 |
| 食物Buff | 来自晚餐(5道菜套餐) |
| 满足度(Gusto) | 稻米属性之一，延长食物Buff持续时间 |

## 7. 存档

- 每天睡觉自动存档
- 关卡内检查点

## 8. 设计特点

1. **3天/季=超高频决策**: 每个季节只有3天→每个行动都影响稻米最终品质→没有"拖延"的余地
2. **一年一收=长期规划**: 12天才能见一次收获+属性成长→节奏感完全不同
3. **昼夜探索风险**: 不是"晚上也可以继续"而是"晚上敌人更强+需光源"→鼓励白天探索晚上回家
4. **集体晚餐=社交+剧情**: 不是每天说话的社交而是每天晚饭时的团聚→所有角色围坐一桌吃饭→剧情自然展开
5. **天气=真威胁**: 下雨不只是"不用浇水"，而是"收割期下雨=灾难"→需琥珀祈祷→资源决策
