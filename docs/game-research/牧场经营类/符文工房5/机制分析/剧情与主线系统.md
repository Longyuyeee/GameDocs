# 剧情与主线系统 - 符文工房5(Rune Factory 5)

> 游戏版本: Rune Factory 5 (Switch/PC, 2021-2022年)
> 数据来源: therunefactory.fandom.com, Game8, Wikipedia, RPG Site

## 1. 世界观设定

- **诺拉德王国(Kingdom of Norad)**——RF系列的传统舞台
- **里格巴斯(Rigbarth)**——边境小镇，人类与兽人共存
- **SEED**——维持和平的游骑兵组织(表面上)

## 2. 核心势力

| 势力 | 定位 |
|------|------|
| SEED | 和平维持组织→实际上是Gideon控制世界的工具 |
| 龙族 | Livia(光龙Lumenivia)+Radea(妹)→可变形为人 |
| 地球使者(Earthmate) | 主角身份——与符文和龙族有深层连接 |
| 深渊恐怖(Fathomless Dread) | 被封印的远古邪恶→操控Gideon→最终BOSS |

## 3. 主线剧情结构

### Act 1: 觉醒与加入SEED
- 主角失忆在森林中醒来→救Hina→进入里格巴斯
- 灵魂球测试→加入SEED成为游骑兵
- Whispering Woods击败九尾狐→Misasagi化人→农龙解锁

### Act 2: 调查与冒险
- Belpha Ruins→符文异常调查
- Kelve Lava Caves→Scarlett认可
- Meline Crystal Caverns→帮助失忆神Lucas
- Atohl's End→追捕逃犯

### Act 3: Oswald阴谋与龙族真相
- 捕获Oswald→被神秘黑龙袭击→Oswald逃脱
- Oswald自愿投降→揭露**Gideon才是幕后黑手**
- Livia揭示自己是**龙(Lumenivia)**
- 敌对龙Radea是Livia的妹妹→被Gideon欺骗

### Act 4: 符文危机
- 恢复Livia的龙之力(4个符文点稳定化)
- Radea释放黑雾→吸取居民记忆和符文
- 击败Radea→真相揭露→Gideon想**控制全人类意志**实现世界和平

### Act 5: 最终决战
- Lucas(神)恢复记忆和力量→开启传送门
- 浮空要塞(SEED总部)→Gideon战→深渊恐怖附身Radea
- Calamity's Edge→最终战→深渊恐怖附身主角→全镇人的纽带拯救
- 结局: 和平恢复→主角继续经营/冒险

## 5. 设计特点

1. **18地牢=系列最长主线**: RF4是约12个→RF5扩展到18个
2. **Gideon=灰色反派**: 他的动机是"通过控制所有人意志实现世界和平"——经历了Sechs帝国崩溃的创伤→非纯粹邪恶
3. **深渊恐怖=系列惯例的最终BOSS**: RF4是Ethelberd→RF5是深渊恐怖→"远古邪恶操控当代人"的传统
4. **Lucas=神=可攻略**: 失忆神→恢复力量→可恋爱结婚→与前作Ventuswill(龙/不可攻略)不同
5. **3D化后的叙事**: 全3D过场动画→比RF4的2D更沉浸→但Switch版性能问题影响体验
