# 制作与打造系统 - 符文工房5(Rune Factory 5)

> 游戏版本: Rune Factory 5 (Switch/PC, 2021-2022年)
> 数据来源: therunefactory.fandom.com, Game8, do-it-cat.com

## 1. 四大制作技能

| 技能 | 制作站 | 产物 |
|------|------|------|
| 锻造(Forging) | 锻炉(Forge) | 武器/农具 |

## 3. 工具升级

| 等级 | 所需技能 | 材料 |

- 锻炉从Palmo处购买

## 4. 锻造技能升级策略

| 方法 | 效果 |
|------|------|
| 使用废料(Scrap Metal/芜菁种子)在最高级配方上→故意失败 | **失败也获得技能EXP!** |
| 装备心之吊坠(Heart Pendant) | 技能EXP加成 |
| 重复失败→用产生的废料继续失败 | 自我循环 |
| 食谱面包(Recipe Bread) | 吃→解锁新配方 |

## 5. 设计特点

1. **故意失败=有效升级**: RF系列的传统——"在最高级配方上用废料故意失败"是最快升级法
2. **心之吊坠=必备**: 银吊坠+爱情水晶→锻造→心之吊坠→技能EXP+——关键装备
3. **食谱面包=配方解锁**: 不是等级够了就自动学会→需要吃面包随机解锁
4. **继承RF4的完整制作体系**: 四大制作技能+7厨具→几乎1:1继承RF4
