# 资源采集与科技树 - 史莱姆牧场2(Slime Rancher 2)

> 游戏版本: Slime Rancher 2 v1.0+ (PC/Xbox)
> 数据来源: TheGamer, GitHub Walkthrough, Official Wiki

## 1. 资源类型

| 资源 | 获取方式 | 用途 |
|------|------|------|
| Plort | 史莱姆产出 | 出售(NB)/科技材料/区域门锁 |
| 矿石(Ores) | 吸尘背包资源采集器→岩石节点 | 制作/升级材料 |
| 木材(Wood) | 资源采集器→树木节点 | 制作材料 |
| 辉石矿(Radiant Ore) | 特定区域节点 | 高级升级 |
| 奇异钻石(Strange Diamonds) | 极稀有节点/宝箱 | **量子无人机**(核心) |
| 月露花蜜(Moondew Nectar) | 星光滨(夜间采集) | 振翅史莱姆专属食物 |
| 棱柱Plort(Prisma Plorts) | 棱柱异常天气 | 终局→夜市兑换 |

## 2. 资源采集器

| 升级 | 成本 | 效果 |
|------|:--:|------|
| 基础资源采集器 | 10棉花Plort+450NB | 可采集矿石/木材/辉石矿 |
| 升级版 | 递增 | 更高采集效率 |

## 3. 精炼厂(Refinery)

- 将采集的原材料→精炼→高级材料
- 制作辉光投影器(Radiant Projector)等终局装备

## 4. 蓝图(Gadget)系统

| 类别 | 代表蓝图 | 解锁方式 |
|------|------|------|
| 传送(Warp Tech) | 连接炮/回家传送门 | 探索+材料 |
| 装饰(Decorations) | 各种牧场装饰 | 购买/宝箱 |
| 实用(Utility) | 辉光投影器/资源检测器 | 剧情/Viktor |
| 无人机(Drone) | 量子无人机站 | 奇异钻石+科技 |

## 5. 资源检测器(Patch 1.2)

- 制作→装备→追踪特定制作材料
- 在世界中高亮显示该资源位置

## 6. 科技树结构(简化)

```
探索→扫描新物品→解锁蓝图+百科
  ↓
收集资源+Plort→精炼厂→制作蓝图物品
  ↓
量子无人机(终局)→全自动牧场
  ↓
辉光投影器(终局)→灰色迷宫探索
```
