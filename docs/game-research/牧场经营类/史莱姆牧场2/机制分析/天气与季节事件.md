# 天气与时间系统 - 史莱姆牧场2(Slime Rancher 2)

> 游戏版本: Slime Rancher 2 v1.0+ (PC/Xbox)
> 数据来源: slimerancher.fandom.com, Official Wiki

## 1. 昼夜循环

| 时段 | 光线 | 影响 |
|:--:|:--:|------|
| 白天 | 明亮 | 正常探索/环尾史莱姆变石像(需太阳盾) |
| 黄昏 | 渐暗 | 磷光史莱姆开始出现 |
| 夜晚 | 黑暗 | **磷光/振翅出现**(振翅仅夜间觅食!) |

## 2. 无季节系统

- SR2**无春夏秋冬**
- 替代: **天气事件**驱动变化

## 3. 天气事件

| 天气 | 效果 | 特殊史莱姆 |
|------|------|:--:|
| 晴天(Clear) | 正常 | — |
| 风天(Wind) | — | **德维什(Dervish)出现** |
| 花粉(Pollen) | 花粉飘散 | **藤蔓(Tangle)出现** |
| 棱柱异常(Prisma Disruption) | 彩虹天气 | **棱柱Plort出现**(Boss战前必需!) |
| 雷暴(Thunderstorm) | 雷/雨 | 特殊资源 |

## 4. 天气对经营的影响

| 天气 | 经营策略 |
|------|------|
| 风天 | 去捕捉德维什史莱姆(限定!) |
| 花粉 | 去捕捉藤蔓史莱姆(限定!) |
| 棱柱异常 | **全力收集棱柱Plort**(终局必需!) |
| 晴天 | 常规经营 |

## 5. 设计特点

1. **天气=限定史莱姆窗口**: 不是"天气影响作物生长"而是"天气决定哪些史莱姆出现"
2. **棱柱异常=终局天气**: Boss战前必须利用此天气收集棱柱Plort
3. **无季节=节奏更自由**: 不需要管理季节种植窗口→聚焦于探索+天气事件
4. **昼夜影响史莱姆行为**: 环尾(日光变石像)/磷光+振翅(仅夜间)——史莱姆生态的真实感
