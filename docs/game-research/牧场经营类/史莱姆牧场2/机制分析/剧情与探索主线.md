# 剧情与探索主线 - 史莱姆牧场2(Slime Rancher 2)

> 游戏版本: Slime Rancher 2 v1.0+ (PC/Xbox)
> 数据来源: slimerancher.fandom.com, Mejoress, Official Wiki

## 1. 世界观

- **彩虹岛(Rainbow Island)**——神秘的人工岛屿，充斥着史莱姆和古代科技
- **Beatrix LeBeau**——主角/史莱姆牧场主
- **Gigi**——发送邀请的神秘人物，留下了全岛的无人机信息

## 2. 主线剧情

```
到达彩虹岛→发现温室(主基地)
  ↓
探索→遇到Gordo→喂食→解锁新区域
  ↓
Viktor/Mochi/Ogden通过通讯联系→分析无人机代码
  ↓
发现全岛散布的无人机(Gigi留下的信息)
  ↓
进入灰色迷宫(Grey Labyrinth)
  ↓
到达棱柱核心(Prismacore)→遇到Gigi
  ↓
帮助Gigi稳定核心:
  - 收集棱柱Plort(Prisma Plorts)
  - 在Gigi的夜市(ProntoMart)兑换代币
  - 制作消除器(Nullifier): 20暗影+20双子+20瞌睡+20振翅Plort
  - 敲响5个钟
  - Boss战: Tarr集群+巨大威胁→用水枪消灭
  ↓
结局: 核心稳定→制作人员名单→继续无限经营
```

## 3. 关键NPC通讯

| NPC | 功能 | 发现 |
|------|------|------|
| Viktor | 接收无人机编码信息 | 这些代码来自高级AI |
| Mochi | 分析无人机技术 | 比她知道的所有无人机都先进 |
| Ogden | 怀疑岛屿本质 | 彩虹岛可能是人造的 |
| **Gigi** | **邀请发送者** | 需要帮助稳定棱柱核心 |

## 4. 设计特点

1. **通讯=无NPC实体的叙事**: 所有故事通过远程通讯+无人机日志推进——不需要NPC在场
2. **Gigi=隐形的邀请者**: 整个游戏在追寻她的踪迹→终局才真正见面
3. **Boss战=塔arr集群**: 不是传统Boss——是Tarr失控→需要策略性使用水枪
4. **孤岛探索感**: 与社交型牧场不同——SR2是孤独的探索者体验
