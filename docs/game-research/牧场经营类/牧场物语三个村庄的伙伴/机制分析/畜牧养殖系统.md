## 2. 日常照料系统(Care Tasks)

每日对动物进行照料可提升好感度(Affection)和生产品质。

### 照料项目

| 照料项目 | 操作方式 | 好感度增加 | 备注 |
|----------|----------|-----------|------|
| 刷毛(Brush) | 使用刷子 | +3~5 | 每日1次 |
| 抚摸(Pet) | 空手触摸 | +1~2 | 每日1次 |
| 喂食(Feed) | 放入饲料 | +1 | 每日必需 |
| 放牧(Graze) | 放出户外 | +2 | 晴天1小时以上 |
| 零食(Treats) | 给予零食 | +3~8 | 每日最多1次 |
| 挤奶/剪毛/收集 | 使用工具 | - | 获取生产物 |
| 说话(Talk) | 空手对话 | +1 | 每日1次 |

### 零食系统(Treats)

零食可提升动物的 **副产物等级(+级)**,每日最多给予 **1个零食**。

| 零食名 | 效果 | 价格/材料 |
|--------|------|-----------|
| 白色饼干(White Cookie) | 产量增加(提升产奶量/产毛量) | 200G |
| 软饲料(Soft Treat) | 提升毛/奶品质 | 300G |
| 辣味饲料(Spicy Feed) | 提升副产物等级 | 350G |
| 甜饲料(Sweet Treat) | 提升好感度额外加成 | 蜂蜜+小麦粉 |
| 彩色零食(Colored Treats) | 各种不同属性加成 | 配方多样化 |

## 3. 品质系统

### 好感度(Affection) vs 星级(Star Quality)

| 系统 | 范围 | 提升方式 | 影响 |
|------|------|----------|------|
| 好感度(Affection) | 0~255 | 日常照料 | 提升生产物星级 |
| 副产物等级(Byproduct Level) | E~S(1~1000) | 零食、时间积累 | 解锁+级品质 |

### 星级与售价关系

| 生产物星级 | 价格倍率 | 0.5★牛奶价 | 5★牛奶价 |
|-----------|----------|------------|----------|
| 0.5★ | ×1.00 | 150G | - |
| 1.0★ | ×1.10 | 165G | - |
| ... | ... | ... | - |
| 5.0★ | ×1.90 | 285G | 285G |

### 副产物等级与+级品质

| 等级 | 所需点数 | 效果 |
|------|----------|------|
| E | 0~199 | 基础品质,无+ |
| D | 200~399 | 基础品质,无+ |
| C | 400~599 | 基础品质,无+ |
| B | 600~799 | 基础品质,无+ |
| A | 800~999 | 开始出现+级品质概率 |
| S | 1000 | **最高等级,必定+级品质** |

> **+级品质(如牛奶+)**: 价格为基础品的约1.5倍,是后期主要收入来源之一。

## 4. 放牧系统(Grazing)

| 条件 | 说明 |
|------|------|
| 天气要求 | **晴天**(雨天/台风天不行) |
| 时间要求 | 户外 **1小时以上** 算有效放牧 |
| 好感度收益 | +2点 |
| 其他收益 | 动物心情变好,品质提升概率增加 |
| 冬季放牧 | 不可(冬季禁止放牧) |

**放牧技巧**:
- 早晨6:00开门放出,到中午收回即可满足1小时要求
- 放牧期间动物会吃草,可节省饲料
- 夜间必须收回屋内(防止野生动物袭击,虽然本游戏无此机制)

## 5. 动物节日(Animal Festival)

每月 **13日** 举办动物节日,分不同级别:

| 级别 | 参赛要求 | 奖励 |
|------|----------|------|
| Beginner(初级) | 初次参赛 | 奖杯 + 村庄点数 |
| Veteran(老手) | 获得过初级冠军 | 奖杯 + 村庄点数 |
| Expert(专家) | 获得过老手冠军 | 奖杯 + 村庄点数 |
| Legendary(传说) | 获得过专家冠军 | 最高级奖杯 + 大量点数 |

**评判标准**:
- 动物的好感度(占比40%)
- 动物的星级/副产物等级(占比40%)
- 动物的外观/健康度(占比20%)

| 装饰品 | 效果 | 获取方式 |

## 9. 畜牧产品加工链

```
牛奶(Milk)
  ├─→ 直接出货 (150G)
  ├─→ 牛奶加工机 → 奶油(Cream, 250G)
  ├─→ 奶酪制造机 → 奶酪(Cheese, 450G)
  ├─→ 酸奶制造机 → 酸奶(Yogurt, 350G)
  └─→ 黄油制造机 → 黄油(Butter, 300G)

羊毛(Wool)
  └─→ 纺织机(Loom) → 毛线(Yarn, 500G)
       └─→ 纺织机 → 布料(Cloth, 800G)

鸡蛋(Egg)
  ├─→ 直接出货 (120G)
  └─→ 蛋黄酱制造机 → 蛋黄酱(Mayonnaise, 350G)
```

## 10. 畜牧策略总结

| 阶段 | 推荐动物 | 策略重点 |
|------|----------|----------|
| 初期(第1年) | 鸡、奶牛 | 快速回本,积累基础资金 |
| 中期(第1~2年) | 羊驼、乌骨鸡 | 高利润产品,建造加工机 |
| 后期(2年目+) | 水牛、所有高级品种 | 追求S级品质+级产品 |
| 长期 | 全部满级 | 装饰品叠加+零食优化 |
