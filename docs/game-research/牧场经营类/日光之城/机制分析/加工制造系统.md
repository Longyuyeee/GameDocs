# 加工制造系统 - 日光之城(Sun Haven)

> 游戏版本: Sun Haven v1.4+ (PC/Switch)
> 数据来源: sunhaven.wiki.gg, Fandom Sun Haven Wiki, TheGamer, cozygamingpanda.com

## 1. 概述

Sun Haven 的加工制造系统是**游戏经济的核心引擎**。原料直接出售收益极低，必须经过加工→烹饪才可获取可观利润。系统覆盖**14个烹饪工作站** + **多个制作站**，分布在三个区域。

## 4. 建造/装饰系统

- **城镇建筑修复**: Sun Haven多个建筑需材料+金币修复(狗屋/咖啡馆/理发店等)
- **家具/装饰**: 通过制作桌、家具节拍卖、节日商人和商店获取
- "风水(Feng Shui)"技能(Mining T7): 每个房屋装饰每晚+4金币(上限20/60/100装饰品)
- 谷仓/动物设施: Town Hall购买，每个农场最多4个谷仓

## 6. 设计特点

1. **14个烹饪工作站**: 区域差异化+功能细分，远超同类游戏(星露谷仅有厨房)
2. **三级加工价值链**: 原料→中间食材(面粉/糖/黄油)→成品菜肴，每一级都增值
3. **技能树锁定工作站**: 核心加工站须通过技能树解锁(Jam Maker/Seed Maker/Keg/Alchemy Table等)，将制造与角色build绑定
4. **三区域独立加工站**: 每区域有自己的榨汁机变体，作物不能跨区域加工
5. **制作消耗法力**: 部分精灵/魔法工作站消耗法力使用，引入第二种"费用"
6. **无洒水器但有替代**: 雨云法术+保水土壤+地震翻耕替代自动化灌溉
