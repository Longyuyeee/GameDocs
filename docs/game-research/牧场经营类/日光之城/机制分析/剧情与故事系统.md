# 剧情与故事系统 - 日光之城(Sun Haven)

> 游戏版本: Sun Haven v1.4+ (PC/Switch)
> 数据来源: sunhaven.wiki.gg, Fandom Sun Haven Wiki, Steam Community, TheGamer

## 1. 主线剧情结构

### 1.1 世界观设定

Sun Haven的世界由**三条守护巨龙**掌管:

| 巨龙 | 守护区域 | 属性 | 特征 |
|------|---------|:--:|------|
| Elios(艾利欧斯) | Sun Haven(人类城) | 太阳龙 | 最慷概祝福的守护者，最多人信仰 |
| Nivara(尼瓦拉) | Nel'Vari(精灵村) | 自然龙 | 守护大树与精灵族，与Elios关系最近 |
| Dynus(戴努斯) | Withergate(恶魔城) | 月龙 | 孤僻的守护者，主线主要冲突来源 |

世界长期处于三族(人类/精灵/怪物族)割裂状态，彼此间缺乏信任与交流。玩家作为外来者农夫，被卷入跨种族冲突。

### 1.2 故事架构

```
序幕: 到达Sun Haven
  ↓
第一章: Sun Haven — 获得Elios祝福，发现黑暗蔓延
  ↓
第二章: Nel'Vari — 精灵村和解，结识Nivara
  ↓
第三章: Withergate — 恶魔城潜入，对抗Dynus
  ↓
高潮: 对抗Dynus(战斗或祭坛)
  ↓
结局: 三城和平大使 → 后日谈
```

### 1.3 故事主冲突

**核心冲突**: Dynus(月龙)正在**用黑暗吞噬世界**。他使用**光耀石晶体(Glorite Crystals)**为黑暗魔法供能，理由是"怪物族在黑暗中才能繁荣，需要帝国和家园"。这种"守护者的极端保护主义"将三族推向战争边缘。

玩家作为被Elios祝福的农夫，最终发现:
- Dynus的动机并非纯粹的邪恶
- 怪物族确实长期遭受人类歧视和迫害
- 真正的解决方案不是消灭Dynus，而是促成三族和平共处

### 2.4 后日谈: 和平大使

Dynus被击败/感化后:
- 黑暗从世界撤退(仅保留Withergate周边)
- Lucia任命玩家为**Sun Haven和平大使**
- 玩家负责在三个城市间建立正式外交关系
- Dragon's Meet(巨龙集会): 玩家直接与Elios对话
- 三族从割裂走向共存的进程开启

## 3. 叙事驱动方式

| 机制 | 说明 |
|------|------|
| **主线强制推进(软锁)**: | 解锁新区域需完成前置任务链，但不限时，可任意节奏 |
| **Snaccoon物品锁**: | 各地图节点需特定物品(如10小麦+10土豆)解锁通行 |
| **法力注入机制锁**: | 需Lucia教授法力注入才能推进精灵线 |
| **好感度可选**: | 不影响主线推进，影响个人剧情 |
| **自由触发**: | 主线外大部分内容可自主选择时机触发 |

## 4. 结局分支

- **战斗路线**: 与Dynus正面战斗(90,000HP)，击败后他承认黑暗法术消耗了力量，但认可你的实力
- **祭坛路线**: 完成7+祭坛的供奉，展示诚意与能力，Dynus同样同意撤退黑暗
- **两种路线的结果一致**: Dynus撤退黑暗，但要玩家承诺未来以更强实力再次挑战他
- 无多结局分歧(主线结局确定)，但角色关系和经营成就提供玩家自定目标

## 5. NPC个人剧情

### 5.1 好感度剧情结构

- 每位可攻略NPC有**心色事件(Heart Events)**，在特定好感度阈值触发
- 事件揭示角色背景/性格/秘密/成长
- 事件触发通常需要特定时间/地点条件

### 5.2 关键角色个人线亮点(精选)

| 角色 | 剧情主题 |
|------|---------|
| Lucia | 大法师引导新手农夫的师徒关系，背负守护世界的责任 |
| Claude | 作曲家寻找灵感与自我认同，音乐节是他个人剧情高潮 |
| Nathaniel | 卫兵队长的责任与个人情感的冲突 |
| Darius | 恶魔王子的身份负担，父子关系与王位继承 |
| Donovan | 下水道的流浪冒险者，寻找归属 |
| Kai | 失忆的纳迦，寻找身世之谜(夏季Y1海滩发现) |
| Wesley | 精灵历史学家，三族历史的见证者与记录者 |

## 6. 支线/隐藏剧情

- **布告栏支线**: 每日随机NPC委托任务，提供短篇叙事
- **Snaccoon隐藏**: 野外隐藏Snaccoon需特定物品解锁，奖励物品/区域
- **精灵文献收集**: 图书馆/书籍收集推进世界观理解
- **祭坛收集(Altar Room)**: 供奉特定物品解锁奖励和剧情
- **Dynus祭坛(7+个)**: 每个祭坛提供独特奖励，完成度影响最终Boss策略

## 7. 剧情与经营的关联

| 经营→剧情 | 剧情→经营 |
|-----------|-----------|
| 种植40颗葡萄+法力注入→解锁精灵村 | 解锁精灵村→精灵作物/精灵农场/法力球货币 |
| 收集10光耀石晶体→获得进入Dynus巢穴许可 | 击败Dynus→光耀石工具/装备解锁 |
| 祭坛供奉→剧情推进 | 祭坛奖励→独特装备/属性加成 |
| 布告栏任务→提升NPC好感 | NPC好感→解锁更多剧情事件 |

## 8. 设计特点

1. **三条巨龙守护者体系**: 以龙作为世界观核心，比传统牧场游戏的模糊世界观更具神话质感
2. **灰色道德冲突**: Dynus不是纯粹反派，其"以黑暗保护受压迫种族"的动机具有道德复杂性
3. **战斗/和平双路线**: 最终Boss可选择战斗或非战斗方式，适应不同玩家偏好
4. **三城和平大使结局**: 主线高潮后不是"王子公主幸福生活"，而是赋予玩家外交官角色，为后日谈提供叙事框架
5. **阶段性区域解锁**: 每个新区域伴随完整任务线，非简单的"过桥即到"
6. **Snaccoon门控**: 可爱化的门控机制，用收集/制作驱动探索节奏
7. **没有多结局但有多路径**: 主线结局确定但达成方式可选，角色关系和经营成就是玩家自定的"第二结局"
