# 时间季节与节日系统 - 米斯特里亚的田野(Fields of Mistria)

> 游戏版本: Fields of Mistria (PC, Early Access 2024-至今, 2025年7月更新)
> 数据来源: fieldsofmistria.com, Game8, GameRant

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 每天长度 | **~12现实分钟**(比星露谷短) |
| 每季节 | 28天 |
| 每年 | 4季×28天=112天 |

## 2. 季节

| 季节 | 特色作物 | 关键节日 |
|:--:|------|------|
| 春 | 芜菁/土豆/草莓/花椰菜 | **樱花节** |
| 夏 | 番茄/玉米/菠萝/西瓜 | **流星节**(夏28) |
| 秋 | 茄子/南瓜/红薯 | 丰收节(计划)/万圣节(计划) |
| 冬 | 冬季专属 | **动物节**/极光夜 |

## 5. 体力系统

| 要素 | 说明 |
|------|------|
| 体力(Stamina) | 工具/战斗消耗 |
| 食物恢复 | 料理恢复体力 |
| 体力上限 | 通过技能升级/力量果实提升(推测) |

## 6. 设计特点

1. **12分钟/天=快节奏**: 比星露谷约14分钟更短→更紧凑
2. **EA节日仍在添加**: 目前4个→1.0计划10+个
3. **流星节=社交关键节点**: 4♥+参加→8心事件偏向浪漫→影响社交走向
