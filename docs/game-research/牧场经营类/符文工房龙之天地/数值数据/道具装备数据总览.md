# 道具装备数据总览 - 符文工房 龙之天地
![神器装备](https://runefactory-gaz.com/img/system/ss_treasure.jpg)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

## 一、农具系统

本作取消了传统RF的独立农具（锄头、水壶等），统一整合为**大地手杖(Terra Tiller)**和**六大神器**的种植功能。

### 大地手杖(Terra Tiller)

| 项目 | 内容 |
|:----|:----|
| **获取方式** | 主线任务"灾变之后(After the Disaster)"完成后获得 |
| **核心功能** | 村庄建造模式：清除障碍物、铺设田地/水田/道路、放置建筑、安装装饰物 |
| **召唤Woolby** | 在建造模式下召唤Woolby制作地形砖块 |
| **是否可升级** | **不可升级**(无技能树) |
| **RP消耗** | 基础建造和种植操作**不消耗RP** |

## 二、六大神器(Sacred Treasures)数据

本作的核心装备系统——六大神器兼具**战斗**和**农业/功能**双重用途，每件神器拥有独立的技能树。

### 2.1 神器总览

| 神器 | 获取章节 | 战斗功能 | 农业/功能 | 属性 |
|:---:|:--------:|:--------:|:--------:|:---:|
| **鼓(Drum)** | 春之里初期 | 范围眩晕、HP恢复 | 催熟作物、复活枯萎 | 无属性/物理 |
| **剑(Sword)** | 到达夏之里 | 火焰剑斩、火箭 | 采种(高等级种子概率) | 火属性 |
| **扇(Fan)** | 到达秋之里 | 追踪旋风(风属性) | 批量直线收获 | 风属性 |
| **阳伞(Parasol)** | 到达冬之里 | 水属性范围攻击、滑翔 | 大范围灌溉 | 水属性 |
| **鬼面(Oni Mask)** | 星界神殿完成后 | 暗属性近战连击、破大石 | 无种植功能(摧毁巨石) | 暗属性 |
| **锡杖(Staff)** | 进入天界后 | 光属性追踪光弹、治愈debuff | 无种植功能(揭示隐藏路径) | 光属性 |

> **注意**：梅枝(Plum Branch)和大地手杖(Terra Tiller)无技能树，不计入六大神器。实际可升级神器为**6件**。

### 2.2 神器升级路径

每件神器的升级通过**技能树(Skill Tree)**系统实现：

1. **使用神器获得经验**（战斗或农业使用累积XP）
2. **在技能树界面消耗XP解锁技能节点**
3. **推进主线**解锁更高阶技能节点

### 2.3 RP消耗系统

| 行为 | RP消耗 |
|:----|:-----:|
| 基础种植（播种/浇水/收获） | **0**（不消耗） |
| 神器战斗技能 | 消耗RP |
| 神器农用技能（鼓之舞/剑之舞等） | 消耗RP |
| 闪避(Instant Dodge) | 消耗少量RP |

> **设计亮点**：基础农业操作完全免费，仅神器特殊技能消耗RP，大幅降低了传统RF中"每天浇水消耗RP"的挫败感。

### 3.3 符咒自动追踪机制

符咒作为本作的远程魔法武器，具有以下特性：
- **自动追踪**：发射的魔法弹自动追踪附近敌人
- **充能攻击**：同时发射多枚符咒，覆盖多个目标
- **终极攻击(Ultimate Attack)**：4连击后的强力终结技
- **弱点穿透(Weakness Magnifier)**：克制属性伤害加成，可继承至其他武器
- **魔防穿透(M.DEF Piercer)**：部分无视敌方魔法防御，可继承至其他武器
