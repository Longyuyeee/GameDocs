## 三、配方获取系统

### 3.1 蛙石(Frog Statues) — 主要来源
隐藏在全地图各处的蛙石(~200+个),调查即可获得配方:
- 春季高原: 24+配方
- 夏季区域: 36+配方
- 秋季区域: 30+配方
- 冬季区域: 36+配方
- 天空岛: 30+配方
- 故事迷宫+冥界: 40+配方

### 3.2 路边神社(Little Red Shrines) — 高价值配方
分布在各区域的路边小神社,通常含武器/高级防具配方:
- 春季: 防麻痹耳环, 驱魔耳环, 动物耳头带
- 夏季: 消防标准, 竹永缎带
- 秋季: 长弓, 银之刃, 流浪武者肩甲, 岚之札
- 冬季: 角弓, 白金剑, 激流之札, 石裂刀
- 天空岛: 觉醒耳环, 猎鹰弓, 春之戒, 红宝石肩甲
- 冥界: 圣光符, 圣岚符, 神鸣符

### 3.5 共同烹饪(Combine Cooking)
与特定角色共同烹饪可解锁特殊料理:
- Pilika + 巧克力麻薯 = 熊麻薯
- Hina + 黄油卷 = 化石面包
- Kanata + 蛋包饭 = 天上飨宴
- Matsuri + 烤饭团 = 最强烤饭团

### 3.6 其他来源
- **钓果报告(Watarase)**: 报告Pescapedia钓果奖励
- **弓靶(Yoichi)**: 破坏全部靶子→与一之弓配方
- **地藏像(Jizo)**: 扫除30个地藏+睡觉→清酒/桃子蜜/口嚼酒
- **剧情推进**: 部分主线节点自动解锁
- **甜点贤者**: 提交特定甜点解锁更多甜品配方
