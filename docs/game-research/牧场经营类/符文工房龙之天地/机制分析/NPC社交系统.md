# NPC社交系统
![龙之天地角色](https://runefactory-gaz.com/img/character/chara_main.png)

> 游戏版本: 符文工房 龙之天地(Rune Factory: Guardians of Azuma) (Switch/Switch2/Steam, 2025年6月5日)

---

## 一、系统总览

本作的社交系统在系列基础上大幅扩展，提供 **16名可攻略角色**(系列最多) + 同性别婚姻 + 全新的**世界转移法(跨世界线系统)**。社交与村庄发展系统深度绑定，NPC满意度直接影响村庄经济。

---

## 二、牵绊等级(Bonds Level)系统

### 2.1 等级与解锁内容

| 牵绊等级 | 解锁内容 |
|---------|---------|
| Lv1 | 可邀请加入队伍(Party) |
| Lv2 | 解锁日常对话选项 |
| Lv3 | 解锁牵绊任务(Bonding Quest)第一段(蓝心) |
| Lv4 | 新对话选项、约会邀请 |
| Lv5 | 解锁第二段牵绊任务 |
| Lv6 | 可**共同烹饪**，解锁特殊料理变体 |
| Lv7 | **可告白**，确认关系后成为恋人 |
| Lv8-Lv9 | 约会活动解锁(牵手、拥抱) |
| Lv10 | **最高牵绊等级**，可求婚 |
| Lv10+(1.1.0) | 更新后牵绊等级上限提升至**100** |

### 2.2 牵绊值提升方式

| 方式 | 提升量 | 每日上限 |
|------|-------|---------|
| 日常对话 | 少量 | 1次/人 |
| 赠送礼物 | 中-高(视喜好) | 1次/人 |
| 约会(Hangout) | **最高效** | 1次/天 |
| 共同战斗 | 少量 | 无明确限制 |
| 完成委托请求 | 中 | 随任务 |
| 共同烹饪(Lv6+) | 中 | 1次/天 |

> **重要改动**：本作取消了送错礼物降低好感的负反馈机制，玩家可以自由尝试各种礼物而无需担心降好感。

### 2.3 牵绊任务(Bonding Quests)

| 任务阶段 | 解锁等级 | 任务类型 |
|---------|---------|---------|
| 蓝心任务(了解你) | Lv3 | 了解NPC背景和个人故事 |
| 红心任务(恋爱) | Lv7+ | 恋爱相关剧情事件 |

### 3.2 告白到结婚全流程

```
Step 1: 牵绊等级 Lv7
  ├─ 完成所有蓝心牵绊任务
  ├─ 选择告白选项 "我爱你"
  ├─ 对方要求次日见面
  └─ 确认后成为恋人

Step 2: 牵绊等级 Lv10
  ├─ 完成所有红心牵绊任务
  └─ 达到最高牵绊等级

Step 3: 求婚
  ├─ NPC Woolby 叫醒玩家，提示命运纺织神社(Fateweaver Shrine)
  ├─ 在村庄神社后方种植梅树(Plum Tree)
  │   ├─ Subaru用：红梅(Red Plum)
  │   └─ Kaguya用：白梅(White Plum)
  ├─ 制作"神灵酒(Divine Spirits)"：1水稻 + 1紫藤(Wisteria)
  ├─ 等待梅树生长，收获树枝
  └─ 邀请伴侣前往命运纺织神社

Step 4: 婚后
  ├─ 伴侣与玩家同住
  ├─ 每日赠送礼物/料理
  └─ 其他恋爱关系自动解除
```

### 3.3 同性别婚姻

| 项目 | 内容 |
|------|------|
| 支持程度 | **发售即支持**，非后续更新追加 |
| 范围 | 所有16名可攻略角色均可，**不限制主角性别** |
| 孩子 | 同性别伴侣也可养育孩子(通过特定方式) |

### 3.4 育儿系统

| 项目 | 内容 |
|------|------|
| 触发时机 | 婚后约**5天**，伴侣问是否要孩子 |
| 同性伴侣 | 获得一个**萝卜(Turnip)**，种在神社后方 |
| 孩子出生 | **祈祷后5天**出生，时间跳跃**5年** |
| 自定义 | 选择孩子的**性别、性格、名字** |
| 外貌遗传 | 第1个孩子继承主角发型+伴侣瞳色 |
| 第2个孩子 | 继承伴侣发型+主角瞳色 |
| 最多孩子数 | **2个** |
| 孩子功能 | 可成为队伍成员，有自己的喜好/厌恶 |
| 特殊解锁 | 第1个孩子出生解锁**通关后BOSS和迷宫** |

## 四、世界转移法(跨世界线系统)

### 4.1 系统概述

"Rewoven Fates"是本作最独特的社交系统创新，允许玩家在**单存档**中体验多条恋爱路线。

| 项目 | 内容 |
|------|------|
| 解锁条件 | 与任一角色结婚后约10天 |
| 触发方式 | 命运纺织神社出现**发光球体** |
| 功能 | 进入**平行世界线(Alternate Timeline)** |

### 4.2 跨世界线机制

| 特性 | 说明 |
|------|------|
| 当前婚姻 | 在新世界线中**取消** |
| 牵绊等级 | **保留**(无需重新培养) |
| 恋爱情感 | 重置，可与其他角色恋爱结婚 |
| 原世界线 | **可随时切换返回** |
| 存档上限 | 最多保存**30条世界线** |
| 经济/物品 | 跨世界线共享？(待验证) |

> 这个系统实际上允许玩家在**一个存档文件**中体验所有的恋爱故事线，无需开新存档。

## 五、队伍系统(Party System)

### 5.1 编队规则

| 项目 | 内容 |
|------|------|
| 解锁条件 | 牵绊等级Lv1即可邀请 |
| 队伍规模 | 最多**6人**(3主力 + 3后备) |
| 编成对象 | 村民NPC + 驯服的魔物 |
| 编队方式 | Party菜单中设置 |

### 5.2 战斗AI定位

| 角色类型 | 行为模式 |
|---------|---------|
| **攻击型(Attack)** | 积极攻击敌人，高伤害输出 |
| **防御型(Defense)** | 保护玩家，吸引敌人仇恨 |
| **支持型(Support)** | 提供增益效果、状态辅助 |
| **恢复型(Heal)** | 回复队伍HP |
| **扰乱型(Disrupt)** | 干扰敌人行动，降低敌人能力 |

### 5.3 队伍切换

战斗中可随时在前排3名队员间切换控制，后备队员会自动替换战败队员。

## 七、NPC与村庄的关联

### 7.1 村民特质

| 特质 | 效果 | 适用委任 |
|------|------|---------|
| 工匠(Journeyman Smith) | 经营铁匠铺 | 锻造 |
| 交易(Trading) | 提升店铺收入 | 商店经营 |
| 烹饪(Cooking) | 提升料理品质 | 餐馆经营 |
| 农业(Farming) | 高效种植 | 农田委任 |
| 畜牧(Livestock) | 高效照顾魔物 | 牧场管理 |

### 7.2 村民满意度

| 因素 | 影响 |
|------|------|
| 建筑/装饰 | 提高风景评分，提升满意度 |
| 工资支付 | 资金不足时满意度下降 |
| 村庄等级 | 高等级提高满意度上限 |
| 玩家互动 | 日常交流维持满意度 |

> 满意度过低的村民可能**离开村庄**，因此需要持续关注村庄建设。

### 7.3 结婚后状态

- 结婚后伴侣与玩家同住
- 每日赠送礼物/料理
- 其他恋爱关系自动解除
- 伴侣仍可参与村庄工作

## 八、社交策略建议

### 快速提升牵绊
1. **每日约会(Hangout)** 是最有效的方式
2. 先送出**樱花(Cherry Blossom)** 建立好感基础
3. 牵绊Lv6后通过**共同烹饪**额外提升

### 路线规划
1. 前期广泛交流(牵绊Lv3解锁牵绊任务)
2. 选取1-2名目标集中培养(Lv7告白)
3. 婚后通过世界转移法体验其他路线
