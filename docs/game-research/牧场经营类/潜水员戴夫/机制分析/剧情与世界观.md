# 剧情与世界观 - 潜水员戴夫(Dave the Diver)

> 游戏版本: Dave the Diver v1.0+ (PC/Switch, 2023年)
> 数据来源: dave-the-diver.fandom.com, IGN Walkthrough, Game8

## 1. 世界观

- **蓝洞(Blue Hole)**——神秘的海底洞穴，地形和生物每日变化
- **番长寿司(Bancho Sushi)**——Dave和Bancho经营的寿司店
- **海人(Sea People)**——海底的神秘文明

## 2. 主要角色

| 角色 | 身份 | 功能 |
|------|------|------|
| Dave(戴夫) | 潜水员/寿司店主 | 主角——白天潜水晚上经营 |
| Bancho(番长) | 寿司师傅 | 厨房核心/剧情推进 |
| Cobra(眼镜蛇) | 武器商人/朋友 | 提供情报/武器 |
| Dr Bacon(培根博士) | 科学家 | 研究/科技 |
| Duff | 武器制作师 | 分析→制作→升级武器 |
| 海人(Sea People) | 海底文明 | 剧情核心——第1章"海人踪迹" |

## 3. 主线章节

| 章节 | 内容 |
|:--:|------|
| Ch.1 | 海人踪迹(Traces of the Sea People) |
| Ch.2+ | 蓝洞深层探索/海人文明揭示 |
| 后续 | Boss战/VIP客人/活动事件 |

## 4. 主题

- **海洋保护**: 可持续捕鱼(完美捕获)vs过度捕捞
- **寿司文化**: 日本寿司店经营的真实还原
- **海洋神秘**: 蓝洞每日变化→每次潜水都是新体验

## 5. 设计特点

1. **双主角分工**: Dave(潜水)+Bancho(料理)——不是"一个人做所有事"
2. **海人=海底文明叙事**: 类似亚特兰蒂斯——为潜水探索提供剧情驱动力
3. **轻松幽默的基调**: 不是硬核生存——而是"在海边开寿司店的冒险"
