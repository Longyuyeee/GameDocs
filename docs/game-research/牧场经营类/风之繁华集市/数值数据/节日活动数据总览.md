## 二、主要节日详解

### 2.1 鲜花节（春8~12日）

| 项目 | 内容 |
|:----|:-----|
| 活动时间 | 春8日~12日送花期间，春11日中午12点在时钟广场 |
| 活动内容 | 送花给予心仪的NPC，比拼谁获得的花朵最多 |
| 推荐送花 | 月泪草、玩具花、魔术蓝草、魔术红草等 |
| 获胜条件 | 送约50朵高品质花可稳胜 |
| 获胜奖励 | 大幅提升全员好感度 |
| 提示 | 春季提前种植大量花卉囤货 |

### 2.2 春之感恩祭/甜心节（春14日）

| 项目 | 内容 |
|:----|:-----|
| 活动内容 | 送巧克力料理(曲奇/蛋糕)给心仪对象 |
| 推荐礼物 | 巧克力曲奇、巧克力蛋糕、布丁 |
| 好感度效果 | 大幅提升 |
| 提示 | 提前准备可可豆制作巧克力 |

### 2.3 茶会（春30日）

| 项目 | 内容 |
|:----|:-----|
| 活动时间 | 中午12点/时钟广场 |
| 必备物品 | 茶叶或罐装红茶 |
| 活动效果 | 全体参与NPC好感度提升 |
| 提示 | 提前种茶树收春茶叶 |

### 2.4 宠物评选赛（夏3日/冬）

| 项目 | 内容 |
|:----|:-----|
| 参赛条件 | 拥有猫或狗，且经过训练 |
| 评分标准 | 训练度(亲密度)、品种 |
| 冠军奖励 | 宠物品牌认证 + 奖杯 |

### 2.5 动物评选赛（夏/秋/冬）

| 项目 | 内容 |
|:----|:-----|
| 参赛条件 | 拥有鸡/牛/羊/羊驼 |
| 评分标准 | 亲密度+健康状态+品种 |
| 冠军奖励 | 动物品牌认证 + 概率产出特殊产品 |
| 金牌奖励 | 对应产品获得品牌认证(售价×3) |

### 2.6 赛马大会（夏16日/冬）

| 项目 | 内容 |
|:----|:-----|
| 参赛条件 | 第二年获得马后，训练度☆3~4 |
| 比赛规则 | 速度竞赛 |
| 冠军条件 | 约27秒成绩 |
| 训练方法 | 劳埃德家左边训练场每日训练 |
| 冠军奖励 | 赛马品牌认证 + 奖杯 |

### 2.7 烹饪节（秋15~16日）

| 项目 | 内容 |
|:----|:-----|
| 主菜分项(秋15日) | 需带主食类料理参赛，约4☆品质可夺冠 |
| 甜品分项(秋16日) | 需带甜品类料理参赛 |
| 单项冠军奖励 | 主菜→松露；甜品→罐装蓝莓红茶 |
| **两项全胜奖励** | **粉红钻石**(非常贵重！) |

### 2.8 葡萄酒祭/果汁节（秋21日）

| 项目 | 内容 |
|:----|:-----|
| 活动内容 | 带葡萄酒类料理参加品鉴 |
| 推荐 | 提前用葡萄在红色风车酿酒 |

### 2.9 南瓜节/万圣节（秋28日）

| 项目 | 内容 |
|:----|:-----|
| 活动内容 | 小孩子来玩家门口要点心 |
| 推荐准备 | 用巧克力做的料理、糖果 |

### 2.10 星夜节（冬25日）

| 项目 | 内容 |
|:----|:-----|
| 活动内容 | 与约会对象一起度过星夜 |
| 条件 | 提前约好对象 |
| 好感度效果 | 大幅提升 |

### 2.11 跨年倒计时（冬31日）

| 项目 | 内容 |
|:----|:-----|
| 活动时间 | 晚上20点/时钟广场 |
| 活动内容 | 与全镇居民一起迎接新年 |

### 2.12 农作物评选赛（每年多次）

| 项目 | 内容 |
|:----|:-----|
| 参赛条件 | 持有高品质作物(推荐☆3以上) |
| 评分标准 | 星星品质 + 稀有品种 |
| 冠军奖励 | 品牌认证(售价×3) + 奖杯 |

## 三、每周六集市

### 3.1 集市基本规则

| 项目 | 内容 |
|:----|:-----|
| 开放时间 | 每周六上午10点~晚上19点 |
| 开放地点 | 小镇广场集市区域 |
| 核心机制 | 摆摊售卖自产商品 |
| 收入特点 | **本作唯一的收入来源（无出货箱）** |
| 营业模式 | 摆摊→摇铃铛招客→顾客前来购买 |
| 上半场 | 10:00~15:00 |
| 下半场 | 15:00~19:00 |

### 3.2 集市等级（Bazaar Rank 1~7）

| 等级 | 名称 | 摊位格子 | 主要解锁内容 |
|:---:|:----|:--------:|:------------|
| Lv.1 | 微風集市 | 3格 | 基础摆摊 |
| Lv.2 | 成长集市 | 4格 | 摊位装饰系统、新店铺进驻 |
| Lv.3 | 热闹集市 | 5格 | 蓝色风车解锁、包装系统 |
| Lv.4 | 繁荣集市 | 6格 | 黄色风车解锁、耀珠商店 |
| Lv.5 | 盛大集市 | 7格 | 摊位格子扩展、精品店 |
| Lv.6 | 辉煌集市 | 8格 | 最高级店铺 |
| Lv.7 | 世界第一大集市 | 9格(满) | 全部功能解锁、游戏通关 |

### 3.3 集市等级升级条件

| 目标等级 | 营业额要求(约) | 其它条件 |
|:--------:|:-------------:|:---------|
| Lv.1→2 | 累计10,000~20,000G | 无 |
| Lv.2→3 | 单周20,000~30,000G | 无 |
| Lv.3→4 | 单周40,000~50,000G | 完成特定委托 |
| Lv.4→5 | 单周60,000~80,000G | 累计加工100件 |
| Lv.5→6 | 单周100,000G+ | 品牌认证数量要求 |
| Lv.6→7 | 单周150,000G+ | 全店铺解锁+通关条件 |

### 3.4 新店铺进驻

| 集市等级 | 新进驻店铺 | 功能 |
|:--------:|:-----------|:----|
| Lv.1 | 基础店铺(种子、食品) | 基础物资购买 |
| Lv.2 | 装饰店 | 摊位装饰(桌布、招牌) |
| Lv.3 | 包装店 | 送礼包装(200G/个) |
| Lv.4 | 耀珠商店(弗乔) | 出售六色耀珠 |
| Lv.5 | 精品店 | 高级商品出售 |
| Lv.6 | 最高级店铺 | 稀有道具 |
| Lv.7 | 隐藏店铺 | 终极商品 |
