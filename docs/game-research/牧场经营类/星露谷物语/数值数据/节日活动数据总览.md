## 2. 春季节日详情

### 2.1 复活节 (Egg Festival) — 春13

| 项目 | 内容 |
|------|------|
| 时间 | 9:00-14:00 |
| 地点 | 鹈鹕镇广场 |
| 核心活动 | **彩蛋寻宝比赛 (Egg Hunt)** |

**彩蛋寻宝：**
- 与阿比盖尔比赛寻找镇广场上的彩色鸡蛋
- **获胜条件：** 找到**9个或以上**彩蛋即可获胜
- 获胜奖励：**草帽 (Straw Hat)**
- 阿比盖尔通常会找到8个，所以9个即可稳赢

**节日商店：**

| 商品 | 价格 | 说明 |
|------|:----:|------|
| 草莓种子 (Strawberry Seeds) | 100g/个 | **春季唯一获取途径**，春13购买后当年可再收1-2次 |
| 草帽 | 0g | 获胜奖励 |

> **草莓种子策略：** 复活节购买的草莓种子，种下后每4天收获一次，春28前可收获2-3次。若提前用生长激素，可收获3次。

### 2.2 沙漠节 (Desert Festival) — 春15-17 *(1.6新增)*

| 项目 | 内容 |
|------|------|
| 时间 | 全天(沙漠区域) |
| 地点 | 卡利科沙漠 |
| 核心货币 | **卡利科彩蛋 (Calico Egg)** |

**机制：**
- 在沙漠各处寻找/挖掘/购物获得卡利科彩蛋
- 卡利科彩蛋可在节日商店兑换特殊物品

**每天出席的村民：**

| 天数 | 出席的村民 |
|:---:|-----------|
| 春15 | 阿比盖尔、艾利欧特、艾米丽、海莉、莉亚、玛鲁、潘妮、山姆 |
| 春16 | 亚历克斯、艾利欧特、哈维、肯特、玛鲁、潘妮、山姆、塞巴斯蒂安、谢恩 |
| 春17 | 阿比盖尔、亚历克斯、艾米丽、海莉、哈维、莉亚、塞巴斯蒂安、谢恩 |

**卡利科彩蛋商店兑换表：**

| 物品 | 所需彩蛋数 | 说明 |
|------|:---------:|------|
| 阿比盖尔的弓 (Abigail's Bow) | 75 | 发型装饰 |
| 魔法发胶 (Magic Hair Gel) | 30 | 发型 |
| 三角帽 (Tricorne Hat) | 20 | 帽子 |
| 沙漠节横幅 | 5 | 装饰品 |
| 各种稀有物品 | 20-100 | 随机刷新 |

**沙漠节雕像效果：**
- 沙漠中散布雕像，靠近后获得各种临时Buff
- 包括：+攻击、+防御、+运气、+速度等

### 2.3 花舞节 (Flower Dance) — 春24

| 项目 | 内容 |
|------|------|
| 时间 | 9:00-14:00 |
| 地点 | 煤灰森林 |
| 核心活动 | **与NPC跳舞** (需达到4心好感) |

**舞蹈配对：**

## 3. 夏季节日详情

### 3.1 夏威夷宴会 (Luau) — 夏11

| 项目 | 内容 |
|------|------|
| 时间 | 9:00-14:00 |
| 地点 | 海滩 |
| 核心活动 | **往汤里放食材 (Potluck Soup)** |

**汤评分机制：**

| 放入的食材 | 评分结果 | 好感影响 |
|-----------|---------|---------|
| **金星/铱星品质物品** | 最佳 | **全村民+120好感** |
| 普通品质物品 | 较好 | 全村民+60好感 |
| 垃圾/树枝/石头 | 最差 | **全村民-100好感** |
| 什么都不放 | 中等 | 无变化 |

> **推荐食材：** 金星品质的蔬菜/鱼/采集品，或铱星松露、铱星紫蘑菇等。

### 3.2 鳟鱼大赛 (Trout Derby) — 夏20-21 *(1.6新增)*

| 项目 | 内容 |
|------|------|
| 时间 | 6:10-2:00(两天) |
| 地点 | 煤灰森林 |
| 核心活动 | **钓虹鳟鱼获取金色标签** |

**机制：**
- 在煤灰森林河流钓鱼，钓到虹鳟鱼(Rainbow Trout)时有概率获得**金色标签(Golden Tag)**
- 用金色标签到节日商店兑换奖励

**金色标签商店：**

| 物品 | 所需金色标签 | 说明 |
|------|:-----------:|------|
| 渔夫帽 (Fisher's Hat) | 2 | 帽子 |
| 虹鳟鱼壁挂 (Mounted Trout) | 3 | 家具(墙饰) |
| 其他随机钓鱼物品 | 1-5 | 视库存而定 |

## 4. 秋季节日详情

### 4.1 星露谷展览会 (Stardew Valley Fair) — 秋16

| 项目 | 内容 |
|------|------|
| 时间 | 9:00-15:00 |
| 地点 | 鹈鹕镇广场 |
| 核心活动 | **农庄展览 (Grange Display) + 小游戏** |
| 核心货币 | **星星币 (Star Tokens)** |

**农庄展览评分：**

| 评分规则 | 说明 |
|---------|------|
| 展示物品 | 放入最多9件物品 |
| 评分范围 | 0-125分 |
| 重复物品 | 放入9件相同物品最高得9分 |
| 不同类型物品 | 每种不同类别有额外加分 |
| 品质加成 | 铱星>金星>银星>普通 |

**评分公式（简化）：**
```
总分 = 物品基础分之和 + 类别数×5 + 数量分 + 品质分
```

| 分数区间 | 星星币奖励 |
|:-------:|:---------:|
| 90-125分 | 1,000星星币 |
| 75-89分 | 500星星币 |
| 60-74分 | 250星星币 |
| 30-59分 | 100星星币 |
| 0-29分 | 50星星币 |

> **1,000星星币策略：** 展示9种不同类别的高品质物品，如铱星蔬菜、铱星水果、铱星鱼、铱星矿物、工匠品、烹饪品、采集品、动物产品、宝石。

**星星币商店：**

| 商品 | 价格 | 说明 |
|:----:|:----:|------|
| **星之果实 (Stardrop)** | **2,000** | **永久+34体力(仅1次)** |
| 软呢帽 (Fedora) | 500 | 帽子 |
| 稻草人#1 (Rarecrow #1) | 800 | 稀有稻草人 |
| 各种家具/装饰 | 50-800 | 随机 |
| 节日装饰 | 100-500 | 季节性 |

**小游戏：**

| 小游戏 | 玩法 | 星星币奖励 |
|-------|------|:---------:|
| 钓鱼游戏 | 钓指定鱼类 | 5-15/次 |
| 力量测试 | 打靶测力 | 5-10/次 |
| 转盘赌 | 绿色/橙色概率 | 视下注而定 |

> **转盘赌概率：** 绿色胜率约75%，橙色约25%。稳赢策略：**一直押绿色**。

### 4.2 亡灵节 (Spirit's Eve) — 秋27

| 项目 | 内容 |
|------|------|
| 时间 | 22:00-23:50 |

**迷宫流程：**
1. 进入广场南侧的迷宫

**节日商店：**

### 5.4 冬日星盛宴 (Feast of the Winter Star) — 冬25

| 项目 | 内容 |
|------|------|
| 时间 | 9:00-14:00 |
| 地点 | 鹈鹕镇广场 |
| 核心活动 | **秘密圣诞老人礼物交换** |

- 提前一周收到系统通知，告诉你送礼对象
- 需要准备一份礼物送给指定的NPC

**礼物品质规则：**

| 送礼品质 | 好感变化 | 回礼 |

| 喜欢礼物 | 对方+225好感(45×5倍) | 中等回礼 |
| 普通/不喜欢 | 少量好感或无变化 | 普通回礼 |

| 商品 | 价格 | 说明 |
|------|:----:|------|
