# 角色属性与战斗数据总览 - 符文工房4 豪华版
![RF4 战斗](https://www.runefactory4.com/img/system/ss_battle01.jpg)

> 游戏版本: 符文工房4 豪华版(Rune Factory 4 Special) (Switch 2019年 / PC 2021年)

## 一、基础属性

| 属性 | 含义 | 初始值 | 成长方式 |
|------|------|:---:|------|
| HP(生命值) | 生存值 | 100 | 防御技能每级+少量 |
| RP(符文点数) | 行动资源 | ~100 | 几乎所有技能升级均+RP |
| STR(力量) | 物理攻击力 | ~10 | 武器技能升级 |
| INT(智力) | 魔法攻击力 | ~10 | 魔法/抗性技能升级 |
| VIT(体力) | 物理防御力 | ~10 | 防御/抗性技能升级 |
| M.Def(魔法防御) | 魔法防御力 | ~10 | 魔法技能升级 |

## 二、武器攻击力表(顶级)

| 武器 | 类型 | 基础攻击力 | 锻造材料(核心) |
|------|:---:|:--------:|------|
| 符文传说(Rune Legend) | 短剑 | 最高级 | 传说材料 |
| 荣耀(Glory) | 长剑 | 最高级 | 传说材料 |
| 盖·博尔格(Gae Bolg) | 枪 | 最高级 | 传说材料 |
| 巨人锤(Gigant Hammer) | 锤 | 最高级 | 传说材料 |
| 符文之刃(Rune Edge) | 双剑 | 最高级 | 传说材料 |
| 芬里尔(Fenrir) | 拳套 | 最高级 | 传说材料 |

> 注: 精确攻击力数值需确认。RF4武器属性可通过材料继承进一步提升。

## 三、材料属性继承系数

| 材料类型 | 继承属性 | 继承比例 |
|---------|---------|:-----:|
| 矿石类(Iron/Silver/Gold/Platinum) | ATK | 100% |
| 魔法水晶(Magic Crystal) | M.ATK | 100% |
| 鳞片/甲壳(Scale/Shell) | DEF | 100% |
| 爪/牙(Claw/Fang) | 暴击率 | 50% |
| 布料/毛皮(Cloth/Fur) | M.DEF | 100% |

## 四、元素相克

```
火(Fire) > 冰/植物
水(Water) > 火/岩石
土(Earth) > 风/飞行
风(Wind) > 飞行/昆虫
光(Light) ↔ 暗(Dark) (互克)
```

## 五、难度设定

| 难度 | 敌人HP | 敌人攻击力 |
|------|:----:|:-------:|
| Easy(简单) | ×0.8 | ×0.8 |
| Normal(普通) | ×1.0 | ×1.0 |
| Hard(困难) | ×1.5 | ×1.2 |
