## 3. 恋爱与婚姻

### 3.1 约会
- LP ≥7(700+)可邀请候补约会
- 按L/R键选择"一起去约会吧"
- 带往候补喜欢的景点

### 3.2 结婚条件
1. LP ≥10(1000+)
2. 购买双人床(Double Bed): 15,000G + 80木材
3. 锻造婚戒(Engagement Ring): 1红宝石+2银, 锻造Lv20
4. **观看候补对应的全部城镇事件(Town Events)**(RF4独有——最关键的隐藏条件)
5. 主线至少完成第二幕

### 3.3 结婚物品
- 约会信物: 告白吊坠(Confession Pendant)
- 结婚信物: **婚戒(Engagement Ring)**

## 4. 城镇事件与结婚锁(详见特色文档)

RF4独有设计: 每位候补的结婚都被特定随机城镇事件锁定。玩家可能需要等待数天至数周才能触发需要的事件。

这是RF4社交系统**最大的差异化特征**,也是**最受争议的设计**。

## 5. 香水系统(Cologne)

- 每天访问特定NPC可获赠随机香水
- 佩戴候补喜欢的香水类型→对话LP+3(追加)
- 可在装备栏中切换

## 6. 委托系统

- 通过伊莱莎(Eliza)接取委托
- 不像RF3的三个公告板系统
- 委托奖励: PP点数(主要) + 少量金币 + LP(有时)
- 可以同时接取多个委托(不需逐个完成)

## 7. 设计特点

1. **12候补(6男6女)=对称完整**: 比RF3的11女+0男更包容
2. **城镇事件锁结婚**: 最独特也最受争议的设计(详见特色文档)
3. **香水系统**: 小细节但增加"每天获得小奖励"的循环
4. **组队冒险**: LP≥5可带候补进入迷宫战斗,社交收益=战力提升
5. **RF4比起RF3**: 多了男性候补线,但多了城镇事件RNG的焦虑
