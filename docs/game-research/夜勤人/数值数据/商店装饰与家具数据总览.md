## 六、商店装饰效果

| 装饰类型 | 效果 |
|------|------|
| 小费加成类 | 与收银机叠加→总小费可达50%+ |
| 防窃类 | 减少窃贼出现概率→保护高价物品 |
| 加速类 | 顾客移动更快→更多交易/天 |
| 营业时间类 | 延长营业窗口→更多顾客 |
| 氛围类 | 视觉主题→无功能影响 |
