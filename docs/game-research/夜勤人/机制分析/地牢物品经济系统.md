# 地牢物品经济系统 - 夜勤人(Moonlighter)

> 游戏版本: Moonlighter v1.0+ (PC/Switch/PS4/Xbox, 2018年)
> 数据来源: moonlighter.fandom.com, ProdigyGamers

## 1. 概述

夜勤人的地牢不仅是战斗场所，更是**商品供应链**。所有商店出售的物品来自地牢宝箱和敌人掉落。地牢→商店→金币→装备→更深地牢的循环中，物品经济是核心驱动。

## 6. 设计特点

1. **地牢=唯一供应链**: 无种植/无畜牧/无采集——所有商品来自地牢→探索=生产
2. **价值跨域指数增长**: 魔像(3g)→科技(26,400g)——8,800倍差距→驱动"尽快解锁下一地牢"
3. **Jelly悖论=终极新手考验**: 商店游戏中最经典的材料管理陷阱
4. **Boss掉落=限量版**: 每Boss一局一次→Boss战利品有收藏+出售双重价值
