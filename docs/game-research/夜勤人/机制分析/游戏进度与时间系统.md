# 游戏进度与时间系统 - 夜勤人(Moonlighter)

> 游戏版本: Moonlighter v1.0+ (PC/Switch/PS4/Xbox, 2018年)

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 一天 | 白天(商店营业)+夜晚(地牢探索) |
| 营业时间 | 开店→关店(玩家自主选择) |
| 地牢时间 | 无限制(漫游者=隐性的时间压力) |
| 无季节循环 | 无春夏秋冬——以地牢推进替代 |

## 2. 双时相循环

```
[白天] 商店:
  开店→摆放物品→定价→顾客进入→观察反应→调整价格
  →补货→收银→关店
    ↓
[夜晚] 地牢:
  进入地牢→清房间→开宝箱→拾取物品→遭遇敌人
  →使用药水→面对Boss→继续或撤退(吊坠)
    ↓ 返回
[白天] 商店(重复)
```

## 3. 进度门控

| 门控类型 | 说明 |
|------|------|
| Boss击败 | 解锁下一地牢(魔像→森林→沙漠→科技→第五门) |
| 金币积累 | 商店升级(8,000→60,000→120,000→250,000) |
| 材料收集 | 装备制作(需要特定地牢材料) |
| 城镇设施 | 布告栏解锁(铁匠/女巫/零售/小贩/银行) |

## 4. 无时间压力vs漫游者隐性压力

| 维度 | 压力 |
|------|:--:|
| 商店 | 无(想营业多久就多久) |
| 地牢 | 漫游者(每层时间限制→强制快节奏) |
| 债务 | 无(与Recettear的30天硬截止相反!) |
| Boss | 无(随时可打) |
| 任务 | 无时间限制 |

## 5. 设计特点

1. **无限时间=自由节奏**: 没有Recettear的债务压力——可无限天慢慢经营——适合休闲
2. **漫游者=隐性的"别磨蹭"**: 唯一的时间限制——防止每层都慢慢清
3. **地牢推进=核心进度**: 不是"季节过去解锁新区域"而是"击败Boss→下一地牢"
4. **金币门控=经营驱动**: 商店扩张(60,000/120,000/250,000)=中期核心驱动
5. **无季节/无节日=专注双循环**: 不做季节和节日系统→将全部设计集中在"商店+地牢"上
