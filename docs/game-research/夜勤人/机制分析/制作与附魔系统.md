# 制作与附魔系统 - 夜勤人(Moonlighter)

> 游戏版本: Moonlighter v1.0+ (PC/Switch/PS4/Xbox, 2018年)
> 数据来源: moonlighter.fandom.com

## 1. 铁匠铺制作(Vulcan's Forge)

### 1.1 机制
- NPC: 安德烈(Andrei)
- 解锁: 500g(布告栏)
- 工艺: 地牢材料+金币→新武器/防具
- 升级: 已有装备→更强版本(需要下一地牢的材料)

### 1.2 愿望单(Wishlist)
- 锁定想要的装备→查看所需材料
- 地牢中愿望单物品**高亮显示**
- 家中储物箱材料**自动可用**(无需携带)

## 2. 女巫附魔(The Wooden Hat)

### 2.1 机制
- NPC: 艾丽丝(Eris)
- 解锁: 500g(布告栏)
- 消耗: 各色Jelly+强化水晶(Empowering Crystals)
- 效果:
  - 武器: 攻击力/特殊效果
  - 防具: 防御力/抗性
  - 部分附魔可**揭示地牢秘密门**

### 2.2 Jelly材料

| Jelly类型 | 来源地牢 | 出售价(金币) | 建议用途 |
|:------:|------|:-----:|------|
| 丰富Jelly(Rich Jelly) | 魔像 | 6g | **保留→药水** |
| 毒液Jelly(Venom Jelly) | 森林 | 22g | **保留→药水** |
| 火Jelly(Fire Jelly) | 沙漠 | 110g | **保留→药水** |
| 电Jelly(Electric Jelly) | 科技 | 440g | **保留→药水** |
| 强化水晶(Empowering Crystal) | 各大地牢 | 110g | **保留→附魔** |

> **核心原则**: Jelly出售价极低≠制作药水的价值极高——**永远保留Jelly做制作材料，不要出售**

## 4. 商店装饰制作(小贩)

| 装饰类型 | 效果 |
|------|------|
| 小费类 | 增加收银机小费% |
| 防窃类 | 减少窃贼出现概率 |
| 加速类 | 顾客移动更快(更多交易/天) |
| 营业时间类 | 延长可营业时长 |

## 5. 设计特点

1. **材料即进度**: 不是"有钱就能升级"——必须下地牢找对应材料
2. **愿望单=目标系统**: 锁定装备→看到所需材料→驱动地牢探索方向
3. **Jelly悖论**: 极其便宜(6-440g出售)但极其宝贵(制作药水节省数千)——考验玩家资源判断
4. **附魔=装备保值**: 不是"旧装备扔掉"而是"精炼让它更强"→延长装备生命周期
