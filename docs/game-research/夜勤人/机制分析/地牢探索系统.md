# 地牢探索系统 - 夜勤人(Moonlighter)

> 游戏版本: Moonlighter v1.0+ (PC/Switch/PS4/Xbox, 2018年)
> 数据来源: moonlighter.fandom.com, TrueAchievements, Steam Solo

## 1. 地牢总览

夜勤人有**5个地下城**(+DLC漫游者地牢)，每个地牢是**程序化生成的Roguelite关卡**。

| # | 地牢 | 层数 | Boss | 解锁条件 |
|:--:|------|:--:|------|------|
| 1 | 魔像地牢(Golem Dungeon) | 3层 | 魔像王(Golem King) | 初始 |
| 2 | 森林地牢(Forest Dungeon) | 3层 | 食人花(Mutae) | 击败魔像王 |
| 3 | 沙漠地牢(Desert Dungeon) | 3层 | 娜迦(Naja) | 击败食人花 |
| 4 | 科技地牢(Tech Dungeon) | 3层 | 能量流(Energy Flux) | 击败娜迦 |
| 5 | 第五门(Unknown/5th Door) | — | 最后维度海盗 | 收集4个守护者钥匙 |
| 6 | 漫游者地牢(Wanderer Dungeon) | 9层 | 激怒漫游者 | DLC |

## 2. Roguelite机制

| 机制 | 说明 |
|------|------|
| 程序化生成 | 每层房间布局随机 |
| 死亡惩罚 | 失去**背包中部分物品**(保留装备) |
| 生命恢复 | 地牢中有治疗池(Healing Pool) |
| 逃脱机制 | 可使用**吊坠(Pendant)**立即返回(保留所有背包物品) |
| 特殊房间 | 治疗池/废弃营地(剧情笔记)/**秘密房间**(跳入闪亮坑洞) |
| 漫游者(Wanderer) | 在每层停留太久→绿色无敌幽灵出现→**15,000伤害(秒杀)**→强制快节奏 |

## 5. 设计特点

1. **Roguelite轻量化**: 不是"死就全丢"的硬核Roguelike——保留装备，仅丢失部分拾取物
2. **漫游者=反磨蹭机制**: 每层有时间限制→强制快节奏→避免"每间房都慢慢清"
3. **Boss战后不再可打**: 鼓励推进而非刷同一个Boss
4. **吊坠逃脱=风险规避**: 获得好物品后可安全撤退——策略选择而非"必须打完"
5. **秘密房间=奖励探索**: 跳入坑洞→隐藏房间→额外宝箱——鼓励冒险
