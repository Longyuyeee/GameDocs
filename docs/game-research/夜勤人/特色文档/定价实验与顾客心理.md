# 定价实验与顾客心理 - 夜勤人(Moonlighter)

> 游戏版本: Moonlighter v1.0+ (PC/Switch/PS4/Xbox, 2018年)

## 一、概述

夜勤人的定价系统本质是**通过顾客表情反应进行价格实验**。不知道物品的"真实价值"→观察顾客的4种表情→逐步调整→找到最优价。这个过程本身就是一个迷你游戏。

## 二、价格发现循环

```
放入新物品(首次)→设直觉价→观察首个顾客
  ├── 😍 → 价格太低! → 大幅提价(×2-3)
  ├── 😊 → 完美! → 锁定
  ├── 😐 → 略高 → 微降5-10%
  └── 😡 → 太高! → 大幅降价(÷2)
→ 重复直到 😊 稳定
```

## 三、价格实验策略

### 3.1 首次定价试探

| 物品类型 | 建议初始价 |
|------|:--:|
| 基础材料(藤蔓/根) | 10-20g |
| 矿石(水晶石/铁锭) | 50-100g |
| 中级材料(玻璃透镜/魔像核心) | 100-200g |
| 高级材料(Boss掉落) | 500-1,000g |

### 3.2 调整粒度

| 反应 | 调整幅度 |
|:--:|:--:|
| 😍 → 😊 | +50-100% |
| 😐 → 😊 | -10-20% |
| 😡 → 😐 → 😊 | 先÷2, 再-10% |

## 四、长期价格管理

| 问题 | 表现 | 对策 |
|------|------|------|
| 需求衰减 | 😊价格越来越难维持 | 暂停该物品出售→展示柜恢复流行度 |
| 同一物品多表 | 一张桌子卖完→下一张需重新定价 | 同桌统一价格 |
| 富客溢价 | 富人愿付>😊价格 | 展示柜设2倍价→等富客 |

## 五、与Recettear讨价还价的对比

| 维度 | 夜勤人(观察反应) | 露塞提娅(主动议价) |
|------|:-----:|:-----:|
| 玩家角色 | 被动观察者 | 主动谈判者 |
| 反馈方式 | 4种表情 | 顾客还价(接受/拒绝) |
| 最优价发现 | 反复试错 | Pin Price隐藏目标 |
| 顾客差异 | 3种(普通/英雄/富人) | 16种×独立价格因子 |
| 复杂度 | 中低 | 极高 |
| 适合受众 | 轻松/休闲 | 硬核/策略 |

## 六、设计分析

1. **表情反馈=直觉化**: 不需要教程解释"130%基准价"——看到😍就涨价/看到😡就降价——直觉友好
2. **隐性流行度=长期策略层**: 在简单反馈下有一个隐藏的供需系统——表层简单深层复杂
3. **展示柜=防窃+溢价+恢复流行度**: 三重功能合一——商店游戏中最紧凑的多功能设计
4. **实验失败成本低**: 错误定价→顾客不满→轻微需求惩罚——不像Recettear的"愤怒→几天不来"那么严重
