# 多周目驱动的设计分析 - 火山的女儿(Volcano Princess)

> 游戏版本: Volcano Princess v1.0+ (PC, 2023年)

## 一、概述

火山的女儿通过**不可能一周目全收集**的设计来驱动多周目。58结局×3领主×18情感对象×432回合的有限资源——每个选择都排除其他可能性，每个结局都是唯一的路径。这是养成品类中对"重玩价值"最极致的探索。

## 二、多周目驱动机制

| 机制 | 效果 |
|------|------|
| 3领主→必须选1 | 一周目只能体验1条线 |
| 58结局→432回合 | 一周目只能达成组合中的1个 |
| 挚友锁 | 部分NPC好感度到了自动锁友情→无法恋爱→下周目换目标 |
| 互斥结局 | 黑暗路线(魔族女王)vs光明路线(火山女神)不可兼得 |
| 继承点 | 结局星级越高→NG+起点越好 |
| 服装保留 | 已购买的服装NG+继承→减轻重复感 |

## 三、周目间的情感递进

| 周目 | 情感 | 驱动力 |
|:--:|:--:|------|
| 一周目 | "我能不能当一个好父亲?" | 学习+探索 |
| 二周目 | "这次我要让你过上更好的生活" | 优化+尝试新路线 |
| 三周目 | "我要让你成为最好的自己(火山女神)" | 追求最高结局 |
| 四周目+ | "我要看遍你所有的可能" | 全收集+完美主义 |

## 四、与牧场游戏的多周目对比

| 游戏 | 多周目驱动 | 继承内容 |
|------|:--:|------|
| 星露谷 | 弱(无限年→一周目可全体验) | 无 |
| 符文工房4 | 弱-中(NG+可选) | 部分 |
| 火山的女儿 | **极强(58结局×3领主×18感情)** | 服装+继承点 |

## 五、设计分析

1. **"不可能全收集"=最诚实的设计**: 不像某些游戏的"NG+只是更难的同样内容"——火山的女儿明确告诉玩家"一周目不可能看所有结局"
2. **继承点=品质驱动**: 结局越好→继承越多→下次越容易——奖励玩家的优化
3. **服装保留=视觉传承**: 女儿穿着上一周目的衣服——视觉上的"我们曾经一起经历过"
4. **三领主=三次"完全不同的女儿"**: 不同领主→不同成长路线→不同性格的女儿——不是"同一个女儿的不同结局"
