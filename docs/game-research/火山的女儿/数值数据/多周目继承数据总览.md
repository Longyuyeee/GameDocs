# 多周目继承数据总览 - 火山的女儿(Volcano Princess)

> 游戏版本: Volcano Princess v1.0+ (PC, 2023年)
> 数据来源: Steam社区指南, VGover

## 一、继承点(Inheritance Points)

| 结局星级 | 继承点(估算) | 可解锁内容 |
|:--:|:--:|------|
| 1★ | 少量 | 基础NG+选项 |
| 3★ | 中量 | 更多继承选项 |
| 5★ | 大量 | 高级继承 |
| **7★(火山女神)** | **最多** | **全部NG+内容** |

## 二、可继承内容

| 类别 | 继承比例 | 说明 |
|:--:|:--:|------|
| 金币 | 部分% | 减少初期经济压力 |
| 服装 | **全部** | 已购买服装全部保留 |
| 道具(部分) | 部分 | 特定道具可继承 |
| 结局解锁记录 | — | 解锁画廊/成就 |
| 好感度 | **不继承**(重置) | 每局重新培养关系 |

## 三、多周目解锁内容

| 内容 | 解锁条件 |
|------|:--:|
| 新对话选项 | NG+ |
| 隐藏事件 | 特定结局达成 |
| 全画廊 | 全结局收集 |
| 特殊服装 | 成就奖励 |

## 四、推荐周目规划

| 周目 | 领主选择 | 重点目标 | 预计结局 |
|:--:|:--:|------|:--:|
| 一周目 | 克劳德 | 熟悉流程/积累继承点/铁匠结婚 | 3-5★ |
| 二周目 | 雷贝莎 | 战斗路线/刷大部分结局(最后一月回档) | 4-6★ |
| 三周目 | 公主(格温妮丝) | 全结局收集/**火山女神(7★)** | 7★ |
| 四周目+ | 任意 | 剩余结局/完美收集 | 全58结局 |

## 五、属性继承机制

- 属性**不继承**(每局独立成长)
- 但**经验(玩家知识)继承**: 知道哪些活动优先/哪些NPC容易锁挚友/哪些结局条件互斥

## 六、设计分析

1. **继承≠碾压**: 属性和好感度重置→每局仍是新的挑战
2. **服装保留=最有意义的继承**: 已购买的服装保留→视觉上的"传承感"
3. **最少3周目=58结局**: 无法一周目全收集→必须多周目——继承系统降低重复疲劳
4. **7★=最高继承点**: 最高结局=最多NG+奖励——驱动玩家挑战真结局
