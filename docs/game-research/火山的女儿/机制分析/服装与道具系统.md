# 服装与道具系统 - 火山的女儿(Volcano Princess)

> 游戏版本: Volcano Princess v1.0+ (PC, 2023年)
> 数据来源: Steam社区指南, VGover

## 1. 服装系统

| 要素 | 说明 |
|------|------|
| 服装数量 | 数十套(日常/职业/特殊) |
| 获取方式 | 购买(服装店-丽贝卡)/事件奖励/成就 |
| 功能 | **提升属性/修正活动效率** |
| 换装 | 可随时更换 |

### 服装对玩法的影响

| 服装类型 | 加成方向 |
|:--:|------|
| 战斗服装 | 体魄+/剑术效率+ |
| 学术服装 | 智力+/理学效率+ |
| 社交服装 | 魅力+/礼仪效率+ |
| 特殊服装 | 全属性+/特殊效果 |

## 2. 礼物系统

### 2.1 通用礼物(全员喜欢)

| 礼物 | 好感度 | 获取 |
|------|:--:|------|
| **睡火莲** | +5 | 种植(最常用!) |
| 金块 | +3 | 采矿/购买 |
| 金剪刀 | +3 | 购买 |
| 肖像画 | 全员喜欢 | 野马湖画(消耗7体力) |

### 2.2 花类礼物(NPC差异化)

| 花 | 喜欢该花的NPC |
|:--:|------|
| 红玫瑰 | 克劳德/哈克特/浴场老板 |
| 勿忘我 | 妮娜/公主/守卫/说书人 |
| 茉莉 | 雷贝莎/克雷格/修女 |
| 野花 | 莫娜/王子/划桨爷爷 |
| 金兰 | 本森/朗/服装店老板 |

## 3. 道具/消耗品

| 类型 | 代表 | 用途 |
|:--:|------|------|
| 体力恢复 | 面包(小麦制作)/各种食物 | 体力循环 |
| 属性书 | 各种属性书 | 直接提升属性 |
| 装备 | 武器/防具 | 战斗加成 |
| 特殊道具 | 海盗证/天堂徽章等 | 特殊结局条件 |

## 4. 设计特点

1. **服装=属性Build**: 类似RPG装备——不同服装适配不同培养路线
2. **睡火莲=社交通用解**: 全员+5——最优化送礼策略的基石
3. **肖像画=体力→好感度转换**: 7体力→全员好感——社交效率的体力投资
4. **花类=NPC性格映射**: 红玫瑰(热情男性)/勿忘我(温柔女性)——花的语言
