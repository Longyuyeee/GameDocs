# 属性与成长系统 - 火山的女儿(Volcano Princess)

> 游戏版本: Volcano Princess v1.0+ (PC, 2023年)
> 数据来源: Steam社区指南, 3DM, 9Game

## 1. 三维基础属性

| 属性 | 英文 | 影响范围 | 主要提升方式 |
|:--:|------|------|------|
| 力量(Stamina) | 体力/体魄相关 | 战斗/体力上限/剑术弓术效率 | 剑术课/弓术课/探索 |
| 头脑(Mind) | 智力/精神相关 | 理学科目/信仰效率/黑暗抵抗 | 理学课/信仰课/书柜 |
| 魅力(Charisma) | 社交/外表相关 | 社交效率/礼仪效率/爱慕者增量 | 礼仪课/音律课/舞会 |

## 4. 隐藏属性

| 属性 | 范围 | 影响 | 提升方式 |
|:--:|:--:|------|------|
| **黑暗值(Darkness)** | 0-200+ | 结局分支(魔族/修女等) | 暗巷探索/黑手荷官/特定事件选择 |
| 声望(Fame) | 0-10000+ | 高星结局条件 | 探索/比赛/事件 |
| 爱慕者(Admirers) | 0-1000+ | 魅力系结局 | 舞会/社交/魅力高 |

## 5. 属性→结局映射逻辑

```
基础属性(力量/头脑/魅力) → 结局大分支(7个职业方向)
    +
四维属性(体魄/智力/情感/想象) → 结局星级(1-7★)
    +
学科等级(8门) → 具体职业结局(58个)
    +
黑暗值 → 特殊分支(魔族/修女等)
    +
好感度 → 情感结局(独立于职业)
```

## 6. 设计特点

1. **三级属性体系**: 基础(3维)→大属性(4维)→学科(8门)——层级递进的成长结构
2. **隐藏黑暗值**: 不直接显示→需通过事件选择间接管理——道德选择的隐性后果
3. **432回合的有限成长**: 属性成长受回合数限制→必须优先排序——不能"全都要"
4. **属性阈值触发**: 不是"越高越好"——达到阈值即可→超过部分浪费
