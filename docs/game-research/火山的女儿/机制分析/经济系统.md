# 经济系统 - 火山的女儿(Volcano Princess)

> 游戏版本: Volcano Princess v1.0+ (PC, 2023年)
> 数据来源: Steam社区指南, VGover

## 1. 货币体系

| 货币 | 获取 | 用途 |
|------|------|------|
| 金币(Gold) | 工作/比赛奖励/出售物品 | 购买种子/礼物/服装/道具/课程 |
| 继承点(Inheritance Points) | 结局评价→NG+ | **解锁多周目内容** |

## 2. 收入来源

| 来源 | 效率 | 说明 |
|------|:--:|------|
| **化肥倒卖** | ★★★★★ | **打折日在铁匠铺低买高卖——最赚钱!** |
| 黑手荷官(工作) | ★★★★ | 金币+爱慕+黑暗值(多产出) |
| 面包出售 | ★★★ | 种小麦→做面包→出售 |
| 比赛奖金 | ★★★ | 火山节等 |
| 探险掉落 | ★★ | 冥滩怪物掉落出售 |

## 3. 种田经济

| 作物 | 用途 | 策略 |
|------|------|:--:|
| 小麦 | →面包(体力恢复) | 体力循环的基础 |
| 花(各种) | →送礼(好感度) | 社交核心 |
| 高价值作物 | →出售 | 辅助收入 |

## 4. 主要消费出口

| 消费项 | 费用 |
|------|:--:|
| 种子/肥料 | 低-中 |
| 礼物(睡火莲等) | 中 |
| 服装 | 中-高 |
| 课程 | 低(前期主要支出) |
| 道具/装备 | 中-高 |

## 5. 设计特点

1. **化肥倒卖=设计上的"漏洞经济"**: 打折日低买高卖→玩家社区发现的最优赚钱法——可能是设计意图也有可能是Bug
2. **面包=体力=更多行动=更多成长**: 自给自足的体力经济闭环
3. **黑手荷官=多产出工作**: 同时给金币+爱慕+黑暗值——一个行动三个收益——效率最高
4. **经济不是核心限制**: 相比日程(432回合)的硬限制——金币是软约束
