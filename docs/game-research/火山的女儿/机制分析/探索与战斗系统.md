# 探索与战斗系统 - 火山的女儿(Volcano Princess)

> 游戏版本: Volcano Princess v1.0+ (PC, 2023年)
> 数据来源: Steam社区指南, VGover

## 1. 冥滩(Hell Beach)

| 属性 | 说明 |
|------|------|
| 定位 | 核心探索区域——战斗+声望+剧情 |
| 体力消耗 | **8点/次(最高!)** |
| 层数 | 多层→**终点=阿斯贝尔讨伐战** |
| 频率 | **天天去**(社区推荐!) |
| 奖励 | 战斗经验+声望+怪物掉落+事件推进 |

## 2. 暗巷(Dark Alley)

| 属性 | 说明 |
|------|------|
| 定位 | 黑暗值获取+特殊剧情 |
| 黑暗值 | 探索→增加黑暗值 |
| 关键 | 黑暗路线结局(魔族女王/帮派头目)的必需 |
| 频率 | 路线依赖(仅黑暗路线需频繁去) |

## 3. 战斗系统

| 要素 | 说明 |
|------|------|
| 类型 | 回合制/简单策略 |
| 属性影响 | 体魄(HP)/剑术(近战伤害)/弓术(远程) |
| 击杀计数 | 部分结局需要击杀特定数量怪物(30/80/100) |
| Boss | 阿斯贝尔讨伐战→火山女神结局必需 |

## 4. 声望与荣誉

| 来源 | 声望值 |
|------|:--:|
| 探索战斗 | 中 |
| 比赛获奖 | 高 |
| 事件 | 不等 |

## 5. 设计特点

1. **冥滩=8体力=最大的时间投资**: 天天去=每天最高体力消耗→但回报是声望+战斗经验+结局条件
2. **暗巷=黑暗路线的代价**: 增加黑暗值→但同时推进黑暗结局条件
3. **战斗=课程成果的检验**: 剑术/弓术等级直接影响战斗能力→课程→战斗→结局——完整闭环
