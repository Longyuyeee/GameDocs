# 日程与行动系统 - 火山的女儿(Volcano Princess)

> 游戏版本: Volcano Princess v1.0+ (PC, 2023年)
> 数据来源: Steam社区指南, VGover, 3DM, 9Game

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 培养周期 | **12年**(5岁→18岁) |
| 每年 | 12个月 |
| 每月 | 3个行动回合(上旬/中旬/下旬) |
| 每回合 | 安排3个行动→消耗体力 |
| 总计行动回合 | 12年×12月×3 = **432回合** |

## 2. 行动类型

| 类型 | 代表活动 | 体力消耗 | 主要收益 |
|:--:|------|:--:|------|
| **课程(Lesson)** | 剑术/理学/礼仪/绘画等8门学科 | 中 | **学科经验**(核心!) |
| **工作(Job)** | 黑手荷官/剧院/农场等 | 中-高 | 金币+部分属性 |
| **社交(Social)** | 与NPC对话/送礼/约会 | 低 | 好感度 |
| **探索(Explore)** | 冥滩(Hell Beach)/暗巷 | 高(8点) | 战斗经验+声望 |
| **休息(Rest)** | 家里床/莫娜的饭/湖畔木桩 | 0(恢复) | 体力恢复 |
| **特殊(Special)** | 剧场/画架/湖畔/书柜/故事角 | 3-7 | 特定属性+隐藏剧情 |

## 3. 体力系统

| 要素 | 数值 |
|------|:--:|
| 初始体力 | ~100/回合 |
| 课程消耗 | ~10-20/课 |
| 探索消耗 | **8/次**(最高!) |
| 特殊活动消耗 | 3-7/次 |
| **每日免费恢复** | 家里床+莫娜的饭+湖畔木桩=**24点** |
| 种小麦做面包 | 额外体力恢复 |

## 6. 天赋推荐路线

```
土地馈赠3级 → 美食馈赠3级 → 精力充沛3级
  → 农神祝福2级 → 八面玲珑3级
  → 甜言蜜语 → 人见人爱
```

## 7. 设计特点

1. **432回合=有限资源**: 不是"无限天"——每回合的选择都有机会成本——紧迫的策略深度
2. **课程→学科→职业**: 8学科直接映射到58个结局→课程选择=职业规划
3. **免费体力恢复=每日最优解**: 3个免费恢复点→必须利用→减少金币消耗
4. **种田=体力经济**: 种小麦→面包→体力→更多行动——自给自足的体力循环
