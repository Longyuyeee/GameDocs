## 三、社交机制

### 3.1 好感度等级

| ♥ | 好感度 | 解锁 |
|:--:|:-----:|------|
| 1-2♥ | 0-40 | 基础对话/送礼 |
| 3-5♥ | 40-90 | 角色事件1 |
| 6-8♥ | 90-130 | 角色事件2 |
| 9-10♥ | 130-140 | 角色事件3 |
| **10♥满** | **140+** | **情感结局条件达成** |

### 3.2 送礼系统

| 礼物类型 | 好感度 | 代表 |
|:--:|:--:|------|
| **通用最爱** | +5全员 | **睡火莲/金块/金剪刀** |
| 专属最爱 | +7-10 | 每NPC有独特最爱(见上表) |
| 花类 | 因NPC而异 | 红玫瑰(多数男性)/勿忘我(多数女性) |
| 肖像画 | 全员喜欢 | 野马湖画肖像(消耗7体力) |
| 讨厌物品 | -2 | 如泽讨厌"沾血浴巾" |

### 3.3 挚友锁/友情锁

⚠️ 关键机制: 部分NPC在好感度较低时触发"挚友"(Best Friends)锁定——之后无法进入恋爱线!

| NPC | 触发好感度 | 应对 |
|:--:|:--:|------|
| Hackett | **~90(最低!)** | 快速推到140+ |
| Kenneth | ~110 | 同上 |
| Zed | ~110 | 同上 |

## 四、设计特点

1. **三领主=三个周目**: 每次只能选一个领主→驱动最少3周目
2. **挚友锁=社交策略**: 不是"随便送礼"——对易锁NPC必须快速推高好感——增加社交紧迫感
3. **百合线**: 女性角色有完整的恋爱线(妮娜/莫娜/公主等)——包容性设计
4. **睡火莲=通用钥匙**: 全员+5——类似星露谷的兔脚/珊瑚岛的粉钻
5. **肖像画=高成本高回报**: 7体力→全员喜欢——体力与好感的资源转换
