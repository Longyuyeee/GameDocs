# 教派管理与信仰系统 - 咩咩启示录(Cult of the Lamb)

> 游戏版本: Cult of the Lamb v1.0+ (PC/Switch/PS/Xbox, 2022年, 含Relics/Sins/Woolhaven DLC)
> 数据来源: cult-of-the-lamb.fandom.com, IGN, GameTruth, GamePressure

## 1. 概述

咩咩启示录是**教派经营×Roguelite地牢**的独特融合。玩家扮演被"待终者(The One Who Waits)"复活的圣羊，建立并管理一个动物教派，同时深入地牢消灭四位主教。两个系统形成紧密的资源循环。

## 2. 信仰(Faith)系统

| 参数 | 说明 |
|------|------|
| 定位 | 教派的核心"货币"——类似牧场游戏的金币 |
| 获取 | 布道(+10-20/日)/装饰品/仪式/完成任务 |
| 消耗 | 低信仰→**异见者出现**→扩散异见→信仰-5~15% |
| 崩溃 | 异见者最终偷金币+离开教派 |
| 紧急按钮 | Holy Day仪式(+80信仰)/Brainwashing(锁定满信仰2天) |

## 3. 饥饿系统

| 参数 | 说明 |
|------|------|
| 消耗 | 信徒每日进食→更多信徒=更快消耗 |
| 种植 | 核心食物来源——3+信徒后建农场+专职农夫 |
| 烹饪 | 煮熟食物+25-50%饱食值 |
| 紧急 | Feasting Ritual(立即喂饱全部+25信仰)/Ritual Fast(3天不饥饿) |

## 4. 信徒(Follower)管理

### 4.1 信徒属性

| 属性 | 说明 |
|------|------|
| 忠诚度(Loyalty) | 布道/祝福/送礼/任务→提升。高忠诚=更快工作/更多虔诚/更强恶魔 |
| 特性(Traits) | 随机——好(Zelous无视异见)/坏(Spy偷金币3天后离开) |
| 工作分配 | 基于特性(Industrious→采集/Faithful→虔诚) |
| 长老(Elders) | 45-55天后停止工作→升天/牺牲/肥料化 |

### 4.2 状态异常

| 状态 | 原因 | 解决 |
|------|------|------|
| 饥饿 | 没饭吃 | 烹饪/Feasting仪式 |
| 疾病 | 低卫生/尸体/洗脑 | 床上休息/Healing Bay |
| 疲惫 | 夜间工作/任务/恶魔使用 | 送回床上 |
| **异见** | **低信仰** | **监狱+再教育(或杀/牺牲/升天)** |
| 老年 | 45-55天 | 升天/牺牲/复活重制 |
| 受伤 | 追随者打架 | 床上休息/Healing Bay |

## 5. 每日经营流程

```
收集一夜产出→喂食信徒→清理营地(粪便/呕吐物/尸体)
  → 布道(+10-20信仰+忠诚度)
  → 检查疾病/老人/异见者
  → 分配工作
  → 地牢探险(Crusade)
  → 返回: 立即处理战利品/金币买治疗/材料建造
```

## 6. 设计特点

1. **信仰=最脆弱的资源**: 不是"金币可以攒"——信仰持续消耗→必须每日维护
2. **异见者=传染性危机**: 一个异见者不处理→传播→全教派信仰崩溃——紧急管理的压力
3. **信徒=资源也是负担**: 更多信徒=更多生产力=更多食物+床位+卫生需求——不是越多越好
4. **教派经营=反向的牧场经营**: 不是"让动物快乐产奶"而是"管理一群会生病/造反/老死的信徒"
