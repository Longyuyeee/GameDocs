# 信徒生命周期系统 - 咩咩启示录(Cult of the Lamb)

> 游戏版本: Cult of the Lamb v1.0+ (2022年)
> 数据来源: cult-of-the-lamb.fandom.com

## 1. 信徒获取

| 方式 | 频率 | 说明 |
|------|:--:|------|
| **地牢救援** | ★★★★★ | 最常见——每轮1-3人 |
| Helob(蜘蛛商人) | ★★★ | 地牢商店→购买信徒 |
| Boss击败 | ★★ | 每区域前3个小Boss+Witnesses |
| 信徒任务 | ★★ | 常伴随状态异常 |
| 传教远征 | ★★ | Missionary II/III建筑 |
| 交配帐+孵化器 | ★ | 信徒繁殖→孵化蛋 |
| Ritual of Decay | ★(DLC) | 腐化信徒 |

## 2. 一生阶段

```
招募 → 工作年龄(~45天) → 长老(~45-55天)→停止工作 → 死亡
                                    ↓ 或:
                              升天(Ascend)→全员忠诚↑
                              牺牲(Sacrifice)→永久力量
                              复活(Resurrection)→重置年龄!
```

## 3. 忠诚度等级

| 等级 | 提升方式 | 效果 |
|:--:|------|------|
| 低 | — | 慢速工作/易异见 |
| 中 | 布道/祝福/贿赂 | 正常 |
| 高 | 长期培养 | 快速工作/高虔诚 |
| **最高(Disciples)** | 满忠诚 | **自动管理异见/特殊肥料** |

## 4. 关键管理技巧

| 技巧 | 说明 |
|------|------|
| 不超募 | 每个信徒+10-20%食物需求——3-5人起步够了 |
| 优先处理异见 | 1个异见不处理→传播→全教派崩溃 |
| 长老管理 | 45天+→提前决定(升天/牺牲/复活)而非等自然死亡 |
| 复活=保留投资 | 已培养的高级信徒→复活重制年龄→保留技能 |

## 5. 设计特点

1. **信徒不是"永久资产"**: 会老死→必须管理生命周期→与牧场动物不同
2. **长老=必须处理的"问题"**: 不是"动物老了产出少"——是完全停止工作——必须做出道德/策略选择
3. **复活=最宝贵的教义**: 75骨→保留一个培养数十天的高级信徒——投入产出比最高
4. **Disciples=终局自动化**: 满忠诚信徒可自动管理异见——教派管理从"主动"变"被动"
