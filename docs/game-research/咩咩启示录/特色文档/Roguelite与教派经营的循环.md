# Roguelite×教派经营的双循环设计 - 咩咩启示录(Cult of the Lamb)

> 游戏版本: Cult of the Lamb v1.0+ (2022年)

## 一、概述

咩咩启示录的"双循环"是牧场品类中**最紧密的系统整合**——不是"两种玩法拼在一起"而是"两种玩法互相需要对方的存在才能运作"。

## 二、双循环结构

```
[营地] 教派经营:
  布道→信仰→信徒忠诚→生产力→食物+材料+金币
    ↓ 用金币+材料升级武器/解锁教义
[地牢] Roguelite探险:
  战斗→获取信徒+骨+金币+材料+塔罗牌
    ↓ 带回营地
[营地] 教派经营(继续):
  新信徒→更多生产力→更多资源→升级→更强地牢探险
```

## 三、两个系统的互相依赖

| 地牢→营地 | 营地→地牢 |
|------|------|
| 信徒救援(最稀缺资源!) | 武器升级(金币+材料) |
| 骨(仪式消耗) | 诅咒/能力解锁(虔诚) |
| 金币(主要收入来源) | 恶魔召唤(信徒产出) |
| 材料(木材/石材/金) | 心/护甲提升(教义+食物) |
| 食物/种子(营地补充) | — |

## 四、时间管理的张力

| 活动 | 消耗 | 风险 |
|------|:--:|------|
| 地牢探险 | 时间(营地无人管理) | 营地饥饿→异见→信仰崩溃 |
| 营地经营 | 时间(不推进主线) | 不消灭主教→不推进→停滞 |
| **平衡点** | 2-3天地牢→1天营地 | — |

## 五、与夜勤人(Moonlighter)的对比

| 维度 | 夜勤人 | 咩咩启示录 |
|------|:--:|:--:|
| 经营 | 商店(定价+卖货) | **教派(管理信徒+信仰)** |
| 地牢 | Roguelite(清房间) | Roguelite(清房间) |
| 资源 | 物品→出售 | 信徒+骨+材料→经营 |
| 时间 | 无限天 | **持续流逝(不管在哪)** |
| 惩罚 | 死亡=丢部分背包 | 死亡=丢25%+营地可能崩溃 |
| 融合深度 | 较浅(物品是纽带) | **极深(信徒/骨/金币/时间全部联动)** |

## 六、设计分析

1. **时间流逝=最核心的联动**: 地牢探险时营地持续运转→信徒可能在此期间饥饿/生病/异见→回来面对混乱——所有系统中融合最深的设计
2. **信徒=最稀缺资源=探索驱动力**: 不是"去地牢打装备"而是"去地牢救人"——救人是最优先的房间类型
3. **骨=最紧张的策略资源**: 仪式消耗→需要在"复活信徒"和"紧急信仰"之间抉择
4. **教义=永久Build=长线策略**: 20个教义选择不可逆→每个教派有独立的策略方向
