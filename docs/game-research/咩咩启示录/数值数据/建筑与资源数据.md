## 三、资源加工链

```
木材 → 精炼厂 → 木板(×2价值)
石材 → 精炼厂 → 石块(×2价值)
金矿 → 精炼厂 → 金条(×2-3价值)
作物 → 烹饪台 → 熟食(+25-50%饱食值)
粪便 → 肥料 → 农场(产量↑)
```
