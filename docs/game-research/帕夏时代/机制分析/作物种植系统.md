# 作物种植系统 - 帕夏时代(Roots of Pacha)

> 游戏版本: Roots of Pacha v1.3+ (PC/Switch/PS4/PS5)
> 数据来源: rootsofpacha.wiki.gg, Fandom Wiki, Neoseeker, TechRaptor

## 1. 概述

帕夏时代的种植系统核心差异在于**种子来源**和**灌溉技术**。野生采集是种子唯一初期来源，后期可购买种子。游戏有独特的灌溉系统(渠沟+泵)替代传统洒水器，以及作物进化系统。

## 2. 种子获取机制

| 阶段 | 获取方式 | 说明 |
|------|---------|------|
| 初期 | 野外采集 | 采集野生作物→获得1个野生品质作物+1个种子 |
| 中期 | 植物苗圃(Plant Nursery) | 交付Igrork 5作物等待2天后解锁，20 CP/种子 |
| 后期 | 作物进化(Evolution) | 太阳与月亮矿洞完成后解锁进化作物 |

- **农场上的作物不产出种子**——这是核心差异，防止无限循环
- 野外采集的种子只能在对应季节/生物群系获取

## 4. 品质与肥料系统

### 4.1 肥料类型

| 肥料 | 效果 |
|------|------|
| 羽毛肥(Feather Meal) | 缩短生长时间**15%** |
| 优质羽毛肥(Quality Feather Meal) | 缩短生长时间**30%** |
| 粉肥(Powder Meal) | 浇水后保湿**1天** |
| 优质粉肥(Quality Powder Meal) | 浇水后保湿**2天** |
| 角肥(Horn Meal) | 轻微提高高产概率 |
| 优质角肥(Quality Horn Meal) | 显著提高高产概率 |

- **同一地块可施加多种肥料**(叠加效果)
- 肥料通过堆肥器(Composter)制作

## 5. 灌溉系统

| 组件 | 材料 | 功能 |
|------|------|------|
| 灌溉泵(Irrigation Pump) | 50铜+50锡+50石 | 为整个系统提供动力(仅需1个) |
| 渠沟(Trench) | 5植物纤维+2石 | 连接最多9个，灌溉相邻地块 |
| 灌溉延长器(Irrigation Extender) | 25硬木+2纱线 | 延长渠沟通行长度 |

- 灌溉系统需要先完成"冶炼(Smelting)"创意解锁
- 最优布局: 两条渠沟间隔3格→中间3格被浇灌—更省资源

## 6. 设计特点

1. **种子不可自我复制**: 农场作物不产种子→必须持续野外采集或购买→维持探索价值
2. **无洒水器灌溉**: 灌溉渠系统替代传统洒水器，石器时代主题一致
3. **多重肥料叠加**: 不同于大多数牧场游戏的"一地块一肥料"，帕夏允许叠加多种肥料
4. **作物进化终局**: 完成主线后可"进化"出高阶作物，为后期经营提供新目标
5. **冬季可种植**: 冬季有5种专属作物+部分跨季作物，避免冬季停摆
