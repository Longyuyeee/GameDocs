# 加工制造系统 - 帕夏时代(Roots of Pacha)

> 游戏版本: Roots of Pacha v1.3+ (PC/Switch/PS4/PS5)
> 数据来源: rootsofpacha.wiki.gg, Fandom Wiki, Neoseeker, TechRaptor

## 1. 概述

帕夏时代的加工制造系统围绕**石器时代技术逐步发明**的叙事框架。所有加工站/建筑都不是"购买"的，而是通过**创意(Ideas)系统**由NPC"发明"后解锁。整个系统呈现人类从石器时代走向金属时代的科技树进程。

## 2. 创意(Ideas)系统——科技树机制

### 2.1 创意触发方式

| 触发类型 | 示例 |
|---------|------|
| 执行特定动作 | 找到燧石→触发"燧石工具"创意 |
| 完成前置创意 | 完成发酵罐→触发"奶酪布"创意 |
| 达到特定日期/季节 | 春11→"日晷"创意 |
| 达到繁荣度阈值 | 24,000→"压榨机"创意 |

- 所有创意**不会错过**，可按自己节奏解锁

### 2.2 创意执行流程

```
NPC冒出创意想法(烟雾信号)
  → 交付材料给NPC
  → 等待研究天数
  → 新工具/建筑/配方解锁
```

## 5. 完整加工链

```
植物纤维 ————————————→ 纺锤 → 纱线/干草捆
鱼 ──→ 太阳干燥器 → 干鱼
  └──→ 发酵罐 → 发酵鱼 ──→ 酿造器
  └──→ 熏制器 → 熏鱼
肉 ──→ 太阳干燥器 → 干肉
  └──→ 熏制器 → 熏肉
水果 ──→ 压榨机 → 果汁 ──→ 酿造器 → 葡萄酒 ──→ 木桶 → 陈酿酒
  └──→ 太阳干燥器 → 干果
奶 ──→ 奶酪布 → 奶酪 ──→ 木桶 → 陈酿奶酪
  └──→ 发酵罐 → 酸奶 ──→ 奶酪布 → 发酵奶酪
作物 —→ 压榨机 → 汁/油
  └──→ 发酵罐+酿造器+醋发酵罐 → 醋 ──→ 泡菜器 → 泡菜
蜂蜜 ——————→ 酿造器 → 蜂蜜酒 ──→ 木桶 → 陈酿蜂蜜酒
```

## 6. 设计特点

1. **科技树叙事**: 从太阳干燥(最原始)→压榨→发酵→酿造→木桶陈酿，呈现人类食品加工技术进化史
2. **繁荣度门控**: 加工站不是有钱就买，而是部落繁荣度→NPC创意→交付材料的递进流程
3. **加工链完整且有深度**: 醋的生产需要发酵罐→酿造器→醋发酵罐三级链，是设计上的复杂但有趣的加工深度
4. **材料循环**: 粪便(动物)→堆肥器→肥料→作物增产→更多加工原料——完整循环经济
5. **无自动化(除灌溉)**: 加工需要手动放置原料→等待→取出，保持玩家参与感
6. **后期加工(木桶陈酿)**: 最晚解锁(230,000繁荣度)，提供终局追求目标
