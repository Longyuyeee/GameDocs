# 畜牧养殖系统 - 帕夏时代(Roots of Pacha)

> 游戏版本: Roots of Pacha v1.3+ (PC/Switch/PS4/PS5)
> 数据来源: rootsofpacha.wiki.gg, Fandom Wiki, TheGamer, Neoseeker, TechRaptor

## 1. 概述

帕夏时代的动物系统围绕**石器时代驯化**(Domestication)概念设计。所有动物从**野生状态通过长笛驯化**——这是与购买动物的牧场游戏根本不同的设计。系统包含驯化→饲养→繁殖→颜色变异四个层次。

## 2. 驯化流程(核心机制)

```
与野生动物互动(触发"动物友谊"创意)
  → 交付Okka 20植物纤维→获得长笛(Flute)
  → 对野生动物使用"调谐(Attune)"→节奏小游戏
  → 每天1次调谐(长笛每日4次使用), 积累♪进度
  → 完全调谐后可邀请动物回村
  → 建造动物棚屋(Animal Shed)
```

### 2.1 长笛调谐小游戏
- 白色圆环收缩至内圈时点击
- 表现对应1-2音符(♪)进度
- 长笛每日有4次使用(绿色圆点指示)

## 5. 繁殖系统

### 5.1 繁殖条件
- 同物种×异性×双方均6♥好感度以上
- 解锁"繁殖围栏(Breeding Pen)"创意(Tetih)
- 材料: 50燧石+30木+20植物纤维(需~50,000繁荣度)

### 5.2 遗传机制(Stats Inheritance)
- 子代**继承双亲属性+随机+1属性提升**
- 每代提升属性:
  - 基础品质(Base Quality): 产出物品质
  - 基础产量(Base Production): 产出频率
  - 基础速度(Base Speed): 坐骑速度
- 每只动物最多繁殖**3次**(同代限制)

## 9. 设计特点

1. **驯化非购买**: 所有动物从野生状态通过调谐驯化，而非从商店购买——石器时代设定完美呈现
2. **节奏小游戏**: 长笛调谐为动物获取增添了主动技能元素
3. **遗传育种系统**: 子代继承+变异→多代繁殖优化属性，最深层的动物养成机制
4. **传说颜色图鉴**: 第2年起特定日期刷新传说颜色，激励长期经营和收藏
5. **动物无死亡/出售机制**: 石器时代部落不会"卖动物"，所有动物都是部落财产
6. **无战斗**: 洞穴探索有谜题无怪物战斗，动物仅经营/骑乘用途
