# 经济系统 - 帕夏时代(Roots of Pacha)

> 游戏版本: Roots of Pacha v1.3+ (PC/Switch/PS4/PS5, 2023年正式版)
> 数据来源: rootsofpacha.wiki.gg, Fandom Wiki, Neoseeker, TechRaptor

## 1. 货币体系

**Roots of Pacha 没有金钱/货币。** 这是该游戏最核心的设计差异——以**贡献点数(Contribution Points/CPs)**替代传统的金钱经济。

| 经济元素 | 说明 |
|---------|------|
| 贡献点数(CP) | 通过向贡献箱放置物品获得，是"购买"一切的唯一"货币" |
| 贡献箱(Contribution Box) | 位于篝火北侧/Pacha之家的核心设施 |
| 结算机制 | 每晚睡觉时统计当日贡献，CP自动计入 |
| 繁荣度(Prosperity) | 部落整体繁荣水平，累积CP驱动，达到阈值解锁新创意/建筑 |

## 2. 收入来源(CP获取)

| 来源 | 前期效率 | 中后期效率 | 说明 |
|------|:------:|:------:|------|
| 干燥鱼(Dried Fish) | ★★★★★ | ★★★ | 太阳干燥器加工后CP翻倍(如肥鱼37→67CP) |
| 农作物加工(干燥/腌制/压榨/酿造) | ★★★ | ★★★★★ | 加工品CP价值远超原料 |
| 畜牧产品加工 | ★★ | ★★★★★ | 羊奶→酸奶→奶酪链大幅增值 |
| 宝石(Gems) | ★★ | ★★★★ | 矿洞宝石固定60CP/颗 |
| 钓鱼(稀有鱼) | ★★★ | ★★★ | 海滩/森林用鱼叉捕获稀有鱼价值数百CP |
| 采集/基础原料 | ★ | ★ | 木材/纤维/石头CP极低，不建议贡献 |

## 3. CP消费出口

| 消费项 | CP消耗 | 说明 |
|--------|:-----:|------|
| 创意研究(Ideas) | 不等 | 解锁新工具/建筑/配方，部分需消耗CP |
| 工具升级 | 100-1,260 CP | 燧石→黑曜石→铜→青铜各级工具 |
| 动物棚屋 | CP+材料 | 小型/中型/大型 |
| 建筑物 | CP+材料 | 灌溉泵/谷仓/酿酒厂等 |
| 种子购买(植物苗圃) | 20 CP/种子 | 解锁苗圃后可购买种子 |
| 动物用品 | CP+材料 | 刀刃剪毛器(160 CP)/挤奶器(120 CP) |
| 装饰/家具 | 不等 | 艺术站解锁后可制作 |

## 6. 设计特点

1. **零金钱设计**: 整个游戏没有金币/货币概念，所有交易以部落贡献为基础——完美契合石器时代部落共同体设定
2. **贡献驱动替代利润驱动**: 玩家的目标不是"赚最多的钱"，而是"为部落做最大的贡献"——改变了经营动机
3. **加工增值链完整**: 尽管没有金钱，干燥/压榨/发酵/酿造/泡菜/熏制/陈酿七条加工链同样完整
4. **繁荣度锁进度**: 创意/建筑/工具升级不是有钱就买，而是需要部落整体繁荣度达标→驱动全方位经营
5. **原料贡献惩罚**: 木材/石头等基础原料CP极低，明确引导"加工后再贡献"
