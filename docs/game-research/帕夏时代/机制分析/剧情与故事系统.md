# 剧情与故事系统 - 帕夏时代(Roots of Pacha)

> 游戏版本: Roots of Pacha v1.3+ (PC/Switch/PS4/PS5)
> 数据来源: rootsofpacha.wiki.gg, Fandom Wiki, Neoseeker, Steam Dev Logs

## 1. 世界观设定

帕夏时代设定在**石器时代**。玩家是**帕夏族(Pachan Clan)**的一员，这个部落由创世神**帕夏(Pacha)**引领至一片新土地。在一次地震后，部落与外界隔绝，决定在此定居并繁荣发展。

### 三大部族

| 部族 | 信仰 | 核心价值 | 生活环境 |
|------|------|---------|------|
| 帕夏(Pachan) | 帕夏(创世神) | 部落共同体/合作发展 | 村庄(森林与大地之间) |
| 莫格拉尼(Mograni) | 火灵(Fire Spirit) | 追随力量/个人主义 | 稀树草原(后扩展至冻原) |
| 雅库安(Yakuan) | 海灵(Sea Spirit) | 追随传统/保守 | 海滩(后扩展至龟岛) |

### 背景冲突

莫格拉尼和雅库安**曾经是同一个部落**，但长久以来分裂并互相敌视。帕夏的意志是让他们**重新统一**。

## 2. 主线剧情结构

```
序幕: 地震→部落定居新土地(春1, Y1)
  ↓
开启: 丛林桥梁修好→发现金字塔预言壁画(春8, Y1)
  ↓
发展: 帕夏族繁荣→遇到莫格拉尼族(春14)→遇到雅库安族(春12)
  ↓
核心循环: 完成预言壁画中的目标
  ├── 每个预言: 5个目标(开发创意/达到阈值/交好人物)
  ├── 完成奖励: 独特装备+能力提升
  └── 逐步揭示莫格拉尼和雅库安分裂的历史
  ↓
高潮: 完成所有预言→浇水激活帕夏树种子(Pachan Tree Seeds)
  ↓
结局: 三族联合庆典→莫格拉尼+雅库安+帕夏重新团结
  ↓
后日谈(v1.3): 神秘访客→虫捕/新洞穴/冻原/龟岛/新动物
```

## 3. 叙事驱动方式

| 机制 | 说明 |
|------|------|
| **预言驱动**: | 金字塔壁画中的预言是任务主线，每个预言5个目标 |
| **繁荣度门控**: | 部落繁荣度达到阈值→触发新创意→推进技术→解锁新区域 |
| **日期触发**: | 春8(丛林)、春12(雅库安)、春14(莫格拉尼)——关键NPC/区域按固定日期解锁 |
| **自由节奏**: | 预言完成不限时间，主线无时间限制 |
| **关系门控**: | 部分预言需要与特定NPC达到好感度 |

|------|:--:|------|------|

## 5. 预言系统详解

丛林金字塔中的预言壁画构成主线任务:
- 每个预言包含**5个目标**
- 目标类型: 发展特定创意、达到繁荣度阈值、与特定NPC交好、收集特定物品等
- 完成5个目标→获得**独特装备**(提升能力)
- 逐步解锁金字塔下的讲台→剧情过场→揭示三族分裂历史
- 完成所有预言后，金字塔下的帕夏树种子被挖出→浇水激活→三族庆典

## 6. 结局

- **单一结局(三族统一)**: 帕夏时代无多结局设计——主线的感情核心是三族和解
- 结局后(v1.3): 神秘访客引入新机制(虫捕/新洞穴/新区域)，提供"后日谈"玩法
- 开放无限经营: 主线完成后可继续经营/社交/探索新区域

### 7.2 心意事件触发条件

- 2花/4花/6花(约会)/8花/10花(结合)各有事件
- 触发受时间(夜间/白天)、天气(晴/风/雪)、地点(森林/海滩/草原)、星期限制
- 节日和雨天通常不触发事件

## 8. 支线/隐藏内容

- **矿洞壁画**: 隐藏房间提供免费CP和成就
- **动物传说颜色**: 第2年起特定日期刷新的传说动物——提供收集目标
- **竞赛日**: 每季6日/20日的坐骑赛跑
- **神秘访客(v1.3)**: 主线后的虫捕/新洞穴支线

## 9. 剧情与经营的关联

| 经营→剧情 | 剧情→经营 |
|-----------|-----------|
| 繁荣度达标→部落吸引外部族注意→相遇 | 完成预言→获得装备→提升能力 |
| 加工技术进步(酿酒/奶酪)→提供图腾供品→获得动物力量 | 动物力量→解锁新矿洞区域→更多资源 |
| 动物驯化+繁殖→竞赛日→族际交流 | 三族统一→解锁冻原/龟岛→新作物/动物 |
| 作物种植→供给部落→繁荣度提升→解锁更多创意 | 创意→新加工站→经济增值循环 |

## 10. 设计特点

1. **预言替代任务日志**: 壁画预言作为任务载体——石器时代不可能有"任务板"或"公告栏"
2. **三族和解主题**: 不是"打Boss救世界"，而是"弥合分裂促进团结"——更成熟的故事主题
3. **渐进叙事**: 从地震求生→丛林发现→遇到他族→理解历史→促成和解——每一步都自然推进
4. **石器时代语境**: 所有叙事包装(预言壁画/图腾力量/结合仪式/创意发明)都恰如其时代设定
5. **v1.3后日谈**: 主故事完成后不是"无尽的重复"，而是引入新机制提供新鲜感
6. **无反派设计**: 三个部族都是"好的"，只是因历史而分裂——更接近现实中的文化冲突
