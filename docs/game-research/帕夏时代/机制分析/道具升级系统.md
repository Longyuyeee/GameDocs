## 1. 工具种类与升级

## 2. 工具升级路线

```
手斧(万能初始)
  → 燧石系列(Flint, 矿洞第4天+)
    → 黑曜石系列(Obsidian, 16,000繁荣度+"黑曜石加工"创意)
      → 铜系列(Copper, 70,000繁荣度+"锻造"创意)
        → 青铜系列(Bronze, 120,000繁荣度+"冶炼"创意)
```

## 3. 武器/战斗装备

**帕夏时代无战斗系统，无武器/防具。** 洞穴探索通过解谜和动物图腾力量(猫头鹰/猴子/熊)而非战斗推进。这是一个纯经营+探索+解谜的游戏。

## 7. 坐骑系统

- 驯化大型动物(野牛/野猪/鸵鸟/山羊/马/原驼)后可骑乘
- "动物骑乘(Animal Riding)"创意(10,000繁荣度)解锁
- 坐骑速度受繁殖属性"基础速度(Base Speed)"影响
- 竞赛日(Race Day)每季6日和20日: 坐骑赛跑活动

## 8. 设计特点

1. **万能初始工具→专业分化**: 手斧能砍树/采矿/钓鱼/锄地，石器时代"一器多用"的真实感
2. **创意锁而非金钱锁**: 升级材料+CP双重要求，且需要对应创意先解锁
3. **无战斗装备**: 纯和平设计，洞穴谜题替代战斗
4. **长笛驯化工具**: 将"购买动物"转化为"长笛调谐"的技能玩法，赋予狩猎采集时代感
5. **动物专属工具**: 剪毛器/挤奶器独立于普通工具，强调畜牧的专业化
