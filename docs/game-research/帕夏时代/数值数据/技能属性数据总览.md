# 技能属性数据总览 - 帕夏时代(Roots of Pacha)

> 游戏版本: Roots of Pacha v1.3+ (PC/Switch/PS4/PS5)
> 数据来源: rootsofpacha.wiki.gg, Fandom Wiki, Neoseeker

## 一、无传统技能树系统

帕夏时代**没有传统的技能等级系统**（无农耕/采矿/战斗/钓鱼技能等级）。玩家能力提升完全通过以下方式实现：

| 提升方式 | 说明 |
|---------|------|
| 工具升级 | 燧石→黑曜石→铜→青铜，提高效率/范围 |
| 创意(Ideas)解锁 | NPC发明新技术→解锁新配方/建筑/能力 |
| 动物图腾力量 | 矿洞图腾赋予猫头鹰/猴子/熊变身能力 |
| 预言装备 | 完成预言获得独特装备提升能力 |
| 食物Buff | 食用加工食品获得临时/永久属性提升 |

## 四、无属性增长机制

帕夏时代没有STR/INT/VIT等RPG属性。角色成长完全体现在：
- 更好的工具→更快的工作效率
- 更多的加工站→更丰富的加工选择
- 更多的动物图腾→更强的探索能力
- 预言装备→被动能力提升
