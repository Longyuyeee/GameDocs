## 二、无武器/防具系统

**帕夏时代无战斗系统，无武器，无防具。**

洞穴探索通过解谜和工具(斧头/锄头/水壶)破障推进，无任何战斗装备。

## 六、工具升级路线总图

```
Tier 1: 手斧(Handaxe) — 万能初始，砍树/采矿/锄地/钓鱼
  ↓ 获得燧石 (矿洞, 春4+)
Tier 2: 燧石系列(Flint) — 升级斧/锤
  ↓ 16,000繁荣度+"黑曜石加工"
Tier 3: 黑曜石系列(Obsidian) — 全工具升级
  ↓ 70,000繁荣度+"锻造"
Tier 4: 铜系列(Copper) — 全工具升级
  ↓ 120,000繁荣度+"冶炼"
Tier 5: 青铜系列(Bronze) — 最高级工具
```
