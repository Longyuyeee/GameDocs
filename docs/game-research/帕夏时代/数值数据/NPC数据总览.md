## 三、好感度阶段与解锁

| 花级 | 关系 | 解锁内容 |
|:---:|------|------|
| 0-1♥ | 陌生人 | 基础对话 |
| 2♥ | 相识 | 2花心意事件 |
| 4♥ | 朋友 | 4花心意事件 |
| 6♥ | **可约会** | NPC主动询问是否成为情侣，可多人约会无惩罚 |
| 8♥ | 亲密 | 8花心意事件 |
| 10♥ | **可结合(Union)** | 需中型小屋+结合花环(Union Wreath)→4天后仪式 |

## 四、社交规则

| 规则 | 说明 |
|------|------|
| 送礼频率 | 每日最多1次，每周最多2次同一角色 |
| 对话 | 每日首次+好感度 |
| 好感衰减 | 长期不理睬会下降 |
| 节日互动 | 共舞/参与活动+额外好感度 |
| 性别限制 | 无(任何性别可与任何性别角色恋爱) |
| 嫉妒机制 | 无(可同时多人约会) |
| 婚姻限制 | 只能与1人结婚(结合) |
