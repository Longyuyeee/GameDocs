# 怪物数据总览 - 帕夏时代(Roots of Pacha)

> 游戏版本: Roots of Pacha v1.3+ (PC/Switch/PS4/PS5)

## 无战斗系统，无怪物

帕夏时代是**纯和平主义**石器时代部落经营游戏，不包含任何战斗系统或敌对生物。

- 无怪物/敌人生成
- 无Boss战
- 无HP/伤害计算
- 无掉落表

## 替代内容

洞穴探索中的"挑战"完全由以下非战斗元素构成:

| 元素 | 说明 |
|------|------|
| 图腾谜题 | 雕齿兽/猫头鹰/猴子/熊/马——每个图腾有独特的解谜机制 |
| 工具障碍 | 藤蔓(需斧头)、杂草(需锄头)、裂隙(需水壶射程)——检查工具等级 |
| 暗房 | 需火把点亮石柱→照明路径 |
| 动物调谐 | 长笛节奏小游戏——驯化动物的技能挑战 |

由于无战斗/怪物系统，本节仅做标注说明。
