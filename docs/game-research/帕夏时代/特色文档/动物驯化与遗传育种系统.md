# 动物驯化与遗传育种系统 - 帕夏时代(Roots of Pacha)

> 游戏版本: Roots of Pacha v1.3+ (PC/Switch/PS4/PS5)

## 一、概述

帕夏时代的动物系统是整个牧场游戏品类中**最深层的动物养成设计**。与"商店买动物→喂养→收获"的传统模式不同，帕夏的动物系统包含: **野外调谐驯化→饲养→多代遗传育种→颜色变异收集** 四层深度。

## 二、驯化(Domestication): 从野生到部落

### 2.1 驯化流程

```
野外遇到动物 → 触发"动物友谊"创意
  → 交付Okka 20植物纤维 → 获得长笛(Flute)
  → 对目标动物使用长笛"调谐(Attune)"
  → 节奏小游戏(白色圆环收缩至内圈时点击)
  → 表现评级→1-2♪音符进度
  → 每日最多4次调谐(长笛绿色圆点指示)
  → 完全填满♪进度条 → 邀请动物回村
  → 建造畜棚 → 动物成为部落成员
```

### 2.2 与传统"购买动物"的对比

| 维度 | 购买动物(星露谷等) | 调谐驯化(帕夏) |
|------|------------------|--------------|
| 获取方式 | 商店金币购买 | 野外长笛调谐 |
| 技能元素 | 无 | 节奏小游戏 |
| 获取限制 | 金币够就任意买 | 每日4次调谐×多天 |
| 叙事意义 | "我买了头牛" | "我与这头牛建立了联系" |
| 动物来源 | 商店无限供应 | 野外特定生物群系特定季节 |

## 三、遗传育种(Genetic Breeding)

这是帕夏动物系统最深层的机制，也是与所有其他牧场游戏的最大差异。

### 3.1 繁殖条件

- 同物种 × 异性 × 双方均6♥好感度
- 建造繁殖围栏(Breeding Pen): 50燧石+30木+20植物纤维
- 需~50,000繁荣度

### 3.2 属性继承

```
子代属性 = 双亲属性均值 + 随机 +1点提升
```

三大可继承属性:

| 属性 | 影响 | 代际提升 |
|------|------|------|
| 基础品质(Base Quality) | 产出物品质等级 | 每代+1(随机) |
| 基础产量(Base Production) | 产出频率(如每天→每天2次) | 每代+1(随机) |
| 基础速度(Base Speed) | 坐骑移动速度 | 每代+1(随机) |

- 每只动物最多繁殖**3次**(同代限制)
- 多代选择性育种可实现**大幅属性提升**

### 3.3 育种策略深度

```
Gen1: 抓到野生野猪(速度2) + 野生野猪(速度2)
  → Gen2: 速度3-4的野猪(继承+随机提升)
    → Gen3: 速度5-6的野猪
      → Gen4: 速度7-8的竞赛级坐骑
```

这种多代优化机制让动物系统从"产资源的工具"变成了"可培养的血脉"。

## 四、颜色变异(颜色图鉴)

### 4.1 稀有颜色(Rare Colors)
- 条件: 已驯化8只动物 + 同物种4种颜色变体
- 刷新: 特定季节特定日期

### 4.2 传说颜色(Legendary Colors)
- 条件: 已驯化12只动物 + 同物种6种颜色变体 + **第2年起**
- 特定刷新时间:

| 传说变体 | 物种 | 日期 | 季节 |
|---------|------|:--:|:--:|
| 火焰野猪 | 野猪 | 6日 | 夏/冬 |
| 星空草原马 | 马 | 18日 | 春/秋 |
| 蜘蛛山羊 | 山羊 | 26日 | 春/秋 |
| 彩虹鸵鸟 | 鸵鸟 | 14日 | 夏/冬 |
| 冰霜原驼 | 原驼 | 2日 | 春/秋 |

传说颜色收集是帕夏时代最具挑战性的终局目标之一。

## 五、完整Animal Pipeline

```
野外发现(探索)
  → 长笛调谐(节奏技能)
  → 邀请回村
  → 建造畜棚
  → 每日喂食+抚摸(好感度养成)
  → 使用工具采集(挤奶/剪毛/捡蛋)
  → 加工产品(奶酪/纱线/肥料)
  → 6♥+异性配对→繁殖围栏→子代属性提升
  → 多代选育→竞赛级坐骑/高品质产出
  → 第2年→传说颜色追踪收集
```

## 六、设计分析

1. **长笛调谐=主动驯化**: 将"购买动物"转化为基于节奏技能的小游戏，赋予动物获取仪式感和技能门槛
2. **遗传育种是完整的养成系统**: 多代属性遗传不是简单的外观装饰——每代都有可量化的性能提升
3. **繁殖次数上限(3次)**: 防止无限育种导致属性通胀，同时要求策略性选择配对
4. **传说颜色=Y2驱动力**: 第2年起的限定刷新提供重新造访旧区域的理由
5. **坐骑速度竞赛**: 竞赛日(每季2次)让"繁殖高速坐骑"从个人满足变成竞技目标
6. **动物不卖**: 部落共同体不会"卖动物"——驯化的动物是部落成员，不是商品
