# 创意发明与科技树系统 - 帕夏时代(Roots of Pacha)

> 游戏版本: Roots of Pacha v1.3+ (PC/Switch/PS4/PS5)

## 一、概述

帕夏时代的"科技树"以**创意(Ideas)系统**呈现。与传统的"花钱买配方/买设备"不同，所有技术进步都是部落成员"想出"的——NPC会产生创意(烟雾信号)→玩家提供材料帮助实现→等待研发→新技术解锁。

这一设计将"技术进步"从经济消费变成了**社区协作叙事**，完美呈现了人类从石器时代走向金属时代的过程。

## 二、创意系统机制

### 2.1 创意触发类型

| 触发方式 | 示例 | 设计逻辑 |
|---------|------|------|
| **发现新材料** | 矿洞找到燧石→"一块燧石"→燧石工具 | 材料驱动发明 |
| **执行新行为** | 与野生动物互动→"动物友谊"→长笛 | 行为驱动发明 |
| **前置创意完成** | 发酵罐→"奶酪布"→奶酪制作 | 科技树链条 |
| **日期到达** | 春11→"日晷"→时间显示 | 时间驱动 |
| **繁荣度达标** | 24,000→"压榨机"→果汁/油 | 部落整体进步 |

> 所有创意**不会错过**——系统确保最终都能触发

### 2.2 创意执行流程

```
NPC产生想法(头上冒烟雾信号)
  → 与NPC对话→接收材料清单
  → 收集并交付材料
  → 等待研发天数(通常1-2天)
  → 新配方/建筑/工具/能力解锁
  → (部分)触发下一个连锁创意
```

## 三、科技演进史: 从石器到金属

### 阶段一: 采集时代(春Y1初期)

```
可用技术: 手斧(万能工具)、水袋(基础浇水)
获取食物: 野外采集+基础钓鱼(手斧)
加工: 太阳干燥器(最原始的食品保存)
```

### 阶段二: 驯化革命(春Y1中)

```
重大突破: 长笛→动物驯化→畜牧
新能力: 动物骑乘(移动革命)
新建造: 小型畜棚
```

### 阶段三: 石器加工(春Y1后期)

```
重大突破: 燧石→黑曜石加工→工具站
材料升级: 燧石工具→黑曜石工具
新建造: 加工站(压榨机/研钵/发酵罐)
```

### 阶段四: 发酵与加工(夏Y1)

```
重大突破: 发酵→酿造→奶酪
加工链: 奶→酸奶→奶酪 / 果汁→葡萄酒 / 蜂蜜→蜂蜜酒
新经济: 贸易(与外族交换)
```

### 阶段五: 金属时代(秋Y1+)

```
重大突破: 铜矿发现→锻造→冶炼
材料升级: 铜工具→青铜工具
新建造: 灌溉系统/谷仓/木桶陈酿
```

### 阶段六: 终局技术(Y2+)

```
重大突破: 木桶陈酿→泡菜系统→水车自动化
高级加工: 陈酿葡萄酒/奶酪/香料酒
作物进化: 太阳与月亮矿洞→进化作物
```

## 五、设计分析

1. **科技树作为叙事**: 从太阳干燥→发酵→酿造→陈酿的技术进化路径，就是人类食品保存技术的发展史
2. **NPC专业化**: 每个NPC有专长领域→让部落中每个成员都有独特贡献→强化"部落共同体"主题
3. **创意不可跳过**: 高级技术的前置条件链条长(泡菜需要: 发酵罐→酿造器→醋发酵罐→泡菜器)，确保玩家经历完整技术演进
4. **煤炭时代的跳过**: 帕夏从燧石→黑曜石→铜→青铜(跳过铁→钢)，因为部落共同体规模适于青铜时代的技术上限
5. **无"无用"创意**: 每条创意都解锁有意义的新能力/配方，没有"为了填充科技树"的凑数项
6. **材料收集作为"研究成本"**: 不是花钱买科技，而是收集特定材料→保留了探索/采集的驱动
