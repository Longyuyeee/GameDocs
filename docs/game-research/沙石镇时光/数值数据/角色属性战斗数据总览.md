# 角色属性战斗数据总览 - 沙石镇时光(My Time at Sandrock)

> 游戏版本: My Time at Sandrock v1.0+ (PC/Switch/PS5/Xbox)
> 数据来源: mytimeatsandrock.fandom.com, TechRaptor, NeoSeeker

## 一、基础属性

| 属性 | 含义 | 影响 |
|------|------|------|
| HP | 生命值 | 归零=晕倒(野外)/游戏结束(危险废墟) |
| 体力(Stamina) | 工作能量 | 采矿/伐木/采集/战斗消耗 |
| 耐力(Endurance) | 翻滚/特殊动作 | 翻滚消耗/攻击恢复 |
| 韧性(Toughness) | 破防/减伤 | 降低承受伤害/抵抗击退 |
| 攻击力(Attack) | 物理伤害基础 | 武器伤害+技能加成 |
| 暴击率(Crit Chance) | 暴击概率 | Strong Attack Lv1=+5% |
| 暴击伤害(Crit Damage) | 暴击倍率 | 基础+技能+武器加成 |
| BREAK | 破防值 | 累积破防→额外伤害 |
| 防御力(Defense) | 物理减伤 | 防具+技能 |

## 四、伤害计算公式(推测)

```
最终伤害 = (武器基础伤害 + 攻击力加成) × (1 + 暴击伤害加成%) × 部位修正 - 目标韧性/防御减免
暴击判定: 暴击率% → 成功 = 伤害×暴击倍率
BREAK判定: 累积BREAK值 ≥ 目标韧性 → 破防 → 额外伤害窗口
```

## 五、危险废墟评分计算

```
总得分 = 击杀分(普通100/精英300) + Boss分(1,500) - 时间惩罚 - 治疗惩罚
S/A/B/C 评级 = 基于总得分区间
```
