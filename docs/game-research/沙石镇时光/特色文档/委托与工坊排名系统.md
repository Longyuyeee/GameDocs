# 委托与工坊排名系统 - 沙石镇时光(My Time at Sandrock)

> 游戏版本: My Time at Sandrock v1.0+ (PC/Switch/PS5/Xbox)

## 一、概述

委托(Commissions)系统是沙石镇时光的**核心游戏循环**，替代传统牧场游戏的耕种/畜牧出货。玩家作为工坊主(Builder)，每天从商业公会布告栏接取委托→制造/收集所需物品→交付→获得戈尔+公会徽章+工坊声望。

这一系统将Sandrock与其他所有牧场游戏从根本上区分开来——你不是"农夫"，你是"承包商"。

## 二、委托系统机制

### 2.1 委托流程

```
清晨起床 → 检查当日委托板 → 接取能力范围内的委托
  → 检查所需材料库存 → 不足→采矿/采集/回收
  → 机器制造/组装站组装 → 前往委托人交付
  → 获得戈尔+公会徽章+声望经验
```

### 2.2 委托要素

| 要素 | 说明 |
|------|------|
| 刷新 | 每日刷新 |
| 酬金 | 100-2,000+戈尔(随难度/稀有度) |
| 类型 | 制造物品/提供材料/修复建筑等 |
| 时效 | 大部分不限时(但同日交付有加成) |
| 城市委托 | 每周高酬金特殊委托 |
| 委托人类别 | 个人委托(NPC)vs城市委托(市政) |

### 2.3 接取策略

| 因素 | 考量 |
|------|------|
| 可完成性 | 只接能当天或近期完成的→避免取消惩罚 |
| 酬金效率 | 高酬金+你已经囤积材料→最优 |
| 同时接取数 | 排名越高同时可接越多(最高3个) |
| 同日交付 | Quick Delivery知识+5-10%报酬 |
| 高品质 | Quality Bonus知识+10%报酬 |

## 三、工坊排名系统

### 3.1 排名机制

| 要素 | 说明 |
|------|------|
| 评定周期 | 每月 |
| 评定标准 | 当月完成委托数量+品质+难度 |
| 排名区间 | 第1名/前3名/前10名/中游/末位 |
| 排名奖励 | 公会徽章+额外委托槽+声望 |

### 3.2 各排名收益

| 排名 | 委托槽 | 额外收益 |
|:--:|:-----:|------|
| 第1名 | 3个 | 最多公会徽章/Chart-topper技能加成 |
| 第2-3名 | 2-3个 | Chart-topper+5-10%声望/经验 |
| 中游 | 1-2个 | 基础 |
| 末位 | 1个 | 最低收益 |

### 3.3 竞争NPC

玩家并非唯一工坊主。**Mi-an**(美安)是主要竞争对手——她也在完成委托。**Yan**(商业公会管理员)在后期揭示为Duvos成员。

## 四、公会徽章经济

| 用途 | 说明 |
|------|------|
| 公会商店 | 兑换稀有材料/蓝图/物品 |
| 排名奖励 | 排名越高获得越多 |
| 委托奖励 | 每次交付额外获得 |

## 五、知识技能深度绑定委托效率

| 技能 | 效果 |
|------|------|
| Reputation Knowledge | +10%双倍社交经验(委托来源) |
| Quick Delivery Lv1-2 | 同日交付+5%戈尔/+10%经验 |
| Advertising Lv1-3 | 广告+10%声望/经验/戈尔 |
| Chart-topper Lv1-2 | Top3排名+5%声望/+10%经验 |
| Commissions & Contracting Lv1-3 | +1委托槽/合同资源+20%/民兵团费用-20% |
| Quality Bonus Lv1-2 | 高品质交付+10%经验/+10%戈尔 |

## 六、设计分析

1. **委托替代耕种作为核心循环**: 从"种地→出货箱→金币"变为"接委托→制造→交付→戈尔"——完全不同的节奏
2. **排名创造竞争感**: 每月排名结算提供周期性目标和压力——“这个月能冲到第1吗？”
3. **多品质交付体系**: 不是"做出来就行"而是"做得更好给更多钱"→品质机制有经济意义
4. **NPC竞争对手(Mi-an)**: 不是静态排名而是有活生生的竞争对手→增强沉浸感
5. **知识技能全面服务委托**: 社交知识树几乎全部围绕委托效率设计——build方向明确
6. **与星露谷"特殊订单"对比**: 星露谷的Special Orders是可选额外，Sandrock的委托是主线
