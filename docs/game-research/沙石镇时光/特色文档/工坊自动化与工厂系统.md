# 工坊自动化与工厂系统 - 沙石镇时光(My Time at Sandrock)

> 游戏版本: My Time at Sandrock v1.0+ (PC/Switch/PS5/Xbox)

## 一、概述

沙石镇时光的工坊从**手动操作**逐步进化到**工厂自动化**。这不仅是QoL升级，而是整个游戏节奏的转变——从"亲手操控每台机器"到"设置生产策略→机器自动执行"。

## 二、机器运作逻辑

### 2.1 机器独立性

每台机器是**独立的运作单元**:
- 有自己的生产队列(可排队多个订单)
- 消耗**水(Water)**作为冷却/运转介质
- 消耗**燃料(Fuel)**(木头/动力石等)提供动力
- 缺水/缺燃料→停机→不生产

### 2.2 时间连续性

机器在游戏时间中**持续运作**:
- 不因睡觉而"跳过"
- 睡前排队→睡觉期间→早上产出
- Feather Duster可加速机器速度(+50%)

## 三、自动化演进三阶段

### 阶段一: 手动时代(初期)

```
所有机器手动操作
玩家: 采集→放入机器→等待→取出→重复
限制: 需要时刻在工坊附近
```

### 阶段二: 排队优化(中期)

```
知识技能+高级机器+多台并行
玩家: 批量设置生产队列→外出做委托→回来收获
关键技能: Time Management(机器+20%速度)、Water Conservation(节水)
```

### 阶段三: 工厂自动化(终局)

```
主线"Sandrock Strikes Back"后解锁
工厂(Factory):
  - 20制造批次+20回收任务同时
  - 需火力发电机供电
  - 产出自动送入交付箱
  - 代价: 不享受品质加成
```

## 四、工厂vs手动: 策略抉择

| 维度 | 手动(工坊庭院) | 工厂(Factory) |
|------|:---------:|:--------:|
| 操作量 | 高(每台机器单独操作) | 低(中控面板批量设置) |
| 生产速度 | 基准+技能加成 | 基准 |
| 品质加成 | **享受全部品质技能** | **不享受** |
| 容量 | 每台机器独立容量 | 20+20批量 |
| 供电 | 每台独立燃料 | 集中火力发电机 |
| 最佳用途 | 高品质委托物品 | 批量生产基础材料 |

**策略结论**: 高品质关键物品→手动；基础材料/回收→工厂。两者互补而非替代。

## 五、机器品质与精炼

### 5.1 机器品质

| 品质 | 效果 |
|------|------|
| 普通 | 基准产出 |
| 优秀 | 概率更高品质产出 |
| 完美 | 更高概率+额外产出概率 |

- 高品质机器通过组装站技能加成获得

### 5.2 精炼(Refining)装备

- 精炼机(Refiner)消耗材料提升装备属性
- 可重复精炼(无上限但消耗递增)
- Advanced Refining: 5%免费/5%双倍效果

## 六、关键自动化相关知识技能

| 技能 | 效果 | 重要性 |
|------|------|:----:|
| Time Management Lv1-3 | 机器速度+10/20% | ★★★★★ |
| Water Conservation Lv1-3 | 节水+10/20%/露水-2 | ★★★★★ |
| Fuel Saving Lv1-2 | 燃料-10/20% | ★★★★ |
| High Quality Assembly Lv1-2 | 组装站高品质概率 | ★★★★ |
| High Quality Worktable Lv1-2 | 工作台高品质概率 | ★★★★ |
| Advanced Building Lv1 | 再提升机器品质概率 | ★★★ |
| Efficient Filtration Lv1-2 | 回收/精炼20%额外产出 | ★★★ |

## 七、设计分析

1. **从小到大**: 手动→排队→工厂——自动化是"成长"的一部分，而非初始就有
2. **品质vs效率的交易**: 工厂快但不出品质——保留了手动操作的价值
3. **水=能效**: 用水而非电力驱动机器——沙漠设定中的资源约束设计
4. **Feather Duster加速**: 主动技能加速机器——保留玩家"手动干预"的快感
5. **火力发电机**: 终局工厂需要独立供电设施→促使玩家建造完整工业链
