# 沙漠废土生存主题 - 沙石镇时光(My Time at Sandrock)

> 游戏版本: My Time at Sandrock v1.0+ (PC/Switch/PS5/Xbox)

## 一、概述

沙石镇时光是牧场经营品类中**最极端的设定偏离**。从星露谷的翠绿田园、珊瑚岛的热带海岛、帕夏时代的史前森林，到Sandrock的后末日沙漠废土——主题从"经营一个农场"彻底转变为"在沙漠中重建文明"。

## 二、主题如何影响每个系统

| 系统 | 传统牧场(星露谷) | 沙石镇时光 |
|------|:------------:|:--------:|
| 核心职业 | 农夫(Farmer) | **工坊主(Builder)** |
| 核心活动 | 耕种/浇水/收获 | 接委托/制造/组装/交付 |
| 水 | 体力消耗(浇水) | **机器燃料**(稀缺资源) |
| 种植 | 核心经济 | 辅助材料来源 |
| 动物 | 商店购买 | 商店购买(养殖非核心) |
| 资源获取 | 采矿+采集 | **废墟采矿+废料回收** |
| 进度驱动 | 季节/节日 | **主线任务/工坊排名** |
| 环境 | 绿色田园 | **沙漠废土** |
| 故事 | 社区发展 | **反谍战/英雄反转** |

## 三、沙漠主题的机制表达

### 3.1 水=生命线
- 所有机器消耗水→缺水=停机
- 不是"每天浇水"而是"持续供水"
- 宠物收集水/露水收集器/水世界购买→多渠道但都有限

### 3.2 废料回收=资源核心
- 不是"采矿→冶炼"而是"废墟→废料堆→回收机→材料"
- 旧世界(Old World)废墟提供文明崩溃后的遗物
- 数据碟——旧世界科技的数据载体→研究新蓝图

### 3.3 生物结皮=土壤改造
- 沙漠不是天然可耕种→需要稻草网格
- 反复种植改良土壤→最终形成生物结皮(Biocrust)
- 叙事: "把沙漠变成绿洲"而非"在肥沃土地上轻松耕种"

### 3.4 季节弱化
- 沙漠没有春夏秋冬的剧烈变化
- 大部分作物全年可种
- 用"主线推进"替代"季节推进"

## 四、后末日世界观的叙事表达

| 元素 | 说明 |
|------|------|
| 旧世界(Old World) | 科技文明崩溃后留下的废墟 |
| 数据碟(Data Discs) | 旧世界科技数据→研究中恢复蓝图 |
| 废墟(Ruins) | 旧世界建筑残骸→资源/遗物/危险 |
| 自由城市联盟 | 幸存者建立的松散城市联盟 |
| 杜沃斯(Duvos) | 想吞并联盟的军事帝国 |
| 教堂(Church of Light) | 后末日新兴宗教 |
| 工坊主(Builder) | 重建文明的关键职业 |

## 五、与星露谷的"田园牧歌"对比

| 维度 | 星露谷 | 沙石镇 |

## 六、设计启示

1. **主题驱动机制创新**: 沙漠→水稀缺→机器耗水→水管理成为核心策略→颠覆"浇水"的传统设计
2. **废土≠末日绝望**: Sandrock是"重建中的废土"而非"垂死的废土"——希望感才是经营游戏的根本
3. **旧世界遗物=新世界科技**: 数据碟/废墟的设定巧妙解释了"为什么古代人比我们先进"
4. **工坊主身份重构**: 脱离"农夫"身份→吸引对种植不感兴趣的玩家群体
5. **主线驱动有助于叙事深度**: 传统牧场"无剧情终点"→Sandrock"有主线高潮"→完成后的后日谈更有满足感
