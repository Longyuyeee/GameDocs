# 工坊与建造系统 - 沙石镇时光(My Time at Sandrock)

> 游戏版本: My Time at Sandrock v1.0+ (PC/Switch/PS5/Xbox)
> 数据来源: mytimeatsandrock.fandom.com, Fandom Wiki, Steam, GamerHour

## 1. 概述

**工坊(Workshop)是沙石镇时光的核心系统，替代传统牧场游戏的农场。** 玩家不是农夫，而是"工坊主(Builder)"——通过组装站制作机器和物品，完成商业委托来发展工坊。

## 2. 组装站(Assembly Station)

### 2.1 核心机制

- 从**手册(Handbook)**选择蓝图→放置在组装站上
- 从背包/仓库放入所需材料
- **即时完成**(Lv1-2)，Lv3解锁自动组装(批量生产)

### 2.2 组装站升级

| 等级 | 名称 | 尺寸 | 关键功能 |
|:--:|------|:---:|------|
| Lv1 | 基础组装站(Basic) | ~9×10格 | 基础机器/物品制作 |
| Lv2 | 中级组装站(Intermediate) | ~12×13格 | 中期主线必需品 |
| Lv3 | 高级组装站(Advanced) | ~21×14格 | **解锁自动组装**: Lv1蓝图45分钟/Lv2蓝图1小时 |

- 升级地点: Construction Junction(Heidi的店，市政厅旁)
- 升级需要戈尔+材料

## 4. 工厂(Factory) — 终局自动化

- 解锁: 完成主线"Sandrock Strikes Back"后
- 需要: 工厂控制面板+工厂信息板(Construction Junction购买)
- 必须放置在**房间(Room)**内
- 可同时处理**20个制造批次+20个回收任务**
- 需火力发电机供电
- **注意**: 工厂内机器不享受品质加成(需要加成→放在外面手动运行)

## 5. 蓝图获取

| 来源 | 说明 |
|------|------|
| Qi研究中心(Research Center) | 用数据碟(Data Discs)换取研究→获得新蓝图 |
| 主线任务奖励 | 关键机器蓝图由主线推进解锁 |
| 支线任务 | 特殊蓝图 |
| 商店购买 | Construction Junction等 |

## 6. 手册(Handbook)

- 组织方式: 全部(All)/机器(Machine)/其他(Other)
- 颜色标记: 🟢绿色=基础 / 🔵蓝色=中级 / 🟣紫色=高级
- 新蓝图自动加入手册

## 8. 关键知识技能(Workshop Tree)

| 技能 | 效果 |
|------|------|
| 学习者(Learner Knowledge) | +10%组装站经验 |
| 高品质组装(High Quality Assembly) | 提高组装站产出高品质概率 |

| 量产(Mass Production) | 工作台/机器+10%经验 |

## 9. 设计特点

1. **即时组装**: 非等待型制作(不占时间)，与星露谷/帕夏的"等待N小时"不同
2. **机器链**: 回收机→熔炉→研磨机→处理器→高级工作台——线性升级而非并行
3. **蓝图研究门控**: 核心机器通过数据碟研究解锁，而非等级/金币门控
4. **工厂=终局自动化**: 后期自动化解放手动操作但牺牲品质——有策略选择
5. **工坊定位**: 玩家是"工坊主"而非"农夫"——作物/养殖仅为辅助
