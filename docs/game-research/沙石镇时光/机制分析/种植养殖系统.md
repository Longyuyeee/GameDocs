# 种植与养殖系统 - 沙石镇时光(My Time at Sandrock)

> 游戏版本: My Time at Sandrock v1.0+ (PC/Switch/PS5/Xbox)
> 数据来源: mytimeatsandrock.fandom.com, TheGamer, TechRaptor, Gamer Guides

## 1. 概述

沙石镇时光中**种植和养殖是辅助系统**，而非核心玩法。核心是工坊和委托。种植在主线任务"The Grass is Greener"后解锁，沙漠环境需要特殊机制(生物结皮/稻草网格)。

## 2. 种植解锁流程

```
主线: The Old Man and the Tree
  → Moisture Farm Blues (帮Zeke修水培农场)
  → The Grass is Greener (获得基础种植套件蓝图)
  → 制作基础种植套件: 3铜棒+6青铜锭+3橡胶管+2旧零件
  → 拜访Zeke→种植教程→获得首批沙米种子
```

## 3. 种植机制

### 3.1 播种流程
1. 进入种植模式(持基础种植套件)
2. 放置**稻草网格(Straw Grid)**——每格1稻草(Straw)
3. 选择种子种植(部分作物需2×2或3×3网格)
4. 浇水(蓝色水分条，不干透即可)
5. (可选)施肥
6. 成熟后收获

### 3.2 生物结皮(Biocrust) — 土壤升级

| 等级 | 土壤类型 | 所需经验 | 最大施肥量 | 特点 |
|:--:|------|:-----:|:--------:|------|
| Lv1 | 沙质土壤(Sandy Soil) | 0 | 2 | 初始 |
| Lv2 | 干旱土壤(Arid Soil) | 100 | 3 | — |
| Lv3 | 肥沃土壤(Fertile Soil) | 400 | 4 | — |
| Lv4 | **生物结皮(Biocrust)** | 1,000 | 5 | **无需稻草网格即可种植** |

- 每次收获贡献土壤经验
- **玫瑰柳(Rose Willow)**最佳: 200土壤经验/次
- 生物结皮可挖掘并重新放置

## 4. 养殖

- 建造畜棚后可饲养动物(鸡/牛/羊等)
- 产出蛋/奶/毛等副产物
- "畜棚知识(Barnyard Knowledge)"技能: 牲畜出售价格+20%
- 养殖在Sandrock中定位为补充性收入来源

## 5. 设计与传统牧场对比

| 维度 | 传统牧场(星露谷) | 沙石镇时光 |
|------|:------------:|:--------:|
| 核心玩法 | 耕种/畜牧 | 工坊/委托 |
| 种植定位 | ★★★★★ 核心 | ★★ 辅助 |
| 土壤系统 | 无需土壤升级 | 4级生物结皮升级 |
| 种植方式 | 自由翻耕 | 稻草网格限制 |
| 水资源 | 每日浇水(体力消耗) | 间歇浇水(机器消耗) |
| 动物 | 核心收入来源 | 补充收入 |
| 施肥 | 提高品质/产量 | 提高产量(效果争议) |

## 6. 设计特点

1. **沙漠种植的叙事一致性**: 稻草地网格+生物结皮——不是"随便翻块地就能种"，必须有意识地改造沙漠土壤
2. **种植非核心**: 委托是收入主体，种植是材料补充和土壤改造
3. **生物结皮系统**: 通过反复种植改良土地——主题上契合"改造沙漠"的核心叙事
4. **树作物=长期投资**: 栗子/咖啡茶树20+天生长期+反复复收——需要长期规划
