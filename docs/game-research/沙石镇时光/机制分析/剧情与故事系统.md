# 剧情与故事系统 - 沙石镇时光(My Time at Sandrock)

> 游戏版本: My Time at Sandrock v1.0+ (PC/Switch/PS5/Xbox)
> 数据来源: mytimeatsandrock.fandom.com, Fandom Wiki, Neoseeker, Steam Community

## 1. 世界观设定

沙石镇(Sandrock)是**自由城市联盟(Free Cities Alliance)**中的一个沙漠小镇，位于波西亚(Portia)之后的世界末日废土背景。世界在大灾难后重建，工坊主(Builder)是重建文明的关键角色。

### 核心势力

| 势力 | 定位 |
|------|------|
| 自由城市联盟(Free Cities Alliance) | 幸存者城市共同体 |
| 沙石镇(Sandrock) | 沙漠前哨小镇，资源匮乏 |
| 杜沃斯(Duvos) | 军事帝国，敌对方，试图控制联盟 |
| 民兵团(Civil Corps) | 沙石镇自卫武装(Justice/Unsuur) |
| 教堂(Church of Light) | 宗教势力(Miguel) |

## 2. 主线剧情结构

### Act 1: 新工坊主的到来
- 玩家作为新任工坊主接替前任Mason
- 结识关键NPC(Mi-an/Justice/Heidi/Qi等)
- 修复基础设施(建造Eufaula Salvege起重机)
- 水危机出现: 修复Moisture Farm
- 首次接触: Geegler袭击

### Act 2: 洛根的"背叛"
- **Logan(洛根)**——曾经的英雄，现在炸毁水源、攻击火车
- 沙石镇视洛根为恐怖分子
- Gecko Station废墟探索→发现真相线索
- The Breach危险废墟攻略
- Duvos渗透迹象出现
- 关键人物秘密: Miguel/Pen/Yan/Matilda的身份开始揭露

### Act 3: 杜沃斯阴谋
- Duvos帝国的渗透企图暴露
- 关键叛变: Stu Matilda(市长)/Pen/Yan/Miguel实际上为Duvos效力
- Logan的真相: 他一直在对抗Duvos渗透→"英雄变罪犯"是叙事核心反转
- 沙石镇保卫战(Sandrock Strikes Back)
- 击败Duvos入侵→叛徒被驱逐/逮捕

### Act 4: 和平重建
- Logan回归→成为可攻略角色
- Duvos威胁消退→沙石镇进入和平重建期
- Musa(城市投资者)到访
- 工厂自动化解锁
- 后日谈: 持续的工坊经营/社交

## 4. 叙事驱动方式

| 机制 | 说明 |
|------|------|
| **主线任务锁定区域**: | 新废墟/区域必须通过主线任务解锁(非时间锁/好感锁) |
| **组装站能力门控**: | 主线要求建造特定物品→必须升级组装站→驱动工坊发展 |
| **工坊排名压力**: | 排名太低→较少委托→主线任务也可能受影响 |
| **关系门控(部分)**: | 特定NPC剧情(如Fang/Logan)需好感度触发 |

## 5. Logan叙事设计(核心反转)

Logan故事线是Sandrock最具叙事野心的设计:

```
前期: Logan炸毁水源/袭击火车 → 玩家和全镇追捕
中期: 废墟探索发现Duvos渗透证据 → 暗示Logan另有动机
后期: Duvos阴谋暴露 → Logan原来一直在对抗内鬼→从"恐怖分子"变为"守护者"
结局: Logan回归 → 英雄平反 → 成为可攻略角色
```

这一"误解→追捕→发现真相→英雄回归"的情感弧线是Sandrock叙事的核心驱动力。

## 7. 结局

- **单一主线结局**: 沙石镇保卫成功/Duvos被驱逐/Logan回归
- **开放后日谈**: 主线后继续经营/社交/探索/工厂自动化
- **无多结局分歧**: 与大多数农场游戏一致

## 8. 设计特点

1. **"废土重建"替代"田园牧歌"**: 后末日世界的新兴小镇——更成熟的叙事基调
2. **Logan反英雄弧线**: "误解一个好人→追捕他→发现真相→回归"是牧场游戏中最长线的角色叙事
3. **内部渗透阴谋**: Stu Matilda/Pen/Yan/Miguel为Duvos内应——"你以为的好人实际上是坏人"的谍战反转
4. **工坊主的叙事身份**: 你不是"农夫"而是"工坊主"——建造/修复是核心叙事动作
5. **主线驱动探索**: 新区域→废墟→矿物→装备升级——主线/经营/探索三位一体
6. **道德灰色角色**: Miguel(教士+叛徒)/Pen(可攻略+反派)——非单纯好人坏人二分法
