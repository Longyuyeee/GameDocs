## 6. 气候/环境特色

| 元素 | 说明 |
|------|------|
| 沙漠风暴(Sandstorms) | 定期发生，室外活动受限 |
| 无四季变化 | 沙石镇是沙漠，无春夏秋冬——以"剧情推进"替代"季节推进" |
| 昼夜循环 | 6AM-2AM，2AM强制睡觉 |
| 水资源稀缺 | 场景主题——沙漠环境，水是最稀缺资源 |

## 7. 设计特点

1. **沙漠废土设定**: 替代传统牧场的绿色田园，所有场景围绕"沙漠生存+废墟探索"
2. **无季节但有季节作物偏好**: 作物有"适宜季节"(但仍沿用4季计时)——春/冬更适合山地玫瑰
3. **废墟=新区域**: 每个新废墟解锁=新资源层级=新装备层级——驱动进度
4. **城镇功能性建筑分布**: 每个建筑有明确功能(NPC+商店)——非纯装饰
5. **工坊=家园核心**: 所有机器/储物/种植/养殖都在工坊庭院内——一个紧凑的自定义空间
