# Gordo与探索解锁系统 - 史莱姆牧场2(Slime Rancher 2)

> 游戏版本: Slime Rancher 2 v1.0+ (PC/Xbox)

## 1. Gordo=探索门控

Gordo是巨大的、不能移动的史莱姆——喂饱它们→爆炸消失→揭示隐藏通道/传送器/洞穴。它们是SR2的"软锁"——不是等级/金币门控，而是**资源准备门控**。

## 2. Gordo机制

| 参数 | 说明 |
|------|------|
| 体型 | 巨大(不可吸取) |
| 需求 | 约**30+食物**(匹配饮食偏好) |
| 投喂 | 吸取食物→发射到Gordo |
| 吃饱后 | 爆炸→掉落Plort+食物+揭示隐藏 |
| 最爱食物 | **效果×2**(15个最爱=30普通) |

| Gordo | 食物 | 解锁 | 重要性 |
|------|:--:|------|:--:|
| 粉色(Rainbow Fields西北) | 30+任意 | **余烬谷** | ★★★★★(第2区域) |

| 蜂蜜(星光滨) | 30+水果 | 隐藏区域 | ★★★ |

| 剑齿(粉雪崖) | 30+肉类 | 隐藏 | ★★★ |

## 4. Gordo的探索策略

1. **首次Gordo(粉色)**: 任意食物→最易→教学
2. **棉花Gordo**: 建立蔬菜花园后→喂卷心菜
3. **振翅Gordo**: **最难的Gordo**——月露花蜜仅在星光滨夜间出现→需专门采集→数十天准备

## 5. 设计分析

1. **Gordo=资源准备检查**: 不是"你等级够不够"而是"你有没有种植/收集足够食物"——经营检查
2. **振翅Gordo=终极经营检查**: 需要30+月露花蜜——保证玩家必须深入理解振翅饮食系统
3. **双路径进入灰色迷宫**: 振翅Gordo(星光滨)或爆炸Gordo(余烬谷)——不会卡死
