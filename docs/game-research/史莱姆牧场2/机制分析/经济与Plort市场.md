## 3. 动态市场价格

| 机制 | 说明 |
|------|------|
| **价格箭头** | 🟢上涨/🔴下跌——基于最近出售量 |
| **过度出售惩罚** | 同一Plort大量出售→价格持续下跌 |
| **多样化策略** | 养殖多种史莱姆→避免单一Plort跌价 |
| **储存等待** | 使用Silo大量储存→等价格回升→批量出售 |
| **Silo扩容** | Patch 1.2: 100→200/格 |

## 6. 设计特点

1. **Plort=唯一经济基础**: 不是"作物+动物+加工品"——只有Plort——经济简单但Largo系统深
2. **动态价格=反倾销**: 大量出售同一Plort→跌价——鼓励多元化牧场
3. **Silo=市场时机工具**: 储存→等价格回升→批量出售——增加经济策略层
4. **黄金/幸运=野外暴富机会**: 不可养殖但极稀有→遇到就是横财
