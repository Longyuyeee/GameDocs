# 吸尘背包与装备系统 - 史莱姆牧场2(Slime Rancher 2)

> 游戏版本: Slime Rancher 2 v1.0+ (PC/Xbox, EA 2022-2025)
> 数据来源: TheGamer Vacpack Guide, GitHub Walkthrough

## 1. 吸尘背包(Vacpack)——核心工具

吸尘背包是FPS视角的核心操作装置——吸取/发射/扫描/飞行一体化。

| 功能 | 说明 |
|------|------|
| 吸取(Vac) | 吸取史莱姆/食物/Plort/资源 |
| 发射(Shoot) | 发射吸取的物品 |
| 扫描(Scan) | 扫描新物品→解锁百科/蓝图 |
| 飞行(Jetpack) | 短距离飞行(需升级) |
| 水枪(Water Tank) | 喷射水→消灭Tarr/清理 |

## 5. 设计特点

1. **吸尘背包=FPS核心**: 吸取/发射/飞行/水枪——一个装置四合一——完美的FPS牧场界面
2. **容量管理=核心决策**: 6个槽×100单位→地牢式背包管理应用到"牧场采集"
3. **自动喂食+收集=半自动化**: 围栏升级→减少日常手动操作——但不如僵尸自动化极端
4. **黄金/暗影瞄准=终局装备**: 锁定稀有史莱姆→获取最高价值Plort
