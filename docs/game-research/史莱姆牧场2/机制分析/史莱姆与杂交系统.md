## 2. 史莱姆杂交(Largo)系统

### 2.1 杂交机制

```
史莱姆A + 史莱姆B的Plort(结晶) → 杂交Largo
Largo = 两种史莱姆的特征+饮食偏好+产物
```

- Largo同时产出**两种Plort**
- Largo继承两种史莱姆的**饮食偏好**(例如: 虎斑-棉花Largo=吃肉类或蔬菜)
- Largo**不能再次杂交**
- Largo体型更大(需要更多空间)

### 2.2 塔arr(Tarr)威胁

| 条件 | 结果 |
|------|------|
| Largo吃**第三种**Plort | → 变为**Tarr**(黑色彩虹怪物) |
| Tarr接触任何史莱姆 | → 复制→失控→全牧场毁灭 |
| Tarr弱点 | **水**(水枪喷射→消灭) |

> ⚠️ Tarr是牧场管理失败的最大惩罚——必须防水枪随时应对

### 2.3 最优Largo组合

| 组合 | 饮食 | 价值 | 难度 |
|------|:--:|:--:|:--:|
| 蜂蜜-棉花 | 蔬菜+水果 | 中高 | 易 |
| 水晶-爆炸 | 蔬菜+肉类 | 中高 | 中(爆炸伤害) |
| 振翅-蜂蜜 | 月露+水果 | **最高** | 难(振翅专属饮食) |
| 双子-虎斑 | 杂食+肉类 | 高 | 终局 |

## 4. 设计特点

1. **Largo=双倍产出+双倍风险**: 高效(两种Plort)→但Tarr威胁→需防水枪+精心管理
2. **振翅=最贵但最难的史莱姆**: 仅吃月露花蜜(星光滨夜间专属采集)→极高Plort价值
3. **Tarr=经营失败的惩罚**: 不是"动物生病"而是"整个牧场被Tarr吞噬"——最极端的失败状态
4. **蛋黄=极稀有+功能性**: 不要卖出→放入鸡舍→产巨蛋(孵出更多鸡→更多肉类供应)
