# 牧场管理与自动化 - 史莱姆牧场2(Slime Rancher 2)

> 游戏版本: Slime Rancher 2 v1.0+ (PC/Xbox)
> 数据来源: GitHub Walkthrough, TheGamer, Official Wiki

## 1. 牧场设施

| 设施 | 功能 | 最大建造数 |
|------|------|:--:|
| 围栏(Corral) | 饲养史莱姆(最多8-10只普通/4-5只Largo) | 多个 |
| 花园(Garden) | 种植水果/蔬菜 | 多个 |
| 鸡舍(Coop) | 养殖鸡肉→供肉食史莱姆 | 多个 |
| 筒仓(Silo) | 储存Plort/食物(200/格) | 多个 |
| 焚化炉(Incinerator) | 火焰史莱姆的灰烬盘 | 1-2 |
| 池塘(Pond) | 水坑史莱姆 | 1-2 |
| 量子无人机站(Quantum Drone) | **自动化** | 每牧场扩展 |

## 3. 量子无人机(自动化核心)

### 3.1 功能

| 无人机类型 | 功能 |
|:--:|------|
| 基础无人机 | 设定"从X取→放到Y"单一任务 |
| 幽灵无人机(Ghostly Drone) | 巡逻/多任务 |
| 研究无人机(Research Drone) | 自动收集研究材料 |

### 3.2 自动流水线

```
花园(Sprinkler+Growth Amplifier)→自动生长
  ↓
自动喂食器(Auto-Feeder)→从筒仓取食物→喂围栏
  ↓
Plort收集器(Plort Collector)→自动收集Plort
  ↓
量子无人机→从收集器取Plort→放入筒仓或市场
  ↓
(完全自动化的被动收入!)
```

## 4. 日常管理循环

| 阶段 | 每日任务 |
|:--:|------|
| 手动时代 | 喂所有围栏→收集所有Plort→去市场出售→浇水花园→捡鸡蛋→重复 |
| 半自动时代 | 自动喂食+收集→只需出售+种植 |
| 全自动时代 | 无人机做几乎所有事→玩家仅需偶尔检查+探索 |

## 5. 设计特点

1. **量子无人机=终局自动化**: 类似守墓人的僵尸——将手动劳动转变为管理
2. **无人机任务编程**: "从围栏1的收集器→放到筒仓A"——简单的编程逻辑→高级自动化
3. **全自动流水线**: 花园→自动喂食→自动收集→无人机→出售——四步全自动
4. **奇异钻石=无人机门控**: 需要奇异钻石(Strange Diamonds)→极稀有资源→驱动探索
