# 战斗与Boss数据 - 魔女之泉R(Witch Spring R)

> 游戏版本: Witch Spring R (PC/Switch/PS5, 2023年)
> 数据来源: witch-spring.fandom.com, GameFAQs

## 一、战斗公式(推测)

| 公式 | 说明 |
|:--:|------|
| 物理伤害 = STR × 武器倍率 - 目标DEF | 物理Build基础 |
| 魔法伤害 = MAG × 技能倍率 - 目标MDEF | 魔法Build基础 |
| 暴击率 = AGI × 系数 | 敏捷影响暴击 |
| 命中/闪避 = AGI vs AGI | 速度对决 |

## 三、战斗消耗

| 行动 | 消耗 | 说明 |
|:--:|:--:|------|
| 普通攻击 | 0 | 无限使用 |
| 魔法技能 | MP | MP=0→只能用普攻 |
| 道具使用 | 消耗道具 | 战斗中使用制作的道具 |
| 宠物行动 | 宠物HP/TP | 宠物有独立资源 |

## 四、物理vs魔法Build战斗差异

| 维度 | 物理型(STR) | 魔法型(MAG) |
|:--:|:--:|:--:|
| 普通攻击 | 高伤害 | 低伤害 |
| 技能伤害 | 低-中(物理技能) | **高(AOE+爆发)** |
| 持续输出 | ★★★★★(无消耗!) | ★★★(MP限制) |
| Boss战 | ★★★★ | ★★★★★(爆发) |
| 杂兵清场 | ★★★★ | ★★★★★(AOE) |
| 装备依赖 | 高 | 中 |
