# 探索与收集系统 - 魔女之泉R(Witch Spring R)

> 游戏版本: Witch Spring R (PC/Switch/PS5, 2023年)
> 数据来源: witch-spring.fandom.com, GameFAQs

## 1. 探索区域

| 区域 | 类型 | 解锁 | 主要收获 |
|:--:|:--:|------|------|
| 森林(Forest) | 初始 | 开始 | 基础草药/蘑菇 |
| 矿山(Mine) | 中期 | 剧情/属性 | 矿石/宝石 |
| 人类村庄 | 中期 | 剧情选择 | 商店/社交 |
| 冰原/火山等(高级) | 后期 | 高属性/剧情 | 稀有材料/Boss |
| 隐藏区域 | 秘密 | 特殊条件 | 传说材料/隐藏宠物 |

## 2. 收集内容

| 类别 | 数量(估算) | 用途 |
|:--:|:--:|------|
| 材料(草药/矿石等) | 30+ | 制作/炼制 |
| 武器 | 10+ | 物理Build装备 |
| 饰品 | 10+ | 属性加成 |
| 宠物 | 5-10 | 战斗辅助 |
| 魔法技能 | 10+ | 魔法Build |

## 3. 探索与训练的时间竞争

| 活动 | 消耗天数 | 收益 |
|:--:|:--:|------|
| 训练 | ~3天/次 | 属性直接提升 |
| 探索 | 不定 | 材料+剧情+宠物 |
| 战斗 | ~1天 | 经验+掉落 |
| 策略 | — | 早期训练→中期探索→后期冲刺训练 |

## 4. 设计特点

1. **探索=训练之外的另一条成长线**: 不是"只训练就行"——探索获得材料→制作→更好的装备/药水→训练效率更高
2. **区域=属性的门控**: 高级区域需要高属性→探索是训练成果的检验
3. **R版扩展**: 原版WS1的地图较小→R版大幅扩展了探索区域和收集内容
