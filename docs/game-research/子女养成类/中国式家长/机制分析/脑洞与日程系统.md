# 脑洞与日程系统 - 中国式家长(Chinese Parents)

> 游戏版本: Chinese Parents (PC, 2018年)
> 数据来源: 萌娘百科, SteamAH, 3loumao, 微信公众号攻略

## 1. 时间结构(现实主义的养成节奏)

| 参数 | 设定 |
|------|------|
| 每代 | **48回合**(婴儿→高考) |
| 阶段 | 婴儿→幼儿园→小学→初中→高中 |
| 每回合 | 安排**6个活动**(学习/娱乐)+脑洞小游戏 |
| 总计/代 | 48×6 = **288个活动安排** |

## 2. 脑洞小游戏(Brain Hole)——核心创新

每回合在安排之前，先进入"脑洞"层——用行动力(AP)点击方块揭示相邻方块。

| 方块类型 | AP消耗 | 效果 |
|:--:|:--:|------|
| 碎片(Fragment) | 10 | +属性(颜色+层数决定，每层+3%) |
| 天赋(Talent) | 15 | +每回合被动属性收益，**50%遗传下一代!** |
| 悟性(Insight) | 10 | +25悟性(用于解锁技能) |
| 下一层(Next Level) | 免费 | 下降1层，+50AP，**全局+3%属性收益** |
| 爆炸(Explosion) | 15 | 收集相邻全部方块 |
| 同类(Linked) | 20 | 收集同层所有同色碎片 |
| 观察(Reveal) | 18 | 揭示当前层所有锁定方块 |
| 闪电(Lightning) | 免费 | 随机恢复AP(10-45) |
| 随机(Random) | 10 | 随机上述效果之一 |

### 脑洞策略

| 策略 | 说明 |
|------|------|
| **优先下楼** | 每下一层+3%全局属性→尽早下楼到更深层 |
| AP≤50时收网 | 用爆炸/同类/观察清理当前层 |
| 天赋碎片优先 | 天赋=遗传下一代→多代累积→后代越来越强 |
| 悟性积攒 | 积攒悟性→后期高消耗(999)课程一鼓作气学 |

## 3. 日程安排(6活动/回合)

| 类别 | 代表 | 消耗 |
|:--:|------|:--:|
| 学习(Learning) | 语文/数学/英语/理科/文科/体育/编程等 | 悟性(Insight) |
| 娱乐(Entertainment) | 玩游戏/看电视/社交等 | 悟性或零花钱 |
| 社会实践(Work) | 打工(剪纸/编程等) | 行动力→零花钱 |

## 4. 悟性系统

| 要素 | 说明 |
|------|------|
| 获取 | 脑洞方块(25/个)+父母奖励 |
| 消耗 | 解锁课程(低→高:基础课程低消耗→高级课程999消耗) |
| 策略 | **早期积攒→黄冈密卷冲刺期集中消耗** |

## 5. 压力与满意度

| 参数 | 机制 | 惩罚/奖励 |
|:--:|------|------|
| 压力(Stress) | 学习增加/娱乐减少 | 过高→心理阴影(属性惩罚) |
| 父母满意度(Parent Satisfaction) | 学习成绩影响 | >80%→奖励(零花钱/悟性)<80%→训斥 |
| 零花钱(Pocket Money) | 小学开始/打工+父母给 | 商店购物/社交 |

## 6. 设计特点

1. **脑洞=养成游戏的"抽卡"**: 不是"选课程→属性增加"的枯燥循环——而是每回合一个策略小游戏——随机性+策略选择
2. **48回合=中国教育周期的隐喻**: 6年小学+3年初中+3年高中≈现实中的12年教育——压缩在48回合中
3. **悟性="学习能力"的货币化**: 不是"想学就学"——悟性有限→必须选择哪些课程学——逼你做取舍
4. **天赋遗传=多代累积**: 类似Roguelite——每代留下50%天赋→代代更强——类似于真实中的"家庭积累"
