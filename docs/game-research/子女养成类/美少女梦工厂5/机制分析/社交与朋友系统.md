# 社交与朋友系统 - 美少女梦工厂5(Princess Maker 5)

> 游戏版本: Princess Maker 5 (PC, 2007年)
> 数据来源: princessmaker.fandom.com, GameFAQs, 3DM

## 一、朋友系统(PM系列最丰富)

PM5拥有系列中**最丰富的朋友/社交系统**——不是"几个主要好感NPC"而是"数十个朋友各有独立事件/好感度/结局关联"。

### 1.1 朋友获取

| 方式 | 说明 |
|:--:|------|
| 学校同班 | 自动认识同班同学 |
| 社团 | 加入社团→认识社团成员 |
| 打工 | 打工中结识同事 |
| 事件 | 特定事件触发→认识新朋友 |

## 二、好感度系统

| 互动 | 效果 | 频率 |
|:--:|:--:|:--:|
| 对话 | +基础好感 | 每日 |
| 送礼(最爱) | +大量好感 | 每周 |
| 一起活动 | +中等好感 | 不定期 |
| 一起打工/社团 | +持续被动好感 | 每次 |
| 生日礼物 | **+最多** | 每年1次 |

## 三、恋爱系统

| 阶段 | 条件 | 事件 |
|:--:|------|------|
| 朋友 | 好感度基础 | 日常互动 |
| 好友 | 好感度中 | 个人事件触发 |
| 恋人 | 好感度高+告白事件 | 约会 |
| 结婚 | 最高+特定条件 | 结婚结局 |

## 四、社交→结局联动

| 社交结果 | 结局影响 |
|:--:|------|
| 与某人结婚 | 婚姻结局 |
| 与多人保持好友 | 社交型结局 |
| 无好友 | 孤独型结局 |
| 加入特定社团 | 社团相关职业结局 |

## 五、设计特点

1. **现代日本的中学社交**: 不是PM2的"邂逅王子"——而是"加入漫画社认识同学"——更日常更真实的社交
2. **朋友=属性成长的一个维度**: 与技能高的朋友互动→提升自己技能——社交有功利价值
3. **社团=社交×属性**: 加入哪个社团→认识哪类朋友→影响哪类属性——三维联动
4. **礼物=日本的文化细节**: 芳香蜡烛/乐谱/COSPLAY服饰——日本中学女生之间的真实礼物文化
