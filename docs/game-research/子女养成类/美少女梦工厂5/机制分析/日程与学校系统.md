# 日程与学校系统 - 美少女梦工厂5(Princess Maker 5)

> 游戏版本: Princess Maker 5 (PC, 2007年)
> 数据来源: princessmaker.fandom.com, GameFAQs, 3DM

## 1. 系列最大的变革: 现代化背景

PM5是系列中**唯一设定在现代日本**的作品。女儿不是"中世纪公主"——而是现代中学女生。学校、社团、朋友、智能手机——彻底现代化，也是系列最长培养周期。

## 2. 时间结构

| 参数 | 设定 |
|------|------|
| 培养周期 | **8年**(小学→高中) |
| 每周 | **7天**(可安排每日活动) |
| 每学期 | 学校课程+社团+打工 |
| 假期 | 暑假/寒假→特殊活动 |
| 总计 | 8年×52周 = **416周**(系列最长!) |

## 3. 学校系统

| 要素 | 说明 |
|------|------|
| 小学 | 基础课程→建立属性基础 |
| 初中 | 社团加入/朋友关系深化 |
| 高中 | 恋爱/升学/职业选择 |
| 课程 | 学校必修课(自动)+课外班(自选) |
| 社团 | 动漫社/音乐社/体育社等→影响属性+社交 |

## 4. 每周活动安排

| 类别 | 活动 | 消耗 |
|:--:|------|:--:|
| 学校(必修) | 上课 | 自动(占白天) |
| 课外班 | 剑术/魔法/舞蹈/绘画等13技能 | 金币+时间 |
| 打工 | 送报纸/家教/餐厅等 | 时间→金币 |
| 社交 | 与朋友/恋人互动 | 时间 |
| 探索 | 战斗(5个Boss世界) | 时间+体力 |
| 娱乐 | 游戏/购物等 | 时间+金币 |

## 5. 设计特点

1. **现代日本=最长的培养周期**: 416周>PM2/PM3的288回合——更细腻的成长模拟
2. **学校=社交+属性的双重场所**: 上课(自动属性)+社团(选择影响)+朋友(社交)——学校是PM5的核心舞台
3. **社团=PM5独有**: 加入动漫社/音乐社等→影响属性+解锁朋友——PM2/PM3都没有
4. **暑假/寒假=特殊活动窗口**: 假期有独特的探索/社交/打工机会
