# 事件与叙事系统 - 火山的女儿(Volcano Princess)

> 游戏版本: Volcano Princess v1.0+ (PC, 2023年)
> 数据来源: Steam社区指南, VGover, 3DM

## 1. 事件类型

| 类型 | 触发机制 | 数量(估算) |
|:--:|------|:--:|
| 课程事件 | 上特定课程→触发 | 8门×多个事件 |
| NPC角色事件 | 好感度阈值→触发(每人3个) | ~18人×3 = **54+事件** |
| 季节事件 | 特定月份→触发 | 每年多个 |
| 领主事件 | 选择领主→专属事件线 | 3条独立线 |
| 随机事件 | 概率触发 | 多个 |
| 结局事件 | 终局→职业选择→情感结局 | 58结局×各有过场 |

## 2. 事件触发条件

| 触发类型 | 机制 | 示例 |
|:--:|------|------|
| 属性阈值 | 属性达到某值→触发 | 智力500→大学者路线事件 |
| 日期触发 | 特定年/月→触发 | 冬火年11月→雷贝莎讨伐邀约 |
| 好感度触发 | NPC好感达到阈值→角色事件 | 好感50→角色事件1 |
| 前置事件 | 完成前置→解锁后续 | 完成事件1+2→事件3(情感结局) |
| 次数触发 | 完成某活动N次→触发 | 剧场20次→戏剧明星事件 |

## 4. 关键时间节点

| 时间 | 事件 | 重要性 |
|------|------|:--:|
| 冬火年1月 | 角色第三好感事件解锁/父亲评价关键期 | ★★★★★ |
| 冬火年10月 | **火山节**(刷金/银/铜奖) | ★★★★ |
| **冬火年11月** | **雷贝莎讨伐邀约→决定真结局** | **★★★★★★(最关键!)** |
| 冬火年12月 | 最终数值调整/结局选择 | ★★★★★ |

## 5. 设计特点

1. **多重触发条件=事件密度高**: 一个游戏12年→54+NPC事件+课程事件+季节事件——事件密度极高
2. **事件选择=属性微调**: 每个事件的选择影响属性→不是纯叙事→有策略后果
3. **冬火年11月=游戏的分水岭**: 雷贝莎讨伐邀约→接受/拒绝→决定能否进入真结局——单次选择影响全局
4. **课程事件=学科提升的额外奖励**: 不是"上课→数值增加"的枯燥循环——中间穿插事件→赋予课程叙事价值
