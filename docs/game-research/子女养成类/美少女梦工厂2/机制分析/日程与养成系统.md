# 日程与养成系统 - 美少女梦工厂2(Princess Maker 2)

> 游戏版本: Princess Maker 2 Refine (PC, 1993年原版/2016年Refine版)
> 数据来源: princessmaker.fandom.com, SteamSolo, 贴吧

## 1. 时间结构(品类模板奠基)

| 参数 | 设定 |
|------|------|
| 培养周期 | **8年**(10岁→18岁) |
| 每年 | 12个月 |
| 每月 | 安排**3个活动**(上/中/下旬) |
| 总计 | 8×12×3 = **288回合** |

## 2. 活动类型

| 类型 | 代表 | 消耗 | 主要收益 |
|:--:|------|:--:|------|
| 课程(Education) | 剑术/魔法/礼仪/绘画等 | 金币 | **属性+评价** |
| 打工(Job) | 农场/酒馆/家庭教师等 | — | **金币收入+部分属性** |
| 休息(Rest) | 休息/自由活动 | — | 体力/压力恢复 |
| 探索(Adventure) | 野外探险 | 体力 | 战斗经验+金币+宝物 |
| 社交(Social) | 与NPC对话 | — | 好感度 |
| 比赛(Contest) | 收获祭/舞蹈会等 | — | 声望+奖金 |

## 3. 体力与压力

| 参数 | 机制 |
|------|------|
| 体力(Stamina) | 课程/探索消耗→休息/自由活动恢复 |
| 压力(Stress) | 过度安排→累积→高压力=生病/离家出走 |
| 健康阈值 | 压力>体力→女儿可能生病 |
| **父女对话**(每月) | 每月与女儿对话→影响道德/亲密度 |

| 守护神 | 属性偏向 | 策略 |
|:--:|------|------|
| Saturn(土星) | 均衡 | 新手推荐 |
| Uranus(天王星) | 智力+ | 学者路线 |
| Neptune(海王星) | 魅力+ | 社交路线 |
| Mars(火星) | 战斗+ | 战士路线 |
| Venus(金星) | 魅力+感性+ | 艺术家路线 |
| Mercury(水星) | 敏捷+ | — |
| Moon(月亮) | 感性+魔法+ | 魔法路线 |
| Sol(太阳) | 全属性+ | 万能(压力大) |
| Hades(冥王) | 黑暗+ | — |

## 5. 打工次数年龄补正

| 年龄 | 打工类型 | 次数倍率 |
|:--:|------|:--:|
| 10-12岁 | 基础(农场/保姆) | ×1.0 |
| 13岁 | 墓园 | ×1.5 |
| 14岁 | 家庭教师/酒馆 | ×2.0 |
| 15岁 | 酒馆 | ×2.5 |
| 16岁+ | 夜总会 | ×4.0 |

## 6. 设计特点(品类模板奠基)

1. **288回合=养成游戏的时间标准**: 约8年×12月×3——成为后续所有养成游戏的时间参考
2. **守护神=初始Build**: 开局选择→影响8年成长方向→类似RPG的"职业选择"
3. **打工年龄补正=时间策略**: 不同年龄打工效率不同→早期教育/晚期打工的策略分化
4. **父女对话=社交核心**: 每月一次的父女对话→影响道德和亲密度——最早的"养成游戏社交系统"
