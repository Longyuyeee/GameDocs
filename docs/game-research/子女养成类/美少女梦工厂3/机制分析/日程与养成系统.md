# 日程与养成系统 - 美少女梦工厂3 梦幻妖精(Princess Maker 3)

> 游戏版本: Princess Maker 3: Fairy Tales Come True (PC, 1997年原版/2017年Steam版)
> 数据来源: princessmaker.fandom.com, SteamSolo, 贴吧

## 1. 核心设定: 妖精的承诺

PM3的独特开场: 一名妖精(玩家)因为在人类世界得到帮助，承诺抚养那个人的女儿作为回报。**玩家不是父亲——而是一名妖精**。这个设定改变了整个养成叙事的情感基调。

## 2. 时间结构

| 参数 | 设定 |
|------|------|
| 培养周期 | **8年**(10岁→18岁) |
| 每年 | 12个月 |
| 每月 | 安排**3个活动**(上/中/下旬) |
| 总计 | 8×12×3 = **288回合**(与PM2一致) |
| 自由行动 | 每月可安排1次自由行动(购物/社交/探索) |

## 3. 活动类型

| 类型 | 代表 | 消耗 | 主要收益 |
|:--:|------|:--:|------|
| 课程(Class) | 学問/武术/舞蹈/烹饪/音乐/绘画/礼仪/教会/减肥 | 金币 | 属性+课程经验 |
| 打工(Job) | 清扫/保姆/女仆/市场/农场/酒馆/家教/木工/矿工/宫廷侍女 | — | 金币+部分属性 |
| 休息(Rest) | 休息/自由活动 | — | 体力恢复/压力降低 |
| 度假(Vacation) | 与女儿出游 | 金币 | 父女对话/信任+ |
| 自由行动 | 购物/社交 | — | 道具/NPC好感 |

## 4. 体力/压力/健康

| 参数 | 机制 |
|------|------|
| 体力(Stamina) | 课程/打工消耗→休息恢复 |
| 压力(Stress) | 过度安排累积→压力>体力→生病 |
| 生病(Sick) | **所有属性下降直到恢复**——最严重的惩罚 |
| 体重(Weight) | 饮食/运动影响→影响魅力 |

## 5. 父亲职业(影响收入+初始属性)

| 职业 | 收入 | 属性加成 |
|:--:|:--:|------|
| 商人 | 高 | — |
| 学者 | 中 | 智力+ |
| 骑士 | 中 | 体力+ |
| 贵族 | 极高 | 气品+ |
| 流浪者 | 低 | — |

## 6. 星座+血型(开局影响初始属性)

| 要素 | 影响 |
|:--:|------|
| 12星座 | 初始属性偏向+成长修正 |
| 4血型(A/B/O/AB) | 属性成长倍率 |

## 7. 设计特点(与PM2的差异)

1. **玩家=妖精非父亲**: 改变了情感定位——从"父爱"变为"感恩回报"
2. **无战斗系统**: PM2的战斗被移除——PM3是纯养成
3. **自由行动=额外维度**: 每月1次购物/社交——增加了城镇互动
4. **度假=亲子时间**: 与女儿出游→信任+——替代PM2的"父女对话"
