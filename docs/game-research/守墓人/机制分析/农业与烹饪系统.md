# 农业与烹饪系统 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (2018年)
> 数据来源: Steam Community, GameFAQs

## 1. 农业系统

### 1.1 花园(Garden)

| 作物 | 生长天数 | 用途 |
|------|:------:|------|
| 胡萝卜(Carrot) | 短 | 驴子饲料(!)/料理/炼金(生命溶液) |
| 卷心菜(Cabbage) | 中 | 料理/炼金(生命粉) |
| 甜菜(Beet) | 中 | 料理/出售 |
| 小麦(Wheat) | 中 | 面粉→面包 |
| 南瓜(Pumpkin) | 长 | 炼金(生命粉)/料理 |
| 洋葱(Onion) | 短 | 炼金(毒粉)/料理 |
| 大麻(Hemp) | 中 | 麻绳(蓝点!)/炼金(加速粉) |
| 扁豆(Lentils) | 中 | 炼金(生命提取物) |

### 1.2 葡萄园与果园

| 作物 | 用途 |
|------|------|
| 葡萄(Grapes) | 葡萄酒/果汁(酒馆经济) |
| 啤酒花(Hops) | 啤酒(酒馆经济)/炼金(减速粉) |
| 苹果(Apple) | 果汁/料理 |
| 浆果(Berries) | 采集→果汁 |

### 1.3 僵尸农耕

- 僵尸农场(Zombie Farm): 放置僵尸→自动种植/收获
- 僵尸葡萄园(Zombie Vineyard): 自动种植葡萄/啤酒花
- 需要: 首次提供种子+肥料→之后无限自动

## 2. 烹饪系统

| 厨具 | 代表料理 | 效果 |
|------|------|------|
| 烹饪台(Cooking Table) | 基础料理 | 恢复能量 |
| 烤箱(Oven) | 面包/派 | 高能量恢复 |

- 料理主要用途: 恢复能量(Energy)→用于继续工作
- 地牢探索必备: 带足高能量食物

## 3. 肥料系统

| 肥料 | 配方 | 效果 |
|------|------|------|
| 基础肥料 | 作物废料 | 加速生长 |
| 优质肥料 | 生长促进剂(灰烬+生命溶液) | 显著加速 |
| 金肥料 | 调味增强剂(灰烬+毒+生命) | 最高加速+品质提升 |

## 4. 设计特点

1. **农业=辅助系统**: 不是核心收入——而是为炼金/驴子/料理提供原料
2. **大麻=蓝点+炼金**: 麻→麻绳(蓝点)/加速粉——最具策略价值的作物
3. **僵尸农耕=终局自动化**: 设置后无需手动——真正的自动农场
4. **肥料=炼金产品的下游**: 生长促进剂来自炼金——农业依赖炼金科技
