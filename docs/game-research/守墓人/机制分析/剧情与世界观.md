# 剧情与世界观 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (2018年)
> 数据来源: TrueAchievements, NeoSeeker, Mejoress

## 1. 世界观

- **现代人穿越到中世纪**: 主角在回家路上被车撞→醒来在一个中世纪世界→被困为守墓人
- **家是目标**: 整个游戏的核心驱动力是"找到回家的路"
- **黑色幽默的宗教讽刺**: 七宗罪作为日历/教堂作为经济机构/尸体作为资源

## 2. 主线剧情

```
开场: 现代人→被车撞→醒来在未知世界→被赋予守墓人使命
  ↓
墓地评分5→开放教堂→遇到Bishop→学会布道+信仰
  ↓
遇到Snake→地牢钥匙→15层地牢→暗心(Dark Heart)
  ↓
Aristocrat Papers(12金)→终局→与Bishop/商人/Astrologer对话
  ↓
暗心→交给Inquisitor或Snake→最终选择
  ↓
建造传送门→回家? → (结局取决于DLC/选择)
```

## 3. 三大DLC故事线

| DLC | 内容 |
|------|------|
| **Stranger Sins** | 酒馆经营+小镇历史→揭示守墓人世界的前世 |
| **Game of Crone** | 吸血鬼/逃犯/难民营地→新的道德抉择 |
| **Better Save Soul** | 灵魂救赎→改善尸体品质的新方法 |

## 4. 主题分析

### 4.1 反向牧场经营

| 传统牧场 | 守墓人 |
|------|------|
| 种作物 | 处理尸体 |
| 养动物 | 复活僵尸 |
| 出货箱→金币 | 布道→信仰+捐款 |
| 提升农场 | 提升墓地 |
| 友好邻居 | 七宗罪NPC(各有企图) |
| 温暖色调 | 黑暗+讽刺 |

### 4.2 宗教作为经济系统

- 教堂=布道→信仰=研究货币→科技树
- 这不是"信仰"的精神价值——而是把它当作一种经济资源
- 精妙的讽刺: "信仰"驱动"科学"进步

## 5. 设计特点

1. **反向解构**: 不是"你在经营一个美好的农场"而是"你在经营一个黑暗的墓地"——对牧场品类的全面反转
2. **回家=动机**: 不是"我想成为最好的农民"而是"我想回家"——更个人的动机
3. **黑色幽默贯穿始终**: 驴子罢工要胡萝卜/尸体是资源/僵尸是劳动力——每个系统都有笑话
4. **DLC补充世界观**: 本体故事较薄→DLC扩展了世界观和前史
