# 炼金与制作系统 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (2018年)
> 数据来源: Steam Community, namu.wiki, GamePressure

## 1. 炼金系统

### 1.1 设备

| 设备 | 功能 |
|------|------|
| 炼金台I(Alchemy Workbench I) | 粉+液体→药水/注射剂/颜料 |
| 炼金台II(Alchemy Workbench II) | 粉+溶液+提取物→高级药剂 |
| 手动搅拌器(Hand Mixer) | 制作粉/溶液/提取物 |
| 炼金磨(Alchemy Mill) | 磨粉 |
| 蒸馏器(Distiller) | 蒸馏酒精 |

### 1.2 八大元素

| 元素 | 粉来源 | 溶液来源 | 提取物来源 |
|------|------|------|------|
| 加速(Acceleration) | 麻/蜂蜜 | 蜂蜜/蓝果冻 | 麻/蓝果冻 |
| 减速(Slowing) | 啤酒花/蛛网 | 蜡/啤酒花/脂肪 | 蜡/蛛网 |
| 混沌(Chaos) | 蝠翼/蝴蝶 | 蛾/蝠翼 | 蝴蝶/蛾 |
| 秩序(Order) | 蜜蜂 | 蜜蜂/绿果冻/白花 | 绿果冻/白花 |
| 生命(Health) | 卷心菜/南瓜 | 胡萝卜/卷心菜 | 扁豆/南瓜 |
| 毒性(Toxicity) | 洋葱/红花 | 橙果冻/红蘑菇 | 洋葱/橙果冻 |
| **生命(Life)** | 肠/硫磺 | 脑/蛆/黄花 | 心脏/蛆/黄花 |
| **死亡(Death)** | 暗肠 | 黑果冻/暗脑 | 暗心/黑果冻 |

## 4. 设计特点

1. **八元素×三形态=24基础材料**: 极其复杂的炼金原料体系
2. **中世界炼金术的模拟**: 不是"魔法"而是"炼金术"——粉/溶液/提取物的组合
3. **僵尸汁=生命粉+血液**: 游戏中最常制作的高级药剂→驱动僵尸经济
4. **纸张=知识经济**: 蝠翼→纸→笔记→章节→书——完整的"知识生产链"
