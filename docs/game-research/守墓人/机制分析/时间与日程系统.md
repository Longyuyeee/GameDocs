# 时间与日程系统 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (2018年)
> 数据来源: Steam Community, TrueAchievements

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 一天 | 昼夜循环(无季节!) |
| 每周 | **6天**(七宗罪循环) |
| 无季节 | 无春夏秋冬——全年一样 |
| 无年份 | 无限天→无限游戏 |

## 2. 七宗罪日历

| 罪日 | 符号 | NPC到访 | 核心活动 |
|:--:|:--:|------|------|
| Pride(傲慢) | 棕圈 | Bishop | **布道**(信仰+金币收入) |
| Lust(色欲) | 红♀ | Ms. Charm | 任务推进 |
| Gluttony(暴食) | 棕 | Merchant | 贸易/批量出售 |
| Wrath(愤怒) | 深棕♂ | Inquisitor | 墓地扩建/柴火 |
| Sloth(懒惰) | 蓝月 | Astrologer | 蓝书购买 |
| 夜间(Sloth前夜) | — | Snake | 地牢/暗心任务 |

## 3. 一天结构

| 时段 | 活动 |
|------|------|
| 清晨 | 处理尸体(驴子刚送到)→解剖/埋葬/火化 |
| 白天 | 采集/采矿/制作/任务/探索 |
| 黄昏 | 整理/为明天准备 |
| 夜间 | Snake任务(仅限特定夜)/继续工作 |

- **无强制睡觉**: 可以无限工作(只要有能量)
- 能量(Energy): 工作消耗→食物恢复→睡觉完全恢复

## 4. 能量系统

| 要素 | 说明 |
|------|------|
| 能量(Energy) | 工具/战斗消耗 |
| 0能量 | 无法工作(但不晕倒) |
| 恢复方式 | 食物(立即)/睡觉(完全) |
| 上限 | 固定(无永久提升?) |

## 5. 设计特点

1. **六天循环=七宗罪**: 不是"周一至周日"——而是"傲慢日到懒惰日"
2. **无季节=简化**: 不需要管理季节作物→聚焦于尸体/科技树/任务
3. **每天=处理尸体+任务**: 核心循环——处理驴子送来的尸体→然后推进NPC任务
4. **无强制睡觉**: 允许不眠不休地工作(只要有食物)——更灵活的时间管理
