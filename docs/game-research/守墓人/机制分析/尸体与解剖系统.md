# 尸体与解剖系统 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (PC/Switch/PS4/Xbox, 2018年)
> 数据来源: Steam Community Guides, TrueAchievements, NeoSeeker, GameFAQs

## 1. 概述

守墓人的核心"作物"是**尸体**。不是种田收获——而是接收尸体→解剖优化→埋葬/火化→提升墓地品质。尸体品质由三类骷髅头系统决定，解剖是优化尸体品质的核心操作。

## 2. 骷髅头系统

| 骷髅类型 | 效果 | 说明 |
|:--:|------|------|
| ⚪白骷髅(White Skulls) | **+1墓地评分/个** | 目标: 最大化 |
| 🔴红骷髅(Red Skulls) | **-1墓地评分/个** | 目标: 移除 |
| 🟢绿脸(Sick Faces) | =红骷髅(不可逆) | 尸体腐烂>90%→出现。**永远不要让它发生!** |

## 3. 尸体腐烂机制

- 尸体从驴(Donkey)送达起开始腐烂
- 腐烂>90%→绿脸出现→永久降质
- **预防**: 尸体到达后立即放到**托盘(Pallet)**或**解剖台(Autopsy Table)**→暂停腐烂

## 4. 解剖——器官移除效果

| 器官 | 移除效果 | 建议 |
|------|------|:--:|
| 血液(Blood) | **-1红, +1白** | **总是移除** |
| 脂肪(Fat) | **-1红, +1白** | **总是移除** |
| 头骨(Skull) | **+1红** | 仅在可抵消时移除 |
| 皮肤(Skin) | 1白→1红 | 避免移除 |
| 骨头/肉(Bones/Flesh) | 先移除白骷髅 | 降质 |
| 心脏/大脑/肠(Heart/Brain/Intestines) | **随机**(移除2白/1白1红/2红) | **存档测试!** |

## 5. 最优解剖流程(按红骷髅数)

| 红骷髅数 | 流程 | 结果 |
|:--:|------|:--:|
| 1红 | 移除血液+脂肪 | 全白, 白+1 |
| 2红 | 移除血液+脂肪 | 全白, 不变 |
| 3红 | 先测试心脏/大脑/肠→SL找到移除2红的→再血液+脂肪 | 全白 |

## 6. 防腐处理(Embalming)

| 注射剂 | 配方 | 效果 |
|------|------|------|
| 碱液(Lye) | 生命粉+死亡溶液+血液 | **+1白, +1红** |
| 银注射(Silver) | 银粉+秩序溶液+生命 | **-1红 或 +1白** |
| 金注射(Gold) | 金粉+毒溶液+混沌 | **-2红 或 +2白** |
| 暗注射(Dark) | 死亡粉+血液 | +2红(与金注射配合) |

## 7. 埋葬与火化

| 操作 | 条件 | 产出 |
|------|------|------|
| 埋葬(Bury) | 挖坟墓+尸体 | 埋葬证明(Burial Certificate)→出售→1.5银 |
| 火化(Cremation) | 火化堆 | **灰烬(Ash)**+**盐(Salt)** |
| 骨灰龛(Columbarium) | 12灰+12壶 | **+20墓地评分** |

- 策略: 高品质尸体→埋葬; 低品质尸体→火化

## 8. 设计特点

1. **尸体=核心资源**: 不是"种植作物"而是"优化尸体"——牧场品类中最颠覆性的"反向设计"
2. **腐烂=时间压力**: 尸体随时间腐烂→必须在腐烂前处理→紧迫感
3. **解剖=策略迷你游戏**: 不同红骷髅数→不同最优流程——不是固定公式
4. **SL大法=核心操作**: 心脏/大脑/肠的效果随机→保存/读档→最佳结果
5. **黑色幽默**: 整个系统的存在本身就是对"牧场经营"的解构——你在"种植"尸体
