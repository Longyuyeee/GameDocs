# 地牢与战斗系统 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (2018年)
> 数据来源: TrueAchievements, PSNProfiles

## 1. 地牢(Dungeon)

| 属性 | 详情 |
|------|------|
| 解锁 | Snake任务链→研究Keeper's Key→获得Active Key |
| 层数 | **15层** |
| 敌人 | 史莱姆/蝙蝠/铁处女(Iron Maidens)等 |
| 战斗 | 简单动作(剑/锤) |

## 2. 地牢掉落

| 掉落 | 层数 | 用途 |
|------|:--:|------|
| 血钉(Bloody Nails) | Lv4+ | 任务/出售 |
| 金块(Gold Nuggets) | Lv6+ | 金币/制作 |
| 切面钻石(Faceted Diamonds) | Lv8+ | 高价值出售 |
| **金苹果(Golden Apple)** | Lv15宝箱 | 终局任务(或10金币从Clotho购买) |

## 3. 战斗

| 要素 | 说明 |
|------|------|
| 武器 | 剑(速度)/锤(伤害) |
| HP恢复 | 食物/药水 |
| 死亡 | 失去部分物品→返回入口 |
| 策略 | 带足食物+优先清远程敌人 |

## 4. 设计特点

1. **地牢=可选但推荐**: 金苹果/暗心等终局物品来自地牢
2. **简单战斗**: 不是核心系统——更侧重资源获取
3. **15层=适中深度**: 不是80层矿洞→15层足够提供挑战但不过长
