# 自动化演进与僵尸经济 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (2018年)

## 一、概述

守墓人的游戏体验可以分为**三个时代**——从手动劳动逐步过渡到僵尸自动化帝国。这个演进过程是游戏最核心的满足感来源。

## 二、劳动三时代

### 时代一: 手动时代(教堂开放前)

```
玩家: 手动砍树/采矿/搬运一切
尸体: 全部埋葬(低品质也无法优化)
收入: 埋葬证明(1.5银/具)
限制: 无信仰/无蓝点/无科技
```

### 时代二: 半自动时代(教堂开放→僵尸解锁)

```
玩家: 布道→信仰→研究→蓝点→科技树
尸体: 解剖优化→高品质埋葬/低品质火化
收入: 布道捐款+埋葬证明
制作: 手动制作墓碑/围栏/教堂装饰
```

### 时代三: 僵尸帝国(僵尸解锁后)

```
玩家: 管理僵尸流水线(设置→忘记)
僵尸: 自动采矿(2铁矿+1采石场)
     +自动伐木
     +自动搬运(把资源搬到家)
     +自动农耕/葡萄园
     +自动炼金(批量制作粉/溶液)
收入: 布道(最大)+僵尸采矿→铁钉出售+蔬菜箱批量出口
```

## 三、僵尸自动化的满足感梯度

| 里程碑 | 满足感 |
|------|:--:|
| 复活第一个僵尸 | ★★★"它能动!" |
| 僵尸开始采矿 | ★★★★"我不需要自己去采石场了!" |
| 设置搬运工僵尸 | ★★★★★"资源自动送到家门口!" |
| 完整的采石场流水线 | ★★★★★★"设置后忘记——铁矿石无限" |
| 僵尸农耕 | ★★★★★"连作物都不用自己收了" |

## 四、与星露谷自动化的对比

| 维度 | 星露谷 | 守墓人 |
|------|:--:|:--:|
| 浇水自动化 | 洒水器(制作站) | 不需要浇水(无此系统) |
| 采矿自动化 | 无 | **僵尸采矿+搬运** |
| 伐木自动化 | 无 | **僵尸伐木** |
| 农耕自动化 | Junimo小屋(后期) | **僵尸农场** |
| 动物自动化 | 自动采集器 | 不需要(尸体自动送达) |
| 加工自动化 | 无(需手动放入机器) | **僵尸炼金**(自动制作粉/溶液) |

## 五、设计分析

1. **僵尸=终极自动化**: 不是"机器"而是"复活尸体"——将最"黑暗"的元素变成最"便利"的工具
2. **品质驱动效率**: 僵尸的白骷髅=效率——驱动寻找更好的尸体原料做僵尸
3. **搬运工=物流革命**: 不只是采集自动化——是"搬运"自动化——资源从采石场到家不需要手动运输
4. **从手动→管理**: 游戏体验从"操作"转变为"管理流水线"——满足感从"做得好"变为"设计得好"
