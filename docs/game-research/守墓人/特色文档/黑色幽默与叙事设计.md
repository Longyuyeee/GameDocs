# 黑色幽默与叙事设计 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (2018年)

## 一、概述

守墓人的叙事核心是一个巨大的黑色笑话——你是一个被困在中世纪世界的现代人，被迫成为守墓人，而周围所有人都在利用你。整个游戏的幽默感来自于"严肃对待荒谬"。

## 二、黑色幽默的来源

### 2.1 系统级幽默

| 系统 | 笑话 |
|------|------|
| 驴子运送尸体 | 如果你不给它胡萝卜——它会**罢工** |
| 解剖尸体 | "取出心脏时请存档"——手术是随机的 |
| 僵尸工人 | "低品质尸体=低效员工"——HR黑暗版 |
| 布道收捐款 | 宗教=经济——神父收钱比农民卖菜多 |
| 科技树 | 信仰(宗教)驱动科学——颠倒因果关系 |
| 埋葬证明 | 把人埋了→证明卖给酒馆老板——产业链 |

### 2.2 对话级幽默

- Gerry(会说话的头骨): 你的教程向导——正在腐烂
- Bishop: "好墓地=更多捐款=神爱你更多"(因果关系)
- Donkey: "我要胡萝卜。现在。"

## 三、"回家"的叙事讽刺

- 核心动机: "我想回家"(回到现代世界)
- 但: 你在不断升级墓地和教堂——越来越深入这个世界
- 讽刺: 回家的路=更多的守墓人工作

## 四、七宗罪日历的叙事功能

| 罪 | 对应NPC | 讽刺 |
|:--:|------|------|
| Pride(傲慢) | Bishop | 宗教权威的傲慢 |
| Lust(色欲) | Ms. Charm | 用魅力换取利益 |
| Gluttony(暴食) | Merchant | 资本主义的贪婪 |
| Wrath(愤怒) | Inquisitor | 宗教审判的暴力 |
| Sloth(懒惰) | Astrologer | 知识分子的疏离 |
| Envy(嫉妒) | Snake | 阴谋家的嫉妒 |

## 五、设计分析

1. **幽默=系统不是教条**: 守墓人的幽默让"处理尸体"这个黑暗主题变得可以接受——不是恐怖而是有趣
2. **讽刺=对宗教/资本主义的批评**: 但力度恰到好处——不是"说教"而是"开玩笑"
3. **Donkey=最佳角色**: 会罢工要胡萝卜的驴子——整个游戏幽默感的化身
4. **"回家"的叙事=对牧场游戏"无终点"的反转**: 星露谷=无限年经营/守墓人=有明确终点(回家)——但到达终点的过程被无限延长
