## 六、推荐科技路线(阶段)

| 阶段 | 优先科技 | 原因 |
|:--:|------|------|
| 初期 | 锯木→木工→原始锻造 | 可处理木材/金属→制作工具 |
| 教堂开放后 | 火化→基础墓碑 | 墓地评分→更好的布道收入 |
| 中期 | 解剖+防腐 | 优化尸体品质→更高评分 |
| 中后期 | 书籍(纸→笔记→章节) | **蓝点农场!**→解锁高级科技 |
| 后期 | 大理石/大教堂 | 最大墓地评分+布道收入 |
