# 尸体与解剖数据 - 守墓人(Graveyard Keeper)

> 游戏版本: Graveyard Keeper v1.0+ (2018年)
> 数据来源: Steam Community Guides

## 一、器官移除效果表

| 器官 | 移除效果 | 确定性 | 建议 |
|------|------|:--:|:--:|
| 血液(Blood) | -1红, +1白 | 确定 | **总是移除** |
| 脂肪(Fat) | -1红, +1白 | 确定 | **总是移除** |
| 头骨(Skull) | +1红 | 确定 | 可抵消时移除 |
| 皮肤(Skin) | 1白→1红 | 确定 | 避免移除 |
| 肉(Flesh) | 先移除白→降质 | 确定 | 避免移除 |
| 骨头(Bones) | 先移除白→降质 | 确定 | 避免移除 |
| 心脏(Heart) | 随机(2白/1白1红/2红) | **随机** | **SL测试!** |
| 大脑(Brain) | 随机 | **随机** | **SL测试!** |
| 肠(Intestines) | 随机 | **随机** | **SL测试!** |

## 二、按红骷髅数解剖流程

| 红骷髅 | 流程 | 最终净白骷髅 |
|:--:|------|:--:|
| 0红 | 不操作 | 原始白数 |
| 1红 | 血液+脂肪 | 白+1 |
| 2红 | 血液+脂肪 | 不变 |
| 3红 | 测试心/脑/肠→SL找2红→再血液+脂肪 | 不变或更多 |

## 三、防腐注射数据

| 注射 | 配方 | 效果 |
|------|------|------|
| 碱液(Lye) | 生命粉+死亡溶液+血液 | +1白, +1红 |
| 银(Silver) | 银粉+秩序溶液+生命 | -1红 或 +1白 |
| 金(Gold) | 金粉+毒溶液+混沌 | **-2红 或 +2白** |
| 暗(Dark) | 死亡粉+血液 | +2红(配合金用→-2+2=0) |

## 四、墓地评分计算

```
坟墓评分 = 尸体净白骷髅 + 墓碑加成 + 围栏加成
(墓碑≤白骷髅数, 围栏≤白骷髅数)
空坟墓 = -2评分
```

| 墓碑/围栏等级 | 加成 |
|:--:|:--:|
| 木制(T1) | +1 |
| 石制(T2) | +2 |
| 大理石(T3) | +3 |
| 高级(T4+) | +4+ |

## 五、火化产出

| 操作 | 产出1 | 产出2 |
|------|------|------|
| 火化1具尸体 | 灰烬(Ash) | 盐(Salt) |

- 灰烬用途: 肥料/骨灰龛/墨水/生长促进剂
- 盐用途: 调味料(Spice→商人任务)
