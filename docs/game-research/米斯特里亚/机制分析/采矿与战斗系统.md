# 采矿与战斗系统 - 米斯特里亚的田野(Fields of Mistria)

> 游戏版本: Fields of Mistria (PC, Early Access 2024-至今)
> 数据来源: fieldsofmistria.com, Game8, GameRant

## 1. 矿洞

| 属性 | 详情 |
|------|------|
| 层数 | **80+层** |
| 生物群系 | 多种——每段有独特敌人/宝物/矿物 |
| 敌人 | 史莱姆/石像鬼/各生物群系变体 |
| 宝物 | 矿石/宝石/文物/龙文明遗物 |
| 解锁 | 主线推进→更深层 |
| 100层 | **解锁Seridia(特殊候补)** |

## 2. 战斗系统

| 要素 | 说明 |
|------|------|
| 类型 | 2D俯视动作 |
| 武器 | 剑/锤/魔法 |
| 法术 | 火/水/土/风——元素克制 |
| HP | 归零=矿洞返回(失去部分物品?) |

## 3. 矿石升级

| 等级 | 矿石 | 工具/武器 |
|:--:|------|:--:|
| 1 | 铜 | 基础 |
| 2 | 铁 | 升级 |
| 3 | 银 | 中级 |
| 4 | 金 | 高级 |
| 5 | **秘银(Mistril)** | **最高级** |

## 4. 设计特点

1. **80+层=大型矿洞**: EA已80层——最终版本可能100+层
2. **龙文明遗物=考古系统联动**: 矿洞出文物→博物馆捐赠→考古技能
3. **100层=特殊候补解锁**: Seridia(龙祭司)——深层探索=社交解锁
