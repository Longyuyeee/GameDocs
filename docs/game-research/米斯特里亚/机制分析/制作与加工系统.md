# 制作与加工系统 - 米斯特里亚的田野(Fields of Mistria)

> 游戏版本: Fields of Mistria (PC, Early Access 2024-至今)
> 数据来源: fieldsofmistria.com, GameRant

| 类别 | 代表物品 | 数量(估算) |

| 制作站 | 产物 |

## 3. 无需携带材料=革命性QoL

| 传统牧场 | 米斯特里亚 |
|:--:|:--:|
| 需要从储物箱取出材料→放入背包→去制作站 | 材料留在储物箱→到制作站**自动读取** |
| 背包满了→频繁往返 | 无需管理背包材料 |

> 这是该游戏最受好评的QoL设计——材料自动读取

## 4. 设计特点

1. **600+物品=EA阶段已接近星露谷终局量**
2. **自动读取材料=QoL革命**: 大多数牧场游戏至今没做这个
3. **考古台=独特制作站**: 修复龙文明文物→博物馆捐赠→声望
