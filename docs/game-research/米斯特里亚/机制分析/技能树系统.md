# 技能树系统 - 米斯特里亚的田野(Fields of Mistria)

> 游戏版本: Fields of Mistria (PC, Early Access 2024-至今)
> 数据来源: fieldsofmistria.com, Game8, GameRant

## 1. 九棵技能树

| # | 技能树 | 经验获取 | Perk方向 |
|:--:|------|------|------|
| 1 | 农耕(Farming) | 浇水/收获/动物照料 | 产量↑/双倍收获/动物好感↑ |
| 2 | 采矿(Mining) | 碎石/采矿/矿洞探索 | 暴击采矿/双倍矿石/宝石概率↑ |
| 3 | 战斗(Combat) | 击杀怪物 | 攻击力↑/HP↑/新攻击技能 |
| 4 | 钓鱼(Fishing) | 钓鱼 | 完美钓鱼概率↑/稀有鱼↑ |
| 5 | 烹饪(Cooking) | 制作料理 | 双倍料理/品质↑/新食谱 |
| 6 | **魔法(Magic)** | 施法 | 法术效果↑/消耗↓/新法术 |
| 7 | 社交(Social) | 送礼/对话/任务 | 好感度增益↑/礼物效果↑ |
| 8 | 考古(Archaeology) | 挖掘文物/捐赠 | 文物发现率↑/奖励↑ |
| 9 | 制作(Crafting) | 制作物品 | 材料节省/品质↑/新配方 |

## 2. Perk系统

| 机制 | 说明 |
|------|------|
| 触发 | **每5级→1个Perk点** |
| 选择 | 每5级有2-3个Perk选项→**选择1个**(不可重置?) |
| 效果 | 永久被动加成 |
| 代表 | 采矿Lv5→暴击采矿/双倍矿石(选一) |

## 5. 设计特点

1. **9棵树=最细化的技能分类**: 烹饪/考古从其他游戏的一个功能独立为技能树
2. **Perk=有意义的选择**: 每5级2-3选1→"采矿暴击vs双倍矿石"——需根据玩法偏好决策
3. **魔法技能=独特**: 星露谷/沙石镇都没有独立的"魔法"技能树
4. **考古=叙事+收集**: 挖掘文物→与主线龙文明相关→技能/叙事/收集三位一体
