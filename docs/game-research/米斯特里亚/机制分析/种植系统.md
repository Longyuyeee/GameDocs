# 种植系统 - 米斯特里亚的田野(Fields of Mistria)

> 游戏版本: Fields of Mistria (PC, Early Access 2024-至今)
> 数据来源: fieldsofmistria.com, Game8

## 1. 概述

50+季节作物，标准四季节轮作。温室(2025年7月更新)提供全年种植。

## 2. 四季作物

| 季节 | 代表作物 | 特色 |
|:--:|------|------|
| 春 | 芜菁/土豆/草莓/花椰菜 | 草莓=复收王 |
| 夏 | 番茄/玉米/菠萝/西瓜 | 菠萝=最高价值 |
| 秋 | 茄子/南瓜/红薯 | 南瓜=秋季王者 |
| 冬 | 冬季专属作物 | — |

## 3. 特色系统

| 系统 | 说明 |
|------|------|
| 温室(Greenhouse) | 2025年7月更新——全年种植任何季节作物 |
| 肥料 | 加速/品质提升 |
| 洒水器(Sprinklers) | 600+可制作物品之一 |
| 魔法辅助 | 水(9×9浇灌)/土(翻耕)/火(清障) |

## 4. 设计特点

1. **魔法替代洒水器早期需求**: 水魔法浇灌9×9→早期不需投资洒水器
2. **温室=终局解锁**: 主线推进后→材料+金币——与星露谷一致
3. **50+作物=EA已丰富**: 未完成即有50+——1.0预计更多
