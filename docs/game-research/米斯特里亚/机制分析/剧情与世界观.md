# 剧情与世界观 - 米斯特里亚的田野(Fields of Mistria)

> 游戏版本: Fields of Mistria (PC, Early Access 2024-至今)
> 数据来源: fieldsofmistria.com, Game8, GameRant

## 1. 世界观

- **米斯特里亚镇(Mistria)**——被自然环绕的古老小镇
- **古代龙文明**——曾经繁荣的龙族+人类共存的文明→现已衰落
- **龙雕像Caldarus**——位于你农场上的神秘龙守护灵→在梦中与你交流

## 2. 主线

### EA当前阶段
```
到达米斯特里亚→继承荒废农场
  → 发现龙雕像Caldarus→解锁魔法
  → 探索矿洞(80+层)→发现龙文明遗物
  → 捐赠博物馆→修复文物→揭示龙文明历史
  → 解锁Seridia(龙祭司/100层特殊候补)
  → (1.0待续: 龙文明完全揭示/主线结局)
```

### 1.0计划
- 主线结局
- 婚姻+子女系统
- 龙文明完全揭示

## 3. 核心角色

| 角色 | 身份 | 叙事功能 |
|------|------|------|
| Caldarus | 龙守护灵 | 引导者/魔法来源/可攻略 |
| Seridia | 龙祭司 | 龙文明的最后守护者/深层矿洞/可攻略 |
| Adeline | 镇长 | 城镇发展/任务分发 |
| Eiland | 考古学家 | 龙文明研究/文物分析 |

## 4. 设计特点

1. **龙文明=核心叙事**: 不是"你种田吧"——而是"你在种田的过程中发现古代龙的秘密"
2. **Caldarus=引导者+攻略对象**: 类似符文工房的Ventuswill——但可恋爱!
3. **EA叙事未完成**: 主线在EA中尚未结束——1.0将完成
