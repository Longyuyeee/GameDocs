## 二、社交机制

### 2.1 好感度系统

| ♥等级 | 解锁 | 好感度需求 |
|:--:|------|:--:|
| 0-1♥ | 基础对话 | 0 |
| 2♥ | 2心事件 | — |
| 4♥ | 4心事件 | — |
| 6♥ | 6心事件 | — |
| **8♥** | **选择: 浪漫告白 OR 永久挚友** | — |
| 10♥ | 婚姻(1.0计划) | — |

### 2.2 送礼规则

| 方式 | 效果 |
|------|:--:|
| 最爱礼物 | +20 pts(+40生日) |
| 喜欢礼物 | +10 pts(+30生日) |
| 每日送礼 | **无周上限**(与星露谷不同!)——每天1次/人 |
| 每日对话 | +7 pts |
| 任务 | +好感度 |

### 2.3 关键社交特性

- **无嫉妒系统**: 可同时与全部12人约会
- **8心告白=关系分岔点**: 浪漫告白(恋人) OR 永久挚友(朋友)——选出你想交往的
- **~5,000行对话/角色**: 季节服装变体+大量独白
- **实时NPC地图**: 随时看到每个NPC位置
- **婚姻尚未实装**(EA限制)

|------|------|

| 动物节(冬) | 社交+竞赛 |
| 极光夜(冬) | NPC赠送限定毛衣 |

## 四、设计特点

1. **12候补+无嫉妒=社交自由最大化**: 可所有人同时恋爱→8心再决定和谁确认关系
2. **8心=浪漫/挚友二选一**: 不是所有人都是"恋爱线"——可选择保持朋友关系
3. **5,000行/角色=对话深度**: EA阶段已远超很多已发售牧场游戏
4. **龙=Caldarus可攻略**: 守护灵→恋爱对象——独特的跨物种浪漫
5. **实时NPC地图=最需要的QoL**: 不再到处找NPC位置
