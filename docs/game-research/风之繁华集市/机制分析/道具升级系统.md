---

## 一、系统概述

本作的道具系统以**农耕工具**为核心，不包含任何武器或战斗系统。所有工具的用途围绕农业生产展开。重制版做了两项重大改进：**工具不再占用背包格**和**新增二段跳+空中工具连击**机制，大幅提升了操作手感和农场工作效率。

## 三、工具升级系统

### 3.1 升级途径

工具有两种升级途径，可叠加使用：

**途径一：工匠屋升级（Carpenter Upgrade）**

工匠屋在**每周六集市**期间营业。随着集市等级提升，可解锁更高级的工具打造。

| 升级等级 | 所需集市等级 | 升级费用 | 所需材料 |
|:-------:|:----------:|:-------:|:--------|
| 铜（★1→★2） | Lv.2 | 1,000G | 铜矿石×5 |
| 银（★2→★3） | Lv.3 | 2,500G | 银矿石×5 |
| 金（★3→★4） | Lv.4 | 5,000G | 金矿石×5 |
| 秘银（★4→★5） | Lv.5 | 10,000G | 秘银矿石×5 |
| 诅咒（★5→★6） | Lv.6 | 20,000G | 奥里哈钢矿石×5 + 金刚石×3 |
| 祝福（★6→★7） | Lv.7 | 50,000G | 祝福之石×1 + 钻石×5 |

**途径二：黄色加工风车（Processing Windmill）**

使用黄色加工风车自行加工工具升级材料（矿石→升级组件），可节省部分费用但需要时间。

| 加工项目 | 所需材料 | 加工时间 | 用途 |

| 铜锭 | 铜矿石×5 | 2小时 | 工具升级/饰品制作 |
| 银锭 | 银矿石×5 | 3小时 | 工具升级/饰品制作 |
| 金锭 | 金矿石×5 | 4小时 | 工具升级/饰品制作 |
| 秘银锭 | 秘银矿石×5 | 5小时 | 工具升级 |
| 奥里哈钢锭 | 奥里哈钢矿石×3 | 8小时 | 高级工具升级 |

## 四、二段跳+工具连击系统（Double Jump + Tool Combo）

### 4.1 系统概述

重制版新增的核心操作机制。玩家可通过跳跃→空中使用工具的组合动作实现**大幅提升工作效率**的操作技巧。

### 4.2 跳跃机制

| 操作 | 说明 |
|:---|:----|
| 单跳（A键） | 跳跃约1.5格高度，0.5秒滞空 |
| 二段跳（A键×2） | 第一跳最高点再按A，总计约2.5格高度，0.8秒滞空 |
| 空中方向控制 | 跳跃中可控制移动方向 |
| 跳跃取消后摇 | 使用工具后立即跳跃可取消硬直 |

### 4.3 空中工具连击操作一览

| 连击操作 | 效果 | 效率对比（相对地面） | 适用场景 |
|:--------|:----|:---------------:|:--------|
| 跳跃+锄头 | 空中锄地→扇形开垦 | ×3-5倍 | 大面积开荒 |
| 二段跳+水壶 | 高空扇形浇水 | ×3-9倍 | 大面积浇水 |
| 跳跃+镰刀 | 空中扇形收割 | ×3-7倍 | 大面积收割 |
| 跳跃+斧头 | 下落重劈 | ×1.5-2倍伤害 | 砍伐大树 |
| 二段跳+锤子 | 下落重砸 | ×2-4倍效率 | 矿石开采 |

### 4.4 实战连击技巧

- **跑跳镰刀流**：按住移动方向→跳跃→镰刀→落地→跳跃→镰刀循环，实现跑动中连续扇形收割
- **二段跳浇水阵**：二段跳至最高点→使用水壶，以最大扇形范围覆盖下方田地
- **滞空三连锄**：跳跃后在空中连续挥锄（需贤者级锄头），利用滞空时间开垦大面积
- **下落采矿术**：在山崖边二段跳→锤子下砸，可同时敲碎下方多层岩石

- 将仓库与风车直接连接

- 无需手动搬运材料，大幅提升批量加工效率
- 是后期大规模加工的关键设施

## 七、总结

本作工具系统以**农业生产效率**为唯一导向，不含任何战斗要素。重制版的两大改进——**工具独立栏位**和**双跳+空中连击**——分别从管理便利性和操作深度两个维度提升了游戏体验。工具升级路径清晰，从铜到贤者的7级体系提供了长期追求目标。滑翔伞和弹跳台的加入进一步拓展了工具系统的探索维度（详见地图场景系统）。
