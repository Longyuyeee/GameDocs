> 原版：牧场物语 欢迎来到风之集市（NDS，2008年）

## 一、系统定性：无战斗系统

**本作不存在任何形式的战斗系统。** 这是牧场物语系列的一贯传统——本作是纯粹的**农耕生活模拟游戏**，不含武器系统、不含敌人、不含血量、不含战斗技能。游戏的全部冲突管理体现在：

- 体力（Stamina）管理——如何高效分配每日精力
- 时间管理——如何在有限时间内完成最多任务
- 经济管理——如何在集市获得最大收益

因此，本文档将聚焦于本作中的**探索性内容**——采矿、钓鱼、采集、滑翔伞探索等非战斗的户外活动。

## 二、采矿系统（Mining System）

### 2.1 采矿机制

采矿通过**锤子（Hammer）** 敲击山地的岩石/矿点实现。无需进入迷宫或矿洞，矿石直接分布于山地各层的露天矿点。

| 项目 | 说明 |
|:----|:----|
| 所需工具 | 锤子（Hammer） |
| 操作方式 | 走到矿点前按工具键敲击 |
| 每次敲击消耗体力 | 2点 |
| 矿石掉落 | 每次敲击随机获得1-2个矿石 |
| 矿点刷新 | 定期刷新（见地图场景系统文档） |
| 工具等级影响 | 更高级锤子可敲碎更大岩石，提高采矿效率 |

### 2.2 各区域矿石产出

| 区域 | 主要矿石 | 稀有产出 | 锤子最低要求 | 推荐等级 |
|:----|:--------|:--------|:----------:|:-------:|
| 山麓矿点 | 铜矿石、小石头 | （无） | ★1 铜 | 初期 |
| 山腰矿洞 | 铜矿石、银矿石 | 金矿石（低概率） | ★2 银 | 中期 |
| 山顶矿洞 | 金矿石、秘银矿石 | 宝石原石（低概率） | ★3 金 | 中后期 |
| 悬崖矿点（滑翔伞） | 秘银矿石、奥里哈钢 | 宝石原石、钻石 | ★4 秘银 | 后期 |
| 瀑布后洞穴 | 宝石原石 | 钻石、金刚石、宝箱 | ★5 诅咒 | 后期 |

### 2.4 采矿技巧

| 技巧 | 说明 |
|:----|:----|
| 锤子蓄力 | 按住工具键蓄力可一次敲击更大范围（取决于工具等级） |
| 空中锤子 | 跳跃+锤子下砸，对下方岩石造成更大破坏 |
| 台风天采矿 | 虽然户外危险，但山顶矿点在台风天有更高概率出稀有矿石 |
| 矿点标记 | 可用小旗标记已采空的矿点，方便追踪刷新 |
| 双人采矿 | 高好感小矮人可协助采矿（自动采集） |

## 三、钓鱼系统（Fishing System）

### 3.1 钓鱼机制

| 项目 | 说明 |
|:----|:----|
| 所需工具 | 钓竿（Fishing Rod） |
| 获得方式 | 集市Lv.2后，在河边触发剧情获得 |
| 操作方式 | 抛竿→等待鱼咬钩→按提示按键收杆 |
| 消耗体力 | 每次抛竿消耗1点 |
| 鱼饵 | 可使用鱼饵提升咬钩率（小麦粉等） |

### 3.2 钓鱼点与鱼种

| 钓鱼点 | 主要鱼类（高概率） | 稀有鱼类（低概率） | 最佳时间 |
|:------|:----------------|:----------------|:--------|
| 河流下游A | 鲫鱼（60%）、鲤鱼（30%） | 鲶鱼（10%） | 全天 |
| 河流中游B | 鳟鱼（50%）、鲈鱼（30%） | 虹鳟（15%）、宝箱（5%） | 6:00-18:00 |
| 河流上游C | 山女鱼（45%）、岩鱼（35%） | 樱鳟（15%）、传说之鱼（5%） | 6:00-12:00 |
| 瀑布潭 | 鲑鱼（50%）、大马哈鱼（30%） | 帝王鲑（15%）、宝箱（5%） | 全天 |
| 山顶湖 | 虹鳟（40%）、远东哲罗鱼（40%） | **传说之鱼**（5%）、宝箱（15%） | 全天 |
| 隐藏海岸 | 鲷鱼（40%）、比目鱼（35%） | 金枪鱼（15%）、旗鱼（10%） | 夏季限定 |

### 3.4 钓鱼技巧

| 技巧 | 说明 |
|:----|:----|
| 抛竿距离 | 蓄力可抛更远，不同距离对应不同鱼种 |
| 鱼饵使用 | 使用鱼饵后咬钩率提升约30%，持续抛竿5次 |
| 收杆时机 | 鱼漂完全沉入水中的瞬间按收杆，成功率最高 |
| 耐力比拼 | 大型鱼需要持续收杆（连续按A键），耗时会较长 |
| 雨天加成 | 雨天钓鱼，稀有鱼出现概率+50% |
| 宝箱 | 钓鱼有概率钓上宝箱，内含矿石、种子或稀有道具 |

## 五、滑翔伞探索系统（Glider Exploration）

### 5.1 滑翔伞功能

| 项目 | 说明 |
|:----|:----|
| 获取方式 | 春25日后触发"飞翔之梦"委托 |
| 操作 | 在高处按跳跃键后按住滑翔键 |
| 体力消耗 | 每秒消耗1点体力 |
| 风力影响 | 风向/风力影响飞行距离和方向 |

### 5.2 滑翔伞可抵达的探索区域

| 目的地 | 前置条件 | 可收集资源 | 探索难度 |
|:------|:--------|:----------|:-------:|
| 悬崖矿点 | 从山顶起跳 | 奥里哈钢、宝石原石 | 中等 |
| 隐藏海滩 | 从悬崖起跳+顺风 | 稀有海鱼、贝壳 | 较难 |
| 孤岛（山顶湖中） | 山顶湖起跳+上升气流 | 宝箱（含耀珠）、祝福之草 | 难 |
| 神秘花园 | 悬崖起跳+强力上升气流 | 珍稀花卉、特殊种子 | 极难 |

### 5.3 探索度系统

| 探索度指标 | 内容 | 奖励 |
|:---------|:----|:----|
| 区域发现率 | 发现全部地图区域 | 每发现一区域 +1,000G |
| 钓鱼图鉴完成率 | 钓到全部鱼类 | 全图鉴 → 传说钓竿装饰 |
| 矿石图鉴完成率 | 采到全部矿石种类 | 全图鉴 → 矿石模型（家具） |
| 采集图鉴完成率 | 采集到全部野生物品 | 全图鉴 → 花环装饰 |
| 宝箱收集率 | 开启全部野外宝箱 | 全收集 → 隐藏家具 |

## 九、总结

本作没有任何战斗系统，游戏挑战性来源于**资源管理、时间规划和探索发现**。采矿、钓鱼、采集和滑翔伞探索构成了主要的野外活动内容。虽然没有战斗带来的紧张感，但对于追求轻松治愈体验的玩家而言，这种"无压探索"反而正是牧场物语系列的核心魅力所在。滑翔伞系统的加入为探索增添了垂直维度和操作技巧要求，是重制版在探索领域最重要的新增内容。
