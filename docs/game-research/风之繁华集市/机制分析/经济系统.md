# 经济系统分析
![集市](https://www.bokumono.com/series/kazebaza/img/ss/ss01.jpg)

> 游戏版本：牧场物语 来吧！风之繁华集市（Switch/Switch2/Steam，2025年8月28日发售）
> 原版：牧场物语 欢迎来到风之集市（NDS，2008年）

---

## 一、系统概述

本作的经济系统与系列传统作品截然不同——**不存在出货箱（Shipping Bin）**。所有产品必须通过在每周六举办的**集市（Bazaar）**上摆摊出售才能转化为收入。这一核心机制重塑了玩家的整个游戏节奏：6天生产筹备 + 1天集中销售，形成了独特的"生产-加工-销售"循环。

---

## 二、集市系统（Bazaar System）

### 2.1 集市等级（Bazaar Level）

集市共 **7 级**，通过累计集市销售额提升。每升一级解锁新功能/商店。

| 等级 | 所需累计销售额 | 解锁内容 |
|:---:|:---:|---|
| Lv.1 | 初始 | 基础集市开放，3个摊位格 |
| Lv.2 | 约 5,000G | 工匠屋（Carpenter）入驻集市，可升级工具 |
| Lv.3 | 约 15,000G | 解锁第4-5摊位格，咖啡馆（Cafe）解锁 |
| Lv.4 | 约 40,000G | 动物店（Animal Shop）入驻，可购买新牲畜 |
| Lv.5 | 约 80,000G | 解锁第6-7摊位格，种子店升级，珍稀种子解锁 |
| Lv.6 | 约 150,000G | 解锁第8摊位格，饰品店入驻 |
| Lv.7 | 约 300,000G | 解锁第9摊位格（最大），所有商店最高级商品解锁 |

### 2.2 摊位系统（Stall System）

玩家在集市广场拥有固定摊位，可自由布置商品。

**摊位格扩张：**

| 摊位格数 | 解锁条件 | 扩建费用 |
|:-------:|:--------:|:--------:|
| 3（初始） | 游戏开始 | 免费 |
| 4 | 集市Lv.3 | 2,000G |
| 5 | 集市Lv.3 | 3,000G |
| 6 | 集市Lv.5 | 5,000G |
| 7 | 集市Lv.5 | 8,000G |
| 8 | 集市Lv.6 | 12,000G |
| 9 | 集市Lv.7 | 20,000G |

**摊位摆放规则：**
- 每个摊位格可放置 **1种商品**（可堆叠，最多99个）
- 玩家可为每种商品**自由定价**
- 商品显示顺序由玩家手动排列决定
- 可放置招牌吸引顾客（需通过工匠屋制作）

### 2.3 自由定价机制（Pricing Freedom）

这是本作经济系统的核心特色之一。玩家为每件商品设定售价，系统根据以下因素动态影响顾客购买意愿：

| 影响因素 | 效果 |
|:--------|:----|
| 商品基础价值 | 系统参考价，定价过低则利润减少 |
| 定价偏离度 | 高于系统参考价越高，顾客购买概率越低 |
| 商品品星（★） | 高星级商品可接受更高溢价 |
| 品牌认证 | 有品牌认证的商品顾客接受溢价空间更大 |
| 流行趋势 | 流行商品即使定价较高也容易售出 |
| 集市等级 | 高等级集市顾客购买力更强 |

**定价策略参考：**

| 策略 | 定价倍率 | 说明 |
|:---|:--------:|:----|
| 低价快销 | 80%-100% 参考价 | 快速清仓，适合低星商品 |
| 标准定价 | 100%-120% 参考价 | 平衡策略 |
| 品牌溢价 | 120%-150% 参考价 | 有品牌认证时适用 |
| 精品高价 | 150%-200% 参考价 | 配合流行趋势+品牌+高星 |
| 奢侈品定价 | 200%-300% 参考价 | 仅限珠宝、饰品等高价值加工品 |

## 三、品牌系统（Brand System）

### 3.1 品牌认证流程

1. 参加各类**节日大赛**（作物祭、畜牧祭、料理祭等）
2. 在比赛中获得**金奖项**
3. 该产品获得**品牌认证**（Brand Certified）
4. 所有同类型产品在集市出售时享受**价格乘数**

### 3.2 品牌价格乘数

| 认证等级 | 获得条件 | 价格乘数 |
|:-------:|:--------:|:--------:|
| 无品牌 | 未参赛 | ×1.0 |
| 铜牌品牌 | 大赛获铜奖 | ×1.15 |
| 银牌品牌 | 大赛获银奖 | ×1.30 |
| 金牌品牌 | 大赛获金奖 | ×1.50 |
| 传说品牌 | 同一作物多次获金奖 | ×1.80 |

### 3.3 品牌维护

- 品牌认证并非永久有效，需定期在比赛中卫冕
- 若连续两年（游戏内）未在该品类获奖，品牌等级下降一级
- 不同品类独立计算品牌（作物品牌 ≠ 畜牧品牌 ≠ 加工品品牌）

## 四、流行趋势系统（Trend System）

### 4.1 月度流行

每月初，系统随机选定 **2-3 种商品** 为该月流行商品。

| 月份 | 季节 | 流行品类示例 |
|:---:|:---:|:-----------|
| 春1月 | 春 | 春季蔬菜、花类、蛋制品 |
| 春2月 | 春 | 叶菜类、加工点心、羊毛制品 |
| 夏1月 | 夏 | 夏季水果、乳制品、冷饮类 |
| 夏2月 | 夏 | 瓜类、鱼类、果汁类 |
| 秋1月 | 秋 | 根茎类、蘑菇类、加工肉类 |
| 秋2月 | 秋 | 谷物类、酿酒类、坚果类 |
| 冬1月 | 冬 | 乳制品、毛绒制品、珠宝首饰 |
| 冬2月 | 冬 | 加工食品、暖饮类、矿石制品 |

### 4.2 流行效果

- 流行商品在集市出售时 **售价 +30%**
- 流行商品顾客**购买意愿大幅提升**（约提升40%-60%）
- 流行商品即使大量铺货也不易滞销
- 可通过与NPC对话获取流行趋势情报（情报准确度约70%）

## 五、小矮人应援团（Little Sprite Support）

### 5.1 系统概述

小矮人（Little Sprite）是本作中帮助玩家工作的精灵群体。在集市系统中，小矮人应援团是提升销售效率的关键机制。

### 5.2 开心能量（Happy Energy）

小矮人拥有"开心能量"积累系统：

| 积累方式 | 能量增加量 |
|:--------|:---------:|
| 每日对话 | +5 |
| 每日送礼（喜欢的物品） | +15 |
| 完成小矮人委托 | +30 |
| 小矮人参与工作一整天 | +3 |
| 送讨厌物品 | -10 |
| 长时间不理睬 | -每日衰减2 |

### 5.3 快速销售模式（Rapid Selling Mode）

当开心能量积累到阈值时，可在集市触发：

| 能量阈值 | 触发效果 | 持续时间 |
|:--------:|:--------|:-------:|
| 50 | 基础助威：顾客流+20% | 30秒 |
| 100 | 活跃助威：顾客流+40%，售价+10% | 45秒 |
| 150 | 热情助威：顾客流+60%，售价+15% | 60秒 |
| 200 | 狂欢助威：顾客流+100%，售价+20% | 90秒 |

### 5.4 小矮人招募

| 小矮人 | 解锁条件 | 擅长领域 |
|:-----:|:--------:|:--------:|
| 红色小矮人 | 春季，集市Lv.2 | 浇水 |
| 蓝色小矮人 | 夏季，集市Lv.2 | 收获 |
| 绿色小矮人 | 秋季，集市Lv.3 | 照顾动物 |
| 橙色小矮人 | 冬季，集市Lv.3 | 加工风车 |
| 紫色小矮人 | 集市Lv.4 | 集市摆摊 |
| 黄色小矮人 | 集市Lv.5 | 采矿 |

## 六、收入策略分析（Income Strategies）

### 6.1 邪修流——牧草种子循环（Hay Seed Loop）

**原理：** 利用粉碎风车将牧草加工成牧草种子，利用肥料和饲料加速生长，形成高效循环。

| 步骤 | 操作 | 成本/收入 |
|:---:|:----|:---------:|
| 1 | 购买牧草种子 ×2 | -80G |
| 2 | 种植并施肥 | 肥料成本约20G |
| 3 | 收获牧草 | - |
| 4 | 粉碎风车加工 → 牧草种子 ×6 | 加工时间约2小时 |
| 5 | 集市出售多余种子 | 约1,120G |

**循环收益率：** 约 1,020G 利润/周期（投入80G，产出1,120G），收益率约 **1,275%**。

### 6.2 年糕流（Mochi Rice Cake Loop）

**原理：** 从咖啡馆购买大米，用粉碎风车制作年糕，利用冬季流行趋势高价出售。

| 步骤 | 操作 | 成本/收入 |
|:---:|:----|:---------:|
| 1 | 咖啡馆购买大米 ×1 | -150G |
| 2 | 粉碎风车 → 年糕 | 加工费约10G |
| 3 | 年糕基础售价 | 约300G |
| 4 | 冬季流行趋势加成（+30%） | 约390G |
| 5 | 品牌认证加成（金牌×1.5） | 约585G |

**单品利润：** 约425G/个，批量生产效益显著。

### 6.3 珠宝加工流（Jewelry Processing）

**原理：** 采集矿石 → 黄色加工风车 → 制成饰品，高价值出售。

| 饰品名称 | 所需材料 | 加工时间 | 基础售价 | 金牌品牌价 |
|:--------:|:--------:|:--------:|:--------:|:--------:|
| 玛瑙项链 | 玛瑙矿石×3 + 金锭×1 | 6小时 | 15,000G | 22,500G |
| 水晶耳环 | 水晶矿石×2 + 银锭×1 | 5小时 | 18,000G | 27,000G |
| 钻石戒指 | 钻石原石×1 + 金锭×2 | 8小时 | 35,000G | 52,500G |
| 翡翠手镯 | 翡翠矿石×3 + 铜锭×2 | 6小时 | 22,000G | 33,000G |
| 珍珠项链 | 珍珠×3 + 银锭×2 | 4小时 | 28,000G | 42,000G |

### 6.4 台风送蛙流（Typhoon Frog Catching）

**原理：** 台风天特定区域出现稀有黑色青蛙（Black Frog），捕获后可高价出售。

| 项目 | 数据 |
|:---|:----|
| 出现条件 | 台风天气 |
| 出现地点 | 河流沿岸、瀑布下方 |
| 捕获工具 | 捕虫网 |
| 单只售价 | 2,500-3,500G |
| 台风天可捕获数量 | 3-5只 |
| 额外收益 | 台风天风力极强，风车加工速度大幅提升 |

## 七、风力与经济的关系

### 7.1 风力等级对加工速度的影响

风车加工是本作经济的重要环节，而风力直接影响加工速度：

| 风力等级 | 天气条件 | 加工速度倍率 |
|:-------:|:--------:|:----------:|
| 0（无风） | 晴朗无风 | ×0.5（极慢） |
| 1（微风） | 晴朗 | ×1.0（标准） |
| 2（和风） | 多云 | ×1.5 |
| 3（强风） | 阴天/雨前 | ×2.0 |
| 4（暴风） | 暴风雨 | ×3.0 |
| 5（台风） | 台风 | ×5.0（极快） |

**策略启示：** 台风天应优先使用风车加工批量商品，同时外出捕蛙；无风天则安排农活、采矿等不受风力影响的活动。

## 八、冬季经济

冬季无法进行户外种植，收入来源转为以下渠道：

| 收入来源 | 预计日收入 | 说明 |
|:--------|:---------:|:----|
| 畜牧产品 | 2,000-5,000G/日 | 乳制品、蛋类、羊毛持续产出 |
| 采矿 | 1,000-3,000G/日 | 矿石直接出售或加工为饰品 |
| 钓鱼 | 500-2,000G/日 | 鱼类出售 |
| 加工品 | 3,000-8,000G/日 | 利用风车加工库存原料 |
| 温室作物 | 1,000-3,000G/日 | 解锁温室后可冬季种植 |
| 珠宝加工 | 5,000-15,000G/日 | 最高效的冬季收入方式 |

## 十、总结

本作经济系统的核心设计理念是**"自由市场"**——取消了传统牧场物语的出货箱，迫使玩家参与每周的集市活动。自由定价、品牌认证、流行趋势三重机制叠加，创造了深度且有趣的定价策略空间。风车加工系统则提供了"原料→加工→高价出售"的价值链增值路径。经济系统的深度和自由度是本次重制版的最大亮点。
