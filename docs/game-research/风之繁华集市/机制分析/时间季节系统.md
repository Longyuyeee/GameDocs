# 时间季节系统分析
![风之集市节日](https://www.bokumono.com/series/kazebaza/img/ss/ss07.jpg)

> 游戏版本：牧场物语 来吧！风之繁华集市（Switch/Switch2/Steam，2025年8月28日发售）
> 原版：牧场物语 欢迎来到风之集市（NDS，2008年）

---

## 一、系统概述

本作的时间系统遵循牧场物语系列的经典框架，以**每周六集市**为节奏锚点，形成了"6天生产筹备 + 1天销售"的独特游戏循环。四季更替决定了作物种植、畜牧放牧和资源采集的内容变化。

## 二、基础时间框架

### 2.1 时间单位

| 时间单位 | 换算 | 说明 |
|:-------|:----|:----|
| 1帧（Frame） | 约1/60秒 | 最小时间单位 |
| 1分钟（游戏内） | 约2秒（现实） | 时间流速 1:30 |
| 1小时（游戏内） | 约60秒（现实） | 现实1分钟=游戏1小时 |
| 1天（游戏内） | 约24分钟（现实） | 6:00~次日6:00 |
| 1周 | 7天 | 集市周期 |
| 1个月 | 30天 | 季节单元 |
| 1年 | 120天（4季×30天） | 游戏年度 |

### 2.2 每日时间表

| 时间段 | 游戏内时间 | 状态 |
|:-----|:---------|:----|
| 清晨 | 6:00-8:00 | 起床、浇第一次水、照顾动物 |
| 上午 | 8:00-12:00 | 户外作业、采集、采矿 |
| 午间 | 12:00-14:00 | NPC活动（可社交） |
| 下午 | 14:00-18:00 | 继续作业、浇第二次水（16:00后） |
| 傍晚 | 18:00-20:00 | 收工、社交、料理 |
| 夜间 | 20:00-24:00 | 室内活动（加工、整理） |
| 深夜 | 24:00-6:00 | 强制睡眠（体力耗尽则昏倒） |

### 2.3 重要时间节点

| 节点 | 时间 | 说明 |
|:----|:----|:----|
| 起床 | 6:00 | 自动起床 |
| 第一次浇水窗口 | 6:00-12:00 | 最佳第一次浇水时段 |
| 店铺营业 | 8:00起 | 种子/物品购买 |
| 第二次浇水窗口 | 16:00-20:00 | 最佳第二次浇水时段 |
| 店铺关门 | 18:00 | 多数店铺停止营业 |
| 睡觉 | 任意时间 | 可上床睡觉结束一天 |
| 强制昏倒 | 6:00未睡 | 次日体力减半 |

## 三、每周集市节奏（Saturday Bazaar Rhythm）

集市是本作时间设计的**核心节拍器**。整个游戏节奏围绕"6+1"模式运行：

| 星期数 | 名称 | 活动重点 |
|:-----:|:----|:--------|
| Sun（日曜） | 准备日 | 浇水、收获、备料 |
| Mon（月曜） | 加工日 | 风车加工、采集 |
| Tue（火曜） | 社交日 | NPC互动、委托 |
| Wed（水曜） | 集中加工日 | 批量风车加工 |
| Thu（木曜） | 最终备货日 | 确认库存、补充生产 |
| Fri（金曜） | 集市前日 | 摊位规划、商品定价策略 |
| **Sat（土曜）** | **集市日** | **摆摊销售（6:00-18:00）** |

### 3.1 集市日流程

| 时间 | 活动 | 说明 |
|:----|:----|:----|
| 6:00 | 集市开放准备 | 将商品从仓库取出 |
| 6:00-8:00 | 摊位布置 | 摆放商品、设置价格、放置招牌 |
| 8:00-16:00 | 营业时间 | NPC顾客陆续到来，进行销售 |
| 12:00-13:00 | 客流高峰 | 最多顾客时段 |
| 16:00-18:00 | 收摊 | 未售出商品自动回收 |
| 18:00 | 结算 | 统计当日销售额，集市经验增加 |

### 3.2 集市日顾客行为

| 顾客类型 | 到访时段 | 购买偏好 | 价格敏感度 |
|:--------|:--------|:--------|:---------|
| 普通村民 | 8:00-14:00 | 日常食品/用品 | 中等 |
| 旅行商人 | 10:00-15:00 | 稀有品/加工品 | 低（高价品也会买） |
| 美食家 | 11:00-14:00 | 高星级食品/料理 | 极低（追求品质） |
| 收藏家 | 12:00-16:00 | 珠宝/饰品/稀有品 | 极低（追求稀有度） |
| 家庭主妇 | 8:00-12:00 | 便宜食材/日用品 | 高（追求性价比） |
| 小孩 | 10:00-14:00 | 甜点/玩具 | 高（零花钱有限） |

## 四、季节系统

### 4.1 四季总览

| 季节 | 天数 | 气候特征 | 农业活动 | 主要收入来源 |
|:---:|:---:|:--------|:--------|:----------|
| 春（Spring） | 1-30日 | 温暖多雨 | 春播作物、果树种植 | 蔬菜+蛋奶 |
| 夏（Summer） | 31-60日 | 炎热偶有台风 | 夏播作物、牧场放牧 | 水果+牛奶+羊毛 |
| 秋（Autumn） | 61-90日 | 凉爽晴朗 | 秋播作物、丰收季 | 根茎类+加工品 |
| 冬（Winter） | 91-120日 | 寒冷降雪 | ❌无户外种植 | 畜牧+加工+珠宝 |

### 4.2 季节更替事件

| 更替 | 日期 | 影响 |
|:----|:---:|:----|
| 春→夏 | 春30日→夏1日 | 春季作物枯萎，果树林继续存活 |
| 夏→秋 | 夏30日→秋1日 | 夏季作物枯萎，果树林继续存活 |
| 秋→冬 | 秋30日→冬1日 | 所有秋季作物枯萎，户外土地冻结 |
| 冬→春 | 冬30日→春1日 | 冻土解冻，新年种植季开始 |

### 4.3 季节对畜牧的影响

| 季节 | 放牧状态 | 产率影响 | 饲料需求 |
|:---:|:--------|:-------:|:-------:|
| 春 | 可放牧 | 标准（100%） | 放牧时不需 |
| 夏 | 可放牧 | 标准（100%） | 放牧时不需 |
| 秋（前半） | 可放牧 | 标准（100%） | 放牧时不需 |
| 秋（16日后） | 草地枯萎 | 下降至80% | 需饲料 |
| 冬 | ❌不可放牧 | 下降至70% | 需饲料（每日） |

## 六、天气系统

### 6.1 天气类型

| 天气 | 出现概率 | 主要季节 | 持续时间 | 对活动影响 |
|:----|:-------:|:-------:|:-------:|:---------|
| ☀️ 晴朗 | 40% | 全年 | 1-5天 | 全活动正常 |
| ⛅ 多云 | 25% | 春秋 | 1-3天 | 全活动正常，风力增益 |
| 🌧️ 雨 | 20% | 春夏 | 1天 | 户外作业效率↓，风车效率↑ |
| ⛈️ 暴风雨 | 10% | 夏 | 半天-1天 | 山地危险，风车效率极高 |
| 🌀 台风 | 5% | 夏末秋初 | 1天 | **户外活动危险，风车效率最大，黑色青蛙出现** |
| ❄️ 雪 | 15%（冬） | 冬 | 1-3天 | 户外作业极慢，风车效率↑ |
| 🌫️ 雾 | 5% | 春秋 | 半天 | 能见度降低 |

### 6.2 天气预报

| 获取方式 | 时间 | 预报精度 |
|:--------|:----|:--------|
| TV天气预报（家中） | 6:00 | 次日天气（90%准确） |
| TV天气预报（家中） | 18:00 | 次日天气（95%准确） |
| NPC对话 | 全天 | 后天天气（70%准确，随机） |
| 看窗外云彩 | 6:00 | 当天天气（玩家自行判断） |

### 6.3 台风应对策略

| 台风前准备 | 操作 |
|:----------|:----|
| 放置作物屏障 | 使用围栏/屏障保护作物 |
| 收容动物 | 台风自动收容（需猫狗高好感） |
| 备料风车 | 提前备好大量风车加工材料 |
| 准备捕虫网 | 台风天捕黑色青蛙 |
| 仓库整理 | 确保仓库有足够空位 |

## 七、体力与疲劳系统

### 7.1 体力（Stamina）系统

| 项目 | 数据 |
|:---|:----|
| 最大体力 | 初始100，每升级+10，最高200 |
| 恢复方式 | 睡觉（全恢复）、吃东西（部分恢复） |
| 体力消耗 | 每个工具动作消耗1-3点 |
| 空体力后果 | 动作效率下降50% |

### 7.2 疲劳（Fatigue）系统

| 疲劳等级 | 累积条件 | 效果 | 解除方式 |

| Lv.0 | 体力正常 | 无影响 | - |
| Lv.1 | 体力≤25% | 移动速度↓20% | 休息/进食 |
| Lv.2 | 体力≤10% | 动作速度↓50% | 立即休息 |
| Lv.3 | 体力=0继续行动 | 可能昏倒 | 强制送医 |

## 九、时间管理策略

### 9.1 不同季节的每日时间分配

| 季节 | 上午（6-12） | 中午（12-16） | 下午（16-20） | 夜间（20-24） |
|:---:|:-----------|:-----------|:-----------|:-----------|
| 春 | 浇水+耕种 | 采集+社交 | 二次浇水+加工 | 料理+整理 |
| 夏 | 浇水+畜牧 | 采矿+钓鱼 | 二次浇水+风车 | 加工+社交 |
| 秋 | 收获+加工 | 采集+委托 | 育种+备货 | 集市策略规划 |
| 冬 | 采矿+畜牧 | 风车加工 | 珠宝制作 | 社交+料理 |

### 9.2 集市周的时间规划

| 星期 | 高效利用法 |
|:---:|:----------|
| 周日 | 收获/浇水 → 风车加工（利用全天风力） |
| 周一 | 集中加工日：把本周要加工的全放入风车 |
| 周二 | 社交+委托日：接任务，刷好感 |
| 周三 | 采矿日：补充矿石库存 |
| 周四 | 最终加工+备货日：完成所有加工品 |
| 周五 | 定价策略日：检查流行趋势，设定价格 |
| 周六 | 集市日：销售+采购物资 |

## 十、总结

本作的时间系统以**集市周期锚定**为核心设计亮点。每周"6天准备、1天销售"的节奏创造了清晰的游戏循环，避免了传统牧场物语中"每天差不多的重复感"。四季系统则提供了年度战略规划维度——春季奠定基础，夏季扩张产能，秋季丰收变现，冬季转型加工。体力与疲劳系统作为时间资源的软约束，要求玩家合理分配每日活动。整体系列了紧凑而富有节奏感的游戏体验。
