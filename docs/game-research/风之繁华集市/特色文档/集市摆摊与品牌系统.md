# 集市摆摊与品牌系统
![品牌认证](https://www.bokumono.com/series/kazebaza/img/system/market.jpg)

游戏版本：牧场物语 来吧！风之繁华集市 (Switch/Switch2/Steam, 2025年8月28日)

> **文档重要性**：★★★★★ 本作最具特色的核心系统，定义了游戏的全部经济节奏和玩法循环。

---

## 一、核心概念：无出货箱的唯一经济系统

### 1.1 与传统牧场物语的最大区别

在绝大多数牧场物语系列作品中，玩家的核心收入来源是**出货箱**——每天把产品放进自家门口的箱子，次日自动结算金币。这是一个"生产→出货→收钱"的线性、被动流程。

**本作完全取消了出货箱。**

玩家的**全部收入**必须通过**每周六的集市摆摊**来实现。这一设计从根本上改变了游戏的节奏和经济策略。

### 1.2 "无出货箱"设计的影响

| 方面 | 传统出货箱模式 | 本作集市模式 |
|:----|:-------------|:------------|
| 收入频率 | 每日都有收入 | 每周只有一次收入 |
| 定价权 | 固定价格，玩家无定价权 | 玩家自由定价 |
| 库存压力 | 每天清空，无需规划 | 需要囤货到周六 |
| 生产节奏 | 每日均衡生产 | 6天准备，周六集中销售 |
| 策略深度 | 单一（种什么卖什么） | 复杂（种+加工+定价+营销） |
| 失败风险 | 低（每天都可出货） | 高（一周努力一次结算） |

### 1.3 设计哲学：为什么取消出货箱？

这个设计是本作最聪明的地方。它解决了牧场物语系列长久以来的核心痛点——**"每天重复相同操作"** 的疲劳感。

具体来说：
1. **制造预期和紧张感**：6天积累的成果在周六一次检验，让每个周六都成为"大日子"
2. **赋予定价权**：玩家决定价格增加了策略维度，思考"定多少能卖光又赚最多"
3. **创造周循环节奏**：周一到周五准备→周六销售→周日休息→周一开始新循环，形成自然节奏
4. **增加社交元素**：与顾客互动、摇铃铛招客，比扔进出货箱有趣得多
5. **放大成就感**：摊位摆满自产的高品质商品被抢购一空，远胜于冷冰冰的数字

## 二、集市基础流程

### 2.1 一周准备周期

```
周一~周五：耕作、浇水、施肥、照顾动物、风车加工、采集
  ↓
  周六：集中摆摊售卖
  ↓
  周日：休息/规划/采购下周物资
  ↓
  周一：开始新一周的循环
```

### 2.2 周六集市流程

**Step 1：出摊（10:00~10:30）**
- 前往小镇广场
- 选择自己的摊位位置
- 从库存选择商品摆放在摊位格子中

**Step 2：营业（10:30~15:00 上半场 / 15:00~19:00 下半场）**
- 摇铃铛吸引顾客
- 顾客走到摊位前时，按A键展示商品
- 顾客看到价格后决定是否购买
- 成功售出→获得金币→**积累声援量表**
- 下半场前可重新补货/调整

**Step 3：收摊（19:00）**
- 集市结束
- 未售出商品自动退回库存
- 结算当日营业额
- 营业额影响集市等级提升

### 2.3 摆摊技巧

| 技巧 | 说明 |
|:----|:-----|
| 摇铃铛 | 主动摇铃比被动等待更能吸引顾客 |
| 摊位位置 | 靠近入口的位置客流量更大 |
| 价格策略 | 定太高卖不掉，定太低利润少——找平衡点 |
| 商品组合 | 不同商品吸引不同顾客 |
| 招牌设置 | 设定合适的摊位主题（"自家直供"等） |

### 2.4 摊位升级

| 集市等级 | 摊位格子数 | 可解锁功能 |
|:--------:|:---------:|:-----------|
| Lv.1 | 3格 | 基础摆摊 |
| Lv.2 | 4格 | 摊位装饰（桌布、招牌） |
| Lv.3 | 5格 | 包装系统 |
| Lv.4 | 6格 | 高级装饰 |
| Lv.5 | 7~8格 | 精品陈列 |
| Lv.6~7 | 9格(满) | 全功能摊位 |

## 三、定价系统

### 3.1 自由定价机制

本作最与众不同的设计：**玩家为每件商品设定售价**。

定价界面中，系统会显示：
- **建议价格**：基于商品基础价值的推荐售价
- **品质加成**：高星级商品的溢价空间
- **趋势加成**：当月热门商品的额外溢价

### 3.2 定价策略

| 策略 | 做法 | 适用场景 |
|:----|:-----|:---------|
| 低价快销 | 略低于市场价，快速清仓 | 大量普通商品、急需资金 |
| 正常定价 | 按建议价出售 | 日常稳定销售 |
| 高价精品 | 定价高于建议价，博弈高利润 | 高品质/品牌商品、热门趋势 |
| 组合定价 | 不同商品不同价格 | 摊位位置有限时的效率最大化 |

### 3.3 影响售价的因素

| 因素 | 影响范围 | 说明 |
|:----|:--------:|:-----|
| 基础价值 | 物品本身的价值 | 每种物品有固定基础价 |
| 星星品质(☆) | ×1.0~×5.0 | 品质越高价格越高 |
| 品牌认证 | ×1.5~×3.5 | 比赛夺冠获得 |
| 摊位加成 | ×1.0~×1.5 | 装饰/位置加成 |
| 热门趋势 | ×1.0~×1.3 | 月度随机热门商品 |
| 小矮人声援 | ×1.0~×1.25 | 声援时间内的额外加成 |
| 占卜 | ×1.0~×1.15 | 找特定NPC占卜 |

## 四、品牌系统

### 4.1 什么是品牌？

品牌是本作中**产品获得官方认证**的系统。在节日比赛的"黄金级"评选中夺冠后，对应的产品或动物就会获得品牌认证。

### 4.2 品牌获得方式

| 比赛类型 | 可获得品牌的产品 |
|:---------|:---------------|
| 农作物评选赛 | 对应参赛作物 |
| 动物评选赛 | 对应参赛动物产出的产品 |
| 烹饪节 | 对应参赛料理 |
| 赛马大会 | 赛马 |

### 4.3 品牌效果

| 效果 | 说明 |
|:----|:-----|
| 售价提升 | 品牌产品基础售价直接上涨约×3倍 |
| 优先购买 | 品牌商品在集市更易被顾客抢购 |
| 摊位吸引力 | 品牌商品提升整体摊位人气 |
| **品牌等级** | 多次夺冠可提升品牌Rank，最高×3.5 |

### 4.4 品牌等级（Brand Rank）

| Rank | 价格倍率 | 解锁条件 |
|:----:|:-------:|:---------|
| Rank 1 | ×1.5 | 首次在该比赛夺冠 |
| Rank 2 | ×2.0 | 该比赛累计夺冠2次 |
| Rank 3 | ×2.5 | 累计夺冠3次 |
| Rank 4 | ×3.0 | 累计夺冠4次 |
| Rank 5 | ×3.5 | 累计夺冠5次以上 |

### 4.5 ⚠️ 重要规则：品牌与加工

**品牌效果只对原材料有效。一旦加工（烹饪/风车加工），品牌效果立即消失。**

- 正确做法：品牌卷心菜直接摆摊卖
- 错误做法：品牌卷心菜做成沙拉后再卖（品牌加成消失）

**例外**：部分特殊加工品（如冠军动物的奇迹牛奶做成极品奶酪）虽无品牌标签，但原材料品质高，成品仍有较高价值。

### 4.6 品牌经济价值计算

以卷心菜为例：

| 状态 | 7★卷心菜 | 7★+品牌Rank4 | 7★+品牌+摊位+趋势 |
|:----|:--------:|:------------:|:---------------:|
| 售价 | 1,120G | 3,360G | 约8,000G+ |
| 倍率 | ×2.8 | ×8.4 | ×20+ |

**这就是"品牌卷心菜流"的恐怖之处**——7★品质×品牌×摊位×趋势≈20倍以上收益。

## 五、趋势系统

### 5.1 趋势概念

每个季度，系统会随机选取某几类商品作为"热门商品"。热门商品在集市上可获得**最高+30%的价格加成**。

### 5.2 查看趋势

- 菜单 → 「集市信息」→ 「趋势」
- 显示当季度热门商品类型
- 热门商品在摊位上有「Hot/热门」标签

### 5.3 趋势策略

| 策略 | 做法 | 收益 |
|:----|:-----|:----|
| 顺势而为 | 多种植/加工热门类型商品 | +30%稳定收益 |
| 提前布局 | 根据下季度可能趋势提前生产 | 抢占先机 |
| 趋势+品牌 | 品牌商品正好是热门 | ×3.5品牌×1.3趋势=×4.55叠加 |
| 趋势+品质 | 高星热门商品 | ×5.0品质×1.3趋势=×6.5 |

### 5.4 趋势刷新时间

- 每个季度（季节变更时）刷新
- 每季度1~3个商品类型成为热门

## 六、小矮人（精灵）声援系统

### 6.1 声援量表

| 项目 | 内容 |
|:----|:-----|
| 累积方式 | 集市中成功售出商品 |
| 最大储存 | 5条量表 |
| 触发方式 | 量表满后按ZL键召唤小矮人啦啦队 |
| 每次触发消耗 | 1条量表 |

### 6.2 声援时间效果

进入"声援时间"后：

| 效果 | 说明 |
|:----|:-----|
| 自动贩卖 | 不需要移动，连续按A即可自动出售 |
| 无需补货 | 直接从库存中售卖（不需柜台有货） |
| **时间暂停** | 集市剩余时间不减少——可以疯狂售卖 |
| 价格加成 | 声援时间内商品售价×1.25 |

### 6.3 五大小矮人

| 小矮人 | 颜色 | 偏好礼物 | 声援效果 | 出现位置 |
|:-------|:----:|:---------|:---------|:---------|
| 卡卡波乐 | 🟣紫 | 石头、木片等采集物 | **品质UP**（商品品质提升） | 红色风车旁 |
| 奇奇波乐 | 🔵蓝 | 鱼类 | **鲜度UP**（保持新鲜度） | 牧场下方河边 |
| 库库波乐 | 🍯蜂蜜 | 巢蜜与蜂蜜 | **声援时间延长** | 山区 |
| 凯凯波乐 | ⚪白 | 菌菇类 | **销售速度UP**（售卖加速） | 山区 |
| 可可波乐 | 🔴红 | 昆虫、小动物 | **销售数量UP**（单次销量增加） | 咖啡厅屋顶 |

### 6.4 小矮人等级提升

- 向对应小矮人赠送喜欢的物品即可升级
- 等级越高，声援效果越强
- 每4级提升对应物品的品质上限0.5星

### 6.5 声援策略：高营收技巧

**20万营业额秘诀**：

1. **提前准备**：头天早睡，确保周六10点准时抵达
2. **按品质排列**：使用R键按高品质排序
3. **声援量表填充**：用低价商品（牧草99个、年糕60个）快速填充声援量表
4. **替换高价商品**：触发声援前将柜台商品替换为高单价商品
5. **疯狂售卖**：声援时间内连按A，利用时间暂停卖光高价商品

## 七、新鲜度系统

### 7.1 基本规则
- 食物类道具会随真实游戏时间流逝而**降低新鲜度**
- 新鲜度影响售价和顾客购买意愿
- 不同商品的保质期不同

### 7.2 新鲜度管理
- **不要大量囤货**：根据周六集市需要规划一周产量
- **加工延长保质期**：奶酪、酒类等加工品比原材料更耐放
- **风车储物箱**：加工品在储物箱中保持新鲜
- 部分小矮人能力可维持商品新鲜度

## 九、"集市"设计为什么是天才设计？

### 9.1 与传统出货箱的对比

| 维度 | 传统出货箱 | 集市摆摊 |
|:----|:----------|:---------|
| 节奏感 | 每天平淡无波 | 每周有高潮（周六） |
| 玩家参与 | 被动的"扔进去" | 主动的"卖出去" |
| 策略层次 | 种→收→卖（3步） | 种→加工→囤货→定价→营销（5步） |
| 社交元素 | 无 | 与顾客互动、摇铃铛 |
| 失误代价 | 低（明天还能卖） | 高（卖不掉要等下周） |
| 趣味性 | 机械重复 | 每次集市都不同 |
| 成就感 | 看数字增长 | 摊位被抢购一空 |

### 9.2 与其他游戏的对比

- **星露谷物语**的出货箱：方便但平淡，玩家建议增加"摆摊mod"不是没道理的
- **符文工房**的出货箱：同样，玩家主要靠出货箱+任务
- **动物森友会**的Nook商店：类似"卖给商店"，但缺少定价权
- **本作的集市**：独一无二的"定价权+周循环+社交"三位一体

### 9.3 设计总结

集市系统是本作最核心、最创新的设计。它通过取消出货箱，强制玩家参与"完整的经济链路"——不只是生产，还包括销售策略。这个设计：

1. **让农场工作有了明确的目标**：不是为了"出货数字"，而是为了"周六集市"
2. **创造了自然的周循环节奏**：防止了其他农场游戏"每天一样"的重复感
3. **给了玩家真正的定价权**：这是绝大多数农场游戏缺失的核心乐趣
4. **把经济系统变成了玩法本身**：而不是后台的一个数字计算
