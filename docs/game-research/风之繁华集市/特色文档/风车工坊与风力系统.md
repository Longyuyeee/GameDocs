## 一、风车系统概述

风车是本作的**经济中枢系统**。游戏中的三座风车承担着将初级农产品转化为高价值加工品的核心职能，是将"风"这个主题融入游戏系统的最重要体现。

### 1.1 三座风车总览

| 风车 | 颜色 | 位置 | 功能定位 | 类比 |
|:----|:----:|:----|:---------|:-----|
| 发酵风车 | 🔴红 | 牧场内 | 食品加工、乳制品、酿酒、茶罐 | 类似星露谷的"小桶" |
| 加工风车 | 🔵蓝 | 旅馆下方 | 工具升级、饰品、种子、粉油 | 类似"锻造台+种子机" |
| 粉碎风车 | 🟡黄 | 瀑布边 | 顶级工具、建材、香水 | 类似"终极工作台" |

### 1.2 与同类游戏的加工系统对比

| 游戏 | 加工系统 | 特点 | 与本作差异 |
|:----|:---------|:-----|:----------|
| 星露谷物语 | 小桶+罐头瓶+熔炉 | 固定时间产出 | 本作受风力影响时间可变 |
| 符文工房3/4 | 锻造台+料理台+制药 | 配方数量多 | 本作由风车统一管理 |
| 牧场物语·三个村庄 | 加工机(多个独立机器) | 每台机器做一种 | 本作三座风车各管一大类 |
| **本作风车系统** | 三座全能加工站 | **风力影响效率** | **核心差异化特色** |

**本作风车系统的独特之处**：
1. 风力影响加工时间——将天气系统与经济系统直接挂钩
2. 三座分工明确——形成自然的加工产业链
3. 使用次数解锁配方——鼓励持续使用和探索
4. 耀珠系统——后期可大幅提升效率

## 二、三座风车的详细功能

### 2.1 红色风车（发酵风车）

| 项目 | 内容 |
|:----|:-----|
| 解锁条件 | 第1年春6日，完成艾萨克委托（木材×10+石头×10） |
| 核心功能 | 发酵、乳制品、酿酒、腌制、茶罐、肥料 |
| 加工类型 | 时间型（需等待数小时） |
| 使用频率 | ★★★★★（最高） |

**典型加工链**：
```
牛奶 → 奶酪/黄油/酸奶
鸡蛋 → 蛋黄酱
水果 → 果酒/果酱
茶叶+水果 → 茶罐（最高利润）
杂草+树枝 → 肥料
```

### 2.2 蓝色风车（加工风车）

| 项目 | 内容 |
|:----|:-----|
| 解锁条件 | 集市等级Lv.3 + 完成"寻找遗失的包裹"任务 |
| 核心功能 | 工具升级(银→金)、饰品制作、种子生产、粉类油类加工 |
| 加工类型 | 时间型 |
| 使用频率 | ★★★★ |

**典型加工链**：
```
矿石+木材 → 升级工具
矿石+宝石 → 饰品（送礼/售卖）
羊毛/羊驼毛 → 毛线/纱线
小麦 → 面粉；糯米 → 年糕
作物 → 种子（高品质循环）
```

### 2.3 黄色风车（粉碎风车）

| 项目 | 内容 |
|:----|:-----|
| 解锁条件 | 集市等级Lv.4 + 累计风车制作100件物品 |
| 核心功能 | 奥利哈钢顶级工具、高级建材、香水、综合加工 |
| 加工类型 | 长时间型 |
| 使用频率 | ★★★ |

**典型加工链**：
```
金工具+奥利哈钢 → 奥利哈钢工具（最强）
花/水果+粉碎 → 香水（高利润）
高级木材 → 终极建材
```

## 三、风车配方解锁机制

### 3.1 使用次数累计

配方的解锁与**该风车累计使用次数**直接相关：

| 累计使用次数 | 解锁效果 |
|:-----------:|:---------|
| 0~10 | 基础配方可用 |
| 10~30 | 中级配方解锁 |
| 30~50 | 高级配方解锁 |
| 50~100 | 稀有配方解锁 |
| 100+ | 全部配方解锁（同时解锁黄色风车条件） |

### 3.2 快速刷次数技巧

- 用**肥料配方**（杂草+树枝）快速刷红色风车使用次数
- 肥料材料成本为0（地图采集），加工时间短
- 只需在风车中放入材料再取出，即算一次使用

## 四、风力系统

### 4.1 风力等级

游戏中的风力每天随机变化，直接影响风车加工速度和滑翔伞移动。

| 等级 | 名称 | 加工效率倍率 | 风速表现 | 户外影响 |
|:----:|:----|:-----------:|:---------|:---------|
| 1 | 无风/微风 | ×1.0 | 几乎感觉不到风 | 无影响 |
| 2 | 和风 | ×1.3 | 树叶微动 | 无影响 |
| 3 | 强风/大风 | ×1.8 | 树木摇摆，旗帜飘扬 | 行走略微偏移 |
| 4 | 暴风 | ×3.0 | 树木剧烈摇摆 | 行走偏移明显 |
| 5 | 台风 | ×5.0~瞬间完成 | 极端天气 | 行动受限，可能被吹落水中 |

### 4.2 风力信息获取

| 方式 | 获取信息 | 提前量 |
|:----|:---------|:------|
| 右上角时间面板图标 | 当前风力等级 | 实时 |
| 小屋收音机 | 天气预报 | 提前1天 |
| NPC对话 | 风力趋势 | 不定时 |

### 4.3 风力对加工时间的影响

**示例：奶酪加工**

| 风力等级 | 加工时间 |
|:--------:|:--------:|
| 无风(×1.0) | 2小时 |
| 和风(×1.3) | 约1小时32分 |
| 强风(×1.8) | 约1小时6分 |
| 暴风(×3.0) | 40分钟 |
| 台风(×5.0) | 约24分钟 |

**示例：奥利哈钢工具**

| 风力等级 | 加工时间 |
|:--------:|:--------:|
| 无风(×1.0) | 1天6小时 |
| 强风(×1.8) | 约17小时 |
| 暴风(×3.0) | 约10小时 |
| 台风(×5.0) | 约6小时 |

### 4.4 风力利用策略

| 策略 | 做法 | 收益 |
|:----|:-----|:----|
| 风等法 | 把大量加工材料留到大风天再处理 | 加工时间减半以上 |
| 台风冲刺 | 台风日集中加工最难/最贵的配方 | 极速产出 |
| 日常小批量 | 微风日只做少量简单加工 | 基础利用 |
| 天气预报依赖 | 根据收音机预报规划一周加工计划 | 效率最大化 |

### 4.5 风力对滑翔伞的影响

| 风力等级 | 滑翔效果 | 操作难度 |
|:--------:|:---------|:--------:|
| 无风 | 滑翔距离短，持续下降 | 简单 |
| 和风 | 正常滑翔 | 简单 |
| 强风 | 滑翔距离远，速度更快 | 中等 |
| 暴风 | 滑翔速度快但难以控制方向 | 困难 |
| 台风 | **不建议滑翔**，极度危险 | 极高风险 |

### 4.6 台风日的风险

台风不仅仅带来好处——
- 户外的**未收割作物可能受损**
- 放牧的动物压力值增加，可能生病
- 玩家行动受限，可能被吹落水中
- 需要建造**防风栏**保护作物

**防风栏**的作用：
- 在农田周围设置防风栏可大幅降低台风对作物的伤害
- 是进入游戏中后期必备的农场设施

## 五、NDS原版麦克风吹气机制（重制版已移除）

### 5.1 原版机制

NDS原版有一个独特的*硬件互动*机制：
- 玩家对着NDS的**麦克风吹气**
- 风车听到"风声"后会加速旋转
- 音乐声响起表示加速成功
- 效果持续到次日

**效果倍率**：
| 方式 | 加速效果 |
|:----|:--------:|
| 吹一次 | 加工速度约×2 |
| 大风天+吹气 | 叠加效果，效率极高 |
| 吹气+蓝色耀珠 | 再叠加，极限加速 |

### 5.2 为什么重制版移除了？

| 原因 | 说明 |
|:----|:-----|
| 硬件不兼容 | Switch/Switch2/PC无麦克风吹气功能 |
| Steam无法实现 | PC无对应输入设备 |
| 平衡性调整 | 吹气过于OP（太强），移除后更平衡 |
| 统一体验 | 全平台统一操作体验 |

### 5.3 替代方案

重制版用以下设计弥补移除吹气后的体验：
- 天气预报系统更精准
- 耀珠系统保留且效果稳定
- 风力系统本身的设计足够有趣
- 暴风/台风日的加速效果比原版更显著

## 六、耀珠（Sprite Gems）系统

### 6.1 耀珠基础信息

耀珠是安装在各风车**投入口左侧**的强化宝石，每个风车最多安装6颗。

| 耀珠 | 颜色 | 价格(G) | 效果 | 购买时期 |
|:----|:----:|:-------:|:----|:--------:|
| 蓝色耀珠 | 🔵 | 10,000 | 风车加工时间**减半** | 第2年秋冬 |
| 橙色耀珠 | 🟠 | 10,000 | 加工**数量上限提升至30个** | 第2年春冬 |
| 绿色耀珠 | 🟢 | 10,000 | **种类上限提升至12种** | 第3年春夏 |
| 黄色耀珠 | 🟡 | 10,000 | 每制作5个**偶尔+1个额外产出** | 第3年春秋 |
| 红色耀珠 | 🔴 | 10,000 | 加工品**品质上升** | 第3年秋冬 |
| 紫色耀珠 | 🟣 | 10,000 | 可制作的**种类增加** | 第3年春夏 |

### 6.2 耀珠安装

- 每个风车安装口独立
- 同一颗耀珠只能装在一个风车上
- 可随时拆卸重新安装
- 解装在集市等级Lv.4后的弗乔商店

### 6.3 耀珠策略指南

| 优先级 | 耀珠 | 推荐理由 |
|:-----:|:----|:---------|
| ★★★★★ | 蓝色(时间减半) | 加工时间减半=产能翻倍，任何风车都适用 |
| ★★★★ | 黄色(额外产出) | 平均每5个+1个产出=免费+20%利润 |
| ★★★★ | 红色(品质上升) | 更高品质=更高售价 |
| ★★★ | 橙色(数量上限) | 需要批量加工时很有用 |
| ★★ | 绿色(种类上限) | 需要多种材料同时加工才用得上 |
| ★★ | 紫色(增加种类) | 解锁更多配方，后期填充用 |

### 6.4 推荐安装顺序

```
第一优先：红色风车(使用最频繁)
  → 装：蓝色耀珠 + 黄色耀珠
  →
第二优先：蓝色风车(次频繁)
  → 装：蓝色耀珠
  →
第三优先：黄色风车(使用较少)
  → 装：待定
```

## 七、风车产业链：从初级到终极

### 7.1 初级产业链（第一年春~夏）

```
采集杂草+树枝 → 红色风车→ 肥料(提升作物品质)
小麦 → 蓝色风车 → 面粉(烘焙原料)
牛奶 → 红色风车 → 奶酪/黄油(基础加工品)
```

### 7.2 中级产业链（第一年秋~第二年）

```
水果+葡萄酒 → 红色风车 → 果酒(高利润)
糯米 → 蓝色风车 → 年糕(热门集市商品)
矿石+木材 → 各风车 → 升级工具(扩大产能)
秋茶叶+水果 → 红色风车 → 茶罐(游戏中最高利润加工品之一)
```

### 7.3 高级产业链（第二年以后）

```
金工具+奥利哈钢 → 黄色风车 → 奥利哈钢工具(顶级效率)
花/香草+加工 → 黄色风车 → 香水(高价值送礼/售卖)
多种水果+顶级乳制品 → 红色风车 → 极品水果酸奶(最高价值)
四季茶叶+多种水果 → 各风车 → 全套茶罐+酒类品牌矩阵
```

## 八、风车与风——统一的主题设计

### 8.1 "风"作为统一主题

本作是牧场物语系列中**唯一以风力为核心主题**的作品。从游戏标题"来吧！风之繁华集市"开始，"风"贯穿整个游戏：

| 系统 | 与风的关联 |
|:----|:----------|
| 风车加工 | 风力直接影响加工效率 |
| 风力等级 | 每天随机变化，决定生产计划 |
| 滑翔伞 | 借助风力探索地图 |
| 天气系统 | 风力是天气预报的核心内容 |
| 台风风险 | 高风险高回报的天气事件 |
| 游戏标题 | "风之"直接点题 |
| 小镇名称 | "煦风镇"(Zephyr Town)——Zephyr意为和风 |
| 世界设定 | 风是导致集市衰落和重振的隐喻 |

### 8.2 与其他风车/加工系统的对比

| 游戏 | 风车系统 | 本作的独特性 |
|:----|:---------|:------------|
| 符文工房3 | 锻造台(固定时间) | 无天气影响，稳定但单调 |
| 星露谷物语 | 小桶(固定时间) | 无变量，追求数量最大化 |
| 牧场物语·三个村庄 | 加工机(固定时间) | 同样稳定，无策略深度 |
| **本作** | **风力可变时间** | **天气→策略→经济的深层联动** |

本作的风车系统最大的差异化在于：它把**天气这个通常被农场游戏视为"背景"的要素，变成了核心策略变量**。玩家不是被动接受天气，而是主动利用天气——这才是高明之处。
