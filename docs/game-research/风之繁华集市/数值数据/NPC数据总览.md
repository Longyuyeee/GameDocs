### 3.3 好感度提升技巧

1. **每日对话**：基础好感度获取，每天一次
2. **送礼**：送出最喜欢/喜欢物品，生日送礼效果翻倍
3. **包装**：集市等级Lv.3后解锁包装（200G/个），额外增加好感度
4. **完成委托**：地图上感叹号标识的任务
5. **关联角色**：部分可攻略角色需要同时与其亲友保持良好关系
6. **最佳互动时段**：早8点至晚22点
7. **重制版新增：范围打招呼**——一次性增加多人好感度

## 四、结婚系统

### 4.1 结婚条件

| 条件 | 要求 |
|:----|:-----|
| 好感度 | ♥10（红色，65,530以上） |
| 双人床 | 集市家具屋购买(约10,000G) |
| 恋爱事件 | 全部触发 |
| 青之羽毛 | 第二年春1日后可购买或获得 |
| 关联人物好感 | 相关亲友好感度30,000以上 |

### 4.2 结婚流程
1. 好感度达到♥10
2. 购买双人床
3. 触发全部恋爱事件（各角色3个事件）
4. 第二年春1日后，持有青之羽毛
5. 向对方求婚
6. 举办婚礼（教堂/广场）
7. 婚后：伴侣协助农场工作，**生子系统**

### 4.3 重制版特色
- **支持同性婚姻**：无论主角性别，均可与任意12个候补结婚
- **全语音剧情**：重要恋爱剧情全程语音演绎
- **新角色追加**：新太和黛安娜有全新的恋爱剧情线
