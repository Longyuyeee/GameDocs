# 技能属性数据总览

游戏版本：牧场物语 来吧！风之繁华集市 (Switch/Switch2/Steam, 2025年8月28日)

> **说明**：本作**无传统技能系统**。没有角色等级、技能树、技能点数等传统RPG养成体系。游戏的成长体现在以下三个核心系统中。

---

## 一、核心成长系统

本作没有"角色等级"概念，以下三个系统共同构成玩家的成长线：

| 成长系统 | 对应传统RPG概念 | 说明 |
|:---------|:--------------:|:-----|
| 集市等级 | 主线进度 | 通过集市营业额提升，解锁新功能 |
| 名声/称号 | 声望系统 | 重振"世界第一大集市"称号 |
| 品牌认证 | 装备/成就 | 产品获得品牌加成的认可系统 |

---

## 二、集市等级系统（Bazaar Rank 1~7）

集市等级是游戏的核心进度系统，共7级。

### 2.1 等级提升条件

| 等级 | 名称 | 升级条件 | 解锁内容 |
|:---:|:----|:---------|:---------|
| Lv.1 | 微風集市 | 初始状态 | 基础摆摊(3个摊位格子) |
| Lv.2 | 成长集市 | 累计一定营业额 | 摊位装饰系统解锁、新店铺进驻 |
| Lv.3 | 热闹集市 | 更高营业额目标 | 蓝色风车解锁、包装系统(200G/个)、新NPC |
| Lv.4 | 繁荣集市 | 达成指定条件 | 黄色风车解锁、更多摊位格子、耀珠商店 |
| Lv.5 | 盛大集市 | 高级条件 | 摊位格子扩展至最大、精品店进驻 |
| Lv.6 | 辉煌集市 | 困难条件 | 最高级店铺解锁 |
| Lv.7 | 世界第一大集市 | 终极目标 | 游戏通关、全部店铺解锁 |

### 2.2 营业额要求参考

| 集市等级 | 每周目标营业额(大约) | 累计要求 |
|:-------:|:------------------:|:--------:|
| Lv.1→2 | 10,000~20,000G | 达到目标即可 |
| Lv.2→3 | 20,000~30,000G | 累计达到一定金额 |
| Lv.3→4 | 40,000~50,000G | 特定条件+营业额 |
| Lv.4→5 | 60,000~80,000G | 更高要求 |
| Lv.5→6 | 100,000G+ | 高难度要求 |
| Lv.6→7 | 150,000G+ | 游戏终极目标 |

### 2.3 摊位格子扩展

| 集市等级 | 可用摊位格子数 |
|:-------:|:-------------:|
| Lv.1 | 3格 |
| Lv.2 | 4~5格 |
| Lv.3 | 5~6格 |
| Lv.4 | 6~7格 |
| Lv.5 | 7~8格 |
| Lv.6 | 8~9格 |

## 三、名声/称号系统

### 3.1 名声概念
- 小镇原名：**「世界第一大露天集市」**
- 由于前任牧场主离世，集市迅速没落
- 玩家通过每周集市赚取营业额，**恢复小镇昔日荣光**
- 名声等级影响顾客数量、商品售价、新功能解锁

### 3.2 名声收益

| 名声阶段 | 效果 |
|:--------|:-----|
| 低级 | 少量顾客光临、基础售价 |
| 中级 | 更多顾客、更高售价、摊位装饰可用 |
| 高级 | 精品顾客出现、品牌商品被抢购 |
| 顶级(世界第一大集市) | 最大客流量、最高售价倍率、全部功能解锁 |

## 四、品牌认证系统

### 4.1 品牌获得
- 在**黄金级比赛**（动物评选赛/农作物评选赛/烹饪节）中夺冠
- 对应的产品或动物获得品牌认证

### 4.2 品牌效果
- 品牌产品基础售价直接上涨（约×3倍）
- 集市摆摊时更易被顾客抢购
- 摊位吸引力提升

### 4.3 品牌等级（Rank 1~5）

| 品牌等级 | 价格倍率 | 解锁条件 |
|:-------:|:-------:|:---------|
| Rank 1 | ×1.5 | 首次夺冠 |
| Rank 2 | ×2.0 | 多次夺冠积累 |
| Rank 3 | ×2.5 | 更高成就 |
| Rank 4 | ×3.0 | 多次高级别比赛胜利 |
| Rank 5 | ×3.5 | 最高级别冠军 |

> **注意**：品牌商品一旦加工（烹饪/风车加工），品牌效果会**立即消失**！建议品牌产品直接销售。

## 五、体力（GUTS）系统

### 5.1 基础体力

体力是执行所有劳作（浇水、施肥、耕地、采集等）的基础消耗资源。

| 项目 | 数值 |
|:----|:----|
| 初始体力上限 | 约100（基础值） |
| 体力归零 | 角色直接晕倒，强制进入下一天 |
| 体力过半 | 疲劳度较高，使用工具可能出现动作失败 |

### 5.2 体力上限提升

通过使用**力量树果实**可永久提升体力上限：

| 使用次数 | 累计效果 |
|:--------:|:--------:|
| 第1次 | +10上限 |
| 第2次 | +10上限 |
| ... | ... |
| 第10次(上限) | +100上限(累计) |

> 力量树果实在以下途径获得：挖掘土壤、抽奖、耕地、称号奖励等。获取没有上限，但只有前10次有效，每次增加体力上限。

### 5.3 体力恢复

| 方式 | 恢复量 | 条件/限制 |
|:----|:------|:---------|
| 早睡(22点前) | 完全恢复 | 次日6点起床 |
| 正常睡眠(22~1点) | 正常恢复 | 起床时间延后 |
| 熬夜(1~5点) | 恢复量递减 | 10点起床 |
| 晕倒(5点后) | 体力清空 | 强制下一天 |
| 泡澡 | 约半管体力 | 消耗30分钟，冷却8小时，每日最多2次 |
| 吃料理 | 依料理品质 | 星星越高恢复越多 |
| 咖啡店购买 | 中等恢复 | 前期可用 |

### 5.4 体力消耗参考

| 行动 | 体力消耗 |
|:----|:--------:|
| 耕地1格 | 约2 |
| 浇水1格 | 约1 |
| 收获作物 | 约1 |
| 砍树 | 约3~5 |
| 采矿(挥锤) | 约4 |
| 钓鱼 | 约1~2 |
| 跳跃使用工具 | 消耗更多但覆盖范围大 |

### 5.5 新手体力管理策略

1. **早睡早起 + 泡澡循环**：
   - 22点睡 → 6点起
   - 浇水施肥照顾动物（8~9点完成）
   - 泡澡恢复大半体力
   - 外出采集/采矿/钓鱼
   - 晚18点左右再次浇水 + 泡澡（注意8小时冷却）

2. **特殊料理增益**：
   - 红茶：浇水时提升稀有农作物出现概率
   - 蜂蜜水：减少体力消耗值

## 六、其他属性相关

### 6.1 新鲜度系统
- 食物类道具会随时间变质
- 影响售价和送礼效果
- 需合理规划库存，不要囤积过多

### 6.2 小矮人等级
- 向对应小矮人赠送喜欢物品即可提升等级
- 等级提升后，该小矮人相关物品的品质上升（每4级提升0.5星）
- 采集物、捕虫、钓鱼等物品的星级上限受小矮人等级影响

### 6.3 无战斗属性
本作无战斗系统，因此不存在：
- HP/MP
- 攻击力/防御力
- 魔法/技能
- 经验值/等级
- 武器/防具
