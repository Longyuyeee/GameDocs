# 角色属性战斗数据总览

游戏版本：牧场物语 来吧！风之繁华集市 (Switch/Switch2/Steam, 2025年8月28日)

---

## 一、无战斗系统

本作作为温馨生活模拟游戏，**不存在任何形式的战斗系统**。因此以下内容在本作中不存在：

| 不存在的系统 | 说明 |
|:------------|:-----|
| ❌ 战斗系统 | 无任何战斗发生 |
| ❌ 武器 | 无剑、魔法等攻击性武器 |
| ❌ 敌人/怪物 | 无任何敌对生物 |
| ❌ HP | 无生命值概念 |
| ❌ MP/SP | 无法力/技能值 |
| ❌ 攻击力/防御力 | 无战斗属性 |
| ❌ 等级/Lv | 无角色等级 |
| ❌ 经验值/EXP | 无经验值系统 |

## 二、角色唯一核心数值系统：体力（GUTS）与疲劳

本作角色唯一的生存/活动限制数值是**体力系统**。

### 2.1 体力系统

| 项目 | 数值 |
|:----|:----|
| 初始体力上限 | 约100（基础值） |
| 体力表示 | 屏幕左上角的GUTS槽 |
| 体力归零 | 角色直接晕倒，强制进入下一天 |
| 体力过半副作用 | 使用工具可能出现动作失败 |

### 2.2 疲劳机制

本作**无独立疲劳值显示**，但疲劳影响通过以下机制体现：

| 状态 | 表现 | 应对方法 |
|:----|:-----|:---------|
| 体力充足(>50%) | 所有工具正常使用 | 正常劳作 |
| 体力中等(25%~50%) | 动作失败概率增加 | 吃料理、泡澡恢复 |
| 体力低下(<25%) | 动作频繁失败 | 立即休息/恢复 |
| 体力归零 | 强制晕倒→下一天 | 避免发生 |
| 熬夜(凌晨1~5点) | 次日起床晚、体力恢复少 | 尽量22点前睡觉 |
| 凌晨5点后不睡 | 强制晕倒 | 避免通宵 |

### 2.3 体力消耗表

| 行动 | 单次体力消耗(约) |
|:----|:--------------:|
| 耕地1格 | 2 |
| 播种1格 | 1 |
| 浇水1格 | 1 |
| 施肥1次 | 1 |
| 收获1个 | 1 |
| 砍树1棵 | 3~5 |
| 采矿(挥锤)1次 | 4 |
| 钓鱼1次 | 1~2 |
| 抱鸡/刷动物毛 | 0.5 |
| 跳跃中使用工具 | 消耗更多(扩大范围) |

### 2.4 体力恢复方式

| 方式 | 恢复量 | 条件/限制 | 推荐度 |
|:----|:------|:---------|:------:|
| 睡眠(22点前) | 完全恢复 | 6点起床 | ★★★★★ |
| 睡眠(22~1点) | 正常恢复 | 起床延后1~3小时 | ★★★★ |
| 睡眠(1~5点) | 减少恢复 | 10点起床 | ★★ |
| 泡澡 | 约半管 | 30分钟，冷却8小时，每天最多2次 | ★★★★★ |
| 吃料理 | 依品质 | 需提前准备食材 | ★★★★ |
| 咖啡店购买食物 | 中等 | 消耗金币 | ★★★ |
| 力量树果实 | 永久+上限 | 最多10次有效 | ★★★★★ |

### 2.5 体力上限提升

通过**力量树果实(力量木之実)**永久提升体力上限：

| 使用次数 | 体力上限增加 |
|:--------:|:-----------:|
| 第1次 | +10 |
| 第2次 | +10 |
| 第3~9次 | 各+10 |
| 第10次(最后有效) | +10 |
| **累计最大** | **+100** |

获取途径：挖掘土壤、耕地、抽奖、称号奖励、特殊事件等。

## 三、其他相关数值

### 3.1 金币
- 初始持有：500G
- 收入来源：仅通过**周六集市摆摊**获取（无出货箱）
- 用途：购买种子、动物、工具升级材料、生活用品

### 3.2 新鲜度
- 食物类道具会随时间和天气变质
- 影响售价和送礼效果
- 建议不要大量囤积新鲜食品

### 3.3 小矮人等级（精灵等级）
- 向小矮人赠送喜爱的物品即可提升等级
- 每4级提升对应物品品质上限0.5星
- 影响采集物、捕虫、钓鱼等物品的最高品质

### 3.4 时间系统
- 每日时间流动：现实1秒 ≈ 游戏1分钟（约）
- 每日6:00起床（22点前睡）/ 不同就寝时间影响起床时间
- 凌晨5:00为日期切换点
- 春1日~冬30日为一年，共120天

## 四、总结

本作角色属性系统极度简洁：
- **体力（GUTS）**是唯一需要管理的角色资源
- **无战斗、无等级、无技能树**
- 游戏核心发展体现在**集市等级**、**品牌认证**、**好感度**三个维度
- 角色成长 = 经济实力 + 社交关系 + 玩法经验

> 这种设计体现了牧场物语系列一贯的"慢生活"理念，去除了战斗压力，让玩家专注于农场经营和社交互动。
