# 时间季节系统
![三村节日](https://www.bokumono.com/series/tots/img/ss/ss06.jpg)

游戏版本: 牧场物语 三个村庄的珍贵朋友(Story of Seasons: Trio of Towns) (Nintendo 3DS, 2016年)

---

## 1. 整体时间结构

| 时间单位 | 数值 | 备注 |
|----------|------|------|
| 1天(1 Day) | 约15分钟(现实时间) | 时间按帧推进 |
| 1季(1 Season) | **30天** | 与Stardew Valley的28天不同 |
| 1年(1 Year) | **120天** | 4季×30天 |
| 游戏开始 | 1年目 春1日 | |

> **注意**: 本游戏每季为30天,非28天。年度总天数120天。

---

## 2. 每日时间结构

### 时间流速

| 时间段 | 时间流速 | 备注 |
|--------|----------|------|
| 6:00~18:00 | 正常流速 | 白天,户外活动 |
| 18:00~24:00 | 正常流速 | 傍晚~夜晚 |
| 0:00~6:00 | 正常流速 | 深夜(大多数活动受限) |

### 每日作息表

| 时间 | 事件 |
|------|------|
| 6:00 AM | **起床**(强制) |
| 6:00~8:59 AM | 农场晨间工作(浇水、照料动物) |
| 9:00 AM | 商店开始营业 |
| 9:00~11:59 AM | 适合外出购物/做兼职/社交 |
| 12:00 PM(正午) | 中期时间段 |
| 12:00~4:59 PM | 下午活动(钓鱼/采矿/加工) |
| **5:00 PM(17:00)** | **出货箱结算** |
| 5:00~5:59 PM | 出货后活动 |
| 6:00 PM(18:00) | 商店陆续关门 |
| 6:00~11:59 PM | 夜晚活动(如钓鱼、社交) |
| 0:00 AM(凌晨) | 强制疲劳提醒 |
| 6:00 AM(次日) | 新一天开始 |

**强制睡眠时间**: 凌晨0:00以后,角色会显示疲劳状态。如果再拖下去会在2:00~3:00自动晕倒,次日体力减半。

## 3. 体力系统(Stamina)

| 属性 | 说明 |
|------|------|
| 初始最大体力 | 100 |
| 体力上限提升 | 通过房屋扩建(浴室)等 |
| 消耗体力 | 农作(耕地-2/格,浇水-1/格,收获-1/格) |
| 体力恢复 | 吃东西、泡澡、睡觉 |

### 体力消耗表

| 行动 | 体力消耗 |
|------|----------|
| 锄头耕地1格 | -2 |
| 水壶浇水1格 | -1 |
| 收获作物1格 | -1 |
| 斧头砍树1下 | -5 |
| 锤子敲矿1下 | -3 |
| 镰刀割草1格 | -1 |
| 钓鱼1次 | -2 |

### 体力恢复方式

| 方式 | 恢复量 | 备注 |
|------|--------|------|
| 吃料理 | 视料理品质 | 5~80+ |
| 泡澡(浴室扩建后) | 50/次 | 每日1次 |
| 睡觉 | 100%恢复 | 必须在6AM前 |
| 吃药材 | 10~20 | 可在诊所购买 |

## 4. 天气系统

| 天气类型 | 出现季节 | 效果 |
|----------|----------|------|
| 晴天(Sunny) | 全年 | 无任何负面效果,可放牧 |
| 阴天(Cloudy) | 全年 | 无特别影响 |
| 雨天(Rainy) | 春夏秋 | 不用浇水,不可放牧,钓鱼收益增加 |
| 暴雨(Heavy Rain) | 春夏 | 部分作物可能受损 |
| **台风(Typhoon)** | 夏末秋初 | **可摧毁作物!** 作物可能被吹走 |
| 雪天(Snowy) | 冬季 | 不用浇水,不可放牧 |
| 暴风雪(Blizzard) | 冬季 | 可能破坏作物 |

### 台风应对

| 防护手段 | 效果 | 备注 |
|----------|------|------|
| 温室(Greenhouse) | 完全防护 | 最佳方案 |
| 台风预告 | 前日电视/收音机预报 | 提前收获已成熟作物 |
| 无防护作物 | 随机损失30~70% | 重大经济损失 |

## 6. 年度机制

### 果树年增长

| 树龄 | 效果 | 说明 |
|------|------|------|
| 第1年 | 首次结果 | 品质较低(0.5★~1.0★) |
| 第2年 | 品质提升 | 提高约0.5~1.0★ |
| 第3年+ | 逐年提升 | 持续改善品质 |

### 爸爸的秘诀(Dad's Tips)跨年追踪

| 秘诀编号 | 类型 | 追踪内容 |
|----------|------|----------|
| #1 | 种植 | 累计出货次数 |
| #2 | 经济 | 累计出货金额 |
| #3 | 经济 | 累计500,000G出货 |
| #4 | 综合 | 累计1,000,000G出货 |

> Dad's Tips从游戏开始即计数,跨越年份不会重置。

## 7. 生日系统

| 类别 | 说明 |
|------|------|
| 主角生日 | 由玩家在游戏开始时设定 |
| NPC生日 | 每个NPC有固定生日(游戏文档可查) |
| 生日礼物效果 | 送礼效果 **×3倍**(FP增加量) |
| 配偶生日 | 结婚后配偶生日有特殊对话 |

**生日送礼策略**: 在NPC生日当天赠送其最爱礼物(Loved Gift),可大幅提升好感度(+500×3=+1,500FP),是快速提升好感的最有效手段。

## 9. 时间管理策略

| 目标 | 推荐时间安排 |
|------|-------------|
| 高效农作 | 6:00~8:00浇水施肥 |
| 动物照料 | 8:00~9:00刷毛喂食放牧 |
| 购物/社交 | 9:00~12:00(商店营业时间) |
| 兼职 | 9:00~12:00(兼职通常半天) |
| 采矿/伐木 | 12:00~16:00 |
| 出货+全天总结 | 17:00前放入出货箱 |
| 料理/加工 | 18:00~22:00 |
| 钓鱼/夜间社交 | 18:00~23:00 |

**每日黄金时间**: 9:00~12:00是社交和购物的最佳时段(所有商店都在营业)。

## 10. 与Stardew Valley的时间对比

| 对比项 | 牧场物语 三个村庄 | Stardew Valley |
|--------|-------------------|----------------|
| 每季天数 | **30天** | 28天 |
| 全年天数 | 120天 | 112天 |
| 游戏内1小时 | ≈60秒 | ≈43秒 |
| 完全通关所需年数 | 3~5年(典型) | 2~3年(典型) |
| 体力系统 | 有(100基础) | 有(270基础) |
| 强制睡眠 | 2:00AM晕倒 | 2:00AM晕倒 |
| 台风 | 有(可毁作物) | 无(但有暴风雨) |
