## 2. 好感度系统(FP/Friendship Points)

### FP数值与花朵阶段

好感度通过 **FP(Friendship Points)** 衡量,以花朵颜色表示阶段:

| 花朵阶段 | 花朵颜色 | FP范围 | 解锁内容 |
|----------|----------|--------|----------|
| 第1阶段 | 白色(White) | 0~10,000 | 基础对话 |
| 第2阶段 | 紫色(Purple) | 10,000~20,000 | 特殊对话1 |
| 第3阶段 | 浅蓝(Light Blue) | 20,000~30,000 | 特殊对话2,接支线 |
| 第4阶段 | 深蓝(Dark Blue) | 30,000~40,000 | 特殊事件 |
| 第5阶段 | 浅绿(Light Green) | 40,000~50,000 | **可告白(告白对象)** |
| 第6阶段 | 红色(Red) | 50,000+(告白成功后) | **恋爱状态(仅限伴侣)** |

### FP增减方式

| 行动 | FP变化 | 每日限制 |
|------|--------|----------|
| 对话(Talk) | +200 | 每日1次 |
| 送礼(喜欢) | +300 | 每日1次 |
| 送礼(最爱) | **+500** | 每日1次 |
| 送礼(讨厌) | -500 | 每日1次 |
| 送礼(最讨厌) | **-1,000** | 每日1次 |
| 节日参赛 | +1,000~3,000 | 按节日 |
| 兼职同做 | +200 | 按兼职 |
| 生日送礼(最爱) | **+1,500**(500×3) | 仅限生日当天 |

## 4. 告白系统(Confession)

### 告白条件

| 条件 | 说明 |
|------|------|
| 花朵阶段 | 必须达到 **浅绿花(Light Green, 40,000~50,000FP)** |
| 必需道具 | **告白吊坠(Confession Pendant)** |
| 吊坠价格 | **150,000G**(苗服模式 **105,000G**) |
| 告白地点 | 任意地点(向目标NPC使用吊坠) |

### 告白吊坠获取

| 方式 | 详情 |
|------|------|
| 购买地点 | 西部镇(Westown)杂货店 |
| 购买条件 | 村庄等级达到一定程度 |
| 价格 | 150,000G |
| 使用方式 | 在目标NPC面前使用(持有吊坠时选择"使用") |

### 告白成功后的流程

```
告白成功
  → 进入"恋人"状态(红花朵)
  → 继续提升好感
  → 触发求婚事件
  → 准备结婚物品(因村庄而异)
  → 举办婚礼
```

## 5. 婚姻系统(Marriage)

### 各村结婚物品

| 村庄 | 结婚物品 | 象征意义 |
|------|----------|----------|
| 西部镇(Westown) | 结婚戒指(Wedding Ring) | 西式婚礼传统 |
| 露草里(Tsuyukusa) | 发梳(Hair Comb) | 日式婚礼传统 |
| 露露可可村(Lulukoko) | 蓝羽(Blue Feather) | 热带岛屿传统 |

### 结婚流程

| 步骤 | 说明 |
|------|------|
| 1. 告白成功 | 使用告白吊坠 |
| 2. 恋爱状态 | 红花朵阶段 |
| 3. 求婚预备 | 好感度继续提升到一定阈值 |
| 4. 触发求婚事件 | 特定天气/地点触发 |
| 5. 准备结婚物品 | 购买对应村庄的结婚物品 |
| 6. 举办婚礼 | 大型事件,全NPC参加 |
| 7. 婚后生活 | 伴侣搬入农场,帮助农务 |

### 婚后福利

| 福利 | 说明 |
|------|------|
| 配偶帮忙农务 | 每天有一定概率帮忙浇水/喂动物 |
| 配偶提供料理 | 偶尔制作料理给主角 |
| 特殊对话 | 每日互动有特殊情侣对话 |
| 可以生孩子 | 婚后一段时间后可生育(本作有子女系统) |
| 结婚纪念日 | 每年有特殊事件 |

## 6. 送礼系统(Gift)

### 礼物品质与FP的关系

| 礼物星级 | FP倍率 |
|----------|--------|
| 0.5★(普通) | ×1.0 |
| 1.0★~2.0★ | ×1.1~1.3 |
| 3.0★~4.0★ | ×1.5~1.7 |
| 5.0★ | ×1.9(最大) |

### 礼物类型分级

| 分级 | FP增加(基础) | 说明 |
|------|-------------|------|
| 最爱(Loved) | +500 | NPC最喜欢的礼物,FP效率最高 |
| 喜欢(Like) | +300 | 大部分NPC对此反应良好 |
| 普通(Neutral) | +100 | 无特别反应 |
| 讨厌(Dislike) | -500 | NPC不喜欢的礼物 |
| 最讨厌(Hated) | -1,000 | 送这个会大幅降低好感 |

> **送礼时机**: 每日仅首次送礼有效。下雨天 NPC 不会在外活动,需进入建筑物寻找。

## 7. 日常社交策略

### 高效好感度提升路径

| 策略 | 每日耗时 | FP/日 | 备注 |
|------|----------|-------|------|
| 对话+喜欢礼物 | 5分钟/人 | 500 | 基础方案 |
| 对话+最爱礼物 | 5分钟/人 | 700 | 需准备对应物品 |
| 节日参赛+送礼 | 半天 | 2,000+ | 节日当日最佳 |
| 生日最爱礼物 | 1次 | 1,500 | 每年1次,务必把握 |
| 追逐事件 | 不定 | 1,000~3,000 | 触发特殊事件 |

### 批量社交路线

| 时间段 | 推荐拜访村庄 | 理由 |
|--------|-------------|------|
| 9:00~10:00 | 西部镇(Westown) | 商店刚开,NPC集中 |
| 10:00~11:00 | 露草里(Tsuyukusa) | 茶馆/种子店NPC在岗 |
| 11:00~12:00 | 露露可可村(Lulukoko) | 工坊/茶屋NPC在岗 |
| 夜间(18:00+) | 餐厅/室内 | NPC回到室内场所 |

## 8. 特殊事件

### 恋爱事件

每位结婚候选人有 **4~5个独立恋爱事件**,在好感度达到特定阶段时触发:

| 事件编号 | 触发条件 | 说明 |
|----------|----------|------|
| 第1事件 | 紫色花阶段 | 初次深入了解 |
| 第2事件 | 浅蓝花阶段 | 进一步互动 |
| 第3事件 | 深蓝花阶段 | 关键剧情 |
| 第4事件 | 浅绿花阶段 | 告白前奏 |
| 婚后事件 | 婚后 | 特殊婚后剧情 |

### 村庄事件

| 事件类型 | 触发条件 | 说明 |
|----------|----------|------|
| 村庄升级事件 | 村庄等级提升 | 全NPC庆祝 |
| 节日事件 | 节日当天 | 全社区参与 |
| 日常互动 | 随机 | NPC之间的对话 |
