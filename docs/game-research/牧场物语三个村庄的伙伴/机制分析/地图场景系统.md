# 地图场景系统
![三村地图](https://www.bokumono.com/series/tots/img/ss/ss05.jpg)

游戏版本: 牧场物语 三个村庄的珍贵朋友(Story of Seasons: Trio of Towns) (Nintendo 3DS, 2016年)

---

## 1. 整体地图结构

游戏地图由 **3个村庄** + **连接区域** + **农场** 构成,呈辐射状分布。

```
                   [山间区域(Mountain)]
                  /         |         \
                 /          |          \
         [西部镇]      [中央农场]     [露草里]
         (Westown)      (Farm)       (Tsuyukusa)
                 \          |          /
                  \         |         /
                   [森林区域(Forest)]
                        |
                   [露露可可村]
                   (Lulukoko)
```

### 区域解锁顺序

| 区域 | 解锁时间 | 解锁条件 |
|------|----------|----------|
| 农场(Farm) | 春1日 | 游戏开始 |
| 西部镇(Westown) | 春1日 | 游戏开始 |
| 露露可可村(Lulukoko) | **春15日** | 完成新手引导 |
| 森林区域(Forest) | 春15日 | 随露露可可村解锁 |
| 露草里(Tsuyukusa) | **夏5日** | 完成前期任务 |
| 山间区域(Mountain) | 夏5日 | 随露草里解锁 |

## 5. 连接区域

| 资源 | 说明 |
|------|------|

| 资源 | 说明 |
|------|------|

## 8. 快速移动系统

| 移动方式 | 解锁条件 | 说明 |
|----------|----------|------|
| 步行 | 初始可用 | 最慢,但可沿途采集资源 |
| **骑马(Horse)** | 达成 **爸爸的秘诀#3**(出货500,000G) | 移动速度大幅提升 |
| 区域间移动 | 到达区域边界自动切换场景 | 读取场景需加载 |

> 骑马是唯一加速移动手段,建议尽快达成Dad's Tip #3以解锁。
