## 1. 加工机总览(Maker Machines)

加工机是游戏后期经济的核心。将原材料转化为高附加值产品。

| 机器名称 | 英文 | 主要功能 | 制造优先级 |

| 蛋黄酱制造机 | Mayonnaise Maker | 蛋→蛋黄酱 | ★★★ |

## 3. 加工流程图解

### 谷物加工链

```
小麦(Wheat)
  → 磨粉机(Mill)
    → 小麦粉(Flour) [230G]
      → 厨房 → 面包(Bread) [350G]
      → 厨房 → 蛋糕(Cake) [500G+]

水稻(Rice)
  → 磨粉机(Mill)
    → 精米(Polished Rice) [280G]
      → 厨房 → 饭团(Rice Ball) [350G]
      → 厨房 → 寿司(Sushi) [600G+]
```

### 牛奶加工链

```
牛奶(Milk) [150G]
  ├→ 牛奶加工机(Cream Maker) → 奶油(Cream) [250G]
  ├→ 奶酪制造机(Cheese Maker) → 奶酪(Cheese) [450G]
  ├→ 酸奶制造机(Yogurt Maker) → 酸奶(Yogurt) [350G]
  └→ 黄油制造机(Butter Maker) → 黄油(Butter) [300G]
```

### 蛋加工链

```
鸡蛋(Egg) [120G]
  └→ 蛋黄酱制造机(Mayonnaise Maker) → 蛋黄酱(Mayonnaise) [350G]
    └→ 厨房 → 各类料理
```

### 纤维加工链

```
羊毛(Wool) [400G]
羊驼毛(Alpaca Wool) [600G]
美洲驼毛(Llama Wool) [700G]
兔毛(Rabbit Wool) [350G]
棉花(Cotton) [230G]
亚麻(Flax) [180G]
  └→ 纺织机(Loom)
    → 毛线(Yarn) [500G+]
    → 布料(Cloth) [800G+]
    → 茶布(Tea Cloth) [600G]
    → 蓝布(Blue Cloth) [700G]
    → 丝绸布(Silk Cloth) [1,000G+]
```

### 水果加工链

```
水果(Fruit)
  ├→ 葡萄酒制造机(Wine Maker) → 葡萄酒(Wine) [500G+]
  │   (不同水果产出不同品种的酒)
  └→ 果酱制造机(Jam Maker) → 果酱(Jam) [400G+]
      (不同水果产出不同果酱)
```

### 蔬菜加工链

```
蔬菜(Vegetable)
  └→ 腌制机(Pickle Maker) → 腌菜(Pickles) [350G+]
```

### 香草加工链

```
香草(Herb)
  └→ 香料制造机(Spice Maker) → 香料(Spice) [400G+]
      (用于高级料理配方)
```

## 8. 加工经济策略总结

| 阶段 | 重点加工机 | 核心产品 | 日利润参考 |
|------|-----------|----------|-----------|
| 初期(第1年夏前) | 无(优先攒钱造肥料机) | 直接出货 | - |
| 中期(第1年夏~冬) | 蛋黄酱+奶酪+纺织机 | 蛋黄酱、起司、布料 | 2,000~5,000G/日 |
| 中后期(第2年春~夏) | 葡萄酒+果酱 | 果酒、果酱 | 5,000~10,000G/日 |
| 后期(第2年秋+) | 全部满配 | S级品质+级产品 | 15,000~30,000G/日 |

**建议建造顺序**:

```
1. 肥料制造机 (最优先,提升作物品质)
2. 蛋黄酱制造机 (低成本,高回报)
3. 奶酪制造机 (牛奶增值)
4. 纺织机 (毛/棉增值)
5. 种子制造机 (锁定高星种子)
6. 葡萄酒制造机 (高利润,但建造贵)
7. 其他按需建造
```
