# 作物种植系统
![三村作物](https://www.bokumono.com/series/tots/img/ss/ss02.jpg)

游戏版本: 牧场物语 三个村庄的珍贵朋友(Story of Seasons: Trio of Towns) (Nintendo 3DS, 2016年)

---

## 1. 星级系统(Star Rank)

所有作物拥有 **0.5★~5.0★** 的星级评价。星级直接影响出货价格和礼物效果。

| 星级 | 价格倍率 | 备注 |
|------|----------|------|
| 0.5★ | ×1.00 | 初始种植基础值 |
| 1.0★ | ×1.10 | 每0.5★递增10% |
| 1.5★ | ×1.20 | |
| 2.0★ | ×1.30 | |
| 2.5★ | ×1.40 | |
| 3.0★ | ×1.50 | |
| 3.5★ | ×1.60 | |
| 4.0★ | ×1.70 | |
| 4.5★ | ×1.80 | |
| 5.0★ | ×1.90 | 最高星级 |

**星级提升方法**:
- 使用肥料提升作物品质
- 使用种子制造机(Seed Maker)将高品质作物加工成高星级种子
- 高星级种子种出高星级作物的概率更高

## 2. 肥料系统(Fertilizer)

肥料是提升作物星级的核心手段。每块耕地每天可使用 **1次** 肥料。

### 5项肥料属性

| 属性 | 日文/英文 | 效果 |
|------|-----------|------|
| 色泽(Color) | 色/Color | 影响外观评价 |
| 甜度(Sweetness) | 甘味/Sweetness | 影响口感评价 |
| 香味(Fragrance) | 香り/Fragrance | 影响香气评价 |
| 水分量(Juiciness) | 水分/Juiciness | 影响多汁度评价 |
| 大小(Size) | 大きさ/Size | 影响大小评价 |

### 肥料类型

| 肥料名称 | 效果 | 制作材料 |
|----------|------|----------|
| 色泽肥料(Color Fert) | 色泽+20 | 草×1 + 铁×1 |
| 甜度肥料(Sweet Fert) | 甜度+20 | 蜂蜜×1 + 铁×1 |
| 香味肥料(Fragrance Fert) | 香味+20 | 花×1 + 铁×1 |
| 水分肥料(Juice Fert) | 水分+20 | 水×1(水壶灌水) + 铁×1 |
| 大小肥料(Size Fert) | 大小+20 | 蘑菇×1 + 铁×1 |
| 全能肥料(All-in-One) | 全属性+10 | 由5种肥料混合(后期配方) |

**施肥策略**:
- **均衡型**: 轮流使用5种肥料,所有属性均衡成长
- **专注型**: 单属性集中施肥,快速提升某一项(用于节日参赛)
- 作物节日前需提前规划施肥方向

### 肥料制造机(Fertilizer Maker)

| 项目 | 数值 |
|------|------|
| 购买费用 | 20,000G |
| 材料需求 | 茶布×15 + 铁×10 + 蓝布×5 |
| 建造优先级 | **最高**(游戏中最关键机器) |
| 功能 | 将原料+铁加工成各种肥料 |

## 4. 黄金作物(Golden Crops)

黄金作物是普通作物的变异版本,具有更高的出货价格但更容易枯萎。

| 特性 | 说明 |
|------|------|
| 出现方式 | 种植时低概率出现 |
| 枯萎概率 | 比普通作物高约50% |
| 出货价格 | 普通作物的 **5~10倍** |
| 节日加分 | 作物节日中有额外加分 |
| 生长周期 | 与普通作物相同 |

**黄金作物价格示例**:

| 作物 | 普通价(0.5★) | 黄金价(0.5★) |
|------|--------------|---------------|
| 黄金芜菁(Golden Turnip) | 130G | 1,300G |
| 黄金马铃薯(Golden Potato) | 180G | 1,800G |
| 黄金番茄(Golden Tomato) | 230G | 2,300G |
| 黄金玉米(Golden Corn) | 280G | 2,800G |

**黄金作物出现概率提升方法**:
- 使用高品质肥料
- 提高耕地品质
- 使用高星级种子

## 8. 茶叶系统(Tea Leaves)

| 特性 | 说明 |
|------|------|
| 种植季节 | **全年** |
| 生长周期 | 种植后约20天开始收获 |
| 再生周期 | **每5天**可以收获一次 |
| 出货价(0.5★) | 250G |
| 加工用途 | 茶布(制作肥料制造机的必需材料) |

**茶叶的战略重要性**:
- 肥料制造机需要 **茶布×15**,而茶布由茶叶在纺织机加工而成
- 全年可收获的特性使其成为稳定收入来源
- 是游戏早期最推荐的种植作物之一

## 9. 种子制造机(Seed Maker)

| 项目 | 数值 |
|------|------|
| 购买费用 | 20,000G |
| 材料需求 | 木材×20 + 小木材×10 + 闪光石×10 + 黄玉×3 + 红宝石×2 |

**核心机制**: 放入高星级作物 → 产出高星级种子

| 放入作物星级 | 产出种子星级概率分布 |
|-------------|---------------------|
| 0.5★~1.0★ | 大部分0.5★,小概率1.0★ |
| 2.0★~3.0★ | 混合产出0.5★~2.0★ |
| 4.0★~5.0★ | 产出种子最低1.0★,高概率2.0★+ |

**战略价值**: 这是星级滚雪球的核心——好种子→好作物→更好的种子→更好的作物。

## 11. 作物节日(Crop Festivals)

每个季节有对应的作物节日:

| 节日 | 季节·日期 | 主题属性要求 |
|------|-----------|-------------|
| 春之蔬菜祭 | 春22日 | 色泽(Color)、大小(Size) |
| 夏之花祭 | 夏20日 | 香味(Fragrance)、色泽(Color) |
| 秋之收获祭 | 秋15日 | 甜度(Sweetness)、大小(Size) |
| 冬之星祭 | 冬25日 | 全部属性均衡 |

**节日评分规则**:

| 评分项 | 加分条件 |
|--------|----------|
| 高星级 | 星级越高得分越高 |
| 黄金/巨大作物 | 大幅加分 |
| 主题属性 | 对应属性高额外加分 |
| 作物稀有度 | 稀有作物加分 |

**获奖奖励**:
- 第1名: 奖杯 + 村庄点数 + 村民好感度
- 第2~3名: 村庄点数 + 村民好感度
- 参与奖: 少量村庄点数

## 12. 种植策略总结

| 目标 | 推荐策略 |
|------|----------|
| 初期赚钱 | 芜菁(快速周转)→草莓(再生) |
| 中期赚钱 | 玉米→玉米面包,水稻→料理 |
| 提升作物品质 | 尽早建造肥料制造机,每天施肥 |
| 节日获胜 | 提前1个月集中施肥对应属性 |
| 冬季收入 | 温室+全年作物+蘑菇+畜牧+加工 |
| 长远发展 | 种子制造机锁高星,果树逐年成长 |
