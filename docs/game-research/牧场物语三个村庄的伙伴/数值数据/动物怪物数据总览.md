### 2.2 品质影响因素

| 因素 | 影响程度 | 说明 |
|------|---------|------|
| 动物好感度 (Affection) | 高 | 好感度越高，产出高品质产品概率越大 |
| 动物心情 | 中 | 每天放牧+刷毛可维持好心情 |
| 饲料质量 | 中 | 使用饲料作物（如玉米）喂养可提升品质 |
| 农场圈加成 | 高 | 特定装饰品可每日提升副产品品质 |
| 动物健康度 | 中 | 生病时会产出低品质产品 |

## 六、动物养成策略

### 6.1 日常流程

1. **早晨**: 先说话 (+1 FP) → 刷毛 (+3 FP) → 放牧 (晴天) 或喂食 (雨雪天)
2. **中午**: 收集副产品
3. **傍晚**: 收牧 → 再次喂食
4. **睡前**: 给予零食 (+4 FP)

### 6.2 快速提升好感度

- 每天完成说话 + 刷毛 + 零食 = 每日最低 +8 FP
- 搭配放牧 + 喂食可达到每日 +13 FP
- 约 77 天可从 0 FP 达到 10,000 FP (最高常规好感度)

### 6.3 品质提升策略

1. **短期**: 利用农场圈的"鸡王雕像"等装饰每日提升副产品品质
2. **中期**: 提升动物好感度至信赖等级以上
3. **长期**: 培育高品质后代（父母品质影响子代初始品质）
