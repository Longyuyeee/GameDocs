## 六、作物品质与价格机制

### 6.1 品质星级与价格倍率

| 星级 | 价格倍率 | 备注 |
|------|---------|------|
| 0.5★ (基础) | 1.0x | 默认出货价 |
| 1.0★ | ~1.1x | — |
| 1.5★ | ~1.2x | — |
| 2.0★ | ~1.3x | — |
| 2.5★ | ~1.4x | — |
| 3.0★ | ~1.5x | — |
| 3.5★ | ~1.6x | — |
| 4.0★ | ~1.7x | — |
| 4.5★ | ~1.8x | — |
| 5.0★ | ~2.0x | 最高品质 |
| + (Plus) | ~2.5x | 金色作物专属标记 |
| S | 最大值 | 出货面板记录用，非直接出售 |

### 6.2 金作物 (Golden Crop) 说明

- 金作物需要对应作物的金种子（Golden Seed）
- 金种子通过 Seed Maker（种子加工机）有概率产出
- 金作物基础价格约为普通作物的 5 倍
- 生长周期更长，但单位土地收益更高
- 金作物收获时有 "+" 标记，品质上限更高

## 七、种植策略建议

### 春季推荐
- **早期快速收益**: 芜菁（5天成熟，适合开局）
- **中期稳定收益**: 土豆（7天成熟，性价比适中）
- **高收益花卉**: 康乃馨（日均利润高达58G/d）
- **多次收获**: 草莓（适合整个季节持续产出）

### 夏季推荐
- **顶级收益**: 金蜜瓜/金西瓜（日均利润可达500G/d以上）
- **多次收获**: 菠萝（单次收获价高，17+4d循环）
- **性价比**: 洋葱（低成本快速收益）

### 秋季推荐
- **快速收益**: 菠菜（5天成熟，日均利润可达45G/d）
- **高收益**: 金牛蒡（日均利润可达137G/d）
- **果树投资**: 柠檬树/橄榄树（长期稳定收益）

### 冬季推荐
- **高收益**: 蓝玫瑰（虽然910G种子贵，但出货价高）
- **花卉**: 雪花/龙胆（冬季少数高收益选择）

### 全年作物推荐
- **山葵**: 日均利润最高（可达149G/d），性价比之王
- **莲藕**: 12天成熟，收益稳定
- **辣椒/仙人掌**: 10天成熟，收益不错
