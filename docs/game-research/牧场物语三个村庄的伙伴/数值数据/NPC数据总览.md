# NPC 数据总览 — 牧场物语 三个村庄的珍贵朋友 (Story of Seasons: Trio of Towns)
![三村角色](https://www.bokumono.com/series/tots/img/character/chara_main.png)

- **游戏版本**: 牧场物语 三个村庄的珍贵朋友 (Story of Seasons: Trio of Towns) (Nintendo 3DS, 2016年)
- **好感度系统**: 每个 NPC 拥有隐藏的好感度点数 (FP, Friend Points)，通过对话、送礼、完成任务等方式提升。
- **婚恋对象**: 标注了 [结婚候补] 的 NPC 为可恋爱结婚对象。

---

## 一、好感度系统说明

### 1.1 好感度等级 (花朵颜色)

| 花朵颜色 | 好感度范围(FP) | 说明 |
|---------|--------------|------|
| 黑色 (无花) | 0~999 | 初始阶段 |
| 紫色花 | 1,000~2,999 | — |
| 蓝色花 | 3,000~5,999 | — |
| 绿色花 | 6,000~9,999 | — |
| 黄色花 | 10,000~19,999 | — |
| 橙色花 | 20,000~39,999 | — |
| 红色花 | 40,000~59,999 | 最高友好度 |
| 彩虹花 (婚姻候补) | 60,000+ | 恋爱状态 |

### 1.2 送礼效果

| 送礼类型 | FP变化 | 备注 |
|---------|-------|------|
| 最喜欢的礼物 (最爱) | +500 FP | 大幅提升 |
| 喜欢的礼物 | +300 FP | 中幅提升 |
| 普通礼物 | +100 FP | 小幅提升 |
| 不喜欢的礼物 | -500 FP | 中幅下降 |
| 讨厌的礼物 | -1,000 FP | 大幅下降 |

### 1.3 好感度提升方式

| 行为 | FP增加量 | 频率限制 |
|------|---------|---------|
| 每天对话 | +100 FP | 1次/天 |
| 送礼 (喜欢) | +300 FP | 1次/天 |
| 送礼 (最爱) | +500 FP | 1次/天 |
| 完成村民请求 | +300~1,000 FP | 依任务而定 |
| 参加节日活动 | +100~500 FP | 依结果而定 |

## 六、恋爱与结婚

### 6.1 结婚条件

| 条件 | 说明 |
|------|------|
| 好感度 | 达到红色花朵 (40,000 FP+) 或以上 |
| 告白 | 赠送"蓝色羽毛" |
| 住房 | 房屋至少升级一次（有厨房和双人床） |
| 事件 | 触发该角色的全部恋爱事件 |
| 时间 | 至少经过一个游戏年 |

### 6.2 结婚后可恋爱对象

| 村庄 | 男性候补 | 女性候补 |
|------|---------|---------|
| Westown | 韦恩 (Wayne)、福特 (Ford) | 莉赛特 (Lisette) |
| Tsuyukusa | 日向 (Hinata)、柚月 (Yuzuki) | 霞 (Kasumi) |
| Lulukoko | 路德斯 (Ludus) | 伊露卡 (Iluka)、希露卡 (Siluka) |

共 **6位男性候补** 和 **3位女性候补** 可选（玩家可选择男/女主角开始游戏，决定了可追求的对象）。
