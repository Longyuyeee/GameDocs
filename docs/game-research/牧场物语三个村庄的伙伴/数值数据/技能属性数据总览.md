# 技能属性数据总览 — 牧场物语 三个村庄的珍贵朋友 (Story of Seasons: Trio of Towns)

- **游戏版本**: 牧场物语 三个村庄的珍贵朋友 (Story of Seasons: Trio of Towns) (Nintendo 3DS, 2016年)
- **无离散技能系统**: 本作不同于《星露谷物语》或《符文工房》系列，**不存在独立的技能等级系统**（如耕种技能Lv.1→Lv.10）。玩家能力的提升主要通过**村镇等级 (Village Rank)** 和**工具升级**间接实现。

---

## 一、村镇等级 —— 核心进度系统

### 1.1 系统概述

本作中，真正的"技能/进度"系统是**三个村庄的村镇等级 (Village Rank)**。玩家需要通过多种方式提升每个村庄的等级，解锁新功能、新区域和新物品。

### 1.2 等级序列

| 等级 | 英文 | 说明 |
|------|------|------|
| E | Rank E | 初始等级 |
| D | Rank D | — |
| C | Rank C | — |
| B | Rank B | — |
| A | Rank A | — |
| S | Rank S | 最高等级 |

### 1.3 提升方式

| 方式 | 提升量 | 说明 |
|------|--------|------|
| 对话 (Talk) | 微量 | 每天与村民对话 |
| 送礼 (Gift) | 中量 | 送出村民喜欢的礼物 |
| 参加节日 (Festival) | 中~大量 | 参与并取得好成绩 |
| 兼职工作 (Part-time Job) | 中量 | 在村庄的店铺打工 |
| 出货 (Shipping) | 微量~中量 | 向该村庄出货产品 |
| 完成请求 (Request) | 大量 | 完成村民的委托任务 |

### 1.4 各等级解锁内容

#### Westown (西镇)

| 等级 | 解锁内容 |
|------|---------|
| E→D | 基础工具升级解锁、种子店扩充 |
| D→C | 铁匠铺升级、新服装解锁 |
| C→B | 高级工具升级、牧场扩建选项 |
| B→A | 稀有种子解锁、特殊饰品制作 |
| A→S | 最终工具升级、隐藏剧情 |

#### Tsuyukusa (露草/月草)

| 等级 | 解锁内容 |
|------|---------|
| E→D | 基础食材店解锁、菜谱学习 |
| D→C | 新烹饪配方、装饰品解锁 |
| C→B | 高级食材解锁、农具改造 |
| B→A | 稀有花卉种子、特殊装饰 |
| A→S | 最终食谱、隐藏区域 |

#### Lulukoko (露露可可)

| 等级 | 解锁内容 |
|------|---------|
| E→D | 基础加工机解锁、果树苗 |
| D→C | 新加工配方、钓鱼扩展 |
| C→B | 高级加工机、新作物种子 |
| B→A | 稀有果树、特殊加工配方 |
| A→S | 最终加工机、隐藏剧情 |

## 二、体力/精力系统 (Stamina)

### 2.1 体力条

- 体力以 **心形图标** 显示在屏幕左上角
- 初始体力上限：**100 点**（5颗心，每颗心=20点）
- 体力可通过以下方式增加

### 2.2 体力上限提升

| 方式 | 增加量 | 条件 |
|------|--------|------|
| 完成"老爸的牧场建议书"Tip 1 | +20 | 早期任务 |
| 完成"老爸的牧场建议书"Tip 2 | +20 | 中期任务 |
| 完成"老爸的牧场建议书"Tip 3 | +30 | 后期任务 |
| 食用特定料理 | 临时提升 | — |
| 力量护符装备 | 减少消耗 | 非直接增加 |

### 2.3 体力消耗

| 行动 | 体力消耗 | 备注 |
|------|---------|------|
| 耕地 (锄头1级) | -2 | 蓄力减少消耗 |
| 播种 | -1 | — |
| 浇水 (水壶1级) | -2 | 蓄力可一次浇多格 |
| 收获作物 | -1 | — |
| 砍树 (1次) | -3 | 视树木大小 |
| 敲矿 (1次) | -3 | — |
| 钓鱼 (1次) | -1 | — |
| 刷毛 | -1 | — |
| 挤奶 | -1 | — |
| 使用镰刀 | -2 | 蓄力可一次割多格 |

### 2.4 体力恢复方式

| 方式 | 恢复量 | 备注 |
|------|--------|------|
| 自然恢复 (睡觉) | 全满 | 次日早晨 |
| 泡温泉 | 持续恢复 | 特定区域 |
| 吃东西 | 视道具而定 | — |
| 回复药剂 | +50/+100 | 购买或制作 |
| 料理 | +30~+200 | 视料理而定 |

## 三、耕作熟练度 (Farming Mastery)

### 3.1 基础机制

本作不存在可查看的"耕作技能等级数字"，但**耕作行为的重复次数**会影响以下方面：

| 行为 | 熟练效果 | 说明 |
|------|---------|------|
| 耕地 | 蓄力范围随工具等级提升 | 非直接技能等级 |
| 浇水 | 蓄力范围随工具等级提升 | — |
| 收获 | 高好感+高品质动物产出更好产品 | — |

### 3.2 重复操作的隐性增益

虽然不是数值化的技能系统，但以下行为重复后会感受到效率提升：

- **工具蓄力速度**: 高等级工具有更大的作用范围
- **移动速度提升**: 跑鞋+骑行马来提升
- **作物品质提升**: 通过土壤改良和肥料长期培育

## 四、角色属性

### 4.1 进度面板 (Player Stats)

玩家可以在菜单中查看以下信息：

| 项目 | 说明 |
|------|------|
| 姓名 | 创建角色时设定 |
| 性别 | 男/女可选 |
| 生日 | 创建角色时设定 |
| 资金 | 当前持有金钱 |
| 游戏天数 | 从第1天开始 |
| 总游戏时间 | 累计游玩时间 |
| 出货总额 | 累计出货金额 |
| 最大出货 | 单次最高出货金额 |

### 4.2 隐藏属性

| 属性 | 影响 | 提升方式 |
|------|------|---------|
| 运气 (Luck) | 高品质作物/产品出现概率 | 饰品、料理 |
| 魅力 (Charm) | 对话时的好感度增加量 | 服装、饰品 |
