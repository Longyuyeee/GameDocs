## 2. 矿洞系统

### 2.1 泉矿洞(Spring Mine)

| 参数 | 详情 |
|------|------|
| 位置 | 母亲山温泉旁边 |
| 开放时间 | **全年** |
| 总层数 | **255层**(Floors) |
| 下楼方式 | 用锤子敲石头→找楼梯(隐藏在地下) |
| 特殊物品 | 女神之玉(9个,特定层收集) |
| 矿石分布 | 铜1F+/银40F+/金80F+/秘银120F+/奥利哈钢170F+ |

### 2.2 湖矿洞(Lake Mine)

| 参数 | 详情 |
|------|------|
| 位置 | 母亲山湖中(冬季结冰后可行走) |
| 开放时间 | **仅冬季** |
| 总层数 | **255层** |
| 特殊物品 | **诅咒工具**(29F-79F),河童之玉(9个收集) |

### 2.3 矿洞机制
- 每层随机布局,楼梯位置随机
- 敲石头消耗体力,下楼→更深层→更稀有矿
- 255F往返需要大量体力和食物储备
- 冬天="采矿季"(无农活,全天挖矿)

## 4. 快速旅行

矿石镇地图较小(约30秒可横穿),无快速旅行系统。骑马是唯一加速方式。

## 5. 设计特点

1. **小镇尺度**: 地图紧凑,步行数秒即可到达任何地点——"镇上每个人都能路过打个招呼"
2. **255层矿洞**: GBA时代最深矿洞之一,设计了"越深越稀有"的驱动力
3. **冬季限定湖矿洞**: 用季节锁限制高级工具→制造"等待冬天"的期待感
4. **商店休息日**: 各商店不同休息日→需记住并规划购买日,增加时间管理深度
5. **无传送系统**: 纯步行/骑马,保持了小镇的真实感和节奏感
