# 战斗/探索系统 - 牧场物语 重聚矿石镇
![矿石镇矿洞](https://www.bokumono.com/series/fomt/img/ss/screenshot08.jpg)

> 矿石镇是无战斗系统的纯经营游戏。本章标注「无此系统」,但记录其探索与收集玩法。

## 1. 战斗系统

**无战斗系统(无此系统)**

矿石镇是纯粹的牧场经营游戏,无怪物、无战斗、无HP战斗损耗、无武器系统。所有"冒险"元素围绕矿洞探索展开。

## 2. 矿洞探索(替代战斗的"冒险"系统)

### 2.1 矿洞=体力管理挑战
- 不是打怪,而是**管理体力**
- 每敲一次石头消耗2-3体力
- 深层探索需要携带大量食物/体力药
- 255层的泉矿洞=体力+食物储备+运气的综合考验

### 2.2 矿洞收集品

| 收集品 | 数量 | 位置 | 完成奖励 |
|------|:--:|------|------|
| 女神之玉(Goddess Jewels) | 9个 | 泉矿洞特定层 | 女神祝福(体力快速恢复) |
| 河童之玉(Kappa Jewels) | 9个 | 湖矿洞(冬季) | 河童祝福 |
| 诅咒工具(Cursed Tools) | 6件 | 湖矿洞29F-79F | 升级为祝福工具→贤者工具 |
| 贤者之石(Mythic Stone) | — | 60F+(全祝福工具后) | 制作贤者工具 |

## 3. 钓鱼系统

| 维度 | 详情 |
|------|------|
| 钓竿等级 | 铁→铜→银→金→秘银→祝福→贤者 |
| 钓点类型 | 海(海滩)/河(城镇河流)/湖(母亲山) |
| 鱼种 | 约40+种(含传说级鱼) |
| 钓竿作用 | 升级后能钓到更大/更稀有的鱼 |
| 传说鱼 | 特定季节+特定天气+特定钓点 |

## 5. 设计特点

1. **纯经营**: 矿石镇代表"无战斗"牧场游戏的最纯粹形态
2. **矿洞=体力考验**: 用体力管理和食物储备替代战斗HP——本质是资源管理,不是战斗
3. **钓鱼和采集=辅助系统**: 不占主要游戏比重,但提供食谱材料/送礼物品
4. **与符文工房系列的对比**: 符文工房在矿石镇基础上加ARPG战斗→两种不同设计方向的分支
