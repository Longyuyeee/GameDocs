# 作物种植系统 - 牧场物语 重聚矿石镇

> 数据来源: ranchstory.farm, andaplus.com, irvangen.com, stratswiki.com

## 1. 种植机制

- **田地**: 初始1块→可扩建至**3块田地**(木匠屋: 首次2000G,二次5000G)
- **浇水**: 水壶手动浇水,升级后蓄力覆盖更大范围
  - 铜(Copper): 1×3格
  - 银(Silver): 2×3格
  - 金(Gold): 3×3格
  - 秘银(Mithril): 3×5格
- **季节**: 4季×30天,冬季无法户外种植(无温室)
- **跨季**: 当季最后一天未收获的作物换季后枯萎
- **无洒水器/无加工机械**: 纯手动农耕,作物→直接出货

## 2. 品质系统

- ★1 ~ ★5 五星品质
- ★5售价约为★1的**1.4倍**
- 品质提升: 在种子上撒肥料(杂货店购买)
- 简单模式(Simple Mode): 所有售价约+20%

- 果树占据3×3空间,不可移动

## 7. 品质升级与肥料

- **肥料(Fertilizer)**: 杂货店购买,撒在已播种的作物上
- 每季初规划: 先买种子→播种→施肥(每日持续)
- 高品质作物→更高售价→更多收益
- 无品质加工链(不像星露谷: 高品质作物→高品质酒)

## 8. 设计特点

1. **纯手动农耕**: 无水壶之外的任何自动化,每个动作都需要体力
2. **解锁作物驱动目标**: 出货100个基础作物→解锁高级作物→激励种植多样性
3. **霍安商店**: 高价特殊种子(500-1000G),需要更高投入→更高回报
4. **变异花朵**: 蓝魔→真魔(5%概率),引入随机性
5. **冬季无种植**: 强迫玩家在冬天转投采矿+畜牧,制造季节节奏转换
6. **果树=长期投资**: 高前期成本+无需维护+永久回报
