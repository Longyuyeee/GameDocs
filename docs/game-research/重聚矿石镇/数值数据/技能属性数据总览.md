# 技能属性数据总览 - 重聚矿石镇

> 数据来源: ranchstory.farm, irvangen.com

## 技能系统说明

矿石镇**没有明确的技能等级系统**(不像星露谷的5技能/符文工房的20+技能树)。角色成长通过以下方式体现:

## 工具使用经验(EXP%)

| 工具 | EXP积累方式 | 满100%效果 |
|------|-----------|----------|
| 锄头(Hoe) | 耕地 | 可跳过中间等级直接升级 |
| 水壶(Watering Can) | 浇水 | 同上 |
| 斧头(Axe) | 砍树/树桩 | 同上 |
| 锤子(Hammer) | 砸石头/矿石 | 同上 |
| 镰刀(Sickle) | 割草/收割牧草 | 同上 |
| 钓竿(Fishing Rod) | 钓鱼 | 同上 |

- EXP%满100%后可越过铜/银/金,直接升级到秘银(需铁匠铺交付材料+金币,等待4天)

## 料理学习进度

- 全120种菜谱可学习
- 食谱来源: 电视(周二)、NPC好感度(约15份)、安娜甜点课(5份)、Lou(10份)、灵感(使用食材次数达标解锁)
- 学会全部120菜谱=解锁美食家(Bon Vivant)结婚候补

## 体力/HP系统

| 属性 | 初始值 | 提升方式 | 上限 |
|------|:---:|------|:--:|
| 体力(Stamina/HP) | ~100 | 力量果实×10(每颗+10HP) | ~200 |
| 疲劳(Fatigue) | 0 | 持续工作不休息→累积 | 脸变蓝=消耗×2 |

### 力量果实获取途径

| # | 来源 | 条件 |
|:--:|------|------|
| 1 | 赛马节 | 900徽章兑换(春/秋各1次) |
| 2 | 海滩日 | 飞盘比赛赢 |
| 3 | 矿洞 | 泉矿洞特定层挖出 |
| 4 | 钓鱼 | 钓到特定鱼王 |
| 5 | 女神祝福 | 收集9个女神之玉 |
| 6-10 | 其他途径 | 送礼/好感度/事件等 |

## 牧场评价(Farm Degree)

矿石镇有一个隐藏的"牧场度"系统，影响部分NPC对话和事件解锁:

| 评价因素 | 影响 |
|---------|------|
| 出货总量 | ★★★★★ |
| 动物好感度总和 | ★★★ |
| 作物出货种类 | ★★★ |
| 鱼/矿/料理出货 | ★★ |
| 房屋扩建程度 | ★★ |

> 该评价不直接显示在游戏中,仅通过爷爷评价(第3年)间接反映。
