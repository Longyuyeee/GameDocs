# 重制版改动 - 牧场物语 重聚矿石镇
![重聚矿石镇封面](https://www.bokumono.com/series/fomt/img/top/mainvisual.jpg)

> 原版: 牧场物语 矿石镇的伙伴们 (GBA, 2003年)
> 重制: 牧场物语 重聚矿石镇(SoS: Friends of Mineral Town) (Switch 2019年/PC 2020年)

## 1. 画面与性能

| 改动 | 原版(GBA) | 重制版(Switch/PC) |
|------|---------|----------------|
| 画面 | 2D像素(俯瞰视角) | 全3D(自由视角,可缩放) |
| 分辨率 | 240×160 | HD(720p-1080p+) |
| 帧率 | 60fps(2D) | 30-60fps(3D,性能模式) |
| 角色模型 | 像素小人 | 3D建模+表情动画 |
| 支持平台 | GBA | Switch + Steam PC |

## 2. 新增核心功能

| 功能 | 说明 |
|------|------|
| **同性婚姻** | 原版男女候补各6限定;重制版玩家可选任何性别追求任何候补 |
| **新增2名候补** | Brandon(布兰登,画家,秋7生日),Jennifer(詹妮弗,自然主义者,冬2生日) |
| **新增动物** | 羊驼(Alpaca),安哥拉兔(Angora Rabbit),咖啡色牛等更多变种 |
| **宠物种类扩展** | 原版仅猫/狗;重制版新增多种猫狗品种+颜色选择 |
| **新作物** | 红豆(Adzuki Beans),辣椒(Chili Pepper),日甜花(Sunsweet)等 |
| **一键打招呼** | 进入有NPC的区域自动触发对话(省去逐一对话的时间) |
| **自动存档** | 睡觉时自动保存;另有多档位手动存档 |
| **背包可视化** | 道具栏图标和分类更清晰 |
| **便捷换工具** | 快速切换工具栏,不占背包格 |

## 3. 移除/简化的内容

| 原版内容 | 重制版改动 |
|---------|---------|
| **恋爱竞争对手系统** | **移除**。原版12候补之间有竞争对手关系(如Gray和Marie互相竞争),玩家婚后对手自动结婚。重制版全部取消,候补关系简化 |
| **NDS联机功能** | 移除(原版矿石镇伙伴们for girl可联机) |
| **原版某些隐藏事件** | 部分简化或移除 |
| **恋爱对手婚后剧情** | 无了(因为对手系统被砍) |

## 4. 平衡性调整

| 调整 | 详情 |
|------|------|
| **简单模式(Simple Mode)** | 新增加,作物/产物售价约+20% |
| **赛马节徽章获取** | 调整,更容易获取大量徽章 |
| **体力消耗** | 略降低,游戏节奏更舒适 |
| **动物好感度增长** | 速度略快于原版 |
| **部分种子价格** | 有微调 |

## 5. 设计分析

### 5.1 好的改动
- **同性婚姻**: 包容性设计,扩大玩家群体
- **自动存档**: 减少因忘记存档导致的损失(QoL关键)
- **一键打招呼**: 解决每天逐一对话NPC的繁琐操作
- **取消对手系统**: 原版的抢婚机制在现代玩家看来可能引起焦虑,移除后社交更加自由

### 5.2 保留的好设计
- **纯经营定位**: 没有在重制中加入战斗系统,保留了矿石镇的"纯粹"
- **255层矿洞**: 完整的诅咒→祝福→贤者升级路径完全保留
- **商店休息日系统**: 保留各商店不同的闭店日(增加时间管理深度)

### 5.3 争议改动
- **3D画面**: 部分老玩家认为2D像素更有"牧场韵味"
- **角色3D建模**: 部分角色画风争议(与原版2D立绘差异大)

## 6. 与同时期重制的对比(橄榄镇)
