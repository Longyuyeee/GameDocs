# 制造机(Makers)系统的设计得失 - 橄榄镇

> 游戏版本: Story of Seasons: Pioneers of Olive Town (2021年)

## 一、概述

橄榄镇的**制造机(Makers)**系统是系列历史上最大胆的创新——几乎所有物品都需要通过专用机器加工才能获得(木材/金属锭/纱线/布料/奶酪等)。这个系统在发售后成为游戏评价最两极分化的要素。

## 二、制造机列表与依赖关系

```
原木 → 木材制造机 → 木材
矿石 → 锭制造机 → 金属锭
羊毛/羊驼毛 → 纱线制造机 → 纱线 → 布料制造机 → 布料
牛奶 → 奶酪制造机/黄油制造机 → 奶酪/黄油
蛋 → 蛋黄酱制造机 → 蛋黄酱
水果 → 果酱制造机 → 果酱
作物 → 种子制造机 → 种子(品质可能提升)
草/纤维 → 线制造机 → 线
```

## 三、设计优势

1. **前所未有的加工深度**: 原木→木材/矿石→锭/羊毛→纱线→布料——比前作多了一层"中间加工品"
2. **加工增值**: 原料→加工品的价值提升明确——驱动制造机使用
3. **种子品质递进**: 种子制造机=唯一的种子品质提升途径——赋予其核心策略价值
4. **视觉满足感**: 摆放大量机器在农场上→制造品的产出动画→工业化的视觉反馈

## 四、设计缺陷(发售后被大量批评)

1. **空间占用过大**: 每台机器占用1格，数十台机器→农场被机器塞满而非作物/动物——违背"农场"的视觉预期
2. **每台机器单独操作**: 没有批量投料/收取→每天排队操作数十台机器→日常重复劳动极重
3. **加工时间过长**: v1.0版本的加工时间设置过慢→大量等待
4. **"工厂"非"农场"**: 核心体验从"种田养动物"变为"管理机器流水线"——偏离了系列核心
5. **机器无自动化**: 没有洒水器式的自动化——必须手动→且必须大量

## 五、社区反馈与修复

| 问题 | 社区反应 | 1.0.5补丁修复 |
|------|------|:--:|
| 加工时间过长 | 最严重的抱怨 | ✓ 显著缩短 |
| 产出量过低 | 强烈不满 | ✓ 提高产量 |
| 机器太多占空间 | 持续抱怨 | 部分缓解 |
| 操作繁琐 | 核心问题 | 未完全解决 |

## 六、设计教训

1. **加工深度≠更复杂的操作**: 可以有复杂加工链(木材+矿石+纱线→高级物品)，但不应每台机器手动操作
2. **工业化需自动化配套**: 如果引入大量机器→必须同时引入批量操作/自动化途径
3. **核心体验一致性**: 牧场的核心是"种田养动物"——机器是工具而非主角
4. **空间管理≠乐趣**: 让玩家为机器留空间而非作物→破坏了"打造农场"的核心幻想
5. **1.0.5修复太迟**: 发售后3个月才出补丁→首发评价已经形成→对销量产生负面影响

## 七、与其他牧场游戏的加工系统对比

| 游戏 | 加工方式 | 操作量 |
|------|------|:--:|
| 星露谷 | 酿造桶/罐头瓶/奶酪机(放置即运作) | 低(批量收取) |
| 矿石镇 | 蛋黄酱/奶酪/毛线机(每台独立) | 中(3台) |
| **橄榄镇** | **木材/锭/纱线/布料/奶酪/果酱/调味料等10+种机器** | **极高(数十台×每台手动)** |
| 珊瑚岛 | 酿造桶/罐头瓶/纺织机等(放置即运作) | 低-中 |
