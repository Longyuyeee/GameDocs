## 3. 制造机(Makers)系统——核心机制

### 3.1 制造机是橄榄镇的标志性特征

橄榄镇引入了**数量巨大的制造机**用于加工几乎所有物品:

| 制造机 | 原料 | 产出 | 加工时间 |
|------|------|------|:--:|
| 木材制造机(Lumber Maker) | 原木 | 木材 | 数小时 |
| 线制造机(Thread Maker) | 草/纤维 | 线 | 短 |
| 锭制造机(Ingot Maker) | 矿石 | 金属锭 | 数小时 |
| 种子制造机(Seed Maker) | 作物 | 种子 | 短 |
| 纱线制造机(Yarn Maker) | 羊毛 | 纱线 | 中 |
| 布料制造机(Cloth Maker) | 线 | 布料 | 中 |
| 奶酪制造机(Cheese Maker) | 牛奶 | 奶酪 | 中 |
| 黄油制造机(Butter Maker) | 牛奶 | 黄油 | 中 |
| 蛋黄酱制造机(Mayo Maker) | 蛋 | 蛋黄酱 | 短 |
| 果酱制造机(Jam Maker) | 水果 | 果酱 | 中 |
| 调味料制造机(Seasoning Maker) | 香草/香料 | 调味料 | 中 |

### 3.2 制造机的争议

- **大量制造机占用空间**: 玩家需要数十台机器运作→农场被机器占满
- **时间管理负担**: 每台机器需要分别投料+收取→日常重复操作量大
- **1.0.5更新优化**: 后续补丁提高了产出量/缩短了时间

## 5. 精灵(Sprite)系统

- 精灵可被派往采集资源(木材/矿石/草等)
- 减少日常重复劳动
- 类似矿石镇的"收获精灵"系统

## 6. 设计特点

1. **制造机=双刃剑**: 深度加工链(最大优势) vs 空间占用+管理负担(最大抱怨)
2. **精灵=缓解重复劳动的尝试**: 替代星露谷风格的"自动化"，但保留玩家参与
3. **大量加工选项**: 几乎所有物品都有加工版本——经济深度在系列中最复杂
4. **1.0.5补丁修复**: 初始版本的制造机效率过低→大量差评→后续更新缓解——值得分析的教训
