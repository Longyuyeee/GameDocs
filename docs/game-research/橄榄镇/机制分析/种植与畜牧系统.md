## 3. 作物品质

- 品质等级: 0-10星(通过种子制造机+肥料逐步提升)
- 种子制造机: 作物→种子(可能产出更高级品质种子)
- 肥料: 加速生长/提高品质

## 4. 设计特点

1. **大量复收作物**: 大多数高价值作物是复收型→减少重复翻耕的负担
2. **四季种植**: 冬季有专属作物(不像BTN冬季完全停摆)
3. **品质递进系统**: 种子制造机→逐步提升种子品质→长线品质追求
4. **更多动物种类**: 羊驼/兔等——系列动物种类最丰富的作品之一
