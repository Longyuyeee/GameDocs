## 二、社交机制

### 2.1 好感度阶段

| 心色 | 解锁 |
|:--:|------|
| 0-2♥ | 基础对话 |
| 3♥ | 触发初期事件 |
| 5♥ | **可表白(赠送表白信物)** |
| 6-8♥ | 恋爱事件 |
| 9♥ | 亲密 |
| 10♥ | **可求婚(需订婚戒指+房屋)** |

### 2.2 社交方式

| 方式 | 效果 |
|------|:--:|
| 每日对话 | +基础好感 |
| 送礼 | +好感(最爱>喜欢>中立>讨厌) |
| 节日互动 | +额外好感 |
| 拍照(部分NPC) | +好感 |
| 完成任务 | +好感 |

## 三、求婚条件

| 条件 | 说明 |
|------|------|
| 10♥好感度 | 最高 |
| 全部心色事件 | 所有事件触发 |
| 房屋扩建(含双人床) | 木匠铺 |
| 订婚戒指 | 购买 |
| 婚礼 | 3天后举行 |
