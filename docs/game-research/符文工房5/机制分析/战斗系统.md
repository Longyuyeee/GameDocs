# 战斗系统 - 符文工房5(Rune Factory 5)

> 游戏版本: Rune Factory 5 (Switch/PC, 2021-2022年)
> 数据来源: therunefactory.fandom.com, Game8

## 1. 概述

RF5的战斗系统从RF4的2D俯视改为**全3D动作战斗**。武器类型、技能系统、符文能力(符文/符文技能)延续RF4框架，但操作和空间感发生了根本变化。

## 2. 八种武器类型

| 武器 | 特点 | 3D变化 |
|------|------|------|
| 短剑(Short Sword) | 快速连击 | 3D中锁定+闪避更重要 |
| 长剑(Long Sword) | 均衡 | — |
| 双剑(Dual Blades) | 极速/低伤 | 3D中的追击性能 |
| 拳套(Fists) | 近身/投技 | — |
| 锤/斧(Hammer/Axe) | 慢速/高伤 | — |
| 长枪(Spear) | 中距/穿刺 | 3D中的Z轴距离优势 |
| 法杖(Staff) | 远程魔法 | 3D中的魔法弹道 |
| 农具(Farm Tools) | 可战斗 | 同RF4 |

## 3. 符文能力(Rune Abilities)

- 继承RF4的符文(Rune)系统
- 各类法术/特殊技——消耗RP
- 通过技能等级提升解锁新符文

## 4. 3D战斗的变化

| 维度 | RF4(2D) | RF5(3D) |
|------|:--:|:--:|
| 移动 | 上下左右4方向 | 自由3D移动 |
| 锁定 | 手动方向 | 可锁定敌人 |
| 闪避 | 移动避让 | **翻滚/侧闪**(3D动作化) |
| 视野 | 固定俯视 | 自由旋转/缩放 |
| 空间感 | 平面 | 高低差(部分地牢) |

## 5. 怪物系统

- 可捕捉/驯服怪物→怪物伙伴
- 怪物可帮农活(RF系列传统)
- 怪物可骑乘(大型怪物)
- 通缉怪物(Wanted Monsters)→SEED任务

## 6. SEED徽章

| 等级 | 解锁 |
|:--:|------|
| 初级 | 基础任务 |
| 中级 | 更多指令/权限 |
| 高级 | 特殊任务/奖励 |
| 最高 | 全部功能 |

## 7. 设计特点(与RF4对比)

1. **全3D动作化**: 从2D俯视→3D动作——战斗操作感变化巨大
2. **锁定+翻滚=更动作化**: 类似3D塞尔达的战斗风格
3. **八种武器平衡**: RF4的武器平衡在3D中需要重新调整——长枪的Z轴优势在3D中更明显
4. **Switch性能问题**: 3D战斗的帧率不稳定→影响战斗体验——RF5的最大技术短板
