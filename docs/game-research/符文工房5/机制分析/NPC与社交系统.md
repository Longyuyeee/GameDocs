## 二、社交机制

### 2.1 好感度阶段

| 等级 | 事件 |
|:--:|------|
| 0-2♥ | 基础对话/送礼 |
| 3♥+ | 触发前3个恋爱事件(粉色心图标) |
| **7♥** | **可表白(Confession)**——成功率30%(7♥)/60%(8♥)/100%(9♥+) |
| 交往中 | 约会×3(含节日约会) |
| **10♥** | **可求婚**——需主线通关+订婚戒指(锻造Lv20)+双人床 |

### 2.2 表白机制

- "I love you!" BTW菜单选项
- 每日限1次表白尝试
- **多角恋惩罚**: 已有1个伴侣→成功率50%/5个伴侣→10%
- 失败→等明天重试

## 五、设计特点(与RF4对比)

1. **12位候补(6男6女)**: RF4是6位→翻倍
2. **同性婚**: 系列首次——欧美版加入
3. **表白成功率递减**: 多角恋惩罚→社交策略深度
4. **求婚需主线通关**: 相比RF4的"3章主线后日谈"→RF5需要主线通才能结婚——进度门控更高
5. **3D心色事件**: 全3D过场动画——RF4的2D立绘切换→3D表演
