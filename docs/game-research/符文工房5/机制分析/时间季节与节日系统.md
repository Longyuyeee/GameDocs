# 时间季节与节日系统 - 符文工房5(Rune Factory 5)

> 游戏版本: Rune Factory 5 (Switch/PC, 2021-2022年)
> 数据来源: therunefactory.fandom.com, Game8

## 1. 时间结构

| 参数 | 设定 |
|------|------|
| 每天 | 6AM - 5:59AM |
| 每季节 | 30天 |
| 每年 | 4季×30天=120天 |
| 时间流速 | 标准(RF系列一致) |

## 2. 季节

| 季节 | 特色作物 | 节日 |
|:--:|------|------|
| 春 | 草莓/芜菁/土豆/卷心菜 | 春之祭/钓鱼赛 |
| 夏 | 菠萝/番茄/玉米/南瓜 | 夏之祭/游泳赛 |
| 秋 | 茄子/红薯/胡萝卜 | 秋之祭/料理赛 |
| 冬 | 冬季作物 | 冬之祭/星夜节 |

## 4. SEED节日指令

- 作为SEED游骑兵→可通过SEED指令解锁额外节日
- SEED徽章等级越高→可举办越多节日

## 5. 体力(RP)系统

| 要素 | 说明 |
|------|------|
| RP(Rune Points) | 使用工具/技能消耗 |
| RP上限 | 通过技能等级提升永久增加 |
| RP恢复 | 温泉/食物/睡觉 |
| RP=0 | 消耗HP继续使用(危险!) |

## 6. 设计特点

1. **30天/季=RF传统**: RF系列一致30天/季→比星露谷28天长2天
2. **SEED指令=节日自定义**: 通过SEED系统解锁额外节日→更高自由度
3. **RP系统保留**: RF系列的RP(而非Stamina)→魔法消耗也使用RP
4. **温泉恢复**: 里格巴斯有温泉→RP恢复点
