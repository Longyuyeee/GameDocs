# 种植系统 - 符文工房5(Rune Factory 5)

> 游戏版本: Rune Factory 5 (Switch/PC, 2021-2022年)
> 数据来源: therunefactory.fandom.com, Game8, RPG Site

## 1. 概述

RF5的种植系统延续了RF4的核心框架——四季作物、土壤品质、肥料系统。最大的变化是**全3D农田**——不再是RF4的2D俯视方格，而是3D空间中的农田(农龙Farm Dragon)。

## 2. 农龙(Farm Dragon)

| 属性 | 说明 |
|------|------|
| 功能 | 类似RF4的季节田——额外种植空间 |
| 解锁 | 主线早期(Whispering Woods后) |
| 特点 | 骑乘→飞回城镇 |
| 空间 | 额外农田(补充主农田) |

## 4. 土壤品质系统(继承RF4)

| 品质参数 | 影响 | 提升方式 |
|------|------|------|
| 生长速度 | 作物成熟天数 | 肥料/土壤管理 |
| 品质等级 | 作物售价 | 肥料/长期耕作 |
| 土壤健康 | 不耕种时恢复 | 休耕 |

## 5. 肥料系统

| 肥料 | 效果 |
|------|------|
| 普通肥料 | 提升土壤品质 |
| 加速肥料 | 缩短生长时间 |
| 品质肥料 | 提升作物品质上限 |

## 6. 设计特点

1. **农龙=RF4季节田的3D化**: 额外种植空间+可骑乘返回——3D化增加了移动维度
2. **作物体系延续RF4**: 相同季节/生长/复收逻辑——老玩家无缝衔接
3. **3D农田=更大空间**: 3D空间可扩展性优于2D方格→更大型的种植操作
4. **视觉效果提升**: 作物在3D环境中生长→视觉满足感更强
