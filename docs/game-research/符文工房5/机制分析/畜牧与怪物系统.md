# 畜牧与怪物系统 - 符文工房5(Rune Factory 5)

> 游戏版本: Rune Factory 5 (Switch/PC, 2021-2022年)
> 数据来源: therunefactory.fandom.com, Game8

## 1. 传统畜牧

| 动物 | 产出 | 好感工具 |
|------|------|:--:|
| 鸡(Chicken) | 蛋 | 抚摸 |
| 牛(Cow) | 牛奶 | 刷子+挤奶器 |
| 羊(Sheep) | 羊毛 | 刷子+剪刀 |
| 蜜蜂(Bee) | 蜂蜜 | — |

- 动物好感度→产出品质(同RF4)
- 动物产品加工: 蛋黄酱机/奶酪机/酸奶机/纺车

## 2. 怪物系统(核心特色)

### 2.1 怪物驯服

| 方式 | 说明 |
|------|------|
| 刷子(Brush) | 给怪物刷毛→增加好感 |
| 礼物 | 给怪物送喜欢的物品 |
| 怪物小屋(Monster Barn) | 需建造→容纳怪物 |
| 爱心捕获(Love Capture) | 部分怪物需特殊方式 |

### 2.2 怪物功能

| 功能 | 说明 |
|------|------|
| 农活帮助 | 浇水/收获/翻耕(不同怪物专长不同) |
| 骑乘 | 大型怪物可骑乘(3D世界快速移动) |
| 战斗伙伴 | 带怪物进地牢→战斗中帮助 |
| 产出 | 部分怪物产出特殊物品(如蜂蜜/丝线/等) |

### 2.3 怪物好感度

| 好感度 | 效果 |
|:--:|------|
| 0-3♥ | 基础帮助 |
| 4-7♥ | 效率提升 |
| 8-10♥ | 最高效率 |

## 3. 设计特点

1. **怪物=多功能**: 农活帮手+骑乘坐骑+战斗伙伴→一物三用——RF系列经典设计
2. **3D骑乘**: 3D世界骑乘怪物探索→比RF4的2D骑乘更沉浸
3. **继承RF4的怪物框架**: 刷子驯服/怪物小屋/爱心捕获→几乎1:1继承
4. **更多大型怪物可骑乘**: 3D空间→骑乘移动的价值更高(地图比RF4大)
