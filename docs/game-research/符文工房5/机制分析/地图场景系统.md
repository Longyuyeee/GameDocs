# 地图与场景系统 - 符文工房5(Rune Factory 5)

> 游戏版本: Rune Factory 5 (Switch/PC, 2021-2022年)
> 数据来源: therunefactory.fandom.com, Game8, Wikipedia

## 1. 地图结构

```
里格巴斯(Rigbarth)及周边
├── 里格巴斯镇(Rigbarth Town) — 核心枢纽
│   ├── 主角家/农田→SEED哨站(Silo)
│   ├── 商店街
│   │   ├── Studio Palmo(Palmo)—家具/双人床
│   │   ├── 铁匠铺—Martin
│   │   ├── Crystalabra—稀有物品
│   │   └── 一般商店/药店
│   ├── 旅店( Murakumo)
│   ├── 餐馆
│   ├── 侦探事务所(Cecil)
│   └── 广场(节日场地)
├── Whispering Woods — 第1地牢/九尾狐
├── Belpha Ruins — 第2地牢/符文异常
├── Kelve Lava Caves — 第3地牢/火山
├── Meline Crystal Caverns — 第4地牢/水晶
├── Atohl's End — 第5地牢/诅咒之地
├── Gadeus Grasslands — 第6地牢/草原
├── Bandit King's Old Base — 第7地牢
├── 四大符文深层(4个) — Belpha/Meline/Kelve/森林深处
├── Everlasting Darkness — Radea黑暗领域
├── Basara's Hideaway — 藏身处
├── Thundering Wastes — Lucas恢复记忆
├── Cloudheim → SEED Floating Fortress — 浮空要塞
└── Calamity's Edge — 最终BOSS空间
```

## 2. 地牢解锁梯度

```
初始: Whispering Woods
  ↓
主线推进: Belpha→Kelve→Meline→Atohl
  ↓
中期: Gadeus→Bandit King's
  ↓
Livia恢复: 4深层地牢
  ↓
后期: Everlasting→Basara→Thundering→Cloudheim→浮空要塞
  ↓
终局: Calamity's Edge
  ↓
通关后: Rigbarth Maze(终极挑战)
```

## 3. 3D场景下的移动

| 方式 | 说明 |
|------|------|
| 步行 | 基础 |
| 骑乘怪物 | 快速/可跨越地形 |
| 农龙(Farm Dragon) | Whispering Woods后→飞回城镇 |
| 传送 | 各地牢门口→快速旅行 |

## 4. 设计特点

1. **18地牢=系列最多**: RF4约12个→RF5 18个→内容量大幅增加
2. **3D开放区域**: 不是RF4的"屏幕→屏幕"切换→而是连续的3D空间
3. **里格巴斯=3D城镇**: 与RF4的2D Selphia不同→更沉浸但也更空旷
4. **Switch性能问题**: 3D开放区域→加载时间+帧率不稳定→RF5最大的技术批评点
