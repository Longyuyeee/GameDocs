## 2. 各区域解锁条件

### 2.1 Sun Haven(起始区域)
- 无需解锁，游戏开始即拥有
- 初始农场可直接使用

### 2.2 Nel'Vari(精灵村)
解锁步骤:
1. 完成所有Sun Haven主线任务(红色太阳图标)
2. Elios回归后选择Nel'Vari任务线(蓝色嫩芽图标)
3. 与图书馆Amanda和酒馆Edwin对话翻译精灵文献
4. 在西部森林找到Anne的宝藏(用镐子清除碎石)
5. 种植并浇灌40颗葡萄种子，全部用法力注入(Infuse)
6. 收获后向西进入精灵森林→向北遇到树灵Salix和Umus
7. 注入法力至Umus获得进入许可

### 2.3 Withergate(恶魔城)
解锁步骤:
1. 推进主线至需要喂食火车站北侧Snaccoon(10小麦+10土豆)
2. 完成Catherine的任务线(送番茄汁给Liam、面包给Claude)
3. 穿越迷雾迷宫(墙壁不可见，靠感觉或用高坐骑看)
4. 过迷宫后遇到Charon(渡船人)
5. 完成"国王丢失的宠物"任务:
   - 用投石机击败Heatviper
   - Linus方案: 50条砂岩鱼→给守卫
   - Rastus方案: 250个砂岩矿石→给守卫
6. Charon带你前往Withergate
7. 被踢出城市后，打通下水道遇到Donovan

### 2.4 Brinestone Deeps(深海水晶域)
- 在玩家农场东侧海滩水域出现漩涡
- 进入漩涡即到达深海水晶域(需主线进度推进)

## 5. 可交互场景元素

| 元素 | 说明 |
|------|------|
| 可破坏物 | 碎石(需镐子)、树桩(需高级斧)、杂草 |
| 宝箱(Treasure Chests) | 散落世界各地和矿洞内，含金币/物品 |
| 上锁宝箱(Locked Chests) | 需钥匙开启，掉落更好 |
| 传送点 | 特定地点间的快速移动 |
| 矿洞钥匙门 | 每5层有奖励钥匙(战斗试炼地牢) |
| Snaccoons | 各地图需要特定物品换取通行/解锁 |

## 6. 季节/天气/昼夜对场景影响

| 因素 | 场景变化 |
|------|---------|
| 季节 | Sun Haven区域植被色调变化，精灵/恶魔区域不变 |
| 天气 | 雨天免除浇水 |
| 昼夜 | 夜晚色调变暗，商店8PM关门，需在0:00前睡觉 |
| 灾害 | 春:幼龙害虫/夏:作物着火/秋:荆棘/冬:冰封 |

## 8. 设计特点

1. **四区域递进式解锁**: Sun Haven→Nel'Vari→Withergate→Brinestone Deeps，每个区域解锁伴随完整任务线
2. **三农场并行经营**: 每个区域独立农场，驱动跨区域每日往返的经营节奏
3. **Snaccoon门控机制**: 各地图节点需要特定物品解锁，类似"祭坛/社区中心"的门控设计
4. **30天一次性解锁vs长期经营**: 区域解锁为一次性任务，但解锁后形成三农场并行的长期经营循环
5. **无缝世界**: 大部分区域间通过步行+过渡区域连接，非独立关卡
