## 二、好感度机制

### 2.1 基本规则

- **送礼**: 每周可送2次礼物 + 生日额外1次
- **对话**: 每日首次对话+少量好感度
- **布告栏任务**: 完成指定NPC的任务+好感度
- **好感度等级**: 0-10心(恋爱后可达更高)
- **好感度阶段**:
  - 0-7心: 朋友阶段
  - 8心: 可赠送表白信物(恋爱)
  - 10心: 可求婚

### 2.2 好感度增益技能

| 技能 | 效果 |
|------|------|
| "感激(Gratuity)"(Exploration T7) | 布告栏任务每日+10金币(上限200/350/500) |
| "欣赏(Appreciation)"(Exploration T5) | 每日每个恋爱对象每心+10/15/20金币(上限+1,000g/天) |
| "人气(Popularity)"(Exploration T9) | 任何对象10心时，每日+1/2/3社区代币 |
| "心意承诺(Heartfelt Promise)"(Exploration T10) | 婚戒效果增强20/35/50% |
| "好邻居(Good Neighbor)"(Exploration T1) | 布告栏任务+1/2额外社区代币 |
| "感激的邻居(Thankful Neighbors)"(Exploration T4) | 给可攻略NPC送礼+1/2/3额外社区代币 |

### 2.3 社交联络技能

| 技能 | 效果 |
|------|------|
| "笔友(Pen-Pals)"(Exploration T3) | 连续4/7/10周收到NPC每周来信 |

## 四、恋爱/婚姻系统

### 4.1 恋爱流程

```
提升好感→8心→赠送表白信物→恋爱状态→10心→求婚→结婚
```

- 可**同时与多人约会**(无嫉妒机制)
- 只能与**1人结婚**
- 结婚后可获得配偶**纪念品(Keepsake)**，提供各种永久加成

### 4.2 婚后内容

- 配偶可搬入玩家家中
- 子女系统(细节待确认)
- 配偶纪念品加成因角色而异

## 五、社区/任务系统

### 5.1 布告栏任务(Bulletin Board Quests)
- 每日刷新，来自各NPC的随机请求
- 奖励: 金币(>商人售价)+社区代币+好感度+技能经验
- 与Exploration技能树深度联动

### 5.2 社区代币(Community Tokens)
- 完成布告栏任务获得
- 兑换社区奖励/特殊物品
- 技能加成: Good Neighbor(+1/2每任务)、Popularity(每日+1/2/3)、Thankful Neighbors(送礼+1/2/3)

### 5.3 主线/支线任务
- 红色太阳图标: 主线任务
- 蓝色嫩芽图标: Nel'Vari任务线
- 紫色图标: Withergate任务线
- 橙色: 支线/可选任务

## 六、NPC行为AI

- **日程系统**: 各NPC有独立日程路线，随时间和天气变化
- **季节性行为**: 节日期间NPC聚集在节日场地
- **天气影响**: 雨天NPC更多待在室内
- 商店NPC在8PM后不可交互(购买)

## 七、设计特点

1. **21位可攻略角色**: 远超星露谷(12位)和珊瑚岛(28位中位数)，四区域均有候补
2. **种族多样性**: 人类/精灵/恶魔/阿玛瑞/纳迦 —— 各候补种族背景差异化
3. **分版本逐步新增**: v1.2(Kai+Vivi+Wesley)、v1.3(Lucius)、v1.4(Karish)，显示持续内容更新策略
4. **部分候补需解锁**: 不是所有候补一开始就出现，需推进剧情/达到特定条件
5. **无嫉妒机制**: 可同时多人约会，降低社交惩罚
6. **技能深度绑定社交**: Exploration技能树大幅强化社交收益(金币/代币)，使社交从"剧情消费"变为"经济策略"
7. **纪念品系统**: 每个恋爱对象提供独特永久加成，赋予恋爱选择策略深度
