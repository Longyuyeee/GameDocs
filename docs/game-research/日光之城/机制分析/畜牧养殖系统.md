# 畜牧养殖系统 - 日光之城(Sun Haven)

> 游戏版本: Sun Haven v1.4+ (PC/Switch)
> 数据来源: sunhaven.wiki.gg, TheGamer, GameRant

## 1. 概述

Sun Haven 的动物系统分布在三个区域，每个区域各有特色动物。动物**不需要谷仓也能饲养**(自由放养)，但谷仓提供容量加成和内置喂食站。动物**不会破坏作物**，可放在农场任意位置，甚至屋内。

## 3. 养殖全流程

```
获取动物 → 放置农场 → 每日喂食 → 好感度成长 → 每日产出 → (可选繁殖)
```

- **购买地点**: Sun Haven-Kitty's Animal Store(农场北侧)、Nel'Vari-Fonzo's Animal Shop(西北区)、Withergate-Ur's Farm Animal Shop(下水道)
- **喂食需求**: 每天1次，不喂食会降低健康值和好感度，健康值过低将停止产出
- **产出收集**: 每日手动互动收集，或购买自动收集器(Auto-Collector, 1,000g)

## 4. 好感度/成长机制

- **好感度不因不抚摸而衰减**——仅不喂食时下降
- 好感度提升方式: 每日喂食 + 抚摸互动
- "快乐动物(Happy Animals)"技能(Farming T2): 8/16/24%概率额外动物产品
- **金心系统**: "金心(Golden Heart)"技能(Farming T9):
  - 动物可突破好感度上限至金心
  - 8/12/16%概率产出金品质产品
  - 最高好感度时配合满级金心技能，可达**68%概率**产出金品质产品

## 9. 设计特点

1. **无疾病/死亡机制**: 动物不会生病或死亡，仅停止产出
2. **动物可随时右键拾取移动**: 不影响好感度/健康/产出
3. **区域差异化**: 三个区域各有专属动物，驱动跨区域经营
4. **技能深度绑定**: 金心、快乐动物、满员等技能让畜牧系有完整build方向
5. **精灵动物产出原料**: 树桩产木材/丝蛾产丝绸——将畜牧与制造系统融合
