# 种族与职业系统 - 日光之城(Sun Haven)

> 游戏版本: Sun Haven v1.4+ (PC/Switch)
> 数据来源: IGN, Fandom Wiki, gamerjournalist.com, re-actor.net

## 一、概述

Sun Haven 是牧场游戏中极少提供**多种族角色创建**的作品。游戏开始时可从**7个种族**中选择，每个种族有独特的被动/主动技能。同时还提供**5种初始职业**选择，形成 7×5=35 种种族×职业组合。

种族和职业的选择**不锁死游戏内容**，任何种族/职业都可体验全部内容，差异主要在于玩法侧重和效率加成。

## 二、七大种族详情

### 1. 人类(Human)
| 属性 | 详情 |
|------|------|
| 技能名 | 专家工匠(Expert Crafter) |
| 类型 | 被动(永久生效) |
| 效果 | +20%制作速度 |
| 推荐指数 | ★★★★★ |
| 适合玩家 | 新手、全能型、制作流 |

> 公认最强种族之一。制作速度加成贯穿整个游戏，所有制作站(铁砧、烹饪、炼金等)都享受加成。

### 2. 精灵(Elf)
| 属性 | 详情 |
|------|------|
| 技能名 | 精灵之眼(Elven Eyes) |
| 类型 | 被动(永久生效) |
| 效果 | +10-15%弩伤害 |
| 推荐指数 | ★★★★ |
| 适合玩家 | 远程战斗流、探索流 |

> 弩是Sun Haven中安全性最高的战斗方式(远程无需近身)。精灵种族加成让弩路线更加强势。

### 3. 恶魔(Demon)
| 属性 | 详情 |
|------|------|
| 技能名 | 暗影冲刺(Shadow Rush) |
| 类型 | 主动技能 |
| 效果 | +60-75%移动速度，持续5-10秒 |
| 冷却时间 | 10分钟 |
| 推荐指数 | ★★ |
| 适合玩家 | 探索/赶路流 |

> 公认最弱种族。速度爆发有用但10分钟冷却太长。游戏已有坐骑系统提供持续移速加成。

### 4. 天使(Angel)
| 属性 | 详情 |
|------|------|
| 技能名 | 奇迹术(Miracle Spell) |
| 类型 | 主动技能 |
| 效果 | 治疗自身15-20HP(+15%最大HP)，同时治疗附近盟友 |
| 冷却时间 | 10分钟 |
| 推荐指数 | ★★★(单人)/★★★★★(多人) |
| 适合玩家 | 辅助/合作玩家 |

> 多人模式中极强——可同时治疗多名队友。单人模式因冷却长而价值下降。

### 5. 阿玛瑞(Amari)
| 属性 | 详情 |
|------|------|
| 技能名 | 原始本性(Primal Nature) |
| 类型 | 被动(永久生效) |
| 效果 | 攻击敌人时小概率恢复HP(吸血) |
| 推荐指数 | ★★★★ |
| 适合玩家 | 近战战斗流、矿洞探索 |

> 永久吸血被动，战斗续航极佳。近战矿洞长时间探索时价值突出。

### 6. 元素使(Elemental)
| 属性 | 详情 |
|------|------|
| 技能名 | 元素汲取(Elemental Tap) |
| 类型 | 主动技能 |
| 效果 | 恢复自身10-20法力(+10%最大法力)，同时恢复附近盟友 |
| 冷却时间 | 10分钟 |
| 推荐指数 | ★★★★(单人)/★★★★★(多人) |
| 适合玩家 | 法术流、多人合作 |

> 法力是法术流派的核心资源(无洒水器→依赖雨云法术+地震)。元素使为法术续航提供强力支援。目前仅火焰元素变体可玩。

### 7. 纳迦(Naga)
| 属性 | 详情 |
|------|------|
| 技能名 | 人鱼之触(Mermaid's Touch) |
| 类型 | 被动(永久生效) |
| 效果 | 钓鱼甜蜜点范围扩大25-30% |
| 推荐指数 | ★★★ |
| 适合玩家 | 钓鱼爱好者 |

> 钓鱼专精种族。使钓鱼小游戏显著更容易，适合将钓鱼作为主要收入来源的玩法。

## 三、五大职业

| 职业 | 起始加成方向 | 推荐搭配种族 |
|------|------------|------------|
| 工具大师(Tool Master) | 工具使用效率 | 人类(制作速度叠加) |
| 探索者(Explorer) | 探索/移动 | 恶魔(速度主题) |
| 咒术师(Spellcaster) | 法术强度 | 元素使(法力协同) |
| 钓客(Angler) | 钓鱼效率 | 纳迦(钓鱼专精) |
| 决斗者(Duelist) | 战斗能力 | 阿玛瑞/精灵(战斗吸血/弩加成) |

## 四、种族×职业最佳组合

| 种族 | 推荐职业 | 理由 |
|------|---------|------|
| 人类 | 工具大师 | 制作速度双重叠加 |
| 精灵 | 决斗者 | 弩伤害+战斗基础 |
| 恶魔 | 探索者 | 速度加成主题统一 |
| 天使 | 任意(多人) | 治疗职业不限 |
| 阿玛瑞 | 决斗者 | 近战吸血+战斗 |
| 元素使 | 咒术师 | 法力恢复+法术强度 |
| 纳迦 | 钓客 | 钓鱼难度降低+钓鱼效率 |

## 五、设计分析

1. **轻度差异化**: 种族差异不锁死内容(不强制精灵只能玩弩)，仅提供效率偏向——保留玩家自由
2. **主动技能冷却设计**: 10分钟冷却的主动种族技能设计偏保守，防止种族选择成为"必选"压力
3. **人类推荐为默认最优**: 20%制作速度是最"普适"的加成，几乎所有玩法都涉及制作——设计上鼓励新手选人类
4. **多人合作激励**: 天使(治疗)/元素使(法力)在多人模式中价值翻倍，鼓励组队时种族多样化
5. **纳迦的钓鱼设计**: 钓鱼小游戏难度是许多玩家的门槛，纳迦提供"降低门槛"的种族选项
6. **美术差异化**: 每个种族有独立的角色美术(不同肤色/耳朵/尾巴等)，不仅是数值差异
