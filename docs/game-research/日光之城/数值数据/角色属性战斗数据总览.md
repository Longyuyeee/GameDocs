## 一、基础属性一览

| 属性 | 英文 | 含义 | 影响的行为 |

## 七、伤害计算公式(推测)

### 物理伤害
```
最终伤害 = (武器基础伤害 + 攻击力加成) × (1 + 统一性加成%) × 暴击倍率(如有) - 目标护甲减免
```

### 魔法伤害
```
最终法术伤害 = (法术基础伤害 + 法术伤害加成 + Intellect加成) - 目标魔法防御减免
```

### 状态异常触发
- 火球术: 100%触发燃烧DOT(2/3/4秒)
- Hot Shot: 100%弩箭触发燃烧(2/3/4秒)
- 闪电链: 100%连锁弹跳(无额外异常)
