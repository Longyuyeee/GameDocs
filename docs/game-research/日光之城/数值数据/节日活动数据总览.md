## 四、节日通用规则

| 规则 | 说明 |
|------|------|
| 参与时间 | 大部分节日需12PM后与开始NPC对话 |
| 结束时间 | 晚上11PM强制结束，直接传送回家 |
| NPC对话 | 节日NPC对话不加好感度(纯氛围) |
| 回家后 | 无法再做任何农场杂务 |
| 建议准备 | 参加前完成当天农场工作，清空背包留空间 |
| 离开方式 | 与开始NPC再次对话可提前离场 |
