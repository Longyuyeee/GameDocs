## 二、"Reading Steiner"能力

| 属性 | 说明 |
|------|------|
| 能力持有者 | **冈部伦太郎**(唯一的完全RS能力者) |
| 能力描述 | 在世界线切换时，冈部保留前一条世界线的所有记忆——其他所有人"不记得" |
| 叙事功能 | **这是玩家与世界线系统的接口**——冈部记得=玩家记得=你需要用这个知识来"改变"当前世界线 |
| 弱RS持有者 | 某些角色在特定条件下有微弱的"既视感"——如红莉栖在β场中对α场的关系有模糊的记忆 |
