## 一、世界线概念

命运石之门的世界线(World Line)系统不是"平行宇宙"——它是**同一条时间线上的不同可能性**。"世界线变动率"(Divergence Number)是一个测量当前世界线相对于"原始世界线"偏离程度的标量值。

## 三、世界线收束原理

| 概念 | 说明 | 游戏体现 |
|------|------|------|
| **Attractor Field** (吸引场) | 某些事件在所有世界线上"必定发生" | α场=真由理必死，β场=红莉栖必死 |
| **世界线收束** | 同一吸引场内，小事件可改变，但"核心事件"不变 | 冈部怎么尝试救真由理→她都死(只是日期和方式有微小差异) |
| **Divergence (偏离)** | 积累足够的小改变才能"跳出"当前吸引场 | 撤回全部D-Mail=跳出α场→回到β场 |
| **Steins;Gate** | 唯一不属于任何吸引场的世界线 | 1.048596%——双星存活 |

## 六、变动率测量仪(Divergence Meter)

| 属性 | 数据 |
|------|------|
| 创建者 | 未来的冈部伦太郎(2025) |
| 携带者 | 阿万音铃羽(从2036年带回) |
| 功能 | 实时测量当前世界线变动率(精确到小数点后6位) |
| 物理原理 | Kerr黑洞参照系差异——通过引力波检测时间线曲率 |
| 叙事功能 | 冈部判断"是否成功改变了世界线"的唯一客观指标 |
