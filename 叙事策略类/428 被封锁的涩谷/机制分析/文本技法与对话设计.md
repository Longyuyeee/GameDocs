# 文本技法与对话设计 — 428: 被封锁的涩谷

> 游戏版本: 428: Shibuza Scramble v2018 (PC/PS4)
> 数据来源: [428 Wiki](https://428shibuyascramble.fandom.com/), [Spike Chunsoft](https://www.spike-chunsoft.com/), [Steam社区](https://steamcommunity.com/app/648580/)

## 一、"音画小说"——文字即画面

428 是"音画小说"（Sound Novel）的里程碑——**文字本身承担了传统视觉小说中"画面"的功能**：

| 文字的功能 | 示例 |
|------|------|
| 场景描述 | "涩谷十字路口——10:05——红灯。他站在人群中，手在发抖。" |
| 动作描述 | "他跑了——照片切换到下一个路口——他在喘气。" |
| 内心独白 | "加纳:'为什么是我？'" |

---

## 二、5种语调——5个主角的"声音"

| 主角 | 文字风格 |
|------|------|
| 加纳 | 焦虑、被动——"我只是想活过今天" |
| 小玉 | 热血、直接——"我必须阻止他们" |
| 远藤 | 冷静、职业——"我是警察——这是我的职责" |
| 大沢 | 观察者、分析——"这个故事比我想象的要大" |
| アルファルド | 神秘、冷静——每句话都有隐含的重量 |

---

## 三、照片+文字=独特的叙事节奏

| 媒介 | 节奏效果 |
|------|------|
| 照片的切换 | 画面被锁定——你需要**阅读**来"看到"动作 |
| 文字的节奏 | 快=紧张（短句+快速换行），慢=沉思（长段描述） |
| KEEP OUT=文字暂停 | 你不能继续——**文字在等待你"选择"** |
