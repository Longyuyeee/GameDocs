## 二、分支拓扑示意

### 2.1 总体结构：线性主路径+局部惩罚循环

```
START → [调查1] → [法庭1-1] → [法庭1-2] → [调查2] → [法庭2] → END(唯一结局)
                      ↑ ↓(错误)              ↑ ↓(错误)
                    [扣血→重试]            [扣血→重试]
```

### 2.2 2-4"伪分支"特例

2-4最终选择是三部曲中唯一的"看起来有分支"的设计：
- **选项A**：宣告有罪 → 真宵安全，真凶王都楼伏法，成步堂保持正义 → **正确**
- **选项B**：宣告无罪 → 真宵死亡，王都楼逃脱 → **Game Over**

实质：选B = 游戏结束(非叙事分支)
