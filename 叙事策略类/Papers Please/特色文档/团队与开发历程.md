> 开发商: Lucas Pope (单人) | 发行商: 3909 LLC
> 发行年份: 2013年8月8日 | 销量: 500万+ | Steam好评率: 97%
> 数据来源: [Lucas Pope GDC演讲](https://www.gdcvault.com/), [开发者博客](https://dukope.com/), [Wikipedia](https://en.wikipedia.org/wiki/Papers,_Please)

## 1. 开发团队

| 角色 | 说明 |
|------|------|
| **开发者** | **Lucas Pope 单人**——前Naughty Dog工程师（参与过Uncharted系列） |
| **发行商** | 3909 LLC（Pope自己的发行公司） |
| **团队规模** | **1人**——全部设计/编程/美术/写作/声音由Pope一人完成 |

## 2. 开发历程

| 时间 | 事件 |
|:--:|------|
| 2012 | Pope离开Naughty Dog——移居日本——开始独立开发。灵感: 过境时看到移民官"盖章"的动作——"这个人在3秒内决定别人的命运" |
| 2012-2013 | 约9个月开发周期。使用**Haxe+OpenFL**(不是Unity)——Pope自己适配 |
| 2013年8月 | Steam发售($9.99)。立即走红——"盖章=道德选择"的设计被广泛赞誉 |
| 2014 | 销量突破100万——独立游戏里程碑 |
| 2017 | iOS/Android移动版 |
| 2023 | 10周年——销量突破500万——"单人开发的叙事里程碑" |

## 4. 设计启示

1. **"盖章"作为一切**——用单一动作承载全部道德/策略/叙事
2. **单人可行**——1人×9个月=定义了"官僚模拟"品类
3. **"规则=反派"**——不需要写一个"坏人"——让规则成为敌人
