# 文本规模数据总览 — Papers, Please

> 游戏版本: Papers, Please v1.4 (PC/Steam, 2013)
> 数据来源: [Papers Please Wiki](https://papersplease.fandom.com/), [HowLongToBeat](https://howlongtobeat.com/), [Steam社区](https://steamcommunity.com/app/239030/)

## 一、文本总量——品类中最少

| 指标 | 数值 |
|------|:--|
| 总文本量（英文） | **约1-2万词**（估算——远少于任何其他叙事策略游戏） |
| 规则手册文本 | ~2000词——但**动态更新** |
| 申请人对话 | ~5000-8000词——破碎、简短、重复的模板 |
| 结局文本 | ~500-1000词——简短但有力 |
| 家庭账单 | 数字+标签——几乎无文本 |

## 二、"最少文本"的叙事力量

Papers, Please 用1-2万词创造了与140万词的苏丹的游戏同等的叙事强度——证明**叙事力量不在文本量，在文本的"道德位置"**。
