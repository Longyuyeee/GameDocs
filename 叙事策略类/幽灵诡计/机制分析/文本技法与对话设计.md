# 文本技法与对话设计 — 幽灵诡计

> 游戏版本: Ghost Trick: Phantom Detective v2023 (PC/Switch/PS4)
> 数据来源: [Ghost Trick Wiki](https://ghosttrick.fandom.com/), [Capcom官方](https://www.capcom-games.com/), [Steam社区](https://steamcommunity.com/app/1967430/)

## 一、巧舟的标志性幽默——在最严肃的情境中

| 幽默类型 | 示例 |
|------|------|
| 希瑟尔的内心独白 | "我死了。这不太理想。"——**用最轻的语气说最重的事实** |
| 角色的夸张动画 | 巧舟的"颜艺"——角色的表情在0.5秒内从平静→极度震惊 |
| 黑猫的"忠告" | "我不知道你为什么会在这里——但我知道你接下来会做什么。"——猫有态度 |

## 二、沉默主角+话多配角=巧舟公式

| 角色 | 说话量 |
|------|:--:|
| 希瑟尔(主角) | 内心独白（你"听"到）——**不对外界说话** |
| 凛音 | 多——她是"能说话的活人" |
| 黑猫 | 中——只说"该说的"——不多不少 |
| 配角们 | 各有独特的语癖——巧舟的"角色声音"设计 |
