# 角色弧线与动机设计 — 幽灵诡计

> 游戏版本: Ghost Trick: Phantom Detective v2023 (PC/Switch/PS4)
> 数据来源: [Ghost Trick Wiki](https://ghosttrick.fandom.com/), [Capcom官方](https://www.capcom-games.com/), [Steam社区](https://steamcommunity.com/app/1967430/)

## 一、主角(希瑟尔/Sissel)——"我不知道我是谁"

希瑟尔(Sissel)的弧线=**从无到有的自我发现**：

| 弧线阶段 | 状态 |
|:--:|------|
| 起点 | 刚死——失忆——"我是谁？我为什么死了？" |
| 拯救他人 | 通过拯救他人的生命——发现了关于自己的碎片 |
| 自我揭示 | **"原来我是...——那一夜是..."** |
| 终点 | "我现在知道了一切——我可以选择了" |

## 二、关键配角

| 角色 | 功能 |
|------|------|
| **凛音(Lynne)** | 警探——"第一个被希瑟尔拯救的人"——她的命运与希瑟尔的真相交织 |
| **黑猫** | 希瑟尔的"向导"——但**猫知道得比它说的多** |
| **操纵者** | 幕后的"另一股力量"——"原来我不是唯一有Ghost Trick的人" |

| 维度 | 幽灵诡计 | 逆转裁判 |
