# 叙事视角与信息控制 — 幽灵诡计

> 游戏版本: Ghost Trick: Phantom Detective v2023 (PC/Switch/PS4)
> 数据来源: [Ghost Trick Wiki](https://ghosttrick.fandom.com/), [Capcom官方](https://www.capcom-games.com/), [Steam社区](https://steamcommunity.com/app/1967430/)

## 一、"幽灵视角"——你只能看不能干涉

| 你能做的 | 你不能做的 |
|------|------|
| 附身物体+操（物理影响） | 说话——无法直接与人交流 |
| 看到活人的世界 | 被活人看到（你是幽灵——他们看不见你） |
| 在物体之间移动 | 自由移动——只能在"有核心的物体"之间跳 |

---

## 二、信息控制——"你只知道你看到的"

| 你"知道"的 | 你"不知道"的 |
|------|------|
| 你刚死了——失忆 | 你是谁——你过去做了什么——为什么被杀了 |
| 你有"死者之力" | 这个能力从哪来——"为什么是我？" |
| 眼前发生的（你"看到"的） | 在事件发生前的过去 |

**信息不对称=核心叙事驱动力**——"我想知道我是谁"驱动了整个游戏。

---

## 三、电话——"唯一的交流窗口"

| 电话的功能 | 叙事效果 |
|------|------|
| 你可以通过电话线移动 | 你的"交通系统"——从一台电话到另一台 |
| 你可以"听到"电话另一端 | 但你无法"说"——你是聆听者 |
