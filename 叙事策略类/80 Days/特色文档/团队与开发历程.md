## 3. 核心技术

自研Ink脚本语言——编译后运行时按需加载——50万+字可在移动设备流畅运行。2016开源——成为最广泛使用的分支叙事引擎之一。

## 4. 设计哲学

| 哲学 | 来源 |
|------|------|
| "世界不是围绕你转的" | 核心事件你可能永远遇不到 |
| "诱导非最优决策" | Meg Jayanth GDC——"策略坏决定=最好故事" |
| "被沉默者获得声音" | 仆人/女性/殖民地原住民=叙述者 |

## 5. 设计启示

1. **Storylet=模块化叙事未来**——条件触发式原子模块=涌现叙事
2. **极简UI=极大叙事**——地图+文字=文学改编终极形态
3. **反完满主义**——不能100%的游戏=真实旅行
4. **文学改编新范式**——用交互重新诠释原著精神
