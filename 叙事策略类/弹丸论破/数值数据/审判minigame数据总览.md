# 审判 Minigame 数据总览 — 弹丸论破

> 游戏版本: Danganronpa: Trigger Happy Havoc v2016 (PC/Steam)
> 数据来源: [Danganronpa Wiki](https://danganronpa.fandom.com/wiki/Class_Trials/Danganronpa_1)

## 一、审判minigame类型

| Minigame | 交互方式 | 核心技能 | 难度曲线 |
|------|------|------|:--:|
| **无休止议论** (Non-Stop Debate) | 射击——言弹→弱点 | 观察力/反应/推理 | 逐步增加(轮数+杂音密度) |
| **闪耀字谜** (Hangman's Gambit) | 射击移动字母→拼词 | 词汇/反应/推理 | 中等(字母移动变快) |
| **机关枪论战** (Bullet Time Battle) | 节奏游戏——按节拍锁定+射击 | 节奏感/反应 | 高(节奏加快+更多辩解) |
| **高潮推理** (Closing Argument) | 拼图——选择正确漫画分镜 | 叙事理解/逻辑 | 逐步增加(页数增多) |

## 二、无休止议论数据

| 参数 | Ch1 | Ch3 | Ch5 | Ch6 |
|------|:--:|:--:|:--:|:--:|
| 辩论轮数 | 5 | 7 | 8 | 10+ |
| 每轮发言人数 | 3-5 | 4-6 | 5-7 | 6-9 |
| 弱点出现间隔(秒) | ~8秒 | ~6秒 | ~4秒 | ~3秒 |
| 杂音(White Noise)密度 | 低 | 中 | 高 | 极高 |
| 吸收言弹(从证言中提取新言弹) | 无 | 有(低) | 有(高) | 有(极高) |

## 三、闪耀字谜数据

| 参数 | 说明 |
|------|------|
| 字母总数(每谜题) | 4-8个 |
| 飞行速度 | 慢→快(随章节推进) |
| 击中错误字母 | 无直接惩罚——但浪费时间=精神焦虑 |
| 使用范围 | 每章1-3次——当关键概念需要被"解锁"时 |

## 四、机关枪论战数据

| 参数 | 说明 |
|------|------|
| 发生时机 | 审判的"高潮"——真凶被锁定时 |
| 节奏阶段数 | 3-5个阶段(每个阶段=新辩解) |
| 每阶段锁定数 | 3-5次锁定 |
| 音乐 | 专属BGM——"Scrum Debate"或角色特定配乐 |
| 失败后果 | 影响值-1→如果归零→全员处刑 |
| 叙事功能 | **"与黑(真凶)的心理对决"**——节奏=战斗=心理防线被逐步击溃 |
| 最后一击 | **"在此终结！"**——最后一轮的最后一锁=真凶的心理防线完全崩溃 |

## 六、影响值/难度调节

| 难度 | 影响值(初始) | 每错误损失 | 特点 |
|:--:|:--:|:--:|------|
| Gentle(简单) | 7心 | -1心 | 更多提示——弱点更明显 |
| Kind(普通) | 5心 | -1心 | 标准 |
| Mean(困难) | 3心 | -1-2心 | 更少提示——杂音更多——速度更快 |

## 七、审判技能装备系统

| 技能类型 | 功能 | SP消耗 |
|------|------|:--:|
| 增加影响值上限 | +1-2心(基础5→最高7) | 2-4 |
| 延缓辩论速度 | 弱点停留时间延长 | 2 |
| 增加言弹伤害 | "一枪削弱"多个弱点 | 3 |
| 自动瞄准 | 减少手动瞄准需求 | 3 |
| 减少杂音 | White Noise减少或消失 | 2 |
