## 二、角色"才能"与行为驱动

| 才能类型 | 角色 | 行为驱动 |
|:--:|------|------|
| **战斗/暴力型** | 大神樱(格斗家)/大和田纹土(暴走族)/桑田怜恩(棒球选手)/灭族者翔(杀人魔) | 物理能力——"当争论失败时，力量是最后的语言" |
| **智力/分析型** | 雾切响子(???)/十神白夜(贵公子)/瑟雷丝(赌徒) | 冷静分析——"在别人还在恐慌时，我已在计算" |

## 三、幸存者最终去向

| 幸存者 | 终局状态 |
|------|------|
| 苗木诚 | **"希望"的象征**——走出学园——面对被绝望摧毁的外部世界 |
| 雾切响子 | 与苗木一起走出去——她的过去终于被揭示 |
| 十神白夜 | "贵公子的傲慢被打破"——接受了"你们不是我的棋子" |
| 朝日奈葵 | "最悲伤的幸存者"——她失去了大神樱(最好的朋友)——但她依然坚持到了最后 |
| 腐川冬子 | Genocide Jack被"控制"——腐川可以不再依赖她 |
| 叶隐康比呂 | **"搞笑的幸存者"**——"我是怎么活下来的？我也不确定。" |
