# 多结局与重玩设计 — 弹丸论破

> 游戏版本: Danganronpa: Trigger Happy Havoc v2016 (PC/Steam)
> 数据来源: [Danganronpa Wiki](https://danganronpa.fandom.com/), [Steam社区指南](https://steamcommunity.com/)

## 一、"只有一条路"的单线结构

弹丸论破是纯单线叙事游戏——只有一条故事路径和一个最终结局。你不能"选择不同的凶手"或"拯救某个角色"。案件的结果是固定的。

### 1.1 为什么是单线？

| 原因 | 说明 |
|------|------|
| **案件逻辑的严谨性** | 如果允许"不同的凶手"，就需要为每个可能性写独立剧本——6个案件×N个分支=指数级的工作量 |
| **固定的死亡顺序=叙事节奏** | 每个角色的死亡时间点被精心安排——改变它会破坏整个叙事的情感弧线 |
| **Chapter 6 的终极反转** | "我们自愿被关在这里"的终局依赖于前面所有固定事件作为"证据" |

## 二、"Game Over"≠结局

游戏唯一的"分支"是：在班级审判中，如果影响值归零→全员处刑→Game Over。但这不是"另一种结局"——这是**被鼓励去回避的失败状态**。

| 正常的结局 | Game Over |
|------|------|
| Chapter 6完成→揭示真相→幸存者逃出学园 | 审判中的任何错误累积导致的全员处刑 |
| 6个幸存者进入"外面的世界" | 17个(全员)学生在学园中死亡——无人幸存 |

## 三、好感度——"只改变体验，不改变结果"

### 3.1 学校模式(School Mode)

Steam版附赠的"学校模式"是一个额外的"if路线"——黑白熊不宣布自相残杀规则，学生们和睦地在学园里生活。在这个模式中：
- 你可以与所有角色(包括在主线中"会死去"的角色)建立好感度
- 完成所有角色的好感度事件→解锁额外对话和"if结局"
- **这不是"真结局"——这是"幻想结局"——"如果这个世界不是这样的"**

### 3.2 重玩驱动

| 重玩原因 | 说明 |
|------|------|
| 在审判中尝试"错误答案" | 看黑白熊的不同嘲讽——以及Game Over的处刑动画 |
| 学校模式 | 与主线中会死去的角色建立"不可能的好感度" |
| 重新体验故事——知道真相 | **"重读"**——第二次玩时，每个角色的每一句话都有了不同的含义（但这在弹丸论破中级弱于海市蜃楼之馆） |

## 四、与本品类其他游戏的多结局设计对比

| 维度 | 弹丸论破 | 逆转裁判 | 海市蜃楼之馆 | 苏丹的游戏 |
|------|------|------|------|------|
| 故事结局数 | **1**（单线） | **1**（单线——各案件独立但无分支） | **8**（1真+7 Bad End） | **多个**(取决于卡牌选择+DDL) |
| "if模式" | **学校模式**——非正史的"幻想" | 无 | 无 | 无 |
| 重玩的驱动力 | 学校模式(建立"不可能的好感") / 重新理解故事 | **几乎无**——单线通关=故事结束 | **重读**——带着真相重读→每句话新含义 | **重选**——不同的卡牌策略→不同的命运 |
| 失败= | Game Over(回避)——不是"另一个结局" | Game Over(回避) | **Bad End=叙事信息** | 死亡/疯狂(有时是"内容") |
