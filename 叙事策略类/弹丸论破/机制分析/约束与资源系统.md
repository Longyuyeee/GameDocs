# 约束与资源系统 — 弹丸论破

> 游戏版本: Danganronpa: Trigger Happy Havoc v2016 (PC/Steam)
> 数据来源: [Danganronpa Wiki](https://danganronpa.fandom.com/), [Steam社区指南](https://steamcommunity.com/)

## 一、"封闭学园"——物理约束

弹丸论破的最基本约束是**封闭空间**：希望之峰学园的所有出口被巨大的铁板封锁，窗户被金属板钉死。学生们不能离开——物理上被困在这个"暴风雪山庄"中。

| 约束层 | 规则 | 叙事功能 |
|------|------|------|
| **物理封闭** | 无法逃离学园 | "外部世界是否存在？"——不确定性的燃料 |
| **区域解锁** | 每章案件后解锁一个新楼层/区域 | 将探索转化为"进度奖励"——但新区域=新谋杀机会 |
| **监视** | 黑白熊通过摄像头监视一切——包括你在宿舍里的行为 | 隐私不存在——"他总是看着"——创造了持续的压迫感 |

## 二、黑白熊的"游戏规则"——规则约束

### 2.1 校规

| 规则 | 惩罚 |
|------|------|
| 杀害另一位同学 | **如果班级审判指认正确→你被处刑** / **如果指认错误→全员(除你外)被处刑，你毕业** |
| 不参与班级审判 | 全员被处刑 |
| 违反其他校规(如破坏监视摄像头) | 处刑 |
| 夜晚进入特定区域 | 警告→处刑 |
| 对黑白熊使用暴力 | 处刑 |

### 2.2 规则设计=叙事约束

黑白熊的规则设计让"强迫16个人自相残杀"成为可能——规则=**将"不杀人=大家一起被困到死"和"杀人=有机会逃出"放在一个天平上**。

## 三、调查约束——"你可能错过关键证据"

### 3.1 搜查篇的有限时间

| 调查约束 | 效果 |
|------|------|
| 必须在特定时间内完成调查 | 有限的时间=你不能调查一切 |
| 调查对象的"亮度" | 关键线索有视觉提示——但有些隐藏线索需要主动寻找 |
| 对话选择有限 | 与同学对话时有选项——不是所有信息都能在一次对话中获得 |

### 3.2 "遗漏言弹"的后果

如果调查时遗漏了某个关键线索→你没有它作为"言弹"→在审判中你可能无法击破某个关键矛盾→血量减少→更接近全员处刑的结局。

## 四、审判约束——"影响值"与时间压力

### 4.1 影响值(Influence)=血量的心理杠杆

| 机制 | 效果 |
|------|------|
| 影响值=5心(初始) | 你只有5次错误空间 |
| 每次错误射击→-1心 | **枪声=你的错误被听到——所有人都知道"你搞错了"** |
| 0心=全员处刑 | 焦虑累积——"我还有多少次机会？" |

### 4.2 时间约束

| 时间压力类型 | 机制 |
|------|------|
| **无休止辩论** | 发言不停滚动——你必须**在弱点消失前射击**(数秒内) |
| **机关枪论战** | 对方的辩解语句持续出现——锁定时间有限 |
| **审判总时间** | 没有硬性时间限制——但拖延=心理压力("下一个错误是不是更近了？") |

## 五、"同班同学=既是盟友也是敌人"——社交约束

### 5.1 认知负担

你(作为苗木)必须同时：
- **信任同学**(在调查中合作)
- **怀疑同学**(任何人可能是凶手)
- **保护同学**(错误指认→全员死亡)
- **谴责同学**(必须指出谁是"黑")——然后**看着他/她被处决**

### 5.2 情感约束

每次处刑后，幸存者人数减少——你失去了一个"朋友"(即使他/她是凶手——你们是"被逼的")。这种累积的失去感是弹丸论破最强大的情感约束。

## 六、与本品类其他游戏的约束系统对比

| 维度 | 弹丸论破 | 逆转裁判 | 这是我的战争 | 苏丹的游戏 |
|------|------|------|------|------|
| 核心约束 | **封闭空间+规则+影响值(审判血量)** | 惩罚(5次错误=Game Over) | **生存约束**(物资/生命/精神) | **DDL约束**(7天×卡牌) |
| 失败= | **全员死亡**(不指认/指认错误=大家一起被处刑) | 委托人被判有罪(Game Over) | 角色死亡/全员崩溃 | 死亡/疯狂/被处死 |
| 压迫感的来源 | **"你不仅要救自己——你要救所有人"** | "你要救你的委托人" | "你要救你的避难所" | "你要在7天内完成不可能的卡牌" |
| 空间约束 | **封闭学园**——暴风雪山庄 | 开放——可自由移动 | 避难所+夜间探索地点 | 叙境内空间(有限区域) |
