# 后果与反馈系统 — 十三机兵防卫圈

> 游戏版本: 13 Sentinels: Aegis Rim v2020 (PS4/PC, 2022)
> 数据来源: [13 Sentinels Wiki](https://13sentinels.fandom.com/), [Atlus官方](https://www.atlus.com/), [Steam社区](https://steamcommunity.com/app/1582030/)

## 一、"信息揭示"——最核心的后果与反馈

十三机兵的后果系统与传统游戏不同——核心"后果"是**你知道了什么**：

| 反馈类型 | 表现 |
|------|------|
| **即时反馈** | 你在一个角色的故事中发现了一个"碎片"——它连接了另一个角色故事中的"另一块碎片" |
| **延迟反馈** | 你5小时前玩的线索，在5小时后——**"原来那件事是这个意思"** |
| **meta反馈** | 你构建的"真相全景"——每个人拼出的图都不同 |

---

## 二、玩家的"真相拼图"

| 拼图状态 | 体验 |
|------|------|
| 初期(20%角色) | "我完全不知道在发生什么——但我被吸引了" |
| 中期(50%角色) | "开始看到连接了——A和B在同一个事件中——但他们的视角不同" |
| 后期(80%角色) | **"我拼出了真相——但还有一块关键的——最后那个角色"** |
| 100% | "全图——所有的碎片都在这里了" |

---

## 三、"恍然大悟"——十三机兵特有的反馈时刻

| 时刻 | 揭示了什么 |
|------|------|
| "原来1985年的XX和2025年的YY是同一个人" | 时间线连接 |
| "原来那个事件是XXX造成的" | 因果揭示 |
| "原来这个世界的本质是..." | **meta揭示——作品的核心真相** |

**与命运石之门的对比**: 命运石之门的真相揭示是"线性→高潮"——每个世界线揭示一部分真相。十三机兵的真相揭示是"网状→拼图"——你收集碎片，然后在某个时刻**所有碎片同时归位**。
