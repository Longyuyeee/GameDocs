# 多结局与重玩设计 — Suzerain

> 游戏版本: Suzerain v2.0 (PC/Steam)
> 数据来源: [Suzerain Wiki](https://suzerain.fandom.com/), [AV Club](https://www.avclub.com/suzerain-interview), [RockPaperShotgun](https://www.rockpapershotgun.com/suzerains-20-update-is-a-massive-overhaul-to-the-already-excellent-political-rpg)

## 一、9个主要结局

| 结局类型 | 核心判据 |
|------|------|
| 民主改革成功 | 宪法改革通过+连任+国家稳定 |
| 独裁巩固 | "我成了我曾经反对的人" |
| 政变推翻 | 军方/反对派推翻你 |
| 战争失败/胜利 | 外交立场→战争→结果 |
| 经济崩溃 | 改革失败+民意崩溃 |
| 被弹劾 | 失去议会支持 |
| 辞职/流亡 | 选择放弃权力 |
| 暗杀/死亡 | — |

## 二、重玩=不同的政治哲学

| 周目 | 目标 |
|:--:|------|
| 一周目 | "活过四年"——任何结局——"我第一次做总统——我尽力了" |
| 二周目 | "这次我要走不同的意识形态路线"——"上次我是改革派——这次我是独裁者" |
| 三周目 | "我要尝试疯狂的策略"——"我能同时得罪所有人——还能活下来吗？" |

## 三、DLC "Kingdom of Rizia"——国王视角

| DLC内容 | 与正传的差异 |
|------|------|
| 扮演国王(Romus Toras) | 而非总统——"君主制=不同的资源/约束/后果" |
| 家族政治 | "你的孩子=你的继承者——也是你的政治筹码" |
| 地缘政治扩展 | 更大范围的外交/军事博弈 |
