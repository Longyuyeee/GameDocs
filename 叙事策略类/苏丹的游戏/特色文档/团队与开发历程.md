### 1.2 团队背景

双头龙工作室的前身是**十字星工作室**（约200人规模，成立于2016年，曾开发《螺旋圆舞曲》等手游）。2023年底，在大厂裁员潮和行业寒冬中，团队缩减至约10人并更名为双头龙工作室，核心成员以极低成本启动了《苏丹的游戏》项目。这是一个"从大厂缩减到微型团队"的求生故事。

制作人远古之风的开发哲学被团队总结为一句话：**"冰箱里有什么就做什么"**——在极度有限的资源下，利用团队已有的叙事写作能力和对卡牌策略的熟悉度，创造一款"不需要昂贵3D美术、不需要复杂物理引擎"但"叙事深度远超AAA"的游戏。

## 2. 开发历程

| 时间 | 事件 |
|:--:|------|
| **2023年12月** | 双头龙工作室成立（从十字星工作室缩减而来）。初始概念：融合"一千零一夜"的卡牌策略×"道德恐怖"叙事。选择卡牌驱动是因为"我们没有钱做3D" |
| **2024年初** | 文本先行——团队先写出了核心故事框架和大量事件文本（做到"故事能自圆其说"），然后再设计游戏机制。这个创作顺序是"文本驱动设计"而非"机制驱动文本"的典型案例 |
| **2024年中** | 在摩点发起众筹。**10分钟内募资超50万人民币**，首周超27.5万美元。对几乎破产的团队来说，"这一次众筹让我们活下来了" |
| **2024年下半年** | 众筹成功后，团队得以专注开发。期间持续扩展文本量（从约100万字扩展到发售时的140万字）和事件系统 |
| **2025年3月31日** | Steam正式发售（$24.99）。48小时内售出**10万份**，登上全球热销榜 |
| **2025年4月** | 首周售出**25万份**（约$670万营收）。NGA/Bilibili/贴吧爆发式社区讨论 |
| **2025年7月** | 官方宣布**全球突破100万份**——2025年国产独立游戏最大黑马 |
| **2025年至今** | 持续版本更新，添加新事件、新角色线、新结局变体。文本量从140万字增长到约200万字 |

### 2.1 开发周期

- 从立项到发售：**约15个月**（2023年12月 - 2025年3月）

- 开发成本：极低——主要成本是人力和时间，没有昂贵的资产制作

## 5. 设计启示

### 5.1 对独立游戏开发者的启示

1. **"冰箱里有什么就做什么"** — 10人团队没有试图做3A规模的3D游戏，而是利用团队在叙事写作和系统设计上的核心优势，创造了"卡牌×海量文本"这一独特的品类融合。小团队的生存策略不是"做更多"而是"把已有的做到极致"。

2. **文本先行→机制后行** — 与传统"先设计机制再填充文本"的游戏开发流程不同，苏丹的游戏是先确保故事结构的完整性和深度，然后设计机制来支撑故事。这种"故事驱动开发"的方法论对叙事类游戏有普遍参考价值。

3. **众筹不只是钱——是验证** — 摩点众筹的爆发性成功不仅解决了资金问题，更重要的是验证了"卡牌×道德恐怖"这个概念的商业可行性。对独立团队来说，众筹首先是一个"概念验证工具"。

4. **"卡牌驱动"可以超越"收集式卡牌游戏"** — 卡牌在这里不是"战斗力数值"而是"道德选择的载体"。一个成熟的游戏品类被重新定义了——卡牌可以不只是Slay the Spire式的战斗工具，也可以是叙事载体。

### 5.2 对叙事策略品类的参考价值

1. **"机制即隐喻"的标杆** — 苏丹卡既是游戏机制（每周必须完成的任务），也是叙事隐喻（暴君的任意权力）。机制和隐喻的融合度是本品类的核心评价维度，苏丹的游戏在这个维度上达到了品类最高水平之一。

2. **文本量与叙事深度的关系** — 140万字+支撑了50+结局变体和多路线体验。证明"大量文本"是小团队可以通过"专注写作+放弃昂贵资产"做到的差异化竞争策略。

3. **"被迫的共谋"作为道德设计创新** — 这不是"好人vs坏人"或"选择A或B"——而是"你发现自己开始利用暴君的规则为自己谋利"。这是叙事策略品类中此前罕见的道德设计方向。

4. **中国独立游戏的全球化路径** — 以中文为原文，英文/日文翻译——中国的叙事游戏正在找到全球受众。苏丹的游戏的100万+销量证明了"非英语母语的叙事深度游戏"在全球市场可以成功。
