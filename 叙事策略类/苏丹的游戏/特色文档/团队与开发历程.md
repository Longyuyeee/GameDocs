# 团队与开发历程 — 苏丹的游戏

> 开发商: 双头龙工作室(Double Cross Studio, 杭州) | 发行商: 2P Games
> 发行年份: 2025年3月31日 | 销量: 100万+ (3个月)
> 数据来源: [触乐](http://www.chuapp.com/article/290581.html), [GameRes](https://www.gameres.com/911804.html), [TechNode](https://technode.com/2025/04/10/)

| 角色 | 说明 |

## 2. 开发历程

| 时间 | 事件 |
|:--:|------|
| 2023-2024 | 团队在几近破产状态下开发游戏。初始概念：一款融合"一千零一夜"和卡牌策略的"道德恐怖"游戏 |
| 2024 | 在摩点（中国众筹平台）发起众筹——10分钟内筹超50万人民币，首周超27.5万美元——**团队"活过来了"** |
| 2025年3月31日 | Steam正式发售($24.99)。48小时内售出**10万份**，登上全球热销榜 |
| 2025年4月 | 首周售出**25万份**($670万营收)。NGA/Bilibili/贴吧——中国玩家社区爆发讨论 |
| 2025年7月 | 发售3.5个月，官方宣布**全球突破100万份**——2025年国产独立游戏最大黑马 |

## 4. "一百万份"——从破产到活下来

触乐报道指出：在众筹成功之前，双头龙工作室**几乎破产**。一位团队成员说："如果众筹失败了，我们可能就散伙了。"游戏的成功不仅是商业故事——它是**一个极度小团队在无资源条件下做出品类创新产品并商业成功的教科书案例**。

## 5. 设计启示

1. **"卡牌驱动"可以超越"收集式卡牌游戏"** —— 卡牌在这里不是"战斗力数值"而是"道德选择的载体"。一个成熟的游戏品类被重新定义了
2. **"暴君是系统不是反派"** —— 苏丹的个人化程度极低，他是一个"规则"而非一个"角色"。这使游戏从"对抗一个具体坏人"升级为"对抗一个政治结构"
3. **10人团队+140万字=可行** —— 只要你设计的不是"单一大主线"而是"事件驱动的多模块剧本"，小团队也可以支撑海量文本
4. **"被迫的共谋"作为道德核心** —— 不是"好人vs坏人"，不是"选择A或B"——而是"你开始利用暴君规则为自己谋利"。这是此前故事游戏中罕见的道德设计
