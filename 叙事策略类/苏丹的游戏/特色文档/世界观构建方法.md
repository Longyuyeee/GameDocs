# 世界观构建方法 — 苏丹的游戏

> 游戏版本: Sultan's Game v2025.3 (PC/Steam)
> 数据来源: [触乐](http://www.chuapp.com/article/290581.html), [GameRes](https://www.gameres.com/911804.html), [RPGFan](https://www.rpgfan.com/review/sultans-game/), [Bilibili Wiki](https://wiki.biligame.com/sultansgame/)

## 一、世界观的"最小化"策略

### 1.1 一个宫殿装下整个世界

苏丹的游戏的世界观构建策略是**极度聚焦**——不是构建一个庞大的幻想世界，而是聚焦一个**宫廷**：

| 聚焦 | 包含的内容 | 刻意不包含的内容 |
|------|------|------|
| **空间** | 苏丹的宫廷（宫殿/后宫/花园/地牢） + 周边城市 | 广阔的国家/邻国/全球政治——世界"外面"是什么完全留白 |
| **人物** | 苏丹、妃子、近卫、祭司、商人、使者——约15-20个核心角色 | 农民/军队/外国使节/普通市民——宫廷之外的世界是模糊的 |
| **时间** | "现在"——暴君统治的当下 | 王朝历史/未来/其他时代 |
| **魔法** | 苏丹卡的魔法系统+黑暗仪式+宗教力量 | 更广泛的魔法世界/魔法学校/魔法生态 |

**设计意图**: "最小化"不是能力限制——是设计选择。通过极度聚焦宫廷，游戏让每个细节都承载多重叙事重量。一个地牢不只是"一个地牢"——它是在极度有限的场景列表中唯一代表"囚禁/惩罚"的空间，因此它的每一次出现都充满了意义。

### 1.2 "一千零一夜"作为世界观的脚手架

游戏不需要构建一个完整的世界——它借用了"一千零一夜"的文化框架：

| 借用元素 | 叙事功能 |
|------|------|
| **视觉风格** | 中东/阿拉伯风格——华丽的宫殿/头巾/弯刀/沙漠——立即可识别 |
| **权力结构** | 苏丹=绝对君主——不需要解释"为什么苏丹有绝对权力"，文化框架已经告诉你了 |
| **叙事传统** | "故事延迟死亡"的核心隐喻——Scheherazade的故事是所有人都知道的 |
| **魔法系统** | 一千零一夜中的魔法/精灵/诅咒——魔法卡牌在这个世界观中不显得突兀 |

## 二、信息布设方式：让玩家自己拼出世界

### 2.1 分层信息揭示

苏丹的游戏不会"告诉"你这个世界是什么样的——它让你通过以下方式自己拼出来：

| 信息源 | 揭示的世界观内容 | 揭示方式 |
|------|------|------|
| **苏丹卡的文字** | 苏丹的性格/宫廷的氛围/权力的性质 | 每张卡牌的简短文字是"世界观的碎片" |
| **角色的对话** | 角色之间的关系/宫廷政治/历史事件 | 对话中的暗示、潜台词、不完整的信息 |
| **地点描述** | 空间的功能/氛围/历史 | 每次进入新地点时的简短描述 |
| **事件结果** | 权力运作的方式/魔法的规则/社会的反应 | 你做的每件事，游戏展示的"发生了什么" |
| **探子报告** | 隐藏的政治关系/角色的秘密 | 探子是"主动获取被隐藏的世界观信息"的机制 |

### 2.2 "日常中的异常"——世界观的入侵

游戏中最有效的世界观构建时刻是**日常被异常打断**：

| 日常框架 | 异常入侵 | 世界观揭示 |
|------|------|------|
| "这周抽卡消卡——和往常一样" | 抽到了一张你从未见过的金卡 | "苏丹卡的系统比你以为的更庞大" |
| "和妃子普通的互动" | 她突然提到"苏丹最近不太对劲" | "苏丹的行为是有原因的——这个原因可能与魔法有关" |
| "普通的消卡流程" | 你的探子报告"有个角色在秘密见另一个人" | "宫廷里有你不知道的政治网络" |

**与命运石之门的"日常中的异常"对比**:
- 命运石之门: 前半的日常被"第一条D-Mail改变了世界线"打断——异常是外部的
- 苏丹的游戏: 日常被"你开始发现卡牌系统比你想象的要深"打断——异常来自你对系统的更深理解

## 三、魔法系统的叙事化设计

### 3.1 魔法规则=叙事规则

苏丹的游戏的魔法系统（苏丹卡+黑暗仪式+宗教力量）不是"一个独立的系统"——它的规则直接映射到叙事：

| 魔法规则 | 叙事映射 |
|------|------|
| 苏丹卡必须服从——不服从=死亡 | 权力不可逃避——在独裁下你没有"选择退出"的选项 |
| 四种卡牌=四种道德维度 | 权力有不同的面貌（杀戮/纵欲/奢靡/征服）——每一种都在腐蚀你 |
| 品级递进=道德代价递进 | 越大的权力（更高的品级）需要越大的道德牺牲 |
| 黑暗仪式的"代价" | 任何超越凡人的力量都有代价——且代价不由你选择 |

### 3.2 密教路线——魔法的终极揭示

密教神路线是唯一完整揭示"苏丹卡魔法系统的真正来源"的路线。这个设计很重要：**不同的路线揭示了世界观的不同层面**。

| 路线 | 揭示的世界观层面 |
|------|------|
| **逃亡** | 苏丹统治的边界——"宫廷之外还有世界" |
| **谋反** | 苏丹统治的内部结构——"近卫是怎样构成的，权力是怎样运作的" |
| **正教神** | 苏丹统治的宗教维度——"苏丹的权力是否有神学的根基？" |
| **密教神** | **苏丹卡的真正来源**——"这套魔法系统是谁设立的？目的是什么？" |

**设计启示**: 多路线不只是"多结局"——每条路线是**世界观的一个观察视角**。完整理解游戏的世界观需要体验多条路线。这是叙事策略品类的核心设计技法之一。

## 四、世界观的"留白"

### 4.1 刻意不解释的内容

苏丹的游戏世界观构建的艺术不仅在于"解释了什么"，也在于"没有解释什么"：

| 留白内容 | 叙事效果 |
|------|------|
| **苏丹的过去**——他为什么变成了这样？ | 不可解释的恶比可解释的恶更可怕 |
| **宫廷之外的详细地理** | 你的整个世界就是这个宫廷——和你的处境一样被局限 |
| **苏丹卡系统的终极起源** | 即使在密教神路线中，魔法系统的"最初来源"也是模糊的 |
| **结局后的世界** | "你推翻了苏丹——然后呢？"的答案由你自己补充 |

### 4.2 留白的叙事功能

留白不是"信息不足"——它是设计选择。当关键信息被留白时，玩家的想象力填充了最符合自己价值观的解释。这种"协同创作"让每个玩家对苏丹的世界的理解都是个人的。

## 五、与其他叙事策略游戏的世界观构建对比

| 维度 | 苏丹的游戏 | 极乐迪斯科 | 命运石之门 | 海市蜃楼之馆 |
|------|------|------|------|------|
| 世界观规模 | **极小**——一个宫廷 | **小**——一个城市（马丁内斯） | **中**——东京+世界线 | **小**——一座洋馆（跨越时间） |
| 信息布设 | 分层揭示——不同路线揭示不同层面 | 开放式探索——玩家决定揭示顺序 | 线性揭示——章节制逐步展开 | 严格线性——按时间顺序揭示 |
| 世界观与机制的关系 | **机制=世界观的具象化**——卡牌是魔法道具 | **机制=世界观的内部化**——24技能是世界观的声音 | **机制=世界观的操作**——D-Mail改变世界线 | N/A——纯线性无机制 |
| 留白策略 | 高留白——很多关键信息刻意不解释 | 中留白——大量描写但保留推理空间 | 低留白——SciADV世界观完整解释 | 低留白——完整解释一切 |
