# 角色弧线与动机设计 — 苏丹的游戏

> 游戏版本: Sultan's Game v2025.3 (PC/Steam)
> 数据来源: [Bilibili Wiki](https://wiki.biligame.com/sultansgame/), [NGA](https://bbs.nga.cn/read.php?tid=44128735)

## 一、主角：无名的大臣

### 1.1 "空白面具"设计

与逆转裁判（成步堂有名字和性格）、命运石之门（冈部有中二病人格）不同——苏丹的游戏中的主角是一个**半空白角色**。他有背景故事（怒斥暴君而被惩罚的大臣），但性格和道德立场完全由**玩家的卡牌选择**来决定。

| 维度 | 设计方式 | 叙事故意 |
|------|------|------|
| 名字 | 无预设名字 | 你是"任何被暴君惩罚的人" |
| 道德立场 | 由善恶名累积决定 | 你不是"好人"或"坏人"——你是在做选择中成为"什么样的人" |
| 与角色的关系 | 由对话和行为选择构建 | 每个随从对你的态度是你"赚来的" |

### 1.2 从"受害者"到"权力玩家"

主角的弧线遵循一个通用的叙事模板（具体由玩家选择填充）：

| 阶段 | 状态 | 触发 |
|:--:|------|------|
| 1. 受害者 | "为什么是我？我只是说了真话。" | 游戏开始——被苏丹惩罚 |
| 2. 适应者 | "好的...我学会了规则。我可以活下去。" | 完成最初几张低品级卡牌 |
| 3. 权力觉醒 | "等等——这张杀戮卡可以解决我自己的问题。" | 第一张"用作工具的苏丹卡" |
| 4. 抉择 | "我继续做暴君的工具——还是做暴君的终结者？" | 中后期——路线分歧点 |
| 5. 终点 | 5条路线的5种人格终点 | 取决于路线选择+善恶累积 |

## 二、关键配角

### 2.1 梅姬(妻子)——道德温度的"标尺"

| 功能 | 说明 |
|------|------|
| **道德锚点** | 她对你的纵欲/杀戮行为有越来越明确的不满——是玩家"道德温度"的最直接反馈 |
| **逃亡线的起点** | 第一张苏丹卡被消除后——她提议"我们逃吧" |
| **信任的度量计** | 她的态度变化=玩家道德变化的可视化仪表 |

### 2.2 苏丹四妃——游戏中的"政治纵欲网络"

| 妃子 | 品质 | 关键道具/功能 | 叙事角色 |
|------|:--:|------|------|
| **金妃莎姬** | 金 | 权力游戏的继承人任务线 | "最高层的政治棋子" |
| **银妃萨达尔尼** | 银 | **出城信物**(逃亡线必需) | "出逃的钥匙" |
| **铜妃安苏亚** | 铜 | 铜妃剧情线 | 中层政治关系 |
| **石妃法德耶** | 石 | 法里斯撕标签关键 | 底层棋子暗藏关键功能 |

**设计意图**：每个妃子=一张政治地图上的节点。纵欲卡不是为了"性"——是为了部署你的政治网络。最关键的银妃（出城信物）对应银品质——"不高不低的品质对应最关键的逃生工具"。

### 2.3 四近卫——谋反的"四道锁"

| 近卫 | 撕标签条件 | 设计含义 |
|------|------|------|
| 奈布哈尼 | 奴隶购买事件/比美事件(送织女) | 最"简单"——代表忠诚可以被利益收买 |
| 哲巴尔 | 侠名>5+黑街识破身份+拳斗 | 代表需要"侠义行为"来赢得忠诚 |
| 法里斯 | 赌马→赌狗→权势>9→调查投毒 | 代表需要"政治智慧"来瓦解 |
| 塞里曼 | 银妃系列任务后饶命 | 代表需要"仁慈"来转化敌人 |

## 三、反派的特殊性

苏丹不是传统"反派"——他是**"系统"**：

| 维度 | 分析 |
|------|------|
| **可见性** | 苏丹几乎不出场——他的存在由"卡牌"和"每周的处刑威胁"来体现 |
| **个性化** | 他没有"动机"——他无聊、残忍、有绝对权力。他不是一个需要被"理解"的反派 |
| **对抗方式** | 你不是"战胜"苏丹——你是**在苏丹的系统中找到漏洞、利用漏洞、最终颠覆系统** |
