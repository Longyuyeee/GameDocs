# 约束与资源系统 — 苏丹的游戏

> 游戏版本: Sultan's Game v2025.3 (PC/Steam)
> 数据来源: [Bilibili Wiki](https://wiki.biligame.com/sultansgame/), [NGA攻略](https://bbs.nga.cn/read.php?tid=44128735)

## 一、约束系统概述

苏丹的游戏的约束设计是"外部暴力强制+内部资源稀缺"的双层结构。最外层是**苏丹的绝对权力**——你不能拒绝抽卡、不能拒绝执行、不能逃跑（初期）。内层是**有限的回合、金币、人力和时间**。

## 二、外部约束：苏丹的绝对权力

### 2.1 "7天倒计时"——生存的钟摆

| 设计要素 | 机制 | 叙事效果 |
|------|------|------|
| **抽卡不可拒绝** | 每周苏丹必然发一张新卡——你无法说"这周我不想玩" | 暴君的任意性——"你的意愿不存在" |
| **卡牌类型随机** | 你不知道下周会是什么类型——可能连续三周杀戮卡 | 不确定性焦虑——"我准备的是错的" |
| **品级影响消除难度** | 金卡可能需要牺牲高层角色——石卡轻松解决 | "有的周是地狱，有的周是喘息" |
| **逾期=死亡** | 没有任何例外——7天没消卡=处刑动画 | **终极约束**——你的一切行动都在死亡阴影下 |

## 三、内部约束：资源管理

### 3.1 五大核心资源

| 资源 | 获取方式 | 消耗方式 | 短缺后果 |
|------|------|------|------|
| **金币** | 治理家业(每回合掷骰≥3→5金币)、事件奖励、交易 | 购买书籍、触发事件、满足奢靡卡、扩建 | 无法推进事件、无法购买情报、奢靡卡无法消除→死亡 |
| **人力(随从)** | 事件招募、主动邀请、剧情解锁 | 每个随从每回合只能分配到一个位置 | 人力资源不足→无法同时处理多个危机 |
| **情报** | 浴场(日常固定来源)、事件、间谍活动 | 提供额外骰子重试次数、解锁特定事件 | 检定成功率下降→关键事件失败 |
| **声望(善恶)** | 道德选择——杀人=恶名/侠义之举=善名 | 善名≥20触发净化之火(一次性不限品质销卡)；恶名过高触发暗杀 | 善名不足→关键"安全阀"不可用；恶名过高→随从背叛 |
| **魔力** | 传家护甲(+2)、特定事件、命运商店 | 谋反结局必须魔力≥20 | 魔力不足→谋反路线不可行 |

### 3.2 金币的绝对稀缺性

金币是苏丹的游戏中最紧张的资源：
- **每回合至少需要3金币**（维持基本运转）
- 治理家业是唯一稳定的金币来源——但需要掷骰
- 掷骰检定：每颗骰子1/3概率成功——可能连续失败
- **结果**：你可能被困在"没钱→无法推进→无法获得更多钱"的死循环中

## 四、卡牌作为"可转化"的约束

苏丹的游戏最独特的约束设计是：**卡牌本身是可以被"创造性使用"的资源。**

| 约束表面的含义 | 你的转化策略 |
|------|------|
| "你必须杀人" | → 你杀的是政敌→**获得了政治利益** |
| "你必须挥霍金币" | → 你用来扩建住宅→**获得了长期金币产出** |
| "你必须纵欲" | → 你与妃子建立关系→**获得了政治网络和关键道具(如出城信物)** |
| "你必须征服" | → 你招募了部队→**建立了对抗苏丹的军事力量** |

## 五、时间约束：回合的稀缺

| 时间维度 | 约束 |
|------|------|
| **每日回合有限** | 每天只能做有限的动作——"今天去治理家业还是去浴场收集情报？" |
| **每周必须消卡** | 7天内必须完成至少一张苏丹卡——"我还有时间处理其他事吗？" |
| **全局回合有限** | 游戏有总体时间框架——"如果我想谋反，我需要在苏丹发现之前积累足够的部队和魔力" |

## 六、与本品类其他游戏的约束对比

| 维度 | 苏丹的游戏 | 这是我的战争 | 命运石之门 |
|------|------|------|------|
| 外部约束 | **暴君的意志**（政治权力） | 战争的物质稀缺（物理现实） | 世界线收束（物理法则） |
| 核心资源 | 金币/随从/情报/声望 | 食物/药品/材料/武器 | 36字节信息/48小时 |
| 约束能否被"利用" | **是**——卡牌是"被迫的武器" | 否——资源只能消耗 | 是——D-Mail改变世界线 |
| 终极压力的根源 | "苏丹会不会发现我在谋反？" | "我的角色还能撑多久？" | "我能不能欺骗世界本身？" |
