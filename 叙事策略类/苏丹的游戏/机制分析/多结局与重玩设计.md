# 多结局与重玩设计 — 苏丹的游戏

> 游戏版本: Sultan's Game v2025.3 (PC/Steam)
> 数据来源: [Bilibili Wiki](https://wiki.biligame.com/sultansgame/), [3DM](https://www.3dmgame.com/gl/3965562.html)

## 一、结局系统总览

苏丹的游戏拥有约**50+结局变体**，分布在**5条主要路线+2条特殊路线**中。结局判定采用**"路线条件+微调变量"**模型：路线条件决定你进入哪个大方向，微调变量(善恶名、随从存活、选择记录)决定结局的细节版本。

### 1.1 五大路线+二特殊

| # | 路线 | 核心条件 | 苏丹的结局 | 玩家位置 |
|:--:|------|------|------|------|
| 1 | **谋反** | 撕4近卫标签+3支部队+魔力≥20 | 可杀/囚/辱/流放 | **新苏丹(或新宰相)** |
| 2 | **逃亡** | 开门方法+选定目的地+善名 | 苏丹继续执政 | **自由的逃亡者** |
| 3 | **屠龙** | 阿迪莱线+部队+道具 | 可能被龙杀死/苏丹继续执政 | **英雄/烈士** |
| 4 | **正教神** | 伊曼+神圣烙印+大猎捕 | 苏丹可能皈依 | **先知** |
| 5 | **密教神** | 拜玲耶+恶之华+混沌之王 | **苏丹变成小宠物** | **更黑暗的力量掌控者** |
| 6 | **全折卡** | 14天内消除所有卡 | 苏丹继续——但你已被女术士"收编" | **游戏赢家(新的循环开始)** |
| 7 | **深渊** | 恶名≥30+邪恶化身 | 苏丹继续 | **四种黑暗结局之一** |

### 1.2 谋反路线的微调变量

在"成功谋反"这个大方向下，根据细节选择产生不同结局：

| 微调变量 | 变化 |
|------|------|
| **苏丹的处置方式** | 杀/囚/辱/流放——每种产生不同的政治后果和结局描述 |
| **新王的身份** | 谁成为新苏丹？——你/某个贵族/某个随从？ |
| **改革vs传统比例** | 你如何处理旧苏丹的体制？——激进改革/保守维护 |
| **善恶名** | 善名主导=仁君结局；恶名主导=你成了新的暴君 |

## 二、重玩设计

### 2.1 "命运点数"——跨周目继承

| 继承机制 | 说明 |
|------|------|
| **命运点数(Fate Points)** | 通过成就系统("一千零一夜")获取——每完成特定里程碑获得点数 |
| **命运商店** | 使用命运点数购买跨周目优势：传家护甲(+2魔力)、金妻子升级、初始金币加成、秘密双胞胎 |
| **设计意图** | 不是"让游戏更简单"——是**"给你更多选择但不变更游戏核心难度"** |

### 2.2 重玩的核心驱动力

| 驱动力 | 说明 |
|------|------|
| **不同路线=完全不同的故事** | 谋反线是中世纪政治惊悚；逃亡线是绝望的求生；密教线是黑暗奇幻恐怖 |
| **卡牌随机性=每次不同的策略** | 不同顺序的卡牌迫使不同的资源分配策略 |
| **角色关系网的重组** | 不同选择导致不同的随从存活/背叛/关系状态 |
| **"如果当初..."的好奇心** | "如果我没有杀那个人会怎样？"——50+结局提供了足够的"如果" |

## 三、与本品类游戏的结局对比

| 维度 | 苏丹的游戏 | 这是我的战争 | 命运石之门 |
|------|------|------|------|
| 结局数量 | 50+变体 | 无定数(每个角色个性化尾声) | 6条明确路线 |
| 判定机制 | 路线条件+微调变量 | 累积行为→系统生成 | 6个FLAG精确条件 |
| "最好"结局 | **不存在**——每条路线都有代价 | **不存在**——即使最好的结局也带着战争创伤 | 是——Steins;Gate(1.048596%) |
| 跨周目继承 | 命运点数+命运商店 | 无(每次全新随机) | 无(每次全新开始但6路线可逐条解锁) |
