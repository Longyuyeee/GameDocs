# 结局条件数据总览 — 苏丹的游戏

> 游戏版本: Sultan's Game v2025.3 (PC/Steam)
> 数据来源: [Bilibili Wiki](https://wiki.biligame.com/sultansgame/), [3DM](https://www.3dmgame.com/gl/3965562.html)

## 一、五大路线结局条件速查

| 路线 | 必要条件 | 推荐准备 | 苏丹命运 | 结局变体数 |
|------|------|------|------|:--:|
| **谋反** | 撕4近卫标签 + 3支部队 + 魔力≥20 | 魔戒+生命权杖+秘密双胞胎 | 杀/囚/辱/流放(4选1) | 20+ |
| **逃亡** | 开门方法(3选1) + 目的地(2选1) + 善名 | 银妃信物/大炸弹/钻地机 | 苏丹继续执政 | 10+ |
| **屠龙** | 阿迪莱线 + 部队 + 道具 | 白犀牛事件不冲突 | 可能被龙杀死 | 5+ |
| **正教神** | 伊曼 + 神圣烙印 + 大猎捕 + 匕首之路 | 神面升到3-4级 | 苏丹可能皈依 | 5+ |
| **密教神** | 拜玲耶 + 恶之华 + 混沌之王 | 黑暗集会全通 | **苏丹变成宠物** | 5+ |

## 三、谋反路线苏丹处置选项

| 处置方式 | 条件 | 后果 |
|------|------|------|
| **杀** | 无特殊条件 | 最直接——但可能引发内战 |
| **囚** | 需要特定建筑 | 政治妥协——部分旧势力保留 |
| **辱(凌辱)** | **需要生命权杖** | 极端羞辱——极大权力但极大恶名 |
| **流放** | 无特殊条件 | 折中方案——苏丹可能卷土重来 |

## 五、各结局达成难度

| 结局 | 难度 | 原因 |
|:--:|:--:|------|
| 逃亡 | ★★☆☆☆ | 条件最简单——"逃离"是最"安全"的选择 |
| 谋反 | ★★★★☆ | 需要撕4标签+部队+魔力——多线并行 |
| 屠龙 | ★★★☆☆ | 单线推进——但需要避免冲突事件 |
| 正教神 | ★★★★☆ | 需要多次检定——失败可能锁死 |
| 密教神 | ★★★★★ | 需要高恶名+全黑暗集会——道德代价最高的路线 |
