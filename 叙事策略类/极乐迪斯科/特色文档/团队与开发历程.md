# 团队与开发历程 — 极乐迪斯科

> 开发商: ZA/UM (爱沙尼亚) | 核心创作者: Robert Kurvitz (创意总监/世界观), Aleksander Rostov (艺术总监)
> 发行年份: 2019 (原版) / 2021 (Final Cut) | 平台: PC (Steam/GOG)/PS4/PS5/Xbox/Switch
> 数据来源: [PC Gamer](https://www.pcgamer.com/how-disco-elysium-was-made/), [GamesIndustry.biz](https://www.gamesindustry.biz/disco-elysium-the-framerate-of-thought), [Disco Elysium Wiki](https://discoelysium.fandom.com/)

## 1. 开发团队

### 1.2 团队规模与背景

| 指标 | 说明 |
|------|------|
| 核心团队 | 约 35 人 |
| 背景 | 爱沙尼亚塔林(Tallinn)——来自"前苏联最小的共和国" |
| 核心成员来历 | Kurvitz 和 Rostov 最初是一个艺术/音乐团体"ZA/UM"的成员——游戏工作室就是从文化团体演变为商业公司的 |
| 资金来源 | 最初自筹资金——爱沙尼亚小型投资人的支持。后来获得更多投资 |

### 1.3 "一个被拒绝了 10 年的世界观"

Robert Kurvitz 在青少年时期就创造了 Elysium 世界。他花了 10 年时间试图让这个世界"被看到"：
- 18 岁时写了一个桌游设定（没人感兴趣）
- 之后写了一本 Elysium 世界观的小说《Sacred and Terrible Air》(2013 年出版，爱沙尼亚语)（销量平平）
- 2015 年前后决定将世界观做成电子游戏——因为"没有人会读小说，但有人会玩游戏"
- 2019 年《极乐迪斯科》发售——**他的世界观终于在 14 年后被世界看到了**

## 2. 开发历程

| 时间 | 事件 |
|:--:|------|
| 2015年左右 | ZA/UM 从文化团体转型为游戏工作室，开始开发 Disco Elysium (当时项目名不叫这个) |
| 2016 | 开发中——游戏原计划使用 Unity，后改为自己的引擎(基于开源技术) |
| 2017-2018 | 核心开发期——游戏在内部迭代中逐渐成形:从传统 RPG 到"无战斗全对话"的转变 |
| 2019年10月15日 | **正式发售**——PC(Steam/GOG) |
| 2020 | 获得 The Game Awards 4项大奖(Best Narrative/Best RPG/Best Indie/Fresh Indie) |
| 2021年3月30日 | **Final Cut**发售——全语音版。Lenval Brown一人配音50万字 |
| 2022年底 | 核心创作者(Kurvitz/Rostov/Hindpere)与公司管理层发生法律纠纷——被迫离开 ZA/UM |
| 2023-2025 | 多个原团队成员成立新工作室(Longdue/Dark Math/Summer Eternal等)——"精神续作"宣布但正宗续作的前景不明 |

### 2.1 "从传统 RPG 到无战斗叙事"的转型

游戏最初设计包含战斗系统。但在开发过程中，团队发现了**技能系统**的核心趣味——"技能在你脑子里说话"远比"战斗"更独特。团队做出了决定：**砍掉所有战斗**，纯对话/检定/叙事。这不是"缺失"——是"聚焦"。

### 2.2 引擎与工具

| 工具 | 说明 |
|------|------|
| **Unity (最终使用)** | 游戏最终使用 Unity 引擎——非自定义引擎 |
| **Ink/自定义对话系统** | 对话树的规模(120万字)需要专门的工具链 |
| **油画风艺术(Photoshop)** | Rostov 的"油画像"风格——每个 NPC 都是一个独立的油画肖像 |

## 4. ZA/UM 的法律纠纷——"失去了自己创造的世界"

2022年底，Kurvitz、Rostov和Hindpere因与公司管理层的法律纠纷被迫离开 ZA/UM。他们失去了 Elysium 世界的知识产权。多个原团队成员后来成立了独立工作室(Longdue/Dark Math/Summer Eternal)，宣布新的"精神续作"项目——但他们无法继续在 Elysium 世界中创作。

**这对独立游戏开发者的启示**：**IP 所有权的法律结构**对创作者至关重要——如果你失去了你创造的世界，你就失去了一切。

## 5. 设计启示

1. **"被拒绝了 10 年的世界观"最终成功**——Kurvitz 从 18 岁起创造的 Elysium 世界，在桌游/小说都失败后，通过游戏这个媒介"被看到"。**媒介很重要——同一个内容在不同媒介中可能有完全不同的命运。**

2. **"砍掉战斗"不是损失——是聚焦**——最初有战斗系统的原型并不有趣。"技能在脑子里说话"才是核心。**找到你的游戏最独特的东西——然后砍掉所有不是那个的东西。**

3. **"失败=内容"的革命性设计哲学**——在 RPG 史上，这可能是最重要的设计创新之一。"检定失败"不是"你不够好"——而是"这是 Harry 人生的另一面"。

4. **35 人团队×120万字×5年=可能**——证明了小团队+深厚世界观+长开发周期可以产生品类革命级的作品。但 IP 法律保护必须跟上。

5. **"政治的叙事化"**——将政治意识形态做成 Harry 的可选人格属性，而不是"游戏告诉你什么是对的"。**政治在游戏中的意义不是"立场"——是"角色定义"。**
