## 三、检定公式

```
检定结果 = 2d6 + 技能等级 + 修正值

特殊结果:
- 2 (Snake Eyes/蛇眼, 1+1) = 自动失败(无论技能多高)
- 12 (Double Six/双六, 6+6) = 自动成功(无论DC多高)
```

## 四、检定类型对比

| 特征 | 白色检定 | 红色检定 |
|------|:--:|:--:|
| 失败后可重试 | **是**——升级技能后可重试 | **否**——一次机会，永不可逆 |
| 失败后果 | 通常是轻微/暂时的——错过这一段内容 | **永久性**——叙事分支不可逆地改变 |
| 分布 | 约 90% 的检定 | 约 10% 的检定 |
| 视觉 | 白色背景 | 红色背景 |
| 数量(估计) | 200+ | ~10 个关键节点 |
| 典型 | "试着理解这幅画" | "抓住逃跑的 Ruby" |

## 五、思维内阁数据（53个思维）

### 5.1 获取方式分类

| 获取方式 | 数量(估) | 示例 |
|------|:--:|------|
| 对话中获得 | ~35 个 | Sorry Cop, Apocalypse Cop, Hobo Cop |
| 发现/探索获得 | ~10 个 | Mazovian Socio-Economics, Jam Rock Shuffle |
| 事件触发获得 | ~8 个 | Finger on the Eject Button |

### 5.2 思维内化数据

| 参数 | 数值 |
|------|:--:|
| 思维总数 | **53 个** |
| 同时可内化的思维 | **最多 4 个** |
| 已内化的思维槽位 | **最多 12 个** |
| 内化所需游戏内时间 | **不等**——通常 1-6 小时游戏内时间 |
| 内化时可装备的数量 | **4 个** (剩余的存储在思维柜中) |
| 遗忘思维 | 可花费 **1 技能点** 遗忘已内化的思维 |

### 5.3 思维效果分类

| 效果类型 | 示例 | 说明 |
|------|------|------|
| **属性加成** | +1 Empathy | 永久性技能值提升 |
| **解锁新选项** | 解锁特定政治/对话选项 | 新增被动检定机会 |
| **世界观改变** | 改变 Harry 看世界的方式 | 某些 NPC 的反应改变 |
| **负效果** | -1 Logic + 非线性思维 | 一些思维有tradeoff |
