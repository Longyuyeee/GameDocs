## 二、Kim好感度系统

Kim是游戏中唯一有明确"好感度"系统的角色——但他的好感度不是通过数字显示，而是通过他的反应和对话感知：

| Kim的反应 | 好感度状态 | 触发行为 |
|------|:--:|------|
| "侦探——这是个好想法。" | 高——他尊重你 | 专业行为/聪明推理/道德正直 |
| 沉默+taking notes | 中——他在观察 | 尚可的调查/有些奇怪但可以接受 |
| "这...这不是警察的做法。" | 低——他在质疑你 | 愚蠢行为/种族主义/腐败 |
| "我不认为我应该继续和你合作。" | 极低——他受够了 | 持续的低能/彻底的道德失败 |

## 三、技能作为"角色"——内在的关系网络

游戏中最独特的"角色关系"不是NPC之间的——而是**24个技能在你脑海中的"对话关系"**：

| 技能关系动态 | 示例 |
|------|------|
| **盟友技能** | Logic + Rhetoric: "他说的有道理——你有反驳吗？" |
| **对立技能** | Electrochemistry vs Volition: "再喝一杯" vs "你已经够了" |
| **竞争技能** | Drama vs Empathy: "她在说谎" vs "她在害怕" |
| **互补技能** | Perception + Visual Calculus: "你看到了弹壳" + "弹道从那个窗口来" |
