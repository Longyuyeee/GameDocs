## 二、NPC 与 Harry 的技能互动

| NPC | 与哪些技能的互动最多 |
|------|------|
| Kim Kitsuragi | Esprit de Corps, Empathy, Authority, Suggestion |
| Klaasje | Drama, Empathy, Volition, Logic——**几乎全技能被触发** |
| Evrart Claire | Rhetoric, Authority, Empathy, Suggestion, Composure |
| Joyce Messier | Encyclopedia, Logic, Rhetoric, Composure |
| Cuno | Esprit de Corps(作为警察的本能), Empathy, Authority |
| Measurehead | Encyclopedia, Rhetoric, Authority, Composure |
| The Deserter | **Rhetoric**——与他讨论"革命为什么失败"——以及根据你的政治立场不同的对话 |

## 三、主要 NPC "命运"变量

| NPC | 结局中的可能状态 |
|------|------|
| Kim | 继续信任 Harry / 推荐 Harry 回归警队 / 失去对 Harry 的尊重 |
| Klaasje | 逃脱司法审判 / 被捕 / 不知所踪 |
| Cuno | 被引导远离毒品 / 继续堕落 / **与 Harry 成为"搭档"**(如果 Kim 受伤) |
| Ruby | 逃脱 / 被捕 / (特定路径中死亡) |
| The Deserter | 必定被捕——但与 Harry 的最后对话因政治立场而异 |
| Jean Vicquemare | 对 Harry 的评价取决于整个游戏的行为 |
