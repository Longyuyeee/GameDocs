# 谋杀调查主线剧情 — 极乐迪斯科

> 游戏版本: Disco Elysium Final Cut v2021 (PC/Steam)
> 数据来源: [Disco Elysium Wiki](https://discoelysium.fandom.com/), [fayde.co.uk](https://fayde.co.uk/), [Post45学术分析](https://post45.org/2024/04/)

## 一、案件发生：枪击（游戏开始前数天）

### 1.1 关键人物与动机

| 角色 | 身份 | 与案件的关系 |
|------|------|------|
| **Ellis "Lely" Kortenaer** (死者) | 奥兰治雇佣兵上校，Wild Pines(野松资本)派遣来破坏码头罢工 | 被枪杀者 |
| **Klaasje Amandou** | 自称"Oranje迪斯科舞者"的年轻女性——实际是逃亡的企业间谍 | 死者死前与她发生性关系；策划了尸体伪装 |
| **Iosef Dros** (逃兵) | 前Revachol公社政委，在海上堡垒藏身43年 | **真正的枪手**——一枪从海上堡垒射穿Kortenaer后脑 |
| **Ruby** | 码头工会同情者，货车司机，技术专家 | 执行了尸体伪装的技术部分 |
| **Titus Hardie** | 码头工人民兵领袖 | 带领Hardie Boys参与尸体伪装 |

### 1.2 完整时间线

```
【枪击前数小时】
- Kortenaer在Whirling-in-Rags旅馆与Klaasje吸毒+发生性关系
- 逃兵通过狙击镜监视Klaasje的窗户(他痴迷地监视她数周)
- 竹节虫的信息素影响了逃兵的精神状态(加剧了偏执和暴力倾向)

【枪击时刻】
- Kortenaer和Klaasje在她房间(面朝大海的大窗户)
- 逃兵从海上堡垒一枪射穿Kortenaer后脑，子弹从嘴巴穿出
- Kortenaer当场死亡
- Klaasje被溅血——她以为子弹是冲自己来的

【掩盖：枪击后立即】
- Klaasje没有报警——她害怕暴露自己作为企业间谍的身份
- 她求助Ruby(工会同情者)
- Ruby加入了掩盖行动
- Titus Hardie和Hardie Boys也被叫来
- 他们伪造现场：将Kortenaer尸体挂在旅馆后院的树上
- 给尸体重新穿上盔甲(掩盖枪伤)
- 制造"私刑处决"的假象

【掩盖后至游戏开始】
- 当地居民(包括Cuno)发现了树上的尸体
- 有人剥走了尸体上的贵重盔甲(如T-500护甲)
- RCM第41分局接到报告
```

## 二、调查阶段：逐日推进

### Day 1：失忆与尸检

| 事件 | 关键检定/选择 | 揭示信息 |
|------|------|------|
| Harry在Whirling-in-Rags旅馆房间醒来——完全失忆，只穿内裤 | 与"远古爬行动物脑"和"边缘系统"的对话——可以选择拒绝醒来（开场1分钟内Game Over） | 建立核心设定：Harry失去了所有记忆 |
| 遇到Kim Kitsuragi——第57分局中尉，临时搭档 | Kim的自我介绍——他显然在"看管"这个崩溃的同僚 | Kim是道德锚点——他对Harry的第一印象取决于你的行为 |
| 检查后院悬挂的尸体 | Visual Calculus检定：分析尸体位置、盔甲、伤口 | 初步判断：多人私刑？但盔甲穿着方式不对——是死后穿上的 |
| 与Garte(旅馆老板)对话 | 他告诉你Harry欠130雷亚尔(旅馆费+打碎的东西) | 建立金钱约束+Harry的自我毁灭背景 |
| 询问Cuno(向尸体扔石头的街头少年) | 他骂骂咧咧但提供了"有人听到了枪声"的信息 | 第一层矛盾：如果是私刑，为什么会有枪声？ |

### Day 2：证人网络

| 事件 | 关键检定/选择 | 揭示信息 |
|------|------|------|
| 询问Hardie Boys(工会民兵) | Authority: 让他们承认参与了"某些事" | 他们声称Kortenaer试图强奸Klaasje——所以被"处决" |
| 与Evrart Claire(工会领袖)会面 | 他给你各种"任务"——包括让一位罢工工人"消失" | 揭示了码头罢工的政治背景 |
| 与Joyce Messier(Wild Pines谈判代表)会面 | Encyclopedia: 她揭示Kortenaer的身份——雇佣兵上校 | 确认：尸体不是普通工人——是职业军人 |
| 询问Klaasje | **关键对话**——Empathy发现她在害怕；Drama发现她在说谎的迹象 | **核心转折**：她不是强奸受害者——Kortenaer是双方自愿的性行为中被杀的 |

### Day 3-4：真相浮现

| 事件 | 关键检定/选择 | 揭示信息 |
|------|------|------|
| 弹道分析——子弹轨迹指向海上堡垒 | Visual Calculus+Perception: 发现子弹入射角 | 枪手不在旅馆附近——在远处 |
| Klaasje的深度审讯 | **"你是企业间谍吗？"** ——她最终承认：她是逃亡的Oranje企业间谍(Klaasje Amandou是假名)，害怕被雇主"清理" | **核心揭示**：掩盖不是为了包庇强奸者——是为了阻止RCM追查到她 |
| 追踪Ruby | 她躲藏在废弃的Feld大楼中 | Ruby是掩盖行动的技术执行者——她设计了悬挂尸体的装置 |
| Ruby的对峙 | **红色检定**——她极度恐惧，认为你是腐败警察来杀她的。她嘴里藏了一颗毒药 | 要么说服她放下枪(Rhetoric)，要么她自杀 |
| 发现"灰域发射装置" | Ruby持有——这个装置是导航灰域的 | 连接到了更深的Elysium世界观 |

### Day 5-6：法庭(Tribunal)

*这是游戏中最关键、不可逆转的场景*

| 事件 | 关键检定/选择 | 后果 |
|------|------|------|
| 回到Whirling-in-Rags——撞上Hardie Boys与剩余雇佣兵的对峙 | **无法回避的剧情事件** | 一场三方对峙：工会民兵 vs 失控雇佣兵 vs RCM |
| 枪战爆发 | Hand/Eye Coordination——射击雇佣兵；Composure——保持冷静 | **Harry中弹，昏迷数天**。Kim可能中弹受伤或安全——取决于你的选择 |
| **如果Kim中弹** | 他被送进医院——**Cuno自愿成为Harry的临时搭档** | 游戏的情感核心被撕裂——你的"道德锚点"消失了 |
| **如果Kim安全** | 他继续作为搭档 | 你和Kim的关系在枪战后进入新阶段 |

### Day 7+：海上堡垒与最终对峙

| 事件 | 关键检定/选择 | 揭示信息 |
|------|------|------|
| 与Kim(或Cuno)前往海上堡垒 | Physical Instrument: 爬过铁索链 | 发现逃兵的藏身处——43年的孤居 |
| 与Iosef Dros(逃兵)对峙 | **游戏中最核心的对话** | **完全自白**：他杀了Kortenaer。动机=(1)对资本主义象征的意识形态仇恨+(2)对Klaasje的性嫉妒+(3)竹节虫信息素+43年孤独导致的疯狂 |
| 逃兵揭示：他曾是Revachol公社的政委 | Rhetoric: "革命失败了——但你还在杀人？" | **游戏最深的悲剧**：逃兵=被背叛的革命理想+43年的精神腐烂 |
| **竹节虫(Phasmid)出现** | Inland Empire检定——与竹节虫进行心灵对话 | **游戏的情感终点**：竹节虫揭示(1)它影响了逃兵的头脑(2)灰域与人类思维同时存在(3)"从你的废墟中走出来，寻找平静" |
