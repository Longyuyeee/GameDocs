# 约束与资源系统 — 极乐迪斯科

> 游戏版本: Disco Elysium Final Cut v2021 (PC/Steam)
> 数据来源: [Disco Elysium Wiki](https://discoelysium.fandom.com/), [fayde.co.uk](https://fayde.co.uk/)

## 一、约束系统概述

极乐迪斯科的约束与任何其他叙事游戏都不同——**没有战斗系统、没有生存压力、没有倒计时DDL**。它的约束有三层：Harry的失忆(认知约束)、有限的技能点(成长约束)、以及时间-金钱的轻微压力(游戏节奏约束)。

## 二、认知约束：Harry的失忆

### 2.1 失忆的叙事功能

Harry Du Bois在游戏开始时完全失忆——不记得自己是谁、在哪里、为什么穿着一条内裤倒在旅馆地板上。这个设定是极乐迪斯科最根本的约束：

| 失忆=什么功能 | 说明 |
|------|------|
| **世界构建工具** | Harry的"无知"=玩家的"无知"——通过Encyclopedia等技能向玩家介绍Elysium这个世界 |
| **人格重构工具** | Harry不记得自己是谁→**玩家通过24个技能的配置+对话选择来"构建"Harry的人格** |
| **道德空白的起始** | Harry没有任何预设的道德立场→每个玩家的Harry在政治和道德上是不同的 |
| **"The Expression"** | 在镜子中看到自己的脸时，Harry意识到自己有一种"悲伤的表情"——这是他过去的一部分，但不记得为何悲伤 |

### 2.2 失忆的"边界"

Harry的失忆是有边界的——某些肌肉记忆(Harry曾经是一名非常优秀的警察)和能力(枪械使用)仍然存在。失忆清除的是**身份**，不是**能力**。

## 三、成长约束：技能点

### 3.1 技能点的稀缺性

Harry在游戏开始时分配12点技能点(4个属性各1基础+8点可分配)。之后的技能点获取：

| 来源 | 获取量 | 说明 |
|------|:--:|------|
| 每次升级 | 1点 | 通过对话和完成检定获取XP→升级 |
| 思维内阁完成 | 不定 | 某些思维提供被动技能加成 |
| 药物/酒精 | 临时+1到特定属性 | 有副作用——可能影响其他检定 |

**设计意味**：你无法将所有技能升满——必须选择"你是什么样的侦探"。一个高Intellect+Psyche的Harry和一个高Physique+Motorics的Harry看到的世界完全不同。

## 四、时间与金钱：轻微的节奏约束

### 4.1 金钱

| 金钱来源 | 数量 | 说明 |
|------|:--:|------|
| 典当物品 | 不等 | 向当铺老板出售捡到的物品 |
| 捡空瓶(Tare) | 10分/瓶 | Martinaise各处散落可回收瓶子 |
| 任务报酬 | 20-100雷亚尔 | 各种委托 |
| 向Kim借钱 | 不定 | 只有特定场景可借 |
| **你的旅馆账单** | **欠Garte 130雷亚尔(旅馆费+打碎的东西)** | 不一定是必须支付的——但有后果 |

金钱压力是轻微的——游戏不会因为没钱而Game Over，但没钱限制了住宿选择和与某些NPC的互动。

### 4.2 时间推进

时间是极乐迪斯科最重要的**潜性约束**：

| 时间如何推进 | 说明 |
|------|------|
| 对话中 | 每段对话推进微小的游戏内时间 |
| 阅读/思考 | 阅读书籍、内化思维会推进时间 |
| **凌晨2点后时间停滞** | 2AM后时间不再推进——防止玩家错过时间窗口 |

某些事件在特定时间内发生（如每天的特定NPC对话变化），但总的来说时间约束比《这是我的战争》宽松得多。重点是**节奏**而非**压力**。

## 五、与本品类其他游戏的约束对比

| 维度 | 极乐迪斯科 | 这是我的战争 | 命运石之门 |
|------|------|------|------|
| 核心约束 | **认知(失忆)+技能稀缺** | 物资稀缺+时间 | 世界线收束(物理法则) |
| 时间压力 | **极低**(主要为节奏) | 中(冬天逼近) | 无(但循环有限) |
| "失败"的含义 | **内容解锁** | 角色死亡/资源损失 | 进入特定世界线 |
| 资源类型 | 技能点/雷亚尔/时间 | 食物/药品/材料/武器 | 36字节信息/48小时 |
| 约束的情感效果 | "我是谁？我能成为谁？" | "我能活下去吗？" | "我能改变命运吗？" |
