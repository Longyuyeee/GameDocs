# 选择与分支架构 — 极乐迪斯科

> 游戏版本: Disco Elysium Final Cut v2021 (PC/Steam, 原版2019)
> 数据来源: [Disco Elysium Wiki](https://discoelysium.fandom.com/), [fayde.co.uk](https://fayde.co.uk/), [ZA/UM官方](https://discoelysium.com/)

## 一、"技能即选择"——叙事选择的革命

极乐迪斯科的选择系统与传统叙事游戏完全不同。这里**没有"A/B选择菜单"**。游戏用24个技能（Skills）作为24个**在你脑海中说话的"角色"**——每个技能有自己的个性、语调、价值观。当你在对话中时，这些技能会主动"插话"（Passive Checks），提供不同的选项和解读。

### 1.1 技能如何成为"选择者"

| 技能介入方式 | 机制 | 示例 |
|------|------|------|
| **被动检定** | 技能等级达标时自动触发，技能"对你说话" | Encyclopedia(博学多闻): "实际上，Klaasje这个名字在Oranje语中的意思是..." |
| **主动检定(白色)** | 对话中的可重试检定——2d6+技能+修正值 vs DC | "试着说服Kim让你喝这瓶酒"——失败可升级后重试 |
| **主动检定(红色)** | **不可重试**——一次机会，永久改变叙事 | "抓住跳跃中的Ruby"——失败=Ruby逃脱 |
| **技能互相对话** | 两个技能同时激活时，它们可能在你的脑海中**互相争论** | Logic vs Drama: "他说的是真话" vs "他在演戏" |

### 1.2 技能作为"24个角色"

每个技能有不同的**个性**(personality)和**语调**(tone)。Lenval Brown在Final Cut中为每个技能配了微妙不同的声音风格：

| 属性组 | 技能 | 人格 | 在Harry脑海中的"声音" |
|:--:|------|------|------|

| | Rhetoric | 辩论家 | "这正是我们需要打破的叙事结构" |
| | Drama | 戏剧女王 | "我的老爷！这是多么精彩的表演！" |
| | Conceptualization | 艺术家 | "那个雕塑...它让我想起了一种特定的灰蓝色" |
| | Visual Calculus | 法医分析师 | "脚印间距2.5米——跑步者，不是散步者" |

## 二、骰子系统

### 2.1 基础公式

```
检定结果 = 2d6 + 技能等级 + 修正值(衣物/思想/药物/状态)

vs 难度等级(DC: Difficulty Class)
```

| DC | 名称 | 含义 |
|:--:|------|------|
| 6-8 | Trivial/Medium | 日常对话选项 |
| 9-11 | Challenging | 需要投入技能点和/或辅助 |
| 12-14 | Formidable | 需要专门的技能投资+修正值 |
| 15-18 | Legendary/Heroic | 几乎不可能——需要极高的技能+多重修正 |
| 19-20 | Impossible | 理论最大值——需要全部条件完美配合 |

### 2.2 双六面骰的特殊结果

| 结果 | 条件 | 含义 |
|:--:|:--:|------|
| **Snake Eyes(1+1)** | 两个骰子都是1 | 自动失败——不管技能多高 |
| **Double Six(6+6)** | 两个骰子都是6 | 自动成功——不管DC多高 |

## 三、"失败即内容"——叙事分支的核心哲学

极乐迪斯科最重要的选择设计原则是：**检定失败不是"游戏惩罚"——它是"内容解锁"**。许多最精彩的对话和故事内容只在失败时触发：

| 案例 | 成功的结果 | 失败的结果 |
|------|------|------|
| 试图高五Kim | 尴尬的高五 | **Kim说:"请不要再这样做了"**——这是建立他们关系最真实的时刻之一 |
| 试图说服Cuno合作 | Cuno勉强同意 | Cuno对你竖中指——但你看到了他墙上的涂鸦，揭示了隐藏信息 |
| 试图抓住吊死者的靴子 | 成功取下尸体 | **失败→Harry呕吐——但呕吐后他获得了+1 Physical Instrument的buff(因为"清理了系统")** |
| 在镜子前审视自己 | 正常 | **失败→看到"Expression"——那个悲伤的表情，Harry意识到自己已经崩溃了** |

### 3.1 为什么失败更值得设计？

Robert Kurvitz的哲学是：在传统RPG中，失败=错过。但在现实中，"失败"往往是定义一个人的时刻。"检定失败"在极乐迪斯科中是一个**叙事工具**——不是惩罚。它让Harry变成一个"会失败的人"，而不是"拥有技能的英雄"。

## 四、与本品类游戏的对比

| 维度 | 极乐迪斯科 | 命运石之门 | 这是我的战争 |
|------|------|------|------|
| 选择机制 | 24个技能作为"内心声音"进行选择 | Phone Trigger(手机回复) | 行为即选择(无菜单) |
| 失败处理 | **失败=内容解锁** | 失败=进入特定世界线 | 失败=角色死亡/崩溃/资源损失 |
| 技能系统 | 24个"会说话的技能" | 无 | 12+角色各有特长 |
| 分支数量 | 天文数字(每场对话数十个技能检定) | 6条预设路线 | 涌现式(无数微选择) |
