# 角色弧线与动机设计 — 海市蜃楼之馆

> 游戏版本: The House in Fata Morgana v2016 (PC/Steam)
> 数据来源: [Bilibili剧情解析](https://www.bilibili.com/read/cv26114249/), [萌娘百科](https://moegirl.icu/)

## 一、米歇尔——"不被承认的灵魂"

### 1.1 最根本的设定

**米歇尔天生拥有女性的身体，但内心认同为男性。** 这个设定不是"附带属性"——它是整个故事的核心。

| 要素 | 分析 |
|------|------|
| **家人的反应** | 家人否认他的心理性别——他被囚禁、虐待、被强制"成为女性" |
| **被送到馆中** | 因为"不正常"——被当作异类放逐 |
| **在馆中的十年** | 独自一人，只有莫尔加娜（魔女）的声音在耳边——但她试图诱使他诅咒世界。他拒绝了。 |
| **与吉赛儿的相遇** | 起初用匕首比喻（言语）伤害她——因为他不相信任何人会接受真实的他 |
| **被接受的时刻** | **吉赛儿是第一个真正接受"米歇尔=男性"的人**——不是"宽恕"，是"我从未觉得需要宽恕什么" |

### 1.2 他的弧线

| 阶段 | 状态 | 转变 |
|:--:|------|------|
| 否认 | 家人否认他的性别身份——他被摧毁了 | "我是不应该存在的人" |
| 愤怒 | 用言语伤害吉赛儿——因为"如果我最真实的一面被看到了，我就会被抛弃" | 自我保护 = 主动推远 |
| 接受 | 吉赛儿接受了他——不是"你终于正常了"，是"你一直很正常" | "原来有人能看到我——真正的我" |
| 牺牲 | 米歇尔被教会骑士团杀害 | 他的死不是"悲剧结局"——是他的"完整"被承认后的终结 |
| 回归 | 终章在现代与吉赛儿重逢——**"无论时光荏苒，无论你的颜姿将化作何形，我终将与你相遇"** | 千年轮回在此闭合 |

## 二、吉赛儿——"等待了千年的人"

### 2.1 她的牺牲

| 牺牲 | 内容 |
|------|------|
| **第一次牺牲** | 被米歇尔的父亲侵犯——被"流放"到馆中 |
| **第二次牺牲** | 回到馆中，相信米歇尔——两人相爱 |
| **第三次牺牲** | 米歇尔被杀——她与莫尔加娜签订契约：以永生换取等待米歇尔再生 |
| **千年的代价** | 1603年、1707年、1869年——她每次都找到了转世的白发少女（莫尔加娜的圣女人格），但每次都不被"认出"。**数百年的孤独让她逐渐失去自我——从"等待米歇尔的吉赛儿"变成"冷漠旁观的女仆"** |

### 2.2 "讲述假故事"的动机

当吉赛儿在第四章讲述"被粉饰的童话版本"时——她不是在"骗玩家"。她是**在骗自己**。真实的记忆太痛苦——"米歇尔经历了那些"对她来说不可承受——所以她创造了一个"更干净的版本"。

## 三、莫尔加娜——"被背叛的圣女"

### 3.1 从圣女到魔女

| 阶段 | 内容 |
|:--:|------|
| **圣女** | 无偿用自己的血救人——她的血液有治愈力量 |
| **被母亲出卖** | 母亲将她交给领主——为了换取金钱和地位 |
| **被囚禁在瞭望塔** | 被放血制药——她的身体成为了"资源" |
| **被三个人背叛** | 亚麻色青年（梅尔前世）假装温柔接她出逃→却将她交给"异国野兽"（幸正前世）→最终被"领主"（雅各布前世，曾救过她的青年）因野心囚禁 |
| **孤独死去** | 在庆典之日无人所知地死亡——她的诅咒从此降下 |

### 3.2 "白发少女"=莫尔加娜的"善良面"

莫尔加娜在疯狂前分离出了自己的善良人格——创造了一个"圣女人格"的分身：白发少女。这个分身被吉赛儿误认为是"米歇尔的转世"，在三个时代中出现——但每次都没有记忆。这是莫尔加娜最悲惨的部分：**即使是"魔女"，也有一部分自己渴望被爱、被记住。**

## 四、三个男人——加害者也是受害者

| 角色 | 对莫尔加娜的伤害 | 他为什么这样做 | 他自己的悲剧 |
|------|------|------|------|
| **梅尔**（1603年） | 前世：假装温柔→背叛 | 被母亲的秘密（白发少女=私生女妹妹）压垮 | 爱上一个"不应该爱"的人→被真相摧毁 |
| **幸正**（1707年） | 前世：将莫尔加娜抓回 | **天生具有无法抑制的杀戮欲望**——不是"邪恶"，是"诅咒" | 唯一能抑制他的白发盲女被村民杀死→他血洗村庄 |
| **雅各布**（1869年） | 前世：因野心囚禁莫尔加娜 | 他想"用铁路为她建设更好的世界"——但从未说出口 | 玛利亚（复仇者）挑拨+自己的沉默→失去一切 |

## 五、与本品类其他游戏的对比

| 维度 | 海市蜃楼之馆 | 极乐迪斯科 | 命运石之门 |
|------|------|------|------|
| 主角类型 | **无选择"见证者"**——玩家不控制主角 | 构建型(24个技能构建Harry的人格) | 有性格主角(冈部) |
| 核心关系 | **米歇尔↔吉赛儿**——"千年等一回" | Harry↔Kim——"看着我崩溃但从不放弃" | 冈部↔红莉栖——"我在你所在的地方" |
| 反派的本质 | **历史中的背叛**——不是一个人，是权力结构的累积 | 意识形态+灰域(遗忘) | 物理法则(世界线收束) |
| "悲剧"的来源 | **命运的不可改变**——悲剧已经发生，你在"理解"它 | 遗忘(灰域)——"被历史抛弃的东西会消失" | **选择的代价**——拯救A=在某世界线失去B |
