# 故事结构与章节设计 — 海市蜃楼之馆

> 游戏版本: The House in Fata Morgana v2016 (PC/Steam)
> 数据来源: [Bilibili剧情解析](https://www.bilibili.com/read/cv26114249/), [萌娘百科](https://moegirl.icu/), [豆瓣](https://www.douban.com/review/15537723/)

## 一、整体结构："门"作为叙事框架

海市蜃楼之馆采用**"门"的框架**——每扇门=一个时代的一段悲剧。表面上是4扇独立的"门"，实际上暗藏了一层"颠倒"的真假结构。

### 1.1 章节总览

| 章节 | 时代 | 地点 | 核心人物 | 叙事功能 | 真实/虚假 |
|:--:|:--:|------|------|------|:--:|
| 序章 | "现在" | 馆中暖炉前 | 你/女仆 | 建立框架——"请务必抓紧下仆的手" | 真实 |

| **第二扇门** | 1707年 | 西班牙(瓦伦西亚) | 幸正(野兽)/白发盲女/保琳 | **第二重悲剧**——人性与兽性 | 真实（但信息不完整） |

### 1.2 "真假四扇门"——最核心的叙事诡计

游戏最精妙的结构设计：**第四章（雾海之馆）是一个"假故事"。** 女仆吉赛儿在讲述前四扇门时，故意将第四章中的人物身份、性别、发色和事件"改写"了。她将真正的故事（第六章）隐藏在了"看起来像童话"的假故事后面。

| 假故事（第四章） | 真故事（第六章） |
|------|------|
| 白发少女"吉赛儿"遇到白发青年"米歇尔" | 黑发少女吉赛儿遇到米歇尔（女性身/男心） |
| 米歇尔为保护她而死 | 米歇尔被教会骑士团杀害 |
| "触之即死"的浪漫诅咒 | **真实的暴力**——吉赛儿被米歇尔的父亲侵犯，被"流放"到馆中 |
| 童话般的悲剧 | **真实的、非浪漫化的恐怖** |

**为什么这样设计？**

| 层面 | 分析 |
|------|------|
| **女仆的心理** | 吉赛儿讲述"童话版"是因为真实的记忆太痛苦——她把自己的创伤"改写成"了更可接受的版本 |
| **对玩家的误导** | 前三扇门的"真实感"让玩家相信第四扇门也是真实的——当第六章揭示真相时，冲击力加倍 |
| **叙事主题** | "人们如何讲述自己的创伤"——吉赛儿的假故事不是谎言，是"自我保护" |

## 二、"馆"作为统一叙事空间

海市蜃楼之馆本身是一个**跨时代的不变场景**——从1099年到1869年再到现代，这座被诅咒的馆一直存在。馆不是"背景"——它是**"容器"**：容纳了所有时代的悲剧、所有灵魂的轮回、所有未竟的对话。

## 三、悲剧的嵌套结构

### 3.1 "三层嵌套"的悲剧

```
最外层：莫尔加娜的悲剧（被背叛的圣女→化为魔女诅咒世界）
    │
    ├── 中层：三个男人的悲剧（梅尔/幸正/雅各布——三人前世今生都参与了伤害莫尔加娜）
    │
    └── 核心：米歇尔与吉赛儿的悲剧（跨越千年的分离、误解、重逢）
```

### 3.2 "迟来的理解"——叙事结构的情感机制

游戏的每一扇门都是"信息的延迟释放"——玩家先看到的是一个"坏人"（梅尔背叛妹妹、幸正屠杀村庄、雅各布囚禁妻子），然后通过后续章节理解他们为什么变成了那样：

| 角色 | 第一印象 | 后来的理解 |
|------|------|------|
| 梅尔 | 懦弱的背叛者，抛弃妹妹 | 被母亲的秘密和禁忌之爱撕裂的少年——无法承受"白发少女是私生女妹妹"的事实 |
| 幸正 | 嗜血的野兽，杀害未婚妻 | 有着无法控制的先天杀戮欲望，白发盲女是他唯一的"锁链"——当锁链被村民杀死，他崩溃了 |
| 雅各布 | 冷漠的资本家，囚禁妻子 | 沉迷铁路建设是因为想给妻子更好的生活——但他从未"说出口"。玛利亚的挑拨+自己的沉默=悲剧 |
