# 叙事视角与信息控制 — 海市蜃楼之馆

> 游戏版本: The House in Fata Morgana v2016 (PC/Steam)
> 数据来源: [Bilibili剧情解析](https://www.bilibili.com/read/cv26114249/), [萌娘百科](https://moegirl.icu/), [VNDB](https://vndb.org/w16768)

## 一、三层嵌套的叙事视角

海市蜃楼之馆的叙事视角是它最复杂的设计维度之一。游戏在不同层面上运作着三个叙事层次：

### 1.1 视角层次图

```
┌─────────────────────────────────────────┐
│ 第一层：「你」（玩家/主角/一无所有的灵魂） │ ← 被叙述者，也是"敲门的人"
│   ↑ 女仆对你说话                          │
├─────────────────────────────────────────┤
│ 第二层：女仆（吉赛儿）                     │ ← 主要叙述者
│   ↑ 但女仆在第四章"说谎"                   │
├─────────────────────────────────────────┤
│ 第三层：各时代主角的第一人称体验            │ ← 故事内人物视角
│   ↑ 但女仆在选择性地呈现这些视角             │
└─────────────────────────────────────────┘
```

### 1.2 "你"——一个空白的容器

游戏的叙述设计从第一句话开始就确立了视角："你在一座馆中醒来，什么都不记得。"

| 视角特征 | 设计意图 |
|------|------|
| **无记忆、无身份、无面孔** | "你"是一个容器——可以容纳任何人的故事进入 |
| **女仆直接对"你"说话** | 打破第四面墙——"你"=玩家。女仆不是在对"角色"说话 |
| **"请不要松开下仆的手"** | "手"=连接的隐喻——"你"通过女仆的"手"连接到故事 |

## 二、女仆——不可靠叙述者的七层伪装

### 2.1 吉赛儿为什么是一个"不可靠叙述者"

| 不可靠的层面 | 表现 | 原因 |
|------|------|------|
| **身份隐瞒** | 在大部分游戏中，女仆不揭示自己的真实身份 | 她的身份揭示（吉赛儿=女仆）是游戏的核心叙事揭示之一 |
| **信息选择性** | 只展示每扇门的"表面悲剧" | 她选择不透露三扇门共享同一个"白发少女"灵魂的事实 |
| **有意篡改** | 第四章将真实故事改写为童话版本 | 自我保护的叙事——她无法直接面对真实创伤 |
| **情感控制** | 女仆的叙述语调始终保持冷静 | 但在某些时刻语调崩溃——暴露了"叙述者也是有情感的人" |
| **对"你"的态度矛盾** | 一方面邀请你理解——另一方面阻止你"知道太多" | "如果你知道了全部真相，你还会留在这里吗？" |
| **沉默** | 在某些关键问题上选择不回答 | 沉默="现在还不是时候"——信息发布的时间控制 |
| **自我欺骗** | 她一定程度上"相信"自己讲述的假故事 | 不是"恶意欺骗"，是"自己也无法承受真相" |

### 2.2 与典型不可靠叙述者的对比

| 维度 | 海市蜃楼之馆 | 极乐迪斯科（Harry的技能） | 《Fight Club》 |
|------|------|------|------|
| 不可靠的来源 | **情感创伤**（无法面对真相→改写） | **药物/酒精/精神崩溃** | **精神分裂**（不知道自己是两个人格） |
| 对读者的态度 | **保护性欺骗**——"我不想让你看到太痛的东西" | 技能互相对话——"你甚至不相信你自己" | 隐瞒——"你自己也不知道" |
| "欺骗"的善意性 | **高**——她的谎言是"温柔" | 不适用（Harry自己也是混乱的） | 低——"你应该知道真相" |

## 三、信息控制——"何时知道什么"

### 3.1 信息发布的时间线

| 章节 | 新增信息 | 保留的信息 |
|:--:|------|------|
| 门一至三 | 每扇门是一个"独立悲剧" | 三个白发的"同源"事实被隐藏 |
| 门四 | 女仆讲述的"童话版本" | **第四章是被篡改的**——这个事实被隐藏 |
| 第五章 | **第一个选择节点出现** | 为什么"这里"有选择？——因为这是假的 |
| 第六章 | **真实的第一零九九年**——推翻第四章 | 女仆的身份、米歇尔的身份 |
| 第七章 | 以前三扇门的女仆视角回顾 | 魔女的起源 |
| 第八章 | **莫尔加娜的起源** | 米歇尔的过去 |
| 第九章 | **米歇尔的过去与性别认同** | — |
| 终章 | — | — |

### 3.2 "信息控制"的核心原则：永远比玩家多想一层

| 原则 | 实例 |
|------|------|
| **给的信息都是真的——但不完整** | 前三扇门的每个悲剧都是真实的——但"为什么这些人会相遇"被隐藏 |
| **假信息包裹在真信息中** | 第四章的假故事被夹在三扇"真门"和一扇"真门"之间——利用格式的惯性 |
| **关键信息在玩家"准备好"时才释放** | 米歇尔的性别认同直到第九章——在玩家已经"认识"并"在乎"了米歇尔之后 |

## 四、Meta 叙事——第四面墙的裂缝

### 4.1 直接对玩家说话

女仆从未说她在对"一个游戏角色"说话——她一直在对**你**说话。严格来说，海市蜃楼之馆从头到尾都是一个**持续的第四面墙打破**。"你"=玩家坐在屏幕前，"下仆的手"="请继续阅读"。

### 4.2 文字的 Meta 时刻

在某些场景回放中，正常阅读的文本被白块替代——出现了"不在正常阅读流程中的文字"。这是文本本身在告诉玩家：**"有些信息被从叙事中删除了。"** 这不仅是叙事中的角色不知道的信息——这也是玩家看不到的信息。

## 五、与本品类其他游戏的视角设计对比

| 维度 | 海市蜃楼之馆 | 逆转裁判 | 命运石之门 | 极乐迪斯科 |
|------|------|------|------|------|
| 叙述者 | **不可靠的第一人称叙述者**（女仆） | 第三人称（玩家=成步堂的"手"） | **固定第一人称**（冈部的内心独白） | **24个技能=24个叙述声音** |
| "你"是谁 | "一无所有的灵魂"=**你本人** | 成步堂龙一=你控制他 | 冈部伦太郎=你体验他的世界 | Harry Du Bois=你构建他的人格 |
| 第四面墙 | **从头到尾都在打破**（女仆一直在对"你"说话） | 偶尔打破（"异议あり！"的系统交互） | 几乎不打破（沉浸式第一人称） | **彻底打破**（技能知道你"在玩游戏"） |
| 信息不对称 | **叙述者vs.玩家**——叙述者知道更多但不说 | 玩家vs.证人——证人说谎，玩家找破绽 | **世界线vs.玩家**——世界线收束=系统的"旨意" | 技能之间的互相矛盾——"真相"不存在 |
