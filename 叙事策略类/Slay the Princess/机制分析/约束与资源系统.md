# 约束与资源系统 — Slay the Princess

> 游戏版本: Slay the Princess v2023 (PC/Steam)
> 数据来源: [Slay the Princess Wiki](https://slay-the-princess.fandom.com/), [Black Tabby Games开发者访谈](https://www.blacktabbygames.com/), [Steam社区](https://steamcommunity.com/app/1989270/)

## 一、唯一的"资源"——你的选择

Slay the Princess **没有传统的资源系统**——没有金钱/生命/物品/技能。唯一的"资源"是**你的选择次数**：

| "资源" | 约束 |
|------|------|
| **选择机会** | 每个循环节点有2-5个选择——一旦决定，不可回退 |
| **时间** | 无DDL——但**每一次拖延=公主对你的看法在改变** |
| **刀** | 带刀/不带刀——**一个选择改变了一切** |

---

## 二、叙述者——"规则的源头"（也是你的对手）

叙述者是 Slay the Princess 中最独特的"约束"——他不是"告诉你规则的人"——他是**规则本身**：

| 叙述者的功能 | 表现 |
|------|------|
| **命令** | "去杀公主。这是你的任务。" |
| **限制** | 他描述的世界=你能看到的世界——**他"不说"的东西你无法知道** |
| **评判** | 当你偏离"杀公主"的路径——他的声音从冷静→焦虑→愤怒→崩溃 |
| **谎言** | 叙述者**不一定在说真话**——但他控制着"你知道什么" |

---

## 三、空间约束——一室一林

| 你能去的地方 | 叙事意义 |
|------|------|
| **树林** | 过渡——你在这里与叙述者对话——"为什么我要杀公主？" |
| **小屋** | 核心场景——公主在地下室里——你选择如何接近她 |
| **地下室** | 公主的所在——"她就是一切——整个游戏都是关于'进入那个地下室'" |

**设计意图**: 限制空间=放大选择。如果在开放世界中，你的选择被稀释到100个小决定中。在一间小屋里，**每一个选择都是重大的**。

**与Papers, Please的对比**: Papers, Please 的空间约束=检查亭（你被困在体制的位置上）。Slay the Princess 的空间约束=小屋（你被困在"与她的关系"中）。两者都用空间放大叙事张力。
