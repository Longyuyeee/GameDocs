# 谜题与线索系统 — 人狼村之谜

> 游戏版本: Raging Loop v2017 (PC/Steam)
> 数据来源: [Raging Loop Wiki](https://ragingloop.fandom.com/), [豆瓣深度分析](https://www.douban.com/review/15144917/), [NGA评测](https://bbs.nga.cn/read.php?tid=22433268)

## 一、"狼人杀=活谜题"——不是推理，是博弈

人狼村之谜的谜题不是"谁是凶手"——而是**"如何在不可能的条件下存活"**：

| 谜题类型 | 内容 |
|------|------|
| **身份谜题** | "谁是狼？"——但每轮轮回中身份**随机变化**——"上一轮的狼=这一轮的村民" |
| **规则谜题** | "规则外杀人"如何运作？——"谁在打破规则——为什么？" |
| **轮回谜题** | "为什么会有轮回？谁在触发轮回？"——最关键：**主角不是轮回的触发者** |

---

## 二、核心反转——"你不是轮回者"

| 反转 | 揭示 |
|------|------|
| 你以为 | 主角房石阳明=轮回者——能力=回到过去——"因为我是主角" |
| **实际** | **千枝实=真正的轮回者**——主角只是"被卷入她的轮回"。主角的"死亡回溯"=千枝实死了——她的轮回触发了——主角的记忆被保留 |

**设计精妙之处**: 这是对"轮回系动画/游戏"主角中心主义的颠覆——"你以为你是轮回的主角——你只是她的'追随者'"。
